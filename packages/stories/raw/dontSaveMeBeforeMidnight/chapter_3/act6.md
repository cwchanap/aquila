# 第六幕：十七分鐘

**旁白**：監控室在行政樓地下半層，要往下走一段很窄的樓梯。牆壁上的油漆剝落了大半，露出底下灰色的水泥。空氣裡有一種潮濕的電子設備味——塑膠外殼受熱之後散發的那種微甜的焦味，混著地下水的霉。

**旁白**：邵叔帶他下來的時候說了一句：「你跟著就好，不用說話。」

**旁白**：顧言沒有說話。但他的眼睛，一直在動。

**顧言** [determined]：(內心)監控室。鐘樓的攝像頭在這裡。裝卸區的攝像頭在這裡。整個校園的眼睛，都在這幾面螢幕上。

**旁白**：監控室不大，大概十來坪。三面牆上掛著九面螢幕，型號新舊混雜——左邊三台是較新的液晶螢幕，畫面清晰，色彩正常；右邊三台是舊式的映像管螢幕，畫面偏綠，邊緣有一圈暗角；中間三台混合著雜訊和切換後的藍屏。

**旁白**：桌前坐著一個年輕人，穿著深藍色的技術志工背心，胸前別著一張列印的臨時工作證。他正在一台舊筆電前操作，手指在鍵盤上飛快地敲著，但螢幕上的畫面進展得很慢。

**邵叔** [serious]：麻煩調一下鐘樓和裝卸區今天的紀錄。從早上六點開始。

**技術人員**：好，我調一下⋯⋯系統今天比較慢，大雨一直讓它切備援線。你們等一下。

**旁白**：技術人員的聲音帶著一種忙到麻木的平靜。他一邊操作一邊解釋，像是在做一份他已經重複了很多次的工作。

**旁白**：螢幕上的畫面開始跳。鐘樓的鏡頭切到裝卸區，又跳到南翼走廊，然後整個黑了一秒，再亮起來的時候變成了校門口的畫面。技術人員罵了一聲，又重新切回鐘樓。

**技術人員**：今天雨太大，舊系統一直在主線和備用線之間切。一切線就要重新同步緩衝，那段時間看不到即時回放。

**顧言**：(內心)切線。緩衝。看不到。

**顧言**：多久？

**旁白**：他沒有要追問的意思。只是一個短句，平得像在問今天星期幾。但他的眼睛盯著技術人員的螢幕，一眨不眨。

**技術人員**：這次⋯⋯十七分鐘。

**顧言** [shocked]：(內心)十七。

**顧言** [shocked]：(內心)十七分鐘。

**旁白**：那三個字卡在他的腦子裡，甩不掉。

**顧言**：(內心)第一輪。鐘樓停電前後。監控有空白。我那時候以為是停電造成的。但——停電前就已經斷了？還是停電後才斷的？我記不清了。我只記得當時沒人在意那段空白。

**顧言**：(內心)第二輪。裝卸區。校務車出發前後。畫面有缺失。我那時候以為是角度問題。但——裝卸區的攝像頭不是壞了，是那段時間根本沒有紀錄。

**顧言** [determined]：(內心)兩次。兩段空白。都在關鍵事件前後。

**顧言** [determined]：(內心)現在他告訴我：舊系統切線，固定會有十七分鐘看不到。

**顧言** [determined]：(內心)十七分鐘。一個固定的、可預測的空白。任何人只要知道這個系統什麼時候會切線，就可以把任何行動塞進那十七分鐘裡。

**旁白**：林主任站在邵叔身後兩步遠的地方，雙手插在西裝褲口袋裡，肩膀微微聳起，像在忍受一件他不想在場的事。

**林主任** [stern]：老系統了，下這麼大雨很正常。今天不要把所有小問題都當陰謀。

**旁白**：他的語氣比在指揮區時放鬆了一些。少了那種被邵叔壓住之後的僵硬。像是一個人在覺得安全的環境裡，又找回了那套官腔。

**顧言**：(內心)林主任。他又在壓。

**顧言**：(內心)第一輪他壓我的問題。第二輪他壓車輛表的異常。第三輪他在壓監控空白。

**顧言** [determined]：(內心)每一次有什麼東西對不上，他都會跳出來說「很正常」。

**旁白**：技術人員終於把鐘樓的畫面調出來了。畫質很差，灰綠色的，像隔了一層髒玻璃。時間戳記顯示 06:48 到 07:05 之間，畫面是正常的——鐘樓外牆、維修門、北側樓梯入口，都是空的。

**旁白**：然後 07:05 過後，畫面開始卡。

**旁白**：不是突然斷掉的那種黑，而是一格一格地跳。像有人按了暫停，等了幾秒，再按播放，再暫停。技術人員皺著眉，把播放速度調慢。

**旁白**：在兩格跳動之間——大概只有零點幾秒的間隙——顧言看見了。

**顧言** [shocked]：(內心)那個⋯⋯

**旁白**：鐘樓維修門前。有人。那人背對鏡頭，穿著深色雨衣，正在往門上貼一張紙。白色的紙。格式像是臨時告示。

**旁白**：然後畫面斷了。

**旁白**：不是暗掉，不是黑屏。是直接跳到了下一個時間戳——07:22。中間十七分鐘的紀錄，一片空白。

**顧言** [determined]：(內心)有人貼了告示。但看不到是誰。他剛好出現在斷線前的最後幾格。

**顧言** [determined]：(內心)是巧合？還是他知道攝像頭什麼時候會斷？

**旁白**：技術人員切到裝卸區的畫面。

**旁白**：這段更糟。06:30 到 07:15 之間是正常的——黑色器材箱整齊排列在雨棚下，有工作人員進出搬東西。然後 07:15 之後，畫面變成一片雪花。等到重新恢復的時候，已經是 07:32。

**旁白**：又是十七分鐘。

**旁白**：但恢復後的畫面裡，器材箱的位置變了。

**顧言** [shocked]：(內心)⋯⋯移動了。

**顧言**：(內心)斷線前，那三個黑色器材箱在雨棚左側，排成一行。恢復之後，最右邊那個箱子的位置往後移了大約一米。中間過程——完全空白。沒有人影，沒有搬運，什麼都沒有。只有結果。

**顧言** [determined]：(內心)箱子被人移動了。但移動的過程，被十七分鐘的空白吃掉了。

**旁白**：第三段。南翼醫務室走廊。

**旁白**：這段的畫面比前兩段好一些——至少沒有完全斷訊，只是畫質很差，人影模糊得像水裡的倒影。時間戳記從 07:40 到 08:10。

**旁白**：07:52，走廊畫面裡出現了一個人。穿著深灰色西裝。金屬框眼鏡反射著走廊燈光。

**顧言**：(內心)⋯⋯林主任？

**旁白**：不。不完全確定。那個人從鏡頭右側進入，走向南翼醫務室方向，然後畫面又開始卡。等畫面穩下來的時候，走廊已經空了。

**顧言** [determined]：(內心)行政組的人員。進出南翼醫務室。時間在斷線前後。畫面不完整。

**顧言** [determined]：(內心)鐘樓有人貼告示。裝卸區有箱子被移動。南翼走廊有人進出。全部發生在十七分鐘的空白前後。全部只有殘片，沒有過程。

**顧言** [determined]：(內心)如果有人知道舊系統的切線時間，他就可以在這十七分鐘裡做任何事。

**旁白**：邵叔站在螢幕前，雙手背在身後，表情平靜。他看了三段畫面，一句話也沒有說。

**邵叔** [serious]：這些紀錄先保留。不要覆蓋。

**技術人員**：好。

**邵叔** [serious]：今天每一次切線，你都記下時間。精確到秒。

**技術人員**：⋯⋯好。

**旁白**：技術人員的語氣有些遲疑。他大概沒有被這樣要求過。但他還是點了點頭，在鍵盤旁邊的便條紙上寫了「07:05-07:22」和「07:15-07:32」兩個時間段。

**旁白**：邵叔轉身，看了一眼顧言。

**邵叔** [concerned]：走吧。帶你去個安靜的地方。

**旁白**：顧言從椅子上站起來，跟在邵叔身後往樓梯走。經過林主任身邊的時候，他聞到了一股很淡的氣味——機器的，微熱的，像這棟樓裡到處都有的那種電子設備散熱的味道。林主任的衣服上沾著它，大概是在這間悶熱的監控室裡待久了。

**顧言** [determined]：(內心)十七分鐘。我記住了。

**顧言** [determined]：(內心)一個舊系統的固定漏洞。一個可以被任何人利用的窗口。關鍵不是漏洞存不存在。關鍵是——誰知道它什麼時候會打開。

**旁白**：他走上樓梯。身後的監控室裡，九面螢幕繼續閃爍著灰綠色的、充滿雜訊的畫面。
