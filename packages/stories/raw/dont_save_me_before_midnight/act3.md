# 第三幕：九點半的校史館

**旁白**：校史館在鳴海高中最北邊，一棟獨立的三層老樓，外牆爬滿了深綠色的藤蔓，看起來像一塊被苔蘚覆蓋的石碑。雨天的時候，藤蔓會變得更深、更濕，像活的東西在呼吸。

**旁白**：顧言推開鐵門的時候，門軸發出一聲刺耳的嘎吱聲。館內的燈光是暖黃色的，帶著一種老式檯燈的昏沉感，和外面灰濛濛的雨天完全不同。空氣裡有木頭和舊紙的味道，混著一點點防蟲劑的刺鼻氣味。

**廣播**：校慶活動提醒——校史館百年回顧展現已開放，歡迎師生及校友前往一樓展廳參觀。二樓為史料修復區，暫不對外開放。

**顧言**：(內心)人果然不多。

**旁白**：一樓展廳裡只有三三兩兩的參觀者，大部分是外校來的家長，戴著胸牌，在手機上對著展板拍照。展板是新的，白底黑字，上面用燙金字體印著「百年鳴海，世紀同行」，每個字都經過精心設計，連感嘆號的角度都統一過。

**旁白**：顧言沒有看展板。他穿過展廳，走到後方的準備區入口。門上貼著一張手寫的紙條：「志工請先簽到，箱子搬到B區架子。」

**旁白**：準備區比展廳大得多，是一個被臨時隔出來的空間，堆滿了展覽用的道具、布景板和各種大小的收納箱。燈光是白色的日光燈管，和展廳的暖黃完全不同，照得所有東西都失去了層次，變得扁平而透明。

**同學**：你就是來幫忙搬箱子的？那邊有手套，自己拿。重的東西兩個人一起搬，不要逞強。

**顧言**：(內心)好。

**旁白**：他拿了一副棉質手套戴上，開始搬箱子。箱子不重，大部分是展覽用的說明牌和備用布條，只有幾箱裝著老舊的檔案資料，比其他的沉一些。他搬了五六箱，手臂開始有點酸，但這種程度的體力活反而讓他安心——至少腦子可以放空。

**旁白**：搬第七箱的時候，他注意到角落裡有一排沒有整理的舊檔案盒。盒子是牛皮紙色的，上面用毛筆寫著年份和分類，墨跡已經暈開了，像水漬，又像眼淚。最上面那個盒子的蓋子沒有蓋好，微微張開，露出一角泛黃的紙邊。

**顧言**：(內心)這些東西也展出嗎？

**旁白**：他把箱子放到架子上，蹲下來整理那排檔案盒。第一個盒子裡是校刊，日期是十五年前的。第二個盒子裡是教職員通訊錄，紙頁發脆，翻的時候會發出細碎的聲響。第三個盒子打開的時候，裡面的東西散了出來。

**旁白**：幾本筆記本、一疊活動照片、一張折了兩折的大紙。

**顧言**：(內心)照片。

**旁白**：他猶豫了一下，把那疊照片拿起來。照片的邊緣已經泛黃，有些粘在一起，他用手指小心地分開。大部分是校慶活動的合影，人很多，表情模糊，看不出誰是誰。

**旁白**：然後他翻到一張不一樣的。

**旁白**：這張照片比其他的保存得好，雖然邊角也有泛黃的痕跡，但畫面本身清晰得不像話。照片裡是四個人，站在一棵大榕樹下，背景是鳴海高中的舊校門——和現在的校門不同，那個校門更矮、更窄，上面還有繁體字的校名。

**旁白**：最左邊是一個年輕男人，穿著白襯衫，袖子捲到手肘，手裡抱著一個三四歲的小男孩。小男孩的臉圓圓的，頭髮剪得很短，正咧著嘴笑，露出一排小小的乳牙。

**顧言**：(內心)⋯⋯

**旁白**：他的手指僵住了。

**顧言**：(內心)這是我。

**顧言**：(內心)這是我小時候。旁邊這個人是⋯⋯

**旁白**：抱著他的那個男人正在笑。不是那種為了拍照擠出來的笑，是真的在笑——眼角有細紋，嘴角的弧度很自然，整個人看起來放鬆、柔軟，像一個普通的父親抱著自己的孩子，在普通的下午，站在普通的校門口。

**旁白**：那個人是顧澤。他的父親。

**顧言**：(內心)不像。

**顧言**：(內心)新聞裡的父親不是長這樣的。新聞裡的照片永遠是同一張——那張被裁切過的、像素模糊的證件照，眼神空洞，表情木然，像一個已經被定了罪的人。

**顧言**：(內心)但這張照片裡的人⋯⋯他在笑。他看起來很普通。他看起來⋯⋯

**旁白**：顧言的喉嚨動了一下。他把照片舉到離眼睛更近的地方，像是想看清楚父親瞳孔裡的倒影。

**旁白**：照片裡第二個人站在父親旁邊，是個略微發福的中年男人，穿著深藍色的學校行政制服，胸前的名牌反光看不清字。他在鏡頭前微微側著身，像是在躲避什麼。

**顧言**：(內心)這個人⋯⋯不認識。

**旁白**：第三個人站在最右邊，個子不高，皮膚很黑，笑的時候露出一口白牙，手裡拿著一把扳手，像是剛修完什麼東西。

**顧言**：(內心)邵叔？

**顧言**：(內心)是邵叔。年輕了很多，但那個臉型、那個笑容，不會認錯。他說過他認識我爸，原來那時候就⋯⋯

**旁白**：第四個人站在照片的中間位置，是一個女人。她穿著一件淺灰色的外套，頭髮紮成低馬尾，臉上的表情很淡，幾乎看不出情緒——不是冷漠，而是某種刻意的節制，像是一個人已經練習過很多次如何在鏡頭前不暴露自己。

**顧言**：(內心)這個人不認識。沒見過。

**顧言**：(內心)她是誰？為什麼會和我爸、邵叔站在一起？

**旁白**：他把照片翻過來，背面是空白的，沒有日期、沒有名字、沒有任何線索。他又翻回正面，目光再次落在父親的臉上。

**旁白**：那個笑容。他一直在看那個笑容。

**顧言**：(內心)這個人看起來不像縱火犯。

**顧言**：(內心)他看起來像一個父親。

**顧言**：(內心)他抱著我，在笑。就好像這是一個正常的日子，正常的人生，沒有什麼會被燒掉。

**旁白**：他的手指微微收緊，照片的邊角被捏出了一個小折痕。他立刻鬆開了手，用拇指把折痕壓回去，動作很輕，像在撫平一道傷口。

**顧言**：(內心)爸。

**旁白**：他很少叫這個字。嘴裡不叫，心裡也不叫。但現在他叫了，在腦子裡，很輕，像怕被誰聽見。

**旁白**：他正準備把照片再翻過來看看有沒有漏掉的細節，一隻手突然從他背後伸過來，把照片抽走了。

**旁白**：動作很快，快到他的手指還保持著捏住照片邊緣的姿勢，但手裡已經空了。

**林主任**：你在看什麼？

**旁白**：聲音從正後方傳來，不高，但很急。顧言轉過身，看見一個穿著深灰色西裝、胸前別著校徽的男人站在他身後。林主任，校史館的行政主管，也是這次百年校慶展覽的總負責人。

**旁白**：他大概四十多歲，頭髮梳得很整齊，眼鏡是金屬細框的那種，看起來像是那種在辦公室裡泡茶看報紙的行政人員。但此刻他的表情不對——不是那種「學生不該亂翻東西」的嚴厲，而是一種更深層的東西。

**旁白**：他在緊張。

**顧言**：(內心)⋯⋯林主任？

**林主任**：這些是還沒有分類的檔案，不應該在開放區域。誰讓你動這些的？

**顧言**：我以為是要整理的⋯⋯

**林主任**：不用整理，這些會統一處理。

**旁白**：林主任的手指緊緊捏著那張照片，指節發白。他低頭看了一眼照片，然後迅速翻到背面，像是在確認什麼，接著又翻回正面，目光在四個人的臉上快速掃過。

**顧言**：(內心)他在看那張照片。

**顧言**：(內心)不，他在看我爸。

**林主任**：校慶是喜事，別把以前那些不愉快的東西拿出來。

**旁白**：他說這句話的時候，語氣是平的，像是在背誦一段規定。但他的手在抖。不明顯，如果不盯著看根本不會發現，但顧言離他不到一公尺，看得很清楚——林主任的右手在抖，照片的邊角跟著顫動，像一面被風吹皺的小旗。

**顧言**：(內心)他不是在生氣。他在害怕。

**顧言**：(內心)為什麼？一張照片而已，為什麼會怕？

**旁白**：林主任把照片塞進西裝外套的內側口袋裡，動作很快，像是在藏什麼見不得光的東西。然後他推了推眼鏡，深吸了一口氣，表情恢復了正常——至少看起來正常了。

**林主任**：你繼續搬外面的箱子就好，裡面的東西不用管。

**顧言**：好。

**旁白**：林主任轉身離開，腳步比正常走路快了半拍。他的背影消失在準備區和展廳之間的通道裡，皮鞋踩在磨石子地板上的聲音越來越遠，越來越急，像一個在趕路的人。

**顧言**：(內心)「校慶是喜事，別把以前那些不愉快的東西拿出來。」

**顧言**：(內心)不愉快的東西。

**顧言**：(內心)他在說我爸。

**顧言**：(內心)他不是在說火災，不是在說傷亡，不是在說安全問題。他是在說——這個人的臉出現在展覽裡，就是不愉快的。

**旁白**：顧言蹲在原地，手裡還是空的。指尖殘留著照片紙的觸感，薄薄的、脆脆的，帶著一種老舊紙張特有的粗糙。他盯著自己空空的手掌看了幾秒。

**顧言**：(內心)不意外。

**顧言**：(內心)每次都一樣。只要有人提起我爸，所有人的第一反應都是藏起來。把他藏進「顧某」兩個字裡，把他藏進模糊的官方報告裡，把他藏進「不愉快的東西」這種形容詞裡。

**顧言**：(內心)好像他從來就不是一個人，只是一個需要被處理的問題。

**顧言**：(內心)可是那張照片裡的人⋯⋯他在笑。

**顧言**：(內心)他抱著我，他在笑。他看起來不像一個問題。

**顧言**：(內心)他看起來像一個人。

**旁白**：他的眼眶有點熱，但他沒有哭。他已經很久沒有哭了，久到他幾乎忘記眼淚的溫度是什麼感覺。那股熱意很快就退下去了，退到一個很深的地方，和其他所有他不願意面對的東西一起沉在水底。

**旁白**：他站起來，拍了拍膝蓋上的灰。

**旁白**：準備區的另一頭傳來嘈雜的聲音。他抬頭看過去，幾個學生正在合力搬動一個很大的玻璃展示櫃——不，不是展示櫃，是一個底座。一個很大的、封閉的、帶有金屬鎖扣的底座。

**同學**：小心小心——這東西怎麼這麼重？

**同學**：裡面裝了什麼啊？水泥嗎？

**同學**：時間膠囊啊，今天晚上的重頭戲。據說裡面放了學校一百年來的重要文件，還有什麼⋯⋯校友基金會的什麼記錄之類的。反正很重，四個人才搬得動。

**旁白**：顧言看著那個底座被慢慢推向展廳方向。它是深色的金屬外殼，四角有固定的螺絲，正面嵌著一塊小銅牌，上面刻著「鳴海高中百年時間膠囊」幾個字。銅牌下面是一個鎖孔，很大，不是那種普通的掛鎖孔，而是需要鑰匙才能打開的嵌入式鎖。

**顧言**：(內心)時間膠囊。

**顧言**：(內心)一個學校的時間膠囊，需要這麼重的底座和這麼大的鎖？

**顧言**：(內心)裡面到底裝了什麼？校友基金會的記錄？什麼樣的記錄需要用這種方式封存？

**旁白**：底座經過他身邊的時候，地板傳來一陣沉悶的震動。真的很重。不是那種「為了儀式感做得厚重」的重，而是裡面確實裝了什麼東西的重。

**顧言**：(內心)算了，不關我的事。

**旁白**：他轉回身，繼續搬箱子。一箱、兩箱、三箱。每一箱都比之前重了一點，也可能只是他的手臂比之前酸了一點。

**旁白**：搬第十一箱的時候，他停了一下。

**顧言**：(內心)林主任為什麼那麼怕那張照片？

**顧言**：(內心)他不是怕照片本身。他是怕照片被看見。怕被我看見。

**顧言**：(內心)可是他為什麼要怕？那張照片裡有什麼？

**顧言**：(內心)有我爸。有我。有邵叔。還有一個我不認識的女人。

**顧言**：(內心)四個人站在校門口，拍了一張正常的合照。

**顧言**：(內心)哪裡「不愉快」了？

**旁白**：他沒有答案。也許林主任只是不想在校慶期間惹麻煩，也許他只是個過度謹慎的行政人員，也許那張照片對他來說真的只是一個「可能讓氣氛變差」的東西。

**顧言**：(內心)也許吧。

**顧言**：(內心)但我忘不掉的是那個笑容。

**顧言**：(內心)我爸在笑。

**顧言**：(內心)不是新聞裡那張臉，不是報告裡那個名字，不是展板上「顧某」兩個字。是一個真的人，穿著白襯衫，袖子捲到手肘，抱著自己的兒子，在校門口笑。

**顧言**：(內心)那個人看起來⋯⋯不像做過那些事的人。

**顧言**：(內心)又或者，我只是在看我想看到的東西。

**旁白**：他把第十一箱放上架子，轉身去拿第十二箱。

**旁白**：準備區的日光燈管閃了一下，像電壓不穩。他抬頭看了一眼，燈管恢復了正常。旁邊那排舊檔案盒還在原地，但他剛才翻開的那個盒子已經被合上了，蓋子壓得很緊，像是有人特意整理過。

**顧言**：(內心)林主任回來過了。

**旁白**：他沒有再去碰那些盒子。

**旁白**：遠處傳來展廳裡的廣播聲，有人在介紹校史展的百年回顧內容，聲音透過牆壁傳進來，變得模糊而遙遠，像一個人在水底說話。展廳裡的參觀者多了起來，腳步聲和交談聲混在一起，形成一種低沉的嗡嗡聲，像雨聲。

**旁白**：顧言搬完最後一箱，摘下手套。手指關節有點僵硬，指尖的觸感還在——不是手套的棉布感，而是更早之前，那張照片的紙張觸感。薄、脆、粗糙，帶著時間的重量。

**顧言**：(內心)邵叔在那張照片裡。

**顧言**：(內心)他說他認識我爸，原來不只是認識。他們看起來很熟。那邵叔知不知道這張照片？知不知道林主任把它收走了？

**顧言**：(內心)還有那個女人。她是誰？

**顧言**：(內心)四個人裡面，我認識三個。我爸、我、邵叔。只剩她不認識。

**顧言**：(內心)她的表情很奇怪。不像是在合照，倒像是在⋯⋯站崗。

**旁白**：他搖了搖頭，把這些念頭推開。想太多也沒用，照片已經被收走了，他連再看一眼的機會都沒有。

**顧言**：(內心)算了。

**顧言**：(內心)反正也不是第一次了。

**顧言**：(內心)每次只要和那個人有關的東西出現，就會有人把它收走。藏起來。鎖起來。變成「不愉快的東西」。

**顧言**：(內心)好像他這個人從來就不應該存在過。

**旁白**：他走出準備區，穿過展廳。經過「十年前南翼樓火災紀念角」的時候，他沒有停腳步，但餘光掃到了展板上的內容——和餐廳旁那塊一樣，「前校友顧某涉嫌人為縱火」，一個字都沒變。

**顧言**：(內心)「顧某」。

**顧言**：(內心)你連他的名字都不願意寫。

**顧言**：(內心)可是我見過了。

**顧言**：(內心)我見過他笑的樣子。在照片裡。不是「顧某」，是顧澤。他穿著白襯衫，抱著我，在笑。

**顧言**：(內心)你們可以抹掉他的名字、藏起他的照片、把他的存在歸類為「不愉快的東西」。

**顧言**：(內心)可是我記得了。

**旁白**：校史館的鐵門在他身後關上，門軸又發出那聲刺耳的嘎吱聲。他站在門外的台階上，雨已經小了很多，幾乎停了，只剩下屋簷上殘留的水珠偶爾滴落，在地面的積水裡打出小小的圓圈。
