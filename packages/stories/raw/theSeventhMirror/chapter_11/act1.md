# 第一幕：原件不要帶來

```bg

```

**旁白**：15:30。澪坐在書桌前。窗簾拉了一半。下午的光斜進來，照在桌面上那面小鏡子旁邊。

**旁白**：手機先震了一下。水瀨佳乃的訊息。很短。

**訊息**：18:30。清澄區民館三樓。

**訊息**：帶影本，不要帶原件。

**訊息**：不要告訴還不確定的人。

**朝倉澪** [determined]：（內心）⋯⋯不要告訴還不確定的人。

**旁白**：最後一句沒有指向任何人。但澪自然想到了琴音。

**朝倉澪** [sad]：（內心）⋯⋯她不知道今晚的事。她不需要知道。這不是隱瞞。這是——

**旁白**：她沒有把那句話想完。手機又震了。第二則。灣岸署的號碼。

**訊息**：關於上次筆錄提到的小鏡子，請在明天上午十點前回覆方便帶原件到警署確認的時間。

**朝倉澪** [scared]：（內心）⋯⋯明日上午十時。

**旁白**：她看著那行字。看很久。像在看一個已經在倒數的鐘。

**朝倉澪** [determined]：（內心）⋯⋯他們知道是一面小鏡子。還沒扣押。但他們要我了。明天十點以前。回覆時段。帶過去。可能當場收走。暫時保管。甚至不還。

**朝倉澪** [determined]：（內心）⋯⋯在那之前，我要留下普通人能完成的紀錄。

**旁白**：她把兩則訊息都截圖。存在手機裡。然後把手機螢幕朝下扣在桌上。

**旁白**：她從抽屜深處拿出一台舊數位相機。不是手機。是她高中時用過的那台。沒有自動上傳。沒有雲端同步。拍完只存在記憶卡裡。

**朝倉澪** [determined]：（內心）⋯⋯不能用現在的手機拍。手機會備份。備份會上去。上去就不知道誰看得到。

**旁白**：她把小鏡子從信封裡取出來。放在白紙上。旁邊放一把普通直尺。

**旁白**：她先拍正面。鏡面朝上。螢幕很小，她看了兩遍，確認對焦。然後翻過來。拍背面。

**旁白**：背面那些刮痕在下午的光裡看起來比她記憶中更淺。不像刻字。更像被什麼尖的東西反覆劃過同一個位置，劃到留下痕跡但沒有完全刻穿。

**朝倉澪** [determined]：（內心）⋯⋯不是一次劃出來的。同一個位置。同一個方向。重複了很多次。不是摔到地上磨到的。不是放在口袋裡自然磨損的。是人為的。

**旁白**：她把相機湊近。拍刮痕的近照。一張。兩張。側邊一張。換角度再一張。

**旁白**：然後她拿出一張透明描圖紙。覆在鏡背上。用細字筆描出每一道痕跡的位置和方向。手很穩。像在做查核報告。

**朝倉澪** [determined]：（內心）⋯⋯位置。方向。長度。間距。先記。不解讀。

**旁白**：描完之後她把描圖紙拿起來，對著窗光看。幾條短線。不規則。有些間距很近，像被擠在一起。有些分開。她看不出那是一個字。也看不出那是一個數字。更像是多段記號被疊在同一個小區域裡。

**朝倉澪** [determined]：（內心）⋯⋯我不知道這是什麼。我只知道它不是自然磨損。

**旁白**：她在筆記本上寫。尺寸。長寬。厚度。鏡面直徑。刮痕分布範圍。最後一行她寫了兩個詞，中間用斜線隔開。

**朝倉澪**：（內心）人為？／疑似 7？

**旁白**：問號。她沒有寫句號。因為她不確定。她不確定那是不是 7。她不確定那是不是一個數字。她只確定有什麼東西被刻在上面了。

**朝倉澪** [determined]：（內心）⋯⋯不要過度解讀。不要現在就決定它是什麼。先留紀錄。照片。描圖。尺寸。筆記。夠了。剩下的等有東西可以比對再說。

**旁白**：她沒有清潔刮痕。也沒有再用任何硬物碰觸鏡背。

**旁白**：她拿出一個乾淨的白色信封。把小鏡子放進去。封口。在信封正面寫上日期。沒有寫裡面是什麼。

**旁白**：然後她拉開書桌最下面那層抽屜。抽屜有鎖。很小。不是保險箱。只是普通抽屜鎖。她把信封放進去。關上。轉鑰匙。鑰匙放進口袋。

**朝倉澪** [determined]：（內心）⋯⋯明天十點以前。我要決定回覆什麼時段。帶過去。讓他們看。讓他們確認。如果他們要暫時保管——

**朝倉澪** [scared]：（內心）⋯⋯至少我有照片。有描圖。有尺寸。有筆記。普通人能做的，我做了。

**旁白**：她把舊相機的記憶卡退出來。放進另一個小信封。也寫上日期。放進同一個上鎖的抽屜。

**旁白**：描圖紙她折好，夾進筆記本。筆記本放進背包。

**朝倉澪** [determined]：（內心）⋯⋯今晚帶影本。照片列印出來。描圖帶去。筆記帶去。原件留在家裡。上鎖。

**旁白**：她看了一眼手機。螢幕還朝下扣著。沒有新的震動。

**朝倉澪** [sad]：（內心）⋯⋯琴音。

**旁白**：她想到琴音。不是因為水瀨那句話。是因為她已經習慣了——出門前不告訴她去哪裡，把行程拆成不完整的句子，讓距離變成日常。不是因為確定了什麼。是因為還不確定。

**朝倉澪** [sad]：（內心）⋯⋯這不是新破綻。這只是現在的樣子。

**旁白**：16:25。她把背包收好。照片還沒列印。她打算在區民館附近的便利店印。

**旁白**：窗外的光正在變薄。下午快過完了。

**旁白**：桌上只剩一把直尺和一張白紙。白紙上什麼都沒有。鏡子已經不在了。

**旁白**：她站起來。拿起背包。確認口袋裡的鑰匙。

**旁白**：抽屜裡鎖著一面小鏡子。鏡子背面有什麼東西被反覆刻過。她不知道那是什麼。她只知道明天十點以前，她得告訴警方什麼時候帶它過去。

**旁白**：而今晚，她要先去一個她沒有告訴任何人的地方。
