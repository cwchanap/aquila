# 1. 前置一致性修正

本章必須與 Chapter 1、Chapter 2 同步，否則小鏡子、手機與外殼的程序處理會產生漏洞。

## 1.1 Chapter 1 匿名訊息定稿

匿名訊息不可直接寫「鏡子」。定稿如下：

> 想知道朝倉悠真在哪裡，今晚 21:17。  
> 灣岸新交通，新木場方向，第七車。  
> **帶上悠真留下的那件東西。**  
> 不要報警。

這樣的效果：

- 澪知道「那件東西」指小鏡子；
- 千田知道那是小鏡子，因為他掌握朝倉父親留下的舊線索；
- 警方只知道她被要求帶某件私人物，不會立刻知道是鏡子；
- Chapter 3 小鏡子暫時保留在澪身上才合理。

## 1.2 Chapter 2 千田確認物件定稿

Chapter 2 中，千田不要多次說出「鏡子」。建議台詞：

> 「那件東西。背面給我看。」

澪從斜背包或小包裡取出小鏡子，千田只看背面刮痕，不拿、不碰，也不讓車內監視器清楚拍到。

這樣 Chapter 3 中，警方不會立刻知道「那件東西」就是小鏡子。

## 1.3 Chapter 2 銀色外殼處理延續

Chapter 2 已定稿：銀色外殼是掌心長度的薄型資料外殼，一側被強行撬裂，破邊像刀片。千田臨倒前把外殼壓進澪掌心，澪在恐慌中未立刻鬆手，才被系統判定為「持有尖銳物」。

Chapter 3 必須接住這個動作：

- 澪不是冷靜持有疑似凶器；
- 她是在急救與驚嚇中，手還停留在上一秒被千田塞住的位置；
- 她放下外殼是被安全系統與現場人員命令後做出的動作。

# 2. 本章核心定位

Chapter 2 已經完成死亡交付。Chapter 3 的任務不是再寫一次千田倒下，而是讓外部世界開始替澪定義事件。

本章的核心句：

> 澪還在說「我是在救他」，但系統、影像、證物標籤、問話紀錄已經開始說「她像是殺了他」。

本章不是逃亡章，也不是時間循環揭露章。它是：

> **官方版本誕生章。**

更精確地說，本章是「制度密室」章。澪離開了第七車，卻被帶進站務室、控制室、問話室、證物袋、畫面定格與供詞紀錄構成的新密室。

---

# 3. 本章一句話總結

千田浩介在第七車內失去意識後，澪被車廂安全系統要求放下染血的銀色外殼。列車被遠端鎖定並停靠封閉維修月台，救護與警方介入。澪試圖說明自己只是被匿名訊息引來、只是接過千田交給她的東西、只是想急救，但控制中心的即時壓縮影像呈現出完全不同的故事：第七車內只有她和千田，千田倒下前，她手裡握著像刀片的金屬物。銀色外殼被封袋，手機被取證，血衣被收走，琴音已讀未回。深夜，醫院傳來千田死亡確認，澪第一次明白，真相還沒被查清楚，畫面已經先替她定罪。

# 4. 本章核心任務

| 任務 | 說明 |
|---|---|
| 完成 Chapter 2 後果 | 車廂遠端鎖定、救護介入、澪與千田被分開。 |
| 固定銀色外殼命運 | 澪短暫看見外殼細節，但外殼由鑑識封袋。 |
| 處理手機與血衣 | 手機短暫取證後暫還；血衣與手部血跡被拍照採樣。 |
| 保留小鏡子 | 小鏡子放在斜背包內層，不與外套一起被收走；警方暫時不知道它就是訊息中的「那件東西」。 |
| 形成官方敘事 | 控制中心使用即時壓縮流，而不是車內本地原始檔。 |
| 強化錯誤密室 | 官方暫時只看到「第七車內兩人、澪手持尖銳物、千田倒下」。 |
| 讓澪成為重大參考人 | 她沒有被正式逮捕，但被問話、取證、限制行動。 |
| 保持日下部曖昧 | 他明確只做兩件事：修正證物用詞、要求保全原始檔。其餘不明顯幫忙。 |
| 保留悠真主線 | 警方試圖把悠真案與千田案分開，刺激澪必須自己連線。 |
| 給 Chapter 4 行動目標 | 原始影像在事故車車載資料單元中，車輛將移至有明車輛基地。 |

本章真正作用：

> 澪不是在第 3 章失去自由，而是在第 3 章失去「自己解釋自己行為的權力」。

# 5. 本章讀者體驗

讀者在本章應經歷以下感受：

1. **焦急**：澪想救千田，卻被系統命令遠離。  
2. **荒謬**：她做的每個正確動作，都被影像整理成可疑行為。  
3. **憤怒**：控制中心與警方不是明顯壞人，但他們只相信流程與畫面。  
4. **不安**：日下部突然介入，像是要壓住她，也像是在保留某些空隙。  
5. **窒息**：匿名訊息、銀色外殼、血衣、遺言，每個線索都反過來變成她的嫌疑。  
6. **懸念**：真正完整畫面可能存在，但澪目前碰不到。  

本章結束時，讀者的核心問題應從：

> 千田到底為什麼倒下？

升級成：

> 澪要如何證明監視器裡的「她」不是事實？

# 6. 作者層面的固定真相

以下內容作者必須先知道，但正文不可一次說破。

## 6.1 本章仍在第一輪，尚未揭露時間循環

第 3 章仍發生在第一輪星期一夜晚至深夜。澪尚未經歷東京灣白光，也不知道自己會回到星期一早上 6:13。

本章不得揭露：

- 時間循環；
- 神鏡計畫；
- 東京方案真正含義；
- 外星訊號；
- 琴音是刺傷千田的人；
- 千田是 Chapter 1 匿名訊息的發送者；
- 車內本地原始檔一定能翻案；
- 悠真是受試者。

本章只讓讀者知道：

> 官方看到的畫面，和澪親身經歷的事情不一樣。

## 6.2 銀色外殼命運固定

採用混合方案：

> 澪短暫保住外殼幾分鐘，看見上面的刻痕、缺口與接點；但外殼很快由警方鑑識封袋。她沒有取得資料內容，只留下記憶與少量可疑細節。

這樣做的理由：

1. 死亡現場的染血尖銳物不可能讓她輕易帶走。  
2. 她不會太早拿到核心資料。  
3. 時間循環規則上合理：物品不能回到過去，但記憶可以。  
4. 劇情上有壓迫感：證據不在她手上，而在控制證物流程的人手上。  

## 6.3 澪在外殼被收走前能看到的細節

不要讓澪看見完整資料內容。她只能看到外部特徵，而且記憶是在慌亂中留下的。

| 細節 | 本章表面意義 | 後期回收 |
|---|---|---|
| 壓裂金屬邊 | 像刀片，所以害她被系統誤判 | 破邊是強行拆取資料模組造成，不是真凶器。 |
| 內側一排金色接點 | 像電子零件 | 證明它是資料匣，不是武器。 |
| 血下露出「T」和像「7」的斜痕 | 意義不明 | 後期連接 TOKYO-7 / 東京方案。 |
| 被撬開的缺口 | 像粗暴損壞 | 千田從設備中硬拆出資料外殼。 |

正文中建議只讓澪明確記住：

> 破邊、金色接點、血下面一個像 T 和 7 的痕跡。

不要在本章讓她完整讀出「TOKYO-7」。

## 6.4 監視器三層影像

| 影像層級 | 狀態 | 本章呈現 |
|---|---|---|
| 車內本地原始檔 | 可能有完整畫面，存在車載安全資料單元中 | 本章拿不到，只被要求保全。 |
| 控制中心即時壓縮流 | 用於即時安全判斷，有補幀、時間戳校正與片段重組 | 本章主要依據，對澪極不利。 |
| 官方匯出畫面 | 由控制中心資料整理出的版本 | 本章末初步形成，將成為外部敘事來源。 |

重要原則：

> 真相不是沒有被拍到，而是被拍到後，最先被拿來判斷的版本已經被系統整理過。

本章不要讓任何人說「影像造假」。應寫成更冷、更現實的情況：

- 控制中心說「目前可確認的畫面如此」；
- 警方說「我們先依現有資料判斷」；
- 澪說「不是這樣」；
- 沒有人有時間、意願或權限立刻幫她調本地原始檔。

## 6.5 車載原始檔為什麼不能立刻看

必須給出簡單但可信的阻力。

定稿設定：

> 車內本地原始檔存在事故列車的安全資料單元中，站務室只能看控制中心即時壓縮流。要取得本地原始檔，必須封存事故車，由灣岸新交通技術班在有明車輛基地抽取資料，並確認資料完整性。

這個設定的效果：

- 澪無法立刻靠原始檔翻案；
- 日下部要求保全原始檔變得合理；
- Chapter 4 有具體行動目標：有明車輛基地 / 技術班 / 事故車。

可用台詞：

> 站務主管：「中央流可以先看。車載原始檔要等車輛封存後，由技術班抽出來。」  
> 日下部：「那就現在封存。不要覆蓋。」

## 6.6 手機處理固定

澪的手機不能被完全忽略，因為它包含：

- 匿名訊息；
- 澪與琴音的訊息；
- 位置與時間紀錄；
- Chapter 1 中澪做的安全備份。

本章定稿處理：

1. 澪主動拿手機給警方看匿名訊息；
2. 警方拍照、記錄、短暫備份必要資料；
3. 匿名帳號已消失或無法追蹤；
4. 截圖與雲端備份不能立刻證明訊息真偽；
5. 因澪尚未被正式逮捕，手機深夜暫時歸還；
6. 澪被要求不得刪除資料、保持聯絡、隔天配合完整取證。

這樣既合理，又保留澪後續行動能力。

## 6.7 小鏡子處理固定

小鏡子不能放在外套內袋，因為外套會被證物流程收走。

定稿處理：

- 小鏡子放在澪斜背包內層的小布套，或手機包夾層；
- 斜背包沒有沾血，現場初步檢查只確認沒有危險物，沒有立即完整扣押；
- 匿名訊息只寫「悠真留下的那件東西」，警方不會立刻知道那是小鏡子；
- 本章只追問一次「那件東西是什麼」；
- 因程序優先處理車內金屬外殼、血衣與手機，這個問題自然延後；
- 澪沒有正式承認那是鏡子；
- 日下部注意到她護著包，但不在其他人面前揭穿。

建議加入一句情感兼邏輯的句子：

> 小鏡子不在外套裡，而在斜背包最內層的小布套中。外套被收走時，她第一次感到慶幸：至少悠真留下的東西還沒有被裝進透明袋裡。

## 6.8 澪為什麼暫時不說小鏡子

澪不是想騙警方。她的沉默來自剛剛看到的一連串程序轉譯。

她的心理理由：

- 銀色外殼被封袋後，從「千田交給她的東西」變成「刃物狀金屬片」；
- 染血外套被收走後，從「急救痕跡」變成「血跡證據」；
- 她害怕小鏡子一旦交出去，也會從「悠真留下的東西」變成別人的證物標籤；
- 她尚未確定誰可信，包括日下部。

可用內心句：

> 她不是不想說。她只是剛剛看見每一樣東西被交出去後，都變成了別人的語言。

## 6.9 血衣與手部採樣固定

澪做過急救，身上必然有血。程序上必須處理。

本章定稿處理：

- 澪的雙手被拍照；
- 指甲、掌心、袖口被初步採樣；
- 染血外套或上衣被裝入證物袋；
- 警方或站務處給她一件臨時外套；
- 她感覺自己救人的痕跡，被包裝成嫌疑證據。

這能強化本章主題：

> 她做過的正確事情，正在被保存成錯誤故事的材料。

## 6.10 千田死亡確認時間固定

本章前半不應直接把千田稱為「死者」。更合理的處理：

| 時間 | 狀態 |
|---|---|
| 21:36–21:45 | 千田被救護帶走，現場稱「心肺停止 / 意識不明」。 |
| 22:10–23:50 | 警方以重大傷害 / 重大事故方向問話。 |
| 00:10 前後 | 醫院傳來死亡確認。 |
| 00:30 後 | 問話氣氛升級，但死因與凶器未鑑定，澪仍未正式逮捕。 |

建議場景：門外有人敲門，把一張紙交給警員。警員看完後低聲對日下部說：

> 「醫院那邊確認了。」

澪不用聽到完整內容，只要看見室內所有人的態度變了。

## 6.11 為什麼澪不會被正式逮捕

本章結尾澪不應被完全釋放，也不應被正式關押。建議狀態是：

> 她被帶去問話，列為重大參考人，暫時被允許離開，但被要求保持聯絡、近期不要離開都內、配合後續傳喚與取證。

合理理由：

- 千田死亡確認較晚才到；
- 死因與傷口型態尚未鑑定；
- 銀色外殼是否為凶器尚未鑑定；
- 澪主動按緊急通話並嘗試急救；
- 她沒有逃離現場；
- 控制中心畫面有時間碼不協調，需要技術確認；
- 日下部以「先問話、先保存證物」壓住現場直接逮捕衝動。

本章末的壓迫感應是：

> 她法律上還沒被關住，但故事版本已經開始關住她。

# 7. 本章時間與地點

## 7.1 時間

第一輪，星期一夜晚至深夜。  
時間範圍約：**21:32–翌日 01:30**。

# 8. 主要單位分工

本章涉及多個單位。為避免程序混亂，行動權限要分清楚。

| 單位 | 能做的事 | 不該做的事 |
|---|---|---|
| 鐵道安全人員 | 隔離車廂、要求澪退後、保護現場、標記地上外殼位置 | 不正式封存疑似凶器，不做刑事判斷 |
| 救護人員 | 處理千田、確認生命徵象、送醫 | 不判斷誰是兇手 |
| 站務 / 控制中心 | 調出中央流、提供列車資料、說明車載資料抽取流程 | 不決定證物性質 |
| 警方先遣 | 控制現場、初步詢問、安排鑑識 | 不跳過鑑識直接定案 |
| 鑑識人員 | 封存外殼、衣物、採樣 | 不負責問話 |
| 日下部 | 接手問話、修正程序用詞、要求保全車載原始檔 | 不像全知者解釋真相，不明顯保護澪 |

這些不必在正文中逐條說明，但場景行動要符合分工。

# 10. 詳細場景拆分

## Scene 1：放下尖銳物

**地點**：第七車內  
**時間**：21:32 左右  
**功能**：承接 Chapter 2，讓澪從急救者被迫轉成危險乘客。

### 場景內容

廣播重複：

> 「第七車乘客，請立即放下手中的尖銳物。」

澪一開始不理解。她還在按千田傷口，還在喊控制中心開門。她以為廣播是在對別人說話，直到她低頭看見左手裡仍握著銀色外殼。

澪的合理反應：

1. 想把外殼丟遠，但怕它滑到座椅下，被人說她破壞現場；
2. 想繼續壓住千田傷口，但廣播要求她退後；
3. 想對控制中心解釋，卻發現自己每個動作都被監視器看著；
4. 最後她慢慢把銀色外殼放在地板上的可見位置，手仍在發抖。

### 外殼細節

外殼被放下時，澪短暫看到：

- 破裂的三角形金屬邊；
- 內側一排金色接點；
- 被撬開的缺口；
- 血跡下方露出一個像「T」和「7」的痕跡。

她沒有時間理解。她只是本能記住。

### 千田狀態

千田不再回答。澪不能判定他已死，只能感覺：

- 他的呼吸變淺；
- 他的手從座椅邊緣滑落；
- 他的眼睛沒有聚焦；
- 血從外套內側與座位縫隙滲出。

### 本場景痛點

> 澪如果繼續急救，就像不服從系統命令；她如果退後，就像放棄千田。

這讓讀者感到制度流程的殘酷。

---

## Scene 2：封閉月台與證物袋

**地點**：封閉維修月台 / 第七車門外  
**時間**：21:34–21:45  
**功能**：救護介入，現場被接管，澪與證物分離。

### 場景內容

列車減速，不停在普通月台，而是停靠一座封閉維修月台。外面不是乘客，而是：

- 鐵道安全人員；
- 救護人員；
- 控制中心工作人員；
- 隨後趕到的警察。

車門打開前，廣播要求澪：

- 雙手舉起；
- 遠離傷者；
- 不要再碰地上的金屬物；
- 不要靠近車門。

澪想說她是在急救，但對方只要求她「先照做」。

### 外殼證物鏈

外殼流程固定如下：

1. 澪把外殼放在地板上的可見位置；
2. 鐵道安全人員先用透明防護罩或標記牌隔離，不直接拿走；
3. 警方到場後，由戴手套的鑑識人員封入透明證物袋；
4. 初始標籤寫成：

> **現場暫稱：刃物狀金屬片 / 第七車 / 朝倉澪手邊發現**

這個標籤非常重要。它不是事實，但它會影響之後所有人怎麼看它。

澪想糾正：

> 「那不是刀，是他給我的。」

但在旁人耳中，這句更像她承認自己拿著它。

### 血衣與手部採樣

澪被帶到車門外後，流程開始處理她身上的血：

- 雙手被拍照；
- 掌心、指甲、袖口被初步採樣；
- 染血外套被要求脫下，裝入證物袋；
- 站務人員給她一件薄薄的臨時外套。

這裡必須補小鏡子位置：

> 小鏡子不在外套裡，而在斜背包最內層的小布套中。外套被收走時，她第一次感到慶幸：至少悠真留下的東西還沒有被裝進透明袋裡。

### 千田傷口一瞥

救護人員剪開千田外套時，澪被要求退後。她只能從人群縫隙看見一瞬間：

- 千田外套內側似乎早有一塊白色膠布或布料；
- 血不是剛剛才噴出，而是早就浸在衣服裡；
- 座位邊緣的血痕起點比她以為的位置更低。

這是後期「上車前已受傷」的重要公平線索。  
但此刻澪太慌，只能把畫面記住，不能立刻推理。

### 本場景重點

> 車門開了，但澪不是獲救。她只是從車廂系統移交給另一個系統。

## Scene 3：第一段畫面

**地點**：灣岸新交通站務室 / 臨時控制室  
**時間**：22:10–23:00  
**功能**：讓澪第一次看到「控制中心版本」的事件。

### 場景內容

澪被帶到一個乾淨、冷白光的小房間。桌上有紙杯水、監控螢幕、列車平面圖。這裡不像審訊室，反而像站務管理空間，讓壓迫感更現代。

一名警員或站務主管問她：

- 為什麼上車；
- 是否認識千田；
- 為什麼前往第七車；
- 訊息裡說的「悠真留下的那件東西」是什麼；
- 為什麼手裡有金屬片；
- 為什麼沒有報警；
- 匿名訊息是誰發的。

### 匿名訊息處理

澪拿出手機，展示匿名訊息與截圖。

固定處理：

- 原始聊天帳號已消失或無法回覆；
- 訊息截圖仍在；
- 澪也有上傳到雲端的備份；
- 警方不能立刻確認截圖真偽；
- 訊息內容包含「不要報警」，反而讓澪更可疑。

官方視角：

> 她不是完全沒有證據，但她的證據不足以證明她無罪，反而證明她是有目的地去那節車廂。

### 手機取證

手機不能被完全略過。

流程：

1. 警方拍攝匿名訊息畫面；
2. 記錄帳號資訊與時間；
3. 初步備份必要對話；
4. 記下澪與琴音的最近通訊；
5. 告知澪之後需要配合完整取證。

澪此時或稍後看到：

> 白石琴音　已讀 21:19

這個細節只一閃而過，不讓澪當場深究。

### 小鏡子問題只追問一次

警員問：

> 「訊息裡說的『那件東西』是什麼？」

澪猶豫。她不是想騙人，而是剛看見外殼和外套被轉成別人的標籤。她害怕小鏡子也被吞進同樣的流程。

本章不要安排太多巧合打斷。最自然的處理是程序排序：

> 先處理車內交付物、金屬外殼、傷者狀態與監視器。私人物之後列清單再確認。

於是這個問題暫時被擱置。不是沒人注意，而是優先級還沒輪到它。

可加入澪內心句：

> 她不是不想說。她只是剛剛看見每一樣東西被交出去後，都變成了別人的語言。

### 控制中心版本影像

控制中心調出的不是本地原始檔，而是即時壓縮流片段。

畫面呈現：

1. 澪進入第七車；
2. 她靠近千田；
3. 防災同步測試期間畫面有短暫不自然補幀；
4. 下一段畫面裡，她手中已有銀色金屬物；
5. 千田身體前傾倒下；
6. 澪手上沾血，靠近千田；
7. 系統判定她持有尖銳物。

澪立刻指出：

> 「不是這樣。中間少了他把東西交給我的那一段。」

對方回答不能太壞，只要很冷：

> 「目前中央備份流裡沒有妳說的那段。」

這句比「妳說謊」更可怕。因為它代表：

> 只要系統版本裡沒有，那她的記憶就暫時不存在。

### 本場景注意事項

- 不要讓警方正式播放完整證據影片給澪看；
- 可以讓她看見幾張定格截圖、螢幕片段、或問話時被展示的一小段；
- 不要讓澪立刻知道本地原始檔一定存在；
- 可以讓她感覺音訊對不上、燈光閃切不自然，但她說不出技術原因；
- 不要在本章量化「11 秒」，只讓她覺得有「一小段不見了」。

## Scene 4：日下部的用詞

**地點**：站務室 / 警署臨時問話室  
**時間**：23:00 前後  
**功能**：日下部正式介入，建立「像反派的保護者」形象，但不能太早露底。

### 場景內容

澪以為來的人會是陌生刑警。門打開時，出現的是 Chapter 1 曾警告她不要追查的日下部悟。

他的第一反應不能溫柔。  
他先看現場資料、外殼證物袋、澪被包紮或清理過的手，再看她。

建議台詞：

> 「妳還是上車了。」

或：

> 「我說過，有些地方不是靠近真相，是靠近危險。」

### 日下部的表面介入理由

為避免他像瞬間傳送到劇情現場，本章需要一個表面理由：

- 澪與悠真失蹤案已在日下部注意範圍內；
- 澪名字出現在灣岸新交通重大傷害事件中後，同事通知他；
- 死者 / 傷者遺言含有「東京」與疑似災害或危險語句，案件被上報；
- 日下部以「朝倉澪與既有失蹤案相關」為由介入。

不要在本章說破他與朝倉父親舊案的真正關聯。

### 日下部明確幫助只保留兩項

為避免他太早顯得是好人，本章他明確做出的幫助只保留兩件。

#### 1. 修正證物用詞

初始標籤是：

> 刃物狀金屬片

日下部看一眼證物袋，說：

> 「先不要寫『兇器』。正式紀錄寫『金屬外殼狀物』。」

這句表面是程序細節，實際上是防止官方敘事過早鎖死。

#### 2. 要求保全車載原始檔

站務主管說：

> 「中央流可以先看。車載原始檔要等車輛封存後，由技術班抽出來。」

日下部說：

> 「那就現在封存。不要覆蓋。」

這是他第二個明確幫助，也是 Chapter 4 的行動鉤子。

### 其他「幫助」不要明顯歸給日下部

| 事件 | v2.1 處理 |
|---|---|
| 小鏡子沒有立刻被收 | 不是日下部刻意保，而是程序優先處理銀色外殼、血衣、手機；問題自然延後。 |
| 手機暫時歸還 | 因澪尚未正式逮捕，完成初步取證後暫還；日下部最多提醒她不要刪資料。 |
| 澪未被逮捕 | 因死亡確認較晚、凶器未鑑定、傷口與外殼不吻合、她主動求救；日下部不直接像保人。 |
| 問話轉移 | 由程序進行，不要讓日下部像把她從壞人手上救走。 |

### 日下部注意小鏡子

澪緊張時，可能下意識護住斜背包。日下部看見，但沒有說破。

這能延續 Chapter 1 他看到鏡子時停頓的伏筆。  
同時，讀者會懷疑：

> 他到底是想保護那件東西，還是也在找它？

### 本場景注意事項

- 日下部不能太明顯是好人；
- 他不能解釋神鏡計畫；
- 他不能直接替澪翻案；
- 他要讓讀者既討厭他，又覺得他不像普通壞警察。

## Scene 5：被整理過的供詞

**地點**：警署 / 臨時問話室  
**時間**：23:30–00:30  
**功能**：讓澪發現，語言也會像影像一樣被整理成對她不利的版本。

### 場景內容

澪被正式問話。不是嚴刑逼供，而是更可怕的「正常程序」。

她說：

> 「我是收到訊息才去的。」

紀錄裡變成：

> 朝倉澪根據匿名訊息前往指定車廂。

她說：

> 「他把東西交給我。」

紀錄裡變成：

> 朝倉澪承認接觸金屬外殼狀物。

她說：

> 「我是在壓他的傷口。」

紀錄裡變成：

> 朝倉澪雙手及袖口附著千田浩介血液。

這裡要讓讀者看見：

> 程序不一定撒謊，但它會把事情壓縮成更容易傷害她的格式。

### 悠真線被警方切開

問話中需要補一個情感刺激。

警員可以說：

> 「朝倉悠真的失蹤案，目前和本案沒有確認關聯。」

或更刺痛：

> 「也就是說，妳相信一名陌生男子，比相信警方對妳弟弟失蹤案的判斷？」

這會刺痛澪。因為警方對悠真的判斷是「自願離家」，但千田在 Chapter 2 已經告訴她那封訊息不是悠真打的。

澪此刻明白：

> 如果她被千田案困住，所有人會更快把悠真失蹤案切回「自願離家」。她必須自己把兩件事連起來。

### 「不要救東京」的處理

澪提到千田遺言：

> 「他說……不要救東京。」

普通警員不要過度反應，只應當作意味不明發言或疑似危險語句追問：

- 「東京」指哪裡？
- 千田是否提到災害？
- 她是否認識任何政治團體？
- 悠真失蹤與這句話有什麼關係？

日下部可有一個微小反應，例如筆停半秒，或視線從紀錄紙移向澪，但不要解釋。

### 死亡確認

約 00:10，門外有人敲門，把一張紙或平板交給警員。警員看完，低聲對日下部說：

> 「醫院那邊確認了。」

澪不用聽清楚完整內容。她只看見室內所有人的態度變了。

此後問話語氣升級，但仍不直接逮捕她，因為：

- 死因未鑑定；
- 外殼是否為凶器未鑑定；
- 傷口與外殼大小不完全吻合；
- 控制中心畫面仍有同步問題。

### 手機暫還與琴音已讀

手機被初步取證後暫時還給澪。她被告知：

> 「不要刪除任何資料。明天還要配合完整取證。」

日下部可以冷冷補一句：

> 「妳現在刪掉任何東西，都會變成另一個問題。」

澪接回手機時，看見訊息列表：

> 白石琴音　已讀 21:19

本章只呈現這個細節，不讓澪立刻懷疑琴音。

### 小鏡子問題延後

如果前面已問過一次「那件東西是什麼」，這裡不要再重複追問，避免太刻意。

澪只在內心感覺：

> 那個問題還會回來。只是現在還沒輪到它。

這樣小鏡子的保留更自然，也更有壓迫感。

### 日下部可用台詞

> 「回答妳確定的事。不要替死者解釋。」

這句表面很冷，實際上是在阻止她胡亂推測，避免供詞被扭曲。

### 本場景主題

> 不是只有影像會被整理。連她說出的話，也會被整理成另一個故事。

## Scene 6：監視器裡的兇手

**地點**：警署走廊 / 站務控制室外 / 深夜街道  
**時間**：00:30–01:30  
**功能**：章末鉤子，讓澪第一次看見自己被畫面固定成兇手，並帶出 Chapter 4 行動目標。

### 澪暫時離開，但不是自由

問話暫時中止。澪沒有被正式逮捕，但她不是自由的。

她被告知：

- 近期不要離開都內；
- 需要配合再次問話；
- 手機資料已備份，不得刪除相關訊息；
- 染血外套暫時作為證物保全；
- 銀色外殼將送鑑定；
- 千田浩介死亡已確認，死因仍待鑑定。

用語建議不要太正式地寫「不得離開東京」。更自然的是：

> 「近期不要離開都內。需要離開，先通知我們。」

### Chapter 4 具體行動鉤子

澪在離開前，從站務人員或日下部與技術人員對話中聽見：

> 「事故車明天上午會移到有明車輛基地。車載原始檔要等技術班抽。」

這句很重要。它讓 Chapter 4 有明確行動方向。

澪知道：

> 如果中央流少了那段，唯一可能證明千田交付外殼的地方，就是事故車上的本地原始檔。

她未必立刻決定闖車輛基地，但這會成為下一章調查動力。

### 日下部章末台詞

原主題句可改得更像刑警語氣。

建議使用：

> 「第一份畫面最麻煩。後面的人都會照著它問問題。」

這句比「畫面一旦有了第一個版本，所有人都會開始替它找理由」更自然，但保留主題。

### 章末畫面

澪經過控制室或警署走廊時，透過半開的門、反光玻璃或螢幕，看見那段控制中心畫面的定格。

畫面中：

- 千田倒在座位邊；
- 澪站在他身旁；
- 她左手握著銀色外殼；
- 外殼破邊在冷光下像刀；
- 她手上都是血；
- 監視器角度看不見她剛才是在急救。

對她來說，那不是自己。  
但對其他人來說，那就是全部。

### 章末前的三個具體矛盾

為了接 Chapter 4，本章末澪必須抓住三個矛盾：

1. **畫面少了一段**：千田把外殼塞給她的動作不在中央流裡；
2. **外殼不像刀**：破邊像刀，但裡面有金色接點；
3. **血跡像早已存在**：千田外套內側的血與膠布不像剛剛才形成。

再加上一個具體目標：

4. **事故車原始檔**：事故車明天會移到有明車輛基地，車載原始檔由技術班抽取。

### 章末建議句

> 監視器裡的她握著銀色外殼，站在千田浩介身邊。那個女孩看起來冷靜、沉默，像早就知道男人會倒下。澪想移開視線，卻發現所有人都正在看那個女孩，而不是她。

# 12. 誤導設計

## 誤導一：日下部像是來壓案的人

表面上：

- 他警告澪不要亂說；
- 他要求她配合問話；
- 他看起來比其他警察更知道內情；
- 他修正用詞時語氣冷硬，不像替她說話。

讀者會懷疑他是神鏡計畫的人。

真相：

> 他確實知道部分舊案，也在監視澪，但本章的兩個動作其實是在防止官方敘事過早定死。

## 誤導二：銀色外殼就是凶器

表面上：

- 它有尖銳破邊；
- 它沾血；
- 它在澪手裡；
- 證物袋一開始標為「刃物狀金屬片」；
- 控制中心也依此判斷澪手持尖銳物。

真相：

> 它是資料外殼。破邊能割傷皮膚，但不能造成千田身上的深刺傷。

## 誤導三：警方只是照程序，所以可能是中立的

本章不要把警方寫成全壞。多數人只是依照畫面、證物與流程行事。

這條誤導讓後期更可怕：

> 系統型陰謀不需要每個人都是壞人。只要每個人都只相信自己那一小段流程，錯誤版本就會自動完成。

## 誤導四：「不要救東京」像恐怖攻擊線索

警方可能把它往危險言論方向理解。  
讀者也可能暫時以為千田牽涉東京災難或恐怖事件。

真相：

> 「東京」不是城市，而是神鏡計畫最終同步方案。

## 誤導五：澪暫時沒被逮捕，好像還有機會證明清白

本章末她沒有被正式逮捕，表面像是還有空間。  
但真正可怕的是：

> 影像版本已經開始存在。即使她暫時走出警署，也走不出那個版本。

# 13. 角色功能

## 13.1 朝倉澪

本章要展現澪的三個核心狀態。

| 狀態 | 表現 |
|---|---|
| 急救者 | 她想救千田，不想退開。 |
| 受害者 | 她被匿名訊息引來，被系統畫面陷害。 |
| 偵查者 | 即使慌亂，仍本能記住外殼刻痕、血跡位置、影像不協調。 |

澪本章心理曲線：

| 階段 | 心理 |
|---|---|
| 車廂廣播 | 為什麼叫我放下？我是在救人。 |
| 外殼被封袋 | 那不是刀，是他給我的。 |
| 血衣被收走 | 我救人的痕跡正在變成嫌疑。 |
| 小鏡子仍在包裡 | 至少悠真留下的東西還沒被拿走。 |
| 初看控制畫面 | 那不是完整事情。 |
| 被問話 | 我的每句話都被改成另一種意思。 |
| 日下部介入 | 他到底是在壓我，還是在保留什麼？ |
| 死亡確認 | 事情從「救不救得回」變成「誰造成他的死」。 |
| 章末定格 | 我不是那個畫面裡的人，但世界會先相信她。 |

她本章不能做的事：

- 不能冷靜推理出所有真相；
- 不能知道千田上車前被琴音刺傷；
- 不能拿回銀色外殼；
- 不能說服警方完全相信她；
- 不能進入逃亡狀態太早。

她本章必須做到的事：

- 保留對外殼細節的記憶；
- 保留對血跡起點的印象；
- 發現控制中心畫面缺少一小段；
- 保住小鏡子；
- 意識到官方版本會吃掉事實；
- 得到下一章調查目標：有明車輛基地 / 車載原始檔。

## 13.2 日下部悟

本章是日下部重要登場章之一。

他要呈現成：

> 讓人討厭，但不能完全討厭。讓人懷疑，但不能完全否定。

本章他最好的功能不是解釋，而是**改變程序用詞與保全資料**。

他明確做的兩件事：

1. 不讓外殼被正式寫成「凶器」；
2. 要求保全車載原始檔，不只留中央流。

這些動作不浪漫，但很有偵探小說現實感。  
日下部的保護不是擁抱主角，而是**讓程序沒有立刻把她封死**。

## 13.3 千田浩介

千田本章大多不再說話，可能已失去意識或被救護帶走。  
但他的存在透過三件物品 / 痕跡延續：

| 物品 / 痕跡 | 功能 |
|---|---|
| 銀色外殼 | 他留下的危險證物。 |
| 血跡 | 他真正受傷地點與倒下地點不一致的證據。 |
| 遺言 | 全書核心謎語。 |

本章要讓讀者感覺：

> 千田失去意識後仍在場，因為每個人都在爭奪如何解釋他留下的痕跡。

## 13.4 白石琴音

琴音本章不建議正式登場。  
她只透過手機已讀未回存在。

原因：

- 保持她前期安全港形象；
- 避免太早讓讀者懷疑她；
- 讓 Chapter 4 或後續她「擔心澪」的登場更有雙重意味。

本章可留的唯一痕跡：

> 21:19 已讀未回。

這個時間點後期會非常痛。

| 行動目標 | 事故車將移至有明車輛基地，車載原始檔由技術班抽取。 |

## 18.2 澪失去的東西

| 類型 | 內容 |
|---|---|
| 證物 | 銀色外殼被封袋。 |
| 衣物 | 染血外套或上衣被收走。 |
| 現場控制權 | 第七車與原始影像由交通系統 / 警方接管。 |
| 解釋權 | 官方初步畫面已把她推向嫌疑人。 |
| 安全感 | 她知道自己不是被誤會一次，而是被一個版本困住。 |

## 18.3 Chapter 4 建議開場方向

Chapter 4 可以接以下任一方向：

1. **澪回到家，第一次整理自己的版本**  
   她把記住的外殼細節、血跡位置、千田台詞寫下來，開始自己的反調查。

2. **新聞開始出現**  
   媒體報導灣岸新交通死亡事件，沒有公布澪姓名，但「女大學生」與「金屬片」已被提及。

3. **琴音終於聯絡**  
   琴音晚來的關心讓讀者暫時安心，但第 3 章「21:19 已讀未回」會讓重讀時變痛。

4. **有明車輛基地成為目標**  
   澪知道事故車會被移走，若她想證明交付片段存在，就必須接近技術班或找到能確認原始檔的人。

5. **日下部警告**  
   日下部私下提醒澪不要再碰千田案，表面壓制，實際上逼她注意自己被監視。

Chapter 4 的核心任務可以是：

> 澪在第一輪剩下的七天裡，試圖靠有限記憶翻案，卻逐步發現千田死亡、悠真失蹤、東京灣白光並不是三件事，而是一條線。

# 19. 本章最終定位

第 3 章的定位是：

> 一場影像審判的開始。

第 2 章讓澪站在倒下的千田旁。  
第 3 章讓世界開始相信她就是應該站在那裡的人。

本章結束後，故事不再只是「弟弟失蹤」。澪現在同時面對四個問題：

1. 弟弟悠真在哪裡？  
2. 千田浩介為什麼倒下並死亡？  
3. 她要如何推翻一段看起來比她更可信的監視器畫面？  
4. 車載原始檔是否還來得及保住？  

本章最重要的主題句可以是：

> 畫面不會說謊。  
> 但畫面會被剪成只剩一種真相。
