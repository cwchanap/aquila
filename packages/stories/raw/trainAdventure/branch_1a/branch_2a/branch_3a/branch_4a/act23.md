# 第二十三幕：部分真相

**旁白**：話音落下，灰色的世界開始消退。色彩逐漸回歸，銀月湖的景色重新浮現。

**旁白**：當視野完全恢復時，李杰和恵美依然站在神樹前，手掌貼在樹幹上，另一隻手緊緊相握。

**旁白**：艾拉如就站在不遠處，臉上帶著期待的表情。

**艾拉如**：（微笑）怎麼樣？儀式完成了嗎？

**旁白**：李杰和恵美呆呆地站在原地，誰也沒有回答。兩人的手依然握在一起，但已經感覺不到任何溫度。

**旁白**：艾拉如的笑容逐漸僵住了。他看著李杰茫然的表情，又看向恵美蒼白的臉色，似乎意識到了什麼。

**艾拉如**：（表情變得凝重）發生什麼事了？

**旁白**：沉默。恵美低著頭，身體微微顫抖著，似乎連開口的力氣都沒有了。

**艾拉如**：（試探性地問）恵美...是不是...李杰他不符合契約的條件？

**旁白**：恵美的身體顫了一下，然後緩緩地、無力地點了點頭。

**艾拉如**：（倒吸一口涼氣，臉上滿是難以置信）這...這怎麼可能...

**旁白**：艾拉如後退了一步，用手撐住額頭，似乎在努力消化這個消息。他的眼神在李杰身上掃過，像是在重新審視什麼。

**艾拉如**：（喃喃自語）三個人...你帶來的三個人中，居然有兩個都不符合精靈契約的資格...這...

**旁白**：李杰聽到這句話，腦中轟的一聲。

**李杰**：（內心獨白）三個人？兩個不符合？這是什麼意思？

**旁白**：恵美聽到艾拉如的話後，整個人彷彿失去了所有支撐。她鬆開李杰的手，搖搖晃晃地走到一旁，無力地坐在地上。

**旁白**：她雙手抱著膝蓋，把臉埋在膝蓋上，肩膀微微顫抖著。她看起來就像一個被全世界拋棄的孩子，脆弱而無助。

**李杰**：（內心獨白）清水她...到底發生了什麼？她為什麼會這麼痛苦？

**旁白**：李杰心中有無數的疑問想問恵美，但看著她那副崩潰的樣子，他實在不忍心再給她施加壓力。

**李杰**：（轉向艾拉如，語氣嚴肅）艾拉如先生...你們到底在說什麼？什麼叫三個人中有兩個不符合？清水她...到底是怎麼回事？

**旁白**：艾拉如深深地看了李杰一眼，然後嘆了一口長長的氣。

**艾拉如**：看來...已經瞞不住了。

**旁白**：艾拉如走到恵美身邊，在她旁邊的草地上坐下。他的表情複雜，既有愧疚，也有無奈。

**艾拉如**：李杰，我必須向你道歉。我和恵美...一直在隱瞞一些事情。

**李杰**：（皺眉）隱瞞？

**艾拉如**：從一開始，我們就沒有告訴你全部的真相。我們引導你來到這裡，但沒有讓你知道...這一切的真正目的。

**旁白**：李杰的心沉了下去。他突然意識到，自己可能從頭到尾都被蒙在鼓裡。

**艾拉如**：恵美...她不是第一次來到這個世界。

**李杰**：（震驚）什麼？

**艾拉如**：她之前就來過這裡。而且，她曾經成功地回到了你們的世界。

**旁白**：李杰的眼睛睜得很大，難以置信地看著坐在地上的恵美。

**艾拉如**：她回去的方法，就是建立精靈契約。那次，她和另一個人類一起來到神樹前，建立了契約，然後兩人一起前往終焉祭壇，成功回到了你們的世界。

**李杰**：（喃喃自語）所以...所以她早就知道回去的方法...早就知道需要兩個人...

**艾拉如**：是的。這一次，她又被送到了這個世界。所以她再次尋找人類夥伴，希望能像上次一樣，建立契約回去。

**旁白**：李杰回想起這些天的經歷。恵美對精靈語的熟悉、她在某些時候表現出的鎮定、她對回家方法的平靜反應...一切都有了解釋。

**艾拉如**：但是...她這次帶來的兩個人，包括你在內，都不符合神樹的資格。只有上次那個人是符合的。

**李杰**：（困惑）資格？什麼樣的資格？為什麼我不符合？

**艾拉如**：（沉默了片刻，搖頭）神樹從不解釋它的判斷。我們也只能接受它的決定。

**旁白**：艾拉如的語氣平靜，但李杰注意到他的眼神有一瞬間閃過複雜的神色。不過，李杰現在滿腦子都是混亂的思緒，沒有深究這一點。

**旁白**：李杰沉默了。他的腦中一片混亂，無數的疑問交織在一起。

**李杰**：（強壓住怒氣）為什麼...為什麼一開始不告訴我？為什麼要瞞著我？

**艾拉如**：（苦笑）因為...如果我們一開始就告訴你真相，你會怎麼想？

**旁白**：李杰愣住了。

**艾拉如**：一個剛剛遇到的陌生女孩，告訴你她曾經來過這個世界，知道回去的方法，需要你和她一起建立某種契約...你會相信嗎？

**艾拉如**：更重要的是，其他人會怎麼想？在那個車站裡，所有人都在尋找回家的方法。如果恵美主動暴露她知道回去的路，你覺得會發生什麼？

**旁白**：李杰腦中浮現出那些絕望的人們的臉——那對煩躁爭吵的夫妻、那個自私膽怯的中年男人、還有那些被恐懼支配的陌生面孔...如果他們知道恵美握有回家的關鍵...

**李杰**：（低聲）他們會...對她不利。

**艾拉如**：（點頭）是的。所以恵美選擇了沉默。她只能裝作和其他人一樣迷茫，一樣在尋找出路。

**艾拉如**：而當她遇到你，覺得你是一個值得信任的人後，我們商量決定...引導你來到這裡，但不告訴你全部的真相。

**艾拉如**：我們原本打算，等契約建立成功後，再向你解釋一切。那時候，你們已經綁在一起了，真相也就不那麼重要了。

**旁白**：李杰深吸了一口氣，努力讓自己冷靜下來。

**旁白**：他理解艾拉如說的話。如果是他處在恵美的位置上，他也會選擇保密。在那個充滿絕望和混亂的車站裡，暴露這樣的秘密無異於自尋死路。

**李杰**：（嘆氣）我...我明白了。

**旁白**：李杰看向依然把臉埋在膝蓋上的恵美，心中湧起一股複雜的情緒。

**李杰**：（內心獨白）所以...她之前那些奇怪的表現，都是因為這個原因。她不是第一次經歷這些，所以才會對某些事情表現得太過平靜。

**李杰**：（內心獨白）而她為什麼會來到這個世界兩次...這真的只是倒霉嗎？還是有什麼其他原因？

**旁白**：李杰想起在銀月湖畔的那些日子，恵美臉上那些真誠的笑容，還有她努力為他煮粥的樣子。那些，應該都不是假的吧？

**旁白**：艾拉如站起身來，拍了拍身上的草屑。

**艾拉如**：（對恵美說）恵美，我們...需要再找一個人。

**旁白**：恵美抬起頭，眼眶紅紅的，臉上滿是淚痕。她看起來疲憊不堪，但還是點了點頭。

**清水恵美**：（聲音沙啞）嗯...我知道...

**艾拉如**：（溫和地）先回去休息一下吧。今天對你們來說...都太沉重了。

**旁白**：艾拉如率先轉身，朝著來時的路走去。恵美搖搖晃晃地站起來，低著頭跟在後面。

**旁白**：李杰站在原地，看著神樹在微風中輕輕搖曳的枝葉。那棵曾經給他詭異感覺的樹，現在看起來只是一棵普通的、古老的樹而已。

**旁白**：他深吸一口氣，轉身跟上了前面兩人的腳步。

**旁白**：三個人默默地走在銀月湖北岸的小路上，每個人都沉浸在各自的思緒中。那段美好的湖畔時光，彷彿已經是很久以前的事了。
