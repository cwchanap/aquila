# 第三十六幕：祭壇上的死者

**旁白**：時間彷彿凝固了一般。李杰站在那裡，看著跪在地上的恵美，沉默了很久很久。

**旁白**：他的腦海中不斷閃過各種念頭——斎藤的期待、恵美朋友的絕望、艾拉如的信任、還有那個殘酷的選擇...

**旁白**：終於，李杰長長地嘆了一口氣。

**李杰**：（疲憊地）我...我會幫妳。

**旁白**：恵美猛地抬起頭，眼中閃過難以置信的光芒。

**清水恵美**：（顫抖地）真...真的嗎？

**李杰**：（轉過頭，不想看她的眼睛）但我不保證一定能成功。而且...

**李杰**：（語氣沉重）我不知道這樣做是對是錯。我只是...只是不想看到妳這樣痛苦。

**清水恵美**：（站起來，眼淚再次湧出）謝謝...謝謝你...李杰先生...

**清水恵美**：（哽咽）我知道...我知道這個請求很過分...我真的...真的很感激...

**李杰**：（別過臉）別說了...我不想聽...

**旁白**：李杰的聲音中帶著一絲痛苦。他知道自己做了一個可能會後悔一輩子的決定。

**旁白**：但就在這時，恵美突然上前一步，緊緊抱住了李杰。

**李杰**：（驚訝）清水？

**旁白**：李杰還沒反應過來，恵美就踮起腳尖，在他的唇上輕輕印下一個吻。

**旁白**：那只是一個很短暫的吻，柔軟而溫暖，帶著淡淡的鹹味——那是恵美淚水的味道。

**旁白**：然後，恵美退開了。

**李杰**：（完全呆住，臉迅速漲紅）妳...妳...

**清水恵美**：（微笑著，眼中還含著淚）這是...道歉。

**清水恵美**：（輕聲）對不起...讓你做這種壞事...這個吻...算是我的歉意...

**李杰**：（結結巴巴）這...這種事...怎麼能...

**旁白**：李杰的手不知道該放在哪裡，臉紅得像要燒起來一樣。他完全不知道該如何反應。

**李杰**：（內心獨白）第...第一次...居然是在這種情況下...

**旁白**：恵美看著李杰慌亂的樣子，臉上露出一個有些調皮的笑容。

**清水恵美**：（擦掉眼淚）我們...該回去了。不能讓凱蘭先生和斎藤先生等太久。

**李杰**：（還沒回過神）啊...嗯...對...

**旁白**：恵美轉身走向休息的地方。李杰呆呆地站在原地，手指無意識地碰了碰自己的嘴唇。

**旁白**：那個吻的觸感，似乎還殘留在那裡。

---

**旁白**：當兩人回到休息地點時，斎藤剛好醒過來，正伸著懶腰。

**斎藤蓮**：（打了個哈欠）呼...睡得真好...咦，你們兩個去哪了？

**清水恵美**：（自然地笑著）我們去周圍看了看。

**凱蘭**：（站起來）既然大家都休息好了，我們繼續出發吧。還有大約兩到三個小時的路程。

**旁白**：李杰點了點頭，但他始終沒有看向恵美。他的心跳還沒有完全平復下來。

---

**旁白**：又走了將近三個小時後，前方的景色突然開闊起來。

**旁白**：一座巨大的石造建築出現在視野中。那是一座古老的神殿，由黑色的巨石堆砌而成，上面刻滿了複雜的符文和圖案。

**旁白**：神殿的正面矗立著一座巨大的拱門，門框上纏繞著發光的藤蔓，散發著淡淡的藍色光芒。

**旁白**：整座建築散發著一股古老而神秘的氣息，讓人不自覺地感到敬畏。

**凱蘭**：（停下腳步）我們到了。那就是終焉祭壇。

**斎藤蓮**：（瞪大眼睛）好...好壯觀...

**清水恵美**：（看著神殿，表情複雜）是啊...

**旁白**：李杰注意到恵美的手微微顫抖。她一定在想著那個被困在這裡的朋友...

**凱蘭**：（開始走向拱門）跟我來。

**旁白**：四人走向那座巨大的拱門。當他們踏入拱門的瞬間——

**旁白**：周遭的一切聲音瞬間消失。森林的風聲、樹葉的沙沙聲、甚至自己的呼吸聲都變得微弱。取而代之的是一種令人窒息的死寂。

**斎藤蓮**：（小聲）這裡...好安靜...

**凱蘭**：（警戒地）小心點...

**旁白**：眼前是一個寬敞的大廳。數根巨大的石柱從地面延伸到看不見的穹頂，每根石柱上都雕刻著奇異的符文。

**旁白**：地板由光滑的黑石鋪成，能清晰地映出他們的倒影。空氣冰冷而稀薄，每一次呼吸都像是在吸入冰塊般刺痛。

**凱蘭**：（指向前方）祭壇在深處。我們要穿過這條長廊，然後往下走。

**旁白**：四人沿著大廳前進。前方是一條幽暗的長廊，像一道深不見底的傷口，從建築的體內裂開。

**旁白**：長廊很長，兩側的牆壁刻滿了複雜的符文。腳步聲在狹窄的空間中迴盪，形成詭異的回音。

**旁白**：走了一段路後，前方出現了一個向下的樓梯。

**凱蘭**：（停在樓梯口）往下走。祭壇在地底深處。

**旁白**：四人開始下樓梯。一層、兩層、三層...樓梯似乎沒有盡頭，不斷向下延伸。

**旁白**：空氣越來越冷，壓力越來越大，像是在往地底深處下沉。

**斎藤蓮**：（喘著氣）還...還要走多久...

**凱蘭**：快到了。

**旁白**：十層...十五層...二十層...終於，樓梯到達了盡頭。

**旁白**：眼前是另一條走廊。這條走廊比上面的更加寬闊，兩側的牆壁上鑲嵌著發光的寶石，散發著淡淡的藍色光芒。

**凱蘭**：（指向前方）就在前面。

**旁白**：走廊盡頭，矗立著一扇巨大的古老石門。石門高達數米，表面刻滿了精緻的浮雕和符文。門縫中透出微弱的金色光芒。

**凱蘭**：（推門）就是這裡了。

**旁白**：四人合力推開石門。門緩緩打開，發出低沉的轟鳴聲。

**旁白**：眼前的景象讓所有人屏住了呼吸。

**旁白**：那是一個極其寬敞的空間——不，應該說是聖地。

**旁白**：穹頂高聳入雲，牆壁四周雕刻著精美的壁畫。空氣中瀰漫著一股神聖的氣息，讓人不由得心生敬畏。

**旁白**：在空間的正中央，矗立著一座白色大理石祭壇。祭壇四周環繞著淡淡的金色光芒，像是有無形的力量在守護著它。

**旁白**：凱蘭帶領著三人走向祭壇。

**旁白**：但是...

**旁白**：當四人看清楚祭壇上的景象時——

**旁白**：一個年輕的男性躺在祭壇上。他看起來大約二十歲左右，身穿白色的長袍，長袍上繡著精緻的花紋，看起來像是某種儀式用的服裝。

**旁白**：他的胸口插著一把刀。

**旁白**：鮮血從傷口流出，染紅了白色的長袍，在祭壇的白色大理石上凝結成一灘暗紅色的血跡。

**旁白**：他閉著眼睛，臉色蒼白如紙，一動不動地躺在那裡。
