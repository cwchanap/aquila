# 第四十三幕：歸途的代價

**旁白**：房間裡陷入短暫的沉默。

**旁白**：大輔收起平板電腦，表情變得認真。

**斎藤大輔**：那麼...關於蓮...

**斎藤大輔**：（看著李杰和恵美）你們知道他的情況嗎？

**旁白**：李杰的身體僵住了。

**旁白**：恵美也低下頭，緊緊咬著嘴唇。

**旁白**：該怎麼說...該怎麼告訴大輔...

**旁白**：他們為了逃命...把斎藤留在了那個世界...

**旁白**：兩人沉默著，誰也沒有開口。

**斎藤大輔**：（察覺到氣氛不對）是...不是出了什麼事？

**旁白**：李杰深吸一口氣，握緊拳頭。

**旁白**：他看向恵美，恵美的眼中滿是愧疚和痛苦。

**旁白**：李杰知道，自己必須說出來。

**李杰**：（緩慢地）斎藤先生...蓮他還在那個世界。為了讓恵美回來，我犧牲了他。

**李杰**：（快速補充）但他還活著！那個地方一次只能讓一個人回來，必須要有人留在那裡。

**李杰**：（抬起頭）如果能再找到一個人，蓮就可以回來了...

**旁白**：房間裡一片死寂。

**旁白**：李杰不敢看大輔的表情。

**旁白**：恵美的眼淚無聲地滑落。

**旁白**：伊藤莉子站在一旁，也不知道該說什麼。

**旁白**：良久——

**斎藤大輔**：（深深嘆了口氣）這樣啊...至少他還活著對吧？

**斎藤大輔**：我在這個組織待了25年，見過太多失蹤案的家屬，很多人連生死都不知道。

**斎藤大輔**：但你告訴我蓮還活著，光是這點就已經讓我很高興了。

**清水恵美**：（哽咽）對不起...都是因為我...

**斎藤大輔**：（搖手認真地）別這麼說。在那種情況下，你們能活著回來就已經很不容易了。

---

**斎藤大輔**：（靠向椅背）其實...5年前，也就是2045年，組織派了一支調查隊。

**李杰**：（驚訝）調查隊？

**斎藤大輔**：對。我們想辦法...讓他們也上了那班列車。目的是調查兩批失蹤乘客的情況。

**斎藤大輔**：那支隊伍有5個人，都是精英。如果你們能早幾年回來，也許就能給他們提供更多情報了。

**斎藤大輔**：（苦笑）不過我不是在責怪你們，只是有點可惜而已。

**清水恵美**：（雙手合十）我會祈禱...

**清水恵美**：祈禱那支隊伍能平安...也能救出蓮先生...

**斎藤大輔**：（溫和地笑了）謝謝妳。

---

**旁白**：李杰沉默了片刻，然後鼓起勇氣開口。

**李杰**：斎藤先生...我的父母...他們...

**旁白**：大輔的表情變得凝重。

**斎藤大輔**：你的父親...李明遠先生，10年前在一場交通事故中去世了。

**斎藤大輔**：你的母親王美玲女士現在住在上海，身體還算健康。

**旁白**：李杰的身體搖晃了一下，感覺胸口被什麼東西狠狠撞擊。

**李杰**：（顫抖閉上眼睛深深吸氣）爸...媽媽她現在應該70歲了...

**旁白**：李杰的手緊緊握著椅子扶手。

**旁白**：父親已經不在了...母親獨自一人在上海...

**旁白**：這25年...她是怎麼熬過來的...

**清水恵美**：（小聲問）那...我的家人...

**旁白**：恵美的聲音很輕，充滿了不安。

**斎藤大輔**：清水小姐，妳的父母都已經過世了。

**斎藤大輔**：不過...妳的弟弟清水浩介還活著，現在住在船橋。

**清水恵美**：（眼淚湧出又抬起頭）浩介...

**斎藤大輔**：他23年前加入了我們組織。當時我在調查第一批失蹤乘客，找到了浩介先生。

**斎藤大輔**：他那時候已經自己調查了好幾年，但因為資源有限一直沒有成果。

**斎藤大輔**：後來他就加入我們，全心投入這個案子。現在他60歲了，這些年一直在組織裡工作。

**旁白**：李杰看著恵美。

**旁白**：他知道恵美經歷了兩次失蹤事件——45年前和25年前。

**旁白**：她的弟弟浩介...這25年來...不，應該說45年來...一直在尋找她...

**斎藤大輔**：（看著恵美）我已經通知他了，等一下可以視訊通話，妳可以跟他聊聊。

**清水恵美**：（哭著點頭）謝謝...謝謝您...

---

**斎藤大輔**：（打了個哈欠揉著眼睛）好了，今天的情報量有點大，我這把老骨頭有點撐不住了。

**斎藤大輔**：莉子，帶他們去飯店休息吧。明天早上我們再詳談。

**伊藤莉子**：（敬禮）是！

**斎藤大輔**：（對李杰和恵美）你們也累了吧，好好休息一晚，有什麼問題明天再說。

**李杰**：（站起來鞠躬）謝謝您...

**清水恵美**：（也站起來鞠躬）非常感謝...

**斎藤大輔**：（揮手溫和地笑著）別客氣。歡迎回到地球。

**旁白**：莉子走到門口，打開門。

**伊藤莉子**：（回頭）走吧，我帶你們去飯店。

**旁白**：李杰和恵美跟著莉子走出辦公室。

**旁白**：離開前，李杰回頭看了一眼。

**旁白**：大輔坐在那張巨大的椅子上，看起來有些疲憊。

**旁白**：但他的眼神依然溫柔。

**旁白**：門緩緩關上。

**旁白**：李杰和恵美踏入走廊，朝著未知的明天前進。
