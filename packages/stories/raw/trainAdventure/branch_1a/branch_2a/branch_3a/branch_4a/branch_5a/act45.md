# 第四十五幕：不合格者的秘密

**旁白**：清晨的陽光從窗簾縫隙中透進來。

**旁白**：李杰緩緩睜開眼睛。

**旁白**：恵美還靠在他懷裡，睡得很熟。

**旁白**：她的臉龐在晨光中顯得格外柔和。

**旁白**：李杰不敢動，怕吵醒她。

**旁白**：就在這時——

**旁白**：手機突然響起。

**旁白**：恵美驚醒，迷迷糊糊地睜開眼睛。

**清水恵美**：（揉眼睛）嗯...

**旁白**：李杰拿起手機，是莉子打來的視訊電話。

**李杰**：（接通）早安...

**伊藤莉子**：（螢幕中笑著）早安～睡得好嗎？

**李杰**：（尷尬）嗯...還...還不錯...

**伊藤莉子**：（壞笑）是嗎～看你們的樣子，昨晚應該很愉快吧～

**旁白**：李杰的臉瞬間紅了。

**伊藤莉子**：（笑夠了）好啦好啦，準備好後打給我，我帶你們去吃早餐。你們有一小時時間，不然我就直接衝進去拖你們出來喔！

**李杰**：（急忙）知道了知道了！我們馬上準備！

**旁白**：莉子笑著掛斷電話。

**清水恵美**：（臉紅）莉子小姐...真的很調皮...

**李杰**：（苦笑）是啊...我們快點準備吧。

---

**旁白**：兩人輪流使用浴室。

**旁白**：刷牙、洗臉、整理儀容。

**旁白**：李杰換上昨天莉子給的新衣服——簡單的T恤和牛仔褲。

**旁白**：恵美也穿上準備好的休閒洋裝。

**旁白**：大約40分鐘後，兩人準備就緒。

**李杰**：（拿起手機）我打給莉子。

**旁白**：電話接通，莉子說她馬上過來。

**旁白**：幾分鐘後，門鈴響了。

**伊藤莉子**：（站在門口笑著）準備好了嗎？走吧，我帶你們去吃早餐！

**旁白**：莉子帶著他們穿過走廊，來到昨天的那棟建築。

**旁白**：但這次她沒有帶他們去大輔的辦公室，而是走向另一個房間。

**伊藤莉子**：（推開門）到了！

**旁白**：房間比辦公室小，但很溫馨。

**旁白**：中央擺著一張木製餐桌。

**旁白**：桌上已經擺好了豐盛的日式早餐。

**旁白**：烤鮭魚、味噌湯、白飯、納豆、煎蛋、還有各種醃菜。

**旁白**：大輔已經坐在桌前，看到他們進來，笑著揮手。

**斎藤大輔**：（爽朗地）早安！來來來，坐下吃吧！

**李杰**：（鞠躬）早安，打擾了。

**清水恵美**：（也鞠躬）早安...

**旁白**：四人圍坐在桌前。

**旁白**：莉子熟練地幫大家盛飯，大輔則開始夾菜。

**斎藤大輔**：（夾了一塊鮭魚給李杰）別客氣，多吃點。你們昨天應該累壞了吧？

**李杰**：（接過）謝謝...

**旁白**：早餐在輕鬆的氣氛中進行。

**旁白**：烤鮭魚外皮香脆，魚肉鮮嫩。味噌湯溫暖而鮮美。

**旁白**：這是他們回到地球後的第一頓正式早餐。

---

**旁白**：吃到一半時，大輔放下筷子。

**斎藤大輔**：（認真地）對了，李杰、恵美，我想問問...你們在那個世界有沒有發現什麼有用的資訊？

**斎藤大輔**：任何細節都好，這些情報對我們的調查很重要。

**旁白**：李杰放下碗，思考了片刻。

**李杰**：（開口）有的。那個世界有一個精靈村落，村落的領袖叫艾拉如。

**李杰**：他們似乎在那裡生活了很長時間。村落裡有一棵神樹，能讓兩個人建立精靈契約。

**斎藤大輔**：（點頭）精靈契約...具體是怎麼運作的？

**李杰**：兩個人同時觸摸神樹，如果雙方都符合資格，就能建立契約。建立契約後，才能前往終焉祭壇，打開回來的門。

**李杰**：（停頓）一開始，我想和恵美建立契約。但是...不知道為什麼，我不符合資格。

**清水恵美**：（補充）艾拉如先生和我都不知道為什麼李杰先生不符合資格...這種情況很少見...

**李杰**：所以我們等了一年的冷卻期，讓恵美和斎藤蓮建立契約。之後我們一起前往終焉祭壇。

**李杰**：（表情變得凝重）但是到了祭壇...一個叫凌墨的男人突然出現。他拿著黑劍，殺了護送我們的精靈劍士凱蘭。

**李杰**：（低聲）情況很危急...我只能選擇犧牲斎藤蓮，打開門帶恵美逃走。

**旁白**：大輔沉默地聽著，表情嚴肅。

**斎藤大輔**：（皺眉）那個凌墨...是什麼來歷？

**旁白**：就在這時，恵美突然開口。

**清水恵美**：（小聲）其實...凌墨是我第一次嘗試建立契約的人...

**旁白**：李杰猛地轉頭看向恵美。

**李杰**：（震驚）什麼？

**清水恵美**：（低著頭）45年前...我第一次來到那個世界時...凌墨也在那班列車上。

**清水恵美**：他是我大學時的朋友。我們一起嘗試建立契約，但是...他也不符合資格。

**清水恵美**：（眼眶泛紅）後來我沒辦法，只好和翔建立契約。然後...凌墨也跟著我們一起回來了。

**清水恵美**：（聲音顫抖）回來之後...我馬上又上了第二班列車，想要回去救翔...

**清水恵美**：從那之後...我就再也沒見過凌墨了...對我來說...也就一年半沒見...

**清水恵美**：但我沒想到...他會出現在終焉祭壇...而且變成那樣...

**旁白**：李杰的手緊緊握著筷子。

**旁白**：凌墨...也不符合資格...

**旁白**：和他一樣...

**李杰**：（內心獨白）艾拉如說過...不符合資格的情況非常罕見...

**李杰**：（內心獨白）如果凌墨也不符合資格...那我們之間...是不是有什麼共同的特徵？

**旁白**：李杰想起凌墨那冰冷的眼神、那把染血的黑劍、還有他殺人時毫無猶豫的動作。

**旁白**：如果他和凌墨有什麼相似之處...

**旁白**：那會是什麼？

**斎藤大輔**：（嚴肅地）這個資訊很重要。不符合資格...而且是兩個人...

**斎藤大輔**：（看著李杰）你們之間一定有什麼共通點。我會讓組織的研究團隊調查這件事。

**伊藤莉子**：（擔心地）李杰先生...你不會變成像凌墨那樣吧...

**李杰**：（勉強笑了）我...我不知道...

**旁白**：但他的聲音聽起來一點也不確定。

**旁白**：不符合資格的人...

**旁白**：凌墨...和他...

**旁白**：究竟有什麼相同之處？

**旁白**：這個問題，像一塊沉重的石頭，壓在李杰的心上。
