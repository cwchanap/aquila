# 第四十八幕：新生活的開始

**旁白**：飛機降落在成田國際機場。

**旁白**：李杰和恵美拖著行李，走出入境大廳。

**旁白**：人群中，一個熟悉的身影舉著牌子站在那裡。

**清水恵美**：（眼睛一亮）浩介！

**旁白**：清水浩介看到他們，露出溫暖的笑容。

**清水浩介**：（揮手）恵美！李杰先生！這邊！

**旁白**：三人會合。

**旁白**：浩介主動接過行李。

**清水浩介**：（笑著）飛機坐得還舒服嗎？

**清水恵美**：（點頭）嗯！很舒服！

**李杰**：（鞠躬）浩介先生，麻煩你特地來接我們了。

**清水浩介**：（擺手帶路）別客氣！你們兩個初來乍到，當然要來接！走吧，車在停車場，我帶你們去新家。

---

**旁白**：浩介開著一輛銀色的休旅車。

**旁白**：車子駛上高速公路，朝著東京市區前進。

**旁白**：窗外是熟悉又陌生的景色。

**旁白**：高樓大廈、廣告看板、飛馳的車輛...

**旁白**：一切都和25年前不太一樣了。

**清水浩介**：（邊開車邊說）對了，組織幫你們準備了一間公寓，在品川區，交通很方便，一年免租金。

**李杰**：（感激）真的嗎...太感謝了...

**清水浩介**：（笑了）而且啊...是一間公寓喔，不是兩間。你們兩個不是情侶嗎？住一起不是很正常？

**旁白**：李杰和恵美對視一眼，臉都紅了。

**李杰**：（內心獨白）果然...大輔先生的安排...

**旁白**：兩人的臉更紅了。

**清水浩介**：（哈哈大笑）別害羞啦！我看得出來你們感情很好！

---

**旁白**：車子開了大約一小時，終於到達公寓。

**旁白**：這是一棟新的大樓，外觀現代簡約。

**旁白**：浩介帶著他們搭電梯，來到12樓。

**清水浩介**：（掏出鑰匙開門）就是這裡了。

**旁白**：門打開，是一間約60平方米的一房一廳公寓。

**旁白**：客廳寬敞明亮，有大片落地窗。

**旁白**：廚房、浴室、臥室，應有盡有。

**旁白**：家具和電器都已經準備好了。

**清水恵美**：（驚嘆）好漂亮的房子...

**李杰**：（環顧四周）真的...比我想像中還好...

**清水浩介**：（拍了拍冰箱指著電視）這些家電都是最新款的。那台電視是8K的，還有AI語音助手。桌上的平板可以控制家裡所有的智能家電。

**旁白**：李杰看著那些設備，頭開始痛了。

**李杰**：（苦笑嘆氣）8K...AI...智能家電...我已經跟不上時代了...25年的科技進步...太可怕了...

**清水恵美**：（更沮喪數著手指）我更慘...上次回來已經過了20年，那時候我完全沒時間適應，只想著要救翔。現在又過了25年，加起來都45年了...

**清水恵美**：（抱頭）我感覺自己像個老奶奶...完全不懂現在的世界...

**清水浩介**：（忍不住笑調皮地）老奶奶倒也不錯！妳想想，如果妳沒去那個世界，現在應該是63歲的真·老奶奶了！

**旁白**：李杰忍不住笑出聲。

**李杰**：（笑著）浩介先生說得對...

**清水恵美**：（氣呼呼舉起拳頭輕輕打了浩介的頭）浩介！你怎麼可以這樣說！

**旁白**：力道很輕，更像是撒嬌。

**清水浩介**：（笑著揉頭認真地）好好好，我錯了！不過說真的，慢慢適應就好，組織會提供協助的。而且你們還年輕，學習能力應該沒問題。

**李杰**：（點頭）嗯...只能慢慢學了...

---

**清水浩介**：（看了看手錶拿出一個文件袋）對了，我待會還要去工作。你們先好好休息，熟悉一下環境。這裡面有一些重要的文件，包括住址證明、銀行帳戶資料、還有每個月的生活補助說明。

**李杰**：（接過）生活補助？

**清水浩介**：（點頭微笑）對。組織每個月會提供40萬日圓的補助，維持一年。應該足夠你們生活和適應這個世界了。

**清水恵美**：（震驚）40萬...這麼多...

**清水浩介**：（拍了拍李杰的肩膀）別客氣，這是組織應該做的。你們提供的情報很有價值。週末我會再來找你們，我們一起吃個飯，好好聊聊。

**清水恵美**：（點頭）好！

**李杰**：（鞠躬）謝謝浩介先生。

**清水浩介**：（揮手走向門口）好啦好啦，我走了！你們好好享受兩人世界！別這麼客氣！你可是我姐夫...不對，是我姐的男朋友！

**旁白**：恵美的臉瞬間紅了。

**清水恵美**：（慌張）浩...浩介！

**清水浩介**：（哈哈大笑）

**旁白**：浩介關上門離開了。

**旁白**：公寓裡只剩下李杰和恵美兩人。

---

**旁白**：兩人站在客廳，突然有些不知所措。這是他們第一次真正意義上的「同居」。

**清水恵美**：（小聲指著臥室）那個...李杰先生...房間只有一個...

**李杰**：（也有些尷尬臉紅）我可以睡沙發...

**清水恵美**：（搖頭說不下去臉紅得像蘋果）不用...我們已經...

**旁白**：李杰看著她，心跳加速。

**旁白**：在夏威夷，他們確認了彼此的心意。

**旁白**：在夏威夷，他們也睡過同一張床。

**旁白**：但那是在飯店。

**旁白**：而現在...這是他們的家。

**旁白**：真正的、屬於他們兩個人的家。

**李杰**：（溫柔地走近）恵美...

**清水恵美**：（抬頭看他）嗯...

**李杰**：（微笑）我們...一起加油吧。

**李杰**：重新適應這個世界，一起建立新的生活。

**清水恵美**：（眼眶泛紅點頭）嗯...一起...

**旁白**：李杰輕輕牽起恵美的手。

**旁白**：兩人相視而笑。

**旁白**：窗外，東京的街道車水馬龍。

**旁白**：陽光灑進客廳，溫暖而明亮。

**旁白**：他們的新生活...

**旁白**：從這裡開始了。
