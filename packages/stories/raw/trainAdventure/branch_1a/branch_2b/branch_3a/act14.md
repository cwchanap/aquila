# 第十四幕：重逢與隔閡

**旁白**：大約過了十分鐘，恵美的情緒漸漸平復下來。

**旁白**：她的眼睛紅腫，睫毛上還掛著淚珠，臉頰上有明顯的淚痕。她用手背抹了抹臉，深呼吸著，努力讓自己冷靜。

**高橋美咲**：（溫柔地扶著她的手臂）來，恵美，先站起來吧。

**旁白**：美咲小心翼翼地將恵美扶起來。恵美雙腿還有些發軟，靠在美咲肩上才勉強站穩。

**高橋美咲**：（拍拍她的背）恵美，別太難過了。我們...我們一定會找到辦法的。

**高橋美咲**：不管發生了什麼，我們都會一起面對，好嗎？

**清水恵美**：（聲音沙啞）謝謝...美咲...

**旁白**：恵美的聲音聽起來疲憊而遙遠，像是隔著一層厚厚的玻璃。

**旁白**：她沒有多說什麼，只是靜靜地站在那裡，目光茫然地望著眼前焦黑的廢墟，彷彿靈魂還沉浸在某個遙遠的悲傷之中。

**旁白**：焦炭的氣味依然濃重，風吹過時，灰燼在空中飛舞。遠處破損的木門發出吱呀的聲響，迴盪在死寂的村落中。

---

**旁白**：李杰在一旁看著這一切，心中猶豫不決。

**李杰**：（內心）我該問嗎？這個時候問會不會太殘忍？

**李杰**：（內心）但是...如果不問清楚，我們根本不知道接下來該怎麼辦。而且...她明顯知道些什麼。

**旁白**：李杰深吸一口氣，決定開口。但他選擇了一種比較委婉的方式。

**李杰**：（語氣盡量保持平和）清水...妳以前...來過這個地方嗎？

**旁白**：恵美的身體微微一顫，像是被冰冷的水潑到。

**旁白**：她緩緩轉過頭，用那雙紅腫的眼睛看著李杰。眼神中有驚訝，有猶豫，還有一絲防備。

**李杰**：我只是...看妳的反應，妳好像對這裡...有很深的感情。

**李杰**：如果妳知道些什麼，也許可以幫助我們理解現在的狀況...

**旁白**：恵美沉默了很久。她的視線從李杰臉上移開，落在地面的焦黑石板上。

**旁白**：她的手指不自覺地握緊又鬆開，像是在進行某種內心的掙扎。

**旁白**：美咲察覺到氣氛的變化，停止了說話，只是安靜地站在旁邊。

**旁白**：最後，恵美深吸一口氣，抬起頭。她的眼神變得平靜，但那是一種築起高牆後的平靜。

**清水恵美**：李杰先生...等時機到了，我會告訴你我知道的一切。

**清水恵美**：但是現在...請讓我保持沉默。

**旁白**：她的聲音很平靜，卻帶著一種不容置疑的堅定。

---

**李杰**：（內心）她果然知道些什麼...但她不願意說。

**旁白**：李杰看著她的眼睛。那雙眼睛裡不再只有膽怯和依賴，而是多了一層深沉的、難以穿透的隔閡。

**旁白**：這個曾經怯生生跟在他身後的女孩，此刻彷彿成了一個陌生人。

**李杰**：我明白了。

**旁白**：李杰的語氣很平淡，但心中卻泛起了複雜的情緒。

**李杰**：（內心）她在隱瞞...而且是很重要的事情。

**李杰**：（內心）我不知道她的目的是什麼，但從現在開始，我必須更加小心。

**旁白**：一顆名為「不信任」的種子，在李杰心中悄悄發芽。

---

**高橋美咲**：（似乎想要緩和氣氛，拍了拍手）好啦！既然我們都到這裡了，不如四處看看吧？

**高橋美咲**：說不定能找到什麼有用的東西！

**旁白**：美咲的語氣輕快，彷彿試圖用她的樂觀來驅散這片廢墟帶來的陰霾。

**高橋美咲**：而且...也許能找到關於其他人的線索？拓海他們...還有山本大叔...

**旁白**：就在美咲話音剛落的瞬間，遠處突然傳來了一個熟悉的聲音。

**？？？**：李杰！恵美！

**旁白**：三人同時一愣。

**李杰**：（猛地轉身）這聲音...

**旁白**：濃霧中，一個高大的身影正快步朝他們走來。腳步聲急促而沉重，踩在碎石上發出清脆的響聲。

**旁白**：隨著距離拉近，那個人的面容漸漸清晰——灰白的頭髮、寬闊的肩膀、風塵僕僕的神情。

**李杰**：（驚訝地）山本大叔？！

**旁白**：那正是山本浩二！

**旁白**：他的外套有些破損，袖口處有明顯的撕裂痕跡。臉上沾著灰塵和泥土，額頭上還有一道淺淺的擦傷。但看起來並無大礙。

**旁白**：他加快腳步，臉上露出如釋重負的笑容，眼角甚至有些濕潤。

**山本浩二**：（喘著氣，聲音激動）太好了！你們還活著！

**山本浩二**：我還以為...我還以為再也見不到你們了！

**旁白**：山本大叔的聲音中帶著難以掩飾的喜悅和慶幸，在這片死寂的廢墟中，顯得格外溫暖。
