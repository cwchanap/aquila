# 第十幕：迷失的方向

**旁白**：霧氣依然濃厚，將兩人籠罩在一片白茫茫之中。李杰靠著樹幹閉目休息，斎藤蓮則縮在一旁，不時發出痛苦的低吟。

**旁白**：寂靜持續了大約十分鐘。就在李杰以為可以稍微休息一下時——

**斎藤蓮**：啊！我的⋯⋯我的地圖！還有打火機！

**旁白**：斎藤蓮突然尖叫起來，雙手慌張地在身上各個口袋摸索。他的臉色變得慘白，眼中閃過驚恐的神色。

**斎藤蓮**：不見了⋯⋯怎麼會⋯⋯明明還在的⋯⋯

**旁白**：他翻遍了所有口袋，甚至把外套脫下來仔細檢查，但顯然什麼也沒找到。

**李杰**：怎麼了？發生什麼事了？

**旁白**：李杰睜開眼睛，故作不知情地看著斎藤蓮慌張的樣子。

**斎藤蓮**：我⋯⋯我⋯⋯

**旁白**：斎藤蓮的話說到一半突然停住了。他意識到自己說漏了嘴，臉色變得更加難看。

**斎藤蓮**：沒⋯⋯沒什麼⋯⋯我⋯⋯我剛才在做夢⋯⋯說夢話而已⋯⋯

**李杰**：做夢？我剛才明明聽到你說什麼地圖和打火機⋯⋯你有附近的地圖嗎？

**旁白**：李杰裝作好奇的樣子，目光緊盯著斎藤蓮的反應。

**斎藤蓮**：沒有！我沒有什麼地圖！你聽錯了！

**旁白**：斎藤蓮的聲音變得尖銳，帶著明顯的慌張。他別過頭，不敢看李杰的眼睛。

**斎藤蓮**：如果我有地圖，我還會被困在這種鬼地方嗎？還會跟你一起在這裡受罪嗎？

**李杰**：這樣啊⋯⋯我還以為你有什麼線索呢。

**旁白**：李杰沒有繼續追問，只是點了點頭，重新閉上眼睛靠在樹幹上。

**旁白**：但他的手下意識地摸了摸口袋裡的地圖和打火機。

**李杰**：(內心)他在說謊。而且說謊的技巧還真是差勁。

**李杰**：(內心)地圖和打火機⋯⋯他為什麼要隱瞞？這些東西對他來說意味著什麼？

**李杰**：(內心)還有，他為什麼會有這片森林的地圖？他是不是早就知道這個地方？

**旁白**：李杰心中的疑問越來越多，但他決定暫時不動聲色。現在不是追問的時候，先觀察斎藤的行動再說。

**旁白**：斎藤蓮坐在一旁，臉色陰晴不定。他時不時偷偷瞄向李杰，又快速移開視線，顯然心裡七上八下。

**旁白**：時間一分一秒地流逝。兩人就這樣在霧中沉默地等待著。

**旁白**：十分鐘過去了。二十分鐘過去了。但霧氣沒有絲毫減弱的跡象，能見度依然極低。

**李杰**：這霧看起來不會散了。

**旁白**：李杰站起身，拍了拍身上的塵土。

**李杰**：斎藤，你的腳現在怎麼樣？能走嗎？

**斎藤蓮**：我⋯⋯我試試看⋯⋯

**旁白**：斎藤蓮撐著樹幹試圖站起來，但右腳剛一用力，劇痛立刻讓他發出一聲悶哼。

**斎藤蓮**：嘶⋯⋯不行⋯⋯還是很痛⋯⋯

**李杰**：看來還是不能走。來吧，我扶你。

**旁白**：李杰走過去，伸手扶起斎藤蓮。斎藤蓮的體重再次壓在他身上。

**斎藤蓮**：你⋯⋯你要去哪裡？

**李杰**：不能一直待在這裡。我們得找個地方休息，至少要有個遮蔽的地方。

**斎藤蓮**：那⋯⋯那我們回去吧⋯⋯回我們來的方向⋯⋯

**李杰**：回去？

**旁白**：李杰環顧四周，濃厚的霧氣遮蔽了一切。他已經完全失去了方向感。

**李杰**：(內心)糟糕⋯⋯我根本不知道哪邊是來的方向。剛才在霧中搜尋清水的時候，已經繞了好幾圈⋯⋯

**李杰**：(內心)現在連十字路口在哪個方向都不清楚了。

**斎藤蓮**：喂⋯⋯你在想什麼？我們回去啊⋯⋯

**李杰**：好，我們走吧。

**旁白**：李杰沒有說出自己已經迷路的事實。他扶著斎藤蓮，隨便選了一個看起來像是路的方向開始前進。

**旁白**：腳下的地面時而堅硬，時而鬆軟。霧氣中，周圍的樹木像幽靈般若隱若現。

**斎藤蓮**：你⋯⋯你確定是這個方向嗎⋯⋯

**李杰**：應該沒錯。

**旁白**：李杰說得很肯定，但心裡其實毫無把握。在這種能見度下，每個方向看起來都一樣。

**旁白**：他們在霧中緩慢前進。斎藤蓮一瘸一拐的，每走一步都要停頓一下。李杰必須用很大的力氣才能支撐住他。

**斎藤蓮**：痛⋯⋯走得好慢⋯⋯

**李杰**：忍耐一下。應該快到了。

**旁白**：但李杰自己也不知道「快到」是指什麼。快到十字路口？快到某個遮蔽處？還是快到更深的危險之中？

**旁白**：霧氣在他們身邊流動，像活物一樣纏繞著他們。前方的路看不清楚，後方的路也早已消失在白茫茫之中。

**旁白**：他們就像兩個在迷宮中盲目行走的人，不知道前方等待著他們的是什麼。

**李杰**：(內心)清水⋯⋯妳現在在哪裡？是不是也在這片霧中迷路了？

**李杰**：(內心)還有大叔和那些大學生⋯⋯他們現在的狀況如何？

**李杰**：(內心)我們⋯⋯到底能不能活著離開這片森林？

**旁白**：這些疑問在李杰心中盤旋，但沒有任何答案。

**旁白**：他只能繼續往前走，扶著斎藤蓮，一步一步地在霧中摸索前進。

**旁白**：直到——

**旁白**：前方的霧氣中，隱約出現了一個黑色的巨大輪廓。

**旁白**：那不是樹木，也不是岩石。而是某種⋯⋯建築物。

**李杰**：那是⋯⋯

**旁白**：李杰停下腳步，瞇起眼睛試圖看清楚那個輪廓。

**旁白**：在霧氣的遮掩下，那個建築物的全貌若隱若現。但可以確定的是——

**旁白**：那裡，有人造的建築。

**旁白**：而在這片詭異的森林裡，這意味著什麼？
