# 第十三幕：霧中的足跡

**旁白**：李杰環顧屋內，看著縮在角落裡的陽菜和翔太，以及坐在椅子上喘氣的斎藤蓮。他的身體已經疲憊到了極點，但心中卻無法平靜下來。

**李杰**：(內心)兩個孩子的父母都在外面⋯⋯如果不快點找到他們，可能會出事。

**旁白**：他深吸一口氣，做出了決定。

**李杰**：陽菜、翔太，我出去找你們的爸爸媽媽。你們待在這裡，不要出去，知道嗎？

**小林陽菜**：(眼睛一亮)真的嗎？大哥哥你要去找爸爸媽媽？

**小林翔太**：(跑過來拉著李杰的衣角)謝謝大哥哥！

**李杰**：(摸了摸翔太的頭)我會盡快回來的。

**斎藤蓮**：(抬起頭)喂，你要出去？那我怎麼辦？

**李杰**：你就待在這裡休息。幫我照看一下這兩個孩子。

**斎藤蓮**：什麼？我可沒答應要當保姆！而且我的腳⋯⋯

**李杰**：你的腳正好需要休息。反正你也走不了，就待在這裡吧。

**旁白**：李杰沒有給斎藤蓮反駁的機會，轉身走向門口。

**小林陽菜**：(擔心地)大哥哥⋯⋯你要小心⋯⋯

**李杰**：(回頭微笑)放心。

**旁白**：他推開門，再次踏入濃霧之中。冰冷的空氣撲面而來，讓他精神一振。

**旁白**：關上門後，李杰站在門廊上，開始仔細觀察周圍的地面。

**李杰**：(內心)霧這麼濃，視線不好。但地面應該會留下痕跡⋯⋯

**旁白**：他蹲下身，仔細端詳著泥土地面。果然，在濕潤的泥土上，他看到了幾組腳印。

**旁白**：有些腳印很小，應該是孩子們的。有些比較大，明顯是成年人的。

**旁白**：他沿著腳印的方向追蹤，發現有兩組成年人的腳印都朝著北邊延伸。一組稍大一些，另一組稍小一些。

**李杰**：(內心)應該是他們的父母。看腳印的深淺和間距⋯⋯較大的那組走得比較快，可能是父親。較小的這組走得比較慢，而且有些凌亂⋯⋯

**旁白**：李杰沿著腳印開始向北走。霧氣依然濃厚，但至少有了明確的方向。

**旁白**：他保持著警惕，一邊跟隨腳印，一邊觀察著周圍的環境。樹木在霧中若隱若現，偶爾傳來風吹樹葉的聲音。

**旁白**：十分鐘。二十分鐘。三十分鐘。

**旁白**：就在李杰開始懷疑自己是否走錯方向時，前方的霧氣中出現了一個人影。

**旁白**：那是一個女人，靠著一棵大樹坐在地上。她的頭低垂著，雙手抱著自己的腳踝，整個人看起來虛弱不堪。

**李杰**：(加快腳步)喂！妳還好嗎？

**旁白**：那個女人聽到聲音，猛地抬起頭。她的臉上滿是驚恐，但看到是李杰後，眼中閃過一絲希望。

**中年女性**：你⋯⋯你是⋯⋯

**李杰**：(走到她面前蹲下)我叫李杰。是從火車上下來的乘客。妳是不是⋯⋯陽菜和翔太的母親？

**中年女性**：(眼眶立刻紅了)你⋯⋯你見到我的孩子了？他們還好嗎？

**李杰**：他們很好，在小屋裡等著。我來找妳。

**中年女性**：(眼淚流了下來)太好了⋯⋯太好了⋯⋯我還以為⋯⋯我還以為再也見不到他們了⋯⋯

**李杰**：妳叫什麼名字？

**中年女性**：(擦著眼淚)我⋯⋯我叫小林智子⋯⋯

**李杰**：小林女士，妳的腳怎麼了？

**小林智子**：我⋯⋯我出來找我先生⋯⋯但是霧太濃了，我迷路了⋯⋯

**小林智子**：然後不小心被樹根絆倒⋯⋯腳踝扭傷了⋯⋯痛得站不起來⋯⋯

**旁白**：李杰低頭看向她的腳踝，確實已經腫了起來，和斎藤蓮的情況差不多。

**李杰**：妳先生呢？他往哪個方向走了？

**小林智子**：我⋯⋯我不知道⋯⋯我跟著他的腳印走，但後來霧越來越濃，我就看不清了⋯⋯

**旁白**：李杰站起身，環顧四周。他突然意識到，這裡的霧氣似乎比之前在四岔路口時稍微淡了一些。

**李杰**：(內心)霧的濃度⋯⋯變弱了？不，應該說是這一帶的霧本來就比較淡。但還是很濃，能見度依然很低。

**旁白**：他蹲下身，仔細檢查地面上的腳印。除了智子的腳印，還有另一組更深、更大的腳印繼續向北延伸。

**李杰**：(內心)這應該是她先生的腳印。他繼續往北走了⋯⋯

**李杰**：小林女士，我會幫妳找妳先生的。但現在先帶妳回小屋。妳的孩子們很擔心妳。

**小林智子**：(搖頭)不⋯⋯不行⋯⋯我先生還在外面⋯⋯我必須找到他⋯⋯

**李杰**：妳現在這個狀態根本走不了。我先送妳回去，然後再出來找妳先生。

**小林智子**：可是⋯⋯

**李杰**：相信我。我動作比較快，而且現在知道他往哪個方向走了。妳留在這裡只會讓情況更糟。

**旁白**：小林智子看著李杰堅定的眼神，咬了咬嘴唇，最終點了點頭。

**小林智子**：(聲音顫抖)拜託你了⋯⋯一定要找到他⋯⋯

**李杰**：我會的。來，我扶妳起來。

**旁白**：李杰伸手扶起小林智子。她的體重壓在他身上，受傷的腳懸空著不敢著地。

**旁白**：他們開始往回走。小林智子每走一步都要停頓一下，李杰必須用很大的力氣才能支撐住她。

**小林智子**：(愧疚地)對不起⋯⋯給你添麻煩了⋯⋯

**李杰**：別說這些。專心走路。

**旁白**：他們沿著來時的腳印緩慢前進。霧氣在他們身邊流動，前方的路看不清楚，但至少還有腳印指引方向。

**旁白**：小林智子受傷的腳讓她每走一步都痛苦萬分。李杰必須承擔她大部分的體重，自己的體力也在快速消耗。

**旁白**：他們走走停停，每走一段路就要休息一下。時間在這緩慢的行進中一點一滴地流逝。

**旁白**：二十分鐘。三十分鐘。四十分鐘。

**旁白**：李杰的肩膀開始發酸，雙腿也越來越沉重。但他咬牙堅持著，一步一步往前走。

**小林智子**：(虛弱地)對不起⋯⋯我⋯⋯我走得太慢了⋯⋯

**李杰**：(喘著氣)沒關係⋯⋯慢慢來⋯⋯

**旁白**：又過了大約二十分鐘，小屋的輪廓終於在霧中顯現。從找到智子到回到小屋，總共花了將近一個小時。

**李杰**：(鬆了一口氣)到了。

**小林智子**：(看到小屋，眼中再次泛淚)孩子們⋯⋯

**旁白**：李杰扶著她走上門廊，敲了敲門。

**李杰**：是我，李杰。

**旁白**：門立刻被打開了。陽菜和翔太站在門口，看到母親的瞬間，眼淚就湧了出來。

**小林陽菜**：媽媽！

**小林翔太**：媽媽！

**旁白**：兩個孩子衝過來，緊緊抱住母親。小林智子也蹲下身，將兩個孩子摟在懷裡。

**小林智子**：(哭著)對不起⋯⋯媽媽讓你們擔心了⋯⋯

**旁白**：李杰站在一旁，看著這溫馨的一幕。但他知道，還有一個人在外面等待救援。

**旁白**：他的目光望向北方，霧氣深處，那裡還有一個迷失的父親。
