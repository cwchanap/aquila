# 第三十一幕：遺忘的循環

**旁白**：李杰站在三扇門前，凝視著散發詭異光芒的左右兩門。他深吸一口氣，握緊手中的白劍。

**李杰**：（堅定地）右邊。

**斎藤蓮**：（緊張地）你...你確定嗎？

**李杰**：（點頭）我們走。

**旁白**：李杰邁步走向右邊的門。斎藤猶豫了片刻，最終還是跟了上去。

**旁白**：當李杰的手觸碰到門把的瞬間，一股強烈的紫色光芒從門縫中爆發出來，瞬間吞沒了兩人。

**李杰**：（驚呼）什麼——

**旁白**：光芒越來越強烈，刺眼得無法睜開眼睛。李杰感覺自己的身體在旋轉，在飛行，在墜落。周圍的一切都在扭曲，在融化，在重組。

**旁白**：然後，一切歸於黑暗。

---

**旁白**：不知道過了多久。李杰緩緩睜開眼睛。

**李杰**：（迷糊地）...嗯？

**旁白**：他的視線逐漸聚焦。天花板。白色的天花板。上面有柔和的燈光在閃爍。

**李杰**：（坐起身）這是...

**旁白**：他環顧四周。這是一節火車車廂。兩排座椅整齊地排列著，藍色的絨布坐墊，不鏽鋼的扶手。車窗外一片漆黑，只能看到自己模糊的倒影。車廂裡很安靜，只有火車行駛時規律的隆隆聲。

**李杰**：（困惑地）我...我怎麼在火車上？

**旁白**：他的頭有些暈眩，腦中一片混沌。他努力回想剛才發生了什麼事，但腦海中一片空白。

**李杰**：（內心）我記得...我在聚會上...喝了點酒...然後上了火車回家...對...我應該是在回家的路上...

**旁白**：但總覺得哪裡不對勁。李杰伸手摸向口袋，掏出手機。

**旁白**：他按下電源鍵，但螢幕沒有反應。他再按了幾次，螢幕依然一片漆黑。手機完全沒電了。

**李杰**：（皺眉）沒電了？

**旁白**：李杰看向車窗外。窗外依然是一片漆黑，什麼都看不見。不對。那不是普通的黑暗。那是隧道。火車正在隧道裡行駛。

**李杰**：（內心）隧道...中央線確實有幾個小隧道...但這個也太長了吧...

**旁白**：李杰環顧車廂。車廂裡空蕩蕩的。除了他自己，只有另外一個人。那是一個年輕男子，坐在幾排座位之外。他戴著眼鏡，看起來二十歲左右。那個男子也剛剛醒來，正困惑地環顧四周。

**李杰**：（內心）這個人...我好像在哪裡見過...

**旁白**：但李杰怎麼也想不起來在哪裡見過他。就在這時，那個戴眼鏡的男子注意到了李杰。

**眼鏡男**：（站起身，緊張地）你...你是誰？這...這到底怎麼回事？這裡為什麼一個人都沒有？

**旁白**：男子的聲音裡帶著驚慌和困惑。

**眼鏡男**：（更加慌張）還有這個隧道！為什麼這麼長？我明明是在回立川的路上...你是誰？為什麼只有我們兩個人在這輛車上？這到底是怎麼回事？

**旁白**：男子快步走向李杰，聲音裡帶著一絲歇斯底里。

**李杰**：（搖頭）我...我也不知道。我也剛醒來。我記得我是從聚會回家的路上...

**眼鏡男**：（稍微冷靜一點）聚會...你也是？

**李杰**：（點頭）我應該是要回三鷹。你呢？

**眼鏡男**：（推了推眼鏡）立川。我住在立川。

**旁白**：兩人對視了片刻。

**李杰**：（伸出手）我叫李杰。

**眼鏡男**：（猶豫了一下，握手）...斎藤蓮。

**旁白**：握手之後，兩人沉默了片刻。斎藤不安地看著空蕩蕩的車廂，李杰則低頭看著自己的雙手。

**旁白**：就在這時，一種奇怪的感覺湧上李杰心頭。

**李杰**：（困惑地看著周圍）...奇怪...

**斎藤蓮**：（警戒地）怎麼了？

**李杰**：（看著自己的座位周圍）我...我好像少了什麼東西...

**旁白**：李杰轉身查看自己剛才坐的位置。座位上空蕩蕩的，什麼都沒有。他又檢查了座位下方，還有旁邊的置物架。

**旁白**：什麼都沒有。

**斎藤蓮**：（疑惑）少了什麼？你的包包嗎？

**李杰**：（搖頭，困惑地）不...不是包包...是...

**旁白**：李杰說不出來。他不知道自己在找什麼。但那種感覺很強烈——彷彿有什麼非常重要的東西應該在他身邊，但現在卻不見了。

**旁白**：那是一種空虛感。不是物理上的空虛，而是一種更深層的、難以言喻的失落感。

**李杰**：（低聲）我不知道...就是覺得...好像少了什麼很重要的東西...

**旁白**：他下意識地伸出右手，手掌微微握緊，彷彿在握著什麼。但手中空無一物。

**旁白**：那個姿勢看起來很自然，彷彿他的手習慣了握著某個東西。但現在，那個位置是空的。

**斎藤蓮**：（看著他的手，困惑）你...你在找什麼？

**李杰**：（慢慢放下手，搖頭）我不知道...可能是我想太多了...

**旁白**：但那種感覺揮之不去。李杰能感覺到，有什麼非常、非常重要的東西消失了。那個東西應該一直在他手中，一直陪伴著他。

**旁白**：但那是什麼？為什麼他會有這種感覺？

**旁白**：他完全想不起來。

**斎藤蓮**：（看著窗外）這個隧道...到底有多長啊...

**旁白**：火車依然在黑暗的隧道中前進，沒有停下的跡象。車窗外一片漆黑，只有偶爾閃過的隧道燈光。

**旁白**：而在這節空蕩蕩的車廂裡，只有兩個困惑的乘客。他們不知道自己為什麼會在這裡，不知道火車要開往何處。

**旁白**：李杰再次看了看自己空蕩蕩的雙手。那種失落感依然強烈，彷彿他的手本該握著什麼。

**旁白**：一種難以言喻的不安感湧上心頭。彷彿...有什麼重要的事情被遺忘了。彷彿...這一切都不應該是這樣的。但當他試圖去回想時，腦中卻只剩下一片空白。

**旁白**：火車繼續在黑暗中前進，通往未知的目的地。
