# 第三十五幕：意外的方法

**旁白**：李杰盯著那扇緊閉的門，腦中不斷思考著各種可能性。

**李杰**：（轉向斎藤）你有USB線嗎？

**斎藤蓮**：（皺眉）什麼？

**李杰**：USB充電線。你有嗎？

**斎藤蓮**：（激動地）你在開玩笑嗎？誰會隨身帶那種東西？

**斎藤蓮**：（繼續）我出門前都會把手機充到一百趴！根本不需要帶充電線！

**旁白**：李杰嘆了口氣。他不想跟這個人爭論。

**李杰**：（低聲）...算了。

**旁白**：他轉身，開始在這個小小的金屬空間裡四處查看。

**旁白**：牆角。地面。門框周圍。每一個角落他都仔細檢查。

**旁白**：然後，在靠近通道入口的地上，他看到了一個灰色的東西。

**旁白**：李杰走過去，蹲下身，伸手拿起那個東西。

**旁白**：那是一條繩子。

**李杰**：（拿起繩子）...這是？

**旁白**：但這條繩子的狀態很糟糕。它看起來已經很舊了，表面粗糙，顏色發灰，有些地方甚至已經開始腐爛。

**旁白**：李杰輕輕拉了拉繩子。雖然腐爛，但韌性還在，應該還能用。

**旁白**：他拉直繩子，目測長度。大約兩米左右。

**李杰**：（內心）夠長了...

**旁白**：李杰拿著繩子回到門邊。他開始在繩子的一端打結，試圖做出一個圓圈。

**旁白**：繩子很粗，不太容易打結。但經過幾次嘗試，他總算做出了一個簡單的套索。

**旁白**：李杰從洞口往裡看，確認了門把的位置。

**旁白**：然後，他把繩子從洞口伸進去。

**旁白**：繩子的前端晃晃悠悠地垂在空中。李杰試圖擺動繩子，讓套索能夠套進門把裡。

**旁白**：但這太難了。

**旁白**：繩子太軟，又太長，根本無法控制方向。每次套索接近門把時，都會因為繩子的軟度、長度和慣性而偏離目標。

**旁白**：李杰試了五次、十次、十五次...

**旁白**：每一次都失敗了。

**斎藤蓮**：（在後面抱怨）不行的話就算了吧...這根本不可能..

**斎藤蓮**：（繼續）我們是不是應該想其他辦法...

**旁白**：李杰的額頭開始冒汗。他咬緊牙關，再次嘗試。

**旁白**：但繩子依然無法套進門把。

**斎藤蓮**：（更大聲）我說你聽到沒有？這樣不行啊...

**李杰**：（突然轉身，沉著臉）你能不能閉嘴？

**旁白**：李杰的聲音很冷，帶著明顯的怒意。

**李杰**：（繼續）你在那邊抱怨有什麼用？你有幫上任何忙嗎？沒有的話就給我閉嘴！

**旁白**：斎藤被李杰的語氣嚇到了，愣在原地。

**斎藤蓮**：（委屈地）我...我只是...

**李杰**：（轉身，不再理他）...

**旁白**：斎藤張了張嘴，但最終什麼都沒說。他只是站在原地，臉色難看。

**旁白**：李杰繼續嘗試用繩子套門把。但無論如何，都無法成功。

**旁白**：就在李杰快要放棄的時候——

**斎藤蓮**：（憤怒地）夠了！

**旁白**：斎藤突然大步走上前。

**斎藤蓮**：（激動地）你這樣根本不可能成功！繩子這麼軟怎麼可能套得進去？！

**李杰**：（冷冷地）那你有更好的辦法嗎？

**斎藤蓮**：（抓狂）當然有！隨便都比你這樣好！

**旁白**：斎藤伸手就要去搶李杰的手電筒。

**李杰**：（往後退）你幹什麼？！

**斎藤蓮**：（憤怒地）給我手電筒！

**旁白**：斎藤猛地從李杰手中搶走手電筒。

**李杰**：（驚訝）喂！

**旁白**：房間瞬間陷入黑暗。只剩下門後那個房間微弱的光線從洞口透出來。

**斎藤蓮**：（氣呼呼地拿著手電筒）我們直接用這個砸！砸爛那個門把！

**李杰**：（皺眉）你瘋了嗎？手電筒摔壞的話，我們就什麼光源都沒有了。

**斎藤蓮**：（大吼）那也比你在這裡浪費時間好！

**旁白**：斎藤舉起手電筒，就要往洞裡扔。

**李杰**：（急忙）等等！

**旁白**：李杰一把抓住斎藤的手腕。

**李杰**：（快速說）如果真的要扔，至少用繩子綁住！這樣就算沒打中，我們也能把手電筒拉回來。

**旁白**：斎藤愣了一下。

**斎藤蓮**：（遲疑）...用繩子綁？

**李杰**：（點頭）對。綁在手電筒上，然後扔向門把。手電筒夠硬，如果撞到門把的話...

**旁白**：李杰沒有說完，但斎藤已經明白了。

**斎藤蓮**：（鬆開手）...那就快點綁。

**旁白**：李杰接過手電筒，打開照明。然後他開始用繩子綁手電筒。

**旁白**：他把繩子繞在手電筒的中段，打了好幾個結，確保它不會鬆脫。

**李杰**：（拉了拉繩子）應該可以了。

**斎藤蓮**：（突然開口）等等，我站門前面。

**李杰**：（疑惑）什麼？

**斎藤蓮**：（走向門）你扔的時候，我盯著門把。手電筒一打到，我就立刻轉這邊的門把。

**斎藤蓮**：（繼續，語氣還是有點不爽）說不定能趁著撞擊的力道把門打開。不然你扔一次沒開，還要扔第二次第三次，煩死了。

**旁白**：李杰看著斎藤，沒想到他在生氣的狀態下還能想到這點。

**李杰**：（簡短地）...好。

**旁白**：斎藤走到門前，雙手握住門把。他的眼睛緊緊盯著門內側那個把手。

**斎藤蓮**：（深吸一口氣）我準備好了。

**旁白**：李杰站在洞口旁邊，握著綁了繩子的手電筒。

**旁白**：他深吸一口氣，調整姿勢。

**旁白**：然後——

**旁白**：他用力將手電筒扔進洞裡！

**旁白**：手電筒在空中劃過一道弧線，光束在黑暗中晃動。

**旁白**：「砰」——

**旁白**：一聲清脆的撞擊聲！

**旁白**：手電筒準確地擊中了門內側的把手！

**斎藤蓮**：（大喊）就是現在！

**旁白**：斎藤猛地轉動門外的把手！

**旁白**：「喀噠」——

**旁白**：門鎖打開的聲音！

**斎藤蓮**：（用力推門）開了！

**旁白**：門被推開了。

**旁白**：斎藤踉蹌著衝進房間，差點摔倒。李杰緊跟在後，拉著繩子把手電筒拿了回來。

**旁白**：兩人踏進房間的瞬間——

**旁白**：「啪」——

**旁白**：天花板上的燈突然全部亮起！

**旁白**：明亮的白光瞬間充滿整個房間，刺得兩人都瞇起眼睛。

**斎藤蓮**：（驚叫）啊！什麼？！

**旁白**：斎藤嚇得往後跳了一步，差點撞到李杰。

**斎藤蓮**：（驚恐地）燈...燈自己亮了！為什麼會自己亮？！

**旁白**：李杰也愣了一下，但他很快就冷靜下來。

**李杰**：（環顧四周）...應該是感應式的。有人進來就會自動開燈。

**斎藤蓮**：（還在驚魂未定）感...感應式...

**旁白**：斎藤的聲音還在發抖。他緊張地環顧四周，彷彿害怕會有什麼東西突然冒出來。

**旁白**：但李杰已經不再關注他了。

**旁白**：因為現在，在明亮的燈光下，他終於能清楚地看到這個房間的全貌。

**旁白**：而他看到的景象，讓他感到十分困惑。
