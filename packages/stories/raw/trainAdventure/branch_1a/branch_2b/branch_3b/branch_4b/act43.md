# 第四十三幕：鏡像困局

**旁白**：李杰不知道自己跪了多久。他的膝蓋發麻,雙手還撐在地板上。

**旁白**：他盯著斎藤的屍體,看著那灘還在緩緩擴大的血跡。然後,他深吸一口氣。

**李杰**：（內心,強迫自己）冷靜...必須冷靜...

**李杰**：（內心）現在崩潰解決不了任何問題...

**旁白**：李杰緩緩站起身。他的腿有些發軟,但他強迫自己站穩。他看著地板上的四具屍體,看著那些血跡。

**李杰**：（內心）清理...還有意義嗎？

**李杰**：（內心,想起昨天）昨天我們清理了...埋葬了他們...但今天早上,一切又回來了...

**李杰**：（內心,決定）如果...如果明天又會重置...那清理也沒有意義...

**旁白**：李杰轉身,走向廚房。他拿起一瓶水,擰開瓶蓋,大口喝下。水流過喉嚨,但他感覺不到任何清涼。他的腦子裡一片混亂。

**李杰**：（內心,感到迷失）接下來...我該做什麼？

**李杰**：（內心）我完全不知道...不知道這裡是哪裡...不知道為什麼會重置...不知道該怎麼離開...

**李杰**：（內心,感到無力）我什麼都不知道...

**旁白**：李杰放下水瓶,雙手撐在流理台上。他低著頭,看著自己的影子在流理台上搖晃。

**旁白**：疲憊。巨大的疲憊感襲來。

**李杰**：（內心）也許...也許我應該休息一下...

**李杰**：（內心）睡一覺...明天再想...

---

**旁白**：李杰走上樓梯,回到自己的房間。他脫下鞋子,躺在床上。天花板還是那麼白,那麼空蕩。

**旁白**：他閉上眼睛。但眼皮後面,立刻浮現出那些畫面。

**旁白**：父親的臉。母親的臉。那些扭曲的姿勢。斎藤割開自己喉嚨的瞬間。血液噴湧而出的樣子。

**旁白**：李杰猛地睜開眼睛。

**李杰**：（內心,感到痛苦）不行...睡不著...

**李杰**：（內心）一閉上眼睛...那些畫面就...

**旁白**：他坐起身,雙手抱著頭。他在床上坐了一會兒,然後站起來。

**李杰**：（內心）不能這樣...我不能就這樣什麼都不做...

**李杰**：（內心,想起之前的經歷）之前的逃脫房間...那些都是謎題...

**李杰**：（內心）那麼這裡...會不會也是某種謎題？

**李杰**：（內心,開始思考）重置...四具屍體...那三樣物品...一定有什麼線索...

**李杰**：（內心,下定決心）我得仔細搜索這座房子...一定有什麼我遺漏的...

**旁白**：李杰走出房間,來到走廊。他先檢查了其他兩個房間。都和他的房間一樣——簡單的床、衣櫃、桌子,沒有任何裝飾。

**旁白**：他打開衣櫃,裡面是空的。他檢查桌子的抽屜,也是空的。沒有任何可疑的東西。

**旁白**：李杰走下樓梯。他需要再仔細搜索一樓——廚房、餐廳、浴室...

**旁白**：但就在他走到樓梯中間時——他聽到了腳步聲。

**旁白**：李杰的動作停住。那是緩慢的、拖曳的腳步聲。像是有什麼沉重的東西在地板上移動。

**旁白**：李杰慢慢抬起頭,看向客廳的方向。然後,他的呼吸停止了。他看到了。

**旁白**：四個身影。

**旁白**：父親李明遠,緩緩地站了起來。他的動作很僵硬,像是關節生鏽了一樣。母親林婉如,也在站起來。她的頭垂著,頭髮遮住臉,但身體在慢慢直立。

**旁白**：斎藤的哥哥,那個肥胖的身軀搖搖晃晃地站起來。還有斎藤蓮。他的喉嚨還開著那道血淋淋的傷口,但他也在站起來。

**旁白**：四具屍體,全都站起來了。他們的眼睛是空洞的,沒有焦點,只是直直地盯著前方。

**旁白**：然後,他們開始移動。緩慢的、拖曳的腳步。一步,一步。他們轉向樓梯。轉向李杰。

**李杰**：（內心,恐懼）不...不...這不可能...

**李杰**：（內心,幾乎崩潰）他們明明已經死了...為什麼...為什麼會動...

**旁白**：四個身影慢慢向樓梯走來。他們的動作不協調,像是剛學會走路的嬰兒。但他們在前進。一步一步,向李杰走來。

**李杰**：（驚恐地後退）不要過來！

**旁白**：但那四個身影沒有停下。他們繼續前進,開始爬樓梯。李杰轉身,朝樓上跑去。

**旁白**：他衝進自己的房間,砰的一聲關上門。他靠著門,雙手顫抖著轉動門鎖。

**旁白**：樓梯上傳來緩慢的腳步聲,越來越近。

**李杰**：（內心,恐慌）怎麼辦...怎麼辦...

**李杰**：（內心,看向窗戶）窗戶！我可以從窗戶跳出去！

**旁白**：李杰衝向窗戶,用力拉開窗簾。他的手握住窗戶的把手,用力推開。窗戶打開了。

**旁白**：但李杰看到的,不是外面的世界。不是那片紫色的天空,不是那兩個巨大的月亮,不是那片灰色的荒地。

**旁白**：他看到的,是另一個房間。一個和他現在所在的房間幾乎一模一樣的房間。

**旁白**：同樣的床,同樣的衣櫃,同樣的桌子。但是...

**旁白**：李杰瞇起眼睛仔細看。那個房間裡的物品位置,是左右對調的。

**旁白**：就像照鏡子一樣。完全的鏡像反射。

**李杰**：（內心,完全困惑）這...這是什麼...

**李杰**：（內心,感到恐懼）窗戶外面...應該是外面...應該是建築物外面...

**李杰**：（內心,幾乎崩潰）為什麼...為什麼會是另一個房間...

**旁白**：門外傳來沉重的撞擊聲。那些東西到了。他們在撞門。

**旁白**：「碰——碰——碰——」門在震動。

**旁白**：李杰轉身看向門,然後又看向窗戶。他伸手,想要觸碰窗戶外面那個房間。他的手伸出窗外——然後觸碰到了。

**旁白**：觸碰到了那個房間的床。真實的觸感。不是空氣,不是幻覺。那是真實存在的另一個房間。

**李杰**：（內心,感到一陣寒意）這...這到底是...

**旁白**：「碰——碰——碰——」撞門聲越來越大。

**旁白**：李杰後退,關上窗戶。他站在房間中央,看著門,看著窗戶。

**旁白**：門外是殭屍。窗外是另一個相同的房間。他被困住了。完完全全地困住了。

**李杰**：（內心,聲音顫抖）這到底是什麼地方...

**李杰**：（內心,感到絕望）我...我該怎麼辦...

**旁白**：「碰——碰——碰——」撞門聲還在繼續。

**旁白**：李杰環顧房間,尋找任何可能的出路。但這個房間太小了,太簡單了。

**旁白**：只有床、衣櫃、桌子、窗戶、門。沒有其他的出口。沒有任何地方可以躲。

**李杰**：（內心,開始思考）等等...窗戶外面是另一個房間...

**李杰**：（內心）如果...如果我爬過去...

**李杰**：（內心,猶豫）但那個房間...會不會也是一樣的情況...

**李杰**：（內心）會不會...那個房間的門外,也有殭屍...

**旁白**：「碰——！」門發出巨大的聲響。門框開始裂開。他們快要撞開門了。

**旁白**：李杰咬著牙,做出了決定。他衝向窗戶,再次推開窗戶。

**旁白**：他爬上窗台,一隻腳跨過窗框,踩在那個鏡像房間的床上。

**旁白**：然後,他整個人爬了過去。

---

**旁白**：李杰雙腳落在鏡像房間的地板上。他站穩身體,環顧四周。這個房間...真的是完全的鏡像。

**旁白**：床在右邊,而不是左邊。衣櫃在窗戶的左側,而不是右側。桌子在床的右邊,而他原本房間的桌子是在床的左邊。

**旁白**：就連窗簾的褶皺,都是左右相反的。

**旁白**：李杰走向桌子,打開抽屜。空的。和原本房間一樣。他走向衣櫃,打開櫃門。也是空的。

**旁白**：他轉身看向他剛才爬過來的窗戶。窗戶還開著。透過窗戶,他能看到原本那個房間。那個房間的門還在震動。

**旁白**：「碰——碰——碰——」撞門聲從窗戶那邊傳來,但聲音變得很遙遠,很模糊。

**旁白**：李杰走向這個鏡像房間的門。他伸手握住門把,試著轉動。

**旁白**：門把轉動了。門沒有鎖。

**旁白**：李杰猶豫了一下,但沒有打開門。他不知道門外是什麼,也不確定現在是否應該出去。

**李杰**：（內心,謹慎）先不要輕舉妄動...我得先弄清楚這裡的狀況...

**旁白**：他鬆開門把,後退幾步,再次環顧這個房間。完全的鏡像。除了物品位置左右對調之外,其他的一切都和原本的房間一模一樣。

**旁白**：同樣的床,同樣的衣櫃,同樣的桌子,同樣的白色天花板,同樣的空蕩。

**李杰**：（內心,感到困惑）這到底是...什麼意思...

**李杰**：（內心）為什麼窗戶外面會是一個鏡像房間...

**李杰**：（內心）這...這也是謎題的一部分嗎...

**旁白**：李杰走回窗邊,看向原本的房間。那個房間的門還在震動,但現在震動變得更劇烈了。

**旁白**：「碰——！碰——！碰——！」突然,門被撞開了。

**旁白**：四個身影擠進房間。父親、母親、斎藤、斎藤的哥哥。他們進入房間,空洞的眼睛掃視著,尋找李杰的蹤跡。

**旁白**：李杰屏住呼吸,站在窗邊,看著他們。那些殭屍在房間裡四處走動,搖搖晃晃的,撞到床,撞到衣櫃。

**旁白**：但他們沒有走向窗戶。他們似乎...看不到這個窗戶。或者說,他們根本不知道李杰已經逃到另一個房間了。

**旁白**：李杰慢慢後退,遠離窗戶。他靠著牆坐下,雙手抱著膝蓋,看著那個窗戶,看著窗戶那邊的殭屍。

**李杰**：（內心,疲憊）我...我現在在哪裡...

**李杰**：（內心）這個鏡像房間...到底是什麼...

**李杰**：（內心,感到無助）我該怎麼辦...
