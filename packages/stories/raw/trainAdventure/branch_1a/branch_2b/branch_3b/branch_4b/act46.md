# 第四十六幕：測試

**旁白**：李杰沿著街道慢慢走著。五月的夜晚涼爽宜人,微風吹過,帶著淡淡的花香。

**旁白**：他下意識地伸手摸向口袋,想拿出手機看看時間。但口袋裡什麼都沒有。只有那三樣物品——螺絲刀、打火機、瑞士軍刀。

**李杰**：（內心,苦笑）對了...我根本沒有手機...

**李杰**：（內心）在這個世界,我是蓮...不是李杰...

**旁白**：李杰繼續往前走。這是一個安靜的住宅區。兩旁是整齊的獨棟房屋,有些窗戶亮著溫暖的燈光。

**旁白**：他能聽到遠處傳來的電車聲,能看到路燈下偶爾走過的行人。一切都是那麼...正常。

**旁白**：李杰走了大約十五分鐘,來到一個車站。他抬頭看向車站的招牌——「三軒茶屋站」。

**李杰**：（內心）三軒茶屋...

**李杰**：（內心,回想）我以前在東京的時候...從來沒來過這個地方...

**旁白**：車站前有幾家便利商店,還有幾家居酒屋。招牌閃爍著,門口站著三三兩兩的上班族,正在抽菸聊天。

**旁白**：電車駛入車站,發出刺耳的剎車聲。乘客從車廂裡走出來,有些人朝車站外走去,有些人走向公車站。

**旁白**：李杰站在車站外,看著這一切。東京。真實的東京。人們走來走去,建築物林立,霓虹燈閃爍。

**旁白**：就像他記憶中的那樣。就像...什麼都沒發生過一樣。

**李杰**：（內心,感到一陣恍惚）這真的是...東京...

**李杰**：（內心,想起那個荒地）那個紫色的天空...那兩個月亮...那些殭屍...

**李杰**：（內心,摸著口袋裡的物品）那一切...真的存在嗎...

**旁白**：李杰站了一會兒,然後嘆了口氣。他轉身,沿著來時的路往回走。

**李杰**：（內心）沒有錢包...沒有手機...

**李杰**：（內心）我什麼都做不了...還是先回去吧...

**旁白**：李杰沿著街道走回那棟房子。他推開門,走進玄關。客廳的燈還亮著,但已經沒有人了。父母應該已經上樓休息了。

**旁白**：李杰脫下鞋子,輕輕走上樓梯。他來到自己的房間門口,握住門把,推開門。然後,他愣住了。

**旁白**：房間裡,斎藤大輔坐在床邊。他沒有笑容。他的臉很嚴肅,雙手交叉放在腿上,靜靜地看著門口。

**旁白**：李杰站在門口,看著大輔。這個表情...和晚餐時完全不同。晚餐時,大輔一直笑嘻嘻的,快樂又輕鬆。

**旁白**：但現在,他的眼神很沉穩,很認真。雖然因為他胖嘟嘟的臉和圓滾滾的身材,看起來還是有點滑稽,但那份嚴肅卻是真實的。

**李杰**：（遲疑地走進房間）大輔...

**李杰**：（突然意識到,趕緊改口）呃...大哥...？你怎麼在這裡...？

**旁白**：大輔的眼神閃過一絲微妙的變化，但很快就恢復平靜。他沒有立刻回答。他只是靜靜地看著李杰,眼神深邃。李杰關上門,站在房間中央,看著大輔。

**李杰**：（不安地）發生什麼事了嗎...？

**旁白**：大輔深吸一口氣,然後開口。

**斎藤大輔**：（聲音低沉,沒有平時的輕鬆）蓮,我問你一個問題。

**李杰**：（點頭）什麼問題...？

**斎藤大輔**：（盯著李杰）你還記得...去年你做了那件蠢事的時候...你最大的秘密是什麼嗎？

**旁白**：李杰愣住了。

**旁白**：去年...蠢事...秘密...？

**李杰**：（困惑地）我...我不知道你在說什麼...

**李杰**：（遲疑地）什麼蠢事...？

**旁白**：大輔的眼睛微微眯起。他的嘴角沒有笑意,只是靜靜地看著李杰。

**旁白**：沉默。房間裡只有兩人的呼吸聲。李杰感到一陣不安。他不知道大輔在問什麼,也不知道該怎麼回答。

**旁白**：過了好一會兒,大輔站起身。

**斎藤大輔**：（平靜地）我知道了。

**李杰**：（更加困惑）你...你知道什麼...？

**斎藤大輔**：（走向門口,沒有回答問題）我已經得到我想要的答案了。

**旁白**：大輔走到門口,握住門把。然後,他回過頭,看向李杰。他的表情還是很嚴肅,但眼神裡有一絲...複雜。

**斎藤大輔**：（認真地）蓮...不,不管你是誰...

**斎藤大輔**：（語氣變得溫和一些）保重。好好照顧自己。

**斎藤大輔**：（停頓）還有...努力一點。

**旁白**：說完,大輔打開門,走了出去。門輕輕關上。

**旁白**：李杰站在房間中央,看著門,完全摸不著頭緒。

**李杰**：（內心,困惑）他...他這是什麼意思...？去年的蠢事...秘密...我根本不知道他在說什麼...

**李杰**：（內心,突然一驚）而且...我剛才...差點直接叫他大輔...他...他會不會注意到了...

**李杰**：（內心,感到一陣寒意）而且...他最後那句話...「不管你是誰」...他...他是不是發現了什麼...？

**旁白**：李杰站了一會兒,然後搖搖頭。

**李杰**：（內心,疲憊）算了...我現在太累了...想不了那麼多...

**李杰**：（內心）先睡一覺再說...

**旁白**：李杰走向窗戶。他拉開窗簾,打開窗戶。窗外——

**旁白**：不再是那個鏡像房間。不再有殭屍在房間裡遊蕩。窗外是正常的夜景。

**旁白**：他能看到對面鄰居的房子,能看到街燈,能看到遠處的路。一切都是正常的。就像這本來就是一個普通的窗戶一樣。

**李杰**：（內心,嘆氣）連窗戶...都變了...

**李杰**：（內心,感到迷失）這到底...是怎麼回事...

**旁白**：李杰關上窗戶,拉上窗簾。他轉身,看著這個房間。簡單的床,簡單的衣櫃,簡單的桌子。和那個荒地世界的房間一模一樣,只是左右對調。

**旁白**：李杰走向床,躺了下來。他閉上眼睛。疲憊如潮水般襲來。

**旁白**：太多事情發生了。太多無法理解的事情。父母的死亡。殭屍。重置。斎藤的自殺。鏡像房間。鏡像世界。還有那個嚴肅的大輔,那些奇怪的問題。

**李杰**：（內心,在意識模糊前）明天...明天再想...

**李杰**：（內心,聲音越來越微弱）明天...一定要弄清楚...這到底是怎麼回事...

**旁白**：李杰的呼吸漸漸平穩。他睡著了。房間裡安靜下來。只有窗外偶爾傳來的汽車聲,和遠處的蟲鳴聲。

**旁白**：夜,很深了。
