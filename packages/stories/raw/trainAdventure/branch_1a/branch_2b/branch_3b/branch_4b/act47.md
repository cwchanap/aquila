# 第四十七幕：不存在的自己

**旁白**：李杰睜開眼睛。陽光從窗簾的縫隙透進來,灑在床上。他愣了一下,然後坐起身。

**旁白**：這是...蓮的房間。不是那個荒地世界的房間。不是他原本世界的房間。這是鏡像世界的房間。

**李杰**：(內心,嘆氣)所以...這一切都是真的...

**旁白**：李杰從床上爬起來,走向窗戶。他拉開窗簾,陽光照進來,刺得他眯起眼睛。

**旁白**：窗外是正常的日本街道。鄰居的房子,街燈,遠處的路。一個平凡的星期日早晨。

**李杰**：(內心)今天是星期日...

**李杰**：(內心,想起什麼)我想...去一個地方...

**李杰**：(內心)三鷹...我的公寓...

**李杰**：(內心,思索)2023年...那時候我應該也住在三鷹...

**李杰**：(內心,好奇)那麼...現在那個公寓...誰住在那裡...?

**旁白**：李杰換好衣服,走出房間。他走下樓梯,來到客廳。

**旁白**：客廳裡,父親和母親坐在餐桌旁,正在吃早餐。電視開著,播放著早間新聞。

**李明遠/斎藤浩一**：(看到李杰,溫和地笑著)蓮,早安。今天起得挺早的。

**林婉如/斎藤美香**：(也笑著)來,坐下吃早餐吧。

**旁白**：李杰走向餐桌,坐下。桌上擺著簡單的日式早餐。米飯、味噌湯、烤魚、醃菜。

**李杰**：(小聲)早安...大哥呢...?

**林婉如/斎藤美香**：(盛了一碗味噌湯給李杰)大輔一大早就出門了,說要和朋友去吃早午餐。

**李明遠/斎藤浩一**：(笑著搖頭)那孩子才剛回來幾天,朋友就約不停。

**旁白**：李杰拿起筷子,開始吃飯。父母偶爾交談幾句,氣氛很平靜,很溫馨。但李杰卻感到一陣不真實。

**旁白**：這些人...真的是他的父母嗎...?還是只是長著父母面孔的陌生人...?

**林婉如/斎藤美香**：(關心地看著李杰)蓮...你這兩天看起來有點不太對勁...

**李明遠/斎藤浩一**：(也擔心地)是啊,昨天你說的那些奇怪的話...還有你的表現...

**林婉如/斎藤美香**：(溫柔地)你...真的沒事嗎?要不要媽媽帶你去看醫生?

**旁白**：李杰抬起頭,看著父母。他們的眼神充滿關心,充滿擔憂。就像...真的父母一樣。

**李杰**：(勉強笑了笑)我沒事...只是...最近比較累...

**李明遠/斎藤浩一**：(皺眉)是不是熬夜打遊戲打太多了?

**李杰**：(搖頭)沒有...我...我只是睡不好...

**林婉如/斎藤美香**：(嘆氣)那你今天好好休息吧。別又宅在房間裡,多出去走走。

**李杰**：(點頭)嗯...我等一下...想出去走走...

**旁白**：父母看著李杰,臉上還是有些擔心,但沒有再多問。

**旁白**：李杰繼續吃著飯,心裡卻在想著別的事情。

**李杰**：(內心,不安)我能撐多久...?

**李杰**：(內心)我不是蓮...我不可能完全假裝成他...

**李杰**：(內心,擔心)遲早...他們會發現的...

**李杰**：(內心,想起大輔)大輔昨晚...好像已經懷疑了...

**李杰**：(內心,稍微安心一點)但至少...蓮基本上沒有什麼親密的朋友...

**李杰**：(內心)所以我只需要應付爸媽...

**李杰**：(內心,嘆氣)但這也不是長久之計...

**旁白**：吃完早餐,李杰幫忙收拾碗筷,然後回到房間。

**旁白**：他從衣櫃裡找到一個錢包。

**旁白**：打開一看,裡面有幾張千圓鈔票,還有一些零錢。

**旁白**：還有一張交通卡。

**李杰**：(內心)至少...我現在有錢了...

**旁白**：李杰把錢包放進口袋,走出房間。

**李杰**：(對著樓下喊)爸,媽,我出門了!

**林婉如/斎藤美香**：(從樓下回應)路上小心!

**旁白**：李杰走出家門,沿著街道走向車站。三軒茶屋站。他走進車站,用交通卡刷卡進站。

**旁白**：月台上已經有一些人在等車。週日的早晨,人不多。電車進站,李杰上車。

**旁白**：他找了個座位坐下,看著窗外飛逝的風景。一站又一站。渋谷、新宿、中野...熟悉的站名。熟悉的風景。就像他記憶中的東京。

**旁白**：終於,電車到達三鷹站。李杰下車,走出車站。他沿著記憶中的路線,走向那棟公寓。

**旁白**：十五分鐘後,他站在那棟公寓樓下。一棟老舊的十四層樓公寓。灰色的外牆,狹窄的入口。一模一樣。

**李杰**：(內心,看著這棟樓)就是這裡...

**李杰**：(內心)我住的地方...

**旁白**：李杰走進大樓,按下電梯按鈕。電梯門打開,他走進電梯,按下九樓的按鈕。電梯緩緩上升。

**旁白**：叮——電梯門打開,李杰走出電梯,來到九樓走廊。他站在一扇門前。902號房。他的房間。

**旁白**：李杰深吸一口氣,舉起手,敲了敲門。咚咚咚。沒有回應。李杰又敲了幾次。還是沒有回應。

**李杰**：(內心)沒人在...?

**李杰**：(內心)還是...出門了...?

**旁白**：李杰站在門前,猶豫著要不要繼續等。就在這時,他聽到電梯叮的一聲。

**旁白**：電梯門打開,一個中年婦女走出電梯,手裡提著購物袋。她看到站在902號房門前的李杰,皺起眉頭。

**中年婦女**：(警惕地)你是誰?為什麼站在我家門口?

**旁白**：李杰愣住了。她的家...?

**李杰**：(趕緊道歉)對不起...我...我可能弄錯了...

**中年婦女**：(走向902號房,看著李杰)你找誰?

**李杰**：(遲疑地)我...我以為...這裡住的是...

**李杰**：(停頓,然後搖頭)對不起,我認錯地方了。

**中年婦女**：(拿出鑰匙)這間公寓我住了五年了。你該不會是想推銷什麼吧?

**李杰**：(連忙搖頭)不是,不是...我真的只是認錯了...

**李杰**：(確認地)您...您真的住在這裡...已經五年了...?

**中年婦女**：(不耐煩地)對啊,怎麼了?

**李杰**：(低聲)沒什麼...對不起打擾了...

**旁白**：李杰轉身,快步走向電梯。中年婦女看著他的背影,搖搖頭,然後開門進屋。

**旁白**：李杰走出公寓樓,站在街道上。他抬頭看著那棟樓。他的公寓。但...現在不是了。

**李杰**：(內心,感到一陣茫然)五年...她說她住了五年...

**李杰**：(內心)2023年...那時候我應該還住在那裡...

**李杰**：(內心,意識到)但在這個世界...我根本不存在...

**李杰**：(內心,感到恐懼)李杰...在這個世界...不存在...

**李杰**：(內心)只有斎藤蓮...

**旁白**：李杰站在街道上,看著周圍的風景。熟悉的街道。熟悉的建築。但...這不是他的世界。他...不屬於這裡。

**李杰**：(內心,聲音顫抖)我...我到底該怎麼辦...

**李杰**：(內心,感到絕望)我...還能回去嗎...

**旁白**：李杰深吸一口氣,轉身走向車站。他不知道接下來該去哪裡。他不知道該做什麼。他只知道...他不屬於這個世界。
