# 第四十九幕：露營

**旁白**：八月中旬，暑假已經過了一半。李杰收到如月颯真傳來的訊息。

**如月颯真**：（訊息）蓮，這個週末我們要去山裡露營，你要來嗎？

**李杰**：（內心，看著訊息）露營...

**李杰**：（內心，想了想）以前的蓮...應該不會參加這種活動...

**李杰**：（內心，笑了笑）但我不是蓮...而且...我也想試試...

**李杰**：（訊息）好啊，我去。

**旁白**：週六早晨,李杰背著背包,來到約定的車站。如月颯真、氷室陸斗、七海詩織、高橋彩花和渡邊真理已經在那裡等著了。

**如月颯真**：（看到李杰，揮手）蓮！早安！

**李杰**：（微笑著走過去）早安。

**氷室陸斗**：（驚訝地）哇，蓮你真的來了！我還以為你會在最後一刻說不來呢！

**七海詩織**：（笑著）陸斗，別這樣說嘛。蓮最近變了很多啊。

**高橋彩花**：（看著李杰，友善地）蓮，你有帶睡袋嗎？

**李杰**：（點頭）有，我都準備好了。

**渡邊真理**：（高興地）太好了！那我們出發吧！

**旁白**：六個人搭上前往山區的電車。車廂裡,大家興奮地討論著這次露營的計劃。

**如月颯真**：（看著手機上的地圖）我們會在這個露營地紮營。那裡有溪流，還可以釣魚。

**氷室陸斗**：（興奮地）釣魚！我要釣到最大的魚！

**七海詩織**：（笑著）陸斗，你上次釣魚的時候連魚竿都拿不穩呢。

**氷室陸斗**：（不服氣地）那是意外！這次我一定可以的！

**旁白**：大家都笑了。李杰坐在窗邊,看著窗外飛逝的風景,嘴角帶著淡淡的笑容。

**旁白**：這種感覺...真好。就像...真的回到了大學時代。

**旁白**：兩小時後,他們到達了山區。從車站走到露營地還需要四十分鐘的山路。大家背著沉重的背包,沿著山路慢慢往上爬。

**渡邊真理**：（喘著氣）好累...還有多遠...

**如月颯真**：（看著地圖）快到了，再走二十分鐘就到了。

**氷室陸斗**：（已經累得不行）我...我不行了...我要休息...

**七海詩織**：（拍拍他的背）加油啊，陸斗。到了之後我們就可以吃午餐了。

**李杰**：（走在中間，步伐穩健）要不要我幫你拿一些東西？

**氷室陸斗**：（驚訝地看著李杰）蓮...你居然還有力氣...

**李杰**：（笑著）還好，我平時有運動。

**高橋彩花**：（也有些驚訝）蓮，你以前不是最討厭運動的嗎？

**李杰**：（頓了一下，然後自然地說）最近...開始喜歡了。

**旁白**：終於,他們到達了露營地。這是一片空曠的草地,旁邊有一條清澈的溪流。遠處是綿延的山脈,天空湛藍,白雲飄浮。

**七海詩織**：（深吸一口氣）好美啊！空氣真清新！

**如月颯真**：（放下背包）好，我們先搭帳篷吧。

**旁白**：大家開始忙碌起來。李杰和如月颯真一起搭帳篷。

**如月颯真**：（一邊搭帳篷一邊說）蓮，你會搭帳篷嗎？

**李杰**：（點頭）會一點。我來幫你。

**旁白**：李杰熟練地將帳篷桿穿過帳篷布，固定好四個角。

**如月颯真**：（驚訝地）哇，蓮，你動作好快！你以前搭過帳篷嗎？

**李杰**：（笑了笑）嗯...很久以前搭過。

**旁白**：另一邊，氷室陸斗和兩個女生正在努力搭另一個帳篷。

**氷室陸斗**：（困惑地）欸...這個桿子要插哪裡...

**高橋彩花**：（看著說明書）應該是這裡...不對，是那裡...

**渡邊真理**：（嘆氣）我們搭了二十分鐘還沒搭好...

**七海詩織**：（走過來看了看）要不要請蓮來幫忙？他那邊已經搭好了。

**旁白**：李杰走過去，看了看他們的帳篷。

**李杰**：（溫和地）這個桿子要先穿過這裡，然後從這裡固定...

**旁白**：在李杰的幫助下，第二個帳篷很快就搭好了。

**氷室陸斗**：（擦著汗）蓮，你太厲害了！你是不是以前當過童軍啊？

**李杰**：（搖頭，笑著）沒有，只是碰巧知道而已。

**旁白**：帳篷搭好後,已經是中午了。大家決定先吃午餐。

**如月颯真**：（從背包裡拿出食材）我們帶了咖哩飯的材料。

**七海詩織**：（興奮地）太好了！我好餓！

**李杰**：（看著食材）需要幫忙嗎？

**如月颯真**：（點頭）可以幫忙切菜嗎？

**旁白**：李杰蹲下來,拿起刀和砧板,開始切菜。他的刀工很熟練,切出來的蔬菜大小均勻。

**七海詩織**：（在旁邊看著，驚訝地）蓮...你切菜好快...而且切得好整齊...

**李杰**：（專注地切著菜）嗯，我平時會自己做飯。

**渡邊真理**：（小聲對彩花說）蓮真的變了好多...以前他連廚房都不進的...

**高橋彩花**：（點頭，也小聲說）對啊...而且現在感覺他...好可靠...

**旁白**：午餐很快就做好了。大家圍坐在一起,吃著熱騰騰的咖哩飯。

**氷室陸斗**：（大口吃著）好好吃！果然在野外吃東西特別香！

**如月颯真**：（笑著）因為你爬山爬得很累，所以什麼都覺得好吃。

**七海詩織**：（看著李杰）蓮，你覺得怎麼樣？

**李杰**：（微笑）很好吃。謝謝大家。

**旁白**：吃完午餐,大家休息了一會兒。下午,如月颯真和氷室陸斗去溪邊釣魚。李杰和三個女生留在營地,準備晚上的篝火。

**七海詩織**：（撿著樹枝）蓮，你幫我們一起撿柴火吧。

**李杰**：（點頭）好。

**旁白**：李杰走進樹林,撿拾乾燥的樹枝。他抱著一大堆柴火走回營地。

**高橋彩花**：（驚訝地）蓮，你一個人抱這麼多！

**李杰**：（放下柴火）還好，不重。

**渡邊真理**：（認真地看著李杰）蓮...你真的...和以前完全不一樣了...

**李杰**：（頓了一下）是嗎...

**渡邊真理**：（點頭）以前的你...總是很傲慢，對誰都不友善...

**高橋彩花**：（接著說）而且你總是說一些很難聽的話，讓人很不舒服...

**七海詩織**：（溫柔地）但是現在的你...很溫和，也很可靠。大家都很喜歡現在的你。

**旁白**：李杰看著她們，心裡有些複雜。

**李杰**：（內心）她們喜歡的...是我...還是那個已經不存在的蓮...

**李杰**：（內心，搖搖頭）算了...不管是誰...至少現在...我是被接受的...

**李杰**：（對女生們微笑）謝謝。

**旁白**：傍晚，如月颯真和氷室陸斗釣魚回來了。

**氷室陸斗**：（舉著一條小魚，得意地）你們看！我釣到魚了！

**如月颯真**：（笑著）就這一條小魚，你說了一下午。

**氷室陸斗**：（不服氣地）至少我釣到了！你一條都沒釣到！

**旁白**：大家都笑了。

**旁白**：夜幕降臨,篝火點燃了。火光在黑暗中跳躍,照亮了每個人的臉。大家圍坐在篝火旁,烤著棉花糖,聊著天。

**如月颯真**：（看著星空）今晚的星星好美。

**七海詩織**：（抬頭看）真的！好多星星！在東京根本看不到。

**氷室陸斗**：（躺在草地上）感覺整個銀河都在眼前...

**旁白**：李杰也抬頭看向天空。滿天的星星,閃爍著微弱的光芒。就像...那個有兩個紫色月亮的世界一樣...但又不一樣。

**旁白**：這裡是真實的世界。這裡是...他現在生活的世界。

**高橋彩花**：（突然說）蓮，你在想什麼？

**李杰**：（回過神）沒什麼...只是在想...

**李杰**：（看著篝火）這樣的時刻...很珍貴。

**渡邊真理**：（微笑）是啊。和朋友一起露營，真的很開心。

**七海詩織**：（看著李杰，溫柔地）蓮，以後我們還要一起出來玩喔。

**如月颯真**：（點頭）對啊！下次我們去海邊！

**氷室陸斗**：（興奮地）好主意！我要去衝浪！

**旁白**：大家都笑著,討論著下次的計劃。李杰坐在那裡,看著這些笑臉。他的心裡,第一次感到...真正的平靜。

**李杰**：（內心，輕聲）也許...

**李杰**：（內心）這就是...屬於我的生活...

**旁白**：篝火繼續燃燒著。火光照亮了夜晚,也照亮了李杰的心。這一夜,他睡得很安穩。

**旁白**：沒有噩夢。沒有殭屍。沒有紫色的月亮。只有星空,篝火,和朋友的笑聲。
