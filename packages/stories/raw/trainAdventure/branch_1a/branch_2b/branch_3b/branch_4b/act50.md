# 第五十幕：消失

**旁白**：露營之後，李杰和朋友們的關係變得更加親密。九月，新學期開始了。李杰每天去上課，和如月颯真、氷室陸斗他們一起吃午餐，討論作業，偶爾一起去居酒屋喝酒。

**氷室陸斗**：（舉著啤酒杯）來！為了期中考全部過關，乾杯！

**如月颯真**：（笑著）陸斗，你上次就是這麼說的，結果還是有一科被當掉。

**氷室陸斗**：（不服氣地）那是意外！這次我一定可以的！

**七海詩織**：（笑著對李杰說）蓮，你要不要也來教教陸斗？你上學期成績那麼好。

**李杰**：（微笑）好啊，有什麼不懂的隨時問我。

**旁白**：十月，學校舉辦了文化祭。李杰和同學們一起經營班級的攤位，賣章魚燒。

**高橋彩花**：（興奮地）好多人來買！蓮，再做十份！

**李杰**：（一邊翻著章魚燒一邊說）知道了，馬上就好。

**渡邊真理**：（驚訝地）蓮，你做章魚燒好熟練！

**李杰**：（笑著）只是碰巧而已。

**旁白**：十一月，氣溫開始下降。李杰和朋友們一起去看紅葉。

**如月颯真**：（拿著相機拍照）這裡的紅葉真美！

**七海詩織**：（站在紅葉樹下）蓮，我們一起拍張照吧！

**旁白**：李杰站在詩織旁邊，朝著鏡頭微笑。喀嚓。照片定格了這個瞬間。

**旁白**：十二月，期末考試結束了。李杰這學期的成績依然很好，平均A。

**氷室陸斗**：（看著成績單，羨慕地）蓮，你怎麼做到的...我才勉強過關...

**李杰**：（拍拍他的肩膀）你已經很努力了。下學期繼續加油。

**旁白**：然後是新年。李杰和父母一起去神社參拜。投錢，搖鈴，雙手合十。

**李杰**：（內心，閉著眼睛）希望...一切平安...

**旁白**：一月，二月，三月...時間一天天過去。李杰漸漸地...真的習慣了這個世界。習慣了斎藤蓮的身份。習慣了這個家庭。習慣了這些朋友。

**旁白**：那些關於荒地世界、殭屍、紫色月亮的記憶，漸漸變得模糊。就像一場遙遠的夢。

---

**旁白**：2024年6月。李杰收到了一個消息。大輔要回日本了。

**林婉如/斎藤美香**：（高興地對李杰說）蓮！好消息！你哥哥要回來了！

**李明遠/斎藤浩一**：（也很高興）他在中國的工作結束了，下週就回日本。

**李杰**：（微笑）真的嗎？那太好了。

**旁白**：李杰的心裡，有些複雜。

**李杰**：（內心）大輔...已經快一年沒見了...

**李杰**：（內心）他...還記得嗎...我不是真正的蓮...

**旁白**：一週後，李杰和父母一起前往成田機場。大輔的航班預計下午三點到達。他們提前到了機場，在抵達大廳等候。

**林婉如/斎藤美香**：（興奮地）大輔說他帶了很多中國的特產回來。

**李明遠/斎藤浩一**：（笑著）那孩子總是這樣，每次回來都大包小包的。

**旁白**：李杰站在一旁，安靜地看著出口。時間一分一秒過去。三點。三點十分。三點二十分。陸續有乘客從出口走出來。但沒有大輔的身影。

**旁白**：三點半。

**林婉如/斎藤美香**：（開始擔心）怎麼還沒出來...該不會是行李太多，過海關比較慢吧...

**李明遠/斎藤浩一**：（拿出手機）我打電話問問。

**旁白**：父親撥通了大輔的電話。嘟...嘟...嘟...沒有人接。

**李明遠/斎藤浩一**：（皺眉）奇怪...沒人接...

**旁白**：他又撥了一次。

**旁白**：還是沒有人接。

**林婉如/斎藤美香**：（更加擔心）會不會是手機沒電了？

**李明遠/斎藤浩一**：（安慰她）可能吧...不要太擔心，可能只是延誤了。

**旁白**：李杰也拿出手機，撥打大輔的號碼。嘟...嘟...嘟...同樣沒有人接。

**李杰**：（內心，感到不安）為什麼...

**李杰**：（內心）為什麼沒人接...

**旁白**：四點。

**旁白**：機場的航班顯示屏上，大輔的航班狀態顯示為「已到達」。

**旁白**：但大輔還是沒有出現。

**林婉如/斎藤美香**：（焦急地）已經一個小時了...到底發生什麼事了...

**李明遠/斎藤浩一**：（也開始擔心）我去問問航空公司的櫃檯。

**旁白**：父親走向航空公司的服務櫃檯。

**旁白**：李杰站在原地，心裡湧起一股強烈的不安。

**李杰**：（內心）這種感覺...

**李杰**：（內心，握緊拳頭）不對勁...

**李杰**：（內心）有什麼事情...不對勁...

**旁白**：父親從櫃檯回來了。

**李明遠/斎藤浩一**：（困惑地）他們說...大輔確實在乘客名單上，而且已經辦理了登機手續...

**林婉如/斎藤美香**：（驚訝地）那他人呢？為什麼沒出來？

**李明遠/斎藤浩一**：（搖頭）他們說不清楚...可能在機場的其他地方...

**李杰**：（突然開口）我去找找看。

**林婉如/斎藤美香**：（看著李杰）蓮，你要去哪裡？

**李杰**：（平靜地，但眼神銳利）我去洗手間、餐廳那邊看看。也許大哥在那裡。

**李明遠/斎藤浩一**：（點頭）好，你去看看。我和你媽媽在這裡等。

**旁白**：李杰轉身，快步走開。他穿過人群，走向機場的其他區域。洗手間，餐廳，商店...他一個一個地方找。但沒有大輔的身影。

**李杰**：（內心，焦急地）大輔...你到底在哪裡...

**旁白**：李杰來到一個安靜的角落。這裡人很少。他停下腳步，深吸一口氣。

**李杰**：（內心，聲音顫抖）為什麼...

**李杰**：（內心）為什麼我會有這種不好的預感...

**李杰**：（內心，想起什麼）難道...

**旁白**：李杰的腦海中，浮現出那些曾經被遺忘的畫面。荒地。廢墟。紫色的月亮。殭屍。還有...那輛神秘的列車。

**李杰**：（內心，冷汗流下）不...不會吧...

**李杰**：（內心，恐懼地）難道...大輔...

**旁白**：就在這時，李杰的手機響了。他連忙拿起手機。是父親打來的。

**李杰**：（接起電話）爸？

**李明遠/斎藤浩一**：（聲音焦急）蓮，你那邊有找到嗎？

**李杰**：（沉重地）沒有...我找遍了附近的地方...都沒有...

**李明遠/斎藤浩一**：（嘆氣）我們這邊也沒有...航空公司說會幫忙查...但...

**李杰**：（沉默了一下）我知道了。我再找找看。

**旁白**：李杰掛斷電話。他站在那裡，看著機場來往的人群。每個人都行色匆匆。每個人都有自己的目的地。但大輔...大輔去了哪裡？

**李杰**：（內心，低聲）大輔...

**李杰**：（內心，聲音帶著絕望）你...該不會...

**旁白**：李杰閉上眼睛。他不想相信。但那種不安的感覺，越來越強烈。就像...就像當初他自己搭上那輛列車的時候...一樣。
