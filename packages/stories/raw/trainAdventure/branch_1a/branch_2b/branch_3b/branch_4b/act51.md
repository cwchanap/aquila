# 第五十一幕：訊息

**旁白**：李杰回到抵達大廳。父母還在那裡等待，臉上滿是焦急。

**李明遠/斎藤浩一**：（看到李杰，期待地）蓮，怎麼樣？

**李杰**：（搖頭）沒有...我找遍了附近的地方...都沒有...

**林婉如/斎藤美香**：（眼眶泛紅）怎麼會這樣...大輔到底去哪裡了...

**旁白**：他們繼續在機場搜尋。父親詢問了航空公司的工作人員，母親問了其他乘客。李杰則走向剛從同一航班出來的旅客。

**李杰**：（禮貌地）不好意思，請問您是從上海來的航班嗎？

**旅客A**：（點頭）是的，怎麼了？

**李杰**：（拿出手機，給對方看大輔的照片）請問您有看到這個人嗎？他應該和您搭同一班飛機。

**旅客A**：（看了看照片，搖頭）沒有印象...抱歉...

**旁白**：李杰又問了幾個乘客。

**旅客B**：（看著照片）沒見過...

**旅客C**：（同樣搖頭）不好意思，我沒注意...

**旁白**：沒有人見過大輔。就像...大輔從來沒有在那班飛機上一樣。

**旁白**：兩個小時過去了。機場的人流漸漸稀疏。父親疲憊地坐在椅子上。

**李明遠/斎藤浩一**：（嘆氣）我們...報警吧...

**林婉如/斎藤美香**：（點頭，聲音顫抖）只能這樣了...

**旁白**：他們來到機場的警察局。一個年輕的警察接待了他們。

**警察**：（聽完說明後，為難地）我理解您們的擔心，但是...失蹤案件需要滿24小時後才能正式立案...

**李明遠/斎藤浩一**：（激動地）但是我兒子明明應該在這班飛機上！航空公司確認了他辦理了登機手續！

**警察**：（安撫地）我知道，我知道...但這是規定...如果24小時後還是沒有消息，請您再來報案...

**林婉如/斎藤美香**：（絕望地）那我們現在該怎麼辦...

**警察**：（同情地）您們可以繼續在機場尋找，或者回家等待。有時候失蹤的人會自己聯繫家人...

**旁白**：父母沮喪地離開了警察局。他們的背影看起來特別疲憊。

**李明遠/斎藤浩一**：（無力地）回家吧...也許大輔會打電話來...

**林婉如/斎藤美香**：（擦著眼淚，點頭）嗯...

**李杰**：（看著父母）爸，媽，你們先回去休息。我...我再找一下。

**李明遠/斎藤浩一**：（看著李杰）蓮...你也別太勉強自己...

**李杰**：（堅定地）我會的。你們先回去。

**旁白**：父母離開了機場。李杰看著他們消失在人群中的身影，深吸一口氣。

**李杰**：（內心）大輔...你到底在哪裡...

**旁白**：李杰轉身，走向機場的深處。他知道有些區域是限制進入的。但他必須試試。

**旁白**：他找到一個偏僻的走廊，趁著警衛不注意，溜了進去。這裡是機場的後勤區域。工作人員來來往往，忙著處理行李和貨物。

**李杰**：（走向一個工作人員）不好意思...

**工作人員A**：（轉過頭，警戒地）你是誰？這裡是限制區域，你不能進來。

**李杰**：（急切地）對不起，但我在找人。我哥哥應該在今天下午的上海航班上，但是他沒有出現...

**工作人員A**：（皺眉）你說什麼？

**李杰**：（拿出手機，給對方看照片）這是我哥哥。請問您有看到他嗎？

**工作人員A**：（看了看照片，搖頭）沒見過。你趕快離開這裡，不然我要叫保安了。

**旁白**：李杰又問了幾個工作人員。但所有人的回答都一樣。沒有人見過大輔。

**工作人員B**：（搖頭）沒印象...

**工作人員C**：（同樣搖頭）不好意思，我沒見過...

**旁白**：李杰走到一個安靜的角落，靠在牆上。他感到一陣深深的無力。

**李杰**：（內心，疲憊地）為什麼...

**李杰**：（內心）為什麼沒有人見過他...

**李杰**：（內心，聲音顫抖）他...他真的上了那班飛機嗎...

**旁白**：就在這時，李杰的手機震動了。他低頭看去。一則新訊息。發送者...大輔。

**李杰**：（瞪大眼睛，心跳加速）大輔！？

**旁白**：李杰顫抖著手，點開訊息。訊息裡...是一張照片。照片顯示的是...一個房間。

**旁白**：一個普通的日式房間，看起來像是書房兼臥室。木地板，一張書桌，書架，一張單人床。牆上貼著幾張動漫角色海報。書桌上有一台遊戲機。書架上擺滿了漫畫書。

**旁白**：窗簾拉著，看不到外面。燈光昏暗。沒有人。只有空蕩蕩的房間。

**李杰**：（盯著照片，困惑地）這是...什麼...

**李杰**：（內心）大輔為什麼要傳這張照片給我...

**李杰**：（內心）這是哪裡的房間...

**旁白**：李杰立刻撥打大輔的電話。嘟...嘟...嘟...沒有人接。

**旁白**：李杰又發了訊息。

**李杰**：（訊息）大哥！你在哪裡？這是哪裡？你還好嗎？

**旁白**：訊息發送出去。但沒有回覆。

**李杰**：（焦急地）接電話啊...回訊息啊...

**旁白**：李杰又打了幾次電話。都是無人接聽。

**旁白**：李杰再次看向那張照片。他放大照片，仔細觀察每個細節。房間...木地板...書桌...書架...床...動漫海報...遊戲機...漫畫書...

**李杰**：（內心，努力回想）這個房間...

**李杰**：（內心）我好像...在哪裡見過...

**旁白**：李杰皺起眉頭。一種奇怪的熟悉感。但他就是想不起來。

**李杰**：（內心，用力回想）在哪裡...我在哪裡見過這個房間...

**旁白**：李杰閉上眼睛，努力搜索記憶。荒地世界...？鏡像世界...？還是...更早之前...？但記憶一片模糊。

**李杰**：（內心，挫敗地）想不起來...

**旁白**：李杰睜開眼睛，再次看著照片。昏暗的燈光。空無一人的房間。窗簾遮住的窗戶。一種詭異的寂靜感。

**李杰**：（內心，不安地）這張照片...

**李杰**：（內心）到底是什麼意思...

**李杰**：（內心，恐懼地）大輔...你想告訴我什麼...

**旁白**：李杰在機場又找了一個多小時。但沒有任何線索。大輔就像憑空消失了一樣。

**旁白**：天色漸漸暗了下來。李杰疲憊地走出機場。

**李杰**：（內心，無力地）找不到...什麼都找不到...

**旁白**：李杰坐上電車，回到東京市區。他沒有直接回家。他知道父母現在一定很難過。他們不會有心情吃晚餐。

**旁白**：李杰走進一家便利商店，買了一個便當。他坐在便利商店外的長椅上，打開便當。但他吃不下。

**旁白**：他看著手機裡那張照片。空蕩蕩的房間。昏暗的燈光。

**李杰**：（內心，低聲）大輔...

**李杰**：（內心，絕望地）你...到底發生了什麼事...

**旁白**：夜晚的街道上，行人匆匆。李杰坐在長椅上，看著那張神秘的照片。

**旁白**：他不知道。這張照片...將會把他帶回那個他以為已經遺忘的世界。
