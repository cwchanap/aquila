# 第五十七幕：便條

**旁白**：氷室陸斗從廚房跑出來，拿著一把菜刀。

**氷室陸斗**：（興奮地）蓮！找到了！

**七海詩織**：（跟在後面，無奈地）陸斗...用菜刀會不會太誇張...

**如月颯真**：（從儲物間出來，拿著工具箱）這裡有工具！

**旁白**：他打開工具箱。裡面有鐵鎚、螺絲起子、老虎鉗、扳手。

**渡邊真理**：（看著工具）這些應該可以用！

**高橋彩花**：（走過來）需要幫忙嗎？

**如月颯真**：（把鐵鎚遞給李杰）蓮，你來。

**李杰**：（接過鐵鎚，深吸一口氣）好...

**旁白**：李杰走到密碼鎖前。朋友們圍在他身邊。他舉起鐵鎚，對準密碼鎖。

**李杰**：（內心）對不起了...大輔...

**旁白**：李杰揮下鐵鎚。鏗！金屬撞擊的聲音在走廊裡迴響。密碼鎖的外殼裂開了一道縫。

**氷室陸斗**：（興奮地）有用！再來！

**旁白**：李杰再次揮下鐵鎚。鏗！鏗！鏗！密碼鎖逐漸變形，碎片掉落在地上。

**如月颯真**：（遞上螺絲起子）試試撬開。

**旁白**：李杰把螺絲起子插進裂縫，用力一撬。喀啦！密碼鎖的面板脫落了，露出裡面複雜的電路。

**渡邊真理**：（遞上老虎鉗）剪斷電線！

**旁白**：李杰用老虎鉗剪斷幾根電線。密碼鎖的燈熄滅了。

**氷室陸斗**：（試著轉動門把）可以開了嗎？

**旁白**：門把轉動了。喀嚓。門...打開了。

**眾人**：（屏住呼吸）

**旁白**：走廊的光照進黑暗的房間。李杰慢慢推開門。

**旁白**：房間裡很暗。窗簾拉著，只有一絲光線從縫隙透進來。

**李杰**：（走進房間，打開電燈）

**旁白**：燈亮了。房間和照片裡一模一樣。

**旁白**：書桌靠著牆，上面放著電腦和顯示器。桌旁有PS5和Switch。

**旁白**：書架上塞滿了漫畫書——哆啦A夢、火影忍者。

**旁白**：牆上貼著兩張海報。一張是長門有希，另一張...

**李杰**：（走近牆壁，看著海報）清水惠美...

**旁白**：海報裡的清水惠美，笑得很燦爛。看起來就像一張普通的照片被放大列印出來的。

**李杰**：（內心，困惑）為什麼...大輔會有清水惠美的照片...

**李杰**：（內心）而且還做成海報...貼在牆上...

**李杰**：（內心）他是在...試著告訴我什麼嗎...

**七海詩織**：（走進房間，環顧四周）這房間...好整齊...

**氷室陸斗**：（也走進來）就像沒人住過一樣...

**如月颯真**：（看著書桌）電腦和遊戲機都還在...

**渡邊真理**：（走到書架前）我們找找看...有沒有什麼線索...

**高橋彩花**：（點頭）對...也許能找到什麼...

**旁白**：六個人開始在房間裡仔細搜索。

**旁白**：李杰打開書桌的抽屜。裡面有文具、筆記本、一些收據。沒有什麼特別的。

**旁白**：如月颯真檢查電腦。按下電源鍵，但電腦沒反應。可能很久沒充電了。

**旁白**：氷室陸斗翻看書架上的漫畫。一本本拿出來檢查，看有沒有夾著什麼東西。

**旁白**：渡邊真理檢查床下和衣櫃。

**七海詩織**：（在書桌旁的牆上）蓮...這裡有東西...

**旁白**：李杰走過去。

**旁白**：七海詩織指著牆上，清水惠美海報的旁邊。有一張小小的便條紙，用透明膠帶貼在牆上。

**旁白**：便條紙是黃色的，上面寫著一些字。

**李杰**：（湊近看）這是...

**旁白**：便條上的字，不是日文。也不是英文。也不是中文。是一種李杰從未見過的文字。奇怪的符號，彎曲的線條，像是某種古老的文字。

**李杰**：（內心，震驚）這些字...我不認識...

**李杰**：（內心，更加震驚）但是...我知道它的意思...

**七海詩織**：（困惑地）這是什麼文字...好奇怪...

**高橋彩花**：（也走過來，在書架後面）我這裡也有一張！

**旁白**：高橋彩花從書架後面取下另一張便條紙。也是黃色的，也寫著同樣奇怪的文字。

**旁白**：李杰接過兩張便條。他仔細看著上面的文字。他不認識這些字。但腦海中自動浮現出意思。

**旁白**：第一張便條的意思是：

**旁白**：「中國上海市浦東新區世紀大道1000號1410室」

**旁白**：第二張便條的意思是：

**旁白**：「東京都三鷹市下連雀3丁目35-1 三鷹公寓902號室」

**李杰**：（手在顫抖）這...這是...

**如月颯真**：（擔心地）蓮？你還好嗎？

**氷室陸斗**：（好奇地）那是什麼文字？

**李杰**：（聲音顫抖）我...我不知道...

**渡邊真理**：（困惑地）不知道？可是你看起來...好像很震驚...

**李杰**：（看著便條，喃喃自語）我不認識這些字...但是...我能理解它的意思...

**眾人**：（更加困惑地看著他）

**七海詩織**：（小心地問）那...它寫的是什麼...

**李杰**：（看著第一張便條，聲音顫抖）這是...一個地址...在中國...

**李杰**：（看著第二張便條）這是...三鷹的一個地址...

**如月颯真**：（皺眉）你能理解但不認識...這是什麼意思...

**李杰**：（搖頭）我也不知道...我就是...知道它的意思...

**高橋彩花**：（擔心地）這兩個地址...對你有什麼意義嗎...

**李杰**：（深吸一口氣，握緊便條）這兩個地址...都很重要...

**氷室陸斗**：（困惑地）可是...為什麼你哥哥會用這種奇怪的文字...

**李杰**：（苦笑）我也想知道...

**如月颯真**：（認真地）那你打算怎麼做？

**李杰**：（看著兩張便條）我...我想我需要去這兩個地方看看...

**氷室陸斗**：（拍拍李杰的肩膀）需要我們陪你去嗎？

**李杰**：（搖頭）不用...這個...我自己去就好...

**高橋彩花**：（擔心地）可是...

**李杰**：（微笑）真的不用...你們已經幫了我很多了...

**如月颯真**：（看著李杰，想了想）好吧...但如果需要幫忙，隨時說。

**七海詩織**：（握著李杰的手）對...我們隨時都在。

**旁白**：李杰看著朋友們擔心的臉。

**李杰**：（真誠地）謝謝你們...真的...

**旁白**：天色已經暗了。朋友們陸續離開。李杰送他們到門口。

**氷室陸斗**：（揮手）蓮！有事就說！

**七海詩織**：（擔心地）明天見...要好好休息...

**如月颯真**：（最後離開，對李杰說）蓮...有些事情，如果你不想說，我們不會勉強你...

**李杰**：（愣了一下）颯真...

**如月颯真**：（微笑）但記得...我們是朋友。

**旁白**：如月颯真離開了。李杰關上門。他回到大輔的房間。看著手中的兩張便條。

**李杰**：（內心）那些文字...我從未見過...

**李杰**：（內心）但我卻能理解它的意思...

**李杰**：（內心）中國...上海...那是我原本父母的家...

**李杰**：（內心）三鷹...那是我原本的公寓...

**李杰**：（內心）大輔...你到底知道多少...

**李杰**：（內心）你為什麼要用這種文字...

**李杰**：（內心）這種...我看不懂卻能理解的文字...

**旁白**：李杰把兩張便條小心地放進口袋。

**李杰**：（內心，堅定地）明天...我先去三鷹...

**李杰**：（內心）去我原本的公寓...

**李杰**：（內心）也許...那裡會有答案...

**旁白**：李杰關上大輔房間的燈。走回自己的房間。

**旁白**：空蕩蕩的房子裡，只有他一個人。

**旁白**：他躺在床上，看著天花板。腦海中不斷回想著今天發生的事。

**旁白**：清水惠美的海報。那些奇怪的便條。那兩個地址。

**李杰**：（內心，疲憊地）一切...都聯繫起來了...

**李杰**：（內心）列車、荒地、清水惠美、大輔、父母...

**李杰**：（內心）還有...我自己...

**李杰**：（內心）答案...一定就在那裡...

**旁白**：李杰閉上眼睛。

**旁白**：外面傳來蟬鳴聲。夏夜漫長。

**旁白**：但李杰知道，明天，他將踏上新的旅程。
