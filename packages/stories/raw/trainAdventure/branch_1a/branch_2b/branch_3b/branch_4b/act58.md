# 第五十八幕：藍色禮物盒

**旁白**：隔天早上，李杰坐在餐桌前。他看著那兩張便條。黃色的紙，奇怪的文字。

**李杰**：（內心）三鷹...下連雀3丁目...我原本的公寓...

**旁白**：李杰回想起上次去那裡的情景。那是幾個月前。他一個人去調查。他來到三樓，站在302號室的門前敲門。沒有人回應。

**旁白**：然後，一個中年婦女提著購物袋走上樓梯。她警惕地問李杰為什麼站在她家門口。她說她住在這裡已經五年了。

**李杰**：（內心）有人住在那裡...我要怎麼進去搜索...如果處理不當...可能會驚動她...那我就再也沒機會進去了...

**旁白**：李杰站起來，在房間裡來回踱步。

**李杰**：（內心）我需要一個理由...一個能進去看看的理由...水管工？維修人員？對...就說是定期管道檢查...舊公寓的管道經常有問題...這個理由應該合理...

**李杰**：（內心，苦笑）我從來沒做過這種事...

**旁白**：李杰想了很久。

**李杰**：（內心，堅定地）我必須去...那個公寓是舊公寓...應該沒有監控攝影機...我可以...偽裝成水管工...先進去看看情況...然後再想下一步...

**旁白**：李杰換了一套乾淨的工作服和深色長褲。他拿了一個工具包。裡面放了一些扳手和螺絲起子。看起來像個水管工。

**李杰**：（內心，緊張地）希望...能順利...

---

**旁白**：李杰坐上電車，前往三鷹。電車搖晃著，窗外的風景飛速掠過。李杰握著工具包，手心出汗。

**旁白**：三鷹站到了。李杰下車，走在熟悉又陌生的街道上。下連雀3丁目。那棟舊公寓就在前方。灰色的牆，生鏽的欄杆，狹窄的樓梯。

**李杰**：（內心）就是這裡...我原本...住在這裡...

**旁白**：李杰走上樓梯。九樓。走廊很暗，只有幾盞昏黃的燈。902號室。李杰站在門前，深吸一口氣。

**李杰**：（內心）冷靜...冷靜...

**旁白**：他按下門鈴。叮咚。等了一會兒。門裡傳來腳步聲。門打開了一條縫。一個小女孩探出頭來。大約十歲左右。綁著馬尾，穿著粉紅色的T恤。

**小女孩**：（小心地）你是誰？

**旁白**：一個小男孩也跑到門邊。大約八歲左右，穿著藍色短褲。

**小男孩**：（好奇地）姐姐，是誰啊？

**李杰**：（愣了一下，快速反應）呃...你好...我是...大樓的水管工...

**小女孩**：（歪著頭）水管工？

**李杰**：（拿出工具包，給他們看）對...我們在做定期的管道檢查...看看有沒有漏水或堵塞的問題...

**小男孩**：（興奮地）哇！工具！

**小女孩**：（猶豫地）可是...媽媽不在家...她說不能讓陌生人進來...

**李杰**：（溫和地）我理解...那我可以先問你們幾個問題嗎？不會很久的...

**小女孩**：（想了想）嗯...好吧...

**李杰**：（拿出筆記本）謝謝你們...請問...你們家的水管最近有沒有什麼問題？

**小女孩**：（搖頭）沒有欸...

**小男孩**：（也搖頭）我們家的水管都很好！

**李杰**：（記錄著）那就好...那...最近有沒有聽到奇怪的聲音？比如水管的聲音...

**旁白**：李杰試探性地往裡面看。玄關、客廳、廚房。和他記憶中的布局一樣。但家具、裝潢都不一樣了。

**李杰**：（內心）就是這裡...我原本住的地方...

**李杰**：（繼續問）那...最近有沒有發生什麼...奇怪的事情？

**小女孩**：（困惑地）奇怪的事情？

**小男孩**：（想了想）什麼叫奇怪的事情？

**李杰**：（解釋）比如...漏水啊...或是...不尋常的聲音之類的...

**小女孩**：（想了想）嗯...沒有欸...

**旁白**：李杰有些失望。

**李杰**：（點頭）好的...那就沒什麼問題了...謝謝你們的配合...

**小女孩**：（突然想起）啊！對了！

**小男孩**：（也興奮地）對！那個盒子！

**李杰**：（心跳加速）盒子？你們說...有人在枕頭下面放了一個盒子？

**小女孩**：（點頭）對！前幾天媽媽在我枕頭下面發現了一個奇怪的盒子！

**小男孩**：（插嘴）藍色的！很漂亮！

**李杰**：（試圖保持冷靜）藍色的盒子...在枕頭下...這確實很奇怪...你們有看清楚那個盒子長什麼樣子嗎？

**小女孩**：（開心地）我本來以為是聖誕老公公送的禮物！

**小男孩**：（點頭）對啊！我也想要一個！

**小女孩**：（有點失望）但是我們怎麼都打不開...

**小男孩**：（用力比劃）我很用力地拉，但是打不開！

**小女孩**：（繼續說）媽媽說看起來很可疑...就準備丟掉...

**李杰**：（內心，震驚）枕頭下的盒子...而且打不開...

**李杰**：（緊張地問）丟掉了？已經丟了嗎？還是...還在家裡？

**小女孩**：（搖頭）還沒有...媽媽說今天要丟...現在還在家裡...

**小男孩**：（好奇地）你為什麼想知道那個盒子？

**李杰**：（內心，興奮）還沒丟...還在家裡...

**李杰**：（溫和地笑著）啊...其實...有時候奇怪的東西可能會影響水管系統...

**小女孩**：（困惑地）盒子會影響水管嗎？

**李杰**：（編理由）呃...有可能...不過...我可以幫你們把垃圾拿去丟嗎？包括那個盒子...

**小男孩**：（更困惑）為什麼？

**李杰**：（解釋）因為...我們水管工也有義務幫助居民保持環境整潔...而且我剛好要下樓...順便幫你們丟...

**小女孩**：（想了想）真的嗎...

**李杰**：（點頭）當然...而且...你們兩個小朋友...拿垃圾下樓也不太方便對吧...

**小男孩**：（點頭）對啊！垃圾好重！

**小女孩**：（猶豫地）那...好吧...我去拿...謝謝叔叔...

**旁白**：小女孩轉身進屋。過了一會兒，她拎著一個垃圾袋出來。

**小女孩**：（遞給李杰）給你...那個盒子也在裡面...

**李杰**：（接過，感覺有點重）好的...我幫你們拿去丟...對了，你們要記得跟媽媽說有水管工來檢查過了...

**小男孩**：（揮手）謝謝叔叔！

**小女孩**：（也揮手）謝謝你！

**李杰**：（尷尬地笑）不...不客氣...這是我應該做的...你們也要好好照顧自己...

**旁白**：兩個小孩關上門。李杰站在走廊上，握著那個垃圾袋。

**李杰**：（內心）藍色的盒子...在枕頭下...他們也打不開...這絕對不是巧合...

**旁白**：李杰快步走到走廊盡頭的樓梯間。他把垃圾袋放在地上，打開。垃圾袋裡有一些廚餘、紙屑、塑膠袋。李杰仔細翻找。然後，他看到了。

**旁白**：一個小小的盒子。藍色的。大約手掌大小，方方正正。表面光滑，像是某種特殊的材質。上面有精緻的金色花紋。

**李杰**：（拿起盒子，手在顫抖）這就是...他們說的那個藍色盒子...

**旁白**：盒子很輕，但有種奇怪的質感。摸起來不像塑膠，也不像金屬。溫溫的，像是有生命一樣。

**李杰**：（內心）為什麼...會在枕頭下...是大輔留下的嗎...還是...有其他人...

**旁白**：李杰試著打開盒子。但盒子沒有明顯的開口。表面光滑，找不到縫隙。

**李杰**：（皺眉）怎麼打開...這上面完全看不到任何縫隙或是按鈕...

**旁白**：李杰仔細檢查盒子的每一面。轉動、按壓、搖晃。但盒子紋絲不動。

**李杰**：（內心，焦慮）那兩個小孩說他們也打不開...難道...只有特定的人才能打開嗎...

**旁白**：李杰試了各種方法。用指甲摳、用力擰、甚至輕輕敲擊。但盒子依然緊閉。

**李杰**：（內心，沮喪）為什麼...我也打不開...裡面到底藏著什麼...

**旁白**：李杰把盒子舉到眼前，仔細觀察。藍色的表面，金色的花紋。花紋像是某種古老的文字，又像是裝飾圖案。

**李杰**：（內心，突然注意到）等等...這些花紋...

**旁白**：李杰仔細看著那些金色的花紋。有些花紋...和便條上的文字很像。那些奇怪的符號，彎曲的線條。

**李杰**：（內心，興奮）和便條上的文字一樣...難道...這些花紋也是文字...

**旁白**：李杰試著理解那些符號的意思。但腦海中什麼都沒有浮現。

**李杰**：（內心，困惑）沒有意思...便條上的文字...我能理解意思...但這些...雖然看起來很像...卻什麼意思都沒有...只是裝飾嗎...還是...我理解不了...

**李杰**：（喃喃自語）一定有辦法打開...大輔留下這個...一定是有原因的...

**旁白**：但無論他怎麼嘗試，盒子就是打不開。

**李杰**：（內心，決定）算了...先帶回去...慢慢研究怎麼打開...

**旁白**：李杰把盒子小心地放進口袋。他看著手上的垃圾袋。

**李杰**：（苦笑）我還得把這個垃圾丟掉...不然會顯得很可疑...

**旁白**：李杰把垃圾袋綁好，拿去樓下的垃圾場。然後，他離開了那棟公寓。走在三鷹的街道上。他摸著口袋裡的盒子。

**李杰**：（內心）大輔...你到底留下了多少線索...你是想告訴我什麼...為什麼我也打不開這個盒子...裡面到底藏著什麼...

**旁白**：李杰看著手機上的時間。下午兩點。

**李杰**：（內心）下一個地址...是上海...但我現在沒有護照...沒辦法去中國...

**李杰**：（內心，沮喪）我該怎麼辦...

**旁白**：李杰坐在公園的長椅上。摸著口袋裡的盒子。藍色的，打不開的盒子。

**李杰**：（內心）那兩個小孩也打不開...為什麼...那兩個小孩...好像...有點眼熟...

**旁白**：李杰想了想。十歲左右的女孩，綁著馬尾。八歲左右的男孩，穿著藍色短褲。

**李杰**：（內心）也許...只是我的錯覺...算了...應該是我想太多了...

**旁白**：遠處傳來孩子的笑聲。公園裡有幾個家庭在野餐。陽光很好，微風輕拂。但李杰感覺不到任何溫暖。他只感到迷茫和孤獨。

**李杰**：（內心，疲憊地）大輔...爸...媽...你們...都去哪裡了...我...該怎麼找到你們...

**旁白**：李杰站起來。他決定先回家。好好想想下一步該怎麼做。

**李杰**：（內心）上海的地址...中國上海市浦東新區世紀大道1000號1410室...我必須去那裡...也許...那裡會有答案...

**李杰**：（內心，想起）但是...護照...我...或是蓮...有護照嗎...我需要先回家找找看...如果有護照...我就可以訂機票去上海...如果沒有...申請護照需要時間...那就麻煩了...

**旁白**：李杰走向車站。口袋裡的盒子，輕輕地撞擊著他的大腿。像是在提醒他，這一切都是真的。

**李杰**：（內心）大輔...你留下了兩個地址...三鷹的盒子...我已經找到了...那麼...上海那裡...會有什麼在等著我呢...
