# 第六十五幕：離別之前

**旁白**：教室裡安靜下來。

**旁白**：李杰看著手中的兩顆藥丸。紅色的。藍色的。

**旁白**：他的手在顫抖。

**李杰**：（內心）如果我選藍色...我可以繼續留在這裡...

**李杰**：（內心）和爸媽在一起...和朋友們在一起...過著正常的生活...

**李杰**：（內心）但這一切...都是假的...

**旁白**：李杰閉上眼睛。深吸一口氣。

**李杰**：（內心）不...我不能這樣...

**李杰**：（內心）我不能因為害怕痛苦...就選擇逃避真相...

**李杰**：（內心）我不能...永遠活在謊言裡...

**旁白**：李杰睜開眼睛。他的眼神變得堅定。

**李杰**：（深吸一口氣，看著大家）這兩顆藥丸...我們都不要吃。

**如月颯真**：（點頭）對，來路不明的東西當然不能亂吃。

**氷室陸斗**：（也點頭）對啊，誰知道是什麼...萬一有毒怎麼辦。

**李杰**：（把藥丸收進口袋）我會把它們收好...也許以後能找專家鑑定...

**七海詩織**：（鬆了一口氣）那就好...我還以為你真的要吃...

**高橋彩花**：（輕聲）這樣最安全...

**渡邊真理**：（點頭附和）對啊...

**旁白**：李杰把紅色和藍色的藥丸都收進口袋。然後把兩個盒子也收好。

**李杰**：（內心）對不起...我騙了你們...

**李杰**：（內心，看著口袋）我已經決定了...我會吃紅色藥丸...

**李杰**：（內心）但不是現在...在那之前...我想和你們...好好相處...

**李杰**：（轉移話題）對了，上課要開始了，我們先進教室吧。

**旁白**：李杰背起背包，走進教室。五個同學跟在後面，彼此交換了擔心的眼神。

---

**旁白**：那天晚上。

**旁白**：李杰回到家。空蕩蕩的房子，只有他一個人。

**旁白**：父母失蹤已經快三週了。警方沒有任何線索。

**旁白**：李杰坐在房間裡，拿出那兩顆藥丸。

**李杰**：（看著紅色藥丸）只要吃下去...我就能回到原本的世界...

**李杰**：（內心）回到那個...爸媽已經死去的世界...

**旁白**：李杰握緊了藥丸。

**李杰**：（內心）但我還不想...這麼快就離開...

**李杰**：（內心）這個世界雖然是假的...但這裡的人...對我的感情...是真的...

**李杰**：（內心，看著房間裡的照片）這個世界的爸媽...雖然不是我真正的父母...

**李杰**：（內心）但他們對我的愛...是真實的...

**旁白**：李杰站起來，走到窗邊。看著窗外的夜景。東京的燈火，一望無際。

**李杰**：（內心）我想...再多待一段時間...

**李杰**：（內心）在離開之前...好好珍惜這裡的時光...

**李杰**：（內心）和爸媽在一起...和朋友們在一起...

**李杰**：（內心）把這些美好的記憶...帶回原本的世界...

**旁白**：李杰把紅色藥丸小心地放進抽屜最深處。藍色藥丸也一起放進去。

**李杰**：（內心）等到時機到了...我就會做出選擇...

**旁白**：李杰關上抽屜。轉身走出房間。

**旁白**：他決定了。在服用紅色藥丸之前，他要好好享受這個世界的最後時光。

---

**旁白**：接下來的幾天，李杰努力讓自己的生活回到正常。

**旁白**：上課、吃飯、和朋友聊天。他試著不去想那些複雜的事情。

**旁白**：但他心裡很清楚。這些日子，都是有期限的。

**旁白**：他不知道自己還能在這個世界待多久。也許幾個月，也許半年，也許一年。

**旁白**：但總有一天，他會吃下那顆紅色藥丸。回到原本的世界。

**旁白**：所以...他想在那之前...留下一些美好的回憶。

---

**旁白**：十月中旬。某個晴朗的午後。

**旁白**：午休時間。李杰和五個同學坐在食堂裡。

**氷室陸斗**：（大口吃著拉麵）啊，真好吃！學校食堂的拉麵越來越好吃了！

**七海詩織**：（笑著）陸斗，你每天都說好吃...

**渡邊真理**：（小聲對彩花說）他好像什麼都覺得好吃...

**高橋彩花**：（也小聲回應，笑著）對啊...

**如月颯真**：（看著窗外）天氣真好...秋天的感覺越來越明顯了。

**旁白**：李杰坐在一旁，看著朋友們。

**李杰**：（內心）我想...和他們一起去旅行...

**李杰**：（內心）趁著還有時間...留下一些美好的回憶...

**李杰**：（突然開口）那個...大家...

**如月颯真**：（轉頭）嗯？怎麼了？

**李杰**：（有些緊張）我...我在想...要不要...今年冬天...一起去旅行？

**旁白**：五個人都停下動作，驚訝地看著李杰。

**氷室陸斗**：（放下筷子，震驚）蓮...你說什麼？

**七海詩織**：（也很驚訝）旅行...？

**李杰**：（點頭）對...我想...今年冬天...二月左右...我們一起去北陸...

**李杰**：（繼續說）去溫泉酒店...然後滑雪...你們覺得怎麼樣？

**旁白**：食堂裡突然安靜了幾秒。

**如月颯真**：（緩緩地）蓮...你...主動提出要組織旅行？

**高橋彩花**：（輕聲，有些不敢相信）這...這是真的嗎...

**渡邊真理**：（也很驚訝，小聲對彩花說）蓮從來沒有主動提過這種事...

**氷室陸斗**：（突然大笑）哈哈哈！太陽從西邊出來了嗎？蓮你居然會主動約大家出去玩！

**七海詩織**：（笑著打陸斗的手臂）陸斗！別這樣說！

**李杰**：（有些尷尬）我...我以前...很少主動嗎...

**如月颯真**：（苦笑）也不是很少...是幾乎沒有...

**高橋彩花**：（溫和地）不過...這是好事啊...

**渡邊真理**：（點頭附和）對啊...我們都很高興...

**氷室陸斗**：（興奮地）當然要去啊！北陸！溫泉！滑雪！聽起來就很讚！

**七海詩織**：（也很興奮）我也想去！好久沒有大家一起旅行了！

**如月颯真**：（看著李杰，笑著）好啊，我也沒問題。蓮，你來安排吧。

**李杰**：（鬆了一口氣，微笑）好...那我會開始規劃...

**氷室陸斗**：（拍拍李杰的肩膀）蓮，你最近真的變了不少啊。

**李杰**：（愣了一下）變了...？

**七海詩織**：（點頭）對啊...感覺你...比以前更主動了...

**如月颯真**：（看著李杰，有些擔心）是因為...你哥哥的事情嗎？

**旁白**：李杰沉默了一會。

**李杰**：（低聲）也許吧...

**李杰**：（抬起頭，微笑）我只是...覺得...應該要珍惜和大家在一起的時間...

**高橋彩花**：（溫柔地）這樣想很好啊...

**渡邊真理**：（也點頭）對啊...

**氷室陸斗**：（大笑）好啦好啦！不要這麼嚴肅！我們是要去旅行，要開心一點！

**七海詩織**：（笑著）對啊！北陸！溫泉！我好期待！

**如月颯真**：（也笑著）那就這樣決定了。蓮，你規劃好之後告訴我們。

**李杰**：（點頭，真心地笑了）嗯...謝謝大家...

**旁白**：五個同學看著李杰的笑容。雖然他們感覺到李杰最近有些不同，但看到他笑得這麼開心，他們也很高興。

---

**旁白**：那天晚上。

**旁白**：李杰坐在書桌前，打開筆記型電腦。

**旁白**：他開始搜尋北陸的溫泉酒店和滑雪場。

**李杰**：（看著螢幕）金澤...富山...福井...

**李杰**：（點開一個溫泉酒店的網頁）這家看起來不錯...有露天溫泉...

**旁白**：李杰認真地看著每一家酒店的介紹。價格、設施、交通。他一一記錄下來。

**李杰**：（內心）滑雪場...最好是初學者也能玩的...

**李杰**：（繼續搜尋）白山...這裡不錯...有初級滑道...

**旁白**：李杰一邊做筆記，一邊計劃行程。

**李杰**：（內心）二月...應該是最適合滑雪的季節...

**李杰**：（內心）六個人...需要訂三間房...

**旁白**：李杰看著螢幕上的照片。白雪皚皚的山脈。溫暖的溫泉。笑得燦爛的遊客。

**李杰**：（內心，微笑）我想...和大家一起...創造一些美好的回憶...

**李杰**：（內心）趁著...還在這個世界的時候...

**旁白**：李杰繼續規劃著。他認真地看著每一個細節，想讓這次旅行完美。

**旁白**：窗外，秋夜的涼風吹進房間。東京的夜晚，燈火依然璀璨。

**旁白**：李杰不知道自己還能在這個世界待多久。

**旁白**：但至少現在，他有了一個期待。

**旁白**：一個和朋友們一起創造美好回憶的機會。

**旁白**：在他回到真實世界之前。
