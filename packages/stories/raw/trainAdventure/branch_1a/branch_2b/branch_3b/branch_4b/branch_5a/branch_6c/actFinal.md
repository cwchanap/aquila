# 終幕：遊戲結束

**旁白**：李杰握著那顆紅色藥丸，手在顫抖。

**旁白**：大輔靜靜地坐在對面，等待著他的選擇。

**旁白**：車廂裡只有列車行駛的聲音，還有兩人的呼吸聲。

**李杰**：（緩慢地，聲音低沉）大輔...

**斎藤大輔**：（看著李杰）嗯？

**李杰**：（抬起頭）我...不吃。

**旁白**：大輔的眼睛微微睜大。

**斎藤大輔**：（驚訝）什麼？

**李杰**：（把藥丸放回口袋）紅色也好，藍色也好...我都不吃。

**旁白**：李杰站起來，看著大輔。

**李杰**：（平靜地）你說這是遊戲。你說系統要我做出選擇。

**李杰**：（聲音變得堅定）那我就...不選。

**李杰**：（看著大輔的眼睛）我不想按照系統告訴我的去做。

**旁白**：大輔盯著李杰看了好一會兒。

**旁白**：然後，他深深地嘆了一口氣。

**斎藤大輔**：（苦笑）是嗎...

**斎藤大輔**：（站起來）我應該想到的...

**斎藤大輔**：（看著李杰，眼神複雜）你...還是和以前一樣固執啊，李杰。

**李杰**：（困惑）以前...？

**旁白**：大輔沒有回答。他只是微笑著，慢慢地往後退。

**斎藤大輔**：（溫和地）那麼...再見了。

**斎藤大輔**：（停頓）祝你好運。

**旁白**：話音剛落——

**旁白**：大輔的身影開始變得透明。

**李杰**：（伸手）等等...大輔...

**旁白**：但已經來不及了。大輔的身體像霧氣一樣，慢慢消散在空氣中。

**旁白**：幾秒鐘後，他完全消失了。車廂裡只剩下李杰一個人。

**李杰**：（低聲）大輔...

**旁白**：李杰站在原地，看著大輔消失的地方。

**旁白**：然後——

**旁白**：燈光閃爍了一下。

**旁白**：李杰感到一陣眩暈。他閉上眼睛，身體搖晃著。

**旁白**：當他再次睜開眼睛的時候——

---

**旁白**：他回到了那節車廂。

**旁白**：但不是空蕩蕩的車廂。是有乘客的車廂。

**旁白**：李杰環顧四周。大學生們坐在中段，還在聊天。其他乘客分散在車廂各處。

**健談男大生**：（笑著）所以我就跟他說，你這樣不行啊...

**活潑女大生**：（笑）對啊對啊...

**旁白**：一切都很正常。就像普通的末班車一樣。

**李杰**：（內心，緊張）我...我回來了...？

**旁白**：李杰慢慢走向車門，看著窗外。

**旁白**：黑暗的隧道。偶爾閃過的燈光。熟悉的景色。

**李杰**：（內心，失落）還是...這個世界...

**旁白**：列車繼續前行。李杰能聽到車輪與軌道摩擦的聲音。

**旁白**：「下一站，中野。」

**旁白**：廣播聲響起。熟悉的聲音。正常的聲音。

**李杰**：（內心）中野...然後是三鷹...

**李杰**：（內心，苦澀）還是...這個世界...

**旁白**：列車駛入中野站。幾個乘客下車，又有幾個乘客上車。

**旁白**：一切都和平常的末班車一樣。沒有任何異常。

**李杰**：（坐在座位上，低頭）...

**旁白**：列車離開中野站，繼續前往三鷹。

**旁白**：李杰看著窗外飛逝的景色。心情沉重。

**李杰**：（內心，失望）我還是...在這個假的世界...

**李杰**：（內心）同學們...已經死了...

**李杰**：（內心）爸媽...還是失蹤...

**李杰**：（內心，痛苦）什麼都沒有改變...

**旁白**：李杰靠在椅背上，閉上眼睛。

**旁白**：疲憊與失望如潮水般襲來。太多事情發生了。太多無法理解的事情。

**李杰**：（內心，疲憊）但...至少...

**李杰**：（內心）列車...沒有把我帶到另一個奇怪的地方...

**旁白**：列車平穩地行駛著。車輪與軌道的摩擦聲規律地響著。

**旁白**：咔噠，咔噠，咔噠...

**旁白**：李杰的呼吸漸漸平穩。

**李杰**：（內心，沮喪）我選擇了不吃藥丸...我反抗了系統...

**李杰**：（內心）但我...還是在這個假世界裡...

**李杰**：（內心，自嘲）什麼都沒有改變...

**旁白**：就在這個時候——

**旁白**：一陣刺耳的金屬摩擦聲突然響起。

**李杰**：（猛然睜開眼睛）什麼...

**旁白**：列車劇烈搖晃起來。

**健談男大生**：（驚叫）怎麼回事？！

**活潑女大生**：（尖叫）啊啊啊！

**旁白**：車廂裡的乘客們驚慌失措，抓住扶手和座椅。

**旁白**：列車的搖晃越來越劇烈。金屬扭曲的聲音刺耳地響起。

**李杰**：（抓住座椅，驚恐）不...不可能...

**旁白**：然後——

**旁白**：一聲巨響。

**旁白**：列車猛烈地傾斜。

**旁白**：李杰的身體被甩向一側。他撞上車窗，劇痛襲來。

**旁白**：玻璃碎裂的聲音。金屬撕裂的聲音。人們的尖叫聲。

**旁白**：一切都混雜在一起，變成一片混亂。

**李杰**：（內心，恐懼）出軌了...列車...出軌了...

**旁白**：車廂在翻滾。天旋地轉。

**旁白**：李杰感到身體撞上堅硬的東西。一次，兩次，三次。

**旁白**：劇痛。難以忍受的劇痛。

**旁白**：然後——

**旁白**：一切都靜止了。

**旁白**：李杰躺在扭曲的車廂裡。他能感覺到溫熱的液體從身體流出。

**旁白**：他能聽到遠處傳來的呻吟聲，哭泣聲。

**旁白**：他想動，但身體不聽使喚。

**李杰**：（內心，虛弱）為什麼...

**李杰**：（內心）為什麼會這樣...

**旁白**：視線開始模糊。天花板的碎片，扭曲的座椅，破碎的玻璃。

**旁白**：一切都在遠去。

**李杰**：（內心，最後的思緒）為什麼...

**李杰**：（內心）我以為...至少...什麼都不會改變了...

**旁白**：但沒有回答。

**旁白**：只有逐漸遠去的聲音。

**旁白**：還有那越來越深的黑暗。

---

**旁白**：2026年9月15日，深夜。

**旁白**：JR中央線一列末班車在中野站與三鷹站之間出軌。

**旁白**：車廂嚴重損毀，多人受傷。

**旁白**：其中一名乘客，身分確認為斎藤蓮，28歲，當場死亡。

**旁白**：事故原因不明。

**旁白**：調查還在進行中。

---

**旁白**：黑暗中，什麼都沒有了。

**旁白**：沒有列車。

**旁白**：沒有藥丸。

**旁白**：沒有選擇。

**旁白**：只有寂靜。

**旁白**：永恆的寂靜。
