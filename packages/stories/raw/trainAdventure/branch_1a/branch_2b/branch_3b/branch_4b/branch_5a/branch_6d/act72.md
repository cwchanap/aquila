# 第七十二幕：死亡遊戲

**旁白**：李杰在荒地上行走。一步,一步。他不知道走了多久。

**旁白**：那座建築物的輪廓逐漸清晰。從遠處的模糊影子,變成了具體的形狀。

**旁白**：當李杰終於靠近時,他停下了腳步。

**旁白**：那是一座學校。

**旁白**：一座三層樓的現代化校舍,白色的外牆,大片的玻璃窗,典型的日本高中建築風格。

**旁白**：但它孤零零地矗立在這片紫色天空下的荒地中,四周什麼都沒有。沒有操場,沒有圍牆,沒有其他建築。

**李杰**：（內心,感到詭異）高中...為什麼會有一座高中在這裡...

**旁白**：李杰走向學校的大門。那是一扇玻璃門,可以看到裡面的走廊和樓梯。

**旁白**：他伸手,握住門把,用力推開。

**旁白**：門開了。

---

**旁白**：李杰走進去,來到一樓的大廳。然後,他停住了。

**旁白**：大廳裡,站著一群人。

**旁白**：李杰的視線掃過他們,然後他的呼吸停止了。

**旁白**：他認識這些人。

**旁白**：有五個年輕人站在一起,看起來二十歲左右,正在困惑地討論著什麼。

**旁白**：如月颯真。氷室陸斗。七海詩織。高橋彩花。渡邊真理。

**李杰**：（內心,震驚）他們...他們是...

**李杰**：（內心）那五個...在鏡像世界裡...當我以斎藤蓮的身分生活時...同一個大學班級的同學...

**旁白**：李杰的視線移向其他人。

**旁白**：角落裡,站著一個肥胖的年輕男子,穿著白色的襯衫。他的臉圓圓的,五官擠在一起,看起來有點滑稽。他正低著頭,像是在思考什麼。

**李杰**：（內心,心跳加速）大輔...斎藤大輔...

**旁白**：靠近窗戶的地方,一個二十多歲的年輕男子正來回踱步,臉上寫滿焦慮。

**李杰**：（內心）海斗...中野海斗...

**旁白**：而在另一側,一個年輕女子和一個穿著黑色外套的男子站在一起,正在低聲交談。

**旁白**：女子有著及肩的黑色長髮,看起來文靜溫柔。男子戴著帽子,神情冷漠。

**李杰**：（內心,無法置信）恵美...清水恵美...還有那個黑衣男子...

**李杰**：（內心,回想起來）我在第一幕...在新宿車站的月台上...看到過他...

**旁白**：李杰的目光最後落在大廳角落的一個中年男子身上。

**旁白**：那個男子獨自一人站在那裡,雙手抱胸,表情陰沉,不與任何人互動。

**李杰**：（內心,認出他）那個自私的中年男...我在第二幕...在隧道裡見過他...

**旁白**：所有人都注意到了李杰的到來。他們轉頭看向他,臉上都露出驚訝的表情。

**中野海斗**：（眼睛瞪大,驚喜）李杰?! 你怎麼會在這裡?!

**旁白**：海斗立刻跑了過來,臉上滿是激動。

**中野海斗**：（抓住李杰的肩膀,欣喜）天啊,真的是你! 我還以為...我還以為我再也見不到你了!

**李杰**：（勉強擠出笑容,內心複雜）海斗...我也沒想到會在這裡看到你...

**旁白**：李杰的視線越過海斗,看向那五個同學。他們正困惑地看著李杰和海斗。

**如月颯真**：（困惑,對其他同學）你們...認識這個人嗎?

**氷室陸斗**：（搖頭）不認識...

**七海詩織**：（看著李杰,皺眉）我沒見過他...

**李杰**：（內心,感到一陣複雜）他們...不認識我...

**李杰**：（內心,恍然）對了...我現在是李杰...不是斎藤蓮了...

**李杰**：（內心）在鏡像世界裡,我是以斎藤蓮的身分和他們相處...但現在...我回到了李杰的身分...

**旁白**：李杰的視線移向角落的大輔。大輔正看著他,那張滑稽的圓臉上露出一個奇怪的表情——像是在努力回想什麼,眉毛一挑一挑的,嘴角微微抽動。

**旁白**：兩人的目光對上了幾秒鐘。大輔突然眨了眨眼睛,做了一個誇張的鬼臉,然後轉開了頭,不再看李杰。

**李杰**：（內心,感到疑惑）大輔...他看我的眼神...好像認識我...那個鬼臉是什麼意思...?

**李杰**：（內心,不解）但為什麼? 當我是李杰的時候...我應該不認識大輔才對...

**李杰**：（內心）除非...他還記得在列車上的那次對話...

**中野海斗**：（焦急地問）李杰,你知道這裡是哪裡嗎? 你知道發生了什麼事嗎?

**李杰**：（搖頭,聲音平靜）我不知道...我也很困惑...

**李杰**：（看著海斗,反問）倒是你...海斗,你怎麼會在這裡?

**中野海斗**：（苦笑,無奈）我也不知道啊...我昨晚在公寓睡覺...然後醒來就在這裡了...

**旁白**：那五個同學也走了過來。

**高橋彩花**：（緊張地問）你...你是從外面進來的嗎?

**李杰**：（點頭）是的。我從外面走進來的。

**渡邊真理**：（急切）外面是什麼情況?! 這裡到底是哪裡?!

**李杰**：（沉默了一下,緩緩說）外面...是一片荒地。紫色的天空,兩個月亮,灰色的沙漠。

**如月颯真**：（震驚）什麼...兩個月亮?!

**氷室陸斗**：（不敢相信）這...這不可能...

**七海詩織**：（聲音顫抖）我們...我們到底在哪裡...

**旁白**：詩織的聲音裡帶著恐懼。其他人的臉色也變得蒼白。

**中野海斗**：（對李杰解釋）我們五個...是在北陸旅行的時候...在旅館睡著後...醒來就在這裡了...

**高橋彩花**：（補充,聲音急促）醒來的時候,我們就在這座學校的教室裡...然後我們走出來,看到其他人也陸續來到大廳...

**渡邊真理**：（指向大門）我們試著打開那扇門...想看看外面...但門打不開...

**如月颯真**：（沮喪）無論怎麼推,怎麼拉,都打不開...就像鎖死了一樣...

**氷室陸斗**：（看著李杰,困惑）但你...你卻能從外面進來...

**七海詩織**：（急切地問）你是怎麼進來的?! 門開了嗎?!

**李杰**：（點頭,平靜地）門沒有鎖。我推開就進來了。

**旁白**：所有人都愣住了。他們互相看著彼此,臉上滿是困惑和不解。

**中野海斗**：（喃喃自語）這到底是怎麼回事...我們打不開的門...他卻能打開...

**旁白**：就在這時——

**旁白**：大廳的廣播突然響起。

**旁白**：「滋滋——」雜訊聲過後,一個冰冷的、機械化的女性聲音響徹整個大廳。

**廣播**：歡迎...來到這個死亡遊戲。

**旁白**：所有人都僵住了。

**旁白**：大廳裡一片死寂。只有廣播的聲音在迴盪。

**廣播**：各位參與者,請注意聽好規則。

**廣播**：這裡是遊戲場地。你們每個人都是玩家。

**廣播**：遊戲的目標...是到達祭壇。

**旁白**：詩織發出一聲驚恐的抽泣。彩花緊緊抓住真理的手臂。

**中野海斗**：（聲音顫抖,低聲）死亡...遊戲...? 祭壇...?

**如月颯真**：（急切地問）什麼祭壇?! 在哪裡?!

**旁白**：但廣播沒有回答。只是繼續用那冰冷的聲音說下去。

**李杰**：（內心,冷靜地想）又是一個遊戲...到達祭壇...但沒有說祭壇在哪裡...也沒有說規則...

**李杰**：（內心）我才剛逃出那個假世界...現在又被困在另一個遊戲裡...

**旁白**：李杰的臉色沒有變化,但他的雙手緊緊握成拳頭。

**旁白**：廣播繼續響起,冰冷的聲音宣告著他們的命運。

**廣播**：遊戲即將開始。祝各位...好運。

**旁白**：「滋——」廣播聲消失了。

**旁白**：大廳裡,只剩下眾人急促的呼吸聲,和恐懼的沉默。
