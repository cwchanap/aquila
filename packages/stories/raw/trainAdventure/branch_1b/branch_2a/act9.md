# 第九幕：林中小徑與夜半小屋

**旁白**：李杰看著眼前緊緊相依的母子三人,又望向那條通往未知黑暗的鐵軌。紫月的光芒灑在他們身上,將他們的影子拉得很長。

**旁白**：幾秒鐘的沉默後,他做出了決定。

**李杰**：我跟你們一起走。

**溫柔母親**：(抬起頭,眼中閃過驚訝)你⋯⋯你願意?

**李杰**：但是,走鐵軌太危險了。如果列車再回來,或者有其他危險⋯⋯

**旁白**：說著,他從口袋裡掏出那張從車站裡找到的泛黃地圖。他蹲下身,將地圖攤開在地上。

**李杰**：剛才我在車站裡找到了這個。你看,地圖上標示著一條通往北邊「亡者峽谷」的路徑。

**李杰**：我們沿著這條路走,應該會比沿著鐵軌安全。

**溫柔母親**：(看著地圖,眼中重新燃起一絲希望)太好了⋯⋯真的⋯⋯謝謝你。

**懂事姐姐**：(小聲)有大哥哥一起,我就不那麼害怕了⋯⋯

**天真弟弟**：(拉著李杰的衣角)大哥哥,你手上的那個是什麼?

**李杰**：(看了看手中的鏽劍)這是⋯⋯保護我們的東西。

**旁白**：在李杰的帶領下,他們離開了死寂的月台,沿著車站外圍尋找地圖上標示的路口。

**旁白**：兩輪紫月依然懸在天空,為這個詭異的世界提供著唯一的光源。周圍是一片深不見底的黑暗森林,風吹過時,樹葉發出沙沙的聲響。

**旁白**：很快,他們在車站外不遠處找到了地圖上標示的路口。

**旁白**：令李杰有些驚訝的是,這條深入森林的小徑竟異常平整。路面由堅實的泥土和碎石鋪成,邊緣甚至還有整齊的石塊標記。

**李杰**：(內心)這條路⋯⋯看得出有被長期維護的痕跡。這與車站的破敗形成了鮮明對比⋯⋯這個世界,真的有人在這裡生活嗎?

**旁白**：四人踏上了向北的小徑。腳下的碎石發出輕微的聲響,在寂靜的森林中格外清晰。

**旁白**：李杰走在最前面,手裡握著那柄鏽劍,警惕地觀察著四周。身後是那對母子三人,緊緊跟隨著。

**旁白**：孩子們似乎從最初的驚嚇中恢復了一些。雖然還是緊緊抓著母親的手,但至少不再哭泣了。

**天真弟弟**：(小聲問)媽媽,爸爸會不會已經在下一站等我們了?

**溫柔母親**：(溫柔地摸了摸兒子的頭)一定的。爸爸那麼厲害,他一定會在那裡等我們的。

**懂事姐姐**：媽媽,我會照顧好弟弟的。

**溫柔母親**：(微笑)陽菜真懂事⋯⋯

**旁白**：身後家人間的輕聲細語,在這死寂的森林裡,反而給了李杰一絲奇特的安心感。至少,他不是一個人面對這個世界。

**旁白**：走了一段路後,那位母親忽然開口。

**溫柔母親**：對了⋯⋯還一直沒請問你的名字。我叫小林智子,這是我的女兒陽菜,還有兒子翔太。

**李杰**：(回過頭,點了點頭)我叫李杰。杰出的杰。

**小林智子**：李先生,謝謝你願意留下來幫我們。

**李杰**：叫我李杰就好。

**小林陽菜**：(小聲)李杰哥哥⋯⋯

**小林翔太**：李杰哥哥!

**李杰**：(難得露出一絲笑容)嗯。

**旁白**：又走了一段路後,小林智子為了緩解緊張的氣氛,再次開口。

**小林智子**：李杰,你是做什麼工作的?

**李杰**：我是一名軟體工程師。

**小林智子**：是嗎?真厲害。不像我,只是一個全職的家庭主婦。

**小林智子**：我們家住在埼玉,孩子們都在東京這邊上學。昨天帶他們回我父母家吃飯⋯⋯

**李杰**：住埼玉的話,昨晚這麼晚搭車,還真是辛苦。

**小林智子**：(苦笑了一下)是啊,老人家一高興,就聊得晚了。沒想到就遇上了這種事⋯⋯唉。

**小林智子**：說起來,李杰你⋯⋯聽你的口音,好像不是關東地區的人?

**李杰**：嗯,我在中國長大的。大學畢業之後才來日本工作。

**小林智子**：原來是這樣啊⋯⋯一個人來這裡打拼,也很不容易呢。

**李杰**：還好。習慣了。

**旁白**：他們繼續前進。森林越來越茂密,兩側的樹木高大而古老,枝葉交錯在頭頂上方,幾乎遮蔽了天空。

**旁白**：只有零星的紫月光芒從樹葉間隙灑落,在地面上投下斑駁的光影。小徑依然平整,但周圍的氣氛越來越壓抑。

**旁白**：時間在行走與偶爾的交談中緩慢流逝。沒有手機,沒有手錶,他們只能憑感覺判斷時間。

**旁白**：一個小時。兩個小時。或許更久。

**旁白**：所有人都已經精疲力盡。李杰的腿開始發酸,握劍的手也開始發麻。而身後的母子三人,狀況更糟。

**小林翔太**：(拖著腳步,聲音疲憊)媽媽⋯⋯我走不動了⋯⋯

**小林智子**：(蹲下來,語氣溫柔但也掩不住疲憊)再堅持一下,好不好?我們⋯⋯我們很快就能找到休息的地方了⋯⋯

**小林陽菜**：(也很累,但強撐著)弟弟,我牽你。

**旁白**：李杰停下腳步,回頭看著這一家人。孩子們的臉色蒼白,小林智子的膝蓋還在滲血,傷口周圍沾滿了灰塵。

**李杰**：(內心)不能再繼續走了⋯⋯必須找個地方休息。但是這片森林⋯⋯

**旁白**：他望向四周。除了這條小徑,到處都是茂密的樹林。如果在野外過夜,誰知道會遇到什麼危險。

**李杰**：(對小林智子說)我們休息一下。

**小林智子**：(點頭,聲音虛弱)好⋯⋯

**旁白**：就在他們準備在路邊坐下休息時,李杰的目光忽然被前方吸引。

**旁白**：小徑的前方,樹林中的空地上,竟出現了一座木屋。

**李杰**：(驚訝)那是⋯⋯

**小林陽菜**：(抬起頭,驚訝地指著前方)媽媽,你看!那裡有房子!

**旁白**：那是一棟雙層的木屋,靜靜地矗立在林中的空地上。在紫月的光芒下,木屋的輪廓清晰可見。

**旁白**：窗戶裡沒有燈光,但結構完好。它甚至還帶有一個小小的門廊,門廊前有幾級木製台階。屋頂是尖頂設計,煙囪筆直地伸向天空。

**旁白**：整棟屋子看起來維護得很好,木材沒有腐朽的跡象,窗戶玻璃也是完整的。在這樣一個詭異的世界裡,突然出現這樣一棟體面的屋子,顯得極其不協調。

**李杰**：(內心)在這荒郊野外⋯⋯怎麼會有這種房子?而且看起來保養得這麼好⋯⋯

**小林智子**：(充滿希望)太好了⋯⋯有房子⋯⋯我們可以在裡面休息⋯⋯

**小林翔太**：(高興)我們可以睡覺了嗎?

**旁白**：儘管心中充滿疑惑,但看著疲憊不堪的母子三人,李杰知道他們迫切需要一個地方休息。而且,夜已經很深了,繼續趕路只會更危險。

**李杰**：我們先過去看看情況。如果沒人,我們今晚就在那裡休息。

**小林智子**：(點頭)好⋯⋯麻煩你了。

**旁白**：李杰握緊手中的鏽劍,警惕地走在前面。他們慢慢接近那棟木屋。

**旁白**：空地上很安靜。沒有燈光,沒有人聲,只有風吹過樹林的沙沙聲。木屋就像一座沉睡的巨獸,在黑暗中等待著他們的到來。

**旁白**：李杰走上門廊的台階。木板在他的重量下發出輕微的「吱呀」聲。他伸手推了推門把——

**旁白**：門沒有鎖。
