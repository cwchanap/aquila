# 第十五幕：衣櫃裡的秘密 

**旁白**:站在十字路口,李杰思考了片刻。北邊的線索已經斷了,但小屋和孩子們的失蹤是更大的謎團。

**旁白**:那間屋子裡一定還藏著什麼他沒發現的東西。他必須回去,必須再仔細搜查一次。

**李杰**:(握緊劍,低聲)回去。

**旁白**:他轉身,踏上了往南的小徑。疲憊的雙腿機械地邁動著,每一步都像踩在刀尖上。

**旁白**:紫月在樹冠間移動,光影在小徑上緩緩流轉。森林比他離開時更加寂靜,連蟲鳴都消失了,只剩下他自己的腳步聲和沉重的呼吸。

**旁白**:走了許久,那棟木屋再次出現在視野中。它靜靜地矗立在空地上,在紫月的光芒下顯得格外陰森。

**旁白**:李杰沒有從正門進入。他繞到側面,熟練地從那扇沒上鎖的窗戶翻進屋內。

**旁白**:雙腳落地的瞬間,一股冰冷的空氣撲面而來,讓他忍不住打了個寒顫。屋子裡的溫度比外面還要低。

**李杰**:(呼出白霧)好冷...

**旁白**:他走到門後,雙手用力抓住那根粗木條。木條很沉,他咬緊牙關,用盡全力才將它從門板和牆壁之間拖出來。

**旁白**:「嘎——」沉重的木條落在地板上,發出一聲悶響。伴隨著「喀」的輕響,門鎖彈開,主要通道被打開了。

**李杰**:(喘著氣)先補充一些補給...

**旁白**:他走到廚房,打開櫥櫃。裡面整齊地放著幾塊用油紙包裹的麵包,還有一個幾乎滿水的皮水袋。

**旁白**:他拿起麵包,準備放進背包。就在轉身的瞬間,紫月的光芒從窗戶斜射進來,以一個刁鑽的角度照亮了地板。

**旁白**:李杰的動作停住了。

**李杰**:(愣住)這是...

**旁白**:地板上,到處都是泥土的痕跡。那不是一行或兩行清晰的腳印,而是一大片混亂、互相重疊、無法辨認的印記。

**旁白**:它們集中在屋子的中央區域,從門口延伸到桌邊,又從桌邊散開到各個角落。像是好幾個人在這裡來回走動,互相追逐,踩踏出一片混亂。

**李杰**:(皺眉,內心)怎麼可能...我第一次進來的時候,為什麼完全沒有注意到?

**旁白**:他回想起第一次進屋的情景。當時天色很暗,他只用紫月微弱的光芒勉強辨認室內的佈局。

**旁白**:而且當時他被桌椅、床鋪的整潔所迷惑,先入為主地認為這裡很乾淨,根本沒有仔細檢查地板。

**李杰**:(低聲)該死...我太大意了。

**旁白**:他蹲下身,伸手觸碰那些泥土。已經半乾了,但還保留著一些濕潤。應該是在他離開後不久留下的。

**旁白**:這片混亂的痕跡,徹底推翻了他之前所有的猜測。這間屋子裡,絕對發生過什麼激烈的事情。

**李杰**:(站起身,握緊劍)必須再仔細找找...

**旁白**:他順著痕跡最密集的方向走去。在屋子的一個角落裡,他發現了一扇之前沒注意到的門。

**旁白**:那是一扇通往屋後的後門。門沒有關嚴,留著一絲縫隙。一股冷風從縫隙中吹進來,帶著潮濕泥土的氣味。

**李杰**:(伸手拉開門)

**旁白**:門外是一小片濕潤的泥地。紫月的光芒照亮了地面,清晰地顯示出好幾行混雜在一起的腳印。

**旁白**:腳印從旁邊的樹叢中延伸出來,在門外的泥地上重疊、交錯,然後消失在後門的門檻下。

**李杰**:(低聲,震驚)有東西...從後門闖進來了。

**旁白**:一股寒意從脊背竄上來。他突然意識到,自己可能一開始就錯了。

**李杰**:(內心)屋外的腳印那麼清晰...但屋內卻是一片混亂。

**旁白**:這意味著真正的襲擊,發生在屋內。通過這扇後門。

**旁白**:他緩緩關上後門,轉身重新審視這間屋子。目光從桌椅掃過,從床鋪掃過,從廚房掃過...

**旁白**:然後,他的視線停住了。

**旁白**:在臥室的角落,那張單人床的旁邊,靜靜地立著一個厚重的木製衣櫃。

**李杰**:(內心,自責)衣櫃...我之前兩次進出,竟然都完全忽略了它。

**旁白**:衣櫃是深色的木頭,高約兩米,寬度足以容納一個成年人。櫃門緊閉,表面有些許劃痕,但整體保存完好。

**旁白**:他深吸一口氣,強迫自己冷靜下來。握緊手中的劍,一步一步地走向那個衣櫃。

**旁白**:腳步聲在空蕩的屋內格外清晰。他的心臟劇烈地跳動,每一次心跳都像鼓聲般轟鳴在耳邊。

**李杰**:(停在衣櫃前,深呼吸)

**旁白**:他伸出左手,顫抖著觸碰到冰冷的木質櫃門。表面的木紋在指尖下清晰可感。

**旁白**:李杰用力抓住櫃門的把手,然後緩緩地、用盡全力地向外拉。

**旁白**:「嘎——」

**旁白**:一聲刺耳的摩擦聲響起,在寂靜的屋內格外刺耳。櫃門緩慢地打開,陳舊的鉸鏈發出抗議般的聲響。

**旁白**:伴隨著櫃門的開啟,一股淡淡的、卻無法忽視的血腥味從縫隙中飄散出來。

**李杰**:(屏住呼吸)

**旁白**:櫃門完全打開的瞬間,紫月的光芒照進了衣櫃深處。

**旁白**:李杰的瞳孔猛地收縮,整個人如遭雷擊般僵在了原地。

**旁白**:衣櫃裡,以一個極其扭曲、不自然的姿勢蜷縮著的,是一具女性的屍體。

**旁白**:頭部向後仰著,雙腿彎曲,整個身體被硬塞進狹小的空間裡。手臂無力地垂著,手指微微彎曲,像是在死前想要抓住什麼。

**旁白**:在她的頸部,有一道深深的、猙獰的傷口。血液早已凝固,在紫月的光芒下泛著暗紅色的光澤。

**旁白**:那張臉——

**旁白**:是小林智子。
