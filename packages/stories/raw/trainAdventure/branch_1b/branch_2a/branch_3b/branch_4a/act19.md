# 第十九幕：曦光之庭 

**旁白**:精靈似乎想起了什麼,表情變得有些凝重。

**精靈**：說起來,就在幾小時前,又有一輛列車抵達了車站。

**李杰**：(心中一動)列車?

**精靈**：比兩年前那次晚了很多。我們在上面...發現了兩具屍體。

**旁白**：李杰沉默片刻,在腦中回想起在「終焉祭壇」的情景。

**李杰**：(內心)那座車站的月台上什麼都沒有...沒有屍體,沒有血跡,什麼痕跡都沒有。

**旁白**：他記得當時的困惑——按照規律,應該有人在那裡下車,但月台卻空無一人。

**李杰**：(內心)如果那兩個人都沒有下車,那就意味著...他們都留在了列車上,然後死了。

**旁白**：但隨即,一個矛盾的地方浮現在他腦中。

**李杰**：(內心)等等...他剛才說,兩年前的列車上只有一具屍體。可是今天卻有兩具?

**旁白**：這個差異讓他感到困惑。兩年前是一具,今天是兩具。為什麼會不一樣?

**李杰**：(內心)是列車的規則改變了?還是...有什麼我不知道的因素?

**旁白**：他想不出合理的解釋,只能將這個疑問暫時埋在心底。

**精靈**：他們是你認識的人嗎?

**李杰**：(搖頭)算不上認識。只是和他們搭上了同一班車,一起來到這個世界的乘客而已。

**旁白**：李杰心中充滿了疑問。那兩個人是誰?是那個膽小的眼鏡男和焦慮的中年女人嗎?他們是怎麼死的?

**李杰**：我可以去看看那輛列車嗎?

**精靈**：當然可以。

**旁白**：他看了看李杰疲憊的樣子。

**精靈**：不過,看你的樣子非常疲憊。列車會在這裡停留一整天,時間還很充裕。

**精靈**：不如,你先隨我回村裡,補充些食物和水,好好休息一下吧。

**旁白**：李杰想了想,對方說得有道理。自己的身體也確實到了極限,雙腿幾乎要失去知覺,眼皮沉重得快要睜不開。

**李杰**：好,那就麻煩你了。

**旁白**：他猶豫了一下,決定先自我介紹。

**李杰**：我叫李杰。

**精靈**：(微微一笑,行了一個優雅的禮)我是伊蘭尼爾。

**旁白**：伊蘭尼爾的動作流暢而莊重,右手放在胸前,微微躬身,像是在進行某種古老的儀式。

**伊蘭尼爾**：我們的村莊,「曦光之庭」,就在附近。

**旁白**:李杰穿好衣服,將劍和背包收拾好,跟隨著伊蘭尼爾走進了森林深處。

**旁白**:伊蘭尼爾的腳步輕盈無聲,彷彿在空氣中滑行。李杰盡力跟上,但疲憊的身體讓他的步伐顯得笨拙。

**旁白**:走了大約十分鐘後,前方的景象突然開闊起來。

**旁白**:一座隱藏在巨樹之間、與自然完美融為一體的村莊出現在眼前。

**李杰**：(停下腳步,驚嘆)這是...

**旁白**：許多房屋直接建造在樹幹上,木質的結構與樹木本身渾然一體,彷彿是樹木自然生長出來的一部分。

**旁白**：精巧的吊橋連接著不同高度的房屋,在微風中輕輕搖晃。散發著柔和光芒的燈籠掛在枝葉間,在陽光的照耀下也依然閃爍著溫暖的光。

**旁白**：整個村莊寧靜而祥和,空氣中飄散著花香和木頭的氣息。

**旁白**：李杰的到來,立刻引起了村民們的好奇。

**旁白**：許多精靈從屋子裡探出頭來,或是從吊橋上停下腳步,對這位稀客指指點點。他們的眼中滿是純粹的好奇,沒有恐懼,也沒有敵意。

**精靈小孩**：(拉著母親的手,用清脆的聲音問)媽媽,這就是人類嗎?他的耳朵是圓的耶!

**旁白**：一個年幼的精靈從樹屋裡探出小小的腦袋,銀色的頭髮在陽光下閃閃發光。

**年長精靈**：(從吊橋上探身,溫和地問)年輕人,外面的世界...是什麼樣子的?和我們的森林一樣綠嗎?

**李杰**：(微笑)有些地方很綠,有些地方...就不太一樣了。

**旁白**：他耐心地回答了幾個簡單的問題,盡量用他們能理解的方式解釋。精靈們聽得津津有味,不時發出驚嘆聲。

**旁白**：伊蘭尼爾將他帶到了自己位於樹幹高處的家中。

**旁白**：他們爬上一道螺旋的木梯,來到一間建在粗壯樹幹上的房屋。房門由淺色的木材製成,上面雕刻著精美的藤蔓圖案。

**伊蘭尼爾**：(推開門)請進。

**旁白**：屋內陳設簡樸,卻充滿了自然的氣息。家具都是由木材和藤蔓編織而成,沒有任何金屬或塑料。

**旁白**：窗戶很大,陽光透過薄薄的布簾灑進來,在地板上投下溫暖的光斑。空氣中飄散著淡淡的草本植物香氣。

**伊蘭尼爾**：請坐。嚐嚐我們自己做的果茶和麵包吧。

**旁白**：他從櫃子裡取出一個陶製的茶壺和兩個杯子,還有一盤包裹著布巾的麵包。

**李杰**：(坐在柔軟的坐墊上)謝謝你,這已經非常豐盛了。

**旁白**：他接過伊蘭尼爾遞來的茶杯,溫熱的觸感讓他凍僵的手指漸漸恢復知覺。

**李杰**：這裡...真的很不可思議。

**旁白**：他環顧四周,看著這個與世隔絕的精靈村莊。

**李杰**：你們一直都住在這裡嗎?

**伊蘭尼爾**：(點頭,碧綠的眼眸中流露出一種古老的平靜)嗯。

**旁白**：他倒了兩杯淡金色的茶水,茶水散發著果香和花香混合的氣息。

**伊蘭尼爾**：我們的祖先追尋著陽光,在神樹的庇佑下建立了「曦光之庭」。

**伊蘭尼爾**：我們的生活很簡單。唱歌,編織,守護神樹...數百年來,幾乎沒有改變。

**伊蘭尼爾**：那輛列車,算是我們與外界唯一的、也是最神秘的聯繫了。

**旁白**：伊蘭尼爾看著李杰,眼中充滿了好奇。

**伊蘭尼爾**：你的世界...是什麼樣子的?和我們的森林很不一樣吧?

**李杰**：(喝了一口溫熱的果茶,思考著該如何解釋)很不一樣。

**旁白**：他停頓了一下,組織語言。

**李杰**：這麼說吧...我們的光,來自一種看不見的力量,我們稱之為「電」。

**李杰**：我們讓這種力量像水流一樣在細細的金屬絲裡流動,所到之處,就能讓特製的玻璃球發出穩定而強烈的光芒。

**伊蘭尼爾**：(專注地聽著,微微點頭)看不見的力量...

**李杰**：我們也有路,是用融化的黑色石頭鋪成的平坦「道路」。

**李杰**：人們乘坐著由金屬製成的、無需馬匹就能靠燃燒油料來自走的「車」,在上面高速移動。

**李杰**：我們的世界太大了,所以我們創造了一個看不見的、遍佈全球的「網路」,讓我們可以瞬間與千里之外的陌生人交談。

**伊蘭尼爾**：(靜靜地聽著,眼中流露出神往與一絲不解)燃燒油料的車...遍佈全球的網路...

**旁白**：他沉默了片刻,似乎在消化這些陌生的概念。

**伊蘭尼爾**：聽起來,是個既高效又複雜的世界。很難想像。

**李杰**：(苦笑)有時候,連我們自己都覺得太複雜了。

**旁白**:溫暖的茶水和帶著果香的麵包,讓李杰感到一股暖流傳遍全身。

**旁白**:窗外是精靈村莊祥和的景象,孩子們在吊橋上嬉戲,成年精靈們在樹下編織或聊天。耳邊是伊蘭尼爾平靜的呼吸聲。

**旁白**:在這與世隔絕的曦光之庭裡,李杰暫時忘卻了死亡、謎團與追殺,享受著這份來之不易的、奢侈的和平。

**旁白**:疲憊如潮水般湧來,他的眼皮越來越重。
