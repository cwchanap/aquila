# 第二十三幕：第十節車廂的對峙

**旁白**：聽到腳步聲,凌墨猛地轉過身。

**旁白**：當他看清來人的臉時,眼中閃過一絲驚訝。

**旁白**：他警戒地盯著李杰,身體微微繃緊。

**凌墨**：你是誰?

**旁白**：他的聲音低沉而冷靜,但帶著明顯的戒備。

**旁白**：李杰沒有立刻回答,而是快步走向車廂中央。

**旁白**：那把黑劍靜靜地躺在座位旁,散發著不祥的氣息。

**李杰**：別碰那把劍。

**旁白**：李杰的聲音很急促。

**旁白**：凌墨皺起眉頭,看了一眼那把劍,又看向李杰。

**凌墨**：你很可疑。

**旁白**：他往後退了一步,擺出防禦的姿態。

**旁白**：雖然沒有武器,但他的站姿透露出隨時準備戰鬥的警覺。

**凌墨**：我再問一次。你是誰?你知道些什麼?

**旁白**：李杰深吸一口氣,知道必須說服這個男人。

**李杰**：我叫李杰。我...是從未來回來的。

**旁白**：凌墨的眼神更加銳利。

**凌墨**：未來?

**李杰**：我知道接下來會發生什麼。我知道你來這個世界,是為了照顧恵美。

**旁白**：聽到恵美的名字,凌墨的表情微微變化。

**李杰**：我也知道,恵美打算從列車乘客中找一個人,作為祭壇的替代品。

**李杰**：用那個人來交換她的男友翔。翔現在被困在祭壇裡,作為獻祭者。

**旁白**：凌墨的眼睛眯了起來。

**凌墨**：你怎麼會知道這些?

**李杰**：因為我經歷過。我搭上這班車,目睹了所有慘劇。

**李杰**：然後我被送回了一週前。這是我第二次搭上這班車。

**旁白**：凌墨沉默了幾秒,審視著李杰的表情。

**李杰**：這次,我想救所有人。所以我需要你的合作。

**李杰**：那把黑劍...它能給人魔鬼般的力量,讓人變得幾乎無敵。

**李杰**：但它也會控制人的心智,讓人變得瘋狂。絕對不能讓任何人碰它。

**旁白**：凌墨看了一眼那把劍,眼中閃過一絲忌憚。

**旁白**：然後,他緩緩放下了防禦姿態。

**凌墨**：時間旅行...聽起來很荒謬。

**旁白**：他停頓了一下。

**凌墨**：但我在那個世界待了一年半,什麼奇怪的事都見過了。

**凌墨**：我姑且相信你說的話。

**旁白**：李杰鬆了一口氣。

**凌墨**：不過...我的目標很明確。

**凌墨**：保護恵美,然後幫她找到一個替代品,救出翔。

**旁白**：他的聲音變得冷漠。

**凌墨**：這意味著,至少要犧牲一個人。

**凌墨**：如果你堅持要救所有人,那我們無法合作。

**旁白**：李杰感到胸口一緊。

**旁白**：他張了張嘴,想要反駁,但不知道該說什麼。

**李杰**：(內心) 他說得沒錯...要救恵美的男友,就必須有人作為獻祭...

**李杰**：(內心) 但是...真的沒有其他辦法了嗎...

**旁白**：就在這時——

**旁白**：連接第九節和第十節車廂的門,突然被推開了。

**旁白**：一個中年男人站在門口,手中握著一把手槍。

**旁白**：槍口正對著李杰。

**旁白**：那張臉,李杰再熟悉不過——正是中村隼人。

**旁白**：中村隼人臉上掛著陰險的笑容。

**中村隼人**：真是精彩的對話啊。我全都聽到了。

**中村隼人**：謝謝你們提供的情報。特別是那把劍的事。

**旁白**：李杰和凌墨同時僵住了。

**李杰**：你——

**旁白**：話還沒說完——

**旁白**：槍聲在車廂內炸響。

**旁白**：李杰感到胸口一陣劇痛,像是被重錘狠狠砸中。

**旁白**：他低頭看去,鮮血正從胸口湧出,迅速染紅了襯衫。

**旁白**：耳邊傳來凌墨的喊聲,但已經變得模糊不清。

**旁白**：視線開始旋轉,天花板和地板在眼前顛倒。

**旁白**：然後,他感覺自己倒了下去。

**旁白**：冰冷的地板貼著臉頰。

**旁白**：意識正在遠去,就像潮水退去一般。

**旁白**：他努力想要保持清醒,但眼皮越來越重。

**旁白**：最後一個念頭閃過腦海——

**李杰**：(內心,微弱) 我...又失敗了嗎...

**旁白**：然後,一切陷入黑暗。
