# 第二十四幕：再次重來

**旁白**：不知道過了多久。

**旁白**：李杰的意識在黑暗中漂浮,然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧?我的手機完全沒訊號!連「圈外」都不是,是「沒有服務」耶!這底下是多深啊?我從來沒見過這種狀況...

**田中健太**：我的也是,什麼都傳不出去。LINE、IG、什麼都不行。喂,這隧道也太長了吧?以前有這麼長嗎?我怎麼一點印象都沒有?

**旁白**：李杰猛地睜開眼睛。

**旁白**：他坐在搖晃的車廂裡,窗外是純粹的黑暗。

**李杰**：(內心) 又...又回來了...

**旁白**：他低頭看向自己的胸口。

**旁白**：襯衫完好無損,沒有血跡,沒有傷口。

**旁白**：就像剛才的一切都沒有發生過。

**李杰**：(內心) 我被中村射中...然後失去意識...然後又回到了列車駛入隧道的時間點...

**李杰**：(內心) 這到底是怎麼回事?!

**旁白**：他的手緊緊握成拳頭。

**旁白**：但現在不是思考這個的時候。

**李杰**：(內心) 我還有機會...我必須改變結局!

**旁白**：他站起身,朝著車廂後方衝去。

**旁白**：穿過一節又一節車廂,腳步急促而堅定。

**旁白**：終於,他推開了第十節車廂的門。

**旁白**：凌墨正站在車廂中央,轉過頭看向他。

**旁白**：和之前一模一樣的場景。

**凌墨**：你是誰?

**旁白**：李杰沒有立刻回答。

**旁白**：他快速掃視車廂,然後走到連接第九節和第十節車廂的門旁邊。

**旁白**：站在那裡,背靠著牆壁。

**旁白**：這個位置,從第九節車廂的門口看不到。

**李杰**：別碰那把劍。

**旁白**：凌墨皺起眉頭,擺出防禦姿態。

**凌墨**：你很可疑。我再問一次。你是誰?你知道些什麼?

**李杰**：我叫李杰。我是從未來回來的。

**李杰**：我知道你來這個世界,是為了照顧恵美。

**李杰**：我也知道,恵美打算從列車乘客中找一個人,作為祭壇的替代品。

**李杰**：用那個人來交換她的男友翔。

**旁白**：凌墨的表情微微變化。

**凌墨**：你怎麼會知道這些?

**李杰**：因為我經歷過。這是我第二次搭上這班車。

**李杰**：這次,我想救所有人。所以我需要你的合作。

**李杰**：那把黑劍能給人魔鬼般的力量,但也會控制人的心智。絕對不能讓任何人碰它。

**旁白**：凌墨看了一眼那把劍,然後放下了防禦姿態。

**凌墨**：時間旅行...聽起來很荒謬。

**凌墨**：但我在那個世界待了一年半,什麼奇怪的事都見過了。

**凌墨**：我姑且相信你說的話。

**旁白**：他停頓了一下。

**凌墨**：不過...我的目標很明確。

**凌墨**：保護恵美,然後幫她找到一個替代品,救出翔。

**凌墨**：這意味著,至少要犧牲一個人。

**凌墨**：如果你堅持要救所有人,那我們無法合作。

**旁白**：李杰沒有回答。

**旁白**：他只是靜靜地站在那裡,目光若有似無地瞥向第九節車廂的門。

**旁白**：幾秒鐘後——

**旁白**：連接第九節和第十節車廂的門,被推開了。

**旁白**：中村隼人走了進來,手中握著手槍。

**旁白**：槍口對準了凌墨。

**中村隼人**：真是精彩的對話啊。我全都——

**旁白**：他的話突然停住了。

**旁白**：臉上露出困惑的表情。

**中村隼人**：(內心) 奇怪...我明明聽到有人在說話...但車廂裡只有這個男人?

**旁白**：就在他愣神的瞬間——

**旁白**：李杰從他身後撲了過去!

**旁白**：用盡全身力氣,狠狠撞向中村隼人的後背!

**中村隼人**：什麼——

**旁白**：中村隼人猝不及防,整個人失去平衡,重重摔倒在地。

**旁白**：手槍從手中飛了出去,在地板上滑行。

**旁白**：凌墨反應極快,瞬間衝上前,撿起了手槍。

**旁白**：槍口對準了倒在地上的中村隼人。

**旁白**：李杰騎在中村隼人身上,用力壓住他的雙臂。

**中村隼人**：你這混蛋!放開我!

**旁白**：他拚命掙扎,但李杰死死按住他,不讓他起身。

**李杰**：(喘著氣) 凌墨...我們可以用這個人...

**李杰**：用他作為祭壇的獻祭品。

**旁白**：凌墨眼中閃過一絲驚訝,然後露出了笑容。

**旁白**：那是一種如釋重負的笑容。

**凌墨**：好主意。

**凌墨**：你按住他,我去找繩子。

**旁白**：他轉身,快步走向車廂門口。

**凌墨**：我很快就回來。

**旁白**：說完,他推開門,離開了第十節車廂。

**旁白**：只剩下李杰和中村隼人。

**中村隼人**：你們這些混蛋...放開我!我要殺了你們!

**旁白**：他還在瘋狂掙扎,但李杰用盡全力壓制著他。

**李杰**：(內心) 總算...總算抓住他了...

**李杰**：(內心) 這次...一定要改變結局!

**旁白**：車廂在搖晃。窗外依然是純粹的黑暗。

**旁白**：而在這狹小的空間裡,命運的齒輪,正朝著不同的方向轉動。
