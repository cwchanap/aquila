# 第二十九幕：車廂內的對話

**旁白**：列車繼續在黑暗中行駛。車廂內只剩下李杰、山本浩二、倒地的健太,以及被綁在角落的中村隼人。

**旁白**：山本浩二靠在座位旁,沉默了許久。然後,他開口了。

**山本浩二**：李先生...我能問你幾個問題嗎?

**李杰**：請說。

**旁白**：李杰忍著腿上的疼痛,靠在對面的座位上。

**山本浩二**：你...真的是從未來回來的嗎?

**李杰**：是的。這是我第三次經歷這趟旅程。

**山本浩二**：那你...你是哪裡人?

**李杰**：我來自上海。在上海讀完大學後,就來東京工作了。已經在這裡五年了。

**山本浩二**：上海啊...聽說那裡的高樓真多,發展得很快。我沒去過,但在電視上看過。

**旁白**：他苦笑了一聲。

**山本浩二**：我啊,是八王子人。土生土長的。高中畢業後就開始當建築工人,到現在已經二十五年了。蓋過不少大樓,但從來沒想過自己會困在這種鬼地方。

**李杰**：你...有家人嗎?

**山本浩二**：有。我老婆是全職主婦,在家裡照顧我們的兒子。兒子今年剛從高中畢業,正在準備考大學。

**旁白**：他的聲音變得溫柔起來。

**山本浩二**：那小子啊,從小就很懂事。成績也不錯,想考早稻田。我和他媽一直在存錢,就是為了讓他能好好念書。

**旁白**：他停頓了一下,眼神變得有些黯淡。

**山本浩二**：我今天...本來是要回家吃晚飯的。老婆說要做我最愛吃的炸豬排...

**旁白**：他深深地嘆了一口氣。

**山本浩二**：你說...我還能再見到他們嗎?

**旁白**：李杰沉默了幾秒。

**李杰**：還有機會。

**山本浩二**：嗯?

**李杰**：列車會在下一站再次開門。到時候需要有兩個人下車。

**山本浩二**：兩個人...那我們...

**李杰**：如果健太在列車到站前還沒有醒來,我會留在車上陪著他。你和中村下車。

**山本浩二**：什麼?!不行!應該留下這個混蛋,讓你下車才對!

**李杰**：我的腿受了傷,走不了路。就算下車,我也很難逃脫。而且...

**旁白**：李杰看向倒在地上的健太。

**李杰**：萬一健太醒來...他會殺掉所有人。我必須留在這裡,確保有人能阻止他。

**山本浩二**：但是...!

**李杰**：你下車後,把中村拖出去。用他作為祭壇的獻祭品,和其他人一起回到地球。

**旁白**：山本浩二握緊拳頭,沉默了許久。

**山本浩二**：...我明白了。

**旁白**：車廂內再次陷入沉默。只有列車行駛的聲音,在黑暗中迴響。

**旁白**：過了一陣子——

**旁白**：被綁在角落的中村隼人發出了一聲低沉的呻吟。他緩緩睜開眼睛,茫然地看著周圍。

**中村隼人**：這...這是...

**旁白**：他的視線掃過車廂,看到了李杰、山本浩二,還有倒在地上的健太。

**中村隼人**：發生什麼事了?為什麼你們還在車上?其他人呢?

**旁白**：他試圖動彈,但發現自己被緊緊綁住了。手槍和刀具都已經被收走,放在遠處的座位上。

**中村隼人**：喂!你們聽到我說話了嗎?!為什麼要綁著我?!

**旁白**：李杰和山本浩二都沒有回應,只是靜靜地看著他。

**中村隼人**：該死...你們到底想幹什麼...

**旁白**：他掙扎了幾下,但USB線纏得很緊,根本掙脫不開。

**中村隼人**：聽著!你們放開我!我可以幫你們!真的!我知道怎麼離開這個鬼地方!

**中村隼人**：那邊那個年輕人...他怎麼了?為什麼倒在地上?

**旁白**：他看向健太,眼中閃過一絲好奇。

**中村隼人**：李杰,對吧?我記得你的名字。你知道發生什麼事了嗎?告訴我啊!

**旁白**：李杰依然沒有回應,只是冷冷地盯著中村。

**中村隼人**：別這樣啊!我們都是被困在這裡的可憐人!應該互相幫助才對!你們綁著我有什麼用?我又不會傷害你們!

**中村隼人**：我剛才...我剛才只是太害怕了!所以才拿出槍!我真的不是故意要傷害任何人!求你們了,放開我!

**旁白**：他的聲音越來越急切,帶著明顯的慌張。

**中村隼人**：我上有老下有小!我還有家人在等我!我不能死在這裡!求求你們...

**山本浩二**：閉嘴。

**旁白**：山本浩二冷冷地開口,聲音低沉而充滿厭惡。

**山本浩二**：你剛才拿槍指著我們的時候,可不是這副嘴臉。

**中村隼人**：我...我那是...!

**山本浩二**：我也有家人在等我。但我沒有拿槍威脅任何人。

**旁白**：他的眼神冰冷。

**山本浩二**：所以別在這裡裝可憐。你做了什麼,自己心裡清楚。

**旁白**：中村隼人的聲音漸漸變小,最終停了下來。他意識到,無論說什麼,這兩個人都不會理他。

**旁白**：車廂內再次陷入沉默。

**旁白**：列車繼續前行,駛向未知的終點。
