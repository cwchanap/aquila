# 第三十二幕：破碎的白劍

**旁白**：白劍與黑劍在空中相撞，爆發出耀眼的光芒！

**旁白**：高山翔與田中健太？同時被震退，雙雙落在月台上。

**旁白**：月台是黑色的大理石，表面光滑如鏡。下方是翠綠色的湖泊，散發著柔和的光芒。

**旁白**：兩人對視片刻，再次衝向彼此！

**旁白**：田中健太？的動作狂野而凌厲。黑劍在空中劃出一道道黑色的弧線，每一劍都帶著致命的殺意。

**旁白**：高山翔則相對保守。他的劍法更注重防禦，白劍在身前舞動，形成一道道光芒組成的屏障。

**旁白**：「鏗！鏗！鏗！」

**旁白**：黑劍與白劍不斷碰撞，金屬的撞擊聲在空曠的大廳中迴盪。每一次碰撞都激起強烈的能量波動。

**旁白**：田中健太？突然加快速度！他的身形化作一道黑影，從側面繞到高山翔身後！

**旁白**：黑劍筆直刺向高山翔的後背！

**旁白**：高山翔瞬間反應過來，身體向前翻滾，堪堪躲過這一擊。

**旁白**：黑劍刺空，在空氣中發出尖銳的破風聲。

**旁白**：高山翔落地後立刻轉身，白劍橫掃而出！

**旁白**：白色的劍光直奔田中健太？的腰部！

**旁白**：田中健太？舉起黑劍，擋住這一擊。

**旁白**：「鏗——！」

**旁白**：巨大的衝擊力讓兩人同時後退數步。

**田中健太？**：哈...這具身體...還真是不順手...

**旁白**：他歪著頭，血紅色的眼睛閃爍著不滿的光芒。

**田中健太？**：力量、速度、反應...都比我原本的身體差太多了...如果是我全盛時期的身體，你早就死了。

**旁白**：儘管如此，他的攻勢依然凌厲。

**旁白**：高山翔沒有回應，只是調整呼吸，擺好架勢。

**旁白**：李杰靠在車廂門邊，看著月台上的戰鬥。

**李杰**：(內心) 健太...不，那個古代戰士...明顯更有經驗...

**李杰**：(內心) 翔的劍法...看起來不太熟練...但還能勉強跟上...

**旁白**：月台上，兩人再次交鋒。

**旁白**：田中健太？這次選擇正面進攻。他舉起黑劍，從上往下猛烈劈砍！

**旁白**：高山翔舉起白劍，硬接這一擊！

**旁白**：「鏗——！」

**旁白**：強大的力量讓高山翔的雙腿微微彎曲，腳下的大理石地面出現了細微的裂痕。

**旁白**：田中健太？緊接著橫掃！黑劍帶著黑色的能量，直奔高山翔的頸部！

**旁白**：高山翔低頭閃避，黑劍從他頭頂掠過，削斷了幾縷黑髮。

**旁白**：高山翔趁機反擊，白劍刺向田中健太？的胸口！

**旁白**：田中健太？側身閃避，白劍擦過他的肩膀，在白袍上留下一道淺淺的劃痕。

**旁白**：兩人同時後退，再次拉開距離。

**旁白**：李杰看著他們的戰鬥，眉頭緊皺。

**李杰**：(內心) 這樣下去...不知道要打多久...我能做什麼...？

**旁白**：他的目光在車廂內掃視，突然落在不遠處座位旁的一把手槍上。

**旁白**：那是中村隼人的手槍。山本浩二當時只拿走了一把，還有一把留在這裡。

**李杰**：(內心) 對了...還有槍...！

**旁白**：他咬緊牙關，忍著腿上的劇痛，開始朝手槍的方向爬去。

**旁白**：每移動一步，傷口就撕裂般疼痛。冷汗不斷從額頭滑落，視線開始模糊。

**旁白**：但他還是堅持著，一點一點地爬向手槍。

**旁白**：月台上的戰鬥還在繼續。

**旁白**：田中健太？發動更加猛烈的攻勢！他的速度越來越快，黑劍化作無數道殘影！

**旁白**：高山翔被迫不斷後退，白劍勉強擋住一次又一次的攻擊。

**旁白**：李杰終於爬到座位旁，伸手抓住了手槍。冰冷的金屬觸感讓他稍微清醒了一些。

**旁白**：他撐起上半身，雙手握著手槍，瞄準月台上的田中健太？。

**李杰**：(內心) 可惡...他們移動得太快了...根本瞄不準...

**旁白**：田中健太？和高山翔的身影不斷交錯，速度快得幾乎只能看到殘影。

**旁白**：李杰的手指放在扳機上，但始終沒有扣下。

**李杰**：(內心) 如果射偏了...或者射到翔怎麼辦...

**李杰**：(內心) 要不...射其他地方...分散那傢伙的注意力...？

**旁白**：他的目光掃向月台的其他位置。

**李杰**：(內心) 不...槍聲可能也會分散翔的注意力...這樣反而會害了他...

**旁白**：他咬緊牙關，最終放下了手槍。

**李杰**：(內心) 該死...我什麼都做不了...

**旁白**：他只能繼續看著戰鬥，手緊緊握著槍，等待著機會。

**旁白**：月台上，高山翔的動作開始變慢。他的呼吸越來越急促，額頭冒出汗水。

**旁白**：田中健太？則越戰越勇。他的笑聲在大廳中迴盪。

**田中健太？**：怎麼了？撐不住了嗎？

**旁白**：他突然加速，黑劍如同黑色閃電般劈向高山翔！

**旁白**：高山翔舉起白劍抵擋——

**旁白**：「鏗——！」

**旁白**：這次的碰撞格外猛烈！白色與黑色的能量在空中爆發，形成耀眼的光芒！

**旁白**：高山翔的身體被震飛，重重砸在月台上。他艱難地撐起身體，白劍還緊握在手中。

**田中健太？**：結束了。

**旁白**：他舉起黑劍，黑色的能量在劍身上瘋狂湧動。

**旁白**：高山翔掙扎著想要站起來，但已經來不及了。

**旁白**：黑劍帶著毀滅性的力量，從上往下猛烈劈砍！

**旁白**：高山翔舉起白劍，做最後的抵抗——

**旁白**：「轟——！」

**旁白**：黑劍與白劍相撞的瞬間，白劍從中間斷裂！

**旁白**：「喀嚓——」

**旁白**：白色的劍身裂成兩半，碎片在空中飛散。

**旁白**：高山翔的身體開始變得透明。

**旁白**：從腳開始，他的身體逐漸化作光點，消散在空氣中。

**旁白**：他的眼神依然平靜，沒有恐懼，也沒有不甘。

**旁白**：僅僅幾秒鐘，高山翔就完全消失了。

**旁白**：只留下破碎的白劍，散落在黑色的月台上。

**李杰**：翔——！

**旁白**：李杰的喊聲在大廳中迴盪。他舉起手槍，瞄準田中健太？。

**旁白**：「砰！砰！砰！」

**旁白**：連續三聲槍響！

**旁白**：但田中健太？只是微微側身，子彈全部落空，打在遠處的石柱上。

**旁白**：他轉過頭，血紅色的眼睛看向李杰。

**旁白**：然後，他笑了。

**田中健太？**：就剩你一個了...不過，殺你太無聊了。

**旁白**：他轉身，朝著月台深處走去。黑劍拖在地上，發出刺耳的摩擦聲。

**田中健太？**：慢慢享受痛苦吧。你很快就會失血過多而死的。

**旁白**：他的身影逐漸消失在黑暗中。

**旁白**：李杰咬緊牙關，用盡全力爬出車廂。

**旁白**：他的身體重重摔在月台上，傷口再次撕裂，鮮血湧出。

**旁白**：就在他爬出車門的瞬間——

**旁白**：「嘶——」

**旁白**：車門關上了。

**旁白**：列車開始震動，緩緩啟動。

**旁白**：李杰回頭看了一眼，然後繼續朝著田中健太？離開的方向爬去。

**旁白**：列車駛離月台，消失在黑暗中。

**旁白**：月台上，只剩下李杰一個人。

**旁白**：他爬得越來越慢。視線越來越模糊。

**旁白**：腿上的傷口還在流血，在黑色的大理石地面上留下一道長長的血痕。

**旁白**：他的呼吸越來越微弱。

**旁白**：最後，他的手臂失去了力量。

**旁白**：整個人趴在冰冷的地面上。

**旁白**：意識逐漸模糊。

**旁白**：在失去意識前的最後一刻，他看到遠處銀白色的魔法火焰，在黑暗中靜靜燃燒。

**旁白**：然後，一切都陷入了黑暗。
