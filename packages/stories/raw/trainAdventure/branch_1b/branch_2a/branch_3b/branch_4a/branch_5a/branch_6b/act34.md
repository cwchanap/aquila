# 第三十四幕：情報與闖入者

**旁白**：李杰在中村隼人對面的座位坐下，目光緊緊盯著他。

**旁白**：凌墨已經離開，去告訴其他乘客不要接近第十節車廂。

**旁白**：車廂內只剩下李杰和被綑綁的中村。

**旁白**：沉默持續了一陣子。

**中村隼人**：喂...李杰，對吧？

**旁白**：中村試探性地開口。

**中村隼人**：我們...我們可以好好談談吧？你沒必要這樣對我。

**旁白**：李杰沒有回應，只是靜靜地看著他。

**中村隼人**：聽著，我知道你是時間旅行者。你一定經歷了很多，對這個世界也很了解，對吧？

**中村隼人**：那你應該明白...這個世界充滿機會！我們可以合作！

**旁白**：他的聲音變得急切。

**中村隼人**：我們可以一起成為這個世界的王！統治那些精靈！想想看，那些資源，那些力量...全都是我們的！

**旁白**：李杰依然沒有說話，只是面無表情地看著中村。

**中村隼人**：只要你放開我，我們就可以聯手！我有計畫！我知道怎麼獲得力量！

**旁白**：李杰的眼神微微變化。

**旁白**：他突然意識到，或許可以從中村口中套出一些情報。

**李杰**：你有計畫？

**旁白**：他終於開口，語氣平淡。

**李杰**：說來聽聽。如果可行，我可以考慮。

**旁白**：中村的眼睛瞬間亮了起來。

**中村隼人**：真的嗎？！太好了！

**旁白**：他激動地想要坐直身體，但被USB線緊緊束縛著。

**中村隼人**：聽我說！我原本的計畫是這樣的——

**旁白**：他壓低聲音，像是在分享什麼祕密。

**中村隼人**：這班列車會停在一個叫終焉祭壇的地方。那裡有一對劍，靜靜地放在祭壇上。

**中村隼人**：那對劍擁有強大的力量！只要拿到它們，就能變得無敵！

**中村隼人**：到時候，我們可以直接去征服精靈的村莊！那些精靈根本不是對手！

**旁白**：他的語氣越來越狂熱。

**中村隼人**：我已經做好準備了！帶了槍，帶了刀，就是為了這一刻！

**旁白**：他停頓了一下，眼神閃爍。

**中村隼人**：雖然...我不太明白...那把黑劍應該是那對劍中的一把...

**中村隼人**：但我不知道為什麼它會在這輛列車上，而不是在終焉祭壇...

**旁白**：他抬起頭，看向李杰。

**中村隼人**：不過這更好啊！我們可以先拿到黑劍，然後去終焉祭壇拿另一把！

**中村隼人**：兩把劍在手，我們就無敵了！整個世界都是我們的！

**旁白**：李杰盯著中村，眼神變得更加冷淡。

**李杰**：你怎麼知道這些？

**旁白**：他的聲音很平靜，但帶著壓迫感。

**李杰**：你來過這個世界嗎？還是...你也是時間旅行者？

**旁白**：中村的表情僵了一下。

**旁白**：然後，他露出了一個意味深長的笑容。

**中村隼人**：呵呵...這個嘛...

**旁白**：他沒有正面回答，只是冷笑著。

**中村隼人**：等我們一起統治這個世界後，我就告訴你。

**中村隼人**：現在，你只需要知道...我的計畫是可行的。

**旁白**：李杰沒有再追問。他靠回座位上，深深地嘆了一口氣。

**李杰**：(內心) 這些情報...大部分我都知道了...

**李杰**：(內心) 終焉祭壇、黑劍的力量、精靈村莊...

**李杰**：(內心) 唯一不知道的是...黑劍原本應該和白劍放在一起...

**李杰**：(內心) 但為什麼黑劍會在列車上？這到底是怎麼回事...

**旁白**：他陷入了沉思。

**旁白**：黑劍和白劍...宿敵...五萬年前的戰爭...

**旁白**：這些線索在腦海中不斷閃現，但始終無法串聯起來。

**旁白**：就在這時——

**旁白**：連接第一節車廂和第二節車廂的門，突然「砰」的一聲被推開！

**旁白**：一個身影衝了進來！

**旁白**：那是一個四十歲左右的男人，臉色通紅，表情焦躁不安。

**旁白**：是那個煩躁的丈夫。
