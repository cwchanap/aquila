# 第三十五幕：混亂

**旁白**：煩躁的丈夫快速掃視了一眼李杰和被綑綁的中村。

**旁白**：然後，他突然轉身，朝著下一節車廂衝去！

**李杰**：等等！

**旁白**：李杰立刻站起身，想要追上去。

**旁白**：但就在這時——

**旁白**：連接第一節車廂和第二節車廂的門再次被推開！

**旁白**：凌墨衝了進來！

**凌墨**：伊藤！

**旁白**：他大喊著那個男人的名字。

**旁白**：李杰和凌墨在車廂中央撞在一起！

**旁白**：兩人同時失去平衡，差點摔倒。

**李杰**：可惡...！

**旁白**：李杰穩住身形，轉頭看向凌墨。

**李杰**：到底怎麼回事？！

**旁白**：凌墨喘著氣，臉上滿是焦急。

**凌墨**：我剛才在第一節車廂...告訴所有人後面的車廂很危險，不要過去...

**凌墨**：那幾個大學生一直在問問題...我在回答他們的時候...

**凌墨**：伊藤突然站起來，什麼都沒說，就直接跑了！

**凌墨**：我根本不知道發生了什麼！

**旁白**：就在這時——

**旁白**：一陣大笑聲響起。

**中村隼人**：哈哈哈哈！

**旁白**：被綑綁的中村笑得前仰後合。

**中村隼人**：看來...那個伊藤...也聽到了黑劍的呼喚啊！

**中村隼人**：那把劍...它在呼喚著...呼喚著那些內心充滿慾望的人...

**旁白**：凌墨皺起眉頭。

**凌墨**：黑劍的呼喚？什麼意思？

**旁白**：但還沒等中村回答——

**旁白**：連接第一節車廂和第二節車廂的門，第三次被推開了。

**旁白**：一個女人走了進來。

**旁白**：她的臉色蒼白，眼神空洞，表情呆滯。

**旁白**：是那個焦慮的妻子——伊藤的妻子。

**凌墨**：妳...妳在做什麼？

**旁白**：凌墨困惑地看著她。

**旁白**：女人沒有回答。

**旁白**：她的視線落在座位旁的地板上——那裡放著幾把刀。

**旁白**：那是李杰之前從中村身上搜出來的武器。

**旁白**：她彎下腰，撿起其中一把折疊刀。

**旁白**：動作僵硬，像是被什麼東西操控著。

**凌墨**：喂！妳想幹什麼？！

**旁白**：凌墨意識到不對勁，立刻上前。

**旁白**：但已經來不及了。

**旁白**：女人突然轉身，舉起刀，朝著李杰刺去！

**旁白**：李杰完全沒有預料到這一擊。

**旁白**：他還在思考剛才中村說的話，注意力根本不在這個女人身上。

**旁白**：刀尖狠狠刺入他的腹部！

**李杰**：嗚——！

**旁白**：劇烈的疼痛瞬間襲來。李杰的臉色變得慘白。

**旁白**：凌墨立刻衝上前，一拳打在女人的臉上！

**旁白**：女人被擊倒，摔在地上，刀從手中飛出。

**旁白**：就在這時，又一個身影衝進車廂。

**旁白**：是山本浩二。

**山本浩二**：發生什麼事了？！

**旁白**：他看到倒在地上的女人和捂著腹部的李杰，立刻明白情況不妙。

**山本浩二**：我來幫忙！

**旁白**：他衝上前，和凌墨一起壓住還在掙扎的女人。

**旁白**：女人的眼神依然空洞，嘴裡發出低沉的呻吟。

**旁白**：她還在拚命想要站起來，彷彿有什麼東西驅使著她。

**凌墨**：李杰，你撐住！我馬上去找點能止血的東西——

**李杰**：等等...！

**旁白**：李杰用顫抖的聲音打斷了他。

**旁白**：他一隻手緊緊按著腹部的傷口，鮮血已經滲透了衣服。

**李杰**：先...先去阻止伊藤...！

**李杰**：他要去拿黑劍...絕對不能讓他拿到...！

**旁白**：凌墨愣了一下，隨即明白過來。

**旁白**：如果伊藤拿到黑劍，後果不堪設想。

**凌墨**：我明白了！

**旁白**：他轉身，朝著第三節車廂衝去。

**旁白**：山本浩二繼續壓制著那個女人。

**山本浩二**：你還好嗎？！

**旁白**：李杰沒有回答。他咬緊牙關，用盡全力按住傷口。

**旁白**：鮮血不斷從指縫間湧出。

**旁白**：視線開始模糊。

**旁白**：但他還是強撐著意識，不讓自己昏過去。

**李杰**：(內心) 不能...不能昏過去...

**李杰**：(內心) 伊藤...絕對不能讓他拿到黑劍...

**旁白**：就在這時，連接第一節車廂的門再次打開。

**旁白**：田中健太和佐藤拓海衝了進來。

**旁白**：他們看到李杰捂著腹部，鮮血不斷湧出的模樣，臉色瞬間變得慘白。

**田中健太**：這...這是怎麼回事？！

**佐藤拓海**：他流血了！我們快去找能幫忙的東西！

**旁白**：兩人沒有多說，立刻轉身衝回第一節車廂。

**旁白**：車廂內，中村的笑聲還在迴盪。

**中村隼人**：哈哈哈...太精彩了...真是太精彩了...

**旁白**：他的聲音在李杰的耳邊越來越遠。

**旁白**：一切都變得模糊不清。
