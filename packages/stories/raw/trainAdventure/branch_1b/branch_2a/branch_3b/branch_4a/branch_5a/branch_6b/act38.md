# 第三十八幕：記憶的碎片

**旁白**：不知道過了多久。

**旁白**：李杰的意識在黑暗中漂浮，然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧？我的手機完全沒訊號！連「圈外」都不是，是「沒有服務」耶！這底下是多深啊？我從來沒見過這種狀況...

**田中健太**：我的也是，什麼都傳不出去。LINE、IG、什麼都不行。喂，這隧道也太長了吧？以前有這麼長嗎？我怎麼一點印象都沒有？

**旁白**：李杰睜開眼睛。

**旁白**：他坐在搖晃的車廂裡，窗外是純粹的黑暗。

**李杰**：(內心，疲憊) 又...又回來了...

**旁白**：他低頭看向自己的胸口。

**旁白**：衣服完好無損，沒有血跡，沒有傷口。

**旁白**：就像剛才被黑劍刺穿心臟、失血、死亡...所有的痛苦都沒有發生過。

**李杰**：(內心) 這已經是第幾次了...第四次？還是第五次？

**旁白**：他深吸一口氣，用手按了按太陽穴。

**旁白**：身體沒有受傷，但精神上的疲憊卻越來越沉重。一次又一次的失敗，一次又一次的死亡...

**李杰**：(內心) 我看著中村被殺...看著奈奈被殺...看著恵美被殺...

**李杰**：(內心) 我看著凌墨、山本、雄大消失...他們可能都已經死了...

**李杰**：(內心) 每一次...每一次都失敗...

**旁白**：他的雙手在微微顫抖。不是因為恐懼，而是因為極度的疲憊和絕望。

**旁白**：無止境的輪迴。無止境的失敗。無止境的死亡。

**李杰**：(內心) 我...還能撐多久...

**旁白**：他咬緊牙關，強迫自己站起身。

**旁白**：雖然精神已經瀕臨崩潰，但他還是朝著車廂後方走去。

**旁白**：因為他知道，如果他不做點什麼，所有人都會死。

**旁白**：穿過一節又一節車廂，腳步沉重而疲憊。

**旁白**：第一節車廂...第二節車廂...

**旁白**：當他走進第三節車廂時——

**旁白**：他看到了一個身影。

**旁白**：一個穿著黑色衣服的年輕男人，正從第四節車廂走回來。

**旁白**：李杰停下腳步。

**李杰**：(內心) 那是...凌墨？

**李杰**：(內心) 他怎麼會在這裡？他應該在後面的車廂...

**旁白**：凌墨也看到了李杰，停下腳步。

**旁白**：兩人對視了幾秒鐘。

**李杰**：你是...？

**凌墨**：我叫凌墨。

**旁白**：凌墨的表情有些困惑，但眼神中閃爍著某種...認知。

**李杰**：你在做什麼？

**凌墨**：我...我剛才在後面的車廂搜索...

**凌墨**：然後...突然...

**旁白**：他皺起眉頭，像是在回想什麼。

**凌墨**：我腦海中出現了一些模糊的記憶...

**李杰**：記憶？

**凌墨**：對...很奇怪的記憶...

**凌墨**：我記得...有人告訴我...後面車廂有一把黑色的劍...

**凌墨**：那把劍很危險...會控制人的心智...

**凌墨**：還有...一個中年男人...他想要那把劍...他很危險...

**凌墨**：但我不記得是誰告訴我的...也不記得什麼時候聽到的...

**凌墨**：就像...就像我經歷過這一切，但又沒有經歷過...

**旁白**：李杰瞪大眼睛，震驚地看著凌墨。

**李杰**：你...你記起來了一些事情...

**凌墨**：記起來？我不確定...這些記憶很模糊...像是夢一樣...

**凌墨**：但我有種感覺...這些都是真的...

**凌墨**：所以我想回來...找出是怎麼回事...

**旁白**：李杰沉默了片刻。

**旁白**：這是前所未有的情況。

**旁白**：凌墨...居然記起了上一次循環的片段。

**李杰**：你說得對。後面第十節車廂確實有一把黑劍。

**李杰**：它是邪惡的，會控制人的心智，讓人變得瘋狂。

**李杰**：而你記憶中的那個中年男人...叫中村隼人。

**李杰**：他想要那把劍的力量。如果他拿到了，會殺死很多人。

**凌墨**：中村隼人...

**旁白**：凌墨皺著眉頭，像是在努力回想。

**凌墨**：對...就是這個名字...我記得這個名字...

**旁白**：就在這時——

**旁白**：連接第二節車廂和第三節車廂的門，突然被推開。

**旁白**：中村隼人走了進來。

**旁白**：他的視線完全集中在李杰身上，沒有注意到旁邊的凌墨。

**旁白**：他的臉上掛著得意的笑容。

**中村隼人**：謝謝你提供的情報啊。

**中村隼人**：時間旅行者？黑劍？第十節車廂？聽起來真是太棒了。

**中村隼人**：我現在就去拿那把劍。

**旁白**：他的右手伸進口袋，掏出一把手槍。

**旁白**：槍口對準李杰。

**李杰**：可惡...！

**李杰**：(內心) 又被偷聽到了...！

**旁白**：凌墨站在一旁，悄悄地移動著腳步，準備行動。

**旁白**：中村完全沒有察覺，所有注意力都在李杰身上。

**中村隼人**：再見了，時間旅行者——

**旁白**：就在這時——

**旁白**：凌墨動了！

**旁白**：他的身形化作一道黑影，從側面瞬間衝到中村面前！

**旁白**：「砰——」一腳踢在中村的手腕上！

**中村隼人**：啊——！

**旁白**：手槍飛了出去，掉在地上。

**旁白**：凌墨順勢一拳打在中村的臉上，中村整個人向後倒去，重重摔在地上。

**李杰**：凌墨！

**旁白**：李杰立刻衝上前，壓住掙扎的中村。

**旁白**：他快速搜索中村的口袋。

**旁白**：從外套內側口袋，掏出一把手槍。

**旁白**：從褲子右側口袋，掏出一把折疊刀。

**旁白**：從褲子左側口袋，又掏出一把折疊刀。

**旁白**：從後腰，掏出最後一把手槍。

**旁白**：李杰將所有武器扔到遠處。

**中村隼人**：你...你怎麼知道...

**旁白**：中村的臉色變得蒼白，眼神中滿是震驚。

**中村隼人**：你怎麼知道我把武器藏在哪裡？！

**中村隼人**：不公平！你們兩個都是時間旅行者！兩個打一個！

**中村隼人**：這根本不公平——！

**旁白**：李杰沒有理會他的咒罵。

**李杰**：凌墨，去前面車廂找其他人，跟他們要幾條USB充電線來綁他。

**凌墨**：好。

**旁白**：凌墨轉身，快速朝第一節車廂走去。

**旁白**：過了幾分鐘，他回來了，手中拿著好幾條長長的USB線。

**凌墨**：前面的女生給的。

**旁白**：他和李杰一起將中村牢牢綁住。

**旁白**：中村還在咒罵，但已經無法動彈。

**旁白**：李杰和凌墨站起身，將中村放在角落的座位上。

**李杰**：凌墨...謝謝你。

**凌墨**：不用謝。雖然我不太明白發生了什麼...但那些記憶告訴我，我必須阻止他。

**旁白**：李杰點了點頭。

**李杰**：現在...我們需要好好討論一下如何處理黑劍的問題。

**李杰**：那把劍...比我們想像的還要危險。

**凌墨**：我知道。在那些模糊的記憶中...我看到了很多人因為那把劍而死。

**凌墨**：我們該怎麼辦？

**旁白**：李杰沉思片刻。

**旁白**：這一次，凌墨記起了一些事情。

**旁白**：這是前所未有的變化。

**旁白**：也許...這次會不一樣。

**旁白**：也許...這次他們能成功。
