# 第三十九幕：古劍的詛咒

**李杰**：那把黑劍...不只是一把武器。

**李杰**：它被一個古代戰士的靈魂附身。那個靈魂會誘惑人去拿起劍，然後控制他們的心智。

**凌墨**：控制心智...

**旁白**：凌墨皺起眉頭，像是在努力回想那些模糊的記憶。

**李杰**：在我過去經歷的時間循環中...我親眼看到健太、伊藤誠被那把劍控制。

**李杰**：拓海也差點被劍誘惑，想要去拿它。

**李杰**：他們拿起劍之後，眼睛會燃燒著紅色的火焰，完全失去理智，變成殺戮機器。

**凌墨**：...

**旁白**：李杰沉默了片刻，腦海中浮現出更早之前的記憶。

**李杰**：(內心) 第一次時間循環...我在森林裡發現了小林雄大和明日香的屍體...

**李杰**：(內心) 雄大被劍刺穿心臟，一擊致命。明日香的胸口也有劍傷...

**李杰**：(內心) 當時我以為是第三方...但現在想想...

**旁白**：他的目光落在凌墨身上。

**李杰**：(內心) 凌墨...你在第一次循環裡，是不是也拿過那把劍？

**李杰**：(內心) 是你殺了雄大和明日香？還有小林智子和孩子們的失踪...也和你有關？

**旁白**：但他沒有說出口。

**旁白**：因為現在的凌墨，對那些事情一無所知。

**李杰**：(內心) 如果我告訴他...他會怎麼想？他會相信嗎？

**李杰**：(內心) 而且...告訴他這些又有什麼意義？他對那段記憶一片空白，只會讓他陷入不必要的痛苦和混亂。

**旁白**：李杰搖了搖頭，將這些念頭壓下。

**旁白**：現在不是追究過去的時候。

**旁白**：他又想起了另一個循環——中村隼人拿起黑劍的那一次。

**李杰**：(內心) 中村...他在第二次循環拿到了黑劍...

**李杰**：(內心) 他說他想征服這個世界，想成為這個世界的王...

**旁白**：那個畫面歷歷在目。中村躺在血泊中，失去了下半身，但依然瘋狂地笑著，說著要征服世界。

**李杰**：(內心) 當時我以為他只是瘋了...被劍控制了...

**李杰**：(內心) 但現在回想起來...中村拿到劍之後的表現，和最近幾次循環裡他的行為...完全一致。

**李杰**：(內心) 他偷聽對話、想要劍的力量、想要殺人...這些都是他自己的意志。

**李杰**：(內心) 而健太、伊藤拿到劍之後，眼睛燃燒著紅光，完全失去理智...那才是被劍控制的樣子。

**李杰**：(內心) 為什麼？為什麼中村拿到劍後沒有被控制？

**李杰**：(內心) 難道...中村本身就足夠邪惡，所以劍無法控制他？

**李杰**：(內心) 還是說...劍選擇了不同的宿主？

**旁白**：太多的謎團，太多無法解釋的矛盾。

**旁白**：李杰揉了揉太陽穴，強迫自己不再糾結於過去。

**李杰**：凌墨，那把劍很可能會再次控制其他乘客。

**李杰**：而且...不一定會是同一個人。它可能會選擇不同的目標。

**凌墨**：那我們該怎麼辦？

**李杰**：我們需要其他人的協助。尤其是強壯的成年男性，像山本浩二和小林雄大。

**李杰**：他們有力量，也比較冷靜。如果有人被劍控制，我們需要他們幫忙制伏。

**凌墨**：明白了。我們現在去第一節車廂找他們？

**李杰**：對。而且我們還需要告訴所有人，後面的車廂很危險，絕對不能過去。

**旁白**：凌墨點了點頭。

**旁白**：兩人轉身，朝著第一節車廂走去。

**旁白**：他們穿過第二節車廂，來到連接第一節和第二節車廂的門前。

**旁白**：李杰推開門——

**旁白**：下一瞬間——

**旁白**：一雙有力的手突然從側面伸出，死死抓住李杰的肩膀！

**李杰**：什麼——！

**旁白**：李杰還沒反應過來，另一雙手已經扣住了他的手臂，將他整個人壓向座位！

**旁白**：「砰——」他的臉被按在座位上，完全動彈不得。

**凌墨**：李杰！

**旁白**：凌墨剛想衝上前——

**旁白**：一個高大的身影從側面出現，雙手迅速扣住了凌墨的手臂！

**凌墨**：什麼——！

**旁白**：山本浩二用專業的手法，瞬間將凌墨的雙手反扣在背後，然後將他壓向另一側的座位。

**山本浩二**：對不起，年輕人。在我們搞清楚情況之前，請你配合一下。

**旁白**：山本的語氣雖然堅定，但並不粗暴，反而帶著一絲歉意。

**旁白**：凌墨掙扎了幾下，但山本的力量和技巧遠超他的想像，完全無法掙脫。

**李杰**：等等——你們誤會了——！

**小林雄大**：先別動，聽我們說。

**旁白**：雄大的聲音沉穩，但也帶著警惕。

**小林雄大**：我不知道你們到底在做什麼，但你們的行為太可疑了。

**小林雄大**：為了我的家人...為了不讓他們有任何卷入危險的可能...我必須這麼做。

**旁白**：李杰被壓在座位上，側過臉看向第一節車廂裡的其他人。

**旁白**：健太、拓海、明日香、美咲四個大學生站在一起，臉上寫滿懷疑和警惕。

**旁白**：小林智子護著陽菜和翔太，眼神戒備地看著李杰和凌墨。

**旁白**：伊藤誠和伊藤奈奈站在一旁，交頭接耳地竊竊私語。

**旁白**：一個戴眼鏡的年輕男人，從後面緩緩走了出來。

**旁白**：他推了推眼鏡，臉上帶著謹慎的表情，一步一步走向被制伏的李杰和凌墨。

**眼鏡男**：山本先生、小林先生，做得好。

**眼鏡男**：非常感謝你們制伏了這兩個人。

**旁白**：他在李杰和凌墨面前停下，居高臨下地看著他們。

**眼鏡男**：我叫斎藤蓮。雖然我們還沒正式認識，但在這種情況下...

**斎藤蓮**：現在...我們可以好好審問一下，你們到底在搞什麼鬼了。

**旁白**：他的聲音帶著一絲得意，像是終於抓住了什麼把柄。

**山本浩二**：你們兩個的行為，確實很可疑。

**山本浩二**：綁架乘客，把他拖到後面車廂...這可不是正常人會做的事。

**山本浩二**：雖然很抱歉要用這種方式，但在弄清楚之前，我們必須這麼做。

**旁白**：李杰深吸一口氣，努力保持冷靜。

**李杰**：我們沒有綁架任何人！我們制伏的是中村隼人——那個自私的中年男！

**李杰**：他偷聽了我們的對話，想要去拿一把危險的武器！我們必須阻止他！

**田中健太**：危險的武器？什麼武器？

**旁白**：健太的臉上滿是懷疑。

**佐藤拓海**：而且...你們怎麼知道後面有什麼東西？

**旁白**：拓海皺著眉頭，目光審視著李杰。

**旁白**：所有人的視線都集中在李杰和凌墨身上。

**旁白**：懷疑、警惕、不信任...

**旁白**：氣氛緊張到了極點。
