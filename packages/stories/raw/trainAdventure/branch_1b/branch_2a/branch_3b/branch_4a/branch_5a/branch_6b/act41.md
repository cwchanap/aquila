**旁白**：就在這時——

**李杰**：(內心) 嗯？

**旁白**：那是凌墨的手指。

**旁白**：凌墨面無表情，但他的雙手...已經從USB線中掙脫了出來！

**李杰**：(內心) 他掙脫了...！

**旁白**：他試著用力扭動手腕，前後擺動。

**旁白**：USB線發出細微的聲音——

**旁白**：「啪」——

**旁白**：斷了。

**李杰**：(內心) 斷了...線斷了...！

**李杰**：(內心) 恵美...是你故意拿了不夠牢固的線嗎...？

**旁白**：但李杰沒有立刻起身。

**旁白**：他和凌墨依然保持著被綁的姿勢，雙手藏在背後。

**旁白**：因為山本和雄大就站在不遠處，目光警惕地看著他們。

**李杰**：(內心) 現在還不能動...山本先生和小林先生會立刻制伏我們...

**旁白**：就在這時——

**旁白**：恵美突然站了起來，聲音中帶著一絲顫抖。

**清水恵美**：那個...山本先生、小林先生...

**山本浩二**：怎麼了，同學？

**清水恵美**：我...我有點害怕...

**清水恵美**：這兩個人...他們可能真的很危險...我不想待在這裡...

**旁白**：她的聲音聽起來很真實，像是真的被嚇到了。

**清水恵美**：可以...可以把我們女生和孩子帶到第二節車廂嗎？

**清水恵美**：至少...至少讓我們遠離這兩個人...

**小林雄大**：這...

**旁白**：雄大看了看自己的妻子和兩個孩子。

**旁白**：小林智子緊緊抱著陽菜和翔太，臉上也寫滿擔憂。

**小林雄大**：你說得對。為了安全起見，我們應該把女性和孩子帶到第二節車廂。

**山本浩二**：嗯，這樣比較好。

**小林雄大**：智子、陽菜、翔太，跟我來。

**小林智子**：好...好的。

**旁白**：恵美也跟了上去。

**伊藤奈奈**：誠，我們也去吧...

**伊藤誠**：...好吧。

**旁白**：伊藤夫婦也跟著離開了。

**旁白**：第一節車廂裡，現在只剩下山本浩二，還有被綁著的李杰和凌墨。

**旁白**：山本站在車廂中央，目光警惕地看著兩人。

**李杰**：(內心) 還是不行...山本先生還在...

**旁白**：「嗚——」

**山本浩二**：嗯？列車在減速...？

**旁白**：列車緩緩停了下來。

**旁白**：「咔嚓——」

**旁白**：山本瞪大眼睛。

**山本浩二**：車門...開了？

**山本浩二**：我得去看看其他人的情況...

**李杰**：(內心) 機會來了...！

**旁白**：他們快步走向車門。

**凌墨**：嗯。

**旁白**：第四節...

**李杰**：(內心) 拜託...拜託來得及...！

**旁白**：凌墨也跟著停下。

**凌墨**：我們...來晚了...
