# 第四十三幕：無動於衷

**凌墨**：李杰...我們能不能去其他車廂談一談？

**旁白**：凌墨壓低聲音，眼神中帶著一絲擔憂。李杰搖了搖頭，眼神依然空洞。

**李杰**：我們...永遠無法打敗那把劍...

**旁白**：他的聲音空洞而虛弱，像是從遙遠的地方傳來。

**凌墨**：你在說什麼...？

**李杰**：沒用的...不管怎麼做...都是一樣的結果...

**旁白**：凌墨皺起眉頭，臉上的表情從擔憂轉為憤怒。

**凌墨**：你給我清醒一點！

**旁白**：「啪——」一記響亮的耳光打在李杰的臉上。車廂裡的其他人都嚇了一跳，紛紛轉頭看向他們。

**田中健太**：欸...！

**鈴木明日香**：怎麼了...？

**旁白**：但李杰只是微微偏了偏頭，臉上沒有任何表情，眼神依然空洞。他甚至沒有抬手摸一下臉頰。

**凌墨**：該死...！

**旁白**：凌墨咬緊牙關，又是一記耳光。「啪——」李杰的頭被打得偏向一邊，但他依然沒有任何反應。

**凌墨**：你聽到我說話了嗎！

**旁白**：「啪——」又是一記耳光，然後是一拳，打在李杰的肩膀上。車廂裡的氣氛變得越來越緊張。

**佐藤拓海**：那個...他們在做什麼...？

**高橋美咲**：好像...在打架...？

**旁白**：小林翔太被這突如其來的暴力嚇到了，眼淚在眼眶裡打轉。

**小林翔太**：嗚...媽媽...

**旁白**：他開始哭了起來。智子趕緊抱住兒子，輕聲安撫他。

**小林智子**：沒事沒事，翔太，沒事的...

**小林陽菜**：媽媽...那個哥哥為什麼要打人...？

**小林智子**：別看，陽菜...

**旁白**：山本浩二和小林雄大站起身來，快步走向凌墨和李杰。

**山本浩二**：住手！你在做什麼！

**小林雄大**：在火車上打人是怎麼回事！

**山本浩二**：你嚇到孩子了！

**旁白**：凌墨停下手，喘著氣，看著自己的雙手。他的手微微發抖。

**凌墨**：我...

**旁白**：他環顧四周，看到所有人驚恐和不安的眼神。翔太還在哭，智子緊緊抱著他。凌墨深吸一口氣，低下頭。

**凌墨**：對不起。我失控了。

**旁白**：他的聲音平靜，但帶著一絲懊悔。

**山本浩二**：你到底想做什麼？

**小林雄大**：有什麼問題可以好好說，為什麼要動手？

**旁白**：凌墨沒有回答，他轉頭看向李杰。李杰依然坐在那裡，眼神空洞，臉上沒有任何表情。即使被打了好幾下，他依然沒有任何反應，像是一具沒有靈魂的軀殼。

**李杰**：(內心) ...我不想再想了...不想再想怎麼解決這一切...反正都是一樣的...

**旁白**：凌墨咬緊牙關，轉向李杰。

**凌墨**：李杰...聽我說...我不知道你經歷了什麼...我只記得...很模糊的記憶...有人拿起了黑劍...殺死了幾個人...這些...都是你告訴我的...

**旁白**：李杰依然沒有反應。

**凌墨**：因為我沒有具體的記憶...所以我需要你的幫助...我需要你告訴我該怎麼做...我們該怎麼解決這個問題...

**旁白**：但李杰只是靜靜地坐在那裡，像是沒有聽到凌墨的話。

**山本浩二**：黑劍...？你們在說什麼...？

**小林雄大**：殺人...？什麼意思...？

**旁白**：山本和雄大臉上寫滿困惑。就在這時——凌墨突然抬起頭，環顧車廂。

**凌墨**：等等...中村呢？

**旁白**：他的聲音突然變得緊張。

**凌墨**：中村隼人在哪裡？

**旁白**：所有人都愣了一下，然後開始環顧四周。

**田中健太**：中村...？那個鬼鬼祟祟的大叔嗎...？

**鈴木明日香**：咦...他不見了...

**佐藤拓海**：剛才...他還在這裡的吧...？

**高橋美咲**：什麼時候走的...？

**旁白**：凌墨的臉色驟然變得蒼白。

**凌墨**：該死...！

**旁白**：他猛地轉身，看向車廂後方的連接門。

**凌墨**：他趁我們在這裡的時候...！

**旁白**：凌墨沒有再說什麼，他轉身朝著車廂後方衝去。

**山本浩二**：喂！你要去哪裡！

**凌墨**：我得去阻止他！

**旁白**：凌墨的身影消失在連接門後。車廂裡的其他人面面相覷，不知道發生了什麼事。

**小林雄大**：這...到底是怎麼回事...？

**山本浩二**：我完全搞不懂...

**旁白**：而李杰——依然坐在原位，眼神空洞，沒有任何反應。即使知道凌墨衝去阻止中村，即使知道危險可能再次降臨——他依然無動於衷。

**李杰**：(內心) ...隨便了...反正...都一樣...

**旁白**：他閉上眼睛，深深地嘆了口氣。絕望的黑暗，完全吞噬了他。
