# 第四十八幕：最後一搏

**旁白**：不知道過了多久。

**旁白**：李杰的意識在黑暗中漂浮，然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧?我的手機完全沒訊號!連「圈外」都不是,是「沒有服務」耶!這底下是多深啊?我從來沒見過這種狀況...

**田中健太**：我的也是,什麼都傳不出去。LINE、IG、什麼都不行。喂,這隧道也太長了吧?以前有這麼長嗎?我怎麼一點印象都沒有?

**旁白**：李杰睜開眼睛。

**旁白**：他坐在搖晃的車廂裡,窗外是純粹的黑暗。

**李杰**：(內心) 又回來了...

**旁白**：他低頭看向自己的身體。沒有傷口,沒有血跡。一切都恢復了。

**李杰**：(內心) 這是第六次了...

**旁白**：他深吸一口氣,用手按了按太陽穴。精神上的疲憊無法消除,但他必須繼續。

**李杰**：(內心) 中村說過...只要意志力夠強...就能駕馭黑劍...

**李杰**：(內心) 他做得到...那我呢...？

**旁白**：他咬緊牙關,思考著這個可能性。

**李杰**：(內心) 我從來沒有試過...拿起那把劍...

**李杰**：(內心) 如果我能控制它...也許就能結束這一切...

**旁白**：他站起身,朝著車廂後方走去。

**旁白**：穿過一節又一節車廂,腳步堅定。

**旁白**：終於,他來到第一節車廂。

**旁白**：就在這時,凌墨從第二節車廂走了進來。

**旁白**：凌墨看到李杰,停下腳步。他的表情有些困惑,但眼神中閃爍著某種認知。

**凌墨**：你是...

**旁白**：他皺起眉頭,像是在努力回想什麼。

**李杰**：凌墨,我需要你幫個忙。

**旁白**：凌墨的眼睛微微睜大。

**凌墨**：你...你怎麼知道我的名字？

**旁白**：凌墨看著李杰,眼神變得更加困惑。

**凌墨**：而且...我腦海中出現了一些模糊的記憶...

**凌墨**：我記得...有人告訴我...後面車廂有一把黑色的劍...

**凌墨**：那把劍很危險...會控制人的心智...

**凌墨**：還有...一個中年男人...他想要那把劍...他很危險...

**凌墨**：但我不記得是誰告訴我的...也不記得什麼時候聽到的...

**旁白**：他停頓了一下,看著李杰。

**凌墨**：就像...就像我經歷過這一切,但又沒有經歷過...

**凌墨**：是你...是你告訴我的嗎？

**旁白**：李杰點了點頭。

**李杰**：你說得對。我們見過。那把黑劍在第十節車廂。

**李杰**：而你記憶中的那個中年男人...叫中村隼人。

**凌墨**：中村隼人...

**旁白**：凌墨皺著眉頭,像是在努力回想。

**凌墨**：對...就是這個名字...我記得這個名字...

**李杰**：我需要你留在這裡,盯著他。不要讓他去後面的車廂。

**旁白**：凌墨的眼神充滿懷疑。

**凌墨**：等等...你到底在說什麼？你打算做什麼？

**旁白**：李杰沒有回答他的問題。他只是認真地看著凌墨。

**李杰**：相信我。我要去結束這場混亂。

**旁白**：說完,他轉身朝著車廂後方走去。

**凌墨**：喂——！

**旁白**：凌墨想叫住他,但李杰已經推開車門,走向第二節車廂。

**旁白**：凌墨站在原地,臉上滿是困惑。

**旁白**：李杰穿過一節又一節車廂。第二節,第三節,第四節...

**旁白**：他的心跳越來越快。

**李杰**：(內心) 我不確定...我不知道自己能不能控制那把劍...

**李杰**：(內心) 但我必須試試...這似乎是唯一的辦法了...

**李杰**：(內心) 只要意志力夠強...只要我能保持清醒...應該就能做到...

**旁白**：他咬緊牙關,繼續前進。

**旁白**：第五節,第六節,第七節...

**旁白**：終於,他推開了第十節車廂的門。

**旁白**：車廂裡空無一人。

**旁白**：在車廂的中央,走道上,那把黑劍靜靜地躺在地上。

**旁白**：劍身散發著微弱的黑色光芒,不祥的氣息在空氣中瀰漫。

**旁白**：李杰站在車廂入口,看著那把劍。

**旁白**：他的手心開始冒汗。

**李杰**：(內心) 就是這把劍...

**李杰**：(內心) 它會控制拿起它的人...讓人變得瘋狂...

**李杰**：(內心) 但如果我能控制它...

**旁白**：他深吸一口氣,邁開步伐。

**旁白**：一步,兩步,三步...

**旁白**：他走到黑劍旁邊,停了下來。

**旁白**：低頭看著那詭異的黑色光芒。

**李杰**：(內心) 我能做到...我一定能做到...

**旁白**：他蹲下身,伸出右手。

**旁白**：手指顫抖著,慢慢接近劍柄。

**旁白**：就在手指觸碰到冰冷的劍柄的瞬間——

**旁白**：整個世界轟然崩塌。

**旁白**：周圍的車廂、燈光、窗外的黑暗,全部在瞬間化為純粹的、絕對的黑。

**旁白**：那不是普通的黑暗,而是一種能吞噬一切的虛無。

**旁白**：李杰的意識漂浮在虛空中,什麼都看不見,什麼都聽不見。

**旁白**：然後——

**旁白**：一個聲音響起。

**？？？**：哈哈哈...哈哈哈哈...

**旁白**：那聲音低沉、沙啞,帶著嘲諷和得意。

**？？？**：你終於...拿起我了...

**旁白**：李杰想說話,但發不出聲音。他的意識在虛空中掙扎。

**？？？**：我等了...好久...好久...

**？？？**：那個膽小鬼...那個自以為是的傢伙...他們都拿起我了...

**？？？**：但你...你一直在躲...一直不敢碰我...

**旁白**：那聲音越來越響,在李杰的腦海中迴盪。

**？？？**：你以為...你的意志力夠強...就能控制我？

**？？？**：哈哈哈...太天真了...

**？？？**：他們被我控制...是因為他們太弱...

**？？？**：但你...無論你多強...你都會被我控制...

**？？？**：因為...你是我們其中一員...

**？？？**：你和他們...不一樣...

**旁白**：劇烈的疼痛襲來。李杰感到大腦彷彿被撕裂。

**李杰**：啊啊啊——！

**旁白**：他想放開劍,但手指已經無法動彈。

**旁白**：那柄劍彷彿與他的手掌融為一體,無法分離。

**旁白**：視野開始旋轉,天地顛倒。意識如同被拉入深淵,墜落、墜落、不斷墜落...

**旁白**：最後,一切歸於死寂。

**旁白**：李杰失去了意識。
