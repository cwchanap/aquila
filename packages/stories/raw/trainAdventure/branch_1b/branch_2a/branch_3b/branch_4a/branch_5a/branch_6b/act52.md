# 第五十二幕：痛苦的凌墨

**旁白**：不知道過了多久。李杰的意識在黑暗中漂浮，然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧?我的手機完全沒訊號!連「圈外」都不是,是「沒有服務」耶!這底下是多深啊?我從來沒見過這種狀況...

**田中健太**：我的也是,什麼都傳不出去。LINE、IG、什麼都不行。喂,這隧道也太長了吧?以前有這麼長嗎?我怎麼一點印象都沒有?

**旁白**：李杰睜開眼睛。他坐在搖晃的車廂裡,窗外是純粹的黑暗。又回來了...這是第八次了。

**旁白**：他深吸一口氣,用手按了按太陽穴。上一次循環的記憶湧上心頭——凌墨被控制、紅色的眼睛、清水不該知道的名字、還有黑劍的話："你是我們其中一員"。

**李杰**：(內心) 冷靜...我需要冷靜下來...這次必須改變策略。必須找到能控制黑劍的人...否則大家都會死...

**旁白**：他環顧車廂。大學生們正在討論手機沒訊號的問題,斎藤蓮試圖解釋但被山本浩二反駁。伊藤夫婦在爭吵,小林家的孩子在哭泣,中村隼人焦躁地喊著要按緊急按鈕。

**李杰**：(內心) 山本大叔和小林先生...他們應該是最有可能的...

**旁白**：李杰站起身,走向山本浩二。此時山本剛結束與斎藤的爭論,沉穩地坐了下來。

**李杰**：大叔,不好意思。

**山本浩二**：(轉頭看向李杰) 嗯?怎麼了,小兄弟?

**李杰**：我有件很重要的事要跟你說。還有...(看向小林雄大)那位先生,能過來一下嗎?

**旁白**：小林雄大看了妻子一眼,然後走了過來。

**小林雄大**：發生什麼事了嗎?

**李杰**：(壓低聲音)我知道這聽起來很荒謬,但請相信我。這輛列車會帶我們去一個異世界。第十節車廂裡有一把黑色的詛咒之劍,會控制人變得瘋狂殺人。

**山本浩二**：(皺起眉頭)小兄弟,你在說什麼...

**小林雄大**：(警戒地看著李杰)你怎麼知道這些?而且...你憑什麼要我們相信你?

**李杰**：(看著小林雄大)小林先生,你們一家四口住在立川。你的妻子叫智子,女兒叫陽菜,兒子叫翔太。

**旁白**：小林雄大的臉色瞬間變了。

**小林雄大**：你...你怎麼會知道我們的名字？！

**李杰**：(看向山本浩二)你叫山本浩二,獨自一人搭車。你坐這條線三十年了。

**山本浩二**：(震驚地)你連我的名字都...

**李杰**：我沒時間解釋。但如果有意志力足夠強的人能控制那把劍,也許能阻止悲劇。否則...大家都會死。

**旁白**：兩人對視了一眼,震驚和懷疑交織,但李杰眼中的認真讓他們沒有立刻否定。

**山本浩二**：小兄弟...這不像是開玩笑的樣子...

**小林雄大**：你真的確定嗎?

**李杰**：(堅定地點頭)我確定。而且...(轉頭看向角落)那個剛才喊著要按緊急按鈕的人,你們要小心他。

**李杰**：還有,黑劍可能會控制其他乘客。我們必須保持警戒。(看向山本浩二)大叔,能跟我一起去第十節車廂嗎?

**山本浩二**：(沉默了幾秒,然後點頭)好。我相信你。

**小林雄大**：我留在這裡看著大家。如果有任何異狀,我會立刻警告。

**李杰**：謝謝。

**旁白**：李杰轉身,準備朝著後方車廂走去。但就在這時,一個念頭閃過他的腦海。

**李杰**：(內心) 等等...凌墨...他沒有來...

**旁白**：他環顧車廂。凌墨不在這裡。在之前的幾次循環,凌墨都有模糊的記憶,總是會從後面走到第一節車廂來。但這次...

**李杰**：(內心) 不好...！

**旁白**：一股不安的感覺在李杰心中升起。

**李杰**：大叔,我們得快點!

**山本浩二**：怎麼了?

**李杰**：我有不好的預感!

**旁白**：李杰立刻衝向車廂後方。山本浩二緊跟在後。他們快速穿過第二節、第三節、第四節...空蕩蕩的車廂在他們身後掠過。

**旁白**：第五節。第六節。第七節。就在他們到達第七節車廂時——

**旁白**：一聲淒厲的尖叫從下一節車廂傳來。

**？？？**：啊啊啊啊——！！！

**山本浩二**：這是...！

**李杰**：第八節車廂！

**旁白**：兩人立刻衝向第八節車廂的門。李杰推開車門——眼前的景象讓他愣住了。

**旁白**：凌墨躺在車廂地板上,身體劇烈地抽搐著。他雙手緊緊抱著頭,指甲深深地陷入頭皮,甚至滲出了血絲。他的臉扭曲成痛苦的表情,額頭上青筋暴起,汗水不斷滴落。

**凌墨**：啊啊啊——！！

**旁白**：他在地板上痛苦地翻滾,像是承受著極大的折磨。

**凌墨**：嗚——啊啊啊！！

**旁白**：他的聲音嘶啞而絕望,整個身體不停地顫抖。

**山本浩二**：(震驚地)這...這是怎麼回事？！

**李杰**：凌墨！

**旁白**：李杰立刻衝上前,跪在凌墨身邊。

**李杰**：凌墨！你怎麼了？！

**旁白**：凌墨的眼睛緊緊閉著,眼皮下的眼球在快速轉動,像是正在經歷某種可怕的幻覺。淚水從眼角滑落。

**凌墨**：啊——！！啊啊——！！

**旁白**：李杰緊緊抓住凌墨的肩膀,試圖讓他清醒。

**李杰**：凌墨！聽我說！我在這裡！你要撐住！

**旁白**：但凌墨彷彿聽不見任何聲音。他只是不斷地掙扎、尖叫。就像是有什麼東西正在撕裂他的意識。
