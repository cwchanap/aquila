# 第五十四幕：對峙

**小林雄大**：(舉起雙手)大家冷靜！請聽我說！這位小兄弟和山本大叔剛才是去——

**中村隼人**：冷靜？！他們拿著劍,背著一個昏迷的人,你要我們怎麼冷靜？！

**旁白**：中村的聲音充滿憤怒和恐懼,他指著李杰和山本。

**中村隼人**：你們看！他們肯定做了什麼！那個人明明好好的,現在卻昏迷不醒！

**斎藤蓮**：(站起身,推了推眼鏡)中村先生說得對。根據常識,持械和昏迷者同時出現,這絕對不是巧合。

**斎藤蓮**：而且,這把劍看起來很古老,很可能是某種危險物品。我們有理由懷疑他們的意圖。

**旁白**：車廂內的氣氛越來越緊張。乘客們開始議論紛紛,看向李杰和山本的眼神充滿懷疑和恐懼。

**伊藤奈々**：老公...我好怕...我們會不會有危險...

**伊藤誠**：(緊張地)該不會...他們想拿劍威脅我們吧？！

**田中健太**：欸欸,這...這不會是什麼殺人犯之類的吧...

**高橋美咲**：天啊天啊,怎麼辦怎麼辦...我們要報警嗎？可是手機沒訊號...

**旁白**：恐慌像病毒一樣在車廂內蔓延。

**中村隼人**：(看向李杰)你！解釋清楚！你們到底想幹什麼？！

**中村隼人**：還有,這到底是什麼鬼地方？！外面那是什麼月亮？！這根本不是東京！

**旁白**：中村的聲音越來越大,他走近李杰,臉上充滿憤怒。

**中村隼人**：我看...你們這些人都很可疑！那個拿劍的大叔、你、還有剛才幫你說話的人！

**中村隼人**：(轉向其他乘客)你們也這麼想吧？！把這些危險人物趕下車！我們留在車上安全！

**中村隼人**：列車肯定會繼續開到下一站,到時候就有人來救我們了！總比跟這些可疑的人待在一起好！

**斎藤蓮**：我同意。這是最理智的做法。讓他們下車,我們留在安全的車廂裡等待救援。

**旁白**：李杰看著中村煽動的樣子,腦中突然閃過之前循環的記憶——

**旁白**：中村知道終焉祭壇的存在。他知道那裡有一把白色的神劍。他一直在等待機會去祭壇拿那把劍。

**李杰**：(內心)他...他想把我趕下車...然後自己留在列車上...

**李杰**：(內心)列車不會馬上開往終焉祭壇...還要經過亡者峽谷...

**李杰**：(內心)而且...每次列車移動,車上的人數都必須減半...

**李杰**：(內心)他想讓我下車,減少人數,這樣列車就能繼續開...最終到達終焉祭壇...

**李杰**：(內心)然後他就可以去拿神劍...！

**李杰**：(內心)絕對不能讓他得逞...！

**山本浩二**：(低聲對李杰說)小兄弟...那個人想把我們趕下車...

**山本浩二**：如果我們真的下車,他就可以一個人留在列車上...

**李杰**：(點頭,壓低聲音)沒錯。但是...列車不會直接開到終焉祭壇。

**旁白**：李杰湊近山本,用只有他能聽見的聲音說。

**李杰**：(小聲)還要經過亡者峽谷,然後才會到達終焉祭壇。祭壇就在那裡。

**李杰**：(小聲)那個一直在煽動大家的人...他知道那裡有另一把劍。一把白色的神劍。他想等列車到站後,去祭壇拿劍。

**山本浩二**：(震驚地)什麼？！還有另一把劍？！

**李杰**：(小聲)對。而且...黑劍現在失去了力量。如果他拿到神劍...我們根本沒辦法對抗他。

**旁白**：山本浩二的臉色變了。他看向那個煽動的中年男人,眼神變得警戒。

**山本浩二**：(小聲)那我們該怎麼辦...？

**李杰**：(小聲)大叔...你現在下車,去車站大廳找地圖。然後往北走,跟著鐵軌延伸的方向。

**李杰**：(小聲)一直走,就能到達終焉祭壇。必須在列車到達之前,先拿到那把白色的神劍。

**山本浩二**：那你呢？還有小林先生...？

**李杰**：(堅定地)我和小林先生留在車上。

**李杰**：(小聲)列車要繼續移動,車上的人數必須減半。如果我們都下車,他就能讓列車開動。

**李杰**：(小聲)我們必須留在這裡拖住他。不讓列車那麼快到達祭壇。

**山本浩二**：(小聲)那個人...他到底是誰？為什麼知道這些？

**李杰**：(搖頭,小聲)我也不清楚...但他絕對有企圖。大叔,拜託你了。

**旁白**：山本浩二看著李杰的眼神,沉默了幾秒,然後點了點頭。

**山本浩二**：(看向小林雄大)小林先生...能請你幫忙嗎？

**山本浩二**：留在這裡,和李杰一起...別讓中村得逞。

**小林雄大**：(疑惑地)咦？你要去哪裡？

**山本浩二**：相信我。我去做該做的事。你們留在這裡拖住中村。

**旁白**：小林雄大看了看李杰,又看了看山本。雖然不明白發生了什麼,但他看得出事態緊急。

**小林雄大**：(堅定地點頭)我明白了。

**旁白**：山本浩二握緊手中的劍,快步走向車門。

**中村隼人**：(指著他)欸！你要去哪裡？！

**旁白**：但山本沒有回答,直接走下了列車,消失在紫色的月光中。

**中村隼人**：(指著山本離開的方向)他去哪裡了？！

**中村隼人**：(轉向李杰,憤怒地)你對他說了什麼？！

**中村隼人**：(指著李杰和小林)而且你們！我剛才是叫你們三個可疑的人下車！不是叫他一個人下車！

**中村隼人**：你們兩個應該跟他一起離開車廂！為什麼還留在這裡？！

**小林雄大**：(站起身,擋在李杰旁邊)這位先生,請你冷靜一點。

**斎藤蓮**：對啊！你們肯定在密謀什麼！太可疑了！

**旁白**：車廂內所有人的目光都集中在李杰和小林身上。懷疑、恐懼、憤怒...各種情緒交織在一起。

**田中健太**：(小聲)這...這到底是怎麼回事啊...

**鈴木明日香**：(抓著健太的手)我好怕...

**旁白**：李杰背上的凌墨依然沒有醒來。車廂外,紫色的月光冷冷地照著這個詭異的世界。

**旁白**：而在角落裡,清水恵美靜靜地看著這一切,眼神複雜而深沉。

**李杰**：(內心)撐住...大叔一定能在列車到達之前拿到神劍...

**李杰**：(內心)我和小林先生...必須拖住那個人...不能讓列車那麼快開動...

**李杰**：(內心)絕對不能讓他得逞...！
