# 第五十七幕：孤立

**旁白**：伊藤誠看著李杰手中的刀，臉色越來越蒼白。

**伊藤誠**：(顫抖著)你...你這個瘋子...

**旁白**：他慢慢後退，退到車廂門口。

**伊藤誠**：我...我不跟你計較...我先走了...

**旁白**：他轉身，快步離開第十節車廂，消失在走廊裡。

**旁白**：李杰握著刀，靠在座位上，大口喘息。他的手還在微微顫抖。

**李杰**：(內心)我竟然...拿刀威脅人...

**旁白**：他閉上眼睛，深吸一口氣，試圖讓自己冷靜下來。

**李杰**：(內心)不...我沒有選擇...如果下車，中村的計劃就成功了...

**旁白**：過了幾分鐘，李杰的呼吸逐漸平穩。他站起身，握緊手中的菜刀，開始往第一節車廂走。

**旁白**：第九節。第八節。第七節。空蕩蕩的車廂在他身後掠過。

**旁白**：第六節。第五節。第四節。

**旁白**：當他走到第三節車廂時——

**旁白**：他透過車窗，看到月台上站著幾個人影。

**李杰**：(驚訝地)——！

**旁白**：紫色的月光下，他看清了那些人影——小林雄大、小林智子，還有兩個孩子。

**旁白**：還有...清水恵美。

**李杰**：(內心)小林一家...還有清水...他們下車了？！

**旁白**：就在這時，伊藤夫婦也從第一節車廂走了出來，踏上月台。

**旁白**：車門「嘶」的一聲，緩緩關閉。

**李杰**：(內心)等等...！

**旁白**：列車開始震動。引擎發出低沉的轟鳴聲。

**旁白**：列車開始移動了。

**李杰**：(內心)怎麼會...到底發生了什麼？！

**旁白**：李杰立刻轉身，衝向第一節車廂。他快速穿過第二節車廂的走廊。

**旁白**：當他推開第一節車廂的門時——

**旁白**：车廂內的四個大學生——田中健太、鈴木明日香、高橋美咲、佐藤拓海——同時抬起頭，看向他。

**旁白**：他們的臉上，全是恐懼和憤怒。

**田中健太**：(站起身，顫抖著)你...你想幹什麼？！

**鈴木明日香**：(退後幾步)不要過來！離我們遠一點！

**高橋美咲**：(害怕地)你...你到底想做什麼...為什麼要拿刀...

**佐藤拓海**：(擋在女生前面)你想對我們做什麼？！

**旁白**：李杰愣住了。他看著這些驚恐但充滿敵意的年輕人，一時不知道該說什麼。

**李杰**：我...我不會傷害你們...

**田中健太**：(憤怒地)鬼才相信你！你剛才還拿刀威脅那個大叔！

**鈴木明日香**：那個帶著孩子的一家人都被你嚇跑了！還有那個女生！

**高橋美咲**：(顫抖著)你...你為什麼要這樣...我們又沒有對你做什麼...

**旁白**：但大學生們根本不相信他的話。他們緊緊靠在一起，充滿戒備地盯著李杰。

**旁白**：李杰環顧車廂。中村隼人坐在角落，冷冷地看著他，嘴角帶著一絲冷笑。斎藤蓮坐在中村旁邊，推了推眼鏡，表情平靜。

**旁白**：凌墨依然躺在座位上，昏迷不醒。

**中村隼人**：(平靜地站起身)看到了嗎？這就是這個人的真面目。

**中村隼人**：拿著刀到處威脅人。那個帶著孩子的一家人和那對夫婦都被他嚇得下車了。

**中村隼人**：現在...我們應該想辦法制服他，把刀奪下來。為了大家的安全。

**旁白**：但沒有人敢動。大學生們雖然憤怒，但也很害怕。

**李杰**：(握緊刀柄)不要過來。

**旁白**：他舉起菜刀，刀刃在燈光下閃爍。

**田中健太**：(後退)該死...！

**佐藤拓海**：(擋在女生前面)你敢！

**鈴木明日香**：(憤怒地)你真的瘋了！

**高橋美咲**：(顫抖著)我們...我們沒有惹你...

**旁白**：大學生們雖然害怕，但依然充滿敵意地盯著李杰。他們緊緊靠在一起，隨時準備逃跑。

**旁白**：李杰看著他們恐懼和憤怒的表情，握緊了刀柄。

**李杰**：(內心)沒辦法了...如果不這樣，他們會聽中村的話，把我趕下車...

**李杰**：(內心)我必須讓他們害怕...讓他們不敢輕舉妄動...

**旁白**：他的手微微顫抖，但還是握緊了刀柄。

**旁白**：就在這時，李杰的目光落在斎藤蓮和中村隼人身上。

**旁白**：斎藤依然坐在那裡，表情平靜。他推了推眼鏡，看著李杰，眼中沒有任何恐懼。

**旁白**：中村也是一樣。他冷冷地看著李杰，嘴角甚至帶著一絲冷笑。

**李杰**：(內心)等等...他們兩個...都不怕...？

**旁白**：李杰的腦中突然閃過一個念頭。

**李杰**：(內心)大學生們雖然憤怒，但還是害怕...

**李杰**：(內心)但是中村和斎藤...他們完全不怕我...

**李杰**：(內心)而且...之前搜查中村的時候...完全沒有找到任何武器...

**李杰**：(內心)三把手槍，兩把刀...那麼多武器...不可能憑空消失...

**李杰**：(內心)除非...

**旁白**：李杰的眼神變得銳利。

**李杰**：(內心)斎藤...是中村的同伙...！

**李杰**：(內心)中村把武器交給了斎藤...所以我們才找不到...！

**旁白**：李杰看著中村和斎藤。斎藤坐在中村旁邊，時不時小心翼翼地看向中村，似乎在等待什麼指示，又似乎有些畏懼。

**李杰**：(內心)如果是這樣...那我現在很危險...

**李杰**：(內心)車上的人越來越少...如果他們兩個聯手...

**旁白**：李杰環顧車廂。現在車上只剩下八個人——他自己、中村、斎藤、四個大學生，還有昏迷的凌墨。

**旁白**：列車的震動變得輕微。速度在減緩。

**李杰**：(內心)列車...要到下一站了...

**李杰**：(內心)下一站...又要有四個人下車...

**旁白**：李杰看著那四個充滿敵意的大學生。

**李杰**：(內心)他們...他們害怕我，也憎恨我...

**李杰**：(內心)一到站，他們肯定會立刻下車...

**李杰**：(內心)那樣的話...車上就只剩下我、中村、斎藤...還有昏迷的凌墨...

**旁白**：李杰的臉色變得蒼白。

**李杰**：(內心)四個人...

**李杰**：(內心)凌墨昏迷不醒...等於只有我一個人...

**李杰**：(內心)對上中村和斎藤...還有他們手中的三把槍...

**李杰**：(內心)我...我會死的...

**旁白**：列車繼續減速。車廂內的燈光微微閃爍。

**旁白**：中村隼人靠在座位上，冷靜地看著李杰，嘴角帶著一絲笑意。斎藤蓮推了推眼鏡，表情依然平靜。

**旁白**：李杰握緊刀柄，額頭滲出冷汗。

**李杰**：(內心)該怎麼辦...

**李杰**：(內心)我必須想辦法...在列車到站之前...

**李杰**：(內心)但是...我能做什麼...？

**旁白**：車廂內陷入詭異的沉默。只有列車引擎的轟鳴聲和車輪與鐵軌摩擦的聲音。

**旁白**：田中健太、鈴木明日香、高橋美咲、佐藤拓海緊緊靠在一起，充滿戒備和敵意地盯著李杰。

**旁白**：中村隼人和斎藤蓮坐在角落，冷靜地看著這一切。斎藤時不時小心翼翼地瞥向中村，眼中帶著一絲畏懼。

**旁白**：凌墨躺在座位上，依然昏迷不醒，完全不知道車廂內的危機。

**旁白**：李杰站在車廂門口，握著菜刀，腦中飛快地思考著。

**李杰**：(內心)一定...一定有辦法的...

**李杰**：(內心)我不能就這樣放棄...

**旁白**：但他的腦中一片空白。他想不出任何辦法。

**旁白**：列車繼續向前行駛，帶著車廂內的這些人，駛向未知的下一站。

**旁白**：紫色的月光透過車窗照進來，將每個人的影子拉得很長。

**旁白**：空氣中瀰漫著恐懼、緊張，還有絕望。
