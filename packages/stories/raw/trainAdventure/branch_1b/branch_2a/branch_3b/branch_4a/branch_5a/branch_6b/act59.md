# 第五十九幕：抉擇

**旁白**：斎藤看著李杰沾滿鮮血的臉，看著地上中村的屍體，整個人嚇得說不出話。

**旁白**：他握著刀的手劇烈顫抖。

**李杰**：(冷靜地)放下刀。

**旁白**：李杰慢慢向斎藤走去，手中的菜刀上還滴著血。

**斎藤蓮**：(崩潰地)不...不要...

**旁白**：斎藤的手一鬆，折疊刀「噹」的一聲掉在地上。他直接跪下，雙手舉起，眼淚從臉上流下。

**斎藤蓮**：(顫抖著)求求你...不要殺我...我什麼都說...

**李杰**：(停下腳步)中村的武器...你藏在哪裡？

**斎藤蓮**：(顫抖著)在...在我的背包...夾層裡...之前大家搜查背包的時候...我假裝找不到...

**旁白**：李杰看向斎藤放在座位上的背包。

**李杰**：為什麼要幫中村？

**斎藤蓮**：(低頭)我...我是被迫的...中村...他知道我的秘密...我最大的秘密...如果不幫他...他就會把秘密說出去...

**旁白**：李杰皺眉。

**李杰**：什麼秘密？

**旁白**：斎藤抬起頭，眼中充滿恐懼和絕望。

**斎藤蓮**：(顫抖著)我...我不能說...就算你殺了我...我也不能說...

**旁白**：他的聲音堅定，雖然害怕，但眼神中帶著某種決絕。

**李杰**：(內心)寧死也不說的秘密...

**旁白**：李杰看著斎藤，沉默了幾秒。

**李杰**：(嘆氣)算了。

**旁白**：李杰彎腰撿起斎藤掉在地上的折疊刀，然後走向斎藤的背包。斎藤跪在地上，看著李杰拿起他的背包，但完全不敢阻止。

**旁白**：李杰打開背包，翻找了一會兒，在夾層裡找到了兩把手槍和一把小刀。他把這些武器全部放進自己的口袋和腰間。

**旁白**：現在，所有的武器都在李杰手上。斎藤已經沒有任何威脅了。

**旁白**：李杰轉身看向昏迷的凌墨。

**斎藤蓮**：(小聲)那個人...他怎麼了？

**李杰**：我也不知道。他突然就昏倒了。你知道些什麼嗎？

**斎藤蓮**：(搖頭)我...我什麼都不知道...中村只告訴我...要幫他把那些人趕下車...其他的...他什麼都沒說...

**旁白**：李杰沉默地看著斎藤，然後轉身坐在座位上。

**旁白**：車廂內陷入沉默。只有列車引擎的轟鳴聲。

**旁白**：地上，中村的屍體躺在血泊中，已經不再動彈。

**旁白**：凌墨依然躺在座位上，昏迷不醒。

**旁白**：斎藤跪在地上，不敢動，也不敢看李杰。

**旁白**：李杰靠在座位上，閉上眼睛，深呼吸。

**李杰**：(內心)總算...解決了中村...但是...接下來該怎麼辦？

**旁白**：過了一段時間，列車開始減速。

**旁白**：李杰睜開眼睛，看向窗外。

**旁白**：紫色的月光照進車廂。窗外的景色逐漸清晰。

**旁白**：峽谷。黑色的岩壁。遠處隱約可見一些骸骨。

**李杰**：(內心)這裡是...

**旁白**：列車緩緩停下。車門「嘶」的一聲打開了。

**旁白**：李杰站起身，走到車門旁，看向月台。

**旁白**：亡者峽谷站。

**李杰**：(內心)這裡...我來過...之前有一次循環，我自己從這裡步行去終焉祭壇...

**旁白**：李杰回想起那次循環。他在前一站下車後，步行經過這個亡者峽谷站，然後沿著鐵軌穿過峽谷，孤獨地走了很久才到達祭壇。

**李杰**：(內心)等等...車上現在有三個活人...我、凌墨、斎藤...列車的規則是...每到一站，必須有人下車...如果沒人下車...列車就不會繼續開...

**旁白**：李杰看著打開的車門，陷入沉思。

**李杰**：(內心)也就是說...必須有一個人下車...是讓斎藤下車...還是我自己下車...？

**旁白**：李杰轉身看向車廂內。

**旁白**：斎藤依然跪在地上，渾身顫抖。

**旁白**：凌墨躺在座位上，昏迷不醒。

**旁白**：中村的屍體躺在血泊中。

**李杰**：(內心)如果讓斎藤下車...

**李杰**：(內心)那我就能帶著凌墨，坐列車直接去終焉祭壇...

**李杰**：(內心)山本應該還在路上...列車比步行快得多...

**李杰**：(內心)我可能會比他先到達祭壇...然後在那裡等他...

**旁白**：他看向車門外的峽谷。

**李杰**：(內心)但是...如果我自己下車...

**李杰**：(內心)我可以去找山本...他應該還在從寂海站步行來祭壇的路上...

**李杰**：(內心)還有其他下車的人...小林一家、恵美、伊藤夫婦、那四個大學生...

**李杰**：(內心)雖然不知道他們在哪裡...但也許能找到他們...然後一起去祭壇...

**旁白**：李杰握緊手中的菜刀，陷入思考。

**李杰**：(內心)但是...如果我下車...斎藤和凌墨會怎麼樣...？凌墨還昏迷著...如果斎藤對他做什麼...

**旁白**：李杰看向斎藤。斎藤低著頭，不敢看他。

**李杰**：(內心)不...斎藤現在嚇成這樣...應該不敢做什麼...而且...他手上已經沒有武器了...

**旁白**：車門依然開著。冷風吹進車廂。

**旁白**：李杰站在車門旁，看著外面的峽谷，又看看車廂內的斎藤和凌墨。

**李杰**：(內心)該怎麼選...讓斎藤下車...還是我自己下車...

**旁白**：時間一分一秒過去。李杰握緊菜刀，眉頭緊皺。

**旁白**：他必須做出決定。
