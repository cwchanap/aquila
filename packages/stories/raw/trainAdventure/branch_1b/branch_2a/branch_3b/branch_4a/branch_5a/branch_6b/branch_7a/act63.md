# 第六十三幕：各自的道路

**旁白**：屋內陷入短暫的沉默。李杰的問題讓三個人面面相覷。

**小林雄大**：(遲疑地)我們...打算繼續往森林深處走。

**小林智子**：(點頭)恵美小姐...她似乎有一種能力，可以感知正確的方向。

**旁白**：智子看向恵美，眼神中帶著信任。

**小林智子**：她帶我們來到這裡，然後說我們應該繼續往前走...深入森林...

**清水恵美**：(小聲，眼神閃躲)我...我只是...有一種直覺...

**旁白**：李杰看著恵美。她的眼神依然迴避著他，不敢直視。

**李杰**：(內心)感知方向的能力...這當然是她編出來的藉口...

**李杰**：(內心)她知道精靈村的位置...她在二十年前就去過那裡...

**李杰**：(內心)她現在是在用這個謊言，把小林一家帶到精靈村...然後...

**旁白**：他想起凌墨告訴他的話——恵美打算用無辜者交換她男友的自由。

**李杰**：(內心)但即使我現在揭穿她...雄大和智子會相信我嗎？

**旁白**：李杰看了看自己衣服上的血跡，又看了看恵美那張年輕無害的臉。

**李杰**：(內心)一個衣服上沾滿血的陌生男人...和一個看起來柔弱的年輕女孩...

**李杰**：(內心)他們會相信誰？答案顯而易見...

**旁白**：李杰深吸一口氣，決定先不揭穿恵美的謊言。

**李杰**：(轉移話題)你們...有看到山本先生嗎？就是那個中年大叔。

**小林雄大**：(搖頭)沒有。我們下車之後就沒見過他。

**小林智子**：(小聲)說實話...我們也不太想見到他...

**李杰**：為什麼？

**小林智子**：他手裡拿著一把劍...雖然那把劍看起來已經褪色了，但還是讓人很害怕...

**小林雄大**：(點頭)對...我們不知道他拿那把劍想做什麼...所以...不太敢接近他。

**旁白**：雄大和智子的眼神中閃過恐懼。

**李杰**：(內心)黑劍...他們看到山本拿著黑劍...所以害怕他...

**李杰**：我明白了。

**旁白**：李杰點了點頭，沒有再多問。

**李杰**：那...伊藤夫婦呢？他們也是在第一站下車的吧？

**小林智子**：(點頭)對...但他們不願意跟我們一起走。

- *小林雄大：(苦笑)伊藤先生...他覺得我很可疑。

**小林雄大**：在車上的時候...我一開始站在李杰這邊...幫他說話...

**小林雄大**：但後來...我為了保護家人...突然轉向...背叛了他...

**旁白**：雄大的聲音帶著深深的內疚。

**小林雄大**：我把他告訴我的事...關於異世界、關於詛咒之劍...全都說了出來...

**小林雄大**：我讓大家以為他是精神病患者...讓大家都想把他趕下車...

**小林智子**：(輕聲)雄大...你當時也是為了保護我們和孩子們...

**小林雄大**：(搖頭)但那還是背叛...而且我一開始還幫他說話...後來卻突然反過來指控他...

**小林雄大**：伊藤先生覺得...這種反覆無常的人...根本不值得信任...

**小林智子**：而且...伊藤先生似乎也懷疑恵美小姐的能力。

**小林智子**：他說什麼「感知方向」根本是胡說八道，不可能有這種能力。

**清水恵美**：(低下頭，沒有說話)

**小林智子**：所以...伊藤夫婦決定自己走。他們說要沿著鐵軌走到下一站。

**小林雄大**：我們勸過他們...但伊藤先生堅持要離開。他說不想和我們這些「可疑的人」待在一起。

**旁白**：李杰聽完，腦海中逐漸拼湊出完整的畫面。

**李杰**：(內心)伊藤夫婦沿著鐵軌走向下一站...

**李杰**：(內心)小林一家和恵美在這裡...準備往精靈村走...

**李杰**：(內心)山本應該還在去祭壇的路上...

**李杰**：(內心)四個大學生在森林小屋休息，之後也會去祭壇...

**李杰**：(內心)凌墨和斎藤在列車上...

**旁白**：他深吸一口氣，做出了決定。

**李杰**：我明白了。謝謝你們告訴我這些。

**旁白**：李杰站起身，走向廚房。

**小林雄大**：(疑惑地)你...你要做什麼？

**李杰**：拿一些食物和水。我要繼續趕路。

**旁白**：李杰打開櫃子，拿出一些麵包、餅乾和一個皮革水袋。

**旁白**：他把這些東西裝進背包裡，動作迅速而熟練。

**小林智子**：(小聲)你...你要去哪裡？

**李杰**：(背對著他們)我有自己的事要做。

**小林雄大**：(站起身)李杰...你接下來打算做什麼？

**旁白**：雄大的聲音中帶著一絲擔心。

**李杰**：(轉過身，看著他們)你們照顧好孩子們就行了。

**李杰**：至於我...不用擔心。

**旁白**：李杰沒有說出自己的目的地。

**李杰**：(內心)我要先去精靈村...找艾拉如談談...

**李杰**：(內心)我需要休息...需要時間思考接下來該怎麼做...

**李杰**：(內心)但我不能告訴他們...尤其是當恵美在這裡的時候...

**旁白**：李杰背起背包，走向門口。

**小林雄大**：李杰！等等！

**旁白**：雄大追上前幾步。

**小林雄大**：你...你真的不是來找我們麻煩的...對吧？

**旁白**：他的眼神中有複雜的情緒——內疚、不安、還有一絲希望。

**李杰**：(停下腳步，沒有回頭)我說過了。我不是來找麻煩的。

**李杰**：你們...好好照顧孩子們。

**旁白**：李杰打開門，走了出去。

**旁白**：身後，小林雄大站在門口，看著李杰的背影消失在濃霧之中。

**小林智子**：(輕聲)他...真的走了...

**清水恵美**：(站在壁爐旁，眼神複雜地看著門外)

**旁白**：恵美的手緊緊握著，指節泛白。

**清水恵美**：(內心)李杰...他知道些什麼...？

**清水恵美**：(內心)他...會阻止我嗎...？

**旁白**：濃霧再次將小屋包圍。兩輪紫月依舊懸在天空，冷冷地注視著這一切。

---

**旁白**：李杰在濃霧中前進。他知道精靈村的方向——就在這條路的盡頭。

**旁白**：他之前來過這裡。和山本、健太、拓海、凌墨一起。

**旁白**：那時候，他們找到了精靈村，見到了艾拉如。

**李杰**：(內心)艾拉如...希望她能給我一些答案...

**李杰**：(內心)關於恵美的計劃...關於祭壇...關於這個世界的一切...

**旁白**：他加快腳步。疲憊的身體在抗議，但他沒有停下。

**旁白**：身後，小屋的輪廓漸漸消失在濃霧之中。

**旁白**：前方，精靈村在等待著他。
