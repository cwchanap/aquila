# 終幕：終焉

**旁白**：車門依然開著。冷風吹進車廂，帶著峽谷深處的氣息。

**旁白**：李杰站在車門旁，握緊手中的菜刀，看著外面的亡者峽谷站。

**旁白**：時間一分一秒過去。

**旁白**：最終，他做出了決定。

**李杰**：(轉身看向斎藤)你...下車吧。

**旁白**：斎藤跪在地上，抬起頭，眼中充滿驚訝和恐懼。

**斎藤蓮**：(顫抖著)什...什麼...？

**李杰**：我說，你下車。

**旁白**：李杰的聲音平靜，但帶著不容置疑的語氣。

**斎藤蓮**：(不敢相信)你...你不殺我？

**李杰**：我需要帶著凌墨去終焉祭壇。你走吧。

**旁白**：斎藤愣了幾秒，然後慢慢站起身。他的腿還在顫抖。

**斎藤蓮**：(小聲)謝...謝謝...

**旁白**：他快步走向車門，踉蹌地下了車。

**旁白**：他的腳剛踏上月台——

**旁白**：車門「嘶」的一聲，立刻開始關閉！

**旁白**：斎藤站在月台上，回頭看著車廂內的李杰。

**旁白**：李杰靠在座位旁，握著菜刀，看著他。

**旁白**：車門完全關閉。

**旁白**：列車開始啟動，緩緩駛離月台。

**旁白**：斎藤的身影逐漸變小，最終消失在窗外。

**李杰**：(內心)希望...你能活下去...

**旁白**：李杰轉身，看向躺在座位上的凌墨。凌墨依然昏迷不醒。

**李杰**：(走到凌墨身邊)凌墨...我會帶你去祭壇...

**旁白**：他坐在凌墨旁邊的座位上，握緊手中的菜刀。

**旁白**：車廂內只剩下他、昏迷的凌墨，還有地上中村的屍體。

**旁白**：列車駛過峽谷，進入黑色的森林。

**旁白**：樹木扭曲，枝幹詭異。空氣中瀰漫著腐朽的氣息。

**旁白**：地面上散落著骨頭和破碎的石碑。

**旁白**：李杰看著窗外，腦中一片空白。

**李杰**：(內心)殺了中村...我真的殺了人...

**旁白**：他看向自己的雙手。手上還有血跡，已經乾涸變黑。

**李杰**：(內心)沒辦法...如果不殺他...被殺的就是我...

**旁白**：列車繼續前行。

**旁白**：不知過了多久，窗外的森林景色突然消失。

**旁白**：列車駛入了一座巨大的建築內部。

**旁白**：四周的黑暗被銀白色的光芒取代。那是懸掛在穹頂的魔法火焰，散發著溫暖而柔和的光輝。

**旁白**：李杰透過車窗，看到下方是一片澄澈的翠綠色湖泊。湖水散發著柔和的光芒。

**李杰**：(低聲)終焉祭壇...

**旁白**：列車緩緩停下。車門「嘶」的一聲，自動打開。

**旁白**：寬闊的黑石月台延伸在湖面上方。巨大的石柱從地面延伸到穹頂，柱上雕刻著奇異的符文。

**旁白**：李杰站起身，握著菜刀，走到車門旁。

**旁白**：他看向凌墨。凌墨依然昏迷。

**李杰**：(內心)先下去看看...也許...也許能找到山本...

**旁白**：他踏上月台。

**旁白**：銀白色的光芒灑在他身上，照亮了他臉上和衣服上的血跡。

**旁白**：就在這時——

**旁白**：遠處傳來腳步聲。

**旁白**：李杰猛地抬起頭，警覺地握緊菜刀。

**旁白**：腳步聲越來越近。清晰，平穩，在空曠的大廳中迴盪。

**旁白**：然後，一個身影出現在遠處。

**旁白**：那是一個穿著白色長袍的年輕人。手中握著一把白色的長劍。

**旁白**：他緩緩走來，白袍在空中飄動，劍身在魔法火焰的光芒下泛著微光。

**李杰**：(內心)是他...翔...

**旁白**：高山翔走到月台上，停在距離李杰十公尺的地方。

**旁白**：他的目光落在李杰身上，眼神銳利而冷漠。

**旁白**：然後，他的視線掃過李杰臉上的血跡，衣服上的血跡，還有李杰手中沾血的菜刀。

**高山翔**：(冷漠地)你身上...有很濃的死亡氣息。

**旁白**：他的聲音平靜，但帶著壓迫感。

**高山翔**：你殺了人。

**旁白**：李杰握緊菜刀，全身肌肉緊繃。

**李杰**：是...我殺了一個人...

**高山翔**：(舉起白劍)那麼，你必須被消滅。

**高山翔**：這座建築不允許沾染鮮血的人存在。你的存在，會破壞這裡的秩序。

**李杰**：(急忙說)等等！我是被迫的！

**李杰**：那個人...他想殺我...我只是自衛...

**高山翔**：(冷冷地)你有證據嗎？

**旁白**：李杰愣住了。

**李杰**：證據...？

**高山翔**：有人能證明你是自衛嗎？

**旁白**：李杰的腦中一片空白。

**李杰**：(內心)凌墨...還在昏迷...斎藤...已經下車了...

**李杰**：(內心)沒有...沒有人能為我作證...

**旁白**：他看著高山翔，張開嘴，卻說不出話。

**高山翔**：沒有證人。那麼，你就是兇手。

**旁白**：高山翔握緊白劍，擺出攻擊姿勢。

**李杰**：(顫抖著)不...等等...我真的是自衛...

**旁白**：但高山翔已經不再聽了。

**旁白**：他的身影瞬間消失！

**李杰**：——！

**旁白**：下一秒，高山翔出現在李杰面前！

**旁白**：白劍泛著聖潔的光芒，筆直地刺向李杰的胸口！

**旁白**：李杰想要舉起菜刀防禦——

**旁白**：但已經來不及了。

**旁白**：白劍的速度太快。

**旁白**：「噗」的一聲——

**旁白**：劍刃貫穿了李杰的胸口。

**李杰**：(瞪大眼睛)啊...

**旁白**：劇烈的疼痛從胸口傳來。他能感覺到劍刃穿過肋骨，刺穿心臟。

**旁白**：鮮血從傷口湧出，順著白劍流下。

**旁白**：李杰的手鬆開，菜刀「噹」的一聲掉在地上。

**旁白**：他的雙腿失去力量，整個人向後倒去。

**旁白**：高山翔抽出白劍。劍身上沾滿鮮血，在銀白的光芒下顯得格外刺眼。

**旁白**：李杰倒在冰冷的黑石地面上。

**旁白**：他的視線開始模糊。銀白色的光芒越來越暗。

**李杰**：(內心，微弱)沒關係...

**李杰**：(內心，微弱)還能...再來一次...

**李杰**：(內心，微弱)我會...重新醒來...在列車上...

**旁白**：他的眼皮越來越重。

**旁白**：就在意識徹底消散的前一刻——

**旁白**：一段記憶突然浮現在他腦海中。

**旁白**：那是很久以前的一次循環。

**旁白**：他登上列車，走進車廂。

**旁白**：車廂內，一具屍體倒在血泊中。那是那個自私的中年男人——中村隼人。

**旁白**：胸口有一個致命的傷口——被利劍貫穿，一擊斃命。

**旁白**：傷口平整而乾淨。

**李杰**：(內心，微弱)原來如此...

**李杰**：(內心，微弱)原來...那個時候的中村...

**李杰**：(內心，微弱)是被...翔...殺死的...

**旁白**：他終於明白了。

**旁白**：那次循環中，中村也殺了人。也許是殺了伊藤奈々，也許是其他人。

**旁白**：然後，中村乘著列車來到終焉祭壇。

**旁白**：高山翔感受到他身上的死亡氣息，用白劍貫穿了他的胸口。

**旁白**：就像現在的自己一樣。

**李杰**：(內心，微弱)原來...我和中村...走上了同樣的道路...

**旁白**：李杰的眼皮越來越重。

**旁白**：銀白色的光芒徹底消失了。

**旁白**：他的呼吸停止了。

**旁白**：他帶著「還能再來一次」的想法，永遠閉上了眼睛。

**旁白**：但他不知道的是——

**旁白**：這一次，他再也不會醒來了。

**旁白**：李杰的身體躺在月台上，胸口的傷口還在流血。

**旁白**：高山翔站在那裡，手握白劍，面無表情地看著李杰的屍體。

**高山翔**：(冷漠地)又一個被黑暗吞噬的靈魂。

**旁白**：他轉身，朝著大廳深處走去。

**旁白**：白袍在空中飄動，劍身上的鮮血滴落在地面上。

**旁白**：月台上，只剩下李杰的屍體，還有車廂內昏迷的凌墨。

**旁白**：銀白色的魔法火焰依然燃燒著，照亮著這座聖潔而殘酷的建築。

**旁白**：翠綠色的湖水依然平靜，倒映著穹頂的光輝。

**旁白**：列車靜靜地停在月台上，車門依然開著。

**旁白**：一切都回歸寂靜。

**旁白**：李杰的故事，就此結束。
