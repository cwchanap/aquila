# 第二十一幕：曦光之庭的秘密

**旁白**：李杰站在月台上,看著眼前的列車。

**旁白**：黑色的車身在陽光下閃爍,車窗映照著他的倒影。裡面躺著兩具屍體,象徵著死亡和絕望。

**旁白**：然後他回頭望向森林深處。

**旁白**：那裡有神樹,有可能回家的希望。雖然路途遙遠,時間緊迫,但這或許是他唯一的機會。

**旁白**：他做出了決定。

**李杰**：(轉向伊蘭尼爾)伊蘭尼爾...我決定了。

**旁白**：伊蘭尼爾靜靜地看著他,等待他的答案。

**李杰**：我想去神樹那裡尋找答案。

**伊蘭尼爾**：(點頭)既然你已經決定,那就這麼辦吧。

**旁白**：他的語氣平靜,沒有驚訝,也沒有勸阻,彷彿早就預料到了這個選擇。

**李杰**：在出發前,能麻煩你幫個忙嗎?

**伊蘭尼爾**：什麼忙?

**李杰**：(看向列車)車上的那兩個人...我想為他們找個地方安葬。

**伊蘭尼爾**：(沉默片刻,然後點頭)好。

**旁白**：兩人走進車廂,合力將兩具屍體抬了出來。

**旁白**：那個中年男人的身體已經僵硬,血跡乾涸在衣服上。那個女人蜷縮的姿勢讓人心痛,臉上凝固著恐懼。

**旁白**：他們將屍體搬到車站旁的森林邊緣,在一片陽光能照到的空地上。

**旁白**：伊蘭尼爾用他的短刀開始挖掘,李杰則用雙手。泥土鬆軟,散發著草木的氣息,與之前那些死寂的地方完全不同。

**旁白**：他們挖了兩個並排的淺坑,將屍體小心地放進去,然後開始填土。

**旁白**：做完這一切後,天色已經完全暗了下來。

**旁白**：夕陽的餘暉透過樹葉灑落,將森林染成一片金紅色。遠處傳來鳥兒歸巢的鳴叫,空氣變得涼爽而清新。

**伊蘭尼爾**：(拍掉手上的泥土)天黑了。

**旁白**：他看著漸暗的天色。

**伊蘭尼爾**：森林的夜晚有它自己的危險。今天你就在村裡休息吧。

**伊蘭尼爾**：我們明天一早再出發前往神樹。

**李杰**：好。

**旁白**：他感受到身體傳來的疲憊。

**李杰**：說實話,我的身體也到了極限。

**李杰**：之前在永無止境的黑夜裡待了太久,生理時鐘已經完全亂了。

**伊蘭尼爾**：(微笑)那你今晚好好休息。

**旁白**：伊蘭尼爾再次將李杰帶回了自己的家中。

**旁白**：他們沿著吊橋,穿過已經點起燈籠的村莊。精靈們的屋子裡透出溫暖的光芒,偶爾傳來輕聲的歌唱和談笑。

**旁白**：回到伊蘭尼爾的樹屋,他為李杰安排了一間舒適的客房。

**旁白**：房間不大,但很溫馨。一張柔軟的床鋪著乾淨的亞麻布,窗戶能看見滿天星辰。空氣中飄散著木頭和草本植物的香氣。

**李杰**：(躺在床上,閉上眼睛)...

**旁白**：在經歷了那麼多恐怖和奔波之後,能夠躺在一張真正的床上,讓李杰感到了無比的奢侈和安心。

**旁白**：柔軟的床墊包裹著他疲憊的身體,溫暖的毯子驅散了寒意。很快,他就沉入了深沉的睡眠。

**旁白**：這一夜,他沒有做夢。

**旁白**：第二天,當李杰醒來時,陽光正透過樹葉的縫隙灑進房間。

**旁白**：他睜開眼睛,花了幾秒鐘才想起自己在哪裡。窗外是樹枝和綠葉,空氣中飄來食物的香氣。

**旁白**：他起身走出房間,看見伊蘭尼爾已經在桌上擺好了早餐。

**旁白**：烤得香脆的麵包,散發著淡淡的麥香。一碗冒著熱氣的野菜湯,裡面有綠色的葉子和白色的菇類。還有一小碟蜂蜜和幾顆新鮮的漿果。

**伊蘭尼爾**：(坐在桌旁)早安。來,趁熱吃吧。

**李杰**：(坐下)謝謝。

**旁白**：他拿起一塊麵包,咬了一口。外皮酥脆,內裡柔軟,帶著天然的甜味。

**旁白**：熱湯溫暖了胃,讓他整個人都清醒過來。

**伊蘭尼爾**：(看著李杰)你昨天說的那些...關於外面世界的事情。

**旁白**：他停頓了一下,似乎在組織語言。

**伊蘭尼爾**：你對森林以外的世界,提到了很多我們無法想像的東西。

**伊蘭尼爾**：我很好奇,那裡真的沒有森林嗎?

**李杰**：(有些驚訝)當然有,只是沒有這裡這麼廣闊。

**旁白**：他喝了一口湯,然後問道。

**李杰**：你們...從來沒想過要離開這片森林,去外面看看嗎?

**伊蘭尼爾**：(神色黯然地搖頭)我們無法離開。

**旁白**：他的語氣變得沉重。

**伊蘭尼爾**：對於我們精靈來說,永恆之森就是整個世界。

**伊蘭尼爾**：神樹的力量庇護著我們。一旦離開森林太遠,我們的生命力就會迅速流失,最終歸於塵土。

**旁白**：李杰被這個事實震驚了。

**李杰**：(內心)原來這個看似自由的村莊,其實是一個巨大的牢籠...

**旁白**：他想起這些精靈們友善的笑容,想起他們數百年不變的生活。原來這一切並非選擇,而是束縛。

**李杰**：原來是這樣...

**旁白**：他沉默了片刻,然後換了個話題。

**李杰**：說起來,在來到永恆之森前,我經過了四個車站。

**伊蘭尼爾**：(好奇)都是什麼樣的地方?

**李杰**：每一個都...很詭異。

**旁白**：他開始回憶那段旅程。

**李杰**：第一個叫「寂海」,車站一側是森林,另一側是黑色的海洋。天空有兩個紫色的月亮。

**李杰**：第二個是「幽冥之都」,也是一座廢棄的車站。同樣是一側森林,一側黑海。

**李杰**：然後是「亡者峽谷」,那裡就只是一個建在無底峽谷邊緣的月台。

**李杰**：最後一個,就是「終焉祭壇」。

**旁白**：他想起那座宏偉而詭異的建築。

**李杰**：一座巨大、冰冷的黑色石頭建築,像個陵墓。

**旁白**：就在他複述這段旅程時,一個他之前從未注意到的細節,突然在他腦中浮現。

**李杰**：(內心)等等...

**旁白**：他停下說話,陷入思考。

**李杰**：(內心)「寂海」是海,「亡者峽谷」是峽谷,「終焉祭壇」是祭壇,現在的「永恆之森」也是森林...

**旁白**：它們的環境都和名字相符。

**李杰**：(內心)只有「幽冥之都」...「都」是都市的意思,但那裡除了森林和海,什麼都沒有。

**旁白**：這個矛盾讓他感到一絲不安。

**李杰**：(內心)是我錯過了什麼嗎?

**旁白**：他想起那座廢棄的車站,想起空蕩蕩的月台。他只是匆匆經過,沒有仔細探索周圍。

**旁白**：或許在那片森林深處,真的有一座城市的廢墟?

**旁白**：但現在想這些也為時已晚。列車已經開走,他再也回不去了。

**伊蘭尼爾**：(打斷他的思緒)聽起來,除了這裡,外面的世界似乎都很荒涼。

**李杰**：是啊。

**旁白**：他收回思緒,點了點頭。

**李杰**：除了那些屍體,我沒有遇到任何其他的活物。那是一段非常糟糕的經歷。

**伊蘭尼爾**：(站起身)那麼,你在這裡好好休息,就是最好的安排了。

**旁白**：早餐過後,伊蘭尼爾拿起他的細劍,還有一張由獸皮繪製的地圖。

**旁白**：地圖上繪製著精細的線條,標記著河流、山谷、還有各種符號。在地圖的中心,有一個特別醒目的標記——那應該就是神樹的位置。

**伊蘭尼爾**：(對李杰點頭)我們出發吧。

**旁白**：他將地圖捲起來,插在腰間。

**伊蘭尼爾**：前往森林的中心。
