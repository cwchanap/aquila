# 第二十四幕：二十年的迴響

**旁白**：列車駛入了隧道。

**李杰**：又是隧道...

**旁白**：他下意識地看了一眼手機。螢幕上的信號格顯示著微弱的連線。網頁載入時不時會轉圈圈，但並沒有像那一晚一樣，徹底顯示為「沒有服務」。

**李杰**：信號還在...沒有完全消失。

**旁白**：隧道的黑暗在車窗外飛逝而過。李杰緊握著手機，手心微微出汗。

**旁白**：一分鐘。兩分鐘。三分鐘。

**旁白**：前方出現了光亮。列車駛出了隧道，重新進入夜色籠罩的郊區。車窗外是熟悉的景色——零星的住宅燈光、高架橋、還有遠處的山脈輪廓。

**李杰**：五分鐘...就出來了。看來今晚只是普通的夜晚。

**旁白**：他徹底放鬆下來，深深地靠在座椅上。緊繃了許久的神經終於得到了片刻的休息。

**旁白**：他解鎖手機，百無聊賴地打開了常逛的「3ch」論壇。

**旁白**：首頁上是一些無聊的新聞討論和政治口水戰。他隨意地往下滑動，尋找著能打發時間的話題。

**旁白**：就在這時，一個帖子的標題突然像磁鐵一樣吸住了他的目光。

**李杰**：這是...

**旁白**：那是在「懸案與都市傳說」版塊裡的一個帖子。標題很簡單，但字字都敲在他的心上。

**旁白**：李杰的手停在螢幕上，整個人愣住了。

**李杰**：20年前...9月23日...和我將要經歷的那一天，一模一樣。

**旁白**：這絕對不是巧合。同樣的日期，同樣的路線，同樣的末班車。

**李杰**：所以在我之前...已經有人被那輛列車帶走了嗎？

**旁白**：他的心跳開始加速。這個帖子證實了一件事——那不是只會發生一次的異常事件。它是有規律的，是會重複的。

**旁白**：他繼續往下滑，看向帖子下方的留言區。

**李杰**：如果什麼都不做...那個發帖的人，永遠都不會知道真相。

**旁白**：他看著自己打出的文字，手指懸停在「發送」按鈕上。

**李杰**：這樣做...真的好嗎？

**旁白**：但只猶豫了一秒，他就按下了發送鍵。

**旁白**：留言立刻出現在帖子下方，顯示為最新的第六樓。

**李杰**：就這樣吧...至少給他一個機會知道真相。

**旁白**：他關掉了手機螢幕，將手機塞回口袋。

**旁白**：車廂內響起了熟悉的JR到站提示音。那是一段輕快而悅耳的旋律，在安靜的車廂中格外清晰。

**廣播**：まもなく、三鷹、三鷹です。お出口は、右側です。

**旁白**：列車平穩地減速，伴隨著輕微的煞車聲，準確地停靠在月台旁。

**旁白**：車門打開，一股帶著涼意的、屬於深夜郊區的空氣吹了進來。

**旁白**：李杰站起身，隨著稀稀落落的人群走下車，踏上了熟悉的月台。

**旁白**：與新宿的喧囂不同，深夜的三鷹站安靜了許多。只有零星的乘客快步走向出口，腳步聲在空曠的月台上迴盪。

**李杰**：回來了...真的回到三鷹了。

**旁白**：他穿過驗票閘門，刷卡時發出的「嗶」聲是如此真實。

**旁白**：走在空曠的車站大廳裡，自己的腳步聲顯得格外清晰。牆上的時鐘顯示著午夜一點十分。

**旁白**：他走出車站，夜晚的涼風吹在臉上。

**旁白**：車站外是熟悉的街道——24小時營業的便利商店亮著明亮的燈光，遠處是一棟棟安靜的公寓樓。

**旁白**：偶爾有計程車駛過，車燈在柏油路上投下移動的光影。

**李杰**：這一切...都還在。

**旁白**：這種真實感讓他感到既安心，又有一種恍如隔世的不真實。

**旁白**：他沿著熟悉的路走向自己的公寓。路過便利商店時，透過玻璃窗能看見店員正在補貨。自動門感應器發出輕微的「叮咚」聲。

**旁白**：五分鐘後，他站在自己公寓的門前。

**旁白**：當他用鑰匙打開門時，一股久違的、名為「家」的安心感徹底包圍了他。

**旁白**：熟悉的玄關、熟悉的擺設、熟悉的氣味。

**李杰**：真的...回來了。

**旁白**：他脫掉鞋子，直接走進浴室。熱水從蓮蓬頭噴灑而出，溫暖的水流沖刷著疲憊的身體。

**旁白**：水蒸氣瀰漫在浴室裡，模糊了鏡子和瓷磚。他閉上眼睛，讓熱水沖走這幾天積累的疲憊和緊張。

**旁白**：洗完澡後，他連頭髮都沒擦乾，就將自己重重地摔在柔軟的床上。

**旁白**：床墊的觸感、被子的溫度、枕頭的柔軟——這些再普通不過的感覺，此刻卻讓他感到無比珍貴。

**李杰**：終於...能好好睡一覺了。

**旁白**：他閉上眼睛，幾乎是瞬間就陷入了深沉的睡眠。

**旁白**：沒有噩夢，沒有驚醒。

**旁白**：只有久違的、完全的、屬於活人的安眠。
