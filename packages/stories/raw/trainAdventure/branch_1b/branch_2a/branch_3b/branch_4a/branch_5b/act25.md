# 第二十五幕：秋葉原的約定

**旁白**：第二天是週日。李杰一直睡到了正午才醒來。

**旁白**：陽光透過窗簾的縫隙照進房間，在地板上投下細長的光束。空氣中飄散著淡淡的陽光氣味。

**旁白**：他慢悠悠地起床，走進浴室。刷牙、洗臉，看著鏡子裡熟悉的自己。

**李杰**：真的回來了...這不是夢。

**旁白**：大腦還處於一種慵懶的、不願思考的狀態。昨晚的睡眠讓他的身體得到了充分的休息，但關於那個世界的記憶依然清晰。

**旁白**：他走回臥室，拿起放在床頭櫃上的手機。

**旁白**：螢幕上顯示著幾條未讀訊息。他習慣性地點開了通訊軟體CLINE。

**旁白**：一條來自陌生人的好友申請彈了出來，附帶著一條訊息。

**旁白**：申請人的ID是「Shimizu_K」。

**旁白**：李杰點開了訊息內容。

---

**訊息**：你好，我在3ch論壇上看到了你的留言。我是帖子的發文者，清水。你說你知道一些關於二十年前事故的情報，是真的嗎？

---

**旁白**：李杰的心跳漏了一拍，睡意全無。

**李杰**：來了...真的有人聯繫我了。

**旁白**：他立刻通過了好友申請，然後快速地打字回覆。

**李杰**：是的。事情有些複雜，不方便在網路上說。如果可以的話，我們能見面談談嗎？

**旁白**：訊息發送出去後，他盯著螢幕，等待著回覆。

**旁白**：幾乎是瞬間，對方的回覆就來了。

**清水**：當然可以！你什麼時候方便？在哪裡見？

**旁白**：李杰想了想。今天是週日，他有一整天的時間。

**李杰**：今晚如何？我正好想去秋葉原逛逛，我們可以約在那裡。

**清水**：沒問題。那就約晚上七點，在秋葉原站的電器街口吧。我會穿一件黑色的風衣，戴著眼鏡。

**李杰**：好，到時候見。

**旁白**：敲定了會面時間和地點後，李杰放下手機，坐在床邊。

**李杰**：清水...二十年前失蹤的人，會和我經歷的事情有關嗎？

**旁白**：他感到自己的人生似乎被推上了一條全新的軌道。一條充滿未知和危險的軌道。

**李杰**：算了，先別想那麼多。距離約定的時間還早，去秋葉原逛逛吧。

**旁白**：他換好衣服，拿上錢包和手機，走出了公寓。

**旁白**：樓下的街道沐浴在週日午後的陽光中。附近的公園裡有小孩在玩耍，便利商店的自動門不時開關。

**旁白**：一切都那麼平靜而日常。

**旁白**：他走進家附近那家熟悉的松屋，點了一份牛肉飯套餐。

**旁白**：店裡客人不多，只有幾個看起來剛加完班的上班族，還有一對年輕情侶。

**旁白**：熱騰騰的牛肉飯端上來，醬汁的香氣撲鼻而來。他大口大口地吃著，感受著這份屬於現實世界的美味。

**李杰**：真好吃...果然還是現實世界的食物最棒。

**旁白**：吃完飯後，他走到最近的車站，搭上了前往秋葉原的總武線列車。

**旁白**：列車在午後的陽光中行駛，穿過一個個熟悉的車站。窗外的景色不斷變化——住宅區、商業區、高樓大廈。

**旁白**：大約三十分鐘後，列車抵達了秋葉原站。

**旁白**：他隨著人流走出車站，立刻被撲面而來的喧鬧氛圍所包圍。

**旁白**：這就是秋葉原——日本御宅文化的聖地。

**旁白**：街道兩旁是高聳的電器商城和動漫周邊店，巨大的廣告牌播放著遊戲和動畫的宣傳影片。

**旁白**：街上到處都是扛著購物袋的觀光客、穿著女僕裝發傳單的店員、還有沉浸在自己世界裡的御宅族。

**李杰**：還是老樣子...這裡永遠都這麼熱鬧。

**旁白**：下午的時光，他讓自己完全沉浸在秋葉原那獨有的、喧鬧而充滿活力的氛圍中。

**旁白**：他走進一家女僕咖啡廳。門口站著兩位穿著粉色女僕裝的店員，正在向路人發放傳單。

**女僕A**：歡迎光臨～主人要不要進來坐坐呀～

**旁白**：李杰點點頭，推開了咖啡廳的門。

**旁白**：店內是一片夢幻的粉紅色世界。牆上貼著各種可愛的裝飾，桌上擺著毛絨玩具。

**女僕B**：歡迎回來，主人～請問主人是一位嗎～？

**李杰**：是的，一個人。

**女僕B**：好的～請主人跟我來～

**旁白**：他被領到一張靠窗的小桌子。女僕遞給他一本菜單，上面都是些裝飾得很可愛的食物照片。

**旁白**：他隨意點了一份蛋包飯。

**旁白**：幾分鐘後，女僕端著一個盤子走了過來。盤子裡是一份用番茄醬畫著小熊圖案的蛋包飯。

**女僕B**：主人，這是您點的萌萌小熊蛋包飯～

**旁白**：女僕露出甜美的笑容，從圍裙口袋裡掏出一根小魔法棒。

**女僕B**：接下來要為您施展讓它變得美味的魔法哦，請跟我一起念：『萌え萌え、きゅん！』

**旁白**：李杰有些尷尬地看著周圍。其他桌的客人都在認真地配合著女僕的「魔法儀式」。

**李杰**：...『萌え萌え、きゅん。』

**旁白**：他小聲地跟著念了一遍，聲音幾乎聽不見。

**女僕B**：太棒了～主人的魔法施展成功了～請慢用～

**旁白**：女僕開心地鞠了一躬，然後去服務其他客人了。

**李杰**：真是...和平到讓人有點不習慣啊。

**旁白**：他看著眼前這份可愛的蛋包飯，想起了在異世界那些生死一線的時刻。

**旁白**：那裡沒有女僕咖啡廳，沒有蛋包飯，只有死亡和恐懼。

**旁白**：而現在，他坐在這裡，被可愛的氛圍包圍，享受著和平的午後時光。

**旁白**：這種對比讓他感到一陣恍惚。

**旁白**：他拿起叉子，吃了一口蛋包飯。柔軟的雞蛋、溫熱的飯粒、酸甜的番茄醬——味道很普通，但卻讓他感到安心。

**旁白**：離開咖啡廳後，他又鑽進一家大型的街機中心。

**旁白**：自動門打開的瞬間，震耳欲聾的音樂、遊戲音效和玩家的喊叫聲立刻將他包圍。

**旁白**：店內充斥著各種聲音——格鬥遊戲的出招音效、音樂遊戲的節奏、夾娃娃機的音樂、還有玩家們興奮或懊惱的喊叫。

**旁白**：閃爍的螢幕、旋轉的燈光、此起彼伏的歡呼聲，構成了一個充滿活力的混亂世界。

**李杰**：這裡還是老樣子...吵得要命。

**旁白**：他走到格鬥遊戲區，那裡聚集著最多的玩家。有人正在激烈地對戰，旁邊圍著一圈觀眾。

**旁白**：李杰找了一台空著的機台，投下硬幣。

**旁白**：遊戲開始的音樂響起，螢幕上出現了角色選擇畫面。他熟練地選擇了自己慣用的角色。

**旁白**：他的手指在搖桿和按鈕上飛舞，操作著角色進行著一場又一場的戰鬥。

**旁白**：連招、防禦、反擊——這些動作已經刻進了他的肌肉記憶中。

**路人**：哇！大哥你的連招好厲害啊！

**旁白**：一個看起來像是高中生的男孩，不知道什麼時候站在了他身後，正驚訝地看著螢幕。

**李杰**：...謝謝。

**旁白**：他只是點點頭，繼續著遊戲。螢幕上，他的角色正在進行著最後一場Boss戰。

**路人**：還要再來一局嗎？要不要對戰一場？

**旁白**：李杰打完最後一關後，搖了搖頭。

**李杰**：啊，不了，我差不多該走了。

**路人**：這樣啊...可惜了，真想跟你學學。

**李杰**：曾幾何時，我也和朋友這樣在街機廳泡一個下午...感覺像是上輩子的事了。

**旁白**：他看了一眼手機上的時間——下午六點半。

**李杰**：差不多該去約定的地點了。

**旁白**：夕陽西下，秋葉原的霓虹燈逐一點亮。

**旁白**：巨大的電子廣告牌開始播放絢麗的影像，將天空染成了賽博龐克般的色彩——藍色、紫色、粉紅色，交織在一起。

**旁白**：街道上的人流依然熙熙攘攘，但氛圍從白天的喧鬧變成了夜晚的炫目。

**旁白**：李杰深吸一口氣，動身前往約定好的地點。

**旁白**：秋葉原站的電器街口，是這個區域最顯眼的地標之一。一塊巨大的廣告牌高懸在上方，播放著最新遊戲的宣傳影片。

**旁白**：他在人群中尋找著那個約定好的身影——穿黑色風衣、戴眼鏡的男人。

**旁白**：很快，他就看到了。

**旁白**：在廣告牌下方，一個大約三十五歲左右的男人正站在那裡。

**旁白**：他身形高瘦，穿著約定好的黑色風衣，戴著一副黑框眼鏡。他的表情有些不安，視線不停地掃視著來往的人群，像是在尋找什麼。

**李杰**：就是他了。

**旁白**：李杰深吸一口氣，整理了一下思緒。

**旁白**：他知道，接下來的對話可能會改變很多事情。

**旁白**：他邁開腳步，朝著那個男人走了過去。
