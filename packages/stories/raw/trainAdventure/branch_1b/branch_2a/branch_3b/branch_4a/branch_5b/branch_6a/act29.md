# 第二十九幕：時間的裂縫

**旁白**：李杰坐在閱覽器前,心臟劇烈地跳動著。

**旁白**：那張泛黃的照片在螢幕上靜靜地凝視著他,彷彿在訴說著什麼不為人知的秘密。

**旁白**：他深吸一口氣,拿出手機,撥通了清水浩介的視訊電話。

**旁白**：電話響了幾聲後接通。螢幕上出現清水浩介焦急的臉龐。

**清水浩介**：喂,李先生?這麼快就...有什麼發現嗎?

**旁白**：李杰沒有立刻回答。他的手微微顫抖著,將手機鏡頭小心地對準了縮微膠卷閱覽器的螢幕。

**旁白**：螢幕上,那張模糊的黑白照片被放大了。

**李杰**：清水先生,你看一下這個。

**旁白**：視訊電話的另一端,傳來清水浩介輕微的呼吸聲。

**旁白**：他似乎在仔細地辨認著那張照片。空氣彷彿凝固了幾秒。

**清水浩介**：是的...沒錯,這就是我姐姐恵美二十年前的照片。

**旁白**：他的聲音沉靜而帶著一絲傷感,像是在觸碰一個塵封已久的回憶。

**旁白**：李杰沉默了幾秒。他知道接下來要說的話,將會徹底改變眼前這個男人對過去的認知。

**李杰**：不...清水先生。我在那輛列車上看到的她,就是這個樣子。

**清水浩介**：你說什麼?這不可能...你確定你看清楚了嗎?那可是二十年前的樣子!

**旁白**：這句話像一顆石子投入平靜的湖面,激起了巨大的漣漪。

**旁白**：清水浩介的聲音突然變得尖銳,帶著不可置信的震驚。

**李杰**：我非常確定,就是同一個人。請問...你姐姐是個什麼樣的人?

**旁白**：清水浩介沉默了片刻,似乎在整理自己的思緒。

**清水浩介**：恵美她...性格很內向,有點膽小怕生,但其實很溫柔。

**旁白**：膽小、文靜...這完全吻合了李杰對那個女孩的印象。

**旁白**：就在這時,一個被他忽略的線索,如同一道閃電般擊中了他的大腦。

**旁白**：那個在曦光之庭遇到的精靈——伊蘭尼爾。

**旁白**：他說過,列車是每「兩年」來一次。但車站的時刻表上卻明明白白地寫著「二十年」。

**旁白**：當時李杰以為這只是曆法定義的不同,畢竟那是一個完全不同的世界。

**旁白**：但如果...如果不是定義不同,而是時間流速真的不一樣呢?

**旁白**：李杰的眼睛驟然睜大,冷汗從額頭滲了出來。

**旁白**：異世界的兩年,正好等於這裡的二十年...時間流速是一比十!

**旁白**：這樣的話,恵美在二十年前進入那個世界,對她而言,其實只過去了兩年!

**旁白**：所以她的外表才沒有任何變化!一切都說得通了!

**旁白**：這個發現讓他感到一陣戰慄,彷彿窺見了世界運行規則背後的某種真相。

**旁白**：但同時,這也解開了最大的謎團。

**旁白**：李杰抬起頭,對著手機,語氣變得無比堅定。

**李杰**：清水先生...我在那個世界,看到過一些關於列車運行的資訊,顯示它很快就會再次出現。

**李杰**：如果一切順利,這個週六的深夜,我們就能搭上那班車。你...也許就能見到你的姐姐了。

**旁白**：視訊電話的另一端,清水浩介激動得說不出話。

**旁白**：他的嘴唇顫抖著,眼眶微微泛紅,只是用力地點頭。

**清水浩介**：...我去。我一定要去!李先生,我們在哪裡見?

**旁白**：他的聲音帶著顫抖,像是一個溺水的人終於抓住了救命的繩索。

**李杰**：週六晚上十一點半,新宿車站西口。不見不散。

**清水浩介**：好,我一定到。不見不散。

**旁白**：通話結束後,李杰長長地嘆了口氣。

**旁白**：他將手機放在桌上,閉上眼睛,讓自己的心跳慢慢平復下來。

**旁白**：片刻後,他拿出手機,仔細地拍下了那則尋人啟事。

**旁白**：他又繼續翻閱了前後幾天的報紙。

**旁白**：確實又找到了幾則零星的失蹤報導,但無論是照片還是姓名,都與他記憶中的任何一名乘客對不上。

**旁白**：就好像其他乘客從未在這個世界留下過任何痕跡一樣。

**旁白**：他將縮微膠卷小心地取下,還給了圖書館員。

**圖書館員**：找到了嗎?你說的那個「未被報導的事故」?

**旁白**：圖書館員抬起頭,眼神中帶著好奇。

**李杰**：沒有...什麼都沒找到。大概是我搞錯了吧。謝謝你的幫忙。

**旁白**：他轉身離開圖書館,踏上了回家的路。

**旁白**：夕陽西下,天空染上了一層橘紅色。街道上的人群匆匆而過,每個人都沉浸在自己的世界裡。

**旁白**：而李杰的腦中,兩個巨大的謎團正在盤旋。

**旁白**：第一個謎團——清水恵美。

**旁白**：為什麼上週日晚上的電車裡,他也看到了她?如果她二十年前就去了那個世界,這意味著她曾經回來過?

**旁白**：那她為什麼又要再次搭上那輛列車?

**旁白**：這點,週六見到清水先生時,或許可以問問他,最近有沒有接到過來自姐姐的、任何形式的聯絡。

**旁白**：第二個謎團——失蹤的報導。

**旁白**：為什麼沒有任何關於列車失蹤的新聞?就算是分散的尋人啟事,熱度也低得不正常。

**旁白**：彷彿他們的失蹤,就跟每天都會發生的普通失蹤案一樣,沒有引起任何波瀾。

**旁白**：這背後...到底隱藏著什麼?

**旁白**：李杰加快了腳步,在暮色中消失在人群之中。
