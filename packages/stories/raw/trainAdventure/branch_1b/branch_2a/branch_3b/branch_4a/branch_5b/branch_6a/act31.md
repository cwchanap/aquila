# 第三十一幕：月台上的重逢

**旁白**：清水浩介看到李杰,遠遠地就朝他揮了揮手。

**旁白**：李杰快步走了過去。

**李杰**：抱歉,我沒遲到吧?

**清水浩介**：沒有,是我來早了。反正閒著也是閒著,不如早點來等。

**旁白**：李杰注意到,浩介背著一個幾乎和他上半身一樣大的登山背包,鼓鼓囊囊的。

**旁白**：裡面塞滿了補給、工具、食物和能量飲料,甚至還有一把閃著寒光的折疊小刀插在側袋。

**李杰**：你帶了這麼多東西啊...

**清水浩介**：畢竟要去那種地方,多準備一點比較安心。你說的那些...怪物,真的很危險嗎?

**李杰**：...很危險。不過那個世界裡有一間可以補充物資的小屋。

**清水浩介**：是嗎?那裡還有補給點?

**旁白**：浩介露出有些驚訝的表情。

**清水浩介**：早知道我就不背這麼多了。不過也好,有備無患總是對的。

**李杰**：帶刀是對的。那個世界...確實需要防身武器。

**李杰**：(內心) 我刻意沒有告訴他,今晚的月台上...我們就會見到他18歲時的姐姐。

**李杰**：(內心) 我從沒跟他提過時間旅行的事,到時候恵美小姐不認識我,浩介肯定會起疑...

**李杰**：(內心) 算了,到時候再隨機應變吧。

**旁白**：距離末班車還有一些時間,他們走進車站附近一家24小時營業的連鎖咖啡店。

**旁白**：店內溫暖明亮,和即將前往的黑暗世界形成了鮮明對比。

**店員**：歡迎光臨,兩位客人需要點什麼?

**李杰**：兩杯特濃的冰黑咖啡,謝謝。

**店員**：好的,兩杯特濃冰黑咖啡。這麼晚了還要提神,兩位很辛苦呢。

**清水浩介**：麻煩給我加一份糖漿。我不太習慣喝純黑咖啡。

**店員**：沒問題!請稍等。

**旁白**：兩人找了個靠窗的位子坐下,看著外面來來往往的人群。

**旁白**：李杰端起咖啡杯,感受著冰涼的觸感。這可能是他最後一次享受這種普通的日常了。

**清水浩介**：說實話,李杰,我現在還是覺得像在做夢一樣。

**李杰**：我理解。我剛回來的那天也是這種感覺,花了很長時間才相信這不是夢。

**清水浩介**：你說...那個世界真的有精靈?還有會說話的神樹?

**李杰**：千真萬確。我本來也不相信,但親眼看到後,就不得不相信了。

**清水浩介**：那...我姐姐她,在那個世界,真的還活著?

**旁白**：浩介的聲音帶著一絲顫抖,像是害怕聽到否定的答案。

**李杰**：我看到她的時候,她很好。和一個看起來很可靠的中年男人在一起。

**清水浩介**：中年男人?

**李杰**：嗯,四五十歲左右,給我的感覺很沉穩,有擔當。你姐姐跟著他,應該不會有危險。

**清水浩介**：...那就好。至少她不是一個人。

**旁白**：浩介低頭看著自己的咖啡,眼眶微微泛紅。

**清水浩介**：你...害怕嗎?回到那個地方。

**李杰**：害怕。但更多的是...想知道答案。你呢?

**清水浩介**：我只想再見她一面,問她一句,這些年...過得好不好。

**旁白**：李杰看著浩介,能感受到他話語中的沉重。

**李杰**：你們...感情很好吧?

**清水浩介**：嗯。雖然她比我大三歲,但個性很溫柔,從小就很照顧我。

**清水浩介**：她失蹤那年,我才十五歲。那天她說要去東京找朋友,讓我別等她吃晚餐...

**清水浩介**：結果就再也沒回來。

**旁白**：浩介的聲音越來越低,像是在回憶那段痛苦的往事。

**清水浩介**：父母一開始還報警尋找,但警察說可能是離家出走,讓我們等等看。

**清水浩介**：等了一個月、一年、五年...最後父母也放棄了。只有我一直相信,她還活著。

**李杰**：...所以你才會每年都在論壇上發帖?

**清水浩介**：對。雖然大家都當我是瘋子,但我不在乎。只要有一絲希望,我就要試試看。

**清水浩介**：沒想到...真的等到了。

**旁白**：浩介抬起頭,眼神中帶著堅定。

**清水浩介**：李先生,謝謝你。如果沒有你,我可能永遠都找不到姐姐。

**李杰**：不用謝我。我只是...剛好知道一些線索而已。

**李杰**：(內心) 而且我還不確定,我們能不能安全地把她帶回來...

**旁白**：兩人安靜地喝完咖啡,提前動身,前往中央線的月台。

**旁白**：深夜的新宿車站依然人來人往,但相比白天已經冷清了許多。

**旁白**：他們來到月台,尋找一個較為安靜的角落站定。

**旁白**：深夜的月台上,人群三三兩兩,都帶著疲憊的神色。

**旁白**：有人低頭滑手機,有人小聲交談,有人靠在柱子上打盹。

**旁白**：月台廣播輕柔地播報著列車即將到站的資訊。

**旁白**：李杰環顧四周,試圖記住每一個人的臉。

**李杰**：(內心) 這些人...今晚會有多少人上那班車?又有多少人能活到最後?

**旁白**：就在兩人靜靜等待時,一個穿著黑色連帽衫的男人,從樓梯上走了下來。

**旁白**：他的步伐沉重而遲緩,似乎很疲憊。頭低得很低,連帽衫的帽子遮住了大半張臉。

**旁白**：李杰的瞳孔一縮。

**李杰**：(內心) 那個身影...!

**旁白**：那個身影,他絕不會認錯。正是那個神秘黑衣男!

**旁白**：他的呼吸不由自主地屏住了,警惕感瞬間拉滿。

**旁白**：然而,更讓他震驚的,是身旁浩介的反應。

**旁白**：清水浩介的眼睛瞪得老大,臉上露出了難以置信的狂喜。

**旁白**：他手中的能量飲料瓶子「哐噹」一聲掉落在地,發出清脆的聲響。

**旁白**：他衝上前幾步,顧不得周圍人的目光。

**清水浩介**：是你嗎,凌墨?這些年你到底去哪了!
