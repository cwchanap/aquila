# 第三十三幕：開演前的舞台

**清水浩介**：凌墨...你和姐姐回來之後,為什麼...不來找我?

**旁白**：浩介的情緒稍微平復,但聲音中仍帶著一絲不解和委屈。

**凌墨**：我們沒臉見你。

**旁白**：凌墨避開了他的視線,聲音低沉。

**凌墨**：恵美一心只想著要回去救人,我們不確定這次能不能活著回來...

**凌墨**：不想讓你再經歷一次失去的痛苦。

**清水浩介**：可是...可是我這二十年都在找你們!如果你們早點告訴我——

**凌墨**：而且,該怎麼解釋我們一點都沒老的事?

**凌墨**：最重要的是,我知道你的個性。如果你知道了這一切,一定會跟我們一起去。

**凌墨**：我不能再把你捲進這種危險裡了。

**清水浩介**：...我明白了。

**旁白**：浩介沉默了片刻,上前一步,輕輕拍了拍凌墨的肩膀。

**清水浩介**：但這次,我不會再讓你們獨自面對了。

**凌墨**：時間差不多了,恵美應該快到了。

**凌墨**：我們必須躲起來,不能讓她發現。

**李杰**：你們躲起來。她不認識我,我像普通乘客一樣待在月台上就好,這樣比較自然。

**凌墨**：好。小心點。如果她發現什麼不對勁,可能會改變計劃。

**李杰**：我會注意的。

**旁白**：凌墨拉著浩介,兩人迅速隱沒在月台柱子的陰影角落裡。

**旁白**：李杰獨自站在月台上,拿出手機假裝在看。

**旁白**：過了一會兒,正如他記憶中的那樣,清水恵美獨自一人來到了月台上。

**旁白**：她穿著淺色的連衣裙,找了個長椅的角落坐下。

**旁白**：她戴著耳機,看似在聽音樂,但目光卻時不時地瞟向李杰這邊。

**李杰**：(內心) 她在觀察我...果然。

**旁白**：李杰假裝沒有注意到,視線完全集中在手機螢幕上。

**旁白**：不久,一陣喧鬧聲由遠及近。

**旁白**：四個大學生嘻嘻哈哈地來到了月台上。

**旁白**：李杰聽著他們的對話,一切都和他的「記憶」分毫不差,鮮活得彷彿就在昨天。

**健談男大生**：不是啊你們聽我說!剛剛那家店真的超誇張的!

**健談男大生**：什麼精釀啤酒專賣店,結果端上來的味道根本就像洗碗水!我跟你們說,我喝了一口差點吐出來!

**健談男大生**：老闆還一臉得意地問我們「如何?很有深度吧?」深度個屁啊!

**活潑女大生**：對對對!我也覺得超難喝!

**活潑女大生**：而且你們有看到老闆那個表情嗎?「這款啤酒可是用比利時進口的啤酒花釀造的」——

**活潑女大生**：結果一杯要八百円!我們在全家買的罐裝酒都比那個有感覺啊!

**多話女大生**：對啊對啊!而且啊,你們有注意到嗎?那家店的椅子超不舒服的,坐起來腰超酸!

**多話女大生**：還有啊,廁所也超小間的,我進去的時候差點轉不過身!

**多話女大生**：然後店員的態度也很怪,一直用那種很高傲的眼神看我們,好像我們不懂欣賞一樣...

**健談男大生**：對吧對吧!我就說那種店根本就是騙錢的!

**健談男大生**：什麼「精釀文化」「品味生活」,說穿了就是賣貴而已!下次絕對不去了!

**健談男大生**：我寧願去我們學校附近那家居酒屋,至少份量實在!

**沉穩男大生**：其實...也不能這麼說吧。精釀啤酒確實有它的市場,只是那家店可能不太適合我們。

**沉穩男大生**：下次去我推薦的那家試試看?我之前帶學妹去過,氣氛不錯,價格也合理。

**健談男大生**：喔?你說的是哪家?是上次你在LINE群組裡分享的那家嗎?

**活潑女大生**：對耶!我也想知道!是不是涉谷那邊的?

**多話女大生**：啊我知道我知道!是不是那家有超大投影螢幕的?然後還可以點歌的那家?

**多話女大生**：我朋友之前去過說超好玩的!而且聽說他們的雞翅超好吃,一份才五百円!比剛剛那家便宜多了!

**李杰**：(內心) 來了...那個膽小眼鏡男。

**旁白**：李杰看到,一個戴著無框眼鏡、穿著精心搭配休閒西裝的年輕男子悄悄靠近了那群大學生。

**旁白**：他刻意咳了兩聲,清了清喉嚨。

**膽小眼鏡男**：各位...不好意思打擾一下。

**膽小眼鏡男**：聽你們的對話,就知道你們還很年輕啊。

**旁白**：他用一種自以為成熟穩重的語氣插話。

**膽小眼鏡男**：真正的好店,可不是隨便在Instagram或Google上就能找到的。

**膽小眼鏡男**：像我這種經常跑夜店、對東京夜生活很熟的人都知道...

**膽小眼鏡男**：真正有品味的地方,都藏在巷子深處。

**膽小眼鏡男**：下次有機會的話,我可以帶你們去一家只有行家才知道的店。那裡的氣氛...絕對是你們想像不到的等級。

**旁白**：李杰看著那四個大學生同時轉過頭,眼神裡閃過一絲錯愕與不耐。

**旁白**：活潑女大生和多話女大生交換了一個眼神,嘴角抽動了一下,努力憋住笑。

**旁白**：健談男大生愣了一秒,只是禮貌性地點了點頭。

**旁白**：沉穩男大生則面無表情地看了眼鏡男一眼,隨後就把視線移開。

**健談男大生**：啊...是喔,謝謝啊...

**活潑女大生**：欸欸,這誰啊...

**旁白**：她小聲地對同伴說。

**多話女大生**：不知道耶,感覺好奇怪喔,突然冒出來...而且他那個笑容好僵硬...

**旁白**：李杰看著那個膽小眼鏡男尷尬地站在原地,保持著僵硬的微笑。

**旁白**：然而幾秒後,四個大學生已經自顧自地聊起了別的話題,完全把他晾在一旁。

**健談男大生**：欸對了,你說的那家店,我們什麼時候去啊?下週末有空嗎?

**沉穩男大生**：可以啊,我先訂位。

**活潑女大生**：我也要去我也要去!

**多話女大生**：我也是我也是!而且我們可以先去吃飯再去喝酒!

**多話女大生**：我知道附近有一家拉麵店超好吃的,他們的叉燒軟到入口即化...

**旁白**：眼鏡男的笑容逐漸凝固,臉上閃過一絲尷尬與不甘。

**旁白**：他假裝若無其事地拿出手機,低頭滑動著螢幕,拇指卻一動不動。

**李杰**：(內心) 雖然知道他們即將面臨什麼...但看到他們現在這麼有活力的樣子,真好。

**李杰**：(內心) 我一定要救下他們。所有人。

**旁白**：他下意識地瞟了一眼柱子的陰影處。

**旁白**：看到浩介不知何時已經戴上了口罩和帽子,將自己遮得嚴嚴實實。

**李杰**：(內心) 連偽裝道具都準備了嗎...真是周到。

**旁白**：遠處,傳來列車駛近的聲音。

**廣播**：往高尾方向的末班車即將進站,請各位乘客退到黃線後方等候。

**旁白**：李杰深吸一口氣,知道他艱鉅的任務,即將開始。

**旁白**：列車的車頭燈在隧道口出現,越來越近。

**旁白**：列車駛入月台,車門打開。

**旁白**：李杰隨著人流走進車廂,找了一個靠窗的座位。

**旁白**：他感覺到清水恵美的視線,正若有若無地落在他身上。

**旁白**：一股寒意突然竄起。

**李杰**：(內心) 她要去救男友,需要一個犧牲品...

**李杰**：(內心) 她一直在觀察我...難道,她的目標...是我?

**旁白**：基於這個猜測,他刻意沒有選擇靠近凌墨和浩介預計會上車的車廂門口。

**旁白**：他坐在另一節車廂,與他們保持距離。

**廣播**：車門即將關閉,請注意。

**旁白**：車門緩緩關閉,隔絕了月台上的世界。

**旁白**：李杰看著窗外倒退的燈火,知道旅程再次開始。

**旁白**：但他絕對無法想像,因為他的介入和凌墨的存在——

**旁白**：接下來發生的事,將會徹底顛覆他的所有認知與計畫。
