# 第三十六幕：失序的拯救

**旁白**：李杰沒有時間呆立,他立刻衝到凌墨和恵美身邊,跪下查看恵美的傷勢。

**旁白**：一把看似普通的水果刀,刀柄還露在外面,但刀刃幾乎完全沒入了腹部。

**旁白**：鮮血正不斷地從傷口與刀柄的縫隙中湧出,地板已經被染成一片暗紅。

**凌墨**：李杰,怎麼辦?

**旁白**：凌墨扶著恵美,聲音因極力壓抑而顫抖。

**李杰**：不能拔刀,會大出血。必須先壓迫止血...

**旁白**：李杰檢查傷口,語氣沉重。

**李杰**：(內心) 傷得很深...這個位置,可能傷到了內臟...

**旁白**：車廂的另一頭,慷慨大叔和沉穩男大生正費力地壓制著還在瘋狂掙扎的多話女大生。

**慷慨大叔**：快!壓住她!她瘋了!

**沉穩男大生**：美咲!妳看清楚!妳在做什麼!快住手!

**旁白**：沉穩男大生幫忙壓制,對著多話女大生低吼。

**旁白**：兩人將還在咒罵尖叫的多話女大生拖到角落捆了起來,又用一塊布堵住了她的嘴。

**旁白**：多話女大生的眼神依然瘋狂,身體不停地扭動掙扎。

**凌墨**：需要繃帶...什麼都好,有能止血的東西嗎?

**旁白**：凌墨看著恵美越來越虛弱,臉色蒼白得嚇人。

**李杰**：(內心) 可惡...浩介那個幾乎裝滿了戶外用品的背包裡,肯定有完整的急救箱...

**李杰**：(內心) 但他卻偏偏消失了!

**李杰**：我這裡只有一些簡單的東西,先頂著用!

**旁白**：李杰立刻從自己的背包裡拿出僅有的一小卷紗布和消毒噴霧。

**旁白**：他試圖按住傷口周圍止血。

**旁白**：但紗布一接觸到傷口,立刻就被鮮血染透,根本無濟於事。

**李杰**：(內心) 該死...完全止不住!

**凌墨**：誰有乾淨的毛巾或急救包?拿出來。

**旁白**：凌墨抬起頭,用冰冷的目光掃視其他乘客。

**旁白**：他的聲音不大但充滿了不容置疑的威嚴。

**旁白**：然而,其他的乘客都只是驚恐地縮在車廂的另一頭,紛紛搖頭。

**旁白**：沒有人敢上前。

**煩躁丈夫**：我、我們什麼都沒有...

**焦慮妻子**：對不起...我們真的沒有...

**旁白**：處理完多話女大生後,慷慨大叔和沉穩男大生走了過來。

**慷慨大叔**：對不起...真的對不起。我們...我們應該早點注意到那個女孩的不對勁...

**旁白**：慷慨大叔看著恵美,滿臉自責。

**沉穩男大生**：是我...都怪我。我不該...

**旁白**：沉穩男大生的表情痛苦,聲音顫抖著。

**李杰**：(內心) 他這話是什麼意思?他不該什麼?

**李杰**：(內心) 但現在根本不是追問的時候...

**李杰**：這樣下去不行,她失血太多了。

**李杰**：你先在這裡繼續壓著傷口,我去車站裡看看,說不定能找到醫療箱!

**凌墨**：好!拜託你了!

**旁白**：凌墨用力點頭,手上繼續按著已經被血染透的紗布。

**旁白**：李杰衝下列車,飛奔進「寂海」站那破舊的車站建築。

**旁白**：月光從屋頂的破洞中灑下,照亮了飛舞的塵埃。

**旁白**：他在剝落的牆壁上,發現了那張褪色的老舊地圖。

**李杰**：(內心) 這是...那個小屋的位置!

**旁白**：地圖上標示著森林深處確實有一棟小屋的圖案,但沒有任何比例尺。

**李杰**：(內心) 距離多遠?要走多久?該死,什麼資訊都沒有!

**旁白**：他又發瘋似地翻遍了售票處和辦公室。

**旁白**：櫃子、抽屜、架子...任何可能藏東西的地方。

**旁白**：但裡面除了腐朽的木頭和垃圾,什麼都沒有。

**李杰**：(內心) 沒有...什麼都沒有!

**旁白**：他只能立刻衝回列車。

**旁白**：然而,當他再次踏入第一節車廂時,車廂內詭異的寂靜讓他停住了腳步。

**旁白**：那把水果刀還插在恵美的腹部。

**旁白**：而凌墨手中,卻握著一把不同的刀——

**旁白**：一把閃著寒光的黑色軍用匕首,刀刃上同樣沾著血。

**旁白**：在他面前,那個沉穩男大生正捂著自己同樣在流血的腹部,滿臉痛苦地跪倒在地。

**旁白**：其他的乘客更加驚恐地縮在車廂的盡頭,鴉雀無聲。

**李杰**：(內心) 這...這是怎麼回事?!

**旁白**：慷慨大叔死死地抓住凌墨的手腕,對著他大吼。

**慷慨大叔**：住手!你瘋了嗎?!快把刀放下!

**旁白**：凌墨的眼神空洞而冰冷,彷彿失去了理智。
