# 第四十四幕：盲目的追尋

**旁白**：又在濃霧中急行了一個多小時後，李杰的雙腿已經開始發軟。

**旁白**：但當他終於從森林中衝出，看到了「幽冥之都」那熟悉的、死寂的輪廓時——

**旁白**：一股希望在胸中燃起。

**李杰**：（內心）終於到了！山本先生一定在這裡！

**旁白**：他沒有絲毫停歇，立刻飛奔進車站建築，開始了地毯式的搜索。

**旁白**：他踹開洗手間的門，門板撞在牆上發出巨響。裡面空無一人，只有破碎的鏡子反射著手電筒的光。

**旁白**：他跑進候車大廳，只有自己的腳步聲在空曠的空間裡迴盪，一聲又一聲，空洞而淒涼。

**旁白**：他搜遍了售票處、儲藏室、辦公室，每一個可能藏人的角落。

**旁白**：最終，他衝上了月台。

**旁白**：月台上空蕩蕩的，只有霧氣在流動。鐵軌在月光下泛著冰冷的銀光，延伸向遠方的黑暗。

**旁白**：整座車站，除了他和無盡的霧氣，再沒有任何活物的蹤跡。

**李杰**：山本大叔！山本大叔！

**旁白**：他大喊著，聲音在空蕩的月台上迴盪，但沒有任何回應。

**旁白**：只有風吹過時，車站建築發出的吱呀聲。

**李杰**：（內心）怎麼會...一個人也沒有？

**旁白**：李杰扶著膝蓋，大口地喘氣，胸口劇烈起伏。

**李杰**：（內心）山本先生不在這裡...按照我的計畫，這一站應該會下來四個人才對，他們也都不在。

**李杰**：（內心）難道他們已經離開了？還是...

**旁白**：一個可怕的想法閃過腦海，但他立刻甩開了它。

**旁白**：就在這時，一個關鍵的細節閃過他的腦海。

**李杰**：（內心）等等...地圖！我上次是在站長室找到地圖的！

**旁白**：他立刻轉身，衝回大廳，找到了那間半開著門的站長室。

**旁白**：門吱呀一聲被推開，裡面滿是灰塵的味道。

**旁白**：當他藉著手電筒的光照向那面牆時——

**旁白**：他的心瞬間沉了下去。

**旁白**：牆上那片泛黃的區域...如今只剩下被撕扯後留下的、參差不齊的白色牆紙。

**旁白**：地圖不見了。

**李杰**：（內心）有人來過這裡！

**旁白**：李杰快步走上前，用手摸了摸那片被撕扯的痕跡。

**旁白**：撕痕看起來很新，邊緣還沒有積灰。

**李杰**：（內心）會是誰？山本先生...一定是他！

**李杰**：（內心）以山本先生的個性，如果他下了車，一定會像我一樣，先把整個車站徹底搜索一遍。

**李杰**：（內心）他絕對會發現這張地圖...一定是他拿走的。

**旁白**：這個發現讓他鬆了一口氣，同時也意識到，情況比他想像的要複雜得多。

**旁白**：山本先生，或者其他從列車上下來的人，顯然已經來過這裡，並且帶著新的情報離開了。

**李杰**：（內心）我現在的首要目標是找到山本先生。

**李杰**：（內心）既然他不在我剛才走來的那條路上，也不在這座車站裡，那他一定是從車站外的那個岔路口，選擇了另外兩條路之一。

**旁白**：一個艱難的抉擇擺在了他的面前。

**旁白**：他閉上眼睛，努力回憶起那張已經消失的地圖上的內容。

**旁白**：三條路...中間是他來時的路，通往森林深處的小屋和月光林地。

**李杰**：（內心）一條路，是往北，通往我上次也去過的「亡者峽谷」。

**李杰**：（內心）另一條路，是往西，通往那個完全未知的、名為「銀月湖」的地方。

**旁白**：李杰緊緊握著拳頭，指甲掐進了手心。

**李杰**：（內心）山本先生會選擇哪一條？

**李杰**：（內心）他會選擇跟隨列車的主線，還是去探索一個聽起來相對安全的地方？

**旁白**：他站在廢棄車站的出口，前方是通往未知命運的岔路口。

**旁白**：濃霧籠罩著一切，看不到任何盡頭。

**旁白**：風吹過，樹葉發出沙沙的聲音，像是在催促他做出決定。

**旁白**：他必須賭一把。

**旁白**：但這一次，如果選錯了，可能就再也找不到山本浩二了。
