# 第四十七幕：血染的村莊

**旁白**：李杰一瘸一拐地走在前往月光林地的主要道路上。

**旁白**：幸運的是，這條路不像之前那些小徑那麼崎嶇，路面相對平坦，走起來容易一些。

**旁白**：但腳踝的傷讓他無法走快，每走一段路就必須停下來休息。

**旁白**：他靠著樹幹坐下，喘著粗氣，揉著腫脹的腳踝。

**旁白**：休息片刻後，又咬著牙繼續前進。

**旁白**：走走停停，時間一點一滴地流逝。

**旁白**：天空中的雙月依舊高懸，但位置似乎移動了不少。

**旁白**：濃霧時而濃厚，時而稀薄，但始終籠罩著這片森林。

**旁白**：李杰不知道自己走了多久，只覺得每一步都格外漫長。

**旁白**：終於，在走了將近十個小時後——

**旁白**：前方的霧氣開始散去，那些散發著柔和光芒的植物再次映入眼簾。

**旁白**：月光林地到了。

**李杰**：（內心）終於...到了...

**旁白**：但當他真正看清眼前的景象時，整個人僵在了原地。

**旁白**：村莊入口處，到處都是戰鬥的痕跡。

**旁白**：地面上留著深深的劍痕和焦黑的痕跡，吊橋的繩索被砍斷，搖搖欲墜。

**旁白**：空氣中瀰漫著濃重的血腥味，混雜著燒焦的味道。

**旁白**：李杰的心臟猛地一縮。

**李杰**：（內心）這...這是怎麼回事？！

**旁白**：他的目光掃過村莊入口，然後——

**旁白**：看到了一個熟悉的身影，倒在地上。

**旁白**：那是山本浩二。

**李杰**：山本大叔！

**旁白**：李杰忘記了腳上的傷痛，跌跌撞撞地衝了過去，跪倒在山本浩二身邊。

**旁白**：山本浩二的身上，佈滿了大大小小的劍傷。

**旁白**：胸口、肩膀、手臂...至少有五六處深深的傷口，鮮血已經凝固，將他的衣服染成暗紅色。

**旁白**：他的臉色蒼白如紙，眼睛緊閉，已經沒有了呼吸。

**旁白**：但他的手中，還緊緊握著那把折疊刀。

**李杰**：山本大叔...山本大叔...

**旁白**：李杰伸出顫抖的手，想去探他的脈搏，但手指剛碰到冰冷的皮膚，就知道一切都晚了。

**旁白**：他緩緩放下手，喉嚨像被什麼東西堵住了，說不出話來。

**旁白**：深吸了一口氣，他強迫自己站起來，繼續往村莊裡走。

**旁白**：腳步沉重得像灌了鉛。

**旁白**：越往裡走，景象越是慘烈。

**旁白**：樹屋的階梯上躺著一個精靈劍士。

**旁白**：那是凱蘭——之前護送艾拉如長老來治療恵美的劍士。

**旁白**：他金色的短髮沾滿了血污，深綠色的皮甲被刺穿，手中的長劍斷成兩截。

**旁白**：他的眼睛還睜著，眼神中殘留著憤怒和不甘。

**旁白**：廣場上，幾個精靈平民倒在血泊中，他們的眼睛還睜著，臉上殘留著驚恐的表情。

**旁白**：木屋的牆壁上濺滿了血跡，門板被劈開，裡面一片狼藉。

**旁白**：到處都是屍體，到處都是鮮血。

**旁白**：李杰的腿在發抖，但他強迫自己繼續走。

**旁白**：然後，他看到了田中健太和佐藤拓海。

**旁白**：兩人倒在一棵大樹下，相距不到兩米。

**旁白**：與山本浩二身上密密麻麻的傷口不同，他們身上的傷口很少。

**旁白**：田中健太的胸口有一道筆直的劍痕，從左肩貫穿到右腰，深可見骨。

**旁白**：佐藤拓海的頸部有一道致命的橫切，幾乎將頭顱斬斷。

**旁白**：一擊斃命。

**旁白**：他們甚至來不及反應，就被殺死了。

**李杰**：（內心）是誰...是誰做的...

**旁白**：李杰緊緊握著拳頭，指甲深深掐進掌心。

**旁白**：他繼續往村莊中心走去。

**旁白**：在那裡，他看到了凌墨。

**旁白**：凌墨躺在村莊中心的廣場上，身下是一大片刺眼的血跡。

**旁白**：他的胸口、腹部，至少有三四處深深的傷口，鮮血還在緩慢地流出。

**旁白**：而恵美，正跪在他身邊，緊緊握著他的手。

**旁白**：她的臉上滿是淚痕，整個人都在顫抖。

**李杰**：凌墨！

**旁白**：李杰不顧一切地衝了過去，跌跌撞撞地跪倒在凌墨身邊。

**李杰**：凌墨！發生什麼事了？！是誰做的？！

**旁白**：凌墨緩緩睜開眼睛，眼神已經開始渙散。

**旁白**：他沒有回答李杰的問題，只是微微抬起手，想要抓住什麼。

**旁白**：李杰立刻伸出手，讓凌墨握住。

**旁白**：凌墨的手冰冷而無力，但還是用盡最後的力氣，緊緊握著李杰的手。

**旁白**：他的嘴唇微微動了動，似乎想說什麼。

**李杰**：凌墨，你說，我聽著！

**旁白**：李杰立刻將耳朵湊到凌墨嘴邊。

**旁白**：凌墨用盡全身最後的力氣，用氣若遊絲的聲音，一字一句地說道。

**凌墨**：...請...幫我...保護...恵美...

**旁白**：聲音微弱得幾乎聽不見，但每一個字都清晰地傳進了李杰的耳中。

**李杰**：我答應你！我一定會保護她！

**旁白**：凌墨聽到這句話，嘴角微微上揚，露出一個淡淡的、安心的微笑。

**旁白**：然後，他的手慢慢鬆開，從李杰的手中滑落。

**旁白**：頭微微側向一邊，眼睛緩緩閉上。

**旁白**：胸口最後的起伏，停止了。

**清水恵美**：凌墨...凌墨！不要...不要離開我...

**旁白**：恵美撲在凌墨身上，放聲大哭。

**旁白**：她的哭聲撕心裂肺，在空蕩蕩的村莊中迴盪。

**旁白**：李杰跪在那裡，看著凌墨毫無生氣的臉。

**旁白**：喉嚨像被什麼東西堵住了，呼吸變得困難。

**旁白**：眼眶灼熱，視線變得模糊。

**旁白**：但眼淚，最終沒有落下。

**旁白**：只是手，緊緊地握成了拳頭，指甲深深刺進掌心，刺出了血。

**旁白**：周圍，是一片死寂。

**旁白**：那些曾經發光的植物，現在看起來格外暗淡。

**旁白**：月光灑在這片血染的村莊上，冰冷而蒼白。

**旁白**：這裡，曾經是充滿希望的地方。

**旁白**：而現在，只剩下死亡和絕望。
