# 第五十二幕：死亡列車

**旁白**：李杰小心翼翼地靠近火車，繞到車尾的位置。

**旁白**：從軌道後方，他爬上了月台。

**旁白**：月台上空無一人，地面上鋪滿了灰塵和落葉。

**旁白**：李杰走向火車的最後一節車廂，伸手推開車門。

**旁白**：門發出刺耳的吱呀聲，緩緩打開。

**旁白**：車廂內昏暗，只有頂部的應急燈發出微弱的光。

**旁白**：李杰舉起手電筒，往車廂內照去。

**旁白**：然後——

**旁白**：他看到了一具屍體。

**旁白**：那是一個戴著眼鏡的年輕男子，正是列車上的膽小眼鏡男。

**旁白**：他倒在座位旁邊，雙眼圓睜，臉上還殘留著驚恐的表情。

**旁白**：胸口有一道深深的劍傷，筆直地貫穿了心臟。

**旁白**：鮮血已經凝固，染紅了周圍的地板。

**旁白**：一擊斃命。

**李杰**：（內心）這...這是怎麼回事...

**旁白**：李杰的手在發抖，但他強迫自己冷靜下來。

**旁白**：他深吸了一口氣，繼續往前走。

**旁白**：穿過一節又一節車廂。

**旁白**：第九節、第八節、第七節...都空無一人。

**旁白**：但到處都是打鬥的痕跡。

**旁白**：座椅被劈開，窗戶破碎，牆壁上有深深的劍痕。

**旁白**：空氣中瀰漫著濃重的血腥味。

**旁白**：李杰的心跳越來越快，呼吸越來越急促。

**旁白**：終於，他來到了第一節車廂。

**旁白**：推開車廂門的瞬間——

**旁白**：眼前的景象讓他倒抽了一口冷氣。

**旁白**：車廂裡到處都是屍體。

**旁白**：那個煩躁的丈夫，倒在靠近門口的地方。

**旁白**：他的身上有四五處劍傷，胸口、腹部、手臂...鮮血將衣服完全染紅。

**旁白**：他的臉上還殘留著憤怒和不甘的表情。

**旁白**：而那個焦慮的妻子，倒在他旁邊不遠處。

**旁白**：她的身上同樣有多處傷口，頸部、肩膀、背部...

**旁白**：她的手還伸向那個煩躁的丈夫的方向，似乎想要抓住什麼。

**旁白**：再往裡面——

**旁白**：小林太太，還有那個小男孩。

**旁白**：兩個人倒在車廂中間，相距不遠。

**旁白**：小林太太身上有多處劍傷，頸部、胸口、手臂...

**旁白**：她的眼睛還睜著，眼中滿是悲傷和絕望。

**旁白**：而那個小男孩...小林家那個天真的孩子...

**旁白**：他倒在母親身旁，身上的傷口雖然不多，但每一處都很深。

**旁白**：他的臉上還殘留著驚恐和不解，眼睛緊緊閉著。

**李杰**：小林太太...還有...

**旁白**：李杰的聲音在顫抖，喉嚨發緊。

**旁白**：他緊緊握著拳頭，指甲深深刺進掌心。

**李杰**：（內心）小林先生呢...他的屍體...為什麼不在這裡？

**旁白**：他環顧車廂，沒有看到小林先生的身影。

**旁白**：只有小林太太和孩子的屍體，倒在血泊中。

**李杰**：（內心）他去了哪裡...

**旁白**：就在這時——

**旁白**：他注意到車廂角落還有一個身影。

**旁白**：那是一個女孩，躺在座位下方。

**旁白**：她的手腳被繩子緊緊綁著，嘴巴也被布條塞住。

**旁白**：身上的衣服染滿了血跡。

**李杰**：（內心）那是...

**旁白**：李杰立刻衝了過去，跪在女孩身邊。

**旁白**：是美咲——那個多話的女大生。

**旁白**：她的臉色蒼白，雙眼緊閉，一動不動。

**李杰**：美咲！美咲！

**旁白**：李杰伸手探向她的脖子，尋找脈搏。

**旁白**：手指剛碰到皮膚——

**旁白**：感受到了微弱但清晰的跳動。

**李杰**：（內心）還活著！她還活著！

**旁白**：李杰立刻開始解開綁在她手腳上的繩子。

**旁白**：繩子綁得很緊，勒進了皮膚，留下深深的印記。

**旁白**：他費了好大力氣，才將繩子全部解開。

**旁白**：然後小心翼翼地取下塞在她嘴裡的布條。

**旁白**：美咲依然沒有醒來，但呼吸漸漸變得平穩了一些。

**李杰**：美咲...醒醒...能聽到我說話嗎？

**旁白**：李杰輕輕拍著她的臉頰，試圖喚醒她。

**旁白**：但美咲依然緊閉著雙眼，沒有任何反應。

**旁白**：李杰從背包裡拿出水袋，小心地將水滴在她的嘴唇上。

**旁白**：幾滴水順著嘴角流進口中。

**旁白**：美咲的嘴唇微微動了動，喉嚨發出細微的吞嚥聲。

**李杰**：（內心）她只是失去意識...應該沒有生命危險...

**旁白**：他仔細檢查美咲身上的血跡。

**旁白**：大部分血跡都已經乾涸，而且看起來不像是她自己的血。

**旁白**：應該是在混亂中沾染到的。

**旁白**：她身上沒有明顯的外傷，只有手腕和腳踝被繩子勒出的傷痕。

**李杰**：（內心）其他人都被殺了...但她卻活了下來...

**旁白**：李杰看著周圍那些冰冷的屍體，又看著懷中昏迷的美咲。

**旁白**：心中湧起無數的疑問。

**旁白**：但現在不是思考的時候。

**旁白**：他必須先想辦法讓美咲醒過來。

**旁白**：必須知道...這裡到底發生了什麼。

**旁白**：李杰小心翼翼地將美咲抱起，讓她靠在座位上。

**旁白**：然後從背包裡拿出精靈的藥膏，輕輕塗在她手腕和腳踝的傷口上。

**旁白**：藥膏散發出淡淡的草藥香氣，清涼的觸感讓美咲的眉頭微微皺了一下。

**旁白**：李杰又拿出水袋，小心地將水滴在她的嘴唇上。

**旁白**：一滴、兩滴、三滴...

**旁白**：美咲乾裂的嘴唇開始微微顫動，喉嚨發出細微的吞嚥聲。

**旁白**：李杰繼續餵她水，一點一點地，不敢給得太急。

**旁白**：漸漸地，美咲的呼吸變得平穩了一些，臉色也不再那麼蒼白。

**旁白**：幾分鐘後——

**旁白**：美咲的眼皮開始微微顫動。

**李杰**：美咲...能聽到我說話嗎？

**旁白**：美咲緩緩睜開眼睛，眼神渙散而迷茫。

**旁白**：她看著李杰，似乎還沒有完全恢復意識。

**美咲**：（聲音沙啞而微弱）這...這裡是...

**旁白**：她的聲音幾乎聽不見，喉嚨因為長時間缺水而乾澀。

**李杰**：別急，先喝點水。

**旁白**：李杰再次將水袋遞到她嘴邊，讓她慢慢喝。

**旁白**：美咲貪婪地吸著水，但李杰控制著水量，不讓她喝得太快。

**旁白**：喝了幾口水後，美咲的眼神漸漸清醒了一些。

**旁白**：她環顧四周，看到那些倒下的屍體時，整個人猛地一顫。

**美咲**：（驚恐地）他們...他們...

**旁白**：她的身體開始劇烈顫抖，眼中滿是恐懼。

**旁白**：李杰輕輕按住她的肩膀，試圖讓她冷靜下來。

**李杰**：美咲，冷靜一點。你現在安全了。

**旁白**：美咲看著李杰，眼淚突然湧了出來。

**美咲**：（哭泣著）李先生...李先生...

**旁白**：她緊緊抓住李杰的手臂，像是抓住了唯一的救命稻草。

**旁白**：李杰沒有催促她，只是靜靜地等著她情緒穩定下來。

**旁白**：窗外的濃霧依舊濃厚，雙月的光芒透過霧氣，照在這列死亡的列車上。

**旁白**：而在這片寂靜中，美咲的哭泣聲，是唯一的聲響。
