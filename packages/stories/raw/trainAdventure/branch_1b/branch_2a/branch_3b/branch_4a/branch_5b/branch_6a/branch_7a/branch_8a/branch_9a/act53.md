# 第五十三幕：列車上的悲劇

**旁白**：美咲的哭泣聲漸漸變小,但身體依然在顫抖。李杰靜靜地坐在她身邊,等待著她情緒穩定。

**旁白**：幾分鐘後,美咲深吸了一口氣,用手背擦去眼淚。

**高橋美咲**：你...你是誰...

**李杰**：我叫李杰，我們在列車上見過。別怕，妳現在安全了。

**高橋美咲**：李杰...

**旁白**：美咲似乎想起了什麼，眼神中閃過一絲認出的神色。

**李杰**：美咲，能告訴我發生了什麼事嗎？

**旁白**：美咲低下頭,雙手緊緊握著李杰的手臂。

**旁白**：她的聲音在顫抖,每說一句話都需要很大的勇氣。

**高橋美咲**：我看到...那個中年男人,那個總是一臉陰沉的男人...他和那個戴眼鏡的男人一起離開了車廂...

**旁白**：李杰點了點頭,沒有打斷她。

**李杰**：後來呢？之後發生了什麼？

**高橋美咲**：過了一會兒火車停了,外面的站牌寫著「幽冥之都」...小林先生對太太和兒子說他要去找女兒,然後就下車了,往火車來的方向走...

**李杰**：小林先生下車了...所以他不在這裡...

**高橋美咲**：我們就留在車上等著...

**旁白**：她深吸了一口氣,身體開始顫抖。

**高橋美咲**：過了一會兒...那個中年男人回來了...他手裡拿著一把黑色的長劍...

**李杰**：黑劍...

**旁白**：李杰的腦海中立刻浮現出那把在第十節車廂看到的詛咒之劍。

**高橋美咲**：他看到我,只是看了我一眼,什麼都沒說,然後就轉向其他人...

**旁白**：美咲的身體開始劇烈顫抖,聲音也變得更加恐懼。

**高橋美咲**：大家都嚇壞了,開始尖叫...那個中年男人直接就殺了那對夫婦,煩躁的男人和他太太...就在我眼前,鮮血到處都是...

**旁白**：她用手捂住臉,肩膀不停地抽動。

**李杰**：然後呢?

**高橋美咲**：然後火車門突然關上了,火車開始動了...我完全動不了,只能眼睜睜地看著,什麼都做不了...

**旁白**：李杰看向車廂門口那兩具屍體,心中一沉。

**高橋美咲**：小林太太抱著兒子,一直在求那個男人,說她願意做任何事,只求放過她的孩子...

**旁白**：美咲的聲音變得更加痛苦,幾乎說不下去。

**高橋美咲**：但那個男人他什麼都沒說...他就那樣...

**旁白**：她無法繼續說下去,只是搖著頭,眼淚滂沱而下。

**高橋美咲**：我想尖叫想逃,但是根本做不到...我就那樣看著他殺了所有人...

**旁白**：李杰看著美咲手腕和腳踝上深深的勒痕,拳頭緊緊握起。

**李杰**：我明白了...

**旁白**：李杰看著那兩具倒在血泊中的屍體,又看著美咲。

**李杰**：他為什麼沒有殺妳?明明有機會...

**旁白**：美咲抬起頭,眼神充滿了困惑和恐懼。

**高橋美咲**：我也不知道...也許他覺得我沒有威脅...或者只是想折磨我,讓我看著其他人死去,然後讓我在恐懼中等待...

**旁白**：李杰深吸了一口氣,強迫自己保持冷靜。

**李杰**：之後呢?火車開到這裡後,那個男人做了什麼?

**高橋美咲**：火車一直開...開了很久很久...我不知道過了多久,可能幾個小時...然後火車停了...

**旁白**：美咲深吸了一口氣,繼續說道。

**高橋美咲**：那個男人站起來走向車門,說他要去「征服這個世界」...然後他就下車了...

**李杰**：征服這個世界...這個瘋子...

**高橋美咲**：我以為他走了,我就安全了...但是沒過多久,我聽到外面有打鬥的聲音...

**李杰**：打鬥?

**高橋美咲**：我從座位下面偷偷看向窗外...看到一個穿白袍的男人,手裡拿著一把白色的劍,正在和那個中年男人打鬥...

**李杰**：白袍男人?

**旁白**：李杰皺起眉頭,在腦海中搜尋著。他認識的人裡,沒有穿白袍的。

**高橋美咲**：他們打得很激烈,那個白袍男人的劍法很厲害...但是那把黑劍好像有什麼奇怪的力量,沒過多久白袍男人就輸了...

**旁白**：李杰聚精會神地聽著,不放過任何細節。

**高橋美咲**：他的劍被那把黑劍劈成兩半...然後他的身體突然就消失了,就像煙霧一樣,地上只剩下那把斷成兩截的白劍...

**李杰**：消失了?這到底是怎麼回事...

**高橋美咲**：之後那個中年男人撿起那把斷劍看了一眼,然後就扔在地上...他笑了,笑得很瘋狂,然後就走進森林裡不見了...

**高橋美咲**：我就一直躲在座位下面,不敢動也不敢出聲,好害怕他會回來...我不知道過了多久,好渴好餓...後來我就什麼都不記得了...

**旁白**：美咲說完後,整個人像洩了氣的皮球一樣靠在座位上,緊緊抓住李杰的手臂,像是害怕他會消失。

**旁白**：李杰看著她,心中湧起一股憤怒和無力。

**旁白**：那個拿著黑劍的瘋子殺了這麼多人...雖然他已經失去下半身,在月光林地等待死亡,但造成的傷害卻無法挽回...

**高橋美咲**：李先生...

**旁白**：美咲抬起頭,用那雙充滿恐懼的眼睛看著他。

**旁白**：她環顧四周,看著那些倒在血泊中的屍體,身體開始劇烈顫抖。

**高橋美咲**：李先生...我不想待在這裡...這裡到處都是...到處都是...

**旁白**：她的聲音因恐懼而顫抖,眼淚再次湧出。

**高橋美咲**：求求你...能不能帶我離開這裡...我真的...真的不想再看到他們了...

**旁白**：她緊緊抓住李杰的手臂,像是抓住唯一的救命稻草。

**高橋美咲**：我知道你有很重要的事...但是...我一個人待在這裡...我會瘋掉的...

**旁白**：李杰看著美咲那雙充滿恐懼和絕望的眼睛,心中一緊。

**旁白**：他知道他必須做出選擇。

**旁白**：是立刻前往永恆之森進行時空穿越...

**旁白**：還是先帶美咲離開這裡,確保她的安全...

**旁白**：但看著美咲那雙充滿恐懼的眼睛,看著她因創傷而不停顫抖的身體...

**旁白**：李杰發現...他無法在這一刻拋下她。

**旁白**：窗外的濃霧依舊濃厚,雙月的光芒冰冷地灑在這片死亡之地上。

**旁白**：而李杰,已經做出了他的決定。
