# 第六十四幕：古老的真相

**旁白**：李杰沉默了片刻,然後抬起頭看向瑟蘭。

**李杰**：瑟蘭,我想問你...有沒有辦法救出成為祭品的人?

**旁白**：瑟蘭的表情變得凝重。

**瑟蘭**：據我所知,沒有。那扇門需要人類作為祭品才能保持開啟,這是它的設計原理。

**李杰**：設計原理?

**旁白**：瑟蘭走到窗邊,望向遠方的黑暗。

**瑟蘭**：終焉祭壇和那扇門,並不是我們精靈族建造的。它們是更古老的遺物...來自五萬年前的人類文明。

**旁白**：李杰和莉拉都愣住了。

**莉拉**：五萬年前?人類的...文明?

**瑟蘭**：是的。在我們精靈族來到這個世界之前,這座幽月之島曾是一片戰場。

**旁白**：瑟蘭轉過身,聲音帶著歷史的厚重感。

**瑟蘭**：當時,魔族和人類為了爭奪這座島嶼,展開了漫長的戰爭。而終焉祭壇那座建築,正是雙方都想要佔領的戰略要地。

**李杰**：為什麼?那座建築有什麼特別的?

**瑟蘭**：我也不完全清楚。但據說那座建築本身就是古代文明的遺跡,蘊含著強大的力量。

**旁白**：瑟蘭在地圖上指了指。

**瑟蘭**：最終,人類佔領了那座建築。他們在建築深處建造了祭壇,並創造了那扇連接人類世界的門。

**莉拉**：(驚訝)所以...那扇門是人類創造的?

**瑟蘭**：沒錯。而那個祭壇的機制——需要人類獻祭才能開啟——也是人類自己設計的。

**旁白**：李杰皺起眉頭,思考著這些資訊。

**李杰**：為什麼要這樣設計?為什麼要用人類作為祭品?

**旁白**：瑟蘭看著他。

**瑟蘭**：也許...是為了防止其他種族濫用那扇門吧。

**李杰**：(恍然大悟)防止其他種族...

**瑟蘭**：如果門可以隨意開啟,精靈、魔族、甚至其他種族都能自由往返於人類世界。那對人類來說,無疑是巨大的威脅。

**瑟蘭**：但如果只有人類能夠開啟那扇門,而代價是犧牲同族的生命...那麼,這扇門就不會被輕易打開。

**旁白**：李杰緊緊握著拳頭。

**李杰**：(低聲)所以...這是人類自己設下的防護機制...

**旁白**：他的心沉了下去。如果真是這樣,那麼想要救出恵美,就必須找到另一個人類作為替代品。

**旁白**：但在這個世界裡,除了他之外,還有其他人類嗎?

**旁白**：莉拉睜大了眼睛,顯然對這些歷史感到震驚。

**莉拉**：我...我從來不知道這些事情...終焉祭壇居然有這麼古老的歷史...

**瑟蘭**：大部分族人都不知道。這些故事,有些是我從艾拉如長老那裡聽來的,有些是從魔族商人口中得知的。

**旁白**：瑟蘭看向莉拉。

**瑟蘭**：魔族在這個世界生活的時間比我們精靈族更久。他們的歷史記載中,保存著許多我們不知道的秘密。

**莉拉**：原來如此...

**旁白**：李杰想起了另一件事。

**李杰**：那...終焉祭壇裡的那些劍呢?你知道它們的作用嗎?

**旁白**：瑟蘭搖了搖頭。

**瑟蘭**：關於那些劍,我知道的不多。只聽艾拉如長老簡單提過,那些劍是維持世界穩定所需的。

**李杰**：維持世界穩定?

**瑟蘭**：是的。據說那些劍封印著某種力量,不能隨意拔出或破壞。

**旁白**：瑟蘭的表情變得更加凝重。

**瑟蘭**：你說永恆之森的神樹失去了力量...也許,這和其中一把劍被破壞有關。

**李杰**：白色的劍...

**瑟蘭**：白色的劍?

**李杰**：我們當初在終焉祭壇看到一把白色的劍,已經斷裂了。

**旁白**：瑟蘭皺起眉頭。

**瑟蘭**：白劍...斷裂了?

**旁白**：他沉思了片刻。

**瑟蘭**：艾拉如長老說過那些劍很重要,不能被破壞。如果白劍真的斷裂了...

**旁白**：他搖了搖頭。

**瑟蘭**：但我不清楚具體會造成什麼影響。也許...神樹失去力量,還有曦光之庭族人的離開,可能和這件事有關?

**旁白**：他的語氣充滿了不確定。

**瑟蘭**：不過這只是我的猜測。關於那些劍的真正作用,恐怕只有艾拉如長老才知道。

**旁白**：他抬起頭,看向李杰。

**瑟蘭**：這個世界...可能正在發生我們無法理解的變化。

**旁白**：屋內陷入了沉默。

**旁白**：李杰看著自己的雙手,腦海中翻騰著無數想法。

**旁白**：恵美躺在祭壇上,成為開啟那扇門的祭品。而想要救她,就需要另一個人類來替代她的位置。

**旁白**：但這個世界裡,還有其他人類嗎?

**旁白**：就算有,他又憑什麼要求別人去犧牲自己?

**旁白**：窗外,永恆的黑夜依然籠罩著大地。兩輪紫月冷漠地懸掛在天空,彷彿在注視著這些渺小生命的掙扎。

**旁白**：而在遙遠的終焉祭壇,恵美依然沉睡在金色的光芒中,等待著不知何時才會到來的救援。
