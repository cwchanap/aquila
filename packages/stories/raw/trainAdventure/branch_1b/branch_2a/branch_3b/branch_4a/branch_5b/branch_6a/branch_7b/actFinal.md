# 終幕：詛咒的終結

**旁白**：李杰站在荒原上，腦海中翻騰著無數疑問。

**旁白**：但就在這時——

**旁白**：一陣劇烈的暈眩感毫無預警地襲來。

**李杰**：嗚……！

**旁白**：他的視野開始旋轉，腦袋像是被無形的手撕裂般劇痛。

**旁白**：那種感覺，和之前拿起黑劍時一模一樣。

**李杰**：（抱頭）不……不要……

**旁白**：他試圖集中精神，試圖保持清醒。

**旁白**：但那股邪惡的力量正在侵蝕他的意識，像黑色的潮水般吞噬著他的理智。

**旁白**：手中的黑劍開始散發出詭異的黑霧，纏繞著他的手臂、身體、頭部。

**李杰**：（掙扎）放開……放開我……！

**旁白**：他用盡全力將黑劍扔在地上。

**旁白**：劍身撞擊地面，發出清脆的聲響。

**旁白**：但暈眩沒有減輕。

**旁白**：反而更加強烈。

**旁白**：黑霧從地面的黑劍中不斷湧出，像有生命般爬向李杰。

**李杰**：不……不要……

**旁白**：他踉蹌後退，但雙腿已經無力，跌坐在地上。

**旁白**：頭痛越來越劇烈，意識逐漸模糊。

**旁白**：耳邊響起那個低沉、沙啞的聲音。

**？？？**：你以為……扔掉我就能擺脫嗎……

**？？？**：愚蠢……你的靈魂……早已被我標記……

**？？？**：你……是我的了……

**李杰**：啊……啊啊……

**旁白**：最後一絲意識消散。

**旁白**：李杰倒在地上，雙眼失去焦距。

**旁白**：黑劍自動飛回他的手中。

**旁白**：他的身體緩緩站起，但那已經不是他自己了。

---

**旁白**：不知過了多久。

**旁白**：意識斷斷續續地浮現，又沉入黑暗。

**旁白**：李杰無法控制自己的身體，只能像旁觀者一樣，模糊地感知著周圍。

**旁白**：第一次恢復意識時——

**旁白**：他發現自己站在一片陰暗的森林中，周圍是高聳的樹木和濃密的迷霧。

**李杰**：（微弱）這裡是……哪裡……

**旁白**：還來不及細想，意識又被黑暗吞沒。

**旁白**：第二次恢復意識時——

**旁白**：場景變了。他在另一片森林裡，樹木更加扭曲，空氣中飄散著腐臭的味道。

**李杰**：（掙扎）我……在移動……？

**旁白**：身體不受控制，像是被某種力量驅使著，在森林中穿行。

**旁白**：意識再次墜入黑暗。

**旁白**：第三次、第四次、第五次……

**旁白**：每一次醒來，都是不同的地方。

**旁白**：有時是河邊，有時是山谷，有時是廢墟。

**旁白**：他的身體在黑劍的控制下，不停地移動著，尋找著什麼。

**旁白**：或者說——獵殺著什麼。

---

**旁白**：終於，在某一次意識浮現時——

**凌墨**：李杰！

**旁白**：一個熟悉的聲音響起，清晰而急切。

**旁白**：那聲音像是一道閃電，劈開了籠罩意識的黑暗。

**旁白**：李杰猛地清醒過來。

**旁白**：視野逐漸清晰。

**旁白**：他發現自己站在一個陌生的地方。

**旁白**：周圍是木製的建築，風格優雅而自然，樹木與建築完美融合。

**旁白**：這是……精靈的村落。

**旁白**：但此刻，這個美麗的村落已經變成了地獄。

**旁白**：地上橫七豎八地躺著屍體。

**旁白**：幾個身穿銀色輕甲的精靈劍士，胸口都有致命的劍傷，鮮血染紅了地面。

**旁白**：還有……人類的屍體。

**旁白**：李杰的視線顫抖著掃過那些熟悉的臉孔。

**旁白**：那個健談的男大生，倒在木屋旁，眼睛還睜著。

**旁白**：那個活潑的女大生，蜷縮在地上，手中還握著什麼東西。

**旁白**：那個沉穩的男大生，靠在樹幹上，頭無力地垂著。

**旁白**：那個多話的女大生，趴倒在不遠處，一動不動。

**旁白**：還有那個慷慨的大叔，躺在村落入口，像是在試圖保護其他人。

**旁白**：所有人……都死了。

**旁白**：全都是劍傷。

**旁白**：乾淨俐落的劍傷。

**李杰**：不……不……

**旁白**：他低頭看向自己的手。

**旁白**：黑劍緊握在手中，劍身上沾滿了還未乾涸的鮮血。

**旁白**：鮮紅的，屬於那些無辜者的血。

**李杰**：我……我殺了……他們……

**旁白**：那個聲音再次響起，充滿憤怒和殺意。

**凌墨**：李杰！你這個惡魔！

**旁白**：李杰抬起頭，循聲望去。

**旁白**：凌墨站在不遠處，手中握著一柄白色的長劍。

**旁白**：那柄劍……和高山翔使用的一模一樣。

**旁白**：劍身散發著聖潔的白光，與李杰手中的黑劍形成鮮明對比。

**旁白**：凌墨的眼神冰冷如刃，臉上沒有任何猶豫。

**李杰**：凌墨……我……

**旁白**：還來不及說完——

**旁白**：凌墨動了。

**旁白**：他的速度比高山翔更快，身影如同鬼魅般消失在原地。

**旁白**：李杰的瞳孔驟然收縮。

**李杰**：（內心）太快——！

**旁白**：下一瞬間，白色劍刃已經刺向他的喉嚨。

**旁白**：李杰根本來不及反應。

**旁白**：「噗——！」

**旁白**：劍刃刺入肩膀，鮮血飛濺。

**李杰**：啊啊啊——！

**旁白**：劇烈的疼痛讓他慘叫出聲。

**旁白**：凌墨一腳踢在他的腹部，將他踢飛出去。

**旁白**：李杰重重摔在地上，肩膀的傷口血流如注。

**旁白**：但還沒結束。

**旁白**：凌墨再次逼近，白劍斬下。

**旁白**：就在這時，李杰的身體自動做出反應。

**旁白**：黑劍橫掃而出，架住了那一擊。

**旁白**：「鏗——！」

**旁白**：但李杰完全無法控制這個動作。

**旁白**：那是黑劍在保護它的宿主。

**李杰**：（掙扎，內心）不……我不想……不想再殺人了……

**旁白**：但身體不受控制，自動揮劍反擊。

**旁白**：凌墨冷靜地格擋，白劍與黑劍不斷碰撞。

**旁白**：「鏗！鏗！鏗鏗鏗——！」

**旁白**：但這次的戰鬥，完全不同。

**旁白**：李杰的身體因為重傷而遲緩，每一個動作都帶著劇痛。

**旁白**：意識還在暈眩中，無法完全發揮黑劍的力量。

**旁白**：更重要的是，他的心已經徹底崩潰了。

**李杰**：（絕望，內心）我殺了他們……全都是我殺的……

**李杰**：（內心）那些大學生……那個大叔……

**李杰**：（內心）還有之前那些人……小林夫婦、孩子們的父母……

**李杰**：（內心）全都是我……全都是我殺的……

**旁白**：而凌墨，比高山翔更強。

**旁白**：他的每一劍都精準到可怕，速度快如閃電，力量沉重如山。

**旁白**：白色劍身上的光芒越來越耀眼，彷彿要淨化一切邪惡。

**凌墨**：去死吧，惡魔！

**旁白**：他猛地一個旋身，白劍劃出一道完美的弧線。

**旁白**：李杰的身體想要防禦，但傷勢太重，動作遲緩了一瞬。

**旁白**：就是這一瞬。

**旁白**：「噗——！」

**旁白**：白色劍刃刺穿了李杰的胸口。

**旁白**：劍身貫穿心臟，從背後穿出。

**李杰**：啊……

**旁白**：他的聲音卡在喉嚨裡，只能發出微弱的氣音。

**旁白**：手中的黑劍「鏗」的一聲掉落在地。

**旁白**：劍身上的黑霧逐漸消散，像是失去了力量。

**旁白**：李杰低頭，看著胸口那柄白色的劍。

**旁白**：鮮血從傷口湧出，染紅了他的衣服。

**凌墨**：（冷漠）你不該拿起那柄劍的。

**旁白**：他抽出白劍。

**旁白**：李杰的身體失去支撐，緩緩倒下。

**旁白**：視野逐漸模糊，周圍的景象開始扭曲。

**旁白**：他看到凌墨轉身離去的背影。

**旁白**：看到地上那些無辜者的屍體。

**旁白**：看到自己手邊，那柄詛咒的黑劍。

**李杰**：（微弱）我……真的……好蠢……

**李杰**：（內心）為什麼……要拿起那柄劍……

**李杰**：（內心）如果沒有拿起它……就不會……變成這樣……

**旁白**：意識逐漸消散。

**旁白**：呼吸越來越微弱。

**旁白**：最後，一切歸於寂靜。

---

**旁白**：李杰死了。

**旁白**：死在精靈村落的廢墟中，死在那些他親手殺害的無辜者身旁。

**旁白**：他本想拯救所有人。

**旁白**：結果卻親手毀滅了一切。

**旁白**：那柄黑劍靜靜地躺在他身邊，等待著下一個愚蠢的拾起者。

**旁白**：因為有些東西，從一開始就不該被觸碰。

**旁白**：有些力量，從一開始就不該被渴望。

**旁白**：這是一個關於貪念、關於選擇、關於詛咒的故事。

**旁白**：也是一個關於——

**旁白**：拿起了不該拿起之物的悲劇。
