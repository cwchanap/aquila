# 第二十九幕：不安的夜晚

**旁白**：夜幕完全降臨，李杰站起身走到廚房倒了一杯水。

**旁白**：透過窗戶能看見對面大樓的燈光，還有遠處天空中閃爍的星星，一切都那麼平靜。

**旁白**：他知道此刻同事們正在居酒屋裡推杯換盞、笑聲不斷，而原本的他應該也在那裡。

**李杰**：但我現在在這裡，安全的家裡。

**旁白**：晚上七點、八點、九點，時間一分一秒地流逝。

**旁白**：李杰回到沙發上打開電視，螢幕上播放著一部深夜動畫，色彩鮮豔的畫面配著輕鬆的日常劇情。

**旁白**：但他的注意力完全不在螢幕上，視線不時飄向牆上的時鐘。

**旁白**：晚上十點、十一點，時鐘的指針緩慢而堅定地移動著，每一次跳動都像是在他心上敲擊。

**旁白**：午夜零點，李杰坐在沙發上握著遙控器，視線死死盯著牆上的時鐘。

**李杰**：現在...應該快結束了吧。

**旁白**：零點十五分，按照記憶原本的他此刻應該已經離開居酒屋，正走向新宿車站。

**李杰**：那些人...現在也在趕往車站嗎？

**旁白**：他的腦中浮現出那些面孔——文靜少女會不會正戴著耳機低頭看著手中的冊子？

**旁白**：大學生們會不會正吵吵鬧鬧地擠在月台上？那對夫妻會不會正在互相抱怨？

**旁白**：還有大叔、小林一家...

**李杰**：不...不要想了。

**旁白**：他用力搖頭試圖甩掉這些念頭，但那些面孔卻越來越清晰。

**旁白**：零點三十分，末班車應該快進站了。

**旁白**：他能想像那個場景——月台上聚集著稀疏的乘客，橙色列車緩緩駛入，車門打開。

**旁白**：所有人魚貫而入找到座位或抓住吊環，沒有人知道這將是他們人生中最後一次搭乘普通的列車。

**李杰**：停下...

**旁白**：零點四十分，列車應該已經出發了。

**旁白**：他閉上眼睛，但腦中的畫面卻更加清晰——列車駛入隧道，車廂內的燈光突然熄滅。

**旁白**：乘客們開始竊竊私語，手機螢幕在黑暗中亮起，五分鐘、十分鐘、十五分鐘過去，隧道還沒有盡頭。

**旁白**：不安開始蔓延。

**李杰**：別想了！

**旁白**：他猛地站起身走到冰箱前，拿出一罐啤酒「啵」的一聲打開，仰頭灌下冰涼的液體試圖用酒精麻痺思緒。

**旁白**：但沒有用，腦中的畫面繼續播放——二十分鐘、二十五分鐘、三十分鐘，隧道依然沒有盡頭。

**旁白**：車廂內開始有人尖叫、有人哭泣，然後前方出現了光亮。

**旁白**：不是熟悉的城市燈光，而是兩輪紫色的月亮。

**李杰**：啊...

**旁白**：他放下啤酒罐雙手抱住頭。

**旁白**：凌晨一點，列車應該已經到達寂海車站——那個一側是黑色海洋、一側是幽深森林的荒涼車站。

**旁白**：他能想像乘客們震驚、恐懼、絕望的表情，能想像車門打開時從異世界吹來的冰冷風。

**旁白**：他能想像那些人站在月台上看著這個完全陌生的世界，不知所措。

**李杰**：但這...和我無關。

**旁白**：他又打開一罐啤酒。

**李杰**：我不在那裡，我在這裡，安全的家裡。他們只是陌生人，我為什麼要在意？

**旁白**：凌晨兩點，李杰坐在沙發上，面前已經空了三罐啤酒。

**旁白**：螢幕上的動畫還在播放但他已經完全不知道劇情進展到哪裡，視線空洞地盯著某個不存在的點。

**旁白**：腦中卻不斷浮現那些畫面——文靜少女縮在座位上顫抖，大學生們慌亂地四處張望，母親緊緊抱住孩子，大叔試圖安撫其他人。

**李杰**：我...我只是想活下去而已。

**旁白**：他對著空無一人的房間說道。

**李杰**：我也沒有能力救任何人，就算我在那裡也只會多一個受害者而已。

**旁白**：沒有人回答，只有時鐘的滴答聲在安靜的房間裡迴盪。

**旁白**：凌晨三點，時鐘的指針指向三，李杰抬頭看著那個時間然後長長地嘆了口氣。

**李杰**：結束了...一切都結束了。

**旁白**：末班車早就過去了，那些人的命運已經註定，而他成功地逃過了那一劫。

**旁白**：他又打開一罐啤酒但這次只喝了一口就放下，酒精讓他的腦袋昏沉但思緒卻沒有停止。

**旁白**：那些面孔還在他腦海中徘徊，那些尖叫聲還在他耳邊迴響。

**李杰**：明天...明天就能回到正常生活了，這一切都會過去的。

**旁白**：他關掉電視，房間陷入一片黑暗，只有窗外的路燈透進些許微光。

**旁白**：他搖搖晃晃地走進臥室倒在床上，閉上眼睛試圖入睡。

**旁白**：但腦中的畫面卻更加清晰——紫色的月亮、黑色的海洋、荒涼的車站，還有那些驚恐的面孔一張又一張。

**旁白**：他們盯著他無聲地質問著——為什麼你能逃脫？為什麼你不救我們？為什麼你選擇了遺忘？

**李杰**：我...我只是...

**旁白**：酒精的作用終於發揮了，他的意識開始模糊、思緒逐漸沉入黑暗。

**旁白**：但即使在夢中，他依然能聽見那列車駛入隧道的聲音，還有那些乘客們驚恐的尖叫。
