# 第三十幕：平靜的假面

**旁白**：週一早上鬧鐘準時響起，刺耳的鈴聲在安靜的臥室中迴盪。

**旁白**：李杰伸手關掉鬧鐘揉了揉發脹的太陽穴，宿醉的感覺讓他有些難受——頭痛、口乾、還有一種說不出的疲憊感。

**旁白**：他坐起身看著窗外的晨光，陽光透過窗簾的縫隙照進來在地板上投下細長的光束。

**旁白**：新的一週開始了，一切如常。

**李杰**：...只是一場夢而已。

**旁白**：他這樣告訴自己——昨晚的那些畫面只是酒精和疲憊造成的幻覺。

**旁白**：他起床走進浴室打開水龍頭，冷水沖在臉上帶來一絲清醒。

**旁白**：他抬頭看向鏡子，鏡子裡的自己看起來有些疲憊、眼睛下方有淡淡的黑眼圈、臉色也有些蒼白。

**旁白**：但還算正常，至少表面上看起來和平常沒什麼兩樣。

**旁白**：他刷牙、洗臉，然後回到臥室換衣服——白襯衫、深色西裝褲、黑色皮鞋，熟悉的上班族制服。

**旁白**：他站在鏡子前整理著領帶，動作熟練而機械，就像過去無數個週一早晨一樣。

**旁白**：穿戴整齊後他拿起公事包，那是一個黑色的皮革公事包邊角已經有些磨損但保養得很好。

**旁白**：裡面裝著筆記型電腦、文件夾還有一些日常用品。

**旁白**：他走到玄關手握上門把，就在這一刻停頓了一秒。

**旁白**：腦中閃過昨晚的畫面——時鐘指向凌晨三點、啤酒罐散落在桌上，還有那些他試圖不去想的面孔。

**李杰**：...

**旁白**：他深吸一口氣打開了門。

**旁白**：走出公寓，秋天的晨風帶著一絲涼意吹在臉上，清新的空氣讓他的頭腦稍微清醒了一些。

**旁白**：街道上已經有不少趕著上班的人——穿西裝的上班族、穿制服的學生、騎腳踏車送報的老人。

**旁白**：便利商店門口有穿著制服的高中生在買早餐，店員熱情地招呼著客人，自動門不停地開開關關。

**旁白**：郵差騎著紅色的機車經過，報紙被準確地塞進每家的信箱。

**旁白**：一切都那麼日常、那麼真實、那麼安全。

**旁白**：他邁開腳步走向車站，腳步聲在安靜的住宅區裡迴響——一步、兩步、三步。

**旁白**：每一步都那麼踏實，每一步都將他帶離昨夜的不安。

**旁白**：陽光灑在柏油路上投下他長長的影子，那個影子跟隨著他一起走向三鷹車站。

**旁白**：五分鐘後他來到了車站入口，熟悉的玻璃門、熟悉的驗票閘門、熟悉的月台指示牌。

**旁白**：他從口袋裡掏出IC卡在閘門的感應器上輕輕一刷，「嗶」的一聲閘門打開。

**旁白**：他走進車站沿著熟悉的路線走向月台。

**旁白**：月台上已經聚集了不少等車的人，大家都低頭看著手機、或是閉目養神、或是無神地盯著鐵軌。

**旁白**：沒有人交談、沒有人微笑，這就是東京的早晨——疲憊、沉默、機械化。

**旁白**：李杰站在黃線後等待著列車的到來，遠處傳來列車駛近的聲音。

**旁白**：那是鐵輪與鐵軌摩擦的聲音，規律而熟悉。

**旁白**：橙色的車身出現在視野中緩緩減速，最後準確地停在月台旁。

**廣播**：中央線、新宿方面行きの電車が参ります。

**旁白**：車門打開，人群開始移動。

**旁白**：李杰跟隨著人流走進車廂，車廂內已經擠滿了上班族和學生。

**旁白**：空氣中混雜著各種氣味——香水味、咖啡香、汗味、還有晨間清新劑的味道。

**旁白**：他找到一個可以站立的位置抓住頭頂的吊環，列車發出「即將關門」的提示音。

**旁白**：車門緩緩關閉，列車啟動，規律的震動傳來。

**旁白**：窗外的景色開始倒退——月台、廣告牌、還有那些沒趕上車的人。

**旁白**：李杰看著窗外面無表情，列車穿過一個個熟悉的車站——吉祥寺、西荻窪、荻窪、阿佐谷。

**旁白**：每一站都有人上下車，每個人都專注於自己的事情——看手機、看書、或是閉目休息。

**旁白**：沒有人知道就在昨晚有一列車開往了另一個世界，沒有人知道有一群人正在那個充滿死亡和恐懼的世界中掙扎。

**旁白**：而他李杰是唯一知道真相的人，但他選擇了沉默。

**李杰**：這樣...就好。

**旁白**：他在心中對自己說。

**旁白**：列車繼續前行——中野、東中野、大久保，窗外的景色從住宅區變成商業區，建築越來越高，街道越來越繁華。

**旁白**：終於列車駛入了新宿站，這是東京最大的交通樞紐之一，也是全世界最繁忙的車站。

**旁白**：李杰沒有下車，他要去的是六本木。

**旁白**：列車短暫停留後再次啟動，穿過代代木、千駄谷，最後抵達六本木站。

**旁白**：大約四十分鐘的車程，對於東京的上班族來說這是再正常不過的通勤時間。

**旁白**：李杰隨著下車的人流走出車廂，穿過驗票閘門，搭乘電扶梯往上。

**旁白**：一層又一層，從地下深處回到地面。

**旁白**：走出車站他看見熟悉的六本木街景——高聳的辦公大樓、時尚的商店、還有來往的人群。

**旁白**：他的公司就在前方那棟玻璃帷幕大樓裡，大樓在晨光中閃閃發亮映照著藍天白雲。

**旁白**：他走進大樓在大廳刷了員工卡，「嗶」的一聲閘門打開。

**旁白**：保全人員朝他點頭致意，他也點頭回應。

**旁白**：搭乘電梯到十二樓，電梯裡還有其他同事但沒有人說話。

**旁白**：大家都低頭看著手機或是盯著電梯門上的樓層數字。

**旁白**：「叮」的一聲十二樓到了，電梯門打開是熟悉的辦公室。

**旁白**：白色的牆壁、整齊的辦公桌、成排的電腦螢幕，還有同事們忙碌的身影。

**旁白**：有人在打電話、有人在敲鍵盤、有人端著咖啡在走廊裡走動。

**前台**：早安，李桑！休假過得怎麼樣？

**旁白**：前台的女職員笑著跟他打招呼，那是一個親切的笑容帶著職業性的禮貌。

**李杰**：早安，還不錯，謝謝。

**旁白**：他也露出笑容回應道，表情自然、語氣平穩，就像一個真的度過了愉快假期的人。

**前台**：那就好！今天要加油哦！

**李杰**：會的。

**旁白**：他朝前台點點頭走向自己的座位，那是一個靠窗的位置可以看見外面的街景。

**旁白**：辦公桌上擺放著電腦、鍵盤、滑鼠、還有一個裝著筆的筆筒，一切都和他離開前一模一樣。

**旁白**：他放下公事包拉開椅子坐下，按下電腦的電源鍵。

**旁白**：主機發出低沉的啟動聲螢幕亮起，熟悉的登入畫面出現。

**旁白**：他輸入帳號和密碼進入系統，桌面載入，各種工作用的軟體圖示整齊排列。

**旁白**：他點開郵件軟體，信箱裡堆積著上百封未讀郵件——會議通知、工作報告、客戶回覆、系統更新。

**旁白**：他嘆了口氣開始一封一封地瀏覽。

**同事D**：李桑！休假回來啦！身體好點了嗎？

**旁白**：隔壁座位的同事探過頭來笑著問道，那是一個年輕的男同事戴著眼鏡看起來很有精神。

**李杰**：嗯，好多了，謝謝關心。

**同事D**：那就好！對了，週日的聚會你沒來真可惜，大家都喝得很開心呢！部長還講了很多以前的趣事。

**李杰**：是嗎...聽起來很有趣。

**旁白**：李杰勉強擠出笑容。

**同事D**：下次一定要來哦！好啦，我先忙了，有什麼需要幫忙的儘管說！

**李杰**：好的，謝謝。

**旁白**：同事轉回去繼續工作，辦公室裡只剩下鍵盤的敲擊聲、滑鼠的點擊聲、還有偶爾的低聲交談。

**旁白**：李杰盯著電腦螢幕手指放在鍵盤上，螢幕上是密密麻麻的代碼和技術文件。

**旁白**：這就是他的生活——朝九晚六，寫代碼、開會、處理bug，平凡、安全、可預測。

**旁白**：他深吸一口氣開始工作，手指在鍵盤上飛快地移動，代碼一行行出現在螢幕上，邏輯清晰、結構完整。

**旁白**：這是他熟悉的世界，一個用0和1構建的、理性的、可控的世界，不像那個充滿紫月和死亡的異世界。

**旁白**：時鐘在牆上靜靜地走著，辦公室的日光燈發出穩定的白光，窗外是繁華的東京街景。

**旁白**：一切都回到了正軌。

**旁白**：就好像那一切從未發生過，就好像昨晚那些人沒有消失在異世界，就好像他不是唯一知道真相的人。

**旁白**：李杰繼續敲著鍵盤，臉上沒有任何表情，眼神專注而空洞。

**旁白**：就像一個真正享受了假期、精神飽滿地回到工作崗位的普通上班族。

**旁白**：沒有人發現異常，沒有人知道他內心的掙扎，沒有人知道他選擇了遺忘。

**旁白**：平靜的謊言，就這樣開始了。
