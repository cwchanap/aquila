# 第三十九幕：裂痕

**旁白**：黑西裝男A繼續盯著李杰震驚的臉，似乎在享受這種掌控感。

**黑西裝男A**：你的父親李明遠，和渡邊陸曾經在中國的同一家公司工作。

**旁白**：他的聲音依然平淡，但每個字都像錘子一樣砸在李杰心上。

**黑西裝男A**：前後大約四年時間，他們有過密切的工作往來。

**旁白**：李杰感到喉嚨發緊，幾乎說不出話來。

**旁白**：父親...和渡邊陸認識？在中國工作過？

**黑西裝男A**：20年前，渡邊陸回到日本。不久之後，他搭上了那班列車，然後失蹤了。

**旁白**：他頓了頓，眼神變得更加深邃。

**黑西裝男A**：但在8年前，他回來了。回到了這個世界，在日本正常生活，有工作，有家庭。

**旁白**：李杰的腦中一片混亂。

**旁白**：8年前回來...那就是在異世界待了12年？

**旁白**：不對，時間流速不同，那邊的兩年等於這裡的20年...

**旁白**：那渡邊陸在異世界待了多久？他是怎麼回來的？

**旁白**：而且，他回來後為什麼又要搭上那班車？

**李杰**：你...你為什麼要告訴我這些？

**旁白**：李杰終於找回自己的聲音，但聲音沙啞而顫抖。

**旁白**：黑西裝男A突然露出一個詭異的笑容。

**旁白**：那不是友善的笑，而是一種冰冷的、帶著嘲諷的弧度。

**旁白**：但他沒有回答，只是靜靜地看著李杰，像在看一個有趣的實驗品。

**旁白**：李杰盯著那張冷漠的臉，突然明白了什麼。

**旁白**：一個可怕的念頭在腦中浮現。

**李杰**：我懂了。

**旁白**：李杰的嘴角也勾起一個笑容，但那是一個苦澀而諷刺的笑。

**李杰**：你們打算在拿到筆電裡的資訊後，殺掉我們兩個，對吧？

**旁白**：他的聲音變得冷靜，甚至帶著一絲嘲諷。

**李杰**：反正我們都要死了，所以你說多少都無所謂。死人不會洩密，對吧？

**旁白**：黑西裝男A的臉色瞬間變了。

**旁白**：那種冷漠的表情出現了一絲裂痕，眼神變得更加銳利，帶著一種被看穿的憤怒。

**清水浩介**：不...不要...

**旁白**：浩介趴在地上，整個人開始劇烈顫抖。

**清水浩介**：求求你們...不要殺我...我什麼都不知道...我只是想找我姐姐...

**旁白**：他的聲音充滿了恐懼，眼淚再次流下來。

**清水浩介**：拜託...我有老母親要照顧...我不能死...求求你們...

**旁白**：黑西裝男A沒有看浩介，他的目光依然鎖定在李杰身上。

**旁白**：幾秒鐘的沉默後，他緩緩開口。

**黑西裝男A**：如果你能提供有用的信息...我們不會殺你。

**旁白**：他的聲音恢復了那種職業性的冷靜。

**黑西裝男A**：你的生死，取決於你的價值。

**旁白**：但李杰知道他在說謊。

**旁白**：那雙眼睛裡沒有任何信任或承諾，只有冰冷的算計。

**旁白**：即使筆電裡真的有他們想要的資訊，即使李杰配合到底，最後等待他的依然是死亡。

**旁白**：因為他知道得太多了。

**旁白**：房間再次陷入死寂。

**旁白**：只有那個永恆的嗡嗡聲，還有浩介壓抑的抽泣聲。

**旁白**：時間一分一秒地流逝。

**旁白**：李杰不知道過了多久——也許十分鐘，也許二十分鐘。

**旁白**：黑西裝男A依然坐在椅子上，但他開始頻繁地看手機。

**旁白**：先是解鎖螢幕，看一眼時間，然後鎖上。

**旁白**：過了一會兒又重複同樣的動作。

**旁白**：他的表情依然冷漠，但動作透露出一絲不安。

**旁白**：李杰注意到了這個細節。

**旁白**：黑西裝男B離開已經很久了。

**旁白**：從三鷹到這裡，來回應該不需要這麼長時間。

**旁白**：黑西裝男A終於按捺不住，拿起手機撥了一個號碼。

**旁白**：手機貼在耳邊，他等待著。

**旁白**：一秒、兩秒、三秒...

**旁白**：沒有人接聽。

**旁白**：他掛斷，再次撥打。

**旁白**：依然沒有回應。

**旁白**：黑西裝男A的眉頭皺了起來，那是他第一次露出明顯的情緒波動。

**旁白**：李杰看到了這一切，嘴角勾起一個微弱的笑容。

**李杰**：看來...你的同伴出事了？

**旁白**：他的聲音充滿了嘲諷。

**李杰**：去了這麼久還沒回來，電話也不接...該不會遇到什麼麻煩了吧？

**旁白**：黑西裝男A猛地抬頭，銳利的目光刺向李杰。

**旁白**：但他沒有說話，臉色變得更加陰沉。

**旁白**：他再次撥打電話，這次等待的時間更長。

**旁白**：但結果依然一樣——沒有人接聽。

**旁白**：黑西裝男A站起身，椅子在地板上發出刺耳的摩擦聲。

**旁白**：他快步走向門口，一隻手已經放在了腰後的槍套上。

**旁白**：在打開門之前，他回頭看向李杰和浩介。

**黑西裝男A**：待在這裡。別想耍花樣。

**旁白**：他的聲音充滿了威脅。

**旁白**：然後他推開金屬門，快步走了出去。

**旁白**：門在他身後關上，發出沉重的咔嚓聲。

**旁白**：房間裡只剩下李杰和浩介。

**旁白**：李杰盯著那扇緊閉的金屬門，腦中飛速運轉。

**旁白**：黑西裝男B出事了...這意味著什麼？

**旁白**：是意外？還是...有人在幫他們？

**旁白**：不管是什麼原因，這可能是他們唯一的機會。

**清水浩介**：李先生...我們...我們會死嗎？

**旁白**：浩介的聲音顫抖著，充滿了絕望。

**李杰**：不會的。

**旁白**：李杰的聲音出乎意料地堅定。

**李杰**：我們會活著離開這裡。

**旁白**：他不知道自己哪來的自信說出這句話。

**旁白**：但此刻，他必須相信這一點。

**旁白**：因為如果放棄希望，那才是真正的死亡。

**旁白**：金屬房間裡，兩個被綁住的人在黑暗中等待著。

**旁白**：等待著不知道會是救贖還是毀滅的命運。
