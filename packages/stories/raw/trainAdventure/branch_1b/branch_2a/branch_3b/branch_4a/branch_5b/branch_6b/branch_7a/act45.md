# 第四十五幕：失蹤者的哥哥

**旁白**：李杰推開店門，一陣淡淡的木頭香氣撲面而來。

**旁白**：店內裝潢簡約，只有五、六張桌子，此刻只有兩桌有客人。

**旁白**：在最裡面的角落，一個中年男子坐在桌旁。

**旁白**：他看起來四十多歲，穿著深灰色的西裝，頭髮整齊地梳向後方。

**旁白**：臉型方正，眼神銳利而沉穩，給人一種不怒自威的感覺。

**旁白**：他的旁邊坐著一個年輕人。

**旁白**：那個年輕人看起來二十多歲，身材圓潤，臉上帶著輕鬆愉快的笑容。

**旁白**：中年男子看到三人進來，立刻站起身，向他們揮手。

**宇佐美誠一郎**：李先生，這邊。

**旁白**：他的聲音低沉而穩重。

**旁白**：李杰、中野和凌墨走到桌旁。

**宇佐美誠一郎**：初次見面，我是麻布警察署的宇佐美誠一郎。

**旁白**：宇佐美伸出手。

**李杰**：您好，我是李杰。

**旁白**：李杰和他握手，能感受到對方堅定的力道。

**李杰**：這位是我的同事中野海斗，這位是凌墨。

**中野海斗**：您好，宇佐美警部。很高興見到您。

**旁白**：中野露出友善的笑容，伸手和宇佐美握手。

**中野海斗**：謝謝您願意幫忙調查這件事。

**凌墨**：你好。

**旁白**：凌墨簡短地點頭，沒有伸手。

**宇佐美誠一郎**：請坐。

**旁白**：宇佐美示意三人坐下。

**旁白**：李杰、中野和凌墨在對面坐下。

**旁白**：桌子是長方形的，雙方面對面坐著。

**旁白**：宇佐美看向身旁的胖年輕人。

**宇佐美誠一郎**：這位是齋藤大輔先生。

**旁白**：胖年輕人立刻露出燦爛的笑容，熱情地向三人鞠躬。

**齋藤大輔**：你們好你們好！我叫齋藤大輔！請多多指教！

**旁白**：他的聲音充滿活力，語氣輕快。

**齋藤大輔**：叫我大輔就好！不用客氣！

**旁白**：他的熱情和宇佐美的嚴肅形成鮮明對比。

**宇佐美誠一郎**：齋藤先生是這次失蹤案件中，一位失蹤乘客的哥哥。

**旁白**：宇佐美的語氣變得認真。

**旁白**：李杰心中一驚。

**旁白**：失蹤乘客的哥哥...所以他也在尋找家人。

**宇佐美誠一郎**：他的弟弟齋藤蓮，在9月23日晚上搭上了那班列車。

**旁白**：宇佐美從西裝內袋掏出一張照片，放在桌上。

**旁白**：李杰看向照片。

**旁白**：照片中是一個戴著眼鏡的年輕男子，大約二十多歲，表情有些拘謹。

**旁白**：李杰的眼睛瞬間睜大。

**旁白**：他認得這張臉——那個在列車上故作聰明、試圖用偽科學解釋一切的眼鏡男！

**李杰**：這個人...我在列車上見過他。

**旁白**：李杰的聲音帶著驚訝。

**齋藤大輔**：真的嗎？！

**旁白**：大輔整個人向前傾，眼中閃過強烈的期待。

**齋藤大輔**：那...那你知道蓮現在怎麼樣了嗎？他還好嗎？

**旁白**：他的聲音急切而緊張。

**旁白**：李杰沉默了幾秒，然後緩緩搖頭。

**李杰**：對不起...我不知道。

**旁白**：大輔的表情瞬間黯淡下來，肩膀也垮了下去。

**齋藤大輔**：這樣啊...

**旁白**：他的聲音變得低落。

**旁白**：宇佐美伸手輕輕拍了拍大輔的肩膀，然後看向李杰。

**宇佐美誠一郎**：齋藤先生在發現弟弟失蹤後，立刻向警方報案。

**旁白**：宇佐美的語氣恢復專業。

**宇佐美誠一郎**：我接手了這個案件，開始調查那班列車的異常情況。

**旁白**：他停頓了一下。

**宇佐美誠一郎**：在調查過程中，齋藤先生展現了驚人的情報收集能力。

**旁白**：宇佐美看向大輔，眼中帶著一絲讚許。

**宇佐美誠一郎**：他找到了許多失蹤者的線索，也是他讓我知道了李先生的存在。

**齋藤大輔**：哈哈，沒什麼啦！

**旁白**：大輔搔了搔頭，露出有些不好意思的笑容。

**齋藤大輔**：我只是...為了找蓮，到處問、到處查而已！

**齋藤大輔**：我在網路上找啊找，看到有人在討論那班車的怪事。

**齋藤大輔**：然後又去高尾警察署問，那邊的人提到有個叫李杰的人也報案過類似的事...

**旁白**：他的語氣輕快，但眼中的擔憂還是掩飾不住。

**宇佐美誠一郎**：但是...

**旁白**：宇佐美的表情變得凝重。

**宇佐美誠一郎**：在調查進行了幾週後，我突然被上級告知，不能再繼續調查這個案件。

**李杰**：什麼？為什麼？

**旁白**：李杰皺起眉頭。

**宇佐美誠一郎**：沒有給出明確的理由。

**旁白**：宇佐美的語氣帶著一絲不滿。

**宇佐美誠一郎**：只是簡單地說「此案不在我們管轄範圍內」，然後就要我結案。

**旁白**：他看向李杰、中野和凌墨。

**宇佐美誠一郎**：這讓我感覺到，上層可能在隱瞞什麼。

**宇佐美誠一郎**：有某種勢力不希望這件事被調查。

**凌墨**：所以你決定私下調查。

**旁白**：凌墨第一次開口，聲音平靜而低沉。

**宇佐美誠一郎**：沒錯。

**旁白**：宇佐美點頭。

**宇佐美誠一郎**：作為一名警察，我不能對這種異常情況視而不見。

**宇佐美誠一郎**：而且...

**旁白**：他看向大輔。

**宇佐美誠一郎**：齋藤先生為了尋找弟弟付出了很多努力。我不能讓他一個人面對這些。

**齋藤大輔**：宇佐美警部...

**旁白**：大輔的眼眶有些泛紅。

**宇佐美誠一郎**：所以，我希望能獲得更多的幫助。

**旁白**：宇佐美看向李杰。

**宇佐美誠一郎**：李先生，你也經歷過那班列車，也遭遇了黑西裝男的攻擊。

**宇佐美誠一郎**：我們或許可以互相分享資訊，一起找出真相。

**旁白**：李杰沒有立刻回答。

**旁白**：他看向中野，中野給了他一個鼓勵的眼神。

**旁白**：他又看向凌墨，凌墨面無表情，但沒有反對的意思。

**宇佐美誠一郎**：當然，你不需要現在就給我答覆。

**旁白**：宇佐美拿起桌上的菜單。

**宇佐美誠一郎**：我們先點些東西吃吧。今天我請客。

**旁白**：他把菜單遞給李杰。

**宇佐美誠一郎**：一邊吃一邊慢慢談，不用緊張。

**齋藤大輔**：對對對！先吃飯先吃飯！

**旁白**：大輔立刻恢復了活力，笑容重新回到臉上。

**齋藤大輔**：這家店的烤魚超好吃的！還有天婦羅！一定要試試！

**旁白**：他熱情地推薦著，彷彿剛才的低落從未發生過。

**旁白**：李杰接過菜單，心中卻還在思考著宇佐美的提議。

**旁白**：和警察合作...這會是個好選擇嗎？

**旁白**：還是會帶來更多的危險？

**旁白**：他看著菜單上的文字，但腦中想的全是別的事情。
