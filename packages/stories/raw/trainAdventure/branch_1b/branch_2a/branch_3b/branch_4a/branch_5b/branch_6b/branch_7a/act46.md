# 第四十六幕：靜月亭的晚餐

**旁白**：中野接過菜單，臉上露出輕鬆的笑容。

**中野海斗**：那真是太好了！既然宇佐美警部請客，我就不客氣了！

**旁白**：他認真地翻閱著菜單。

**中野海斗**：我要這個特選和牛燒肉定食，還有刺身拼盤！

**旁白**：他的語氣充滿期待。

**中野海斗**：真的很感謝您願意幫忙，宇佐美警部。

**宇佐美誠一郎**：不客氣。

**旁白**：宇佐美淡淡地笑了笑。

**齋藤大輔**：哇！和牛！我也要我也要！

**旁白**：大輔立刻舉手。

**齋藤大輔**：再加一份天婦羅套餐！

**旁白**：李杰看了看菜單，點了一份普通的烤魚定食。

**李杰**：我點這個就好。

**旁白**：凌墨則簡短地說。

**凌墨**：豬排定食。

**旁白**：宇佐美招手叫來服務生，點完所有餐點。

**旁白**：等待上菜的時候，宇佐美放鬆地靠在椅背上。

**宇佐美誠一郎**：你們平常都有什麼興趣愛好？

**旁白**：他的語氣變得輕鬆，像是在進行日常的閒聊。

**中野海斗**：我的話...喜歡玩電玩遊戲！

**旁白**：中野眼睛一亮。

**中野海斗**：最近在玩那個新出的RPG，叫《幻想編年史》，超好玩的！

**齋藤大輔**：哇！我也在玩那個！

**旁白**：大輔整個人向前傾，興奮地說道。

**齋藤大輔**：你玩到哪裡了？我已經打到第三章的BOSS了！

**中野海斗**：我也是！那個BOSS超難打的對吧？

**齋藤大輔**：對對對！我死了快十次才過！

**旁白**：兩人立刻熱烈地討論起來。

**齋藤大輔**：對了對了，你有看最近那部新番動畫嗎？

**齋藤大輔**：《星海旅人》！超熱血的宇宙冒險故事！

**中野海斗**：有啊有啊！我每週都在追！

**中野海斗**：上週那集主角覺醒新能力的場景，真的燃爆了！

**齋藤大輔**：對吧對吧！我看到那段都起雞皮疙瘩了！

**旁白**：兩人越聊越起勁，話題從遊戲跳到動畫，又跳到漫畫。

**旁白**：李杰在旁邊聽著，偶爾也會插上幾句。

**李杰**：我也有玩《幻想編年史》，不過進度比較慢。

**中野海斗**：杰你也在玩啊！那我們可以一起打副本！

**李杰**：好啊，有空的話。

**旁白**：李杰淡淡地笑了笑。

**齋藤大輔**：李先生你有看《星海旅人》嗎？

**李杰**：有看過幾集，畫面很不錯。

**齋藤大輔**：對吧！這季的動畫製作真的很用心！

**旁白**：凌墨坐在一旁，面無表情地聽著。

**旁白**：他一句話也沒說，只是偶爾喝一口茶。

**旁白**：宇佐美則在旁邊微笑著，看著這群年輕人熱烈地討論。

**旁白**：他沒有加入話題，但也沒有打斷，只是靜靜地觀察著。

**旁白**：沒多久，服務生開始上菜。

**旁白**：一盤盤精緻的料理擺上桌面。

**旁白**：和牛的油花在燈光下閃著光澤，刺身的色澤鮮豔誘人。

**旁白**：天婦羅炸得金黃酥脆，烤魚散發著淡淡的醬油香氣。

**中野海斗**：哇，看起來好好吃！

**旁白**：中野雙手合十。

**中野海斗**：我開動了！

**齋藤大輔**：我開動了！

**旁白**：大輔也迫不及待地拿起筷子。

**旁白**：兩人幾乎同時夾起第一口食物。

**中野海斗**：嗯...！這個和牛...入口即化！

**旁白**：中野的表情充滿幸福。

**齋藤大輔**：天婦羅也超棒的！外酥內嫩！

**旁白**：大輔興奮地說著。

**齋藤大輔**：果然沒錯，這家店真的是一流的！

**中野海斗**：宇佐美警部，真的太謝謝您了！

**中野海斗**：這麼好吃的料理，真的很不好意思！

**宇佐美誠一郎**：別客氣，慢慢吃。

**旁白**：宇佐美淡淡地笑著。

**旁白**：李杰也開始吃起自己的烤魚定食。

**旁白**：魚肉烤得恰到好處，鮮嫩多汁。

**旁白**：他看向凌墨，凌墨正安靜地吃著豬排。

**李杰**：凌墨，味道怎麼樣？

**旁白**：李杰輕聲問道。

**旁白**：凌墨停下筷子，抬起頭看向李杰。

**旁白**：他沒有說話，只是豎起大拇指。

**旁白**：然後繼續低頭吃飯。

**旁白**：李杰忍不住笑了出來。

**旁白**：這就是凌墨的風格——簡潔到不能再簡潔。

**旁白**：餐桌上，中野和大輔還在熱烈地討論著。

**齋藤大輔**：對了對了，中野先生，你有玩過《銀河戰記》嗎？

**中野海斗**：玩過玩過！那個戰略遊戲對吧？

**齋藤大輔**：對！我超愛那個遊戲的！

**旁白**：兩人邊吃邊聊，氣氛輕鬆愉快。

**旁白**：宇佐美也慢慢吃著自己的料理，偶爾抬頭看看大家。

**旁白**：他的眼神溫和，嘴角帶著淡淡的笑意。

**旁白**：時間在這溫暖的氛圍中悄然流逝。

**旁白**：大約一個小時後，大家都吃得差不多了。

**旁白**：桌上的盤子幾乎都空了。

**宇佐美誠一郎**：吃飽了嗎？

**旁白**：宇佐美看向大家。

**中野海斗**：吃得很飽！真的非常感謝！

**齋藤大輔**：超滿足的！

**旁白**：大輔拍了拍自己圓滾滾的肚子。

**李杰**：謝謝款待。

**凌墨**：謝謝。

**旁白**：宇佐美招手叫來服務生結帳。

**旁白**：幾分鐘後，一行人走出餐廳。

**旁白**：夜晚的六本木依然燈火通明，街道上人來人往。

**宇佐美誠一郎**：今天就到這裡吧。

**旁白**：宇佐美轉身面對三人。

**宇佐美誠一郎**：李先生，關於合作的事，你可以慢慢考慮。

**宇佐美誠一郎**：如果願意的話，隨時聯絡我。

**旁白**：他遞給李杰一張名片。

**李杰**：好的，我會認真考慮的。

**旁白**：李杰接過名片。

**齋藤大輔**：李先生，中野先生，凌墨先生！

**旁白**：大輔熱情地揮手。

**齋藤大輔**：今天真的很開心！希望以後還能再見面！

**中野海斗**：我也是！大輔先生，有機會一起打遊戲啊！

**齋藤大輔**：好啊好啊！加個好友吧！

**旁白**：兩人立刻掏出手機，交換了遊戲帳號。

**旁白**：李杰和凌墨在旁邊看著，都露出了淡淡的笑容。

**宇佐美誠一郎**：那麼，大家路上小心。

**旁白**：宇佐美點了點頭。

**旁白**：眾人互相道別後，各自朝不同的方向離開。

**旁白**：宇佐美和大輔一起往左邊的街道走去。

**旁白**：李杰、中野和凌墨則往車站的方向前進。

**旁白**：走了一段路後，三人來到六本木站。

**中野海斗**：杰，你覺得怎麼樣？

**旁白**：中野輕聲問道。

**中野海斗**：宇佐美警部和大輔先生，看起來都是好人。

**李杰**：嗯...我也這麼覺得。

**旁白**：李杰點點頭。

**李杰**：但還是要謹慎一點。

**凌墨**：我會繼續觀察。

**旁白**：凌墨簡短地說道。

**旁白**：三人在車站門口停下。

**中野海斗**：那我先回去了。杰，週末好好休息！

**李杰**：你也是，中野。週一見。

**凌墨**：路上小心。

**旁白**：中野揮揮手，走進車站。

**旁白**：李杰和凌墨也各自搭上不同方向的電車。

**旁白**：夜色中，東京的燈光依然閃爍。

**旁白**：今晚的會面，為這個複雜的謎團增添了新的線索。

**旁白**：但前方的路，依然充滿未知。
