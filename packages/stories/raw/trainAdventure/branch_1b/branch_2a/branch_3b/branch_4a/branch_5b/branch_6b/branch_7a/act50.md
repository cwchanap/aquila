# 第五十幕：銀座的邀約

**旁白**：李杰皺著眉頭，緩緩睜開眼睛。

**旁白**：頭痛欲裂，口乾舌燥。

**旁白**：中野躺在沙發上，還在呼呼大睡。

**旁白**：大輔則蜷縮在另一側的地毯上，發出輕微的鼾聲。

**旁白**：茶几上散落著空啤酒罐和零食袋，昨晚的狂歡痕跡一覽無遺。

**李杰**：中野...大輔先生...該起床了...

**旁白**：李杰輕聲叫著。

**中野海斗**：嗯...再睡一下...

**旁白**：中野翻了個身，繼續睡。

**齋藤大輔**：唔...幾點了...

**旁白**：大輔揉著眼睛坐起來。

**李杰**：中午十二點半了。

**齋藤大輔**：什麼？！

**旁白**：大輔猛地清醒，看向手機。

**齋藤大輔**：完了完了...我原本想早點回家的...

**旁白**：他慌張地站起來，開始收拾自己的東西。

**李杰**：中野，我們要走了。

**旁白**：李杰走到沙發邊，拍了拍中野的肩膀。

**中野海斗**：嗯...好...

**旁白**：中野勉強睜開眼睛，坐起身。

**中野海斗**：你們...要回去了嗎？

**旁白**：李杰和大輔簡單整理了一下儀容。

**旁白**：李杰瞇著眼睛，感覺頭痛更嚴重了。

**齋藤大輔**：李先生，你也是搭京濱東北線嗎？

**李杰**：對，我在品川轉車。

**齋藤大輔**：太好了！那我們一起走吧！

**旁白**：兩人並肩走在街道上。

**旁白**：週日中午的住宅區很安靜，偶爾有幾個路人經過。

**旁白**：李杰看了看身旁的大輔。

**旁白**：他想起昨晚大輔說的那些話。

**旁白**：關於那個怪老闆...困在另一個世界...

**旁白**：那究竟是什麼意思？

**李杰**：大輔先生...

**旁白**：李杰試探性地開口。

**李杰**：你之前的公司...老闆失蹤的事...

**齋藤大輔**：嗯？老闆失蹤？

**旁白**：大輔轉頭看向李杰，表情有些困惑。

**李杰**：你昨晚有提到，你的老闆失蹤了。

**齋藤大輔**：啊...對喔...

**旁白**：大輔抓了抓頭。

**齋藤大輔**：他確實失蹤了。就突然不來公司了，然後就再也沒出現過。

**李杰**：你知道他為什麼會失蹤嗎？

**齋藤大輔**：不知道啊。

**旁白**：大輔聳聳肩。

**齋藤大輔**：有一天就突然不見了。警察也找過，但什麼線索都沒有。

**旁白**：他的語氣聽起來很自然，沒有任何異樣。

**李杰**：是這樣啊...

**旁白**：李杰沉默了幾秒。

**李杰**：對了，昨晚你還說...

**旁白**：他深吸一口氣。

**李杰**：你說你老闆困在另一個世界...

**齋藤大輔**：咦？

**旁白**：大輔停下腳步，臉上滿是疑惑。

**齋藤大輔**：我有說過這種話嗎？

**李杰**：有，你喝醉的時候說的。

**齋藤大輔**：這...

**旁白**：大輔皺起眉頭，努力回想。

**齋藤大輔**：我完全不記得了...

**旁白**：他尷尬地笑了笑。

**齋藤大輔**：我昨晚喝太多了，斷片了...

**齋藤大輔**：可能是把什麼動漫的劇情搞混了吧？

**旁白**：他抓了抓頭。

**齋藤大輔**：我最近在追一部異世界轉生的動畫，說不定是把那個劇情和現實搞混了。

**齋藤大輔**：真不好意思，讓你聽了一堆胡言亂語...

**旁白**：大輔的表情看起來真誠而無辜。

**旁白**：李杰看著他，試圖從他的眼神中找到破綻。

**旁白**：但大輔的眼神清澈，沒有閃躲，也沒有不安。

**李杰**：是嗎...那可能真的是我聽錯了。

**旁白**：李杰沒有繼續追問。

**齋藤大輔**：哈哈，以後我要少喝一點！

**旁白**：大輔笑著說。

**齋藤大輔**：喝醉後亂講話，真的很丟臉啊！

**旁白**：兩人繼續往前走。

**旁白**：但李杰的心中卻充滿了疑問。

**旁白**：大輔真的不記得了嗎？

**旁白**：還是他在刻意隱瞞什麼？

---

**旁白**：大森町站很快就到了。

**旁白**：兩人走進車站，刷卡進站。

**齋藤大輔**：李先生，你要在品川站下車對吧？

**李杰**：對，然後轉車去三鷹。

**齋藤大輔**：那我們可以一起坐到品川！

**旁白**：兩人一起上了往北的列車。

**旁白**：週日中午的車廂裡人不多，兩人找到座位坐下。

**齋藤大輔**：對了對了，李先生！

**旁白**：大輔一坐下就開始說話。

**齋藤大輔**：昨天玩的那個遊戲副本真的超難的！

**齋藤大輔**：最後那個頭目的血量也太厚了吧！

**李杰**：嗯，確實挺難的。

**旁白**：李杰簡短地回應。

**齋藤大輔**：不過我們三個配合得很好！

**齋藤大輔**：你的補血時機抓得很準，不然我們早就滅團了！

**旁白**：大輔興奮地比劃著。

**齋藤大輔**：對了，你有看最新一集的《星海旅人》嗎？

**齋藤大輔**：上週那個劇情大反轉，我完全沒想到！

**李杰**：還沒看。

**齋藤大輔**：什麼？！你一定要看！真的超精彩的！

**旁白**：大輔滔滔不絕地講述著劇情。

**旁白**：李杰靠在椅背上，偶爾點頭回應。

**旁白**：列車搖晃著前進，窗外的風景不斷變換。

**齋藤大輔**：還有還有，中野先生真的人很好！

**旁白**：大輔繼續說著。

**齋藤大輔**：他教我的時候超有耐心的，每個步驟都解釋得很清楚。

**齋藤大輔**：而且辦公室的氣氛也很好，大家都很友善！

**李杰**：嗯。

**齋藤大輔**：雖然你們說工作會很忙，但我覺得應該還好啦！

**齋藤大輔**：我以前也加過班，應該撐得住！

**旁白**：大輔笑著說。

**齋藤大輔**：對了，你平常下班後都做什麼？

**李杰**：看動畫、玩遊戲。

**齋藤大輔**：哇，我也是！我們真的很合得來耶！

**旁白**：大輔開心地說。

**齋藤大輔**：那你最近還有在追其他動畫嗎？

**旁白**：李杰簡短地回答著大輔的問題。

**旁白**：大輔則一直保持著高昂的興致，從遊戲聊到動畫，從工作聊到生活。

**旁白**：李杰發現自己根本不需要主動找話題。

**旁白**：只要偶爾回應幾句，大輔就能自己繼續聊下去。

**齋藤大輔**：欸欸，李先生，你有女朋友嗎？

**旁白**：大輔突然問道。

**李杰**：沒有。

**齋藤大輔**：我也沒有！

**旁白**：大輔笑著說。

**齋藤大輔**：之前一直忙著工作和打遊戲，根本沒時間談戀愛...

**旁白**：他的語氣變得有些落寞。

**齋藤大輔**：不過沒關係！現在工作穩定了，說不定能遇到好對象！

**旁白**：他很快又恢復了笑容。

**旁白**：列車繼續前進，大輔的聲音在車廂中迴盪。

**旁白**：李杰閉上眼睛，任由大輔的話語在耳邊流淌。

**旁白**：宿醉的疲憊感還沒完全消退。

**旁白**：但大輔的聲音意外地讓人覺得安心。

**旁白**：至少，這個開朗的胖子不像是在隱瞞什麼。

**旁白**：幾站之後，列車到達品川站。

**李杰**：我到了，大輔先生再見。

**齋藤大輔**：再見！李先生週一見！

**旁白**：李杰下了車，站在月台上。

**旁白**：他看著列車緩緩駛離，大輔在車窗內揮手道別。

**旁白**：列車消失在視線中，往六本木方向駛去。

**旁白**：李杰深吸一口氣，走向轉乘的月台。

---

**旁白**：回到三鷹的家，已經是下午兩點。

**旁白**：李杰脫下外套，倒在沙發上。

**旁白**：宿醉的疲憊感讓他全身無力。

**旁白**：手機突然震動了一下。

**旁白**：是宇佐美的訊息。

**旁白**：「李先生，最近有個有趣的進展。」

**旁白**：「我在調查過程中，認識了一位從美國來的調查員。」

**旁白**：「他叫納撒尼爾史特林，也在追查類似的失蹤案件。」

**旁白**：「他對你的經歷很感興趣，想和你見面聊聊。」

**旁白**：「今天傍晚方便嗎？」

**旁白**：李杰看著訊息，揉了揉太陽穴。

**旁白**：今天實在太累了...

**旁白**：他開始打字。

**旁白**：「宇佐美警部，不好意思，今天宿醉有點嚴重。」

**旁白**：「到時候見。」

**旁白**：李杰回覆：「好的，明天見。」

**旁白**：他把手機放到一邊，閉上眼睛。

**旁白**：美國的調查員...納撒尼爾史特林...

**旁白**：為什麼會對這個案子感興趣？

**旁白**：他又知道些什麼？

**旁白**：這些疑問在腦中盤旋。

**旁白**：但疲憊最終戰勝了好奇心。

**旁白**：李杰很快就睡著了。

---

**旁白**：週一轉眼就到了。

**旁白**：李杰在公司處理了一整天的工作。

**旁白**：大輔和中野在辦公室裡依然熱絡地聊天，就像週末什麼事都沒發生過一樣。

**旁白**：不知道今晚的會面，會帶來什麼樣的資訊。
