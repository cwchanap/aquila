# 第五十一幕：失蹤的經理

**李杰**：你好，史特林先生。

**旁白**：李杰握住對方的手，能感受到那股堅定的力道。

**納撒尼爾史特林**：請叫我納撒尼爾就好。

**旁白**：納撒尼爾微笑著說，鬆開手。

**納撒尼爾史特林**：很高興終於見到你。宇佐美警部跟我說了很多關於你的事。

**宇佐美誠一郎**：來，先坐下吧。

**旁白**：三人在桌邊坐下。

**旁白**：壽司師傅在吧檯後方專注地處理著食材，動作優雅而精準。

**旁白**：店內只有他們這一桌客人，環境安靜而私密。

**納撒尼爾史特林**：李先生，容我先自我介紹一下。

**旁白**：納撒尼爾坐直身子，語氣變得認真。

**納撒尼爾史特林**：我是一名商人，在東京經營一家貿易公司。

**納撒尼爾史特林**：公司的辦公室在霞關，主要做日美之間的商品進出口。

**李杰**：原來如此。

**旁白**：李杰點點頭。

**納撒尼爾史特林**：我雇用了大約二十名日本員工。

**旁白**：納撒尼爾停頓了一下，眼神變得黯淡。

**納撒尼爾史特林**：但在八個月前...其中一位員工失蹤了。

**旁白**：他的聲音變得低沉。

**納撒尼爾史特林**：不只是他，還有他的妻子和兩個孩子。

**納撒尼爾史特林**：一家四口，就這樣憑空消失了。

**旁白**：李杰的心跳瞬間加速。

**旁白**：一家四口...八個月前...

**旁白**：那不就是...9月23日那天嗎？

**李杰**：他們是...在9月23日失蹤的嗎？

**納撒尼爾史特林**：對。

**旁白**：納撒尼爾看向李杰，眼神銳利。

**納撒尼爾史特林**：就是你搭上那班列車的同一天。

**旁白**：李杰沉默了幾秒。

**旁白**：腦中的畫面開始浮現。

**旁白**：那個晚上，列車到達幽冥之都站後，車廂裡爆發了衝突。

**旁白**：當他趕回月台時，看到一個女人跪在地上，膝蓋流著血。

**旁白**：兩個孩子哭著撲向她。

**旁白**：而那個女人絕望地看著遠去的列車，她的丈夫還被困在車上。

**旁白**：後來，他陪著那個母親和兩個孩子一起走。

**旁白**：在林中小徑上，那個母親介紹了自己——智子，還有女兒陽菜和兒子翔太。

**旁白**：他們在林中小屋休息，李杰獨自前往亡者峽谷尋找她的丈夫。

**旁白**：然後...峽谷旁的鐵軌上。

**旁白**：那具冰冷的屍體。

**旁白**：胸口被劍貫穿的致命傷。

**旁白**：那張因死亡而蒼白的臉。

**旁白**：小林先生...

**旁白**：李杰感到胃部一陣翻攪。

**旁白**：但隨即，另一個念頭閃過。

**旁白**：等等...他後來時間旅行了。

**旁白**：回到了列車剛到達那個世界的時候。

**旁白**：也就是說...小林先生的命運可能改變了？

**旁白**：他可能...還活著？

**納撒尼爾史特林**：李先生？

**旁白**：納撒尼爾的聲音把李杰拉回現實。

**李杰**：抱歉，我在回想。

**旁白**：李杰深吸一口氣。

**納撒尼爾史特林**：宇佐美警部說，你在那班列車上。

**旁白**：納撒尼爾從西裝內袋掏出一個信封。

**旁白**：他從信封裡拿出一張照片，放在桌上。

**納撒尼爾史特林**：這是我的員工，小林雄大。

**旁白**：李杰看向照片。

**旁白**：那是一個男人的證件照，穿著深色西裝，表情嚴肅。

**旁白**：方正的臉型，短髮整齊地梳向後方。

**旁白**：眉宇間帶著一股沉穩的氣質。

**旁白**：李杰的呼吸停滯了一瞬間。

**旁白**：果然是他。

**旁白**：那個他在列車上見過的父親。

**旁白**：智子的丈夫。

**旁白**：陽菜和翔太的父親。

**旁白**：那張他在峽谷鐵軌旁看到的、因死亡而蒼白的臉。

**旁白**：原來...他叫雄大。

**納撒尼爾史特林**：你...見過他嗎？

**旁白**：納撒尼爾緊盯著李杰，語氣充滿期待。

**李杰**：我...有印象。

**旁白**：李杰緩緩說道。

**李杰**：那天晚上，車上確實有一家四口。

**李杰**：父親、母親，還有兩個孩子。

**納撒尼爾史特林**：對！就是他們！

**旁白**：納撒尼爾向前傾身。

**納撒尼爾史特林**：你知道他們現在在哪裡嗎？他們還好嗎？

**旁白**：李杰感到喉嚨一緊。

**旁白**：他該怎麼說？

**旁白**：說他看到雄大死在峽谷旁的鐵軌上？

**旁白**：但那是時間旅行之前的事...

**旁白**：現在的時間線裡，他可能還活著...

**旁白**：而且，他根本不能告訴別人自己時間旅行的事。

**李杰**：我...不確定。

**旁白**：李杰選擇了含糊的回答。

**李杰**：到達那個世界後，大家很快就分散了。

**李杰**：我後來沒有再見過他們。

**旁白**：這不算說謊。

**旁白**：至少在這個時間線裡，他確實不知道那家人的狀況。

**納撒尼爾史特林**：這樣啊...

**旁白**：納撒尼爾的眼神黯淡下來。

**旁白**：他拿起照片，看著照片中的男人。

**納撒尼爾史特林**：小林是個很優秀的人。

**旁白**：他的聲音變得柔和。

**納撒尼爾史特林**：他在我公司工作了七年。從普通職員一路升到經理。

**納撒尼爾史特林**：現在負責管理一個十人的團隊。

**旁白**：納撒尼爾嘆了口氣。

**納撒尼爾史特林**：工作認真，待人親切。

**納撒尼爾史特林**：他的妻子智子也是個溫柔的人。

**納撒尼爾史特林**：兩個孩子...陽菜和翔太...都很懂事。

**旁白**：他的語氣充滿感傷。

**納撒尼爾史特林**：我去過他們家幾次。

**納撒尼爾史特林**：那是一個溫暖的家庭。

**旁白**：宇佐美在旁邊默默聽著，沒有打斷。

**旁白**：李杰看著照片中那張熟悉的臉。

**旁白**：小林雄大。

**旁白**：不只是峽谷旁的一具屍體。

**旁白**：他是一個經理，一個父親，一個丈夫。

**旁白**：李杰感到心中一陣沉重。

**納撒尼爾史特林**：自從他們失蹤後，我一直在尋找線索。

**旁白**：納撒尼爾收起照片。

**納撒尼爾史特林**：警方找不到任何蛛絲馬跡。

**納撒尼爾史特林**：就好像他們從這個世界上蒸發了一樣。

**納撒尼爾史特林**：直到我遇到了宇佐美警部。

**旁白**：他看向宇佐美。

**宇佐美誠一郎**：我在調查失蹤案件時，發現了一個共同點。

**旁白**：宇佐美接過話題。

**宇佐美誠一郎**：所有失蹤者都在9月23日晚上搭了同一班列車。

**宇佐美誠一郎**：包括小林一家。

**納撒尼爾史特林**：所以我想...或許你能提供一些資訊。

**旁白**：納撒尼爾看向李杰。

**納撒尼爾史特林**：任何關於那個世界的事...任何關於小林一家的線索...

**納撒尼爾史特林**：我都想知道。

**旁白**：他的眼神懇切。

**旁白**：李杰看著眼前這個真誠的美國商人。

**旁白**：他能理解對方的心情。

**旁白**：失去一個重要的員工，一個視如家人的朋友。

**旁白**：那種無助和焦慮。

**旁白**：但他該說多少？

**旁白**：該如何在不透露時間旅行的情況下，提供有用的資訊？

**旁白**：壽司師傅開始上菜。

**旁白**：一盤精緻的握壽司放在三人面前。
