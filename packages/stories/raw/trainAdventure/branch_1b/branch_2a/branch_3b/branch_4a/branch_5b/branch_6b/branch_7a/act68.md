# 第六十八幕：歸途上的陰影

**旁白**：晚上九點，晚餐結束了。

**旁白**：六個人在餐廳門口互相道別。夜晚的銀座依然繁華，街道上人來人往。

**清水浩介**：今天真的很開心。好久沒這樣聚會了。

**清水由香里**：謝謝大家！下次有機會一定要來我們家玩！

**原田繪里花**：一定會的。今天很高興認識各位。

**凌墨**：李桑，原田桑，保重。

**清水恵美**：再見！路上小心！

**旁白**：大家揮手告別。

**旁白**：浩介和由香里往日比谷線的方向走去，他們要搭日比谷線到茅場町，再換東西線回西船橋。

**旁白**：繪里花也朝著地鐵的方向離開，她要搭地鐵回品川。

**原田繪里花**：李桑，那我先走了。今晚很開心。

**李杰**：嗯，路上小心。

**旁白**：繪里花對他微笑，然後轉身離開。

**旁白**：李杰、凌墨和恵美則一起走進銀座站。

**旁白**：三個人走在車站的通道裡。周圍是下班的人潮，腳步聲在寬敞的地下空間中迴盪。

**清水恵美**：今天真的很開心！浩介大哥看起來過得很幸福呢。

**凌墨**：嗯。

**清水恵美**：欸，我想去那邊的珍珠奶茶店買杯飲料。你們等我一下好嗎？

**旁白**：恵美指著通道旁的一家珍珠奶茶店。

**凌墨**：好，我們在這裡等妳。

**旁白**：恵美快步走向奶茶店。

**旁白**：凌墨和李杰站在通道邊等待。人群持續流動，但沒有人注意他們。

**旁白**：凌墨突然轉過頭，認真地看著李杰。

**凌墨**：李桑，剛才在餐廳...你問我們有沒有被跟蹤。

**旁白**：凌墨的聲音壓得很低。

**凌墨**：到底發生什麼事了？

**旁白**：李杰猶豫了一下，看了看奶茶店的方向。恵美還在排隊點餐。

**李杰**：...從五月開始，我偶爾會感覺到有人在跟蹤我。

**凌墨**：跟蹤？

**李杰**：不是每天，也不是固定的時間。就是...偶爾會有那種被注視的感覺。

**旁白**：凌墨的表情變得嚴肅。

**凌墨**：你看到跟蹤的人了嗎？

**李杰**：沒有。每次回頭都找不到可疑的人。可能只是我太敏感了。

**凌墨**：...

**旁白**：凌墨沉默了幾秒。

**凌墨**：你有什麼想法嗎？是誰在跟蹤你？

**李杰**：不知道。也許...和那些黑西裝男人有關？

**凌墨**：有可能。

**旁白**：凌墨皺起眉頭。

**凌墨**：李桑，如果有任何危險，一定要聯絡我。不要一個人面對。

**李杰**：我知道。謝謝你，凌墨桑。

**旁白**：凌墨點點頭。

**凌墨**：還有...不要告訴恵美這件事。

**李杰**：嗯？

**凌墨**：她好不容易才走出來。我不想讓她再擔心這些事。

**旁白**：李杰明白了。凌墨是在保護恵美。

**李杰**：我懂。對了...恵美桑她...真的已經...

**旁白**：李杰不知道該怎麼措辭。

**凌墨**：她已經決定要往前走了。

**旁白**：凌墨的聲音很溫柔。

**凌墨**：翔不會回來了。恵美花了很長時間才接受這個事實。但她現在明白，她必須繼續生活。為了翔，也為了她自己。

**李杰**：...這樣啊。

**凌墨**：我會陪著她。我們會一起建立新的生活。

**旁白**：李杰點點頭。他能感受到凌墨對恵美的關心和承諾。

**旁白**：這時，恵美拿著一杯珍珠奶茶走了回來。

**清水恵美**：讓你們久等了！

**旁白**：她笑著走過來。凌墨的表情立刻恢復了平靜。

**凌墨**：沒事。我們該走了。

**旁白**：三個人繼續往前走，來到丸之內線的月台入口。

**旁白**：這裡分成兩個方向——一個通往池袋方向，一個通往荻窪/新宿方向。

**凌墨**：李桑，你要往荻窪方向吧？

**李杰**：對。你們要去池袋？

**清水恵美**：嗯，我們要在池袋換東武線回川越。

**旁白**：三個人在岔路口停下。

**清水恵美**：李桑，今天真的很開心！下次再一起出來玩吧！

**李杰**：好。你們也路上小心。

**凌墨**：保重。有事隨時聯絡。

**旁白**：凌墨認真地看著李杰，眼神中帶著擔憂。

**李杰**：我會的。

**旁白**：凌墨和恵美轉身，走向池袋方向的月台。

**旁白**：李杰看著他們的背影消失在人群中，然後轉身走向另一個方向。

---

**旁白**：李杰走下荻窪方向的月台。

**旁白**：月台上等車的人不少，大多是下班的上班族。

**旁白**：幾分鐘後，列車進站了。

**旁白**：李杰上車，找了個靠窗的座位坐下。

**旁白**：列車啟動，駛離銀座站。

**旁白**：經過赤坂見附、四谷...

**旁白**：四谷站到了。李杰站起身，走下車。

**旁白**：這裡是丸之內線和中央線的轉乘站。李杰跟著人群走向中央線的月台。

---

**旁白**：中央線月台。

**旁白**：李杰等待開往高尾方向的列車。

**旁白**：月台上人不多，稀稀落落地站著幾個等車的乘客。

**旁白**：過了幾分鐘，列車進站了。

**旁白**：李杰上車，找了個靠窗的座位坐下。

**旁白**：列車啟動，駛離四谷站。

**旁白**：李杰看著窗外飛逝的景色，思緒飄遠。

**旁白**：今天晚上的聚會很愉快。見到老朋友，聊聊近況，享受美食...這就是普通的、珍貴的日常。

**旁白**：但同時，那些疑問依然存在。

**旁白**：被跟蹤的感覺，那個可疑的中年男人，還有...

**旁白**：就在這時，李杰突然感覺到了。

**旁白**：那種熟悉的、被注視的感覺。

**旁白**：他的背脊一陣發涼。

**旁白**：有人在看著他。

**旁白**：李杰努力保持鎮定。他沒有立刻回頭，而是裝作看手機的樣子，用餘光觀察車廂內的其他乘客。

**旁白**：車廂裡人不多。大部分是下班的上班族，還有幾個學生模樣的年輕人。

**旁白**：沒有什麼特別可疑的人。

**旁白**：列車繼續前進，經過新宿、中野...

**旁白**：那種被注視的感覺越來越強烈。

**旁白**：李杰開始更仔細地觀察。他假裝站起來伸懶腰，趁機環顧整個車廂。

**旁白**：然後，他注意到了。

**旁白**：在車廂的另一端，靠近門口的位置，站著一個年輕女孩。

**旁白**：她看起來二十多歲，身材很好，穿著休閒的牛仔褲和黑色夾克。

**旁白**：黑色的長髮披在肩上，五官精緻，但表情冷淡，沒有任何笑容。

**旁白**：她低著頭，手裡拿著手機，手指在螢幕上快速滑動，像是在玩遊戲或是看什麼東西。

**旁白**：李杰盯著那個女孩看了幾秒。

**旁白**：她一直低著頭玩手機，沒有抬起頭來。

**旁白**：也許只是巧合？也許她只是剛好要去三鷹或更遠的地方？

**旁白**：但李杰的直覺告訴他，不對勁。

**旁白**：這個女孩從四谷就上車了。她一直站在那個位置，一直低頭玩手機。

**旁白**：而且...她給人一種很奇怪的感覺。那種過於刻意的自然，那種假裝在做其他事情的樣子。

**旁白**：列車的廣播響起。

**廣播**：下一站，三鷹，三鷹。

**旁白**：李杰的心跳加速。

**旁白**：如果這個女孩真的在跟蹤他，那麼她應該也會在三鷹下車。

**旁白**：李杰站起身，拿起包包，走向車門。

**旁白**：他刻意不看那個女孩的方向，但用餘光注意著她的動作。

**旁白**：列車減速，平穩地停靠在月台旁。

**旁白**：車門打開。

**旁白**：李杰的手握緊包包帶，準備踏出車廂。

**旁白**：他的餘光緊盯著車廂另一端那個低頭玩手機的女孩。

**旁白**：她會下車嗎？

**旁白**：還是...這一切只是他的多疑？
