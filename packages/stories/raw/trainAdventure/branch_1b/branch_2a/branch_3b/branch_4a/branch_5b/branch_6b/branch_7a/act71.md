# 第七十一幕：台場的夜晚

**旁白**：逛完街後,已經是下午四點多了。

**旁白**：秋天的傍晚來得很早,天色已經開始漸漸變暗。

**李杰**：去吃晚餐吧。想吃什麼?

**原田繪里花**：嗯...我想吃義大利麵!

**李杰**：那我們去台場吧。那邊有家不錯的義大利餐廳,還能看海景。

**原田繪里花**：台場!好啊!我好久沒去台場了!

---

**旁白**：兩人搭上百合海鷗號,前往台場。

**旁白**：列車行駛在高架軌道上,窗外是東京的城市景觀。

**旁白**：高樓大廈、街道、公園...一切都在腳下展開,像是在俯瞰整個城市。

**旁白**：此時已是傍晚時分,天空染上了橙紅色的晚霞。

**旁白**：經過彩虹大橋時,繪里花興奮地指著窗外。

**原田繪里花**：李桑你看!彩虹大橋!還有晚霞!

**李杰**：嗯。

**旁白**：夕陽的餘暉灑在海面上,波光粼粼。遠處可以看到自由女神像的複製品和摩天輪的輪廓。

**旁白**：台場的景色在黃昏時分顯得格外美麗。

**旁白**：台場海濱公園站到了。

---

**旁白**：餐廳位於購物中心的三樓,裝潢簡約而溫馨。

**旁白**：最重要的是,這裡有大片的落地窗,可以看到台場的海景。

**旁白**：兩人坐在靠窗的位置。

**旁白**：窗外是東京灣,海水在黃昏的餘暉下閃耀著金紅色的光芒。遠處是彩虹大橋,還有對岸高樓林立的東京市區。

**旁白**：天色漸暗,城市的燈光開始陸續亮起。

**旁白**：海鷗在空中盤旋,偶爾傳來悠長的叫聲。

**旁白**：兩人點了餐——繪里花點了海鮮義大利麵,李杰點了波隆納肉醬麵。

**旁白**：等待餐點的時候,他們靜靜地看著窗外的景色。

**原田繪里花**：好漂亮...黃昏的台場真美...

**旁白**：繪里花喃喃地說。

**李杰**：嗯。

**旁白**：餘暉從窗外灑進來,照在桌上的白色桌布上,染上了一層淡淡的金色。

**旁白**：這一刻很安靜,但不會讓人覺得尷尬。

**旁白**：只是兩個人一起,靜靜地享受這份時光。

---

**旁白**：服務生端來了他們點的義大利麵。

**原田繪里花**：看起來好好吃!

**旁白**：繪里花開心地說。

**旁白**：兩人開始用餐,邊吃邊聊著各種話題——工作、電影、最近的新聞...

**旁白**：對話很輕鬆,氣氛很溫馨。

**旁白**：窗外的天色逐漸從金紅色轉為深藍,城市的燈光越來越明亮。

**旁白**：彩虹大橋的燈光完全點亮了,在夜空中顯得格外美麗。

**原田繪里花**：今天真的很開心!好久沒有這樣逛街了!

**李杰**：嗯,我也是。

**旁白**：繪里花笑著喝了一口水。

**原田繪里花**：對了,李桑,你聖誕節有什麼計畫嗎?

**旁白**：李杰的心跳突然漏了一拍。

**旁白**：聖誕節...那是他和海斗約定要告白的日子。

**李杰**：...還沒想好。妳呢?

**原田繪里花**：我也還不確定。不過聖誕節的東京應該很漂亮吧!到處都是燈飾!

**李杰**：嗯。

**原田繪里花**：如果到時候李桑有空的話...我們要不要一起去看燈飾?

**旁白**：繪里花有些害羞地問。

**旁白**：李杰看著她,心中突然有種衝動。

**旁白**：也許...也許這就是機會?

**旁白**：也許他應該在那天,在燈飾下,告訴她...

**李杰**：...好。

**原田繪里花**：真的嗎!太好了!

**旁白**：繪里花開心地笑了,笑容像陽光一樣明亮。

**旁白**：李杰看著她的笑容,心中充滿了溫暖。

**旁白**：他喜歡這樣的時光。喜歡和她在一起的感覺。

**旁白**：喜歡看到她的笑容,喜歡聽到她的聲音。

**旁白**：也許...這就是喜歡吧。

---

**旁白**：吃完晚餐後,已經是晚上七點左右了。

**旁白**：外面已經完全天黑了,但台場的燈光讓這裡依然明亮。

**原田繪里花**：李桑,我們去海邊走走吧!

**李杰**：好。

**旁白**：兩人走出購物中心,來到台場海濱公園。

**旁白**：公園沿著東京灣而建,有寬闊的步道和草地。

**旁白**：夜晚的台場燈火通明,彩虹大橋的燈光在夜空中顯得格外美麗。

**旁白**：海風吹來,帶著一絲涼意。

**原田繪里花**：有點冷呢...

**旁白**：繪里花拉緊了圍巾。

**李杰**：要不要回去?

**原田繪里花**：不要!難得來台場,我想多待一會兒!

**旁白**：繪里花堅定地說。

**旁白**：兩人沿著海濱步道慢慢走著。

**旁白**：海浪拍打著岸邊,發出規律的聲音。

**旁白**：遠處的東京鐵塔也亮起了燈光,在夜色中閃耀著。

**旁白**：彩虹大橋、摩天輪、高樓大廈...一切都被燈光點亮,像是一幅美麗的夜景畫。

**原田繪里花**：好漂亮...

**旁白**：繪里花停下腳步,看著眼前的景色。

**旁白**：橋上的燈光倒映在海面上,隨著波浪輕輕搖曳。

**旁白**：李杰也停下來,站在她旁邊。

**旁白**：兩人並肩站著,靜靜地看著夜景。

**旁白**：這一刻很安靜。只有海浪聲和遠處傳來的城市喧囂。

**旁白**：但在這裡,卻有種遠離塵囂的寧靜感。

**原田繪里花**：李桑...

**李杰**：嗯?

**原田繪里花**：謝謝你今天陪我。

**旁白**：繪里花輕聲說。

**原田繪里花**：我很開心。

**李杰**：...我也是。

**旁白**：李杰看著她。

**旁白**：夜晚的燈光映在她的臉上,讓她看起來格外溫柔。

**旁白**：他想說些什麼,想告訴她...

**旁白**：但話到嘴邊,又嚥了回去。

**旁白**：還沒準備好。

**旁白**：他還沒有足夠的勇氣。

**旁白**：也許...再過一段時間。等他準備好的時候。

**旁白**：海風吹過,帶起繪里花的髮絲。

**旁白**：她伸手撥了撥頭髮,轉頭看著李杰。

**原田繪里花**：李桑,你說...我們以後也會常常這樣出來嗎?

**李杰**：...嗯?

**原田繪里花**：就是...像今天這樣,一起看電影、逛街、吃飯...

**旁白**：繪里花的聲音有些不確定。

**原田繪里花**：我很喜歡這樣的時光。和李桑在一起,感覺很輕鬆,很開心。

**李杰**：我也是。

**旁白**：李杰認真地說。

**李杰**：我也很喜歡和妳在一起。

**旁白**：繪里花愣了一下,然後笑了。

**原田繪里花**：那太好了!那我們以後也要常常這樣!

**旁白**：也許,只要準備好了,任何時候都可以。

**李杰**：還好。為什麼突然問這個?
