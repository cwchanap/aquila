# 第七十九幕：意外發現

**旁白**：提交證據給警方之後，李杰又回到了平靜的生活。

**旁白**：一月的東京，天氣依然寒冷，但陽光明媚的日子漸漸多了起來。

**旁白**：李杰每天上班、下班，和繪里花偶爾約會，和海斗聊天吃飯。

**旁白**：那個神秘的網址，那些黑西裝男人，齋藤大輔的死...這些事情仍然像陰影一樣存在，但至少現在交給警方處理了。

**旁白**：李杰試著不去想太多，只想好好享受現在的生活。

---

**旁白**：一月下旬，某個週四下午。

**旁白**：辦公室裡，李杰正在處理一個報表，海斗在旁邊的座位上敲著鍵盤。

**中野海斗**：杰，這個週末有什麼計劃嗎？

**旁白**：海斗突然問道，轉過椅子看著李杰。

**李杰**：沒什麼特別的...可能就在家裡休息吧。

**中野海斗**：那要不要一起出去玩？好久沒一起出去了。

**旁白**：海斗撓了撓頭。

**中野海斗**：嗯...其實我也不知道去哪裡。不然我去你家玩？我還沒去過你家呢。

**李杰**：我家？

**旁白**：海斗笑著說。

**中野海斗**：我們可以在你家看電影、打遊戲，比在外面跑來跑去輕鬆多了。

**李杰**：也好...我最近買了一台五十二吋的電視，看電影效果應該不錯。

**旁白**：李杰想了想，點點頭。

**李杰**：而且我前陣子還養了一隻貓。

**中野海斗**：貓？你養貓了？真的？太好了！

**旁白**：海斗的眼睛發亮。

**李杰**：嗯，一隻橘貓。前陣子從社區的動物領養中心領養的，看起來很親人就帶回家了。

**中野海斗**：我超喜歡貓的！那更要去看看了！

**旁白**：海斗興奮地說，然後看向繪里花的座位。

**中野海斗**：繪里花！這週末我們要去杰家玩，妳有興趣嗎？

**旁白**：繪里花抬起頭，看向海斗，臉微微紅了。

**原田繪里花**：去...去李桑家？真的可以嗎？

**李杰**：嗯...如果妳想來的話...

**旁白**：李杰也有些不好意思。

**中野海斗**：當然好啊！我們可以一起做飯、看電影，很有趣的！而且杰養了一隻貓！妳喜歡貓嗎？

**旁白**：海斗笑著說。

**原田繪里花**：喜歡！我超喜歡貓的！那...我可以去嗎？好期待喔...

**旁白**：繪里花看向李杰，眼神裡帶著期待，聲音裡透著興奮。

**李杰**：當然可以。

**旁白**：李杰點點頭。

**中野海斗**：太好了！那就這麼說定了！我們可以買點食材，在你家做壽喜燒！

**旁白**：海斗拍了拍手。

**中野海斗**：我最近很想吃壽喜燒，但一個人吃太孤單了。三個人一起吃，才有氣氛嘛！

**原田繪里花**：聽起來好棒！我也很久沒吃壽喜燒了呢！

**旁白**：繪里花也笑了，眼睛亮晶晶的。

**李杰**：那好吧，就這麼辦。

---

**旁白**：週六上午十一點，李杰在家裡打掃。

**旁白**：他的公寓不大，一房一廳一廚一衛，大約三十五平方公尺。客廳和臥室之間只有一道拉門，廚房是開放式的，連著客廳。

**旁白**：客廳裡，新買的五十二吋電視掛在牆上，下面是一個簡單的電視櫃。沙發是深灰色的兩人座，前面擺著一張小茶几。

**旁白**：一隻胖胖的橘貓趴在沙發上，懶洋洋地打著哈欠。

**李杰**：小橘，等一下有客人來，你要乖一點。

**旁白**：李杰一邊擦桌子一邊對貓說。

**旁白**：橘貓看了他一眼，翻了個身，繼續睡覺。

**旁白**：李杰笑了笑，繼續打掃。

---

**旁白**：中午十二點，門鈴響了。

**旁白**：李杰打開門，海斗和繪里花站在門外，手裡提著好幾個購物袋。

**中野海斗**：打擾了！

**旁白**：海斗笑著走進來。

**原田繪里花**：打擾了...哇，好寬敞！

**旁白**：繪里花跟在後面，有些緊張但又好奇地看著李杰的公寓。

**李杰**：歡迎。進來吧。

**旁白**：兩人脫下鞋子，走進客廳。繪里花把手提袋放在廚房的椅子上，然後走回來。海斗看著牆上的電視，眼睛發亮。

**中野海斗**：哇！電視好大！五十二吋真的很爽！

**原田繪里花**：李桑的房間...很整潔呢。比我想像中還要乾淨！

**旁白**：繪里花環顧四周，臉上帶著驚訝和讚賞的笑容。

**李杰**：還好吧...剛才特地打掃了一下。

**中野海斗**：咦，貓呢？

**李杰**：在那邊。

**旁白**：李杰指向沙發。橘貓還趴在沙發上，瞇著眼睛看著這些陌生人。

**中野海斗**：哇！好可愛！來，讓我摸摸。

**旁白**：海斗走過去，蹲下來伸出手。橘貓聞了聞海斗的手，然後慢慢靠過來，讓海斗摸牠的頭。

**中野海斗**：好乖！叫什麼名字？

**李杰**：小橘。

**中野海斗**：小橘...真是個簡單直接的名字。

**旁白**：海斗笑著說。

**原田繪里花**：哇！好可愛喔！牠的毛色好漂亮...可以讓我也摸摸嗎？

**旁白**：繪里花蹲下來，眼睛發亮，小心翼翼地伸出手。

**旁白**：小橘看了看繪里花，然後主動蹭了蹭她的手。

**原田繪里花**：哇！牠好親人喔！而且好軟...

**旁白**：繪里花輕輕地摸著小橘的背，臉上露出幸福的笑容。小橘舒服地發出呼嚕聲。

---

**旁白**：中午過後，三人開始準備壽喜燒。

**旁白**：海斗帶來了牛肉片、豆腐、大蔥、香菇、白菜、粉絲，還有壽喜燒的醬汁。

**原田繪里花**：哇！食材好豐富！海斗桑準備得真周到！

**旁白**：繪里花看著滿桌的食材，驚喜地說。

**旁白**：繪里花負責洗菜和切菜，李杰準備鍋子和餐具，海斗則在客廳設置電火鍋。

**中野海斗**：杰，你家有便攜式瓦斯爐嗎？

**李杰**：有，在廚房櫃子裡。

**旁白**：李杰從櫃子裡拿出瓦斯爐，放在茶几上。

**中野海斗**：完美！

**旁白**：海斗架好鍋子，倒入醬汁，開火。醬汁的香氣很快就飄散開來，甜甜的醬油味混雜著糖和味醂的香氣。

**原田繪里花**：哇！好香喔！聞起來就很好吃的樣子！

**旁白**：繪里花端著切好的蔬菜走出廚房，忍不住深吸了一口氣。

**中野海斗**：來，我們開始吧！

**旁白**：海斗把牛肉片放進鍋裡，肉片在醬汁中慢慢變色，發出滋滋的聲音。

---

**旁白**：三人圍坐在茶几旁，一邊煮一邊聊天。

**旁白**：小橘也湊過來，對鍋裡的食物很感興趣，不停地嗅著空氣中的香味。

**原田繪里花**：小橘也想吃呢！好可愛喔...

**旁白**：繪里花看著小橘，笑得很開心。

**中野海斗**：杰，要不要給小橘也吃一點？

**李杰**：不行，貓不能吃太鹹的東西。

**原田繪里花**：對喔，貓咪不能吃人類的食物呢。小橘乖乖，等一下給你貓罐頭好不好？

**旁白**：繪里花輕輕地摸了摸小橘的頭。壽喜燒煮得差不多了，三人開始夾菜吃。牛肉嫩滑，蔬菜清甜，醬汁濃郁。

**原田繪里花**：嗯～好好吃！這個豆腐也太入味了吧！

**旁白**：繪里花吃了一口豆腐，臉上露出滿足的笑容，眼睛都瞇起來了。

**原田繪里花**：還有這個牛肉...入口即化！海斗桑帶來的食材真的很棒呢！

**中野海斗**：對吧！壽喜燒就是要這樣一起吃才有感覺！我們來看部電影吧？

**旁白**：海斗笑著說，然後打開電視。

**李杰**：你想看什麼？

**中野海斗**：StreamHub上有什麼推薦的嗎？

**旁白**：李杰拿起遙控器，打開StreamHub。

**李杰**：動作片？還是喜劇？

**原田繪里花**：喜劇好像不錯呢！吃飽了看輕鬆的電影最舒服了！

**中野海斗**：那就看個輕鬆的喜劇吧！

**旁白**：李杰選了一部評價不錯的喜劇電影，按下播放。

---

**旁白**：電影開始播放，三人一邊吃著壽喜燒，一邊看著螢幕。

**旁白**：電影裡的笑點讓海斗和繪里花笑得很開心，李杰也跟著笑了幾次。

**旁白**：小橘趴在沙發上，懶洋洋地看著這些人類。

**旁白**：氣氛很輕鬆，很溫馨。

**旁白**：過了一會兒，海斗放下碗筷。

**中野海斗**：杰，你家有飲料嗎？我有點渴。麻煩你幫我拿一罐可樂！

**李杰**：有，冰箱裡有可樂和茶。好。

**旁白**：李杰站起身，走向廚房。

---

**旁白**：李杰打開冰箱，拿出一罐可樂。

**旁白**：就在他關上冰箱門準備回客廳的時候，他注意到廚房角落有什麼動靜。

**旁白**：小橘不知道什麼時候跑到那裡，正在玩一個提袋。

**旁白**：那是繪里花的包。她剛才進門時把包放在廚房的椅子上，現在被小橘拖到地上。

**李杰**：小橘，不可以玩別人的東西。

**旁白**：李杰走過去，想把小橘抱開。

**旁白**：就在這時，他看到從包裡露出一角照片。

**旁白**：小橘的爪子勾著照片，似乎想把它拖出來玩。

**旁白**：李杰皺了皺眉，把小橘抱到一邊，然後拿起那張照片。

**旁白**：照片有些舊了，邊緣微微泛黃。照片裡是兩個女孩，站在某個遊樂園的摩天輪前，笑得很開心。

**旁白**：其中一個女孩...是繪里花。

**旁白**：李杰停下動作，仔細看著照片。

**旁白**：另一個女孩...黑色長髮，笑容明媚，穿著牛仔褲和黑色夾克。

**旁白**：李杰愣住了。

**旁白**：那個女孩...是那個在電車上跟蹤他的女孩。

**旁白**：那個在十一月某個夜晚，在中央線電車上，一直盯著他看的女孩。

**旁白**：為什麼繪里花會和那個女孩有合照？

**旁白**：她們是什麼關係？

**中野海斗**：杰！可樂呢？

**旁白**：海斗的聲音從客廳傳來，打斷了李杰的思緒。

**旁白**：李杰看著手中的照片，腦中充滿了疑問。

**旁白**：客廳裡傳來電影的音效和海斗的笑聲。

**旁白**：繪里花也在那裡，正笑著看電影。

**旁白**：那個讓他心動的女孩。

**旁白**：那個溫柔善良的女孩。

**旁白**：李杰低頭看著手中的照片，照片裡的兩個女孩笑得那麼開心。

**旁白**：這是巧合嗎？還是...

**旁白**：李杰皺起了眉頭，陷入了沉思。
