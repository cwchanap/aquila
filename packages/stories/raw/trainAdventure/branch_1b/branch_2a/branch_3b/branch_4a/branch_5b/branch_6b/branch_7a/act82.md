# 第八十二幕：綁架

**旁白**：下午六點，李杰換上旅館提供的浴衣，準備去一樓的宴會廳吃晚餐。他走出房間，沿著二樓的走廊往樓梯方向走去。

**旁白**：走廊很安靜，鋪著深色的地毯，牆上掛著日式畫作。兩側是一間間客房，大部分門都關著。

**旁白**：就在這時，李杰看到一隻貓。一隻黑白相間的貓，從走廊另一端快速跑過來，然後鑽進了一個房間。那個房間的門是開著的。

**李杰**：嗯？

**旁白**：李杰有些好奇。那個房間的門為什麼開著？他走向那個房間，想看看是怎麼回事。

**旁白**：李杰走到門口，往裡面看了一眼。下一秒，他整個人僵住了。

---

**旁白**：房間裡，一個年輕女孩倒在榻榻米上。她穿著黑色的夾克和牛仔褲，黑色的長髮散落在地上，一動不動。

**李杰**：喂！妳還好嗎？

**旁白**：李杰立刻衝進房間，跑到女孩身邊蹲下。他伸手搖了搖女孩的肩膀，但她沒有反應。

**李杰**：喂！醒醒！

**旁白**：李杰試圖叫醒她，但女孩依然沒有動靜。他趕緊檢查她的呼吸——還好，還在呼吸，只是失去了意識。

**旁白**：李杰鬆了口氣，然後仔細看了看女孩的臉。下一秒，他的瞳孔驟然收縮。

**旁白**：這張臉...他見過。黑色長髮，二十多歲，清秀的五官，表情冷淡...這是那個在電車上跟蹤他的女孩！

**旁白**：李杰的腦中閃過那天晚上在中央線上看到她的畫面——她站在車廂另一端，低頭玩手機，但視線一直放在他身上。

**旁白**：還有繪里花包裡的那張照片——這個女孩和繪里花站在摩天輪前，笑得那麼開心。

**旁白**：為什麼她會在這裡？為什麼她會倒在這個房間裡？李杰的心中充滿了疑惑，但現在不是想這些的時候。她需要幫助。

**李杰**：我得叫人來...

**旁白**：李杰轉身看向房間角落的內線電話，準備打電話給旅館櫃台。但就在他起身走向電話的瞬間——一陣劇烈的電流從背後襲來。

**旁白**：李杰的身體猛地一僵，整個人向前倒下。他的視線開始模糊，身體失去了控制。

**旁白**：然後，一切陷入黑暗。

---

**旁白**：不知道過了多久。李杰的意識逐漸恢復。他聽到了聲音——一個年輕女孩的聲音，很冷靜。

**女孩**：醒醒。

**旁白**：李杰努力睜開眼睛。視線很模糊，他只能看到昏暗的光線和模糊的輪廓。他的頭很痛，身體很沉重，手腳被什麼東西綁住了，動不了。

**旁白**：李杰眨了眨眼，視線漸漸清晰。他發現自己坐在地上，雙手被繩子綁在背後，背後靠著另一個人——那個女孩。他們背靠背被綁在一起，坐在一個狹小的房間裡。

**旁白**：房間很小，大約只有三坪大，沒有窗戶，只有一盞昏暗的燈泡懸掛在天花板上。牆壁是灰色的水泥，地板是冰冷的磁磚。房間裡什麼都沒有，只有他們兩個。

**女孩**：你醒了。

**旁白**：女孩的聲音從背後傳來，很平靜。

**李杰**：...我...

**旁白**：李杰的喉嚨很乾，聲音沙啞。

**李杰**：這是哪裡...發生了什麼事...

**女孩**：我們被綁架了。

**旁白**：女孩的聲音很平靜，沒有什麼情緒波動。

**李杰**：綁架？

**女孩**：對。被一個叫「影會」的組織綁架了。

**李杰**：影會？

**女孩**：一個邪惡的組織。他們做了很多壞事。綁架、殺人、人體實驗...

**旁白**：女孩的聲音依然平靜，彷彿在陳述事實。

**女孩**：我一直在調查他們，但被他們發現了。抱歉，連累了你。

**李杰**：妳是誰？

**旁白**：李杰強迫自己冷靜下來，開始思考現在的狀況。

**女孩**：我叫長谷川澪。

**李杰**：長谷川...

**旁白**：李杰想起了那張照片。

**李杰**：妳和原田繪里花認識？

**長谷川澪**：咦？你怎麼知道...

**旁白**：澪的聲音裡帶著驚訝。

**李杰**：我看過妳們的合照。

**長谷川澪**：...是嗎...

**旁白**：澪沉默了一會兒。

**長谷川澪**：你就是李杰吧？

**李杰**：妳怎麼知道我的名字？

**長谷川澪**：委託人告訴我的。而且...我一直在監視你。

**李杰**：監視我？為什麼？

**旁白**：李杰突然想起了什麼。那天晚上，在中央線上看到她。還有在三鷹站，她也下了車。他以為她在跟蹤他...

**旁白**：還有那天被黑西裝男人襲擊後，他在自己家裡醒來。他一直不明白是誰把他送回家的...原來是她。她一直在暗中保護他。

**長谷川澪**：因為你可能有危險。影會一直在盯著你。

**李杰**：影會為什麼要盯著我？我和他們有什麼關係？

**長谷川澪**：不清楚。但他們似乎對你很感興趣。

**李杰**：妳知道什麼就直說。現在不是隱瞞的時候。

**長谷川澪**：我真的不知道太多。我只是被人委託監視你，確保你的安全。但我也不知道為什麼。

**李杰**：被誰委託？

**長谷川澪**：不能說。

**李杰**：影會到底是什麼組織？他們為什麼要抓我？

**長谷川澪**：我只知道他們很危險。其他的，我幫不上忙。

**旁白**：澪的語氣始終很平靜，彷彿這一切都在她的預料之中。

**旁白**：李杰沒有再追問。從澪的反應看來，她確實不知道太多。但有一件事是確定的——有人知道李杰有危險，所以才委託澪監視他。

**旁白**：那張照片，那些不經意的擔心，還有今天在列車上的那番話...這一切會不會都有關聯？

---

**旁白**：就在這時，房間的門突然打開了。刺眼的光線從門外射進來，李杰下意識地瞇起眼睛。

**旁白**：一個人影走了進來。李杰看清了那個人——一個中年男人，四十多歲，身材中等，穿著深色的西裝，頭髮梳得整整齊齊。他的臉上掛著笑容，但那笑容看起來很陰險。

**旁白**：在他身後，跟著兩個穿黑色西裝的壯漢。他們面無表情，站在門口，像是保鑣。

**中年男人**：李杰桑，終於醒了啊。

**旁白**：中年男人笑著說，聲音很溫和，但李杰能感覺到那種虛偽。

**李杰**：你是誰？

**中年男人**：我？我叫中村隼人。

**旁白**：中村隼人...李杰沒聽過這個名字。但看著這張臉，他卻覺得很熟悉。削瘦的臉型，陰沉的眼神，還有那種詭異的笑容...

**旁白**：李杰的腦中閃過一個畫面——九月在銀座的壽司餐廳，走廊裡，這個男人從對面走來，那種陰險的、帶著某種意味的笑容...

**旁白**：還有在三號車廂上，那個自私膽怯的中年男人，總是抱怨其他人的決定。是他。

**旁白**：李杰的心一沉。他落入了一個危險的陷阱。
