# 第八十三幕：威脅

**李杰**：你想做什麼？

**旁白**：李杰強迫自己冷靜下來，盯著中村。

**中村隼人**：做什麼？當然是有些事情要和你談談。

**旁白**：中村笑了笑，那笑容依然虛偽。

**中村隼人**：其實我只需要你一個人。但這位長谷川小姐...她實在太煩人了。她一直在監視你，保護你，讓我們很難下手。所以花了不少時間設陷阱，才終於把她抓住。

**旁白**：中村看向被留在隔壁房間的澪的方向。

**李杰**：你到底想幹什麼？

**中村隼人**：別急。我先讓你看點東西。

**旁白**：中村從西裝口袋裡拿出一支手機，操作了幾下。

**旁白**：房間角落的一台投影機亮了起來，在白色的牆壁上投射出影像。

**旁白**：螢幕上顯示著一個房間——和剛才關著李杰和澪的房間很像，灰色的水泥牆，冰冷的地板，昏暗的燈光。但這個房間裡，綁著兩個人。

**旁白**：一個是海斗，一個是繪里花。他們背靠背被綁在房間中央，雙手被繩子綁在背後,嘴巴被膠帶封住。

**李杰**：！

**旁白**：李杰的瞳孔驟然收縮。

**李杰**：繪里花...海斗...

**旁白**：下一秒，隔壁房間傳來一聲悶響。

**長谷川澪**：不可能！

**旁白**：澪的聲音，第一次帶著明顯的震驚。

**旁白**：李杰明白了——投影機的影像穿過牆壁，兩個房間都能看到這個畫面。中村故意讓他們一起看。

**中村隼人**：看來你們都很驚訝啊。

**旁白**：中村滿意地笑了。

**中村隼人**：我來找你，有兩個目的。第一，向你報仇。第二，我需要你的力量，幫我做一件事。

**李杰**：報仇？你在說什麼？

**旁白**：李杰完全不明白。

**李杰**：我和你有什麼仇？我根本不認識你！

**中村隼人**：不認識？哈哈...也許你真的不記得了。

**旁白**：中村的笑容變得更加陰險。

**中村隼人**：沒關係。這次，我知道的比你多。我終於可以站在上風了。

**李杰**：你到底在說什麼？什麼力量？什麼報仇？

**中村隼人**：慢慢來，李杰桑。我會讓你想起來的。但在那之前...我要讓你好好品嚐一下痛苦的滋味。

**旁白**：中村站起身，轉身對門口的黑西裝男人打了個手勢。

**中村隼人**：去準備吧。

**旁白**：黑西裝男人點點頭，轉身離開。

**李杰**：你想幹什麼？

**旁白**：李杰的心臟狂跳。

**中村隼人**：你很快就會知道了。

**旁白**：牆上的投影畫面突然變化。一個黑西裝男人走進了關著繪里花和海斗的房間。他手裡拿著一個塑膠桶。

**旁白**：黑西裝男人把桶放在地上，打開蓋子，開始往地板上倒液體。那是油。黑色的、黏稠的油。他沿著房間的一側倒了一條長長的油線，從門口一直延伸到房間深處。

**旁白**：繪里花和海斗被綁在房間中央，距離油線大約三公尺。黑西裝男人倒完油後，從口袋裡拿出一個打火機。

**李杰**：不...

**旁白**：李杰的聲音顫抖起來。

**旁白**：投影畫面中，黑西裝男人點燃了打火機，然後丟在油線的起點。火焰瞬間沿著油線燃燒起來，發出「呼」的一聲。橘紅色的火光照亮了整個房間。

**旁白**：繪里花和海斗驚恐地掙扎著，但繩子綁得很緊，他們動不了。嘴巴被封住，他們連叫喊都做不到。火焰緩緩地向前蔓延，朝著他們的方向燒過去。

**李杰**：住手！

**旁白**：李杰猛地站起來，但黑西裝男人立刻上前，把他按回椅子上。

**中村隼人**：別激動，李杰桑。

**旁白**：中村笑著說。

**中村隼人**：火焰會慢慢蔓延。按照我的計算，大概五分鐘後，火就會燒到他們身上。

**李杰**：你這個混蛋！為什麼要這麼做？他們和這件事沒有關係！

**旁白**：李杰咬緊牙關，眼中充滿了憤怒。

**中村隼人**：沒有關係？也許沒有。但我就是想讓你痛苦。我想看著你無能為力，看著你在乎的人在你眼前死去。

**旁白**：中村歪著頭，笑得更開心了。然後俯下身，靠近李杰。

**中村隼人**：你會明白我的痛苦的，李杰桑。很快，你就會明白了。

**李杰**：你到底在說什麼？我什麼都沒做過！

**中村隼人**：你會想起來的。

**旁白**：中村直起身，看向門口的兩個黑西裝男人。

**中村隼人**：你們可以走了。

**旁白**：兩個黑西裝男人點點頭，轉身走出房間。門關上了。

**旁白**：房間裡只剩下李杰和中村。李杰依然坐在椅子上，雙手被綁著。中村走到一旁，拉了張椅子坐下，翹起二郎腿。

**中村隼人**：五分鐘。我會在這裡陪你好好欣賞這場表演。

**旁白**：中村的視線轉向牆上的投影。投影機還在運作，畫面持續播放著那個房間的情況。

**旁白**：火焰繼續蔓延，一點一點地逼近繪里花和海斗。他們拼命地掙扎，但繩子紋絲不動。

**旁白**：繪里花的眼中充滿了恐懼，海斗的臉上滿是汗水。

**旁白**：李杰看著牆上的投影，心臟狂跳，腦中一片空白。

**旁白**：五分鐘。他只有五分鐘。但他該怎麼辦？
