# 第十四幕：地圖的指引

**旁白**：就在這時，李杰突然想起了什麼。他伸手摸向口袋。

**李杰**：（恍然大悟）對了！我們在最後一間房子裡找到了一張地圖。剛才顧著看冊子，差點忘了。

**鈴木明日香**：（驚喜）地圖？快拿出來看看！

**旁白**：李杰從口袋裡掏出那張摺疊的羊皮紙，小心翼翼地攤開在草地上。同時，他也拿出了之前在幽冥之都車站找到的那張舊地圖。

**李杰**：（將兩張地圖並排放好）讓我比較一下這兩張⋯⋯

**旁白**：兩張地圖的風格相似，都用精靈文字標註。舊地圖標示著從幽冥之都出發的幾條主要路線，而新地圖則更加詳細。

**李杰**：（仔細對照）這張新地圖⋯⋯標示的範圍比較廣。看起來銀月湖是在地圖的中間位置。

**鈴木明日香**：（湊近看）上面寫了什麼？

**李杰**：（指著地圖北方）往北⋯⋯有一個地方叫做「終焉祭壇」。

**鈴木明日香**：（皺眉）終焉祭壇？聽起來好可怕⋯⋯

**李杰**：（點頭，然後指向地圖南方）而往南⋯⋯這裡寫著⋯⋯

**旁白**：他的聲音突然停住了，眼睛緊緊盯著地圖上的文字。

**鈴木明日香**：（好奇）怎麼了？

**李杰**：（聲音帶著驚訝）往南的地方⋯⋯標示著「月光林地」。

**鈴木明日香**：（愣了一下）月光林地⋯⋯這個名字⋯⋯

**李杰**：（拿起精靈冊子，翻到記載三個村落的那一頁）就是冊子裡提到的那個！精靈族三大村落之一，位於永恆森林北部的月光林地！

**鈴木明日香**：（震驚）那⋯⋯那就是說，月光林地在我們的南邊？

**李杰**：（沉思）如果地圖沒錯的話⋯⋯應該是這樣。但問題是⋯⋯

**旁白**：他仔細查看兩張地圖，試圖找到更多線索。但地圖上並沒有標示任何關於永恆森林或是碧翠之島門扉的位置。

**李杰**：（皺眉）奇怪⋯⋯這張地圖沒有標示永恆森林的位置，也沒有提到通往碧翠之島的門扉。

**鈴木明日香**：（困惑）為什麼會這樣？

**李杰**：（思考）也許⋯⋯也許這張地圖標示的只是這個異世界的一部分。永恆森林和門扉可能在更遠的地方。

**李杰**：（看著地圖上的「月光林地」）但是⋯⋯如果這個月光林地真的是冊子裡提到的那個精靈村落⋯⋯

**鈴木明日香**：（接話）那就代表精靈真的能在這個世界生存？

**李杰**：（點頭）或者⋯⋯這個「月光林地」和冊子裡的是同名異地？畢竟我們不確定這張地圖標示的範圍到底有多大。

**旁白**：李杰再次仔細查看地圖，試圖找到任何其他線索，但除了「終焉祭壇」和「月光林地」這兩個地點，地圖上沒有標示其他重要的地標。

**李杰**：（嘆了口氣）看來這張地圖給的資訊有限。我們知道往北可以去終焉祭壇，往南可以去月光林地，但⋯⋯

**鈴木明日香**：（小聲）但我們不知道哪條路是對的⋯⋯

**李杰**：（看向她）不過至少我們有了兩個選擇。如果往南去月光林地，也許能確認那裡是不是真的精靈村落。

**李杰**：（繼續）而往北去終焉祭壇⋯⋯雖然不知道那是什麼地方，但既然被標在地圖上，應該也有它的重要性。

**鈴木明日香**：（擔心）那我們該怎麼選？

**李杰**：（沉思片刻）月光林地⋯⋯如果那裡真的是精靈村落，也許能找到一些答案。冊子上說艾拉如被放逐了，但也有部分族人跟隨他離開。

**李杰**：（看著地圖）如果這個月光林地是艾拉如和他的追隨者建立的新村落⋯⋯那就解釋得通了。他們離開了永恆森林，來到這個異世界。

**鈴木明日香**：（恍然）可是冊子上說這個世界對精靈是致命的啊⋯⋯

**李杰**：（點頭）對，這就是矛盾的地方。也許⋯⋯也許他們找到了某種方法可以在這裡生存？或者⋯⋯

**旁白**：他停頓了一下，眉頭皺得更深。

**李杰**：（低聲）或者，那裡已經沒有活著的精靈了⋯⋯只是一座廢棄的村落。

**鈴木明日香**：（害怕）那⋯⋯那我們要去嗎？

**李杰**：（看著地圖，陷入長時間的沉思）我需要好好想想⋯⋯

**旁白**：兩人安靜地坐在湖邊。李杰盯著地圖，腦海中不斷思考著各種可能性。明日香則望著湖面，等待著他的決定。

**旁白**：銀月湖的水面依然平靜如鏡，發光的花朵在湖面上輕輕搖曳。紫色的月光灑在兩人身上，在草地上投下長長的影子。

**旁白**：北方的終焉祭壇，南方的月光林地——兩條截然不同的道路，擺在他們面前。而每一條路，都可能通向未知的危險，或是回家的希望。

**旁白**：選擇的時刻，即將到來。
