# 第一幕：喧囂的月台

**旁白**：深夜，新宿車站的中央線月台。電子顯示板在黑夜中閃爍著冷白色的光芒，數字跳動間透出一種機械式的冷漠。午夜十一點五十分——最後一班往高尾方向的列車，即將進站。

**旁白**：月台上的空氣厚重而黏膩，混雜著居酒屋飄散出的酒精味、便利店炸雞的油膩香氣。週末的狂歡正在落幕，人群三三兩兩地散落在長椅與立柱之間。

**旁白**：有人靠著立柱打盹,有人盯著手機螢幕發出疲倦的笑聲,還有人茫然地望著漆黑的隧道口,彷彿在等待某種救贖。

**旁白**：李杰站在黃線後方，微微仰頭，任由深夜的涼風拂過臉頰。他的西裝領帶早已鬆垮，襯衫領口敞開，露出疲憊的線條。

**李杰**：(內心)又是一個喧鬧的夜晚⋯⋯派對的酒精和笑語彷彿還在耳邊迴盪。但一轉身，就只剩下自己和這片冰冷的月台。

**李杰**：(內心)二十八歲，不高不低，不成功也不失敗。就像這末班車一樣——永遠卡在終點之前，不知道是該慶幸還是該嘆息。

**旁白**：他輕輕嘆了口氣，呼出的白霧在空氣中浮現又消散。李杰有個習慣，喜歡觀察陌生人。從他們的表情、姿態、衣著的皺褶中，猜測他們的故事。

**旁白**：視線掃過月台，首先映入眼簾的是一群朝氣蓬勃的大學生。他們是這片灰色空間裡最鮮豔的色彩——笑聲肆無忌憚，動作誇張。

**健談男大生**：欸，我跟你說，剛才那家店的清酒真的超純的！我現在頭還在暈⋯⋯

**活潑女大生**：哈哈，你明明才喝兩杯就開始胡言亂語了！還說什麼要請大家去夏威夷玩，笑死我了！

**多話女大生**：對啊對啊，我也聽到了！而且他還說要把他的遊戲機賣了湊旅費，我看他明天醒來一定會後悔死。對了，你們不覺得剛才那個學長長得很帥嗎？雖然話少了點，但很有型耶⋯⋯

**沉穩男大生**：(淡淡地)妳每次看到帥哥都這麼說。先看好妳的包包吧，別又弄丟了。

**多話女大生**：哎呀，你很囉唆耶！我有在看啦！

**旁白**：李杰看著他們，嘴角微微上揚。曾幾何時，他也像他們一樣，以為夜晚永遠不會結束，以為世界就在腳下。

**旁白**：視線往旁邊移一點，一個身材微胖、穿著格子襯衫的年輕男子正侷促地站在立柱旁。他時不時地調整一下鼻樑上的黑框眼鏡，眼神游移，似乎想加入那群大學生，卻又遲遲不敢開口。

**膽小眼鏡男**：(自言自語)那個⋯⋯其實我剛才也覺得那家店不錯⋯⋯只是人有點多⋯⋯

**旁白**：他的聲音太小，很快就被淹沒在大學生的笑鬧聲中。他尷尬地低頭看著自己的鞋尖，腳步不安地蹭著地面。

**旁白**：就在這時，他像是下定了決心，深吸一口氣，往前跨了一小步，對著那個健談的男大生開口。

**膽小眼鏡男**：(語氣有些僵硬)嘿，那個⋯⋯我是說，剛才那家店的調酒也很有名喔，如果你們下次想去的話，我可以推薦⋯⋯

**旁白**：健談男大生愣了一下，轉頭看了他一眼，隨即露出一個有些敷衍的笑容。

**健談男大生**：喔，是喔？謝啦。

**旁白**：說完，他又轉過頭去跟朋友們繼續剛才的話題。

**多話女大生**：欸，我們明天去哪裡續攤？我聽說涉谷那邊新開了一家酒吧，聽說調酒師超有名的！

**沉穩男大生**：可以啊，我先訂位。

**活潑女大生**：我也要去我也要去！

**多話女大生**：我也是我也是！而且我們可以先去吃飯再去喝酒！我知道附近有一家拉麵店超好吃的，他們的叉燒軟到入口即化⋯⋯

**旁白**：膽小眼鏡男的笑容逐漸凝固，臉上閃過一絲尷尬與不甘。他假裝若無其事地拿出手機，低頭滑動著螢幕，拇指卻一動不動——顯然什麼也沒在看。他的耳朵微微泛紅，額頭上滲出一層薄汗。

**李杰**：(內心)年輕真好⋯⋯充滿了無所謂的精力，就算明天要後悔，今晚也要盡情燃燒。那群人的互動還挺有趣的——有人主導話題，有人熱烈回應，有人冷靜補充。這就是青春的樣子吧。

**李杰**：(內心)不過那個傢伙⋯⋯那種急於表現、渴望被認同的樣子，反而顯得更加淺薄。大概是剛出社會沒多久吧，還保留著想要融入群體的焦慮，卻又不知道該怎麼做。

**旁白**：李杰收回目光，視線隨意掃過月台，然後——他的目光停住了。

**李杰**：(內心)嗯？那個女孩⋯⋯

**旁白**：在喧鬧人群的另一端，月台長椅的最角落，坐著一個安靜得幾乎要融入夜色的女孩。她穿著一件淺米色的棉質連衣裙，裙擺輕輕搭在膝蓋上，露出纖細的小腿和一雙乾淨的白色帆布鞋。

**旁白**：她戴著一副復古風格的有線耳機，白色的耳機線垂在胸前，隨著她的呼吸微微晃動。她的長髮柔順地披在肩上，在月台的燈光下泛著淺淺的栗色光澤。

**旁白**：她的腿上放著一本有些舊的素描本，正用一支細細的鉛筆，專注地在紙上描繪著什麼。她的側臉小巧精緻，睫毛在眼瞼上投下淺淺的陰影，嘴唇微微抿著，眉頭有時會輕輕蹙起。

**旁白**：她周身散發著一種與環境格格不入的寧靜氣質。彷彿有一道透明的屏障將她與外界的嘈雜隔絕開來。無論是大學生的喧鬧，還是列車進站的廣播聲，都無法打擾她此刻的專注。

**李杰**：(內心)在這種地方還能如此專注⋯⋯真是少見。她看起來⋯⋯很特別。那閃亮的頭髮，小巧的側臉，還有那種彷彿活在另一個世界裡的神情⋯⋯有種讓人忍不住想保護的感覺。我在想什麼啊⋯⋯

**旁白**：李杰的視線短暫地在她身上停留了幾秒，隨後不著痕跡地移開，假裝只是隨意掃視月台。他的目光落在不遠處的一根立柱上——柱子的陰影裡，站著一個全身黑衣的男人。

**旁白**：他穿著一件黑色連帽衛衣，帽簷壓得極低，遮住了大半張臉。雙手插在口袋裡，身體靠著柱子，一動也不動，像一座沒有生命的雕像。周圍的人來來往往，沒有人注意到他的存在。

**李杰**：(內心)那個人⋯⋯總覺得有點違和感。為什麼要把自己裹得那麼嚴實？而且那個姿勢⋯⋯不像是在等車，更像是在⋯⋯觀察什麼。算了，大概是我想多了。這座城市裡，什麼樣的人都有。

**旁白**：這時，月台的廣播響起，打斷了李杰的思緒。

**廣播聲**：往高尾方向的末班車即將進站，請各位乘客退到黃線後方等候。

**旁白**：遠處的隧道口開始透出微弱的光芒。列車的頭燈劃破黑暗，伴隨著逐漸清晰的軌道摩擦聲，一節節銀白色的車廂緩緩駛入月台。冰冷的金屬車身反射著月台的燈光，像一條冰冷的銀蛇。

**旁白**：列車平穩地停下，車門發出「嘶」的一聲打開。冷氣從車廂內湧出，帶來一股混合著消毒水與陳舊座椅味道的氣息。等待的人群開始緩緩移動，一個接一個地走進車廂。

**李杰**：(內心)終於來了⋯⋯回家吧。

**旁白**：李杰隨著人流走進車廂，選了一個靠窗的空位坐下。他放下背包，靠在椅背上，疲憊地閉上眼睛。幾秒後，他睜開眼，不經意地一瞥——那個安靜的女孩也上了同一節車廂，就坐在斜前方不遠處。

**旁白**：她依舊低著頭，耳機還掛在耳朵上,素描本放在膝蓋上。手指輕輕撫過紙面，像是在檢查剛才畫好的線條。她的側臉在車廂燈光下顯得更加柔和，睫毛在臉頰上投下細碎的陰影。

**旁白**：那四個大學生也吵吵鬧鬧地走了進來。健談男大生依然滔滔不絕，活潑女大生不時發出爽朗的笑聲，多話女大生在旁邊快速地補充各種細節，沉穩男大生則安靜地跟在最後，偶爾點頭回應。

**旁白**：他們在車廂中段找到位置坐下，笑聲依然迴盪在空氣中。膽小眼鏡男刻意地走在他們後面幾步，假裝與他們同行，但沒有人回頭看他。他在靠近門口的位置坐下，拿出手機，開始假裝專注地滑動螢幕。

**旁白**：但眼神卻時不時地瞟向那群大學生，眼底閃過一絲落寞與不甘。

**廣播聲**：車門即將關閉，請注意。

**旁白**：車門緩緩合攏，發出沉悶的「咔嚓」聲，像某種儀式的完成。列車輕輕震動了一下，開始緩緩啟動。窗外的城市燈火開始向後倒退，霓虹燈的光影在玻璃上一閃而過，漸漸變得模糊、遙遠。

**李杰**：(內心)又是一個平凡的夜晚⋯⋯又是一段平凡的旅程。明天醒來，一切照舊。

**旁白**：此刻的他還無法想像，這輛將他帶離熟悉城市的列車，將會把他帶往一個怎樣的命運。而那些此刻看似平凡的乘客們——那個安靜的女孩、那四個青春洋溢的大學生、那個故作深沉的膽小眼鏡男，還有那個神秘的黑衣男。

**旁白**：他們即將一同經歷一場，超越所有人理解的旅程。

**旁白**：車廂內的燈光微微閃爍了一下，隨後恢復正常。列車駛入了黑暗的隧道，窗外的世界徹底被吞沒。只剩下車輪與軌道摩擦的聲音，在寂靜中無止盡地迴響。
