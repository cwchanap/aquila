# 第五幕：意外的同行者

**旁白**：寂海車站的月台上，李杰剛踏上冰冷的水泥地面，身後就傳來了急促的腳步聲。他轉過身，驚訝地看著那個在月台上畫畫的文靜少女，正站在他面前。

**旁白**：她的臉色蒼白，雙手緊緊抓著那本素描本。她的眼神閃躲不定，像是做了一個連自己都無法理解的決定。

**李杰**：(語氣疑惑，帶著一絲警惕)妳...為什麼要跟下來？待在車上不是更安全嗎？

**文靜少女**：(低著頭，聲音很小)我...我不知道...車上...車上很吵...大家都在爭執...

**文靜少女**：(停頓了一下)然後...我看到你下車...覺得...跟著你比較好...

**李杰**：(內心)她的聲音雖然小，但沒有我想像中那麼慌亂。只是...為什麼要跟著一個陌生人？

**文靜少女**：(抬起頭，眼神中帶著一絲堅定)你...你說要下來找答案對吧...我也想知道...這裡是哪裡...

**文靜少女**：(聲音依然很小)而且...一個人的話...會害怕...

**李杰**：(看著她，心中的警惕稍微放鬆了一些)好吧。既然妳已經下來了，那就跟著我吧。我叫李杰。妳呢？

**文靜少女**：(聲音微弱)我叫...清水恵美。

**李杰**：(點點頭)好，恵美。別走太遠，這裡的情況我們還完全不清楚。

**旁白**：李杰帶著恵美，開始在寂海車站的月台上緩慢前進。月台上的風帶著一股大海特有的鹹腥味，還有一種說不上來的、像是腐爛木頭的味道。

**旁白**：就在這時，原本安靜的車廂內突然傳來了爭執聲。

**健談男大生**：(在車門口大喊)喂！大叔！你看！那兩個人都沒事！月台上根本沒有怪物！

**沉穩男大生**：(皺著眉頭)冷靜點。沒看到怪物並不代表這裡安全。

**活潑女大生**：可是我們不能就這樣坐在這裡等死啊！萬一這車永遠不開了呢？

**多話女大生**：對啊對啊，我也不想一直待在車上，感覺好壓抑喔...

**慷慨大叔**：(看著李杰和恵美的背影，又回頭看了看車廂內的乘客)大家先別急...

**自私中年男**：(縮在座位上，語氣刻薄)想死的就儘管下去！我可不去！誰知道那片森林裡藏著什麼！

**慷慨大叔**：(咬牙下定決心)走！我們一起走！總比在這裡等死好！

**活潑女大生**：(緊緊抓著健談男大生的衣袖)那...那我也去...你們別丟下我...

**沉穩男大生**：(最後一個踏出車門，平靜地說)既然決定了，就不要後悔。

**多話女大生**：(一邊下車一邊嘀咕)嗚...我好害怕...但還是跟著他們好了...一個人留在車上更可怕...而且如果真的有危險，人多至少可以互相照應...

**旁白**：就在眾人以為所有願意下車的人都已經下來時，膽小眼鏡男突然從座位上站起來，快步走向車門。

**膽小眼鏡男**：(一邊下車，一邊不屑地推了推眼鏡，但額頭上的冷汗暴露了他的恐懼)哼，一群愚蠢的羊，自以為做了正確的選擇。

**膽小眼鏡男**：(故作鎮定)我可不是跟著你們！我只是要自己出去尋找更科學的解釋罷了！這種超自然現象，一定有合理的物理學解釋！

**旁白**：車廂內，自私中年男縮在座位上，看著越來越多人下車。他的臉上閃過掙扎的神色，手放在座椅扶手上，想站起來，又坐了回去。

**自私中年男**：(小聲嘀咕，聲音充滿恐懼)不行...外面太危險了...我不出去...待在車上肯定更安全...

**旁白**：月台上很快聚集了八個人。李杰、恵美、慷慨大叔、四名大學生，以及嘴上不饒人但還是跟下來的膽小眼鏡男。

**旁白**：他們站在月台邊緣，望著列車內剩下的乘客——那對中年夫婦緊緊抱在一起，那個帶著孩子的家庭縮在角落，還有幾個獨自乘客面無表情地坐著。

**旁白**：車廂內，剩下的乘客們臉色蒼白地看著車門外的人群。有人眼中充滿羨慕，有人充滿不解，有人充滿恐懼，還有人眼中閃過對自己選擇的動搖。

**旁白**：沉默只持續了幾秒。就在這批「先行者」還未完全站穩腳跟，甚至還沒來得及討論下一步該怎麼辦時——

**旁白**：一陣刺耳的「嘶——」聲毫無預兆地響起，劃破了寂海車站的死寂。

**李杰**：(猛然轉頭，內心)這是...車門的聲音！

**旁白**：車門！剛剛還敞開著的列車車門，在沒有任何廣播預告的情況下，突然開始緩緩關閉！

**健談男大生**：(驚叫)門...門關了！

**活潑女大生**：(抓住健談男大生的手臂)怎...怎麼回事？！

**旁白**：車門「咔嚓」一聲完全關閉。透過車窗，李杰看到車廂內的乘客們先是一驚，身體本能地往後縮。

**旁白**：但隨即，他們的臉上浮現出狂喜的神色——有人激動地抱在一起，有人對著天花板做出感謝的手勢，有人甚至笑出了聲。

**旁白**：列車開始提升速度，車輪與鐵軌摩擦發出沉重的轟鳴聲。引擎的聲音從低沉變得高亢，彷彿一頭甦醒的野獸。

**旁白**：透過車窗，李杰看到車廂內的乘客們對著月台上的八人揮手。有些人臉上帶著解脫的笑容，有些人則露出幸災樂禍的表情。

**旁白**：自私中年男隔著車窗對外面大喊著什麼，臉上帶著得意的笑容。雖然聽不到聲音，但從他的口型和表情，不難猜出是在嘲笑他們的選擇。

**旁白**：列車漸行漸遠，速度越來越快。車廂內的燈光在夜色中漸漸縮小，像一隻逐漸遠去的螢火蟲。

**旁白**：最終，那點光芒消失在來時的方向——那片幽深森林的深處，被黑暗徹底吞沒。

**旁白**：月台上陷入一片寂靜。只剩下海風呼嘯的聲音，和遠處海浪拍打礁石的低沉回響。

**旁白**：八個倖存者呆立在原地。他們被丟棄在這片未知的「寂海」車站，面對著無盡的黑暗、詭異的森林、死寂的海洋。

**旁白**：以及頭頂那兩輪如同眼睛般注視著他們的詭異紫色殘月。

**李杰**：(緩緩轉頭看向列車消失的方向，內心)它...就這麼走了...真的走了...

**清水恵美**：(聲音很輕，但比之前更平靜)我們...回不去了嗎...

**旁白**：李杰注意到，恵美的聲音雖然依舊小聲，但已經沒有剛才那種顫抖了。她靜靜地站在那裡，望著列車消失的方向，眼神中閃過一絲難以捉摸的情緒。

**慷慨大叔**：(深吸一口氣，沉聲說道)事已至此，抱怨也沒用。我們得想辦法活下去。

**旁白**：冰冷的海風吹過月台，帶來陌生植物的氣味和海水的鹹腥。八個人站在空蕩蕩的車站上，面對著未知的命運。
