# 第十二幕：血跡與選擇

---

---

**旁白**：李杰撿起那副破碎的眼鏡,仔細端詳。黑色細框,左側鏡腿彎曲變形,右側鏡片完全碎裂,尖銳的裂口在昏暗的光線下閃著冷光。他的腦海中,某個身影模糊地閃過。

**李杰**：(內心獨白) 這副眼鏡...款式雖然普通,但印象中...好像在哪見過?

**旁白**：他努力回想,直到一個尖酸刻薄的聲音在腦海中響起——「你們是白痴嗎!」那個瘦高的身影,驕傲地推了推鼻樑上相似的黑框眼鏡。

**李杰**：(眼神一凜,脫口而出) 這是眼鏡男的!

**清水恵美**：(聲音顫抖) 眼鏡男...?他...他是不是出事了?

**李杰**：很有可能。在這裡發現他的眼鏡,他肯定就在附近。我們找找看!

**旁白**：李杰和恵美開始在周圍的濃霧中搜索。這片區域的樹木更加高大密集,能見度極差。他們彎下腰,仔細檢查著每一寸泥土和樹根,腐葉的氣味混雜著潮濕的霧氣,讓人呼吸不暢。

**旁白**：約莫五分鐘後,李杰在路邊一棵粗大的樹幹上,發現了一抹可疑的深色痕跡。

**李杰**：清水,妳看這裡!

**旁白**：在樹幹離地約一公尺高的地方,有一道明顯的擦痕。痕跡的末端,染著一小片已經半乾的暗紅色——那是血跡。血跡旁邊的樹皮,似乎被什麼尖銳的東西劃開了一道口子。

**清水恵美**：(看到血跡,臉色更加蒼白) 血...是血...眼鏡男...是不是被...被野獸襲擊了?森林裡...會不會有...很可怕的動物?

**李杰**：(內心獨白) 野獸...確實有可能。這片森林這麼原始,什麼都有可能發生。這比想象的還要危險。

**李杰**：這裡確實不安全。我們得小心了。這樣吧,我們還是先回小屋一趟,找找有沒有能防身的東西。

**清水恵美**：(從背包裡掏出兩把明晃晃的廚房刀,遞了一把給李杰) 我...我剛才走的時候...在廚房順手拿的...一把給你...一把給我...

**旁白**：李杰看著恵美手裡那兩把在油燈下閃閃發亮的廚房刀,眼神中閃過一絲驚訝。這個膽小怯懦的女孩,竟然會如此有先見之明?她的準備,超出了他的預期。

**李杰**：妳...準備得很充分。

**旁白**：有了防身工具,兩人壯著膽子又在血跡周圍搜尋了大約一個小時,希望找到眼鏡男的更多線索,或者發現其他的倖存者。然而,除了濃霧和無盡的樹木,他們一無所獲。偶爾傳來的不明聲響,讓兩人的神經繃得更緊。

**清水恵美**：(小聲地說) 李杰先生...這裡...這裡太危險了...我們...還是不要離開主路太遠比較好...

**旁白**：李杰思考了一下。清水說得沒錯。他們對這個世界一無所知,盲目地在森林深處搜尋,只會讓自己陷入更大的危險。雖然眼鏡男的遭遇讓他感到不安,但說實話,李杰對這個總是愛嘲諷別人的眼鏡男並沒有太深的感情。他的首要任務,是保證自己和清水的安全。

**李杰**：好,我們聽妳的。繼續沿著這條路往前走。

**旁白**：李杰收起心中的雜念,帶著恵美再次踏上主路。他們不敢再有片刻鬆懈,手裡緊握著廚房刀,警惕地掃視著周圍濃霧籠罩的森林。

**旁白**：又走了很長一段時間,濃霧似乎更加濃重了。他們來到了第二個岔路口。

**李杰**：(內心獨白) 這裡...是地圖上沒有的岔路。

**旁白**：眼前的景象再次與他們殘破的地圖產生了偏差。地圖上根本沒有這個岔路口的資訊。但實際上,除了繼續前行主路,這個岔路口現在有兩條明顯的支線——一條通往左邊,一條通往右邊。沒有任何標識,也無從判斷這些路的去向。三條路徑都被濃霧吞沒,看不到盡頭。

**旁白**：李杰陷入了沉思。他努力回憶地圖上的每一個細節,試圖從模糊的圖像中找出任何提示,但一無所獲。這是一個完全陌生的選擇,沒有邏輯,沒有依據。

**李杰**：(內心獨白) 往左?往右?還是繼續直走?如果走錯了...

**旁白**：就在他猶豫不決時,恵美輕輕拉了拉他的衣角。

**清水恵美**：李杰先生...我們...我們走左邊那條路...好不好?

**李杰**：(轉頭看她,語氣疑惑) 左邊?為什麼?妳有什麼發現嗎?

**清水恵美**：(眼神有些閃爍,怯生生地搖了搖頭) 我...我也不知道...就是...就是一種感覺...覺得那邊...好像比較...比較安全一點...

**旁白**：直覺?李杰看著恵美,回想起她在森林裡異常的平靜,以及她能聽懂地圖上文字的短暫異樣。她的直覺,是否比他的邏輯判斷更值得信賴?他雖然理性,但面對這種超自然的情況,理性似乎不再是唯一的標準。

**李杰**：(內心獨白) 冒險嗎?還是相信她的直覺?

**旁白**：他看著清水期待的眼神,又看向三條被濃霧吞沒的路徑。左邊、右邊、還是直走?每一條路都通往未知,而錯誤的選擇可能意味著死亡。李杰緊握著手中的廚房刀,陷入了沉默。

**旁白**：濃霧在周圍翻騰,彷彿在催促著他做出決定。
