# 第十四幕：無法同步的時代

**清水恵美**：(輕聲說,望著遠方的濃霧) 能和家人在一起...真好呢...

**旁白**：她的眼神變得有些悠遠,彷彿陷入了某段回憶中。

**清水恵美**：我也想...想去看海...等他...等他的傷好了以後,就跟他一起去。

**李杰**：(內心獨白) 他?男朋友嗎...

**旁白**：李杰心中閃過一絲說不清道不明的驚訝,但很快便平復下來。

**李杰**：(內心獨白) 也對,像她這樣可愛的女孩,有男朋友也是理所當然的。只是...為什麼聽起來這麼悲傷?

**旁白**：李杰察覺到氣氛變得有些沉重,便決定換個輕鬆點的話題。

**李杰**：說起來,妳平常都聽些什麼音樂?

**清水恵美**：我喜歡聽一些...比較老的歌。

**李杰**：老的?比如?

**清水恵美**：八、九十年代的那些...像是...中森明菜、ZARD...之類的。

**李杰**：(驚訝地睜大了眼睛) 真的假的?我也超喜歡那個年代的歌!感覺現在的流行歌,總少了點味道。

**清水恵美**：(終於露出一絲發自內心的微笑) 嗯!我也是這麼覺得!舊的旋律比較能觸動人心...

**旁白**：找到共同話題讓氣氛好了起來,李杰也輕鬆了不少。他們又走了不知道多久,腳步在腐葉上踩出單調的節奏。

**李杰**：(為了打發時間) 妳學文學的,平常都看些什麼書?喜歡哪些作家?

**清水恵美**：我喜歡...夏目漱石、川端康成...還有太宰治...

**李杰**：都是經典。太宰治我在高中的時候讀過《人間失格》,當時覺得很沉重。

**清水恵美**：嗯...他的作品確實有點悲傷...但是...很真實...

**李杰**：妳還有其他興趣愛好嗎?除了看書和聽音樂。

**清水恵美**：嗯...我喜歡散步...有時候會去公園坐著發呆...

**李杰**：(笑了) 發呆也算興趣嗎?

**清水恵美**：(有點不好意思) 我...我就是喜歡安靜地待著...看看天空,看看樹...

**清水恵美**：李杰先生工作那麼忙,應該沒什麼時間休閒吧?

**李杰**：(嘆了口氣) 是啊。以前還會打打籃球,現在連週末都在加班,根本沒時間運動。有時候真羨慕妳們學生,至少還有時間做自己喜歡的事。

**清水恵美**：但是...李杰先生的工作一定很有意義吧?

**李杰**：有意義?說實話,大部分時間都在做報表、開會、應付客戶...偶爾會想,這樣的生活到底是為了什麼。

**清水恵美**：(輕聲) 會累的話...就休息一下吧...

**旁白**：兩人就這樣一邊聊著,一邊前行。平凡的對話讓這段漫長的路程不那麼難熬。

**李杰**：那電影和動畫呢?最近那部很紅的《星塵迴響》,妳看了嗎?聽說評價超高。

**清水恵美**：(迷茫地搖了搖頭) 電影...動畫?對不起...我...我幾乎不看那些...所以...不太清楚你說的是什麼...

**李杰**：(內心獨白) 連這麼有名的作品都不知道?她平常到底都過著什麼樣的生活...

**旁白**：這個小小的插曲讓李杰心中的疑惑又加深了一層。他覺得眼前的恵美,彷彿與他所處的時代,有著一層看不見的隔閡。但他沒有追問。或許每個人都有自己的生活方式。

**旁白**：短暫的休息結束了。兩人再次起身,繼續沿著小徑前行。走了一段路後,清水忽然開口。

**清水恵美**：李杰先生...你在日本最喜歡吃什麼?

**李杰**：(有些意外這個問題) 嗯?最喜歡的...大概是拉麵吧。尤其是冬天下班後,來一碗熱騰騰的豚骨拉麵。

**清水恵美**：(眼睛亮了起來) 我也喜歡拉麵!我最喜歡醬油拉麵...學校附近有一家小店,老闆娘人很好,每次都會多給我加蔥花...

**李杰**：(笑了) 聽起來是個好地方。等我們回去,妳帶我去吃吧。

**清水恵美**：(開心地點頭) 好啊!不過...那家店有點舊,裝潢也不太好...李杰先生不會介意嗎?

**李杰**：怎麼會。好吃的店往往都是那種不起眼的小地方。我剛來日本的時候,有個學長帶我去吃過一家巷子裡的居酒屋,破破爛爛的,但是燒鳥超好吃。

**清水恵美**：(好奇地問) 後來還有去嗎?

**李杰**：(苦笑) 沒有。那家店後來搬走了。我一直想再去找,但始終沒找到。

**清水恵美**：好可惜喔...

**旁白**：他們又走了不知道多久,腳步在腐葉上踩出單調的節奏。忽然,前方的濃霧似乎開始變得稀薄,隱約有光透了進來,還伴隨著一種柔和的聲音——像是風吹過樹葉,又像是遠處的水流聲。

**李杰**：(停下腳步,警惕地看向前方) 妳聽到了嗎?那個聲音...

**清水恵美**：(點點頭) 嗯...好像是...風聲?還有水聲?

**李杰**：前面有光!我們可能要走出這片森林了!

**旁白**：兩人心中燃起希望,互相攙扶著,加快了腳步。濃霧越來越薄,光線越來越亮,那柔和的聲音也越來越清晰。當他們最終穿過最後一層薄霧,眼前的景象,讓他們徹底呆立在原地,震驚得無法言語。那是一個他們絕對無法想像的地方。
