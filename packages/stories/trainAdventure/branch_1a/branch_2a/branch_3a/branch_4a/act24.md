# 第二十四幕：意外的重逢

**旁白**：三人默默地走在回程的路上。銀月湖的美景依然如故，但此刻誰也無心欣賞。

**旁白**：走了大約十分鐘後，恵美突然停下腳步。她深吸一口氣，轉身面對李杰。

**清水恵美**：李杰先生...對不起。

**旁白**：恵美的聲音很小，眼眶還紅紅的。她低著頭，像個做錯事的孩子。

**清水恵美**：我...我一直騙你。從一開始就...對不起...

**李杰**：（沉默了一會兒，嘆氣）我理解妳的處境。在那種情況下，妳也只能這麼做。

**旁白**：李杰說著這些話，但他的心情卻很複雜。理智上，他確實理解恵美的選擇。但情感上，被欺騙的感覺還是讓他有些不舒服。

**旁白**：尤其是想到在銀月湖畔的那些日子，她對自己的溫柔和笑容...有多少是真實的，又有多少只是演技？

**清水恵美**：（抬起頭，眼中含著淚光）謝謝你...謝謝你願意原諒我...

**旁白**：恵美露出一個勉強的笑容，但那笑容中帶著深深的疲憊和愧疚。

**旁白**：李杰點了點頭，轉身繼續往前走。他不想讓恵美看到自己臉上複雜的表情。

**旁白**：又走了一段路後，李杰開口了。

**李杰**：艾拉如先生，我想問一下...清水第一次帶來的那個人，那個不符合資格的...是什麼樣的人？

**艾拉如**：（想了想）那是恵美的朋友。一個年輕的男生，年紀和恵美差不多，比你年輕一些。

**艾拉如**：他們好像是在同一所大學讀書的。那個男生的性格...其實和你挺像的。

**李杰**：（疑惑）像我？

**艾拉如**：嗯。都是那種比較冷靜、安靜的類型。不太愛說話，但思考問題很理性。

**旁白**：李杰心中一動。如果那個人和自己性格相似，而且也不符合資格...難道資格的判定和性格有關？

**李杰**：那...第二個人呢？那個符合資格的。

**艾拉如**：第二個人也是恵美的大學朋友。不過他和第一個人完全不同。

**艾拉如**：那是個很外向、很健談的年輕人。總是笑嘻嘻的，充滿活力。和他在一起的時候，整個氣氛都會變得輕鬆起來。

**旁白**：李杰若有所思地點了點頭。

**李杰**：那會不會...神樹需要的是那種外向、健談的人？所以我和第一個人才會不符合資格？

**艾拉如**：（聳肩）也許吧。但說實話，我也不確定。神樹的判斷標準一直是個謎。

**旁白**：恵美一直安靜地走在兩人身後，聽著他們的對話。當聽到關於那兩個朋友的描述時，她的表情變得更加沉重了。

**清水恵美**：（小聲問）艾拉如先生...我們接下來...該怎麼辦？

**艾拉如**：（停下腳步，看著遠處的銀月湖）先回去休息吧。我們需要好好想想下一步該怎麼做。

**艾拉如**：可能...需要想辦法尋找其他人類。但具體怎麼做，我現在也沒有頭緒。

**旁白**：三人繼續前行。很快，銀月湖畔的精靈營地就出現在視野中。

**旁白**：但就在他們準備進入營地時，一個精靈急匆匆地跑了過來。

**精靈衛士**：（氣喘吁吁）艾拉如大人！出事了！

**艾拉如**：（皺眉）怎麼了？

**精靈衛士**：我們...我們抓到了一個人類！一個男性人類！他的行為非常可疑，在營地周圍鬼鬼祟祟的！

**旁白**：李杰、恵美和艾拉如同時愣住了。

**李杰**：（震驚）人類？這裡怎麼會有其他人類？

**清水恵美**：（驚訝地）居然...有人找到這裡來了...

**艾拉如**：（嚴肅地）帶我們去看看。

**精靈衛士**：是！請跟我來！

**旁白**：精靈衛士帶著三人快速穿過營地，來到一個較大的帳篷前。兩個精靈衛士守在帳篷門口，看到艾拉如後立刻行禮。

**艾拉如**：人在裡面嗎？

**精靈衛士**：是的，大人。我們已經把他綁起來了。

**旁白**：艾拉如點點頭，掀開帳篷的簾子走了進去。李杰和恵美緊隨其後。

**旁白**：帳篷內光線昏暗，但李杰很快就看清了被綁在中央的那個人。

**旁白**：那是一個戴著眼鏡的年輕男子，看起來二十歲左右。他的衣服有些破爛，臉上沾著泥土和汗水，眼鏡也歪了。

**旁白**：最引人注目的是他的表情——充滿了恐懼和驚慌，整個人瑟瑟發抖。

**旁白**：李杰瞪大了眼睛，難以置信地看著眼前的人。

**李杰**：（震驚地）你...你是...！
