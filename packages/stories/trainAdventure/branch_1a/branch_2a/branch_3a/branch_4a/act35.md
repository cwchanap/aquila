# 第三十五幕：殘酷的代價

**旁白**：離開銀月湖後，四人繼續朝北方前進。

**旁白**：很快，那棵巨大的神樹就出現在視野中。它的樹冠遮蔽了大片天空，枝葉間散發著柔和的光芒。

**旁白**：即使已經見過一次，李杰依然被神樹的壯觀震撼了。但這一次，他心中那種詭異的感覺更加明顯了。

**李杰**：（內心獨白）神聖與邪惡...兩種力量依然混雜在一起...這棵樹...到底是什麼...

**凱蘭**：（停下腳步）我們到了。

**旁白**：凱蘭轉身看向恵美和斎藤。

**凱蘭**：你們準備好了嗎？

**清水恵美**：（深吸一口氣，點頭）嗯...準備好了。

**斎藤蓮**：（緊張地）我...我也準備好了...

**旁白**：恵美和斎藤並肩走到神樹前。他們對視了一眼，然後同時伸出手，將手掌貼在粗糙的樹幹上。

**旁白**：李杰和凱蘭站在一旁，靜靜地看著。

**旁白**：恵美和斎藤閉上了眼睛，表情變得平靜而專注。周圍的空氣彷彿凝固了一般，只有微風輕輕吹過樹葉的沙沙聲。

**旁白**：時間一分一秒地過去。李杰屏住呼吸，緊張地看著兩人。

**李杰**：（內心獨白）不會再出什麼問題了吧...斎藤這次...應該符合資格吧...

**旁白**：又過了幾分鐘，恵美和斎藤同時睜開了眼睛。

**旁白**：兩人的臉上都露出了放鬆的笑容。

**李杰**：（立刻走上前）怎麼樣？沒問題嗎？

**清水恵美**：（微笑，點頭）嗯...一切順利。

**清水恵美**：（轉身面對李杰和凱蘭）我們...可以前往終焉祭壇了。

**斎藤蓮**：（鬆了一口氣）太好了...終於...終於成功了...

**旁白**：李杰也露出了笑容。一年的等待，終於有了結果。

**凱蘭**：（點頭）那我們現在就出發吧。繼續朝北方前進，大約需要六到八小時的路程。

**凱蘭**：路上會經過一片荒原和一些古代遺跡。那裡比較危險，我們需要格外小心。

**李杰**：我明白了。那我們盡快出發吧。

**旁白**：四人轉身，離開神樹，繼續朝北方前進。

---

**旁白**：離開神樹後，道路變得更加崎嶇。周圍的植物逐漸稀少，取而代之的是荒涼的岩石和乾燥的土地。

**旁白**：凱蘭走在最前面，警惕地觀察著周圍的環境。李杰、恵美和斎藤跟在後面。

**旁白**：他們已經走了大約四個小時，每個人都開始感到疲憊。

**斎藤蓮**：（氣喘吁吁）要...要走多久啊...我的腿都快不行了...

**凱蘭**：（回頭看了看）你們看起來都累了。前面有一片比較平坦的地方，我們在那裡休息一下吧。

**清水恵美**：（擦了擦汗）謝謝...

**旁白**：又走了大約十分鐘，他們來到一片相對平坦的岩石區域。這裡有幾塊大石頭可以當作座位，周圍視野開闊，適合休息。

**凱蘭**：就在這裡休息吧。我們可以小睡一會兒，恢復體力。

**李杰**：（點頭）好主意。還有幾個小時的路程，確實需要休息一下。

**旁白**：四人各自找了個舒適的位置坐下或躺下。凱蘭靠在一塊大石頭上，但依然保持警覺，隨時注意周圍的動靜。

**旁白**：斎藤很快就躺下了，不一會兒就傳來了輕微的鼾聲。看來他真的累壞了。

**旁白**：李杰坐在一塊石頭上，閉著眼睛休息。疲憊讓他的意識逐漸模糊...

**旁白**：但就在他快要睡著的時候，他感覺到有人輕輕碰了碰他的手臂。

**旁白**：李杰睜開眼睛，看到恵美蹲在他旁邊。她的臉上帶著緊張和不安的表情。

**清水恵美**：（小聲）李杰先生...你能...能跟我來一下嗎？

**李杰**：（疑惑）嗯？怎麼了？

**清水恵美**：（看了看正在休息的凱蘭和斎藤）我...我有些事情...需要單獨跟你說...

**旁白**：恵美的表情很嚴肅，眼中帶著一絲複雜的情緒。李杰感到一股不祥的預感。

**李杰**：（站起來）好...

**旁白**：兩人走到離凱蘭和斎藤較遠的一塊岩石後面。恵美確認了周圍沒有人能聽到，才轉身面對李杰。

**清水恵美**：（深吸一口氣）李杰先生...等一下我說的話...請你...請你不要太生氣...也不要太驚訝...

**李杰**：（心中湧起不祥的預感）到底...是什麼事？

**旁白**：恵美沉默了一會兒，像是在組織語言。她的手緊緊握著，指節因為用力而發白。

**清水恵美**：（顫抖地）終焉祭壇...雖然可以把人送回原本的世界...但是...

**清水恵美**：（聲音更小了）要打開那扇門...需要...需要付出代價...

**李杰**：（緊張地）什麼代價？

**旁白**：恵美咬著嘴唇，眼眶有些泛紅。

**清水恵美**：（聲音哽咽）其中一個參與者...必須永遠留在祭壇前的大門旁...

**清水恵美**：只有這樣...門才會打開...讓另一個人回去...

**旁白**：李杰的腦中轟的一聲，像是被雷擊中一般。

**李杰**：（難以置信）你說...什麼？

**清水恵美**：（低著頭，眼淚滴落）也就是說...我和斎藤先生...只能有一個人回到原本的世界...

**清水恵美**：另一個人...必須永遠留在那裡...

**旁白**：李杰整個人僵住了。他張了張嘴，卻發不出任何聲音。

**旁白**：這個消息就像一把利劍，狠狠地刺進他的心臟。

**旁白**：李杰的腦中突然閃過一個念頭。他記起艾拉如說過的話——恵美上次和另一個人建立了契約，然後成功回到了原本的世界。

**李杰**：（聲音顫抖）等等...妳上次...妳上次和那個人建立契約...然後妳回來了...

**李杰**：（抬起頭，直視恵美）那個人...妳的朋友...他還在終焉祭壇嗎？

**旁白**：恵美的身體猛地一顫。她緩緩抬起頭，眼中滿是淚水。

**清水恵美**：（哽咽地點頭）嗯...

**李杰**：（難以置信）所以...所以妳這次來...是為了...

**清水恵美**：（哭著說）我...我想救他...我想把他帶回去...

**清水恵美**：我回去之後...每天每夜都在想著他...想著他一個人被困在那個地方...永遠無法回家...

**清水恵美**：（聲音顫抖）那是因為我...因為我的自私...他才會...

**旁白**：恵美捂著臉，肩膀劇烈顫抖。壓抑已久的愧疚和痛苦終於爆發出來。

**李杰**：（內心獨白）所以...這就是她的目的...她不是單純想回家...而是想救出那個為了她而被困住的人...

**旁白**：李杰沉默了很久，心中湧起無數複雜的情緒。

**旁白**：恵美擦了擦眼淚，深吸一口氣，像是下定了什麼決心。

**清水恵美**：（顫抖地）李杰先生...艾拉如先生告訴我...

**清水恵美**：如果想要交換祭壇前的犧牲者...新的犧牲者需要...需要握住原本犧牲者的雙手...

**清水恵美**：只有這樣...才能完成交換...

**李杰**：（皺眉）妳的意思是...

**清水恵美**：（低著頭）我...我需要讓斎藤先生...握住我朋友的手...

**清水恵美**：但是...斎藤先生如果知道真相...他一定不會願意...

**旁白**：恵美抬起頭，眼中滿是懇求和絕望。

**清水恵美**：（顫抖地）所以...所以我需要...需要欺騙他...

**清水恵美**：（聲音幾乎是哀求）李杰先生...我知道...我知道這個請求很過分...

**清水恵美**：但是...拜託你...拜託你幫幫我...幫我想辦法讓斎藤先生...

**旁白**：恵美說不下去了。她雙手合十，深深地鞠躬，眼淚不斷滴落在地上。

**清水恵美**：（哭著）求求你...我真的...我真的很想救他...他是因為我才...

**旁白**：李杰看著眼前崩潰的恵美，心中百感交集。

**旁白**：他想說些什麼，但話到嘴邊又說不出口。

**旁白**：欺騙斎藤...讓他成為犧牲品...這是對的嗎？

**旁白**：但如果不這麼做...恵美的朋友就要永遠被困在那裡...

**旁白**：李杰的手緊緊握著，指甲深深陷進掌心。

**旁白**：良久，他依然沒有給出答案。

**李杰**：（沉默）......

**清水恵美**：（抬起頭，看著李杰沉默的樣子，眼中閃過絕望）李杰先生...

**旁白**：但李杰只是站在那裡，看著遠方，一言不發。

**旁白**：他的腦中一片混亂——恵美的謊言、斎藤的期待、那個被困住的陌生人、還有這個殘酷的選擇...

**旁白**：一切都交織在一起，讓他無法做出決定。

**旁白**：李杰緊緊握著拳頭，指甲深深陷進掌心。

**旁白**：他看著恵美。這個看起來陽光可愛的女孩，現在正跪在地上，眼淚縱橫，為了救回她的朋友而哀求著。

**李杰**：（內心獨白）她...到底還隱藏了多少秘密...

**旁白**：李杰想起這一年來的種種——恵美的謊言、她對某些事情異常的平靜、她總是迴避的眼神、還有她那些若有所思的沉默...

**旁白**：原來，她從一開始就知道這個殘酷的真相。

**旁白**：原來，她帶著斎藤、帶著李杰來到這裡，背後有著這樣的目的。

**旁白**：而現在，她請求李杰幫助她欺騙斎藤，讓他成為那個被永遠困住的犧牲者。

**李杰**：（內心獨白）這個女孩...她的心裡...到底藏著多少東西...

**旁白**：李杰看著恵美哀求的眼神，心中湧起無數複雜的情緒——憤怒、失望、同情、困惑...全都交織在一起。

**旁白**：但他什麼都沒有說。

**旁白**：他只是沉默地站在那裡，看著遠方那片荒涼的岩石地帶。

**旁白**：因為他不知道該說什麼。

**旁白**：因為無論做出什麼選擇，都會有人要付出無法承受的代價。
