# 第三十七幕：破碎的救贖

**旁白**：恵美看到祭壇上的景象，整個人僵住了。

**旁白**：她的臉色瞬間變得慘白，雙眼睜大，嘴唇顫抖著。

**清水恵美**：（顫抖地）不...不可能...

**旁白**：下一秒，恵美突然衝向祭壇，動作急促而慌亂。

**清水恵美**：（尖叫）翔！翔！

**旁白**：她的雙手抓住祭壇的邊緣，整個人趴在祭壇上，看著那個年輕男子的臉。

**清水恵美**：（聲音撕裂）為什麼...為什麼會這樣！

**旁白**：恵美的聲音在空曠的聖殿中迴盪，充滿了絕望和難以置信。

**斎藤蓮**：（呆立在原地，結結巴巴）什...什麼...這...

**旁白**：斎藤完全呆住了，他看著祭壇上的屍體，又看著崩潰的恵美，完全無法理解眼前的狀況。

**旁白**：李杰的心臟猛地一縮。

**李杰**：（內心獨白）那個人...就是恵美說的朋友...翔...

**李杰**：（內心獨白）但是...他怎麼會...死了？

**旁白**：李杰記得恵美說過，犧牲者會被留在祭壇旁邊，永遠無法回家。

**旁白**：但是...她從來沒說過犧牲者會死！

**凱蘭**：（震驚地喃喃自語）這...怎麼會...

**凱蘭**：（往前走了幾步）發生了什麼...他為什麼...

**旁白**：凱蘭的聲音中也充滿了困惑和不安。

**旁白**：恵美的雙腿一軟，整個人跌坐在祭壇前的地面上。

**清水恵美**：（哭著）翔...翔...對不起...對不起...

**旁白**：她的雙手捂著臉，肩膀劇烈顫抖，眼淚從指縫間滑落。

**旁白**：李杰回過神來，快步衝到恵美身邊。

**李杰**：（蹲下身，扶住她的肩膀）清水...

**旁白**：恵美抬起頭，眼淚模糊了視線，她的臉上滿是絕望。

**清水恵美**：（哽咽）為什麼...為什麼他會死...

**清水恵美**：（聲音顫抖）艾拉如先生明明說...只是會被困在這裡...

**清水恵美**：（崩潰地）為什麼...為什麼會變成這樣...

**旁白**：李杰緊緊抱住恵美，感受到她身體的顫抖。

**旁白**：他不知道該說什麼，只能默默地給予支持。

---

**斎藤蓮**：（終於忍不住，大聲）到底...到底是怎麼回事？

**斎藤蓮**：（看向恵美）我們...我們不是來這裡回家的嗎？

**斎藤蓮**：（不耐煩地）管他是誰死在這裡，我只想趕快回去！

**旁白**：斎藤的聲音中帶著恐懼和憤怒，還有一絲冷漠。

**旁白**：他指向祭壇後方——那裡矗立著一個巨大的門框。

**旁白**：門框由古老的石頭雕刻而成，表面刻滿了複雜的符文。

**旁白**：但門框的中央是空的，什麼都沒有。

**斎藤蓮**：（指著門框）那個...那個就是回原本世界的門吧？

**斎藤蓮**：（焦急地）既然到了這裡，我們快點回去啊！

**旁白**：但恵美沒有回答。她只是盯著翔的屍體，淚水不斷滑落。

**清水恵美**：（喃喃自語）是誰...是誰殺了翔...

**清水恵美**：（聲音顫抖）他明明...明明應該只是被困在這裡...

**斎藤蓮**：（更加焦躁）喂！妳有沒有在聽我說話？

**斎藤蓮**：（提高音量）我問妳話！我們要怎麼回去？

**旁白**：恵美依然沒有反應，她的目光始終停留在高山翔身上。

**斎藤蓮**：（憤怒地大吼）妳到底在搞什麼！

**斎藤蓮**：（走上前）別以為妳哭幾聲我就會同情妳！

**斎藤蓮**：（尖銳地）給我說清楚！到底是怎麼回事？

**李杰**：（站起來，擋在恵美前面）夠了，斎藤！

**旁白**：李杰的聲音冰冷而堅定。

**李杰**：（怒視斎藤）你沒看到她現在的狀態嗎？

**斎藤蓮**：（不甘示弱）那又怎樣？我也想知道到底發生什麼事！

**斎藤蓮**：（指著屍體）這傢伙是誰？為什麼會死在這裡？

**斎藤蓮**：（聲音尖銳）我們不是說好要一起回家的嗎？

**旁白**：李杰深吸一口氣，壓抑著怒火。

**李杰**：（冷靜地）斎藤...你想回家對吧？

**斎藤蓮**：（警戒地）那...那當然...

**李杰**：（指向祭壇）很簡單。

**李杰**：（語氣平淡）把這個人的屍體移開，然後你躺到祭壇上。

**李杰**：（看著斎藤）這樣...門就會打開。

**旁白**：斎藤愣住了。他看著李杰，眼中閃過懷疑和恐懼。

**斎藤蓮**：（結巴）你...你說什麼？

**斎藤蓮**：（往後退了一步）躺...躺到祭壇上？

**斎藤蓮**：（聲音顫抖）那...那這個人是怎麼死的？

**李杰**：（避開視線）我不知道。

**斎藤蓮**：（激動地）你不知道？你他媽的騙誰！

**斎藤蓮**：（指著李杰）你一定知道什麼！說清楚！

**旁白**：李杰沒有回答。他只是沉默地站在那裡。

**旁白**：斎藤盯著李杰看了一會兒，然後緩緩走向祭壇。

**旁白**：他伸出顫抖的手，抓住翔的肩膀。

**旁白**：屍體很冷，觸感僵硬。斎藤打了個寒顫，但還是咬著牙，用力將屍體拖下祭壇。

**旁白**：翔的身體沉重地落在地面上，發出沉悶的聲響。

**旁白**：恵美渾身無力地靠在李杰懷中，眼神空洞地看著那具被拖動的屍體。

**旁白**：她的嘴唇微微顫抖，但已經發不出任何聲音。

**旁白**：斎藤沒有理會恵美的狀態。他已經將屍體移到一邊。

**旁白**：斎藤看著空蕩蕩的祭壇，又看向李杰。

**斎藤蓮**：（冷笑）躺上去是吧？

**斎藤蓮**：（質疑地）那...為什麼不是你躺上去？

**斎藤蓮**：（看向恵美）或者她？為什麼一定要是我？

**李杰**：（張嘴想說話）這是因為...

**旁白**：但就在這時——

**旁白**：一陣清晰的腳步聲從聖殿入口傳來。

**旁白**：腳步聲很緩慢，很有節奏，在空曠的空間中迴盪。

**旁白**：所有人同時轉頭，看向聲音的來源。

**旁白**：一個身影從陰影中走出。

**旁白**：那是一個年輕男子，身材修長，穿著黑色的夾克和黑褲。

**旁白**：他的右手握著一把黑色的劍，劍身上殘留著乾涸的血跡。

**旁白**：他的臉部線條冷峻，眼神深邃而漠然。

**旁白**：他就這樣慢慢地、一步一步地，朝著他們走來。

**旁白**：每一步都踩在堅硬的石地板上，發出清脆的回響。

**凱蘭**：（警戒地握住劍柄）是誰！

**旁白**：但那個男子沒有回答。

**旁白**：他只是繼續走著，黑色的劍在微弱的光線下閃爍著冰冷的寒光。

**旁白**：隨著距離的拉近，李杰終於看清了那張臉。

**旁白**：那張熟悉的...卻又陌生的臉。
