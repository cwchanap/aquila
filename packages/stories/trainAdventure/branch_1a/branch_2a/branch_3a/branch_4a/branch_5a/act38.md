# 第三十八幕：絕境逃生

**旁白**：李杰的身體猛地一震。

**旁白**：那張臉——他見過。

**李杰**：（內心獨白）是他...新宿車站...那個靠在柱子旁的黑衣男...

**旁白**：李杰清楚地記得那個夜晚。

**旁白**：在新宿車站月台上，那個穿著黑色連帽衛衣的男子，靠在角落的柱子旁。

**旁白**：帽簷壓得很低，雙手插在口袋裡，一動也不動。

**旁白**：當時李杰就覺得那個人很違和，像是在觀察什麼。

**旁白**：但來到這個世界後，那個男子就再也沒有出現過。

**李杰**：（內心獨白）他一直都在這個世界...而且...他手上的劍...

**旁白**：李杰的視線落在黑劍上那些乾涸的血跡。

**旁白**：一個可怕的念頭在他腦中閃過。

**旁白**：但就在這時，恵美發出了一聲驚呼。

**清水恵美**：（震驚地）凌墨...你為什麼會在這裡...

**旁白**：恵美的聲音顫抖著，眼中滿是難以置信。

**旁白**：黑衣男子——凌墨——沒有回答。

**旁白**：他只是繼續往前走，腳步從容而穩定。

**凱蘭**：（握緊劍柄，低聲）小心...這傢伙很危險...

**凱蘭**：（轉頭對李杰）帶恵美小姐離開這裡！快！

**旁白**：凱蘭上前一步，擺出戰鬥姿態。

**旁白**：他雙手握劍，劍尖指向凌墨，全身肌肉緊繃。

**斎藤蓮**：（尖叫）不...不要...

**旁白**：斎藤的聲音充滿恐懼。他的雙腿發軟，踉蹌著往後退。

**斎藤蓮**：（慌亂地）不要過來！不要過來！

**旁白**：斎藤轉身想逃，但卻跌倒在地，然後手腳並用地爬向聖殿角落。

**旁白**：李杰的腦中快速運轉。

**李杰**：（內心獨白）要打開那扇門...需要有人躺在祭壇上...

**李杰**：（內心獨白）恵美...還是斎藤...

**旁白**：李杰看向癱軟在地的恵美，又看向縮在角落瘋狂顫抖的斎藤。

**旁白**：他的心跳如鼓，每一秒都彷彿無比漫長。

**旁白**：但他已經沒有時間猶豫了。

**李杰**：（內心獨白）對不起...斎藤...

**旁白**：李杰轉身，大步朝斎藤走去。

**斎藤蓮**：（看到李杰靠近，驚恐地）你...你要幹什麼...

**斎藤蓮**：（往後縮）不要...李杰...我們是朋友啊...

**旁白**：李杰沒有說話。

**旁白**：他衝到斎藤面前，右拳握緊，狠狠揮出。

**旁白**：拳頭結結實實地打在斎藤的臉上。

**斎藤蓮**：（悶哼）唔——

**旁白**：斎藤的頭猛地往後仰，眼睛一翻，整個人癱軟下來。

**旁白**：李杰抓住斎藤的衣領，用力將他拖向祭壇。

**旁白**：斎藤的身體沉重，鞋子在地面上拖出刺耳的摩擦聲。

**旁白**：李杰咬著牙，使盡全力將斎藤抬起，放到祭壇上。

**旁白**：就在斎藤的身體接觸到祭壇表面的那一刻——

**旁白**：整個空間震動了。

**旁白**：祭壇後方的空門框突然亮起耀眼的光芒。

**旁白**：白色、金色、銀色的光線交織在一起，在門框中央形成一個旋轉的光渦。

**旁白**：光渦越來越亮，散發出巨大的能量波動。

**旁白**：門——打開了。

---

**旁白**：就在同一時刻——

**旁白**：凌墨已經來到凱蘭面前。

**凱蘭**：（大喝）喝啊！

**旁白**：凱蘭揮劍斬向凌墨，劍刃在空中劃出一道銀光。

**旁白**：那是精靈劍術的精髓，速度快如閃電，力道足以斬斷岩石。

**旁白**：但凌墨只是微微側身。

**旁白**：凱蘭的劍從他身旁掠過，連衣角都沒有碰到。

**旁白**：下一秒——

**旁白**：凌墨的黑劍動了。

**旁白**：那是一個簡單而優雅的橫斬。

**旁白**：沒有多餘的動作，沒有花哨的技巧。

**旁白**：黑劍在空中劃過，留下一道漆黑的軌跡。

**旁白**：然後——

**旁白**：凱蘭的身體從腰部斷成兩截。

**旁白**：上半身和下半身緩緩分離，鮮血噴濺而出。

**凱蘭**：（眼睛睜大，發不出聲音）......

**旁白**：凱蘭的上半身跌落在地，眼中還殘留著震驚和難以置信。

**旁白**：整個過程，不到一秒。

**李杰**：（震驚地瞪大眼睛）凱蘭...

**旁白**：李杰的大腦一片空白。

**旁白**：凱蘭——那個優秀的精靈劍士，那個一路保護他們的戰士——

**旁白**：就這樣，在他眼前被斬成兩半。

**旁白**：凌墨踏過凱蘭的屍體，繼續朝李杰走來。

**旁白**：他的表情依然冷漠，彷彿剛才只是做了一件微不足道的事。

**旁白**：李杰的身體本能地顫抖。

**旁白**：但恵美——恵美還癱坐在地上，完全沒有意識到危險。

**李杰**：（內心獨白）不能猶豫...不能停下...

**旁白**：李杰衝向恵美，彎下腰，雙手從她的肩膀和膝蓋下穿過，將她整個人抱起。

**清水恵美**：（茫然地）李杰先生...

**李杰**：（急促地）抱歉了！

**旁白**：李杰轉身，朝著那扇發光的門狂奔。

**旁白**：他的肺部像要炸開，心臟瘋狂跳動。

**旁白**：身後，凌墨的腳步聲越來越近。

**旁白**：但李杰不敢回頭。

**旁白**：他只是抱緊恵美，用盡全力往前跑。

**旁白**：五米...三米...一米...

**旁白**：李杰縱身一躍，撲進那道光門。

**旁白**：一瞬間，耀眼的白光吞沒了他的視野。

**旁白**：一切都消失在那片光芒之中。
