# 第三十九幕：淚光中的誓言

**旁白**：不知道過了多久。

**旁白**：李杰感覺到意識逐漸回歸。

**旁白**：那片刺眼的白光慢慢消退，視野開始變得清晰。

**旁白**：他發現自己躺在冰冷的地面上。

**旁白**：身體很沉重，四肢無力，但至少還能動。

**李杰**：（低聲呻吟）嗯...

**旁白**：李杰撐著地面，緩緩坐起身。

**旁白**：他環顧四周——這是一個狹小的房間。

**旁白**：牆壁是灰白色的，沒有任何裝飾。空間大約只有三、四坪，顯得侷促而壓抑。

**旁白**：房間裡什麼都沒有，只有一扇門。

**旁白**：那扇門是金屬製的，表面有些斑駁的痕跡，門把是銀色的。

**旁白**：然後，李杰看到了恵美。

**旁白**：她坐在不遠處，背靠著牆壁，雙膝抱在胸前。

**旁白**：她的肩膀微微顫抖，臉埋在膝蓋間，發出輕微的啜泣聲。

**李杰**：（緩緩站起來）清水...

**旁白**：李杰走到恵美身邊，慢慢蹲下。

**旁白**：恵美抬起頭，眼眶紅腫，臉上滿是淚痕。

**清水恵美**：（哽咽）李杰先生...

**旁白**：她的聲音沙啞而脆弱。

**李杰**：（溫柔地）我們...活下來了。

**清水恵美**：（眼淚再次湧出）但是...翔他...

**旁白**：恵美的聲音顫抖著，說不下去。

**旁白**：李杰沉默了片刻，然後伸出手，輕輕撫摸恵美的頭髮。

**李杰**：我知道...我知道妳很難過...

**李杰**：但是...翔一定不希望看到妳這樣。

**清水恵美**：（搖頭，哭著）我...我對不起他...

**清水恵美**：（聲音破碎）都是因為我...他才會...

**旁白**：李杰的心一緊。

**旁白**：他張開雙臂，將恵美輕輕攬入懷中。

**李杰**：（輕聲）不要這樣...我不想看到妳這麼自責...

**李杰**：（更輕地）妳已經盡力了...真的...

**旁白**：恵美趴在李杰的肩膀上，眼淚浸濕了他的衣服。

**旁白**：她的身體顫抖著，但慢慢地，哭聲變得平緩。

**旁白**：李杰輕輕拍著她的背，沒有說話，只是默默陪伴。

**旁白**：過了很久，恵美終於停止了哭泣。

**旁白**：她從李杰懷裡退開，擦了擦眼淚。

**清水恵美**：（聲音沙啞）對不起...讓你看到我這麼難看的樣子...

**李杰**：（微笑）沒關係。

**旁白**：恵美低著頭，手指絞著衣角。

**旁白**：她沉默了很久，像是在組織語言。

**清水恵美**：（輕聲）李杰先生...其實...我一直有件事...沒有告訴你...

**李杰**：（看著她）什麼事？

**清水恵美**：（深吸一口氣）翔他...他的全名是高山翔...

**清水恵美**：（停頓）而且...他不只是我的朋友...

**旁白**：恵美抬起頭，眼中閃著淚光。

**清水恵美**：（顫抖地）他是...我的男朋友。

**旁白**：李杰愣住了。

**旁白**：這個消息像一記悶棍，重重砸在他心上。

**李杰**：（低聲）男朋友...

**清水恵美**：（點頭，眼淚再次滑落）我們...在高中的時候認識的...

**清水恵美**：（聲音輕柔）那時候他坐在我前面...總是會回頭跟我借筆記...

**清水恵美**：（苦笑）明明他自己也有筆記...只是想找理由跟我說話...

**旁白**：恵美的聲音中帶著懷念和哀傷。

**清水恵美**：後來...他跟我告白了...在櫻花樹下...

**清水恵美**：（擦眼淚）我們就這樣...在一起了...

**旁白**：李杰靜靜地聽著，心中湧起複雜的情緒。

**清水恵美**：那天...我們去參加一個朋友的生日派對...

**清水恵美**：派對結束後，我們一起搭電車回家...

**清水恵美**：然後...就來到了那個世界...

**旁白**：恵美的聲音越來越小。

**清水恵美**：我之前...沒有告訴你實話...

**清水恵美**：（低著頭）我說他只是普通朋友...是因為...

**清水恵美**：（聲音顫抖）我怕如果你知道他是我男朋友...你就不會那麼努力保護我了...

**旁白**：李杰的心一沉。

**李杰**：（複雜地）清水...

**清水恵美**：（抬起頭，眼中滿是愧疚）對不起...我知道這樣很自私...

**清水恵美**：（哽咽）但我真的...真的很害怕...

**清水恵美**：（小聲）我不想一個人待在那個世界...

**旁白**：李杰看著恵美淚流滿面的臉，心中百感交集。

**旁白**：有失望、有理解、有心疼...全都交織在一起。

**旁白**：但最終，他嘆了口氣。

**李杰**：（溫和地）我明白...

**李杰**：（看著恵美）在那種情況下...任誰都會害怕...

**李杰**：（輕聲）妳只是想活下去...想回家...這沒有錯。

**清水恵美**：（眼淚湧出）李杰先生...

**李杰**：（微笑）而且...就算妳告訴我真相...我也一樣會保護妳。

**清水恵美**：（愣住）咦...

**李杰**：（認真地）因為...這是我自己的選擇。

**李杰**：不管妳是誰，不管妳有什麼秘密...

**李杰**：（看著她的眼睛）我都會保護妳。

**旁白**：恵美的眼淚再次滑落，但這次，她的嘴角帶著一絲笑容。

**清水恵美**：（哭著笑）謝謝...謝謝你...

**旁白**：但笑容很快消失，取而代之的是深深的哀傷。

**清水恵美**：（低聲）但是...我已經失去翔了...

**清水恵美**：（聲音破碎）我再也...見不到他了...

**旁白**：李杰看著恵美悲傷的臉，心中湧起一股衝動。

**旁白**：他伸出手，輕輕捧起恵美的臉。

**李杰**：（認真地）清水...

**清水恵美**：（疑惑地看著他）李杰先生...

**李杰**：（深吸一口氣）妳失去了他...我知道這對妳來說很痛苦...

**李杰**：（看著她的眼睛）但是...

**李杰**：（輕聲）以後...讓我一直陪在妳身邊...好嗎？

**旁白**：恵美的眼睛睜大，臉上閃過震驚。

**清水恵美**：（顫抖）李杰先生...你...

**旁白**：李杰沒有給她說話的機會。

**旁白**：他緩緩靠近，在恵美的唇上落下一個溫柔的吻。

**旁白**：恵美的身體僵住了，眼睛瞪得很大。

**旁白**：但很快，她閉上了眼睛，眼淚從眼角滑落。

**旁白**：她沒有推開李杰，而是顫抖著，回應了這個吻。

**旁白**：在這個狹小而空蕩的房間裡，兩個受傷的靈魂緊緊依偎。

**旁白**：外面是什麼樣的世界，他們還不知道。

**旁白**：但至少在這一刻，他們不再孤單。
