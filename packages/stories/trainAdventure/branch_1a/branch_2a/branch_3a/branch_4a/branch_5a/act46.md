# 第四十六幕：上海的聯繫

**旁白**：房間陷入沉默。

**旁白**：大家都在思考著李杰和凌墨之間可能的共通點。

**旁白**：突然，大輔眼睛一亮。

**斎藤大輔**：（猛地拍桌）對了！我想到一個很重要的共通點！

**旁白**：所有人都看向他。

**斎藤大輔**：（興奮地）你們兩個...都是中國人！而且都是從上海來的！

**旁白**：李杰愣住了。

**旁白**：對啊...凌墨...那個名字...明顯是中文名字...

**李杰**：（恍然大悟）我怎麼沒想到...我們都有中文名字...

**李杰**：（皺眉）但是...這不太合理吧？

**李杰**：神樹為什麼要歧視中國人？而且那是另一個世界的東西...

**李杰**：它怎麼會知道地球上的國籍和城市？

**斎藤大輔**：（搖頭）我不是說神樹在歧視。我的意思是...

**斎藤大輔**：出生和成長在同一個城市，可能代表你們有其他的共通點。

**斎藤大輔**：比如...文化背景、生活環境、甚至是...某種特殊的經歷？

**旁白**：李杰沉思著。

**旁白**：確實...上海是個特殊的城市...

**旁白**：但這和不符合資格有什麼關係？

**伊藤莉子**：（突然興奮地）啊！我知道了！

**伊藤莉子**：（眼睛發亮）你們該不會...其實是失散多年的兄弟吧？

**伊藤莉子**：就像電視劇裡演的那樣！出生時被分開，長大後才發現彼此的存在！

**旁白**：莉子說得一臉認真。

**李杰**：（嘴角抽搐）失散的...兄弟...

**旁白**：李杰忍不住笑了。

**李杰**：（苦笑）莉子小姐...那是電視劇的劇情...

**李杰**：現實中哪有那麼多巧合...

**伊藤莉子**：（嘟嘴）誰說的！我看過好多這種故事！

**斎藤大輔**：（哈哈大笑）莉子啊，妳電視劇看太多了！

**清水恵美**：（也忍不住笑）雖然...雖然有點離譜...但莉子小姐的想像力很豐富呢...

**旁白**：氣氛稍微輕鬆了一些。

---

**斎藤大輔**：（收起笑容）對了，李杰，你還知道其他倖存者的情況嗎？

**斎藤大輔**：除了恵美和蓮之外，你有見過其他失蹤乘客嗎？

**李杰**：（搖頭）沒有。我在第一個車站就下車了。

**李杰**：之後...我就再也沒見過其他乘客了。只有恵美和蓮。

**斎藤大輔**：（點頭）這樣啊...

**旁白**：大輔若有所思地看著桌面。

**斎藤大輔**：（抬起頭微笑）好的，我明白了。暫時沒有其他問題了。

**斎藤大輔**：你們的情報非常有價值，謝謝你們願意分享這些。

**李杰**：（鞠躬）不會...這是我們應該做的...

**斎藤大輔**：（拿起平板電腦滑動幾下）我已經安排好了。一週後，你們可以搭飛機回日本。

**斎藤大輔**：機票、住宿、還有一些生活費，組織都會負擔。

**清水恵美**：（感激地）真的嗎...謝謝您...

**斎藤大輔**：（揮手）別客氣。這是我們應該做的。

**斎藤大輔**：你們失蹤了這麼久，要重新適應這個世界也需要時間。

**伊藤莉子**：（興奮地）一週時間耶！那你們可以好好享受夏威夷！

**伊藤莉子**：（眨眼）這裡有很多好玩的地方喔～海灘、火山、還有各種景點！

**伊藤莉子**：而且夏威夷的海很美，你們一定要去看看！

**李杰**：（微笑）謝謝...我們會的...

**清水恵美**：（點頭）嗯...能在這裡休息一週...真的很感激...

**斎藤大輔**：（溫和地笑）好好放鬆吧。這一年多...你們辛苦了。

**斎藤大輔**：（站起來）那我先去忙了。莉子，帶他們去看看周圍的環境吧。

**伊藤莉子**：（敬禮）是！包在我身上！

**旁白**：大輔走出房間。

**旁白**：莉子轉向李杰和恵美，笑容燦爛。

**伊藤莉子**：（拉著恵美的手）走走走！我帶你們去看海灘！

**伊藤莉子**：夏威夷的海超級美的！而且現在天氣正好！

**清水恵美**：（被拉著走有些慌張）咦咦？現在就去嗎？

**伊藤莉子**：（回頭對李杰笑）李杰先生也快點跟上！

**李杰**：（苦笑跟上）好好好...

**旁白**：三人走出建築。

**旁白**：陽光灑在身上，溫暖而舒適。

**旁白**：遠處傳來海浪拍打的聲音。

**旁白**：空氣中飄散著鹹鹹的海風味道。

**旁白**：這是...真正的地球。

**旁白**：他們...真的回來了。

**旁白**：李杰看著前方活潑的莉子，還有被拉著走的恵美。

**旁白**：嘴角不自覺地上揚。

**旁白**：雖然還有很多謎團沒有解開...

**旁白**：雖然前方的路還很不確定...

**旁白**：但至少現在...

**旁白**：他們可以享受這片刻的寧靜。
