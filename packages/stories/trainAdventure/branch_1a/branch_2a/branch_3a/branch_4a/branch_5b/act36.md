# 第三十六幕：被背叛的旅程

**旁白**：李杰緊緊握著拳頭,指甲深深陷進掌心。良久,他終於開口。

**李杰**：(沉重地)對不起...清水...我不能幫妳。

**旁白**：恵美的身體顫了一下。她依然跪在地上,低著頭,看不清表情。

**清水恵美**：(輕聲)是嗎...

**旁白**：她的聲音很平靜,平靜得讓人害怕。

**清水恵美**：(緩緩站起身,轉過頭去)我明白了...

**李杰**：(急忙解釋)清水,我理解妳想救妳朋友的心情...

**李杰**：但是...犧牲斎藤來換取他的自由...這不對。

**李杰**：斎藤雖然做過錯事,但他也在努力贖罪。他也有回家的權利...

**李杰**：我不能...我不能幫妳欺騙他,讓他成為那個被永遠困住的人...

**旁白**：恵美沒有回應。她只是靜靜地站在那裡,背對著李杰。

**旁白**：微風吹過,吹動她的長髮。在月光下,她的身影看起來異常孤獨。

**清水恵美**：(輕聲)李杰先生說得對。

**清水恵美**：是我...太自私了...

**旁白**：她的聲音聽起來很疲憊,像是失去了所有力氣。

**李杰**：(想說些什麼)清水...

**清水恵美**：(打斷他)我先過去找斎藤先生了。

**旁白**：恵美慢慢地朝著休息處走去,腳步沉重而緩慢。

**旁白**：李杰看著她的背影,心中湧起複雜的情緒。也許...讓她自己冷靜一下比較好。

**旁白**：就在這時,李杰突然感到後腦勺一陣劇痛。

**旁白**：他的視野瞬間變得模糊,身體失去了平衡。

**李杰**：(驚愕)什...

**旁白**：話還沒說完,他的意識就陷入了黑暗。

---

**旁白**：不知過了多久,李杰終於恢復了意識。

**旁白**：他的頭痛得厲害,後腦勺傳來陣陣刺痛。他緩緩睜開眼睛,映入眼簾的是茂密的樹枝和灌木。

**李杰**：(痛苦地呻吟)嗚...

**旁白**：他發現自己躺在一片灌木叢中,而不是之前的主路上。

**旁白**：李杰掙扎著坐起身來,用手撫著後腦勺。那裡有個明顯的腫包,碰到就痛得厲害。

**李杰**：(內心獨白)是誰...是誰襲擊了我...

**旁白**：他環顧四周,這裡是一片茂密的灌木叢。透過樹枝的縫隙,他隱約能看到不遠處的主路。

**旁白**：李杰搖搖晃晃地站起來,撥開擋路的樹枝,朝著主路走去。

**旁白**：走出灌木叢後,他發現主路就在距離他不到十公尺的地方。

**李杰**：(內心獨白)有人把我拖到這裡來...然後繼續趕路了...

**旁白**：李杰站在原地,努力回想剛才發生的事情。

**旁白**：他記得恵美朝斎藤走去,他在原地看著她的背影...然後...後腦勺突然一陣劇痛...

**李杰**：(內心獨白)當時清水和斎藤都在我前面...能從背後襲擊我的...

**李杰**：(震驚)只有凱蘭...

**旁白**：李杰的臉色變得蒼白。凱蘭...那個被艾拉如派來保護他們的精靈劍士...

**旁白**：為什麼?為什麼凱蘭要襲擊他?

**李杰**：(內心獨白)難道...難道凱蘭和清水...是一夥的?

**李杰**：(內心獨白)不...這不可能...凱蘭沒有理由...

**旁白**：李杰的腦中一片混亂。頭痛和困惑交織在一起,讓他無法冷靜思考。

**旁白**：但現在不是糾結這些的時候。他必須趕到終焉祭壇。

**旁白**：不管發生了什麼,他都必須弄清楚真相。

---

**旁白**：李杰忍著頭痛,沿著主路繼續前行。

**旁白**：他走得很慢,每一步都讓頭痛加劇。但他咬著牙,堅持著往前走。

**旁白**：周圍的景色逐漸變得荒涼。樹木越來越稀疏,取而代之的是光禿禿的岩石和乾裂的土地。

**旁白**：空氣中瀰漫著一股古老而沉重的氣息,讓人感到壓抑。

**旁白**：不知走了多久,前方終於出現了一座巨大的建築。

**旁白**：那是一座古老的石造建築,風格詭異而莊嚴。巨大的拱門矗立在入口處,門框上刻滿了複雜的符文。

**李杰**：(喘著氣)終於...到了...

**旁白**：李杰深吸一口氣,踏入巨大的拱門。

---

**旁白**：一踏入其中,周遭的一切聲音瞬間消失。

**旁白**：風聲、樹葉的沙沙聲、甚至自己的呼吸聲都變得微弱。取而代之的是一種令人窒息的死寂。

**李杰**：(警戒地環顧四周)這裡...

**旁白**：候車大廳寬敞得不像話。數根巨大的石柱從地面延伸到看不見的穹頂,每根石柱上都雕刻著奇異的符文。

**旁白**：地板由光滑的黑石鋪成,能清晰地映出他的倒影。空氣冰冷而稀薄,每一次呼吸都像是在吸入冰塊般刺痛。

**旁白**：李杰停下腳步,注意到大廳深處有兩個方向。

**旁白**：左側和右側各有一條幽暗的長廊,像兩道深不見底的傷口,從建築的體內裂開。

**李杰**：(內心獨白)終焉祭壇的門...應該在哪邊...

**旁白**：就在這時,李杰注意到地面上有什麼東西。

**李杰**：(蹲下查看)這是...血跡?

**旁白**：黑色的石地板上,有一道淡淡的血跡。不是很明顯,但在光滑的地面上依然能看出痕跡。

**旁白**：血跡從大廳入口延伸向左側的長廊。

**李杰**：(站起身,警戒地)是清水或斎藤的血嗎...

**旁白**：李杰做出決定,朝著左側的長廊走去。

---

**旁白**：長廊很長,兩側的牆壁刻滿了複雜的符文。腳步聲在狹窄的空間中迴盪,形成詭異的回音。

**旁白**：長廊內沒有任何光源,黑暗如實質般濃稠。李杰只能憑著微弱的光線,小心翼翼地往前走。

**旁白**：走了許久,前方終於出現了一個向下的樓梯。

**李杰**：(停在樓梯口)往下...

**旁白**：李杰開始下樓梯。一層、兩層、三層...樓梯似乎沒有盡頭,不斷向下延伸。

**旁白**：空氣越來越冷,壓力越來越大,像是在往地底深處下沉。

**旁白**：十層...十五層...二十層...李杰的雙腿開始發軟,頭痛也更加劇烈。但他咬牙堅持,不敢停下。

**旁白**：又下了幾層後,終於——樓梯到達了盡頭。

---

**旁白**：眼前是另一條走廊。

**旁白**：這條走廊比上面的更加寬闊,兩側的牆壁上鑲嵌著發光的寶石,散發著淡淡的藍色光芒。

**旁白**：李杰沿著走廊繼續前進。走廊盡頭,矗立著一扇巨大的古老石門。

**旁白**：石門高達數米,表面刻滿了精緻的浮雕和符文。門縫中透出微弱的金色光芒。

**李杰**：(停在門前,深吸一口氣)這就是...

**旁白**：李杰用力推開石門。門緩緩打開,發出低沉的轟鳴聲。

**旁白**：門縫越來越大,金色的光芒湧了出來,照亮了李杰的臉。

**旁白**：而當門完全打開的那一瞬間——

**旁白**：李杰整個人都僵住了。

**旁白**：他站在門口,一動也不動。眼睛睜得很大,臉上的表情從疲憊變成了震驚,然後是難以置信。

**旁白**：他的嘴巴微微張開,想說些什麼,但卻發不出任何聲音。

**旁白**：因為門後的景象...是他怎麼也想像不到的。
