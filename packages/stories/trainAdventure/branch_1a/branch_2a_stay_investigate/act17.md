# 第十七幕：焚村之夜

**旁白**：李杰在睡夢中感到一陣刺鼻的氣味。最初他以為是夢境的一部分，但隨著氣味越來越濃，他的意識逐漸清醒。

**旁白**：當他睜開眼睛時，發現臥室裡瀰漫著淡淡的灰色煙霧，正從門縫下緩緩滲入。空氣中混雜著木頭燃燒的焦味和一種說不出的刺激感，讓他的喉嚨開始發癢。

**李杰**：(猛地坐起身，警覺性瞬間提升) 煙？這是...著火了？

**旁白**：他迅速跳下床，衝向臥室門。打開門的瞬間，一股更濃的煙霧撲面而來，嗆得他咳嗽了幾聲。大廳裡的煙霧比臥室濃得多，視線已經開始模糊。

**旁白**：透過煙霧，他看到前門的方向閃爍著橘紅色的火光。木門的下半部分已經開始燃燒，火舌舔舐著門框，發出劈啪的聲響。熱浪撲面而來，讓他本能地後退了一步。

**旁白**：就在這時，另一間臥室的門也猛地打開了。艾拉如衝了出來，銀色的長髮散亂，臉上帶著少見的緊張神色。他的長袍上沾著灰塵，顯然也是剛剛被煙霧驚醒。

**艾拉如**：(急切地看向李杰，聲音裡透著擔憂) 李杰！你還好嗎？有沒有受傷？

**李杰**：我沒事！但是前門著火了！我們得快點出去！

**艾拉如**：(點點頭，轉身衝向清水的房間) 清水！清水！快醒醒！

**旁白**：清水恵美從房間裡跑了出來，臉色蒼白，眼神中充滿了驚恐。她緊緊抱著自己的背包，身體微微顫抖。

**清水恵美**：艾拉如先生！李杰先生！發生什麼事了？

**李杰**：(拉起她的手，語氣急促) 別問了！快走！

**旁白**：兩人衝回大廳，艾拉如已經用濕布蓋住了自己的口鼻，正在觀察著火勢。前門的火焰越燒越旺，已經無法從那裡出去了。

**艾拉如**：(轉頭看向兩人，聲音嚴肅) 整個村莊都在燃燒。我們必須從後門離開，現在就走！

**旁白**：三人迅速衝向後門。艾拉如用力推開門，外面的景象讓所有人都倒吸一口氣。

**旁白**：整個月光林地陷入了一片混亂。到處都是火光和濃煙，那些原本發著柔光的植物此刻被火焰吞噬，燃燒時發出噼啪的爆裂聲。建築物接連著火，橘紅色的火舌在夜空中狂舞。

**旁白**：空氣中瀰漫著嗆人的煙霧，混雜著木頭、植物燃燒的氣味，還有一種說不出的刺鼻味道。原本寧靜祥和的村莊，此刻如同煉獄。

**旁白**：精靈們從各個房屋中衝出來，有的抱著孩子，有的攙扶著老人，所有人都朝著同一個方向奔跑——北方。他們的臉上寫滿了恐慌和不解，優雅的舉止此刻被求生的本能取代。

**旁白**：清澈的溪流映照著火光，水面如同流淌的熔岩。那些曾經美麗的藤蔓橋開始燃燒，發出令人心碎的斷裂聲。

**李杰**：(看著眼前的景象，震驚得說不出話) 這...到底發生了什麼...

**艾拉如**：(拉著兩人的手臂，催促道) 別愣著！跟著大家走！我們要去銀月湖！

**旁白**：三人加入了逃難的人群。李杰一邊跑一邊忍不住回頭看，火光照亮了半邊天空，那些精美的建築正在火焰中倒塌。

**李杰**：(喘著氣問道) 銀月湖？那是什麼地方？

**艾拉如**：那是我們精靈的神聖之地，位於村莊北方的深處。湖水來自地底最純淨的泉源，有著古老的守護之力。當村莊遭遇襲擊時，那裡就是我們的避難所。

**旁白**：人群沿著一條向北的小路奔跑。這條路比他們來時的路更加崎嶇，兩旁的樹木在火光的映照下投下詭異的影子。偶爾能聽到遠處傳來的建築倒塌的巨響和火焰的咆哮。

**李杰**：(腦中閃過無數疑問) 到底是誰襲擊了村莊？為什麼要這麼做？

**艾拉如**：(臉色凝重) 我不知道。襲擊者應該是趁著所有人都在睡覺時動的手。這裡是永夜之地，黑暗給了襲擊者最好的掩護。他們可以悄無聲息地潛入村莊，在各處放火。

**李杰**：但是...這火勢蔓延得也太快了吧？我記得這裡到處都是霧氣，應該不容易燃燒才對...

**艾拉如**：(眉頭深鎖) 你說得對。這確實很不尋常。月光林地常年籠罩在濃霧之中，空氣濕潤，就算著火也不該蔓延得這麼快。除非...除非襲擊者使用了某種加速燃燒的東西。

**旁白**：李杰心中一沉。如果真是這樣，那麼這次襲擊絕對不是偶然，而是有預謀的。

**李杰**：(想了話麼，擔憂地問) 艾拉如先生，您的兒子...他還好嗎？他有收到消息嗎？

**艾拉如**：(臉上閃過一絲擔憂，但很快掩飾過去) 我還沒有收到他的消息。不過我已經派人去通知他了。他很強大，一定不會有事的。

**李杰**：他現在在哪裡？我們能去找他嗎？

**艾拉如**：(語氣有些含糊，明顯不想多談) 他...他在執行一項重要的任務。具體位置我也不太清楚。我們現在最重要的是先到銀月湖，確保所有人的安全。

**旁白**：李杰注意到艾拉如的閃躲，但現在不是追問的時候。他轉頭看向身旁的清水，想確認她是否還好。

**旁白**：令他意外的是，清水雖然臉上帶著驚訝，但並沒有表現出應有的恐懼。她的眼睛甚至閃爍著一種奇特的光芒，嘴角帶著一絲若有似無的微笑。

**清水恵美**：(氣喘吁吁，但表情平靜) 李杰先生...你沒事吧？

**李杰**：(喘著氣) 我沒事。妳呢？妳還好嗎？不害怕嗎？

**清水恵美**：(搖搖頭，語氣輕鬆) 嗯...我沒事...有李杰先生在，我不怕...

**李杰**：(內心獨白) 她看起來...一點都不害怕。甚至還有點...期待？這種反應...實在太不尋常了。不過至少她沒有嚇壞，這也算是好事吧。

**旁白**：李杰暗自慶幸清水還能保持這樣的心態。在這種混亂的情況下，如果她崩潰或者陷入恐慌，事情會變得更加複雜。

**李杰**：(努力擠出一個安慰的笑容) 別擔心，我會保護妳的。我們很快就能到安全的地方了。

**旁白**：人群繼續向北奔跑，身後的火光逐漸遠去。前方的道路越來越黑暗，只有精靈們手中舉著的一些發光石提供著微弱的照明。不知道還要跑多久，才能到達那個神秘的銀月湖。
