# 第二十三幕：部分真相

**旁白**：話音落下，灰色的世界開始消退。色彩逐漸回歸，銀月湖的景色重新浮現。

**旁白**：當視野完全恢復時，李杰和恵美依然站在神樹前，手掌貼在樹幹上，另一隻手緊緊相握。

**旁白**：艾拉如就站在不遠處，臉上帶著期待的表情。

**艾拉如**：（微笑）怎麼樣？儀式完成了嗎？

**旁白**：李杰和恵美呆呆地站在原地，誰也沒有回答。兩人的手依然握在一起，但已經感覺不到任何溫度。

**旁白**：艾拉如的笑容逐漸僵住了。他看著李杰茫然的表情，又看向恵美蒼白的臉色，似乎意識到了什麼。

**艾拉如**：（表情變得凝重）發生什麼事了？

**旁白**：沉默。恵美低著頭，身體微微顫抖著，似乎連開口的力氣都沒有了。

**李杰**：（艱難地開口，聲音沙啞）神樹說...我不符合資格。

**艾拉如**：（瞳孔縮小，難以置信地）不符合資格？這...這怎麼可能？

**旁白**：艾拉如走上前，看著神樹那平靜的樹幹。他低聲吟誦了幾句咒語，然後轉過頭，神色複雜地看著兩人。

**艾拉如**：（重重地嘆了口氣）看來是真的。契約...沒有建立成功。

**旁白**：他走到恵美身邊，輕輕拍了拍她的肩膀。

**艾拉如**：（溫柔地說）恵美，冷靜點。這不是妳的錯。

**清水恵美**：（終於忍不住哭了出來，蹲在地上摀著臉）對不起...艾拉如先生...我...我真的不知道會這樣...

**旁白**：李杰站在原地，看著崩潰的恵美，心中的疑問終於爆發了。

**李杰**：（走上前，語氣生硬）艾拉如先生，請告訴我實話。神樹說的「第一次」是什麼意思？還有...為什麼說我不是純粹的人類？

**旁白**：艾拉如沈默了片刻，看著李杰認真的眼神，最終點了點頭。

**艾拉如**：（嘆氣）跟我來吧。我們需要好好談談。

**旁白**：三人回到了銀月湖畔。艾拉如帶著他們來到湖邊一塊大石頭旁坐下。夜風輕輕吹過湖面，泛起陣陣漣漪。

**艾拉如**：（看著遠處的雙月，語氣悠長）其實...恵美並不是第一次來到這個世界。

**李杰**：（雖然心裡有了預感，但親耳聽到還是感到震驚）真的嗎？

**艾拉如**：是的。大約在兩年前，她和另外兩個人類夥伴，也曾經來到過這裡。

**李杰**：（追問）那...那兩個人呢？他們回去了嗎？

**艾拉如**：（沈默了一會兒）其中一個回去了。另一個...和妳一樣，被判定為不符合資格。

**旁白**：李杰心頭一震。

**李杰**：（急切地問）那個不符合資格的人...後來怎麼樣了？

**艾拉如**：（語氣變得沈重）他...留在了這個世界。因為不符合資格的人，是無法通過終焉祭壇的傳送門的。

**旁白**：這句話像是一記重錘，狠狠地砸在李杰的心上。留在這個世界？永遠？

**李杰**：（轉向恵美，語氣複雜）所以...妳之前回去了，但這次又回來了？為什麼？

**清水恵美**：（抬起頭，眼眶紅腫，聲音微弱）因為...因為我必須回來...

**旁白**：她抓著衣角，手指因為用力而發白。

**清水恵美**：上次回去後...我一直感到內疚。那個留在這裡的朋友...是為了救我才留下的。我...我想把他救回去。

**李杰**：（難以置信）所以...妳是故意再次來到這個世界的？

**清水恵美**：（點點頭，眼淚又落了下來）嗯...我找了很久，才找到了再次進入那個車站的方法...

**旁白**：李杰完全說不出話來。為了救朋友，竟然願意再次進入這個危險重重的異世界...這份勇氣和執著，讓他感到震撼。

**李杰**：（語氣緩和了一些）那...神樹說的「不是純粹的人類」又是什麼意思？

**艾拉如**：（表情變得有些奇怪，沈默了片刻）那是一個很複雜的問題。在我們精靈的眼中，靈魂的純粹度並不僅僅取決於你的種族。

**艾拉如**：有些人的靈魂中，摻雜了太多其他的東西——比如過度的執念、未解的仇恨，或者是...某種不屬於人類世界的力量。

**李杰**：（皺眉）我不明白。我只是個普通的工程師，哪來什麼不屬於人類世界的力量？

**艾拉如**：（搖搖頭）這我也解釋不清楚。神樹的判斷是直指靈魂本質的。或許...連你自己都不知道，你的靈魂中隱藏著什麼。

**旁白**：這番話讓李杰感到莫名的寒意。他的靈魂中隱藏著什麼？他自己都不知道的東西？

**艾拉如**：至於那個判定標準...我想，可能是因為恵美。

**李杰**：（疑惑）因為恵美？

**艾拉如**：是的。精靈契約需要兩者的靈魂產生共鳴。恵美因為已經簽訂過一次契約，她的靈魂已經不再是純粹的白紙了。

**艾拉如**：所以，這一次如果要建立契約，對對方的要求會變得極其苛刻。

**旁白**：李杰看向恵美，只見她低著頭，神情頹然。

**李杰**：（內心獨白）所以...我成了她的「犧牲品」嗎？因為她的靈魂已經不純粹了，所以我才不符合資格？

**旁白**：心中湧起一股說不清的情緒。是憤怒？是無奈？還是同情？

**艾拉如**：不過，你也不用太絕望。雖然精靈契約沒成，但並代表你就一定回不去。

**李杰**：（眼睛一亮）真的嗎？還有其他方法？

**艾拉如**：（點點頭，眼神深邃）是的。雖然那條路會比精靈契約難得多...但並非全無可能。

**艾拉如**：這也是為什麼我一直想讓我的兒子回來的原因。他知道一條...不需要精靈契約也能啟動祭壇的密道。

**旁白**：希望的火花再次燃起。雖然前路依然渺茫，但至少還沒有完全斷絕。

**李杰**：（認真地看向艾拉如）艾拉如先生，請務必告訴我們那個方法！

**艾拉如**：（看著他，沈默了很久，最終緩緩開口）好吧。既然神樹已經做出了判斷，我們也只能走那條路了。

**艾拉如**：但是，在告訴你之前，我必須先確認一件事。

**李杰**：什麼事？

**艾拉如**：你的決心。為了回家，你願意付出多大的代價？

**旁白**：空氣中充滿了肅穆的氣氛。李杰深吸一口氣，眼神變得前所未有的堅定。

**李杰**：無論多大的代價，我都願意。

**旁白**：艾拉如看著他，滿意地點了點頭。而一旁的恵美，則露出了複雜的表情——那裡面有希望，但更多的是一絲隱隱的...恐懼？

**旁白**：真相，似乎只揭開了冰山一角。在那深邃的迷霧之下，還有更大的祕密在等待著他們。

**旁白**：三個人默默地走在銀月湖北岸的小路上，每個人都沉浸在各自的思緒中。那段美好的湖畔時光，彷彿已經是很久以前的事了。

**李杰**：（內心獨白）她曾經來過這裡...這是我完全沒想到的。

**李杰**：（內心獨白）但是...她為什麼會來到這個世界兩次...這真的只是倒霉嗎？還是有什麼其他原因？

**旁白**：李杰想起在銀月湖畔的那些日子，恵美臉上那些真誠的笑容，還有她努力為他煮粥的樣子。那些，應該都不是假的吧？

**旁白**：艾拉如站起身來，拍了拍身上的草屑。

**艾拉如**：（對恵美說）恵美，我們...需要再找一個人。

**旁白**：恵美抬起頭，眼眶紅紅的，臉上滿是淚痕。她看起來疲憊不堪，但還是點了點頭。

**清水恵美**：（聲音沙啞）嗯...我知道...

**艾拉如**：（溫和地）先回去休息一下吧。今天對你們來說...都太沉重了。

**旁白**：艾拉如率先轉身，朝著來時的路走去。恵美搖搖晃晃地站起來，低著頭跟在後面。

**旁白**：李杰站在原地，看著神樹在微風中輕輕搖曳的枝葉。那棵曾經給他詭異感覺的樹，現在看起來只是一棵普通的、古老的樹而已。

**旁白**：他深吸一口氣，轉身跟上了前面兩人的腳步。

**旁白**：三個人默默地走在銀月湖北岸的小路上，每個人都沉浸在各自的思緒中。那段美好的湖畔時光，彷彿已經是很久以前的事了。
