# 第二十八幕：真相與威脅

**旁白**：第二天一早，李杰找到了正在整理工具的斎藤。

**李杰**：斎藤，能跟我來一下嗎？有些事情想和你聊聊。

**斎藤蓮**：（抬頭，表情有些警惕）什...什麼事？現在要開始工作了...

**李杰**：（平靜地）不會耽誤太久。跟我來吧。

**旁白**：李杰的語氣雖然平靜，但帶著一股不容拒絕的堅定。斎藤咬了咬嘴唇，最終還是跟了上去。

**旁白**：兩人走到村落邊緣一處較為僻靜的地方，周圍沒有其他人。

**李杰**：（轉身面對斎藤）斎藤，我想問你一件事。

**斎藤蓮**：（故作輕鬆）什...什麼事啊？神神秘秘的...

**李杰**：（盯著他的眼睛，語氣嚴肅）昨晚，我去了你的帳篷。

**旁白**：斎藤的臉色瞬間變得慘白，身體微微顫抖了一下。

**斎藤蓮**：（聲音顫抖）你...你說什麼...

**李杰**：我看到了你手臂上的傷痕。那些燒傷。

**旁白**：斎藤像是被雷擊中了一樣，整個人僵在原地。他的呼吸變得急促，眼神中充滿了恐懼和絕望。

**斎藤蓮**：（激動地）你...你偷看我！你憑什麼這麼做！

**李杰**：（平靜地）我不想偷看。但你的反應太奇怪了，我必須確認這會不會對我們建立契約造成影響。

**李杰**：現在，告訴我，那些燒傷是怎麼來的？

**斎藤蓮**：（咬牙切齒，眼眶紅了）這不關你的事！這是我的私事！

**李杰**：（語氣變得嚴厲）這不只是你的私事。我們現在是一個團隊。如果你隱瞞了什麼重要的事情，可能會害死我們所有人。

**李杰**：那些燒傷...是你在來到這個世界之前就有的，還是來到這裡之後才有的？

**旁白**：斎藤低著頭，沉默不語。他的手緊緊握著，指甲深深陷進掌心。

**李杰**：不說嗎？好，那我現在就去告訴艾拉如先生。看看他會怎麼想。

**斎藤蓮**：（猛地抬頭，尖叫道）別！別告訴他！

**旁白**：斎藤的反應非常激烈。他的眼中充滿了哀求和恐懼。

**斎藤蓮**：（顫抖地）我...我告訴你...我告訴你就是了...

**旁白**：斎藤深吸一口氣，像是下定了什麼決心。

**旁白**：他環顧四周，確認沒有人注意到這邊後，深吸一口氣，開始緩慢地朝李杰走去。

**斎藤蓮**：（勉強笑著）好...好吧...我告訴你...但是...你得答應我不告訴任何人...

**旁白**：斎藤一步一步接近李杰，右手垂在身側，看起來很自然。但李杰注意到，他的袖子裡似乎藏著什麼東西。

**旁白**：一個小小的反光物體——那是刀刃的光澤。

**李杰**：（內心獨白）這傢伙...居然藏了刀？而且藏得這麼明顯...他到底在想什麼...

**旁白**：李杰表面上保持平靜，實際上已經做好了應對的準備。

**斎藤蓮**：（繼續靠近，聲音顫抖）其實...事情是這樣的...我...我...

**旁白**：就在斎藤距離李杰只有兩步之遙時，李杰突然聳了聳肩。

**李杰**：（嘆氣）算了。看來你是不打算好好說了。

**旁白**：話音未落，李杰突然向前踏出一步，右腳猛地踢向斎藤的腹部。

**斎藤蓮**：（驚愕）什...

**旁白**：斎藤完全沒有反應過來。他的身體像蝦米一樣弓起來，然後雙膝跪地。

**旁白**：藏在袖子裡的小刀掉落在地上，發出清脆的響聲。

**斎藤蓮**：（捂著肚子，痛苦地呻吟）嗚...咳...

**旁白**：李杰彎腰撿起那把小刀，仔細打量了一下。那是一把很普通的小刀，刀刃不長，大概是從工地上偷來的工具。

**李杰**：（搖頭，語帶嘲諷）藏刀藏得這麼明顯，你是想嚇唬我還是想傷害我？

**李杰**：（把玩著小刀）說實話，斎藤，你真的不適合做這種事。太業餘了。

**斎藤蓮**：（跪在地上，憤怒地抬起頭）你...你這個混蛋...

**旁白**：斎藤的眼中燃燒著憤怒，但更多的是屈辱和恐懼。他想站起來，但腹部的疼痛讓他動彈不得。

**旁白**：他意識到，自己徹底搞砸了。

**斎藤蓮**：（咬牙切齒）你...你早就發現了...

**李杰**：（冷靜地）當然。你以為你藏得很好嗎？

**旁白**：李杰將小刀收進自己的口袋，然後居高臨下地看著跪在地上的斎藤。

**李杰**：現在，你有兩個選擇。要麼老實告訴我真相，要麼我現在就帶你去見艾拉如先生。

**李杰**：自己選吧。

**旁白**：斎藤低著頭，身體微微顫抖。他的拳頭緊緊握著，指甲深深陷進掌心。

**旁白**：良久，他終於開口了。

**斎藤蓮**：（聲音沙啞，充滿無力感）好...我說...我全都告訴你...

**旁白**：斎藤緩緩抬起頭，眼中滿是絕望和認命的神色。

**斎藤蓮**：但是...你得保證...不管我說什麼...你都不會告訴精靈們...

**李杰**：（沉默片刻）我會根據你說的內容來判斷。

**斎藤蓮**：（苦笑）這樣啊...也好...反正我也沒有選擇了...

**旁白**：斎藤深吸一口氣，準備說出那個他一直隱藏的秘密。
