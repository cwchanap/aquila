# 第二十九幕：陰謀的真相

**斎藤蓮**：（深吸一口氣）這件事...要從我們還在火車上的時候說起...

**旁白**：斎藤低著頭，聲音顫抖著。

**斎藤蓮**：就在火車剛離開隧道的時候...那個...那個看起來很自私的中年男人...悄悄接近了我。

**李杰**：（皺眉）那個中年男人？

**斎藤蓮**：（點頭）就是那個...看起來很膽小、很自私的那個。他...他把我叫到車廂的角落...

**斎藤蓮**：（聲音更加顫抖）然後他對我說...他知道我的秘密。

**李杰**：秘密？

**斎藤蓮**：（緊緊握著拳頭）對...一個非常重要的秘密...一個如果被人知道，我的人生就會完全毀掉的秘密...

**斎藤蓮**：我不知道他是怎麼知道的...他甚至拿出了證據...一些照片，還有一段錄音...

**斎藤蓮**：（崩潰地）我當時嚇壞了...我問他想要什麼...

**斎藤蓮**：他給了我一樣東西。一個精緻的銀色打火機。還有...一張手繪的地圖。

**斎藤蓮**：他對我說...等到了目的地，讓我找機會離開人群，去地圖上標註的一個地方。

**斎藤蓮**：他說那個地方叫「月光林地」...是一群精靈居住的地方。

**李杰**：（震驚）他怎麼會知道這些？

**斎藤蓮**：（搖頭）我不知道...我當時腦子一片混勞，只想著要保住我的秘密...

**斎藤蓮**：他命令我...到了那裡之後，找機會放火燒掉村莊。

**李杰**：（難以置信）放火燒掉村莊？為什麼？

**斎藤蓮**：他說...那是為了「清理障礙」...為了讓他在接下來的計劃中更順利...

**斎藤蓮**：（哭著說）我一開始是拒絕的！我真的拒絕了！我雖然自私，但我不是殺人犯！

**斎藤蓮**：但是...他威脅我...他說如果我不照做，他現在就把證據交給警察...

**斎藤蓮**：而且...他還說...這個世界根本就不是真實的。這裡只是一個遊戲，或者一場夢。

**斎藤蓮**：他說燒掉村莊不會真的死人...那些精靈只是虛擬的數據...

**斎藤蓮**：（捂著臉）我...我當時真的信了...或者說，我被迫讓自己相信了...

**斎藤蓮**：來到這個世界後...趁著大家休息的時候，我偷偷離開了隊伍，按照地圖找到了月光林地...

**斎藤蓮**：我...我用了那個打火機...那不是普通的火...那火燒得非常快，非常猛...

**斎藤蓮**：（聲音哽咽）當我看到精靈們在火中慘叫...看到他們絕望的眼神...我才意識到...這根本不是什麼遊戲...這一切都是真實的！

**斎藤蓮**：我...我殺了人...我親手毀掉了他們的家園...

**李杰**：（冷冷地）所以，你就跑了？

**斎藤蓮**：（羞愧地低頭）我...我被火圍住了...我的手臂也被燒傷了...我拼命逃跑，躲在森林裡...

**斎藤蓮**：我原本以為中村會來找我...會把證據還給我...

**斎藤蓮**：但是...他只是冷冷地笑了...然後說...我不需要知道。

**旁白**：斎藤的眼淚滴落在地上。

**斎藤蓮**：（顫抖地）他還威脅我...如果我不服從命令，或者...或者把這件事告訴任何人...

**斎藤蓮**：他就會在成功回到原本的世界後，把我的秘密...公布到網路上...

**斎藤蓮**：（崩潰地）那樣的話...我就完了...我的人生...我的家人...一切都完了...

**旁白**：李杰沉默地看著崩潰的斎藤。他的心情極其複雜——既憤怒於中村的陰謀，又對斎藤的懦弱感到失望。

**李杰**：（內心獨白）那個中年男人...中村隼人...我一直以為他只是個自私膽小的普通人...沒想到...他居然隱藏著這麼多東西...

**李杰**：（內心獨白）他怎麼知道這個世界的事情？怎麼知道精靈村莊的位置？還有那張地圖...那個打火機...這一切都太不正常了...

**旁白**：李杰強迫自己冷靜下來，繼續追問。

**李杰**：中村有說...他什麼時候會來找你嗎？

**斎藤蓮**：（搖頭）沒有...他只是說...他最終會來的。但沒有說具體的時間...

**斎藤蓮**：（苦笑）我一直在等...等了這麼久...但他從來沒有出現過...

**李杰**：所以...你燒毀了月光林地之後，就一直在等他？

**斎藤蓮**：（點頭，聲音哽咽）我...我其實...很後悔...

**斎藤蓮**：燒毀村莊之後，我躲在森林裡...看著那些火焰...看著精靈們驚慌失措地逃跑...

**斎藤蓮**：我...我感覺自己像個怪物...

**旁白**：斎藤抱著頭，身體劇烈顫抖。

**斎藤蓮**：那些燒傷...就是那個時候留下的。火勢蔓延得太快了，我來不及逃跑，手臂被燒傷了...

**斎藤蓮**：每次看到這些傷疤，我就會想起那一天...想起我做過的事...

**斎藤蓮**：（抬起頭，眼中滿是淚水）我...我其實有想過...想過去月光林地告訴精靈們真相...

**斎藤蓮**：我想道歉...想承擔責任...但是...但是我沒有勇氣...

**斎藤蓮**：我害怕他們會殺了我...害怕中村會把我的秘密公開...我太懦弱了...

**斎藤蓮**：所以...我一直在迷霧森林裡徘徊...不敢前進，也不敢後退...

**斎藤蓮**：直到...直到我走出了森林，來到了銀月湖...

**斎藤蓮**：（聲音更加哽咽）本來想偷偷觀察一下情況，但被精靈們抓住了...

**斎藤蓮**：（深吸一口氣）我想...既然被抓住了，那就...至少幫他們重建村莊吧...

**斎藤蓮**：這是我能做的...最起碼的贖罪...

**旁白**：斎藤說完後，整個人像是失去了所有力氣，癱坐在地上。

**旁白**：李杰看著眼前這個懦弱、自私、卻又在痛苦中掙扎的年輕人，心情無比沉重。

**李杰**：（內心獨白）中村隼人...他到底是什麼人？他的目的是什麼？為什麼要燒毀精靈村莊？

**李杰**：（內心獨白）而且...他為什麼會知道那麼多事情？他是不是...也和這個世界有某種聯繫？

**旁白**：太多的疑問在李杰腦中盤旋。但現在，他必須先處理眼前的問題。

**旁白**：他必須決定...該如何對待斎藤，以及...是否要把這件事告訴艾拉如和恵美。
