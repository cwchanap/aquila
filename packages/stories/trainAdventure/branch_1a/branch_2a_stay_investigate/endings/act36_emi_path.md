# 第三十六幕：祭壇上的死者

**旁白**：時間彷彿凝固了一般。李杰站在那裡，看著跪在地上的恵美，沉默了很久很久。

**旁白**：他的腦海中不斷閃過各種念頭——斎藤的期待、恵美朋友的絕望、艾拉如的信任、還有那個殘酷的選擇...

**旁白**：終於，李杰長長地嘆了一口氣。

**李杰**：（疲憊地）我...我會幫妳。

**旁白**：恵美猛地抬起頭，眼中閃過難以置信的光芒。

**清水恵美**：（顫抖地）真...真的嗎？

**李杰**：（轉過頭，不想看她的眼睛）但我不保證一定能成功。而且...

**李杰**：（語氣沉重）我不知道這樣做是對是錯。我只是...只是不想看到妳這樣痛苦。

**清水恵美**：（泣不成聲）謝謝...李杰先生...真的...謝謝你...

**旁白**：李杰沒有回答，只是彎下腰，扶起癱坐在地上的恵美。

**李杰**：我們先去找大叔吧。他應該知道些什麼。

**旁白**：就在這時，原本安靜的森林裡突然傳來一陣嘈雜聲。

**旁白**：那是金屬碰撞的聲音，還有急促的腳步聲。

**凱蘭**：（從陰影中走出，手握劍柄）李杰先生，清水小姐，發生什麼事了？

**旁白**：凱蘭——艾拉如派來護送他們的精靈劍士，此刻一臉警覺地看著兩人。

**李杰**：（指著祭壇）凱蘭先生，妳看那個。

**旁白**：凱蘭看向祭壇，瞳孔猛地收縮。

**凱蘭**：（震驚）那是...人類？

**旁白**：他快步上前，檢查了一下躺在祭壇上的男子。

**凱蘭**：（臉色鐵青）他已經死了。而且...是剛死不久。

**李杰**：（心頭一緊）是誰做的？

**凱蘭**：（環顧四周，眼神冰冷）我不知道。但這裡不安全。我們必須馬上離開。

**李杰**：但是大叔還沒回來！我們得等他！

**凱蘭**：（嚴肅地）沒時間了。那些「追蹤者」就在附近。如果我們被包圍，誰也走不了。

**清水恵美**：（驚恐地抓住李杰的袖子）李杰先生...

**李杰**：（咬牙）好。凱蘭先生，帶路吧。

**旁白**：凱蘭點點頭，轉身朝著聖殿深處走去。李杰扶著恵美緊隨其後。

**旁白**：聖殿內部的走廊陰暗而潮濕。牆壁上刻滿了古老的符文，散發著微弱的幽光。

**旁白**：腳步聲在寂靜的長廊中迴盪，顯得格外突兀。

**李杰**：（低聲問凱蘭）凱蘭先生，這裡有後門嗎？

**凱蘭**：有一條通往地底的密道。我們可以從那裡繞到森林的另一邊。

**旁白**：三人加快了腳步。然而，就在他們快要到達走廊盡頭時，一個黑影突然擋住了去路。

**凱蘭**：（立刻拔出長劍，橫在身前）是誰！

**旁白**：那個黑影緩緩走出陰影。那是一個穿著黑色連帽衫的男子，帽簷壓得很低，看不清臉。

**旁白**：他的手中握著一把細長的黑劍，劍身上流動著詭異的紅光。

**凱蘭**：（瞳孔收縮）凌墨...

**旁白**：黑衣男子沒有回答，只是緩緩舉起了手中的劍。

**凱蘭**：（對李杰大喊）帶清水走！快！

**李杰**：可是妳...

**凱蘭**：（語氣決絕）這是我的職責！快走！

**旁白**：凱蘭說完，嬌喝一聲，揮劍衝向那個叫凌墨的男子。

**旁白**：兩把劍在空中激烈碰撞，爆發出耀眼的火花。

**李杰**：（一咬牙，抱起恵美）走！

**旁白**：他轉過身，朝著相反的方向狂奔而去。

**旁白**：身後傳來凱蘭的怒吼聲和金屬交織的撞擊聲。李杰不敢回頭，他只能用盡全力奔跑。

**旁白**：聖殿的走廊彷彿沒有盡頭一般。李杰感覺肺部像要炸開，心臟瘋狂跳動。

**旁白**：突然，前方出現了一扇散發著金色光芒的拱門。

**凱蘭**：（遠遠傳來聲音，帶著一絲疲憊）就是那裡！跳進去！

**旁白**：李杰看著那扇光門，又回頭看了一眼。

**旁白**：凱蘭已經渾身是血，但依然死死纏住那個黑衣男子。

**凱蘭**：（最後的嘶吼）走啊！

**旁白**：李杰閉上眼睛，抱緊恵美，縱身躍向那片耀眼的金色光芒。

**凱蘭**：（開始走向拱門）跟我來。

**旁白**：四人走向那座巨大的拱門。當他們踏入拱門的瞬間——

**旁白**：周遭的一切聲音瞬間消失。森林的風聲、樹葉的沙沙聲、甚至自己的呼吸聲都變得微弱。取而代之的是一種令人窒息的死寂。

**斎藤蓮**：（小聲）這裡...好安靜...

**凱蘭**：（警戒地）小心點...

**旁白**：眼前是一個寬敞的大廳。數根巨大的石柱從地面延伸到看不見的穹頂，每根石柱上都雕刻著奇異的符文。

**旁白**：地板由光滑的黑石鋪成，能清晰地映出他們的倒影。空氣冰冷而稀薄，每一次呼吸都像是在吸入冰塊般刺痛。

**凱蘭**：（指向前方）祭壇在深處。我們要穿過這條長廊，然後往下走。

**旁白**：四人沿著大廳前進。前方是一條幽暗的長廊，像一道深不見底的傷口，從建築的體內裂開。

**旁白**：長廊很長，兩側的牆壁刻滿了複雜的符文。腳步聲在狹窄的空間中迴盪，形成詭異的回音。

**旁白**：走了一段路後，前方出現了一個向下的樓梯。

**凱蘭**：（停在樓梯口）往下走。祭壇在地底深處。

**旁白**：四人開始下樓梯。一層、兩層、三層...樓梯似乎沒有盡頭，不斷向下延伸。

**旁白**：空氣越來越冷，壓力越來越大，像是在往地底深處下沉。

**斎藤蓮**：（喘著氣）還...還要走多久...

**凱蘭**：快到了。

**旁白**：十層...十五層...二十層...終於，樓梯到達了盡頭。

**旁白**：眼前是另一條走廊。這條走廊比上面的更加寬闊，兩側的牆壁上鑲嵌著發光的寶石，散發著淡淡的藍色光芒。

**凱蘭**：（指向前方）就在前面。

**旁白**：走廊盡頭，矗立著一扇巨大的古老石門。石門高達數米，表面刻滿了精緻的浮雕和符文。門縫中透出微弱的金色光芒。

**凱蘭**：（推門）就是這裡了。

**旁白**：四人合力推開石門。門緩緩打開，發出低沉的轟鳴聲。

**旁白**：眼前的景象讓所有人屏住了呼吸。

**旁白**：那是一個極其寬敞的空間——不，應該說是聖地。

**旁白**：穹頂高聳入雲，牆壁四周雕刻著精美的壁畫。空氣中瀰漫著一股神聖的氣息，讓人不由得心生敬畏。

**旁白**：在空間的正中央，矗立著一座白色大理石祭壇。祭壇四周環繞著淡淡的金色光芒，像是有無形的力量在守護著它。

**旁白**：凱蘭帶領著三人走向祭壇。

**旁白**：但是...

**旁白**：當四人看清楚祭壇上的景象時——

**旁白**：一個年輕的男性躺在祭壇上。他看起來大約二十歲左右，身穿白色的長袍，長袍上繡著精緻的花紋，看起來像是某種儀式用的服裝。

**旁白**：他的胸口插著一把刀。

**旁白**：鮮血從傷口流出，染紅了白色的長袍，在祭壇的白色大理石上凝結成一灘暗紅色的血跡。

**旁白**：他閉著眼睛，臉色蒼白如紙，一動不動地躺在那裡。
