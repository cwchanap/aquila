# 第三十六幕：沈重的抉擇

**旁白**：李杰緊緊握著拳頭,指甲深深陷進掌心。良久,他終於開口。

**李杰**：(沉重地)對不起...清水...我不能幫妳。

**旁白**：恵美的身體顫了一下。她依然跪在地上,低著頭,看不清表情。

**清水恵美**：(輕聲)是嗎...

**旁白**：她的聲音很平靜,平靜得讓人害怕。

**清水恵美**：(緩緩站起身,轉過頭去)我明白了...

**李杰**：(急忙解釋)清水,我理解妳想救妳朋友的心情...

**李杰**：但是...犧牲斎藤來換取他的自由...這不對。

**李杰**：我們不能決定別人的生死,即便那是為了救我們重要的人。

**清水恵美**：(沒有回頭,語氣冰冷)對不對...重要嗎?

**清水恵美**：在這個扭曲的世界裡,還有什麼是正確的?

**旁白**：她邁開腳步,緩緩走向祭壇。

**李杰**：(緊張地)清水!妳要做什麼!

**旁白**：李杰想要衝上前阻止她,但凱蘭卻伸出手攔住了他。

**凱蘭**：(眼神哀傷)讓她去吧。這是她自己的選擇。

**旁白**：恵美走到祭壇前。她看著躺在祭壇上的高山翔,眼神中充滿了溫柔和決絕。

**清水恵美**：(低聲呢喃)翔...等我...我馬上就來救你。

**旁白**：她轉過頭,看向躲在石柱後的斎藤蓮。

**斎藤蓮**：(驚恐地)妳...妳看我幹什麼!我警告妳...別過來!

**旁白**：恵美沒有理會他的威脅。她從懷中拿出一面小巧的銀色鏡子——那是艾拉如交給她的「引導之鏡」。

**清水恵美**：(對著鏡子低聲唸誦)——以靈魂為誓,以生命為引。

**旁白**：銀鏡爆發出刺眼的紅光!

**斎藤蓮**：(發出淒厲的慘叫)啊——!我的眼睛!我的頭!

**旁白**：斎藤蓮痛苦地摀住腦袋,整個人癱倒在地。無數紅色的光絲從他的口鼻中湧出,被吸入銀鏡之中。

**李杰**：(憤怒地吼道)清水!快停手!

**旁白**：李杰想要推開凱蘭,但凱蘭的力量大得驚人,將他死死按在原地。

**凱蘭**：(低聲說)儀式已經開始了...停不下來的。

**旁白**：短短幾秒鐘後,斎藤蓮的身體徹底停止了掙扎。他的皮膚變得乾癟灰白,整個人彷彿被吸乾了水分的乾屍。

**旁白**：而那面銀鏡,此刻正散發著妖艷而恐怖的紅光。

**清水恵美**：(轉身看向祭壇,語氣近乎瘋狂)翔!你看!我做到了!

**旁白**：她將銀鏡按在高山翔的胸口。

**旁白**：紅光瞬間沒入高山翔的體內。奇蹟發生了——高山翔蒼白的臉色開始恢復紅潤,停止的呼吸重新變得有力。

**旁白**：他緩緩睜開雙眼。

**高山翔**：(茫然地)恵美...是你嗎...

**清水恵美**：(驚喜地抱住他)是我!是我啊!翔!我們終於可以在一起了!

**旁白**：然而,喜悅只持續了一瞬間。

**旁白**：原本神聖肅穆的聖地,突然開始劇烈震動。無數漆黑的裂縫在地面和牆壁上蔓延開來。

**凱蘭**：(臉色大變)不好!契約被扭曲了!這個空間要崩潰了!

**旁白**：一股極其邪惡的氣息從那些裂縫中噴湧而出。

**旁白**：一個冰冷、漠然、充滿殺意的聲音在聖殿中迴盪。

**凌墨**：(聲音如寒風般凜冽)貪婪的人類...竟然妄想玩弄靈魂...

**旁白**：那個黑衣男子——凌墨,不知何時已經出現在了祭壇上方。他手握黑劍,眼神中透著令人膽寒的殺機。

**凌墨**：(舉起黑劍)既然妳們破壞了規則,那就全部留下來陪葬吧。

**凱蘭**：(拔出長劍,對李杰大喊)帶他們走!去右邊的門!快!

**李杰**：(看著眼前的混亂,腦中一片空白)可是...

**凱蘭**：(最後的嘶吼)走啊!不要讓凱蘭白白犧牲!

**旁白**：李杰一咬牙,衝向祭壇,想要拉起恵美和高山翔。

**旁白**：但恵美卻推開了他的手。

**清水恵美**：(微笑著,眼神中充滿了平靜)李杰先生...對不起。你走吧。

**清水恵美**：(握住高山翔的手)我哪也不去。我要留在這裡...永遠陪著翔。

**李杰**：(急切地)妳瘋了嗎!這裡要崩潰了!

**清水恵美**：(搖搖頭)對我來說...這就是最好的結局。

**旁白**：黑色的氣流已經席捲而來。凌墨的黑劍斬落,巨大的衝擊波將李杰震飛了出去。

**李杰**：(在空中大喊)清水——!

**旁白**：他最後看到的畫面,是恵美依偎在高山翔懷中,兩人在黑色的火焰中漸漸消散。

**旁白**：還有凱蘭,他那孤單而決絕的背影,正衝向無敵的凌墨。

**旁白**：李杰撞進了右側的一道光門中。

**旁白**：耀眼的白光瞬間淹沒了他的意識。

---

**旁白**：李杰(警戒地環顧四周)這裡...

**旁白**：候車大廳寬敞得不像話。數根巨大的石柱從地面延伸到看不見的穹頂,每根石柱上都雕刻著奇異的符文。

**旁白**：地板由光滑的黑石鋪成,能清晰地映出他的倒影。空氣冰冷而稀薄,每一次呼吸都像是在吸入冰塊般刺痛。

**旁白**：李杰停下腳步,注意到大廳深處有兩個方向。

**旁白**：左側和右側各有一條幽暗的長廊,像兩道深不見底的傷口,從建築的體內裂開。

**李杰**：(內心獨白)終焉祭壇的門...應該在哪邊...

**旁白**：就在這時,李杰注意到地面上有什麼東西。

**李杰**：(蹲下查看)這是...血跡?

**旁白**：黑色的石地板上,有一道淡淡的血跡。不是很明顯,但在光滑的地面上依然能看出痕跡。

**旁白**：血跡從大廳入口延伸向左側的長廊。

**李杰**：(站起身,警戒地)是清水或斎藤 की血嗎...

**旁白**：李杰做出決定,朝著左側的長廊走去。

**旁白**：長廊很長,兩側的牆壁刻滿了複雜的符文。腳步聲在狹窄的空間中迴盪,形成詭異的回音。

**旁白**：長廊內沒有任何光源,黑暗如實質般濃稠。李杰只能憑著微弱的光線,小心翼翼地往前走。

**旁白**：走了許久,前方終於出現了一個向下的樓梯。

**李杰**：(停在樓梯口)往下...

**旁白**：李杰開始下樓梯。一層、兩層、三層...樓梯似乎沒有盡頭,不斷向下延伸。

**旁白**：空氣越來越冷,壓力越來越大,像是在往地底深處下沉。

**旁白**：十層...十五層...二十層...李杰的雙腿開始發軟,頭痛也更加劇烈。但他咬牙堅持,不敢停下。

**旁白**：又下了幾層後,終於——樓梯到達了盡頭。

**旁白**：眼前是另一條走廊。

**旁白**：這條走廊比上面的更加寬闊,兩側的牆壁上鑲嵌著發光的寶石,散發著淡淡的藍色光芒。

**旁白**：李杰沿著走廊繼續前進。走廊盡頭,矗立著一扇巨大的古老石門。

**旁白**：石門高達數米,表面刻滿了精緻的浮雕和符文。門縫中透出微弱的金色光芒。

**李杰**：(停在門前,深吸一口氣)這就是...

**旁白**：李杰用力推開石門。門緩緩打開,發出低沉的轟鳴聲。

**旁白**：門縫越來越大,金色的光芒湧了出來,照亮了李杰的臉。

**旁白**：而當門完全打開的那一瞬間——

**旁白**：李杰整個人都僵住了。

**旁白**：他站在門口,一動也不動。眼睛睜得很大,臉上的表情從疲憊變成了震驚,然後是難以置信。

**旁白**：他的嘴巴微微張開,想說些什麼,但卻發不出任何聲音。

**旁白**：因為門後的景象...是他怎麼也想像不到的。
