# 第三十七幕：破碎的救贖

**旁白**：恵美看到祭壇上的景象，整個人僵住了。

**旁白**：她的臉色瞬間變得慘白，雙眼睜大，嘴唇顫抖著。

**清水恵美**：（顫抖地）不...不可能...

**旁白**：下一秒，恵美突然衝向祭壇，動作急促而慌亂。

**清水恵美**：（尖叫）翔！翔！

**旁白**：她的雙手抓住祭壇的邊緣，整個人趴在祭壇上，看著那個年輕男子的臉。

**清水恵美**：（聲音撕裂）為什麼...為什麼會這樣！

**旁白**：恵美的聲音在空曠的聖殿中迴盪，充滿了絕望和難以置信。

**旁白**：李杰和斎藤站在不遠處，被這突如其來的變故嚇得手足無措。

**李杰**：（震驚）那就是...那個留在這裡的朋友？

**斎藤蓮**：（臉色發青，往後退了兩步）他...他死了...被殺了...

**旁白**：凱蘭也快步走上前，神情嚴肅地檢查著男子的屍體。

**凱蘭**：（低聲）傷口很深...一刀致命...

**凱蘭**：（轉頭看向四周，眼神冰冷）是誰...是誰在這裡動的手...

**旁白**：恵美依然趴在屍體上，瘋狂地搖晃著他的肩膀，試圖將他喚醒。

**清水恵美**：（哭喊）翔！你醒醒啊！你說過要等我回來的！

**清水恵美**：（眼淚奪眶而出）我們說好要一起去看海的...你不能就這樣丟下我...

**旁白**：然而，回應她的只有死一般的寂靜。男子的身體冷冰冰的，沒有任何生命氣息。

**旁白**：李杰看著這一幕，心口一陣隱隱作痛。他知道，恵美所有的堅持、所有的勇氣、所有的希望...都在這一刻粉碎了。

**旁白**：她不惜再次冒險來到這個世界，就是為了救回這個人。可結果...

**李杰**：（走上前，輕輕拍了拍恵美的背）清水...冷靜點...

**清水恵美**：（猛地轉過頭，眼神中充滿了瘋狂）不！他不該死的！一定還有辦法救他的！

**清水恵美**：（抓住李杰的袖子，語氣急促）李杰先生！你是工程師對不對？你一定懂什麼高科技！快救救他！

**李杰**：（沈重地搖搖頭）清水...他已經沒氣了...

**旁白**：這句話像最後一根稻草，徹底壓垮了恵美。她鬆開了李杰的袖子，整個人癱坐在祭壇下，眼神空洞地望著前方。

**旁白**：就在這時，原本安靜的聖殿深處，傳來一陣輕微的響聲。

**旁白**：那是腳步聲。緩慢而有力，一聲、一聲地敲擊在地板上。

**凱蘭**：（立刻拔出長劍，擋在眾人身前）是誰！出來！

**旁白**：一個身影緩緩從石柱後的陰影中走出。

**旁白**：那是一個年輕男子，穿著一身黑色的風衣。他的臉部輪廓冷峻，眼神深邃得彷彿能吸走所有的光芒。

**旁白**：他的右手握著一把細長的黑劍，劍身上還殘留著幾滴未乾的血跡。

**凱蘭**：（瞳孔猛地收縮）凌墨...

**旁白**：被稱為凌墨的男子沒有理會凱蘭，他的目光穿過眾人，直接落在了癱坐在地上的恵美身上。

**凌墨**：（聲音冰冷，不帶一絲情感）妳不該回來的。

**清水恵美**：（抬起頭，看著凌墨，眼神中充滿了仇恨）是你...是你殺了他？

**凌墨**：（語氣平淡，彷彿在說一件微不足道的事）他不符合這個世界的規則。留著他，只會產生不必要的麻煩。

**清水恵美**：（猛地站起身，尖叫著衝向凌墨）你這混蛋！我要殺了你！

**凱蘭**：（一把拉住恵美）清水小姐！冷靜！妳不是他的對手！

**旁白**：凱蘭將恵美護在身後，手中的長劍對準了凌墨。

**凱蘭**：（嚴肅地）凌墨，妳身為守門人，竟然隨意獵殺人類...妳知道這是違反聖約的嗎？

**凌墨**：（冷笑一聲）聖約？在這種地方，我就是規則。

**旁白**：凌墨說完，身形突然一動，整個人化作一道黑影，瞬間出現在凱蘭面前。

**旁白**：兩把劍再次交鋒，爆發出劇烈的金屬撞擊聲。強大的衝擊波向四周擴散，吹亂了李杰和恵美的頭髮。

**凱蘭**：（咬牙支撐著，對李杰大喊）帶他們走！去那扇門！快！

**李杰**：（猶豫了一下）但是妳...

**凱蘭**：（決絕地）我是精靈的戰士！我的命就是為了守護這扇門！走！

**旁白**：李杰看著凱蘭堅定的眼神，知道自己留在這裡只會拖累她。他一咬牙，一把拉起還在掙扎的恵美，另一隻手抓住已經嚇傻的斎藤。

**李杰**：走！

**旁白**：三人朝著祭壇後方那扇散發著金色光芒的門衝去。

**旁白**：身後傳來凱蘭和凌墨激烈的戰鬥聲。每一次交鋒都像雷鳴般震撼。

**旁白**：就在李杰快要踏入光門時，他忍不住回頭看了一眼。

**旁白**：他看見凱蘭的身體被凌墨一劍貫穿。

**李杰**：（驚呼）凱蘭先生！

**旁白**：凱蘭吐出一口鮮血，但她卻死死抓住了凌墨的劍刃，臉上露出了一個悽慘卻又欣慰的笑容。

**凱蘭**：（用盡最後的力氣大喊）跑！永遠不要回頭！

**旁白**：金色的大門在李杰眼前緩緩關上。最後一刻，他看到了凌墨那冰冷到極致的眼神。

**旁白**：光門徹底關閉。三人墜入了一片耀眼的白光之中。

**斎藤蓮**：（結巴）你...你說什麼？

**斎藤蓮**：（往後退了一步）躺...躺到祭壇上？

**斎藤蓮**：（聲音顫抖）那...那這個人是怎麼死的？

**李杰**：（避開視線）我不知道。

**斎藤蓮**：（激動地）你不知道？你他媽的騙誰！

**斎藤蓮**：（指著李杰）你一定知道什麼！說清楚！

**旁白**：李杰沒有回答。他只是沉默地站在那裡。

**旁白**：斎藤盯著李杰看了一會兒，然後緩緩走向祭壇。

**旁白**：他伸出顫抖的手，抓住翔的肩膀。

**旁白**：屍體很冷，觸感僵硬。斎藤打了個寒顫，但還是咬著牙，用力將屍體拖下祭壇。

**旁白**：翔的身體沉重地落在地面上，發出沉悶的聲響。

**旁白**：恵美渾身無力地靠在李杰懷中，眼神空洞地看著那具被拖動的屍體。

**旁白**：她的嘴唇微微顫抖，但已經發不出任何聲音。

**旁白**：斎藤沒有理會恵美的狀態。他已經將屍體移到一邊。

**旁白**：斎藤看著空蕩蕩的祭壇，又看向李杰。

**斎藤蓮**：（冷笑）躺上去是吧？

**斎藤蓮**：（質疑地）那...為什麼不是你躺上去？

**斎藤蓮**：（看向恵美）或者她？為什麼一定要是我？

**李杰**：（張嘴想說話）這是因為...

**旁白**：但就在這時——

**旁白**：一陣清晰的腳步聲從聖殿入口傳來。

**旁白**：腳步聲很緩慢，很有節奏，在空曠的空間中迴盪。

**旁白**：所有人同時轉頭，看向聲音的來源。

**旁白**：一個身影從陰影中走出。

**旁白**：那是一個年輕男子，身材修長，穿著黑色的夾克和黑褲。

**旁白**：他的右手握著一把黑色的劍，劍身上殘留著乾涸的血跡。

**旁白**：他的臉部線條冷峻，眼神深邃而漠然。

**旁白**：他就這樣慢慢地、一步一步地，朝著他們走來。

**旁白**：每一步都踩在堅硬的石地板上，發出清脆的回響。

**凱蘭**：（警戒地握住劍柄）是誰！

**旁白**：但那個男子沒有回答。

**旁白**：他只是繼續走著，黑色的劍在微弱的光線下閃爍著冰冷的寒光。

**旁白**：隨著距離的拉近，李杰終於看清了那張臉。

**旁白**：那張熟悉的...卻又陌生的臉。
