# 第三十七幕：最終的寂靜

**旁白**：李杰站在門口,整個人像被雷擊中一般,無法移動。

**旁白**：門邊,倒著一具屍體。

**旁白**：那是凱蘭。精靈劍士凱蘭的屍體。

**旁白**：他的身體被從腰部整齊地切成兩半,上半身和下半身分離,倒在血泊中。

**旁白**：他的臉上還保持著驚愕的表情,眼睛睜得很大,似乎到死都不敢相信發生了什麼。

**旁白**：他的劍掉落在一旁,刀刃上沾滿了血跡。

**李杰**：(顫抖地)凱蘭...

**旁白**：李杰的視線越過凱蘭的屍體,看向聖地的深處。

**旁白**：在那裡,他看到了恵美。

**旁白**：恵美趴在祭壇下,一動不動。

**旁白**：她的後背被鮮血染紅,白色的長裙已經變成了暗紫色。她的手還徒勞地伸向祭壇的方向,彷彿想要抓住最後的一絲希望。

**李杰**：(心臟猛地停跳)清水!

**旁白**：李杰踉踉蹌蹌地衝向祭壇,跪在恵美身邊。他伸出顫抖的手,想要翻開她的身體。

**旁白**：但他的手還沒碰到恵美,就停住了。

**旁白**：因為恵美的雙眼睜得很大,瞳孔已經渙散,失去了所有的光彩。

**旁白**：她死了。

**旁白**：在經歷了兩次穿越、無數次掙扎、和長達兩年的等待後,她最終死在了離希望只有幾步之遙的地方。

**李杰**：(發出絕望的哀鳴)不——!

**旁白**：李杰抱起恵美冰冷的屍體,眼淚止不住地往下掉。

**旁白**：為什麼...為什麼會這樣...我們明明已經到了...明明就要成功了...

**旁白**：就在這時,李杰注意到另一扇門前還倒著一個人。

**李杰**：(震驚地)斎藤...

**旁白**：那是斎藤蓮。那個膽小、愛逞強的眼鏡男。

**旁白**：他就這樣趴在門框前,一動不動。

**旁白**：李杰顫抖著蹲下來,想要翻過斎藤的身體查看情況。

**旁白**：但他的手剛碰到斎藤的肩膀,就停住了。

**旁白**：因為斎藤的身體已經冰涼僵硬。

**李杰**：(痛苦地閉上眼睛)連斎藤也...

**旁白**：李杰緩緩站起身來,看著眼前空蕩蕩的門框。

**李杰**：(喃喃自語)這就是...回家的門嗎...

**旁白**：李杰深吸一口氣,伸出手,朝著門框走去。

**旁白**：如果這真的是通往原本世界的門,那麼只要穿過去...

**旁白**：李杰踏入門框。

**旁白**：但...什麼都沒有發生。

**旁白**：他就這樣走過了門框,來到了門框的另一側。

**旁白**：沒有光芒,沒有傳送,沒有任何異常。就像穿過一個普通的空門框一樣。

**李杰**：(轉身,困惑地看著門框)什麼都...沒有?

**旁白**：李杰又走回門框,再次穿過。

**旁白**：依然什麼都沒發生。

**李杰**：(沮喪地)為什麼...為什麼門打不開...

**旁白**：李杰站在原地,腦中一片混亂。

**旁白**：凱蘭死了。恵美死了。斎藤也死了。祭壇上還有一個陌生男子的屍體。

**旁白**：而通往原本世界的門...根本打不開。

**李杰**：(內心獨白)到底...發生了什麼...是誰殺了他們...

**李杰**：(內心獨白)而且...如果門打不開...我要怎麼回去...

**旁白**：就在這時,李杰突然想起了什麼。

**李杰**：(回想起來)對了...剛才在入口大廳...右邊還有一條走廊...

**旁白**：李杰快步走回聖地的入口。他小心翼翼地跨過凱蘭的屍體,走出石門。

**旁白**：回到候車大廳後,他看向右側。

**旁白**：那裡確實有另一條走廊,同樣幽暗深邃。

**李杰**：(握緊拳頭)也許...答案在那裡...

**旁白**：李杰深吸一口氣,朝著右側的走廊走去。

**旁白**：他必須弄清楚真相。

**旁白**：他必須知道...這一切到底是怎麼回事。
