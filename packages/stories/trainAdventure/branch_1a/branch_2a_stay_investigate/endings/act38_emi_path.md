# 第三十八幕：絕境逃生

**旁白**：李杰的身體猛地一震。

**旁白**：那張臉——他見過。

**李杰**：（內心獨白）是他...新宿車站...那個靠在柱子旁的黑衣男...

**旁白**：李杰清楚地記得那個夜晚。

**旁白**：在新宿車站月台上，那個穿著黑色連帽衛衣的男子，靠在角落的柱子旁。

**旁白**：帽簷壓得很低，雙手插在口袋裡，一動也不動。

**旁白**：當時李杰就覺得那個人很違和，像是在觀察什麼。

**旁白**：但來到這個世界後，那個男子就再也沒有出現過。

**旁白**：沒想到，竟然會在這裡，以這種方式再次見面。

**李杰**：（顫抖地開口）你...你是新宿月台上的那個...

**旁白**：黑衣男子終於停下腳步。他緩緩抬起頭，帽簷下的那雙眼睛冰冷得沒有一絲溫度。

**凌墨**：（聲音冷冽，不帶任何情感）妳不需要知道我是誰。妳只需要知道，妳們今天走不掉。

**凱蘭**：（握緊劍柄，擋在李杰身前）凌墨！你這是在公然挑釁精靈族的尊嚴！

**凌墨**：（語氣平淡）尊嚴？在絕對的力量面前，那種東西一文不值。

**旁白**：話音剛落，凌墨突然動了！

**旁白**：他的速度快得驚人，幾乎在眨眼間就衝到了凱蘭面前。

**旁白**：黑色的劍化作一道黑光，猛地向凱蘭劈下。

**凱蘭**：（大喝一聲，舉劍迎擊）鐺——！

**旁白**：劇烈的金屬碰撞聲在聖殿中震盪。凱蘭的身體微微一晃，顯然在力量上落了下風。

**凌墨**：（眼神漠然）太弱了。

**旁白**：他手腕一轉，黑劍突然以一個詭異的角度繞過凱蘭的防禦，刺向她的胸口。

**凱蘭**：（勉強側身避開，但手臂還是被劃開了一道口子）唔...！

**李杰**：（驚呼）凱蘭先生！

**旁白**：凱蘭沒有回頭，她忍著痛，反手一劍橫斬，逼退了凌墨。

**凱蘭**：（急促地對李杰喊道）別管我！快帶他們走！那扇門！

**旁白**：她指向祭壇後方那扇散發著金色光芒的拱門。

**李杰**：（一咬牙，拉起癱坐在地上的恵美）走！斎藤！快！

**旁白**：斎藤蓮此刻已經嚇得臉色慘白，連滾帶爬地跟在李杰身後。

**旁白**：三人繞過祭壇，朝著光門狂奔而去。

**旁白**：身後傳來密集的劍刃交鋒聲和凱蘭的低喝聲。李杰不敢回頭，他知道那是凱蘭在用生命為他們爭取時間。

**凌墨**：（冰冷的聲音在身後響起）妳以為，妳攔得住我嗎？

**旁白**：李杰心頭猛地一緊。他轉過頭，看到了令他心碎的一幕。

**旁白**：凱蘭的長劍被凌墨一腳踢飛，掉落在遠處的地板上。

**旁白**：凌墨手中的黑劍，正緩緩刺入凱蘭的肩膀，將她釘在了石柱上。

**凱蘭**：（吐出一口鮮血，眼神依然堅定）走...快走...！

**李杰**：（眼眶通紅，嘶吼著）凱蘭先生！

**旁白**：但他沒有停下。他抱緊恵美，衝向那扇光門。

**旁白**：距離光門只有五米了...四米...三米...

**旁白**：突然，一股強大的寒意從背後襲來！

**凌墨**：（不知何時已經擺脫了凱蘭，出現在他們身後不遠處）我說過，妳們走不掉。

**旁白**：他揮動黑劍，一道黑色的氣流化作巨浪，朝著三人的後背席捲而來。

**李杰**：（內心獨白）該死！躲不掉了！

**旁白**：就在這千鈞一髮之際，原本空蕩蕩的祭壇突然爆發出一陣耀眼的白光！

**旁白**：那光芒如此強烈，甚至壓過了那道黑色的氣流。

**凌墨**：（微微皺眉，停下腳步）嗯？

**旁白**：光芒中，原本躺在祭壇下的翔的身體，竟然緩緩漂浮了起來。

**旁白**：他的胸口爆發出神聖的白色光芒，擋住了凌墨的攻擊。

**清水恵美**：（淚流滿面，失聲叫道）翔！

**旁白**：翔的眼睛緩緩睜開。那不再是死人的空洞，而是充滿了祥和的光芒。

**翔**：（聲音在聖殿中迴盪，空靈而溫暖）恵美...走吧...回到妳該去的地方...

**旁白**：他揮動雙手，兩股柔和的力量推動著李杰、恵美和斎藤，將他們送入了光門。

**凌墨**：（眼神一冷）多管閒事。

**旁白**：他一劍刺向空中的翔。

**旁白**：但就在劍尖快要觸及翔的瞬間，翔的身體化作無數光點消散在空氣中。

**旁白**：李杰三人在這股力量的推動下，衝進了光門。

**旁白**：金色的大門在他們身後重重關閉。

**旁白**：最後的一刻，李杰看到了凌墨那冰冷、漠然、卻帶著一絲複雜神色的雙眼。

**旁白**：然後，耀眼的白光吞噬了一切。

**旁白**：凱蘭衝向凌墨，長劍揮舞出無數道銀色的光弧。

**旁白**：每一劍都帶著精靈戰士的榮耀和決死。

**旁白**：但凌墨只是微微側身。

**旁白**：凱蘭的劍從他身旁掠過，連衣角都沒有碰到。

**旁白**：下一秒——

**旁白**：凌墨的黑劍動了。

**旁白**：那是一個簡單而優雅的橫斬。

**旁白**：沒有多餘的動作，沒有花哨的技巧。

**旁白**：黑劍在空中劃過，留下一道漆黑的軌跡。

**旁白**：然後——

**旁白**：凱蘭的身體從腰部斷成兩截。

**旁白**：上半身和下半身緩緩分離，鮮血噴濺而出。

**凱蘭**：（眼睛睜大，發不出聲音）......

**旁白**：凱蘭的上半身跌落在地，眼中還殘留著震驚和難以置信。

**旁白**：整個過程，不到一秒。

**李杰**：（震驚地瞪大眼睛）凱蘭...

**旁白**：李杰的大腦一片空白。

**旁白**：凱蘭——那個優秀的精靈劍士，那個一路保護他們的戰士——

**旁白**：就這樣，在他眼前被斬成兩半。

**旁白**：凌墨踏過凱蘭的屍體，繼續朝李杰走來。

**旁白**：他的表情依然冷漠，彷彿剛才只是做了一件微不足道的事。

**旁白**：李杰的身體本能地顫抖。

**旁白**：但恵美——恵美還癱坐在地上，完全沒有意識到危險。

**李杰**：（內心獨白）不能猶豫...不能停下...

**旁白**：李杰衝向恵美，彎下腰，雙手從她的肩膀和膝蓋下穿過，將她整個人抱起。

**清水恵美**：（茫然地）李杰先生...

**李杰**：（急促地）抱歉了！

**旁白**：李杰轉身，朝著那扇發光的門狂奔。

**旁白**：他的肺部像要炸開，心臟瘋狂跳動。

**旁白**：身後，凌墨的腳步聲越來越近。

**旁白**：但李杰不敢回頭。

**旁白**：他只是抱緊恵美，用盡全力往前跑。

**旁白**：五米...三米...一米...

**旁白**：李杰縱身一躍，撲進那道光門。

**旁白**：一瞬間，耀眼的白光吞沒了他的視野。

**旁白**：一切都消失在那片光芒之中。
