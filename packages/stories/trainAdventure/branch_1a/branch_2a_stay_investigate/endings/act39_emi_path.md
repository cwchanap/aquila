# 第三十九幕：淚光中的誓言

**旁白**：不知道過了多久。

**旁白**：李杰感覺到意識逐漸回歸。

**旁白**：那片刺眼的白光慢慢消退，視野開始變得清晰。

**旁白**：他發現自己躺在冰冷的地面上。

**旁白**：身體很沉重，四肢無力，但至少還能動。

**李杰**：（低聲呻吟）嗯...

**旁白**：李杰撐著地面，緩緩坐起身。

**旁白**：他環顧四周——這是一個狹小的房間。

**旁白**：牆壁由灰白色的金屬板組成，天花板上掛著一盞發著白光的長形燈。

**旁白**：房間裡沒有窗戶，只有一扇沉重的金屬門。

**旁白**：空氣中瀰漫著一股淡淡的、像是消毒水的味道。

**李杰**：（內心獨白）這裡...是哪裡？我們回來了嗎？

**旁白**：他急忙看向身旁。

**旁白**：恵美也躺在不遠處。她還閉著眼睛，臉色蒼白，但呼吸很平穩。

**李杰**：（爬過去，輕輕推了推她）清水！清水！醒醒！

**旁白**：恵美發出一聲輕吟，緩緩睜開雙眼。

**清水恵美**：（茫然地看著天花板）李杰先生...這裡是...

**李杰**：（搖搖頭）我也不知道。但感覺不像是精靈的世界。

**旁白**：就在這時，房間的金屬門突然發出「嘶——」的一聲，緩緩向兩側滑開。

**旁白**：三個穿著整齊軍裝的男女走了進來。

**旁白**：他們胸口別著某種徽章，眼神犀利。領頭的是一個年輕女性，英姿颯爽。

**年輕女衛兵**：（看著兩人，表情嚴肅）妳們終於醒了。

**李杰**：（警惕地站起身，護在恵美身前）妳們是誰？這裡是哪裡？

**年輕女衛兵**：（沒有回答，只是打量著李杰）李杰，28歲，軟體工程師。清水恵美，21歲，大學生。沒錯吧？

**李杰**：（震驚）妳怎麼知道我們的名字？

**年輕女衛兵**：（微微一笑）我們一直在等你們。

**旁白**：她走到李杰面前，伸出手。

**年輕女衛兵**：歡迎來到「守望者」基地。這裡是夏威夷。

**李杰**：（愣住）夏威夷？怎麼可能...我們明明是在東京上車的...

**年輕女衛兵**：（聳肩）時間和空間在那個世界是扭曲的。你們離開東京已經一年半了。

**清水恵美**：（驚呼）一年半？！

**旁白**：這句話像重錘一樣砸在兩人心中。一年半...對他們來說，那只是一瞬間的事，但現實世界竟然已經過去了這麼久。

**李杰**：（急切地問）那我爸媽呢？還有清水的家人呢？

**年輕女衛兵**：（眼神中閃過一絲遺憾）關於這個...等一下會有人跟你們詳細說明的。

**年輕女衛兵**：現在，請跟我們來。長官要見你們。

**旁白**：李杰看向恵美，兩人眼中都充滿了不安。

**旁白**：但他們現在毫無反抗之力，只能跟著這些人走出房間。

**旁白**：走廊很長，燈光明亮得有些刺眼。沿途，李杰看到許多穿著白袍的科研人員在忙碌著。

**旁白**：這裡看起來像是一個極其先進的秘密研究機構。

**旁白**：最終，他們被帶到了一個寬敞的辦公室。

**旁白**：辦公室的落地窗外，是一望無際的湛藍大海和金色的沙灘。

**旁白**：一個中年男子坐在巨大的辦公桌後，正在批閱檔案。

**年輕女衛兵**：（敬禮）長官，人帶到了。

**中年男子**：（抬起頭，露出一張和藹的臉）好，妳下去吧。

**旁白**：女衛兵轉身離開，關上了門。

**中年男子**：（示意兩人坐下）坐吧。我是這裡的負責人，妳們可以叫我大輔。

**李杰**：（坐下，聲音顫抖）大輔先生...請告訴我們實話。我們的家人到底怎麼了？

**大輔**：（沈默了一會兒，長長地嘆了一口氣）李杰先生，很遺憾。一年半前的那次列車失蹤事件，被判定為重大超自然災害。

**大輔**：當時在月台上的所有人，包括你的父母，都沒能倖免。

**李杰**：（如遭雷擊，整個人癱在椅子上）沒能...倖免？什麼意思？

**大輔**：（語氣低沈）根據我們搜救隊的現場報告...雖然月台消失了，但在原本月台的位置，發現了...一些遺物和...不完整的屍體。

**大輔**：你的父母...就在其中。

**旁白**：眼淚奪眶而出。李杰摀住臉，肩膀劇烈地顫抖著。雖然心裡早有準備，但親耳聽到這個結果，他依然無法接受。

**大輔**：（看向恵美）至於清水小姐...情況比較複雜。

**清水恵美**：（臉色慘白，抓緊裙角）我家人呢？

**大輔**：妳的父母也在那次事件中失蹤了。但我們沒有找到他們的屍體，所以暫時列為失蹤。

**大輔**：不過...妳的弟弟還活著。他這一年半來一直住在我們這裡，協助我們研究。

**清水恵美**：（驚訝地抬起頭）浩介？他在這裡？

**大輔**：是的。等一下妳就能見到他了。

**旁白**：恵美眼中閃過一絲希望，但隨即又被巨大的悲傷淹沒。父母失蹤...生死未卜...

**大輔**：（看著兩人悲傷的樣子，語氣變得溫柔了一些）我知道這對你們來說很殘忍。但能在那個世界存活一年半並平安歸來，你們已經是奇蹟了。

**大輔**：而且...

**大輔**：（眼神變得深邃）你們的歸來，可能掌握著拯救那個世界的關鍵。

**李杰**：（擦乾眼淚，語氣生硬）拯救世界？我現在只想知道，為什麼會發生這種事？那個黑衣男又是誰？

**大輔**：（沈聲道）那正是我們想要查明的。我們稱那個世界為「鏡像界」。而那個黑衣男...我們懷疑他是鏡像界的執行者，負責清理那些不符合規則的人類。

**李杰**：（咬牙）不符合規則...神樹也說過這句話。

**大輔**：沒錯。而你們，是極少數能對抗那種規則的人。

**旁白**：李杰看著窗外美麗的海景，心中卻是一片荒蕪。

**旁白**：他回來了。回到了這個原本的世界。

**旁白**：但對他來說，家已經不在了。

**清水恵美**：（突然靠過來，輕輕拉住李杰的手）李杰先生...

**李杰**：（轉頭看她）清水...

**清水恵美**：（眼中閃爍著堅定的光芒）我們...雖然失去了很多。但我們還活著。

**清水恵美**：（輕聲說）既然回來了，我們就要替那些沒能回來的人活下去。

**旁白**：李杰看著恵美淚流滿面卻又無比堅強的臉。他想起在銀月湖畔的諾言，想起凱蘭最後的吶喊，想起翔消散前的微笑。

**李杰**：（點點頭，反手握住她的手）妳說得對。我們不能就這樣被打倒。

**李杰**：（看向大輔，語氣變得堅定）大輔先生，我們願意配合妳們。只要能弄清楚真相，只要能讓悲劇不再發生。

**大輔**：（欣慰地笑了）很好。我果然沒看錯你們。

**大輔**：（站起身）走吧。我帶你們去見那些還活著的人。

**旁白**：三人走出辦公室，朝著基地深處走去。

**旁白**：陽光燦爛地灑在走廊上，前方的路依然充滿未知。

**旁白**：但這一次，李杰不再是孤身一人。他身邊有恵美，有這些願意提供幫助的人。

**旁白**：雖然家園破碎，但新的希望，正從廢墟中緩緩升起。

**清水恵美**：（哽咽）但我真的...真的很害怕...

**清水恵美**：（小聲）我不想一個人待在那個世界...

**旁白**：李杰看著恵美淚流滿面的臉，心中百感交集。

**旁白**：有失望、有理解、有心疼...全都交織在一起。

**旁白**：但最終，他嘆了口氣。

**李杰**：（溫和地）我明白...

**李杰**：（看著恵美）在那種情況下...任誰都會害怕...

**李杰**：（輕聲）妳只是想活下去...想回家...這沒有錯。

**清水恵美**：（眼淚湧出）李杰先生...

**李杰**：（微笑）而且...就算妳告訴我真相...我也一樣會保護妳。

**清水恵美**：（愣住）咦...

**李杰**：（認真地）因為...這是我自己的選擇。

**李杰**：不管妳是誰，不管妳有什麼秘密...

**李杰**：（看著她的眼睛）我都會保護妳。

**旁白**：恵美的眼淚再次滑落，但這次，她的嘴角帶著一絲笑容。

**清水恵美**：（哭著笑）謝謝...謝謝你...

**旁白**：但笑容很快消失，取而代之的是深深的哀傷。

**清水恵美**：（低聲）但是...我已經失去翔了...

**清水恵美**：（聲音破碎）我再也...見不到他了...

**旁白**：李杰看著恵美悲傷的臉，心中湧起一股衝動。

**旁白**：他伸出手，輕輕捧起恵美的臉。

**李杰**：（認真地）清水...

**清水恵美**：（疑惑地看著他）李杰先生...

**李杰**：（深吸一口氣）妳失去了他...我知道這對妳來說很痛苦...

**李杰**：（看著她的眼睛）但是...

**李杰**：（輕聲）以後...讓我一直陪在妳身邊...好嗎？

**旁白**：恵美的眼睛睜大，臉上閃過震驚。

**清水恵美**：（顫抖）李杰先生...你...

**旁白**：李杰沒有給她說話的機會。

**旁白**：他緩緩靠近，在恵美的唇上落下一個溫柔的吻。

**旁白**：恵美的身體僵住了，眼睛瞪得很大。

**旁白**：但很快，她閉上了眼睛，眼淚從眼角滑落。

**旁白**：她沒有推開李杰，而是顫抖著，回應了這個吻。

**旁白**：在這個狹小而空蕩的房間裡，兩個受傷的靈魂緊緊依偎。
