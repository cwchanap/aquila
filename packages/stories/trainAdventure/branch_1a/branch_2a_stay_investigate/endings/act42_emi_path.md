# 第四十二幕：失蹤者的羈絆

**旁白**：李杰和恵美的身體都繃緊了。

**旁白**：他們對視一眼，眼中都流露出警戒。

**旁白**：這個神秘組織...掌握著他們的資料...掌握著斎藤的照片...

**旁白**：這些人...到底想做什麼？

**肥胖中年男**：（揮手笑著）哎呀哎呀，別那麼緊張嘛！

**肥胖中年男**：我們不是壞人。你們放輕鬆一點。

**旁白**：但李杰和恵美依然保持著戒備。

**肥胖中年男**：（嘆了口氣）好吧，我理解你們的擔心。

**肥胖中年男**：（看向窗外）其實...我們和你們一樣，也是那場列車失蹤事件的受害者。

**李杰**：（愣住）受害者？

**肥胖中年男**：（轉過身，神情變得沈重）25年前...也就是2026年...那班從新宿出發的深夜列車，失蹤了。

**肥胖中年男**：（指著自己）我當時也在月台旁，眼睜睜看著那班車消失。

**肥胖中年男**：而我最重要的親人...也在那班車上。

**旁白**：李杰和恵美被這番話震撼了。

**李杰**：（急切地問）是誰？妳的親人是誰？

**肥胖中年男**：（沈默了一會兒，然後從平板電腦中調出一張照片）是他。

**旁白**：那是照片上是個年輕人，看起來二十多歲，和斎藤蓮有幾分神似。

**肥胖中年男**：他叫斎藤蓮。是我的弟弟。

**旁白**：李杰和恵美同時瞪大了眼睛，難以置信地看著照片。

**李杰**：（失聲叫道）斎藤蓮？！

**清水恵美**：（驚呼）他是妳弟弟？！

**肥胖中年男**：（沈重地點點頭）是的。他當時才21歲，剛考上大學。

**肥胖中年男**：沒想到那一晚，竟成了永別。

**旁白**：李杰感到腦中一陣混亂。斎藤蓮...是這個肥胖中年男人的弟弟？

**李杰**：（追問道）那妳是...？

**肥胖中年男**：（苦笑了一下）我叫斎藤大輔。現在是這個研究機構的負責人。

**斎藤大輔**：這25年來，我傾注了所有的財力和精力，就是為了找回我的弟弟。

**斎藤大輔**：我成立了這個組織，招募了世界頂尖的科學家，建立了這個基地。

**斎藤大輔**：（看著兩人）就是為了守著那扇「門」。

**旁白**：李杰看著斎藤大輔，原本的警戒消失了，取而代之的是深深的同情。

**旁白**：為了找回失蹤的親人，竟然堅持了25年...這份執著，讓他動容。

**斎藤大輔**：（重新坐回辦公桌後，語氣變得有些急切）所以...請告訴我。你們...在門那邊見過他嗎？

**斎藤大輔**：（眼神中閃爍著希望的光芒）他...他還活著嗎？

**旁白**：李杰的喉嚨像是被什麼東西堵住了。

**旁白**：他該怎麼說？他該怎麼告訴大輔...他弟弟最後的情況？

**旁白**：他想起在終焉祭壇，為了開啟光門，他不得不做出的那個殘酷選擇...

**旁白**：他想起斎藤蓮消失在鏡子裏時，那絕望而恐懼的眼神...

**李杰**：（避開視線，聲音沙啞）大輔先生...我們...

**清水恵美**：（突然開口，眼眶泛紅）我們見過他。

**斎藤大輔**：（猛地站起身，手掌重重拍在桌上）真的？！他在哪？他在那邊過得怎麼樣？

**清水恵美**：（低下頭，眼淚滑落）他...他一直很想念家人...他一直想回家...

**旁白**：斎藤大輔聽了這話，眼眶瞬間濕潤了。

**斎藤大輔**：（顫抖地說）我就知道...我就知道他還活著...我就知道他在等我...

**旁白**：看著大輔激動的樣子，李杰感到前所未有的心虛和內疚。

**李杰**：（內心獨白）對不起...大輔先生...我...我親手...

**旁白**：就在這時，辦公室的門再次打開，那個叫莉子的女衛兵走了進來。

**莉子**：（恭敬地說）長官，伊藤小姐已經到了。

**斎藤大輔**：（收起情緒，擦了擦眼角）好，請她進來吧。

**旁白**：一個年輕女孩走了進來。她看起來大約二十歲左右，長得很漂亮，有一雙明亮的大眼睛。

**旁白**：最引人注目的是，她穿著一件白色的長裙，看起來優雅而安靜。

**斎藤大輔**：（對李杰和恵美介紹道）這位是伊藤莉子。她和你們一樣，也和失蹤事件有著深深的聯繫。

**伊藤莉子**：（對兩人微微一笑，聲音輕柔）你們好。

**李杰**：（禮貌地點頭）你好。

**斎藤大輔**：（指著莉子，語氣變得溫柔）莉子的父母...也在25年前的那班列車上。

**旁白**：李杰和恵美的身體再次僵住了。

**李杰**：（驚訝）她的父母...伊藤誠和伊藤奈々...

**斎藤大輔**：（沈重地）也在那班列車上。

**清水恵美**：（驚訝）也是失蹤乘客...

**斎藤大輔**：莉子是2025年2月出生的。

**斎藤大輔**：她是個早產兒，出生時身體很虛弱。

**斎藤大輔**：所以一直住在醫院的保溫箱裡接受治療。

**斎藤大輔**：（看著莉子）那年8月...也就是莉子出生六個月後...

**斎藤大輔**：她的父母上了那班列車。

**斎藤大輔**：當時莉子還在醫院...還沒能出院回家。

**伊藤莉子**：（平靜地）所以爸媽失蹤的時候，我還在保溫箱裡呢。

**伊藤莉子**：（微笑）連他們的臉都沒見過。

**斎藤大輔**：列車失蹤後，醫院聯繫不上莉子的父母。

**斎藤大輔**：過了一段時間，我們組織接手了這個案子。

**斎藤大輔**：（溫和地笑著）莉子就這樣被組織收養了。

**伊藤莉子**：在醫院住了幾個月，身體穩定後，就被組織帶走照顧了。

**伊藤莉子**：（微笑）不過沒關係。組織把我撫養長大，大輔叔叔也一直照顧我。

**斎藤大輔**：從那時候開始，我就被指派為莉子的監護人。

**斎藤大輔**：（溫暖地笑著）這25年來...我看著她長大。

**伊藤莉子**：（調皮地）大輔叔叔就像我爸爸一樣！

**伊藤莉子**：（認真地）雖然我沒見過親生爸媽...但我一直想知道他們的事...

**旁白**：大輔從抽屜裡拿出另一支平板電腦，滑動幾下，然後遞給李杰和恵美。

**斎藤大輔**：這是莉子父母的照片。你們...有見過他們嗎？

**旁白**：李杰看向螢幕——

**旁白**：照片上是一對夫妻。

**旁白**：男人看起來四十歲左右，表情有些煩躁，眉頭緊皺。

**旁白**：女人也是差不多年紀，臉上帶著焦慮的神情。

**李杰**：（驚訝）這...這是...

**清水恵美**：（顫抖）伊藤先生...伊藤太太...

**旁白**：李杰清楚地記得。

**旁白**：在列車上，有一對夫妻——煩躁易怒的丈夫和焦慮不安的妻子。

**旁白**：他們總是在爭吵，總是抱怨。

**旁白**：那就是...伊藤誠和伊藤奈々。

**伊藤莉子**：（急切地）你們認識他們嗎？

**伊藤莉子**：（眼中閃著希望）他們...他們還活著嗎？
