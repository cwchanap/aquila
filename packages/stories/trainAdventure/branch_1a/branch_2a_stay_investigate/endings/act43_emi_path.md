# 第四十三幕：歸途的代價

**旁白**：房間裡陷入短暫的沉默。

**旁白**：大輔收起平板電腦，表情變得認真。

**斎藤大輔**：那麼...關於蓮...

**斎藤大輔**：（看著李杰和恵美）你們知道他的情況嗎？

**旁白**：李杰的身體僵住了。

**旁白**：恵美也低下頭，緊緊咬著嘴唇。

**旁白**：該怎麼說...該怎麼告訴大輔...

**旁白**：他們為了逃命...把斎藤留在了那個世界...

**斎藤大輔**：（察覺到異樣，皺眉問道）怎麼了？是不是...出什麼事了？

**李杰**：（深吸一口氣，抬起頭，語氣艱難）大輔先生...對不起。

**李杰**：斎藤他...他沒能跟我們一起回來。

**旁白**：這句話像一聲驚雷，在安靜的辦公室裡炸開。

**斎藤大輔**：（手猛地抖了一下，聲音顫抖）什麼...沒能回來？這是什麼意思？

**李杰**：（痛苦地閉上眼睛）當時在祭壇那邊...情況非常危急。

**李杰**：守門官凌墨突然發起攻擊...凱蘭先生為了保護我們...犧牲了。

**清水恵美**：（抽泣著）對不起...大輔先生...都是為了救我...

**李杰**：（繼續說，聲音沙啞）當時光門已經快要關閉了。如果我們不馬上走，誰也回不來。

**李杰**：所以我...我只能選擇打開門...但斎藤他...他被留在了祭壇那邊...

**旁白**：李杰沒敢說出「祭品」兩個字。他不想讓大輔在痛苦之餘，還要承受那種殘酷的真相。

**旁白**：但他知道，在大輔看來，這依然是一個無法接受的結果。

**斎藤大輔**：（呆呆地坐在椅子上，臉色瞬間變得慘白）留在了那邊...？

**斎藤大輔**：（喃喃自語）這不可能...我等了25年...我做了這麼多準備...

**斎藤大輔**：（突然情緒失控，猛地拍桌子站起來）為什麼不救他？！妳們明明都回來了，為什麼偏偏留下他一人？！

**李杰**：（低下頭，無言以對）...

**清水恵美**：（大聲哭喊）對不起！大輔先生！真的對不起！

**斎藤大輔**：（劇烈喘息著，眼中充滿了絕望和憤怒）25年...我為了找他，放棄了原本的生活，放棄了一切...

**斎藤大輔**：（指向窗外，聲音破碎）妳們知道這25年我是怎麼過的嗎？！我每天都在看著那扇該死的門，每天都在期待他能走出來！

**斎藤大輔**：（癱坐在椅子上，捂著臉，泣不成聲）我只是想讓他回家...我只是想再見他一面...為什麼這麼難...

**旁白**：看著這個50多歲的男人在自己面前像個孩子一樣崩潰大哭，李杰感到前所未有的心碎。

**旁白**：25年的執著、25年的等待、25年的希望...在這一刻，全都化為了泡影。

**旁白**：他理解大輔的痛苦。因為他也失去了父母。

**旁白**：但他更感到內疚。因為斎藤蓮的消失，他負有不可推卸的責任。

**旁白**：辦公室裡陷入了死一般的沉寂。只有大輔壓抑的哭聲，在空曠的空間裡迴盪。

**旁白**：過了很久，大輔終於平靜了一些。他拿出手帕，擦乾眼淚，深深地看了李杰一眼。

**斎藤大輔**：（聲音沙啞而疲憊）妳們先下去吧。我需要一個人靜靜。

**李杰**：（站起身，深深鞠了一躬）大輔先生...真的很抱歉。

**清水恵美**：（也跟著鞠躬，泣不成聲）對不起...

**旁白**：兩人走出了辦公室。莉子守在門口，看到他們的神情，眼中閃過一絲複雜的情緒。

**莉子**：（低聲說）跟我來吧。我帶妳們去休息的地方。

**旁白**：李杰和恵美跟在莉子身後，默默地走在走廊上。

**旁白**：陽光依然燦爛，海浪聲依然清脆。

**旁白**：但這一切，對他們來說，都顯得那麼遙遠和陌生。

**旁白**：他們回來了。

**旁白**：但歸途的代價，卻是如此沉重。

**旁白**：沉重到，讓他們甚至不敢去直視未來。

---

**旁白**：李杰的身體搖晃了一下，感覺胸口被什麼東西狠狠撞擊。

**李杰**：（顫抖閉上眼睛深深吸氣）爸...媽媽她現在應該70歲了...

**旁白**：李杰的手緊緊握著椅子扶手。

**旁白**：父親已經不在了...母親獨自一人在上海...

**旁白**：這25年...她是怎麼熬過來的...

**清水恵美**：（小聲問）那...我的家人...

**旁白**：恵美的聲音很輕，充滿了不安。

**斎藤大輔**：清水小姐，妳的父母都已經過世了。

**斎藤大輔**：不過...妳的弟弟清水浩介還活著，現在住在船橋。

**清水恵美**：（眼淚湧出又抬起頭）浩介...

**斎藤大輔**：他23年前加入了我們組織。當時我在調查第一批失蹤乘客，找到了浩介先生。

**斎藤大輔**：他那時候已經自己調查了好幾年，但因為資源有限一直沒有成果。

**斎藤大輔**：後來他就加入我們，全心投入這個案子。現在他60歲了，這些年一直在組織裡工作。

**旁白**：李杰看著恵美。

**旁白**：他知道恵美經歷了兩次失蹤事件——45年前 and 25年前。

**旁白**：她的弟弟浩介...這25年來...不，應該說45年來...一直在尋找她...

**斎藤大輔**：（看著恵美）我已經通知他了，等一下可以視訊通話，妳可以跟他聊聊。

**清水恵美**：（哭著點頭）謝謝...謝謝您...

---

**斎藤大輔**：（打了個哈欠揉著眼睛）好了，今天的情報量有點大，我這把老骨頭有點撐不住了。

**斎藤大輔**：莉子，帶他們去飯店休息吧。明天早上我們再詳談。

**伊藤莉子**：（敬禮）是！

**斎藤大輔**：（對李杰和恵美）你們也累了吧，好好休息一晚，有什麼問題明天再說。

**李杰**：（站起來鞠躬）謝謝您...

**清水恵美**：（也站起來鞠躬）非常感謝...

**斎藤大輔**：（揮手溫和地笑著）別客氣。歡迎回到地球。

**旁白**：莉子走到門口，打開門。

**伊藤莉子**：（回頭）走吧，我帶你們去飯店。

**旁白**：李杰和恵美跟著莉子走出辦公室。

**旁白**：離開前，李杰回頭看了一眼。

**旁白**：大輔坐在那張巨大的椅子上，看起來有些疲憊。

**旁白**：但他的眼神依然溫柔。

**旁白**：門緩緩關上。

**旁白**：李杰和恵美踏入走廊，朝著未知的明天前進。
