# 第四十五幕：不合格者的秘密

**旁白**：清晨的陽光從窗簾縫隙中透進來。

**旁白**：李杰緩緩睜開眼睛。

**旁白**：恵美還靠在他懷裡，睡得很熟。

**旁白**：她的臉龐在晨光中顯得格外柔和。

**旁白**：李杰不敢動，怕吵醒她。

**旁白**：就在這時——

**旁白**：手機突然響起。

**旁白**：恵美驚醒，迷迷糊糊地睜開眼睛。

**清水恵美**：（揉著眼睛）唔...李杰先生...早上好了嗎？

**李杰**：（笑了笑）嗯，早上好。

**旁白**：李杰接起電話。是大輔打來的。

**大輔**：（語氣變得嚴肅）李杰先生，請馬上帶著清水小姐來我的辦公室。

**大輔**：組織的研究團隊，對你們帶回來的數據有了重大發現。

**大輔**：這關係到...你們之所以能回來的真相。

**旁白**：李杰心中一沉。真相？

**李杰**：（看著恵美）大輔先生叫我們過去。好像有重要發現。

**旁白**：兩位迅速洗漱完畢，跟著莉子再次來到了大輔的辦公室。

**旁白**：辦公室裏除了大輔，還有那個肥胖的中年男人凱，以及幾位穿著白大褂的科研人員。

**凱**：（興奮地跳了起來，指著螢幕上的複雜圖表）看！快看！

**凱**：這就是妳們的腦波掃描圖！還有妳們血液中的能量頻率！

**李杰**：（看著那些看不懂的線條）這能說明什麼？

**凱**：（急促地說）妳們知道，為什麼精靈神樹會判定妳不符合資格嗎？

**李杰**：（沈默了一會兒）不知道。它只說我不是純粹的人類。

**凱**：（拍了拍桌子，大聲喊道）因為妳的靈魂中，含有一種「干擾波」！

**凱**：一種和「守門官」凌墨完全一致的干擾波！

**旁白**：這句話像一聲驚雷，在辦公室裏炸開。

**李杰**：（震驚）完全一致？什麼意思？

**凱**：（眼神狂熱）這意味著...妳和凌墨，在某種程度上，是同一種「存在」！

**凱**：妳們的靈魂頻率，都被那個世界打上了標記！

**大輔**：（走上前，神色凝重地接話道）李杰先生，根據我們的推測...

**大輔**：妳並非只是普通的乘客。

**大輔**：妳的家族，或者妳的祖先，可能曾經與鏡像界有過極深的淵源。

**大輔**：所以妳的基因裏，天生就帶著那個世界的印記。

**旁白**：李杰感到一陣眩暈。

**李杰**：（內心獨白）同一種存在...我...和凌墨？

**李杰**：（想起凌墨那冰冷殘酷的眼神，感到一陣惡寒）這不可能...我只是個工程師...

**大輔**：（繼續說道）這也解釋了，為什麼妳能聽懂精靈語，為什麼妳能從凌墨的手下逃脫。

**大輔**：因為在那個世界的法則看來，妳...其實是他們的一員。

**清水恵美**：（震驚地看著李杰）李杰先生...

**李杰**：（顫抖地問）如果是這樣...那為什麼凌墨要殺我們？

**大輔**：（沈聲道）因為妳的存在，威脅到了他的權威。

**大輔**：妳是「不合格者」，是那個世界的叛徒。

**旁白**：李杰坐在椅子上，感覺整個世界都崩塌了。

**旁白**：他一直以為自己只是個倒霉的倖存者。

**旁白**：沒想到，他的血脈裏，竟然流淌著那個恐怖世界的秘密。

**凱**：（不顧氣氛，繼續興奮地嚷嚷）而且！最有趣的是恵美小姐！

**凱**：（指著恵美的掃描圖）妳的靈魂中，竟然有兩層重疊的標記！

**凱**：這說明，妳確實穿越過兩次！

**凱**：而且，妳和李杰先生的標記，正在慢慢融合！

**凱**：這就是為什麼，最後那一刻，你們能產生共鳴，打開光門的原因！

**大輔**：（拍了拍凱的肩膀，示意他安靜）李杰先生，清水小姐。

**大輔**：雖然這個真相很殘酷，但請記住...

**大輔**：（語氣變得溫柔而堅定）不管妳們的血脈裏流著什麼，現在，妳們都在地球上。

**大輔**：妳們是人類，是我們的同胞。

**旁白**：李杰抬起頭，看著大輔那信任的目光。

**旁白**：又轉過頭，看著恵美那充滿擔憂卻又不離不棄的眼神。

**李杰**：（深吸一口氣，握緊拳頭）我明白了。

**李杰**：（語氣變得堅定）不管我是什麼，我都不會讓那個世界的陰影再糾纏我們。

**李杰**：我們會活下去。作為人類，好好活下去。

**大輔**：（欣慰地笑了）很好。

**大輔**：（看向門口）好了，妳們的弟弟浩介已經在休息室等著了。

**大輔**：去見他吧。剩下的研究，我們可以慢慢來。

**旁白**：李杰和恵美走出辦公室，朝著休息室走去。

**旁白**：前方的路依然充滿未知。

**旁白**：但這一次，他們心中已經有了答案。

**旁白**：不合格者的秘密，不再是負擔。

**旁白**：而是他們在這殘酷命運中，最強大的武器。

**清水恵美**：（補充）艾拉如先生和我都不知道為什麼李杰先生不符合資格...這種情況很少見...

**李杰**：所以我們等了一年的冷卻期，讓恵美和斎藤蓮建立契約。之後我們一起前往終焉祭壇。

**李杰**：（表情變得凝重）但是到了祭壇...一個叫凌墨的男人突然出現。他拿著黑劍，殺了護送我們的精靈劍士凱蘭。

**李杰**：（低聲）情況很危急...我只能選擇犧牲斎藤蓮，打開門帶恵美逃走。

**旁白**：大輔沉默地聽著，表情嚴肅。

**斎藤大輔**：（皺眉）那個凌墨...是什麼來歷？

**旁白**：就在這時，恵美突然開口。

**清水恵美**：（小聲）其實...凌墨是我第一次嘗試建立契約的人...

**旁白**：李杰猛地轉頭看向恵美。

**李杰**：（震驚）什麼？

**清水恵美**：（低著頭）45年前...我第一次來到那個世界時...凌墨也在那班列車上。

**清水恵美**：他是我大學時的朋友。我們一起嘗試建立契約，但是...他也不符合資格。

**清水恵美**：（眼眶泛紅）後來我沒辦法，只好和翔建立契約。然後...凌墨也跟著我們一起回來了。

**清水恵美**：（聲音顫抖）回來之後...我馬上又上了第二班列車，想要回去救翔...

**清水恵美**：從那之後...我就再也沒見過凌墨了...對我來說...也就一年半沒見...

**清水恵美**：但我沒想到...他會出現在終焉祭壇...而且變成那樣...

**旁白**：李杰的手緊緊握著筷子。

**旁白**：凌墨...也不符合資格...

**旁白**：和他一樣...

**李杰**：（內心獨白）艾拉如說過...不符合資格的情況非常罕見...

**李杰**：（內心獨白）如果凌墨也不符合資格...那我們之間...是不是有什麼共同的特徵？

**旁白**：李杰想起凌墨那冰冷的眼神、那把染血的黑劍、還有他殺人時毫無猶豫的動作。

**旁白**：如果他和凌墨有什麼相似之處...

**旁白**：那會是什麼？

**斎藤大輔**：（嚴肅地）這個資訊很重要。不符合資格...而且是兩個人...

**斎藤大輔**：（看著李杰）你們之間一定有什麼共通點。我會讓組織的研究團隊調查這件事。

**伊藤莉子**：（擔心地）李杰先生...你不會變成像凌墨那樣吧...

**李杰**：（勉強笑了）我...我不知道...

**旁白**：但他的聲音聽起來一點也不確定。

**旁白**：不符合資格的人...

**旁白**：凌墨...和他...

**旁白**：究竟有什麼相同之處？

**旁白**：這個問題，像一塊沉重的石頭，壓在李杰的心上。
