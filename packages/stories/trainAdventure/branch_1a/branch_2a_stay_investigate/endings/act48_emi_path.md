# 第四十八幕：新生活的開始

**旁白**：飛機降落在成田國際機場。

**旁白**：李杰和恵美拖著行李，走出入境大廳。

**旁白**：人群中，一個熟悉的身影舉著牌子站在那裡。

**清水恵美**：（眼睛一亮）浩介！

**旁白**：清水浩介看到他們，露出溫暖的笑容。

**清水浩介**：（揮手）恵美！李杰先生！這邊！

**旁白**：三人會合。

**旁白**：浩介主動接過行李。

**清水浩介**：（打量著兩人）在夏威夷玩得開心嗎？看你們氣色都變好了。

**清水恵美**：（笑著點頭）嗯！超開心的！

**李杰**：（微笑）謝謝浩介先生的安排。

**清水浩介**：（擺擺手）別客氣。走吧，車已經在外面等了。

**旁白**：浩介開著車，載著兩人駛向東京市區。

**旁白**：窗外的景色飛速後退。高樓大廈、霓虹燈、還有擁擠的人群。

**旁白**：這就是東京。他們失蹤前的那個世界。

**清水浩介**：（看向後座的李杰）李杰先生，我幫你們租了一間公寓。

**清水浩介**：在品川附近，交通很方便，環境也很安靜。

**李杰**：（驚訝）幫我們租了公寓？

**清水浩介**：（點頭）對。考量到你們需要時間適應，組織決定提供一年的住房補助。

**清水浩介**：（對恵美眨眨眼）而且是兩房一廳喔！妳們可以住在一起。

**清水恵美**：（臉紅）浩介！妳在說什麼啊！

**清水浩介**：（哈哈大笑）我是說你們可以互相照應嘛！

**旁白**：不久後，車子在品川的一棟現代化公寓前停下。

**旁白**：浩介帶著兩人上樓，來到一間寬敞明亮的房間。

**清水浩介**：（把鑰匙交給李杰）就是這裡了。家具和家電都已經準備好了。

**清水浩介**：（從口袋掏出一張提款卡）還有這個。

**李杰**：（接過）生活補助？

**清水浩介**：（點點頭微笑）對。組織每個月會提供40萬日圓的補助，維持一年。應該足夠你們生活和適應這個世界了。

**清水恵美**：（震驚）40萬...這麼多...

**清水浩介**：（拍了拍李杰的肩膀）別客氣，這是組織應該做的。你們提供的情報很有價值。週末我會再來找你們，我們一起吃個飯，好好聊聊。

**清水恵美**：（點頭）好！

**李杰**：（鞠躬）謝謝浩介先生。

**清水浩介**：（揮手走向門口）好啦好啦，我走了！你們好好享受兩人世界！別這麼客氣！你可是我姐夫...不對，是我姐的男朋友！

**旁白**：恵美的臉瞬間紅了。

**清水恵美**：（慌張）浩...浩介！

**清水浩介**：（哈哈大笑）

**旁白**：浩介關上門離開了。

**旁白**：公寓裡只剩下李杰和恵美兩人。

**旁白**：兩人站在客廳，突然有些不知所措。這是他們第一次真正意義上的「同居」。

**清水恵美**：（小聲指著臥室）那個...李杰先生...房間只有一個...

**李杰**：（也有些尷尬臉紅）我可以睡沙發...

**清水恵美**：（搖頭說不下去臉紅得像蘋果）不用...我們已經...

**旁白**：李杰看著她，心跳加速。

**旁白**：在夏威夷，他們確認了彼此的心意。

**旁白**：在夏威夷，他們也睡過同一張床。

**旁白**：但那是在飯店。

**旁白**：而現在...這是他們的家。

**旁白**：真正的、屬於他們兩個人的家。

**李杰**：（溫柔地走近）恵美...

**清水恵美**：（抬頭看他）嗯...

**李杰**：（微笑）我們...一起加油吧。

**李杰**：重新適應這個世界，一起建立新的生活。

**清水恵美**：（眼眶泛紅點頭）嗯...一起...

**旁白**：李杰輕輕牽起恵美的手。

**旁白**：兩人相視而笑。

**旁白**：窗外，東京的街道車水馬龍。

**旁白**：陽光灑進客廳，溫暖而明亮。

**旁白**：他們的新生活...

**旁白**：從這裡開始了。

**旁白**：—— 全劇終 ——
