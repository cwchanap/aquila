# 終幕：永恆的終焉

**旁白**：李杰緩緩從黑暗的長廊走出來,回到了寬敞的候車大廳。

**旁白**：他的腳步沉重,每走一步都彷彿踏在無盡的疲憊之上。

**李杰**：(喃喃自語)沒有...什麼都沒有...

**旁白**：右側的長廊裡什麼都沒有。沒有答案,沒有出口,只有更深的黑暗。

**旁白**：李杰站在大廳中央,環顧四周。巨大的石柱依然矗立,黑色的地板依然映照著他孤單的身影。

**旁白**：他深吸一口氣,朝著大廳的另一側走去。

**旁白**：李杰走過空曠的大廳,腳步聲在石柱間迴盪,形成空洞的回音。

**旁白**：就在這時,一股前所未有的孤獨感襲上心頭。

**旁白**：凱蘭、恵美、斎藤...所有人都離開了。這片神聖而詭異的聖地裡,只剩下他一個人。

**李杰**：(內心獨白)我是唯一活下來的人嗎...

**李杰**：(內心獨白)可是...活下來又有什麼意義...

**旁白**：他走到大廳邊緣,看著前方無盡的黑暗。

**旁白**：原本以為只要來到終焉祭壇,就能找到回家的路。

**旁白**：可現在,除了死亡和沈默,他一無所獲。

**旁白**：就在這時,一陣輕微的腳步聲從他身後傳來。

**李杰**：(猛地轉身,充滿希望地喊道)是誰!

**旁白**：但他的希望在瞬間破滅。

**旁白**：出現在他面前的,不是死而復生的同伴,而是那個黑色風衣的男子——凌墨。

**旁白**：凌墨站在那裡,黑劍入鞘。他的表情平靜得像一潭死水,完全看不出剛才在那場血腥屠殺中的狠辣。

**凌墨**：(語氣淡然)妳是最後一個。

**李杰**：(握緊手中的短刀,憤怒地吼道)是你殺了他們!為什麼!他們做了什麼要讓你下此毒手!

**凌墨**：(沒有回答,只是走上前兩步)不合格者...本就不該存在。

**李杰**：(咬牙切齒)不合格者?就因為那個什麼狗屁規則,你就要殺掉所有人嗎!

**凌墨**：(眼神中閃過一絲微不可察的疲憊)規則...即是守護。

**凌墨**：(看著李杰)妳不想回去嗎?

**李杰**：(冷笑)回去?門根本打不開!這根本就是個陷阱!

**凌墨**：(搖搖頭)門...一直都開著。只是妳...看不見。

**旁白**：凌墨指了指大廳中央那根最粗大的石柱。

**凌墨**：妳的執念太深,所以妳的眼睛被蒙蔽了。

**旁白**：李杰順著他的手指看去。原本普通的石柱,此刻在微弱的光線下竟然呈現出一種半透明的質感。

**旁白**：在那石柱的中心,有一個緩緩旋轉的金色光漩。

**李杰**：(愣住)那是...

**凌墨**：那是真正的世界之門。

**凌墨**：(語氣冰冷)但要啟動它,需要一份...特別的能量。

**李杰**：(心中一驚,本能地後退一步)什麼能量?

**凌墨**：(緩緩拔出黑劍)不合格者的靈魂。

**旁白**：李杰的心跳猛地加速。他終於明白了——凌墨之所以殺死那些人,是為了收集啟動大門的能量!

**李杰**：(憤怒地指著他)妳這惡魔!妳殺了恵美她們,就是為了這個?!

**凌墨**：(語氣平淡)只要妳一個人回去,這一切都值得。

**李杰**：(愣住)妳說什麼?

**凌墨**：(眼神深邃地看著李杰)這是...宿命。

**旁白**：凌墨突然動了!

**旁白**：他的速度快得超乎想像,李杰根本來不及反應。

**李杰**：(慘叫)啊——!

**旁白**：黑劍刺穿了李杰的腹部。

**旁白**：劇痛像海浪般席捲全身。李杰感覺生命力正在被那把邪惡的劍快速抽離。

**凌墨**：(靠在李杰耳邊,低聲說)走吧。帶著她們的希望...活下去。

**旁白**：凌墨猛地抽回黑劍,然後用力一掌拍在李杰的胸口。

**旁白**：李杰的身體像斷了線的風箏一般,倒飛了出去,準確地落入了石柱中的金色光漩。

**李杰**：(在光芒中發出最後的吶喊)不——!

**旁白**：金色的光漩爆發出耀眼的光芒,瞬間將李杰吞噬。

**旁白**：大廳重新歸於沈默。

**旁白**：凌墨站在那裡,看著逐漸消散的光漩,長長地嘆了一口氣。

**凌墨**：(低聲自語)下一個...又會是誰呢...

**旁白**：他轉過身,緩緩走向黑暗的深處。

**旁白**：聖地的石門緩緩關上。

**旁白**：只剩下幾具冰冷的屍體,在寂靜中訴說著這場慘劇。

---

**旁白**：李杰的呼吸停止了。

**旁白**：媽媽。她站在那裡。活生生的，真實的，微笑著的媽媽。

**斎藤浩一**：（從客廳探出頭）蓮！歡迎回家！

**旁白**：爸爸也在。他們都在。就像...從來沒有失蹤過一樣。

**李杰**：（聲音顫抖）爸...媽...

**斎藤美香**：（走過來）怎麼了？臉色這麼差...是不是太累了？

**李杰**：（搖頭，聲音哽咽）沒...沒事...

**旁白**：李杰放下行李，走向母親。他伸出手，輕輕碰了碰母親的肩膀。

**旁白**：溫熱的。真實的觸感。

**李杰**：（內心，眼淚幾乎要流下來）媽媽...你真的在...

**斎藤美香**：（困惑地笑了）蓮，你今天怎麼怪怪的？

**斎藤浩一**：（笑著）可能是玩太累了。先去洗澡吧，等等一起吃晚飯。

**旁白**：就在這時，樓梯上傳來腳步聲。

**斎藤大輔**：（從樓上走下來，笑嘻嘻地）蓮！你回來啦！旅行好玩嗎？

**旁白**：李杰轉身看向大輔。

**旁白**：大輔還是那個樣子。胖胖的，開朗的，笑嘻嘻的。完全不像列車上那個穿著白色醫生袍、眼神深邃的男人。

**李杰**：（盯著大輔，試探地）大哥...

**斎藤大輔**：（走下樓梯）怎麼了？幹嘛這樣看著我？

**李杰**：（深吸一氣氣）你...你還記得...我們在列車上的對話嗎...

**斎藤大輔**：（困惑）列車？什麼列車？

**李杰**：（繼續試探）就是...那班末班車...你穿著白色的袍子...

**斎藤大輔**：（笑著搖頭）蓮，你在說什麼啊？我完全聽不懂。

**斎藤大輔**：（拍拍李杰的肩膀）你該不會是做夢了吧？哈哈哈！

**旁白**：李杰看著大輔的眼睛。那裡沒有任何陰影，沒有任何深意。

**旁白**：只有單純的困惑，和開朗的笑容。

**李杰**：（內心）他...真的不記得...

**李杰**：（內心）還是說...他的人格...也被改變了...

**斎藤大輔**：（好奇地）對了，你們去滑雪場玩得怎麼樣？有沒有摔得很慘？

**李杰**：（勉強笑了笑）...還好...

**斎藤大輔**：（笑嘻嘻）下次帶我一起去嘛！我也想去滑雪！

**旁白**：李杰看著大輔。這個大輔，和記憶中的大輔一模一樣。

**旁白**：開朗、愛笑、有點孩子氣。完全不像那個知道一切真相、厭倦了作為AI的大輔。

**李杰**：（內心）他...已經不是列車上的那個大輔了...

**李杰**：（內心）還是說...那個大輔...才是bug...而現在這個...才是真正的大輔...

**旁白**：李杰低下頭,看著自己完好無損的身體。

**旁白**：沒有傷口,沒有疼痛。

**旁白**：剛才在聖地發生的一切,凱蘭的死、恵美的死、斎藤的死...還有凌墨那致命的一劍...

**旁白**：難道真的只是一場噩夢?

**斎藤美香**：(在廚房喊道)蓮!快去洗手!吃飯了!

**李杰**：(回過神來)來了!

**旁白**：李杰走進餐廳。桌上擺滿了他最愛吃的菜。

**旁白**：父親、母親、還有哥哥大輔,圍坐在桌旁,談笑風生。

**旁白**：在這個溫暖而真實的世界裡,一切都是那麼美好。

**旁白**：如果這真的是一場夢...那就讓他永遠不要醒來吧。

**旁白**：—— 全劇終 ——
