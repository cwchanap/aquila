# 第三十七幕：破碎的救贖

**旁白**：恵美看到祭壇上的景象，整個人僵住了。

**旁白**：她的臉色瞬間變得慘白，雙眼睜大，嘴唇顫抖著。

**清水恵美**：（顫抖地）不...不可能...

**旁白**：下一秒，恵美突然衝向祭壇，動作急促而慌亂。

**清水恵美**：（尖叫）翔！翔！

**旁白**：她的雙手抓住祭壇的邊緣，整個人趴在祭壇上，看著那個年輕男子的臉。

**清水恵美**：（聲音撕裂）為什麼...為什麼會這樣！

**旁白**：恵美的聲音在空曠的聖殿中迴盪，充滿了絕望和難以置信。

**旁白**：李杰和斎藤衝上前，停在祭壇旁。

**斎藤蓮**：（倒吸一口涼氣）這...這是...

**旁白**：李杰盯著祭壇上的屍體，感覺大腦一陣眩暈。

**旁白**：那個男子——翔，安靜地躺在那裡。他的臉上沒有痛苦，甚至帶著一絲解脫般的平靜。但那把插在胸口的刀，卻是如此刺眼。

**凱蘭**：（聲音沉重）看來，我們來晚了。

**李杰**：（猛地轉向凱蘭）這是怎麼回事？為什麼他會死在這裡？

**凱蘭**：（搖了點頭）我不知道。祭壇通常是受保護的。沒人能隨意進入這裡。

**清水恵美**：（瘋狂地搖晃著翔的肩膀）醒醒！翔！求求你醒醒！你不是說過會等我嗎？你不是說過要帶我走嗎？

**旁白**：恵美的眼淚像斷了線的珍珠般落下，滴在翔那件染血的白色長袍上。

**旁白**：但翔沒有回應。他的身體已經冷了，生命已經徹底離開了。

**李杰**：（走上前，按住恵美的肩膀）恵美...冷靜點...

**清水恵美**：（猛地甩開李杰的手，轉過頭，眼神中充滿了瘋狂）冷靜？你要我怎麼冷靜！他死了！他死在我的面前！

**清水恵美**：（指著祭壇，歇斯底里）這就是你說的救贖嗎？這就是你要幫我的結果嗎？

**李杰**：（無言以對）我...

**旁白**：就在這時，李杰注意到了祭壇邊緣刻著的一行字。

**旁白**：那不是符文，而是用鮮血寫成的。

**李杰**：（低聲讀出）「唯一的真實，在於死亡的終焉。」

**斎藤蓮**：（湊過來）這什麼意思？

**凱蘭**：（臉色微變）這...這是...

**旁白**：凱蘭沒有說完，整座聖殿突然開始劇烈震動。

**旁白**：穹頂上的碎石落下，牆壁上的符文發出刺眼的紅光。

**凱蘭**：（臉色大變）糟了！儀式被中斷了！或者是...被某種力量扭曲了！

**李杰**：儀式？什麼儀式？

**凱蘭**：沒時間解釋了！我們必須馬上離開這裡！這裡要坍塌了！

**斎藤蓮**：（驚叫）坍塌？

**清水恵美**：（依然趴在祭壇上，緊緊抱著翔的屍體）不...我不走！我要留下來陪他！

**李杰**：（用力抓住恵美的手臂）恵美！跟我們走！

**清水恵美**：（拼命掙扎）放開我！讓我跟他死在一起！

**旁白**：聖殿的震動越來越劇烈，巨大的石柱開始出現裂痕。

**凱蘭**：（焦急地）快！再不走就來不及了！

**李杰**：（看著恵美絕望的神情，心中一沉）對不起，恵美。

**旁白**：李杰猛地一用力，將恵美從祭壇上拉了下來，強行拖向門口。

**清水恵美**：（尖叫著，雙手在空中亂抓）不！翔！翔！

**旁白**：四人拼命朝著出口衝去。身後傳來巨大的轟鳴聲，那是聖殿坍塌的聲音。

**旁白**：當他們衝出石門，回到那條發光的長廊時，身後那扇巨大的石門已經被碎石徹底堵死。

**斎藤蓮**：（靠在牆上，劇烈地喘著氣）呼...呼...差點...差點就交待在裡面了。

**清水恵美**：（跪在地上，泣不成聲）

**李杰**：（看著那堆碎石，心中充滿了複雜的情緒）

**凱蘭**：（臉色凝重地看著李杰）李杰，我有話要跟你說。

**李杰**：（抬起頭）什麼？

**凱蘭**：（深吸一口氣）關於剛才祭壇上的那個男子...以及這一切的真相。

**李杰**：真相？

**凱蘭**：嗯。但這裡不安全。我們要找個安靜的地方。

**斎藤蓮**：（一臉懵逼）真相？還有什麼真相？

**凱蘭**：跟我來。

**旁白**：凱蘭帶著三人在長廊中穿梭，最後來到一個隱蔽的小房間。房間裡只有一張石桌和幾把石凳。

**凱蘭**：（示意大家坐下）

**李杰**：（坐在石凳上，看著凱蘭）說吧。

**凱蘭**：（沈默了一會兒，開口說道）剛才祭壇上的那個年輕男子...他並不是死於他殺。

**清水恵美**：（猛地抬起頭，雙眼通紅）你說什麼？

**凱蘭**：（看著恵美）他是自殺的。

**清水恵美**：（激動地站了起來）胡說！翔不可能自殺！他答應過我...

**凱蘭**：（打斷她）但他必須這麼做。

**凱蘭**：（語氣平靜而殘酷）那是儀式的一部分。為了讓這個世界繼續運轉，為了壓制住那個「存在」，每隔二十五年，就必須有一個「不合格者」主動獻出生命。

**清水恵美**：（僵住了）不合格者...獻出生命？

**凱蘭**：（點了點頭）是的。而翔，就是這一代的「祭品」。

**清水恵美**：（癱坐在地上）

**李杰**：（憤怒地站了起來）所以，這一切都是一場騙局？你們利用他的愛，利用他的善良，讓他去死？

**凱蘭**：（冷冷地看著李杰）這不是騙局，這是責任。如果沒有他的犧牲，這列火車，這個世界，早在二十五年前就崩潰了。

**李杰**：（握緊拳頭）那艾拉如呢？她也是這儀式的一部分嗎？

**凱蘭**：（眼神閃爍了一下）艾拉如...她是守護者。她的責任是確保儀式順利進行。

**李杰**：確保儀式順利進行...（冷笑）所以，她也知道這一切？

**凱蘭**：她知道。

**旁白**：李杰感覺心臟像是被一隻無形的大手緊緊揪住。

**旁白**：他想起艾拉如那恬靜的笑容，想起她對他說過的那些話...原來，那一切都是建立在別人的痛苦和犧牲之上的嗎？

**斎藤蓮**：（結巴）你...你說什麼？

**斎藤蓮**：（往後退了一步）躺...躺到祭壇上？

**斎藤蓮**：（聲音顫抖）那...那這個人是怎麼死的？

**李杰**：（避開視線）我不知道。

**斎藤蓮**：（激動地）你不知道？你他媽的騙誰！

**斎藤蓮**：（指著李杰）你一定知道什麼！說清楚！

**旁白**：李杰沒有回答。他只是沉默地站在那裡。

**旁白**：斎藤盯著李杰看了一會兒，然後緩緩走向祭壇。

**旁白**：他伸出顫抖的手，抓住翔的肩膀。

**旁白**：屍體很冷，觸感僵硬。斎藤打了個寒顫，但還是咬著牙，用力將屍體拖下祭壇。

**旁白**：翔的身體沉重地落在地面上，發出沉悶的聲響。

**旁白**：恵美渾身無力地靠在李杰懷中，眼神空洞地看著那具被拖動的屍體。

**旁白**：她的嘴唇微微顫抖，但已經發不出任何聲音。

**旁白**：斎藤沒有理會恵美的狀態。他已經將屍體移到一邊。

**旁白**：斎藤看著空蕩蕩的祭壇，又看向李杰。

**斎藤蓮**：（冷笑）躺上去是吧？

**斎藤蓮**：（質疑地）那...為什麼不是你躺上去？

**斎藤蓮**：（看向恵美）或者她？為什麼一定要是我？

**李杰**：（張嘴想說話）這是因為...

**旁白**：但就在這時——

**旁白**：一陣清晰的腳步聲從聖殿入口傳來。

**旁白**：腳步聲很緩慢，很有節奏，在空曠的空間中迴盪。

**旁白**：所有人同時轉頭，看向聲音的來源。

**旁白**：一個身影從陰影中走出。

**旁白**：那是一個年輕男子，身材修長，穿著黑色的夾克和黑褲。

**旁白**：他的右手握著一把黑色的劍，劍身上殘留著乾涸的血跡。

**旁白**：他的臉部線條冷峻，眼神深邃而漠然。

**旁白**：他就這樣慢慢地、一步一步地，朝著他們走來。

**旁白**：每一步都踩在堅硬的石地板上，發出清脆的回響。

**凱蘭**：（警戒地握住劍柄）是誰！

**旁白**：但那個男子沒有回答。

**旁白**：他只是繼續走著，黑色的劍在微弱的光線下閃爍著冰冷的寒光。

**旁白**：隨著距離的拉近，李杰終於看清了那張臉。

**旁白**：那張熟悉的...卻又陌生的臉。
