# 第三十八幕：絕境逃生

**旁白**：李杰的身體猛地一震。

**旁白**：那張臉——他見過。

**李杰**：（內心獨白）是他...新宿車站...那個靠在柱子旁的黑衣男...

**旁白**：李杰清楚地記得那個夜晚。

**旁白**：在新宿車站月台上，那個穿著黑色連帽衛衣的男子，靠在角落的柱子旁。

**旁白**：帽簷壓得很低，雙手插在口袋裡，一動也不動。

**旁白**：當時李杰就覺得那個人很違和，像是在觀察什麼。

**旁白**：但來到這個世界後，那個男子就再也沒有出現過。

**旁白**：為什麼...為什麼他會出現在這裡？

**旁白**：而且...他剛才殺了凱蘭。

**旁白**：用那樣殘忍、冷酷的方式。

**李杰**：（聲音顫抖）你是...新宿車站的那個...

**旁白**：男子停下腳步。他抬起頭，帽簷下的目光如冰隼般犀利。

**凌墨**：（聲音冷冽，不帶一絲情感）凌墨。

**李杰**：凌墨...

**凌墨**：（看著李杰）你不該出現在這裡。

**李杰**：（握緊拳頭）什麼意思？

**凌墨**：（語氣淡漠）這不是你該參與的遊戲。

**旁白**：凌墨緩緩舉起手中的黑劍，劍尖直指李杰。

**凌墨**：既然進來了，就別想出去了。

**旁白**：感受到那股徹骨的殺氣，李杰本能地後退一步。

**旁白**：恵美依然靠在他懷裡，像個失去了靈魂的木偶。

**斎藤蓮**：（驚叫）李杰！快躲開！

**旁白**：就在凌墨準備動手的瞬間，聖殿再次劇烈震動。

**旁白**：這一次，震動比之前任何一次都要猛烈。

**旁白**：無數碎石從穹頂落下，巨大的石柱紛紛倒塌。

**凱蘭**：（僅剩的上半身，掙扎著發出微弱的聲音）快...快跑...

**旁白**：李杰咬著牙，抱緊恵美，轉身衝向聖殿深處。

**斎藤蓮**：等等我！

**旁白**：斎藤緊隨其後。

**旁白**：凌墨看著逃跑的三人，眼神中閃過一絲嘲諷。

**凌墨**：（低聲自語）無謂的掙扎。

**旁白**：他身形一閃，化作一道殘影，追了上去。

**旁白**：李杰穿梭在崩塌的碎石之間。

**旁白**：身後傳來凌墨平穩而可怕的腳步聲，那聲音就像死神的倒計時。

**旁白**：前方出現了一道發光的門。那是這層聖殿的出口。

**李杰**：（大喊）斎藤！衝過去！

**斎藤蓮**：我知道！

**旁白**：斎藤用盡全力，一躍而入。

**旁白**：就在李杰即將到達門口時，凌墨出現在他身後。

**凌墨**：結束了。

**旁白**：黑劍猛地劈下。

**旁白**：李杰感到背後傳來一陣劇痛，鮮血染紅了他的衣服。

**旁白**：但他沒有停下。藉著這股衝力，他抱著恵美，一頭栽進了那道光門。

**旁白**：一瞬間，天旋地轉。

**旁白**：當李杰再次睜開眼時，發現自己躺在草地上。

**旁白**：陽光穿透樹葉間的縫隙，灑在他臉上。

**旁白**：微風徐徐，帶著泥土和花草的芬芳。

**旁白**：一切都顯得那麼平靜、那麼美好。

**旁白**：如果忽略他背後的傷口，以及懷中那個崩潰的少女。

**李杰**：（虛弱地）呼...呼...逃出來了嗎？

**斎藤蓮**：（坐在不遠處，驚魂未定）好險...真的好險...

**清水恵美**：（趴在李杰胸口，哭聲已經沙啞）

**旁白**：李杰忍著劇痛，輕輕撫摸著恵美的頭髮。

**李杰**：沒事了，恵美。我們安全了。

**旁白**：但他的內心卻充滿了疑慮。

**旁白**：凌墨是誰？他為什麼要在新宿車站觀察他？

**旁白**：這一切...真的只是一場意外嗎？

**旁白**：李杰閉上眼，任由疲憊席捲全身。

**旁白**：他不知道未來會怎樣，但他知道，這場噩夢遠未結束。

**旁白**：但凌墨只是微微側身。

**旁白**：凱蘭的劍從他身旁掠過，連衣角都沒有碰到。

**旁白**：下一秒——

**旁白**：凌墨的黑劍動了。

**旁白**：那是一個簡單而優雅的橫斬。

**旁白**：沒有多餘的動作，沒有花哨的技巧。

**旁白**：黑劍在空中劃過，留下一道漆黑的軌跡。

**旁白**：然後——

**旁白**：凱蘭的身體從腰部斷成兩截。

**旁白**：上半身 and 下半身緩緩分離，鮮血噴濺而出。

**凱蘭**：（眼睛睜大，發不出聲音）......

**旁白**：凱蘭的上半身跌落在地，眼中還殘留著震驚和難以置信。

**旁白**：整個過程，不到一秒。

**李杰**：（震驚地瞪大眼睛）凱蘭...

**旁白**：李杰的大腦一片空白。

**旁白**：凱蘭——那個優秀的精靈劍士，那個一路保護他們的戰士——

**旁白**：就這樣，在他眼前被斬成兩半。

**旁白**：凌墨踏過凱蘭的屍體，繼續朝李杰走來。

**旁白**：他的表情依然冷漠，彷彿剛才只是做了一件微不足道的事。

**旁白**：李杰的身體本能地顫抖。

**旁白**：但恵美——恵美還癱坐在地上，完全沒有意識到危險。

**李杰**：（內心獨白）不能猶豫...不能停下...

**旁白**：李杰衝向恵美，彎下腰，雙手從她的肩膀 and 膝蓋下穿過，將她整個人抱起。

**清水恵美**：（茫然地）李杰先生...

**李杰**：（急促地）抱歉了！

**旁白**：李杰轉身，朝著那扇發光的門狂奔。

**旁白**：他的肺部像要炸開，心臟瘋狂跳動。

**旁白**：身後，凌墨的腳步聲越來越近。

**旁白**：但李杰不敢回頭。

**旁白**：他只是抱緊恵美，用盡全力往前跑。

**旁白**：五米...三米...一米...

**旁白**：李杰縱身一躍，撲進那道光門。

**旁白**：一瞬間，耀眼的白光吞沒了他的視野。

**旁白**：一切都消失在那片光芒之中。
