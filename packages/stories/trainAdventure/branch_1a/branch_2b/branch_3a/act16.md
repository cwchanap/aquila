# 第十六幕：失落的庇護所

**山本浩二**：我也想去救他們，但火太大了，我根本靠近不了。

**山本浩二**：我只能跟著其他精靈一起逃進森林。

**李杰**：後來呢？

**山本浩二**：後來大霧越來越濃，我和他們走散了。

**山本浩二**：我一個人在森林裡走了好幾個小時，完全迷失了方向。

**山本浩二**：就在我以為要永遠困在這片森林裡的時候，我發現了一棟木屋。

**高橋美咲**：木屋？

**山本浩二**：（點頭）對，一棟木屋。保存得很完好，有床、廚房、還有一些補給品。

**山本浩二**：我想說可以把那裡當作據點，休息一下，然後繼續尋找你們和那三個大學生。

**李杰**：所以你就進去了？

**山本浩二**：對。但當我推開門的時候，發現裡面已經有人了。

**高橋美咲**：（緊張地）誰？

**山本浩二**：一個年輕男生，坐在角落裡，手臂上纏著布條。

**山本浩二**：我認出他了——就是當時和你們一起的那四個大學生之一。

**李杰**：他叫什麼名字？

**山本浩二**：佐藤拓海。

**高橋美咲**：（眼睛一亮，但隨即又露出複雜的表情）拓海...他還活著...

**旁白**：美咲的聲音輕得幾乎只有自己能聽見，語氣中混雜著欣慰、擔憂，還有一絲說不清的情緒。

---

**山本浩二**：他看到我的時候也很驚訝。

**山本浩二**：他說他認得我，因為在你們被攻擊的那天，他其實試圖折返回去救妳。

**高橋美咲**：（愣住）他...他想回來救我？

**山本浩二**：（點頭）對。他說當時他們四個人被攻擊後四散逃跑，但跑了一段路後，他開始擔心妳的傷勢。

**山本浩二**：所以他決定冒險折返。在回去的路上，他看到我在追一個精靈。

**李杰**：所以他當時也在附近？

**山本浩二**：對，但是霧太濃了。

**山本浩二**：他說他找了好幾個小時，完全找不到妳倒下的地方。霧氣把所有的痕跡都遮蓋了，他根本分不清方向。

**山本浩二**：最後他迷路了，受傷的手臂也越來越痛，只能放棄搜索。

**山本浩二**：就在他快要絕望的時候，他找到了那棟木屋，就在裡面休養到現在。

**旁白**：美咲聽著山本大叔的描述，表情從驚訝變成欣慰，又慢慢轉為憤怒。她咬著嘴唇，雙手握緊。

**高橋美咲**：（聲音帶著怨氣）可是當初被攻擊的時候，他也沒有保護我啊！

**高橋美咲**：大家都只顧著自己逃跑！如果他真的在乎我，為什麼不在第一時間就保護我？！

**山本浩二**：（安撫地）小姑娘，當時的情況太突然了，誰都沒有反應的時間...

**高橋美咲**：（深吸一口氣，情緒稍微平復）...算了。至少他還活著。

**高橋美咲**：他...他現在還在那棟房子裡嗎？

**山本浩二**：對，他的傷還沒完全好，走路有點不方便。

**山本浩二**：我跟他說我要出來找你們，他就留在那裡等我。

**高橋美咲**：那我們快去吧！

**山本浩二**：（站起身）好，我帶你們去。路程不算太遠，大概一個多小時就能到。

**李杰**：那我們出發吧。

---

**旁白**：四人——李杰、恵美、美咲、山本大叔——離開了焦黑的村落廢墟，重新踏入濃霧瀰漫的森林。

**旁白**：山本大叔走在最前面帶路，其他人緊跟在後。霧氣依然濃重，能見度不超過五公尺。

**旁白**：一路上，大家都沒有說太多話。

**旁白**：美咲的表情複雜，時而皺眉，時而咬唇，似乎在思考著關於拓海的事。她的步伐比平時快了許多，顯然迫不及待想見到他。

**旁白**：恵美依然沉默，眼神空洞，像是還沉浸在之前的悲傷中。她機械地跟著前面的人走，對周圍的一切都漠不關心。

**旁白**：李杰則不時觀察著周圍的環境，保持著警惕。他的手一直放在口袋裡，隨時準備應對突發狀況。

**旁白**：森林依然死寂，只有腳步踩在落葉和枯枝上的聲音。偶爾有風吹過，樹枝發出輕微的搖曳聲。

**旁白**：大約一個多小時後，濃霧中出現了一棟木屋的輪廓。

**山本浩二**：就是這裡了。

**旁白**：那是一棟簡樸的木屋，外牆是深褐色的木板，屋頂覆蓋著青苔。窗戶透出微弱的光線，表示裡面確實有人。

**旁白**：山本大叔快步走到門前，推開木門。

**山本浩二**：（大聲喊道）拓海！我把他們帶回來了！

**旁白**：門緩緩打開，四人魚貫而入。

**旁白**：但當他們看清屋內的景象時，所有人都愣住了。

**李杰**：（倒吸一口涼氣）這...這怎麼可能...
