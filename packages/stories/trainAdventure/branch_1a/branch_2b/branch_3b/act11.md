# 第十一幕：廢棄車站的發現

**旁白**：李杰扶著斎藤蓮在霧中緩慢前進。斎藤受傷的腳讓他們的速度慢得令人絕望，每走幾步就要停下來休息。

**旁白**：時間一分一秒地流逝。一個小時過去了，兩個小時過去了。霧氣依然濃厚，沒有絲毫減弱的跡象。

**斎藤蓮**：還要走多久⋯⋯我的腳⋯⋯快受不了了⋯⋯

**李杰**：快到了，再忍耐一下。

**旁白**：李杰嘴上這麼說，但心裡完全沒有底。他甚至不知道自己在往哪個方向走。

**旁白**：就在他快要放棄的時候，前方的霧氣中，那個巨大的黑色輪廓越來越清晰了。

**旁白**：那是一座建築物——一座古老而龐大的建築物。

**李杰**：那是⋯⋯車站？

**旁白**：隨著他們越走越近，建築的全貌逐漸在霧中顯現。那是一座廢棄的車站，外牆佈滿了裂痕和青苔，屋頂的瓦片破損不堪。

**旁白**：車站前立著一塊巨大的站牌。雖然表面斑駁，但上面刻著的四個大字依然清晰可辨——「幽冥之都」。

**斎藤蓮**：幽冥之都⋯⋯這⋯⋯這裡是⋯⋯

**旁白**：斎藤蓮的臉色突然變得蒼白，身體開始顫抖。

**斎藤蓮**：不對！這個方向完全不對！你這個笨蛋！你根本走錯方向了！

**李杰**：嗯？你知道這個地方？

**斎藤蓮**：我⋯⋯我⋯⋯

**旁白**：斎藤蓮意識到自己又說漏了嘴，立刻閉上了嘴巴。

**李杰**：你剛才說方向不對，是因為你知道回去的路在哪裡嗎？看來你對這附近很熟悉啊。

**斎藤蓮**：我⋯⋯我只是隨便說說！我怎麼可能知道這種鬼地方！

**旁白**：李杰沒有繼續追問，只是意味深長地看了斎藤蓮一眼。

**李杰**：好吧。既然已經到這裡了，先進去休息一下吧。你的腳也需要休息。

**旁白**：李杰扶著斎藤蓮走進車站大廳。腳下的石板冰冷而堅硬，上面布滿了歲月侵蝕的裂痕。

**旁白**：大廳極其寬闊，天花板高得幾乎看不見頂端。到處都是蜘蛛網和厚厚的灰塵。空氣中飄浮著細小的塵埃，帶著一股腐朽的氣味。

**李杰**：這裡應該很久沒有人來過了⋯⋯你先坐下休息。

**旁白**：李杰在大廳一角找到一張還算完整的長椅，扶著斎藤蓮坐下。

**斎藤蓮**：你⋯⋯你要去哪裡？

**李杰**：我在附近看看，也許能找到什麼有用的東西。你就待在這裡，別亂動。

**旁白**：李杰開始在車站裡搜索。他穿過一排排鏽死的驗票閘門，那些閘門早已無法轉動，金屬表面布滿了紅褐色的鏽斑。

**旁白**：在一個角落，他發現了一間像是站長室的房間。門虛掩著，沒有上鎖。

**旁白**：他推開門，門軸發出刺耳的吱呀聲。一股更濃重的霉味撲面而來，夾雜著紙張腐朽的氣味。

**旁白**：房間裡一片狼藉。桌椅翻倒，文件散落一地，牆上的木板剝落。但有一個地方引起了他的注意。

**旁白**：牆上有一塊明顯被移動過的痕跡。那裡原本似乎掛著什麼東西，但現在已經空空如也。周圍的灰塵被擾動過，有人曾經在這裡拿走了什麼。

**李杰**：(內心)有人來過這裡⋯⋯而且是最近的事。

**旁白**：他繼續搜索。在一個警衛室裡，他看到一個被踹開的鐵皮櫃。櫃門歪斜地掛著，裡面空蕩蕩的，只有幾張破爛的文件和一些腐朽的布料。

**旁白**：櫃子後方的角落也有被人翻動過的痕跡。地上的灰塵亂七八糟的，顯然有人在這裡找過什麼東西。

**李杰**：(內心)看來之前有人已經把這裡搜過一遍了。能用的東西大概都被拿走了⋯⋯

**旁白**：李杰確認周圍沒有什麼有用的物品後，從口袋裡拿出了那張從斎藤那裡撿到的地圖。

**旁白**：他展開地圖，藉著從窗戶透進來的紫月光芒仔細端詳。

**旁白**：地圖畫得簡略，但標示出了幾個主要地點。在標註著「幽冥之都」的位置上，有一條往北延伸的道路，盡頭處畫著一個簡單的房子圖案。

**李杰**：(內心)北邊有房子⋯⋯看起來不遠。如果能到那裡，也許能找到更多線索。

**旁白**：他將地圖收回口袋，然後拿出那個金屬打火機。

**旁白**：打火機在手中沉甸甸的，看起來很舊但保養得很好。李杰試著按下開關，火石摩擦發出清脆的聲音，但沒有火焰。

**李杰**：(內心)沒油了嗎？也是，這種地方怎麼可能有打火機的燃料。這東西帶著也沒用。

**旁白**：他隨手把打火機放在警衛室的桌子上，轉身準備離開。

**旁白**：但就在他要走出房間時，腳下踩到了什麼東西，發出「咔噠」一聲。

**旁白**：李杰低頭一看，發現地上有一塊磚頭看起來和其他的不太一樣。

**旁白**：他蹲下身，用手敲了敲那塊磚。

**旁白**：「咚咚」——聲音很空洞，和周圍實心的磚塊完全不同。

**李杰**：(內心)這塊磚⋯⋯是空心的？裡面有東西？

**旁白**：李杰試著用手摳開那塊磚，但磚頭和地面黏得很緊，根本無法撬動。他又試著用力踩踏，想把磚頭踩碎，但磚頭異常堅硬。

**李杰**：(內心)該死⋯⋯如果有工具就好了。但這裡什麼都沒有⋯⋯

**旁白**：他環顧四周，想找點什麼可以用來破壞磚頭的東西。但警衛室裡空空如也，連一根像樣的鐵棍都沒有。

**旁白**：那個被踹開的鐵皮櫃雖然是金屬的，但太大太笨重，根本無法用來當工具。

**李杰**：(內心)算了⋯⋯先記住這個位置，以後有機會再回來看看。

**旁白**：李杰在心中記下那塊空心磚的位置，然後站起身。

**旁白**：他深深地看了那塊磚一眼，轉身走出警衛室。

**旁白**：而在這片死寂中，那塊空心磚靜靜地躺在地板上，守護著它內部不為人知的秘密。
