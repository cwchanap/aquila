# 第十二幕：重逢的小屋

**旁白**：李杰回到大廳時，斎藤蓮還縮在長椅上。看到李杰回來，他立刻抬起頭。

**斎藤蓮**：找⋯⋯找到什麼了嗎？

**李杰**：我在站長室找到了一張地圖。

**斎藤蓮**：地圖？！

**旁白**：斎藤蓮的眼睛瞬間亮了起來，身體也因為激動而前傾。

**李杰**：嗯，上面標示了附近的路線。往北走，不遠處有一座房子。

**斎藤蓮**：房子？太好了！那我們快走吧！快帶我去！

**旁白**：斎藤蓮的反應異常激動，完全不像之前那副謹慎膽小的樣子。他試圖站起來，但腳傷讓他發出一聲痛呼。

**李杰**：別急。你的腳還沒好，走太快只會更糟。

**旁白**：李杰走過去，再次扶起斎藤蓮。兩人緩緩走出車站，沿著地圖上標示的方向前進。

**旁白**：霧氣依然濃厚，能見度極低。他們只能摸索著前進，每走一步都要小心翼翼。

**旁白**：時間在緩慢的行走中流逝。一個小時，兩個小時。斎藤蓮的腳傷讓他們的速度慢得令人絕望。

**斎藤蓮**：還⋯⋯還要多久⋯⋯我的腳快受不了了⋯⋯

**李杰**：應該快到了。再堅持一下。

**旁白**：又過了一段時間，前方的霧氣中終於出現了一個熟悉的輪廓。

**旁白**：那是一棟雙層的木屋，靜靜地矗立在林中的空地上。屋子結構完好，有個小門廊和幾級木製台階。尖頂的屋頂上，煙囪筆直地伸向天空。

**李杰**：到了。

**斎藤蓮**：(鬆了一口氣)終於⋯⋯

**旁白**：李杰扶著斎藤蓮走上門廊的台階。木板在他們的重量下發出輕微的吱呀聲。

**旁白**：他伸手準備推門，但猶豫了一下，最後還是選擇敲門。

**旁白**：「咚咚咚」——敲門聲在寂靜的森林中格外清晰。

**旁白**：幾秒鐘後，屋內傳來一陣窸窸窣窣的聲音。有人！

**旁白**：一個稚嫩的聲音從門後傳來，帶著明顯的警惕和恐懼。

**懂事姐姐**：誰⋯⋯誰在外面？

**旁白**：李杰愣了一下。這個聲音⋯⋯他聽過。

**李杰**：(內心)這是⋯⋯那個小女孩？火車上那家人的孩子？

**李杰**：我們是從火車上下來的乘客。我叫李杰，還記得嗎？

**旁白**：門後沉默了幾秒。

**懂事姐姐**：(小聲)火車上的⋯⋯大哥哥？

**李杰**：對。我們在霧中迷路了，而且我的同伴受傷了。能讓我們進去休息一下嗎？

**天真弟弟**：(在門後小聲問)姐姐，怎麼辦⋯⋯

**旁白**：門後又是一陣竊竊私語。過了一會兒，門鎖轉動的聲音響起。

**旁白**：門緩緩打開了一條縫。一個小小的身影躲在門後，只露出半張臉。

**懂事姐姐**：(怯生生地)你們⋯⋯不是壞人吧？

**旁白**：李杰看著那張充滿戒備的小臉，不由得苦笑。他撓了撓頭，盡量露出溫和的笑容。

**李杰**：不是壞人。我保證。

**旁白**：小女孩盯著李杰看了好一會兒，似乎在確認他的話是否可信。

**天真弟弟**：(從姐姐身後探出頭)是那個大哥哥！我記得！

**旁白**：小女孩這才放下心來，打開了門。

**旁白**：李杰扶著斎藤蓮走進屋內。屋子裡陳設簡單，但一塵不染。一張木桌，四把椅子，角落裡有石砌的壁爐，旁邊堆著木柴。

**旁白**：在屋子的一側，有一張鋪著乾淨毯子的單人床。床上還有兩個小小的身影剛剛起身的痕跡。

**李杰**：謝謝你們讓我們進來。對了，我剛才說了我叫李杰，但還不知道你們的名字呢。

**懂事姐姐**：(有些害羞)我⋯⋯我叫小林陽菜⋯⋯

**天真弟弟**：(躲在姐姐身後)我叫小林翔太！

**李杰**：陽菜、翔太，很高興認識你們。這位是⋯⋯

**斎藤蓮**：(不耐煩地打斷)斎藤蓮。行了，寒暄就到這裡吧。

**旁白**：斎藤蓮一瘸一拐地走到木桌旁，重重地坐在椅子上。

**斎藤蓮**：(看了一眼屋內，然後看向兩個孩子)就你們兩個小鬼在這裡？你們的父母呢？

**旁白**：斎藤蓮的語氣有些高傲，完全不像是在求助。

**小林陽菜**：(低下頭，聲音帶著哭腔)爸爸和媽媽⋯⋯不在⋯⋯

**小林翔太**：(抓著姐姐的衣角)媽媽說她很快就會回來的⋯⋯

**李杰**：你們的父母去哪了？能告訴我嗎？

**小林陽菜**：(擦了擦眼淚)我們⋯⋯我們到這裡之後⋯⋯爸爸說要出去找食物和水⋯⋯

**小林翔太**：爸爸說很快就回來⋯⋯但是⋯⋯

**小林陽菜**：但是過了很久很久⋯⋯爸爸都沒有回來⋯⋯媽媽很擔心⋯⋯

**小林陽菜**：然後媽媽說她要去找爸爸⋯⋯讓我們在這裡等著⋯⋯不要出去⋯⋯

**小林翔太**：(聲音發顫)媽媽也走了好久好久了⋯⋯都不回來⋯⋯

**旁白**：小林翔太說著說著，眼淚就掉了下來。陽菜趕緊抱住弟弟，自己的眼眶也紅了。

**李杰**：(內心)父親先出去找補給，結果沒回來。母親又出去找父親⋯⋯兩個都沒回來。讓兩個這麼小的孩子獨自待在這裡⋯⋯

**李杰**：他們大概什麼時候離開的？你們記得嗎？

**小林陽菜**：(搖搖頭)不⋯⋯不知道⋯⋯沒有時鐘⋯⋯但是感覺過了好久好久⋯⋯

**小林陽菜**：我們睡了一覺⋯⋯醒來之後⋯⋯他們還是沒有回來⋯⋯

**斎藤蓮**：(不耐煩地)行了行了，別管那些了。反正我們現在也走不了，就先在這裡休息吧。

**李杰**：(皺眉，但沒有反駁)⋯⋯嗯。

**旁白**：李杰沒有說什麼。他扶著斎藤蓮確保他坐穩。

**斎藤蓮**：(鬆了一口氣)終於⋯⋯能休息了⋯⋯

**旁白**：陽菜和翔太站在一旁，眼神有些畏縮地看著斎藤蓮。

**旁白**：李杰環顧四周，心中盤算著接下來該怎麼辦。屋外的霧氣依然濃厚，繼續趕路顯然不是明智的選擇。

**旁白**：而這兩個孩子，孤零零地待在這裡，也讓他無法放心離開。

**旁白**：更重要的是，他們的父母到底去了哪裡？又會不會回來？

**旁白**：無數的疑問在他心中浮現，但此刻，他能做的只有先讓所有人好好休息。

**旁白**：窗外，濃霧依然籠罩著整片森林。而在這座孤零零的小屋裡，四個陌生人的命運，就這樣交織在了一起。
