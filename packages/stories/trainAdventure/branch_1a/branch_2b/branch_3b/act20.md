# 第二十幕：意外的重逢

**旁白**：李杰站在門前，伸手敲了敲木門。

**旁白**：「咚咚咚」——敲門聲在寂靜的夜色中格外清晰。

**李杰**：田中先生？你在裡面嗎？

**旁白**：屋內傳來一陣窸窸窣窣的聲音，像是有人在移動。隨後，腳步聲逐漸接近門口。

**旁白**：門「咯吱」一聲被打開了。

**旁白**：出現在門口的，不是健太，而是一張李杰熟悉的臉——戴著眼鏡，神情有些緊張的斎藤蓮。

**李杰**：（驚訝）斎藤？

**斎藤蓮**：（同樣驚訝）李桑！？你怎麼會在這裡？

**旁白**：兩人在門口對視了幾秒，都從對方臉上看到了意外。

**李杰**：我應該問你才對。你為什麼會在這裡？

**旁白**：斎藤蓮讓開身位，讓李杰進屋。李杰走進屋內，環顧四周。

**旁白**：屋子裡很簡樸，壁爐裡燃著微弱的火光。桌上還放著一些食物的殘渣，看起來斎藤剛吃過東西。

**斎藤蓮**：（坐回椅子上）我...我不想一直待在那個家庭那裡。他們一家四口，加上我這個外人，太擠了。

**斎藤蓮**：所以我腳傷好了一點之後，就想去你說的那個房子找你。

**李杰**：然後呢？

**斎藤蓮**：結果我到那裡的時候，發現屋子裡有兩個大學生。一個男的叫田中健太，一個女的叫鈴木明日香。

**李杰**：（皺眉）明日香？

**旁白**：李杰記得健太提過這個名字。那是他們小組裡受傷的女生。

**斎藤蓮**：對。我問他們有沒有見過你，那個叫健太的說見過。

**斎藤蓮**：他說他在銀月湖遇到了你，而且他要去那裡和你會合。所以我就...

**旁白**：斎藤蓮看了看自己還有些腫脹的腳踝。

**斎藤蓮**：我的腳還沒完全好，走不了那麼遠。所以就決定先在這裡休息。

**李杰**：（困惑）等等，這裡不是我住的那間房子啊。

**斎藤蓮**：（尷尬地）啊...是嗎？我可能...走錯方向了。

**旁白**：李杰嘆了口氣。在這個霧氣瀰漫的世界裡，沒有明確的路標，迷路也不奇怪。

**李杰**：（沉思）如果健太和明日香要去銀月湖，我應該會在路上遇到他們才對...

**李杰**：但我一路過來什麼都沒看到。看來他們也走錯方向了。

**斎藤蓮**：（有些不安）那他們...會不會有危險？

**李杰**：應該不會。他們兩個人一起，而且這條路看起來很安全。等他們發現走錯了，自然會折返。

**旁白**：李杰在另一張椅子上坐下。

**李杰**：我接下來要回幽冥之都車站調查一些東西。

**斎藤蓮**：（驚訝）什麼？回那個鬼地方？為什麼？

**李杰**：（平靜地）我有些事情需要確認。

**旁白**：李杰沒有提起神樹說的那些話。那些關於血脈、罪孽的模糊預言，他還沒搞清楚是什麼意思。

**斎藤蓮**：（緊張地）等等！你不會是要把我一個人丟在這裡吧？

**斎藤蓮**：我的腳還沒好！走不了那麼快！而且...而且一個人在這種地方太可怕了！

**旁白**：斎藤蓮的聲音帶著明顯的恐慌。

**李杰**：（嘆氣）我知道了。我可以在這裡休息一下，等一段時間再出發。

**斎藤蓮**：（鬆了口氣）真的嗎？謝謝...

**李杰**：而且健太和明日香應該不會有事。等他們發現走錯路，自然會找到正確的方向。

**旁白**：李杰站起身，走向廚房的櫃子。裡面還有一些食材和乾糧。

**李杰**：先吃點東西吧。我們都需要補充體力。

**旁白**：李杰開始準備食物。他在壁爐上架起一個小鍋，倒入一些水，加入麵包塊和一些乾肉。

**旁白**：簡單的湯很快就煮好了。熱氣騰騰的香味在小屋裡飄散開來。

**旁白**：兩人坐在桌前，默默地吃著簡單的晚餐。

**斎藤蓮**：（猶豫地）那個...李桑。

**李杰**：嗯？

**斎藤蓮**：你...你平常是做什麼工作的？

**李杰**：（喝了口湯）軟體工程師。在東京工作。

**斎藤蓮**：（眼睛一亮）誒？是嗎？我在讀電腦科學。

**李杰**：大學生？

**斎藤蓮**：（點頭）嗯...三年級了。

**李杰**：你平常有什麼興趣？

**斎藤蓮**：（放鬆了一些）我喜歡玩遊戲。RPG、動作遊戲都玩。還有看動畫。

**李杰**：（點頭）我也是。工作之餘會玩玩遊戲，看看動畫。偶爾去健身房，或者去爬山。

**斎藤蓮**：（驚訝）爬山？你還做運動啊...

**斎藤蓮**：（小聲）我基本上一整天都坐在電腦前面...

**李杰**：（嘴角上揚）所以你是個家裡蹲？

**斎藤蓮**：（立刻炸毛）什麼！？我才不是家裡蹲！我只是...只是比較宅而已！

**旁白**：斎藤蓮漲紅了臉，激動地辯解著。

**李杰**：（淡淡地）不都一樣嗎？

**斎藤蓮**：（氣憤地）完全不一樣好嗎！家裡蹲是那種完全不出門的！我...我偶爾也會出門的！

**旁白**：李杰沒有繼續取笑他，只是低頭繼續喝湯。但嘴角依然掛著一絲笑意。

**旁白**：兩人在這種微妙的氣氛中吃完了晚餐。

**旁白**：飯後，李杰清洗了碗碟，斎藤則坐在椅子上，時不時看向窗外那片永恆的黑暗。

**李杰**：（擦乾手）該休息了。明天...或者說，下一段旅程開始前，我們都需要養足精神。

**斎藤蓮**：（點頭）嗯...

**旁白**：斎藤蓮爬上床，在床上躺下。他的腳還在隱隱作痛，但比起最開始已經好多了。

**旁白**：李杰在地上鋪了一條毯子，選擇了靠近門口的位置躺下。這樣如果有什麼狀況，他可以第一時間反應。

**旁白**：李杰沒有多說什麼，只是閉上了眼睛。

**旁白**：小屋裡很快只剩下兩人均勻的呼吸聲。壁爐裡的火光逐漸減弱，最後只剩下微弱的餘燼。

**旁白**：窗外的紫月依然懸掛在天空中，靜靜地注視著這個詭異的世界。

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
