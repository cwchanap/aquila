# 第二十五幕：血染的祭壇

**旁白**：李杰走出小屋，朝著北方前進。

**旁白**：智子說雄大往北走了，那麼他應該還在北方的某處。

**旁白**：李杰加快腳步，穿過森林，沿著熟悉的道路前進。

**旁白**：大約半小時後，他回到了那個十字路口。

**旁白**：地上的腳印變得混亂不堪。雄大多次來回走動，腳印相互重疊，已經無法分辨哪些是來時的，哪些是去時的。

**李杰**：（內心）沒辦法靠腳印了...

**旁白**：李杰回想起從斎藤那裡得到的地圖。北方的主要路線，應該是通往「亡者峽谷」車站。

**李杰**：（內心）小林先生可能沿著主要道路一直往北走。我也走那條路，應該能找到他。

**旁白**：做出決定後，李杰沿著北方的主要道路繼續前進。

**旁白**：時間在行走中流逝。一個小時。一個半小時。

**旁白**：隨著深入，周圍的環境開始發生變化。森林的濕潤氣息逐漸消失，空氣變得乾燥。

**旁白**：樹木越來越稀疏，取而代之的是裸露的岩石和陡峭的峭壁。

**旁白**：終於，前方出現了一座月台的輪廓。

**旁白**：那是一座極其荒涼的車站。沒有建築，沒有候車大廳，只有一個光禿禿的水泥月台和一塊孤零零的站牌。

**旁白**：月台的邊緣就是峽谷，深不見底的黑暗在下方張開巨口。

**李杰**：（走近站牌）亡者峽谷...

**旁白**：紫月的光芒灑在月台上，將一切染成詭異的紫黑色。風從峽谷深處吹上來，帶著冰冷刺骨的寒意。

**旁白**：李杰環顧四周。月台上空蕩蕩的，沒有任何人影。

**李杰**：（內心）小林先生不在這裡...

**旁白**：但當他低頭看向月台的地面時，注意到了一些凌亂的腳印。

**旁白**：那些腳印在月台上來回走動，顯然有人曾經在這裡停留過。從腳印的大小和形狀判斷，很可能就是雄大留下的。

**李杰**：（蹲下仔細觀察）這些腳印...還很新鮮。

**旁白**：他順著腳印的方向，發現它們最終指向了鐵軌。

**李杰**：（內心）他沿著鐵軌走了？

**旁白**：李杰抬起頭，看向鐵軌延伸的方向。站牌上標示著下一站：終焉祭壇。

**李杰**：（內心）他去了那裡嗎？

**旁白**：李杰沒有猶豫，踏上了鐵軌。

**旁白**：鐵軌兩側是陡峭的峽谷，路基狹窄，碎石遍布。他必須小心翼翼地踩在枕木上，保持平衡。

**旁白**：隨著不斷深入，周圍的環境變得更加壓抑。空氣稀薄，每一次呼吸都讓喉嚨感到刺痛。

**旁白**：不知走了多久。也許一個小時，也許更久。

**旁白**：兩旁陡峭的峽谷岩壁終於開始變得平緩，鐵軌逐漸離開懸崖，沒入一片新的森林之中。

**旁白**：這片森林與之前的完全不同。樹木更加高大，枝幹扭曲成詭異的形狀。樹葉幾乎是黑色的，在紫月下泛著金屬般的光澤。

**旁白**：空氣變得更加壓抑，帶著一股難以形容的腐朽氣息。不是腐爛的臭味，而是一種古老的、死寂的氣息。

**李杰**：（警戒地握緊拳頭）這裡的氣氛...不對勁...

**旁白**：就在這時，前方的樹林間，一座建築的輪廓出現在視野中。

**旁白**：那是一座車站。但它比之前任何車站都更加古老，更加宏偉，也更加不祥。

**旁白**：整座建築完全由黑色岩石建成。不是普通的黑色，而是一種能吸收光線的、深邃的黑。彷彿連紫月的光芒都被它吞噬了。

**旁白**：它沒有窗戶，只有一個巨大而深邃的拱門充當入口。拱門上方刻著複雜的紋路，像是某種古老的文字，又像是扭曲的圖騰。

**旁白**：整座建築看起來不像車站，更像是一座陵墓。或者說，一座祭壇。

**李杰**：（低聲）終焉祭壇...

**旁白**：李杰小心翼翼地走近那座建築。

**旁白**：但當他距離拱門還有十幾米時，他看到了前方地面上的景象，腳步突然僵住了。

**旁白**：在建築入口前方，兩個人影倒在地上，一動不動。

**旁白**：周圍的地面上，是大片大片的暗紅色血跡。

**李杰**：（心跳加速）那是...

**旁白**：他快步衝上前去。當他看清那兩個人時，呼吸停滯了。

**旁白**：其中一個，是小林雄大。

**旁白**：他臉朝下趴在地上，背後有一個可怕的傷口。鮮血已經凝固，將他的衣服染成深黑色。

**李杰**：（跪下，伸手翻過身體）小林先生！

**旁白**：但雄大已經沒有了呼吸。他的臉色蒼白，眼睛半睜著，眼神空洞而呆滯。

**旁白**：而在雄大身旁不遠處，躺著另一個人。

**旁白**：那是一個中年男人，李杰在火車上見過他。那個看起來陰險、冷漠的男人。

**旁白**：他同樣已經死了。身下是一灘已經半凝固的血跡。

**旁白**：但最引人注目的，是他手中緊握著的東西。

**旁白**：那是一把劍。不是普通的劍，而是一把通體雪白、散發著淡淡光芒的劍。

**旁白**：劍身純淨無瑕，彷彿是用某種神聖的材質鑄造而成。在這片黑暗陰森的世界裡，它顯得格外突兀。

**李杰**：（震驚地看著那把劍）這是...什麼劍？

**旁白**：李杰的腦中一片混亂。兩個人都死了。雄大和那個中年男人。

**旁白**：他們之間發生了什麼？是互相廝殺？還是遭遇了其他威脅？

**旁白**：李杰強迫自己冷靜下來，開始檢查兩人的傷口。

**旁白**：雄大背後的傷口很深，像是被利刃貫穿。從傷口的位置判斷，應該是從背後偷襲。

**旁白**：在雄大身旁，那把生鏽的劍斷成了兩截，散落在地上。斷面參差不齊，顯然是在激烈的戰鬥中被折斷的。

**李杰**：（看著斷劍）連劍都斷了...

**旁白**：而那個中年男人，胸口有多處傷痕，顯然經歷了一場激烈的戰鬥。

**李杰**：（內心）他們...遭到了襲擊。但是誰？這裡還有其他人？

**旁白**：李杰站起身，警戒地環顧四周。

**旁白**：終焉祭壇車站一片寂靜，只有風吹過樹林的聲音。

**旁白**：但在這片死寂之中，李杰感受到了一種說不出的危險。

**旁白**：彷彿有什麼東西，正在黑暗中注視著他。
