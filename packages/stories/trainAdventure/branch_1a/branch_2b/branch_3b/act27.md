# 第二十七幕：血色交鋒

**旁白**：李杰沿著熟悉的路徑快速前進,握著白劍的手中傳來源源不斷的力量。

**旁白**：濃霧在他身邊流動,紫月的光芒為這片死寂的世界塗上詭異的色彩。

**旁白**：他和斎藤住的房子就在前方不遠處。透過霧氣,他能隱約看到那座小屋的輪廓。

**李杰**：（內心）快到了...

**旁白**：但就在他距離房子還有十幾米時,眼前的景象讓他驟然停下了腳步。

**旁白**：他的瞳孔瞬間放大,呼吸凝滯。

**旁白**：在房子前方的空地上,一個身穿黑色連帽衫的男人正站在那裡。

**旁白**：連帽衫的帽子鬆鬆地掛在腦後,露出一張年輕的臉。大約二十歲左右,五官端正,皮膚蒼白,但表情空洞,彷彿一具沒有靈魂的軀殼。

**旁白**：他的雙手從口袋中抽出,右手握著一把通體漆黑的長劍。劍身吞噬著周圍的光線,彷彿是用凝固的黑暗鑄造而成。

**旁白**：而在他面前,跪倒在地上的,是鈴木明日香。

**旁白**：明日香的臉上滿是淚痕,雙手顫抖著向前伸出,聲音嘶啞而絕望。

**鈴木明日香**：（哭泣）求求你...不要殺我...求求你...

**鈴木明日香**：（哽咽）我...我什麼都沒做...為什麼...為什麼要這樣...

**旁白**：黑衣男沒有回應。他只是靜靜地站著,黑劍在手中微微抬起。

**旁白**：李杰想要大喊,想要衝上前去。但一切都發生得太快。

**旁白**：就在下一個瞬間——

**旁白**：黑劍如同閃電般劃過空氣。

**旁白**：一道黑色的弧線在紫月下綻放。

**旁白**：鮮血飛濺。

**鈴木明日香**：（發出短促的悶哼）...！

**旁白**：明日香的雙手緩緩垂落,身體搖晃了一下,然後無力地倒在地上。

**旁白**：她的頸部出現一道深深的傷口,鮮血如泉湧般湧出,在地面上迅速擴散成一片暗紅色的血泊。

**旁白**：她的眼睛睜得很大,嘴巴微微張開,彷彿還想說些什麼。但很快,那雙眼睛就失去了光澤。

**李杰**：（怒吼）住手！

**旁白**：李杰的怒吼劃破寂靜。他握緊白劍,如同離弦之箭般衝向黑衣男。

**旁白**：黑衣男緩緩轉過身,帽簷下的陰影中,一雙冰冷的眼睛直視著李杰。

**旁白**：沒有驚訝,沒有慌張,只有一片死寂般的平靜。

**旁白**：李杰衝到他面前,白劍在空中劃出一道凌厲的弧線,直取對方的頸部。

**旁白**：黑衣男抬起黑劍,輕鬆地擋下了這一擊。

**旁白**：「鏘」——

**旁白**：兩把劍相撞的瞬間,爆發出震耳欲聾的金屬撞擊聲。火花在劍刃交接處綻放,照亮了兩人的臉。

**旁白**：李杰感受到一股巨大的反震力從劍上傳來,但白劍賦予他的力量讓他穩穩地站住了。

**李杰**：（咬緊牙關）你這個混蛋...！

**旁白**：他再次揮劍,這次瞄準對方的胸口。黑衣男向後一退,劍刃擦過他的衣襟,帶起一縷布料的碎片。

**旁白**：黑衣男沒有說話,只是反手一劈。黑劍帶著破空聲斬向李杰的側腹。

**旁白**：李杰側身閃避,劍刃貼著他的衣服掠過,在布料上劃出一道口子。

**旁白**：兩人的身影在霧中交錯,劍光閃爍。白色與黑色的劍刃不斷碰撞,每一次撞擊都爆發出刺耳的聲響和耀眼的火花。

**旁白**：李杰的攻勢如同狂風暴雨,一劍接著一劍,毫不停歇。白劍在他手中彷彿活了過來,每一次揮動都精準而凌厲。

**旁白**：但黑衣男同樣不弱。他的動作簡潔而高效,沒有任何多餘的招式。每一次格擋都恰到好處,每一次反擊都直指要害。

**旁白**：兩把劍再次碰撞在一起,僵持在半空中。兩人的臉只有幾十公分的距離。

**李杰**：（憤怒地）為什麼...為什麼要殺她們！

**旁白**：黑衣男依然沒有回答。他突然用力一推,將李杰震退幾步。

**旁白**：隨後他向前踏出一步,黑劍從上而下狠狠劈落。

**旁白**：李杰舉劍格擋,劍刃承受了這沉重的一擊。巨大的力道讓他的手臂微微發麻。

**旁白**：但他沒有退縮。他用力一扭,將黑劍挑開,然後趁機向前一刺。

**旁白**：白劍的劍尖直指黑衣男的心臟。

**旁白**：黑衣男反應極快,身體向側邊一扭。劍尖刺穿了他的衣服,在他的側腹劃出一道傷口。

**旁白**：鮮血從傷口滲出,染紅了黑色的布料。

**旁白**：但黑衣男彷彿感覺不到疼痛。他反手一揮,黑劍橫掃向李杰的頸部。

**旁白**：李杰低頭閃過,黑劍從他頭頂掠過,削斷了幾縷頭髮。

**旁白**：戰鬥進入了白熱化。兩人的身影在霧中快速移動,劍光閃爍不停。

**旁白**：李杰的攻勢越來越猛烈。白劍賦予他的力量讓他感覺不到疲憊,每一次揮劍都帶著足以開山裂石的威力。

**旁白**：終於,在一次激烈的碰撞後,李杰找到了破綻。

**旁白**：黑衣男在格擋他的一記橫斬時,身體的重心稍稍偏移。

**旁白**：就是這個瞬間。

**李杰**：（怒吼）結束了！

**旁白**：李杰全力揮出白劍,劍身帶著刺眼的白光,重重地斬在黑劍的劍身上。

**旁白**：「咔嚓」——

**旁白**：一聲清脆的斷裂聲響起。

**旁白**：黑劍從中間斷成兩截,斷面參差不齊。斷裂的劍身在空中旋轉著飛出,插進遠處的泥土裡。

**旁白**：黑衣男愣了一瞬,低頭看著手中只剩下半截的劍柄。

**旁白**：李杰沒有給他任何機會。

**旁白**：白劍繼續前進,劍刃劃過黑衣男的胸口,留下一道深深的傷口。

**旁白**：鮮血噴湧而出。

**旁白**：黑衣男踉蹌後退幾步,手中的斷劍掉落在地。他捂住胸口,鮮血從指縫間不斷湧出。

**旁白**：他的嘴角微微動了動,彷彿想說些什麼。但最終,什麼聲音都沒有發出。

**旁白**：他的身體搖晃了一下,然後沉重地倒在地上,再也沒有動靜。

**旁白**：李杰大口喘著氣,握劍的手微微顫抖。

**李杰**：（喘息）...結束了...

**旁白**：但就在這時,他感覺到左臂傳來一陣刺痛。

**旁白**：他低頭一看,才發現自己的左臂上有一道深深的傷口。鮮血正順著手臂流下,滴落在地上。

**李杰**：（皺眉）什麼時候...

**旁白**：就在剛才激烈的戰鬥中,黑劍曾經劃過他的手臂。當時因為腎上腺素的作用,他沒有感覺到疼痛。

**旁白**：但現在,疼痛開始湧現。傷口很深,血流不止。

**旁白**：就在這時,一股溫暖的感覺從手中的白劍傳來。

**旁白**：白劍散發出柔和的光芒,光芒沿著他的手臂向上蔓延,籠罩住傷口。

**李杰**：（驚訝地）這是...

**旁白**：溫暖的光芒包裹著傷口,李杰能感覺到一股舒適的暖流在傷口處流動。

**旁白**：疼痛逐漸消失。傷口以肉眼可見的速度癒合。

**旁白**：十幾秒後,原本深可見骨的傷口已經完全癒合,只留下一道淺淺的粉紅色痕跡。

**李杰**：（震驚地看著手臂）它...治癒了我...

**旁白**：他抬起頭,看著手中的白劍。劍身依然散發著柔和的白光,彷彿在回應他的驚訝。

**旁白**：就在這時,李杰注意到,手中白劍的震動...停止了。

**旁白**：從他拾起這把劍開始,劍身就一直在輕微地震動,指引著他前進的方向。

**旁白**：但現在,那種震動完全消失了。劍安靜地躺在他手中,不再有任何動靜。

**李杰**：（內心）震動停止了...難道說...

**李杰**：（看向地上碎裂的黑劍）白劍是在對黑劍做出反應？

**李杰**：（內心）現在黑劍被毀了,所以白劍也不再震動了...

**旁白**：李杰深吸一口氣,將白劍收起。現在不是思考這些的時候。

**旁白**：他快步走向明日香倒下的地方。

**旁白**：明日香面朝下躺在血泊中,身體已經完全冰冷。

**李杰**：（蹲下,伸手輕輕翻過她的身體）明日香...

**旁白**：明日香的眼睛半睜著,眼神空洞而呆滯。她的嘴角還掛著淚痕,臉上殘留著死前的恐懼。

**旁白**：李杰伸手探了探她的鼻息。沒有呼吸。

**旁白**：他又摸了摸她的頸動脈。沒有脈搏。

**李杰**：（低聲）對不起...我來晚了...

**旁白**：他緩緩站起身,環顧四周。

**旁白**：就在不遠處,大約五六米的地方,他看到了另一個身影。

**旁白**：那是清水恵美。

**旁白**：她倒在地上,同樣是頸部有一道深深的傷口。鮮血已經凝固,將她的衣領染成暗紅色。

**李杰**：（衝過去）恵美！

**旁白**：李杰跑到恵美身邊,蹲下查看。

**旁白**：恵美的臉色蒼白,雙眼緊閉,毫無生氣。她的手還保持著向前伸出的姿勢,彷彿在生命的最後一刻,還想抓住什麼。

**旁白**：李杰顫抖著手,探向她的鼻息。

**旁白**：同樣,沒有呼吸。

**李杰**：（咬緊牙關）該死...該死！

**旁白**：兩條年輕的生命,就這樣在他眼前逝去。

**李杰**：（內心）如果我早一點...如果我再快一點...

**旁白**：房子的門緊閉著,沒有任何動靜。周圍的霧氣依然濃厚,看不到其他人影。

**旁白**：他的聲音在霧中迴盪,但沒有任何回應。

**旁白**：寂靜。死一般的寂靜。

**李杰**：（內心）田中和斎藤...他們去哪裡了？還活著嗎？

**旁白**：那是樹枝折斷的聲音。

**旁白**：下一秒,一個人影從灌木叢中搖搖晃晃地站了起來。

**旁白**：那是斎藤蓮。
