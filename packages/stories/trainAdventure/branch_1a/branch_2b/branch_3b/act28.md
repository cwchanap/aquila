# 第二十八幕：倖存者的證詞

**旁白**：李杰扶著斎藤，讓他靠在自己身上。斎藤的身體依然在微微顫抖。

**旁白**：過了好一會兒，斎藤的呼吸才稍微平穩下來。

**李杰**：（低聲）斎藤，告訴我...這裡到底發生了什麼事？

**旁白**：斎藤抬起頭，推了推鼻樑上的眼鏡。鏡片上的灰塵讓他的視線有些模糊。

**斎藤蓮**：（聲音還在發抖）我...我一直待在屋子裡。你離開之後，我就一個人在裡面。

**斎藤蓮**：（深吸一口氣）然後...突然聽到外面有聲音。腳步聲，還有...喘息聲。

**李杰**：什麼時候的事？

**斎藤蓮**：我...我也不確定。這裡又沒有時鐘...

**斎藤蓮**：（想了想）應該是你離開後沒多久吧？可能一兩個小時？我真的不知道...

**斎藤蓮**：我透過窗戶往外看，看到...看到鈴木桑在跑。她看起來很慌張，一直在跑。

**斎藤蓮**：（停頓）還有那個...火車上那個文靜的女孩。她也在跑。

**李杰**：你是說清水？

**斎藤蓮**：對...清水。她們兩個一起跑向這座房子。

**斎藤蓮**：（顫抖）我...我當時就覺得不對勁。她們看起來...像是在逃命。

**旁白**：斎藤的聲音越來越小，眼神閃爍著恐懼。

**斎藤蓮**：所以我...我就從後門溜出去了。

**李杰**：（皺眉）你沒有幫她們？

**斎藤蓮**：（激動地）我能做什麼？！我連發生了什麼事都不知道！

**斎藤蓮**：（聲音發抖）我只是...我只是覺得應該先躲起來。所以我跑到那邊的灌木叢...然後就一直躲在那裡。

**李杰**：你看到了什麼？

**斎藤蓮**：（搖頭）我什麼都沒看到。我不敢看。我把頭壓得低低的，一動也不敢動。

**斎藤蓮**：（停頓）但是...我聽到了聲音。

**李杰**：什麼聲音？

**斎藤蓮**：（深吸一口氣）恵美的聲音。她在喊一個名字...凌墨。

**旁白**：李杰的眉頭皺得更緊了。

**李杰**：凌墨？

**斎藤蓮**：（點頭）她一直在喊那個名字。說什麼「凌墨，你在做什麼」、「求求你恢復正常」之類的...

**斎藤蓮**：（顫抖）她的聲音聽起來很絕望...很害怕...

**旁白**：斎藤的聲音哽咽了，他深吸一口氣，努力讓自己冷靜下來。

**斎藤蓮**：然後...然後她的聲音就停了。

**李杰**：（沉默片刻）然後呢？

**斎藤蓮**：我聽到了一些聲音...不知道是什麼。可能是...可能是她倒下的聲音。

**斎藤蓮**：（停頓）然後...我又聽到鈴木桑的聲音。她在求饒。一直在哭著求饒。

**斎藤蓮**：（低聲）她說「求求你不要殺我」、「我什麼都沒做」...

**旁白**：斎藤的手緊緊抓著李杰的衣服，指節發白。

**斎藤蓮**：然後...她的聲音也停了。就這樣...突然停了...

**旁白**：李杰沉默地看著斎藤。他能想像那個場景——兩個女孩接連被殺，而斎藤就躲在幾米外的灌木叢裡。

**李杰**：之後呢？

**斎藤蓮**：之後...我聽到你的聲音。你在喊「住手」。

**斎藤蓮**：然後就是打鬥的聲音。金屬碰撞的聲音，很大聲，持續了很久...

**斎藤蓮**：（小聲）我還以為...我還以為你也會死...

**旁白**：李杰沒有回應。他的腦中快速整理著這些資訊。

**李杰**：（內心）凌墨...恵美認識那個黑衣男。而且聽起來，她似乎想喚醒他。

**李杰**：（內心）那個黑衣男...難道原本不是這個樣子？

**李杰**：（看向斎藤）那個叫凌墨的人，很可能從銀月湖一路追殺她們到這裡。

**斎藤蓮**：（點頭）應該是...她們是從那個方向跑過來的。

**李杰**：你有看到田中嗎？

**旁白**：斎藤搖了搖頭。

**斎藤蓮**：沒有。我只看到她們兩個。

**李杰**：（皺眉）田中...他去哪裡了？

**旁白**：就在這時，斎藤的目光落在李杰手中的白劍上。

**斎藤蓮**：（疑惑地）那把劍...你哪裡來的？

**李杰**：撿到的。

**斎藤蓮**：（難以置信）撿到的？這種劍...怎麼可能隨便撿到？

**斎藤蓮**：（盯著白劍）而且...你剛才和那個怪物戰鬥的時候...

**斎藤蓮**：（驚訝地）我沒想到你這麼會打架。那個怪物看起來很厲害，但你居然...

**李杰**：（平靜地）我在終焉祭壇找到小林先生的時候，他身邊還有另一具屍體。一個中年男人。

**李杰**：這把劍就在那個人手裡。

**斎藤蓮**：（停頓片刻）...中年男人？

**李杰**：嗯。我不認識他。

**斎藤蓮**：（點頭，語氣平淡）這樣啊...

**旁白**：斎藤低下頭，推了推眼鏡。那個動作看起來很自然，但李杰總覺得哪裡不太對勁。

**旁白**：也許只是錯覺吧。

**斎藤蓮**：（抬起頭）這個世界真的...到處都是死人...

**旁白**：李杰站起身，看向明日香和恵美的屍體。

**李杰**：我們不能就這樣把她們留在這裡。

**斎藤蓮**：（抬頭）什麼？

**李杰**：（轉身看向斎藤）幫我一起埋葬她們。就在附近找個地方。

**斎藤蓮**：（愣了一下）你...你是說現在？

**李杰**：對。現在。

**斎藤蓮**：（猶豫）可是...我們不是應該趕快離開這裡嗎？萬一還有其他危險...

**李杰**：（堅定地）她們不應該就這樣暴屍荒野。

**旁白**：斎藤看著李杰的眼神，知道這不是請求，而是命令。

**斎藤蓮**：（嘆氣）好吧...

---

**旁白**：兩人花了將近一個小時，在房子附近找到一片較為鬆軟的土地。

**旁白**：李杰用白劍挖掘，劍刃輕鬆地切開泥土，就像切開豆腐一樣。

**旁白**：斎藤則在一旁用樹枝和石塊輔助，雖然動作笨拙，但至少在幫忙。

**旁白**：兩個淺坑很快就挖好了。

**旁白**：李杰小心地將明日香的遺體放進其中一個坑裡，然後開始填土。

**旁白**：斎藤站在一旁，看著李杰的動作，沒有說話。

**旁白**：當最後一捧土蓋上時，李杰用石塊在上方堆成一個簡單的標記。

**旁白**：隨後，他對恵美做了同樣的事。

**旁白**：最後，李杰走向凌墨的屍體。

**斎藤蓮**：（驚訝）你...你要埋那個怪物？

**李杰**：（平靜地）不管他做了什麼，他也是人。

**斎藤蓮**：（不滿地）但是他殺了...

**李杰**：（打斷他）我知道。

**旁白**：李杰將凌墨的遺體也埋葬了，但距離明日香和恵美的墳墓稍遠一些。

**旁白**：他沒有為凌墨堆石塊標記，只是簡單地填上了土。

**李杰**：（站在三座墳前，低頭）希望你們能安息。

**旁白**：斎藤看著李杰，張了張嘴，但最終什麼都沒說。

**旁白**：李杰轉過身，擦了擦手上的泥土。

**李杰**：我要去銀月湖。

**斎藤蓮**：（立刻）我跟你一起去！

**李杰**：（看著他）你確定？

**斎藤蓮**：（堅定地點頭）我...我不想一個人待在這裡。那個怪物雖然死了，但誰知道還會不會有其他危險...

**斎藤蓮**：（抓住李杰的手臂）我要跟著你。你有那把劍，比較安全。

**李杰**：（沉默片刻）...隨你。

**旁白**：斎藤鬆了口氣。

**斎藤蓮**：那我們什麼時候出發？

**李杰**：現在。我要去銀月湖看看田中在不在那裡。

**李杰**：（看向恵美的墳墓）還有...我想知道那裡到底發生了什麼事。

**旁白**：李杰握緊手中的白劍，轉身朝著南方走去。

**旁白**：斎藤緊緊跟在他身後，不敢落後半步。

**旁白**：兩人的身影消失在濃霧之中，留下三座孤獨的墳墓，在紫月的照耀下顯得格外淒涼。

**旁白**：而在銀月湖的方向，還有更多未解的謎團，等待著他們。
