# 第三十四幕：通道盡頭的房間

**旁白**：李杰舉著手電筒，走在前方。斎藤緊跟在他身後，兩人的腳步聲在狹窄的通道中迴盪。

**旁白**：通道的牆壁依然是粗糙的岩石，腳下的地面坑坑窪窪，佈滿碎石。

**旁白**：手電筒的光束在黑暗中劃出一條明亮的路徑，但前方依然看不到盡頭。

**斎藤蓮**：（顫抖著）這...這條路到底通往哪裡...

**旁白**：斎藤的聲音在身後響起，帶著明顯的恐懼。

**李杰**：（沒有回頭）不知道。

**斎藤蓮**：（更加不安）那...那我們這樣走下去真的沒問題嗎？萬一...萬一這條路是死路怎麼辦？

**李杰**：（平靜地）那就回頭。

**斎藤蓮**：（急切地）可是萬一回不去了怎麼辦？萬一這條路會塌陷？萬一有什麼東西跟在我們後面？

**旁白**：李杰的眉頭微微皺起。斎藤不停的抱怨讓他感到煩躁。

**李杰**：（嘆氣）...你能不能安靜一點。

**斎藤蓮**：（委屈地）我...我只是很害怕...這裡太可怕了...伸手不見五指...

**斎藤蓮**：（繼續）如果不是你有手電筒...我根本不敢走進這種地方...

**旁白**：李杰沒有回應。他只是繼續向前走，手電筒穩穩地照著前方。

**旁白**：但其實，李杰心裡也很害怕。

**李杰**：（內心）這條通道...不知道通往哪裡...也不知道有多長...

**李杰**：（內心）如果手電筒沒電了怎麼辦？如果前面真的是死路...

**旁白**：但他沒有表現出來。他不想讓斎藤看出他的恐懼。

**旁白**：而且...至少有斎藤在身邊。雖然他不停地抱怨，但總比一個人走要好。

**斎藤蓮**：（又開口）李杰...你說...我們真的能找到出去的路嗎？

**李杰**：（簡短地）不知道。

**斎藤蓮**：如果找不到怎麼辦...

**李杰**：（打斷他）那就繼續找。

**旁白**：斎藤張了張嘴，但最終什麼都沒說。他只是更緊地跟在李杰身後。

---

**旁白**：兩人就這樣沉默地走著。不知道走了多久。

**旁白**：通道依然是直直向前，沒有岔路，沒有盡頭。只有無盡的黑暗和粗糙的岩壁。

**旁白**：李杰的腿開始感到疲憊。他估計他們至少走了二十分鐘以上。

**旁白**：就在他開始懷疑這條路是否真的有盡頭時——

**旁白**：前方出現了一道光。

**李杰**：（停下腳步）...那是什麼？

**斎藤蓮**：（驚喜地）有光！前面有光！

**旁白**：兩人加快腳步，朝著那道光前進。

**旁白**：很快，他們來到了通道的盡頭。

---

**旁白**：出現在他們眼前的，是一個房間。

**旁白**：不，更準確地說，是一個金屬房間。

**旁白**：整個房間的牆壁都是銀灰色的金屬製成，表面光滑而冰冷。天花板上嵌著幾盞LED燈，發出明亮的白光。

**旁白**：這和剛才原始粗糙的洞穴形成了強烈的對比。

**斎藤蓮**：（驚訝地）這...這是什麼地方...

**旁白**：李杰慢慢走進房間，環顧四周。

**旁白**：房間不大，大約只有三、四坪。牆壁上沒有任何裝飾，只有冰冷的金屬。

**旁白**：而在房間的另一側，有一扇金屬門。

**李杰**：（走向門）這裡有門...

**旁白**：李杰來到門前。那是一扇厚重的金屬門，表面同樣光滑而冰冷。

**旁白**：門上有一個圓形的把手。李杰伸手握住把手，試著轉動。

**旁白**：把手紋絲不動。

**李杰**：（用力）...嗯？

**旁白**：李杰加大力道，但把手依然沒有任何反應。它被牢牢地鎖住了。

**斎藤蓮**：（走上前）鎖住了嗎？

**李杰**：（點頭）嗯。

**斎藤蓮**：（失望地）該死...好不容易走到這裡...

**旁白**：李杰退後一步，仔細觀察這扇門。

**旁白**：門框和牆壁之間有一些縫隙——不，那不是普通的縫隙。那是幾個小洞。

**旁白**：洞的大小不一，有的只有手指粗細，有的則有拳頭大小。它們不規則地分佈在門框周圍的牆壁上。

**李杰**：（蹲下，看著洞）這些洞...

**旁白**：李杰舉起手電筒，從其中一個較大的洞口往裡照。

**旁白**：光束穿過洞口，照亮了門後的空間。

**旁白**：那是另一個房間。但和現在這個空蕩蕩的金屬房間不同，那個房間裡擺滿了東西。

**旁白**：李杰看到了一張桌子，上面堆著一些文件。一把椅子靠在桌邊。

**旁白**：房間角落有一個書架，上面擺滿了書籍。旁邊是幾個櫃子。

**旁白**：還有一個微波爐，放在一個小櫃子上。以及...似乎還有更多東西，但因為角度的關係，看不太清楚。

**斎藤蓮**：（也湊過來看）裡面...裡面好像是個辦公室？

**李杰**：（移動手電筒，從不同角度查看）...不只是辦公室。

**旁白**：李杰換了幾個洞口，從不同角度觀察房間內部。

**旁白**：他看到了更多細節——牆上掛著一個時鐘，桌上有一台筆記型電腦，書架旁邊還有一個小冰箱。

**李杰**：（內心）這看起來...像是有人在這裡長期生活過...

**旁白**：李杰移動到門邊的一個洞口。這個洞位置較低，大約在膝蓋的高度。

**旁白**：他從這個角度往裡看，視線終於能夠看到門的內側。

**旁白**：在門的內側，有一個把手。

**李杰**：（低聲）...看到了。

**斎藤蓮**：（疑惑）什麼？

**李杰**：（指著洞口）門的內側有把手。如果能從這裡伸進去轉動把手...應該就能打開門。

**旁白**：但問題是，那個把手距離洞口太遠了。李杰估計至少有一米五以上的距離。

**旁白**：他試著把手伸進洞裡，但只能伸進去一半，根本碰不到把手。

**李杰**：（站起身，思考）需要某種...能夠延伸的東西...

**李杰**：（轉向斎藤）你有繩子嗎？

**斎藤蓮**：（愣了一下）繩子？

**李杰**：對。或是任何長條狀、能夠勾住東西的東西。

**斎藤蓮**：（沒好氣地）我為什麼會有那種東西？我又不是哆啦A夢！

**旁白**：李杰沒有回應斎藤的抱怨。他開始檢查自己的口袋。

**旁白**：他掏出手機——沒電。錢包——沒用。車票——更沒用。

**旁白**：然後，他的手指觸碰到幾條柔軟的線狀物。

**李杰**：（拿出來）...USB線？

**旁白**：李杰掏出了三條USB充電線。它們原本是用來充手機的，現在一直放在他口袋裡。

**旁白**：他把三條線連接在一起，拉直測量長度。

**旁白**：大約一米左右。

**李杰**：（皺眉）...還是太短。

**旁白**：即使把三條線全部連接起來，長度也不夠從洞口伸到門把那裡。

**斎藤蓮**：（看著他）怎麼樣？可以嗎？

**李杰**：（搖頭）不夠長。

**斎藤蓮**：（失望地）那...那怎麼辦...

**旁白**：李杰站在門前，握著那三條USB線，陷入沉思。

**旁白**：他的視線在門、洞口、以及門內的把手之間來回移動。

**旁白**：必須想辦法打開這扇門。但要怎麼做？

**旁白**：他身上沒有其他工具，斎藤也沒有。這個金屬房間裡除了他們兩個人之外，什麼都沒有。

**李杰**：（內心）一定有辦法...一定有什麼辦法...

**旁白**：李杰再次蹲下，從洞口看向門內的把手。

**旁白**：距離太遠。USB線太短。而且就算線夠長，要怎麼準確地勾住把手並轉動它？

**旁白**：這幾乎是不可能的任務。

**斎藤蓮**：（焦慮地）李杰...你想到辦法了嗎？

**旁白**：李杰沒有回答。他只是靜靜地站在那裡，看著那扇緊閉的金屬門。

**旁白**：他的大腦飛速運轉，試圖找出任何可能的解決方案。

**旁白**：但現在，他什麼都想不出來。
