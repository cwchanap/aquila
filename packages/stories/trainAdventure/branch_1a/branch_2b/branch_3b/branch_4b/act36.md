# 第三十六幕：樂譜的秘密

**旁白**：明亮的燈光下,李杰終於看清了這個房間的全貌。

**旁白**：這是一個小房間,大約只有五、六坪大小。但裡面塞滿了各種物品。

**旁白**：靠牆放著一張簡單的木桌和一把椅子。桌上有一台筆記型電腦,螢幕是黑的。

**旁白**：牆邊有一個高大的書架,上面擺滿了各式各樣的書籍。書架旁邊是一架老式的直立鋼琴,黑色的琴身泛著暗淡的光澤。

**旁白**：房間角落有一張單人床,床單整齊地鋪著。床旁邊是幾個木製櫃子。

**旁白**：另一側有個小廚房區域,配備著簡單的爐具和流理台。地上放著一個金屬水桶。

**旁白**：牆上掛著一台老式收音機,旋鈕和天線都完好無損。

**旁白**：而在房間的另一側,有一扇門。

**李杰**：（走向門）...

**旁白**：李杰來到門前,仔細觀察。

**旁白**：這扇門和剛才那扇金屬門不同。它是木製的,表面光滑,看起來很普通。

**旁白**：但門上沒有門把。

**旁白**：門把原本應該在的位置,只有一個很小的圓形洞。洞只有螺絲孔那麼大,邊緣很光滑。

**李杰**：（皺眉）沒有門把...

**旁白**：洞太小了,連手指都伸不進去。李杰試著用力推門,但門紋絲不動,顯然是鎖住的。

**斎藤蓮**：（在房間裡四處張望）這到底是什麼鬼地方...

**旁白**：李杰離開門,走向那台筆記型電腦。

**旁白**：他打開螢幕。螢幕亮了,但立刻顯示一個密碼輸入畫面。

**李杰**：（低聲）被鎖住了...

**旁白**：李杰試著按了幾個常見的密碼,但全部都錯誤。

**旁白**：他環顧四周,眉頭越皺越緊。

**李杰**：（內心）這個房間...到底是怎麼回事？

**李杰**：（內心）為什麼會有這些東西？而且看起來...好像有人在這裡生活過...

**斎藤蓮**：（突然開口）我知道了。

**旁白**：李杰轉頭看向斎藤。

**斎藤蓮**：（指著周圍）這是密室逃脫。

**李杰**：（疑惑）什麼？

**斎藤蓮**：（解釋）密室逃脫啊！你沒玩過嗎？就是那種...把人關在房間裡,然後要解謎才能出去的遊戲。

**斎藤蓮**：（繼續）你看,門沒有把手,電腦被鎖住,房間裡擺了一堆東西...這根本就是典型的密室逃脫設定。

**旁白**：李杰沉默了片刻。

**旁白**：雖然他不太相信這只是個「遊戲」,但斎藤說的也不無道理。

**李杰**：（點頭）...那就找找看吧。

**旁白**：兩人開始在房間裡搜索。

---

**旁白**：李杰首先走向書架。

**旁白**：書架有五層,每層都擺滿了書。有小說、工具書、字典...各種類型都有。

**旁白**：他隨手拿起幾本書翻看,但沒有發現什麼特別的。

**旁白**：斎藤則在檢查櫃子。他打開每一個抽屜,翻找裡面的東西。

**斎藤蓮**：（抱怨）衣服...毛巾...餐具...什麼有用的東西都沒有...

**旁白**：李杰走向鋼琴。

**旁白**：他伸手按了幾個琴鍵。「叮」「咚」——琴聲清脆,音準也沒問題。

**旁白**：就在這時,他注意到鋼琴頂上放著一張對折的紙。

**李杰**：（拿起紙）這是...

**旁白**：李杰展開那張紙。

**旁白**：紙上畫著一段簡單的樂譜。五條橫線組成的五線譜上,有四個音符。

**旁白**：但這段樂譜看起來很奇怪。

**旁白**：沒有譜號,沒有小節線,沒有任何音樂記號。就只有五條線,和四個簡單的音符。

**旁白**：四個音符的位置高低不一,第一個在第二線上,第二個在第四線上,第三個在第三線上,第四個在第一線上。

**李杰**：（困惑）這是什麼...

**旁白**：李杰不懂音樂,但他知道正常的樂譜不會長這樣。

**旁白**：他看著那四個音符,試圖理解它們的意義。

**斎藤蓮**：（走過來）你找到什麼了？

**李杰**：（遞給他）一張樂譜。但看起來很怪。

**旁白**：斎藤接過紙,看了看。

**斎藤蓮**：（皺眉）這什麼鬼...連譜號都沒有...

**斎藤蓮**：（把紙還給李杰）我不懂這個。

**旁白**：李杰拿著紙,在房間裡來回走動,思考這張樂譜的意義。

**旁白**：他的視線不經意地掃過書架。

**旁白**：等等。

**旁白**：李杰停下腳步,走近書架,仔細觀察。

**旁白**：在書架的側邊,靠近牆壁的地方,有一些淡淡的鉛筆痕跡。

**旁白**：五條水平的鉛筆線,每條線都對應著書架的一層。

**李杰**：（低聲）這是...

**旁白**：他拿著樂譜,對比書架側邊的鉛筆線。

**旁白**：五條橫線。

**旁白**：和樂譜上的五線譜完全一樣。

**李杰**：（恍然大悟）...這不是給鋼琴的樂譜。

**斎藤蓮**：（疑惑）什麼？

**李杰**：（指著書架）這張樂譜...是書架的地圖。

**旁白**：斎藤走過來,看著書架側邊的鉛筆線,又看了看李杰手中的樂譜。

**斎藤蓮**：（恍然）你是說...音符代表書架上的書？

**李杰**：（點頭）對。音符的高低代表哪一層,位置代表書在那層的大概位置。

**旁白**：李杰再次看著樂譜。

**旁白**：第一個音符在第二線上,偏左。

**旁白**：他走到書架前,看著第二層,從左邊開始數。

**旁白**：大約在左邊四分之一的位置,有一本深藍色的書。

**李杰**：（拿起那本書）這本...

**旁白**：他翻開書,仔細檢查。書名是《基礎物理學》。

**旁白**：他看著書背,然後注意到一個細節。

**旁白**：書名的其中一個字——「礎」——顏色稍微深一點。就像是墨水多印了一層。

**李杰**：（低聲）「礎」...不對,英文的話...

**旁白**：他看向書背上的英文標題"FOUNDATION"。字母S的顏色明顯比其他字母深。

**李杰**：（內心）S...

**旁白**：李杰繼續按照樂譜上的音符尋找。

**旁白**：第二個音符在第四線上,偏中間。

**旁白**：他找到第四層書架,中間位置。一本紅色的書——《經濟學原理》。

**旁白**：書背上,"ECONOMICS"的字母E顏色較深。

**李杰**：（內心）E...

**旁白**：第三個音符在第三線上,偏右。

**旁白**：第三層書架,右側。一本綠色的書——《設計思維》。

**旁白**：書背上,"DESIGN"的字母D顏色較深。

**李杰**：（內心）D...

**旁白**：第四個音符在第一線上,靠左。

**旁白**：第一層書架,左側。一本灰色的書——《生物學入門》。

**旁白**：書背上,"BIOLOGY"的字母B顏色較深。

**李杰**：（內心）B...

**旁白**：李杰把四個字母連起來。

**李杰**：（低聲）S、E、D、B...SEDB？

**旁白**：這不是任何有意義的英文單字。

**斎藤蓮**：（在旁邊）找到什麼了嗎？

**李杰**：（搖頭）四個字母。但拼不出任何字。

**斎藤蓮**：（失望）那就是錯的囉...

**李杰**：（思考）不...一定有其他線索...

**旁白**：李杰環顧房間。他的視線落在牆上的收音機。

**李杰**：（走向收音機）試試這個。

**旁白**：李杰打開收音機。

**旁白**：「滋滋滋——」

**旁白**：收音機發出刺耳的雜訊。

**旁白**：李杰開始慢慢轉動頻率旋鈕。

**旁白**：「滋——滋滋——滋——」

**旁白**：大部分頻率都是雜訊。但當他轉到某個特定頻率時——

**旁白**：雜訊突然消失了。

**旁白**：然後,一個平靜的聲音從收音機中傳出。

**收音機**：書背會說謊。書頁說真話。看標記下方,不是標記本身。

**旁白**：那個聲音重複了三遍,然後又變回雜訊。

**斎藤蓮**：（驚訝）什麼？剛才...剛才有聲音！

**李杰**：（重複）書背會說謊...書頁說真話...

**旁白**：李杰思考著這句話的意義。

**旁白**：書背上的字母是錯的？要看書頁？

**旁白**：但要看書頁的哪裡？「標記下方」是什麼意思？

**旁白**：李杰正思考時,斎藤突然開口。

**斎藤蓮**：（指著水桶）欸,這個水桶底部有字！

**旁白**：李杰走過去。斎藤把水桶翻過來。

**旁白**：在水桶的底部,用利器刻著幾個字：

**旁白**：「反轉他們所說的」

**李杰**：（低聲）反轉...

**旁白**：李杰突然明白了。

**李杰**：（快步走向書架）我知道了！

**旁白**：他拿起第一本書——《基礎物理學》。翻開書,找到書背上那個深色的「S」對應的位置。

**旁白**：在那一頁,「S」所在的那一行,正好有一個句子。

**旁白**：李杰看著那個深色「S」正下方的那一行文字。在「S」下方的字母是...B。

**李杰**：（內心）B...

**旁白**：他繼續檢查其他三本書。

**旁白**：第二本書,深色「E」下方的字母是...E。

**旁白**：第三本書,深色「D」下方的字母是...D。

**旁白**：第四本書,深色「B」下方的字母...沒有。那一行是空白。

**李杰**：（皺眉）等等...

**旁白**：他重新檢查。仔細看那個深色「B」的位置。

**旁白**：不是看正下方的那一行文字。而是看...這個「B」本身下方的那一個字母。

**旁白**：在書背英文標題"BIOLOGY"中,「B」的下一個字母是...沒有。「B」是第一個字母。

**旁白**：等等,「反轉」...

**李杰**：（恍然）是倒序讀！

**旁白**：李杰重新排列字母。

**旁白**：原本是S-E-D-B,倒過來讀就是B-E-D。

**李杰**：（低聲）BED...床！

**斎藤蓮**：（驚訝）床？

**旁白**：李杰立刻衝向房間角落的那張床。

**旁白**：他仔細檢查床架。床架是木製的,看起來很普通。

**旁白**：但當他檢查床底的板條時,他發現了異常。

**旁白**：其中一根板條比其他的短。

**李杰**：（抓住那根板條）這根...

**旁白**：他用力一拉。板條被拉了出來。

**旁白**：在板條原本的位置,露出了一個空腔。

**旁白**：空腔裡,放著兩個門把。

**旁白**：一個是金屬製的,表面光滑,泛著銀色的光澤。

**旁白**：另一個是木製的,深褐色,表面有細微的蛀蟲洞痕跡。

**斎藤蓮**：（興奮地）找到了！門把！

**旁白**：但李杰沒有立刻拿起門把。他注意到空腔內壁上還刻著一個字。

**旁白**：那是一個英文單字,用利器刻得很深：

**旁白**：「FOUNDATION」

**李杰**：（低聲）Foundation...基礎？

**旁白**：他不明白這個字的意義。但他把它記在心裡。

**斎藤蓮**：（伸手去拿）管它什麼字,我們先把門打開吧！

**旁白**：斎藤拿起那個木製門把,快步走向門口。

**李杰**：（皺眉）等等...

**旁白**：但斎藤已經把木門把塞進門上的洞裡了。

**旁白**：「喀嗒」——門把裝上了,契合得很完美。

**斎藤蓮**：（用力轉動）哈！成功了！

**旁白**：但門沒有打開。

**斎藤蓮**：（用力）...嗯？

**旁白**：斎藤繼續轉動門把,甚至用身體撞門。但門依然紋絲不動。

**斎藤蓮**：（焦急）怎麼回事？為什麼打不開？

**旁白**：李杰走上前,仔細觀察門和門把。

**旁白**：他注意到門板上,在門把孔的旁邊,有一個很小的圖案。

**旁白**：那是一個奇怪的符號,像是某種裝飾。但仔細看,那個符號的紋路...和木門把上的蛀蟲洞圖案很相似。

**李杰**：（內心）「書背會說謊」...

**李杰**：（對斎藤）換另一個。

**斎藤蓮**：（不解）什麼？

**李杰**：（指著木門把）這個是錯的。換金屬的那個。

**斎藤蓮**：（不情願）可是這個明明裝得上...

**李杰**：（堅定）換。

**旁白**：斎藤嘆了口氣,拔下木門把,回去拿金屬門把。

**旁白**：他把金屬門把裝上。

**旁白**：「喀嗒」——同樣契合得很好。

**斎藤蓮**：（轉動門把）...

**旁白**：這次,門把順利轉動了。

**旁白**：「咔」——

**旁白**：門鎖打開的聲音。

**斎藤蓮**：（推門）開了！

**旁白**：門緩緩打開。
