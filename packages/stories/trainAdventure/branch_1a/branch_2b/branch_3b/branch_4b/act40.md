# 第四十幕：荒野中的房屋

**旁白**：李杰站在那片灰色的荒地上,緩緩轉身環顧四周。

**旁白**：一片荒涼。除了灰色的土地、散落的石頭,什麼都沒有。

**旁白**：但當他轉向身後時,他看到了。

**旁白**：一座巨大的山。

**旁白**：那座山就在他們剛走出來的玻璃門後方,灰褐色的岩石構成,表面光禿禿的,沒有任何植被。

**旁白**：山體龐大,佔據了視野的很大一部分。

**李杰**：（內心）那個逃脫房間...還有走廊...應該都在這座山裡面...

**旁白**：玻璃門就嵌在山體的表面,像是某種入口。從外面看,那扇門顯得如此格格不入,現代化的玻璃和金屬框架,鑲嵌在原始的岩石中。

**斎藤蓮**：（跪在地上,聲音發抖,突然大喊）這到底是哪裡？！這他媽的到底是哪裡？！

**旁白**：斎藤的聲音在空曠的荒野中迴盪。

**斎藤蓮**：（開始崩潰,雙手抓著頭髮）我只是想回家！我只是想回家而已！為什麼...為什麼會變成這樣？！

**斎藤蓮**：（哭喊）這不是真的！這一定是夢！一定是夢！

**旁白**：李杰看了他一眼,但沒有說話。

**旁白**：他知道現在說什麼都沒用。斎藤需要時間發洩。

**旁白**：李杰轉身,開始向前走。

**旁白**：他的腳步堅定,踩在乾裂的土地上,發出輕微的沙沙聲。

**斎藤蓮**：（發現李杰在走,抬頭喊）你...你去哪裡？！你要丟下我一個人嗎？！

**旁白**：李杰沒有回答,繼續往前走。

**旁白**：他不知道要去哪裡,但他知道不能留在原地。

**旁白**：他需要找到答案。需要找到出路。

**斎藤蓮**：（慌張,從地上爬起來）等...等等我！

**旁白**：斎藤踉蹌地站起來,追在李杰後面。

**旁白**：兩人在荒涼的土地上行走,紫色的天空在他們頭頂延伸,兩個巨大的月亮靜靜地懸掛著。

---

**旁白**：他們走了很久。

**旁白**：沒有太陽,沒有時鐘,無法判斷時間。但他們的身體告訴他們——已經走了很久很久。

**旁白**：李杰的嘴唇開始感到乾燥。喉嚨像是被火燒一樣,每次吞嚥都感到疼痛。

**旁白**：他的胃也開始抗議,發出咕嚕咕嚕的聲音。

**旁白**：他們已經多久沒吃東西了？從進入那個逃脫房間開始...應該已經過了很多小時...或者更久？

**旁白**：李杰回頭看去。

**旁白**：那座山已經消失在視野盡頭了。

**旁白**：只剩下一望無際的灰色平原。

**斎藤蓮**：（氣喘吁吁,聲音沙啞）李杰...我不行了...我真的不行了...

**斎藤蓮**：（停下腳步,彎腰喘氣）我好渴...好餓...我們...我們到底要走到哪裡去？

**斎藤蓮**：（抬頭,聲音裡帶著絕望）這裡什麼都沒有！什麼都沒有！我們會死在這裡的！

**旁白**：李杰也停下腳步。他的體力也快到極限了。

**旁白**：但就在這時——

**旁白**：他看到了什麼。

**李杰**：（瞇起眼睛,指向遠方）那裡...

**斎藤蓮**：（抬頭,順著李杰的手指看去）什麼...

**旁白**：在遠處,在那片灰色的地平線上,有一個小小的輪廓。

**旁白**：一個建築物。

**斎藤蓮**：（眼睛瞬間睜大,聲音裡充滿希望）是...是房子？！那是房子嗎？！

**旁白**：那確實是一座建築物。從這個距離看不清細節,但可以看出建築的形狀。

**斎藤蓮**：（突然有了力氣,開始奔跑）房子！那裡有房子！一定有人！一定有水！

**旁白**：李杰也加快腳步,跟著斎藤向那個方向走去。

**旁白**：他的心裡燃起一絲希望。

**旁白**：也許...也許那裡會有答案。

---

**旁白**：當他們靠近時,那座建築物的輪廓逐漸清晰。

**旁白**：那是一座兩層樓的現代化房屋。

**旁白**：白色的外牆,灰色的屋頂,方正的造型。看起來就像是日本郊區常見的獨棟住宅。

**旁白**：但它孤零零地矗立在這片荒涼的土地上,四周什麼都沒有。

**旁白**：沒有圍牆,沒有花園,沒有車道,沒有任何配套設施。

**旁白**：就只有這麼一座房子。

**斎藤蓮**：（氣喘吁吁地跑到房子前,看著大門）有門...門沒鎖...

**旁白**：那是一扇普通的木製大門,咖啡色的,和日本的住宅大門一模一樣。

**旁白**：甚至還有門鈴和信箱。

**李杰**：（走到門前,伸手握住門把,低聲）小心點。

**旁白**：李杰小心翼翼地轉動門把。

**旁白**：門沒有鎖。

**旁白**：「吱呀——」

**旁白**：門緩緩打開,發出輕微的聲響。

**旁白**：裡面一片漆黑。

**斎藤蓮**：（在李杰身後,聲音緊張）有...有人嗎？

**旁白**：沒有回應。

**旁白**：李杰摸索著找到牆上的電燈開關,按了下去。

**旁白**：「啪」——

**旁白**：燈亮了。

**旁白**：明亮的白光照亮了整個玄關和客廳。

**旁白**：然後,李杰和斎藤同時看到了。

**旁白**：地板上,躺著三具屍體。

**旁白**：鮮血。

**旁白**：到處都是鮮血。

**旁白**：深紅色的血液在地板上蔓延開來,已經開始凝固,顏色變得暗沉。

**旁白**：三具屍體倒在客廳中央,姿勢扭曲,像是掙扎過。

**旁白**：李杰的視線落在最左邊的屍體上。

**旁白**：一個中年男子,穿著深色的襯衫和長褲。

**旁白**：李杰的呼吸停止了。

**旁白**：那張臉...他認識。

**旁白**：那是他的父親,李明遠。

**旁白**：旁邊,第二具屍體。

**旁白**：一個中年女子,穿著淺色的家居服。

**旁白**：那是他的母親,林婉如。

**旁白**：李杰的雙腿開始發抖。

**李杰**：（內心,聲音顫抖）不...不可能...

**李杰**：（內心）他們應該在上海...他們應該在家裡...

**李杰**：（內心,幾乎崩潰）為什麼...為什麼會在這裡...

**旁白**：就在這時——

**斎藤蓮**：（突然發出淒厲的尖叫）兄貴！！兄貴！！

**旁白**：斎藤衝進客廳,撲向第三具屍體。

**旁白**：那是一個體型肥胖的中年男子,穿著黑色的T恤和牛仔褲。

**斎藤蓮**：（跪在屍體旁邊,雙手顫抖著觸碰那張臉）兄貴...兄貴你怎麼了？！發生什麼事了？！

**斎藤蓮**：（開始搖晃屍體）醒醒啊！拜託！醒醒啊！

**旁白**：李杰僵硬地站在門口。

**旁白**：他的大腦一片空白。

**旁白**：斎藤的哥哥...也在這裡。

**旁白**：和他的父母一起...

**旁白**：死在這個不知名的世界...

**旁白**：這個孤零零的房子裡。

**李杰**：（內心）為什麼？

**李杰**：（內心,絕望）這到底是為什麼？

**旁白**：李杰的雙腿失去力量,他靠著牆壁慢慢滑坐在地上。

**旁白**：他的視線落在父母的屍體上,眼睛漸漸失去焦點。

**斎藤蓮**：（抱著哥哥的屍體,痛哭）為什麼...為什麼會這樣...兄貴...

**旁白**：在那兩個紫色月亮的光芒下,在這座孤獨的房子裡,兩個年輕人跪在三具屍體旁邊。

**旁白**：一個痛哭失聲。

**旁白**：一個完全僵住,無法思考,無法移動。

**李杰**：（內心,不斷迴響）這到底...是怎麼回事？
