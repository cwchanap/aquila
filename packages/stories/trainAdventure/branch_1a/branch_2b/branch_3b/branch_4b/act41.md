# 第四十一幕：血色謎團

**旁白**：不知道過了多久。李杰的呼吸逐漸平穩下來。

**旁白**：他閉上眼睛,深吸了幾口氣,強迫自己冷靜。

**李杰**：（內心）冷靜...必須冷靜...

**李杰**：（內心）現在崩潰解決不了任何問題...

**旁白**：他睜開眼睛,緩緩站起身。斎藤還跪在他哥哥的屍體旁,抱著那具屍體,肩膀不停顫抖。

**旁白**：李杰走向父親的屍體。他蹲下來,伸出顫抖的手,小心翼翼地檢查。

**旁白**：他翻開父親的襯衫,檢查胸口、腹部。沒有傷口。

**旁白**：他又檢查脖子、頭部。還是沒有傷口。

**李杰**：（內心,困惑）怎麼可能...沒有任何外傷...

**旁白**：他轉向母親的屍體,同樣仔細檢查。結果一樣——沒有任何傷口。

**旁白**：李杰的視線落在地板上那些血跡。鮮紅色,還沒有完全凝固,邊緣才剛開始變暗。

**李杰**：（內心）這些血很新鮮...他們應該是最近才...

**李杰**：（內心,更加困惑）但如果身上沒有傷口...這些血是從哪裡來的？

**旁白**：他再次回到父親身邊,開始檢查衣服口袋。

**旁白**：右邊的口袋裡,他摸到了一個硬硬的東西。

**旁白**：李杰拿出來——是一把螺絲刀。

**旁白**：普通的一字螺絲刀,金屬手柄,看起來有些使用痕跡。

**李杰**：（內心,困惑）螺絲刀...？

**李杰**：（內心,更加困惑）爸爸根本不會用這種東西...家裡要修什麼都是請師傅來...為什麼他會帶著螺絲刀...？

**旁白**：他把螺絲刀放在一旁,轉向母親的屍體。

**旁白**：他小心翼翼地檢查母親的口袋。

**旁白**：在外套口袋裡,他找到了一個打火機。

**旁白**：紅色的塑膠打火機,便利店常見的那種。

**李杰**：（內心,盯著打火機）打火機...

**李杰**：（內心,感到不對勁）媽媽不抽菸...家裡用的是電子爐...她根本不會需要打火機...

**李杰**：（內心）這到底是怎麼回事...？

**旁白**：李杰把打火機也放在一旁,站起身。

**旁白**：他看向斎藤還抱著的那具屍體。

**李杰**：（對斎藤,猶豫地）斎藤...我能...檢查一下你哥哥嗎？

**斎藤蓮**：（抬起頭,眼睛紅腫,聲音嘶啞）檢查...？

**李杰**：我在我父母身上找到了一些東西...我想看看...

**旁白**：斎藤沉默了片刻,然後緩緩鬆開手。

**旁白**：李杰蹲下來,檢查那個胖男子的口袋。

**旁白**：在褲子口袋裡,他摸到了一個金屬物品。

**旁白**：拿出來一看——是一把瑞士軍刀。

**旁白**：紅色的外殼,上面有白色的十字標誌,看起來保養得很好。

**斎藤蓮**：（看著那把刀,困惑地）這是什麼...？

**李杰**：瑞士軍刀。在你哥哥的口袋裡。

**斎藤蓮**：（更加困惑,聲音顫抖）不...不對...兄貴從來沒有這種東西...他不喜歡帶刀...

**斎藤蓮**：（盯著那把刀）為什麼...為什麼他會有這個...？

**旁白**：李杰把瑞士軍刀也放在旁邊。

**旁白**：三樣物品並排放在地板上——螺絲刀、打火機、瑞士軍刀。

**李杰**：（內心,盯著那三樣東西）爸爸不會用工具...媽媽不用打火機...斎藤的哥哥從來沒有刀...

**李杰**：（內心,感到一陣寒意）但他們卻都帶著這些...

**李杰**：（內心）這絕對不是巧合...這些東西...一定有什麼意義...

**旁白**：他站起身,環顧客廳。血跡散落在三具屍體周圍,但沒有濺射的痕跡,也沒有拖行的痕跡。

**旁白**：就像是...憑空出現的一樣。

**李杰**：（內心）這不合理...完全不合理...

**旁白**：另一個疑問在他腦海中浮現。

**李杰**：（內心）為什麼斎藤的哥哥會在這裡？

**李杰**：（內心）爸媽應該和他沒有任何聯繫...他們甚至不認識斎藤...

**李杰**：（內心）為什麼他們會一起死在這個地方？

**旁白**：李杰轉向斎藤。

**李杰**：（聲音沙啞但盡量保持冷靜）斎藤...你知道你哥哥為什麼會在這裡嗎？

**斎藤蓮**：（抬起頭,眼睛紅腫,聲音嘶啞）你...你說什麼？

**李杰**：你哥哥...他最近有說過要來什麼地方嗎？或者遇到什麼奇怪的事情？

**斎藤蓮**：（突然爆發,大喊）我怎麼知道？！我什麼都不知道！

**斎藤蓮**：（聲音裡充滿痛苦和憤怒）我們只是...只是一起吃了晚飯！然後各自回家！就這樣！

**斎藤蓮**：（哭喊）為什麼會變成這樣？！為什麼兄貴會死在這種鬼地方？！

**旁白**：李杰沒有再說話。他知道現在問不出什麼。斎藤深吸了幾口氣,用袖子擦了擦眼淚。

**斎藤蓮**：（聲音顫抖,指向李杰的父母）那...那兩個人是誰？

**旁白**：李杰沉默了片刻。

**李杰**：（低聲）我的父母。

**斎藤蓮**：（震驚,眼睛瞪大）什...什麼？

**斎藤蓮**：（不敢相信）你...你的父母？！

**斎藤蓮**：（聲音變得更加混亂）為什麼...為什麼你的父母會和兄貴...

**旁白**：李杰搖搖頭。

**李杰**：我也不知道。

**旁白**：兩人陷入沉默。客廳裡只有斎藤壓抑的抽泣聲。

---

**旁白**：李杰再次站起身。他需要做點什麼。不能就這樣站在這裡。

**李杰**：（低聲）我去看看房子裡有沒有其他東西。

**旁白**：斎藤沒有回應,還是跪在他哥哥身邊。李杰走出客廳,開始探索這座房子。

**旁白**：一樓除了客廳,還有廚房、餐廳、浴室。二樓有三個房間,看起來是臥室。

**旁白**：房子的佈局很正常,就像普通的日本住宅。但家具很少,牆上沒有照片,沒有任何個人物品。

**旁白**：就像一個剛搬進來還沒開始生活的空房子。

**旁白**：李杰來到廚房。他打開冰箱。

**旁白**：裡面有食物——飯糰、三明治、沙拉、飲料。全是便利店的包裝。

**旁白**：他又打開櫥櫃。泡麵、麵包、罐頭、零食。同樣都是便利店常見的商品。

**旁白**：流理台上還有一台微波爐,冰箱旁邊有飲水機。

**李杰**：（內心,感到不對勁）為什麼這裡會有這些...

**李杰**：（內心）就像是...專門準備給人吃的一樣...

**旁白**：但他的胃這時發出強烈的抗議聲。喉嚨乾得像要裂開一樣。

**旁白**：他們已經太久沒有進食了。

**李杰**：（內心）先補充體力...之後再想...

**旁白**：李杰拿起一瓶水,擰開瓶蓋,大口喝下。冰涼的水流過喉嚨,那種感覺讓他幾乎落淚。

**旁白**：他又拿了兩個飯糰,撕開包裝,開始吃。米飯的味道,海苔的香氣,鮭魚的鹹味。

**旁白**：這些平常普通到不行的食物,此刻吃起來卻像是世界上最美味的東西。

**旁白**：李杰拿著食物和水,走回客廳。

**李杰**：（對斎藤,聲音平靜）廚房有食物和水。你也吃點吧。

**斎藤蓮**：（沒有抬頭,聲音空洞）我不餓...

**李杰**：（堅定）你必須吃。我們需要體力。

**旁白**：李杰把一瓶水和飯糰放在斎藤旁邊的地板上。然後他走到牆邊,靠著牆坐下,繼續吃著手中的食物。

**旁白**：過了一會兒,斎藤終於伸手拿起那瓶水,喝了一口。然後又喝了一口。

**旁白**：最後,他拿起飯糰,開始機械性地咀嚼。

---

**旁白**：他們吃完後,兩人都沒有說話。客廳裡的三具屍體還躺在那裡,血跡在燈光下顯得格外刺目。

**李杰**：（打破沉默）我們...得處理他們。

**斎藤蓮**：（聲音顫抖）處理...？

**李杰**：我們不能讓他們就這樣躺在這裡。

**旁白**：斎藤低頭看著哥哥的屍體,眼淚又開始流下來。

**斎藤蓮**：（哽咽）我...我知道...

**旁白**：李杰走回那三樣物品旁邊,把它們一一撿起來。

**旁白**：螺絲刀、打火機、瑞士軍刀。他把它們放進自己的口袋裡。

**李杰**：（內心）也許...這些東西會有用...

**旁白**：李杰在房子後面找到了一個小工具間。裡面有鏟子。

**旁白**：他拿著鏟子走到屋外。紫色的天空下,那兩個巨大的月亮依然懸掛著,投下詭異的光芒。

**旁白**：李杰選了離房子不遠的一塊空地,開始挖掘。灰色的土壤很硬,但他一下一下地挖著。

**旁白**：斎藤也走了出來,默默地接過另一把鏟子,和李杰一起挖。兩人沒有說話。

**旁白**：只有鏟子插入土地的聲音,和兩人沉重的呼吸聲。

**旁白**：不知道挖了多久。坑終於夠深了。

**旁白**：他們回到屋內,小心翼翼地抬起第一具屍體。李杰抬著父親的肩膀,斎藤抬著腳。

**旁白**：他們慢慢走到屋外,將屍體輕輕放入坑中。然後是母親。

**旁白**：李杰的手在顫抖,但他沒有哭。他只是盯著母親的臉,想要記住她的樣子。

**旁白**：最後是斎藤的哥哥。斎藤跪在坑邊,看著哥哥的臉,嘴唇顫抖著。

**斎藤蓮**：（哽咽,低聲）對不起...兄貴...對不起...

**旁白**：李杰開始填土。一鏟,一鏟。灰色的土壤逐漸覆蓋住那三張臉。

**旁白**：斎藤也拿起鏟子,和李杰一起填土。直到三具屍體完全被土壤覆蓋。

**旁白**：直到地面上只剩下一個隆起的土堆。

**旁白**：李杰站在那個土堆前,看著那片新翻的土地。

**旁白**：他的父母就埋在那裡。

**旁白**：在這個陌生的世界,這片荒涼的土地上。

**李杰**：（內心,聲音平靜但充滿悲傷）爸,媽...對不起...

**李杰**：（內心）我會找出真相的...我保證...

---

**旁白**：他們回到屋內。李杰在廚房找到清潔用品,開始清理客廳的血跡。

**旁白**：斎藤也默默地幫忙,用抹布擦拭地板。他們用了很多水,很多清潔劑。

**旁白**：一遍又一遍地擦拭。直到那些深紅色的痕跡逐漸淡去。

**旁白**：直到地板恢復成原本的顏色。但李杰知道,那些痕跡並沒有真正消失。

**旁白**：它們會永遠留在他的記憶裡。

**旁白**：清理完畢後,兩人都精疲力盡。

**李杰**：（疲憊地）今天先休息吧。明天再想接下來要怎麼辦。

**斎藤蓮**：（聲音空洞,點點頭）嗯...

**旁白**：李杰上樓,隨便選了一個房間。房間裡只有一張床,一個衣櫃,什麼裝飾都沒有。

**旁白**：他脫下鞋子,躺在床上。天花板是白色的,空蕩蕩的。

**旁白**：李杰閉上眼睛。父母的臉在他腦海中浮現。那些血跡。那個土堆。

**旁白**：還有最核心的問題——

**李杰**：（內心）為什麼...為什麼他們會在這裡...

**李杰**：（內心）這一切...到底是怎麼回事...

**旁白**：但疲憊終究戰勝了思考。李杰的意識逐漸模糊。

**旁白**：在這個詭異的世界,在這座孤獨的房子裡,他沉入了黑暗的睡眠中。
