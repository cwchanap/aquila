# 第四十二幕：重置與絕望

**旁白**：李杰睜開眼睛。他盯著天花板看了一會兒,才意識到自己睡著了。

**旁白**：房間裡很安靜。紫色的光從窗簾縫隙透進來,在牆上投下詭異的光影。

**旁白**：李杰坐起身,揉了揉臉。身體很疲憊,但腦子卻很清醒。昨天的一切在腦海中迴放——父母的屍體、血跡、埋葬、那三樣奇怪的物品。

**旁白**：他站起身,走出房間。走廊很安靜。斎藤應該在另一個房間休息。

**旁白**：李杰走向浴室。他需要洗把臉,讓自己清醒一點。他打開浴室的門,走到洗手台前。

**旁白**：鏡子裡的自己看起來很憔悴。眼睛下方有明顯的黑眼圈,頭髮凌亂。他打開水龍頭,開始刷牙。

**旁白**：就在這時——

**斎藤蓮**：（從樓下傳來淒厲的尖叫）啊啊啊啊啊！！！

**旁白**：李杰的動作停住。那聲尖叫充滿恐懼,像是看到了什麼不可思議的東西。

**旁白**：李杰扔下牙刷,衝出浴室。他飛快地跑下樓梯,腳步聲在空蕩的房子裡迴響。

**旁白**：當他衝進客廳時,他看到了。然後,他的腳步停住了。他整個人僵在原地,無法動彈。

**旁白**：地板上,躺著三具屍體。鮮血。到處都是鮮血。

**旁白**：深紅色的血液在地板上蔓延開來,和昨天一模一樣的痕跡,一模一樣的位置。

**旁白**：父親李明遠,穿著深色的襯衫和長褲,倒在左邊。母親林婉如,穿著淺色的家居服,倒在中間。

**旁白**：斎藤的哥哥,那個肥胖的中年男子,倒在右邊。三具屍體,姿勢扭曲,就像昨天第一次看到時的樣子。

**旁白**：一模一樣。完全一模一樣。

**李杰**：（內心,聲音顫抖）不...不可能...

**李杰**：（內心,幾乎崩潰）我們明明...明明已經埋了他們...

**李杰**：（內心）血跡...我們明明清理乾淨了...

**旁白**：斎藤蓮跪在地上,就在那三具屍體前面。他一動不動,像一座石像。

**旁白**：他的眼睛睜得很大,盯著那些屍體,嘴巴微張,但發不出任何聲音。他的臉色慘白,整個人在顫抖。

**旁白**：李杰深吸一口氣,強迫自己冷靜下來。他的腦子在飛快運轉。

**李杰**：（內心）冷靜...必須冷靜...

**李杰**：（內心）這不可能是幻覺...斎藤也看到了...

**李杰**：（內心）那麼...這是...重置？

**旁白**：李杰走向父親的屍體,蹲下來。他伸手探入父親的口袋。

**旁白**：他摸到了一個硬硬的東西。拿出來——螺絲刀。和昨天一模一樣的螺絲刀。

**旁白**：李杰又檢查母親的口袋。打火機。紅色的,便利店常見的款式。

**旁白**：然後是斎藤哥哥的口袋。瑞士軍刀。紅色外殼,白色十字標誌。

**旁白**：李杰握著這三樣物品,緩緩站起身。然後他轉身,朝樓上跑去。

**旁白**：他衝進自己的房間,直奔那張桌子。他拉開抽屜。

**旁白**：空的。昨天他明明把那三樣物品放在這裡。但現在,抽屜裡什麼都沒有。

**李杰**：（內心,感到一陣寒意）真的...真的重置了...

**旁白**：李杰衝下樓,來到廚房。他打開冰箱。

**旁白**：飯糰、三明治、沙拉、飲料。全都在。昨天他們吃掉的那些,現在又完好無損地擺在冰箱裡。

**旁白**：李杰打開櫥櫃。泡麵、麵包、罐頭。一樣都沒少。

**旁白**：就像...就像昨天什麼都沒發生過一樣。

**旁白**：李杰靠著冰箱,閉上眼睛。

**李杰**：（內心）一切都重置了...

**李杰**：（內心）屍體、血跡、食物...全都回到原點...

**李杰**：（內心,開始思考）這是什麼意思...？

**李杰**：（內心）又是一個謎題嗎...？

**李杰**：（內心）我們應該做什麼...才能打破這個循環...？

**旁白**：李杰回到客廳。斎藤還跪在那裡,一動不動。但他的嘴唇在動,在喃喃自語。

**斎藤蓮**：（喃喃自語,聲音空洞）怎麼回事...這到底是怎麼回事...

**斎藤蓮**：（繼續喃喃）不可能...不可能的...我們明明埋了他們...

**斎藤蓮**：（聲音開始顫抖）這是夢嗎...這一定是夢...

**斎藤蓮**：（突然抬高音量,近乎崩潰）這他媽的到底是怎麼回事！！

**旁白**：李杰看著斎藤。他知道斎藤的精神已經到了極限。

**李杰**：（試圖安撫,聲音平靜）斎藤...冷靜點...

**斎藤蓮**：（轉頭看向李杰,眼睛裡充滿絕望）冷靜？！你叫我怎麼冷靜？！

**斎藤蓮**：（聲音越來越高）兄貴死了！然後我們埋了他！然後他又活過來了？！不...不是活過來...是又死了一次？！

**斎藤蓮**：（開始大笑,那笑聲裡充滿瘋狂）哈哈哈...這太瘋狂了...太他媽瘋狂了...

**斎藤蓮**：（笑聲停止,突然安靜下來）我不要...我不要再待在這裡了...

**斎藤蓮**：（聲音變得平靜,但那是一種絕望的平靜）我受夠了...

**旁白**：斎藤的視線落在地板上那把瑞士軍刀上。李杰剛才把三樣物品放在那裡。

**旁白**：斎藤緩緩伸手,拿起那把刀。

**李杰**：（察覺到不對勁）斎藤...？

**旁白**：斎藤打開瑞士軍刀,刀刃反射著詭異的紫色光芒。然後,在李杰還沒反應過來之前——

**旁白**：斎藤把刀刃對準自己的喉嚨。

**李杰**：（驚恐,大喊）不要——！！

**旁白**：但太遲了。刀刃劃過喉嚨。

**旁白**：血。鮮紅的血液噴湧而出,濺在地板上,濺在那些已經存在的血跡上。

**旁白**：斎藤的眼睛睜得很大,嘴巴張開,像是想說什麼。但只有氣泡和血從他的喉嚨湧出。

**旁白**：他的身體向前倒下,重重地摔在地板上。就倒在他哥哥的屍體旁邊。

**旁白**：李杰站在那裡,整個人僵住。他的手還伸在半空中,保持著想要阻止斎藤的姿勢。

**旁白**：但他來不及。他完全沒想到斎藤會突然這麼做。

**旁白**：李杰看著斎藤的身體,看著那些從喉嚨湧出的血。斎藤的身體還在抽搐,但動作越來越小。

**旁白**：越來越小。然後,停止了。

**旁白**：客廳裡一片寂靜。地板上,現在有四具屍體。四灘血跡。

**旁白**：李杰慢慢放下手,看著自己的雙手。它們在顫抖。

**李杰**：（內心,聲音空洞）斎藤...

**李杰**：（內心,感到無力）我...我來不及阻止...

**旁白**：李杰的雙腿失去力量。他跪倒在地上,雙手撐著地板。

**旁白**：他盯著斎藤的屍體,盯著那些血。在這個詭異的世界,在這座孤獨的房子裡。

**旁白**：現在,只剩下他一個人了。
