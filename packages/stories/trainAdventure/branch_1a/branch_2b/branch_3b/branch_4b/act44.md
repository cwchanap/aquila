# 第四十四幕：鏡像身份

**旁白**：李杰坐在牆邊,看著窗戶那邊的殭屍在原本的房間裡遊蕩。他們搖搖晃晃地撞來撞去,像是沒有目標的機器。

**旁白**：李杰深吸一口氣,站起身。

**李杰**：（內心）在這裡坐著也解決不了問題...

**李杰**：（內心,看向門）這個房間的門能打開...也許我應該出去看看...

**李杰**：（內心）這個鏡像世界...到底是什麼情況...

**旁白**：李杰走向門,再次握住門把。他深吸一口氣,然後轉動門把,輕輕推開門。

**旁白**：門外是走廊。和原本房子的走廊一樣,但是左右對調的。李杰小心翼翼地走出房間,環顧走廊。

**旁白**：安靜。沒有任何聲音。他慢慢走向樓梯。

**旁白**：就在他走到樓梯口時——他聽到了聲音。說話聲。從樓下傳來的,模糊的對話聲。

**旁白**：李杰的腳步停住。他側耳傾聽。那些聲音很模糊,但他能聽出來,那是正常的對話。

**旁白**：不是殭屍的低吼,不是什麼怪異的聲音。是人類的聲音。

**李杰**：（內心,困惑）有人...？

**李杰**：（內心）這個鏡像房子裡...有其他人？

**旁白**：李杰更加謹慎地走下樓梯。他的腳步很輕,盡量不發出聲音。

**旁白**：隨著他一步步走下樓梯,那些聲音變得越來越清晰。是對話。正常的、輕鬆的對話。還有笑聲。

**旁白**：然後,李杰聽清楚了一個聲音。一個他非常熟悉的聲音。

**男性聲音**：（從樓下傳來,溫和地）蓮怎麼還沒下來？飯都要涼了。

**旁白**：李杰的腳步停住。那個聲音...是父親的聲音。李明遠的聲音。

**女性聲音**：（也從樓下傳來,溫柔地）可能在房間裡休息吧。我去叫他下來。

**旁白**：母親的聲音。林婉如的聲音。李杰的心臟劇烈跳動。

**李杰**：（內心,震驚）爸爸...媽媽...？

**李杰**：（內心,幾乎不敢相信）他們...他們還活著...？

**旁白**：李杰快步走下剩下的樓梯,衝向客廳。然後,他看到了。他整個人愣在原地。

**旁白**：客廳裡,沒有屍體,沒有血跡。取而代之的,是一個溫馨的場景。餐桌旁,坐著三個人。

**旁白**：父親李明遠,穿著深色的襯衫,正拿著筷子。母親林婉如,穿著家居服,臉上帶著溫柔的笑容。

**旁白**：還有一個年輕的胖男子,大約二十多歲,正在大口吃著飯,臉上掛著開朗的笑容。李杰認得那張臉。

**旁白**：那是斎藤蓮的哥哥。那個在原本世界已經死去的,和父母倒在一起的胖男子。但現在,他活生生地坐在那裡,愉快地享用著晚餐。

**旁白**：餐桌上擺滿了飯菜。米飯、味噌湯、烤魚、蔬菜。就像一個普通家庭的晚餐。

**旁白**：父親抬起頭,看到站在客廳入口的李杰。

**李明遠**：（溫和地笑著）蓮,你總算下來了。我們都等你好久了。

**旁白**：母親也轉過頭,看著李杰。

**林婉如**：（溫柔地）快來坐下吃飯吧,飯都要涼了。

**旁白**：那個胖男子也抬起頭。

**胖男子**：（笑嘻嘻地,嘴裡還塞著食物）弟弟,你今天怎麼這麼晚才下來？我都快把你那份也吃掉啦！

**旁白**：李杰站在那裡,嘴巴微張,說不出話來。他們...他們在叫他什麼？

**李杰**：（聲音顫抖）你們...你們在叫我...蓮？

**李明遠**：（困惑地）是啊,怎麼了？你今天怪怪的。

**林婉如**：（關心地）是不是身體不舒服？臉色看起來不太好。

**李杰**：（搖頭,聲音更大）不...我不是蓮...我是李杰...

**李杰**：（指著父親）你是我爸爸,李明遠...

**李杰**：（指著母親）你是我媽媽,林婉如...

**李杰**：（聲音急切）我是你們的兒子,李杰！不是什麼蓮！

**旁白**：三個人面面相覷,臉上都露出困惑的表情。

**李明遠**：（皺眉,擔心地）蓮,你在說什麼？

**林婉如**：（站起身,走向李杰）孩子,你是不是發燒了？

**旁白**：母親走到李杰面前,伸手想要摸他的額頭。李杰下意識地後退一步。

**李杰**：（堅持）我不是蓮...我的名字是李杰...

**李明遠**：（也站起身,聲音變得嚴肅）蓮,別鬧了。你叫斎藤蓮,是我和你媽媽的兒子。

**李明遠**：（指著那個胖男子）這是你哥哥,斎藤大輔。

**李明遠**：（看著李杰,認真地）我們是斎藤家。我叫斎藤浩一,你媽媽叫斎藤美香。你從小就叫斎藤蓮。

**斎藤大輔**：（也站起身,走向李杰,輕鬆地拍拍他的肩膀）欸欸,弟弟,別嚇我們啊。你該不會是熬夜打遊戲打到神智不清了吧？哈哈哈！

**旁白**：李杰看著他們三個人。父親的臉。母親的臉。那個應該已經死去的胖男子的臉。

**旁白**：他們都看著他,眼中充滿關心和困惑。就像...就像他們真的是一家人一樣。

**李杰**：（內心,感到天旋地轉）這...這是怎麼回事...

**李杰**：（內心）爸爸說我是斎藤蓮...說我們是斎藤家...

**李杰**：（內心,看著那個胖男子）他說那個人是我哥哥...斎藤大輔...

**李杰**：（內心,困惑）但我明明是李杰...我是他們的兒子...

**旁白**：李杰的腦子一片混亂。他張了張嘴,想要反駁,想要堅持。但看著父母臉上的擔心,看著他們真切的關心。他的話卡在喉嚨裡。

**李杰**：（內心,掙扎）如果我繼續堅持...他們會怎麼想...

**李杰**：（內心）而且...至少...至少他們還活著...

**李杰**：（內心,感到一陣酸楚）不管這是真實還是幻覺...至少...他們還在我面前...

**旁白**：李杰深吸一口氣,低下頭。

**李杰**：（小聲地）對不起...我...我可能是太累了...

**林婉如**：（鬆了口氣,溫柔地）沒事就好。來,先坐下吃飯吧。

**李明遠**：（點頭）好好休息一下,別想太多。

**斎藤大輔**：（笑著搓搓李杰的頭）哈哈,沒事就好。弟弟,累了就早點睡,別想太多啦。有哥哥在呢！

**旁白**：李杰點點頭,慢慢走向餐桌。他坐在唯一空著的那張椅子上。

**旁白**：母親給他盛了一碗飯,父親給他夾了菜。斎藤大輔繼續吃著飯,偶爾和父母聊著天。

**旁白**：談論著今天的天氣,談論著工作,談論著一些日常瑣事。就像一個普通的家庭晚餐。

**旁白**：李杰拿著筷子,看著碗裡的飯。他的手在顫抖。

**李杰**：（內心,混亂）這到底是怎麼回事...

**李杰**：（內心）這個鏡像世界...我變成了斎藤蓮...

**李杰**：（內心）爸爸媽媽...變成了斎藤家的父母...

**李杰**：（內心,看向斎藤大輔）那個本來應該死去的人...成了我的哥哥...

**李杰**：（內心,感到困惑和恐懼）這是真的嗎...還是幻覺...

**李杰**：（內心）如果是真的...那原本的世界...原本的我...又是什麼...

**李杰**：（內心,看著父母）但...至少他們還活著...

**李杰**：（內心,感到矛盾）就算這是假的...就算這只是一場夢...

**李杰**：（內心,聲音顫抖）至少...我還能再看到他們...

**林婉如**：（溫柔地）蓮,怎麼不吃？不合胃口嗎？

**李杰**：（回過神,勉強笑了笑）沒有...我吃...

**旁白**：李杰夾起一口飯,放進嘴裡。味道。真實的味道。溫熱的米飯,鹹香的烤魚。

**旁白**：這不是幻覺。這些...都是真實的。

**旁白**：李杰慢慢咀嚼著,看著對面的父母。他們還在聊天,偶爾看向他,臉上帶著溫柔的笑容。

**旁白**：就像以前一樣。就像什麼都沒發生過一樣。

**李杰**：（內心,感到一陣悲傷）這個世界...到底是什麼...

**李杰**：（內心,困惑）我...到底該怎麼辦...
