# 第五十二幕：謎題

**旁白**：李杰吃完便當，扔掉便當盒，站起身來。夜色已深，他該回家了。他坐上電車，回到三軒茶屋。從車站走回家，只需要十分鐘。

**旁白**：李杰打開家門。客廳的燈亮著。父親坐在沙發上，母親靠在他肩膀上，眼睛紅腫，還在輕聲啜泣。父親輕輕拍著她的背，安慰著她。

**李杰**：（輕聲）爸，媽...我回來了...

**林婉如/斎藤美香**：（抬起頭，眼淚還掛在臉上）蓮...你回來了...

**李明遠/斎藤浩一**：（疲憊地）蓮...你也累了吧...去休息吧...

**李杰**：（看著他們，心裡難受）爸，媽...別太擔心...大哥一定會沒事的...

**林婉如/斎藤美香**：（哽咽地）我不知道...我真的不知道...他怎麼會突然消失...

**李明遠/斎藤浩一**：（嘆氣）明天一早我們就去報警...警察會幫我們找的...

**李杰**：（點頭）嗯...你們也早點休息...

**旁白**：李杰走回自己的房間。他坐在床邊，看著手機裡那張照片。空蕩蕩的房間，昏暗的燈光...

**李杰**：（內心）大輔...你到底想告訴我什麼...

**旁白**：突然，李杰想到了什麼。

**李杰**：（內心，站起來）大輔的房間...

**李杰**：（內心）也許...他的房間裡會有什麼線索...

**旁白**：李杰走出房間，來到走廊。大輔的房間就在他房間的對面。李杰走到大輔的房門前，伸手握住門把。但門把紋絲不動。

**李杰**：（皺眉）鎖住了...？

**旁白**：李杰低頭看去。門把旁邊，有一個小小的智能密碼鎖。黑色的面板，上面顯示著數字鍵盤。

**李杰**：（驚訝地）這是...什麼時候裝的...

**旁白**：李杰試著按了幾個數字。嗶。紅燈亮起。密碼錯誤。

**李杰**：（內心）密碼...我不知道密碼...

**旁白**：李杰轉身，走回客廳。

**李杰**：（對父母說）爸，媽...大哥的房間...什麼時候裝了密碼鎖？

**李明遠/斎藤浩一**：（抬起頭）密碼鎖？

**林婉如/斎藤美香**：（也抬起頭，困惑地）我不知道...他什麼時候裝的...

**李明遠/斎藤浩一**：（想了想）可能是他去中國工作之前吧...他說要保護自己的隱私...

**李杰**：（問道）那...你們知道密碼嗎？

**李明遠/斎藤浩一**：（搖頭）不知道...他沒告訴我們...

**林婉如/斎藤美香**：（擦著眼淚）他說那是他的私人空間...我們也沒問...

**李杰**：（內心，失望）連父母都不知道...

**李杰**：（對父母說）我知道了...你們休息吧...

**旁白**：李杰回到自己的房間。他坐在書桌前，再次拿起手機，看著那張照片。

**李杰**：（內心）這張照片...

**李杰**：（內心，突然想到）等等...

**旁白**：李杰的眼神變得銳利。

**李杰**：（內心）這張照片...該不會就是線索吧...

**旁白**：李杰想起了一年前的事情。那些謎題。那些需要解開的密碼。那輛神秘的列車。

**李杰**：（內心，聲音顫抖）又來了...

**李杰**：（內心）又是謎題...

**李杰**：（內心，握緊拳頭）我差點就忘了...因為這一年的平靜生活...

**李杰**：（內心）但現在...一切又回來了...

**旁白**：李杰深吸一口氣，強迫自己冷靜下來。他放大照片，仔細觀察每一個細節。房間...木地板...書桌...書架...床...還有那些物品。

**李杰**：（內心，冷靜地分析）我需要找出線索...

**李杰**：（內心）這張照片裡...一定有什麼訊息...

**旁白**：李杰開始清點照片中的物品。

**李杰**：（內心，一個一個數）PS5...Switch...一台桌上型電腦...一個顯示器...

**旁白**：他的目光移向書架。

**李杰**：（內心）書架上...是漫畫...

**旁白**：李杰放大照片，仔細看著書架上的漫畫。

**李杰**：（內心）哆啦A夢...火影忍者...

**李杰**：（內心）好多本...

**旁白**：然後是牆上的海報。

**李杰**：（內心）海報...兩張...

**旁白**：李杰放大第一張海報。是一個女性動漫角色。藍色的長髮，穿著校服，笑容燦爛。

**李杰**：（內心，皺眉）這個角色...我不認識...

**旁白**：李杰又看向第二張海報。這是一個真人。一個年輕女性，長髮，笑容甜美。看起來像是演員或偶像。

**李杰**：（內心）這個人...我也不認識...

**李杰**：（內心，嘆氣）可能是日本的演員吧...

**旁白**：李杰把照片中的物品一一記下。PS5，Switch，桌上型電腦，顯示器。哆啦A夢漫畫，火影忍者漫畫。不認識的女性動漫角色海報。不認識的女演員海報。

**李杰**：（內心）這些東西...有什麼共通點嗎...

**李杰**：（內心，苦惱地）我想不出來...

**旁白**：李杰看著照片，陷入沉思。時間一分一秒過去。已經接近午夜了。

**李杰**：（內心，疲憊地）不行...我現在太累了...腦子轉不動...

**李杰**：（內心）明天...明天我去學校...找同學幫忙...

**李杰**：（內心）颯真他們...可能會知道那些動漫角色和演員是誰...

**旁白**：李杰把手機放在書桌上，躺到床上。他閉上眼睛。但腦海中，那張照片的畫面揮之不去。

**旁白**：空蕩蕩的房間。昏暗的燈光。那些物品。還有...大輔的消失。

**李杰**：（內心，在黑暗中低語）大輔...

**李杰**：（內心，聲音痛苦）你...到底在哪裡...

**旁白**：李杰在黑暗中睜開眼睛，看著天花板。這一夜，他無法入睡。
