# 第五十四幕：北海道

**旁白**：一週過去了。父母向警方報案後，警方展開了調查。但調查沒有任何進展。

**旁白**：這天晚上，父親接到警方的電話。

**李明遠/斎藤浩一**：（接起電話）喂？...是的...什麼？...沒有入境紀錄？

**林婉如/斎藤美香**：（焦急地）怎麼了？

**李明遠/斎藤浩一**：（放下電話，沉重地）警方說...日本沒有大輔入境的紀錄...

**林婉如/斎藤美香**：（驚訝地）什麼意思...他明明辦理了登機手續...

**李明遠/斎藤浩一**：（嘆氣）警方懷疑...他可能沒有真的登機...他們說...需要聯繫中國當局...調查他在中國的行蹤...

**林婉如/斎藤美香**：（崩潰地）這...這到底是怎麼回事...

**旁白**：李杰站在房間門口，聽到了這段對話。

**李杰**：（內心）沒有入境紀錄...

**李杰**：（內心，握緊拳頭）這和我當時一樣...

**李杰**：（內心）我搭上了列車...然後就消失了...

**李杰**：（內心）大輔...你也是嗎...

**旁白**：李杰回到房間，關上門。他坐在書桌前，看著手機裡那張照片。已經看了無數次了。但還是找不到答案。

**李杰**：（內心，疲憊地）不能再這樣下去了...

**李杰**：（內心）我需要...暫時放下這件事...

**李杰**：（內心）至少...在找到清水惠美之前...我什麼都做不了...

**旁白**：李杰看向日曆。2024年6月底。距離2025年9月15日...還有一年多。

**李杰**：（內心，嘆氣）一年多...

**李杰**：（內心）我只能等...

---

**旁白**：隔天早上，李杰收拾好行李，準備出門。

**林婉如/斎藤美香**：（看著李杰）蓮...你真的要去旅行嗎...

**李杰**：（點頭）嗯...早就和朋友約好了...

**李明遠/斎藤浩一**：（疲憊地）去吧...放鬆一下也好...

**林婉如/斎藤美香**：（擔心地）但是...大輔的事情...

**李杰**：（溫柔地）媽，警方會繼續調查的...我在這裡也幫不上忙...

**李明遠/斎藤浩一**：（點頭）蓮說得對...而且...你也需要休息...這幾天你也累壞了...

**李杰**：（對父母說）我會帶著手機...有消息隨時告訴我...

**林婉如/斎藤美香**：（點頭，擦著眼淚）好...路上小心...

**旁白**：李杰背著背包，走出家門。東京的天氣悶熱潮濕。六月底的梅雨季，空氣黏膩得讓人難受。

**李杰**：（內心）逃離這裡...去北海道...

**李杰**：（內心）也許...能讓我暫時忘記這一切...

**旁白**：李杰來到羽田機場。如月颯真、氷室陸斗、七海詩織、高橋彩花、渡邊真理已經在那裡等著了。

**如月颯真**：（揮手）蓮！這裡！

**氷室陸斗**：（興奮地）終於等到你了！北海道，我們來了！

**七海詩織**：（笑著）陸斗，別這麼大聲，很吵...

**高橋彩花**：（笑著對李杰說）蓮，你還好嗎？看起來氣色好一點了。

**李杰**：（微笑）嗯...謝謝...

**渡邊真理**：（興奮地）我好期待北海道！聽說那裡現在才20度左右！

**氷室陸斗**：（誇張地）天堂啊！東京這鬼天氣快把我熱死了！

**旁白**：他們辦理登機手續，通過安檢，登上飛機。一個半小時後，飛機降落在新千歲機場。一走出機場，涼爽的空氣迎面而來。

**氷室陸斗**：（深吸一口氣）啊！這才是夏天該有的溫度！

**七海詩織**：（笑著）好舒服！完全不悶熱！

**如月颯真**：（看著天空）天氣真好，太適合騎車了。

**旁白**：他們租了腳踏車，開始第一天的行程。從新千歲機場出發，沿著海岸線騎向小樽市。藍天，白雲，一望無際的大海。涼爽的海風吹在臉上。路上的車不多，騎起來很舒服。

**氷室陸斗**：（騎在前面，大喊）好爽啊！

**七海詩織**：（也很開心）風好涼！

**旁白**：李杰騎在隊伍的後方，看著朋友們開心的背影。

**李杰**：（內心）也許...這樣就好...

**李杰**：（內心）至少現在...可以忘記那些煩惱...

**旁白**：中午，他們來到一個小漁村。在海邊的餐廳，吃了新鮮的海膽丼和烤扇貝。

**氷室陸斗**：（大口吃著）太好吃了！這海膽也太新鮮了吧！

**高橋彩花**：（笑著）陸斗，慢點吃，沒人跟你搶。

**渡邊真理**：（看著海景）這裡的風景真美...

**如月颯真**：（對李杰說）蓮，你覺得怎麼樣？

**李杰**：（微笑）很好...真的很舒服...

**七海詩織**：（笑著）太好了！這次旅行就是要讓大家好好放鬆！

**旁白**：下午，他們繼續騎車。經過了一片片薰衣草田。紫色的花海在微風中搖曳。空氣中飄著淡淡的花香。

**渡邊真理**：（停下來拍照）好美！

**高橋彩花**：（也拿出手機）我們一起拍張照吧！

**旁白**：六個人站在薰衣草田前，擺好姿勢。喀嚓。

**氷室陸斗**：（看著照片）完美！

**旁白**：傍晚，他們到達小樽市。入住海邊的民宿。民宿是一棟木造的小房子，溫馨舒適。老闆娘是個和藹的中年婦女，熱情地招待他們。

**民宿老闆娘**：（笑著）歡迎歡迎！你們是從東京來的嗎？

**如月颯真**：（點頭）是的，我們是大學生，來北海道騎車旅行。

**民宿老闆娘**：（笑著）真好！年輕就是要多出來走走！晚餐想吃什麼？

**氷室陸斗**：（興奮地）有什麼推薦的嗎？

**民宿老闆娘**：（笑著）今天早上剛捕的螃蟹，新鮮得很！

**七海詩織**：（眼睛發亮）螃蟹！

**旁白**：晚餐時，桌上擺滿了海鮮。帝王蟹、烤魚、味噌湯、白飯。六個人圍坐在一起，開心地吃著。

**氷室陸斗**：（大口吃著螃蟹）這螃蟹也太好吃了吧！

**渡邊真理**：（笑著）陸斗今天一直在吃...

**高橋彩花**：（也笑著）難得來北海道，當然要好好吃！

**如月颯真**：（舉起啤酒杯）來，為這次旅行乾杯！

**眾人**：乾杯！

**旁白**：李杰也舉起杯子，和大家碰杯。他看著朋友們笑得燦爛的臉。聽著他們愉快的笑聲。

**李杰**：（內心，微笑）這樣...真好...

**旁白**：晚餐後，他們坐在民宿外的露台上。海風輕輕吹著。遠處傳來海浪的聲音。夕陽在海平面上慢慢落下，天空染成橘紅色。

**七海詩織**：（看著夕陽）好美...

**渡邊真理**：（靠在欄杆上）真想時間就這樣停住...

**如月颯真**：（笑著）明天還有一整天呢，我們慢慢享受。

**氷室陸斗**：（伸了個懶腰）我要去泡澡了，今天騎了一整天，累死了。

**高橋彩花**：（笑著）那我們也去吧。

**旁白**：大家陸續回房間休息。

**旁白**：李杰站在露台上，看著逐漸暗下來的天空。星星開始出現。一顆，兩顆，三顆...越來越多。

**李杰**：（內心）大輔...

**李杰**：（內心，看著星空）你現在...在哪裡...

**如月颯真**：（走過來）蓮，還不去休息？

**李杰**：（回過神，轉頭）啊...馬上就去...

**如月颯真**：（看著他）你今天看起來心情好多了。

**李杰**：（微笑）嗯...謝謝你們...帶我來這裡...

**如月颯真**：（拍拍他的肩膀）我們是朋友。有什麼事，隨時說。

**李杰**：（點頭）我知道。

**旁白**：兩個人並肩看著星空。海浪聲，星光，還有朋友的陪伴。這一刻，李杰感到內心平靜。至少...暫時的平靜。
