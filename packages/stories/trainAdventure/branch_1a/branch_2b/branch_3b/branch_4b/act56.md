# 第五十六幕：暴力破解

**旁白**：李杰在屋裡來回搜索。客廳、廚房、浴室、陽台。每個角落都仔細檢查過了。

**旁白**：他再次走進父母的臥室。打開衣櫃，翻看抽屜，檢查床底。沒有任何異常。衣服整齊掛著，日常用品都在。一切都像是父母只是暫時出門一樣。

**李杰**：（內心，焦慮地）到底去哪裡了...

**旁白**：李杰站在走廊上，看著大輔的房門。整個屋子裡，唯一還沒搜索的，就是這個房間。但那個智能密碼鎖，堵住了他的去路。

**李杰**：（內心，握緊拳頭）該死...

**旁白**：李杰走到房門前，再次盯著那個密碼鎖。黑色的面板，數字鍵盤，冰冷而堅硬。

**李杰**：（內心）如果能打開這個門...也許...也許就能找到線索...

**旁白**：但他試過了。無數次。每一次都是紅燈。李杰嘆了口氣，回到自己的房間。他坐在書桌前，腦海中不斷思考。

**李杰**：（內心）清水惠美...2025年9月15日...

**李杰**：（內心，焦慮地）但那是明年...還有一年多...

**李杰**：（內心）我要等那麼久嗎...

**李杰**：（內心，更焦慮）而且...如果歷史改變了怎麼辦...

**李杰**：（內心）如果她沒有出現怎麼辦...

**李杰**：（內心，握緊拳頭）我不能只是等待...我必須做點什麼...

**旁白**：但他能做什麼呢？唯一的線索，就是清水惠美。唯一可能有答案的地方，就是大輔的房間。但他進不去。

**李杰**：（內心，疲憊地）我...該怎麼辦...

---

**旁白**：隔天一早，李杰去了警察局。報案大廳裡人不多。值班的警察是個中年男子，看起來很疲憊。

**警察**：（看著李杰）你要報案？

**李杰**：（點頭）是的...我的父母...失蹤了...

**警察**：（拿出筆記本）什麼時候的事？

**李杰**：昨天...我旅行回來後...他們就不在家了...

**旁白**：李杰把情況說明了一遍。父母不在家，電話不接，沒有留下任何訊息。

**警察**：（記錄著）有沒有可能是臨時出門？去親戚家之類的？

**李杰**：（搖頭）不可能...他們一定會告訴我的...而且電話也不接...

**警察**：（想了想）這樣吧，我先記錄下來。如果24小時內還沒有消息，我們會正式立案調查。

**李杰**：（焦急地）但是...

**警察**：（打斷他）很多失蹤案件，最後都是誤會。也許你父母只是臨時有事。先等等看吧。

**旁白**：李杰知道再說什麼也沒用。

**李杰**：（無奈地）好...如果有消息...請聯絡我...

**警察**：（點頭）會的。留下你的聯絡方式。

**旁白**：李杰留下電話號碼，離開警察局。走在街上，他感到前所未有的無力。

**李杰**：（內心）連警察都幫不上忙...

---

**旁白**：李杰照常去學校上課。但他完全無法集中精神。老師在講台上說話，他一個字都聽不進去。腦海中全是父母失蹤的事。

**旁白**：午休時間，李杰一個人坐在教室角落。他拿著便當，但沒有胃口。

**如月颯真**：（走過來）蓮...你還好嗎？

**旁白**：李杰抬起頭。如月颯真、氷室陸斗、七海詩織、高橋彩花、渡邊真理都圍了過來。

**七海詩織**：（擔心地）你今天看起來很沮喪...

**氷室陸斗**：（也擔心）是不是發生什麼事了？

**李杰**：（沉默了一會兒，嘆氣）...我父母...失蹤了...

**眾人**：（驚訝地）什麼？！

**李杰**：（低聲）昨天我從北海道回來...他們就不在家了...電話也不接...沒有留下任何訊息...

**渡邊真理**：（震驚地）怎麼會...

**高橋彩花**：（著急地）報警了嗎？

**李杰**：（點頭）報了...但警察說要等24小時...

**如月颯真**：（嚴肅地）你哥哥之前也失蹤了對吧...現在連父母也...

**李杰**：（苦笑）對...我也不知道是怎麼回事...

**七海詩織**：（握著李杰的手）蓮...我們能幫什麼忙嗎？

**旁白**：李杰看著朋友們擔心的臉。他猶豫了一下。

**李杰**：（緩緩說）其實...我哥哥的房間...被密碼鎖鎖住了...

**氷室陸斗**：（困惑地）密碼鎖？

**李杰**：（點頭）我覺得...房間裡可能有線索...但我進不去...

**李杰**：（拿出手機）我哥哥在失蹤前...傳了一張照片給我...

**旁白**：李杰把照片給大家看。

**如月颯真**：（看著照片）就是上次那張...你哥哥房間的照片？

**李杰**：（點頭）我覺得...這張照片...可能是密碼的提示...但我試了很多次都不對...

**七海詩織**：（想了想）上次我們不是已經認出照片裡的東西了嗎...長門有希...還有清水惠美...

**李杰**：（點頭）對...但我一直想不出來這些東西和密碼有什麼關係...

**如月颯真**：（認真地）蓮...不如我們放學後一起去你家看看？

**氷室陸斗**：（點頭）對啊！多幾個人一起想，說不定能想出來！

**渡邊真理**：（也點頭）我們一起幫你試試看！

**高橋彩花**：（溫柔地）說不定我們能找到什麼線索。

**李杰**：（看著他們，感動地）你們...真的願意嗎...

**七海詩織**：（笑著）我們是朋友啊。

**李杰**：（點頭，感激地）...謝謝你們...

---

**旁白**：下午的課，李杰依然無法集中精神。但心裡多了一份期待。

**旁白**：放學後，六個人一起坐電車回到三軒茶屋。從車站走到李杰家，只需要十分鐘。

**旁白**：李杰打開家門。

**李杰**：（輕聲）我回來了...

**旁白**：屋裡一片寂靜。沒有回應。

**氷室陸斗**：（環顧四周，小聲）你父母...真的不在...

**七海詩織**：（也小聲，擔心地）好安靜...

**李杰**：（低聲）跟我來...

**旁白**：李杰帶著朋友們來到走廊。大輔的房門就在眼前。門把旁邊，那個黑色的智能密碼鎖，在走廊燈光下閃著冷光。

**如月颯真**：（看著密碼鎖）就是這個...

**李杰**：（點頭）對...我試過很多次了...

**渡邊真理**：（拿出手機，再次看照片）那我們再仔細看看照片...

**旁白**：六個人圍在手機旁邊，仔細看著照片。PS5，Switch，電腦，顯示器。哆啦A夢漫畫，火影忍者漫畫。長門有希的海報。清水惠美的照片。

**高橋彩花**：（思考著）密碼...通常是數字吧...

**如月颯真**：（點頭）應該是...那就要找出這些東西和數字的關聯...

**氷室陸斗**：（看著照片）PS5...會不會就是5？

**李杰**：（走到密碼鎖前）我試試...

**旁白**：李杰按下數字5。嗶。紅燈。

**氷室陸斗**：（抓抓頭）不對嗎...

**七海詩織**：（想了想）會不會是發售日期？PS5是2020年...Switch是2017年...

**李杰**：（按下2020）

**旁白**：嗶。紅燈。

**李杰**：（按下2017）

**旁白**：嗶。紅燈。

**李杰**：（按下20202017）

**旁白**：嗶。紅燈。

**渡邊真理**：（看著漫畫）那漫畫呢？哆啦A夢...火影忍者...

**如月颯真**：（皺眉）但我們看不清楚照片裡到底有幾本...

**高橋彩花**：（想了想）清水惠美...她失蹤的日期是2005年9月15日...試試看？

**李杰**：（按下20050915）

**旁白**：嗶。紅燈。

**李杰**：（按下0915）

**旁白**：嗶。紅燈。

**李杰**：（按下915）

**旁白**：嗶。紅燈。

**七海詩織**：（也想著）長門有希...《涼宮春日的憂鬱》...首播是2006年...

**李杰**：（按下2006）

**旁白**：嗶。紅燈。

**氷室陸斗**：（抓抓頭）都不對...

**旁白**：大家沉默了。又試了很多組合。遊戲機的型號。漫畫的可能集數。清水惠美的生日。每一次，都是紅燈。

**李杰**：（疲憊地靠在牆上）不行...都不對...

**如月颯真**：（也有些沮喪）這密碼...到底是什麼...

**旁白**：走廊裡一片安靜。只有密碼鎖偶爾發出的「嗶」聲。

**旁白**：氷室陸斗看著李杰疲憊的臉。突然，他說了一句話。

**氷室陸斗**：（疑惑地）蓮...我有個問題...

**李杰**：（抬起頭）什麼？

**氷室陸斗**：（指著密碼鎖）為什麼你一定要解開密碼？

**李杰**：（困惑地）什麼意思...

**氷室陸斗**：（理所當然地）我是說...為什麼不直接破壞這個鎖？

**眾人**：（驚訝地看著他）

**氷室陸斗**：（繼續說）你想進房間對吧？那就把鎖砸爛啊。暴力破解不是最簡單的方法嗎？

**七海詩織**：（無語地）陸斗...那是別人家的門...

**氷室陸斗**：（抓抓頭）可是...那是蓮自己家啊...而且現在情況這麼緊急...

**如月颯真**：（想了想）陸斗說得...也有道理...

**高橋彩花**：（也點頭）雖然有點粗暴...但如果真的沒有其他辦法...

**旁白**：李杰愣住了。他從來沒有想過這個方法。從一開始，他就想著要解開謎題。要找出密碼。要按照規則來。但...為什麼？

**李杰**：（內心，恍然大悟）對...為什麼我一定要解開密碼...

**李杰**：（內心）這不是遊戲...

**李杰**：（內心）這是現實...

**李杰**：（內心，興奮地）我可以直接破壞那個鎖...

**李杰**：（站直身體，眼睛發亮）陸斗...你說得對...

**氷室陸斗**：（得意地）對吧！暴力才是最好的解決方法！

**七海詩織**：（無奈地）陸斗...你這話聽起來好危險...

**李杰**：（握著氷室陸斗的手）謝謝你...陸斗...你給了我很大的啟發...

**氷室陸斗**：（被握著手，有點不好意思）欸...不用這麼誇張吧...

**如月颯真**：（認真地）蓮，你打算怎麼做？

**李杰**：（堅定地）我...我現在就去破壞這個鎖...

**渡邊真理**：（點頭）我們來幫你！

**氷室陸斗**：（興奮地）對！砸鎖這種事，交給我！

**高橋彩花**：（也點頭）我們一起找找看有什麼工具...

**李杰**：（感動地看著大家）你們...

**如月颯真**：（拍拍李杰的肩膀）我們是朋友。走吧，去找工具。

**旁白**：六個人開始在屋裡尋找可以用來破壞密碼鎖的工具。李杰看著朋友們忙碌的身影。

**李杰**：（內心，感動）謝謝你們...真的...

**李杰**：（內心，堅定地）我終於...可以打開那扇門了...

**李杰**：（內心）大輔...爸...媽...

**李杰**：（內心）等著我...
