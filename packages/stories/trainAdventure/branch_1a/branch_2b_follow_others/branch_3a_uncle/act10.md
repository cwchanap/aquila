# 第十幕：森林中的庇護所

**旁白**：濃霧中,李杰抱著受傷的女孩,清水恵美緊跟在旁,三人艱難地穿過森林。

**旁白**：沉重的呼吸聲和腳步聲是唯一的聲響。李杰的手臂開始發酸,但他咬緊牙關,一步一步地往前走。

**旁白**：懷中的女孩雖然輕,但長時間抱著,加上要小心避開箭傷,讓他的體力消耗很快。

**李杰**：(喘著氣,試圖分散注意力)妳...叫什麼名字?我們總不能一直叫妳「那個女孩」吧。

**多話女大生**：(虛弱地,聲音很小)高橋...美咲...東京人...早稻田大學...二年級...

**李杰**：(溫和地)美咲,撐住。我們會找到地方的。妳已經很勇敢了,再堅持一下。

**高橋美咲**：(微弱地點頭,沒有多餘的力氣說話)嗯...謝謝...

**旁白**：三人沉默地前行。霧氣越來越濃,幾乎無法辨識方向。李杰只能憑著感覺,盡量保持直線前進。

**旁白**：就在李杰感覺自己快要支撐不住時,前方的小徑旁突然出現了一座廢棄的伐木工小屋。

**旁白**：小屋雖然破舊,但看起來還算堅固。門虛掩著,裡面透出一股乾燥、冷清的氣息。

**李杰**：(眼睛一亮)清水,妳看!那裡有座房子!

**清水恵美**：(驚喜地抬起頭)真的!太好了!

**旁白**：兩人加快腳步,衝向小屋。李杰小心翼翼地推開門,將受傷的高橋美咲安置在屋內唯一的一張破舊木床上。

**旁白**：屋內雖然簡陋,但角落裡堆著一些乾燥的乾草和幾條發霉的毯子。更重要的是,牆邊有一個小櫥櫃,裡面竟然擺放著一些基本的醫療用品和幾本泛黃的書籍。

**李杰**：(如獲至寶)醫療用品!

**旁白**：他趕緊打開櫥櫃。裡面有幾瓶看起來還沒開封的消毒水、幾卷乾淨的繃帶,甚至還有一些止痛藥和消炎藥膏。

**清水恵美**：(拿起其中一本書,驚訝地唸道) 「森林傷患處理指南」?這...這簡直是量身定做的。

**李杰**：(顧不得感嘆巧合,迅速拿過消毒水和繃帶)別管那麼多了。先幫美咲處理傷口。

**旁白**：兩人按照指南上的步驟,小心翼翼地為美咲處理大腿上的箭傷。美咲痛得渾身發抖,冷汗直流,但她死死咬著牙,沒有發出一聲尖叫。

**旁白**：處理完畢後,美咲虛弱地睡了過去,呼吸變得平緩。疼痛和失血讓她的身體進入了保護性的昏睡。

**李杰**：(擦了擦額頭的汗,長舒一口氣)暫時處理好了。讓她好好休息吧。只要傷口不感染,應該能撐過去。

**清水恵美**：(為美咲蓋上毯子,小心地將毯子塞好,確保她能安穩地睡著)嗯。

**旁白**：兩人都鬆了一口氣。處理傷口的緊張感消退後,疲憊和飢餓感突然湧了上來。

**清水恵美**：(小聲地,有些不好意思)李杰先生...我們...要不要吃點東西?我...我肚子有點餓了...

**李杰**：(苦笑)我也是。剛才太緊張沒注意,現在放鬆下來才發現自己餓得不行。

**旁白**：李杰這才發現自己從派對結束後就沒吃過東西。兩人走到廚房,在櫥櫃裡仔細搜索。

**旁白**：他們找到了一些保存良好的乾糧、醃菜,以及一小袋看起來很新鮮的米。甚至還有一些乾燥的蔬菜和香菇。

**清水恵美**：(看著這些食材)這些東西...看起來不像是放了很久的樣子...這個房子真的很奇怪...

**李杰**：先填飽肚子再說。妳會煮飯嗎?

**清水恵美**：(點頭)我...我試試看...

**旁白**：清水恵美熟練地檢查爐灶,發現裡面還有乾燥的木柴。她用火柴點燃火,開始煮一鍋簡單的粥。

**旁白**：她在粥裡加了一些乾燥蔬菜和香菇,還放了一點點鹽。很快,溫暖的香氣就在小屋裡飄散開來。

**旁白**：兩人圍坐在木桌旁,安靜地吃著。熱粥的溫暖讓疲憊的身體得到了一絲慰藉。

**李杰**：(真誠地)妳的廚藝不錯。在這種情況下還能煮出這麼好吃的粥,不簡單。

**清水恵美**：(臉微紅,小聲地)謝謝...我...我一個人住的時候...常常自己煮...所以習慣了...

**李杰**：一個人住?妳父母不在東京嗎?

**清水恵美**：(低頭看著碗)嗯...父母在福岡...我來東京上大學...一個人租了個小公寓...

**清水恵美**：剛開始很不習慣...但慢慢就學會照顧自己了...煮飯、打掃...

**李杰**：(理解地點頭)一個人生活確實需要學很多東西。我剛來日本的時候也是,什麼都要自己來。

**旁白**：兩人簡短地聊了幾句,都太疲憊了,沒有多餘的精力深談。吃完飯後,清水恵美找到了另外兩條毯子。

**旁白**：毯子雖然有些舊,但很乾淨,還帶著一股淡淡的薰衣草香氣。她將毯子鋪在地板上。

**李杰**：(站起身)妳睡床吧,我睡地板就好。妳照顧了美咲那麼久,也累了。

**清水恵美**：(堅定地搖頭)不行...美咲需要床...她的傷勢比我們嚴重多了。我...我和你一起睡地板就好...

**清水恵美**：而且地板也不算太硬...有毯子墊著應該沒問題...

**李杰**：(看著她堅定的眼神,沒有再堅持)好吧。那我們都早點休息。不知道明天會發生什麼事。

**旁白**：兩人各自裹著毯子,並排躺在地板上,中間保持著適當的距離。

**旁白**：疲憊讓他們很快就進入了夢鄉。屋外,濃霧依然籠罩著森林,但至少這一晚,他們有了一個安全的庇護所。
