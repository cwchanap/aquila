# 第十一幕：啟程的疑惑

**旁白**：李杰和清水恵美在伐木工小屋裡待了兩天。這兩天,他們輪流照顧受傷的高橋美咲。

**旁白**：在精靈藥膏和精心的照料下,美咲的傷口癒合得很快,精神也好了許多。

**旁白**：第三天早晨,也就是李杰判斷應該是「早晨」的時候,美咲終於可以下床走動了。

**高橋美咲**：(扶著牆壁,慢慢走進客廳,臉色雖然還有些蒼白,但眼神中恢復了光采)李杰先生,恵美小姐...謝謝你們這兩天的照顧。我覺得好多了。

**李杰**：(抬頭看向她,露出一個放鬆的微笑)別客氣。妳恢復得比我想像中還快。

**清水恵美**：(開心地跑過去扶住美咲)美咲!妳終於能走路了!太好了!

**旁白**：三人坐在木桌旁,分享著最後一點熱粥。

**李杰**：(放下碗,語氣變得嚴肅)既然美咲已經好多了,我覺得我們該出發了。

**清水恵美**：(有些緊張)出發?去哪裡?

**李杰**：(從口袋裡拿出那張古老的地圖,鋪在桌上)我在這間屋子裡找到了一份更詳細的地圖。

**旁白**：這是一張刻在獸皮上的地圖,上面的標記非常精確。李杰指著地圖上西邊的一個圓圈。

**李杰**：這裡標註著「銀月湖」。而且旁邊有一行小字,雖然我也看不懂是什麼語言,但感覺這裡很重要。

**高橋美咲**：(湊過去看)銀月湖...

**李杰**：而且我發現,這份地圖上的路徑,似乎能繞開那些「守望者」頻繁出沒的區域。

**清水恵美**：(擔心地)可是...山本大叔還沒回來。我們不等他嗎?

**李杰**：(沉默了片刻,嘆口氣)已經兩天了。如果大叔能回來,他早就回來了。

**李杰**：我們不能一直在這裡等下去。這裡的食物和水也快沒了。

**高橋美咲**：(點頭)李杰先生說得對。我們必須前進。只有前進,才有一線生機。

**旁白**：雖然心中充滿了對山本大叔的擔憂和對未來的疑惑,但三人還是開始收拾行李,準備啟程。

**旁白**：李杰將剩餘的藥草和醫療用品裝進背包。清水恵美則細心地將剩下的米和醃菜打包。

**旁白**：就在三人準備走出小屋時,李杰突然停下了腳步。

**清水恵美**：(疑惑)李杰先生?怎麼了?

**李杰**：(盯著地圖上的某個點,眉頭緊鎖)奇怪...

**高橋美咲**：哪裡奇怪?

**李杰**：(指著地圖中心)這裡本來標註著精靈村落...但妳們看,這裡有一處像是被故意塗抹掉的痕跡。

**旁白**：地圖中心原本應該標記精靈村的位置,現在卻是一團模糊的黑色。

**李杰**：(內心)為什麼要把村落的位置塗掉?難道這地圖是村落被毀滅後才畫的?

**李杰**：(搖搖頭,收起地圖)算了,想不通。走吧。

**旁白**：三人推開門,再次踏入濃霧之中。

**旁白**：這一次,李杰背上了背包,走在前面開路。清水恵美扶著美咲緊隨其後。

**旁白**：森林依然安靜得詭異。濃霧在樹木間緩緩流動,遮蔽了所有遠處的景物。

**旁白**：李杰一邊走,一邊不時低頭查看手中的地圖。

**清水恵美**：(小聲地)李杰先生...你有沒有覺得...霧好像變色了?

**李杰**：(停下腳步,仔細觀察)變色?

**旁白**：他這才注意到,原本灰白色的霧氣中,竟然隱約透著一絲淡淡的紫色。

**旁白**：這種紫色非常微弱,不仔細看根本察覺不到。但隨著他們不斷前進,紫色變得越來越明顯。

**高橋美咲**：(臉色慘白)這...這不是好兆頭...

**李杰**：(握緊手中的短刀,語氣冷靜)別自己嚇自己。我們按照地圖的方向走,應該沒問題。

**旁白**：三人繼續在紫色的濃霧中穿行。每一步都像走在棉花上,感覺不到地面的堅實感。

**旁白**：走了約莫一小時,前方的霧氣中突然傳來一陣微弱的樂聲。

**旁白**：這樂聲悠揚婉轉,像是在指引著方向,又像是在警告著什麼。

**李杰**：(停下腳步) 聽到了嗎?

**清水恵美**：(點頭) 像是...有人在唱歌...

**高橋美咲**：(不確定地) 或者...是風聲?

**李杰**：(眼神一凝,指向樂聲傳來的方向) 我們去那邊看看。

**旁白**：三人調整方向,朝著樂聲的源頭走去。

**旁白**：而在此刻，原本熟悉的森林路徑，正在發生他們無法想像的變化。
