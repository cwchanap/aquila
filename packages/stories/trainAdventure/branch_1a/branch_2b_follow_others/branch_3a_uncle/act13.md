# 第十三幕：焚燒的真相

**旁白**：濃霧漸漸散開，眼前的景象終於完整地呈現在三人面前。

**旁白**：那確實是一座村落，但卻是一座被徹底摧毀的村落。

**旁白**：焦炭的氣味撲鼻而來，混雜著一種說不出的腐朽味道。空氣中還飄浮著細微的灰燼，在紫月光下像是黑色的雪花。

**旁白**：大部分的建築物已經被燒成灰燼，只剩下焦黑的木樑斜插在地面上。牆壁坍塌成碎片，散落一地。

**旁白**：這些建築原本應該很精緻——從殘留的雕花窗框和弧形的屋頂輪廓可以看出，它們有著優雅的曲線和細膩的裝飾。但現在一切都化為焦炭。

**旁白**：石板路上佈滿了燒焦的痕跡，有些地方的石頭甚至被高溫燒裂，露出底下黑色的泥土。碎裂的瓦片、變形的金屬器具散落各處。

**旁白**：少數幾棟房屋看起來還算完整，外牆雖然燻黑但結構尚存。門窗洞開，裡面空無一人。風吹過時，破損的門板撞擊門框，發出低沉的嗚咽聲。

**旁白**：更令人心寒的，是零星散落在村落各處的屍體。數量不多，但每一具都讓人心頭一沉。

**旁白**：有一具倒在房屋的廢墟中，身上壓著燒焦的木樑。另外幾具倒在街道上，手臂伸向某個方向，彷彿在逃跑時倒下。

**旁白**：這些年邁的精靈就這樣靜靜地躺在廢墟之中，永遠地沉睡了。

**高橋美咲**：（聲音帶著失望）只是...一座廢墟啊...

**高橋美咲**：我還以為會有人...會有食物或水...

**旁白**：美咲的語氣中帶著一絲遺憾，但她的情緒並沒有太大的波動。她環顧四周，像是在評估這裡是否有任何可用的資源。

**旁白**：就在這時，一直站在旁邊的恵美，腳步突然踉蹌了一下。

**清水恵美**：（聲音顫抖）怎麼會...

**李杰**：（轉頭看她）清水？

**旁白**：恵美的臉色變得蒼白，雙手握緊成拳。她的視線在廢墟間游移，像是在尋找什麼，又像是不敢相信眼前的景象。

**清水恵美**：（聲音哽咽，眼眶泛紅）為什麼...為什麼會變成這樣...

**旁白**：她的身體微微顫抖著，呼吸變得急促。眼淚在眼眶裡打轉，但她努力忍著，不讓它們掉下來。

**清水恵美**：（喃喃自語）我明明...我明明...

**旁白**：恵美的聲音越來越小，但李杰還是聽到了她接下來的話。

**清水恵美**：（聲音幾乎要哭出來）我要怎麼救翔君...我要怎麼救他...

**高橋美咲**：（關切地走近）恵美？妳還好嗎？

**旁白**：恵美猛地回神，用手背抹了抹眼睛。她深吸一口氣，努力穩定自己的情緒。

**清水恵美**：（聲音還在顫抖）我...我沒事...只是...這裡...

**高橋美咲**：（輕輕拍她的肩膀）我知道...這景象確實很可怕...

**旁白**：但美咲的安慰顯然沒有觸及恵美真正的痛苦。恵美只是點了點頭，垂下眼簾，不再說話。

**李杰**：（內心）翔君？那是誰？

**旁白**：李杰站在原地，眼前被焚燒的村落確實令人震撼，但更讓他在意的，是恵美剛才的反應。

**李杰**：（內心）她明顯知道這個地方。她還認識這裡的人。

**旁白**：李杰回想起這一路以來恵美的種種異常——她堅持要往森林中心走，她對方向的確信，她一路上的急切...

**李杰**：（內心）她從一開始就知道我們要來這裡。她一直在隱瞞什麼。

**旁白**：無數的疑問在李杰心中翻騰。恵美到底是誰？她和這個精靈村落有什麼關係？那個叫翔的人又是誰？

**旁白**：李杰看向恵美。她背對著他，肩膀微微顫抖，看起來如此脆弱。

**李杰**：（內心）現在不是時候...她的狀態...

**旁白**：李杰深吸一口氣，壓下心中的疑問。他決定沉默，至少現在沉默。

**李杰**：（沉聲）我們先看看周圍有沒有倖存者。小心一點，注意安全。

**高橋美咲**：好...我去那邊看看。

**旁白**：美咲朝右側幾棟相對完整的房屋走去。恵美站在原地沒有動，只是看著眼前的廢墟。

**旁白**：焚毀的村落在薄霧中顯得更加淒涼。三個人各懷心事，而在那些倒塌的房屋和焦黑的屍體之間，似乎還隱藏著更多未知的秘密，等待著被揭開。
