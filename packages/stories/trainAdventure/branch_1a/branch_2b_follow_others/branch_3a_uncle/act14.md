# 第十四幕：重逢與隔閡

**旁白**：大約過了十分鐘，恵美的情緒漸漸平復下來。

**旁白**：她的眼睛紅腫，睫毛上還掛著淚珠，臉頰上有明顯的淚痕。她用手背抹了臉，深呼吸著，努力讓自己冷靜。

**高橋美咲**：（溫柔地扶著她的手臂）來，恵美，先站起來吧。

**旁白**：美咲小心翼翼地將恵美扶起來。恵美雙腿還有些發軟，靠在美咲肩上才勉強站穩。

**高橋美咲**：（拍拍她的背）恵美，別太難過了。我們...我們一定會找到辦法的。

**高橋美咲**：不管發生了什麼，我們都會一起面對，好嗎？

**清水恵美**：（聲音沙啞）謝謝...美咲...

**旁白**：恵美的聲音聽起來疲憊而遙遠，像是隔著一層厚厚的玻璃。

**李杰**：（走上前，遞過一瓶水）喝點水吧。

**清水恵美**：（接過水瓶，點了點頭）謝謝。

**旁白**：恵美喝了幾口水，蒼白的臉色稍微恢復了一些生氣。她看了看李杰，又看了看美咲，眼神中閃過一絲複雜的情緒。

**清水恵美**：我...我剛才失態了。對不起。

**李杰**：（平靜地）沒關係。在這種地方，誰都會有情緒崩潰的時候。

**李杰**：重要的是，妳現在還好嗎？

**清水恵美**：（勉強笑了笑）我沒事。只是...看到這裡變成這樣，心裡很難受。

**李杰**：（盯著她的眼睛）清水...

**清水恵美**：（避開他的視線）什麼事？

**李杰**：（沈默了一會兒，最終還是開口問道）妳剛才提到的「翔君」...是誰？

**旁白**：恵美的身體僵住了。握著水瓶的手微微用力，指節因為用力而發白。

**清水恵美**：（沈默了很久，聲音微弱）那是我的...一個朋友。

**李杰**：（追問）也是和我們一樣的人類嗎？

**清水恵美**：（點點頭）嗯。

**李杰**：（看向廢墟）他也失蹤了嗎？就在這個村子裡？

**清水恵美**：（聲音變得有些急促）我不知道。但我必須找到他。

**李杰**：（內心）妳怎麼知道他在這裡？

**旁白**：李杰忍住了沒有問出口。他感覺得出來，恵美已經快要到達極限了。如果再追問下去，她可能會再次崩潰。

**李杰**：（語氣緩和了一些）我明白了。我們會幫妳一起找的。

**清水恵美**：（抬起頭，看著李杰，眼中閃過一絲感激）謝謝你，李杰先生。

**旁白**：雖然恵美這麼說，但李杰能感覺到，她和他們之間，已經產生了一道看不見的隔閡。

**旁白**：這道隔閡，不僅僅是因為那些隱瞞的秘密，更是因為兩者對這個世界的認知完全不同。

**旁白**：對李杰和美咲來說，這裡是一個恐怖、陌生、急於逃離的噩夢。

**旁白**：但對恵美來說，這裡似乎隱藏著她生命中極其重要的一部分。

**李杰**：（看向一旁的美咲）美咲，妳那邊有發現什麼嗎？

**高橋美咲**：（搖搖頭）沒有。那邊的房子也都燒空了。除了一些打碎的罈子，什麼都沒有。

**李杰**：(內心)看來這裡真的沒有倖存者了。

**旁白**：就在眾人感到絕望時，遠處突然傳來了一陣微弱的聲音。

**旁白**：像是有人在低聲呻吟，又像是風吹過空瓶子的聲音。

**高橋美咲**：（緊張地抓住李杰的袖子）你聽到了嗎？

**李杰**：（立刻警覺地握緊短刀）嗯。在那邊。

**旁白**：李杰指了指村落邊緣的一座廢墟。那座房子看起來比其他的都要大，雖然也被燒毀了，但還保留著一部分完整的牆壁。

**李杰**：（低聲對兩人說）跟在我後面。小心一點。

**旁白**：三人放輕腳步，朝著聲音的來源慢慢靠近。

**旁白**：越靠近那座廢墟，聲音就越清晰。那確實是人的呻吟聲，帶著極大的痛苦。

**李杰**：（來到廢墟門口，往裡看去） ...

**旁白**：廢墟內，在一堆倒塌的木樑下，隱約可以看到一個人影在動。

**李杰**：（驚呼）有人！快幫忙！

**旁白**：李杰和美咲趕緊衝過去，合力抬起那根沉重的木樑。

**旁白**：當木樑被移開後，一個渾身是血的精靈出現在他們面前。

**旁白**：他看起來很年輕，臉色慘白得近乎透明，胸口有一道深可見骨的傷痕，鮮血正源源不斷地滲出。

**年輕精靈**：（虛弱地睜開眼睛，看著李杰，聲音細如蚊蚋） ...

**李杰**：（湊過去聽）妳說什麼？

**年輕精靈**：（艱難地抬起手，指向森林的另一個方向） 逃...快逃...

**旁白**：話音剛落，年輕精靈的手便頹然垂下，徹底失去了氣息。

**高橋美咲**：（嚇得往後退了一步，捂住嘴巴）他...他死了...

**清水恵美**：（看著精靈的屍體，眼神中充滿了痛苦和悲哀） 對不起...對不起...

**旁白**：李杰看著精靈指的方向。那是村落的北邊，更深、更幽暗的森林。

**李杰**：（內心）逃？逃向那裡嗎？那邊到底有什麼？

**旁白**：就在這時，原本安靜的村落周圍，突然傳來一陣嘈雜的腳步聲。

**旁白**：那是整齊、有力、充滿殺氣的腳步聲。

**李杰**：（臉色大變）有人來了！快躲起來！

**旁白**：三人迅速躲到廢墟的陰影中。透過破碎的牆縫，他們看到一群全副武裝的面具生物，正朝著村落中心開過來。

**旁白**：他們手中握著長矛和弓箭，眼神冰冷而殘酷。

**李杰**：（壓低聲音，對兩人說）別出聲。

**旁白**：一場新的危機，正悄悄降臨。

**李杰**：（內心）如果妳知道些什麼，也許可以幫助我們理解現在的狀況...

**旁白**：恵美沉默了很久。她的視線從李杰臉上移開，落在地面的焦黑石板上。

**旁白**：她的手指不自覺地握緊又鬆開，像是在進行某種內心的掙扎。

**旁白**：美咲察覺到氣氛的變化，停止了說話，只是安靜地站在旁邊。

**旁白**：最後，恵美深吸一口氣，抬起頭。她的眼神變得平靜，但那是一種築起高牆後的平靜。

**清水恵美**：李杰先生...等時機到了，我會告訴你我知道的一切。

**清水恵美**：但是現在...請讓我保持沉默。

**旁白**：她的聲音很平靜，卻帶著一種不容置疑的堅定。

**李杰**：我明白了。

**旁白**：李杰的語氣很平淡，但心中卻泛起了複雜的情緒。

**李杰**：（內心）她在隱瞞...而且是很重要的事情。

**李杰**：（內心）我不知道她的目的是什麼，但從現在開始，我必須更加小心。

**旁白**：一顆名為「不信任」的種子，在李杰心中悄悄發芽。

**高橋美咲**：（似乎想要緩和氣氛，拍了拍手）好啦！既然我們都到這裡了，不如四處看看吧？

**高橋美咲**：說不定能找到什麼有用的東西！

**旁白**：美咲的語氣輕快，彷彿試圖用她的樂觀來驅散這片廢墟帶來的陰霾。

**高橋美咲**：而且...也許能找到關於其他人的線索？拓海他們...還有山本大叔...

**旁白**：就在美咲話音剛落的瞬間，遠處突然傳來了一個熟悉的聲音。

**？？？**：李杰！恵美！

**旁白**：三人同時一愣。

**李杰**：（猛地轉身）這聲音...

**旁白**：濃霧中，一個高大的身影正快步朝他們走來。腳步聲急促而沉重，踩在碎石上發出清脆的響聲。

**旁白**：隨著距離拉近，那個人的面容漸漸清晰——灰白的頭髮、寬闊的肩膀、風塵僕僕的神情。

**李杰**：（驚訝地）山本大叔？！

**旁白**：那正是山本浩二！

**旁白**：他的外套有些破損，袖口處有明顯的撕裂痕跡。臉上沾著灰塵和泥土，額頭上還有一道淺淺的擦傷。但看起來並無大礙。

**旁白**：他加快腳步，臉上露出如釋重負的笑容，眼角甚至有些濕潤。

**山本浩二**：（喘著氣，聲音激動）太好了！你們還活著！

**山本浩二**：我還以為...我還以為再也見不到你們了！

**旁白**：山本大叔的聲音中帶著難以掩飾的喜悅和慶幸，在這片死寂的廢墟中，顯得格外溫暖。
