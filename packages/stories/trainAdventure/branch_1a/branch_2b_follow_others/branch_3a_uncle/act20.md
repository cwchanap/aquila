# 第二十幕：絕望的追逐

**旁白**：李杰和山本大叔背著恵美，在濃霧中狂奔。

**旁白**：身後，凌墨那冰冷而沈重的腳步聲依然不緊不慢地跟著。

**旁白**：那聲音像是一下又一下敲擊在他們的心尖上，讓人窒息。

**山本浩二**：（大口喘著氣）李杰...不行了...這霧太大了...我們根本分不清方向...

**旁白**：李杰的衣服已經被汗水和鮮血浸透。背上的恵美已經陷入了昏迷，呼吸微弱得幾乎感覺不到。

**李杰**：（咬牙）不能停下！一旦停下就全完了！

**旁白**：然而，命運似乎並不打算放過他們。

**旁白**：前方原本隱約可見的路徑，突然消失在了一片斷崖前。

**李杰**：（猛地收住腳步，石塊落入深淵的聲音清晰可聞）該死！

**旁白**：這是一個直徑數百米的圓形深淵。下方是一片漆黑，什麼也看不見。

**旁白**：他們被逼到了絕路。

**旁白**：而在此刻，凌墨的身影緩緩從霧中走出。

**旁白**：他手中的黑劍在地上拖行，發出刺耳的摩擦聲。

**凌墨**：（聲音沙啞，不帶感情）沒路了。

**山本浩二**：（握緊木鏟，擋在李杰身前）妳這怪物...到底想怎麼樣！

**凌墨**：（指向李杰背上的恵美）把她交給我。

**李杰**：（怒吼）妳做夢！

**凌墨**：（嘆了口氣，眼神中閃過一絲疲憊）妳們不明白...只有殺了她，這個世界才能重啟...

**李杰**：（震驚）重啟？

**凌墨**：（語氣變得瘋狂）這是一個錯誤的世界！一個充滿bug的牢籠！

**凌墨**：（指向高塔）只有徹底清除這些數據，我們才能回到真正的現實！

**旁白**：就在凌墨準備動手時，斷崖下方突然爆發出一陣劇烈的震動。

**旁白**：伴隨著一聲震耳欲聾的咆哮，一條巨大的、通體透明的藍色巨龍從深淵中沖天而起！

**李杰**：（被強大的風壓震退數步）那是...精靈守護神？！

**藍色巨龍**：（發出神聖而威嚴的鳴叫，巨大的尾巴猛地掃向凌墨） ...

**凌墨**：（臉色大變，舉劍迎擊）鐺——！

**旁白**：劇烈的碰撞爆發出耀眼的強光。凌墨被這股排山倒海的力量震飛了出去，消失在遠處的濃霧中。

**藍色巨龍**：（轉過頭，柔和的目光落在恵美身上） ...

**旁白**：巨龍緩緩低下頭，伸出巨大的龍爪，輕輕托住了李杰和恵美。

**李杰**：（震撼地看著眼前的巨龍）妳...妳是在救我們嗎？

**藍色巨龍**：（發出一聲溫柔的低吟，隨即振動雙翼，帶著兩人飛向高空） ...

**山本浩二**：（呆呆地站在斷崖邊，看著飛遠的巨龍）天啊...

**旁白**：在這場絕望的追逐最後，奇蹟發生了。

**旁白**：而在這寂靜的夜晚，三個陌生人的命運，就這樣在銀月湖畔交會了。
