# 第二十五幕：艾拉如的治療

**旁白**：大屋內部寬敞明亮。牆上掛著發光的植物，散發著柔和的銀藍色光芒。空氣中飄散著淡淡的草藥香氣，讓人感到安心。

**旁白**：屋內有幾張木製的床鋪，還有一些櫥櫃和桌椅。所有的家具都透著一種自然的美感，像是從樹木中直接生長出來的。

**艾拉如**：（對凱蘭示意）把她放到那張床上。

**旁白**：凱蘭小心翼翼地將恵美扶到床上，讓她平躺下來。恵美臉色蒼白，呼吸有些急促。

**艾拉如**：（走到床邊，看著恵美的傷口）傷得不輕...不過別擔心，很快就會好的。

**旁白**：艾拉如將手掌輕輕放在恵美側腰的傷口上方，閉上眼睛。

**旁白**：下一瞬間，他的手掌開始散發出溫暖的綠色光芒。光芒像水一樣流動，慢慢包裹住恵美的傷口。

**清水恵美**：（感覺到溫暖，輕聲）好溫暖...

**旁白**：李杰和山本大叔站在一旁，驚訝地看著這一幕。雖然他們之前在村子裡也見過一些神奇的事情，但這種直接的治療術還是第一次親眼所見。

**旁白**：大約過了五分鐘，綠色光芒逐漸消散。艾拉如收回手，長長地吐出一口氣，額頭上滲出了一些細汗。

**艾拉如**：（擦了擦汗，微笑著對兩人說）好了。傷口已經癒合了大半，剩下的需要靠藥物和休息來慢慢恢復。

**李杰**：（驚訝地看向恵美的傷口）癒合了？

**旁白**：李杰走上前。果然，原本深可見骨的傷口此刻已經縮小了很多，邊緣也已經開始結痂，鮮血完全止住了。

**山本浩二**：（感嘆）這簡直是奇蹟...

**艾拉如**：（對凱蘭說）凱蘭，去拿些「星辰草」和「月光露」來。

**凱蘭**：是，大人。

**旁白**：凱蘭快步走向櫥櫃。

**艾拉如**：（指著另一邊的床鋪對李杰和山本大叔說）兩位也先坐下休息吧。我看你們也受了不少傷。

**旁白**：李杰這才意識到，自己身上也有很多處擦傷和淤青。山本大叔的手臂上也有幾道被樹枝劃開的口子。

**艾拉如**：（溫和地說）凱蘭會幫你們處理傷口的。等一下我會讓人準備一些食物。

**李杰**：（坐下，感激地）謝謝您，艾拉如大人。

**旁白**：很快，凱蘭帶著藥草和水回來了。他先是幫李杰和山本大叔清洗並包紮了傷口，然後又調製了一碗綠色的藥汁，讓恵美喝下。

**旁白**：喝下藥汁後，恵美的臉色明顯好轉，眼神也變得清明瞭一些。

**清水恵美**：（虛弱地）謝謝艾拉如大人...救命之恩...

**艾拉如**：（坐在床邊，慈祥地看著她）別說這些客氣話。妳就像我的孩子一樣。

**艾拉如**：（頓了頓，語氣變得嚴肅）不過，孩子...妳能告訴我，村子到底發生了什麼事嗎？為什麼會變成那樣？

**旁白**：聽到這個問題，恵美的眼神中閃過一絲痛苦。她咬著下唇，聲音顫抖地開始講述那一晚的慘劇。

**旁白**：從突如其來的火災，到面具生物的襲擊，再到那個可怕的黑衣男凌墨...恵美說得很詳細，語氣中充滿了絕望。

**旁白**：艾拉如安靜地聽著，眉頭越鎖越深。

**艾拉如**：（沈默了很久，長嘆一口氣）原來是這樣...沒想到...那種黑暗力量竟然已經蔓延到這裡了...

**清水恵美**：（緊張地）那...那您知道是誰做的嗎？

**艾拉如**：（搖頭，表情沈重）不知道。我們沒有看到任何可疑的人。

**艾拉如**：當時情況太混亂了，大家都忙著逃命和救人。等火勢被控制住，放火的人早就不見蹤影了。

**旁白**：李杰猶豫了一下，開口問道。

**李杰**：艾拉如大人...我想問您一個問題。

**李杰：您還記得...凌墨嗎？

**旁白**：艾拉如的身體明顯一震。他猛地轉過頭，眼中閃過驚訝的神色。

**艾拉如**：（聲音帶著驚訝）凌墨...你怎麼會知道這個名字？

**旁白**：艾拉如的目光轉向床上的恵美，眼神中帶著詢問。

**艾拉如**：恵美...妳把一切都告訴他們了？

**清水恵美**：（點頭，聲音低沉）是的...艾拉如大人...我都告訴他們了...

**清水恵美**：李杰先生和山本大叔...他們救了我...他們有權知道真相...

**艾拉如**：（沉默片刻，點頭）我明白了。

**清水恵美**：（聲音顫抖）而且...凌墨他...他出現了...

**清水恵美**：但他...他好像失去理智了...他的眼神空洞，行動緩慢，但卻擁有可怕的力量...

**清水恵美**：他殺了人...用一把黑色的劍...

**山本浩二**：（補充）那傢伙的力氣大得驚人。我用盡全力打他，他連痛都不覺得...

**山本浩二**：而且那把劍...我們完全拿不起來，就像被釘在地上一樣...

**清水恵美**：（聲音哽咽）那把劍...不知道從哪裡冒出來的...凌墨以前從來沒有那種東西...

**旁白**：艾拉如聽到「黑色的劍」這四個字，臉色瞬間變得鐵青。

**旁白**：他的手顫抖著，緊緊抓住窗框。那雙溫和的眼睛裡第一次流露出恐懼的神色。

**艾拉如**：（喃喃自語，聲音顫抖）天啊...怎麼可能...那把劍...怎麼又出現了...

**李杰**：（警覺地）艾拉如大人？您知道那把劍？

**旁白**：艾拉如沒有回答。他只是站在窗邊，身體微微顫抖，眼神中充滿了深深的恐懼和不安。

**李杰**：（內心）不妙...這反應...那把劍絕對不是什麼好東西...

**旁白**：李杰正想再問更多，突然——

**旁白**：大門被猛地推開！一個年輕的精靈慌張地衝進屋內，臉上滿是驚恐。

**年輕精靈**：（喘著氣，聲音急促）艾拉如大人！不好了！

**年輕精靈**：有...有個可疑的男人...拿著一把黑色的劍...

**年輕精靈**：他正朝這裡走過來！動作很慢，但一直在往這個方向移動！

**旁白**：屋內的空氣瞬間凝固。

**旁白**：李杰、山本大叔和恵美的臉色同時變得蒼白。他們腦海中浮現出同一個畫面——那個手持黑劍、眼神空洞的男人。

**清水恵美**：（驚恐地）不...不會吧...凌墨...他怎麼找到這裡的...

**山本浩二**：（站起身，握緊拳頭）該死...他又來了...
