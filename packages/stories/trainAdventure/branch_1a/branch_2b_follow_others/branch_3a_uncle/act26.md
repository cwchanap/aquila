# 第二十六幕：無力的抵抗

**旁白**：李杰、山本大叔、艾拉如和那個年輕精靈衝出大屋。恵美也勉強從床上爬起來，跟著走到門口。

**旁白**：銀月湖邊的景象讓所有人心頭一緊。

**旁白**：凌墨站在湖邊的草地上，距離大屋約五十公尺。他手中握著那把漆黑的長劍，一步一步緩慢地朝著精靈聚居的方向走來。

**旁白**：他的動作依然緩慢，像是每一步都承受著巨大的重量。但那種壓迫感，讓周圍的空氣都變得沈重。

**旁白**：幾個精靈弓箭手已經注意到了他。他們拉開弓弦，箭矢對準凌墨。

**精靈弓箭手A**：（大喊）站住！不許再靠近！

**旁白**：凌墨沒有回應。他繼續往前走，眼神空洞，像是完全沒有聽到警告。

**精靈弓箭手B**：（對同伴喊道）放箭！

**旁白**：數十支箭矢劃破長空，朝著凌墨射去。

**旁白**：然而，詭異的事情發生了。

**旁白**：當箭矢進入凌墨身周三公尺的範圍時，它們突然像是撞上了一堵無形的牆，紛紛在空中爆裂成齏粉！

**精靈弓箭手A**：（驚恐）什麼？！

**旁白**：凌墨依然面無表情。他隨意地揮動了一下手中的黑劍。

**旁白**：一道黑色的弧光閃過，湖邊的地面瞬間被撕開一道十幾公尺長的裂縫！強大的衝擊波將附近的幾個精靈震飛出去。

**李杰**：（咬牙）這傢伙...比之前更強了。

**山本浩二**：（握緊拳頭）我們得阻止他！

**艾拉如**：（伸手攔住兩人，語氣沈重）不。普通的力量對他沒用。那把劍...正在吞噬周圍的一切。

**艾拉如**：（看向李杰）必須立刻疏散族人！

**艾拉如**：（轉頭對年輕精靈喊道）拉響警報！讓所有人往神樹那邊撤離！快！

**年輕精靈**：是！大人！

**旁白**：刺耳的警報聲響徹整個月光林地。精靈們從房屋中衝出來，開始慌亂地往北方撤離。

**李杰**：（對艾拉如說）艾拉如大人，我們留下幫忙！

**艾拉如**：（看著兩人，眼中閃過一絲果決）好。但你們要記住，絕對不要硬碰硬。

**艾拉如**：（停頓了一下）有一個地方...

**艾拉如**：北方的神樹。那是我們精靈族最神聖的地方，也是自然之力最強大的地方。

**艾拉如**：神樹散發的力量可以壓制附近的邪惡力量。我想...那個人應該不會靠近那個區域。

**李杰**：那我們躲到那裡應該就安全了？

**艾拉如**：（點頭）理論上是這樣。但是...

**旁白**：艾拉如的目光掃過湖邊慌亂撤離的精靈們。老人動作緩慢，孩子們需要被抱著，年輕的精靈攙扶著年長者。

**艾拉如**：疏散需要時間。尤其是孩子和老人，他們走得很慢。

**艾拉如**：如果那個人一直跟著我們...我們根本來不及到達神樹...

**旁白**：艾拉如的話語中帶著深深的憂慮。

**艾拉如**：我們需要有人...留下來拖延他。至少...至少要爭取足夠的時間。

**旁白**：就在這時，五個身穿輕甲的精靈劍士從人群中走了出來。他們腰間掛著細長的劍，眼神堅定。

**旁白**：他們站成一排，擋在凌墨前進的路上。帶頭的是之前見過的凱蘭，還有四個年輕的精靈戰士。

**精靈劍士凱蘭**：（拔出劍，聲音堅定）艾拉如大人，我們留下。

**精靈劍士凱蘭**：我們會盡全力拖住他。請您帶領族人前往神樹。

**其他精靈劍士**：（齊聲）我們不會讓他追上去的！

**旁白**：艾拉如看著這五個年輕的戰士，眼中閃過痛苦的神色。他知道，這很可能是訣別。

**艾拉如**：（聲音沙啞）凱蘭...你們...

**精靈劍士凱蘭**：（微笑）這是我們的職責，艾拉如大人。請不要猶豫。

**旁白**：艾拉如：（深吸一口氣，轉向李杰等人）我也會留下。

**艾拉如**：我會試著吸引他的注意力，把他引向另一個方向。

**艾拉如**：在所有族人到達安全區域之前，我們必須阻止他追上撤離的隊伍。

**李杰**：（急切地）但是艾拉如大人，您...

**艾拉如**：（打斷他，語氣溫和但堅定）你們跟著族人一起走。

**艾拉如**：恵美的傷還沒好，她需要有人照顧。還有...

**旁白**：艾拉如看著李杰和山本大叔，眼神中帶著懇求。

**艾拉如**：如果我們失敗了...請你們保護好我的族人。

**艾拉如**：尤其是孩子們...拜託了。

**山本浩二**：（握緊拳頭，沉默）...

**李杰**：（內心掙扎）我們...我們真的要逃跑嗎...

**旁白**：遠處，凌墨還在緩慢地前進。每一步都帶著沈重的壓迫感。

**旁白**：五個精靈劍士拔出劍，擺好戰鬥姿勢。艾拉如也握緊拳頭，準備迎戰。

**旁白**：而在他們身後，精靈們正拖家帶口地往北方撤離。老人、孩子、婦女...所有人的生命都寄託在這幾個留下的人身上。

**旁白**：選擇的時刻到了。
