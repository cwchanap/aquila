# 第九幕：濃霧中的謎團

**旁白**：李杰正準備進一步檢查多話女大生的傷勢時,身後的樹上突然傳來一陣窸窣聲。

**旁白**：樹葉的沙沙聲在這片寂靜的森林中格外清晰。一個模糊的黑影從樹枝上一躍而下。

**旁白**：那道身影輕盈敏捷,落地幾乎沒有發出聲響,隨即迅速竄入濃霧深處。

**山本浩二**：(警覺地轉身,大聲喝道)什麼人!給我站住!

**旁白**：那道黑影的動作極其敏捷,像是森林中的野獸般靈活。眨眼間就要消失在白茫茫的霧氣中。

**旁白**：山本浩二毫不猶豫,立刻拔腿追了上去。他的身手比想像中靈活,多年建築工作鍛煉出的體力在這時展現無遺。

**山本浩二**：(邊跑邊喊,聲音在霧中迴盪)小兄弟!你們照顧這個女孩!我去追!別擔心,我很快就回來!

**李杰**：(大聲回應)大叔小心!別追太遠!

**旁白**：李杰看著山本大叔的身影消失在濃霧中,心中充滿了不安。但在這種情況下,他也沒辦法丟下受傷的女孩不管。

**李杰**：(轉頭看向清水恵美,語氣急促)清水,妳身上有急救包嗎?或者紙巾也行。

**清水恵美**：(慌張地翻找背包)有、有的...我這有紙巾...還有消毒濕巾...

**旁白**：清水恵美從背包裡掏出一大包紙巾和一盒濕巾,手抖得厲害。

**李杰**：(接過紙巾)很好。妳過來幫我按住傷口。

**旁白**：李杰小心翼翼地揭開剛才山本大叔包紮用的外套。女孩大腿上的傷口還在滲血,箭頭深深地沒入肉中。

**李杰**：(低聲)這箭不能隨便拔...會大出血的。我們先止血。

**旁白**：李杰用大量的紙巾疊在傷口周圍,用力按壓。女孩發出一聲低沉的呻吟,眉頭緊皺,冷汗直冒。

**清水恵美**：(看著鮮血染紅了紙巾,臉色慘白)李杰先生...她、她會死嗎?

**李杰**：(咬牙)不會的。只要能及時止血...大叔去追那個黑影了,應該很快就會帶人回來幫忙。

**旁白**：時間一分一秒地過去。整片森林靜得可怕,除了微弱的呼吸聲和樹葉偶尔落下的聲音,什麼也聽不見。

**旁白**：濃霧依然沒有散去的跡象。原本熟悉的森林,在此刻顯得如此陌生而危險。

**李杰**：(內心)太久了...大叔已經去了快十五分鐘了...怎麼還沒回來?

**旁白**：李杰心中的不安感越來越強烈。他抬頭看向大叔消失的方向,除了白茫茫的一片,什麼也看不見。

**李杰**：(看向清水恵美)清水,妳留在這裡。如果有人過來,就大聲呼救。

**清水恵美**：(驚恐地抓住他的袖子)妳、妳要去哪?別丟下我一個人!

**李杰**：(指了指大叔消失的方向)我去接一下大叔。我很快就回來。

**清水恵美**：(拼命搖頭)不要!太危險了!萬一你也回不來怎麼辦?

**旁白**：李杰看著清水恵美驚恐的眼神,又看了看躺在地上一動不動的多話女大生,陷入了掙扎。

**李杰**：(內心)不能丟下她們兩個女孩在這種地方...但大叔那邊肯定出事了...

**旁白**：就在這時,森林深處傳來一聲悽厲的尖叫。

**旁白**：聲音短促而尖銳,隨即戛然而止。

**清水恵美**：(嚇得癱倒在地)那、那是大叔的聲音!

**李杰**：(臉色大變,霍然站起)大叔!

**旁白**：聲音是從大叔消失的方向傳來的,但聽起來比剛才跑得更遠了。

**李杰**：(咬牙,看向清水恵美)我不能再等了。清水,妳抱著她,我們一起往大叔那個方向走!

**清水恵美**：(結巴)可、可是李杰先生...我們不認識路啊...

**李杰**：(堅決地)跟著血跡走!

**旁白**：李杰小心翼翼地將雙手伸到多話女大生身下。一手托著她的背,一手環過她的膝蓋下方,小心避開那支插在大腿上的箭。

**旁白**：他深吸一氣氣,用力將她抱了起來。女孩因為疼痛發出微弱的呻吟,身體無力地靠在他懷中。

**李杰**：(內心)比想像中輕...但是大腿上的箭不能碰,手臂的傷口還在流血...必須盡快找到地方處理。

**多話女大生**：(虛弱地)謝謝...謝謝你們...對不起...給你們添麻煩了...

**李杰**：(溫和地)別說話,保存體力。妳會沒事的。

**李杰**：(看向清水恵美)我們走主幹道。不回隱藏路線了。

**清水恵美**：(疑惑地)為什麼?隱藏路線不是比較近嗎?

**李杰**：(邊走邊解釋)隱藏路線太危險,那些精靈可能還在附近。主幹道至少比較開闊,視線也比較好,不容易被偷襲。

**旁白**：李杰用下巴示意清水恵美幫他從口袋裡取出那張古老的地圖。

**旁白**：清水恵美顫抖著手從他的口袋裡掏出地圖,在微弱的月光下展開。泛黃的紙張在她手中微微顫動。

**清水恵美**：(盯著地圖看了一會兒,困惑地)李杰先生...這張地圖...好像沒什麼用...

**李杰**：(皺眉)怎麼說?

**清水恵美**：(指著地圖)上面雖然有標記建築物和路徑...但是...我們根本不知道自己現在在哪裡...

**清水恵美**：沒有GPS或指南針,我們沒辦法定位...這張地圖就算畫得再詳細,也不知道該往哪個方向走...

**李杰**：(沉默片刻,點點頭)妳說得對...我一時沒想到這點。

**旁白**：清水恵美小心翼翼地將地圖收回李杰的口袋。兩人在濃霧中對視了一眼,眼中都透露出一絲無奈。

**李杰**：(深吸一氣氣,語氣變得果斷)那就只能憑感覺走了。我們隨便選一個方向,往前走。

**清水恵美**：(擔心地)可是...萬一走錯方向...或者...迷路了怎麼辦?

**李杰**：(思考片刻)這樣吧。我們一邊走,一邊在路上做記號。

**旁白**：李杰環顧四周,目光落在路邊的樹枝和石塊上。

**李杰**：看到那些樹枝了嗎?我們把它們折斷,或者移動路邊的石頭,製造一些明顯的痕跡。

**李杰**：如果我們真的迷路了,至少能知道自己有沒有走回原來的地方。

**清水恵美**：(恍然大悟,點頭)我明白了...這樣就算走錯路,也能避免一直在同一個地方打轉...

**李杰**：(看向前方的濃霧)就這麼辦。我們往這個方向走。清水,妳幫忙留意路邊,把能移動的東西都弄亂一點。

**旁白**：清水恵美點點頭,開始在路邊尋找可以做標記的東西。她彎下腰,將幾塊石頭堆成小堆,又折斷了幾根低矮的樹枝。

**旁白**：李杰抱著受傷的多話女大生,清水恵美緊緊跟在旁邊,兩人一邊前進,一邊在路上留下標記。

**旁白**：腳下的路徑隱約可見,每一步都要小心翼翼。身後的森林深處,偶爾傳來一兩聲詭異的鳥鳴,像是某種警告。

**清水恵美**：(一邊折斷樹枝,一邊小聲說)李杰先生...我在這裡放了三根樹枝...這樣應該很明顯了...

**李杰**：(點點頭)很好。記住每個標記的位置和特徵,這樣我們就能認出來。

**旁白**：兩人繼續在濃霧中摸索前進。清水恵美每隔一段距離就停下來,移動石頭或折斷樹枝,留下一個又一個標記。

**李杰**：(內心)山本大叔...希望你平安無事。還有那三個走散的學生...那個沉穩的男生、健談的男生和活潑的女生...

**李杰**：(內心)這片森林到底隱藏著什麼秘密?那些精靈為什麼要攻擊他們?而剛才那個黑影...又是誰?

**旁白**：而在那片霧氣深處,某個未知的威脅,正在等待著下一個獵物。
