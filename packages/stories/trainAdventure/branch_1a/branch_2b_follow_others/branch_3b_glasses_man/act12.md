# 第十二幕：重逢的小屋

**旁白**：李杰和斎藤屏住呼吸，死死盯著那個緩緩開啟的儲物櫃。蒼白的手指扣在金屬門緣，發出令人牙酸的摩擦聲。

**李杰**：(握緊手中的木棍，隨時準備出擊) 出來！

**旁白**：隨著櫃門完全打開，一個嬌小的身影從裡面跌了出來，重重地摔在水泥月台上。

**斎藤蓮**：(看清來人，驚呼出聲) 清水？！

**旁白**：跌出來的正是清水恵美。她臉色慘白，眼神中充滿了驚恐。她緊緊抱著自己的雙肩，身體劇烈地顫抖著。

**清水恵美**：(看到李杰和斎藤，眼淚奪眶而出) 李杰先生...斎藤先生...嗚嗚...

**李杰**：(趕緊跑過去將她扶起) 沒事了，別哭。妳怎麼會在這裡？

**清水恵美**：(抽泣著說) 我們...我們被那些怪物追...跑散了...我看到這裡有個櫃子，就躲了進去...

**斎藤蓮**：(有些尷尬地推了推眼鏡) 妳倒是挺機靈的...不像我...

**李杰**：(安慰地拍了拍她的背) 好了，別怕。妳沒受傷吧？

**清水恵美**：(搖了搖頭) 我沒事...但是大叔他們...

**李杰**：(眼神一凝) 我知道。剛才斎藤說他們被帶進隧道里了。

**旁白**：清水聽到這話，身體顫抖得更厲害了。

**清水恵美**：隧道...那裡面...那裡面好可怕...我剛才聽到隧道里傳來好奇怪的聲音...

**李杰**：(看向隧道深處，語氣沉重) 雖然可怕，但我們不能丟下他們不管。

**旁白**：就在三人準備前往隧道時，車站的廣播系統突然發出一陣刺耳的雜訊。

**廣播**：(發出一種扭曲、機械的聲音) 各位...旅客...歡迎來到...幽冥之都...終點站...

**斎藤蓮**：(嚇得尖叫) 誰在說話！

**廣播**：(繼續播報) 下一班...列車...即將進站...請...做好準備...

**李杰**：(內心獨白) 列車？這種地方還有列車？

**旁白**：隨著廣播聲落下，遠處的鐵軌深處傳來一陣低沈的轟鳴聲，伴隨著鐵軌劇烈的震動。一道雪亮的燈光刺破隧道的黑暗，正朝著月台高速逼近。

**李杰**：(大喊) 快！躲起來！

**旁白**：三人迅速躲到一排長椅後面。片刻後，一列漆黑的列車衝出隧道，在月台邊緩緩停下。這列火車與他們之前搭乘的那列非常相似，但車身上刻滿了奇異的符文，散發著淡淡的紫光。

**旁白**：車門緩緩打開，但卻沒有乘客下來。

**李杰**：(小聲對兩人說) 我們趁現在進隧道。那些守衛可能都被這列火車吸引了注意力。

**旁白**：三人藉著列車的掩護，悄悄溜進了幽暗的隧道中。隧道內部陰暗潮濕，牆壁上掛著一些散發著微光的苔蘚，勉強提供了照明。

**旁白**：走了約莫十分鐘，前方出現了一道微弱的火光。李杰示意兩人停下，自己則小心翼翼地靠近。

**旁白**：隧道的一側有一間廢棄的鐵路工人小屋。火光正是從窗戶里透出來的。李杰湊近窗戶，看清了屋內的景象。

**旁白**：大叔被綁在一張椅子上，昏迷不醒。而那對夫妻——伊藤誠和伊藤奈々，則蜷縮在角落裡，瑟瑟發抖。兩個面具生物正圍著火堆，低聲商量著什麼。

**李杰**：(內心獨白) 找到了！

**旁白**：他回頭對清水和斎藤做了個手勢。

**李杰**：(小聲說) 一會兒我衝進去吸引注意力，斎藤你負責解開大叔的繩子，清水妳去帶那對夫妻跑。聽明白了嗎？

**斎藤蓮**：(臉色發白，但還是點了點頭) 明白。

**清水恵美**：(堅定地點頭) 嗯。

**李杰**：(握緊木棍) 好。預備...衝！

**旁白**：李杰猛地踹開房門，大喝一聲衝了進去！

**李杰**：(揮動木棍) 嘿！看這裡！

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
