# 第十五幕：岔路的相遇

**旁白**：白石小屋內部的裝修極其簡約且自然。牆壁由發光的礦石點綴，提供著柔和而溫暖的照明。室內瀰漫著一種淡淡的草藥清香，讓人感到心曠神怡。

**旁白**：艾拉如指引大叔將受傷的女孩放在一張鋪滿柔軟獸皮的石床上。她從牆上的木架上取下幾瓶綠色的液體，又採摘了一些新鮮的植物葉片，開始熟練地調配藥劑。

**精靈女性**：(一邊動作一邊輕聲說道) 別擔心。這支箭雖然有毒，但還沒有侵入心臟。

**李杰**：(驚訝) 妳知道這箭有毒？

**精靈女性**：(點點頭) 這是「守望者」常用的狩獵武器。他們並不打算立刻殺死獵物，而是讓其失去行動能力。

**旁白**：艾拉如將調配好的藥膏輕輕敷在女孩的傷口上，又給她餵下了一些藥劑。奇蹟般地，女孩原本慘白的臉色開始恢復紅潤，呼吸也變得平穩起來。

**多話女大生**：(悠悠轉醒，聲音雖然虛弱但清晰) 唔...我是在夢裡嗎？

**清水恵美**：(驚喜地拉住她的手) 妳醒了！太好了！

**旁白**：眾人懸著的心終於落了下來。在艾拉如的照顧下，大家都得到了一次難得的修整。

**李杰**：(走到艾拉如身邊，認真地問道) 艾拉如，妳說的「守望者」，到底是什麽東西？

**精靈女性**：(嘆了口氣，眼神變得憂傷) 他們曾是這座城市的守護者。但在幾百年前的一場災難中，他們失去了本心，淪為了黑暗力量的傀儡。

**李杰**：(追問) 災難？什麼災難？

**精靈女性**：(搖了搖頭) 那是一段被詛咒的歷史。我不能說太多。我只能告訴你們，這座城市正在崩塌。而你們的到來，似乎加劇了這個過程。

**李杰**：(內心獨白) 加劇？難道我們就是那個「bug」？

**精靈女性**：(看向窗外的高塔) 如果你們想活下去，就必須在那座高塔崩塌前，找到「時空之匙」。

**斎藤蓮**：(有些不耐煩) 又是匙，又是塔。這簡直像是在玩遊戲。

**李杰**：(沒有理會斎藤，繼續問道) 妳知道匙在那裡嗎？

**精靈女性**：(指向森林南邊的一個峽谷) 亡者峽谷。那裡曾是這座城市的祭壇。但現在，那裡是黑暗力量最濃郁的地方。

**旁白**：就在眾人商討對策時，小屋的門外傳來一陣急促的敲門聲。

**旁白**：李杰立刻握緊木棍，警覺地看向門口。艾拉如則輕輕一揮手，門緩緩打開。

**旁白**：站在門外的，是一群穿著各異的人。他們看起來同樣疲憊、恐懼，眼神中充滿了迷茫。

**陌生青年**：(看到李杰等人，愣了一下) 妳們...也是從火車上下來的？

**李杰**：(點點頭) 是的。妳們是誰？

**陌生青年**：(有些激動) 我叫健太。我們是從另一列火車上下來的乘客！

**旁白**：眾人皆是一震。另一列火車？

**李杰**：(內心獨白) 難道失蹤的列車不只我們這一列？

**旁白**：在寧靜之谷的小屋前，兩群迷失的人，就這樣在命運的指引下相遇了。而在這座死寂的車站深處，某些被塵封已久的秘密，即將被揭開。
