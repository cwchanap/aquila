# 第十六幕：前往銀月湖

**旁白**：小屋外的氣氛變得有些微妙。兩群陌生人互相打量著，眼神中既有戒備，也有見到同類的狂喜。

**健太**：(喘著氣) 我們在那邊被一些怪物追了好久...好不容易才逃到這。

**李杰**：(皺眉) 你們那邊也有怪物？

**健太**：(點頭如搗蒜) 全身黑漆漆的，戴著白面具，力氣大得驚人！

**斎藤蓮**：(推了推眼鏡) 得，看來這些「守望者」是全方位服務啊。

**李杰**：(看向艾拉如) 他們能進來嗎？

**精靈女性**：(搖了搖頭) 寧靜之谷受神聖力量保護。只要不離開這片山谷，暫時是安全的。

**旁白**：這番話讓眾人都鬆了一口氣。健太帶領的人群大約有十幾個，男女老少都有，每個人看起來都極其狼狽。

**慷慨大叔**：(站起身) 既然大家都遇到了，就一起想辦法吧。艾拉如，妳說的「亡者峽谷」，離這裡遠嗎？

**精靈女性**：(眼神憂慮) 路程倒是不遠。穿過這片森林，往南走兩小時就到了。但那裡的危險，絕非你們可以想像。

**李杰**：(內心獨白) 沒辦法。不去那裡，我們永遠回不了家。

**旁白**：李杰將健太拉到一邊，詳細詢問了他們火車失蹤的情況。

**健太**：(回憶道) 我們那班車是下午三點發車的。本來一切正常，進了一個隧道後，光線就變了。等門再開的時候，我們就已經在那座廢棄車站了。

**李杰**：(思索著) 下午三點...我們是晚上十一點。看來，這列火車會隨機在不同的時間段捕獲乘客。

**斎藤蓮**：(插話道) 或者是某種定時的收割。

**旁白**：修整了約一個小時後，受傷的女孩已經可以自行走動。眾人商議決定，由李杰、大叔、斎藤和健太組成先遣隊，前往亡者峽谷尋找「時空之匙」。其餘人則留在艾拉如的小屋內躲避。

**清水恵美**：(拉住李杰的衣袖，眼神不安) 你一定要小心。

**李杰**：(溫和地笑了笑) 放心吧。我會回來的。

**旁白**：眾人告別了艾拉如，踏上了南下的路程。

**旁白**：森林南部的氣息明顯變得陰冷壓抑。原本發光的銀色樹木逐漸消失，取而代之的是焦黑、乾枯的枯木。地面上也開始出現一些巨大的、不知名生物的骨骸。

**斎藤蓮**：(看著腳邊的一具巨獸頭骨，嚥了口唾沫) 我現在後悔還來得及嗎？

**慷慨大叔**：(沉聲道) 閉嘴。注意周圍。

**旁白**：又走了大約一個小時，前方的地勢突然下陷。一個巨大的、被濃重黑霧籠罩的峽谷出現在眾人面前。峽谷兩側的岩壁上刻滿了扭曲的符文，散發著令人不安的血紅色光芒。

**李杰**：(看著眼前的峽谷) 那就是...亡者峽谷？

**旁白**：在峽谷的入口處，矗立著一座巨大的石牌坊。牌坊上掛著幾盞搖搖欲墜的風燈，燈光在黑霧中忽明忽暗。

**健太**：(聲音顫抖) 你們看...那是什麼？

**旁白**：李杰順著他指的方向看去，只見峽谷入口的陰影中，靜靜地站著一個巨大的身影。那身影高達三公尺，全身披掛著沉重的黑色鎧甲，手中握著一把巨大的斬馬刀。

**李杰**：(內心獨白) 守門人？

**旁白**：就在眾人屏息以待時，那個巨大的身影緩緩轉過頭，血紅色的雙眼在黑霧中燃燒。而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
