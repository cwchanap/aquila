# 第十八幕：分道揚鑣

**旁白**：銀月湖畔的重逢雖然令人振奮,但現實的殘酷很快又將眾人拉回了冰冷的現實。那座位於湖中心、被黑色結界籠罩的高塔,成了他們回家路上最後的、也是最大的障礙。

**健太**：(看著高塔,一臉絕望) 我們已經試過所有辦法了。石頭砸、火燒、甚至幾個人合力撞...那層黑色霧氣連晃都沒晃一下。

**斎藤蓮**：(推了推眼鏡,有些神經質地冷笑) 也許我們真的該考慮一下在這裡定居了。我看這湖邊的土挺肥沃的,種點菜應該能活。

**慷慨大叔**：(瞪了他一眼) 說什麼渾話!

**李杰**：(走到湖邊,彎下腰觀察湖水的波動) 剛才艾拉如說過,要進塔需要「時空之匙」。

**慷慨大叔**：(點頭) 沒錯。我們在這個小島上搜了一遍,除了這座塔和一些破爛石碑,什麼都沒有。

**旁白**：李杰走到其中一座殘破的石碑前。石碑上刻滿了扭曲的符文,與他在石台上看到的如出一轍。他試圖用手中的地圖碎片進行比對。

**李杰**：(指著石碑底部的一個凹槽) 你們看這裡。

**旁白**：眾人圍了過來。那是一個菱形的凹槽,裡面刻著一些精細的齒輪狀圖案。

**李杰**：(內心獨白) 這個形狀...我在地圖碎片的邊緣見過!

**旁白**：他趕緊拿出自己的碎片,又向健太借了他手中的那一半。兩片碎片拼湊在一起,背面隱約顯露出一個菱形的圖騰。

**斎藤蓮**：(驚呼) 這圖騰...和這凹槽一模一樣!

**李杰**：(試探著將拼好的地圖背面按向凹槽) ...

**旁白**：就在兩者接觸的一瞬間,整座石碑突然劇烈震動起來,發出一陣低沈的轟鳴聲。原本籠罩在高塔周圍的黑色結界開始瘋狂翻湧,像是被某種強大的力量攪動。

**眾人**：(驚喜) 有戲!

**旁白**：然而,震動僅持續了幾秒鐘就戛然而止。黑色結界恢復了平靜,甚至變得比剛才更加濃郁。

**李杰**：(皺眉) 能量不夠?

**慷慨大叔**：(沈聲道) 或者是需要更多的鑰匙。

**旁白**：就在這時,原本團結的氣氛出現了裂痕。

**伊藤誠**：(突然開口,語氣生硬) 我看地圖還在李先生手裡比較好。畢竟是他發現的。

**健太**：(有些不悅) 喂,這地圖有一半是我們的!憑什麼全給他?

**伊藤奈々**：(也跟著幫腔) 對啊!要是他一個人跑了怎麼辦?我們在這裡等了這麼久...

**斎藤蓮**：(唯恐天下不亂) 哎呀,大家別激動。我看不如這樣,地圖拆開,一人一半。

**李杰**：(冷冷地看著眾人) 拆開了就什麼也打不開。你們想一輩子困在這裡嗎?

**旁白**：爭執愈演愈烈。恐懼和自私在這一刻徹底爆發。原本一起同生共死的同伴,此刻卻為了這份虛無縹緲的「鑰匙」權利爭得面紅耳赤。

**慷慨大叔**：(怒吼一聲) 夠了!

**旁白**：全場瞬間安靜下來。

**慷慨大叔**：(看著這群人,眼神中充滿了失望) 命都快沒了,還在爭這些?

**健太**：(低著頭,小聲咕囔) 我只是不想再被拋下了...

**旁白**：最終,原本的團隊徹底分裂。健太帶領著他的人群,拿回了屬於他們的那一半地圖,決定去湖泊的另一邊尋找其他出口。而李杰、大叔、斎藤和清水則留在了石碑旁。

**李杰**：(看著健太等人遠去的背影,語氣沈重) 希望他們能好運。

**慷慨大叔**：(拍了拍李杰的肩膀) 別想了。人各有志。我們繼續研究這石碑吧。

**旁白**：而在這寂靜的夜晚,三個陌生人的命運,就這樣在銀月湖畔交會了。
