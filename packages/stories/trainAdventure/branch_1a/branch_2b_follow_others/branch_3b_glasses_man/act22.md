# 第二十二幕：工具的需求

**旁白**：眾人沿著那條充滿未來感的白色走廊前行。斎藤大輔走在最前面,他的背影顯得格外孤寂而堅定。李杰則緊跟在他身後,心中一直迴盪著大輔剛才說的話。

**李杰**：(內心獨白) 遺忘...這就是回家的代價嗎?如果忘記了這一切,那我們來到這裡的意義到底是什麼?

**斎藤蓮**：(好奇地東摸西摸) 哇,這牆壁的觸感...簡直不像是石頭,也不像是金屬。大哥,這到底是什麼材質?

**斎藤大輔**：(頭也不回) 這是由純粹的「意識數據」凝結而成的。在這裡,思想就是物質。

**斎藤蓮**：(依舊一頭霧水) 行吧,當我沒問。

**旁白**：走廊的盡頭是一座巨大的圓形拱門。門後是一個寬敞得令人咋舌的空間。無數巨大的透明管道在空中交錯,管道里流動著五彩斑斕的光液。

**斎藤大輔**：(停下腳步,看向空間中央) 到達「核心區」了。

**旁白**：在空間的正中央,懸浮著一個巨大的藍色光球。光球周圍環繞著無數環形的軌道,正以一種複雜而和諧的規律旋轉著。

**李杰**：(震撼) 這就是核心?

**斎藤大輔**：(點頭) 沒錯。它是這座高塔,也是這個鏡像空間的心臟。我們要做的,就是重啟它的引導程序。

**旁白**：大輔走向光球下方的一個控制台。然而,當他準備操作時,臉色突然變得極其難看。

**斎藤大輔**：(憤怒地垂在台上) 該死!關鍵插槽被鎖死了!

**李杰**：(跑上前) 怎麼回事?

**斎藤大輔**：(指著控制台中心一個空的圓孔) 啟動這台機器需要一個實體的「物理秘鑰」。我原本以為它可以通過代碼模擬,但系統增加了硬件驗證。

**慷慨大叔**：(皺眉) 物理秘鑰?那是什麽樣的東西?

**斎藤大輔**：(回憶道) 應該是一個菱形的、散發著金光的金屬體。在塔底的武器庫里應該有備份。

**斎藤蓮**：(一臉絕望) 武器庫?那我們不是還要跑回去?

**斎藤大輔**：(神情嚴峻) 不止如此。武器庫由最強大的「暗影近衛」把守。單憑我一個人的權限,根本進不去。

**李杰**：(立刻明白了) 所以妳需要工具?

**斎藤大輔**：(看向眾人) 沒錯。我們需要一些特殊的工具來破開武器庫的防禦。

**斎藤大輔**：(指向走廊一側的一個儲物間) 那裡面有一些古代精靈留下的工程設備。雖然老舊,但如果能修復的話,應該能派上用場。

**李杰**：(主動請纓) 我去看看。

**斎藤大輔**：(叮囑道) 小心點。儲物間雖然沒有怪物,但裡面佈滿了邏輯陷阱。一旦觸發,你的意識會被瞬間禁白。

**旁白**：李杰帶著斎藤蓮走向了側面的儲物間。大叔和清水則留在原地協助大輔維持控制台的最低運轉。

**旁白**：儲物間內堆滿了各種奇形怪狀的零件。有散發著微光的齒輪,有長滿鏽跡的扳手,還有一些根本看不出用途的金屬塊。

**李杰**：(在一堆廢鐵中搜索) 找找看有沒有類似「脈衝發生器」或者「量子切割刀」的東西。

**斎藤蓮**：(一臉嫌棄地翻動著零件) 妳說的這些東西...這裡真的會有嗎?我看這裡更像是一個大型的廢金屬回收站。

**李杰**：(停下動作,目光落在角落裡一個半掩埋的黑箱子上) 等等...那是什麽?

**旁白**：李杰走上前,用力拉出那個箱子。箱子表面刻著精緻的圖案,中間有一個熟悉的標誌——艾拉如的符文。

**李杰**：(打開箱子,眼睛一亮) 找到了!

**旁白**：箱子裡靜靜地躺著幾把散發著淡淡藍光的「修復工具」。

**斎藤蓮**：(好奇地湊過來) 哇,這就是妳說的工具?看起來挺酷的。

**旁白**：就在李杰伸手拿起工具的一瞬間,儲物間的燈光突然轉為刺眼的紅色,一個刺耳的警報聲響徹整座高塔。

**廣播**：(冰冷的聲音) 檢測到非授權的工具提取。防禦協議啟動。

**李杰**：(臉色大變) 快跑!

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
