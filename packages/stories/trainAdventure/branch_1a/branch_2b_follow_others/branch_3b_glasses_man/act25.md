# 第二十五幕：血染的祭壇

**旁白**：核心區域的拱門在劇烈的撞擊下終於支撐不住，「砰」的一聲巨響，化作無數晶瑩的碎片散落一地。

**旁白**：一股刺鼻的硫磺味伴隨著滾滾黑煙湧入室內。在煙霧中,三個高大的身影緩緩走出。

**清水恵美**：(臉色慘白,護在乘客們身前) 妳、妳們是誰!

**旁白**：走在最前面的,是一個身穿深紫色披風的生物。他的面具與普通的「守望者」不同,通體漆黑,額頭處鑲嵌著一顆散發著血紅光芒的寶石。他手中握著一把細長的銀色長劍,劍身上流動著詭異的紅光。

**紫衣生物**：(發出一種尖銳、刺耳的笑聲) 呵呵...真是美味的靈魂啊。

**伊藤誠**：(絕望地大喊) 求求你!別殺我們!我們什麼都不知道!

**紫衣生物**：(沒有理會他的哀求,轉向那團藍色光球) 終於找到了...最後的核心。

**旁白**：他緩緩舉起長劍,劍尖指向光球。

**清水恵美**：(內心獨白) 不能讓他破壞核心!否則李杰先生他們就永遠回不來了!

**旁白**：雖然害怕得全身發抖,但清水還是從地上撿起了一根剛才遺留下來的藍色工具。

**清水恵美**：(擋在光球前,大聲喊道) 不許動它!

**紫衣生物**：(停下動作,饒有興致地看著清水) 哦?一個勇氣可嘉的小姑娘?

**旁白**：他身後的兩名暗影近衛正要上前,卻被紫衣生物抬手攔住。

**紫衣生物**：(優雅地行了個禮) 我的名字是凌墨。這座城市的「清理官」。小姑娘,妳想守護這一切?

**清水恵美**：(咬牙切齒) 妳們殺了那麼多人...不會有好下場的!

**凌墨**：(語氣平淡) 殺人?不,我只是在回收過期的數據。

**旁白**：話音剛落,凌墨的身形突然消失。下一秒,他已經出現在清水身後。

**清水恵美**：(驚呼) 唔!

**旁白**：凌墨並沒有用劍,而是隨意地揮出一掌。一股強大的黑暗力量將清水重重地擊飛出去,撞在遠處的控制台上。

**清水恵美**：(吐出一口鮮血,委頓在地) 咳、咳...

**旁白**：其餘乘客看到這一幕,紛紛四散而逃。但石室內到處都是暗影近衛,慘叫聲此起彼伏。

**旁白**：原本寧靜的核心區域,此刻瞬間變成了血染的祭壇。

**凌墨**：(走到控制台前,看著上面閃爍的綠色符號) 斎藤大輔那個叛徒...竟然修復到這種程度了。

**旁白**：他舉起長劍,對準控制台中心狠狠刺了下去!

**旁白**：就在劍尖即將刺入的一瞬間,控制台下方的地板突然炸裂。李杰的身影從中一躍而出!

**李杰**：(雙手握著剛從武器庫拿到的「金屬方塊」,對著凌墨橫掃而去) 給我滾開!

**旁白**：金屬方塊發出一陣耀眼的金色強光,將凌墨逼退數步。

**李杰**：(趕緊扶起清水) 沒事吧?

**清水恵美**：(虛弱地微笑) 你...你終於回來了...

**旁白**：大叔、斎藤蓮和健太也緊跟著衝了上來,每個人手中都拿著剛武裝上的奇異兵器。

**李杰**：(眼神冷冽地看向凌墨) 現在,該我們清算了。

**旁白**：而在這座死寂的車站深處,某些被塵封已久的秘密,即將被揭開。
