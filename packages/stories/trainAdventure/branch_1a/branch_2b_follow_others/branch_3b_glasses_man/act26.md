# 第二十六幕：緊急警示

**旁白**：核心區域內的戰鬥一觸即發。李杰手中的金色方塊散發著神聖的光芒,與凌墨手中那把流動著紅光的長劍形成了鮮明對比。

**凌墨**：(收起輕蔑的神情,目光凝重地看著李杰) 妳竟然能從暗影近衛的包圍中拿到「啟動核心」?

**李杰**：(握緊方塊,冷冷地說) 多虧了妳那些手下太過自大。

**斎藤大輔**：(從管道里爬了出來,神色焦急) 李杰!別跟他說廢話!趕快把方塊放入插槽!

**旁白**：凌墨顯然也意識到了威脅,他身形一動,整個人化作一道紫色殘影,直取李杰!

**慷慨大叔**：(揮舞著一把散發著白光的巨大戰錘,擋在李杰身前) 妳的對手是我!

**旁白**：戰錘與長劍碰撞,爆發出雷鳴般的巨響。強大的力量讓整座高塔都為之顫抖。

**斎藤蓮**：(拿著一支類似雷射槍的短棍,對著其餘的暗影近衛瘋狂掃射) 走開!都給我走開!

**旁白**：健太則帶領著其餘倖存的乘客,躲到了光球軌道的後方。

**李杰**：(衝向控制台) 大輔!幫我掩護!

**旁白**：大輔再次坐回控制位,手指在鍵盤上飛速敲擊,為李杰開啟了最後一道安全路徑。

**斎藤大輔**：(大喊) 就在現在!放進去!

**旁白**：李杰猛地將金色方塊按入了那個解鎖的圓形插槽中。

**旁白**：一瞬間,整座石室被強烈的金光淹沒。原本藍色的核心光球在吸收了方塊的力量後,變成了純粹的金黃色。

**廣播**：(那種扭曲的聲音消失了,取而代之的是一個溫柔、空靈的女聲) 引導程序重啟...時空坐標...校準中...

**凌墨**：(被金光震飛,重重撞在牆上) 唔...不!這不可能!

**旁白**：隨著系統的重啟,原本狂暴的黑暗力量開始迅速退散。那些暗影近衛在金光中紛紛瓦解,化作點點塵埃。

**李杰**：(興奮地看向大輔) 成功了嗎?

**斎藤大輔**：(看著螢幕上跳動的數據,臉色卻依然蒼白) 還沒完。數據雖然恢復了,但塔的結構已經受損。我們必須在五分鐘內完成傳送。

**斎藤大輔**：(轉頭看向眾人,語氣急促) 所有人,立刻站到光球下方的圓台上!

**旁白**：眾人紛紛衝向圓台。清水被李杰扶著,雖然腳步有些虛浮,但眼神中充滿了希望。

**廣播**：(冷靜地播報) 警告。檢測到高塔結構性崩潰。傳送門將在300秒後關閉。

**李杰**：(內心獨白) 300秒...最後的五分鐘。

**旁白**：就在這最後關頭,原本倒在牆邊的凌墨突然發出一陣狂笑。

**凌墨**：(捂著胸口站起來,眼神變得瘋狂) 妳們以為...這樣就能逃掉嗎?

**旁白**：他猛地將手中的長劍插入地面。

**凌墨**：以我之魂,燃燒幽冥!

**旁白**：黑色的火焰從他體內噴湧而出,瞬間將整座石室包圍。

**李杰**：(大驚) 妳幹什麼!

**凌墨**：(狂笑著) 如果我回不去,那誰也別想走!

**旁白**：而在這座死寂的車站深處,某些被塵封已久的秘密,即將被揭開。
