# 第三十二幕：空無一人的列車

**旁白**：李杰獨自一人走在深夜的街道上。

**旁白**：新宿的夜晚依然燈火通明,但此刻在他眼裡,卻顯得有些冷清。

**旁白**：他回到公寓,打開燈。

**旁白**：房間裡一切如常。桌上的電腦還開著,螢幕上是他沒寫完的程式碼。

**李杰**：(坐下,看著螢幕) ...

**旁白**：他試圖繼續工作,但腦海中卻不斷浮現出那些碎片般的畫面。

**旁白**：紫色的雙月、發光的森林、戴著面具的守望者、還有那個叫凌墨的男人...

**李杰**：(煩躁地關掉螢幕) 該死...怎麼都忘不掉。

**旁白**：他站起身,走進浴室洗了個冷水澡。

**旁白**：冰冷的水沖刷著身體,讓他稍微冷靜了一些。

**李杰**：(看著鏡子裡的自己) 妳到底在想什麼?這一切都結束了。妳現在在地球上。

**旁白**：他走出浴室,躺在床上。

**旁白**：很快,疲憊席捲而來。他閉上眼睛,進入了夢鄉。

**旁白**：在夢裡,他再次回到了那列漆黑的火車。

**旁白**：火車在無盡的隧道中穿行。車廂裡空蕩蕩的,只有他一個人。

**李杰**：(站在車廂中央,茫然地環顧四周) 有人在嗎?

**旁白**：沒有人回應。只有火車行駛時單調的轟鳴聲。

**旁白**：他一節車廂一節車廂地走過。每一節都空無一人。

**旁白**：直到他來到第一節車廂。

**旁白**：車門是開著的。

**旁白**：李杰走進去,發現大輔正坐在駕駛位上。

**大輔**：(沒有回頭,平靜地說) 妳醒了。

**李杰**：(驚訝) 大輔?妳怎麼在這裡?

**大輔**：(轉過頭,露出一個詭異的微笑) 我一直都在這裡啊。李杰。

**李杰**：(後退一步) 這是哪裡?我們要去哪?

**大輔**：(指向前方) 去妳該去的地方。

**旁白**：李杰看向前方。隧道即將結束,外面是一片耀眼的白光。

**李杰**：(擋住眼睛) 唔!

**旁白**：當光芒消失時,火車停了下來。

**旁白**：李杰睜開眼,發現火車停在一個廢棄的站台前。

**旁白**：站台上矗立著一塊牌子:「幽冥之都」。

**李杰**：(倒吸一口氣) 不!

**旁白**：他想衝出火車,但車門卻緊緊鎖死。

**大輔**：(在身後幽幽地說) 逃不掉的。李杰。這是妳的循環。

**旁白**：火車再次啟動,緩緩駛向黑暗的深處。

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
