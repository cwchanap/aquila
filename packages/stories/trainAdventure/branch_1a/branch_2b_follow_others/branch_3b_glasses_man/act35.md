# 第三十五幕：意外的方法

**旁白**：李杰盯著那扇緊閉的門，腦中不斷思考著各種可能性。

**李杰**：（轉向斎藤）你有USB線嗎？

**斎藤蓮**：（皺眉）什麼？

**李杰**：USB充電線。你有嗎？

**斎藤蓮**：（激動地）你在開玩笑嗎？誰會隨身帶那種東西？

**斎藤蓮**：（繼續）我出門前都會把手機充到一百趴！根本不需要帶充電線！

**旁白**：李杰嘆了口氣。他不想跟這個人爭論。

**李杰**：（低聲）...算了。

**旁白**：他走回門邊，再次觀察那個凹槽。

**李杰**：（內心）如果物理鑰匙不在這裡...那一定有其他方法可以手動開啟...

**旁白**：他注意到門縫的邊緣有一處小小的、半透明的突起。

**李杰**：（伸手按下突起） ...

**旁白**：「嗡——」

**旁白**：一聲輕微的振動，門上方的牆壁突然裂開，露出一個隱藏的拉桿。

**斎藤蓮**：（驚訝）哇！你是怎麼發現的？

**李杰**：（握住拉桿，用力一拉） ...

**旁白**：金屬門發出沈重的磨損聲，緩緩向兩側滑開。

**旁白**：門後不是另一個房間，而是一個巨大的、垂直向下的通道。

**斎藤蓮**：（看著深不見底的通道，嚥了口唾沫） ...我們真的要跳下去嗎？

**李杰**：（看向身後漸漸逼近的紅光）我們沒有其他選擇。

**旁白**：兩人對視一眼，一起躍入了黑暗之中。
