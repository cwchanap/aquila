# 第三十六幕：樂譜的秘密

**旁白**：李杰和恵美決定留下。

**旁白**：他們告訴莉拉，想在這裡等待一段時間，看看會不會有什麼轉機。

**旁白**：莉拉雖然擔心，但也尊重他們的決定。

**旁白**：幾天後，搬遷的日子到了。

**旁白**：村子裡大部分的精靈都收拾好行李，準備前往魔族之島。

**旁白**：李杰和恵美跟著送行的隊伍，來到了村子外的碼頭。

**旁白**：碼頭邊停靠著一艘大型的木船，船身刻著精緻的精靈紋飾。

**旁白**：瑟蘭站在船頭，指揮著族人們登船。他的表情嚴肅，但眼中帶著一絲不捨。

**旁白**：薩爾維斯很感激他們。她用微弱的聲音說，很高興在最後的時光裡，還有人陪伴。

**旁白**：第六週的一個夜晚，薩爾維斯叫住了李杰和恵美。

**薩爾維斯**：謝謝你們...願意留下來...陪我們走完最後一程...

**旁白**：她的聲音很輕，像是隨時會消散。

**清水恵美**：不用謝...這是我們應該做的...

**薩爾維斯**：我...我很快就要去找其他人了...到時候...你們要好好照顧自己...

**李杰**：我們會的。妳也不用擔心。

**旁白**：薩爾維斯笑了笑，閉上了眼睛。

**薩爾維斯**：真好...這個世界雖然黑暗...但至少...還有溫暖的人...

**旁白**：她的呼吸漸漸平緩，最後停止了。

**旁白**：李杰和恵美靜靜地坐在床邊，陪著她走完最後一程。

**旁白**：第二天，李杰和恵美為薩爾維斯舉行了葬禮。

**旁白**：他們在森林裡挖了最後一個墓穴，將她埋葬在其他精靈的旁邊。

**旁白**：恵美再次唱起了安魂曲，聲音在寂靜的森林中迴盪。

**旁白**：李杰站在墓前，看著這片墓地。

**旁白**：八個墓碑，整齊地排列著。

**旁白**：這是月光林地最後的守望者們，安息的地方。

**旁白**：葬禮結束後，李杰和恵美回到了村子。

**旁白**：空蕩的街道，緊閉的門窗，沒有任何生命的氣息。

**旁白**：這個曾經充滿笑聲和溫暖的村子，現在只剩下死寂。

**旁白**：兩人走到廣場中央，坐在長椅上，看著天空中的雙月。

**旁白**：沉默持續了很久。

**清水恵美**：李杰先生...我們...接下來要怎麼辦？

**旁白**：李杰沒有立刻回答。他看著這個空蕩的村子，心中充滿了複雜的情緒。

**李杰**：我也不知道...但至少...我們還活著。

**旁白**：他轉過頭，看著恵美。

**李杰**：清水，妳後悔嗎？如果當初跟著明日香她們一起離開...

**清水恵美**：不...我不後悔。

**旁白**：恵美搖了搖頭，眼神很堅定。

**清水恵美**：如果我離開了...薩爾維斯她們...就要一個人面對死亡了。

**清水恵美**：至少...至少在最後的時刻，她們還有人陪伴...這樣就夠了。

**旁白**：李杰笑了笑。

**李杰**：妳真的...變堅強了。
