# 第三十七幕：白色瓷磚的房間

**旁白**：明亮的燈光下,李杰終於看清了這個房間的全貌。

**旁白**：這是一個小房間,大約只有五、六坪大小。但裡面塞滿了各種物品。

**旁白**：靠牆放著一張簡單的木桌和一把椅子。桌上有一台筆記型電腦,螢幕是黑的。

**旁白**：牆邊有一個高大的書架,上面擺滿了各式各樣的書籍。書架旁邊是一架老式的直立鋼琴,黑色的琴身泛著暗淡的光澤。

**旁白**：房間角落有一張單人床,床單整齊地鋪著。床旁邊是幾個木製櫃子。

**旁白**：另一側有個小廚房區域,配備著簡單的爐具和流理台。地上放著一個金屬水桶。

**旁白**：牆上掛著一台老式收音機,旋鈕和天線都完好無損。

**旁白**：而在房間的另一側,有一扇門。

**李杰**：（低聲）這不是出口...

**旁白**：李杰走到鋼琴前,看著琴架上放著的一張樂譜。樂譜很舊,紙張泛黃,上面密密麻麻地寫滿了音符。

**李杰**：（拿起樂譜）這首曲子...

**旁白**：他雖然不懂音樂,但他能感覺到這張樂譜的特殊之處。

**旁白**：樂譜的標題是《遺忘之城》。

**李杰**：（內心）遺忘之城...

**旁白**：他注意到樂譜的背面,有一行用鉛筆寫的小字：

**旁白**：「鏡子不會說謊，但它會顛倒真相。」

**李杰**：（皺眉）鏡子？

**旁白**：他轉過頭,看向洗手台上方的那面鏡子。

**旁白**：鏡子很乾淨,映照出他疲憊的面孔。

**旁白**：他走上前,仔細觀察鏡子。鏡框的邊緣似乎有一些刻痕。

**李杰**：（伸手觸摸刻痕）這是...數字？

**旁白**：1、4、2、6...

**李杰**：（低聲唸道）1426...

**旁白**：他想起桌上那台電腦。

**旁白**：他走到桌邊,按下電腦的開機鍵。

**旁白**：螢幕亮起,出現一個藍色的輸入框：

**旁白**：「ENTER PASSWORD:」

**李杰**：（輸入）1-4-2-6

**旁白**：「ACCESS DENIED」

**李杰**：（煩躁）不對？

**旁白**：他再次看向那面鏡子。

**旁白**：鏡子...顛倒真相...

**李杰**：（內心）顛倒...那就是反過來？

**旁白**：他試著輸入：6-2-4-1

**旁白**：「ACCESS GRANTED」

**旁白**：螢幕閃爍了一下,進入了系統桌面。

**旁白**：桌面很空,只有一個資料夾,名稱是:「ROOT」。

**旁白**：李杰點開資料夾,裡面只有一個文字檔案:「READ_ME.txt」。

**旁白**：他點開檔案,螢幕上彈出幾行字：

**旁白**：「妳終於來了。李杰。」

**李杰**：（瞳孔縮小）妳怎麼知道我的名字！

**旁白**：「別緊張。這只是一個預設的程序。」

**旁白**：「妳現在所在的,是這個世界的『緩衝區』。只有意識頻率達到臨界點的人,才能進入這裡。」

**李杰**：（對著螢幕喊道）我要怎麼離開！

**旁白**：「離開的方法就在妳身邊。但首先,妳必須找回失去的『記憶數據』。」

**旁白**：「去書架看看吧。在那裡,妳會找到答案。」

**旁白**：檔案自動關閉,電腦螢幕恢復了黑色。

**李杰**：（咬牙）書架...

**旁白**：他轉過身,走向那個裝滿了書的書架。

**旁白**：書架上的書種類繁多,從古典文學到現代科學都有。他一本一本地翻找,試圖找到所謂的「記憶數據」。

**旁白**：就在他翻到一本名為《時間的碎片》的書時,一張照片從書頁中滑落。

**旁白**：李杰彎下腰,撿起照片。

**旁白**：照片上是一群穿著實驗袍的人,正站在一列火車前合影。

**旁白**：在那群人的中央,他看到了一個熟悉的身影。

**李杰**：（震驚）這...這不是大輔嗎？

**旁白**：照片裡的大輔比現在年輕很多,臉上帶著自信的笑容。

**旁白**：而在大輔身旁,站著一個年輕男子。

**旁白**：雖然隔著很多年,但李杰一眼就認出了那個人。

**李杰**：（手抖得厲害）這是我...爸爸？

**旁白**：照片的背面寫著一行字:「2001年8月15日,新宿站測試圓滿成功。」

**李杰**：（內心）2001年...那是我三歲的時候...

**李杰**：（內心）爸爸和大輔...他們早就認識？

**旁白**：就在這時,房間的收音機突然自動打開。

**旁白**：一陣刺耳的雜訊過後,傳來一個低沈的男聲。

**廣播**：「李杰,妳還有三分鐘。」

**李杰**：（驚恐）三分鐘什麼！

**廣播**：「三分鐘後,緩衝區將被清理。如果妳不離開,妳的意識將被徹底格式化。」

**李杰**：（瘋狂地翻找書架）該死！在哪裡！記憶數據在哪裡！

**旁白**：他在慌亂中撞到了那架鋼琴。

**旁白**：鋼琴發出一聲不和諧的重音。

**旁白**：李杰的目光落在琴蓋上。琴蓋上刻著一行字：

**旁白**：「音樂是靈魂的密碼。」

**李杰**：（想起那張樂譜）樂譜！

**旁白**：他衝回鋼琴前,抓起那張《遺忘之城》的樂譜。

**旁白**：他仔細觀察樂譜上的音符。

**李杰**：（恍然大悟）...這不是給鋼琴的樂譜。

**斎藤蓮**：（疑惑）什麼？

**李杰**：（指著書架）這張樂譜...是書架的地圖。

**旁白**：斎藤走過來,看著書架側邊的鉛筆線,又看了看李杰手中的樂譜。

**斎藤蓮**：（恍然）你是說...音符代表書架上的書？

**李杰**：（點頭）對。音符的高低代表哪一層,位置代表書在那層的大概位置。

**旁白**：李杰再次看著樂譜。

**旁白**：第一個音符在第二線上,偏左。

**旁白**：他走到書架前,看著第二層,從左邊開始數。

**旁白**：大約在左邊四分之一的位置,有一本深藍色的書。

**李杰**：（拿起那本書）這本...

**旁白**：他翻開書,仔細檢查。書名是《基礎物理學》。

**旁白**：他看著書背,然後注意到一個細節。

**旁白**：書名的其中一個字——「礎」——顏色稍微深一點。就像是墨水多印了一層。

**李杰**：（低聲）「礎」...不對,英文的話...

**旁白**：他看向書背上的英文標題"FOUNDATION"。字母S的顏色明顯比其他字母深。

**李杰**：（內心）S...

**旁白**：李杰繼續按照樂譜上的音符尋找。

**旁白**：第二個音符在第四線上,偏中間。

**旁白**：他找到第四層書架,中間位置。一本紅色的書——《經濟學原理》。

**旁白**：書背上,"ECONOMICS"的字母E顏色較深。

**李杰**：（內心）E...

**旁白**：第三個音符在第三線上,偏跑。

**旁白**：第三層書架,右側。一本綠色的書——《設計思維》。

**旁白**：書背上,"DESIGN"的字母D顏色較深。

**李杰**：（內心）D...

**旁白**：第四個音符在第一線上,靠左。

**旁白**：第一層書架,左側。一本灰色的書——《生物學入門》。

**旁白**：書背上,"BIOLOGY"的字母B顏色較深。

**李杰**：（內心）B...

**旁白**：李杰把四個字母連起來。

**李杰**：（低聲）S、E、D、B...SEDB？

**旁白**：這不是任何有意義的英文單字。

**斎藤蓮**：（在旁邊）找到什麼了嗎？
