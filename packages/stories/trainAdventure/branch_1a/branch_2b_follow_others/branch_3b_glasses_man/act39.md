# 第三十九幕：逃離之門

**旁白**：李杰走向廁所,斎藤疲憊地跟在後面。

**旁白**：狹小的廁所裡,白色瓷磚在燈光下反射著冷冷的光。

**旁白**：這裡能移動的物品不多——塑膠杯、毛巾、藥櫃裡的瓶子...還有那卷印著詩的衛生紙。

**旁白**：李杰拿著樂譜和UV燈,仔細研究第二小節「Reflection」的音符。

**旁白**：這一小節的音符排列不同,有的音符很長,有的很短,形成一種特殊的節奏pattern...

**李杰**：（內心）長音符...短音符...這像是節奏...但為什麼要強調節奏？

**李杰**：（內心）等等...「Reflection」...映照...鏡子會映照...鏡子會把左右顛倒...

**李杰**：（內心）該負不...要把物品擺成音符的鏡像位置？把左右反過來？

**旁白**：李杰嘗試著將藥櫃裡的瓶子按照樂譜上的高度擺放,但故意將左右順序顛倒。

**旁白**：當最後一個瓶子放下的瞬間——

**旁白**：「咔嚓」

**旁白**：洗手台下方的櫃子突然彈開,露出一個金屬轉盤。

**斎藤蓮**：（激動）李杰！你看！又是這種轉盤！

**李杰**：（走過去,看著轉盤上的符號） ...

**旁白**：轉盤上有三個刻度,分別對應著不同的幾何圖形。

**李杰**：（看向第三小節「Cycle」的樂譜）最後一個謎題。

**旁白**：來到門前,斎藤還站在那裡,一臉茫然和疲憊,不知道發生了什麼。

**李杰**：（開始轉動轉盤,手指在金屬轉盤上快速移動）方形...三角形...圓形...

**旁白**：他把三個轉盤依序轉到：方形■、三角形▲、圓形●。

**旁白**：每轉一個,轉盤就發出「咔」的一聲,清脆而堅定。

**旁白**：當最後一個轉盤轉到位時——

**旁白**：「咔嚓」——

**旁白**：門鎖打開的聲音,像是某種機關被觸發了。轉盤上的符號同時亮起綠光。

**斎藤蓮**：（驚訝,聲音裡帶著不敢相信）真的開了？！真的開了？

**李杰**：（推開門）走吧。

**旁白**：門後,是一條明亮的走廊——不是他預期中的黑暗或詭異空間,而是正常的、現代化的走廊。

**旁白**：這一切看起來如此...正常。如此...安全。

**旁白**：斎藤幾乎是衝進走廊的,跑向那扇通往外界的玻璃門。

**斎藤蓮**：（大喊）外面！是外面！

**旁白**：他用力推開玻璃門。

**旁白**：刺眼的陽光瞬間包圍了兩人。
