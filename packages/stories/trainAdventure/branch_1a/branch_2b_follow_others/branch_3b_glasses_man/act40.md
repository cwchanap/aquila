# 第四十幕：山中的對峙

**旁白**：2027年12月,輕井澤之旅的第三天早晨。

**旁白**：冬日的陽光透過窗簾灑進房間,李杰從睡夢中醒來。

**旁白**：他伸手想摸摸身旁的澪,卻只摸到冰冷的床單。

**李杰**：澪?

**旁白**：房間裡靜悄悄的,浴室的門開著,裡面沒有人。

**旁白**：李杰坐起身,視線落在床頭櫃上的一張紙條。

**旁白**：那是用黑色原子筆寫的,筆跡工整,帶著某種刻意的冷靜。

**李杰**：「我想和你玩個遊戲。請到東京都西多摩郡奧多摩町氷川1142-3。澪和繪里花在我手上。渡邊陸。」

**旁白**：李杰的身體僵住了。

**旁白**：渡邊陸...那個被認為已經死在父親刀下的男人。

**旁白**：他竟然還活著,而且綁架了澪和繪里花。

**旁白**：李杰瘋狂地撥打澪的電話,卻傳來「您撥的電話已關機」的冷漠女聲。

**旁白**：他衝出房間,找到飯店的櫃台。

**李杰**：（焦急地問）請問有沒有看到和我一起的那位女士？

**櫃台服務員**：（禮貌地搖頭）很抱歉,先生。今天早晨並沒有看到那位女士出門。

**李杰**：（咬牙）那監視器呢？我要看監視器！

**旁白**：在李杰的強烈要求下,飯店方調出了走廊的監視器畫面。

**旁白**：畫面顯示,凌晨四點左右,澪獨自一人走出了房間。

**旁白**：她神情木然,動作僵硬,彷彿夢遊一般走出了飯店大門。

**旁白**：在大門外,一輛黑色的轎車早已等候在那裡。澪緩緩拉開車門,坐了進去。

**旁白**：車子隨即啟動,消失在茫茫雪色中。

**李杰**：（看著畫面,心如刀割）澪...妳到底怎麼了...

**旁白**：他知道,這絕對不是普通的綁架。渡邊陸能讓澪乖乖跟著走,一定是掌握了什麼致命的威脅。

**旁白**：他不敢報警。紙條上寫得很清楚,如果報警,他這輩子都別想再見到她們。

**旁白**：李杰立刻退了房,開著租來的車,瘋狂地駛向東京。

**旁白**：一路上,他的腦海中不斷閃過渡邊陸那張陰沈的臉。

**李杰**：（內心）當年到底發生了什麼...為什麼你會活下來...又為什麼要在這個時候出現...

**旁白**：五個小時後,車子駛入了奧多摩山區。

**旁白**：這裡山路崎嶇,兩旁是茂密的森林。隨著海拔的升高,霧氣越來越重。

**旁白**：下午兩點,李杰終於找到了紙條上的地址。

**旁白**：那是一座廢棄的伐木場工廠。巨大的水泥建築矗立在荒山野嶺中,顯得格外突兀和詭異。

**李杰**：（停下車,握緊手中的短刀）我來了。

**旁白**：他走進工廠。裡面光線昏暗,空氣中瀰漫著一股霉味。

**旁白**：就在他走到工廠中央時,原本漆黑的大廳突然亮起了幾盞昏黃的燈。

**旁白**：在燈光下,李杰看到了他最不想看到的畫面。

**旁白**：澪和繪里花被綁在兩張木椅上,嘴巴被膠帶封住,眼神中充滿了驚恐。

**旁白**：而在她們身後,站著一個男人。

**旁白**：他穿著一件黑色的長風衣,手裡握著一把閃著寒光的長劍。

**旁白**：他緩緩抬起頭,露出了一張被燒焦了一半、極其醜陋的面孔。

**渡邊陸**：（聲音嘶啞難聽,像是從地獄傳來的回響）李杰...好久不見。

**李杰**：（憤怒地咆哮）放了她們！你有什麼衝著我來！

**渡邊陸**：（冷笑一聲）衝著妳？妳太高估自己了。

**渡邊陸**：（指著澪）我只是想拿回屬於我的東西。

**李杰**：（震驚）屬於妳的東西？什麼意思？

**渡邊陸**：（語氣變得瘋狂）當年妳父親那一刀,雖然沒殺了我,但卻毀了我的一切！

**渡邊陸**：名聲、地位、金錢...還有這張臉！

**渡邊陸**：（猛地揮劍指向李杰）我要讓妳也嚐嚐,失去一切的滋味！

**旁白**：就在渡邊陸準備動手時,原本安靜的工廠角落,突然傳來一陣輕微的腳步聲。

**旁白**：所有人同時轉頭。

**旁白**：一個身影緩緩從陰影中走出。

**旁白**：那是個年輕女子,綁著俐落的馬尾,手裡握著一把警用配槍。

**年輕女警**：（語氣冰冷）渡邊陸,放下武器。妳被包圍了。

**李杰**：（驚訝）莉子？！

**旁白**：出現的竟然是李杰之前在新宿車站遇到過的年輕女警莉子。

**渡邊陸**：（狂笑）包圍？就憑妳一個小警察？

**旁白**：渡邊陸說完,身形突然一動,整個人化作一道殘影,瞬間衝到了莉子面前。

**旁白**：長劍與子彈同時響起。

**旁白**：工廠內陷入了一片混亂。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
