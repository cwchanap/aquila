# 第四十三幕：無動於衷

**旁白**：李杰坐在床邊,看著陷入昏迷的澪。

**旁白**：剛才那道白光過後,凌墨逃走了,而澪也因為體力不支再次倒下。

**旁白**：這間廢棄的辦公室裡,只剩下他們兩個人,以及那台閃爍著藍光的電腦。

**李杰**：(內心)大輔說...插入隨身碟,一切就會結束...

**旁白**：李杰看著手中的黑色隨身碟。

**李杰**：(猶豫)如果大家都忘記了...那我們這一年來的相遇...真的就不存在了嗎?

**旁白**：他想起在大巴車上的那些笑聲,想起在雪地裡的冒險。

**李杰**：(眼眶微濕)我不想忘記。

**旁白**：就在這時,澪的手指動了一下。

**澪**：(虛弱地睜開眼,聲音微弱)李杰...

**李杰**：(驚喜)澪!妳醒了!

**澪**：(看著他,眼神中透著一絲悲哀)快點...趁他還沒回來...

**李杰**：(指著電腦)大輔說...

**澪**：(打斷他,語氣堅定)別聽他的。他只是想...回收妳的記憶。

**李杰**：(震撼)回收?

**澪**：(點頭)這個世界...是靠我們的記憶維持的。如果我們忘記了...空間就會崩塌。

**李杰**：(後退一步)那大輔...他到底是誰?

**澪**：(閉上眼睛,長嘆一聲)他是這場遊戲的「管理員」。他累了。他想毀掉這一切。

**李杰**：(內心)難道我們只是...管理員手中的玩物?

**旁白**：就在這時,窗外傳來一聲巨響。

**旁白**：整棟工廠建築開始劇烈晃動。

**大輔**：(聲音從四面八方傳來)看來妳們選了最難的一條路。

**大輔**：(冷笑)既然不想遺忘,那就帶著記憶...永遠消失吧。

**旁白**：黑色的氣流從牆壁的縫隙中滲入,像是有生命一般向兩人包圍過來。

**李杰**：(抱起澪)走!我們離開這裡!

**旁白**：他衝出房間,朝著樓梯跑去。

**旁白**：但在他的身後,整層樓正像破碎的拼圖一樣,一塊塊掉落入虛空。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
