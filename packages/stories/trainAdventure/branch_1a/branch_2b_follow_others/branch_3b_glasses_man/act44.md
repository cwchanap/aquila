# 第四十四幕：強者與弱者

**旁白**：李杰背著澪,在搖搖欲墜的工廠走廊上狂奔。

**旁白**：身後,牆壁和天花板正成片坍塌,露出了外面那片混濁的、不斷旋轉的暗紫色星空。

**旁白**：這已經不是現實世界,也不是幽冥之都。這裡是數據崩潰的「邊緣」。

**李杰**：(喘著氣)呼...呼...澪,抓緊我!

**旁白**：澪虛弱地靠在他的背上,雙手環繞著他的頸部,但力氣越來越小。

**澪**：(低聲)李杰...把我放下吧...妳一個人跑比較快。

**李杰**：(咬牙)別胡說!我說過會帶妳回去的!

**旁白**：他衝下樓梯,來到一樓大廳。

**旁白**：大門已經消失了,取而代之的是一道緩緩旋轉的漆黑裂縫。

**旁白**：而在裂縫前,站著一個挺拔的身影。

**凌墨**：(轉過頭,神情冷峻)妳還是來了。

**李杰**：(握緊短刀,警覺地看著他)妳想怎麼樣?

**凌墨**：(看向裂縫)那是唯一的出口。但它只能容納一個人的靈魂。

**李杰**：(心頭一震)什麼!一個人的靈魂?

**凌墨**：(指著澪)她是這個空間的核心。如果她走了,這裡會立刻崩塌。如果你走了...她會永遠消失在虛無中。

**李杰**：(震撼)這就是最後的選擇嗎?

**凌墨**：(冷笑)這就是強者的遊戲。李杰。妳是想當那個活下來的強者,還是那個消失的弱者?

**李杰**：(看向背上的澪) ...

**旁白**：澪似乎也聽到了對話。她掙扎著滑下李杰的背,癱坐在地上。

**澪**：(看著李杰,眼神中充滿了溫柔)李杰...你走吧。

**李杰**：(搖頭)不可能。

**澪**：(微笑,眼眶微濕)我本來就不屬於那個世界。這一年來...能認識你...我已經很滿足了。

**李杰**：(眼淚滑落)澪...

**旁白**：就在兩人依依不捨時,凌墨突然發出一聲痛苦的嘶吼。

**凌墨**：(捂著胸口,身體周圍的黑氣開始失控)該死...系統在排斥我...

**旁白**：他的身體開始閃爍,像是不穩定的全息投影。

**大輔**：(聲音低沈而威嚴)凌墨。妳的任務失敗了。

**旁白**：裂縫中伸出無數黑色的觸手,瞬間將凌墨捲了進去。

**凌墨**：(發出絕望的尖叫)不——!

**旁白**：凌墨消失在黑暗中。大廳陷入了死一般的沉寂。

**旁白**：黑色的氣流已經蔓延到了兩人腳下。

**李杰**：(看向澪,眼神變得前所未有的堅定)既然只能走一個...

**李杰**：(抱住澪)那我們就一起消失吧。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
