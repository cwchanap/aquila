# 第四十五幕：再次醒來

**旁白**：黑暗。無邊無際的黑暗。

**旁白**：李杰感覺不到自己的身體,也聽不到任何聲音。

**旁白**：意識像是一片漂浮在虛空中的羽毛,漫無目的地遊蕩。

**李杰**：(內心)結束了嗎...這就是消失的感覺嗎?

**旁白**：就在他準備放棄所有思考時,一道微弱的光芒出現在遠處。

**旁白**：光芒很小,但卻異常明亮。李杰本能地朝著那個方向前進。

**旁白**：隨著距離的拉近,光芒越來越大,最後變成了一扇散發著溫暖白光的門。

**李杰**：(穿過大門) ...

**旁白**：當他再次睜開眼時,發現自己正躺在一張舒適的床上。

**旁白**：陽光透過窗簾的縫隙灑進房間,照在床邊的木桌上。

**旁白**：李杰坐起身,揉著有些發痛的額頭。

**李杰**：(迷茫地看著四周)這裡是...我家?

**旁白**：房間裡一切如常。桌上的電腦開著,螢幕上是他沒寫完的程式碼。

**旁白**：他拿起手機看了一眼。時間:2024年5月19日,早晨八點。

**李杰**：(喃喃自語)八點...我睡了這麼久?

**旁白**：他走進浴室,看著鏡子裡的自己。

**旁白**：頭髮有些亂,眼下有淡淡的黑眼圈,但除此之外,沒有任何異樣。

**李杰**：(內心獨白)難道...真的只是一場夢?

**旁白**：他回想起夢中的那些情景。漆黑的火車、發光的森林、溫暖的精靈村莊、還有那個叫澪的女孩。

**旁白**：那種真實感,讓他無法相信這只是大腦的臆想。

**旁白**：他走出房間,來到客廳。

**李杰母親**：(從廚房探出頭,笑著說)小杰!妳醒啦!早飯做好了,快來吃!

**李杰**：(愣住)媽?妳怎麼在這裡?

**李杰母親**：(困惑地笑了)這孩子,說什麼胡話呢?我不是一直都在這嗎?

**旁白**：李杰看著母親那熟悉的、慈祥的臉。

**李杰**：(內心)不對...我媽明明在上海...

**旁白**：他猛地衝向玄關,推開大門。

**旁白**：外面的街道熱鬧非凡。行人匆匆,車水馬龍。

**旁白**：這裡是新宿。他失蹤前的那個世界。

**李杰**：(顫抖地掏出手機,撥打那個刻在腦海中的號碼) ...

**旁白**：電話接通了。

**澪**：(電話那頭傳來一個清脆、熟悉的聲音)餵?李杰?找我有事嗎?

**李杰**：(手機滑落,淚流滿面)澪...

**旁白**：他回來了。回到了這個原本的世界。

**旁白**：但不知為何,他的心中卻感到一陣莫名的空虛和恐懼。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
