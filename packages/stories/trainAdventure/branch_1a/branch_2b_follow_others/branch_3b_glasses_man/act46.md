# 第四十六幕：真相與背叛

**旁白**：李杰看著電腦螢幕，心中翻起了驚濤駭浪。

**旁白**：照片上的男子——他的父親李明遠，竟然出現在2001年的新宿車站實驗現場。

**旁白**：而且，他和斎藤大輔站在一起，兩人看起來關係匪淺。

**李杰**：（聲音顫抖）這不可能...我爸只是個普通的公務員...他從來沒去過日本...

**大輔**：（在螢幕上苦笑）那是因為...他的身份被徹底抹除了。

**大輔**：妳父親李明遠,曾是「訊聯科技」的核心工程師之一。他是這個空間最初的構建者。

**李杰**：（震驚）構建者？

**大輔**：沒錯。25年前的那場列車失蹤事件，並非意外。那是妳父親和大輔為了測試「靈魂備份」技術而進行的一場實驗。

**大輔**：（眼神中透著一絲哀傷）可惜，實驗失控了。空間發生了坍塌，近兩百名乘客被困在了這裡。

**大輔**：為了掩蓋真相,公司抹除了所有參與者的記錄。妳父親被送回了中國,改名換姓,重新生活。

**大輔**：而我...則被留在了這裡,成了這個空間的守護者。

**李杰**：（憤怒）所以...我這二十多年來的生活,全都是建立在謊言之上的？

**大輔**：不。妳父親這麼做,是為了保護妳。

**大輔**：(指向螢幕後方) 凌墨,他其實是那次實驗中產生的第一個「變異數據」。他擁有妳父親的一部分權限,所以他才能在這裡橫行霸道。

**大輔**：而妳,李杰。妳體內流淌著妳父親的基因。這就是為什麼妳能開啟這扇門的原因。

**旁白**：李杰感到腦中一陣混亂。真相太過沉重,壓得他喘不過氣。

**斎藤蓮**：（臉色蒼白地看著李杰） 妳...妳竟然是這種人的兒子...

**李杰**：（咬牙）我不知道！我什麼都不知道！

**旁白**：就在這時,房間的警報聲再次響起,這次比之前更加急促。

**廣播**：「清理程序倒計時：60秒。」

**大輔**：（語氣急促）李杰！沒時間解釋了！快去拿那把白劍！

**大輔**：只有那把劍,才能徹底清除凌墨的數據,重置這個空間！

**李杰**：（環顧四周）在哪裡！

**大輔**：(指向實驗室深處的一個密室) 就在那扇門後。快！

**旁白**：李杰衝向密室。

**旁白**：門是感應式的,在他靠近的瞬間自動開啟。

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
