# 第四十七幕：祭壇的真相

**旁白**：李杰衝進密室。

**旁白**：密室的正中央,懸浮著一把散發著柔和白光的長劍。

**旁白**：劍身如白玉般晶瑩剔透,劍柄上鑲嵌著一顆散發著神聖氣息的寶石。

**李杰**：(伸手握住劍柄) ...

**旁白**：嗡——!

**旁白**：一股強大的、溫暖的力量瞬間傳遍李杰全身。他感到原本疲憊的身軀重新煥發了生機,腦海中那些雜亂的碎片也漸漸平息。

**李杰**：(握緊白劍) 好強的力量...

**旁白**：他走出密室,發現實驗室已經開始崩解。

**旁白**：牆壁像燃燒後的紙片一樣,一點點消散在虛空中。

**大輔**：(在螢幕上喊道) 走!去終焉祭壇!

**旁白**：李杰拉起斎藤,衝出了實驗室。

**旁白**：他們沿著原路返回,但通道已經變得面目全非。無數黑色的氣流在空氣中肆虐,試圖阻攔他們的去路。

**李杰**：(揮動白劍) 滾開!

**旁白**：白光劃破黑暗,將黑色氣流瞬間擊碎。

**旁白**：兩人衝出了維修通道,回到了那座巨大的候車大廳。

**旁白**：凌墨正站在祭壇前,冷冷地注視著他們。

**凌墨**：(看著李杰手中的白劍,眼神一凝) 妳竟然拿到了那把劍...

**李杰**：(劍指向他) 凌墨。一切都結束了。

**凌墨**：(狂笑) 結束?這才剛剛開始!

**旁白**：凌墨猛地揮劍,一道紫色的劍氣橫掃而來。

**李杰**：(舉劍格擋) 鐺——!

**旁白**：白光與紫光在空中劇烈碰撞。

**旁白**：強大的衝擊波讓整座候車大廳都為之震顫。

**旁白**：而在此刻，原本熟悉的月台路徑，正在發生他們無法想像的變化。
