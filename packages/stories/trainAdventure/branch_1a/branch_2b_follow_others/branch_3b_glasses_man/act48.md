# 第四十八幕：最後一搏

**旁白**：大廳內的戰鬥進入了白熱化。

**旁白**：李杰揮舞著白劍,每一劍都帶著神聖的光芒。而凌墨則像一道紫色的幽靈,在光影間快速閃爍。

**斎藤蓮**：(躲在石柱後,大聲喊道) 李杰!加油!

**旁白**：凌墨的劍法極其詭異,角度刁鑽。李杰雖然有白劍護身,但身上還是多處掛彩。

**李杰**：(咬牙) 撐住...我必須撐住!

**旁白**：他想起在大巴車上和澪的對話,想起那個溫暖的精靈村莊。

**李杰**：(大喝一聲,白劍爆發出前所未有的強光) 喝啊!

**旁白**：這一劍勢如破竹,直接震碎了凌墨的防禦。

**凌墨**：(驚恐地往後退) 不!這不可能!

**李杰**：(趁勢而上,劍尖直指凌墨的咽喉) 告訴我!怎麼徹底重置這個空間!

**凌墨**：(露出猙獰的笑容) 重置?妳以為拿到這把劍就能贏了嗎?

**凌墨**：(指向祭壇中央的那個凹槽) 妳需要「祭品」。

**李杰**：(心頭一震) 祭品?什麼祭品?

**凌墨**：(瘋狂地大笑) 必須有一個人...永遠留在這裡...用他的靈魂作為重啟程序的最後一行代碼!

**李杰**：(後退一步) ...

**旁白**：就在這時,大廳的穹頂開始崩塌。一塊巨大的石樑正朝著李杰砸下!

**斎藤蓮**：(驚呼) 李杰!小心!

**旁白**：斎藤猛地衝過來,用力推開了李杰。

**旁白**：砰——!

**旁白**：石樑重重地砸在了斎藤蓮的腿上。

**斎藤蓮**：(痛苦地慘叫) 啊——!我的腿!

**李杰**：(大驚,衝過去想要搬開石樑) 斎藤!妳沒事吧!

**斎藤蓮**：(臉色慘白,滿頭大汗) 快、快走...別管我...

**凌墨**：(在旁邊冷冷地看著) 看來...祭品已經選好了。

**李杰**：(憤怒地回頭看向他) 妳說什麼!

**凌墨**：(指著斎藤) 他的靈魂...剛好符合要求。

**旁白**：新的危機，伴隨著斎藤的受傷,讓李杰陷入了前所未有的絕望。
