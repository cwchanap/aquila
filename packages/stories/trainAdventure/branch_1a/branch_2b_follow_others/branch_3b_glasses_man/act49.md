# 第四十九幕：尋找候選人

**旁白**：李杰拼命想要搬開那塊沉重的石樑。

**旁白**：但石樑重達千斤,任憑他如何用力,也只是微微晃動。

**斎藤蓮**：(疼得渾身發抖) 沒用的...李杰...快、快去重置系統...

**李杰**：(眼眶泛紅) 閉嘴!我不會丟下妳的!

**旁白**：凌墨一步步走過來,手中的黑劍在地面上拖行,發出刺耳的聲響。

**凌墨**：(語氣平淡) 這是最好的結局。用他一個人的命,換妳回地球。

**李杰**：(怒吼) 這不是結局!這是屠殺!

**凌墨**：(冷笑) 隨妳怎麼說。不過妳也看到了...這裡要崩潰了。

**旁白**：整座候車大廳正像破碎的玻璃一樣,一片片掉落入下方的虛空。

**旁白**：就在李杰感到絕望時,他的腦海中突然響起了大輔的聲音。

**大輔**：(聲音微弱) 李杰...聽我說...

**李杰**：(驚喜) 大輔!妳還在?

**大輔**：(斷斷續續地) 還有一個方法...不需要祭品...

**李杰**：(急切地問) 什麼方法!快說!

**大輔**：(聲音越來越小) 尋找...「覺醒者」...在其他的數據維度裡...

**李杰**：(愣住) 其他的維度?

**大輔**：(最後的聲音) 去...銀月湖...找艾拉如...她那裡有...

**旁白**：聲音消失了。李杰的手機也徹底熄滅。

**李杰**：(看向斎藤) 斎藤,撐住!我一定會帶妳走的!

**旁白**：他再次嘗試搬動石樑,但力量依然不夠。

**旁白**：就在這時,原本安靜的大廳角落,突然傳來一陣急促的腳步聲。

**旁白**：一個身影跑了過來。

**健太**：(喘著氣) 我來幫忙!

**旁白**：竟然是田中健太。他雖然也滿身是傷,但眼神卻很堅定。

**李杰**：(驚喜) 健太!妳沒死!

**健太**：(一邊用力抬石樑一邊說) 命大!剛好掉進了一個緩衝槽。

**旁白**：兩個人合力,石樑終於被緩慢抬起。

**李杰**：(趕緊把斎藤拖了出來) 走!去碼頭!

**旁白**：三人朝著大廳外面的碼頭狂奔。

**凌墨**：(在後方冷冷地盯著他們的背影) 逃吧。看看妳們能逃到哪裡。

**旁白**：他們朝著祭壇的方向，一步一步地走去。
