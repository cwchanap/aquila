# 第五十一幕：紅眼的凌墨

**旁白**：李杰站在銀鏡前。

**旁白**：鏡面如水波般盪漾,映照出他堅毅的臉龐。

**旁白**：就在他準備踏入鏡中時,一聲刺耳的尖叫從森林深處傳來。

**田中健太**：(驚呼) 他來了!

**旁白**：凌墨衝破了森林的迷霧,出現在山谷邊緣。

**旁白**：此刻的他,雙眼通紅,全身纏繞著濃郁的黑色火焰。

**凌墨**：(狂笑) 妳們以為躲在法陣裡就安全了嗎?

**旁白**：他猛地揮劍,一道巨大的黑色劍氣狠狠撞擊在光團上。

**旁白**：砰——!

**旁白**：光團劇烈震動,法陣表面出現了一道道裂紋。

**艾拉如**：(噴出一口鮮血,臉色慘白) 快...李杰!我撐不住了!

**李杰**：(握緊白劍,看了一眼斎藤和健太) 大家保重!

**旁白**：他一頭扎進了銀鏡之中。

**旁白**：一瞬間,世界天旋地轉。

**旁白**：當李杰再次恢復意識時,發現自己正站在一個由無數綠色代碼構成的空間。

**旁白**：腳下是流動的數據流,頭頂是盤旋的代碼雲。

**李杰**：(驚嘆) 這裡是...核心內核?

**旁白**：一個巨大的紅色核心正懸浮在前方,發出陣陣令人不安的嗡鳴聲。

**凌墨**：(聲音在空間中迴盪) 妳以為進來就能贏嗎?

**旁白**：凌墨的身影竟然也出現在了代碼空間中。

**凌墨**：(眼神瘋狂) 這裡是我的主場!

**旁白**：他打了個響指,無數黑色的代碼鏈條從四面八方射向李杰。

**李杰**：(揮動白劍) 滾開!

**旁白**：兩人在代碼空間展開了最後的對決。

**旁白**：白劍與黑劍在數據流中碰撞,激起一陣陣電火花。

**李杰**：(內心) 必須...必須破壞那個紅色核心!

**旁白**：他拼著受傷,硬接了凌墨一劍,然後藉力衝向了紅色核心。

**李杰**：(最後的吶喊) 給我碎吧!

**旁白**：白劍深深地刺入了紅色核心的中心。

**旁白**：咔嚓——

**旁白**：時間靜止了。

**旁白**：然後,耀眼的白光從核心內部爆發,瞬間淹沒了李杰的視野。

**旁白**：意識...消失了。
