# 第五十二幕：陰謀的真相

**旁白**：十一月二十八日,週四上午。

**旁白**：李杰坐在辦公室裡,眼睛盯著電腦螢幕,但一個字也沒看進去。

**旁白**：昨天那場突如其來的綁架,以及渡邊陸那張毀容的臉,一直在他腦海中盤旋。

**李杰**：(內心)他到底想幹什麼...還有那個叫莉子的警察...

**旁白**：就在這時,中野海斗神色慌張地跑了進來。

**中野海斗**：李君!出大事了!

**李杰**：(猛地抬頭)怎麼了?

**中野海斗**：(喘著氣)山田部長...他剛剛被警察帶走了!

**李杰**：(震驚)山田部長?為什麼?

**中野海斗**：(壓低聲音)聽說和公司的賬目造假有關。而且...警察還查封了他的電腦,說裡面有涉及「國家安全」的機密文件。

**旁白**：李杰心頭一沉。國家安全...

**旁白**：這件事似乎越鬧越大了。從一個普通的失蹤案,演變成了涉及公司高層和警察的複雜博弈。

**李杰**：(內心)難道大輔的失蹤,也和這些「機密文件」有關?

**旁白**：下班後,李杰再次來到了那家廢棄工廠。

**旁白**：既然渡邊陸說澪在那裡,那他就必須去確認。

**旁白**：工廠大門敞開著,裡面靜得可怕。

**李杰**：(握緊手中的短刀,小心翼翼地走進去) ...

**旁白**：就在他走到工廠中央時,一個冰冷的聲音突然從身後傳來。

**黑衣人**：(語氣冷漠)妳終於來了。

**李杰**：(猛地回頭)誰!

**旁白**：在陰影中,走出一個熟悉的身影。

**旁白**：黑色的衛衣,低壓的兜帽,手裡握著那把漆黑的長劍。

**李杰**：(咬牙)凌墨...

**凌墨**：(緩緩抬起頭,眼神中透著一絲瘋狂)妳想知道真相嗎?

**李杰**：(怒吼)少廢話!澪在哪裡!

**凌墨**：(指向工廠深處的一扇門)就在那扇門後。不過,妳得先打敗我。

**旁白**：凌墨說完,身形一動,整個人化作一道黑光,瞬間衝到了李杰面前。

**旁白**：黑劍猛地劈下!

**李杰**：(舉刀格擋)鐺——!

**旁白**：劇烈的碰撞震得李杰手臂發麻。

**凌墨**：(冷笑)太弱了。

**旁白**：他身形再次閃爍,出現在李杰身側,一腳將他踹飛了出去。

**李杰**：(重重撞在牆上)唔!

**凌墨**：(走到李杰面前,舉起黑劍)這就是真相。強者生存,弱者淘汰。

**旁白**：就在凌墨準備下死手時,那扇門突然打開了。

**旁白**：繪里花從裡面衝了出來。

**原田繪里花**：(尖叫)住手!凌墨!不要!

**旁白**：凌墨的手在空中停住了。他轉過頭,冷冷地看著繪里花。

**凌墨**：(語氣冰冷)妳怎麼出來了。

**原田繪里花**：(跑過來護在李杰身前,眼神堅定)妳不能殺他!

**凌墨**：(冷笑)為什麼?難道妳愛上他了?

**原田繪里花**：(臉色慘白)不...是因為...

**旁白**：她湊到凌墨耳邊,低聲說了一句話。

**凌墨**：(瞳孔猛地收縮,難以置信地看著李杰)什麼?!這不可能!

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
