# 第五十四幕：權力的遊戲

**旁白**：十一月三十日,週六下午。

**旁白**：李杰來到約定的地點——那是一座位於郊區的私人莊園。

**旁白**：高大的圍牆,全天候的監控,門口站著幾名全副武裝的保安。

**旁白**：這裡戒備森嚴,完全不像是一般的住宅。

**李杰**：(對保安說)我是來見渡邊先生的。

**旁白**：保安檢查了他的身份證件,然後示意他進去。

**旁白**：李杰穿過寬敞的草坪,走進了一棟充滿現代感的別墅。

**旁白**：客廳裡,渡邊陸正坐在一張沙發上,手裡拿著一根雪茄。

**渡邊陸**：(指了指對面的沙發)坐。

**李杰**：(坐下,開門見山)妳說要告訴我真相。

**渡邊陸**：(噴出一口煙霧)別急。我們先喝杯咖啡。

**旁白**：一名穿著制服的僕人端來兩杯熱騰騰的咖啡。

**渡邊陸**：(喝了一口咖啡,緩緩開口)妳知道「訊聯科技」背後的真正老闆是誰嗎?

**李杰**：(皺眉)不是山田部長嗎?

**渡邊陸**：(冷笑一聲)他?他只是個傀儡。真正的老闆,是一個妳絕對想不到的人。

**李杰**：(追問)是誰?

**渡邊陸**：(眼神變得深邃)是「他」。那個隱藏在所有超自然現象背後的操盤手。

**李杰**：(心頭一震)妳是指那個黑衣男人?

**渡邊陸**：(搖頭)不。他只是個打手。真正的老闆,是掌握著這個世界最高權力的人。

**李杰**：(震撼)最高權力...

**渡邊陸**：(繼續說道)他們在進行一項名為「重生」的實驗。而大輔...就是這項實驗的關鍵。

**李杰**：(眼神一凝)重生實驗?

**渡邊陸**：(點頭)利用空間傳輸技術,將人的靈魂備份到鏡像空間。這樣一來,即便現實世界的身體死亡,靈魂也能在那個世界永生。

**李杰**：(感到一陣惡寒)這簡直是瘋了...

**渡邊陸**：(狂笑)瘋了?這可是無數權貴夢寐以求的技術!

**渡邊陸**：(語氣變得陰沈)可惜,實驗出了差錯。大輔在備份過程中,無意中觸發了一個bug。

**李杰**：(追問)什麼bug?

**渡邊陸**：(看向窗外)他創造了一個完全獨立於現實世界的「幽冥之都」。那裡不再受任何人的控制,包括那些老闆。

**李杰**：(內心)難道這就是為什麼凌墨要清理那些乘客?為了掩蓋這個bug?

**渡邊陸**：(盯著李杰的眼睛)現在,那些老闆想拿回控制權。而妳...就是他們唯一的希望。

**李杰**：(後退一步)我?我能做什麼?

**渡邊陸**：(冷冷地說)妳的血脈裡,流淌著進入核心區域的密鑰。

**旁白**：就在這時,別墅的警報聲突然響起。

**保安**：(在大門口大喊)長官!不好了!有人闖進來了!

**李杰**：(立刻警覺地站起身)是誰!

**旁白**：一個身影撞破窗戶,一躍而入。

**旁白**：黑色的長風衣,手持黑劍。

**李杰**：(驚呼)凌墨!

**凌墨**：(眼神冰冷)妳們知道得太多了。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
