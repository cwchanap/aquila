# 第五十五幕：消失的雙親

**旁白**：十二月一日，週日。

**旁白**：李杰醒來時,感覺頭重腳輕。昨晚那場在別墅裡的驚險逃生,讓他至今感到心驚肉跳。

**旁白**：渡邊陸的話一直在他腦海中迴盪。重生實驗、幽冥之都、還有他父親的秘密...

**李杰**：(內心) 密鑰...難道我真的是這一切的關鍵?

**旁白**：他走到客廳,發現餐桌上放著一張紙條。

**旁白**：那是母親林婉如的字跡:「小杰,我和妳爸去一趟療養院。中午自己弄點吃的。」

**李杰**：(皺眉) 療養院?哪個療養院?

**旁白**：他想起昨天莉子說的話。大輔的父母就在那家秘密療養院!

**李杰**：(立刻抓起外套衝出家門) 不好!

**旁白**：他打了一輛計程車,直奔市郊。

**旁白**：一路上,他不停地撥打父母的電話,但始終無人接聽。

**旁白**：一個小時後,他來到了那家名為「晨曦」的療養院。

**旁白**：這裡大門緊閉,門口站著兩名穿著黑西裝的保安。

**李杰**：(跑過去) 我要找我的父母!李明遠和林婉如!

**保安**：(冷冷地擋住他) 這裡沒有妳說的人。請回吧。

**李杰**：(憤怒) 瞎說!他們剛剛還發訊息說要來這裡!

**旁白**：就在這時,療養院深處傳來一聲巨響。

**旁白**：一團漆黑的煙霧從主樓的窗口噴湧而出。

**李杰**：(臉色大變) 爸! 媽!

**旁白**：他不顧保安的阻攔,用力撞開大門衝了進去。

**旁白**：主樓的大廳裡空無一人,空氣中瀰漫著一股刺鼻的藥味。

**旁白**：他順著樓梯跑上三樓。在那裡,他看到了令人心碎的一幕。

**旁白**：一間病房的門開著。裡面一片狼藉,儀器被砸碎,病床倒在一邊。

**旁白**：在病床邊,散落著幾張照片。

**李杰**：(顫抖地撿起照片) ...

**旁白**：那是他三歲時,和父母在新宿車站的合影。

**旁白**：而照片的背面,赫然寫著渡邊陸的名字。

**李杰**：(跪倒在地上,眼淚滑落) 為什麼...為什麼要帶走他們...

**旁白**：就在這時,原本空蕩蕩的走廊里傳來一個低沈的聲音。

**大輔(AI)**：(聲音從牆上的廣播裡傳出) 李杰。這就是妳父親隱瞞的真相。

**大輔(AI)**：(語氣冰冷) 只有犧牲他們,才能換回妳的自由。

**李杰**：(抬頭看向廣播,眼神中充滿了仇恨) 妳這混蛋! 妳把他們藏在哪裡了!

**大輔(AI)**：(沒有回答,而是冷冷地說) 鑰匙。打開門。

**旁白**：牆壁上的液晶螢幕亮起,出現了一個倒計時器。

**旁白**：五分鐘。

**旁白**：如果他在五分鐘內不能做出選擇,整座療養院都將化為灰燼。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
