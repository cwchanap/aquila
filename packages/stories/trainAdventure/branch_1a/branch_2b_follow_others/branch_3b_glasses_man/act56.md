# 第五十六幕：神祕的聯繫

**旁白**：十二月二日，週一。

**旁白**：李杰坐在辦公室裡,看著電腦螢幕。

**旁白**：雖然表面上在工作,但他的腦海中卻在反覆思索著這幾天發生的事情。

**旁白**：渡邊陸、重生實驗、大輔的失蹤、還有凌墨的追殺...這一切都指向了一個巨大的陰謀。

**李杰**：(內心)大輔說過...我是唯一一個擁有「修正權限」的人...

**旁白**：就在這時,中野海斗走了過來。

**中野海斗**：李君,有人找妳。

**李杰**：(愣了一下)誰?

**中野海斗**：(指了指門口)她說是妳的朋友。

**旁白**：李杰看向門口。在那裡,站著一個年輕女子。

**旁白**：她綁著俐落的馬尾,穿著一套乾淨的職業裝,手裡提著一個精緻的公事包。

**李杰**：(驚訝)莉子?!

**旁白**：竟然是那個年輕女警。

**李杰**：(趕緊走過去)妳怎麼來了?這裡可是公司。

**莉子**：(微微一笑)別擔心。我現在是休假時間。

**莉子**：(壓低聲音)我有重要的線索要告訴妳。關於大輔的父母。

**旁白**：李杰心中一動。

**李杰**：跟我來。

**旁白**：兩人來到公司頂樓的露台。

**旁白**：冬日的陽光雖然微弱,但曬在身上還是有一絲暖意。

**莉子**：(從包裡拿出一份檔案)這是我從警方的秘密檔案室裡查到的。

**莉子**：(指著檔案上的照片)這是大輔的父母。

**旁白**：李杰看向照片。照片上是一對慈祥的中年夫妻,正坐在一列火車前微笑著。

**李杰**：(皺眉)這有什麼特別的嗎?

**莉子**：(眼神變得嚴肅)特別的地方在於...這張照片的拍攝日期。

**莉子**：(低聲唸道) 「2001年8月15日」。

**李杰**：(震驚)2001年?! 那不是...

**莉子**：(點頭)沒錯。那是第一次列車失蹤事件發生的日子。

**莉子**：(繼續說道)大輔的父母,正是那次事件的唯一兩名倖存者。

**李杰**：(手抖了一下)倖存者...

**莉子**：(指著檔案末尾的一行字)檔案裡記載,他們在回來後不久,就被送往了一家秘密療養院。

**莉子**：(盯著李杰的眼睛)而那家療養院的背後投資人...正是妳所在的「訊聯科技」。

**旁白**：這番話像是一道驚雷,在李杰耳邊炸開。

**李杰**：(內心)難道大輔加入這家公司,是為了尋找他的父母?

**莉子**：(語氣沉重)不僅如此。檔案裡還提到,他們在療養院裡接受了一種特殊的治療。

**莉子**：(低聲) 「基因重組」。

**李杰**：(震撼)基因重組?! 這是什麼瘋狂的科學!

**莉子**：(搖頭)我不清楚。但從那之後,大輔的父母就徹底消失了。警方也查不到任何後續記錄。

**旁白**：李杰看著檔案。原本清晰的真相,似乎又被蒙上了一層更厚的迷霧。

**李杰**：(內心)大輔、凌墨、還有我...我們之間...到底有什麼聯繫?

**莉子**：(看著李杰)妳要小心。我感覺...那些老闆已經注意到妳了。

**旁白**：就在這時,李杰的手機震動了一下。

**旁白**：是一則新訊息。發送者是：大輔（AI）。

**大輔**：『時間到了。李杰。來「門」那裡。我給妳看真正的真相。』

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
