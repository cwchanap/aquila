# 第五十七幕：再次進入

**旁白**：十二月三日，週二。

**旁白**：深夜十一點。新宿車站已經停止營運,只有微弱的應急燈在黑暗中閃爍。

**旁白**：李杰站在月台邊緣。

**旁白**：那個特殊的標誌,在手電筒的光束下顯得格外詭異。

**李杰**：(深吸一氣氣) 我來了。

**旁白**：他按照大輔訊息裡的指示,閉上眼睛,將手掌貼在標誌上。

**旁白**：一瞬間,熟悉的冰冷感再次傳遍全身。

**旁白**：當他睜開眼時,發現自己已經站在了那列漆黑的火車上。

**旁白**：車廂裡空蕩蕩的,只有大輔一個人在第一節車廂。

**李杰**：(走過去) 我來了。

**大輔**：(沒有回頭,平靜地說) 妳終於下定決心了。

**李杰**：(直視大輔) 告訴我真正的真相。

**大輔**：(轉過身,眼神中透著一絲疲憊) 真相就是...這列火車,其實是一台「靈魂提取器」。

**李杰**：(震驚) 靈魂提取?

**大輔**：(點頭) 那些權貴們害怕死亡。所以他們想出了這個辦法——在人死前,將意識備份到這個鏡像空間。

**大輔**：(指向窗外) 幽冥之都,其實就是一個巨大的硬盤。

**李杰**：(憤怒) 那那些無辜的乘客呢! 他們做錯了什麼!

**大輔**：(苦笑) 對於他們來說,我們只是「耗材」。為了維持空間的穩定,需要源源不斷的生命能量。

**李杰**：(握緊拳頭) 所以凌墨就是為了收集這些能量?

**大輔**：(搖頭) 凌墨...他本來也是個受害者。但他瘋了。他現在只想毀掉兩個世界。

**大輔**：(從懷中拿出一塊散發著金光的晶片) 這是重啟程序的「物理秘鑰」。只有妳能使用它。

**李杰**：(接過晶片) 為什麼是我?

**大輔**：(眼神深邃) 因為妳的體內...流淌著「第一代」試驗者的基因。

**大輔**：(指向鐵軌深處) 走吧。去終焉祭壇。在那裡,妳可以徹底關閉這個錯誤的程序。

**李杰**：(握緊晶片) 那妳呢?

**大輔**：(微笑) 我只是一個殘留的影子。我的任務...到此結束了。

**旁白**：大輔說完,身形逐漸變得透明,最終化作點點流光消散。

**旁白**：火車再次啟動,發出震耳欲聾的轟鳴聲,衝進了深不見底的隧道。

**李杰**：(看向前方) 這一次,我一定要結束這一切。

**旁白**：火車的速度越來越快。

**旁白**：就在這時,原本空無一人的車廂裡,突然傳來一陣淒厲的慘叫。

**李杰**：(臉色大變) 還有其他人!

**旁白**：他衝向後面的車廂。

**旁白**：而在這寂靜的夜晚,兩個迷失在異世界的人,暫時找到了短暫的休息之地。
