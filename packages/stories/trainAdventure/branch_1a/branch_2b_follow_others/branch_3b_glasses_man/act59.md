# 第五十九幕：夜路的陰影

**旁白**：十一月二十二日,週五晚上。

**旁白**：李杰加班到晚上十點才走出公司大樓。

**旁白**：晚秋的風帶著陣陣寒意,吹在臉上有些刺痛。他縮了縮脖子,拉緊風衣領口,快步走向新宿車站。

**旁白**：由於是週五晚上,街道上依然人流如織。居酒屋和卡拉OK店傳出陣陣喧鬧聲。

**旁白**：在月台上等車時,李杰感到一陣莫名的疲憊。

**李杰**：(揉著太陽穴)最近是不是工作太拼了...

**旁白**：中央線的快速列車進站。由於是末班車前的幾班,車廂裡擠滿了渾身酒氣的上班族。

**旁白**：李杰好不容易在門口找到一個可以靠著的位置。他閉上眼睛,試圖在搖晃的車廂裡小睡一會兒。

**旁白**：就在這時,那種熟悉的、如影隨形的注視感再次出現。

**李杰**：(猛地睜開眼,環顧四周) ...

**旁白**：車廂裡很擁擠,每個人都在低頭看手機或是打瞌睡。

**旁白**：但在他的右前方,距離他大約三公尺的位置,站著一個穿著黑色長風衣的男人。

**旁白**：男人戴著黑色的帽子,帽簷壓得很低,遮住了上半張臉。

**旁白**：儘管看不清眼神,但李杰能感覺到,那個人的目光正死死地盯著自己。

**李杰**：(心跳漏了一拍)是他...

**旁白**：那種冷冽、漠然的氣息,即使隔著厚厚的人群,也讓他感到一陣惡寒。

**旁白**：列車駛入隧道,窗外變得一片漆黑。

**旁白**：車廂裡的燈光閃爍了一下。

**旁白**：當燈光恢復正常時,那個黑衣男人已經不見了。

**李杰**：(揉了揉眼睛,難以置信地看著那個位置)消失了?

**旁白**：原本站著人的地方,現在只有兩個正在聊天的年輕人。

**李杰**：(內心)難道又是幻覺?

**旁白**：列車駛出隧道,到達了三鷹站。李杰走出車廂,踏上月台。

**旁白**：他加快腳步,穿過檢票口,走向回家的街道。

**旁白**：這條路他已經走了幾百次,路燈昏暗,兩旁是安靜的住宅區。

**旁白**：腳步聲在寂靜的夜裡顯得格外清晰。

**旁白**：突然,李杰停下了腳步。

**旁白**：在他的身後,隱約傳來了另一個腳步聲。

**旁白**：頻率和他完全一致,但稍微延遲了一點點。

**李杰**：(猛地回頭)誰!

**旁白**：身後的街道空蕩蕩的。只有路燈投下的昏黃光圈,和被風吹動的落葉。

**李杰**：(握緊拳頭,快步跑向公寓) ...

**旁白**：回到公寓,鎖好門,李杰背靠著門板,大口口地喘著氣。

**李杰**：(看著自己顫抖的手)這不正常...這絕對不正常。

**旁白**：他走到窗邊,拉開窗簾往外看。

**旁白**：在公寓對面的電線桿下,站著一個黑色的影子。

**旁白**：雖然看不清臉,但李杰知道,那個人正抬著頭,盯著他的窗戶。

**李杰**：(內心)他到底想幹什麼...

**旁白**：這場名為「生活」的遊戲,似乎正悄悄轉向一個危險的方向。
