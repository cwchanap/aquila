# 第六十幕：不安的等待

**旁白**：十一月二十五日，週一。

**旁白**：整個上午，李杰的手機都沒有震動。繪里花沒有回覆他的訊息。

**旁白**：他試圖專心工作，打開電腦檢查程式碼，但眼睛卻不自覺地飄向螢幕角落的時鐘。

**旁白**：九點半、十點、十點半...時間一分一秒地過去，手機依然安靜。

**旁白**：李杰的注意力無法集中。他盯著同一段程式碼看了十幾分鐘，卻完全沒有進展。

**旁白**：腦中不斷浮現各種猜測——她到底發生了什麼事？為什麼不回訊息？是不是遇到危險了？

**旁白**：就在這時，山田部長從辦公室走出來。

**山田部長**：李君！昨天交代你的測試報告做好了嗎？

**李杰**：（猛地回神，有些慌張）啊...部長，快、快好了，再給我半個小時。

**山田部長**：（皺起眉頭，看了看他的螢幕）你這一段寫了半天還在原地打轉。是不是最近太累了？

**李杰**：（尷尬地笑了笑）可能吧，昨晚沒睡好。

**山田部長**：（嘆了口氣，語氣緩和了一些）好吧，做完這個報告就休息一下吧。別太拼命了。

**李杰**：謝謝部長。

**旁白**：部長走後，李杰深吸一口氣，強迫自己把注意力集中在報告上。

**旁白**：然而，那種不安感卻像是一條毒蛇，緊緊纏繞著他的心臟。

**旁白**：午休時間。

**旁白**：李杰坐在位置上，看著桌上的便當，一點胃口都沒有。

**旁白**：中野海斗走過來，敲了敲他的桌面。

**中野海斗**：李君，去吃午飯嗎？聽說樓下開了一家新的拉麵店。

**李杰**：（搖搖頭）不用了，我不太餓。

**中野海斗**：（打量著他，關切地問）你臉色好差。發生什麼事了嗎？

**李杰**：（猶豫了一下，還是決定問出口）中野君，妳今天有看到繪里花...不，原田小姐嗎？

**中野海斗**：（愣了一下）原田小姐？她今天請假了啊。你不知道嗎？

**李杰**：（震驚）請假？

**中野海斗**：（點頭）對啊，早上部長在晨會上說的。說她家裡有急事，要請一週的假。

**李杰**：（心頭一沈）一週...

**中野海斗**：你沒事吧？

**李杰**：（勉強笑了笑）沒事。我只是想問她一些工作上的事。

**旁白**：中野走後，李杰立刻拿出手機，再次撥打繪里花的電話。

**旁白**：依然是那陣冰冷的、無人接聽的提示音。

**旁白**：李杰坐在位置上，感到一陣前所未有的孤立無援。

**旁白**：在這個繁華的城市裡，他突然發現，除了工作，他對那個每天坐在他旁邊、對他微笑的女孩，竟然一無所知。

**旁白**：家裡的急事...到底是什麼事？

**旁白**：那種如影隨形的危機感再次襲上心頭。

**旁白**：就在這時，他的手機突然震動了一下。

**旁白**：李杰心跳加速，立刻抓起手機。

**旁白**：是一則新訊息。發送者是：原田繪里花。

**旁白**：李杰屏住呼吸，點開了訊息。
