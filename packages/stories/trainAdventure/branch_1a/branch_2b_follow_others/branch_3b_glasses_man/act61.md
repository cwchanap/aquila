# 第六十一幕：上海的假期

**旁白**：四月二十九日，週六上午。李杰搭乘早班飛機從東京成田機場起飛。

**旁白**：兩個半小時後，飛機降落在上海浦東國際機場。

**旁白**：走出機場，熟悉的空氣撲面而來。上海的春天比東京溫暖一些，陽光明媚，天空湛藍。

**旁白**：李杰提著行李，搭上磁浮列車前往市區，然後轉乘地鐵。

**旁白**：地鐵裡人潮洶湧，到處都是說著上海話和普通話的聲音。這種熟悉感讓他感到放鬆。

**旁白**：大約一個小時後，李杰到達了家附近的地鐵站。他走出車站，沿著熟悉的街道往家的方向走。

**旁白**：這是一個老社區，樓房都有二三十年的歷史了。街道兩旁種著梧桐樹，陽光透過樹葉灑下斑駁的光影。

**旁白**：李杰走進一棟老式公寓，爬上四樓。他站在熟悉的門前，按下門鈴。

**旁白**：「叮咚——」

**旁白**：門很快就開了。

**林婉如**：（驚喜地）小杰！你回來啦！

**旁白**：開門的是李杰的母親。她穿著一件花色的圍裙，手上還沾著麵粉，顯然正在廚房忙碌。

**李杰**：（微笑著擁抱母親）媽，我回來了。

**林婉如**：（高興地拍拍他的背）回來就好！快進來！你爸剛還在說你呢。

**旁白**：李杰走進屋子。客廳裡擺設依舊，充滿了溫馨的生活氣息。電視開著，播放著上海地方新聞。

**李明遠**：（從陽台走進來，放下手中的花剪）回來了。

**旁白**：李杰的父親是個沈默寡言的人，雖然語氣平淡，但眼神中卻透著掩飾不住的喜悅。

**李杰**：爸。

**林婉如**：（接過他的行李）累壞了吧？快坐下休息一會兒。我正在包餛飩，中午我們吃薺菜大肉餛飩！

**李杰**：（笑了）好，我最想念妳做的餛飩了。

**旁白**：午飯非常豐盛。除了熱騰騰的餛飩，還有紅燒肉、清蒸魚、和幾道李杰愛吃的家常菜。

**旁白**：一家三口圍坐在餐桌前，邊吃邊聊。

**林婉如**：（一邊給李杰夾菜一邊問道）在那邊工作還順利嗎？生活還習慣嗎？

**李杰**：都挺好的。工作雖然忙，但挺充實的。

**李明遠**：（喝了一口茶，沈聲道）要注意身體，不要太拼命了。

**李杰**：我知道，爸。

**旁白**：在這個溫暖的家裡，李杰感到前所未有的安心。那些在東京感到的壓力、迷茫、還有那種若有似無的被窺視感，彷彿都留在了那個海島國家。

**旁白**：在這裡，他只是李杰，是父母眼中的孩子。

**旁白**：吃完午飯，李杰回到自己的房間。房間被打掃得乾乾淨淨，床單換成了他最喜歡的藍色。

**旁白**：他躺在床上，看著天花板，聽著客廳裡電視機的聲音和父母低聲交談的聲音。

**旁白**：這一刻，他覺得這才是真實的生活。

**旁白**：下午，李杰陪著母親去附近的菜市場買菜。

**旁白**：菜市場裡熱鬧非凡，攤販們大聲吆喝著。母親熟練地挑選著蔬菜和肉類，不時和熟識的攤主聊上幾句。

**攤主**：（笑著對林婉如說）林阿姨，這是妳兒子吧？長得真俊！在日本工作的吧？

**林婉如**：（自豪地點頭）對，放假回來看看。

**旁白**：走在喧囂的街道上，李杰看著母親忙碌的背影，心中湧起一股暖流。

**旁白**：晚飯後，一家人下樓散步。

**旁白**：夜晚的社區安靜而祥和。老人們在樹下聊天，孩子們在空地上嬉戲。

**旁白**：李杰走在父母中間，聽著他們聊著家長里短。

**旁白**：這三天的假期，是他近年來最輕鬆、最快樂的時光。

**旁白**：然而，在這種平凡的幸福背後，一絲不易察覺的陰影正悄悄潛伏。

**旁白**：就在這時，李杰的手機震動了一下。

**旁白**：是一條微信訊息。發送者是：張偉。

**張偉**：『杰哥，聽說你回來了？出來聚聚吧！我們幾個都在呢！』

**李杰**：(笑了) 果然是這幫傢伙。
