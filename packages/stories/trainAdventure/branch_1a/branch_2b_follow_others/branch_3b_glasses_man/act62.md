# 第六十二幕：父子間的秘密

**旁白**：四月三十日，上午十點。林婉如提著購物袋出門去菜市場。她說中午要做糖醋小排，得去買新鮮的豬肋排。

**林婉如**：小杰，你和爸爸在家等著。我很快就回來。

**李杰**：好，媽慢走。

**旁白**：大門關上的聲音在公寓裡迴盪。李杰坐在客廳的沙發上，手裡拿著手機，隨意地滑著新聞。李明遠剛從陽台走進來，他穿著灰色的舊T恤和運動褲，手裡拿著一本報紙。

**旁白**：父親在對面的單人沙發坐下，展開報紙開始閱讀。客廳裡很安靜，只有翻報紙的沙沙聲，還有遠處傳來的汽車聲。

**旁白**：李杰偷偷瞄了父親一眼。李明遠戴著老花眼鏡，專注地看著報紙。臉上的皺紋在晨光中顯得更深了，兩鬢的白髮也比上次回來時多了一些。

**旁白**：這個畫面讓李杰感到有些不真實。他和父親單獨相處的時間並不多。平時在家裡，大部分時候都是母親在說話，父親只是偶爾應聲。而李杰自己，也更習慣和母親聊天。和父親說話總是有些尷尬，不知道該說什麼。

**旁白**：沉默持續了。李杰看著手機螢幕，但腦中想的卻是別的事——兩年多前，他被黑西裝男人綁架的那一天。那個冰冷的男人說的話，至今還清晰地迴盪在他的記憶裡。

**李杰**：（沈默了片刻，終於開口）爸，我有件事想問您。

**旁白**：李明遠沒有抬頭，只是淡淡地應了一聲。

**李明遠**：什麼事？說吧。

**李杰**：（猶豫了一下）您...您還記得一個叫渡邊陸的人嗎？

**旁白**：客廳裡的空氣彷彿在瞬間凝固了。

**旁白**：翻動報紙的沙沙聲戛然而止。

**旁白**：李明遠握著報紙的手微微顫抖了一下，雖然動作極其輕微，但李杰還是敏銳地捕捉到了。

**李明遠**：（依然盯著報紙，語氣平淡）不認識。沒聽說過。

**李杰**：（盯著父親的側臉，繼續追問）可是有人告訴我，您二十五年前在日本工作的時候，和他有過聯繫。

**旁白**：這一次，李明遠終於抬起了頭。他隔著老花眼鏡看向李杰，眼神中閃過一絲複雜的情緒——驚訝、警惕、還有一絲深深的憂慮。

**李明遠**：（沈聲問道）是誰告訴你的？你在日本遇到什麼人了嗎？

**李杰**：（避開了問題）您真的不認識他嗎？聽說他在一家叫「訊聯科技」的公司工作過。

**旁白**：聽到「訊聯科技」這四個字，李明遠的臉色瞬間變得鐵青。他猛地放下報紙，站起身走向陽台，背對著李杰。

**李明遠**：（聲音有些沙啞）我說過，我不認識這個人。以後不要再提這個名字了。

**李杰**：（內心）果然...他知道渡邊陸。

**旁白**：李杰看著父親的背影，心中湧起一股強烈的不安感。父親的反應太激烈了，完全不像是在面對一個陌生人的名字。

**李杰**：（站起身，走到陽台門口）爸，如果這個人威脅到我們的安全，我有權知道真相。

**李明遠**：（猛地轉過身，眼神銳利得讓李杰感到陌生）真相？你知道什麼叫真相嗎？

**李明遠**：（語氣變得嚴厲）有些事情，不知道才是最好的保護。如果你真的想幫我，就老老實實在那邊工作，不要去管這些陳年舊事。

**李杰**：（倔強地）可是他已經找到我了！兩年前我就被他的人綁架過！

**旁白**：這句話像是一記重錘，狠狠地砸在了李明遠的心上。他的身體劇烈地搖晃了一下，臉色從鐵青轉為慘白。

**李明遠**：（顫抖著，聲音幾乎聽不見）妳...妳說什麼？妳被綁架過？

**李杰**：（點點頭，神情沈重）對。雖然最後脫險了，但他留了一句話給我。

**李明遠**：（急切地抓住李杰的手臂）他說了什麼？

**李杰**：（看著父親的眼睛）他說...「欠下的債，終究是要還的。」

**旁白**：李明遠鬆開了手，整個人癱坐在陽台的椅子上，像是瞬間蒼老了十歲。

**李明遠**：（閉上眼睛，長嘆一聲）債...呵呵...債...

**旁白**：就在這時，玄關傳來開門的聲音。

**林婉如**：（歡快的聲音傳進來）我回來啦！買到了上好的排骨！

**旁白**：母親的聲音打破了客廳裡凝重的氣氛。

**旁白**：林婉如提著購物袋走進來,臉上帶著笑容。

**林婉如**：小杰,來幫媽媽拿一下東西。

**李杰**：好。

**旁白**：李杰站起身,走向母親。

**旁白**：在轉身的瞬間,他回頭看了父親一眼。

**旁白**：李明遠依然坐在沙發上,表情平靜地看著報紙。

**旁白**：但李杰注意到,報紙的頁面,從剛才到現在都沒有翻過。
