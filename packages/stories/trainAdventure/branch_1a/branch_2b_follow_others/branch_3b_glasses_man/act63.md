# 第六十三幕：意外的線索

**旁白**：五月三日，晚上七點。李杰和三個朋友約在徐家匯的一家KTV。

**旁白**：包廂裡燈光昏暗，音響播放著節奏感強烈的音樂。牆上的大螢幕顯示著歌詞，茶几上擺著啤酒、小吃和果盤。

**旁白**：張偉正拿著麥克風唱著一首老歌，聲音走音得厲害，但唱得很投入。王浩和陳宇坐在沙發上，一邊喝啤酒一邊起鬨。

**王浩**：偉哥！唱得太好了！再來一首！

**陳宇**：哈哈哈！這高音簡直要破音了！

**張偉**：去去去！你們行你們來！

**旁白**：張偉唱完最後一句，把麥克風扔給王浩。

**張偉**：輪到你了！

**旁白**：李杰坐在沙發的一角，手裡握著一瓶啤酒，靜靜地看著朋友們打鬧。

**旁白**：他的腦海中還在迴盪著那天早晨和父親的對話。

**李杰**：（內心）渡邊陸...債...

**旁白**：就在這時，陳宇坐到了他身邊，遞給他一塊西瓜。

**陳宇**：杰哥，怎麼一個人在這發呆？來，吃塊西瓜。

**李杰**：（接過西瓜）謝謝。

**陳宇**：看你這幾天好像心事重重的。是不是家裡出什麼事了？

**李杰**：（猶豫了一下，決定試探一下）陳宇，你大學畢業後不是在訊聯科技待過一段時間嗎？

**陳宇**：（點頭）對啊，待了兩年。怎麼突然問起這個？

**李杰**：那你在那裡的時候，有沒有聽說過一個叫「斎藤大輔」的人？

**旁白**：李杰故意說出了大輔的名字。他想看看，在同一個行業、同一家公司，會不會有什麼線索。

**陳宇**：（瞪大眼睛）斎藤大輔？當然知道啊！

**陳宇**：（有些興奮地）那可是我們公司的傳奇人物！

**李杰**：（心跳加速）傳奇人物？

**陳宇**：對啊！他是從日本過來的安全專家。聽說他寫的防護系統，至今都沒人能攻破。

**陳宇**：（壓低聲音）不過這不是最傳奇的。最傳奇的是他的身份。

**李杰**：什麼身份？

**陳宇**：聽說他其實是個天才駭客。大學還沒畢業就被公司特招了。

**陳宇**：（唏噓道）可惜，後來聽說他出了意外，失蹤了。

**李杰**：（握緊酒瓶）失蹤了？什麼時候的事？

**陳宇**：我想想...應該是2026年吧。也就是大概兩年前。

**李杰**：（內心）2026年...那正是大輔在現實世界失蹤的時間。

**陳宇**：不過最奇怪的還不是這個。

**李杰**：還有什麼？

**陳宇**：（神秘兮兮地）聽說他失蹤的前一天，正好是公司一個大項目的測試日。

**陳宇**：那個項目據說和「空間傳輸」有關。當時我級別太低，只是聽前輩們私下討論。

**旁白**：空間傳輸...

**旁白**：這個詞讓李杰感到一陣眩暈。

**李杰**：（追問）那...那個項目的負責人是誰？

**陳宇**：好像姓渡邊。具體叫什麼我不記得了。

**旁白**：姓渡邊...

**旁白**：李杰的手心滲出了冷汗。

**李杰**：（內心）難道真的是渡邊陸？

**旁白**：就在這時，包廂的門開了，張偉提著幾瓶新啤酒走進來。

**張偉**：來來來！繼續喝！杰哥別光顧著聊天啊！

**旁白**：李杰接過啤酒，勉強笑了笑。

**李杰**：好。

**旁白**：他仰頭喝了一大口啤酒。冰涼的液體滑過喉嚨，卻無法冷卻他心中的燥熱。

**旁白**：線索正一點點拼湊起來。

**旁白**：父親工作過的保安公司、大輔工作的訊聯科技、渡邊姓氏的項目負責人...

**旁白**：這一切絕非巧合。

**旁白**：而在這喧囂的KTV裡，李杰感到自己正被捲入一個巨大的漩渦。

**陳宇**：杰哥？妳還好吧？

**李杰**：(回過神來) 沒事。只是覺得大輔的事情挺可惜的。

**陳宇**：是啊。對了，妳怎麼突然對他感興趣？

**李杰**：沒什麼。在那邊遇到一個姓斎藤的人，所以隨口問問。

**旁白**：李杰隱瞞了真相。他不想讓朋友們也捲入這場危險的遊戲中。

**旁白**：但他知道,這場遊戲,才剛剛開始。
