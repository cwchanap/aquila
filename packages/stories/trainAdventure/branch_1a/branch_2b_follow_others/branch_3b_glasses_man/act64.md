# 第六十四幕：最後的家庭時光

**旁白**：五月六日，早上九點。

**旁白**：李杰收拾好行李，把行李箱放在客廳角落。明天晚上就要去機場，今天是在上海的最後一天。

**旁白**：母親從臥室走出來，換上了一件淺藍色的連衣裙，臉上化了淡妝，看起來精神奕奕。父親也換上了乾淨的襯衫和休閒褲，難得沒有穿那件灰色舊T恤。

**李杰**：好。

**旁白**：李杰看著父母，心中湧起一種奇妙的感覺。

**旁白**：父母今天都休息，說要陪他好好逛逛上海。這種三個人一起出去玩的機會，李杰已經記不清上次是什麼時候了。

**旁白**：也許是他高中畢業那年？還是更早？

**旁白**：他們去了豫園，在古香古色的長廊裡穿行。母親興致勃勃地看著那些手工藝品，不時拉著李杰和父親合影。父親雖然嘴上說著麻煩，但每次拍照都站得很直，臉上帶著淡淡的笑容。

**旁白**：中午在九曲橋旁的餐廳吃了南翔小籠包。皮薄汁多，味道鮮美。

**旁白**：下午去了外灘。

**旁白**：雖然是假期，但外灘的人潮依然洶湧。江對面的陸家嘴金融區高樓林立，在陽光下閃爍著現代都市的繁華。

**旁白**：江風徐徐吹過，帶著一絲潮濕的水汽。

**林婉如**：（指著東方明珠）小杰，你看！那裡變了好多。

**李杰**：（點頭）是啊，每次回來都覺得不一樣。

**旁白**：父親站在欄杆邊，看著江面上來往的船隻。

**李明遠**：（沈聲道）上海發展得太快了。有時候覺得，我們這輩人都快跟不上腳步了。

**李杰**：（走到父親身邊）爸，您已經做得很好了。

**旁白**：父親轉過頭看了他一眼，眼神中閃過一絲欣慰，但隨即又被一種沈重所取代。

**李明遠**：（沈默了一會兒）在那邊...真的還好嗎？

**李杰**：（沒想到父親會突然問這個，愣了一下）嗯，挺好的。公司雖然忙，但能學到很多東西。

**李明遠**：（點點頭）那就好。記住，不管發生什麼，保護好自己才是最重要的。

**李杰**：（看著父親嚴肅的神情）爸，您是不是有什麼話想跟我說？

**旁白**：父親盯著江面看了很久，最後只是長長地嘆了一口氣。

**李明遠**：沒什麼。累了就回家。家門永遠為你開著。

**旁白**：這句話讓李杰的眼眶微微一熱。

**旁白**：他知道父親在擔心他，也在隱瞞著什麼。但他此刻不想去追問。他只想享受這最後的、難得的家庭時光。

**旁白**：他們在公園裡找了一張長椅坐下。父親去附近的小店買了三瓶水回來。

**李明遠**：喝點水吧。

**李杰**：謝謝爸。

**旁白**：李杰接過水，擰開瓶蓋喝了一口。

**旁白**：他偷偷看了父親一眼。陽光下，父親的側臉顯得有些蒼白。兩鬢的白髮在光線中格外明顯。

**林婉如**：小杰，來！我們拍張照！

**旁白**：母親招手叫來一個路人，請他幫忙拍照。

**林婉如**：好久沒拍全家福了！

**路人**：準備好了嗎？笑一個！一、二、三！

**旁白**：快門聲響起。那一瞬間，李杰覺得時間好像靜止了。

**旁白**：陽光、湖水、父母的笑容，這一切都被定格在照片裡。

**旁白**：吃完飯，天色已經暗了下來。街燈亮起，城市開始展現出另一種風貌。

**旁白**：回到家，母親開始幫李杰整理行李，塞進各種上海的土產。

**林婉如**：這些帶給同事吃。還有這些乾貨，留著妳自己燉湯喝。

**李杰**：（笑著）媽，太多了，行李箱都快塞不下了。

**林婉如**：（一邊塞一邊叮囑）在那邊要好好吃飯，別天天點外賣。有空多給家裡打電話。

**李杰**：我知道了。

**旁白**：那一晚，李杰睡得很沈。

**旁白**：夢裡沒有火車，沒有黑衣人，只有小時候家門口那棵繁茂的梧桐樹，在風中輕輕搖曳。

**旁白**：第二天晚上，李杰登上了回東京的飛機。

**旁白**：在起飛前的一刻，他看向窗外的上海夜景。

**旁白**：萬家燈火，繁華璀璨。

**旁白**：他知道，這短暫的平靜即將結束。

**旁白**：新的挑戰，或者說，那場未完成的遊戲，正在東京月台上等待著他。

**旁白**：但他不再感到恐懼。

**旁白**：因為他知道，有些東西，是任何超自然力量都無法奪走的。

**旁白**：飛機沖向雲霄，沒入漆黑的夜空中。
