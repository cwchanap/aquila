# 第六十八幕：命運的推手

**旁白**：李杰深吸一口氣，朝北方走去。

**旁白**：離開神樹後，他沿著森林中的道路繼續前進。

**旁白**：這條路比之前走過的路更窄，兩旁的樹木更加茂密。

**旁白**：腳下是柔軟的泥土和落葉，每一步都會發出細微的沙沙聲。

**旁白**：走了大約一個小時後，森林開始變得稀疏。

**旁白**：樹木逐漸變少，露出更多的岩石和山壁。

**旁白**：空氣變得越來越冷，能看到呼出的白霧。

**旁白**：又走了大約三個小時後，地形開始改變。

**旁白**：他來到了一座陡峭的山坡前。山坡上長滿了奇異的紫色灌木,散發著淡淡的光芒。

**李杰**：（喘著氣）呼...呼...

**旁白**：他爬上山坡,眼前的景象讓他瞬間屏住了呼吸。

**旁白**：在山坡的另一側,竟然隱藏著一座巨大的地下遺跡。

**旁白**：巨大的石門敞開著,門縫中透出微弱的藍光。

**李杰**：（內心）終焉祭壇...

**旁白**：他加快腳步,衝向石門。

**旁白**：然而,就在他踏入遺跡的一瞬間,一個黑影突然擋住了他的去路。

**李杰**：（猛地停下腳步,握緊手中的白劍）誰!

**旁白**：黑影緩緩走出陰影。那是個穿著黑色長風衣的男人,臉部被帽簷遮住,看不清長相。

**旁白**：他的手裡握著一把細長的黑劍,劍身上流動著詭異的紅光。

**黑衣人**：（語氣冰冷）妳不該來這裡。

**李杰**：（咬牙）妳是誰!為什麼要在這裡攔著我!

**黑衣人**：（緩緩抬起頭,露出那張熟悉的、冷酷的臉）凌墨...

**李杰**：（震驚）妳...妳為什麼會在這裡!妳不是在精靈村嗎!

**凌墨**：（冷笑一聲）精靈村?那只不過是我的分身罷了。

**凌墨**：（指向遺跡深處）妳想救那個男人?

**李杰**：（握緊劍柄）對!

**凌墨**：（語氣變得瘋狂）那就看看妳有沒有那個本事了!

**旁白**：凌墨說完,身形突然一動,整個人化作一道紫色殘影,瞬間衝到了李杰面前。

**旁白**：黑劍與白劍在空中激烈交鋒,爆發出耀眼的火花。

**旁白**：強大的衝擊波向四周擴散,震碎了周圍的石柱。

**李杰**：（被震退數步,虎口發麻）好強的力量...

**凌墨**：（攻勢不停）受死吧!不合格者!

**旁白**：黑色的劍化作無數道殘影,將李杰團團圍住。

**李杰**：（拼命抵擋）唔...!

**旁白**：就在這千鈞一髮之際,遺跡深處突然傳來一陣巨大的震動。

**旁白**：伴隨著低沈的轟鳴聲,一道金色的光柱沖天而起。

**凌墨**：（停下動作,臉色劇變）什麼?!核心被啟動了?!

**李杰**：（驚喜）大輔!

**旁白**：凌墨沒有理會李杰,轉身衝向遺跡深處。

**李杰**：（一咬牙）等等我!

**旁白**：兩人一前一後,消失在悠長的隧道中。

**旁白**：而命運的齒輪，在這一刻再次轉動。
