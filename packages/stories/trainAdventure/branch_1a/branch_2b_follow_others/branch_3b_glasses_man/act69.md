# 第六十九幕：重返月台

**旁白**：李杰握著白劍，走出儲藏室。

**旁白**：劍身散發著柔和的白光，照亮了狹窄的走廊。

**旁白**：他沿著走廊快步返回，大約二十分鐘後，回到了候車大廳。

**旁白**：斎藤依然坐在原地，而凌墨躺在一旁，雙眼緊閉，一動不動。

**斎藤蓮**：(看到李杰回來，站起身)李桑，你找到了什麼？

**旁白**：他的目光落在李杰手中的白劍上，眼睛瞬間睜大。

**斎藤蓮**：(驚訝)那是...劍？

**李杰**：(點點頭)神劍。在儲藏室找到的。

**斎藤蓮**：(有些興奮)所以...我們現在可以回家了嗎？

**李杰**：(沈默片刻,看向凌墨) 還得帶上他。

**斎藤蓮**：(一臉嫌棄) 為什麼?他可是想殺我們的人!

**李杰**：(語氣堅定) 他也是受害者。而且...只有他知道真正的出口。

**旁白**：李杰收起白劍,彎下腰將昏迷的凌墨背了起來。

**李杰**：走。

**旁白**：兩人穿過空曠的大廳,走向大廳盡頭的那扇拱門。

**旁白**：跨入拱門的瞬間,周圍的空氣再次變得扭曲。

**旁白**：當視野重新清晰時,李杰驚訝地發現,自己竟然站在新宿車站的月台上。

**李杰**：(震驚) 新宿車站?

**斎藤蓮**：(興奮地跳了起來) 回來了!真的回來了!我就知道!

**旁白**：月台上人來人往,熱鬧非凡。廣播裡傳來熟悉的進站鈴聲。

**廣播**：由新宿開往三鷹的中央線末班車即將進站...

**李杰**：(低頭看自己的手) 雙手依然沾滿了血跡...凌墨也還在背上...

**旁白**：他意識到,這並不是真正的現實。

**李杰**：(看向四周,語氣沈重) 斎藤,別高興得太早。看那裡。

**旁白**：斎藤順著他的手指看去。原本普通的自動販賣機,此刻正不斷地滲出黑色的粘液。

**旁白**：月台上的乘客們,臉色蒼白得不正常,眼神空洞,如同行屍走肉一般在月台上徘徊。

**斎藤蓮**：(嚇得往後一躲) 這、這是什麼鬼地方...

**李杰**：(內心獨白) 虛假的現實...看來大輔說的沒錯。

**旁白**：就在這時,原本昏迷的凌墨突然發出一聲微弱的呻吟。

**凌墨**：(緩緩睜開眼睛,看著眼前的景象,露出一抹慘笑) 終於...又回到了這裡。

**李杰**：(沈聲問道) 妳醒了。告訴我,這裡到底是哪裡?

**凌墨**：(掙扎著站起身,靠在柱子上) 這裡是「臨界點」。現實與虛幻的交界處。

**凌墨**：(指著鐵軌深處) 如果想真正回去,就必須穿過那條隧道。

**李杰**：(看著深不見底的隧道) 那裡面有什麼?

**凌墨**：(眼神中閃過一絲恐懼) 妳最想忘記的東西。

**旁白**：就在這時,廣播聲再次響起,聲音變得尖銳、刺耳。

**廣播**：列車即將進站!請各位旅客做好最後的準備!

**旁白**：一輛通體漆黑、散發著死亡氣息的列車衝出隧道,在月台邊緩緩停下。

**李杰**：(握緊白劍,眼神堅定) 走吧。

**旁白**：而在這寂靜的夜晚，兩個迷失在異世界的人，暫時找到了短暫的休息之地。
