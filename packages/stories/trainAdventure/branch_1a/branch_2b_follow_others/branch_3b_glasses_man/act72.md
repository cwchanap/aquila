# 第七十二幕：死亡遊戲

**旁白**：李杰在荒地上行走。一步,一步。他不知道走了多久。

**旁白**：那座建築物的輪廓逐漸清晰。從遠處的模糊影子,變成了具體的形狀。

**旁白**：當李杰終於靠近時,他停下了腳步。

**旁白**：那是一座學校。

**旁白**：一座三層樓的現代化校舍,白色的外牆,大片的玻璃窗,典型的日本高中建築風格。

**旁白**：但它孤零零地矗立在這片紫色天空下的荒地中,四周什麼都沒有。沒有操場,沒有圍牆,沒有其他建築。

**李杰**：（內心,感到詭異）高中...為什麼會有一座高中在這裡...

**旁白**：李杰走向學校的大門。那是一扇玻璃門,可以看到裡面的走廊和樓梯。

**旁白**：他推了推門。門沒有鎖,輕易地就被推開了。

**旁白**：李杰走進去。

**旁白**：校舍內部非常安靜,安靜得只能聽到他自己的腳步聲在空曠的走廊上迴盪。

**旁白**：陽光——不,紫色的月光透過玻璃窗灑進來,在光滑的地板上投下長長的影子。

**旁白**：牆壁上貼著學生們製作的海報和通知,一切都和普通的學校一模一樣。

**李杰**：（內心）這裡有人嗎?

**旁白**：李杰一間教室一間教室地走過。所有的教室都是空的。課桌椅整齊地排列著,黑板上還殘留著粉筆擦過的痕跡。

**旁白**：直到他來到二樓的一間教室。

**旁白**：門是半開著的。

**旁白**：李杰推開門。

**旁白**：教室裡坐著一個人。

**旁白**：一個身穿高中生制服的年輕男子。他正趴在桌子上睡覺,雙手枕在臉下。

**李杰**：（驚訝,試探地喊聲）餵...

**旁白**：那人沒有反應。

**旁白**：李杰走上前,輕輕拍了拍那人的肩膀。

**李杰**：餵,醒醒。

**旁白**：那人動了一下,緩緩抬起頭。

**旁白**：當他看清那人的臉時,李杰整個人都僵住了。

**李杰**：（失聲叫道）颯真?!

**旁白**：那人揉了揉朦朧的睡眼,看著李杰,露出了一抹驚訝而又溫暖的微笑。

**如月颯真**：（驚喜地）蓮!你終於來啦!

**李杰**：（顫抖地問）颯真...你怎麼會在這裡?這到底是怎麼回事?

**如月颯真**：（站起身,伸了個懶腰）大家都在等你呢。快跟我來。

**旁白**：颯真不由分說地拉起李杰的手,跑出教室,朝著樓上的禮堂跑去。

**旁白**：一路上,李杰大腦一片混亂。颯真不是死了嗎?為什麼他會穿著高中制服出現在這裡?

**旁白**：當他們推開禮堂大門時,李杰徹底呆住了。

**旁白**：禮堂裡擠滿了人。

**旁白**：陸斗、詩織、彩花、真理...所有在列車上的人,竟然都在這裡!

**旁白**：他們穿著各自的衣服,正興高采烈地交談著。看到李杰進來,所有人同時轉過頭,爆發出一陣歡呼聲。

**眾人**：（大喊）蓮!歡迎回來!

**李杰**：（愣愣地站在門口）這...這是在幹什麼...

**旁白**：陸斗跑過來,一把摟住李杰的肩膀。

**氷室陸斗**：（興奮地）你太慢啦!大會都快開始了!

**李杰**：（掙扎著問）大會?什麼大會?

**旁白**：就在這時,禮堂台上的燈光突然亮起。

**旁白**：一個熟悉的身影走到麥克風前。

**旁白**：那是穿著白色醫生袍的大輔。他推了推金邊眼鏡,露出了一抹神秘的微笑。

**斎藤大輔**：（聲音響徹整個禮堂）各位乘客,請安靜。

**旁白**：禮堂瞬間安靜下來。

**斎藤大輔**：（看著李杰,眼神深邃）歡迎來到最後的考驗。

**斎藤大輔**：這不是什麼遊戲,也不是什麼修復程式。

**斎藤大輔**：（語氣變得嚴肅）這裡是「審判之地」。

**旁白**：這句話像一聲驚雷,在李杰耳邊炸開。

**李杰**：（震驚）審判之地?!

**斎藤大輔**：（點頭）在進入真實世界之前,你們每個人都必須面對自己內心最深處的恐懼與罪惡。

**斎藤大輔**：（指向台下的眾人）這裡的所有人,都是你記憶中的一部分。

**斎藤大輔**：你的選擇,將決定他們——以及你自己的最終命運。

**旁白**：大輔打了個響指。

**旁白**：禮堂的牆壁突然變成了透明。

**旁白**：外面原本荒涼的世界,此刻竟然被無數黑色的火焰包圍。

**斎藤大輔**：（冰冷地說）如果你無法通過審判,這裡的一切,包括你在乎的所有人,都將被這黑色的火焰徹底抹殺。

**斎藤大輔**：（盯著李杰的眼睛）現在,開始你的最後一場遊戲吧。

**斎藤大輔**：這場名為「真實」的死亡遊戲。

**旁白**：全場再次陷入死一般的寂靜。

**旁白**：李杰站在人群中央,看著那些熟悉的、卻又可能是虛幻的面孔。

**旁白**：他感到前所未有的壓力。

**旁白**：一場關於靈魂、關於記憶、關於救贖的最終博弈,正式拉開了序幕。

**旁白**：—— 全劇終 ——
