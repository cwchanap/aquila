# 第八幕：岔路的抉擇

**旁白**：李杰跟在眼鏡男和浩二身後，走進了森林深處。濃霧如影隨形，很快就將身後的月台徹底吞沒。

**旁白**：腳下的泥土鬆軟而潮濕，每一步都陷得很深。空氣中瀰漫著一股淡淡的、像是泥土和腐爛葉子的氣味。

**旁白**：三人走了約莫十分鐘，前方出現了一個岔路口。

**旁白**：地圖在這裡再次與現實產生了偏差。地圖上顯示的是一條筆直的小徑，但眼前卻分成了左右兩條路。

**斎藤蓮**：(停下腳步,推了推眼鏡) 嘖,地圖果然不靠譜。

**山本浩二**：(皺眉看著前方) 這霧太大了,根本看不清哪條路比較好走。李兄弟,你覺得呢?

**李杰**：(低頭看著手中的地圖碎片) 地圖上這一段被燒毀了,看不清。

**斎藤蓮**：(冷笑) 所以我們現在是在玩「真心話大冒險」嗎?

**李杰**：(沒有理會他的嘲諷,看向左右兩條路) 剛才大叔他們往哪個方向走了?

**山本浩二**：(搖頭) 沒注意。他們跑得太快了。

**李杰**：(內心獨白) 往左?往右?如果選錯了,可能就再也見不到他們了。

**旁白**：就在三人猶豫不決時,左邊的森林深處傳來一聲隱約的慘叫。

**斎藤蓮**：(臉色慘變) 什麼聲音?!

**山本浩二**：(立刻握緊手中的公事包) 像是有人出事了!

**李杰**：(眼神一凝) 在左邊!走!

**旁白**：李杰率先衝進了左側的小徑,浩二和斎藤雖然害怕,但也只能緊緊跟上。

**旁白**：這條路比剛才的路更加崎嶇不平,樹木交錯,枝葉低垂,不時劃過他們的臉龐。

**旁白**：跑了大約五分鐘,前方的霧氣中出現了一個模糊的人影。

**李杰**：(放慢速度,警惕地靠近) 誰在那裡?

**旁白**：那人影蜷縮在一棵大樹下,身體微微顫抖。

**旁白**：當三人走近時,才發現那是——多話女大生。

**多話女大生**：(抬起頭,眼中滿是驚恐,淚水流了滿臉) 救...救救我...

**斎藤蓮**：(驚訝) 是妳?其他人呢?

**多話女大生**：(抽泣著) 他們...他們被抓走了...

**山本浩二**：(焦急) 被抓走了?被誰抓走了?

**多話女大生**：(拼命搖頭) 我不知道...霧裡...霧裡突然出現了很多黑影...

**李杰**：(蹲下身,注意到她的大腿上有一道長長的傷口,鮮血正不斷滲出) 妳受傷了。

**旁白**：傷口看起來像是被什麼利器劃開的,邊緣整齊。

**李杰**：(內心獨白) 利器...不是野獸。這個世界裡...還有其他人?

**山本浩二**：(迅速脫下外套,幫她包紮傷口) 小姑娘,妳先別說話,我幫妳止血。

**旁白**：就在浩二幫她包紮時,周圍的濃霧中傳來一陣奇異的聲響。

**旁白**：像是樹葉被踩碎的聲音,又像是某種低沈的吟唱。

**李杰**：(立刻站起身,環顧四周) 誰?!

**旁白**：霧氣中,幾道模糊的黑影若隱若現。

**旁白**：而時間，正在一分一秒地流逝。
