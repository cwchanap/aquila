# 第九幕：消失在霧中

**旁白**：黑影在濃霧中快速移動，速度快得不合常理。李杰試圖用手機照明，但光束在厚重的霧氣中被迅速折射、消散，只能照亮前方不到三公尺的範圍。

**斎藤蓮**：(聲音顫抖) 他、他們過來了！

**山本浩二**：(護在受傷女孩身前，大聲吼道) 是誰在那裡！出來！

**旁白**：黑影並沒有回應，而是逐漸縮小了包圍圈。李杰能感覺到一股冰冷的、帶著敵意的氣息正從四面八方襲來。

**李杰**：(內心獨白) 這些黑影...不像是人類。但他們的動作卻很有組織。

**旁白**：突然，一個黑影從霧中猛地竄出，直奔李杰而來！

**李杰**：(本能地側身一躲) 唔！

**旁白**：那黑影從他身邊掠過，帶起一陣陰冷的風。李杰看清了它的樣子——那是一個身形修長、全身覆蓋著灰黑色布料的生物，臉部被一塊白色的面具遮住，只露出一雙散發著淡紫色幽光的眼睛。

**李杰**：(驚呼) 這到底是什麼東西！

**斎藤蓮**：(嚇得癱坐在地) 怪物...真的是怪物...

**旁白**：那生物在地面上輕巧地翻了個身，重新站穩，手中的短刃在微弱的光線下閃著冷光。

**山本浩二**：(舉起沉重的公事包，準備迎戰) 別怕！我們人多！

**旁白**：隨著這一聲怒吼，其他的黑影也紛紛現身。大約有五六個同樣裝束的生物，將四人團團圍住。他們並不急於進攻，而是像獵人觀察獵物一樣，靜靜地盯著他們。

**李杰**：(大聲問道) 你們是誰！為什麼要攻擊我們！

**旁白**：其中一個看起來像是領頭的生物，緩緩走上前。他的面具上刻著更為複雜的符號，雙眼的紫光也更為濃郁。

**面具生物**：(發出一種低沈、沙啞的聲音，聽起來像是兩塊石頭在摩擦) 這裡...不是你們該來的地方。

**李杰**：(震撼) 他說話了？

**山本浩二**：(強裝鎮定) 我們只是迷路了。我們想回家。請放我們走！

**面具生物**：(冷笑一聲) 回家？進了「幽冥之都」的人，從來沒有能回家的。

**斎藤蓮**：(崩潰地喊道) 幽冥之都？你在說什麼！這裡到底是哪裡！

**面具生物**：(沒有回答，而是揮了揮手中的短刃) 帶走。

**旁白**：話音剛落，黑影們同時發起了進攻！

**李杰**：(對浩二喊道) 帶她跑！我擋住他們！

**山本浩二**：(一咬牙，背起受傷的女孩) 好！你小心！

**旁白**：浩二背著女孩衝向包圍圈的一個缺口。李杰則揮舞著隨手撿起的一根粗木棍，拼命抵擋著黑影的攻擊。

**李杰**：(揮動木棍) 滾開！

**旁白**：木棍與短刃碰撞，發出沉悶的響聲。李杰感覺手臂被震得生疼，這些生物的力量大得驚人。

**斎藤蓮**：(連滾帶爬地跟在浩二身後) 等等我！別丟下我！

**旁白**：在混亂中，四人被迫分開。浩二背著女孩跑進了右邊的濃霧中，斎藤緊隨其後。

**李杰**：(內心獨白) 該死...他們跑散了！

**旁白**：李杰想去追他們，但卻被兩個面具生物死死攔住。他只能且戰且退，試圖往回跑向月台的方向。

**李杰**：(喘著氣) 呼...呼...

**旁白**：濃霧越來越濃，李杰漸漸分不清方向。他在森林中狂奔，身後的腳步聲依然緊追不捨。

**旁白**：不知道跑了多久，前方的景物突然變得開闊了一些。李杰衝出一片密林，發現自己來到了一個奇怪的地方。

**旁白**：那是一個巨大的圓形石台，四周矗立著幾根殘破的石柱。石台中央，有一團緩緩燃燒的紫色火焰。

**李杰**：(停下腳步，驚愕地看著眼前的一切) 這是什麼地方...

**旁白**：身後的追兵似乎不見了，森林裡變得安靜得可怕。李杰看著那團紫色的火焰，感覺到一種莫名的吸引力。

**李杰**：(緩緩走向石台) ...

**旁白**：當他踏上石台的那一刻，周圍的景色突然發生了變化。濃霧消失了，取而代之的是一片璀璨的星空，以及腳下那座懸浮在半空中的廢墟城市。

**李杰**：(不敢置信地瞪大眼睛) 這、這是...幽冥之都？

**旁白**：而在他身後，一個冷冰冰的聲音再次響起。

**面具生物**：(不知何時出現在石台邊緣) 歡迎來到真正的世界，人類。

**李杰**：(猛地回頭) 你！

**旁白**：而他，究竟還隱藏著多少秘密？
