# 第六十九幕：重返月台

**旁白**：2026年9月15日，午夜11點40分。新宿車站，中央線月台。

**旁白**：李杰站在月台邊，手插在口袋裡，握著那顆紅色藥丸。月台上的空氣厚重而黏膩，混雜著居酒屋飄散出的酒精味、便利店炸雞的油膩香氣。

**李杰**：（內心，看著周圍）一年了...我又回到這裡...

**旁白**：李杰的視線掃過月台。有一群朝氣蓬勃的大學生，笑聲肆無忌憚，動作誇張。

**健談男大生**：不是啊你們聽我說！剛剛那家店真的超誇張的！什麼精釀啤酒專賣店，結果端上來的味道根本就像洗碗水！

**活潑女大生**：對對對！我也覺得超難喝！而且你們有看到老闆那個表情嗎？

**多話女大生**：對啊對啊！而且啊，你們有注意到嗎？那家店的椅子超不舒服的...

**李杰**：（內心，震驚）和一年前...一模一樣...這些人...這些對話...

**李杰**：（內心，猶豫）她...看起來像清水惠美...但又不太一樣...

**李杰**：（內心）年齡不對...清水惠美應該才二十一歲...這個人看起來都快四十歲了...但是...那個側臉...

**旁白**：李杰站在原地，猶豫了幾秒。列車駛入隧道，車窗外一片漆黑。

**李杰**：（內心，深吸一口氣）...問問看吧...

**旁白**：李杰走到那個女性旁邊。

**李杰**：（禮貌地）不好意思...請問...

**旁白**：女性抬起頭。李杰的呼吸停止了。

**旁白**：那張臉，確實和清水惠美很像，但眼角和額頭都有明顯的細紋，神情更加成熟，已經是中年女性的模樣。

**李杰**：（聲音顫抖）請問...妳是...清水惠美嗎...

**旁白**：女性看著李杰，眼神複雜。她沒有立刻回答，只是靜靜地看著他。

**旁白**：幾秒後，她的嘴角浮現一絲微笑。

**女性**：（輕聲）你終於...找到我了。

**李杰**：（震驚）什麼...

**旁白**：就在這一瞬間——車廂裡的燈光閃爍了一下。

**旁白**：然後，那個女性消失了。

**李杰**：（驚愕）...！

**旁白**：不只是她。車廂裡所有的乘客，全部消失了。

**旁白**：大學生們，還有其他所有乘客——全部不見了。

**李杰**：（環顧四周，聲音顫抖）怎麼回事...

**旁白**：空蕩蕩的車廂裡，只剩下李杰一個人。

**旁白**：座椅還在，燈光還亮著，列車還在行駛。但所有的乘客，都消失了。

**李杰**：（內心，恐懼）這...這是什麼...

**旁白**：就在這時，李杰聽到腳步聲。從車廂另一端傳來。

**旁白**：緩慢的，沉重的腳步聲。

**李杰**：（轉身，警戒）誰...

**旁白**：一個身影從車廂的另一端走來。是一個胖胖的男人，穿著白色的醫生袍。

**旁白**：他的臉上掛著笑容，溫和的，親切的笑容。

**旁白**：李杰的心臟停止了跳動。那張臉...他認識。

**旁白**：雖然胖了很多，雖然穿著醫生袍，雖然神情和記憶中不太一樣——但李杰絕對不會認錯。

**李杰**：（聲音顫抖，幾乎是低吼）...大輔...

**旁白**：那個胖男人停下腳步，微笑著看著李杰。

**胖男人**：（溫和地）好久不見了...李杰。

**旁白**：李杰的拳頭緊緊握住。列車在黑暗中前行，只有他們兩個人。

**旁白**：還有那無數的疑問。
