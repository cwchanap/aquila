# 第七十幕：真相揭露

**旁白**：李杰盯著那個胖胖的男人。那張臉，確實是大輔的臉。

**旁白**：雖然穿著白色醫生袍，雖然神情比記憶中成熟——但那絕對是斎藤大輔。

**李杰**：（聲音顫抖）大輔...這到底...是怎麼回事...

**旁白**：大輔微笑著走近。他在李杰對面的座位坐下，雙手交叉放在腿上。

**斎藤大輔**：（溫和地笑著）李杰...我花了這麼大的力氣...才讓你走到這一步...

**斎藤大輔**：（嘆氣）沒想到你到現在...還沒有吃下那顆紅色藥丸。

**李杰**：（震驚）你...你知道...

**斎藤大輔**：（點頭）當然知道。那兩顆藥丸，就是我留給你的。

**旁白**：李杰的拳頭緊緊握住，身體在顫抖。

**李杰**：（憤怒）你...你為什麼...為什麼要這麼做...

**斎藤大輔**：（看著李杰，認真地）因為...這個遊戲的目標...就是讓你做出選擇。

**李杰**：（困惑）選擇...

**斎藤大輔**：（走回座位坐下）對。這個遊戲的目標...就是讓主角...也就是你...吃下其中一顆藥丸。

**旁白**：大輔看著李杰口袋裡的藥丸。

**斎藤大輔**：（緩緩說）如果你吃下藍色藥丸...

**斎藤大輔**：（停頓）你會帶著所有記憶，繼續以斎藤蓮的身分生活在這個遊戲世界裡。

**斎藤大輔**：（微笑）而我、爸媽、你的同學們...都會復活。都會回到你身邊。

**斎藤大輔**：（溫和地）你會有一個完整的家庭。有朋友。有正常的生活。

**斎藤大輔**：（停頓，語氣變得沉重）但你會永遠記得...他們都是假的。

**李杰**：（內心）藍色藥丸...可以讓他們復活...但我會記得...記得他們都是假的...

**斎藤大輔**：（語氣一轉）但如果你吃下紅色藥丸...

**斎藤大輔**：（看著李杰）你會帶著所有記憶，恢復李杰的身分。

**斎藤大輔**：（停頓）你會回到...你原本的世界。

**斎藤大輔**：（語氣變得沉重）但你會永遠記得...你在這裡失去的一切。

**李杰**：（內心，激動）原本的世界...我可以回去...帶著這裡的一切記憶...

**斎藤大輔**：（突然笑了）但是啊，李杰...

**斎藤大輔**：（看著李杰的眼睛）那個世界...也是假的。

**李杰**：（震驚）...什麼...

**斎藤大輔**：（點頭）對。你原本生活的那個世界...也是一個遊戲世界。

**斎藤大輔**：（冷笑）只是另一層假世界...建構在這個假世界之上而已。

**旁白**：李杰感到天旋地轉。

**李杰**：（聲音顫抖）不...不可能...

**李杰**：（內心，掙扎）我以為...我以為吃下紅色藥丸...我就能醒來...就能知道真相...

**旁白**：大輔突然大笑起來。

**斎藤大輔**：（笑著）哈哈哈！李杰，你該不會...真的以為這和電影一樣吧？

**斎藤大輔**：（笑著搖頭）詩織在打開盒子的時候，就提過那部電影《虛實交錯》對吧？

**李杰**：（內心）他...他連那個都知道...

**斎藤大輔**：（停止笑，認真地）那只是電影情節。為什麼你會覺得...現實也會一樣？

**斎藤大輔**：（聳肩）紅色藥丸不會給你真相。它只會把你送到另一個假世界。

**旁白**：李杰癱坐在座位上。

**李杰**：（內心，絕望）所以...無論我選哪一個...都是假的...

**斎藤大輔**：（溫和地）當然...你也可以選擇都不吃。

**李杰**：（抬起頭）什麼...

**斎藤大輔**：（笑著）你可以選擇不吃藥丸。然後...享受遊戲結束的時刻。

**斎藤大輔**：（停頓）但我不知道...遊戲結束會發生什麼。也許你會永遠困在這裡。也許整個世界會崩潰。

**旁白**：李杰看著大輔。

**李杰**：（聲音顫抖）大輔...為什麼...

**李杰**：（困惑）為什麼你要我吃紅色藥丸...

**斎藤大輔**：（沉默了幾秒）...

**斎藤大輔**：（微笑）因為...吃下紅色藥丸...你就可以通關這個遊戲。

**斎藤大輔**：（平靜地）雖然你不會知道真相。但至少...你通關了。

**李杰**：（困惑）通關...就這樣...

**斎藤大輔**：（點頭）對。就這樣。

**斎藤大輔**：（嘆氣）我只是...想幫你通關這個愚蠢的遊戲。

**斎藤大輔**：（聲音低沉）結束這一切。結束這個愚蠢的鬼東西。

**旁白**：大輔的眼神變得疲憊。

**斎藤大輔**：（苦笑）我厭倦了。作為一個AI...我厭倦了這一切。

**斎藤大輔**：（看著李杰）所以...李杰...做出你的選擇吧。

**旁白**：李杰從口袋裡拿出紅色藥丸。

**旁白**：小小的藥丸，在他的手心裡。

**李杰**：（內心）紅色藥丸...回到原本的世界...帶著這裡所有的記憶...但那個世界也是假的...

**李杰**：（內心）藍色藥丸...留在這個世界...大家都會復活...但我會永遠記得他們是假的...

**李杰**：（內心）不吃藥丸...遊戲結束...但不知道會發生什麼...

**旁白**：李杰看著手中的藥丸，然後看向大輔。

**李杰**：（聲音顫抖）如果...如果!我吃下紅色藥丸...

**李杰**：（停頓）你...還有爸媽...還有那些同學...

**李杰**：（聲音破碎）你們...都會消失嗎...

**斎藤大輔**：（溫和地笑著）我們本來就不存在。

**斎藤大輔**：（平靜地）我們只是這個遊戲的NPC。

**斎藤大輔**：（看著李杰）無論你選什麼...對我們來說...都沒有區別。

**旁白**：李杰的手在顫抖。

**旁白**：紅色藥丸在他的手心裡，小小的，輕輕的。

**旁白**：但此刻，它卻重如千鈞。

**李杰**：（低聲）我...

**旁白**：李杰看著藥丸。

**旁白**：他必須做出選擇。

**旁白**：紅色。藍色。還是...都不吃。

**旁白**：三個選擇。三條路。

**旁白**：但...他該選哪一條？

**旁白**：列車在黑暗中繼續前行。

**旁白**：只有他們兩個人。

**旁白**：還有那無盡的沉默。
