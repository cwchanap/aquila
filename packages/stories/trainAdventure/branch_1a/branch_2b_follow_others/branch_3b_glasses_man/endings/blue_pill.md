# 終幕：藍色藥丸

**旁白**：李杰看著手中的藥丸。紅色的，藍色的。

**旁白**：大輔靜靜地坐在對面，等待著他的選擇。列車在黑暗中前行。

**李杰**：（內心）如果我吃紅色...我會回到那個世界...帶著所有這裡的記憶...

**李杰**：（內心）如果我吃藍色...我會留在這裡...大家會復活...但我會永遠記得他們是假的...

**旁白**：李杰閉上眼睛。腦海中浮現一個個畫面。

**旁白**：颯真溫和的笑容。陸斗誇張的動作。詩織興奮的自拍。彩花和真理的小聲討論。

**旁白**：大輔開朗的笑聲。父母溫暖的眼神。

**旁白**：即使是假的...即使知道他們不是真實存在的...

**旁白**：但那份溫暖，那份幸福...卻是真真切切地感受到了。

**李杰**：（內心，眼眶濕潤）如果這就是救贖...

**旁白**：他抬起頭，看著大輔。

**李杰**：（聲音沙啞）我選...藍色。

**旁白**：大輔的眼中閃過一絲難以察覺的哀傷，但隨即被溫和的微笑取代。

**大輔**：（輕聲說）我明白了。

**大輔**：（把藍色盒子推向李杰）那就吃下去吧。

**旁白**：李杰伸出顫抖的手，拿起那顆湛藍色的藥丸。

**旁白**：他看著窗外。黑暗中，有無數光點在飛舞，像是一場盛大的螢火蟲之舞。

**李杰**：（輕聲對大輔說）謝謝你...大哥。

**大輔**：（微笑著點頭）再見了，蓮。

**旁白**：李杰張開嘴，將藍色藥丸吞了下去。

**旁白**：一瞬間，一股冰涼而柔和的力量從喉嚨擴散到全身。

**旁白**：眼前的景象開始變得模糊、扭曲。大輔的身影、空蕩蕩的車廂、甚至那份沉重的悲傷...都在這股力量下漸漸消融。

**旁白**：像是一塊被投入溫水中的冰，慢慢融化，消失不見。

**旁白**：意識陷入了一片虛無。

---

**旁白**：耳邊傳來陣陣嘈雜聲。

**旁白**：那是人們交談的聲音、笑聲、還有金屬碰撞的聲音。

**旁白**：李杰感覺到陽光刺破眼簾。他緩緩睜開雙眼。

**旁白**：他正坐在一間明亮的餐廳裏。窗外是繁華的東京街頭,行人匆匆。

**旁白**：桌上擺著幾杯已經喝了一半的咖啡,還有散亂的筆記和手機。

**如月颯真**：（推了推眼鏡,笑著看過來）蓮,你發什麼呆呢?剛才說到哪了?

**旁白**：李杰——不,現在他是斎藤蓮。他愣愣地看著眼前的男人。

**斎藤蓮**：（有些茫然地）颯真...

**氷室陸斗**：（誇張地拍了拍桌子）喂喂!蓮!你不會是睡著了吧!我們正討論等一下去哪裡吃晚飯呢!

**七海詩織**：（拿著手機,對著蓮做個鬼臉）別管他啦!他最近肯定又在熬夜寫程式!看那黑眼圈!

**高橋彩花**：（和真理笑成一團）就是說啊!蓮,你再這樣下去,小心交不到女朋友喔!

**旁白**：聽著朋友們熟悉的調侃,看著他們鮮活而生動的笑臉,蓮心中的迷茫漸漸散去。

**旁白**：那種像是從高處墜落後的眩暈感,被此刻真實而溫暖的幸福感取代。

**斎藤蓮**：（嘴角露出一抹放鬆的微笑）抱歉...刚才走神了。

**斎藤蓮**：（合上筆記本）晚餐去吃那家拉麵店吧!我請客!

**眾人**：（歡呼）喔——!萬歲!

**旁白**：一行人吵吵鬧鬧地走出餐廳。

**旁白**：走在熱鬧的街頭,蓮感受著拂面而來的微風,聽著周圍充滿生活氣息的聲音。

**旁白**：他忘記了。

**旁白**：他忘記了那個叫李杰的孤獨靈魂。忘記了那個失去雙親的痛苦夜晚。忘記了在那個扭曲的世界裡所經歷的一切恐懼與掙扎。

**旁白**：他忘記了自己曾經面臨過一個殘酷的抉擇。

**旁白**：他忘記了這一切...都是虛假的。

**旁白**：現在的他,只是斎藤蓮。一個普通的大學三年級學生,有著深愛的家人、可靠的朋友、和充滿希望的未來。

**旁白**：在這個美好的世界裡,他將平靜而幸福地活下去。

**旁白**：永遠。

**旁白**：—— 全劇終 ——
