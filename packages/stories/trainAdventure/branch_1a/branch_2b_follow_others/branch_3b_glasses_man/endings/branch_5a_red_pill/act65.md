# 第六十五幕：離別之前

**旁白**：教室裡安靜下來。

**旁白**：李杰看著手中的兩顆藥丸。紅色的。藍色的。

**旁白**：他的手在顫抖。

**李杰**：（內心）如果我選藍色...我可以繼續留在這裡...

**李杰**：（內心）和爸媽在一起...和朋友們在一起...過著正常的生活...

**李杰**：（內心）但這一切...都是假的...

**旁白**：李杰閉上眼睛。深吸一口氣。

**李杰**：（內心）不...我不能這樣...

**李杰**：（內心）我不能沉溺在虛假的美好裡...

**李杰**：（內心）而且...我還有事情沒做完...

**旁白**：李杰睜開眼睛，眼神變得堅定。

**旁白**：他把兩顆藥丸收回盒子裡。

**李杰**：（內心）大輔...我會找出真相的...

**旁白**：下課鐘聲響起。

**如月颯真**：（伸懶腰）終於下課了！蓮，要不要去吃拉麵？

**氷室陸斗**：（興奮）拉麵！我贊成！我知道有一家店的叉燒超讚的！

**七海詩織**：（笑著）陸斗，你眼裡只有吃的嗎？

**渡邊真理**：（小聲對彩花說）蓮好像有心事...

**高橋彩花**：（點頭，擔心地看著李杰）嗯...

**李杰**：（微笑）拉麵嗎？好啊。

**旁白**：六個人走出校門。夕陽將影子拉得很長。

**旁白**：在拉麵店裡，大家聊得很開心。陸斗一直在吹牛，詩織忙著拍照，颯真在旁邊吐嘈，真理和彩花在聊最近流行的化妝品。

**李杰**：（看著大家，心裡湧起一股暖流）

**李杰**：（內心）如果...如果能一直這樣下去...該有多好...

**旁白**：吃完拉麵後，大家在路口道別。

**如月颯真**：蓮，明天見！

**七海詩織**：明天見！記得帶作業喔！

**李杰**：（揮手）明天見。

**旁白**：李杰看著朋友們離去的背影，心裡有一種說不出的苦澀。

**李杰**：（內心）也許...這是最後幾次和他們在一起了...

**旁白**：回家的路上，李杰走得很慢。

**旁白**：他看著熟悉的街道，路邊的便利店，公園裡的鞦韆。

**旁白**：這一切...對他來說...已經不再只是背景...而是他生活的一部分。

**李杰**：（內心）我真的...捨得離開嗎...

**旁白**：回到家，媽媽已經做好了晚餐。

**斎藤母親**：（笑著）蓮，你回來了！快洗手吃飯。今天有你最喜歡的漢堡排。

**斎藤父親**：（放下報紙）蓮，過來坐。

**旁白**：晚餐桌上，氣氛溫馨。爸爸在聊工作上的趣事，媽媽在叮囑他要注意身體。

**李杰**：（低著頭吃著漢堡排）

**斎藤母親**：（擔心）蓮，怎麼了？漢堡排不好吃嗎？

**李杰**：（抬起頭，微笑）沒有...很好吃...真的很好吃。

**旁白**：吃完飯後，李杰回到房間。

**旁白**：他坐在書桌前，拿出那個盒子。

**旁白**：紅色的。藍色的。

**李杰**：（內心）如果...如果我真的離開了...那這裡的「我」會怎麼樣...

**李杰**：（內心）爸媽...會傷心嗎...

**李杰**：（內心）不...他們只是NPC...他們不會傷心的...

**李杰**：（內心）但...對我來說...他們是真實的...

**旁白**：李杰把頭埋進雙手。

**旁白**：就在這時，手機震動了一下。

**旁白**：是群組訊息。

**氷室陸斗**：【照片】這家拉麵真的超讚！下次還要再去！

**七海詩織**：好啦好啦！知道了啦！

**如月颯真**：蓮，你今天話很少耶，還好嗎？

**李杰**：（看著螢幕，手指在鍵盤上停了很久）

**李杰**：我沒事，只是有點累。

**李杰**：對了...大家。

**李杰**：我們去旅行吧。

**旁白**：五個人都停下動作，驚訝地看著李杰。

**氷室陸斗**：（放下筷子，震驚）蓮...你說什麼？

**七海詩織**：（也很驚訝）旅行...？

**李杰**：（點頭）對...我想...今年冬天...二月左右...我們一起去北陸...

**李杰**：（繼續說）去溫泉酒店...然後滑雪...你們覺得怎麼樣？

**旁白**：食堂裡突然安靜了幾秒。

**如月颯真**：（緩緩地）蓮...你...主動提出要組織旅行？

**高橋彩花**：（輕聲，有些不敢相信）這...這是真的嗎...

**渡邊真理**：（也很驚訝，小聲對彩花說）蓮從來沒有主動提過這種事...

**氷室陸斗**：（突然大笑）哈哈哈！太陽從西邊出來了嗎？蓮你居然會主動約大家出去玩！

**七海詩織**：（笑著打陸斗的手臂）陸斗！別這樣說！

**李杰**：（有些尷尬）我...我以前...很少主動嗎...

**如月颯真**：（苦笑）也不是很少...是幾乎沒有...

**高橋彩花**：（溫和地）不過...這是好事啊...

**渡邊真理**：（點頭附和）對啊...我們都很高興...

**氷室陸斗**：（興奮地）當然要去啊！北陸！溫泉！滑雪！聽起來就很讚！

**七海詩織**：（也很興奮）我也想去！好久沒有大家一起旅行了！

**如月颯真**：（看著李杰，笑著）好啊，我也沒問題。蓮，你來安排吧。

**李杰**：（鬆了一口氣，微笑）好...那我會開始規劃...

**氷室陸斗**：（拍拍李杰的肩膀）蓮，你最近真的變了不少啊。

**李杰**：（愣了一下）變了...？

**七海詩織**：（點頭）對啊...感覺你...比以前更主動了...

**如月颯真**：（看著李杰，有些擔心）是因為...你哥哥的事情嗎？

**旁白**：李杰沉默了一會。

**李杰**：（低聲）也許吧...

**李杰**：（抬起頭，微笑）我只是...覺得...應該要珍惜和大家在一起的時間...

**高橋彩花**：（溫柔地）這樣想很好啊...

**渡邊真理**：（也點頭）對啊...

**氷室陸斗**：（大笑）好啦好啦！不要這麼嚴肅！我們是要去旅行，要開心一點！

**七海詩織**：（笑著）對啊！北陸！溫泉！我好期待！

**如月颯真**：（也笑著）那就這樣決定了。蓮，你規劃好之後告訴我們。

**李杰**：（點頭，真心地笑了）嗯...謝謝大家...

**旁白**：五個同學看著李杰的笑容。雖然他們感覺到李杰最近有些不同，但看到他笑得這麼開心，他們也很高興。

**旁白**：那天晚上。

**旁白**：李杰坐在書桌前，打開筆記型電腦。

**旁白**：他開始搜尋北陸的溫泉酒店和滑雪場。

**李杰**：（看著螢幕）金澤...富山...福井...

**李杰**：（點開一個溫泉酒店的網頁）這家看起來不錯...有露天溫泉...

**旁白**：李杰認真地看著每一家酒店的介紹。價格、設施、交通。他一一記錄下來。

**李杰**：（內心）滑雪場...最好是初學者也能玩的...

**李杰**：（繼續搜尋）白山...這裡不錯...有初級滑道...

**旁白**：李杰一邊做筆記，一邊計劃行程。

**李杰**：（內心）二月...應該是最適合滑雪的季節...

**李杰**：（內心）六個人...需要訂三間房...

**旁白**：李杰看著螢幕上的照片。白雪皚皚的山脈。溫暖的溫泉。笑得燦爛的遊客。

**李杰**：（內心，微笑）我想...和大家一起...創造一些美好的回憶...

**李杰**：（內心）趁著...還在這個世界的時候...

**旁白**：李杰繼續規劃著。他認真地看著每一個細節，想讓這次旅行完美。

**旁白**：窗外，秋夜的涼風吹進房間。東京的夜晚，燈火依然璀璨。

**旁白**：李杰不知道自己還能在這個世界待多久。

**旁白**：但至少現在，他有了一個期待。

**旁白**：一個和朋友們一起創造美好回憶的機會。

**旁白**：在他回到真實世界之前。
