# 第六十六幕：雪國初日

**旁白**：2025年2月14日。情人節。

**旁白**：東京車站。早上七點。

**旁白**：李杰和五個同學在新幹線月台會合。每個人都背著大背包，穿著厚外套。

**如月颯真**：（揮手）蓮！這裡！

**李杰**：（走過去）早安，大家。

**氷室陸斗**：（興奮地）終於要出發了！我等這一天等好久了！

**七海詩織**：（笑著）陸斗從上週就一直在說...

**高橋彩花**：（小聲對真理說）他昨天還在群組裡發了十幾則訊息...一直在問要帶什麼...

**渡邊真理**：（點頭，微笑）對啊...

**李杰**：（看著大家，心裡感到一陣溫暖）大家票都拿到了嗎？

**如月颯真**：拿到了。走吧，車快開了。

**旁白**：新幹線「白鷹號」緩緩駛出月台。窗外的風景從高樓大廈漸漸變成田野，然後是連綿不斷的山脈。

**氷室陸斗**：（看著窗外）哇！開始有雪了！

**七海詩織**：（也湊過去看）真的耶！好美！

**旁白**：大家在車上吃著鐵路便當，聊著天。李杰看著窗外的雪景，心裡有一種平靜的感覺。

**李杰**：（內心）這樣的旅行...如果是真實的...該有多好...

**旁白**：中午時分，列車抵達金澤站。一股冷空氣迎面而來。

**氷室陸斗**：（縮了縮脖子）呼！好冷！

**如月颯真**：廢話，這裡可是北陸。

**旁白**：他們轉乘巴士前往滑雪場。山路兩旁積雪很厚，樹木被白雪覆蓋，形成了一幅美麗的冬日畫卷。

**七海詩織**：（不停地拍照）太美了！簡直像夢一樣！

**旁白**：抵達滑雪場後，大家換上滑雪服，租好裝備。

**氷室陸斗**：（笨拙地踩著滑雪板）這...這怎麼動啊？

**七海詩織**：（笑他）陸斗，你剛才不是很神氣嗎？

**如月颯真**：（熟練地滑了一段）大家跟著我，我們先在平地練習。

**旁白**：李杰也是第一次滑雪。他小心翼翼地維持平衡，感受著雪板與雪地的摩擦。

**李杰**：（內心）專注在當下...忘記那些事情...

**旁白**：一個下午的時間，大家都在雪地上摸爬滾打。陸斗摔得最慘，但也笑得最大聲。詩織和彩花互相扶持，真理則在旁邊安靜地練習。

**李杰**：（看著真理）真理，沒事吧？

**渡邊真理**：（微笑）沒事...就是有點累。蓮，你學得很快耶。

**李杰**：（微笑）還好。

**如月颯真**：（滑過來）好了，今天就先到這裡吧。我們去旅館泡溫泉！

**氷室陸斗**：（歡呼）溫泉！萬歲！

**李杰**：（點頭，微笑）好。

**旁白**：兩個人並肩滑下雪道。風吹在臉上，雪板在雪地上滑行的聲音。

**旁白**：李杰感覺到一種久違的輕鬆。至少在這一刻，他可以忘記那些沉重的事情。

**旁白**：傍晚六點。六個人結束了滑雪，搭巴士前往加賀溫泉的旅館。

**旁白**：旅館是一棟傳統的日式建築。木造的外牆，掛著燈籠，積雪覆蓋著屋頂。

**七海詩織**：（看著旅館）好漂亮！

**旁白**：他們辦理入住，分配房間。男生三人一間，女生三人一間。李杰、颯真和陸斗的房間在二樓，榻榻米地板，矮桌，窗外能看到雪景。

**氷室陸斗**：（躺在榻榻米上）啊...累死了...全身都痛...

**如月颯真**：（笑著）誰叫你一直亂滑，摔了那麼多次。

**氷室陸斗**：（笑嘻嘻）但是很好玩啊！明天我一定要學會！

**李杰**：（看著窗外）我們先去泡溫泉吧。

**如月颯真**：（點頭）對，泡完溫泉應該就不會那麼痛了。

**旁白**：露天溫泉。熱氣從水面升起，在冷空氣中形成白色的霧氣。李杰泡在溫泉裡，仰頭看著天空，雪花偶爾飄落，落在溫泉水面上立刻融化。

**氷室陸斗**：（舒服地嘆氣）啊...這才是人生啊...

**如月颯真**：（也很放鬆）確實...泡完溫泉，全身都放鬆了...

**旁白**：遠處傳來女生們的笑聲。她們在另一邊的女湯。

**氷室陸斗**：（笑著）聽起來女生們也很開心。

**如月颯真**：（笑著）這次旅行真的很棒。謝謝蓮規劃這一切。

**李杰**：（微笑）不客氣...能和大家一起來...我也很開心...

**旁白**：李杰閉上眼睛。溫泉的熱氣，雪花的冰涼，朋友的陪伴。

**旁白**：他什麼都不想，只是享受這一刻的平靜。

**李杰**：（內心，幾乎聽不見的聲音）...真好...

**旁白**：晚上八點，旅館的餐廳。六個人圍坐在一張大桌子旁，桌上擺滿了料理：烤螃蟹、生魚片、天婦羅、茶碗蒸、味噌湯、白飯，還有當地特產的加賀野菜。

**氷室陸斗**：（眼睛發亮）哇！好豐盛！

**七海詩織**：（拿起手機）等等！我要先拍照！

**高橋彩花**：（也拿出手機）對對對，這個一定要記錄下來！

**旁白**：拍完照後，大家開始吃飯。

**氷室陸斗**：（大口吃著螃蟹）太好吃了！這螃蟹超新鮮！

**渡邊真理**：（溫和地）陸斗，慢點吃...

**如月颯真**：（笑著）讓他吃吧，他今天消耗太多體力了。

**七海詩織**：（看著李杰）蓮，你今天滑雪學得很快耶。

**李杰**：（微笑）還好...其實也摔了很多次...
