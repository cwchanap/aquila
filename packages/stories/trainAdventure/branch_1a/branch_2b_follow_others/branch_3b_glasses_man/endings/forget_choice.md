# 終幕：遺忘之選

**旁白**：斎藤蓮坐在餐桌前,看著面前熱氣騰騰的咖哩飯。

**旁白**：那是母親最拿手的家常菜,散發著誘人的香氣。

**旁白**：父親坐在主位上,正看著晚報。哥哥大輔坐在他對面,正一邊滑手機一邊嘿嘿傻笑。

**旁白**：在這個溫馨的家庭氛圍中,剛才在月台上的驚心動魄、在精靈村莊的奇幻冒險、還有在終焉祭壇的慘烈戰鬥...

**旁白**：這一切都顯得那麼不真實。像是一個遙遠而模糊的噩夢,正隨著咖哩的香氣一點點消散。

**斎藤母親**：（從廚房探出頭,溫柔地笑著）蓮,快吃呀,涼了就不好吃了。

**斎藤蓮**：（看著母親那熟悉的臉龐,心中湧起一股莫名的酸楚）嗯...我知道了。

**旁白**：他拿起湯匙,舀起一口咖哩送入嘴中。

**旁白**：熟悉的味道,熟悉的溫度。

**旁白**：這就是真實。這就是他的生活。

**旁白**：什麼精靈,什麼守門人,什麼世界之門...

**旁白**：那不過是他在月台上睡著後做的一個長長的夢罷了。

**斎藤大輔**：（抬起頭,看著蓮的神情,開玩笑道）弟弟,你怎麼吃個咖哩飯也一副要哭的樣子？

**斎藤蓮**：（趕緊擦了擦眼角,擠出一個笑容）哪有...只是覺得今天的咖哩特別好吃。

**斎藤大輔**：（哈哈大笑）那當然！這可是媽媽為了慶祝你平安回家特意做的！

**旁白**：慶祝他平安回家...

**旁白**：蓮的心中微微一動。他想起了在那個夢裡,他也曾無數次渴望著回家。

**旁白**：而現在,他就在家裡。

**旁白**：他有愛他的家人,有關心他的朋友,有平靜而安穩的未來。

**旁白**：既然如此,為什麼還要糾結於那個噩夢呢？

**旁白**：他低下頭,開始大口大口地吃著咖哩飯。

**旁白**：每一口都帶著滿足,每一口都帶著安穩。

**旁白**：隨著胃部的充實,腦海中那些關於異世界的記憶開始變得愈發模糊。

**旁白**：清水恵美的臉龐、高山翔的決絕、凱蘭的犧牲...

**旁白**：這些名字和畫面,正像沙灘上的足跡,被回歸現實的海浪一點點抹平。

**旁白**：他不再是那個在異世界掙扎求生的倖存者。

**旁白**：他是斎藤蓮。一個普通的大學三年級學生。

**旁白**：在這個美好的世界裡,他將忘記一切痛苦,忘記一切真相。

**旁白**：他將平靜而幸福地活下去。

**眾人**：（一起）我開動了！

**旁白**：斎藤蓮吃著咖哩飯。熟悉的味道，溫暖，安心。

**斎藤母親**：（關心地）蓮，最近有沒有好好吃飯？看你好像瘦了...

**斎藤蓮**：（笑著）有啦...只是最近功課比較多...

**斎藤大輔**：（開玩笑）弟弟，你該不會又熬夜打電動吧？

**斎藤蓮**：（無奈）才沒有...

**斎藤父親**：（嚴肅地）蓮，健康最重要...不要太拼命...

**斎藤蓮**：（點點頭）我知道了...

**旁白**：斎藤蓮感受著家人的關愛。

**旁白**：雖然腦海中偶爾會閃過一些奇怪的畫面——漆黑的列車、發光的樹木、還有那個叫李杰的男人的聲音。

**旁白**：但那些都已經不重要了。

**旁白**：因為現在，他在這裡。在現實的世界裡。

**旁白**：他吃完飯,回到了自己的房間。房間裡擺滿了他的東西——電腦、書架、還有他在滑雪場買的紀念品。

**旁白**：他坐在書桌前,打開筆記本電腦。

**旁白**：螢幕亮起,那是他正在寫的程式碼。

**斎藤蓮**：(笑了)我知道。

**旁白**：斎藤蓮出門，走在街道上，早晨的陽光很暖和。

**旁白**：他坐上電車，車廂裡擠滿了上班族和學生。到達大學，走進教室，同學們已經在座位上了。

**如月颯真**：（揮手）蓮！早啊！

**斎藤蓮**：（走過去）早...

**七海詩織**：（笑著）蓮，你昨天回去有好好吃媽媽做的咖哩嗎？

**斎藤蓮**：（笑著）當然！超好吃的！

**氷室陸斗**：（羨慕）真好...我媽都只會煮那幾樣...

**高橋彩花**：（和真理笑著）你就知足吧...

**旁白**：上課鐘聲響起，同學們回到座位。斎藤蓮坐在位子上，拿出課本。

**旁白**：教授走進教室，開始上課。斎藤蓮認真地聽講，做筆記。

**旁白**：一切都很正常，就像昨天，就像前天，就像這三年來的每一天。平靜、安穩、沒有波瀾。

**旁白**：在這個世界裡，斎藤蓮的生活繼續著。

**旁白**：他有愛他的父母，有關心他的哥哥，有陪伴他的朋友。

**旁白**：他每天上大學，認真讀計算機科學的課，和同學們聊天，週末和家人一起吃飯。

**旁白**：他是一個普通的大學三年級學生，有著普通的生活、普通的煩惱、普通的快樂。

**旁白**：沒有失蹤的家人，沒有神秘的列車，沒有鏡像的世界。

**旁白**：沒有痛苦，沒有掙扎，沒有真相。只有平靜。

**旁白**：但在某個地方，在某個沒有人知道的角落，有兩個盒子——紅色和藍色，靜靜地躺在斎藤蓮的背包深處。

**旁白**：盒子是空的，但曾經裝著選擇。紅色的藥丸，藍色的藥丸。真實，遺忘。

**旁白**：而斎藤蓮選擇了遺忘。他忘記了那個叫李杰的人，忘記了那個失去雙親的孤獨靈魂，忘記了那段痛苦的旅程。

**旁白**：他成為了斎藤蓮，完完全全的斎藤蓮。

**旁白**：而李杰，李杰不再存在了。他被埋葬在遺忘的深處。

**旁白**：沒有人記得他，沒有人知道他曾經存在過。甚至連他自己，也忘記了自己。

**旁白**：這是一個選擇，一個沒有對錯的選擇。

**旁白**：只是...有人選擇了痛苦的真實，有人選擇了舒適的虛幻。

**旁白**：而斎藤蓮，他選擇了後者。

**旁白**：他選擇了遺忘，選擇了和平，選擇了...活下去。

**旁白**：時間繼續流逝，世界繼續運轉。

**旁白**：在這個沒有李杰的世界裡，斎藤蓮平靜地生活著。笑著，哭著，成長著。像一個普通的人。

**旁白**：而真相，真相被埋葬了。永遠。

**旁白**：【完】
