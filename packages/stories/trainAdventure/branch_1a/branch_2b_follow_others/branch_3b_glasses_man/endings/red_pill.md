# 終幕：紅色藥丸

**旁白**：李杰看著手中的紅色藥丸，小小的，圓圓的，在他的掌心裡。

**斎藤大輔**：（平靜地看著他）做出選擇了嗎？

**旁白**：李杰閉上眼睛，深吸一口氣。

**李杰**：（內心）對不起...大輔...爸媽...大家...

**李杰**：（內心）我...我想回去...

**旁白**：李杰張開眼睛，看著大輔。

**李杰**：（聲音顫抖）我...我選紅色藥丸。

**斎藤大輔**：（微笑）我知道你會這樣選。

**斎藤大輔**：（把紅色盒子推向李杰）那就吃下去吧。

**旁白**：李杰伸出顫抖的手，拿起那顆鮮紅色的藥丸。

**旁白**：他看著大輔。大輔的臉色平靜，眼神中帶著一絲解脫。

**李杰**：（輕聲）大哥...保重。

**斎藤大輔**：（點頭）再見了，蓮。

**旁白**：李杰張開嘴，將紅色藥丸吞了下去。

**旁白**：一瞬間，一股灼熱的力量從喉嚨擴散到全身。

**旁白**：眼前的景象開始扭曲、旋轉。大輔的身影、空蕩蕩的車廂、甚至那份沈重的悲傷...都在這股力量下漸漸消融。

**旁白**：像是一塊被投入火中的冰，瞬間昇華，消失不見。

**旁白**：意識陷入了一片虛無。

---

**旁白**：不知道過了多久。

**旁白**：李杰感覺到重力重新回到了身上。

**旁白**：他睜開雙眼。

**旁白**：他正坐在一列疾馳的電車上。

**旁白**：窗外是東京熟悉的夜景，五彩斑斕的燈光飛速流逝。

**旁白**：車廂裡坐滿了乘客。有人在玩手機,有人在打瞌睡,有人在低聲交談。

**旁白**：廣播裡傳來溫柔的女聲。

**廣播**：下一站...新宿。下一站...新宿。

**旁白**：李杰愣住了。

**旁白**：他看了看自己的手。雙手乾淨,沒有血跡,沒有傷口。

**旁白**：他摸了摸口袋。手機在那裡。錢包在那裡。

**旁白**：他拿起手機看了一眼。時間:2024年5月18日,晚上11點45分。

**李杰**：（失神地喃喃自語）我...回來了?

**旁白**：就在這時,他的手機震動了一下。

**旁白**：是一條視訊通話邀請。

**旁白**：李杰心跳加速，顫抖著接通了電話。

**李杰**：（對著手機）餵？

**旁白**：螢幕閃爍了一下，出現了一張熟悉的、慈祥的臉。

**李杰母親**：（笑著）小杰！下班了嗎？怎麼這麼晚還沒到家？

**旁白**：看到母親的一瞬間，李杰的眼淚奪眶而出。

**李杰**：（哽咽）媽...是我...我剛下班...

**李杰母親**：（有些擔心）哎呀，這孩子，怎麼哭了？是不是工作太累了？

**李杰**：（一邊擦眼淚一邊笑）沒...沒有...只是...突然很想妳。

**李杰母親**：（溫柔地笑了）這孩子...多大的人了。好了，快回家吧。爸爸今天做了你最愛吃的紅燒肉，正等著你呢。

**李杰**：（重重地點頭）嗯！我馬上就回去！

**旁白**：掛斷電話後，李杰深吸一口氣。

**旁白**：電車緩緩駛入新宿車站。

**旁白**：門開了。李杰走出車廂，踏上月台。

**旁白**：新宿車站的月台依然喧囂，依然擠滿了人。

**旁白**：但李杰覺得，這一切都是那麼親切，那麼真實。

**旁白**：他邁開腳步，朝著出口走去。

**旁白**：就在他快要走出月台時，一群大學生從他身邊走過。

**旁白**：兩男三女，有說有笑。

**旁白**：李杰的腳步停住了，心臟停止了跳動。

**旁白**：那五張臉。他永遠不會忘記。

**旁白**：一個沉穩的男生，戴著黑框眼鏡，對周圍的事物充滿好奇，正在觀察路邊的植物。

**旁白**：一個笑嘻嘻的男生，開朗活潑，正在講著笑話逗大家笑。

**旁白**：一個陽光的女生，笑容滿面，笑聲清脆，眼睛彎成月牙，和每個人都聊得很開心。

**旁白**：兩個形影不離的女生，正在小聲討論著什麼，一邊走一邊觀察周圍的風景。

**笑嘻嘻的男生**：（笑著）所以我就跟教授說，這個bug不是我的錯！是電腦的錯！結果教授也笑了，說那你去修電腦吧！哈哈哈！

**陽光的女生**：（笑著打他）陸斗你真的很敢講欸！教授沒有當場把你踢出去就不錯了！

**沉穩的男生**：（笑著搖頭）陸斗啊，你這樣下去期末會被當的。

**其中一個女生**：（小聲對另一個女生說）詩織和陸斗感情真好，每次都這樣鬥嘴。

**另一個女生**：（也小聲回應）對啊，颯真總是要幫他們調解。我們這個小組真的很熱鬧。

**旁白**：李杰站在原地，看著他們。

**旁白**：他們從李杰身邊走過，沒有停下，沒有回頭，只是一群普通的大學生，路過一個陌生的上班族。

**中野海斗**：（困惑）李君？怎麼了？

**旁白**：李杰回過神。

**李杰**：（搖頭，微笑）...沒事。只是...覺得他們看起來很開心。

**中野海斗**：（笑著）是啊！年輕真好！

**旁白**：李杰轉過身，看著那五個人的背影。他們越走越遠，笑聲漸漸遠去。

**李杰**：（內心，輕聲）颯真...陸斗...詩織...真理...彩花...

**李杰**：（內心）在這個世界...我們是陌生人...

**李杰**：（內心）但我...永遠不會忘記你們...

**旁白**：秋風吹過，紅葉飄落。李杰深吸一口氣，轉身繼續往山頂走。

**中野海斗**：（在前面喊）李君！快點啦！

**李杰**：（微笑）來了！

**旁白**：李杰跟上海斗的腳步。

**旁白**：他們一起往山頂走去。

**旁白**：身後，那五個大學生的身影，漸漸消失在山路的轉角。

**旁白**：李杰繼續生活著。每天上班、下班，週末去秋葉原，或者和朋友出去，有時候給父母打電話，有時候一個人在公寓裡看書、看電影。

**旁白**：日子很平靜，很普通。

**旁白**：但有時候，在某個瞬間——看到便利店裡的哆啦A夢商品，聽到電車裡有人在討論PS5遊戲，路過一家精靈耳飾專賣店，經過某個滑雪場的廣告——李杰會停下腳步。

**旁白**：他會想起那個世界，那些人，那些事，那些...已經不存在的回憶。

**李杰**：（內心）那些記憶...是真的嗎...

**李杰**：（內心）還是...只是一場夢...

**旁白**：但李杰知道，無論是真是假，那些記憶都會永遠留在他心裡，陪伴他繼續走下去。

**旁白**：夜晚，李杰坐在公寓的窗邊，看著窗外的東京夜景。萬家燈火，車水馬龍，這個世界繼續運轉著。

**旁白**：李杰拿起手機，看著螢幕。沒有如月颯真、氷室陸斗、七海詩織、渡邊真理、高橋彩花的聯絡方式，沒有斎藤大輔的聯絡方式，只有他的同事、朋友，還有上海的父母。

**李杰**：（輕聲）大輔...

**李杰**：（微笑）謝謝你...讓我...通關了這個遊戲...

**旁白**：李杰放下手機，站起身走向床邊，躺下，閉上眼睛。

**旁白**：明天，太陽依然會升起，他依然會去上班，依然會和同事聊天、吃飯，依然會在這個世界裡繼續生活下去，帶著那些記憶，帶著那些只屬於他一個人的秘密。

**李杰**：（內心，在入睡前的最後一刻）晚安...大家...

**旁白**：李杰睡著了。窗外，東京的夜晚依然喧囂，但在這個小小的房間裡，一切都很安靜，很平和。

**旁白**：列車繼續在軌道上奔馳，載著無數的乘客前往各自的目的地。有人上車，有人下車，有人在月台上等待，有人在車廂裡打盹。

**旁白**：新宿車站的月台，永遠那麼喧囂。

**旁白**：中央線的末班車，永遠準時發車。

**旁白**：而李杰——只是其中一個普通的乘客，帶著他的記憶，帶著他的故事，繼續在這個世界裡，活下去。

**旁白**：【完】
