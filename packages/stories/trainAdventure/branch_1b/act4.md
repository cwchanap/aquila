# 第四幕：封閉的車廂

**旁白**：敞開的車門外是深不見底的寂靜,彷彿一張等待獵物的巨口。冰冷的空氣持續從外面灌進來,帶著濃郁的鹹味和陌生草木的氣息。

**旁白**：車廂內的乘客們像是被施了定身咒,沒有人敢率先踏出那一步。那兩輪紫色殘月透過車窗投下詭異的光芒,將每個人的臉孔染成一片死灰。

**旁白**：恐懼扼住了每個人的喉嚨,時間在令人窒息的沉默中流逝。

**旁白**：十分鐘過去了。二十分鐘。三十分鐘。車廂內依然是死一般的寂靜,沒有人敢動,沒有人敢說話。只有偶爾傳來的抽泣聲和急促的呼吸聲。

**旁白**：將近一個小時後,李杰終於無法再忍受這種煎熬的等待。

**李杰**：(內心)這樣等下去不是辦法。我們對現在的情況一無所知,留在這裡,只是坐以待斃。至少⋯⋯我得先搞清楚這輛列車本身有什麼異常。

**旁白**：在一片死寂中,李杰緩緩站了起來。這個簡單的動作立刻吸引了所有人的目光,像一顆投入死水的石子。

**自私中年男**：(語氣充滿戒備)喂,你⋯⋯你想幹嘛?你該不會想一個人下去吧?

**李杰**：(看著他,平靜地說)我只是去後面的車廂看看。

**自私中年男**：(冷哼一聲)去後面的車廂?誰知道你是想逃跑還是想幹嘛。我告訴你,現在大家都在一條船上,你別想耍什麼花樣!

**李杰**：(冷淡)隨你怎麼想。

**旁白**：李杰沒有理會他們的冷嘲熱諷,徑自轉身,朝著後方車廂的連接處走去。他的腳步聲在安靜的車廂內格外清晰。

**慷慨大叔**：(沉穩地開口)小兄弟,小心點。有什麼異狀立刻回來。

**李杰**：(點了點頭)我會的。

**旁白**：車廂連接處的自動門發出「嘶」的一聲打開。一股更加冰冷的空氣撲面而來。李杰深吸一口氣,跨過車廂之間的連接處,走進了下一節車廂。

**旁白**：第二節車廂。空蕩蕩的。座位整齊排列,頭頂的燈光發出淡淡的白光。幾張廣告傳單散落在地板上,隨著列車細微的震動輕輕晃動。

**李杰**：(內心)沒有人⋯⋯一個人都沒有。

**旁白**：他繼續往前走。第三節車廂。依然空蕩。只有空調的低沉嗡鳴聲在空氣中迴盪。座椅上有人留下的痕跡——一個被遺忘的塑膠袋、一本翻開的雜誌、一個空的咖啡罐。

**旁白**：彷彿乘客只是暫時離開,隨時會回來。但李杰知道,他們再也不會回來了。

**李杰**：(內心)太奇怪了⋯⋯就算是末班車,也不可能只有我們那十幾個人。乘客也太少了⋯⋯

**旁白**：一股寒意從李杰的背脊竄起。但他沒有停下腳步,決定繼續往列車的前端走去。第四節車廂。第五節車廂。全都一樣——空蕩、寂靜、只有他一個人的腳步聲。

**旁白**：當他走到第五節車廂時,他下意識地朝窗外瞥了一眼。就在那一瞬間,藉著兩輪紫色殘月微弱的光芒,他看到了——

**旁白**：一個模糊的人影,正離開月台。那道身影頭也不回地朝著遠處那片深邃的森林走去。距離太遠,光線太暗,他完全無法辨認出那是誰。

**李杰**：(內心)有人下去了!是誰?是我們車廂裡的人嗎?

**旁白**：李杰心中一緊。無論那個人是誰,他都是唯一離開列車的人——也許知道些什麼,也許知道該往哪裡去。他是唯一的線索。

**李杰**：(喃喃自語)等等!

**旁白**：李杰不再猶豫,立刻轉身衝向最近的車門。他的手伸向門邊的開關按鈕。就在他的指尖即將碰到按鈕的那一瞬間——

**旁白**：「嘶——」

**旁白**：「砰!」

**旁白**：整輛列車所有的車門,在同一時間猛然關閉。沉重的撞擊聲在每一節車廂內迴響,像是某種宣告。李杰的手僵在半空中,隔著玻璃,他只能眼睜睜地看著那道身影越走越遠。

**旁白**：越來越小。

**旁白**：最終消失在森林的黑暗之中。

**李杰**：(用力拍打車門)該死⋯⋯

**旁白**：他用力按下開關按鈕。按了一次。兩次。三次。車門紋絲不動。緊急開關也完全沒有反應。

**李杰**：(內心)鎖死了⋯⋯所有的門都鎖死了。

**旁白**：車廂,再次變成了與世隔絕的牢籠。外面的世界近在咫尺,卻遙不可及。李杰靠在車門上,看著窗外那片吞噬了唯一線索的黑暗森林。

**旁白**：一種深深的無力感湧上心頭。

**李杰**：(緩緩呼出一口氣)⋯⋯回去吧。至少要告訴其他人這個情況。

**旁白**：他轉過身,開始往回走。空蕩蕩的車廂此刻顯得更加陰森。頭頂的燈光不知何時開始微微閃爍,發出細微的電流聲。

**旁白**：李杰深吸一口氣,轉身開始往回走。他必須告訴其他人這個情況——所有的門都鎖死了,有人已經離開列車,而他們被困在這裡。

**旁白**：他快步穿過空蕩的車廂,回到了乘客們聚集的地方。車門依然緊閉,紫色的月光依然透過窗戶灑進來。

**旁白**：而那片黑暗森林,依然在遠處靜靜地等待著。
