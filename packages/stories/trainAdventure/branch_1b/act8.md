# 第八幕：獻祭的列車

**旁白**：李杰握著劍,跟著焦急的姐弟倆衝出車站大廳。他的腳步聲在空蕩的月台上迴響,心中湧起一股不祥的預感。

**旁白**：然而,當他踏出大廳的那一刻,眼前的一幕讓他渾身一震,腳步驟然停下。

**旁白**：月台上,那輛他們搭乘的列車——車門已經緊緊關閉,正緩緩地開動,朝著遠方的黑暗駛去。

**李杰**：(震驚)什麼⋯⋯

**旁白**：車窗內,一個男人正瘋狂地捶打著車門玻璃。他的拳頭一下又一下地砸在玻璃上,發出沉悶的「砰砰」聲。臉上滿是絕望與憤怒,嘴巴張大,似乎在吼叫著什麼。

**旁白**：但隔著玻璃,隔著列車的轟鳴聲,什麼都聽不見。

**李杰**：(內心)那是⋯⋯那家人的父親!

**旁白**：而在月台上,一個女人正從地上掙扎著爬起來。她的膝蓋破了皮,鮮血順著小腿流下。衣服上沾滿了灰塵和血跡。

**溫柔母親**：(絕望地尖叫)老公!老公!

**旁白**：她瘋狂地追向已經啟動的列車,雙手在空中亂揮,試圖抓住那冰冷的金屬車身。但列車的速度越來越快,發出巨大的鐵軌摩擦聲,將她的呼喊聲徹底淹沒。

**天真弟弟**：(大哭著跑向媽媽)爸爸!爸爸!

**懂事姐姐**：(拉著弟弟,也在不停地流淚)爸爸⋯⋯

**旁白**：李杰呆立在月台上,看著那輛銀白色的列車加速。窗外,紫色的殘月冷漠地注視著這一切。

**旁白**：列車逐漸遠去,最終消失在鐵軌盡頭的黑暗森林中,只留下空蕩蕩的月台,和一家三口絕望的哭喊聲。

**李杰**：(心跳加速,內心)它走了⋯⋯它把那個父親帶走了,卻把這對母子丟在了這裡。

**旁白**：那位母親頹然地跪倒在月台邊緣,雙手掩面,身體劇烈地顫抖著。兩個孩子抱著她,哭得撕心裂肺。

**李杰**：(緩緩走上前,聲音沙啞)發生了什麼事?

**溫柔母親**：(抬起頭,滿臉淚痕)那個⋯⋯那個胖男人⋯⋯

**旁白**：她斷斷續續地講述了剛才發生的事。在李杰走進車站大廳後,車廂內的爭吵升級成了肢體衝突。自私中年男為了搶奪靠近車門的位置,竟然在推搡中將這位母親推下了車。

**溫柔母親**：我丈夫看到我摔下去,立刻想下來救我⋯⋯但就在他快到門口的時候,車門突然關上了!

**旁白**：她緊緊抱著孩子,哭聲中充滿了自責和憤恨。

**溫柔母親**：他被關在裡面了⋯⋯他一直在拍門,一直想出來⋯⋯但那輛車就那樣開走了⋯⋯

**李杰**：(內心)自私的中年男還在車上,而這個為了保護家人的父親卻被帶走了。這個世界的惡意⋯⋯還真是毫不掩飾。

**李杰**：(抬頭看向遠處,內心)下一站是「亡者峽谷」。如果到了那一站,按照這個規律⋯⋯又會有一半的人被迫下車。只剩下兩個人。

**旁白**：他的拳頭緊握,指節發白。一個可怕的真相浮現在眼前。

**李杰**：(內心)這輛列車⋯⋯它在每一站都強制減少乘客數量。那些被迫下車的人⋯⋯會怎麼樣?那個大叔,那個女孩,還有這家人的父親⋯⋯

**旁白**：他想起「寂海」站的死寂,想起「幽冥之都」的荒廢。那些地方看起來都不像有人能生存的地方。

**李杰**：(內心)留在車上不知道會被帶到哪裡,下車也不知道會面對什麼⋯⋯這根本就是一個無解的困局。

**旁白**：他想起車廂內那些人的臉——那個自私膽小的中年男,那個煩躁易怒的丈夫,還有那個焦慮不安的妻子。他們現在還剩下誰?誰會在下一站被留下?

**李杰**：(低聲,只有自己聽得見)幸好我下來了⋯⋯

**旁白**：就在這時,那位母親用手撐著地面,慢慢站了起來。膝蓋的傷口還在流血,但她似乎已經感覺不到疼痛。

**溫柔母親**：(深吸一口氣,聲音顫抖但堅定)不能⋯⋯不能就這樣放棄⋯⋯

**懂事姐姐**：(擔心地)媽媽,你的腿⋯⋯

**溫柔母親**：(蹲下來,用力擦乾眼淚)媽媽沒事。聽好了,我們現在要去找爸爸。

**天真弟弟**：(抬起頭,眼睛紅紅的)去哪裡找爸爸?

**溫柔母親**：(指著鐵軌消失的方向)列車往那邊開了。下一站叫什麼⋯⋯亡者峽谷對吧?爸爸一定會在那裡等我們。

**懂事姐姐**：(害怕)可是⋯⋯那裡聽起來好可怕⋯⋯

**溫柔母親**：(抱住兩個孩子,眼神無比堅定)不怕。我們是一家人,一定會在一起的。媽媽保證,我們一定能找到爸爸。

**旁白**：她站起身,牽起兩個孩子的手。儘管膝蓋還在流血,儘管身上滿是塵土,但她的背挺得筆直。

**溫柔母親**：(看向李杰)謝謝你願意聽我們說⋯⋯我們要走了。

**李杰**：(看著這對母子)⋯⋯沿著鐵軌走?

**溫柔母親**：(點頭)嗯。列車往那個方向去了,我們就往那個方向走。不管多遠,不管多危險,我都要找到我的丈夫。

**天真弟弟**：(拉著媽媽的手)我們會找到爸爸的對不對?

**溫柔母親**：(摸摸孩子的頭)對,一定會的。

**旁白**：這個決定擺在了李杰的面前。他摸了摸口袋裡的地圖,又看了看眼前這一家三口。

**李杰**：(內心)跟著他們走?沿著鐵軌去「亡者峽谷」⋯⋯那是列車的下一站。如果那個父親真的還活著,確實會在那裡。

**旁白**：但他也記得站牌上的指示——「亡者峽谷」，光聽名字就不是什麼好地方。而且列車會在那裡停下,會發生什麼事根本無法預料。

**李杰**：(內心)還是⋯⋯獨自一人,按照地圖探索「迷霧之森」或「銀月湖」?那些地方或許危險,但至少不會碰上那輛詭異的列車⋯⋯

**旁白**：一個是與這對母子三人結伴同行,目標明確但可能更危險。另一個是獨自探索未知,沒有同伴但保有更多選擇。

**旁白**：兩輪紫月靜靜地懸在天空,冷漠地注視著月台上的四個人。遠處,黑色的海洋泛著詭異的銀光,幽深的森林像一道永恆的黑牆。

**旁白**：風吹過空蕩的月台,發出低沉的呼嘯聲。鐵軌延伸向遠方的黑暗,不知通往何處。

**溫柔母親**：(再次看向李杰)你⋯⋯要跟我們一起走嗎?

**旁白**：李杰沒有立刻回答。他看著手中的鏽劍,看著口袋裡的地圖,又看著眼前這一家三口。

**旁白**：在這個詭異的世界裡,每一個選擇都可能是生死之別。

**旁白**：李杰必須立刻做出選擇。
