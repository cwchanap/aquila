# 第十一幕：峽谷的亡者

**旁白**：李杰衝上月台,氣喘吁吁地停下腳步。然而,眼前的景象讓他心頭一沉。

**旁白**：這座名為「亡者峽谷」的車站,比「幽冥之都」還要荒涼。這裡沒有任何建築——沒有候車大廳,沒有售票處,沒有警衛室。

**旁白**：只有一個光禿禿的水泥月台,還有一塊孤零零的站牌。月台的邊緣就是峽谷,深不見底的黑暗在下方張開巨口。

**李杰**：(環顧四周)什麼都沒有⋯⋯

**旁白**：紫月的光芒灑在月台上,將一切染成詭異的紫黑色。風從峽谷深處吹上來,帶著冰冷刺骨的寒意。

**旁白**：他走到站牌下。站牌是同樣的黑木材質,上面用尖銳的字體刻著——

**旁白**：上一站:「幽冥之都」

**旁白**：下一站:「終焉祭壇」

**旁白**：李杰站在空曠的月台上,感到了前所未有的孤獨和恐懼。這裡安靜得可怕,只有風聲在峽谷間迴盪。

**李杰**：(大喊)小林先生!你在這裡嗎?

**旁白**：他的聲音在廣闊的峽谷中激起一陣陣回聲,卻沒有人回應。

**李杰**：(內心,焦急)如果列車在這裡停過,他應該會下車。但這裡一眼就能望穿,根本沒有藏身之處。

**旁白**：他走到月台的邊緣。下方是深不見底的黑暗。一陣強風吹過,李杰不由自主地後退了一步。

**李杰**：(內心)難道⋯⋯他失足摔下去了?

**旁白**：這個念頭讓他背脊發涼。如果真的摔下去了,那絕對沒有生還的可能。

**旁白**：他不死心地在月台上搜索起來。月台延伸得很長,大部分懸在空中。他一邊走,一邊仔細觀察地面。

**旁白**：忽然,他在月台遠端的角落裡看到了一些東西。

**旁白**：李杰連忙跑過去。那是幾道深深的劃痕,刻在水泥地面上。劃痕很新,像是被某種重物劇烈摩擦留下的。

**旁白**：在劃痕旁邊,他看到了一隻鞋子。一隻黑色的男式皮鞋。

**李杰**：(拿起鞋子,手微微顫抖)這是⋯⋯小林先生的鞋子。我看他穿過。

**旁白**：這意味著,小林先生確實到過這裡。而且,這裡發生過什麼。

**旁白**：他順著劃痕的方向看去。劃痕一直延伸到月台的邊緣。在那裡,他看到了一段斷裂的護欄。鐵製的護欄扭曲著,斷口參差不齊,像是被一股巨大的力量強行扯斷。

**李杰**：(心跳加速)護欄斷了⋯⋯

**旁白**：他壯著膽子走到斷裂處,探頭向下方望去。

**旁白**：在紫月的光芒下,他隱約看到下方幾公尺處,有一塊凸出的岩石。

**旁白**：岩石上掛著一塊深色的布料。

**李杰**：(驚呼)在那裡!

**旁白**：他連忙從背包裡取出剛才在小屋裡找到的繩索。繩索很結實,一端繫在月台上殘存的鐵柱上,另一端纏在腰間。

**旁白**：李杰小心翼翼地沿著峭壁滑了下去。他的動作很笨拙,幾次險些撞上岩石,但他顧不了那麼多了。

**旁白**：終於,他踩到了那塊凸出的岩石。

**旁白**：就在岩石後方的裂縫裡,他找到了小林先生。

**旁白**：李杰踏上了來時的鐵軌。軌道兩側是深不見底的峽谷,只有窄窄的路基可以行走。碎石遍布,極難行走。

**旁白**：他只能盡力加快腳步,沿著鐵軌往回走。紫月的光芒微弱,鐵軌在黑暗中延伸,像一條通往未知的銀色長蛇。

**旁白**：十分鐘。二十分鐘。三十分鐘。他的腿已經幾乎失去知覺,但他不能停下。

**旁白**：將近一個小時後,前方的鐵軌上,他看到了一個倒下的身影。

**李杰**：(心中一緊)那是⋯⋯

**旁白**：他連忙跑上前去。那人穿著熟悉的衣服,是小林先生。

**李杰**：(跪下,翻過身體)小林先生!

**旁白**：但當他看清小林先生的狀態時,心臟彷彿被一隻冰冷的手緊緊攥住。

**旁白**：小林先生已經死了。他的眼睛閉著,臉色蒼白,嘴角還有一絲凝固的血跡。

**旁白**：但真正讓李杰震驚的,是他胸口的傷口。

**旁白**：那是一個猙獰的傷口,正中心臟位置。傷口的邊緣平整而乾淨,像是被某種極其鋒利的武器一擊貫穿。

**旁白**：不是拳頭的鈍擊,不是石塊的砸傷,而是——

**李杰**：(顫抖著伸手觸碰傷口邊緣)這是⋯⋯劍傷?

**旁白**：一把長劍,從正面刺入,貫穿心臟,乾淨俐落。一擊致命。

**李杰**：(呆立在原地)怎麼可能⋯⋯

**旁白**：這不是打鬥造成的傷,不是爭執中的失手,而是蓄意的、精準的殺戮。

**李杰**：(內心,震驚)這是⋯⋯被謀殺的。而且兇手用的是劍⋯⋯

**旁白**：他強迫自己冷靜下來,重新思考整個情況。

**李杰**：(內心)如果小林先生是被劍殺死的⋯⋯兇手是誰?車上那個膽小的男人和中年女人?

**旁白**：他搖了搖頭。

**李杰**：(內心)不可能。列車早已開走,他們不可能在半路下車殺了人,再回到行駛的列車上。列車的門是自動鎖死的,不會中途打開。

**旁白**：那麼,只剩下一個可能——

**李杰**：(低聲,難以置信)有第三方⋯⋯

**旁白**：一個他們完全不知道的存在,在這個世界裡活動著。一個擁有武器、充滿惡意的存在。

**旁白**：他看了看自己手中的鏽劍。同樣都是劍,但能夠如此精準地一擊貫穿心臟⋯⋯那個人絕對是個高手。

**旁白**：一個更可怕的念頭突然湧上心頭。

**李杰**：(內心)如果連小林先生這樣一個成年男性都能被輕易殺害⋯⋯那留在小屋裡的小林智子和孩子們⋯⋯

**旁白**：那個神秘的第三方,很可能就在森林裡某處活動。而小林智子和兩個孩子,毫無防備地待在那間小屋裡。

**李杰**：(猛然站起,驚恐)他們有危險!

**旁白**：一股冰冷的恐懼湧上心頭。不管那個人是誰,不管他的目的是什麼,只要他在這片森林裡,就是威脅。

**李杰**：(大喊)該死!

**旁白**：他不再有絲毫猶豫,轉身朝著來時的方向狂奔。腿部的疼痛,肺部的灼燒,全都被拋在腦後。

**旁白**：只有一個念頭在腦中迴響——必須趕回去!必須在出事之前趕到!
