# 第十三幕：交錯的足跡

**旁白**：李杰走出小屋,站在門外的空地上。紫月的光芒從樹葉間隙灑落,在地面上投下斑駁的光影。

**旁白**：他剛踏出門口,目光就被地面上的痕跡吸引。

**李杰**：(蹲下,仔細觀察)腳印⋯⋯

**旁白**：門外的泥土地上,清晰地留著幾行腳印。他認出其中四行是他們來時留下的——自己的、小林智子的,還有兩個孩子的。

**旁白**：但還有新的足跡。

**旁白**：一行較淺的女性腳印,從屋子的方向延伸出來。尺寸和小林智子的腳印相似。

**旁白**：但這些腳印是單獨的。沒有孩子們的小腳印跟隨。

**李杰**：(內心)小林女士一個人出來了⋯⋯

**旁白**：他順著這行腳印往前追蹤。走了大約十步遠,他發現了第二行腳印。

**旁白**：那是一行更深、更大的腳印。從側面的樹叢中走出來,與小林智子的腳印匯合,然後兩行腳印並排,一同向遠方延伸。

**李杰**：(皺眉)這個人⋯⋯

**旁白**：他走到那行大腳印出現的地方——一片茂密的灌木叢。蹲下身,檢查地面。

**旁白**：灌木叢旁邊有壓痕,像是有人長時間站在那裡。幾根樹枝被折斷,葉子上有手指按壓的痕跡。

**李杰**：(內心)這個人沒有進屋。他是在外面等著⋯⋯等小林女士出來。

**旁白**：這個發現讓他感到無比困惑。小林智子為什麼要拋下孩子,去跟一個在屋外等她的人離開?

**李杰**：(站起身,看著兩行並行的腳印)她認識這個人?還是被脅迫的?

**旁白**：但如果是被脅迫,為什麼腳印如此平穩?沒有拖行的痕跡,沒有掙扎的跡象。兩行腳印步伐均勻,像是兩個人正常行走。

**李杰**：(內心)她是自願跟這個人走的⋯⋯但為什麼?

**旁白**：他開始思考那個人的身份。

**李杰**：(內心)腳印比小林女士的大一些,應該是男性⋯⋯會是在「寂海」站下車的那些人之一嗎?

**旁白**：腦海中浮現出那些人的臉孔。

**李杰**：(內心)那個沉穩的大叔?他看起來可靠,如果是他,小林女士可能會信任他。

**李杰**：(內心)還是那個眼鏡男?他故作聰明,但本質上很膽小⋯⋯不像是會單獨在森林裡行動的人。

**李杰**：(內心)或者是那幾個男大學生中的一個?他們活力充沛,有可能⋯⋯

**旁白**：他的思緒又回到了更早的時候。

**李杰**：(內心)還有⋯⋯在「寂海」站,我看到的那個走向森林的模糊人影。那個人是誰?

**旁白**：還有一個更不安的可能性。

**李杰**：(內心)從一開始就在列車上消失的黑衣男⋯⋯他去了哪裡?

**旁白**：最後,還有一個可能——

**李杰**：(低聲)完全可能是我們根本不認識的人⋯⋯

**旁白**：無數的可能性和謎團在他的腦中交織,讓他感到一陣頭痛。

**李杰**：(搖頭)光想沒有用。必須跟上去。

**旁白**：他決定不再原地空想。最直接的辦法,就是跟著這兩行詭異的腳印走下去。

**旁白**：李杰握緊手中的劍,順著足跡前進。紫月在樹冠間移動,光影在地面上緩緩流轉。

**旁白**：十分鐘。二十分鐘。三十分鐘。他一步一步地追蹤著那兩行腳印。腳印帶著他深入森林,遠離小屋,遠離小徑。

**旁白**：周圍的樹木越來越茂密,光線越來越暗。空氣中飄散著潮濕泥土和腐朽樹葉的氣味。

**旁白**：忽然,前方的腳印開始變得凌亂。

**李杰**：(停下,蹲下觀察)這裡⋯⋯發生了什麼?

**旁白**：原本平穩並行的兩行腳印,突然變得雜亂無章。有深深的腳跟痕,像是有人急停。有拖行的痕跡,像是有人掙扎。

**旁白**：地面上的落葉被翻起,露出底下黑色的泥土。

**旁白**：然後,腳印在一片草叢旁徹底消失了。

**李杰**：(站起身,警惕地環顧四周)⋯⋯

**旁白**：他握緊劍,慢慢走向那片草叢。心跳在耳邊轟鳴。

**旁白**：就在草叢旁邊,他看到了一個人影,倒在地上,一動不動。

**李杰**：(心中一緊)小林女士!

**旁白**：他衝了過去,跪在那人身邊。伸手想要扶起她——

**旁白**：但當他看清那人的臉時,動作驟然停住。

**旁白**：那不是小林智子。

**旁白**：那是一個年輕的女孩,看起來二十歲左右。她穿著時髦的短裙和夾克,染成淺棕色的頭髮散亂地鋪在地上。

**旁白**：她的身下有一大灘暗紅色的血跡。

**李杰**：(震驚)怎麼會⋯⋯

**旁白**：藉著紫月的光芒,他認出了這張臉。

**旁白**：倒在地上的,是本應在「寂海」車站下車的那群大學生中的一個女孩。那個活潑女大生——在月台上和朋友們嘻嘻哈哈,抱怨精釀啤酒難喝的女孩。

**李杰**：(低聲,難以置信)是她⋯⋯

**旁白**：她的眼睛半睜著,眼神空洞而呆滯。嘴巴微微張開,像是想要說些什麼,卻永遠說不出來了。

**旁白**：她的胸口有一個可怕的傷口,鮮血早已凝固。

**李杰**：(伸手,探向她的鼻息)⋯⋯

**旁白**：沒有呼吸。身體已經冰冷。

**李杰**：(緩緩收回手)死了⋯⋯

**旁白**：他跪坐在地上,看著這具年輕的屍體。腦中一片混亂。

**李杰**：(內心)為什麼⋯⋯為什麼她會在這裡?她不是在「寂海」站下車的嗎?為什麼會出現在這個地方?

**旁白**：更重要的是——

**李杰**：(內心)小林女士呢?那兩行腳印明明通向這裡⋯⋯但這裡只有這個女孩的屍體。

**旁白**：他站起身,環顧四周。森林一片漆黑,只有風吹過樹葉的聲音。

**旁白**：沒有其他人。沒有小林智子的蹤影。

**李杰**：(低聲)到底⋯⋯發生了什麼?
