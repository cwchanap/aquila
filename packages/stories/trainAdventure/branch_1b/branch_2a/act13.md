# 第十三幕：交錯的足跡

**旁白**：李杰走出小屋,站在門外的空地上。紫月的光芒從樹葉間隙灑落,在地面上投下斑駁的光影。

**旁白**：他剛踏出門口,目光就被地面上的痕跡吸引。

**李杰**：(蹲下,仔細觀察)腳印⋯⋯

**旁白**：門外的泥土地上,清晰地留著幾行腳印。他認出其中四行是他們來時留下的——自己的、小林智子的,還有兩個孩子的。

**旁白**：但還有新的足跡。

**旁白**：一行較淺的女性腳印,從屋子的方向延伸出來。尺寸和小林智子的腳印相似。

**旁白**：但這些腳印是單獨的。沒有孩子們的小腳印跟隨。

**李杰**：(內心)小林女士一個人出來了⋯⋯

**旁白**：他順著這行腳印看去。腳印通往森林深處,卻不是沿著原本的小徑。

**旁白**：更讓他驚訝的是,在女性腳印旁邊,還有一行腳印。

**旁白**：那是一行沉重的、大尺寸的男性腳印。

**李杰**：(皺起眉頭,內心焦慮)那是誰的?小林先生?不⋯⋯小林先生已經在那邊⋯⋯

**旁白**：這兩行腳印並排前進,步幅平穩,甚至可以說是悠閒。完全不像是強行擄走或匆忙逃命。

**李杰**：(內心,困惑)如果是被強行帶走的,應該會有掙扎的痕跡。但這些腳印⋯⋯就像是兩個人在散步一樣。

**旁白**：他看向不遠處的灌木叢。灌木叢的枝條被折斷了一些,顯示有人不久前剛從這裡經過。

**李杰**：(握緊劍柄,內心警惕)難道⋯⋯她認識那個人?

**旁白**：這個念頭讓他感到不寒而慄。如果小林智子是自願跟著那個人離開的,那兩個孩子去哪了?

**旁白**：他再次看向那兩行腳印。在靠近森林邊緣的地方,腳印變得模糊不清,似乎被刻意掩蓋過。

**李杰**：(內心)不管怎樣,我必須追上去。

**旁白**：李杰握緊手中的劍,順著足跡前進。紫月在樹冠間移動,光影在地面上緩緩流轉。

**旁白**：十分鐘。二十分鐘。三十分鐘。他一步一步地追蹤著那兩行腳印。腳印帶著他深入森林,遠離小屋,遠離小徑。

**旁白**：周圍的樹木越來越茂密,光線越來越暗。空氣中飄散著潮濕泥土和腐朽樹葉的氣味。

**旁白**：忽然,前方的腳印開始變得凌亂。

**李杰**：(停下,蹲下觀察)這裡⋯⋯發生了什麼?

**旁白**：原本平穩並行的兩行腳印,突然變得雜亂無章。有深深的腳跟痕,像是有人急停。有拖行的痕跡,像是有人掙扎。

**旁白**：地面上的落葉被翻起,露出底下黑色的泥土。

**旁白**：然後,腳印在一片草叢旁徹底消失了。

**李杰**：(站起身,警惕地環顧四周)⋯⋯

**旁白**：他握緊劍,慢慢走向那片草叢。心跳在耳邊轟鳴。

**旁白**：就在草叢旁邊,他看到了一個人影,倒在地上,一動不動。

**李杰**：(心中一緊)小林女士!

**旁白**：他衝了過去,跪在那人身邊。伸手想要扶起她——

**旁白**：但當他看清那人的臉時,動作驟然停住。

**旁白**：那不是小林智子。

**旁白**：那是一個年輕的女孩,看起來二十歲左右。她穿著時髦的短裙和夾克,染成淺棕色的頭髮散亂地鋪在地上。

**旁白**：她的身下有一大灘暗紅色的血跡。

**李杰**：(震驚)怎麼會⋯⋯

**旁白**：藉著紫月的光芒,他認出了這張臉。

**旁白**：倒在地上的,是本應在「寂海」車站下車的那群大學生中的一個女孩。那個活潑女大生——在月台上和朋友們嘻嘻哈哈,抱怨精釀啤酒難喝的女孩。

**李杰**：(低聲,難以置信)是她⋯⋯

**旁白**：她的眼睛半睜著,眼神空洞而呆滯。嘴巴微微張開,像是想要說些什麼,卻永遠說不出來了。

**旁白**：她的胸口有一個可怕的傷口,鮮血早已凝固。

**李杰**：(伸手,探向她的鼻息)⋯⋯

**旁白**：沒有呼吸。身體已經冰冷。

**李杰**：(緩緩收回手)死了⋯⋯

**旁白**：他跪坐在地上,看著這具年輕的屍體。腦中一片混亂。

**李杰**：(內心)為什麼⋯⋯為什麼她會在這裡?她不是在「寂海」站下車的嗎?為什麼會出現在這個地方?

**旁白**：更重要的是——

**李杰**：(內心)小林女士呢?那兩行腳印明明通向這裡⋯⋯但這裡只有這個女孩的屍體。

**旁白**：他站起身,環顧四周。森林一片漆黑,只有風吹過樹葉的聲音。

**旁白**：沒有其他人。沒有小林智子的蹤影。

**李杰**：(低聲)到底⋯⋯發生了什麼?
