# 第十四幕：十字路口的抉擇

**旁白**：李杰跪在屍體旁,腦中一片混亂。無數問題如潮水般湧來。

**李杰**：(內心)為什麼⋯⋯為什麼這個女孩會在這裡?

**旁白**：她明明是在「寂海」站下車的。和那個大叔、那個眼鏡男、還有其他三個大學生一起。那已經是好幾個小時前的事了。

**李杰**：(內心)從「寂海」到這裡⋯⋯如果沿著這條小徑走,確實能到達。但為什麼她會來這裡?她又是和誰一起的?

**旁白**：更讓他困惑的是——小林智子和孩子們到底去了哪裡?如果離開小屋的不是小林智子,那她們是怎麼消失的?

**旁白**：他站起身,環顧四周。森林一片漆黑,只有紫月投下微弱的光芒。

**李杰**：(深吸一口氣)冷靜⋯⋯必須仔細調查。

**旁白**：他強迫自己平靜下來,開始仔細勘查這片林間空地。

**旁白**：他在女孩的屍體旁邊蹲下來。女孩的手邊掉落著一個小包,是她在火車上背著的那個名牌包。包是開著的,裡面的化妝品、手機、紙巾撒了一地。

**旁白**：李杰撿起她的手機。螢幕是碎的,按了幾下也沒有反應。

**李杰**：(放下手機)沒用。

**旁白**：他再次看向她的傷口。

**旁白**：和鐵軌上的小林先生一模一樣。乾淨俐落,一擊貫穿心臟。

**李杰**：(內心,背脊發涼)是同一個兇手。那個用劍的高手。

**旁白**：他仔細查看地面上的足跡。在草叢周圍,腳印非常混亂。除了女孩的腳印,還有一行巨大的男性腳印——就是他一路追蹤過來的那行。

**旁白**：男性腳印在女孩倒下的地方停留了一會兒,然後轉身,朝著森林另一個方向走去。

**李杰**：(內心,疑惑)他殺了人,然後離開了。但為什麼要把她帶到這裡來殺?在小屋裡不行嗎?

**旁白**：不對。

**李杰**：(內心)他不是從小屋把她帶出來的。

**旁白**：他猛然意識到自己的錯誤。他一直假設從小屋出來的那行腳印是小林智子的,但如果不是呢?

**旁白**：他連忙低頭,仔細查看女孩腳上的鞋子。

**旁白**：是一雙時髦的運動鞋,白色的,沾滿了泥土和血跡。他估算了一下尺寸。

**李杰**：(內心)這個尺寸⋯⋯

**旁白**：他回想起那兩行腳印——一行較淺的女性腳印,一行較深的男性腳印。

**旁白**：李杰的心臟狠狠一沉。

**李杰**：(內心,震驚)這個尺寸⋯⋯和我追蹤過來的那行女性腳印一樣!

**旁白**：他猛然意識到一個可怕的事實。

**李杰**：(內心)我一直以為那行女性腳印是小林智子的⋯⋯但實際上,那是這個女孩的腳印!

**旁白**：這意味著,這個女孩曾經到過小屋。是她和那個神秘男人一起,從小屋離開,來到了這裡。

**李杰**：(內心)那麼⋯⋯小林女士呢?她和孩子們⋯⋯到底去了哪裡?他們是在這個女孩到達之前就消失了?還是⋯⋯

**旁白**：他試圖尋找更多線索,但離開了這條主要小徑,森林裡到處都是茂密的灌木和樹木,根本無跡可尋。

**李杰**：(低聲,沮喪)這樣下去找不到她們⋯⋯

**旁白**：他站在原地,看著女孩的屍體。這個年輕的生命,就這樣結束在這片陌生的森林裡。

**旁白**：李杰深深地嘆了口氣。他將劍插在地上,開始用雙手挖掘旁邊鬆軟的土地。

**旁白**：泥土在指甲下積聚,手掌被粗糙的樹根劃破。但他不停下。一捧又一捧,慢慢挖出一個淺坑。

**旁白**：十分鐘後,坑挖好了。不深,但足夠容納一個人。

**旁白**：李杰小心翼翼地將女孩抱起來。她的身體已經冰冷僵硬。他輕輕地將她放進坑裡,盡可能讓她看起來舒適一些。

**旁白**：然後他開始填土。一捧又一捧,蓋住那張年輕的臉,蓋住那雙空洞的眼睛,蓋住那個可怕的傷口。

**旁白**：最後,他在上面堆起一個小小的土堆,用幾塊石頭圍住邊緣。

**李杰**：(低聲)至少⋯⋯讓你安息吧。

**旁白**：他站在那個簡陋的墳墓前,沉默了片刻。忽然,他想起了被他留在鐵軌上的小林先生。

**李杰**：(內心)我當時一心只想趕回小屋⋯⋯完全忽略了應該為他做點什麼。

**旁白**：一絲愧疚湧上心頭。但他知道,現在不是回頭的時候。小林智子和孩子們還不知道在哪裡,他必須繼續找下去。

**李杰**：(對著墳墓)對不起⋯⋯

**旁白**：他拍掉手上的泥土,拔起插在地上的劍,讓自己徹底冷靜下來。

**旁白**：李杰站在小徑上,看著前方和後方延伸的道路。他必須做出選擇。

**旁白**：往南,是他來時的方向——回到小屋,回到「幽冥之都」。小林智子和孩子們很可能還在那個方向,他必須找到他們。

**旁白**：往北,是「亡者峽谷」的方向。這個女孩和那個神秘男人從這個方向來,也許在那裡能找到更多線索。

**李杰**：(內心)回去?還是繼續前進?

**旁白**：一個是回到小屋,搜索小林智子和孩子們的下落。另一個是前往「亡者峽谷」,追查這個女孩和神秘男人的來源。

**旁白**：兩個選擇都充滿風險,都可能是錯的。但他必須選擇一個。

**旁白**：紫月在天空中緩緩移動,投下長長的影子。森林在黑暗中等待著他的決定。

**李杰**：(握緊手中的劍)⋯⋯

**旁白**：時間不多了。他必須做出選擇。
