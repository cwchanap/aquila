# 第十六幕：被遺留的冊子

**旁白**:李杰呆立在衣櫃前,雙手顫抖著垂在身側。手中的劍尖觸碰地板,發出輕微的金屬聲響。

**旁白**:衣櫃裡飄出一股混雜著塵埃、木頭腐朽和血腥的氣味。他強迫自己不要移開視線,不要逃避眼前的現實。

**李杰**:(深呼吸,閉上眼睛)冷靜...必須冷靜...

**旁白**:多年的理性訓練在這一刻發揮了作用。震驚過後,他強迫自己的思維重新運轉起來。

**旁白**:李杰睜開眼睛,將劍靠在牆邊。他蹲下身,伸出顫抖的手,輕輕地為小林智子闔上了半睜的雙眼。

**李杰**:(低聲)對不起...我來晚了。

**旁白**:他小心翼翼地將她從狹小的衣櫃中移出。身體已經僵硬,手腳彎曲著,很難擺正。他盡可能溫柔地將她平放在地板上。

**旁白**:紫月的光芒從窗外照進來,落在小林智子蒼白的臉上。那張溫柔的臉,現在凝固著死前的驚恐和絕望。

**旁白**:李杰開始仔細檢查屍體。他的手指掃過她的衣服、手臂和身體,尋找任何傷口或線索。

**李杰**:(皺眉)沒有刺傷...沒有割傷...

**旁白**:這與之前兩名死者完全不同。小林先生的胸口被劍貫穿,那個活潑女大生的胸口也有致命傷,但小林智子沒有這樣的外傷。

**旁白**:他的目光落在她白皙的脖子上。在那裡,有一圈清晰而深刻的暗紫色瘀痕,像一條殘酷的項鍊環繞著她的頸部。

**李杰**:(低聲)窒息...被掐死的。

**旁白**:瘀痕很深,顯示出兇手用了極大的力量。而且從傷痕的分布來看,小林智子幾乎沒有掙扎的機會,就被迅速制伏了。

**李杰**:(內心)要在短時間內掐死一個成年女性...需要壓倒性的力量。而且必須動作迅速,不給對方任何逃脫或呼救的機會。

**旁白**:他站起身,開始在腦中過濾所有已知的男性倖存者名單。

**李杰**:(內心)那個眼鏡男...他看起來很瘦弱,而且膽小。應該沒有這種力量。

**旁白**:他回想起在「寂海」站下車的那群人。

**李杰**:(內心)那兩個男大學生...其中一個看起來比較瘦弱,不像是能輕易制伏成年女性的體格。

**旁白**:他的思維繼續推進。

**李杰**:(內心)但那個沉穩的大叔...他體格健壯,而且看起來很有力量。還有那個看起來像運動社團的男大學生,他也有足夠的體力。

**旁白**:李杰搖了搖頭。

**李杰**:(內心)但這些都只是猜測。更重要的是...還有那些我們完全不了解的「第三方」。殺死小林先生的那個用劍的人,很可能就在附近。

**旁白**:他再次環顧屋內,希望能找到更多線索。但除了地板上的混亂腳印和後門的痕跡,沒有其他發現。

**李杰**:(站起身,深吸一口氣)至少...讓她安息吧。

**旁白**:他彎下腰,將小林智子的屍體抱起。身體已經變得冰冷僵硬,比他想像的要沉重許多。

**旁白**:李杰一步一步地走出小屋。夜風吹過,帶來森林的寒意。他在屋旁不遠處找到了一棵大樹,樹下的土地相對鬆軟。

**旁白**:他將屍體輕輕放在一旁,拔出劍,開始挖掘。劍尖刺入泥土,手掌用力挖開。

**旁白**:一捧又一捧,泥土在他手中堆積。指甲裡塞滿了泥,手掌被劃出了幾道傷口,但他沒有停下。

**旁白**:二十分鐘後,一個淺坑挖好了。李杰將小林智子放進去,盡可能讓她看起來安詳一些。

**李杰**:(低聲)你的孩子們...我一定會找到他們。我向你保證。

**旁白**:他開始填土。泥土一點點覆蓋那張蒼白的臉,覆蓋那圈可怕的瘀痕,覆蓋一切。

**旁白**:最後,他在土堆旁邊放了幾塊石頭作為標記。這是他今晚親手埋葬的第二個人了。

**旁白**:做完這一切,他感到身心俱疲。雙腿幾乎無法支撐身體,他靠在旁邊的一棵樹上,大口地喘息著。

**旁白**:汗水混合著泥土,沿著臉頰滑落。雙手沾滿了泥和血,在紫月的光芒下顯得格外刺目。

**李杰**:(喘息)必須...休息一下...

**旁白**:就在這時,他的目光無意中掃過了旁邊的樹叢。

**旁白**:在那片樹叢的邊緣——也就是當初那行大腳印出現的地方——一抹不自然的粉色在暗沉的背景中格外顯眼。

**李杰**:(皺眉)那是什麼?

**旁白**:他強打起精神,走了過去。撥開低矮的灌木,他看清了那是什麼。

**旁白**:一支智慧型手機和一本小巧的冊子,被遺落在微濕的草地上。手機的外殼是淺粉色的,冊子的封面也是粉色,上面畫著一些可愛的卡通圖案。

**李杰**:(撿起手機)這個...我好像在哪裡見過。

**旁白**:他按下電源鍵。黑色的螢幕沒有任何反應。電池早已耗盡。

**李杰**:(放下手機,拿起冊子)這是...日記?還是素描本?

**旁白**:冊子看起來是屬於某個年輕女孩的。封面很可愛,有點磨損,顯然被使用過一段時間。

**旁白**:李杰用還沾著泥土的手指,翻開了那本冊子。

**旁白**:才看了第一頁,他的表情就瞬間凝固了。

**旁白**:呼吸驟然停止,眼中充滿了難以置信的驚駭。
