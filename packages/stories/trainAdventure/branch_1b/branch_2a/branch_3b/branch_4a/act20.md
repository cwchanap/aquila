# 第二十幕：循環的鐵軌 

**旁白**：短暫的休息過後,李杰感覺體力恢復了不少。

**旁白**：溫暖的茶水和麵包為他補充了能量,疲憊的肌肉也在放鬆中得到緩解。雖然心中還有許多疑問,但至少身體已經能夠繼續行動了。

**伊蘭尼爾**：準備好了嗎?我帶你去車站。

**李杰**：(站起身)好,麻煩你了。

**旁白**：在伊蘭尼爾的帶領下,他穿過寧靜的村莊。

**旁白**：精靈們依然好奇地看著這位人類訪客,但已經習慣了他的存在。有些精靈友善地點頭致意,李杰也禮貌地回應。

**旁白**：他們離開村莊,重新走進森林。陽光透過樹葉灑落,在地面上投下斑駁的光影。鳥鳴聲依然清脆,空氣中瀰漫著花草的香氣。

**旁白**：很快,那條熟悉的鐵軌出現在視野中。

**旁白**：順著鐵軌往前走了大約十分鐘,一座相對樸素的車站出現在眼前。

**旁白**：與「終焉祭壇」那種宏偉而詭異的建築不同,這座車站非常簡單。只有一個月台,一個小小的候車亭,還有一塊木製的站牌。

**旁白**：四周完全被這片廣闊的永恆之森所環抱。陽光明媚,鳥語花香,如果不是那輛停靠在月台上的列車,這裡看起來就像是一個普通的森林休憩站。

**李杰**：(走上月台)...

**旁白**：他首先走向那塊木製的站牌。站牌上刻著簡單的文字,在陽光下清晰可見。

**旁白**：本站：永恆之森

**旁白**：上一站：終焉祭壇

**旁白**：當他看到「下一站」那一行字時,整個人都愣住了。

**旁白**：下一站：寂海

**李杰**：(難以置信地低聲念出)寂海...?

**旁白**：他揉了揉眼睛,再次確認。沒錯,確實是「寂海」。

**李杰**：(內心,震驚)下一站是「寂海」?!怎麼會...

**旁白**：寂海是他們離開隧道後的第一個車站。那個一側是森林、一側是黑色大海的詭異月台。

**李杰**：(內心)難道這是一條無限循環的路線?

**旁白**：他帶著震驚,轉頭看向旁邊的時刻表。

**旁白**：時刻表同樣是木製的,上面刻著密密麻麻的文字和數字。他的目光落在「下一班列車」那一欄。

**旁白**：2045年。

**李杰**：(皺眉,轉向伊蘭尼爾)伊蘭尼爾,你確定列車是每兩年來一次嗎?

**旁白**：他指著時刻表。

**李杰**：這個時刻表上寫著,下一班車是...2045年。從現在算起,差不多是二十年後了。

**伊蘭尼爾**：(肯定地點頭)我非常確定。兩年與二十年,是不可能弄錯的。

**旁白**：他走近時刻表,仔細看了看。

**伊蘭尼爾**：不過,「2045年」...我不懂這個數字的含意。

**伊蘭尼爾**：對我們來說,現在是精靈曆5000年。

**旁白**：李杰瞬間明白了。

**李杰**：(內心)原來如此...精靈曆5000年...人類曆2045年...

**旁白**：兩個完全不同的世界,使用著完全不同的曆法計算方式。

**李杰**：(內心)精靈口中的「兩年」,換算成我們世界的時間,恐怕正好就是二十年。

**旁白**：這意味著,如果他錯過這班列車,就要在這個世界等待二十年。

**旁白**：他深吸一口氣,轉身看向停靠在月台上的列車。

**旁白**：那輛熟悉的列車靜靜地停在那裡。黑色的車身在陽光下泛著金屬光澤,車窗映照著森林的綠色。

**李杰**：我進去看看。

**伊蘭尼爾**：(點頭)小心。

**旁白**：李杰走向最近的一節車廂,推開門。

**旁白**：車廂內的場景讓他心頭一沉。

**旁白**：一具屍體倒在血泊中。那是那個自私的中年男人,臉朝下趴在地上。

**旁白**：李杰走近,小心地將屍體翻過來。

**旁白**：胸口有一個致命的傷口——被利劍貫穿,一擊斃命。傷口的邊緣平整而乾淨,與那個女大生和小林先生的傷口完全一樣。

**李杰**：(內心)同樣的劍傷...

**旁白**：他站起身,繼續搜索車廂。

**旁白**：在不遠處的座位下,他發現了第二具屍體。

**旁白**：那是那個焦慮的中年女人。她蜷縮在狹小的空間裡,身體僵硬。

**旁白**：李杰蹲下身,仔細觀察。

**旁白**：她的衣衫不整,釦子被扯開,裙子凌亂。臉上滿是乾涸的淚痕,表情扭曲著恐懼和絕望。

**旁白**：最觸目驚心的是她脖子上的瘀痕——深深的指印,清晰可見。她是被活活掐死的。

**李杰**：(內心,沉重)看來,在離開「亡者峽谷」後,車上只剩下他們兩人時...

**旁白**：他能想像那個場景。一個自私膽怯的男人,一個孤立無援的女人,在封閉的車廂裡。

**李杰**：(內心)那個男人對她伸出了魔爪...

**旁白**：但隨即,另一個疑問浮現。

**李杰**：(內心)可是,誰殺了那個男人?同樣的劍傷...

**旁白**：他看著男人胸口那道致命的傷口。

**李杰**：(內心)兇手是怎麼在行駛的列車上殺了他,又不留下任何蹤跡的?

**旁白**：列車的門在行駛時是鎖死的,不可能中途打開。窗戶也都是密閉的。

**旁白**：那個神秘的劍客,究竟是誰?又是如何進入列車的?

**旁白**：他找不到任何答案。

**李杰**：(低聲)對不起。

**旁白**：雖然這兩個人在生前並不討人喜歡,但此刻看著他們悲慘的死狀,李杰還是感到一陣悲哀。

**旁白**：他帶著沉重的心情退出了車廂。

**旁白**：伊蘭尼爾正在月台上等著他,靠在木製的欄杆上,看著遠方的森林。

**李杰**：(走過去)伊蘭尼爾。

**伊蘭尼爾**：(轉過身)怎麼樣?

**李杰**：裡面有兩具屍體。就像你說的。

**旁白**：他停頓了一下,問出心中的問題。

**李杰**：你說的神樹...從這裡過去,需要多久?

**伊蘭尼爾**：(指向森林深處)路途很遠。

**旁白**：他的手指指向北方,那裡的樹木更加茂密,幾乎看不見盡頭。

**伊蘭尼爾**：而且森林深處的路並不好走。有很多河流、山谷、還有迷霧。

**伊蘭尼爾**：以你的腳程,來回一趟,恐怕需要大半天的時間。

**旁白**：李杰的心沉了下去。

**李杰**：(內心)列車只會停留一天...

**旁白**：如果他現在去神樹,就算順利,等他回來的時候,列車也肯定已經開走了。

**李杰**：(內心)時間上,我不可能兩者兼得。

**旁白**：他站在月台上,看著眼前的列車。

**旁白**：黑色的車身在陽光下閃爍,車窗映照著他疲憊的臉。裡面躺著兩具屍體,承載著死亡和恐懼。

**旁白**：然後他回頭望向森林深處。

**旁白**：那裡有神樹,有可能回家的希望。但也有未知的危險,還有時間的限制。

**李杰**：(內心)是該搭上這輛滿載死亡的列車,重回循環的起點...

**旁白**：去尋找那些或許還活著的、第一批下車的同伴。那個沉穩的大叔,那個文靜的少女,還有那些大學生。

**李杰**：(內心)還是該放棄列車,去森林的中心...

**旁白**：尋找那一絲渺茫的、關於「回家」的希望。

**旁白**：兩個選擇,兩條截然不同的道路。

**旁白**：陽光依然明媚,森林依然寧靜。但時間在流逝,列車不會永遠等待。

**旁白**：他必須在此刻,做出最終的選擇。
