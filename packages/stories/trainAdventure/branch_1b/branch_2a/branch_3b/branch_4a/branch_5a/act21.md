# 第二十一幕：重返循環 

**旁白**：李杰站在月台上,目光在深邃的森林和冰冷的列車之間來回移動。

**旁白**：森林深處,有神樹,有回家的希望。列車裡面,有死亡,有謎團,但也有那些可能還活著的同伴。

**旁白**：陽光依然明媚,時間卻在流逝。

**旁白**：最終,他轉向伊蘭尼爾,眼中帶著一絲歉意與決心。

**李杰**：伊蘭尼爾,謝謝你的一切。

**旁白**：他深吸一口氣。

**李杰**：我決定...搭上這輛列車。

**伊蘭尼爾**：(靜靜地看著他,碧綠的眼眸中沒有意外,只有一絲理解)我明白了。

**旁白**：他的表情平靜,像是早已預料到這個答案。

**伊蘭尼爾**：雖然我認為神樹是更好的選擇,但我尊重你的決定。

**伊蘭尼爾**：每個人都有自己要追尋的路。

**李杰**：(苦笑)神樹或許能給我「回家」的答案...

**旁白**：他停頓了一下,組織語言。

**李杰**：但它解釋不了這趟旅程的真相,也無法解釋那些乘客的死因。

**李杰**：這輛列車...它本身就是案發現場。

**李杰**：我必須回到起點,看看會不會有新的發現。否則我無法心安理得地獨自尋找出路。

**伊蘭尼爾**：(點頭,語氣帶著一絲古老的智慧)我們精靈相信,有些路途必須走到盡頭。

**伊蘭尼爾**：即便那盡頭是最初的起點。

**旁白**：他看著遠方,眼神悠遠。

**伊蘭尼爾**：或許,你的答案不在於尋找新的終點。

**伊蘭尼爾**：而在於用新的雙眼,去審視那個舊的開端。

**旁白**：這句話讓李杰沉默片刻。

**李杰**：(低聲)謝謝你,伊蘭尼爾。

**旁白**：伊蘭尼爾從腰間的皮袋裡,取出一個用葉子包裹的小巧藥包。

**旁白**：藥包散發著淡淡的草本植物香氣,被精緻地用細繩綁好。

**伊蘭尼爾**：這是安神草,能幫助你在心煩意亂時保持平靜。

**旁白**：他將藥包遞給李杰。

**伊蘭尼爾**：帶上它吧。

**李杰**：(接過藥包,握在手中)我會的。謝謝。

**旁白**：在列車出發前,李杰和伊蘭尼爾合力將那兩具屍體抬下了車。

**旁白**：屍體已經僵硬,抬起來很費力。他們小心翼翼地將它們放在月台上,然後拖到車站旁不遠處的森林邊緣。

**旁白**：在那裡,他們為逝者挖了兩個簡單的墳墓。

**旁白**：泥土鬆軟,陽光透過樹葉灑落,給這個悲傷的時刻增添了一絲溫暖。

**旁白**：當兩個墳墓都填好土後,伊蘭尼爾站在墳前,閉上眼睛。

**旁白**：他用精靈的語言,輕聲為亡魂祝禱。那是一種古老而優美的語言,像是風吹過樹葉的聲音。

**旁白**：雖然李杰聽不懂具體的內容,但能感受到其中的哀悼與祝福。

**旁白**：願他們能擺脫列車的束縛,回歸安寧。

**李杰**：(低頭,默哀片刻)...

**旁白**：做完這一切後,兩人回到月台。

**旁白**：遠處傳來列車引擎低沉的轟鳴聲,彷彿在催促著出發。

**李杰**：(轉向伊蘭尼爾)那麼...再見了。

**伊蘭尼爾**：願森林之靈庇佑你。

**旁白**：李杰走向列車,推開車門,踏入空蕩蕩的車廂。

**旁白**：他選了一個靠窗的座位坐下,將背包和劍放在旁邊。手中緊緊握著那個還帶著清香的藥包,那是他在這個世界上唯一的、代表著溫暖的物品。

**旁白**：伊蘭尼爾站在月台上,靜靜地看著車窗裡的李杰。

**旁白**：車門發出「嘶」的聲音,開始緩緩關閉。

**旁白**：就在門完全閉合前,伊蘭尼爾對著車窗,行了一個莊重的精靈禮節。

**旁白**：他右手放在胸前,微微躬身,然後舉起手,輕輕揮動。

**李杰**：(透過窗戶,也舉手回應)...

**旁白**：列車發出一聲長鳴,緩緩開動。

**旁白**：車輪與鐵軌摩擦的聲音逐漸加快,窗外的景色開始移動。伊蘭尼爾那孤獨而優雅的身影,在月台上越來越小。

**旁白**：陽光灑在他銀色的長髮上,閃閃發光。他一直站在那裡,直到列車完全駛離車站。

**旁白**：最終,他的身影消失在了永恆之森的盡頭。

**旁白**：李杰轉過頭,看著前方。

**旁白**：車廂裡空蕩蕩的,只有他一個人。座位整齊排列,窗戶乾淨明亮,彷彿從未有過任何乘客。

**旁白**：但他知道,這輛列車曾經載著十幾個人,現在大部分都已經死了。

**李杰**：(內心,緊張)列車真的會回到「寂海」嗎?

**旁白**：他開始不受控制地思考起來。

**李杰**：(內心)如果真的回到了「寂海」,我會再次見到那些人嗎?

**旁白**：他回想起在「寂海」車站下車的那些人。

**李杰**：(內心)那個沉穩的大叔,那個文靜的少女...

**旁白**：還有那群把一切當作冒險的大學生們。

**李杰**：(內心)他們是否還活著?時間對他們來說,又是如何流逝的?

**旁白**：更讓他在意的是另一群人。

**李杰**：(內心)我又會不會,再次見到小林一家...

**旁白**：那對姐弟的哭泣聲,智子溫柔的笑容,都還清晰地印在他腦海中。

**李杰**：(內心)甚至,那個從一開始就消失了的黑衣男?

**旁白**：那個神秘的乘客,在第一站就莫名消失,至今下落不明。

**旁白**：就在他思緒紛飛之際,前方的景色開始變暗。

**旁白**：列車前方再次出現了隧道的入口。

**旁白**：那是一個巨大的黑色洞口,彷彿一張準備吞噬一切的巨口。周圍的岩壁粗糙而陰森,隧道內部伸手不見五指。

**李杰**：(看著隧道,握緊手中的藥包)...

**旁白**：列車平穩地駛入了純粹的黑暗之中。

**旁白**：窗外的陽光瞬間消失,取而代之的是完全的黑暗。車廂內的燈光自動亮起,投下昏黃的光芒。

**旁白**：隧道中的空氣潮濕而寒冷。列車的聲音在密閉的空間中變得更加響亮,產生陣陣回音。

**旁白**：李杰靠在座位上,閉上眼睛,試圖讓自己平靜下來。

**旁白**：五分鐘過去了。

**旁白**：十分鐘過去了。

**旁白**：十五分鐘過去了。

**旁白**：就在這時,一股劇烈的頭痛猛地襲來。

**旁白**：那是一種白熱化的、從內部炸開的劇痛。彷彿有人用烙鐵在他的大腦裡攪動,每一根神經都在尖叫。

**李杰**：(痛苦地抓住頭)啊...!

**旁白**：他的視線開始模糊,耳邊響起刺耳的嗡鳴聲。

**旁白**：手中的安神草藥包滑落在地,在昏黃的燈光下滾到座位下方。

**旁白**：李杰想要伸手去抓,但身體已經不聽使喚。

**旁白**：眼前一黑。

**旁白**：他徹底失去了意識。
