# 第二十七幕：黑劍的誘惑

**旁白**：列車在黑暗中行駛,車輪與鐵軌的摩擦聲在寂靜的車廂內迴盪。田中健太和佐藤拓海扶著李杰坐在座位上,幫他簡單地處理了腿上的傷口。山本浩二則守在昏迷的中村隼人身旁,確保他不會有任何動作。

**旁白**：就在這時,健太的目光落在了不遠處座位旁的黑劍上。那把劍靜靜地躺在那裡,黑色的劍身在昏黃的燈光下泛著詭異的光澤。

**田中健太**：那把劍...是什麼東西?看起來好奇怪...

**佐藤拓海**：對啊,這種造型...感覺不像是普通的武器...

**旁白**：兩人不自覺地朝黑劍走近了幾步。

**李杰**：別碰那把劍!

**旁白**：李杰的聲音突然響起,帶著明顯的緊張。

**李杰**：那把劍會控制持有者,讓人失去理智,殺掉所有人。上一個時間線裡,中村就是用這把劍,屠殺了車上所有的乘客。

**田中健太**：什麼?這把劍...會控制人?

**佐藤拓海**：聽起來...好像恐怖電影的情節...

**旁白**：李杰額頭冒著冷汗,不確定地看著那把劍。

**李杰**：(內心) 但說實話...我也不確定中村到底是被劍控制,還是他本來就是個瘋子...那把劍的力量是真的,但控制心智這點...我沒有證據...

**旁白**：就在這時——健太和拓海的表情變得有些恍惚。

**旁白**：他們的目光死死地盯著那把黑劍,像是被什麼東西吸引住了。

**田中健太**：(喃喃自語) 它在...說話...

**佐藤拓海**：(喃喃自語) 力量...它說...會給我力量...

**田中健太**：明日香...美咲...我可以保護她們...

**佐藤拓海**：對...我也可以...只要有這個力量...

**旁白**：兩人緩緩地朝黑劍走去,步伐越來越快。

**山本浩二**：等等!你們兩個!

**旁白**：山本浩二立刻察覺到不對勁,迅速起身衝了過去。他一把抓住拓海的肩膀,想要把他拉回來。

**山本浩二**：冷靜點!別被那把劍影響了!

**佐藤拓海**：放開我!我要保護大家!我需要那個力量!

**旁白**：拓海瘋狂地掙扎,力氣大得驚人。山本浩二咬緊牙關,一拳狠狠砸在拓海的後頸上。拓海的身體一軟,癱倒在地。

**旁白**：就在這一瞬間——連接第九節和第十節車廂的門,突然被推開了。鈴木明日香、高橋美咲和清水恵美走了進來。她們剛好看到山本浩二擊倒拓海的那一幕。

**鈴木明日香**：拓海?!

**高橋美咲**：你對拓海做了什麼?!

**旁白**：美咲的聲音尖銳而憤怒,她衝上前去,蹲在拓海身邊。

**高橋美咲**：大叔!你為什麼打他?!他做錯了什麼?!

**山本浩二**：等等,妳聽我解釋——

**李杰**：田中!別拿那把劍!

**旁白**：李杰的喊聲打斷了山本浩二的話。眾人轉頭看去——健太已經走到了黑劍旁邊,正彎腰伸手準備拿起它。他的表情恍惚,眼神空洞,像是失去了自我意識。

**李杰**：(掙扎著) 該死...!

**旁白**：李杰咬緊牙關,用盡全力撐起身體。他伸出手,抓住健太的小腿,想要把他拉回來。

**田中健太**：(恍惚地) 力量...我需要力量...保護明日香...保護美咲...保護拓海...

**旁白**：他猛地一甩腿,掙脫了李杰的手。李杰失去平衡,重重摔倒在地,腿上的傷口再次撕裂。

**李杰**：啊——!

**旁白**：劇痛讓他的視線瞬間模糊。健太的手,已經觸碰到了劍柄。

**田中健太**：(喃喃自語) 這個力量...是我的了...我可以保護大家...我可以...

**山本浩二**：住手!

**旁白**：山本浩二不再理會美咲的質問,轉身朝健太衝去。他的速度極快,拳頭直直地朝健太的臉揮去。但——已經太遲了。

**旁白**：健太的手指緊緊握住了劍柄。瞬間——一股強大的黑色能量從劍身爆發而出!那是一種濃稠的、邪惡的氣息,如同實質般擴散開來!能量形成一道無形的屏障,狠狠地撞擊在山本浩二的身上!

**山本浩二**：什麼——!

**旁白**：他的身體像被巨錘擊中,整個人倒飛了出去!重重撞在車廂壁上,發出沉悶的巨響。然後滑落在地,一動不動。

**鈴木明日香**：山本先生?!

**高橋美咲**：這...這是怎麼回事?!

**清水恵美**：(顫抖地) 那個...那個能量...

**旁白**：黑色的能量在爆發後,迅速收縮回劍身,全部湧入健太的體內。健太的身體猛地顫抖起來。他單膝跪地,另一隻手撐在地板上。手中依然緊緊握著黑劍,關節因為用力而發白。他的背對著李杰,看不清表情。但從他劇烈起伏的肩膀,和微微發抖的身軀,可以看出他正在承受著什麼。喉嚨深處發出低沉的、壓抑的呻吟。

**鈴木明日香**：健太...?

**旁白**：明日香想要走近,但腳步卻停在原地,不敢再前進。

**李杰**：(內心) 健太...他到底在承受什麼...

**旁白**：幾秒鐘過去了。健太身體的顫抖越來越劇烈,那壓抑的呻吟聲也越來越大。

**旁白**：然後——他的身體突然失去了所有力量。整個人側身倒在地上,發出沉悶的聲響。黑劍從他手中滑落,掉在一旁。

**鈴木明日香**：健太!

**旁白**：明日香立刻衝上前,跪在健太身邊。

**高橋美咲**：健太!你還好嗎?!

**旁白**：車廂內,陷入了混亂。只有列車行駛的聲音,在黑暗中持續迴響。
