# 第二十八幕：幽冥之都的抉擇

**旁白**：明日香跪在健太身邊,伸手想要扶起他。但當她的手觸碰到健太的肩膀時——一股無形的力量猛地將她的手彈開!

**鈴木明日香**：啊!

**旁白**：她整個人被彈得往後跌坐在地。

**高橋美咲**：明日香!

**旁白**：美咲立刻衝上前,想要拉起明日香。但她也伸手嘗試觸碰健太——同樣的,一股力量將她的手彈開。

**高橋美咲**：這...這是怎麼回事?!為什麼碰不到他?!

**李杰**：(咒罵) 該死...!

**旁白**：李杰用力捶打地板,額頭青筋暴起。

**李杰**：(內心) 怎麼又...又是這樣...!事情又朝著最壞的方向發展了...!

**旁白**：列車的速度開始減緩。震動變得輕微,車廂內的燈光微微閃爍。然後,列車平穩地停了下來。

**旁白**：車門「嘶」的一聲,自動打開了。窗外,是一片陰暗的景色。月台籠罩在濃重的霧氣中,遠處隱約可見一些破舊的建築輪廓。空氣中彌漫著一股壓抑的氣息。

**李杰**：幽冥之都...第二站...

**旁白**：李杰掙扎著撐起身體,透過車窗往外看。月台上空無一人。

**李杰**：(內心) 凌墨當然不在這裡...列車的速度那麼快,他不可能跑得比列車還快...

**旁白**：他轉過頭,看向車廂內的眾人。明日香和美咲跪在健太身旁,臉上滿是淚水和恐懼。拓海靠在座位旁,似乎恢復了意識,但臉色蒼白。山本浩二靠在車廂壁上,還沒有完全恢復。恵美站在不遠處,表情複雜地看著這一切。

**李杰**：聽我說。你們其中四個人必須立刻下車。

**旁白**：李杰的聲音很急促,但很堅定。

**李杰**：去找凌墨。告訴他這裡發生的事。然後深入森林,去找精靈領袖艾拉如。只有他能幫我們了。

**高橋美咲**：什...什麼?下車?

**鈴木明日香**：可是...可是健太他...!

**李杰**：我、健太、還有中村會留在車上。

**旁白**：他停頓了一下,深吸一口氣。

**李杰**：還需要一個人留下來。留下來的那個人...要有心理準備...在健太醒來後...可能會被他殺死。

**旁白**：車廂內陷入了死一般的寂靜。所有人都愣住了,完全不知道該如何反應。

**鈴木明日香**：被...被殺死...?

**旁白**：她的聲音顫抖,眼淚開始不受控制地流下來。

**鈴木明日香**：不...不要...健太他...他不會...!

**旁白**：她開始放聲大哭。

**高橋美咲**：這...這到底是怎麼回事...!

**旁白**：美咲也跟著哭了起來,雙手掩著臉。

**佐藤拓海**：可惡...!

**旁白**：拓海用力捶打地板,牙關緊咬。

**佐藤拓海**：可惡!可惡!可惡!

**旁白**：他的拳頭不斷砸在地板上,發出沉悶的聲響。

**李杰**：我知道這很殘酷。但如果不這樣做,所有人都會死在這裡!快!沒時間了!你們必須現在下車!

**山本浩二**：我留下。

**旁白**：山本浩二撐著牆壁站起身,聲音低沉而堅定。

**山本浩二**：我年紀最大,活得也夠久了。讓我來吧。

**鈴木明日香**：不...不要...!我不要下車...!我要留下來陪健太...!

**旁白**：明日香哭喊著,緊緊抓住健太的衣服。山本浩二走上前,一把抓住明日香的手臂。

**鈴木明日香**：放開我!放開——

**旁白**：話還沒說完——山本浩二用力一甩,將明日香整個人扔出了車門!

**鈴木明日香**：啊——!

**旁白**：明日香的身體在空中劃過一道弧線,重重摔在月台上。

**高橋美咲**：明日香!

**佐藤拓海**：美咲,走!

**旁白**：拓海一把抓住美咲的手腕,用力拖著她往車門走去。

**高橋美咲**：等等...!我也想留下來幫忙...!健太他...!

**旁白**：美咲掙扎著想要掙脫,但拓海的力氣更大。他將美咲拖出車門,兩人一起跌落在月台上。

**旁白**：恵美站在車門口,回頭看了一眼車廂內的景象。她的表情很複雜,像是想說什麼,但最終什麼都沒說。她轉身,走下了列車。腳步聲在月台上響起,然後停在了明日香和美咲身邊。

**旁白**：就在恵美踏下月台的瞬間——車門「嘶」的一聲,猛地關上了!

**鈴木明日香**：不——!

**旁白**：明日香立刻爬起來,衝到車門前。她瘋狂地敲打著車門,眼淚模糊了視線。

**鈴木明日香**：開門!拜託!開門!健太還在裡面!健太...!健太...!

**旁白**：她的拳頭不斷敲打著玻璃,發出沉悶的聲響。但車門紋絲不動。

**旁白**：美咲跪在月台上,雙手掩著臉,肩膀不斷顫抖。哭聲在空曠的月台上迴盪。

**旁白**：拓海從後面走上前,伸手環抱住美咲。

**佐藤拓海**：美咲...

**旁白**：他的聲音也在顫抖,但他努力保持鎮定。

**佐藤拓海**：我們...我們會救他們的...一定會的...

**旁白**：美咲靠在拓海懷裡,哭得更大聲了。

**旁白**：恵美站在他們旁邊,靜靜地看著這一切。她的表情很複雜,像是悲傷,又像是無奈。她什麼都沒說,只是站在那裡。

**旁白**：列車開始震動。引擎重新啟動的聲音在夜色中響起。

**旁白**：明日香的敲打聲越來越急促,越來越用力。

**鈴木明日香**：不要...不要走...!健太...!

**旁白**：但列車還是緩緩啟動了。窗外的景色開始移動。

**旁白**：明日香跟著列車跑,不斷敲打著車門。但列車的速度越來越快,她最終停下了腳步。跪倒在月台上,放聲大哭。

**旁白**：列車駛向遠方,朝著未知的方向前進。而在第二站的月台上,四個人影逐漸變得渺小。

**旁白**：車廂內,李杰靠在座位上,閉上了眼睛。

**李杰**：(內心) 我又...又讓她們經歷了這些...這次...真的能改變結局嗎...

**旁白**：車廂在黑暗中搖晃。健太依然倒在地上,一動不動。黑劍靜靜地躺在他身旁,散發著不祥的氣息。而在不遠處,山本浩二靠在座位旁。他的眼神很平靜,像是已經做好了覺悟。

**旁白**：列車繼續前行。駛向更深的黑暗。
