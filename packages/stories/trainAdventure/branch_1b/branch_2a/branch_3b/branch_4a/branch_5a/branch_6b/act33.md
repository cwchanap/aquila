# 第三十三幕：第三次機會

**旁白**：不知道過了多久。

**旁白**：李杰的意識在黑暗中漂浮,然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧?我的手機完全沒訊號!連「圈外」都不是,是「沒有服務」耶!這底下是多深啊?我從來沒見過這種狀況...

**田中健太**：我的也是,什麼都傳不出去。LINE、IG、什麼都不行。喂,這隧道也太長了吧?以前有這麼長嗎?我怎麼一點印象都沒有?

**旁白**：李杰睜開眼睛。

**旁白**：他坐在搖晃的車廂裡,窗外是純粹的黑暗。

**李杰**：(內心,疲憊) 又...又回來了...

**旁白**：他低頭看向自己的左腿。

**旁白**：褲子完好無損,沒有血跡,沒有傷口。

**旁白**：就像剛才的槍傷、失血、爬行...所有的痛苦都沒有發生過。

**李杰**：(內心) 這已經是第三次了...

**旁白**：他深吸一口氣,用手按了按太陽穴。

**旁白**：身體沒有受傷,但精神上的疲憊卻無法消除。一次又一次的失敗,一次又一次的死亡...

**李杰**：(內心) 翔被殺了...健太被附身了...凌墨被留在第一站...

**李杰**：(內心) 這次...必須改變...！

**旁白**：他咬緊牙關,站起身。

**旁白**：雖然身體很疲憊,但他還是朝著車廂後方走去。

**旁白**：穿過一節又一節車廂,腳步略顯沉重。

**旁白**：終於,他推開了第十節車廂的門。

**旁白**：凌墨正站在車廂中央,手伸向那把黑劍——

**李杰**：別碰那把劍！

**旁白**：凌墨的手停在半空中,轉頭看向李杰。

**凌墨**：你是誰？

**旁白**：李杰快速走到連接第九節和第十節車廂的門旁邊,背靠著牆壁站好。

**旁白**：他向凌墨快速說明了自己的身份——從未來回來的時間旅行者。

**旁白**：他告訴凌墨關於黑劍的危險,關於恵美的計畫,關於恵美想用一個乘客交換她的男友翔。

**旁白**：凌墨雖然對時間旅行感到懷疑,但還是選擇相信。

**旁白**：他表明自己的立場——保護恵美,幫她找到替代品。這意味著至少要犧牲一個人。

**旁白**：如果李杰堅持要救所有人,他們就無法合作。

**旁白**：李杰沒有回答。他只是靜靜地站在那裡,目光若有似無地瞥向第九節車廂的門。

**旁白**：就在這時——

**旁白**：連接第九節和第十節車廂的門被推開了。

**旁白**：中村隼人走進來,手中握著手槍,槍口對準凌墨。

**中村隼人**：真是精彩的對話啊。我全都——

**旁白**：他突然愣住了。

**中村隼人**：(內心) 奇怪...我明明聽到有人說話...但只有這個男人？

**旁白**：李杰從他身後撲了過去！

**旁白**：用盡全力撞向中村的後背！

**中村隼人**：什麼——

**旁白**：中村失去平衡,重重摔倒在地。手槍飛了出去。

**旁白**：凌墨瞬間撿起手槍,槍口對準中村。

**旁白**：李杰騎在中村身上,死死壓住他的雙臂。

**中村隼人**：你這混蛋！放開我！

**旁白**：李杰沒有理會他的叫喊。他開始搜查中村的口袋。

**旁白**：從外套內側口袋,掏出一把手槍。

**旁白**：從褲子右側口袋,掏出一把折疊刀。

**旁白**：從褲子左側口袋,又掏出一把折疊刀。

**旁白**：從後腰,掏出最後一把手槍。

**旁白**：李杰將所有武器扔到遠處,然後繼續壓制著中村。

**中村隼人**：你...你怎麼知道...

**旁白**：中村的臉色變得蒼白,眼神中滿是震驚。

**中村隼人**：你怎麼知道我把武器藏在哪裡？！

**旁白**：他瞪大眼睛,難以置信地看著李杰。

**中村隼人**：你該不會...已經經歷過好幾次了吧？！你到底重來過多少次？！

**旁白**：李杰沒有回應。他只是冷冷地看著中村,然後轉頭對凌墨說。

**李杰**：凌墨,我們把他移到第二節車廂。

**李杰**：不能讓任何人接近這把黑劍。

**旁白**：凌墨點了點頭。

**凌墨**：好。

**旁白**：李杰站起身,一把抓住中村的衣領,將他從地上拖起來。

**李杰**：走。

**旁白**：他押著中村,朝著前方車廂走去。

**旁白**：凌墨跟在後面,手中的手槍始終對準著中村。

**旁白**：他們穿過第九節車廂,第八節,第七節...

**旁白**：一節又一節空蕩蕩的車廂。

**旁白**：終於,他們來到第二節車廂。

**旁白**：這節車廂空無一人。

**李杰**：就這裡。

**旁白**：他將中村按在座位上。

**中村隼人**：你們不能這樣對我！我要報警！

**李杰**：凌墨,去前面找繩子。

**凌墨**：好。

**旁白**：凌墨轉身,朝著第一節車廂走去。

**旁白**：李杰用力壓著中村的肩膀,讓他無法動彈。

**中村隼人**：喂...喂！你到底想幹什麼？！

**李杰**：閉嘴。

**旁白**：中村還想說些什麼,但看到李杰冰冷的眼神,最終還是閉上了嘴。

**旁白**：過了幾分鐘,凌墨回來了。

**旁白**：他手中拿著幾條長長的USB線。

**凌墨**：前面車廂的女生給的。她們說這是她們能找到的最長的線。

**李杰**：夠了。

**旁白**：凌墨立刻上前,用USB線開始綑綁中村的雙手。

**旁白**：一條,兩條,三條...他用了好幾條線,纏了好幾圈,確保中村掙脫不開。

**旁白**：然後是雙腳。同樣纏了好幾圈。

**凌墨**：報警？

**旁白**：凌墨冷笑了一聲。

**凌墨**：等你能聯絡到警察再說吧。

**旁白**：綑綁完成後,他們將中村放在角落的座位上。

**旁白**：李杰看著被牢牢綁住的中村,深深地吐了一口氣。

**李杰**：(內心) 這次...總算順利抓住他了...

**李杰**：(內心) 但接下來...還有更多問題要解決...

**李杰**：(內心) 健太...絕對不能讓他碰到黑劍...

**旁白**：他轉頭看向凌墨。

**李杰**：我們得確保沒有人能去第十節車廂。

**凌墨**：好。我去跟前面車廂的人說,後面的車廂很危險,不要靠近。

**李杰**：我也去告訴其他人。我們分頭行動。

**旁白**：兩人點了點頭。

**旁白**：留下被綑綁的中村隼人,一個人坐在角落,臉色鐵青。

**旁白**：車廂在搖晃。窗外依然是純粹的黑暗。

**旁白**：而在這條駛向異世界的列車上,命運的齒輪,正朝著新的方向轉動。
