# 第三十六幕：黑劍之主

**旁白**：李杰的頭越來越沉重。

**旁白**：視線模糊不清，耳邊傳來的聲音也變得遙遠。

**旁白**：失血...太多了...

**旁白**：山本浩二已經將那個女人牢牢綁住。他脫下自己的外套，試圖壓在李杰的傷口上止血。

**旁白**：但鮮血還是不斷滲透布料，根本止不住。

**山本浩二**：該死...這樣下去不行...

**旁白**：他的聲音充滿焦急。

**旁白**：就在這時，田中健太和佐藤拓海從第一節車廂衝了回來。

**旁白**：兩人空手而歸，臉色難看。

**佐藤拓海**：不行...第一節車廂什麼都沒有...沒有急救箱，什麼都沒有...

**田中健太**：我們問了所有人...大家都說沒帶醫療用品...

**旁白**：山本浩二咬緊牙關。

**山本浩二**：可惡...

**旁白**：車廂內陷入沉默。

**旁白**：所有人都不知道該怎麼辦。

**旁白**：李杰的呼吸越來越微弱。

**旁白**：又過了幾秒鐘，連接第一節車廂的門再次打開。

**旁白**：小林雄大走了進來，臉上帶著擔憂。

**小林雄大**：發生什麼事了？我聽到前面很吵...

**旁白**：他的視線落在李杰身上，表情瞬間凝重。

**小林雄大**：這...！

**李杰**：去...去幫凌墨...

**旁白**：李杰用微弱的聲音說道。

**李杰**：伊藤...很危險...凌墨一個人...可能打不過他...

**山本浩二**：我明白了。

**旁白**：他轉向小林雄大。

**山本浩二**：我們一起去吧。

**小林雄大**：什麼...？去哪裡？

**旁白**：小林雄大還搞不清楚狀況，但看到山本浩二嚴肅的表情，他點了點頭。

**小林雄大**：好...我跟你去。

**旁白**：山本浩二站起身，對田中健太和佐藤拓海說。

**山本浩二**：你們兩個留在這裡，看住這兩個人。

**旁白**：他指了指被綁住的女人和中村隼人。

**田中健太**：知道了！

**旁白**：山本浩二和小林雄大轉身，朝著第三節車廂衝去。

**旁白**：車廂內再次陷入沉默。

**旁白**：只剩下李杰、田中健太、佐藤拓海，以及被綁住的女人和中村隼人。

**旁白**：過了片刻，中村隼人突然開口。

**中村隼人**：喂，女士。

**旁白**：他看向被綁在一旁的女人。

**中村隼人**：妳叫什麼名字？

**旁白**：女人緩緩抬起頭，眼神依然空洞。

**伊藤奈奈**：伊藤...奈奈...

**旁白**：她的聲音空洞而呆滯，像是機械般的回答。

**中村隼人**：奈奈啊...那麼，為什麼要刺他？

**旁白**：他用下巴指了指李杰。

**伊藤奈奈**：因為...他想殺我丈夫...

**旁白**：她的聲音毫無感情。

**伊藤奈奈**：他想殺誠...所以...我要殺他...

**旁白**：李杰微微睜開眼睛，看向奈奈。

**李杰**：妳...為什麼會這麼想...

**旁白**：他的聲音虛弱到幾乎聽不見。

**李杰**：我從來沒說過...要殺你丈夫...

**旁白**：奈奈沒有回應，只是茫然地看著前方。

**中村隼人**：呵呵...有意思...

**旁白**：他露出玩味的笑容。

**中村隼人**：奈奈小姐，妳是不是聽到了什麼聲音？

**中村隼人**：從那把黑劍...傳來的聲音？

**旁白**：奈奈的眼神閃爍了一下。

**伊藤奈奈**：聲音...對...聲音...

**旁白**：她的聲音開始變得混亂。

**伊藤奈奈**：它說...它說要保護誠...誠很危險...他們想殺誠...

**伊藤奈奈**：黑色的...黑色的火焰...燃燒...燃燒一切...

**伊藤奈奈**：力量...給我力量...我要保護...保護...

**旁白**：她開始語無倫次，眼神越來越渙散。

**田中健太**：她...她這是怎麼了...

**佐藤拓海**：好像...精神錯亂了...

**旁白**：就在這時——

**旁白**：列車開始減速。

**旁白**：窗外的黑暗中，逐漸出現了熟悉的景象。

**旁白**：左側是幽深的森林，右側是黑色的海洋。天上掛著兩輪紫色的殘月。

**旁白**：列車緩緩停下。

**旁白**：寂海站。

**旁白**：「嘶——」

**旁白**：車門打開了。

**旁白**：但沒有人下車。

**旁白**：所有人都呆呆地看著打開的車門。

**旁白**：幾秒鐘後——

**旁白**：連接第三節車廂和第二節車廂的門，突然「砰」的一聲被踢開！

**旁白**：一個身影衝了進來！

**旁白**：是伊藤誠。

**旁白**：他的右手緊握著一把漆黑的劍。

**旁白**：黑劍。

**旁白**：他的眼睛不再是正常的模樣——瞳孔中燃燒著赤紅色的火焰。

**旁白**：那火焰在他的眼眶中跳動，散發著不祥的光芒。

**旁白**：他的臉上掛著瘋狂的笑容。
