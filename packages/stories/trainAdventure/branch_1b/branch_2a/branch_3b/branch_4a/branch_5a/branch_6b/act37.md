# 第三十七幕：絕望的終局

**伊藤誠**：殺光你們...！全部都給我死——！

**旁白**：他的聲音嘶啞而狂暴，迴盪在整個車廂。

**旁白**：田中健太和佐藤拓海嚇得臉色慘白，本能地轉身就跑！

**田中健太**：快逃！快逃啊——！

**佐藤拓海**：大家快下車！快——！

**旁白**：兩人衝進第一節車廂，聲嘶力竭地大喊。第一節車廂內傳來混亂的喊聲和腳步聲。

**旁白**：李杰躺在地上，視線模糊地看向第三節車廂。沒有凌墨的身影，沒有山本浩二的身影，也沒有小林雄大的身影。

**李杰**：(內心) 他們...都死了...

**李杰**：(內心) 可惡...又失敗了...我又失敗了...！

**旁白**：伊藤誠高舉黑劍，朝著李杰走來，每一步都帶著壓迫感。

**旁白**：就在這時——

**中村隼人**：等...等等！

**旁白**：被綁在一旁的中村突然開口，他的眼睛瞪得滾圓，盯著伊藤手中的黑劍，眼神中滿是狂熱。

**中村隼人**：這力量...這力量太驚人了！

**中村隼人**：伊藤先生！我們可以合作！一起統治這個世界！

**中村隼人**：你有黑劍！我有計畫！我們可以征服所有精靈！成為這個世界的王——！

**旁白**：伊藤誠停下腳步，轉頭看向中村隼人。那雙燃燒著赤紅火焰的眼睛，冷冷地盯著他。中村的聲音戛然而止，空氣彷彿凝固了。

**伊藤誠**：你...沒資格。

**旁白**：他的聲音低沉而冰冷。

**旁白**：下一秒，黑劍劃過一道黑色的弧線！「噗嗤——」劍刃刺穿了中村的胸膛。

**中村隼人**：啊...啊...

**旁白**：中村的眼睛瞪得更大，嘴巴微微張開，想說些什麼，但只能發出微弱的呻吟。鮮血從傷口湧出，染紅了他的衣服。伊藤誠抽出黑劍，中村的身體癱軟下來，頭垂在胸前。死了。

**旁白**：李杰看著這一幕，心中湧起一股複雜的情緒。恐懼、絕望、還有一絲...解脫？至少中村死了...那個想要殺人的傢伙，終於死了...

**旁白**：伊藤誠轉過頭看向李杰。李杰閉上眼睛，等待著死亡。

**旁白**：但預期的劇痛並沒有到來。李杰微微睜開眼睛，看到伊藤誠正盯著他，眼神中閃爍著詭異的光芒。然後，伊藤誠轉身，走向另一個被綁住的人——他的妻子，伊藤奈奈。

**李杰**：不...！

**旁白**：李杰想要喊出聲，但虛弱的身體讓他只能發出微弱的呻吟。伊藤誠舉起黑劍，奈奈抬起頭，空洞的眼神看著自己的丈夫。

**伊藤奈奈**：誠...？

**旁白**：黑劍橫掃而過。「噗——」奈奈的喉嚨被割開，鮮血噴湧而出。她的眼睛慢慢失去光澤，身體無力地倒下。

**旁白**：李杰瞪大眼睛，震驚地看著這一幕。

**李杰**：(內心) 他...他殺了自己的妻子...！

**旁白**：就在這時，「嘶——」列車門關閉的聲音響起。列車開始震動，緩緩啟動。

**李杰**：(內心) 列車...開動了...有人逃出去了嗎...

**旁白**：列車開始加速。

**旁白**：就在這時，連接第一節車廂和第二節車廂的門突然打開了。一個身影衝了進來。

**清水恵美**：凌墨！凌墨你在哪——

**旁白**：恵美的聲音戛然而止。她看到了車廂內的景象——地上的鮮血、中村和奈奈的屍體，還有握著黑劍、眼睛燃燒著赤紅火焰的伊藤誠。她的臉色瞬間變得慘白。

**旁白**：伊藤誠轉過身看向她，然後身形化作一道黑影，瞬間衝到恵美面前！黑劍橫掃——

**李杰**：恵美——！

**旁白**：李杰用盡全力喊出她的名字，但已經來不及了。「噗——」黑劍劃過恵美的喉嚨，鮮血噴灑而出。

**清水恵美**：啊...

**旁白**：恵美的雙手本能地捂住喉嚨，鮮血從指縫間湧出。她的眼睛睜得很大，看向李杰的方向。

**旁白**：嘴巴微微張開，想說些什麼，但只能發出模糊的氣音。然後，她的身體慢慢倒下。「砰——」她倒在地上，再也沒有動靜。

**李杰**：恵美...恵美...！

**旁白**：李杰的聲音充滿絕望。淚水不受控制地從眼眶湧出。

**李杰**：(內心) 為什麼...為什麼我這麼無力...我什麼都保護不了...什麼都做不到...我只能眼睜睜看著他們死...一個接一個...

**旁白**：伊藤誠站在恵美的屍體旁，咬緊牙關，臉上滿是憤怒。

**伊藤誠**：可惡...還是讓那麼多人逃走了...本來可以全部殺光的...

**旁白**：他轉過身，看向躺在地上的李杰。那雙燃燒著火焰的眼睛充滿了殺意。他舉起黑劍，一步一步走向李杰。

**旁白**：李杰已經沒有力氣反抗了。他只能躺在那裡，看著伊藤誠越來越近。

**李杰**：(內心) 結束了...這次...真的結束了...

**旁白**：伊藤誠站在李杰面前，高舉黑劍。劍尖對準李杰的心臟。

**旁白**：下一秒，黑劍筆直刺下！「噗嗤——」劍刃刺穿了李杰的胸膛。

**旁白**：劇烈的疼痛瞬間襲來，但很快就變得麻木。李杰的視線越來越模糊，他看到伊藤誠抽出黑劍，鮮血從傷口湧出。

**旁白**：周圍的聲音越來越遠，一切都開始變得安靜。

**旁白**：然後——一片黑暗。
