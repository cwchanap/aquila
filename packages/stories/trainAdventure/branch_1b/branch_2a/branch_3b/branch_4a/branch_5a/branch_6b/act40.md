# 第四十幕：時間旅行者的謊言

**李杰**：聽我說...我知道這聽起來很荒謬，但我必須告訴你們真相。

**李杰**：我是一個時間旅行者。

**旁白**：第一節車廂裡，所有人都安靜下來，目光聚焦在李杰身上。

**李杰**：每次我死亡，時間就會倒流回到列車剛到達這個世界的時候。

**李杰**：我已經經歷過這一切很多次了。我知道後面會發生什麼。

**李杰**：列車的第十節車廂裡，有一把黑色的劍。那把劍被一個古代戰士的靈魂附身。

**李杰**：它會誘惑乘客去拿起它，然後占據那個人的身體，殺死列車上所有的人。

**李杰**：我回來，就是為了阻止這一切發生，為了拯救所有人！

**旁白**：李杰說完，期待著眾人的反應。

**旁白**：但出乎意料的是——

**旁白**：所有人的臉上並沒有震驚，反而露出一種...恍然大悟的表情。

**田中健太**：哦...原來如此...

**鈴木明日香**：所以是這樣啊...

**旁白**：他們的語氣平淡，像是終於得到了一個合理的解釋。

**旁白**：但不是相信，而是...更加懷疑。

**李杰**：(內心) 什麼...？這反應不對...

**旁白**：斎藤蓮推了推眼鏡，臉上露出一絲冷笑。

**斎藤蓮**：哈...你終於承認了。

**李杰**：什麼？承認什麼？

**斎藤蓮**：承認你是時間旅行者啊。

**斎藤蓮**：我早就告訴過大家了——你和那個黑衣男人，是時間旅行者。

**斎藤蓮**：但你們不是來拯救我們的。你們的真正目的，是要占據這個世界！

**旁白**：李杰終於明白了。

**李杰**：(內心) 原來如此...這就是為什麼山本先生和小林先生會在門口埋伏...

**李杰**：(內心) 斎藤在我們回來之前，就已經把謊言告訴了所有人...

**旁白**：凌墨突然開口，聲音冷靜。

**凌墨**：你怎麼知道這麼多的？

**凌墨**：是那把黑劍告訴你的嗎？

**田中健太**：還想轉移話題...

**鈴木明日香**：都這個時候了還在狡辯...

**旁白**：眾人的目光沒有轉向斎藤，反而更加戒備地看著凌墨和李杰。

**旁白**：斎藤搖了搖頭，推了推眼鏡，臉上露出一個得意的笑容。

**斎藤蓮**：不。黑劍沒有告訴我任何事情。

**斎藤蓮**：我知道這些...是因為我也是一個時間旅行者。

**斎藤蓮**：這一點，我早就告訴過大家了。

**旁白**：李杰瞪大眼睛。

**李杰**：(內心) 什麼...他說他也是時間旅行者...？

**斎藤蓮**：沒錯。我和你一樣，經歷過無數次時間循環。

**斎藤蓮**：但不同的是——我決定把真相告訴大家。

**旁白**：他環視車廂裡的所有人，語氣變得激動而誇張。

**斎藤蓮**：所以在你們回來之前，我就已經把一切都告訴了大家！

**斎藤蓮**：你和那個黑衣男人的真正目的——是要控制第十節車廂的黑劍！

**斎藤蓮**：然後把我們所有人，送到列車的終點站「終焉祭壇」！

**旁白**：他張開雙臂，做出誇張的手勢。

**斎藤蓮**：把我們作為祭品，獻給那座祭壇！讓你們成為這個世界的王！

**斎藤蓮**：這就是為什麼你們要綁架中村先生！因為他發現了你們的陰謀！

**旁白**：李杰聽著這些荒謬的謊言，感到一陣無力。

**李杰**：這完全是胡說八道！

**李杰**：我說的才是真話！我是來拯救大家的！

**斎藤蓮**：是嗎？那你解釋一下，你為什麼要偷偷摸摸地綁架中村先生？

**斎藤蓮**：為什麼要把他拖到後面的車廂？你們想對他做什麼？

**李杰**：因為他想去拿黑劍！我必須阻止他！

**李杰**：那把劍會控制人的心智！會殺死所有人！

**佐藤拓海**：你怎麼證明？

**田中健太**：對啊，我們只看到你們把中村先生綁起來...

**李杰**：你們必須相信我！斎藤說的全是謊言！

**李杰**：我真的是來拯救大家的！

**旁白**：他環視車廂裡的所有人，試圖從他們的眼神中尋找一絲相信。

**旁白**：山本和雄大的臉上寫滿警惕，明顯不相信李杰。

**旁白**：健太、拓海、明日香、美咲四個大學生，眼神中充滿懷疑和戒備。

**旁白**：伊藤夫婦互相看著，竊竊私語。

**旁白**：小林智子護著兩個孩子，眼神中滿是防備。

**李杰**：(內心) 沒有人相信我...

**李杰**：(內心) 但最重要的是...目前沒有人看起來被黑劍控制。

**李杰**：(內心) 至少...至少大家現在還是安全的...

**旁白**：他的目光落在人群中的一個女孩身上。

**旁白**：那個文靜的女孩——清水恵美。

**旁白**：她站在大學生們的旁邊，臉上寫滿擔憂。

**旁白**：她的眼神在李杰和凌墨之間游移，像是想說什麼，但又不敢說。

**李杰**：(內心) 她...她也來過這個世界。她應該知道斎藤說的是胡說八道...

**李杰**：(內心) 但她沒有說出來...她不想讓自己變得可疑...

**旁白**：恵美注意到李杰的目光，眼神中閃過一絲掙扎。

**旁白**：但最終，她低下了頭，沒有說話。

**山本浩二**：麻煩兩位女同學去拿幾條USB線過來。

**山本浩二**：我們需要把這兩個人綁起來，確保大家的安全。

**清水恵美**：我去拿！

**旁白**：恵美快步走向前，聲音中帶著一絲緊張。

**清水恵美**：我...我一個人就可以了。

**旁白**：她轉身，快步走向其他乘客，開始收集USB充電線。

**旁白**：李杰深吸一口氣，做最後的嘗試。

**李杰**：聽我說！我說的才是真話！

**李杰**：後面第十節車廂真的有一把黑劍！它會殺死所有人！

**李杰**：你們必須相信我——！

**旁白**：但沒有人理會他。

**田中健太**：他還在說那些話...

**佐藤拓海**：真是不死心啊...

**旁白**：恵美很快就收集到了好幾條USB線，遞給山本和雄大。

**旁白**：她低著頭，不敢看李杰和凌墨。

**山本浩二**：謝謝你，同學。

**旁白**：山本接過USB線，開始將李杰的雙手綁在背後。

**李杰**：等等！你們這是在犯錯！

**李杰**：斎藤才是在說謊！他根本不是什麼時間旅行者！

**旁白**：雄大也同樣將凌墨的手綁起來。

**旁白**：凌墨沒有掙扎，也沒有說話，只是緊閉著嘴唇，任由雄大將USB線纏在他的手腕上。

**凌墨**：(內心) 李杰說的...應該是真的。

**凌墨**：(內心) 我記得那些模糊的記憶...有人告訴我黑劍很危險...

**凌墨**：(內心) 但現在...沒有人相信我們。

**凌墨**：(內心) 爭辯也沒有用。只會讓情況更糟。

**凌墨**：(內心) 現在最重要的是...保持冷靜。等待機會。

**李杰**：你們會後悔的！黑劍真的會殺死所有人！

**李杰**：至少...至少不要讓任何人去第十節車廂！

**山本浩二**：夠了。安靜點。

**李杰**：我說的是真的...你們一定要相信我...

**旁白**：李杰的聲音越來越無力。

**旁白**：USB線被牢牢地纏在他的手腕上，勒得很緊。

**李杰**：(內心) 我已經盡力了...但沒有人相信我...

**李杰**：(內心) 至少...至少現在大家還是安全的。只要他們不去第十節車廂...

**旁白**：他感到一陣無力和絕望。

**旁白**：這是前所未有的情況。

**旁白**：不是被黑劍控制的人殺死，而是...被其他乘客當作威脅，綁了起來。

**旁白**：斎藤蓮站在一旁，臉上掛著勝利的笑容。

**旁白**：他看著被綁起來的李杰和凌墨，眼神中閃爍著得意的光芒。

**旁白**：車廂裡的氣氛壓抑而緊張。

**旁白**：所有人都警惕地看著這兩個被綁起來的「時間旅行者」。

**旁白**：沒有人知道，真正的危險，還在列車的最深處，靜靜地等待著...
