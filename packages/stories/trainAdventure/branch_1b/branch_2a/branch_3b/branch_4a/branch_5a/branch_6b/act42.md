# 第四十二幕：絕望的深淵

**旁白**：李杰和凌墨立刻轉身，朝著森林的方向狂奔！他們的腳步聲在夜色中迴響，心跳如擂鼓。

**李杰**：快跑！

**凌墨**：我知道！

**旁白**：但身後——一個黑影以驚人的速度追了上來。斎藤蓮，眼中燃燒著紅色的火焰，手持黑劍，像一頭野獸般撲向他們。他的速度遠超常人，每一步都跨出數米的距離。

**李杰**：(內心) 太快了...！

**旁白**：李杰的心臟劇烈跳動，肺部因為奔跑而灼痛。但斎藤的速度更快，他離他們越來越近。十米...五米...三米...李杰能感受到身後的殺氣。

**旁白**：「唰——」一道黑色的劍影在月光下劃過。

**李杰**：嗚——！

**旁白**：劍刃刺穿了李杰的背部，從胸口穿出。鮮血噴濺而出。

**凌墨**：李杰！

**旁白**：李杰的雙腿一軟，跪倒在地上。劇烈的疼痛從胸口蔓延到全身。他感覺到黑劍從他的身體中抽出，鮮血如泉水般湧出。他倒在地上，呼吸變得越來越困難，視線開始模糊。他勉強抬起頭，看向列車的方向。

**旁白**：「咔嚓——」列車的車門緩緩關上。「嗚——」汽笛聲響起，列車開始緩緩移動。

**李杰**：(內心，虛弱) 列車...開走了...凌墨...其他人...你們...能逃掉嗎...

**旁白**：他的視線越來越模糊，只能勉強看到列車緩緩遠去的輪廓。他想伸手，但手臂完全無法移動。

**李杰**：(內心) 拜託...拜託活下去...

**旁白**：他的意識開始消散，黑暗逐漸吞噬了他的視野。最後一個念頭是——

**李杰**：(內心) 對不起...我又失敗了...

**旁白**：然後，一切陷入黑暗。

---

**旁白**：不知道過了多久。李杰的意識在黑暗中漂浮，然後緩緩甦醒。熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧？我的手機完全沒訊號！連「圈外」都不是，是「沒有服務」耶！這底下是多深啊？我從來沒見過這種狀況...

**田中健太**：我的也是，什麼都傳不出去。LINE、IG、什麼都不行。喂，這隧道也太長了吧？以前有這麼長嗎？我怎麼一點印象都沒有？

**旁白**：李杰睜開眼睛。他坐在搖晃的車廂裡，窗外是純粹的黑暗。

**李杰**：(內心) 又...又回來了...

**旁白**：他低頭看向自己的胸口。衣服完好無損，沒有血跡，沒有傷口。就像剛才被黑劍刺穿心臟、失血、死亡...所有的痛苦都沒有發生過。

**旁白**：但這一次——李杰沒有站起來。他只是坐在那裡，雙手無力地垂在身側，眼神空洞，像是失去了所有的光彩。

**李杰**：(內心) ...我受夠了...

**旁白**：他感到一陣深深的疲憊，從靈魂深處湧出的疲憊。

**李杰**：(內心) 第一次...中村用手槍殺死了我...第二次...健太拿到劍...被劍控制...第三次...伊藤誠拿到劍...第四次...斎藤編造謊言，拿到劍...我看到健太他們的屍體...每一次...每一次都失敗...

**旁白**：他緩緩地癱坐在座位上，雙眼無神地看著天花板。

**李杰**：(內心) 我根本...無法阻止那把劍...它會誘惑任何人...控制任何人...不管我做什麼...總會有人拿起它...然後...然後就是災難...

**旁白**：車廂裡的其他乘客注意到了李杰的異樣。健太和明日香停止了對話，轉頭看向他。拓海和美咲也停下動作，用奇怪的眼神打量著他。

**田中健太**：那個...你還好嗎？

**旁白**：李杰沒有回應，他只是坐在那裡，像是一具失去靈魂的軀殼。

**鈴木明日香**：他看起來...好像很累...

**佐藤拓海**：是不是身體不舒服？

**旁白**：所有人的目光都聚焦在李杰身上，但李杰對這些視線完全沒有反應。

**李杰**：(內心) 就算我再努力...也只會重複同樣的結局...就算我被殺...也只會再次循環...一次又一次...永無止境...我...我不想再試了...

**旁白**：他閉上眼睛，深深地嘆了口氣。絕望像黑暗的潮水一樣淹沒了他。

**旁白**：就在這時——車廂連接處的門打開了。凌墨從第三節車廂走了進來，掃視了一眼車廂裡的所有人，目光最後落在李杰身上。他走向李杰，眉頭微皺。

**凌墨**：你...還好嗎？

**旁白**：李杰沒有回應，他連看都沒有看凌墨一眼，只是繼續呆坐著。

**凌墨**：喂...

**旁白**：凌墨伸手想要碰李杰的肩膀——就在這一瞬間——一個聲音，突然在李杰的腦海中響起。那不是人類的聲音，低沉、詭異、帶著一絲嘲諷。

**？？？**：放棄吧...

**旁白**：李杰猛地睜開眼睛。

**李杰**：(內心) 什麼...？

**？？？**：你永遠...無法戰勝我...

**旁白**：那個聲音在他的腦海中迴盪，像是從遙遠的深淵傳來。

**？？？**：無論你做什麼...無論你嘗試多少次...總會有人...拿起我...然後...屠殺所有人...

**旁白**：李杰的身體開始顫抖。那是黑劍的聲音。

**？？？**：你看到了吧...田中...伊藤...斎藤...他們都成為了我的宿主...你以為你能阻止我嗎？

**旁白**：聲音中帶著濃濃的嘲諷。

**？？？**：可笑...你只是一個凡人...在無盡的時間循環中掙扎...而我...是不朽的...只要這把劍存在...就永遠會有人被我誘惑...永遠...

**旁白**：李杰的雙手緊緊握成拳頭，指甲嵌入掌心。但他說不出話，因為他知道——那個聲音說的是對的。

**旁白**：車廂裡的其他人看著李杰，臉上寫滿困惑和不安。凌墨的手停在半空中，不知道該說什麼。

**旁白**：而李杰——只是低著頭，身體微微顫抖，陷入了前所未有的絕望深淵。
