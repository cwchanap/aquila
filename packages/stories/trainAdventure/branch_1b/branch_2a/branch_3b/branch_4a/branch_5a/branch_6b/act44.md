# 第四十四幕：強者與弱者

**旁白**：不知道過了多久。列車緩緩減速，發出一聲長長的汽笛聲。「嗚——」「咔嚓——」車門打開了。車廂裡的氣氛變得緊張起來。

**山本浩二**：車門...開了...

**小林雄大**：我們...要下車嗎...？

**小林智子**：這裡...到底是什麼地方...

**旁白**：所有人都看向車門外，猶豫著該不該下車。就在這時——沉重的腳步聲從車廂連接處傳來。「咔——咔——咔——」

**旁白**：車廂連接處的門被推開，一個身影走了進來。

**旁白**：中村隼人。他手中握著那把黑色的劍，劍身上沾著鮮紅的血跡，在昏暗的燈光下閃爍著詭異的光芒。

**旁白**：他的眼神冰冷而平靜，臉上帶著一絲笑容。

**山本浩二**：中村...先生...？

**小林雄大**：那把劍...是...

**旁白**：中村沒有回答，他只是靜靜地看著車廂裡的所有人，嘴角的笑容越來越大。「唰——」黑劍劃過空氣。

**山本浩二**：什——

**旁白**：劍刃刺穿了山本的胸口，鮮血噴濺而出。

**小林智子**：啊啊啊啊——！

**小林陽菜**：媽媽——！

**小林翔太**：爸爸——！

**清水恵美**：不...不要...

**旁白**：尖叫聲在車廂裡爆發，所有人開始慌亂地逃竄。

**小林雄大**：快跑！大家快跑！

**旁白**：雄大抱起翔太，拉著智子和陽菜，朝著車門衝去。伊藤夫婦也跟著跑向車門。清水恵美緊跟在伊藤夫婦後面。

**旁白**：田中健太、鈴木明日香、佐藤拓海、高橋美咲和斎藤蓮緊跟在後。

**旁白**：中村舉起黑劍，追向逃跑的人群。「唰——」

**小林智子**：嗚——！

**旁白**：智子的背後被劍刺穿，她倒在車門前。

**小林雄大**：智子——！

**旁白**：雄大的眼睛瞬間充血，臉上的表情從驚恐轉為憤怒。

**小林雄大**：你這個混蛋——！

**旁白**：他放下翔太，朝著中村衝去。

**小林陽菜**：爸爸——！不要——！

**旁白**：陽菜哭著抓住父親的衣服。

**小林翔太**：爸爸——！

**旁白**：翔太也哭著抱住父親的腿。

**旁白**：雄大停下腳步，低頭看著兩個孩子。他們眼中充滿恐懼和絕望。他咬緊牙關，眼淚從眼眶中流出。

**小林雄大**：該死...該死...！

**旁白**：他蹲下身，抱起翔太。

**小林雄大**：陽菜！快跟上！

**旁白**：他抱著翔太朝著車門衝去。陽菜跟在後面，但她的腳步太慢了。伊藤誠拉著奈々跳下了列車。

**旁白**：斎藤蓮推開其他人，也跳下了列車。雄大抱著翔太跳下列車，消失在黑暗中。

**清水恵美**：等等——！

**旁白**：恵美伸手想抓住車門，但她慢了一步。

**旁白**：「咔嚓——」車門突然關上。「嗚——」列車開始緩緩移動。

**小林陽菜**：爸爸——！爸爸——！

**旁白**：陽菜跪在車門前，拍打著車門，哭喊著父親。她差一步就能跳下列車，但車門關上得太快了。

**清水恵美**：不...不...怎麼會...

**旁白**：恵美跪坐在地上，雙手顫抖著。

**田中健太**：該死...門關上了...

**鈴木明日香**：怎麼辦...我們要怎麼辦...

**旁白**：車廂裡剩下的人驚恐地看著中村。除了李杰——他依然坐在原位，眼神空洞，對周圍的一切毫無反應。

**旁白**：中村舉起黑劍，嘴角露出冰冷的笑容。

**中村隼人**：接下來...輪到你們了。

**旁白**：「唰——」「唰——」「唰——」「唰——」「唰——」「唰——」

**旁白**：尖叫聲再次響起，但很快就停止了。

**旁白**：小林陽菜，倒在母親身旁。清水恵美，倒在車門前。田中健太和鈴木明日香，倒在一起。佐藤拓海和高橋美咲，倒在座位旁。

**旁白**：車廂裡，只剩下幾具屍體。山本浩二，胸口被劍刺穿，倒在血泊中。小林智子，背後有一道致命的劍傷。小林陽菜，小小的身體躺在母親旁邊。清水恵美，倒在車門前。

**旁白**：田中健太和鈴木明日香，倒在車廂中央。佐藤拓海和高橋美咲，倒在座位旁。還有——李杰。

**旁白**：他依然坐在原位，眼神空洞，沒有任何反應。即使剛才就在他面前發生了屠殺，即使尖叫聲和血腥味充斥著整個車廂——他連看都沒看一眼。

**李杰**：(內心) ...

**旁白**：他的內心一片空白。沒有恐懼，沒有悲傷，沒有憤怒。只有空洞。

**旁白**：中村收起黑劍，轉身看向李杰。他走向李杰，在他對面的座位上坐下。他的衣服上沾著血跡，但他的表情依然平靜。

**中村隼人**：感覺如何？

**旁白**：李杰沒有回應，他只是靜靜地坐在那裡，像是沒有聽到中村的話。

**中村隼人**：呵...看來你已經放棄了。

**旁白**：中村靠在座位上，臉上帶著滿意的笑容。

**中村隼人**：也好。這樣你就不會礙事了。

**旁白**：他開始自顧自地說話。

**中村隼人**：我接下來的計畫...很簡單。先去征服那個精靈村。他們有很多稀有的資源...魔法道具、神樹的果實、還有這把劍...

**中村隼人**：然後我會找到回到原來世界的方法。到時候...我就可以把這些稀有資源帶回去...跟地球上的有錢人交易...

**中村隼人**：想想看...一個擁有魔法力量的世界的資源...那些富豪會付出多少錢？

**旁白**：他的眼中閃爍著貪婪的光芒。

**中村隼人**：我會成為世界上最富有的人...

**旁白**：李杰依然沒有反應。但就在這時——一個念頭，突然閃過他的腦海。

**李杰**：...為什麼...

**旁白**：李杰的聲音虛弱而沙啞，像是很久沒有說話。

**中村隼人**：嗯？

**旁白**：中村轉頭看向他，眼中帶著一絲驚訝。

**李杰**：為什麼...你沒有被劍控制...？

**旁白**：這是一個李杰一直想不通的問題。健太、伊藤、斎藤...所有拿起黑劍的人，都被劍控制了。他們的眼睛會燃燒著紅色的火焰，完全失去自我。

**旁白**：但中村...他的眼睛裡沒有紅色的火焰，他完全清醒，完全理智。

**中村隼人**：哈哈哈哈——

**旁白**：中村突然大笑起來。

**中村隼人**：你問這個？很簡單啊...因為我比他們強。

**旁白**：他的語氣中充滿自信和傲慢。

**中村隼人**：只有弱者...才會被這把劍控制。那些軟弱的人...沒有足夠的意志力...所以他們會被劍的力量吞噬...

**中村隼人**：但我不一樣。我的意志...比這把劍更強。所以我能夠駕馭它...而不是被它駕馭。

**旁白**：李杰的眼神，第一次有了一絲變化。一道光芒，在他空洞的眼神中閃過。

**李杰**：(內心) 意志...只有弱者才會被控制...如果意志夠強...就能克服黑劍的誘惑...原來如此...

**旁白**：這個答案，就像一道閃電，劈開了籠罩在他心中的黑暗。他終於明白了。不是黑劍無法被阻止，而是拿起它的人，意志不夠堅定。

**旁白**：如果有一個意志堅定的人...就能駕馭黑劍，而不是被它控制。

**李杰**：...為什麼...

**旁白**：李杰抬起頭，第一次直視中村的眼睛。

**李杰**：你不殺我...？

**中村隼人**：嗯？

**旁白**：中村挑了挑眉。

**中村隼人**：哦...這個啊...因為我可能還需要你幫忙。萬一...我需要什麼資訊...或者需要一個人質...留著你比較方便。

**旁白**：他站起身來，走向李杰。

**中村隼人**：不過現在...你先睡一會兒吧。

**旁白**：他舉起拳頭。「砰——」一拳打在李杰的下巴上。李杰的頭向後仰，眼前一黑，他失去了意識。

**旁白**：但在失去意識之前——他的心中，第一次燃起了一絲希望。

**李杰**：(內心) 我知道該怎麼做了...

**旁白**：然後，黑暗吞噬了他。
