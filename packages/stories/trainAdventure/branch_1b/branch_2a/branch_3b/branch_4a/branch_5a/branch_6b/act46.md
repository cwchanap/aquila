# 第四十六幕：真相與背叛

**旁白**：莉拉看到李杰，眼神閃過一絲複雜的情緒。她張了張嘴，似乎想說什麼，但最終只是咬緊嘴唇，低下了頭。

**旁白**：斎藤蓮看到李杰的目光，眼鏡片後的眼睛閃躲了一下，立刻移開視線。

**旁白**：李杰深吸一口氣，看向中村。

**李杰**：發生了什麼事？

**旁白**：他的聲音平靜，但帶著一絲壓抑的憤怒。

**中村隼人**：發生了什麼？哈哈...

**旁白**：中村笑了，聲音中充滿了得意。

**中村隼人**：很簡單啊。我殺了所有剩下的乘客，還有那個精靈長老——艾拉如。

**旁白**：李杰的拳頭緊緊握住。

**中村隼人**：多虧了你，我才知道那個老頭有能力擊敗這把劍。

**旁白**：他舉起手中的黑劍，劍身在昏暗的光線下閃爍著不祥的光澤。

**中村隼人**：所以我找到了躲在精靈村的那些乘客，還找到了這個膽小鬼。

**旁白**：他用劍指了指斎藤。

**中村隼人**：有了他的幫助，我輕鬆殺了那個老頭和所有乘客，征服了整個村子。

**旁白**：莉拉猛地抬起頭，眼中充滿憤怒。她狠狠地瞪著斎藤和中村，雙手緊緊握成拳頭。

**旁白**：李杰注意到莉拉的反應，心中一震。

**李杰**：等等...你說「多虧了我」...是什麼意思？

**旁白**：李杰皺起眉頭，聲音中帶著困惑。

**李杰**：我沒有告訴過你任何關於艾拉如長老的事...

**旁白**：中村冷笑一聲，但沒有回答。他只是搖了搖頭。

**中村隼人**：無所謂了。反正一切都結束了。

**旁白**：他轉身，看向莉拉。

**中村隼人**：我要回到地球了，帶著這位精靈女士。

**旁白**：莉拉的臉色更加蒼白，但她沒有說話，只是咬緊嘴唇。

**中村隼人**：至於你...

**旁白**：他看向斎藤。

**中村隼人**：你跟著我。我需要你作為祭品，來維持通往地球的門。

**旁白**：斎藤的臉色瞬間變得慘白。

**斎藤蓮**：什...什麼？這...這不是你之前說的...！

**旁白**：他的聲音顫抖著，眼鏡後的眼睛睜得很大。

**斎藤蓮**：你說...你說只要我幫你...你就會帶我一起回地球...！

**旁白**：中村沒有回答，只是舉起黑劍，劍尖直指斎藤的喉嚨。

**中村隼人**：你可以跟著我，作為祭品，還是現在就死在這裡。選一個吧。

**旁白**：斎藤的臉色變得更加蒼白。他的嘴唇顫抖著，想說什麼，但最終什麼都沒說出來。

**旁白**：他緩緩後退一步，低下了頭。

**斎藤蓮**：我...我跟你走...

**旁白**：他的聲音小得幾乎聽不見。

**旁白**：李杰看著這一幕，憤怒在胸中燃燒。他的拳頭緊緊握住，指甲深深掐進掌心。

**旁白**：他想衝上去，想一拳打在中村的臉上。但他知道自己做不到。

**旁白**：中村手中的黑劍，讓他毫無勝算。

**中村隼人**：好了，我要去祭壇了。

**旁白**：中村收回黑劍，轉身看向李杰。

**中村隼人**：你要是有興趣的話，可以跟著我。說不定能看到些有趣的東西。

**旁白**：他說完，轉身朝著左側的走廊走去。莉拉跟在他身後，步伐沉重。斎藤猶豫了一下，也跟了上去。

**旁白**：李杰站在原地，看著他們三個人的背影。

**旁白**：他的腦海中飛快地思考著。

**李杰**：(內心) 跟上去...還是留在這裡...？

**李杰**：(內心) 如果跟上去...也許能找到機會...也許能知道更多...

**李杰**：(內心) 但留在這裡...什麼都做不了...

**旁白**：他咬緊牙關，做出了決定。

**旁白**：他邁開步伐，朝著左側的走廊走去，跟在他們三人身後。

**旁白**：走廊裡只剩下腳步聲，在黑暗中迴盪。

**旁白**：李杰保持著一段距離，跟在最後面。他只能看到前方三個人的背影。

**旁白**：中村走在最前面，手中握著黑劍。莉拉走在中間，低著頭。斎藤跟在後面，步伐踉蹌。

**旁白**：黑劍在中村手中，散發著不祥的黑色光芒。

**旁白**：他們朝著祭壇的方向，一步一步地走去。
