# 第四十七幕：祭壇的真相

**旁白**：他們走進左側的走廊。

**旁白**：穿過長長的走廊，下了陡峭的石階。周圍的藍色寶石發出微弱的光芒。

**旁白**：不久後，他們來到那扇巨大的石門前。中村推開石門，金色的光芒從門縫中傾瀉而出。

**旁白**：門後是一個極其寬敞的空間。在空間的正中央，矗立著一座白色大理石祭壇。

**旁白**：祭壇上，躺著一個身穿純白色長袍的男子。

**旁白**：李杰的目光立刻被吸引過去。

**李杰**：(內心) 翔...

**旁白**：那個白袍男子安詳地躺在祭壇上，雙手交疊放在胸前。

**旁白**：但他的嘴角，流出大量暗紅色的血跡。血液已經凝固，沿著臉頰流到耳邊，在白色的長袍上留下觸目驚心的痕跡。

**李杰**：(內心) 他的身體...果然和白劍一起死亡了...

**旁白**：中村走向祭壇，看了一眼翔的屍體。

**中村隼人**：把他移開。

**旁白**：他看向斎藤和莉拉。

**旁白**：莉拉咬緊嘴唇，走上前。斎藤猶豫了一下，也跟了上去。

**旁白**：他們兩個人合力抬起翔的屍體，將他放在祭壇旁邊的地面上。

**旁白**：中村看向斎藤。

**中村隼人**：躺上去。

**旁白**：斎藤的臉色變得蒼白。他看著祭壇，眼中充滿恐懼。

**斎藤蓮**：我...我真的要...

**中村隼人**：躺上去。

**旁白**：中村的聲音冰冷而堅定。

**旁白**：斎藤顫抖著，慢慢爬上祭壇。他躺了下來，全身僵硬。

**旁白**：中村轉身，看向祭壇後方。

**旁白**：那裡有一個高大的拱形門框，表面刻滿了精緻的符文。

**旁白**：但門框裡面...什麼都沒有。只是空蕩蕩的黑暗。

**中村隼人**：好了...接下來...

**旁白**：他的聲音中充滿期待。他的眼睛緊緊盯著那個門框，等待著它變成閃光的門。

**旁白**：時間一分一秒地過去。

**旁白**：但什麼都沒有發生。

**旁白**：門框依然是空蕩蕩的黑暗。

**旁白**：中村的表情開始變得困惑。

**中村隼人**：為什麼...為什麼沒有亮...

**旁白**：他走向祭壇，低頭看著斎藤。

**中村隼人**：你有好好躺著嗎？

**斎藤蓮**：我...我照你說的做了...！

**旁白**：斎藤的聲音顫抖著。

**斎藤蓮**：我就是躺在這裡...我什麼都沒做錯...！

**旁白**：中村的表情越來越陰沉。他緊緊握住手中的黑劍。

**中村隼人**：該死...到底是怎麼回事...

**旁白**：他舉起黑劍，盯著劍身。

**中村隼人**：為什麼門沒有打開？！

**旁白**：他的聲音在空曠的空間中迴盪。

**旁白**：然後，他似乎聽到了什麼。他的表情變得更加陰沉。

**旁白**：李杰站在遠處，看著中村。他看不到黑劍說了什麼，但他能看到中村的臉色越來越難看。

**旁白**：突然，中村的臉上露出憤怒的表情。

**中村隼人**：你騙我？！

**旁白**：他的聲音充滿了暴怒。

**中村隼人**：你根本沒告訴我...門需要白劍的存在才能保持打開！

**旁白**：他舉起黑劍，劍尖指向天空。

**中村隼人**：你從一開始就在騙我！

**旁白**：就在這時——

**旁白**：中村的眼睛開始燃燒起來。

**旁白**：紅色的火焰，從他的瞳孔中湧出。

**旁白**：斎藤和莉拉同時感覺到不對勁。他們的臉色變得慘白。

**斎藤蓮**：不...不對...

**旁白**：斎藤立刻從祭壇上跳下來。

**莉拉**：快跑——！

**旁白**：莉拉轉身想要逃跑。

**旁白**：但中村的動作更快。

**旁白**：「唰——！」

**旁白**：黑劍橫掃而過。

**旁白**：斎藤的身體從腰部被斬成兩半。上半身和下半身分離，倒在地上。

**旁白**：「唰——！」

**旁白**：莉拉也沒能逃掉。她的身體同樣被斬成兩半，倒在血泊中。

**旁白**：李杰看著這一幕，全身僵硬。

**旁白**：但他的心中...卻出奇地平靜。

**李杰**：(內心) 我要死了...

**李杰**：(內心) 但這正是我想要的...

**旁白**：他的腦海中閃過之前的一切。中村的話，黑劍的秘密，意志力的關鍵。

**李杰**：(內心) 我知道了...我終於知道該怎麼對付這把劍了...

**旁白**：中村轉身，眼中的紅色火焰更加猛烈。

**旁白**：他看向李杰，嘴角露出扭曲的笑容。

**中村隼人**：輪到你了...

**旁白**：他舉起黑劍，朝著李杰走來。

**旁白**：李杰沒有逃跑。他只是站在原地，靜靜地看著中村。

**旁白**：黑劍在空中劃過。

**旁白**：「唰——！」

**旁白**：劇烈的疼痛瞬間襲來。

**旁白**：李杰的身體倒了下去。視線越來越模糊。

**旁白**：他看到天花板上的壁畫，看到金色的光芒在旋轉。

**旁白**：然後，一切都陷入了黑暗。

**旁白**：意識消失了。
