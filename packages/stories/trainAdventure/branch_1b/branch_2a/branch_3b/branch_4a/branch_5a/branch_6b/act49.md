# 第四十九幕：尋找候選人

**旁白**：不知道過了多久。

**旁白**：李杰的意識在黑暗中漂浮，然後緩緩甦醒。

**旁白**：熟悉的聲音傳入耳中。

**鈴木明日香**：欸...不是吧?我的手機完全沒訊號!連「圈外」都不是,是「沒有服務」耶!這底下是多深啊?我從來沒見過這種狀況...

**田中健太**：我的也是,什麼都傳不出去。LINE、IG、什麼都不行。喂,這隧道也太長了吧?以前有這麼長嗎?我怎麼一點印象都沒有?

**旁白**：李杰睜開眼睛。

**旁白**：他坐在搖晃的車廂裡,窗外是純粹的黑暗。

**李杰**：(內心) 又回來了...

**旁白**：他低頭看向自己的身體。沒有傷口,沒有血跡。一切都恢復了。

**李杰**：(內心) 第七次了...

**旁白**：他深吸一口氣,閉上眼睛,試圖回想剛才發生的事。

**李杰**：(內心) 我拿起了黑劍...然後...失去了意識...

**李杰**：(內心) 之後發生了什麼...我完全不記得...

**李杰**：(內心) 但既然我又回到了這裡...肯定不是什麼好事...

**旁白**：他用力按了按太陽穴。精神上的疲憊越來越沉重。

**旁白**：但他沒有時間休息。他站起身,朝著車廂後方走去。

**旁白**：穿過一節又一節車廂,來到第一節車廂。

**旁白**：不久後,凌墨從第二節車廂走了進來。

**旁白**：凌墨看到李杰,停下腳步。他的表情有些困惑,但眼神中閃爍著某種認知。

**凌墨**：你是...

**旁白**：他皺起眉頭,像是在努力回想什麼。

**凌墨**：我腦海中出現了一些模糊的記憶...

**凌墨**：我記得...有人告訴我...後面車廂有一把黑色的劍...

**凌墨**：那把劍很危險...會控制人的心智...

**旁白**：李杰點了點頭。

**李杰**：你說得對。那把黑劍在第十節車廂。

**李杰**：但我現在想...之前中村說過,他的意志力夠強,可以駕馭那把劍。

**李杰**：也許...真的需要一個意志力足夠強的人才能控制它。

**李杰**：一個能夠駕馭它、克服它控制的人。

**旁白**：凌墨看著李杰,沉默了片刻。

**凌墨**：我可以試試。

**旁白**：李杰看向凌墨,眼神中閃過一絲猶豫。

**李杰**：你確定？那把劍很危險。

**凌墨**：我知道。但如果沒有人去嘗試,情況只會更糟。

**旁白**：李杰思考了幾秒,最後點了點頭。

**李杰**：好。小心點。

**旁白**：凌墨點了點頭,轉身朝著車廂後方走去。

**旁白**：李杰看著凌墨的背影消失在第二節車廂的門後。

**旁白**：然後,他轉身,看向第一節車廂裡的其他乘客。

**旁白**：那些大學生們還在聊天。明日香和健太在抱怨手機沒有訊號,拓海在翻閱雜誌。

**旁白**：李杰的目光掃過每一個人。

**李杰**：(內心) 如果凌墨失敗了...誰會是下一個候選人？

**旁白**：他看向健太。

**李杰**：(內心) 健太...不行。我在之前的循環中,看到他被黑劍控制了。他太膽小,意志力不夠。

**旁白**：他看向拓海。

**李杰**：(內心) 拓海...也不適合。他雖然沉穩,但我看到他也被控制過。缺乏足夠的決斷力。

**旁白**：他看向斎藤。那個戴眼鏡的男生正坐在角落,低頭看著手機。

**李杰**：(內心) 斎藤...更不行。我親眼看到他被黑劍控制。他故作聰明,但本質上很膽小。

**旁白**：他的視線移向其他車廂。

**李杰**：(內心) 伊藤夫婦...他們太焦慮了。我看到過他們被控制的樣子。容易被情緒支配。

**旁白**：然後,他想到了另外兩個人。

**李杰**：(內心) 山本浩二...他沉穩有擔當...

**李杰**：(內心) 還有小林雄大...他是家庭的支柱,意志力應該很強...

**李杰**：(內心) 如果凌墨失敗...他們兩個也許可以試試...

**旁白**：就在這時,李杰注意到中村隼人。

**旁白**：中村坐在座位上,目光在車廂裡的乘客身上掃來掃去,像是在觀察著什麼。

**旁白**：他的眼神銳利而陰沉,嘴角帶著一絲若有似無的冷笑。

**旁白**：那種表情...讓人感覺他正在盤算著什麼見不得人的計謀。

**李杰**：(內心) 中村...這次居然沒有行動...

**李杰**：(內心) 但他那種眼神...肯定在計劃著什麼...

**旁白**：李杰皺起眉頭,心中升起一股不安。

**旁白**：時間一分一秒地過去。

**旁白**：列車繼續在黑暗的隧道中疾駛。

**旁白**：不知過了多久——

**旁白**：窗外的黑暗終於開始改變。

**旁白**：遠處出現了微弱的光芒。

**旁白**：然後,列車開始減速。

**旁白**：列車停了下來。

**旁白**：車廂裡的乘客們開始騷動起來。

**山本浩二**：我們⋯⋯下去看看嗎？也許車站裡有人⋯⋯

**伊藤誠**：你瘋了嗎？外面是什麼鬼地方！誰知道會遇到什麼！

**中村隼人**：對！待在車上比較安全！萬一車門關上,列車開走了怎麼辦！

**鈴木明日香**：可是⋯⋯待在這裡也不知道會發生什麼啊⋯⋯

**田中健太**：我⋯⋯我也不知道該怎麼辦⋯⋯

**高橋美咲**：怎麼辦怎麼辦,到底該怎麼辦⋯⋯

**旁白**：李杰沒有理會周圍的騷動。他轉身,快速朝著車廂後方走去。

**李杰**：(內心) 凌墨還沒回來...

**李杰**：(內心) 我得去看看發生了什麼...

**旁白**：他穿過一節又一節車廂。

**旁白**：第二節,第三節,第四節...

**旁白**：每一節車廂裡的乘客都在驚慌失措地討論著窗外的詭異景象。

**旁白**：但李杰完全不在意。他的目標只有一個——

**旁白**：第十節車廂。

**旁白**：終於,他推開了第十節車廂的門。

**旁白**：然後——

**旁白**：他看到了眼前的景象。

**旁白**：凌墨躺在車廂地板上,一動也不動。

**旁白**：他的右手緊緊握著那把黑劍,劍身散發著詭異的黑色光芒。

**旁白**：凌墨的臉色蒼白,雙眼緊閉,失去了意識。

**李杰**：凌墨——！

**旁白**：李杰衝上前,跪在凌墨身邊。

**旁白**：但凌墨毫無反應。他的呼吸很微弱,像是陷入了極深的昏迷。

**旁白**：而他手中的黑劍,依然散發著不祥的氣息。

**李杰**：(內心) 失敗了...

**李杰**：(內心) 凌墨也無法控制這把劍...
