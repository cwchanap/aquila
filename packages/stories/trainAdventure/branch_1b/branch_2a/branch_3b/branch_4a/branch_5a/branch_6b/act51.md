# 第五十一幕：紅眼的凌墨

**李杰**：清水！快跑！

**旁白**：李杰的喊聲撕破了森林的寂靜。

**旁白**：恵美猛地一驚,像是突然被驚醒。她看了一眼正朝自己走來的凌墨,臉上閃過極度的恐懼。

**清水恵美**：啊——！

**旁白**：她轉身就跑,朝著來時的小路拼命逃去。她的腳步慌亂,幾乎是連滾帶爬地消失在樹林深處。

**旁白**：李杰沒有猶豫。他立刻衝向凌墨,抓住他的肩膀。

**李杰**：凌墨！醒醒！是我！

**旁白**：李杰用力搖晃著凌墨的肩膀。凌墨的身體隨著搖晃而晃動,但他依然沒有反應。

**李杰**：凌墨！聽得到我說話嗎？！快醒過來！

**旁白**：就在這時——

**旁白**：凌墨突然停止了晃動。

**旁白**：他緩緩轉過頭,看向李杰。

**旁白**：李杰的手僵在半空中。

**旁白**：凌墨的眼睛...變了。

**旁白**：原本黑色的瞳孔,此刻變成了詭異的血紅色。那雙眼睛冰冷而殘忍,像是在注視著一個獵物。

**李杰**：(顫抖地) 凌墨...？

**旁白**：凌墨的嘴角緩緩揚起一個扭曲的笑容。

**凌墨？**：真是麻煩...

**旁白**：那個聲音,雖然是凌墨的聲音,但語氣卻完全不同。冰冷、殘酷、充滿惡意。

**凌墨？**：為什麼這個身體...這麼難控制？

**凌墨？**：明明他也是我們其中一員...卻還能抵抗到這種程度...

**旁白**：李杰猛地鬆開手,後退了一步。

**李杰**：(內心) 凌墨...被控制了...

**李杰**：(內心) "我們其中一員"...

**旁白**：那句話讓李杰腦海中閃過一個記憶。

**旁白**：在上一次循環,當他握住黑劍時,那個聲音曾經對他說過——

**旁白**："因為...你是我們其中一員..."

**李杰**：(內心) 黑劍說我是"其中一員"...現在又說凌墨"也是其中一員"...

**李杰**：(內心) 我和凌墨...都是...什麼？

**旁白**：然而,另一個念頭突然閃過李杰的腦海。

**李杰**：(內心) 等等...清水剛才...她叫我"李先生"...

**李杰**：(內心) 但是...我們從來沒有交談過...

**李杰**：(內心) 在這次循環裡,我從一開始就沒有跟她說過話...

**李杰**：(內心) 她怎麼會知道我的名字？！

**旁白**：但凌墨？沒有給李杰更多思考的時間。

**旁白**：他猛地舉起手中的黑劍。

**李杰**：——！

**旁白**：李杰想要躲開,但太遲了。

**旁白**：黑劍在空氣中劃出一道黑色的弧線。

**旁白**：冰冷的劍刃,精準地劃過李杰的喉嚨。

**旁白**：一道深深的傷口在他的脖子上綻開。鮮血噴湧而出,染紅了他的衣領。

**李杰**：(捂住喉嚨,發不出聲音) 唔...

**旁白**：劇烈的疼痛襲來。他感覺自己的呼吸被切斷了,空氣無法進入肺部。

**凌墨？**：哈哈哈...繼續掙扎吧...

**凌墨？**：在這無限的時間循環裡...繼續掙扎吧...

**旁白**：凌墨？的笑聲冰冷而殘酷。他轉身,朝著恵美逃走的方向走去。

**旁白**：李杰的雙腿一軟,跪倒在地上。他用手緊緊捂住喉嚨,但血液還是不斷從指縫間湧出。

**旁白**：他的視線開始模糊。森林、樹木、紫色的月光...一切都在旋轉。

**李杰**：(內心) 清水...快逃...

**李杰**：(內心) 拜託...一定要逃掉...

**旁白**：李杰的身體向前倒下,臉頰貼在冰冷的泥土地上。

**旁白**：遠處,傳來恵美驚恐的尖叫聲。

**旁白**：李杰想要伸手,想要站起來,想要去幫她。

**旁白**：但他的身體已經不聽使喚了。

**旁白**：力量從他的身體中流失,連同那不斷湧出的鮮血。

**旁白**：視線越來越暗。

**旁白**：最後,只剩下一片黑暗。

**旁白**：意識...消失了。
