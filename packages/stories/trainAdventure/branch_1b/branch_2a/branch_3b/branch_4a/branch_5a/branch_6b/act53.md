# 第五十三幕：黑色的轉移

**李杰**：(看著痛苦的凌墨)大叔,我們得先去第十節車廂。凌墨現在這樣...我們也幫不了他。

**山本浩二**：但是...把他一個人留在這裡？

**李杰**：我們必須先確保黑劍的安全。如果被其他人拿走...

**旁白**：山本浩二看了凌墨一眼,然後點了點頭。

**山本浩二**：好。我們快去快回。

**旁白**：兩人立刻衝向第九節車廂,穿過連接處,進入第十節車廂。

**旁白**：車廂裡空蕩蕩的,只有那把黑劍靜靜地躺在地板中央。它散發著詭異的黑色光芒,像是在等待著什麼。

**李杰**：就是那個...

**山本浩二**：(走上前)我來。

**旁白**：山本浩二深吸一口氣,伸手握住了劍柄。

**旁白**：黑色的光芒瞬間爆發,像是活物一般纏繞上他的手臂。山本浩二的身體僵硬了一瞬,額頭上青筋暴起。

**李杰**：大叔！

**山本浩二**：我...沒事...

**旁白**：他緊緊握著劍,呼吸急促,但眼神依然清明。

**李杰**：你還好嗎？！感覺怎麼樣？

**山本浩二**：(喘著氣)有個...很煩人的聲音...在我腦子裡...

**山本浩二**：它在催促我...殺了所有人...

**旁白**：李杰的心臟一沉。

**李杰**：你...

**山本浩二**：(搖頭)但我不會做的。我很清楚...我絕對不會做那種事。

**旁白**：他的聲音堅定,眼神沒有一絲動搖。

**李杰**：(鬆了口氣)太好了...你能控制它...跟中村一樣...

**山本浩二**：中村？

**李杰**：沒什麼。我們快回去看看凌墨。

**旁白**：兩人快速返回第八節車廂。凌墨依然躺在地板上,身體劇烈地抽搐著,尖叫聲從未停止。

**凌墨**：啊啊啊——！！

**李杰**：凌墨...

**山本浩二**：我們該怎麼辦？

**李杰**：先試試能不能叫醒他...

**旁白**：山本浩二握著黑劍,走向凌墨。

**旁白**：就在他靠近的瞬間——

**旁白**：黑劍突然劇烈震動起來。黑色的氣息從劍身上湧出,像是找到了目標一般,直接撲向凌墨。

**李杰**：什麼——？！

**旁白**：黑色氣息鑽入凌墨的身體。他的尖叫聲戛然而止,身體停止了抽搐。

**旁白**：凌墨的眼睛睜開了一瞬,瞳孔中閃過一絲詭異的紅光,然後他失去了意識,軟軟地倒在地上。

**李杰**：凌墨！

**旁白**：李杰立刻衝上前,跪在凌墨身邊。他檢查凌墨的呼吸——還在。脈搏也正常。但凌墨完全失去了意識。

**山本浩二**：這...這是怎麼回事？

**旁白**：山本浩二看著手中的黑劍。劍身上的黑色光芒完全消失了,變成了一把普通的、灰色的舊劍。

**山本浩二**：那個聲音...消失了。

**李杰**：(抬頭看向山本)什麼？

**山本浩二**：剛才那個催促我殺人的聲音...完全消失了。這把劍...好像變成普通的劍了。

**旁白**：李杰看著失去意識的凌墨,又看著變得普通的黑劍。一股不安的感覺湧上心頭。

**李杰**：(內心)黑劍的力量...轉移到凌墨身上了？

**李杰**：(內心)這不對...這絕對不對...

**李杰**：大叔,我們得先把凌墨帶回第一節車廂。

**山本浩二**：好。

**旁白**：李杰彎下腰,用力將凌墨背到背上。凌墨的身體很沉,但李杰咬緊牙關,穩住了腳步。

**李杰**：走吧。

**旁白**：兩人開始朝著第一節車廂走去。李杰背著凌墨,山本浩二握著那把已經變得普通的劍。

**旁白**：他们穿过第七节、第六节、第五节...

**旁白**：就在他们快要到達第一節車廂時,李杰突然感覺到列車開始減速。

**李杰**：(內心)到了...

**旁白**：窗外的黑暗終於出現了光亮。不是熟悉的城市燈光,而是詭異的紫色月光。

**旁白**：列車緩緩駛入月台,停了下來。車門自動打開,冰冷的風吹進車廂。

**旁白**：李杰背著凌墨,山本浩二握著劍,兩人推開連接處的門,走進第一節車廂。

**旁白**：所有乘客的目光瞬間集中在他們身上。

**田中健太**：欸？那是...劍？！

**鈴木明日香**：你們...你們在幹什麼？！

**中村隼人**：(站起身,指著山本)你拿著那把劍想做什麼？！

**旁白**：車廂內瞬間充滿了緊張和恐慌。

**小林雄大**：(站起身,擋在家人面前)大家冷靜！

**山本浩二**：請聽我說——

**旁白**：但就在這時,一個聲音從車廂後方傳來。

**清水恵美**：那個人...怎麼了...？

**旁白**：所有人轉頭看向恵美。她的臉色蒼白,眼神震驚地盯著李杰背上的凌墨。她的手緊緊抓著座椅的扶手,指節發白。

**李杰**：(內心)清水...她認識凌墨...但她在裝作不認識...

**旁白**：李杰看著恵美的表情——那種擔憂無法完全掩飾,但她的問話方式卻刻意保持著距離,像是在問一個陌生人。

**李杰**：他暫時失去意識了...但沒有生命危險。

**旁白**：恵美的眼神閃爍了一下,她似乎想說什麼,但又忍住了。她只是緊緊盯著凌墨,咬著嘴唇,眼中的擔憂越來越深。

**李杰**：(內心)她為什麼要假裝不認識他...？

**中村隼人**：失去意識？！你們到底對他做了什麼？！

**伊藤誠**：對啊！你們拿著劍,還背著一個昏迷的人！這到底是怎麼回事？！

**旁白**：車廂內的恐慌和質疑聲越來越大。窗外,紫色的月光灑在月台上,詭異而冰冷。

**旁白**：李杰看著車門外那個陌生的世界,再看著車廂內慌亂的乘客,還有背上失去意識的凌墨。

**李杰**：(內心)黑劍的力量轉移到凌墨身上...清水明明認識凌墨卻假裝不認識...

**李杰**：(內心)這次循環...到底會發生什麼...？
