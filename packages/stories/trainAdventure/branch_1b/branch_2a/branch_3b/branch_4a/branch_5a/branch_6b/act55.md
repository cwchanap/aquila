# 第五十五幕：搜查

**旁白**：車廂內的氣氛越來越緊張。所有人的目光都集中在李杰和小林雄大身上。

**旁白**：李杰小心地將背上的凌墨放到座位上,讓他平躺著。凌墨依然昏迷不醒,呼吸平穩。

**旁白**：就在這時,李杰腦中突然閃過一個念頭。

**李杰**：(內心)等等...之前的循環...中村帶了武器...

**李杰**：(內心)三把手槍,兩把折疊刀...他一直把它們藏在身上...

**李杰**：(抬起頭,看向中村)各位!

**旁白**：他的聲音突然響起,打斷了車廂內的爭吵。

**李杰**：這個人...他身上藏了武器!

**旁白**：車廂內瞬間一片譁然。

**田中健太**：什麼?!武器?!

**鈴木明日香**：天啊...

**伊藤奈々**：老公...我怕...

**中村隼人**：(臉色一變,但很快恢復)你胡說什麼?!

**中村隼人**：我怎麼可能帶武器?!你不要亂栽贓!

**斎藤蓮**：對啊!你憑什麼說人家有武器?你有證據嗎?

**李杰**：如果你真的清白,就讓大家搜查你的身體。

**李杰**：如果你沒有武器,那就證明我在說謊。

**旁白**：中村隼人的眼神閃爍了一下。他似乎在思考什麼。

**中村隼人**：好啊。我同意讓大家搜查。

**中村隼人**：但是——(指著李杰)如果你找不到任何武器,你要怎麼辦?

**李杰**：如果找不到,我立刻下車。

**中村隼人**：(冷笑)只有你一個人?不夠。

**中村隼人**：你和那個幫你說話的人,你們兩個都要下車。

**旁白**：他指著小林雄大。

**小林雄大**：(皺眉)我...?

**李杰**：(內心)他...他想讓我和小林先生都下車...

**李杰**：(內心)他為什麼這麼有信心...?難道他已經把武器藏起來了...?

**旁白**：李杰看著中村的表情——那種自信,那種篤定,讓他感到一絲不安。

**李杰**：(內心)但是...三把手槍和兩把刀...那麼多武器...他不可能憑空消失...

**李杰**：好。我答應你。

**中村隼人**：(露出得意的笑容)很好。那就開始吧。

**旁白**：小林雄大走上前,開始搜查中村的身體。他檢查了中村的外套口袋、褲子口袋、內袋...

**旁白**：什麼都沒有。

**小林雄大**：(皺眉)沒有...

**李杰**：他的背包!檢查他的背包!

**旁白**：中村隼人冷笑著,將自己的背包遞給小林雄大。

**旁白**：小林雄大打開背包,仔細翻查。裡面只有一些日常用品——錢包、手機、充電線、一瓶水...

**旁白**：沒有任何武器。

**小林雄大**：(搖頭)什麼都沒有...

**中村隼人**：(得意地笑)怎麼樣?我說了我沒有武器吧!

**中村隼人**：現在,按照約定,你們兩個應該下車了!

**斎藤蓮**：對!說話要算話!你剛才自己說的!

**伊藤誠**：是啊!冤枉人還不認錯!趕快下車!

**旁白**：其他乘客也開始附和。李杰咬緊牙關,額頭滲出冷汗。

**李杰**：(內心)不可能...他明明有武器...他一定藏在某個地方...

**李杰**：等等!

**旁白**：他突然開口。

**李杰**：既然我可能冤枉了人,那為了公平起見...大家都應該檢查自己的背包!

**李杰**：如果我真的說錯了,那大家的背包裡都不應該有可疑的東西,對吧?

**中村隼人**：(臉色微變)你...

**斎藤蓮**：這太荒謬了!為什麼我們要配合你?!

**李杰**：因為如果大家都沒問題,就能證明這裡是安全的。

**李杰**：這對大家都好,不是嗎?

**旁白**：車廂內沉默了片刻。

**田中健太**：(聳肩)我沒差啊...反正我也沒帶什麼奇怪的東西...

**鈴木明日香**：我也是...

**小林智子**：如果這樣能讓大家安心...

**旁白**：乘客們開始打開自己的背包,檢查裡面的物品。

**旁白**：田中健太的背包:教科書、筆記本、筆袋。

**旁白**：鈴木明日香的背包:化妝包、雜誌、零食。

**旁白**：高橋美咲的背包:手機充電器、耳機、面紙。

**旁白**：佐藤拓海的背包:運動服、水壺、毛巾。

**旁白**：伊藤夫婦的行李:衣服、盥洗用品、藥品。

**旁白**：小林一家的背包:孩子的玩具、零食、換洗衣物。

**旁白**：斎藤蓮的背包:筆記型電腦、文件、名片夾。

**旁白**：清水恵美的背包:錢包、手機、化妝品、一本小說。

**旁白**：所有人都檢查完了。沒有任何可疑的物品。沒有槍,沒有刀,什麼都沒有。

**中村隼人**：(冷笑)看到了嗎?什麼都沒有!你完全是在胡說八道!

**斎藤蓮**：對!你根本就是在製造恐慌!

**伊藤誠**：這種人太危險了!一定要把他趕下車!

**伊藤奈々**：還有那個幫他的人!他們一家人都很可疑!

**旁白**：她指著小林雄大和他的家人。

**小林智子**：(驚訝地)我們...?

**小林陽菜**：(害怕地抓著媽媽的手)媽媽...

**小林翔太**：(哭泣)我們沒有做壞事...

**旁白**：車廂內的氣氛變得更加敵對。所有人都用懷疑和憤怒的眼神看著李杰、小林雄大,以及小林的家人。

**李杰**：(咬緊牙關)不可能...武器一定在某個地方...

**旁白**：他環顧車廂,試圖找出蛛絲馬跡。但什麼都沒有發現。

**中村隼人**：夠了!按照約定,你們兩個現在就給我下車!

**中村隼人**：大家都看到了!這個人根本就是在說謊!他才是最危險的人!

**旁白**：車廂內響起一片贊同的聲音。李杰感到前所未有的壓力。

**李杰**：(內心)該死...他到底把武器藏到哪裡去了...？

**旁白**：而在角落裡,清水恵美依然靜靜地看著這一切。她的眼神複雜,似乎在思考著什麼。

**旁白**：躺在座位上的凌墨依然昏迷不醒,完全不知道車廂內發生的一切。

**旁白**：紫色的月光透過車窗照進來,為這場對峙增添了一絲詭異的氣氛。
