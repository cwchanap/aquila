# 第五十八幕：殺機

**旁白**：李杰站在車廂門口，握緊手中的菜刀，腦中快速思考著。

**李杰**：(內心)不能現在動手...

**李杰**：(內心)如果現在動手，那四個大學生不知道會做出什麼反應...

**李杰**：(內心)他們可能會驚慌逃跑，也可能會趁亂攻擊我...

**旁白**：他看向那四個充滿敵意的大學生。

**李杰**：(內心)不...我必須等...等到下一站...

**李杰**：(內心)等他們下車...車上就只剩下我、斎藤、中村...還有昏迷的凌墨...

**李杰**：(內心)斎藤手上有武器...可能是槍，也可能是刀...

**李杰**：(內心)我必須快速制服他，奪取他的武器...

**李杰**：(內心)然後...殺掉中村。

**旁白**：李杰的眼神變得冰冷。

**李杰**：(內心)靠我一個人，想要同時制服兩個人...幾乎不可能...

**李杰**：(內心)除非...殺掉其中一個。

**李杰**：(內心)中村才是真正的威脅...只要他死了，斎藤不足為懼...

**旁白**：他看向斎藤和中村。兩人坐在角落，冷靜地看著他。

**李杰**：(內心)斎藤和中村都不是體能型的人...

**李杰**：(內心)只要動作夠快，夠精準...我能殺掉中村...

**旁白**：李杰深吸一口氣。

**李杰**：(內心)而且...就算失敗了...我還能再來一次...

**李杰**：(內心)我比他們冷靜...我的精神狀態比他們好...

**李杰**：(內心)我會贏的...

**旁白**：李杰慢慢移動位置，找到一個能在車門關閉瞬間快速衝向斎藤的位置。

**旁白**：他靠在車廂中段的座位旁，握緊刀柄，目光在斎藤和中村之間來回掃視。

**旁白**：車廂內陷入詭異的沉默。大學生們緊緊靠在一起，充滿戒備地看著李杰。

**旁白**：中村和斎藤坐在角落，表情平靜。斎藤時不時小心翼翼地瞥向中村，眼中帶著一絲畏懼。

**旁白**：凌墨依然躺在座位上，昏迷不醒。

**旁白**：列車繼續向前行駛。窗外的紫色月光越來越明亮。

**旁白**：過了一段時間，列車開始減速。

**旁白**：車廂內的燈光閃爍了幾下。

**旁白**：李杰握緊刀柄，全身肌肉緊繃。

**李杰**：(內心)到了...

**旁白**：列車緩緩停下。又是幽冥之都站。

**旁白**：車門「嘶」的一聲打開了。

**旁白**：田中健太、鈴木明日香、高橋美咲、佐藤拓海四個人立刻站起身，快步走向車門。

**田中健太**：(小聲)快走...

**鈴木明日香**：(小聲)不要回頭...

**旁白**：四個人緊緊靠在一起，快速下了車。他們踏上月台，快步離開。

**旁白**：佐藤拓海是最後一個下車的。

**旁白**：他的腳剛踏上月台——

**旁白**：車門立刻開始關閉！

**旁白**：車廂內只剩下李杰、中村、斎藤，還有昏迷的凌墨。

**李杰**：(內心)就是現在！

**旁白**：車門「嘶」的一聲完全關閉的瞬間——

**旁白**：李杰猛地衝向斎藤！

**斎藤蓮**：——！

**旁白**：斎藤驚恐地站起身，迅速伸手到褲子裡，掏出一把黑色的手槍！

**旁白**：他舉起手槍，顫抖著指向李杰！

**李杰**：(內心)果然有槍！

**旁白**：但是——

**旁白**：斎藤的手指在扳機上猶豫著，不知道該怎麼開槍！

**斎藤蓮**：(慌張地)怎...怎麼用...

**旁白**：槍卡住了！他完全不知道怎麼使用手槍！

**中村隼人**：(憤怒地)你這個蠢貨！

**旁白**：中村立刻衝向斎藤，想要搶過手槍！

**旁白**：但李杰更快！

**旁白**：李杰衝到斎藤面前，用力一揮刀，打掉了斎藤手中的手槍！

**斎藤蓮**：啊——！

**旁白**：手槍飛了出去，掉在地上！

**旁白**：就在同一瞬間，中村衝到了李杰身邊！

**李杰**：——！

**旁白**：李杰反手一刀，砍向中村！

**旁白**：刀刃劃過中村的右手！

**中村隼人**：啊啊啊——！

**旁白**：中村的手掌被割開，鮮血噴濺而出！他尖叫著倒退！

**中村隼人**：(憤怒地)斎藤你這個廢物！連槍都不會用！

**旁白**：李杰趁機一腳踢向中村的腹部！

**旁白**：中村被踢飛，撞在座位上！

**旁白**：就在這時——

**旁白**：斎藤從口袋裡掏出一把折疊刀，猛地刺向李杰！

**李杰**：(內心)糟糕！

**旁白**：李杰想要轉身防禦，但還是慢了一步！

**旁白**：刀刃劃過李杰的左臂，鮮血滲了出來！

**李杰**：(咬牙)嗯...！

**旁白**：李杰忍著疼痛，用力一推，把斎藤推開！

**斎藤蓮**：(顫抖著)我...我...

**旁白**：斎藤踉蹌後退，手中還握著沾血的刀。

**旁白**：就在這時，李杰看到中村爬起身，衝向地上的手槍！

**中村隼人**：(喘息著)我要殺了你...！

**旁白**：中村伸手要去撿槍！

**李杰**：(內心)不能讓他拿到槍！

**李杰**：(內心)只要殺了中村...就不用面對一對二了...

**旁白**：李杰的眼神變得冰冷。

**旁白**：李杰猛地衝向中村，整個人撲了過去！

**中村隼人**：——！

**旁白**：中村的手指剛剛碰到手槍，李杰就撲到了他身上！

**旁白**：李杰用左手按住中村，右手舉起菜刀！

**中村隼人**：(驚恐地)不——！

**旁白**：刀刃在紫色的月光下閃爍！寬而鋒利的刀刃反射著冷光！

**旁白**：然後——

**旁白**：李杰用盡全力一揮！

**旁白**：沉重的菜刀深深劃過中村的喉嚨！

**中村隼人**：(發出含糊的聲音)咕...嗚...

**旁白**：鮮血瞬間噴濺而出，濺在座位上、地板上、李杰的臉上！

**旁白**：中村的眼睛瞪得很大，雙手抓著自己的喉嚨，但傷口太深，血止不住地湧出！

**旁白**：他的身體抽搐了幾下，然後癱軟下來，不再動彈。

**旁白**：中村隼人...死了。

**旁白**：李杰跪在中村的屍體旁，大口喘息著。他的手在顫抖，臉上、衣服上都是血。

**李杰**：(顫抖著)我...我殺了人...

**旁白**：他看著手中沾滿鮮血的刀，整個人愣住了。

**旁白**：就在這時——

**斎藤蓮**：(驚恐地尖叫)啊啊啊啊——！

**旁白**：李杰猛地回過神，轉頭看向斎藤。

**旁白**：斎藤站在車廂另一端，臉色蒼白，握著刀的手劇烈顫抖。

**斎藤蓮**：(崩潰地)你...你殺了他...你真的殺了他...

**旁白**：斎藤的眼中充滿恐懼。他看著地上中村的屍體，看著李杰臉上的血，整個人嚇得說不出話。

**旁白**：李杰慢慢站起身，握緊手中的刀，看著斎藤。

**李杰**：(冷靜地)放下刀。
