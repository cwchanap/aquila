# 第六十幕：重逢的恐懼

**旁白**：李杰站在車門旁，看著外面的峽谷，又看看車廂內的斎藤和凌墨。時間一分一秒過去。

**李杰**：(內心)必須做決定了...山本應該還在路上...如果我能找到他，還有其他人...我們可以一起去祭壇...

**旁白**：他握緊手中的菜刀，深吸一口氣，轉身看向斎藤。

**李杰**：斎藤，你留在車上，照顧好凌墨。

**斎藤蓮**：(愣住)什麼？你...你要下車？

**李杰**：列車會帶你們去終焉祭壇。我會在那裡和你們會合。

**斎藤蓮**：(驚恐)不...不行...我...我不知道怎麼照顧他...萬一...萬一他出什麼事...

**李杰**：(冷冷地看著他)你只需要照顧好他，直到我回來。什麼都不要做。

**旁白**：斎藤張了張嘴，想要抗議，但看到李杰冰冷的眼神，話又嚥了回去。

**李杰**：(走近斎藤，聲音低沉)聽好了。如果凌墨少了一根汗毛，我會把你撕成碎片。

**旁白**：斎藤的臉色瞬間慘白，身體不由自主地顫抖。

**斎藤蓮**：(顫抖著)我...我不會...我不會傷害他的...我發誓...

**李杰**：(盯著他看了幾秒)記住你說的話。

**旁白**：李杰轉身，最後看了一眼昏迷的凌墨，然後走向車門。

**李杰**：(回頭)在祭壇等我。

**旁白**：他跨出車門，雙腳踏上月台。峽谷的冷風吹在臉上，帶著乾燥的岩石氣息。

**旁白**：身後傳來車門關閉的「嘶」聲。列車的引擎發出低沉的轟鳴，車輪與鐵軌摩擦的聲音漸漸響起。

**旁白**：李杰站在月台上，看著列車緩緩駛離。透過車窗，他能看到斎藤驚恐的臉，還有躺在座位上的凌墨。

**旁白**：列車越來越快，最後消失在峽谷深處的黑暗中。

**旁白**：月台上又只剩下李杰一個人。

**李杰**：(深呼吸)好了...該去找山本了。

**旁白**：他看向來時的方向。鐵軌沿著峽谷邊緣延伸，沒入遠處的黑暗森林。

**李杰**：(內心)從這裡往回走...中途會經過那座小屋...可以在那裡補給一些食物和水...

**李杰**：(內心)凌墨和斎藤也需要食物...我得帶些回去...

**旁白**：他開始沿著鐵軌往回走。腳下的碎石發出輕微的聲響，在寂靜的峽谷中格外清晰。

**旁白**：兩輪紫月依然懸在天空，為這片荒涼的土地提供著唯一的光源。遠處的峽谷深不見底，黑色的岩壁上零星散落著巨大的骸骨。

**旁白**：李杰走了很久。一個小時，也許更久。時間在這裡似乎失去了意義。

**旁白**：他逐漸遠離峽谷，鐵軌開始沒入森林。樹木越來越茂密，空氣也變得潮濕起來。

**旁白**：終於，前方的樹林間，那座熟悉的木屋出現在視野中。

**李杰**：(停下腳步)是這裡...

**旁白**：那座雙層木屋靜靜地矗立在林中的空地上。在紫月的光芒下，木屋的輪廓清晰可見。門廊、窗戶、煙囪——一切都和他記憶中一模一樣。

**李杰**：(內心)智子...陽菜...翔太...那時候我們一起走到這裡，智子牽著孩子們的手，我在前面帶路...

**李杰**：(內心)他們現在在哪裡呢...希望他們都還平安...

**旁白**：他慢慢走近小屋。但就在距離門廊還有十幾米時，他停下了腳步。

**旁白**：屋內傳來了說話的聲音。

**健太的聲音**：(從屋內傳來)...該怎麼辦？我們就這樣一直等下去嗎？

**明日香的聲音**：(害怕)我不知道...那個李杰...他和那個黑衣服的男人...他們會不會...

**拓海的聲音**：(沉穩)先別想那麼多。至少我們現在是安全的。

**旁白**：李杰聽出了那些聲音——是那四個大學生。

**李杰**：(內心)他們在這裡...還好，至少他們還活著...

**旁白**：但隨即，他又猶豫了。

**李杰**：(內心)他們...很害怕我。在車上的時候，他們的眼神...充滿了恐懼和敵意。

**李杰**：(內心)如果我就這樣走進去...他們會不會更加恐慌？

**旁白**：他低頭看了看自己。衣服上沾滿了血跡——中村的血。菜刀還插在腰間，刀刃上的血已經乾涸，變成暗紅色。

**李杰**：(內心)這樣子...確實會嚇到他們...

**旁白**：他猶豫了幾秒，然後做出決定。

**李杰**：(內心)不管怎麼樣，都必須和他們解釋清楚。我們需要一起行動，才能活下去。

**旁白**：他把菜刀從腰間拔下，小心地放進背包裡。然後深吸一口氣，走向小屋的門。

**旁白**：他的腳步踏上門廊的木階。木板發出輕微的「吱呀」聲。

**旁白**：屋內的說話聲瞬間停止。

**旁白**：李杰推開門，走進屋內。

**旁白**：四個大學生正聚在壁爐旁。看到李杰的瞬間，他們的臉色全部變了。

**鈴木明日香**：(尖叫)啊啊啊！是他！是他！

**高橋美咲**：(驚恐地往後退)不要...不要殺我們...求求你...

**田中健太**：(立刻站起來，從旁邊抓起一把劍)你...你別過來！

**旁白**：健太雙手握著那把劍，擋在其他人前面。劍身鏽跡斑斑，但他握得很緊，手都在顫抖。

**李杰**：(認出那把劍)那把劍...

**旁白**：那是一把生鏽的長劍。劍身上的鏽跡、劍柄上腐朽的布條——和李杰第一次循環時找到的那把很像。

**李杰**：(內心)他們也在車站裡搜索過...找到了武器...

**田中健太**：(顫抖著，但努力保持鎮定)李杰...你...你馬上離開這裡！

**田中健太**：這次...這次我們有武器！我們不會再退縮了！

**鈴木明日香**：(躲在健太身後，哭著)求求你...求求你不要殺我們...我們什麼都沒做...

**高橋美咲**：(同樣躲著，聲音顫抖)我們...我們不會妨礙你的...求求你放過我們...

**旁白**：佐藤拓海沒有說話，但他的身體緊繃，隨時準備保護身後的兩個女生。

**李杰**：(舉起雙手，慢慢說)等等...我不是來傷害你們的。

**李杰**：(展示空手)你們看，我手上什麼都沒有。我只是想和你們談談。

**田中健太**：(劍尖指著李杰，不敢放鬆)你...你身上都是血！你殺了誰？！

**鈴木明日香**：(哭喊)一定是那個看起來很可疑的叔叔...還有那個戴眼鏡的...你把他們都殺了對不對？！

**李杰**：(搖頭)我沒有殺那個戴眼鏡的。他還活著，在列車上。

**高橋美咲**：(顫抖)那...那另一個人呢？

**旁白**：李杰沉默了幾秒。

**李杰**：那個可疑的叔叔已經死了。他叫中村。

**李杰**：車上那個戴眼鏡的...斎藤，他和中村是一夥的。他們兩個人都想殺我。

**李杰**：我一個人面對他們兩個，沒有其他選擇。情況很複雜。請給我一個機會，讓我解釋清楚。

**田中健太**：(大喊)你離開！我們不想聽你的解釋！

**佐藤拓海**：(終於開口，聲音沉穩)健太。

**田中健太**：(回頭)拓海？

**佐藤拓海**：先聽他說。

**田中健太**：(不可置信)什麼？你瘋了嗎？他殺了人！

**佐藤拓海**：(看著李杰)如果他真的想殺我們，剛進來的時候就會直接動手了。他沒有拿武器，是想和我們談。

**鈴木明日香**：(哭著)可是...可是...

**佐藤拓海**：(對李杰)你說。但是如果你有任何可疑的舉動，我們會拼命的。

**旁白**：李杰點了點頭。

**李杰**：謝謝你願意聽我說。

**李杰**：對了，我還沒自我介紹。我叫李杰。

**佐藤拓海**：(點頭)我是佐藤拓海。

**田中健太**：(還是有些警戒)田中健太...

**鈴木明日香**：(小聲)鈴木...明日香...

**高橋美咲**：(顫抖)高橋美咲...

**李杰**：我知道你們害怕我。在車上的時候，我的行為...讓你們覺得我很危險。

**李杰**：但是，事情的真相比你們想像的更複雜。
