# 第七十一幕：銀月湖的重逢

**旁白**：大廳裡，眾人站在發光的傳送門前。

**田中健太**：(看著門，深吸一口氣)終於...可以回家了...

**鈴木明日香**：(緊張地握著美咲的手)真的...可以嗎...

**李杰**：(轉向他們)你們先回去吧。我們還有事要處理。

**佐藤拓海**：(點頭)明白了。那...我們先走了。

**李杰**：路上小心。

**田中健太**：(看著李杰)李桑...謝謝你。

**李杰**：(微笑)保重。

**高橋美咲**：(看著凌墨和翔)你們...也要小心...

**凌墨**：(點頭)會的。

**旁白**：斎藤站在一旁，沉默地看著眾人。

**斎藤蓮**：(推眼鏡，轉向門)...我先走了。

**旁白**：他徑直走向傳送門，沒有回頭。當他踏入那片白色光芒時，身影瞬間消失。

**鈴木明日香**：(也走向門)那...我們也走了...

**旁白**：明日香和美咲手牽手，一起走進傳送門，消失了。

**田中健太**：(最後看了李杰一眼)保重。

**旁白**：健太也走進光芒中。

**佐藤拓海**：(向李杰點頭)再見。

**旁白**：拓海最後一個走進傳送門，身影消失。

**旁白**：大廳裡恢復了寧靜。只剩下李杰、凌墨和高山翔三人。

**李杰**：(看著門)希望他們能平安到家。

**凌墨**：(點頭)會的。門已經穩定了。

**高山翔**：(急切地)我們...我們快去找恵美吧...

**李杰**：(轉身)走吧。去銀月湖。

---

**旁白**：三人離開終焉祭壇，沿著幽暗的長廊向上走。藍色的寶石照亮著路途，他們的腳步聲在空曠的空間中迴盪。

**旁白**：很快，他們回到了大廳，然後走出建築，來到外面。

**旁白**：外面依然是那片永恆的紫黑色天空，雙月高懸，森林在遠處延伸。

**凌墨**：(停下腳步，看向翔)翔，有些事...我必須告訴你。

**高山翔**：(疑惑)什麼事？

**凌墨**：(深吸一口氣)是關於恵美的。她回到地球後又回來了。當時...她想用「交換」的方式救你。

**高山翔**：(臉色蒼白)交換...？

**李杰**：(平靜地解釋)獻祭一個無辜的人，來換取你的自由。讓那個人代替你，被困在這個世界。

**旁白**：翔的臉色變得更加蒼白，身體微微搖晃。

**高山翔**：(低聲)不...不...她怎麼會...

**凌墨**：(握住翔的肩膀)我一直在勸她，但她聽不進去。所以...我偷偷跟著她，想阻止她。

**高山翔**：(抬起頭，眼眶泛紅)她...她打算犧牲誰...？

**李杰**：(看著他)小林一家。他們是車上的一對夫妻和兩個孩子。她可能會選擇他們中的一個人。

**旁白**：翔猛地抬起頭，眼神中充滿震驚和痛苦。

**高山翔**：(顫抖)孩子...？她...她要犧牲孩子...？

**凌墨**：(低聲)對不起，翔...我沒能阻止她...

**高山翔**：(搖頭，聲音哽咽)不...這不是你的錯...是我...是我害了她...

**李杰**：(走上前)但現在不需要了。你已經醒了，門也開了。沒有人需要被犧牲。

**高山翔**：(看著李杰，眼中充滿複雜的情緒)你...你救了我...也救了那家人...

**李杰**：(搖頭)不只是我。如果不是這把劍，我也沒辦法。

**旁白**：翔低下頭，沉默了幾秒。

**高山翔**：(抬起頭，看著李杰)謝謝你...真的...謝謝你...

**凌墨**：(看著李杰)還有一件事，翔。關於李杰。

**高山翔**：(疑惑)什麼？

**凌墨**：(嚴肅地)李杰...是個時間旅行者。

**旁白**：翔愣住了，不可置信地看著李杰。

**高山翔**：時間...旅行者...？

**李杰**：(點頭)對。我經歷過這一切，然後回到過去，改變了很多事情。

**高山翔**：(震驚)所以...所以你早就知道會發生什麼...

**李杰**：我知道這個世界、列車、你被困在祭壇，還有恵美的計劃。所以我回來改變這一切。

**高山翔**：(低聲)為了救我們...

**李杰**：(平靜地)為了救所有人。

**旁白**：翔看著李杰，眼神中充滿敬意和感激。

**高山翔**：你...你做了不可思議的事...

**凌墨**：(點頭)對。所以...我們欠他很多。

**李杰**：(搖頭)別這麼說。走吧，恵美在等著你。

---

**旁白**：三人穿過森林，沿著李杰記憶中的路線前進。大約走了六個小時，前方的樹林開始變得稀疏。

**李杰**：(停下腳步)快到了。

**旁白**：他們走出森林，銀月湖出現在眼前——湖面依然平靜如鏡，倒映著天空中的雙月，湖面上的發光花朵輕輕搖曳。

**高山翔**：(看著湖面，低聲)好美...

**凌墨**：(點頭)銀月湖。恵美應該在這裡。

**旁白**：李杰環顧四周，很快看到湖邊有幾個人影。

**李杰**：(指著那邊)在那裡。

**旁白**：他們快步走向那些人影。

**旁白**：當距離拉近時，李杰看清了——是小林一家，還有...恵美。

**旁白**：恵美穿著那件淺米色的棉質連衣裙，栗色的長髮在微風中輕輕飄動。她正蹲在湖邊，看著湖面，背對著他們。

**小林陽菜**：(看到他們，警覺地)爸爸！有人來了！

**旁白**：陽菜的喊聲讓所有人都轉過頭來。

**小林雄大**：(站起身，表情複雜)李杰...你回來了...

**小林智子**：(也站起身，緊張地看著李杰身後的兩人)你...你們...

**小林陽菜**：(躲到母親身後，小聲)媽媽...

**旁白**：恵美也轉過身，當她看到李杰身後的兩個人時，臉色瞬間變得蒼白。

**清水恵美**：(顫抖)翔...？

**旁白**：她的聲音充滿不可置信。

**高山翔**：(看著她，眼眶泛紅)恵美...

**旁白**：恵美愣在原地，眼淚瞬間湧了出來。

**清水恵美**：(哽咽)翔...真的是你嗎...？我不是在做夢吧...？

**旁白**：翔快步走向她，張開雙臂。

**旁白**：恵美也跑了過去，撲進他的懷抱。

**清水恵美**：(哭泣)翔...翔...你終於醒了...

**高山翔**：(抱緊她，聲音顫抖)對不起...讓妳等了這麼久...

**旁白**：兩人緊緊抱在一起，恵美的哭聲在湖邊迴盪。小林一家、凌墨和李杰靜靜地站在一旁，給他們一些時間。

**小林翔太**：(小聲問陽菜)姐姐，那個大哥哥是誰？

**小林陽菜**：(也小聲)不知道...但看起來...姐姐很高興...

**旁白**：過了好一會兒，恵美終於鬆開翔，退後一步。她的臉上掛著淚痕，但眼神中充滿喜悅。

**清水恵美**：(擦著眼淚)你...你怎麼醒來的...？我還在想...想辦法...

**高山翔**：(溫柔地)是李杰救了我。他用神劍喚醒了我。

**旁白**：恵美轉頭看向李杰，眼神複雜。

**清水恵美**：李桑...是你...

**李杰**：(點頭，平靜地)對。翔已經醒了，門也開了。

**旁白**：恵美的身體微微一震，眼神複雜。

**清水恵美**：(低聲)他...他真的醒了...

**李杰**：對。所以...妳不需要再執行那個計劃了。沒有人需要被犧牲。

**旁白**：恵美沉默片刻，然後低下頭。

**清水恵美**：(低聲)對不起...我...

**高山翔**：(走到她身邊，握住她的手)恵美，別這樣說...一切都過去了...

**旁白**：小林一家站在遠處，看著這一幕。

**小林雄大**：(猶豫地走上前幾步)李桑...那個...

**旁白**：雄大的表情複雜，似乎想說些什麼，但又不知道該怎麼開口。

**李杰**：(看著他，平靜地)不用說了。

**小林雄大**：(沉默片刻，點了點頭)...謝謝。

**旁白**：雄大退後幾步，回到智子身旁。

**小林智子**：(抱著陽菜和翔太，眼神複雜地看著李杰)

**凌墨**：(走到恵美面前)恵美，我也一直在擔心妳。

**清水恵美**：(抬起頭，看到凌墨，愣住)凌墨...你也...

**凌墨**：(點頭)我一直跟著妳。只是...我沒能阻止妳。

**清水恵美**：(搖頭，哽咽)對不起...對不起...我讓你們擔心了...

**旁白**：翔抱住她，輕輕拍著她的背。

**高山翔**：(溫柔地)別哭了...我們都在這裡...一切都會好起來的...

**旁白**：湖邊恢復了寧靜。小林一家站在遠處，看著那對久別重逢的戀人。凌墨和李杰站在一旁。

**旁白**：雙月的光芒灑在湖面上，發光的花朵依然輕輕搖曳。氣氛有些尷尬而沉默。

---

**旁白**：過了一會兒，恵美終於平靜下來。

**李杰**：(看著眾人)我們在這裡休息一晚吧。明天你們可以先回地球。

**旁白**：小林一家互相看了看，沒有說話。

**小林雄大**：(點頭)...嗯。

**小林智子**：(低聲)湖邊有幾間小屋...

**清水恵美**：(看著翔)...好。

**李杰**：我明天還要去找一個人。所以你們不用等我。

**小林雄大**：(警覺地)找誰？

**李杰**：山本浩二。我之前讓他先去終焉祭壇拿神劍，但我不確定他是否找到了路。我必須確認他的安全。

**旁白**：小林一家沉默地聽著，沒有說話。

**清水恵美**：(低聲)李桑...你...你要去多久？

**李杰**：不確定。可能幾個小時，也可能更久。你們不用等我，門已經開了，可以直接回去。

**高山翔**：(看著李杰)那...那我們明天...

**李杰**：(點頭)對。明天你們先回地球。

**旁白**：氣氛再次陷入沉默。陽菜和翔太躲在父母身後，偷偷看著李杰他們。

**凌墨**：(走上前)我跟你一起去找山本。

**李杰**：(看著凌墨)好。

**旁白**：眾人開始朝著湖邊的小屋走去。

---

**旁白**：夜晚，湖邊的小屋裡亮著微弱的光。

**旁白**：小林一家住在其中一間小屋裡。雄大和智子坐在床邊，看著已經睡著的陽菜和翔太。

**小林智子**：(低聲)明天...我們就能回去了...

**小林雄大**：(點頭)...嗯。

**小林智子**：(沉默片刻)李桑...他真的...

**小林雄大**：(打斷她)...別想太多了。休息吧。

**旁白**：智子沉默地點了點頭。

---

**旁白**：第二天早晨——雖然天空依然是那片永恆的紫黑色，但眾人都已經醒來。

**旁白**：湖邊，眾人聚在一起。

**清水恵美**：(看著李杰)李桑...我們...我們要先回去了...

**李杰**：(點頭)嗯。路上小心。

**旁白**：恵美走到李杰面前，猶豫了一下。

**清水恵美**：李桑...謝謝你...真的...謝謝你...

**李杰**：(平靜地)別客氣。

**清水恵美**：(深吸一口氣)那...保重。

**李杰**：妳也是。

**旁白**：翔也走過來，向李杰深深鞠了一躬。

**高山翔**：李桑...我欠你一條命...以後...以後如果有需要我的地方...

**李杰**：(搖頭)不用。好好照顧恵美就好。

**高山翔**：(直起身)...我會的。

**旁白**：小林雄大帶著家人走上前。

**小林雄大**：(保持距離，正式地點頭)李桑...那我們也先走了。

**李杰**：(點頭)路上小心。

**小林智子**：(站在雄大身後，低聲)...多謝。

**李杰**：不客氣。

**旁白**：陽菜和翔太躲在父母身後，偷偷看著李杰，沒有說話。

**旁白**：凌墨走到恵美和翔面前。

**凌墨**：恵美，翔，回去後好好生活。

**清水恵美**：(眼眶泛紅)凌墨...你也要小心...

**凌墨**：(微笑)會的。

**高山翔**：(看著凌墨)凌墨...謝謝你一直以來...

**凌墨**：(拍了拍翔的肩膀)別說這些了。快回去吧。

**旁白**：恵美、翔和小林一家開始朝著終焉祭壇的方向走去。

**旁白**：李杰和凌墨站在湖邊，目送著他們的背影漸漸消失在森林中。

**旁白**：湖面依然平靜如鏡，倒映著天空中的雙月。

**凌墨**：(看著他們離開的方向)他們...終於可以回家了。

**李杰**：(點頭)是啊。

**凌墨**：(轉向李杰)那我們也走吧。去找山本。

**李杰**：(看著森林深處)走吧。

**旁白**：兩人轉身，朝著相反的方向走去。

**旁白**：在那片永恆的紫黑色天空下，兩組人分別朝著不同的方向前進。

**旁白**：一組，走向回家的路。

**旁白**：另一組，走向未知的森林深處，去尋找最後一個失聯的同伴。
