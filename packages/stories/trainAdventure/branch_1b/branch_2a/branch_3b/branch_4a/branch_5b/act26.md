# 第二十六幕：二十年前的交集 

---

**旁白**：李杰走到那個男人面前。對方也注意到了他，視線從人群中移向他。

**清水浩介**：你好，請問是...李先生嗎？

**李杰**：是的，我就是@LJ_2025。你是清水先生吧？

**清水浩介**：是的，你好，我叫清水浩介。目前在千葉的一家通訊公司上班。

**旁白**：清水浩介明顯鬆了口氣，伸出手來。

**旁白**：李杰握住了他的手。對方的手掌溫暖而有力，但能感覺到一絲緊張的顫抖。

**李杰**：我叫李杰。是一名軟體工程師，從上海來東京工作的。

**清水浩介**：讓你特地跑一趟真不好意思。

**李杰**：沒關係。我住在三鷹，過來也方便。倒是清水先生你，從千葉過來要很久吧？

**清水浩介**：我家在船橋，搭總武線直達，其實還好。

**李杰**：是嗎？我也是搭總武線過來的。

**旁白**：這條共同的通勤線路，無形中拉近了兩人之間的距離。

**旁白**：原本陌生的兩個人，因為一條鐵路線，因為一個二十年前的謎團，站在了這裡。

**李杰**：我們找個地方坐下聊吧。這裡人太多了。

**清水浩介**：好的。

**旁白**：李杰環顧四周，在附近找了一家掛著紅燈籠的居酒屋。

**旁白**：透過玻璃窗能看見裡面溫暖的燈光和忙碌的店員。這是那種充滿昭和風情的老式居酒屋，木製的桌椅、泛黃的海報、還有掛在牆上的書法。

**旁白**：李杰拉開門，一股混合著烤肉焦香、醬油味道和酒精氣息的空氣撲面而來。

**旁白**：店內傳來人們的歡笑聲、碰杯聲、還有店員們高聲的招呼。

**店員**：歡迎光臨！兩位客人這邊請！

**旁白**：一位穿著法被的店員熱情地招呼著他們。

**旁白**：店內幾乎滿座，大部分都是下班後來喝一杯的上班族。空氣中瀰漫著熱鬧而放鬆的氛圍。

**旁白**：他們在吧台找到了兩個空位坐下。

**旁白**：吧台後面是開放式的廚房，師傅正在熟練地翻烤著串燒，火焰不時竄起，發出「呼」的聲響。

**店員**：兩位客人喝點什麼？

**李杰**：兩杯生啤酒，謝謝。

**清水浩介**：再來一盤枝豆和綜合串燒。

**店員**：好的，請稍等！

**旁白**：很快，店員將兩杯冒著綿密白色泡沫的生啤酒放在他們面前。

**旁白**：玻璃杯冰涼透明，啤酒呈現出漂亮的金黃色。

**旁白**：清水浩介拿起杯子，喝了一大口。喉結上下滾動，然後長長地嘆了口氣。

**旁白**：他放下杯子，神情變得沉重起來。

**清水浩介**：再次感謝你願意聯繫我。我的姐姐，名叫清水恵美...二十年前，她和朋友聚會後，就消失了。

**旁白**：清水浩介的聲音帶著壓抑已久的痛苦。

**清水浩介**：而當時和我一起在便利商店打工的一個同事，叫凌墨，那天也和姐姐在一起...之後也同樣失蹤了。警方什麼都查不到，但我一直覺得，事情沒有那麼簡單。

**李杰**：凌墨...這個名字聽起來是中國人？

**旁白**：李杰沒有聽過這個名字，但確實聽起來像是中國人的姓名。

**李杰**：請問，你這位叫凌墨的同事...是中國人嗎？

**清水浩介**：嗯，他是中國來的留學生，為了賺大學學費，晚上在我打工的便利商店兼職。

**旁白**：清水浩介喝了一口啤酒，繼續說道。

**清水浩介**：他那個人很獨立，聽說在日本也沒什麼親戚。所以他失蹤之後...老實說，除了我因為姐姐的事一直記著他，店裡的其他同事，大概也只是以為他找到新工作或回國了，抱怨了幾天就沒再提起了。

**旁白**：清水浩介的眼中充滿了期待和不安。他身體微微前傾，雙手緊握著啤酒杯。

**清水浩介**：李先生，你說你可能掌握一些情報...請問，你到底知道些什麼？

**旁白**：李杰正要開口，店員的聲音打斷了他們。

**店員**：久等了，這是綜合串燒和枝豆。

**旁白**：店員將一個木盤放在吧台上。盤子裡是各式各樣的串燒——雞肉、雞皮、青椒、香菇，每一串都烤得焦香，刷著閃亮的醬汁。

**旁白**：還有一盤翠綠的枝豆，撒著粗鹽，冒著熱氣。

**旁白**：食物的香氣在兩人之間瀰漫開來，但此刻誰都沒有心思去品嚐。

**旁白**：李杰拿起一串烤雞肉，咬了一口。肉質緊實，帶著炭火的焦香和醬汁的甜鹹。

**旁白**：他又喝了一大口冰涼的啤酒，定了定神。

**旁白**：他看著清水浩介懇切的雙眼，那雙眼睛裡充滿了二十年來從未放棄的執著。

**旁白**：李杰開始講述那個發生在一週之後、卻又彷彿早已發生過的，關於一輛末班電車的離奇故事。
