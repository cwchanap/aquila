# 第二十七幕：不完全的真相

---

**旁白**：李杰看著清水浩介懇切的雙眼，決定隱去那些血腥殘酷的細節，只講述一個相對客觀的版本。

**李杰**：總之，那輛列車把我們帶到了一個完全陌生的地方。我在第二個車站下了車，之後一路沿著鐵軌步行，穿過了很多地方，最後到了一個有精靈的森林。是那裡的「神樹」，把我送了回來。

**旁白**：清水浩介震驚地聽著這一切，嘴巴微微張開，眼神中充滿了難以置信。

**旁白**：他久久沒有說話，只是盯著桌上的啤酒杯。

**旁白**：然後，他端起杯子，猛地灌了一大口，喉結上下滾動。啤酒杯重重地放回桌上，發出「咚」的一聲。

**清水浩介**：精靈...神樹...這些...

**旁白**：他的聲音有些沙啞，似乎是在努力消化這過於超現實的資訊。

**清水浩介**：在那邊...你有沒有見過任何中年女性？大概...四十歲左右。

**李杰**：四十歲左右的女性...

**旁白**：李杰在腦中快速回憶著在異世界遇到的每一個人。

**李杰**：小林女士...她的名字是智子，不是恵美。而且她有丈夫和孩子。

**旁白**：他又想到了那個死在列車上的焦慮妻子。

**李杰**：還有那個焦慮的妻子...但她當時和丈夫在一起，也不可能是失蹤了二十年的人。

**李杰**：我確實遇到了兩位年齡相仿的女士，但根據我了解到的情況，她們似乎都不是你要找的人。

**旁白**：清水浩介的眼神暗淡了下去，肩膀也垮了下來。

**清水浩介**：是嗎...那...像凌墨那樣的中年男性呢？

**李杰**：我不知道他們的名字。你說的凌墨，是什麼樣的人？

**清水浩介**：他很安靜，話不多，但做事非常認真努力。人也很好，很樂於助人，工作從來不遲到。

**旁白**：李杰再次在心中過濾著那些男乘客的形象。

**李杰**：安靜、認真、樂於助人...

**旁白**：自私中年男？絕對不符合。那個人自私膽怯，完全不樂於助人。

**旁白**：沉穩父親小林先生？但他有妻子和孩子，不像是獨身一人的留學生。

**旁白**：還有那個慷慨的大叔...但我對他了解也不多，無法確定。

**李杰**：聽你的描述，似乎...也沒有能完全對得上的人。

**清水浩介**：這樣啊...

**旁白**：他長長地嘆了口氣，臉上滿是失望。整個人彷彿一下子老了十歲。

**李杰**：不過，我並沒有探索那個世界的全部，很多事情還不好說。

**旁白**：李杰看著對方失落的樣子，猶豫了一下，忍不住問道。

**李杰**：清水先生，恕我直言...你真的相信我說的這些嗎？穿越到異世界...這聽起來實在太荒唐了。

**旁白**：清水浩介自嘲地笑了笑，搖了搖頭。

**清水浩介**：我姐姐無緣無故地消失了二十年，警察、偵探、靈媒...所有的方法都試過了，卻連一絲線索都沒有。這件事本身，就已經夠「荒唐」了。這個世界上，本來就有很多科學無法解釋的事情，不是嗎？

**李杰**：你說得對。

**旁白**：沉重的話題告一段落。

**旁白**：兩人點了更多的食物和酒——鹽燒秋刀魚、炸雞塊、還有幾串烤雞心。

**旁白**：他們轉而聊起了彼此的工作和生活，試圖用酒精和美食來沖淡這份凝重的氣氛。

**清水浩介**：李先生是做軟體工程的吧？從上海過來的？

**李杰**：對，五年前來的。一開始語言不太通，吃了不少苦頭。

**清水浩介**：現在日語說得很流利了啊。在哪家公司工作？

**李杰**：在六本木的一家IT公司，做後端開發。你呢？通訊公司的工作怎麼樣？

**清水浩介**：就那樣吧，普通的上班族生活。每天擠總武線通勤，加班到很晚，週末偶爾加班。

**旁白**：他苦笑著搖搖頭。

**清水浩介**：不過比起以前在便利商店打工，至少現在有雙休和年假。

**李杰**：你以前在便利商店打工？

**清水浩介**：嗯，高中的時候。那時候家裡經濟不太好，姐姐在讀大學，我就去打工貼補家用。就是在那時候認識凌墨的。

**旁白**：提到凌墨，他的表情又黯淡了些。但很快又恢復過來。

**清水浩介**：對了，李先生平時有什麼興趣愛好嗎？

**李杰**：偶爾會去秋葉原逛逛，打打遊戲。你也看到了，今天下午我就在那邊消磨時間。

**清水浩介**：哈哈，我也喜歡遊戲。不過現在工作太忙，玩得少了。以前學生時代可是天天泡在遊戲裡。

**李杰**：那時候真好啊，有大把時間可以揮霍。

**清水浩介**：是啊...那時候姐姐還在，雖然她總是嫌我玩遊戲太多，但現在想想，那段時光才是最珍貴的。

**旁白**：氣氛又有些沉重起來。李杰連忙岔開話題。

**李杰**：船橋那邊生活環境怎麼樣？我只去過一兩次。

**清水浩介**：還不錯，比東京市中心便宜很多。而且離海近，週末可以去海邊走走。李先生呢？三鷹怎麼樣？

**李杰**：挺安靜的住宅區。離新宿和秋葉原都算方便，房租也還能接受。

**清水浩介**：那不錯啊。我也想過搬到三鷹附近，但搬家太麻煩了。

**旁白**：笑聲逐漸多了起來，氣氛也變得輕鬆了一些。

**清水浩介**：不管怎麼說，雖然還是沒找到姐姐的線索，但能認識你這樣一位經歷奇特的朋友，也算是一種收穫。

**旁白**：他舉起酒杯，臉上帶著微醺的紅暈。

**清水浩介**：乾杯。

**李杰**：乾杯。

**旁白**：兩個杯子碰在一起，發出清脆的聲響。

**旁白**：午夜時分，兩人都已有了幾分醉意，走路時有些搖晃。

**旁白**：在秋葉原站的月台上，他們互相道別。

**清水浩介**：李先生，如果有什麼新的線索，請一定要告訴我。

**李杰**：會的。你也保重。

**清水浩介**：保持聯繫。

**旁白**：清水浩介揮了揮手，轉身走向開往千葉方向的月台。

**旁白**：李杰看著他的背影消失在人群中，然後走向返回三鷹的月台。

**旁白**：列車很快就進站了。這個時間點，車廂內人很少，只有零星幾個深夜回家的乘客。

**旁白**：他找了一個靠窗的座位坐下。

**旁白**：列車啟動，駛離了秋葉原站。車窗外是東京的夜景——高樓大廈的燈光、街道上的霓虹、還有偶爾閃過的便利商店。

**旁白**：李杰靠在窗邊，看著窗外飛速倒退的城市燈火。

**旁白**：酒意漸漸退去，思緒變得清晰起來。

**李杰**：神樹給了我第二次機會...一個可以當作什麼都沒發生過的機會。

**旁白**：他閉上眼睛，腦中浮現出那些在異世界遇到的人們。

**李杰**：我真的能當作什麼都沒發生過嗎？就這樣心安理得地享受這「偷來」的一週？

**旁白**：然後眼睜睜地看著那些人再次踏上那輛列車，走向死亡或失蹤？

**李杰**：還是...去揭開這一切的真相，即便代價是再次失去現在所擁有的一切安寧？

**旁白**：列車平穩地行駛著，將他帶回安穩的家。

**旁白**：窗外的燈火在他臉上忽明忽暗地閃爍，就像他內心兩種想法的交戰。

**旁白**：逃避，還是面對？

**旁白**：安全，還是真相？

**旁白**：列車穿過一個又一個車站。中野、吉祥寺、西荻窪...

**旁白**：終於，熟悉的廣播聲響起。

**廣播**：まもなく、三鷹、三鷹です。お出口は、右側です。

**旁白**：列車緩緩駛入三鷹站，停靠在月台旁。

**旁白**：李杰站起身，走向車門。

**旁白**：當他踏上月台的那一刻，他緊鎖的眉頭舒展開來。

**旁白**：他已經做出了決定。
