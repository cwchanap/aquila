# 第二十八幕：二十年前的臉孔 

**旁白**：第二天是週一。

**旁白**：李杰醒來後，腦中異常清晰。昨晚在電車上做出的決定，讓他有了一個明確的目標。

**李杰**：在週末的悲劇重演之前，我必須盡可能調查清楚二十年前的真相。

**旁白**：而要做到這一點，第一步，就是確保自己有足夠的時間。

**旁白**：他拿起放在床頭櫃上的手機，找到公司部長的號碼，撥了出去。

**旁白**：電話響了幾聲後接通。

**李杰**：山田部長，早上好。非常抱歉，我想申請從今天開始，請一週的個人長假。

**山田部長**：什麼？！一週？！你腦子沒問題吧李君！

**旁白**：話筒裡傳來部長暴怒的聲音，幾乎要把耳膜震破。

**山田部長**：你知道今天是星期一嗎？你知道這個週末就是客戶的最後交付期限嗎？！整個團隊都在趕進度，你現在跟我說要請一週假？！

**李杰**：我明白這會造成很大的困擾，但我確實遇到了——

**山田部長**：我不想聽你的藉口！你知道如果這個項目延期，公司要賠多少違約金嗎？你一個人的薪水夠賠嗎？！

**旁白**：部長的聲音越來越大，帶著毫不掩飾的憤怒。

**山田部長**：我真是瞎了眼才會招你進來！你這種沒有團隊精神、沒有責任感的人，根本不配在這裡工作！

**李杰**：部長，我真的很抱歉，但這是非常緊急的私人事務——

**山田部長**：私人事務？你給我聽好了，李杰！你不是在請假，你是在毀掉自己的職業生涯！

**旁白**：電話那頭傳來重重的拍桌聲和摔東西的聲音。

**山田部長**：你要是敢不來上班，我就把你的事情報到人事部！解雇、降職、扣獎金，你自己選一個！

**李杰**：我知道後果，但這件事我必須去做。真的非常抱歉。

**山田部長**：你…你…好！很好！那你就別回來了！

**旁白**：部長氣得說話都開始結巴。

**山田部長**：我告訴你，就算你回來，也別想有好日子過！你這輩子都別想在這個業界混了！我會讓所有人都知道你是個什麼樣的廢物！

**李杰**：…我明白了。再次為您帶來的困擾道歉。

**旁白**：電話被狠狠地摔斷了，話筒裡只剩下嘟嘟的忙音。

**旁白**：掛掉電話後，李杰感到一陣輕鬆。雖然得罪了上司，但至少爭取到了時間。

**李杰**：如此大規模的失蹤…不可能在歷史上毫無痕跡。

**旁白**：他迅速換好衣服，簡單吃了點早餐，然後動身前往國會圖書館。

**旁白**：國會圖書館位於永田町，是日本最大的圖書館。雄偉的建築在陽光下顯得莊嚴而肅穆。

**旁白**：李杰走進寬闊的大廳，腳步聲在高挑的天花板下迴盪。

**旁白**：他來到檔案查詢櫃台，一位年長的圖書館員正在整理資料。

**圖書館員**：你好，請問需要什麼幫助？

**李杰**：你好，我想申請查閱二十年前的舊報紙。

**圖書館員**：二十年前？好的。請問有具體的日期和報紙名稱嗎？

**李杰**：日期是2005年9月23日前後那一週。我想看看所有主流報紙，比如朝日、讀賣、每日新聞等等。

**圖書館員**：範圍很廣呢。請問是為了研究特定的課題嗎？比如當時的政經情勢？

**旁白**：圖書館員有些驚訝地抬起頭，眼神中帶著職業性的好奇。

**李杰**：我想調查一件…可能沒有被廣泛報導的交通事故。可能是在中央線上，關於一輛末班列車的。

**圖書館員**：沒有被報導的鐵路事故？那可真是罕見。

**旁白**：圖書館員扶了扶眼鏡，眼神中帶著一絲懷疑。

**圖書館員**：好吧，那個年代的報紙檔案都在縮微膠卷上，我帶你去閱覽室。

**旁白**：李杰跟著圖書館員走進一間安靜的閱覽室。

**旁白**：房間裡擺放著幾台老式的縮微膠卷閱覽器，牆邊是一排排裝滿膠卷盒的櫃子。

**旁白**：圖書館員從櫃子裡取出幾盒膠卷，放在桌上。

**圖書館員**：這些是2005年9月20日到30日之間的主要報紙。閱覽器的使用方法在這裡有說明。如果需要列印，請到櫃台登記。

**李杰**：謝謝您。

**旁白**：圖書館員點點頭，離開了閱覽室。

**旁白**：李杰坐在閱覽器前，深吸一口氣，開始了漫長的搜尋。

**旁白**：他小心地將第一卷膠卷裝進機器，打開電源。螢幕上出現了泛黃的報紙影像。

**旁白**：2005年9月23日，朝日新聞。

**旁白**：頭版是關於政治的報導。他快速地翻閱著每一頁，尋找任何關於列車事故或失蹤案件的新聞。

**旁白**：沒有。

**旁白**：他換上讀賣新聞的膠卷。同樣的日期，同樣的失望。

**旁白**：每日新聞、產經新聞、東京新聞…一份接著一份。

**旁白**：時間一分一秒地流逝。窗外的陽光從明亮變得柔和,再逐漸暗淡。

**李杰**：怎麼會…完全沒有任何報導？

**旁白**：那幾天的新聞，除了政治和社會雜聞，沒有任何關於列車事故或集體失蹤的報導。

**旁白**：彷彿那輛列車和上面的人，就那麼憑空蒸發了。

**李杰**：難道真的沒有留下任何痕跡嗎？

**旁白**：他揉了揉疲憊的眼睛，看了看手機。已經下午三點多了。

**旁白**：他改變了策略。

**李杰**：既然找不到「事件」的報導，那就從「人」開始查起。

**旁白**：他開始專注於報紙社會版的角落，尋找小規模的、不起眼的尋人啟事。

**旁白**：那些通常只有豆腐塊大小的廣告,刊登著失蹤人口的照片和簡單資訊。

**旁白**：9月24日…沒有。

**旁白**：9月25日…沒有。

**旁白**：9月26日…

**旁白**：就在他快要放棄的時候，在社會版的一個毫不起眼的角落，一則小小的尋人啟事抓住了他的視線。

**旁白**：標題是：「尋找失蹤少女，清水恵美」。

**旁白**：李杰的手停在滾輪上,心跳開始加速。

**旁白**：文章旁邊，附著一張黑白的、有些模糊的照片。

**旁白**：照片上的女孩留著長髮，穿著一件淡色的休閒上衣，臉上帶著一絲靦腆的微笑。背景似乎是在某個公園或戶外的地方。

**旁白**：李杰的瞳孔猛地收縮，血液彷彿在這一刻凝固了。

**李杰**：這張臉…

**旁白**：照片上的這張臉，他認識。

**旁白**：雖然照片模糊,雖然已經過去了二十年，但那輪廓、那眼神，他絕對不會認錯。

**旁白**：這正是那個在「寂海」車站，跟著慷慨大叔一起下車的…文靜少女。

**李杰**：原來…她真的是清水恵美。
