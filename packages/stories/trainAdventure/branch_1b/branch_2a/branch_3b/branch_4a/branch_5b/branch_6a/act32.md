# 第三十二幕：二十年後的再會 

**旁白**：聽到清水浩介的呼喊,那個穿著黑色連帽衫的男人猛地抬起了頭。

**旁白**：帽簷下的陰影褪去,露出一張年輕、蒼白而又俊俏的臉。

**旁白**：他看到清水浩介時,眼中同樣充滿了震驚,腳步頓時停住。

**凌墨**：...浩介?是你嗎?你為什麼...會在這裡?

**旁白**：他的聲音有些沙啞,帶著一絲不確定。

**清水浩介**：凌墨!真的是你!

**旁白**：浩介激動得聲音都在顫抖。

**清水浩介**：你的樣子...怎麼一點都沒變!我還以為...我還以為你二十年前就已經...

**旁白**：浩介說著,再也控制不住情緒,上前一步,緊緊地抱住了凌墨。

**旁白**：凌墨的身體僵硬了一下,但沒有抗拒,只是任由浩介抱著他。

**凌墨**：...抱歉,讓你擔心了。

**旁白**：他的聲音輕得幾乎聽不見。

**李杰**：(內心) 清水浩介...認識這個黑衣男?他們是什麼關係?

**李杰**：(內心) 凌墨?這個名字我從沒聽過...

**旁白**：過了一會兒,浩介鬆開了他,但眼眶已經泛紅。

**清水浩介**：這些年你到底去哪了?我找了你好久...

**凌墨**：很複雜。但你還沒說,你為什麼會在這裡?

**旁白**：凌墨的表情變得嚴肅。

**凌墨**：今晚的月台很危險。你不應該來這裡。

**清水浩介**：我...我是來找我姐姐的。

**凌墨**：恵美?你知道她要搭這班車?

**清水浩介**：對!是李先生告訴我的,他說——

**旁白**：李杰知道自己不能再沉默下去了。他走了過去。

**李杰**：是我帶他來的。凌墨先生,你好。我叫李杰。

**李杰**：我知道今晚這班列車會再次出現,我是來阻止悲劇重演的。

**旁白**：凌墨銳利的目光轉向李杰,像是要看穿他的內心。

**凌墨**：你?我從未在二十年前的列車上見過你。

**凌墨**：列車每二十年才出現一次,你又是怎麼知道的?

**李杰**：(內心) 在這個人面前,任何謊言都會被輕易戳破...

**旁白**：李杰深吸一口氣,決定說出真相。

**李杰**：我搭乘的列車...就是我們現在即將要搭上的這一班。

**李杰**：我經歷了即將發生的一切,然後被送回了一週前。

**李杰**：所以,我知道接下來會發生什麼,也知道有誰會在這班車上。

**清水浩介**：等等...時間旅行?

**旁白**：浩介徹底愣住了,難以置信地看著李杰。

**清水浩介**：李先生,你之前為什麼不告訴我?

**李杰**：這件事太過離奇,我怕會讓你混亂。對不起。

**清水浩介**：所以...所以你說在列車上看到我姐姐,其實是指...未來?

**李杰**：準確來說,是已經發生過的未來。

**旁白**：凌墨聽完李杰的話,卻異常平靜。

**凌墨**：...原來如此。

**旁白**：他沉默了片刻,似乎在評估著一切。

**凌墨**：能回到過去...看來那個世界的規則,比我想像的還要複雜。

**李杰**：你...相信我?

**凌墨**：我在那個世界待了二十年,什麼奇怪的事都見過了。時間倒流,不算什麼。

**凌墨**：你說要拯救所有人...你有具體的計畫嗎?

**李杰**：有。但那需要所有人的合作。

**凌墨**：可以。我的目標只有一個,保護恵美。

**凌墨**：如果你的計畫,能確保恵美、她的男友,還有我自己的安全,我就聽你的指揮。

**李杰**：男友?恵美小姐有男友?

**旁白**：李杰驚訝地看向凌墨。

**凌墨**：是的。她的男友,二十年前和我們一起,被困在了那個世界。

**清水浩介**：什麼?!姐姐有男友?我怎麼不知道?

**凌墨**：你那時候才十五歲,她沒告訴你也很正常。

**凌墨**：這也是她執意要再次搭上這班車的原因。她要去救他。

**李杰**：那你們...是怎麼回來的?二十年前的那次。

**旁白**：凌墨的眼神變得黯淡,像是在回憶痛苦的往事。

**凌墨**：二十年前,其實是她的男友犧牲了自己。

**凌墨**：透過某種精靈的古老儀式,打開了回來的門,把恵美送了回來。

**凌墨**：我當時也在場,所以跟著她一起逃了出來。

**清水浩介**：那...那太好了!所以你們都平安回來了?

**李杰**：等等,你們在那個世界待了多久?

**凌墨**：大約一年半左右。為什麼這麼問?

**李杰**：(內心) 一年半...但現實世界已經過了快二十年。

**李杰**：(內心) 那就是說,時間流速不是1比10...而是更接近1比13?

**李杰**：(內心) 我之前的推算不完全正確...

**凌墨**：對。但恵美她...她無法接受男友被留在那裡的事實。

**清水浩介**：她的男友...他怎麼了?

**凌墨**：他啟動儀式後,自己被困在了那個世界。無法逃出來。

**李杰**：你們是什麼時候回來的?

**凌墨**：大約半年前。對這個世界來說。

**旁白**：李杰和浩介都被這段曲折的過往震驚了。

**清水浩介**：所以...姐姐這二十年一直活著?她在哪裡?為什麼不回家?

**凌墨**：她一直對男友被困耿耿於懷,認為是自己害了他。

**凌墨**：所以才想用「交換」的方式去救他出來。

**李杰**：交換?什麼意思?

**凌墨**：獻祭一個無辜的人,來換取她男友的自由。

**凌墨**：讓那個無辜者代替她男友,被永遠困在那個世界。

**旁白**：凌墨的聲音帶著一絲憤怒。

**凌墨**：我一直勸她不要這麼做,犧牲無辜者是錯的。而且她男友也絕對不會希望她這樣做。

**凌墨**：但她完全聽不進去。在那之後,她就斷絕了和我的一切聯繫。

**清水浩介**：姐姐她...怎麼會變成這樣...

**旁白**：浩介的聲音充滿了痛苦和不解。

**凌墨**：所以我只能跟在後面。我不能讓她知道我也在車上。

**凌墨**：否則...我怕我無法阻止她犯下大錯。

**李杰**：(內心) 原來如此...所以凌墨是來阻止恵美的。

**李杰**：(內心) 但如果恵美真的打算用無辜者交換她男友的自由...那個被選中的人會是誰?

**旁白**：就在這時,月台上的廣播響起。

**廣播**：往三鷹方向的末班車即將進站,請乘客們在白線後等候。

**旁白**：遠處,傳來列車駛近的聲音。

**旁白**：那個聲音,在寂靜的深夜裡,顯得格外清晰。
