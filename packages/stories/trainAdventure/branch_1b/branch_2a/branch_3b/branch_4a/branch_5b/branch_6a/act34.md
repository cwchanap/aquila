# 第三十四幕：失控的劇本

**旁白**：列車駛入隧道,車廂內的燈光變得昏暗。

**旁白**：李杰立刻拿出手機,給清水浩介發去了一條訊息。

**旁白**：「一切順利嗎?」

**旁白**：手機震動了一下,浩介的回覆很快傳來。

**旁白**：「沒問題。我和凌墨在後面的車廂,正看著你這邊。恵美沒有起疑。」

**旁白**：「等到了那個世界,我們就現身,到時候就看你的了。」

**李杰**：(內心) 很好,第一步進行得很順利。

**旁白**：李杰稍微鬆了口氣,繼續滑著手機,等待著信號消失的那一刻。

**旁白**：大約五分鐘後,螢幕頂端的信號格突然變灰。

**旁白**：「沒有服務」的字樣跳了出來。

**李杰**：(內心) 差不多了,他們該過來了。

**旁白**：他抬起頭,果然看到凌墨正從後方車廂的連接處,快步向他走來。

**旁白**：但李杰立刻察覺到不對勁——只有凌墨一個人,而且他的臉色異常凝重。

**李杰**：(內心) 清水浩介呢?

**凌墨**：浩介...不見了。

**旁白**：凌墨走到李杰身邊,聲音壓得極低,帶著一絲顫抖。

**李杰**：什麼?!

**旁白**：李杰震驚地站了起來。

**凌墨**：我只記得我們一起上了車,找了個角落的位子坐了下來...然後...我的記憶就出現了一段空白。等我回過神來,手機就沒了信號,坐在我身邊的浩介也消失了。

**李杰**：(內心) 記憶空白?這是什麼情況?

**李杰**：(內心) 難道是列車的某種規則?還是...

**旁白**：李杰感到一股寒意從背後升起。他的計畫在第一步就脫軌了。

**旁白**：他強迫自己冷靜下來,深吸一口氣。

**李杰**：現在車上還有多少人?

**凌墨**：我數過了,包括我們和恵美,一共十六個。

**李杰**：...明白了。

**李杰**：(內心) 十六個人...和上次一樣。

**李杰**：(內心) 但如果算上失蹤的浩介,原本應該是十七個...

**李杰**：(內心) 浩介的失蹤,是列車自動修正人數的機制嗎?

**李杰**：浩介的失蹤可能和列車的規則有關。現在情況變了,沒有浩介,要說服他姐姐會很困難,但我們必須試試。

**凌墨**：好。我們一起去。

**旁白**：凌墨點了點頭,兩人轉身,朝著清水恵美的位置走去。

**旁白**：恵美獨自坐在車廂角落,耳機還戴著,但眼神卻一直盯著窗外的黑暗。

**旁白**：她看到他們走來,臉上沒有絲毫驚訝,反而帶著一絲冰冷的敵意。

**清水恵美**：你果然跟來了。又要阻止我嗎?還有你,你又是誰?為什麼會跟他站在一起?

**李杰**：(內心) 這真的是同一個人嗎?

**李杰**：(內心) 那個在月台上安靜膽怯的女孩...現在的眼神卻如此冰冷、充滿攻擊性。

**李杰**：(內心) 為了救翔...她竟然能變成這樣。

**凌墨**：恵美,聽我說。這位李杰先生,他知道一個能讓所有人都安全離開這個世界的方法,包括翔在內。

**李杰**：(內心) 這傢伙...也太會隨機應變了吧。

**李杰**：(內心) 我可沒說過我能救她的男友。但現在只能先順著他的話說下去了。

**清水恵美**：我憑什麼相信你們?凌墨,你既然把他帶來了,想必已經把我的計畫都告訴他了吧。

**清水恵美**：既然你已經知道了,那你也就不再是個合適的目標了。也好,省得我麻煩。我現在的目標不會是你了。

**旁白**：她的目光轉向李杰,眼神中沒有任何溫度。

**清水恵美**：我不在乎其他人的死活,我只要救翔回來。凌墨,我警告你,別再妨礙我,否則,我就立刻死在你面前。

**旁白**：她的聲音平靜得可怕,像是在陳述一個既定的事實。

**旁白**：凌墨看著她決絕的眼神,臉上閃過一絲痛苦。

**旁白**：他長長地嘆了口氣,點了點頭。

**清水恵美**：現在,離我遠點。別讓其他人注意到我們。

**旁白**：李杰也嘆了口氣。他知道,現在絕不是公開計畫的好時機。

**旁白**：他和凌墨默默地退到了一旁,回到自己的座位。

**李杰**：(內心) 她已經徹底被執念控制了...

**李杰**：(內心) 這樣下去,根本無法說服她。

**旁白**：就在這時,李杰突然想起了什麼。

**李杰**：(內心) 等等...上一次...我在第一條時間線裡,只走到了第五節車廂就回頭了。

**李杰**：(內心) 後面的車廂,我從來沒檢查過。會不會有什麼線索?

**李杰**：這樣下去不是辦法。我想去後面的車廂看看,找找有沒有別的線索。你留下來,再試試看能不能和她溝通。

**凌墨**：好,你去吧。小心點。

**旁白**：李杰獨自一人,開始往列車後方走去。

**旁白**：一節、兩節、三節...每一節車廂都一模一樣,空無一人,只有昏暗的應急燈在微微閃爍。

**旁白**：窗外是純粹的黑暗,玻璃上只映照出他自己緊張的臉孔。

**旁白**：在這封閉的空間裡,只聽得見列車行駛時單調的「喀噠」聲,還有他自己越來越沉重的腳步聲。

**李杰**：(內心) 這輛列車...到底有多長?

**旁白**：這輛列車就像一個無盡的、移動的墳墓。

**旁白**：越往後走,那股詭異的壓抑感就越是強烈。

**旁白**：第六節、第七節、第八節...

**旁白**：他一路走到了第十節車廂。

**旁白**：在車廂的角落,他看到了什麼東西。

**旁白**：那是一柄劍。通體漆黑的長劍,靜靜地躺在那裡。它沒有劍鞘,劍身彷彿能吸收周圍所有的光線。

**旁白**：散發著一股不祥的、令人心悸的氣息。

**李杰**：(內心) 這是...什麼?

**旁白**：李杰慢慢走近,盯著那把劍。

**旁白**：一股奇異的衝動在他心中升起。那柄劍...彷彿在呼喚他,讓他去撿起來。

**李杰**：(內心) 為什麼...我會有這種感覺?

**旁白**：但他同時也感到一種強烈的直覺——

**旁白**：一旦他拿起這柄劍,事情就會徹底失控。

**旁白**：他站在原地,陷入了掙扎。

**李杰**：(內心) 到底...該不該拿起這柄劍?

**李杰**：(內心) 這可能是一個陷阱...但也可能是一個機會。

**旁白**：他的手伸了出去,又縮了回來。黑劍在昏暗的燈光下,泛著詭異的光澤。
