# 第三十五幕：第一滴血

**旁白**：李杰站在那柄散發著不祥氣息的黑劍前,內心掙扎了許久。

**旁白**：他的手伸出去,停在半空中,手指微微顫抖。

**李杰**：(內心) 不...不能碰。

**李杰**：(內心) 我的直覺告訴我,這東西很危險。一旦拿了,就沒有回頭路了。

**旁白**：他最終還是選擇相信自己的直覺,移開了視線。

**旁白**：他環顧四周,在車廂的座位底下,意外地發現了一些東西。

**旁白**：一個打火機、一支小巧但光線很足的手電筒，還有一把菜刀。

**李杰**：(內心) 這些東西...是之前的乘客留下的嗎?

**旁白**：菜刀的刀刃寬而鋒利，刀柄沉重。看起來是從餐車遺落的。

**旁白**：他將這三樣工具放進口袋。打火機和手電筒很容易就放進去了，菜刀則插在腰間的皮帶上。

**旁白**：比起那把詭異的黑劍，這些實用的工具更讓他安心。

**旁白**：然後他試圖拉開通往下一節車廂的門。

**旁白**：門紋絲不動。被鎖死了。

**李杰**：(內心) 第十節之後就去不了了嗎...

**旁白**：就在這時,他感覺到列車的震動變得輕微。

**李杰**：(內心) 列車...減速了?已經駛出隧道了嗎?

**旁白**：他決定立刻返回列車前方。

**旁白**：當他走到第五節車廂時,正好迎面遇上了同樣走過來的凌墨。

**凌墨**：有什麼發現嗎?

**旁白**：凌墨看到他,立刻問道。

**李杰**：在第十節車廂找到了一些不太起眼的工具。還有一柄感覺很詭異的黑劍。

**李杰**：如果可以的話,最好別讓任何人靠近那節車廂。我怕有人會把它撿起來。

**凌墨**：黑劍?

**旁白**：凌墨雖然眼中閃過一絲好奇,但還是點了點頭,沒有多問。

**凌墨**：我這邊...還是沒能說服恵美。

**凌墨**：而且...我覺得她有點奇怪,比我印象中的她,要固執得多。

**李杰**：我也有同感。我記憶中的她...和這次給我的感覺,完全不同。

**李杰**：(內心) 到底是哪裡不對勁?

**旁白**：就在兩人交談時,列車平穩地停了下來。

**旁白**：沒有任何廣播,車門「嘶」的一聲,自動打開了。

**李杰**：到「寂海」站了。我們先回第一節車廂。

**旁白**：兩人快步往回走。

**旁白**：然而,當他們剛踏入第二節車廂時——

**旁白**：一陣淒厲的女性尖叫聲和孩子們的哭喊聲,猛地從第一節車廂傳來!

**旁白**：兩人臉色一變,立刻朝著第一節車廂衝了過去。

**李杰**：(內心) 發生什麼事了?!

**旁白**：當他們衝進車廂時,眼前混亂的景象讓他們徹底呆住了。

**旁白**：清水恵美正痛苦地倒在地板上,腹部插著一把小刀。

**旁白**：鮮血正不斷地從傷口湧出,染紅了她淺色的連衣裙。

**旁白**：那個多話女大生則被慷慨大叔死死地按在地上。

**旁白**：沉穩男大生也在一旁幫忙壓制住她的手腳。

**多話女大生**：放開我!讓她去死!都是她!都是這個女人害的!

**旁白**：她瘋狂掙扎著,歇斯底里地尖叫。

**慷慨大叔**：冷靜點,小姑娘!妳冷靜點!到底發生什麼事了?!

**旁白**：慷慨大叔用力按住她,大吼道。

**沉穩男大生**：美咲!清醒一點!妳怎麼了,發生什麼事了?!

**旁白**：他的聲音帶著焦急和不解。

**旁白**：旁邊,那個活潑女大生和那對懂事的姐弟,都嚇得縮在角落裡大哭。

**旁白**：其他的乘客則驚恐地遠遠躲開,交頭接耳。

**煩躁丈夫**：搞什麼鬼啊!那個女孩瘋了嗎?!

**焦慮妻子**：老公你別過去!危險!

**膽小眼鏡男**：快、快報警啊...啊,沒信號...

**旁白**：李杰的目光快速掃過那些驚恐的臉孔。

**旁白**：但其中一張臉的表情卻格格不入。

**旁白**：那個自私中年男,他的嘴角,竟然掛著一絲若有似無的冷笑。

**李杰**：(內心) 那個男人...他在笑?

**凌墨**：恵美!恵美你怎麼樣!

**旁白**：凌墨看到恵美倒下,瞬間衝了過去,跪在她身邊。

**清水恵美**：...凌墨...好痛...

**旁白**：她痛苦地呻吟,抓住了凌墨的手。她的臉色蒼白,嘴唇已經沒有血色。

**凌墨**：撐住!不要睡!我馬上幫你止血!

**旁白**：凌墨的聲音在顫抖,他慌亂地脫下自己的外套,按在恵美的傷口上。

**李杰**：(內心) 這到底是怎麼回事?

**李杰**：(內心) 計畫從第一步開始就完全失控了。

**李杰**：(內心) 為什麼會是那個女大學生攻擊了恵美?

**李杰**：(內心) 還有...那個男人臉上的冷笑又是怎麼回事?

**旁白**：李杰感到一陣眩暈。所有的計畫,所有的準備,在這一刻都變得毫無意義。

**旁白**：第一滴血已經流下。而這,只是開始。
