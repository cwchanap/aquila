# 第三十七幕：絕境中的隊伍

**旁白**：李杰立刻衝上前,一把抓住凌墨握著刀的手腕。

**李杰**：凌墨!冷靜下來!

**旁白**：凌墨的臉上沒有任何表情,眼神空洞。

**旁白**：他只是用一種沒有起伏的語氣,反覆地說著。

**凌墨**：我不會...放過任何...想傷害她的人...

**旁白**：他開始用力掙扎,想要甩開李杰和慷慨大叔的手。

**旁白**：他的力氣不大,但那股偏執的勁頭卻讓人心驚。

**旁白**：他轉頭用冰冷的眼神瞪著慷慨大叔,嘶啞地吼道。

**凌墨**：放手!

**旁白**：這次,慷慨大叔再也壓不住怒火。

**慷慨大叔**：你這小子!

**旁白**：他怒吼一聲,猛地甩開凌墨的手。

**旁白**：然後毫不猶豫地,一記沉重的右勾拳,狠狠地打在了凌墨的下巴上。

**旁白**：凌墨悶哼一聲,雙眼一翻,立刻軟倒下去。

**旁白**：李杰眼疾手快地扶住了他,將他拖出了車廂。

**慷慨大叔**：把他看好...我看這裡所有人都快瘋了。

**旁白**：慷慨大叔喘著粗氣,對李杰說。

**旁白**：李杰將昏過去的凌墨平放在月台的地面上。

**旁白**：然後立刻轉身回到車廂,查看清水恵美的狀況。

**旁白**：她也因為失血過多而昏迷了,但幸好,呼吸還算平穩。

**李杰**：(內心) 至少...還活著。但必須盡快找到急救用品...

**旁白**：另一邊,慷慨大叔正在查看沉穩男大生的傷勢。

**慷慨大叔**：還好,這小子捅得不深,沒有傷到要害。但必須馬上處理。

**李杰**：我在車站裡發現了一張地圖,上面標示著森林裡有一棟小屋。

**李杰**：我不確定那裡有沒有急救箱,但這是我們唯一的希望了。

**慷慨大叔**：好!我跟你一起去!

**旁白**：慷慨大叔立刻下定決心。

**李杰**：我來負責清水小姐。

**慷慨大叔**：那這個小子就交給我了。

**旁白**：他說著,走到凌墨身邊,吃力地將他背了起來。

**健談男大生**：我來扶著拓海!他還能走。

**旁白**：健談男大生主動站出來。

**沉穩男大生**：...麻煩你了,健太。

**旁白**：沉穩男大生咬著牙,對健談男大生點了點頭。

**旁白**：其他人依舊驚恐地看著這一切,不敢作聲。

**活潑女大生**：那...我留下來看著美咲...

**旁白**：她小聲地說,聲音還在顫抖。

**膽小眼鏡男**：咳...這裡也需要有人主持大局。

**膽小眼鏡男**：我就留下來,整合剩下的人。

**旁白**：他故作鎮定地清了清喉嚨。

**李杰**：(內心) 這傢伙...明明是害怕才不敢走吧。

**旁白**：那對夫婦只是緊緊地抱著自己的孩子,不斷地安撫著。

**旁白**：李杰的目光掃過留在車上的人。

**旁白**：再次看到了那個自私中年男...

**旁白**：他依然坐在角落,臉上那絲若有似無的冷笑,彷彿在欣賞一場有趣的戲劇。

**李杰**：(內心) 這個男人...絕對有問題。但現在沒時間管他了。

**旁白**：李杰快速地清點了一下準備離開的人數。

**李杰**：(內心) 我、清水小姐、凌墨、大叔,還有兩個男大學生...一共六個人。

**李杰**：(內心) 車上還剩下十個人...還差兩個人,列車才會開走吧。

**李杰**：(內心) 不過現在也顧不了那麼多了。

**旁白**：他沒有時間再猶豫。

**旁白**：小心翼翼地將昏迷的清水恵美橫抱起來,盡量避免觸碰到她腹部的傷口。

**旁白**：然後對其他人喊道。

**李杰**：我們走!

**旁白**：慷慨大叔背著凌墨,健談男大生扶著沉穩男大生。

**旁白**：一行六人的臨時隊伍,就這樣走下了列車。

**旁白**：月台外,是一片深不見底的黑暗森林。

**旁白**：兩輪紫月高懸在天空,投下詭異的光芒。風吹過樹林發出陰森的沙沙聲。

**李杰**：(內心) 這條路...真的是對的嗎?

**旁白**：他抱著恵美,踏入了黑暗之中。

**旁白**：身後,列車的燈光越來越遠,最終消失在視線裡。

**旁白**：他們,消失在了「寂海」站外那片深不見底的黑暗森林之中。
