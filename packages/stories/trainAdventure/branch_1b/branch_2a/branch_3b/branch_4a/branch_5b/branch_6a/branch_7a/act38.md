# 第三十八幕：扭曲的心智

**旁白**：一行六人艱難地在漆黑的森林中前進。

**旁白**：李杰橫抱著恵美,每一步都走得異常小心,手臂已經開始發酸。

**旁白**：慷慨大叔背著昏迷的凌墨,呼吸沉重。

**旁白**：健談男大生則攙扶著腹部受傷的沉穩男大生,深一腳淺一腳地跟在後面。

**旁白**：四周除了他們沉重的呼吸聲和腳步聲,只剩下死一樣的寂靜。

**旁白**：紫月的光芒透過樹葉的縫隙灑下,在地面投下詭異的斑駁光影。

**慷慨大叔**：喂,年輕人們...還沒請教你們的名字。我叫山本浩二。

**旁白**：山本為了打破這壓抑的氣氛,開口道。

**李杰**：我叫李杰。

**健談男大生**：我叫田中健太,他是佐藤拓海。

**李杰**：我懷裡這位是清水小姐,山本先生你背上的是凌墨。

**山本浩二**：清水...這姓氏倒是很常見。

**李杰**：(內心) 他不知道清水恵美的事...也好,現在不是解釋的時候。

**旁白**：又走了一段路後,山本背上的凌墨發出了一陣痛苦的呻吟。

**旁白**：他緩緩睜開了眼睛。

**山本浩二**：你醒了?小子,還好吧?

**凌墨**：我沒事...放我下來。

**旁白**：凌墨意識到自己被人背著,立刻掙扎起來。

**旁白**：他從山本的背上滑了下來,雖然腳步還有點虛浮,但還是站穩了。

**旁白**：李杰將所有人的名字又向他介紹了一遍。

**旁白**：凌墨的目光在健太和拓海身上掃過,最終停留在拓海痛苦的臉上。

**旁白**：他的眼神複雜,充滿了愧疚與困惑。

**凌墨**：我剛才...怎麼會那麼失控...

**旁白**：他低聲地說,聲音帶著自責。

**李杰**：那不像你,你當時完全像變了個人。

**凌墨**：對不起...佐藤。

**旁白**：凌墨轉向拓海,深深地鞠了一躬。

**佐藤拓海**：沒關係...追根究底,是我沒有看好美咲,讓你擔心清水小姐的安危了。是我的責任。

**旁白**：拓海按著傷口,搖了搖頭。

**李杰**：美咲?就是那個...動手的女孩?到底發生了什麼?

**佐藤拓海**：是的。在你們離開後...清水小姐走過來跟我說,她知道離開這裡的方法。

**佐藤拓海**：但需要我的幫助,作為交換,她可以帶我一起回家。

**李杰**：(內心) 她改變目標了...原本她在觀察我,但後來卻選了佐藤。

**佐藤拓海**：美咲看到了,她以為...清水小姐要騙我,情緒突然變得很激動。

**佐藤拓海**：就衝了上去跟她爭吵。爭吵中,美咲突然就從座位底下拿起一把刀...

**佐藤拓海**：刺向了清水小姐...

**旁白**：拓海的聲音低沉,帶著痛苦。

**旁白**：李杰和凌墨靜靜地聽著。

**李杰**：(內心) 清水小姐為什麼會這麼直接?這和上次那個謹慎的她完全不同...

**李杰**：(內心) 是因為知道我識破了她的計畫,所以狗急跳牆了嗎?

**李杰**：(內心) 但總覺得...還是很奇怪。

**李杰**：那把刀是哪來的?

**佐藤拓海**：不知道,就好像突然出現在那裡一樣。

**旁白**：拓海搖了搖頭。

**田中健太**：對!我發誓那裡之前絕對沒有刀!

**山本浩二**：我也沒見過。

**佐藤拓海**：而且美咲...她平時雖然話多,但絕不是會動刀的人。

**佐藤拓海**：她今天在車上一直很奇怪。

**旁白**：一直沉默的凌墨,終於開口了。

**凌墨**：不只是高橋...我也一樣。

**凌墨**：還有恵美,也很奇怪。我到現在都無法相信,我會對你動手...

**凌墨**：那種憤怒,完全不受控制。

**旁白**：他的目光掃過李杰、健太和山本。

**凌墨**：但奇怪的是,李杰、田中、佐藤,還有山本先生...你們好像都很正常。

**凌墨**：這輛列車上...一定有什麼東西,在影響我們的心智。

**李杰**：(內心) 影響心智...這麼說的話,多話女大生、凌墨、清水小姐...

**李杰**：(內心) 他們的行為確實都很異常。但為什麼有些人會被影響,有些人不會?

**山本浩二**：你是說...有什麼看不見的力量,在控制人的情緒?

**凌墨**：我不確定。但這是我能想到的唯一解釋。

**田中健太**：這、這也太恐怖了吧...那我們會不會也...

**凌墨**：不知道。但如果我再次失控,請不要猶豫,立刻制止我。

**李杰**：我們會的。

**旁白**：一行人繼續在黑暗中前進,每個人的心中都蒙上了一層陰影。
