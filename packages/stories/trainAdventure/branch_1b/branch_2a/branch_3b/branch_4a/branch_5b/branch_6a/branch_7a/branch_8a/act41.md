# 第四十一幕：長老的治癒

**旁白**：四人在濃霧中快速趕路，凱蘭在前開路，莉拉在後警戒。

**旁白**：李杰的手電筒光束在濃霧中晃動，他的呼吸急促，每一步都踩得飛快。

**旁白**：一個多小時後，他們終於回到了那棟林中小屋。昏黃的燈光從窗戶透出，在濃霧中顯得格外溫暖。

**旁白**：當李杰一行人靠近時，屋門從內側被警惕地打開，凌墨的身影出現在門口。

**旁白**：當他看到李杰身後跟著三名精靈時，他緊繃的臉上終於露出了一絲如釋重負的表情。

**李杰**：這位是艾拉如先生，他是來幫忙的。

**旁白**：艾拉如點了點頭，徑直走進屋內。

**旁白**：他的目光掃過在椅子上休息的拓海，然後落在了床上。

**旁白**：當他同時看到床上昏迷不醒的恵美和站在一旁的凌墨時——

**旁白**：這位年長精靈溫和的眼眸中，閃過了一絲極快的、幾乎無法察覺的震驚。

**旁白**：但他立刻恢復了平靜，快步走到床邊。

**凌墨**：艾拉如長老，她怎麼樣？

**旁白**：凌墨的聲音因為焦急而微微發顫。

**艾拉如**：傷得很重，生命力正在不斷流失。我不能保證一定能救活她，但我會盡我所能。

**旁白**：艾拉如輕輕掀開覆蓋在恵美身上的外套，仔細查看傷口。

**旁白**：刀刃還插在腹部，周圍的布料已經被鮮血浸透成暗紅色。恵美的臉色蒼白如紙，呼吸微弱得幾乎聽不見。

**旁白**：艾拉如深吸了一口氣，將他乾枯的手掌，輕輕地懸停在傷口的上方。

**旁白**：他閉上眼睛，口中開始吟唱起一連串古老而又充滿韻律的咒文。

**旁白**：那聲音低沉而悠遠，像是從時間的深處傳來，帶著一種說不出的神聖感。

**旁白**：柔和的白光從他的掌心匯聚，如同一輪小太陽，照亮了恵美蒼白的臉。

**旁白**：光芒越來越亮，整個小屋都被照得如同白晝。溫暖的氣息在空氣中流動，驅散了屋內的陰冷。

**旁白**：在光芒的照耀下，所有人都能清楚地看到——

**旁白**：傷口處的流血速度正在減緩，從湧出變成滲出，最終奇蹟般地完全停止了。

**旁白**：李杰屏住呼吸，眼睛一眨不眨地盯著那個傷口。

**李杰**：（內心）這就是...精靈的治癒術嗎？這種力量...簡直不可思議。

**旁白**：凌墨緊緊握著拳頭，指甲都掐進了手心。他的眼睛濕潤了，但努力不讓眼淚掉下來。

**旁白**：艾拉如停止了吟唱，光芒也隨之散去。

**旁白**：他重重地喘了幾口氣，額頭上滲出了細密的汗珠，整個人看起來蒼老了幾分。

**莉拉**：長老，您還好嗎？

**旁白**：莉拉立刻上前扶住他。

**艾拉如**：無妨...只是對我這把老骨頭來說，神術的負擔還是有些沉重。

**旁白**：他擺了擺手，但手指微微顫抖著。

**旁白**：李杰上前查看，恵美依舊處於深度昏迷，傷口看起來仍然猙獰可怖。

**旁白**：但好消息是，血總算是止住了。她的呼吸也變得平穩了一些，臉色不再那麼蒼白。

**艾拉如**：她的生命暫時穩住了，但傷勢太重，恐怕需要靜養好幾個星期才能醒來。

**旁白**：艾拉如從木盒裡拿出一些由葉子製成的特殊繃帶。

**旁白**：那些葉子薄如蟬翼，散發著淡淡的綠光，觸感柔軟而溫暖。

**旁白**：他小心翼翼地為恵美包紮好傷口，動作輕柔得像是在處理易碎的珍寶。

**旁白**：他又取出一個小瓶子，遞給李杰。

**旁白**：那是一個用透明水晶雕刻而成的小瓶，裡面裝著閃著銀色光澤的液體。

**艾拉如**：這是神樹的露水，每天餵她一滴，連續四週，能幫助她恢復元氣。

**凌墨**：謝謝您，長老。真的...不知道該怎麼感謝您。

**旁白**：凌墨深深地鞠了一躬，聲音哽咽。

**艾拉如**：不必客氣。等她康復後，歡迎你們隨時來月光林地作客。

**旁白**：說完，艾拉如便在凱蘭和莉拉的護送下，轉身離開了小屋。

**旁白**：他的背影在濃霧中逐漸模糊，最終完全消失。

**旁白**：李杰走到窗邊，看著三個精靈消失的方向，心中五味雜陳。

**李杰**：凌墨，你之前...見過精靈嗎？

**凌墨**：嗯，兩年前，我見過艾拉如長老。但他從沒提過自己會使用這種治癒神術。

**李杰**：（內心）艾拉如看到恵美和凌墨時，那一瞬間的震驚...絕對不是我看錯了。

**李杰**：（內心）他認識他們嗎？還是...看到了什麼我看不到的東西？

**旁白**：就在這時，小屋的門突然被人猛地從外面推開了！

**旁白**：健太和那個活潑女大生氣喘吁吁地衝了進來，帶進來一陣冷冽的霧氣。

**旁白**：而在他們身後，還跟著一個小小的、瑟瑟發抖的身影。

**旁白**：李杰定睛一看，那正是小林家的女兒，小林陽菜。

**旁白**：小女孩的臉上滿是淚痕，衣服也濕透了，整個人看起來嚇壞了。

**李杰**：小妹妹，你怎麼會在這裡？你的家人呢？

**李杰**：（內心）是陽菜...她怎麼會一個人在森林裡？小林先生和太太呢？

**旁白**：他立刻走上前，蹲下身想查看小女孩的狀況。

**旁白**：但小林陽菜看到陌生人靠近，嚇得立刻往後退了一步，躲到健太身後。

**旁白**：她警惕地看著李杰，眼中滿是恐懼和不信任。
