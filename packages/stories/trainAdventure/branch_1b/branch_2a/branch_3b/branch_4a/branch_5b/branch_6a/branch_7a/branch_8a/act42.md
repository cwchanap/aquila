# 第四十二幕：失散與重逢

**旁白**：屋內所有人的目光，瞬間都聚焦在了門口的三人身上。

**旁白**：李杰看到那個小女孩時，心頭一震。

**李杰**：（內心）小林陽菜...她果然也在這裡。但她怎麼會離開列車？小林先生和太太呢？

**旁白**：坐在椅子上休息的拓海，在看到門口的健太和明日香時，臉上露出了極度驚訝的表情。

**佐藤拓海**：健太？明日香？還有...這位是...你們怎麼會？！

**旁白**：明日香一進門，目光就立刻鎖定了捂著腹部、臉色蒼白的拓海。

**鈴木明日香**：拓海！你的傷...還好嗎？！

**旁白**：她快步走上前，眼中滿是擔憂。

**旁白**：但隨即意識到屋內還有其他陌生人，她連忙停下腳步，有些尷尬地鞠了一躬。

**鈴木明日香**：啊...對不起，忘了介紹。我叫鈴木明日香...這是陽菜妹妹。

**旁白**：她輕輕拉過躲在健太身後的小女孩。

**李杰**：等等，田中。山本先生呢？他不是跟你們在一起嗎？

**旁白**：聽到這個問題，健太的臉色沉了下去，明日香也再次露出了害怕的表情。

**田中健太**：我們...先是找到了她們。

**旁白**：健太深吸了一口氣，開始講述他們離開小屋後發生的事。

**田中健太**：我們花了快一個小時才走回那個岔路口，那裡的霧濃得嚇人。

**田中健太**：然後...我們就看到明日香和陽菜正蹲在一棵大樹下，陽菜一直在哭。

**旁白**：明日香看到健太時，立刻衝了過去，緊緊地抱住了他。

**鈴木明日香**：健太！我好怕...我還以為再也見不到你們了...

**旁白**：她的聲音帶著哭腔，整個人都在顫抖。

**田中健太**：沒事了，明日香，抱歉，讓妳擔心了。

**旁白**：健太輕輕拍著她的背，低聲安慰著。

**旁白**：等明日香情緒稍微平復後，她講述了列車上發生的事。

**鈴木明日香**：在你們離開後，車廂裡一片死寂。過了一會兒，那個自私的男人突然站起來，說他要去後面的車廂看看有沒有吃的。

**鈴木明日香**：那個戴眼鏡的聽到了，也猶豫了一下跟了過去。

**旁白**：明日香停頓了一下，似乎在回憶那個令她不安的場景。

**鈴木明日香**：我又渴又餓，想著美咲也被綁起來了，就跟小林太太說了一聲，下車去車站裡找找看有沒有販賣機之類的東西。

**鈴木明日香**：但車站裡什麼都沒有。就在我準備放棄回車上時，突然聽到月台上有個女孩在喊爸爸媽媽。

**鈴木明日香**：我跑過去一看，就看到陽菜跪在地上，而那輛列車...已經關上門，慢慢開走了。

**旁白**：明日香指了指身邊的陽菜，眼中閃過一絲心疼。

**旁白**：小女孩聽到這裡，眼眶又紅了，緊緊抓著明日香的衣角。

**鈴木明日香**：她說，她也是想下車找點吃的，她媽媽本來想攔住她，但她說「那個大姐姐已經在那裡了，應該沒事」。

**鈴木明日香**：可她剛下車，車門就突然關了起來。她爸爸媽媽在車裡發瘋似地捶門，但一點用都沒有...

**鈴木明日香**：列車就那樣開走了，把她一個人留在了月台上。

**旁白**：屋內的氣氛變得更加沉重。凌墨緊緊握著拳頭，李杰的眉頭也皺得更深了。

**李杰**：（內心）列車...又帶走了人。這次是小林先生和太太，還有翔太。

**李杰**：（內心）陽菜被留下了...這到底是隨機的，還是有什麼規則？

**鈴木明日香**：我嚇壞了，只能安慰她，說我們順著路走，一定能找到你們。

**鈴木明日香**：但霧越來越大，陽菜就開始哭了...我們走到那個岔路口，完全不知道該走哪條路，只能待在原地，希望你們能找到我們...

**旁白**：明日香說到這裡，聲音又哽咽了。

**旁白**：李杰靜靜地聽完了這一切，再次追問道。

**李杰**：那山本先生呢？

**田中健太**：我們找到她們之後，情況就變了。

**田中健太**：山本先生說，帶著她們去走未知的路太危險了。他讓我們必須先把她們送回小屋，和你們會合，確保她們的安全。

**旁白**：健太的聲音帶著明顯的自責。

**田中健太**：我問他那你怎麼辦，他說他一個人行動比較快，目標也小，去那條通往「幽冥之都」的路偵查一下情況。

**田中健太**：他說之後會自己回小屋來找我們，讓我們別管他，先把人送到安全的地方。所以...我就只能按照他說的，先把她們帶回來了。

**旁白**：屋內陷入了片刻的沉默。窗外的濃霧依舊籠罩著森林，看不到任何盡頭。

**旁白**：小屋內，現在聚集了七個人——李杰、凌墨、健太、拓海、明日香、陽菜，還有昏迷的恵美。

**旁白**：而在濃霧深處的某個地方，山本浩二正獨自前行。
