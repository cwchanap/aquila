# 第四十五幕：重逢的小屋

**旁白**：李杰站在廢棄車站的出口，前方是通往未知命運的岔路口。

**旁白**：濃霧中，三條小徑延伸向不同的方向，都看不到盡頭。

**李杰**：（內心）現在有兩個方向...「亡者峽谷」或「銀月湖」。

**李杰**：（內心）以山本大叔的個性，他可能會考慮跟隨列車的主線，去「亡者峽谷」看看。

**李杰**：（內心）但是...我們已經經過了兩個廢棄的車站了，「寂海」和「幽冥之都」都空無一人。

**旁白**：李杰閉上眼睛，努力站在山本浩二的立場思考。

**李杰**：（內心）一個理性的人，不會再把希望寄託在下一個同樣詭異的車站上。

**李杰**：（內心）他現在的首要目標是尋求援助。一個叫「湖」的地方，聽起來遠比另一個廢棄車站更有可能找到生機或資源。

**李杰**：（內心）我賭他會選擇更有希望的地方。

**旁白**：他睜開眼睛，目光堅定。深吸一口氣，轉身走進了那條通往西邊「銀月湖」的小徑。

**旁白**：這條路比之前走過的任何一條都更加崎嶇。

**旁白**：濃霧依舊沒有散去，能見度極低。腳下的路也從泥土變成了濕滑的青苔和尖銳的碎石。

**旁白**：每走一步都要格外小心，好幾次差點滑倒。

**旁白**：李杰的雙腿像灌了鉛一樣沉重，呼吸也越來越困難。

**旁白**：手電筒的光已經開始變暗，電池快要耗盡了。

**旁白**：在這樣的環境中又獨自走了近三個小時後，他的體力已經接近極限。

**旁白**：就在他幾乎要撐不住的時候——

**旁白**：前方濃霧的盡頭，隱約透出了一個熟悉的黑色輪廓。

**李杰**：（內心）是小屋！

**旁白**：那是一棟和之前一模一樣的林中小屋。木質的牆壁，小小的窗戶，煙囪靜靜地矗立著。

**旁白**：李杰加快腳步，幾乎是跑著衝到了門前。

**旁白**：他輕輕敲了敲門，壓低聲音問道。

**李杰**：有人在嗎？

**旁白**：屋內沒有任何回應。但透過門縫，可以聽到一陣陣沉重而有規律的鼾聲。

**旁白**：李杰心中一喜，小心翼翼地推開了沒有上鎖的門。

**旁白**：門發出輕微的吱呀聲。屋內昏暗，只有從窗戶透進來的微弱月光。

**旁白**：果然，山本浩二正躺在屋內唯一的床上，睡得正香。

**旁白**：他的背包放在床邊，外套掛在椅子上，整個人看起來很放鬆。

**李杰**：（內心）...猜對了。

**旁白**：李杰長長地舒了口氣，積累的緊張和疲憊一下子湧了上來。

**旁白**：他的腿軟了一下，差點跌倒，趕緊扶住了門框。

**旁白**：看著山本浩二疲憊的睡臉，李杰決定不吵醒他。

**旁白**：恵美的傷勢暫時穩住了，現在最重要的是恢復體力。

**旁白**：他悄悄走進屋內，關上了門。

**旁白**：從屋內的水罐中將自己的水袋重新灌滿，咕嚕咕嚕地喝了好幾大口。

**旁白**：然後掏出能量棒，大口大口地吃了起來。

**旁白**：吃完後，他在床邊的一張椅子上坐下，靠著牆壁，閉上了眼睛。

**旁白**：幾乎是一瞬間，他就沉沉地睡了過去。

**旁白**：不知過了多久，他感覺有人在輕輕地搖晃他的肩膀。

**旁白**：他迷迷糊糊地睜開眼，視線一片模糊。

**旁白**：漸漸聚焦後，看到山本浩二正一臉關切地看著他。

**山本浩二**：李杰，醒醒！你怎麼會在這裡？

**旁白**：李杰揉了揉眼睛，讓自己清醒過來。喉嚨有些乾澀。

**李杰**：山本大叔...

**山本浩二**：我們分開之後，到底發生了什麼？清水小姐她們呢？

**李杰**：我們分開後，我繼續往森林深處走，找到了一個精靈村莊。

**李杰**：村莊的長老會使用治癒神術。我帶他回到了大家所在的小屋，他用神術穩住了清水小姐的傷勢。

**李杰**：但她需要很長時間的靜養才能醒來。佐藤和凌墨也都安全。

**李杰**：我是聽田中說你往「幽冥之都」的方向去了，才出來找你的。

**山本浩二**：太好了...真是太好了...

**旁白**：山本浩二鬆了口氣，一屁股坐在床邊，雙手撐著膝蓋。

**旁白**：他的眼眶有些濕潤，但努力不讓眼淚掉下來。

**李杰**：對了，「幽冥之都」車站的地圖，是你拿走的吧？

**山本浩二**：嗯，沒錯。我也是靠著它才找到這裡的。

**旁白**：他說著，從口袋裡掏出了那張地圖，小心地展開。

**旁白**：李杰接過地圖，藉著從窗戶透進來的月光仔細研究起來。

**旁白**：地圖上的線條和文字已經有些模糊，但還能勉強辨認。

**李杰**：你看，我們現在在這裡...離地圖上標示的銀月湖很近。

**李杰**：而且，地圖上畫著一條從銀月湖通往月光林地的路。

**李杰**：如果我們從這條路走，再從精靈村莊走我們來時的路...說不定比原路返回要近得多。

**旁白**：山本浩二湊過來，仔細看著地圖上的路線。

**山本浩二**：既然有路，而且靠近你說的那個村莊，應該會安全一些。好，就聽你的。

**李杰**：那我們走吧，先去銀月湖看看。

**旁白**：兩人收拾好東西，將水袋灌滿，確認裝備齊全。

**旁白**：推開小屋的門，外面的濃霧依舊濃密，但至少現在，他們不再是一個人了。

**旁白**：兩個人的腳步聲在濃霧中響起，朝著銀月湖的方向前進。
