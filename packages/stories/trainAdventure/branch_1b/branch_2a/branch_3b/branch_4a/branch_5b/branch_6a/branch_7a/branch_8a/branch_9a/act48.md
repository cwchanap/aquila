# 第四十八幕：瘋狂的征服者

**旁白**：李杰緩緩站起身,環顧廣場四周。

**旁白**：在廣場的另一側,有幾個精靈站在一個老者的屍體旁邊。

**旁白**：他們的臉上滿是悲痛和憤怒,有些人還在流淚。

**旁白**：而在距離他們幾米遠的地方——

**旁白**：一個中年男子躺在地上,下半身完全消失了。

**旁白**：從腰部以下,只剩下一片焦黑的痕跡和鮮血。

**旁白**：他的手中,緊緊握著一把通體漆黑、散發著不祥氣息的長劍。

**旁白**：那把劍,李杰認得——正是列車第十節車廂裡的黑劍。

**李杰**：(內心)那把劍...果然被他拿走了...

**旁白**：李杰慢慢走近,想要看清那個男人的臉。

**旁白**：當他看清楚時,心中的憤怒瞬間湧了上來。

**旁白**：那正是列車上那個自私的中年男——那個在清水恵美被刺時,臉上掛著冷笑的男人。

**李杰**：(內心)是他...真的是他...

**旁白**：李杰又看向那些精靈圍著的老者屍體。

**旁白**：當他看清那張臉時,腳步猛地停住了。

**旁白**：那是艾拉如——月光林地的長老,那個用神術救了恵美的精靈。

**旁白**：艾拉如的臉上沒有任何傷口,表情安詳,就像睡著了一樣。

**旁白**：他的銀色長髮散落在地上,白色的長袍一塵不染。

**旁白**：但他的胸口已經沒有了起伏。

**旁白**：就在李杰想要再走近一些時——

**莉拉**：小心!不要靠近那個男人!

**旁白**：一個女性精靈大聲喊道。她正是莉拉——之前和凱蘭一起護送艾拉如的弓箭手。

**旁白**：她的肩膀上纏著染血的繃帶,弓已經斷成兩截,但她依然警戒地盯著那個男人。

d：即使他失去了下半身,他仍然擁有那把劍的力量!不要靠近!

**旁白**：話音剛落,那個男人突然發出一陣刺耳的笑聲。

**自私中年男**：哈哈哈...咳咳...真是悲慘啊...

**旁白**：他的聲音嘶啞而扭曲,帶著病態的愉悅。

**自私中年男**：竟然...竟然輸給了這個老頭...一個快要死掉的老傢伙...哈哈哈...

**旁白**：他說著,又劇烈地咳嗽起來,從嘴角流出黑色的血液。

**旁白**：李杰緊緊咬著牙,指甲深深掐進掌心。

**旁白**：憤怒在胸中燃燒,幾乎要將他吞噬。

**李杰**：你到底是誰?!為什麼要做這些事?!

**旁白**：李杰的聲音在顫抖,但每一個字都清晰有力。

**旁白**：自私中年男轉頭看向李杰,臉上露出一個扭曲的笑容。

**旁白**：他費力地抬起手中的黑劍,在空中虛弱地揮舞著。

**自私中年男**：我?哈哈...我叫中村隼人...

**自私中年男**：一個...想要征服這個世界的人...咳咳...

**旁白**：他的聲音雖然虛弱,但語氣中卻充滿了病態的自豪。

**中村隼人**：我想要佔領這裡...成為這個世界的王...咳咳...

**旁白**：他的聲音越來越弱,呼吸變得急促。

**中村隼人**：可惜...咳...好痛...失去下半身真的好痛...

**中村隼人**：有沒有人...可以給我...一刀...讓我解脫...咳咳...

**旁白**：他的聲音帶著哀求,但眼神依然瘋狂而扭曲。

**旁白**：李杰沒有回應他,只是冷冷地看著這個瘋子。

**李杰**：(內心)這個人...已經徹底瘋了。

**旁白**：他轉身,走向艾拉如的屍體。

**旁白**：跪下身,仔細查看艾拉如的遺體。

**旁白**：艾拉如的身上確實沒有任何傷口,臉上的表情安詳而平靜。

**旁白**：就像是用盡了所有力氣後,安心地睡去了。

**李杰**：艾拉如長老...到底發生了什麼?

**旁白**：站在旁邊的一個年輕精靈,眼眶紅腫,聲音哽咽。

**精靈村民**：艾拉如長老...他用盡了所有的生命能量...去攻擊那個惡魔。

**旁白**：他抹了一把眼淚,繼續說道。

**精靈村民**：那把黑劍...讓那個男人對所有物理攻擊都免疫。無論是劍還是箭,都無法傷他分毫。

**精靈村民**：我們的戰士一個接一個倒下,凱蘭隊長也...

**旁白**：他的聲音哽咽了,無法繼續。

**旁白**：莉拉接過話,聲音低沉而痛苦。

**莉拉**：長老見我們完全無法對抗那個男人,就決定用生命之力直接攻擊他。

**莉拉**：那是一種...將自己的生命能量化為純粹力量的禁忌之術。

**莉拉**：長老的攻擊,成功摧毀了那個男人的下半身,也消除了他大部分的邪惡力量。

**莉拉**：但長老也因此...耗盡了所有生命...

**旁白**：她說到這裡,眼淚終於忍不住滑落。

**莉拉**：那個男人失去了下半身,也失去了大部分力量。他活不了多久了,最多幾個小時就會死去。

**莉拉**：但代價...代價太大了...

**旁白**：她的聲音越來越小,最後變成了哽咽。

**旁白**：李杰看著艾拉如安詳的臉,心中的悲傷和憤怒交織在一起。

**旁白**：山本浩二、田中健太、佐藤拓海、凌墨、艾拉如、凱蘭...

**旁白**：還有那些無辜的精靈平民...

**旁白**：都因為這個瘋子而死。

**李杰**：(內心)這個男人...已經活不了多久了。

**李杰**：(內心)但即使他快死了,莉拉說他還保有劍的力量...不能靠近他。

**旁白**：李杰深吸了一口氣,壓下心中的憤怒。

**旁白**：他轉身,走向還跪在凌墨身邊哭泣的恵美。

**李杰**：恵美...

**旁白**：恵美抬起頭,眼睛紅腫,臉上滿是淚痕。

**旁白**：她看著李杰,嘴唇顫抖著,想說什麼卻說不出來。

**旁白**：李杰輕輕扶起她,讓她靠在自己肩上。

**旁白**：身後,中村隼人還在斷斷續續地笑著、哀嚎著。

**旁白**：時而是瘋狂的笑聲,時而是痛苦的呻吟,時而是請求解脫的哀求。

**旁白**：但沒有人理會他。

**旁白**：精靈們圍在艾拉如的屍體旁,低聲哭泣。

**旁白**：李杰扶著恵美,看著這片血染的村莊。

**旁白**：月光依舊冰冷地灑落,照在那些逝去的生命上。

**旁白**：而那個征服者,躺在血泊中,在痛苦和瘋狂中,等待著死亡的降臨。
