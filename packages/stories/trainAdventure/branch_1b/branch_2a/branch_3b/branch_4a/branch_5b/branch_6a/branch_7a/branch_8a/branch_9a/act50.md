# 第五十幕：失去的信任

**旁白**：李杰張開嘴,想要辯解,想要說些什麼。

**旁白**：但還沒等他開口——

**中村隼人**：等等...讓我來告訴大家...咳咳...一個更有趣的秘密...

**旁白**：中村隼人躺在血泊中,臉上掛著扭曲的笑容。

**旁白**：鮮血還在從他的傷口滲出,但他的眼神卻異常興奮。

**中村隼人**：李杰...你早就知道了,對吧?

**中村隼人**：你早就知道這班列車...會把所有人帶到這個詭異的世界...

**旁白**：他每說一句話,都要費力地喘息,但聲音依然清晰。

**中村隼人**：但你什麼都沒說...你隱瞞了這個秘密...看著我們所有人...咳...像傻瓜一樣上了那班車...

**旁白**：周圍的空氣瞬間凝固。

**旁白**：所有人的目光,再次聚焦在李杰身上。

**旁白**：明日香停止了抽泣,小林陽菜也抬起頭,眼中滿是困惑和不解。

**旁白**：就連精靈們,也紛紛轉過頭,用複雜的眼神看著他。

**李杰**：你在胡說八道!

**旁白**：李杰的聲音因憤怒而顫抖。

**李杰**：你根本沒有任何證據!

**中村隼人**：哈哈...證據?誰需要證據呢...咳咳...

**旁白**：他艱難地抬起手,指向李杰。

**中村隼人**：你本來有機會救大家的...但你什麼都沒做...

**中村隼人**：你只是看著...看著我們一個個死去...咳...

**旁白**：他的聲音越來越弱,但每一個字都像刀子一樣刺進李杰的心。

**中村隼人**：我甚至懷疑...你早就知道我會攻擊這個村莊...

**中村隼人**：所以你才會在那個時間點離開...又在那個時間點回來...咳咳...

**中村隼人**：時機剛剛好呢...李杰先生...

**旁白**：他說著,笑得更加瘋狂。

**中村隼人**：而且...你還知道回到原來世界的方法...對吧?

**中村隼人**：你知道如何穿越時間...如何回到過去...咳...

**中村隼人**：但你沒有告訴任何人...你只是自己一個人...享受著這個秘密...

**李杰**：你...你憑什麼這麼說?!

**旁白**：李杰的聲音在發抖,額頭上冒出冷汗。

**李杰**：我承認...我確實知道這班列車會帶我們來到這個世界。

**旁白**：他深吸了一口氣,環顧四周。

**李杰**：但我不知道他會攻擊村莊!我根本不可能預料到這種事!

**中村隼人**：哈哈...那你有證據嗎?咳咳...

**旁白**：中村隼人的笑聲刺耳而嘲諷。

**中村隼人**：你說你不知道...但你有證據證明嗎?

**中村隼人**：就像你說我在胡說八道一樣...咳...你也拿不出證據...對吧?

**旁白**：李杰張開嘴,卻說不出話來。

**旁白**：他想要辯解,想要證明自己的清白。

**旁白**：但面對中村隼人的質問,他找不到任何有力的證據。

**旁白**：周圍的空氣越來越沉重。

**旁白**：所有人的目光,從困惑變成了懷疑,從懷疑變成了質問。

**旁白**：莉拉皺著眉頭,手緊緊握著斷裂的弓。

**旁白**：其他精靈們也紛紛後退了一步,眼中的信任開始動搖。

**旁白**：而明日香——

**旁白**：她緩緩站起身,用通紅的眼睛死死盯著李杰。

**鈴木明日香**：你...你早就知道?

**旁白**：她的聲音在顫抖,既有悲傷,也有憤怒。

**鈴木明日香**：你明明知道...卻不告訴我們?

**旁白**：她往前走了幾步,眼淚再次湧出。

**鈴木明日香**：如果你早點說...如果你早點告訴大家...

**鈴木明日香**：健太他們...美咲她...是不是就不用死了?!

**旁白**：她的聲音越來越大,最後幾乎是喊出來的。

**鈴木明日香**：為什麼?!為什麼你要隱瞞?!

**旁白**：她的質問如同利刃,刺向李杰。

**李杰**：我...我不是..

**旁白**：李杰想要解釋,但話卡在喉嚨裡。

**旁白**：他不知道該怎麼說,不知道該從何說起。

**旁白**：小林陽菜也顫抖著站起來,淚水模糊了她的視線。

**小林陽菜**：大哥哥...你真的...知道嗎?

**旁白**：陽菜的聲音帶著哭腔,充滿了失望和不解。

**小林陽菜**：爸爸...媽媽...還有翔太...

**旁白**：她哭得更加厲害,整個人幾乎站不穩。

**小林陽菜**：如果你早點說...他們是不是...就不會死了...

**旁白**：她的聲音越來越小,最後變成了無聲的抽泣。

**旁白**：李杰看著這兩個女孩,心如刀割。

**旁白**：他想說些什麼,想要安慰她們。

**旁白**：但所有的話,都顯得那麼蒼白無力。

**旁白**：而恵美——

**旁白**：她站在原地,沒有說話,也沒有責備。

**旁白**：但她的眼神...那種複雜的眼神...

**旁白**：既有困惑,也有失望,還有一絲掩藏不住的懷疑。

**旁白**：她看著李杰,嘴唇微微動了動,想說什麼卻最終沒有開口。

**旁白**：然後,她緩緩轉身,一步一步走向明日香和陽菜。

**旁白**：雖然腹部的傷口讓她每走一步都很痛,但她還是堅持著走過去。

**旁白**：站在了她們身邊。

**旁白**：這個動作,勝過千言萬語。

**旁白**：李杰看著恵美的背影,感覺胸口像被什麼東西狠狠撞擊了一下。

**旁白**：他張開嘴,想要叫住她,想要解釋。

**旁白**：但聲音卡在喉嚨裡,怎麼也發不出來。

**中村隼人**：哈哈哈...看到了嗎...咳咳...

**旁白**：中村隼人躺在血泊中,笑得更加瘋狂。

**中村隼人**：沒有人相信你了...李杰...

**中村隼人**：你失去了所有人的信任...咳...真是...太美妙了...

**旁白**：他的笑聲刺耳而尖銳,在空蕩蕩的村莊中迴盪。

**旁白**：李杰緊緊握著拳頭,指甲深深刺進掌心。

**旁白**：他想要辯解,想要說明一切。

**旁白**：但面對中村隼人的言語攻擊,他發現自己根本無力招架。

**旁白**：他不擅長辯論,更不擅長在這種情況下為自己辯護。

**旁白**：而中村隼人...即使躺在那裡奄奄一息...

**旁白**：卻依然能用言語輕易地將他逼入絕境。

**旁白**：周圍的目光越來越冰冷,空氣中的懷疑越來越濃重。

**旁白**：就在這時——

**莉拉**：夠了。

**旁白**：莉拉走上前,擋在李杰面前。

**旁白**：她的肩膀上還纏著染血的繃帶,但眼神依然堅定。

**莉拉**：李杰,我建議你暫時離開村莊。

**旁白**：她的聲音平靜,但帶著不容置疑的堅決。

**莉拉**：現在這裡的每個人...都還處在失去親友的痛苦中。

**莉拉**：大家需要時間冷靜,需要時間療傷。

**莉拉**：你留在這裡...只會讓情況變得更糟。

**旁白**：李杰看著莉拉,看著那雙充滿疲憊但依然清明的眼睛。

**旁白**：他明白,這不是驅逐,而是保護。

**旁白**：保護他,也保護這些深陷悲痛的人們。

**李杰**：我...我明白了。

**旁白**：他深吸了一口氣,環顧四周。

**旁白**：明日香低著頭,不願看他。

**旁白**：恵美輕輕摟著小林陽菜,用手輕撫著她的頭髮。

**旁白**：陽菜把臉埋在恵美懷裡,抽泣不止。

**旁白**：而恵美...她依然沒有回頭看他。

**旁白**：李杰的喉嚨發緊,心像被什麼東西緊緊攥住。

**旁白**：他想說些什麼,但最終只是沉默地點了點頭。

**旁白**：轉身,一步一步走向村莊的出口。

**旁白**：腳步沉重,每走一步都像踩在刀尖上。

**旁白**：身後,中村隼人的笑聲依然在迴盪。

**中村隼人**：哈哈...走吧...走吧...咳咳...

**中村隼人**：孤獨的...李杰先生...

**旁白**：李杰沒有回頭,只是繼續往前走。

**旁白**：穿過那些倒下的屍體,穿過那片血染的土地。

**旁白**：走過村莊入口時,他看到了山本浩二的遺體。

**旁白**：那張曾經溫和而堅毅的臉,現在已經冰冷而蒼白。

**旁白**：李杰停頓了一秒,然後繼續往前。

**旁白**：濃霧再次將他吞沒。

**旁白**：身後的村莊,那些質疑的目光,那些悲傷的哭聲,漸漸遠去。

**旁白**：只剩下他一個人。

**旁白**：走在通往銀月湖的道路上。

**旁白**：腳踝的傷口依然在痛,但這種痛...

**旁白**：和心中的痛比起來,根本不算什麼。

**旁白**：月光冰冷地灑落,照在這條孤獨的道路上。

**旁白**：而李杰的身影,漸漸消失在濃霧之中。

**旁白**：只留下一串孤獨的腳步聲。

**旁白**：迴盪在寂靜的森林裡。
