# 第六十幕：歸途之門

**旁白**：他們沿著昨天走過的路，穿過森林，穿過那條通往神樹的小徑。這一次沒有人說話，大家都沉默著，腳步聲是唯一的聲響。

**旁白**：走了很久很久，沒有人知道確切的時間——在這個沒有日夜的世界裡，他們早就失去了時間的感覺。也許是八個小時，也許是十個小時，甚至更久。

**旁白**：終於，終焉祭壇的建築再次出現在眼前——黑色的岩石，巨大的拱門，陰森的氣息。

**旁白**：一切都和之前一樣。但這一次，他們知道這裡將會發生什麼。

**旁白**：五個人走進建築，穿過幽暗的長廊，來到了祭壇聖所。金色的光芒依然在祭壇周圍流轉，翔的身體依然安詳地躺在那裡，而那扇門框依然空蕩蕩的，沒有任何光芒。

**旁白**：恵美深吸了一口氣，朝著祭壇走去。

**旁白**：她在祭壇前停下，轉身看著其他人。

**清水恵美**：明日香、美咲、陽菜...回到地球後，要好好生活。照顧好陽菜，也照顧好自己。

**旁白**：她的聲音很輕，但很堅定。

**清水恵美**：記得我們一起經歷的這一切，但不要被悲傷困住，要向前看。

**鈴木明日香**：恵美...

**旁白**：明日香的眼淚湧了出來。

**高橋美咲**：恵美...我們會的...我們一定會好好活著...

**小林陽菜**：恵美姐姐...

**旁白**：陽菜緊緊抱住恵美。

**旁白**：恵美蹲下身，輕輕摸著陽菜的頭。

**清水恵美**：陽菜，要堅強。妳的爸爸媽媽，還有翔太，他們都希望妳能幸福地活下去。明日香和美咲會好好照顧妳的，妳要聽她們的話，好好長大。

**小林陽菜**：嗯...

**旁白**：陽菜用力點頭，眼淚止不住地流。

**旁白**：恵美站起身，最後看向李杰。

**清水恵美**：李杰先生...謝謝你為我做的一切。但請不要太勉強自己。

**旁白**：她停頓了一下。

**清水恵美**：如果實在找不到辦法...就回家吧。你的父母在等你。

**旁白**：李杰沒有說話，只是緊緊握著拳頭。

**旁白**：恵美對他露出一個微笑，然後轉身走向祭壇。她慢慢躺下，就在翔的身旁。

**旁白**：她的手輕輕觸碰到翔冰冷的手。

**清水恵美**：翔...我回來了...

**旁白**：她的聲音很小，幾乎聽不見。然後，她閉上了眼睛。

**旁白**：下一秒，祭壇周圍的金色光芒突然變得明亮起來。光芒從祭壇底部升起，像是液體般包裹住恵美的身體。

**旁白**：她的臉依然平靜，像是在安詳地睡著。

**旁白**：而在祭壇後方，那扇空蕩蕩的門框開始發出光芒。柔和的金色光芒從門框邊緣流動，漸漸充滿整個門框。

**旁白**：光芒越來越亮，最後形成一片閃耀的光幕。門，開了。

**旁白**：與此同時，翔的身體開始變得透明，他的輪廓漸漸模糊，像是霧氣般散開。幾秒後，他的身體完全消失了。

**旁白**：祭壇上，只剩下恵美一個人。

**鈴木明日香**：門...門開了...

**旁白**：明日香的聲音在顫抖。

**旁白**：美咲捂著嘴，眼淚不停地流。陽菜緊緊抓著明日香的手，也在哭泣。

**鈴木明日香**：我們...走吧...

**旁白**：明日香深吸了一口氣擦去眼淚，轉身牽著陽菜的手朝著那扇門走去。

**旁白**：美咲跟在她們身後，一步三回頭地看著祭壇上的恵美。她們走到門前，光幕溫柔地包圍著她們。

**旁白**：明日香最後看了一眼恵美，然後帶著陽菜和美咲踏入光芒中。三個人的身影漸漸消失。

**旁白**：聖所裡，只剩下李杰一個人。

**旁白**：他站在祭壇旁邊，看著恵美安詳的睡顏。金色的光芒依然包圍著她，像是在守護著她的夢。

**旁白**：李杰的拳頭緊緊握起，指甲深深刺進掌心。

**李杰**：清水...我一定會找到辦法...一定會...

**旁白**：他的聲音在空蕩的聖所裡迴盪。

**旁白**：而那扇門，依然閃耀著金色的光芒，等待著他的選擇。
