# 第三十七幕：黑白之戰

**旁白**：下一秒——

**旁白**：高山翔動了。

**旁白**：他的身影如同閃電般消失在原地，白色劍身劃出一道刺眼的光痕。

**旁白**：李杰的瞳孔驟然收縮。

**旁白**：那速度快得不可思議，幾乎超越了人類的極限。

**李杰**：（內心）太快了——！

**旁白**：但就在白色劍刃即將斬中他的瞬間，李杰的身體像是被某種本能驅使，自動做出了反應。

**旁白**：黑劍橫掃而出，精準地架住了那一擊。

**旁白**：「鏗——！」

**旁白**：金屬碰撞的尖銳聲響在整個空間中炸開，震得耳膜發疼。

**旁白**：一黑一白兩柄劍交錯在一起，迸發出刺目的光芒。

**旁白**：李杰感受到從劍身傳來的巨大衝擊力，腳步向後滑退了兩步，但很快就穩住了身形。

**李杰**：（震驚，內心）這力量……黑劍在幫我……

**旁白**：那柄黑劍，正在源源不斷地將力量灌注到他的身體裡。

**旁白**：肌肉、骨骼、神經——每一個細胞都彷彿被強化了，變得比平時更加敏銳、更加強大。

**高山翔**：居然擋住了……

**旁白**：高山翔的眼神變得更加冰冷。

**高山翔**：看來你確實不是普通人。

**旁白**：他再次揮劍。

**旁白**：這一次，是一連串密集的斬擊。

**旁白**：白色劍身化為無數道光影，從上、從下、從左、從右，毫無規律地襲來。

**旁白**：每一劍都快如閃電，每一劍都帶著致命的殺意。

**旁白**：李杰深吸一口氣，雙手握劍，全神貫注地應對。

**旁白**：「鏗！鏗！鏗鏗鏗——！」

**旁白**：黑劍與白劍不斷碰撞，火花在空氣中四濺。

**旁白**：每一次格擋都精準到位，黑劍彷彿知道白劍的軌跡，自動引導著李杰的動作。

**李杰**：（咬牙，內心）不能……不能攻擊……

**李杰**：（內心）我沒有理由殺他……他可能是恵美要救的人……

**旁白**：但高山翔沒有任何留手的意思。

**旁白**：他的攻勢越來越猛烈，速度也越來越快。

**高山翔**：只會防守嗎？惡魔！

**旁白**：他猛地一個旋身，白色劍刃橫掃而來，帶著撕裂空氣的尖嘯聲。

**旁白**：李杰舉劍格擋。

**旁白**：「鏗——！」

**旁白**：巨大的衝擊力震得他手臂發麻，身體被震退了幾步。

**旁白**：但李杰穩住身形，沒有倒下。

**旁白**：他調整呼吸，重新擺好架勢。

**高山翔**：你比我想像的要強。但這改變不了結果。

**旁白**：高山翔舉起白劍，劍身上的光芒越來越耀眼，像是在聚集所有的力量。

**旁白**：那是更強的一擊。

**旁白**：足以決定勝負的一擊。

**李杰**：（喘息，內心）不能硬接……

**旁白**：高山翔向前踏出一步，白色劍身化為一道光柱，直刺而來。

**旁白**：李杰沒有選擇硬碰，而是側身閃避，同時用黑劍斜向格擋。

**旁白**：「轟——！！」

**旁白**：白色光柱擦過他的肩膀，黑劍與白劍再次碰撞。

**旁白**：巨大的衝擊力讓李杰無法保持平衡，身體被震退，踉蹌著後退了十幾步。

**旁白**：他的雙腳在地面上劃出兩道痕跡，一直退到候車大廳的入口。

**李杰**：咳……

**旁白**：他單手撐著黑劍，半跪在地上，大口喘著氣。

**旁白**：肩膀有些灼痛，但沒有受重傷。

**旁白**：高山翔從月台緩步走來，白色劍身拖在地上，發出刺耳的摩擦聲。

**李杰**：（喘息）等等……我不知道……我不知道發生了什麼……

**李杰**：（顫抖）這把劍……它控制了我……我不是故意要殺那些人的……

**高山翔**：謊言。

**旁白**：他的聲音冰冷如刃。

**高山翔**：持有那柄劍的人,就是惡魔。沒有任何例外。

**李杰**：（急切）不……你聽我說……我是來找人的……

**李杰**：我是來找恵美的……還有凌墨……

**李杰**：清水恵美和凌墨！你認識他們嗎？！

**旁白**：高山翔的眼神沒有任何波動。

**高山翔**：我不認識什麼恵美,也不認識什麼凌墨。

**旁白**：他舉起白劍,劍尖指向李杰。

**高山翔**：這些名字,不過是你為了活命編造的謊言罷了。

**李杰**：（震驚）什麼……你不認識……

**高山翔**：那柄黑劍,是這個世界的禍根。它必須被摧毀。

**高山翔**：而持有它的你,也必須消失。

**旁白**：他再次抬起白劍，準備發動攻擊。

**旁白**：李杰深吸一口氣，握緊黑劍站起身。

**李杰**：（內心）這樣下去不是辦法……

**李杰**：（內心）我不想殺他，但繼續戰鬥只會有一方倒下……

**李杰**：（內心）必須先撤退……弄清楚狀況再說……

**旁白**：他做出決定。

**旁白**：趁著高山翔準備下一擊的間隙，李杰轉身向大廳出口衝去。

**旁白**：高山翔眼神一凜，正要追擊——

**旁白**：但李杰突然回身，揮出一道黑色劍氣阻擋。

**旁白**：那道劍氣雖然沒有殺傷力，但足以讓高山翔停下腳步防禦。

**旁白**：就這一瞬間，李杰已經衝出大廳，來到建築外的荒原上。

**旁白**：紫色的月亮高懸在天空,詭異的光芒照亮著這片死寂的大地。

**旁白**：他調整呼吸，轉身看向那座巨大的黑色建築。

**旁白**：高山翔追到拱門入口處，白色的身影在黑暗中格外顯眼。

**旁白**：但他停在了門檻內，沒有再追出來。

**旁白**：兩人隔著數十公尺的距離對峙著。

**高山翔**：（冷漠）聽好了,持劍者。

**高山翔**：永遠不要再踏入這座神殿。

**高山翔**：如果你膽敢再次靠近這裡,我會毫不猶豫地取你性命。

**旁白**：說完,他轉身走回建築深處,身影消失在幽暗的長廊中。

**旁白**：只留下李杰一個人,站在荒涼的原野上。

---

**旁白**：李杰緊握著黑劍，站在荒涼的原野上。

**旁白**：雖然身體有些疲憊，但他的意識很清醒。

**李杰**：（低語）他說……他不認識恵美和凌墨……

**李杰**：（內心）怎麼可能……凌墨明明說過,恵美的男友叫「翔」……

**李杰**：（內心）而這個人,也叫高山翔……

**旁白**：他看著那座巨大的黑色建築。

**旁白**：它靜靜地矗立在那裡,像是一座不可侵犯的陵墓。

**李杰**：（內心）難道……他們不是同一個人？

**李杰**：（內心）還是說……他失去了記憶？

**李杰**：（內心）又或者……凌墨和恵美根本就騙了我？

**旁白**：無數的疑問在腦海中翻騰,但沒有任何答案。

**旁白**：他環顧四周。

**旁白**：荒涼的原野一望無際,沒有任何生命的跡象。

**旁白**：紫色的月光冰冷地灑在大地上,一切都顯得死寂而詭異。

**旁白**：沒有列車,沒有倖存者,沒有任何線索。

**李杰**：（低語）我……接下來該怎麼辦……

**旁白**：他看向手中的黑劍。

**旁白**：劍身上還殘留著血跡,在月光下泛著詭異的暗紅色光澤。

**旁白**：這柄劍給了他力量,讓他能與高山翔抗衡。

**旁白**：但這柄劍也奪走了五條人命,讓他成為了所有人眼中的惡魔。

**李杰**：（凝重）我……到底該相信什麼……

**旁白**：寒風吹過,帶走了他的低語。

**旁白**：李杰站在終焉祭壇外的荒原上,握著那柄詛咒之劍。

**旁白**：迷霧重重，前路未明。
