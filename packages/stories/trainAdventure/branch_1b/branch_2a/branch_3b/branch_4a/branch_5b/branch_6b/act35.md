# 第三十五幕：失蹤者名單

**旁白**：李杰盯著照片，強迫自己保持鎮定，臉上做出困惑的表情。

**李杰**：我...我不認識這個人。

**旁白**：他的聲音聽起來有些沙啞，但語氣盡量自然。

**李杰**：你為什麼要給我看這張照片？這個人是誰？

**旁白**：黑西裝男的眼睛瞇了起來，銳利的目光像刀子一樣刺向李杰。

**旁白**：幾秒鐘的沉默後，那個男人突然動了。

**旁白**：一記重拳毫無預警地砸向李杰的臉頰！

**李杰**：啊——！

**旁白**：劇烈的疼痛瞬間爆發，李杰的頭被打得偏向一側，金屬椅子發出刺耳的摩擦聲。

**旁白**：嘴角傳來鐵鏽般的血腥味，他能感覺到臉頰內側被牙齒割破。

**黑西裝男**：別跟我裝傻。

**旁白**：那個男人的聲音冰冷而充滿威脅，沒有一絲情感波動。

**旁白**：李杰喘著氣，臉頰火辣辣地疼，但他不敢再說話。

**黑西裝男**：這個人叫中村隼人。

**旁白**：黑西裝男再次舉起那張照片，確保李杰能清楚看見。

**黑西裝男**：42歲，某證券公司的中階主管。9月23日晚上失蹤。

**旁白**：他把照片放下，從西裝內袋掏出另一張照片——這次是李杰自己的照片。

**旁白**：那是一張看起來像是從遠處偷拍的照片，李杰正走在街上，表情自然，顯然完全不知道被拍攝。

**黑西裝男**：我們在中村隼人的辦公室找到了這張照片。

**旁白**：他把兩張照片並排舉在李杰面前。

**黑西裝男**：所以，你還要說你不認識他嗎？

**旁白**：李杰感到心臟狂跳，但他強迫自己保持冷靜。

**李杰**：我真的不認識什麼中村隼人。

**旁白**：他的聲音有些顫抖，但語氣堅定。

**李杰**：我不知道為什麼他會有我的照片，也許...也許他認錯人了？或是搞錯了什麼？

**旁白**：黑西裝男發出一聲冷笑，那聲音聽起來像是在嘲諷什麼愚蠢的謊言。

**旁白**：他把李杰的照片收起來，又從口袋裡拿出第三張照片。

**旁白**：照片上是一個戴著眼鏡的年輕男人，看起來二十多歲，臉色蒼白，表情帶著一種故作聰明的神態。

**旁白**：李杰認得這張臉——那個膽小的眼鏡男，在列車上見過。

**黑西裝男**：那這個人呢？認識嗎？

**旁白**：李杰搖了搖頭，試圖讓表情看起來困惑而無辜。

**李杰**：不認識...這又是誰？

**旁白**：黑西裝男沒有像剛才那樣動手，只是站在原地，嘴角勾起一個冰冷的弧度。

**黑西裝男**：斎藤蓮，20歲，大學生。

**旁白**：他頓了頓，眼神變得更加銳利。

**黑西裝男**：這張照片，也是在中村隼人的辦公桌上找到的。

**旁白**：他把斎藤蓮的照片和李杰的照片並排舉起。

**黑西裝男**：跟你的照片放在一起。

**旁白**：李杰的呼吸變得急促，但他努力壓抑著內心的恐慌。

**黑西裝男**：有意思的是，中村隼人失蹤了，斎藤蓮也失蹤了。

**旁白**：那個男人向前踏了一步，逼近李杰。

**黑西裝男**：但你還在這裡。

**旁白**：冰冷的話語像針一樣刺進李杰的心臟。

**李杰**：失蹤？！

**旁白**：李杰努力做出震驚的表情，眼睛睜大，聲音提高。

**李杰**：你是說他們都...失蹤了？什麼時候的事？

**旁白**：但他心裡很清楚，這些人沒有失蹤——他們去了另一個世界。

**旁白**：他必須假裝驚訝，不能露出任何知情的跡象。

**黑西裝男**：9月23日晚上。

**旁白**：黑西裝男的聲音沒有任何起伏，就像在陳述一個無關緊要的事實。

**黑西裝男**：同一天，同一個時間，他們都消失了。

**旁白**：他再次發出那種嘲諷的冷笑。

**旁白**：然後從口袋裡拿出第四張照片。

**旁白**：這次李杰看到照片時，臉上的表情不再是假裝的困惑，而是真正的震驚。

**旁白**：照片上是一個中年男人，看起來跟中村隼人差不多年紀，四十多歲的樣子。

**旁白**：但這個人的長相讓人感到不安——眼窩深陷，顴骨突出，嘴角帶著一種陰森的弧度。

**旁白**：整張臉散發著一種說不出的詭異感，讓人看了就覺得毛骨悚然。

**李杰**：這...這是誰？

**旁白**：這次李杰的困惑是真實的，他完全不認識這個人。

**黑西裝男**：渡邊陸，43歲。

**旁白**：黑西裝男把照片舉得更近，讓李杰能清楚看見每個細節。

**黑西裝男**：跟中村隼人和斎藤蓮一樣，9月23日晚上失蹤。

**旁白**：李杰感到腦中一片空白。

**旁白**：渡邊陸？他完全沒有印象。

**旁白**：在列車上，他見過中村隼人——那個自私的中年男人。

**旁白**：也見過斎藤蓮——那個膽小的眼鏡男。

**旁白**：但這個渡邊陸...他完全沒有見過。

**旁白**：如果這個人也在9月23日失蹤，那是不是意味著他也搭上了那班列車？

**旁白**：但李杰很確定，上次在列車上他沒有見過這張臉。

**旁白**：那麼...這個人去哪裡了？

**李杰**：我...我真的不認識這個人。

**旁白**：李杰的聲音顫抖著，這次不是偽裝，而是真實的困惑和恐懼。

**李杰**：我從來沒見過他。

**旁白**：黑西裝男盯著李杰的眼睛，像是在尋找謊言的痕跡。

**旁白**：幾秒鐘的沉默後，他把所有照片都收了起來。

**黑西裝男**：三個人，同一天失蹤。

**旁白**：他的聲音低沉而充滿壓迫感。

**黑西裝男**：而你的照片，出現在其中一個失蹤者的辦公室裡。

**旁白**：他向前傾身，臉幾乎貼到李杰面前。

**黑西裝男**：你最好開始說實話，李杰。

**旁白**：李杰的心臟狂跳，冷汗從額頭滴落。

**旁白**：他不知道該說什麼，也不知道這些人到底掌握了多少資訊。

**旁白**：但有一點他很確定——如果他說出關於異世界的真相，等待他的可能不是自由，而是更可怕的事情。

**旁白**：金屬房間裡只剩下那個低沉的嗡嗡聲，還有李杰急促的呼吸聲。

**旁白**：時間彷彿凝固了。
