# 第三十六幕：破碎的謊言

**旁白**：黑西裝男沒有回應李杰關於渡邊陸的疑問，只是把照片收起來，轉身走向門口。

**旁白**：他從西裝內袋掏出一個黑色的平板電腦，手指在螢幕上滑動著。

**旁白**：幾秒鐘後,他轉回身,把平板舉到李杰面前。

**黑西裝男A**：那這個呢？

**旁白**：螢幕上是一個網頁——3ch論壇的帖子。

**旁白**：李杰的心臟猛地一沉,那是他非常熟悉的頁面。

**旁白**：標題：【尋人/舊聞】有人還記得20年前新宿中央線的末班車嗎？

**旁白**：發帖人：@Looking4Sister

**旁白**：黑西裝男的手指在螢幕上滑動,停在了第六樓的回覆上。

**旁白**：那是李杰自己發的回覆,用著LJ_2025的帳號。

**黑西裝男A**：LJ_2025,這是你吧？

**旁白**：李杰感到一股寒意從脊椎竄上。

**旁白**：他沒想到這些人連網路上的蛛絲馬跡都挖了出來。

**李杰**：我...我不知道你在說什麼。

**旁白**：他努力讓聲音聽起來困惑,但心臟已經狂跳不止。

**李杰**：LJ_2025可以是任何人,怎麼就一定是我？

**旁白**：黑西裝男冷笑一聲,沒有回答。

**旁白**：他把平板收起來,然後毫無預警地——

**旁白**：又是一記重拳砸在李杰臉上！

**李杰**：啊——！

**旁白**：李杰的頭被打得偏向另一側,耳朵嗡嗡作響,眼前一片白光。

**旁白**：嘴角的傷口裂開,鮮血順著下巴滴落在衣領上。

**黑西裝男A**：我最後說一次。

**旁白**：那個男人的聲音冰冷得像刀子。

**黑西裝男A**：別跟我耍小聰明。

**旁白**：李杰喘著氣,嘴裡滿是血腥味,但他不敢再說話。

**旁白**：黑西裝男掏出手機,撥了一個號碼,簡短地說了一句。

**黑西裝男A**：把那個人帶進來。

**旁白**：掛斷電話後,他站在李杰面前,雙手抱胸,眼神冷漠地盯著他。

**旁白**：金屬房間陷入令人窒息的沉默。

**旁白**：只有那個低沉的嗡嗡聲,還有李杰急促的呼吸聲。

**旁白**：大約兩分鐘後,門外傳來腳步聲。

**旁白**：不止一個人,至少兩個。

**旁白**：金屬門發出沉重的咔嚓聲,緩緩打開。

**旁白**：另一個黑西裝男走了進來,他的手裡拖著一個人。

**旁白**：更準確地說,是拽著那個人的衣領,像拖一個破布袋一樣把他拖進房間。

**旁白**：李杰看清了那個被拖進來的人,瞳孔瞬間放大。

**李杰**：清水...先生？！

**旁白**：清水浩介！

**旁白**：他的臉上有明顯的瘀青,眼鏡不知去向,頭髮凌亂,衣服也皺巴巴的。

**旁白**：黑西裝男B把他粗暴地扔在地上,浩介發出一聲痛苦的呻吟。

**清水浩介**：李先生——！

**旁白**：浩介看到李杰,眼中閃過驚恐和求救的光芒。

**旁白**：他掙扎著想爬起來,但黑西裝男B一腳踩在他的背上,把他重新壓回地面。

**清水浩介**：救救我...拜託...

**旁白**：他的聲音顫抖著,充滿了恐懼。

**清水浩介**：我什麼都不知道！我真的什麼都不知道！

**旁白**：他掙扎著抬起頭,眼淚混著鼻涕流下來。

**清水浩介**：什麼中村、斎藤、渡邊,我一個都不認識！

**旁白**：他的聲音越來越高,帶著哭腔。

**清水浩介**：我只是在找我姐姐！我只知道恵美和凌墨！

**旁白**：聽到這兩個名字,李杰的心臟狠狠一縮。

**旁白**：恵美——那個文靜少女。

**旁白**：凌墨——那個神秘的黑衣男。

**旁白**：這些人...連他們都盯上了？

**李杰**：清水先生...

**旁白**：李杰想說什麼,但話語卡在喉嚨裡。

**旁白**：黑西裝男A慢慢走向浩介,從腰後掏出一把黑色的手槍。

**旁白**：槍身在冷白色的燈光下泛著金屬光澤,沉默而致命。

**清水浩介**：不...不要...

**旁白**：浩介看到槍,整個人都癱軟了,身體劇烈顫抖。

**旁白**：黑西裝男A蹲下身,用槍口抵住浩介的太陽穴。

**旁白**：冰冷的金屬觸感讓浩介發出一聲驚恐的尖叫。

**黑西裝男A**：李杰。

**旁白**：那個男人的聲音依然平靜,就像在討論今天的天氣。

**黑西裝男A**：現在,你最好把你知道的一切都告訴我。

**旁白**：他用槍口用力頂了頂浩介的頭,讓後者發出痛苦的呻吟。

**黑西裝男A**：否則,這個人會死。

**清水浩介**：嗚...嗚嗚...

**旁白**：浩介已經完全崩潰了,眼淚不停地流,整個人像篩糠一樣顫抖。

**清水浩介**：李先生...拜託...告訴他們...

**旁白**：他哽咽著,聲音幾乎聽不清。

**清水浩介**：告訴他們真相...拜託...我不想死...

**旁白**：李杰感到胸口像被巨石壓住,幾乎無法呼吸。

**旁白**：他看著浩介驚恐的臉,看著那把冰冷的槍,看著黑西裝男毫無情感的眼神。

**李杰**：我...

**旁白**：他的聲音卡住了,腦中一片混亂。

**旁白**：該說嗎？說了真相之後,他們會放過浩介嗎？會放過他嗎？

**旁白**：還是說,他們會變本加厲,用更殘酷的方式逼問更多細節？

**旁白**：而且,這些人到底想要什麼？

**旁白**：他們知道異世界的存在嗎？還是只是在追查失蹤案件？

**旁白**：如果他說出真相,會不會讓更多人陷入危險？

**旁白**：恵美、凌墨,還有其他可能活下來的人...

**黑西裝男A**：你在猶豫什麼？

**旁白**：那個男人的聲音打斷了他的思緒。

**黑西裝男A**：是在想要編一個什麼樣的故事來騙我？

**旁白**：他用槍口再次用力頂了頂浩介的頭。

**黑西裝男A**：還是在想,這個人的命值不值得你說實話？

**清水浩介**：李先生...求你了...

**旁白**：浩介的哭聲越來越淒厲,整個人趴在地上,身體不停地抽搐。

**清水浩介**：我不想死...我還想找到姐姐...

**旁白**：李杰緊咬著嘴唇,指甲陷進掌心,幾乎要出血。

**旁白**：他的腦中飛速運轉,試圖找到一個最佳的選擇。

**旁白**：但每一個選項都充滿了不確定性,每一條路都可能通向更深的深淵。

**旁白**：時間在一秒一秒地流逝。

**旁白**：黑西裝男的手指,已經放在了扳機上。
