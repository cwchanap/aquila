# 第三十七幕：謊言的代價

**旁白**：李杰看著浩介驚恐的臉，看著那把冰冷的槍口。

**旁白**：他的腦中飛速運轉，在幾秒鐘內做出了決定。

**旁白**：說真話太危險，但完全沉默只會讓浩介死。

**旁白**：那麼...就編一個故事吧。

**旁白**：一個混合了真實與謊言的故事，一個能讓他們相信的故事。

**李杰**：我...我說。

**旁白**：他的聲音沙啞，但語氣變得堅定。

**李杰**：我確實知道一些事情。

**旁白**：黑西裝男A的眼睛瞇了起來，但沒有把槍移開。

**黑西裝男A**：說。

**旁白**：李杰深吸一口氣，開始編織他的謊言。

**李杰**：大約兩週前，我收到一封郵件。

**旁白**：他努力讓聲音聽起來誠懇。

**李杰**：寄件人自稱是中村隼人，說他掌握了一些關於20年前失蹤乘客的重要信息。

**旁白**：黑西裝男A沒有說話，只是靜靜地聽著。

**李杰**：我當時很震驚...因為我也在尋找一個人。

**旁白**：他頓了頓，讓這個謊言聽起來更真實。

**李杰**：那是我童年時的朋友，20年前，我隨家人來日本旅遊時認識的。

**李杰**：後來...他就失蹤了。我一直想知道他發生了什麼事。

**旁白**：這個故事的基礎建立在真實之上——確實有人在20年前失蹤。

**旁白**：但細節全是虛構的。

**清水浩介**：李先生...

**旁白**：浩介趴在地上，用充滿希望的眼神看著他。

**李杰**：中村說他可以給我一些信息，但他要我幫他做一件事。

**黑西裝男A**：什麼事？

**李杰**：他沒說具體是什麼...只是約我在9月23日午夜，去新宿站搭最後一班往高尾的列車。

**旁白**：他混入了真實的時間和地點。

**李杰**：他說會在車上等我，到時候會告訴我詳情。

**旁白**：黑西裝男A的表情沒有變化，但李杰能感覺到對方在仔細聆聽每一個字。

**李杰**：我們沒有見過面，所有聯繫都是通過郵件。

**旁白**：這樣可以解釋為什麼他認不出中村的照片。

**李杰**：但在約定的那一週之前，中村給了我一些信息...作為誠意。

**清水浩介**：什麼信息？！

**旁白**：浩介突然出聲，聲音充滿了急切。

**清水浩介**：關於我姐姐的嗎？！關於恵美的？！

**李杰**：是的...他告訴了我一些關於清水恵美的事。

**旁白**：李杰看向浩介，眼神中帶著歉意。

**旁白**：他必須說些什麼來安撫浩介，但他知道的信息有限。

**李杰**：他說...恵美在20年前確實搭上了那班列車。

**李杰**：她當時和朋友在一起，是從新宿上車的。

**旁白**：這些都是他從與浩介的對話中推測出來的信息。

**清水浩介**：然後呢？！她現在在哪裡？！

**李杰**：中村說...更詳細的信息都在他給我的文件裡。

**旁白**：李杰再次深吸一口氣，準備說出最關鍵的謊言。

**李杰**：那些文件...都在我家的電腦裡。

**旁白**：他看向黑西裝男A。

**李杰**：中村發給我的所有郵件、照片、還有關於失蹤者的資料，全都存在那台電腦上。

**黑西裝男A**：為什麼不在手機上？

**旁白**：那個男人的聲音充滿了懷疑。

**李杰**：因為...資料太多了，而且有些是加密文件。

**旁白**：李杰努力讓謊言聽起來合理。

**李杰**：我不想冒險把這些敏感信息放在手機上，所以全都轉存到電腦裡了。

**旁白**：他頓了頓，補充了最關鍵的一點。

**李杰**：那台電腦有指紋鎖，只有我的指紋才能解鎖。

**旁白**：這就是他不會被立刻殺死的理由——他們需要他活著。

**黑西裝男A**：繼續說。你為什麼沒去搭那班車？

**李杰**：因為...我覺得太可疑了。

**旁白**：李杰做出猶豫的表情。

**李杰**：一個陌生人突然聯繫我，說有20年前的秘密信息...

**李杰**：約我在深夜搭末班車，還不肯透露具體要我幫什麼忙...

**旁白**：他搖了搖頭。

**李杰**：我越想越不對勁，所以最後決定不去了。

**李杰**：我想...也許應該先報警，或是找人陪我一起去。

**旁白**：黑西裝男A盯著李杰的眼睛，那雙眼睛像X光一樣想要看穿他的謊言。

**旁白**：房間陷入令人窒息的沉默。

**旁白**：幾秒鐘後，黑西裝男A緩緩站起身。

**旁白**：他把槍從浩介的太陽穴上移開，讓後者發出一聲如釋重負的哭泣。

**黑西裝男A**：如果你在說謊...

**旁白**：他把槍收回腰後的槍套。

**黑西裝男A**：你會後悔的。

**旁白**：然後他轉向黑西裝男B，簡短地說了一句。

**黑西裝男A**：去他家把那台電腦帶過來。

**旁白**：黑西裝男B點了點頭，顯然早就知道李杰的地址。

**黑西裝男B**：明白。

**旁白**：他轉身準備離開。

**旁白**：黑西裝男A走到李杰面前，粗暴地搜查他的口袋。

**旁白**：他從李杰的褲子口袋裡掏出一串鑰匙，還有錢包和手機。

**旁白**：他把鑰匙扔給黑西裝男B。

**黑西裝男A**：筆記型電腦應該在客廳或臥室。

**旁白**：黑西裝男B接過鑰匙，轉身離開了房間，金屬門在他身後關上。

**旁白**：李杰心中一沉——他們連問都不用問就知道他住哪裡。

**旁白**：這些人對他的監視，比他想像的還要徹底。

**旁白**：房間裡只剩下黑西裝男A、李杰、還有趴在地上的浩介。

**旁白**：黑西裝男A站在李杰面前，雙手抱胸。

**黑西裝男A**：你最好祈禱你說的都是真的。

**旁白**：他的聲音冰冷而充滿威脅。

**黑西裝男A**：如果那台電腦裡什麼都沒有...

**旁白**：他沒有說完，但威脅已經很明顯了。

**旁白**：李杰緊咬著嘴唇，不敢回應。

**旁白**：他知道自己剛才編的故事有多麼脆弱。

**旁白**：當他們把電腦帶回來，用他的指紋解鎖後，會發現什麼？

**旁白**：普通的工作文件、一些遊戲、瀏覽器歷史記錄...

**旁白**：但絕對不會有任何關於中村隼人的郵件，也不會有什麼失蹤者的資料。

**旁白**：到時候，他該怎麼解釋？

**旁白**：清水浩介依然趴在地上，身體還在顫抖，但至少那把槍不再指著他了。

**清水浩介**：李先生...謝謝...

**旁白**：他小聲地說著，聲音充滿了感激和恐懼。

**旁白**：李杰沒有回應，只是閉上眼睛。

**旁白**：他為兩人爭取到了一些時間，但這個謊言能維持多久？

**旁白**：一個小時？兩個小時？

**旁白**：等黑西裝男B從三鷹回來，等他們發現電腦裡什麼都沒有...

**旁白**：那時候，等待他們的會是什麼？

**旁白**：金屬房間裡只剩下那個低沉的嗡嗡聲。

**旁白**：時間一分一秒地流逝。

**旁白**：每一秒都像是倒數計時，走向謊言被揭穿的那一刻。
