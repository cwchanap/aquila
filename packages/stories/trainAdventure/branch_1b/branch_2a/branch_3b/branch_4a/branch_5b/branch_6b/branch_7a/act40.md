# 第四十幕：脫困

**旁白**：就在李杰和浩介等待的時候，金屬門突然再次打開。

**旁白**：門發出沉重的咔嚓聲，一個人影快步走了進來。

**旁白**：那是另一個穿黑色西裝的男人，身材精瘦，動作敏捷。

**旁白**：他的手裡握著一把刀，刀刃在冷白色的燈光下閃著寒光。

**清水浩介**：不——不要殺我！

**旁白**：浩介看到那把刀，整個人瞬間崩潰，聲音變成了尖叫。

**清水浩介**：求求你！我什麼都說了！不要殺我！拜託！

**旁白**：他拼命地想往後退，但手腳被綁著，只能像蟲子一樣在地上掙扎。

**旁白**：但那個黑西裝男沒有攻擊，反而停下腳步。

**黑西裝男？**：是我，凌墨。

**旁白**：那個聲音低沉而平靜，帶著一種熟悉的冷靜。

**旁白**：李杰感到整個世界都停止了。

**李杰**：凌墨...？！

**旁白**：他的聲音充滿了難以置信。

**旁白**：凌墨...那個神秘的男人，應該在異世界的人，怎麼會在這裡？

**旁白**：那個黑西裝男伸手摘下墨鏡和黑色口罩。

**旁白**：一張年輕而冷峻的臉露了出來，大約二十多歲，眼神銳利而深邃。

**旁白**：李杰盯著那張臉，腦中突然閃過一個畫面。

**旁白**：新宿站的月台，時間倒流之前，那個穿黑色連帽衛衣的神秘男人。

**旁白**：就是他——凌墨！

**李杰**：是你...在新宿站我看到的...

**旁白**：李杰喃喃自語，震驚地意識到一切的聯繫。

**清水浩介**：凌墨...是你...

**旁白**：浩介瞪大眼睛，聲音從恐懼變成了如釋重負。

**清水浩介**：你來救我們了...

**凌墨**：稍後再解釋。我們現在得趕快離開這裡。

**旁白**：凌墨的聲音雖然冷靜，但帶著一絲關切。

**旁白**：他快速走到李杰身邊，用刀割斷了綁住他的繩索。

**旁白**：繩索斷裂，李杰的雙手終於恢復自由。

**旁白**：血液重新流回手腕，帶來刺痛的感覺。

**旁白**：凌墨又走向浩介，同樣割斷了綁住他的繩索。

**凌墨**：還好嗎？能走嗎？

**李杰**：沒事...可以走。

**旁白**：李杰搖搖晃晃地站起身，雙腿因為長時間被綁而有些發麻。

**清水浩介**：我...我也可以...

**旁白**：浩介掙扎著爬起來，眼淚還掛在臉上，但眼神中充滿了劫後餘生的慶幸。

**凌墨**：跟我走。安靜點。

**旁白**：凌墨把刀收起來，快步走向門口。

**旁白**：李杰和浩介緊跟在他身後。

**旁白**：他們走出金屬房間，進入一條狹窄的走廊。

**旁白**：走廊兩側是斑駁的牆壁，天花板有些地方已經破損，露出鏽蝕的鋼筋。

**旁白**：這不是什麼高科技的秘密基地，而是一棟廢棄的老建築。

**旁白**：地上散落著碎石和灰塵，空氣中有一股潮濕和腐朽的氣味。

**旁白**：他們穿過走廊，經過幾個空蕩蕩的房間。

**旁白**：有些房間的門已經掉落，露出裡面破爛的傢俱和塗鴉。

**旁白**：凌墨走在最前面，動作輕盈而警覺，不時回頭確認兩人跟上。

**旁白**：終於，他們來到一個樓梯間。

**旁白**：凌墨示意兩人小心腳下，然後開始往下走。

**旁白**：樓梯的扶手已經鏽蝕，每一步都會發出輕微的嘎吱聲。

**旁白**：李杰緊張地聽著周圍的動靜，但除了他們的腳步聲，沒有其他聲音。

**旁白**：那兩個黑西裝男去哪裡了？凌墨是怎麼處理他們的？

**旁白**：下到一樓，凌墨推開一扇生鏽的鐵門。

**旁白**：外面是郊區的荒涼景象——雜草叢生的空地、幾棟廢棄的建築、還有遠處隱約可見的工廠煙囪。

**旁白**：夜幕已經降臨，天空是深藍色的，只有幾顆星星在閃爍。

**旁白**：冷風吹在臉上，讓李杰打了個寒顫，但那是自由的感覺。

**旁白**：不遠處，一輛計程車已經停在路邊，引擎還在運轉，車燈在黑暗中顯得格外明亮。

**凌墨**：車已經在等了，快上車。

**旁白**：三人鑽進計程車，凌墨坐在副駕駛座，李杰和浩介坐在後座。

**計程車司機**：去哪裡？

**凌墨**：中野，我會指路。

**旁白**：車子啟動，駛離了那片荒涼的郊區。

**旁白**：李杰看著窗外逐漸出現的街燈和建築，感到一種不真實的解脫感。

**旁白**：他們真的逃出來了。

**旁白**：浩介坐在旁邊，還在輕微地顫抖，但臉上帶著如釋重負的表情。

**李杰**：凌墨...你怎麼...

**旁白**：李杰忍不住開口，想問出心中無數的疑問。

**旁白**：凌墨回頭看了他一眼，眼神中帶著理解。

**凌墨**：到了再說。計程車不是談論這些事的地方，你應該能理解。

**旁白**：李杰點點頭，雖然心中充滿了疑問，但他知道凌墨說得對。

**旁白**：車子在夜晚的東京街道上行駛著，經過燈火通明的商店街、安靜的住宅區。

**旁白**：大約三十分鐘後，車子駛入中野的一個住宅區，停在一棟老舊的公寓樓前。

**凌墨**：就是這裡。

**旁白**：凌墨付了車資，三人下車。

**旁白**：這是一棟典型的日本老式公寓，外牆是米色的，有些地方已經斑駁。

**旁白**：凌墨帶著他們走上外側的樓梯，來到三樓的一個房門前。

**旁白**：他掏出鑰匙，打開了門。

**凌墨**：進來吧。

**旁白**：三人走進公寓。

**旁白**：這是一個小小的套房，客廳、廚房和臥室連在一起，擺設簡單但整潔。

**旁白**：客廳裡有一張小桌子、幾個坐墊、還有一個書架。

**旁白**：但吸引李杰注意的，是坐在小桌旁的一個年輕女孩。

**旁白**：她看起來二十歲左右，留著及肩的黑色長髮，戴著一副細框眼鏡。

**旁白**：她的臉型秀氣，表情文靜，眼神中帶著一絲不安。

**旁白**：李杰瞬間認出了她。

**旁白**：那張臉，他在列車上見過——那個文靜的少女，總是低頭看著手中的小冊子。

**旁白**：女孩看到他們進來，立刻站起身。

**旁白**：她的目光先是落在凌墨身上，然後移向浩介。

**旁白**：當她看清浩介臉上的瘀青和狼狽的樣子時，眼中閃過一絲痛苦和擔憂。

**清水浩介**：姐姐...我沒事...

**旁白**：浩介的聲音有些虛弱，但努力擠出一個笑容。

**凌墨**：恵美，他們都安全了。

**旁白**：凌墨的聲音比平時溫和了一些。

**旁白**：李杰聽到這個名字，心中最後的疑問也得到了答案。

**旁白**：恵美——清水恵美，浩介的姐姐。

**旁白**：那個在列車上見過的文靜少女。

**清水恵美**：太好了...真的太好了...

**旁白**：恵美的聲音輕柔而顫抖，眼淚開始滑落。

**旁白**：浩介走上前，姐弟兩人緊緊擁抱在一起。

**清水恵美**：我一直很擔心...凌墨說要去救你...我好怕...

**清水浩介**：對不起...讓妳擔心了...

**旁白**：兩人抱在一起，都在哭泣。

**旁白**：李杰站在門口，看著這一幕，心中五味雜陳。

**旁白**：他看向凌墨，那個神秘的男人正靜靜地站在一旁。

**旁白**：現在，是時候得到一些答案了。
