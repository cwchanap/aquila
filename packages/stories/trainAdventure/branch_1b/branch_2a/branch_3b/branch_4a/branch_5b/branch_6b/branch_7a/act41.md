# 第四十一幕：上週六的月台

**旁白**：李杰看著眼前的三人——凌墨、恵美、還有剛剛平復情緒的浩介。

**旁白**：太多疑問在他腦中盤旋。

**旁白**：凌墨為什麼會在這裡？他怎麼知道自己被綁架？他又是如何找到那個地方的？

**李杰**：凌墨，你能解釋一下...到底發生了什麼事嗎？

**旁白**：李杰的聲音充滿了困惑和急切。

**李杰**：你為什麼會在東京？你不是應該...

**旁白**：他欲言又止，看了一眼恵美和浩介。

**凌墨**：坐下吧，這個故事有點長。

**旁白**：凌墨走到小桌旁坐下，示意其他人也坐。

**旁白**：李杰和浩介在桌邊盤腿坐下，恵美則去廚房倒了幾杯水。

**旁白**：她把水杯放在每個人面前，然後安靜地坐在凌墨旁邊。

**凌墨**：事情要從上週六說起。

**旁白**：凌墨的聲音平穩，開始講述那天的故事。

**凌墨**：那天晚上，我原本打算陪恵美去搭那班車。她想要回到那個世界。

**旁白**：李杰看向恵美，她低著頭，雙手握著水杯。

**凌墨**：我們約好在新宿站的月台見面。但當我到達的時候，卻看到了一個意外的人。

**旁白**：他看向浩介。

**凌墨**：他站在月台上，一個人，看起來若有所思。

**清水浩介**：那時候...我剛和李先生談完話。

**旁白**：浩介接過話頭，聲音有些不好意思。

**清水浩介**：李先生告訴了我一些關於那班車的事...那些超現實的經歷。

**旁白**：他停頓了一下，回憶著那個晚上。

**清水浩介**：回家的路上，我突然想起來，那個週六...9月21日...

**清水浩介**：剛好是姐姐失蹤20週年的前兩天。9月23日，就是她消失的那一天。

**旁白**：他的聲音變得有些哽咽。

**清水浩介**：我也不知道為什麼，就突然很想去那個月台看看。

**清水浩介**：不是為了調查什麼，只是...想站在她最後出現的地方，懷念一下。

**旁白**：凌墨點了點頭，繼續說道。

**凌墨**：我原本只是在月台上等恵美，她說會晚一點到。

**凌墨**：但突然有個人走向我，看起來很激動。

**旁白**：凌墨看向浩介。

**凌墨**：就是他。他直接走到我面前，叫出了我的名字。

**清水浩介**：因為...我認得你。20年前，我們一起在便利商店打工。

**旁白**：浩介的聲音有些顫抖。

**清水浩介**：雖然20年過去了，但你...你幾乎沒有變。

**清水浩介**：我一眼就認出來了。所以我走過去叫住你。

**凌墨**：他告訴我他在找失蹤的姐姐。當他說出姐姐的名字時，我就知道了。

**凌墨**：清水恵美。我們在那個世界一起度過了兩年。

**清水浩介**：當時我完全驚呆了...凌墨，那個20年前和姐姐一起失蹤的人...

**清水浩介**：居然就站在我面前。

**旁白**：浩介的眼中閃過當時的震驚。

**凌墨**：我們談了一會兒。他提到了你，李杰。

**旁白**：凌墨轉向李杰。

**凌墨**：他說你也去過那個世界，還回來了。這讓我很感興趣。

**凌墨**：一個陌生人，居然也經歷了同樣的事情...而且還能回來。

**旁白**：李杰點了點頭，開始理解事情的脈絡。

**凌墨**：就在我們談話的時候，恵美也來了。

**旁白**：恵美抬起頭，接過話題。

**清水恵美**：我...我沒想到會在那裡看到浩介。

**旁白**：她的聲音輕柔，帶著當時的震驚和慌亂。

**清水恵美**：20年了...他已經長大了，變成一個成熟的大人...

**旁白**：她的眼眶再次泛紅。

**清水浩介**：姐姐...

**旁白**：浩介伸手握住姐姐的手。

**清水恵美**：我當時...真的不知道該說什麼。

**清水恵美**：我還記得他還是個15歲的少年，現在卻已經35歲了...

**旁白**：她深吸一口氣，努力控制情緒。

**凌墨**：他們兩個抱在一起哭了很久。

**旁白**：凌墨的語氣平淡，但能感覺到他對那個場景的理解。

**凌墨**：恵美告訴浩介，她一直活著，但不敢回來找他。

**清水浩介**：我問她為什麼...為什麼不來找我...

**旁白**：浩介的聲音顫抖著。

**清水恵美**：因為...我要再回去那個世界。

**旁白**：恵美的聲音變得堅定。

**清水恵美**：我不知道這次能不能平安回來。如果回不來...

**清水恵美**：那浩介就要經歷第二次失去我的痛苦。

**旁白**：李杰看著恵美，等待她繼續說下去。

**清水恵美**：我想去救一個人。高山翔...我的...我的男朋友。

**旁白**：她的聲音變得更輕。

**清水恵美**：他還困在那個世界。我想把他帶回來，帶回這個真實的世界。

**清水浩介**：姐姐說，她已經做好了犧牲的準備。

**旁白**：浩介握緊恵美的手。

**清水浩介**：但我告訴她，我不想再失去她了。一次都嫌多，更別說兩次。

**旁白**：凌墨繼續說道。

**凌墨**：就在他們談話的時候，末班車進站了。

**旁白**：他的聲音變得低沉。

**凌墨**：橙色的車身，熟悉的聲音，我們都知道那是什麼。

**凌墨**：車門打開，恵美看著那輛車，然後看看浩介。

**旁白**：恵美輕聲說道。

**清水恵美**：那一刻...我突然意識到，浩介也是我重要的人。

**旁白**：她看向弟弟。

**清水恵美**：我想救翔...但我也不想再讓浩介一個人。

**清水恵美**：我...我不知道該怎麼辦。

**凌墨**：車門開了又關，列車沒等我們做決定。

**旁白**：凌墨的語氣中帶著一絲遺憾。

**凌墨**：它開走了，駛入隧道，消失在黑暗中。

**旁白**：房間陷入短暫的沉默。

**旁白**：李杰看著眼前的三人，開始理解那個夜晚發生的一切。

**旁白**：一次意外的相遇，一場姐弟的重逢，還有一個艱難的抉擇。

**旁白**：而那班車，就這樣離開了，沒有帶走任何人。
