# 第四十二幕：追蹤與營救

**旁白**：李杰看著眼前的三人——凌墨、恵美、還有剛剛平復情緒的浩介。

**旁白**：終於安全了，他鬆了口氣，掏出手機想確認時間。

**旁白**：螢幕亮起，顯示著數個未接來電和訊息通知。

**旁白**：大部分是公司同事的，但有幾通是同一個號碼——中野海斗。

**李杰**：中野...

**旁白**：李杰點開訊息記錄，快速瀏覽著。

**旁白**：「李先生！你還好嗎？！」

**旁白**：「我拖住了那兩個黑西裝男一陣子，但之後就失去你的蹤跡了！」

**旁白**：「請回覆我！我很擔心！到底發生了什麼事？」

**旁白**：最後一則訊息是三個小時前發的。

**凌墨**：怎麼了？

**旁白**：凌墨注意到李杰的表情。

**李杰**：是我的同事...他之前幫我對付過那些黑西裝男。

**李杰**：看起來他當時拖住了他們一陣子，但後來就失去我的行蹤了。

**凌墨**：你應該回覆他。讓他知道你平安。

**旁白**：李杰點點頭，快速打了一則簡訊。

**旁白**：「我沒事，已經安全了。謝謝你的幫忙。明天辦公室詳談。」

**旁白**：發送後，他把手機收起來，看向凌墨。

**旁白**：太多疑問需要解答。

**凌墨**：那天晚上之後，浩介帶著恵美回家了。

**旁白**：凌墨繼續講述著那之後的故事。

**凌墨**：我們交換了聯絡方式。我回到自己的公寓，以為事情就這樣結束了。

**旁白**：他頓了頓，眼神變得凝重。

**凌墨**：但第二天，浩介聯絡我，說有可疑的人在跟蹤他們。

**清水浩介**：對...我帶姐姐去便利商店買東西的時候，注意到有個穿黑色西裝的男人一直跟著我們。

**旁白**：浩介回憶著當時的情況。

**清水浩介**：一開始我以為是巧合，但我們轉了幾條街，那個人還是在後面。

**凌墨**：我讓他先別慌張，繼續正常生活。我會暗中觀察。

**旁白**：凌墨的語氣變得專業。

**凌墨**：接下來幾天，我跟著浩介和恵美，保持距離觀察。

**凌墨**：確實有人在跟蹤他們。一個穿黑西裝的男人，一直保持著距離監視。

**旁白**：他看向恵美。

**凌墨**：但有趣的是，他們的目標似乎不是恵美，而是浩介。

**清水恵美**：我們也覺得很奇怪...為什麼要跟蹤浩介？

**凌墨**：我懷疑他們在追查什麼線索。浩介最近接觸過誰，去過哪裡。

**旁白**：凌墨看向李杰。

**凌墨**：你和浩介見過面，對吧？在秋葉原。

**李杰**：是的...上週日。我們在居酒屋談了很久。

**凌墨**：那可能就是原因。他們在追查你，而浩介成了線索。

**旁白**：李杰感到一陣愧疚，是他連累了浩介。

**凌墨**：為了以防萬一，我聯絡了恵美，讓她在浩介身上放一個GPS追蹤器。

**清水恵美**：我趁浩介不注意，把追蹤器放進他的錢包裡。

**旁白**：恵美小聲說道。

**清水恵美**：雖然有點對不起他...但這樣至少能確保他的安全。

**凌墨**：今天下午，追蹤器顯示浩介突然移動到了郊區。

**凌墨**：速度很快，明顯是坐車。而且目的地是一個廢棄的工業區。

**旁白**：凌墨的聲音變得更加嚴肅。

**凌墨**：我馬上意識到出事了。他被抓走了。

**清水浩介**：對...他們在路上突然攔住我，然後就...

**旁白**：浩介的聲音還有些顫抖。

**凌墨**：我立刻趕到那個地方。是一棟廢棄的建築。

**凌墨**：當我到達的時候，看到一個黑西裝男從建築裡走出來，上了一輛車離開了。

**旁白**：李杰的眼睛睜大。

**李杰**：那應該就是去我家拿筆電的那個人...

**凌墨**：我不知道他要去哪裡，但我的首要任務是救人。

**凌墨**：我進入建築，在裡面搜索了大約一個小時。

**旁白**：凌墨回憶著當時的情況。

**凌墨**：那棟樓很大，有很多房間。我必須小心不發出聲音。

**凌墨**：終於，我聽到了說話聲。是你的聲音，李杰。

**旁白**：他看向李杰。

**凌墨**：你在和另一個人對話。從談話內容判斷，那個人應該是審問你。

**凌墨**：我在門外等待時機。然後聽到那個人說要出去找他的同伴。

**凌墨**：我迅速退到走廊轉角處，找到了一根廢棄的鋼管。

**凌墨**：他走出房間後沿著走廊往出口方向走，完全沒注意到後面。

**旁白**：凌墨的聲音依然平靜，但李杰能想像當時的緊張。

**凌墨**：等他走過我藏身的位置時，我從背後襲擊，一擊擊倒了他。

**清水浩介**：然後凌墨就進來救我們了...

**旁白**：浩介的聲音充滿了感激。

**凌墨**：我之前已經叫了計程車在外面等。時間計算得剛剛好。

**旁白**：凌墨說得輕描淡寫，但李杰知道這需要多麼精準的判斷和冷靜的執行。

**旁白**：房間裡陷入短暫的沉默。

**李杰**：那個黑西裝男...你看到他離開多久後，你就進去救我們了？

**凌墨**：我看到他開車離開後，就立刻進入建築了。

**旁白**：凌墨回憶著當時的情況。

**凌墨**：那棟樓很大，有很多房間。我花了將近一個小時才找到你們所在的房間。

**凌墨**：找到你們、解決掉另一個黑西裝男、帶你們離開...之後的過程大約二十分鐘。

**李杰**：所以...從他離開到我們逃出建築，總共過了一個多小時...

**旁白**：李杰皺起眉頭，計算著時間。

**李杰**：從三鷹來回...如果順利的話，差不多就是這個時間。

**旁白**：他抬起頭看向凌墨。

**李杰**：我們離開的時候他還沒回來，但現在...他應該已經回去了吧？

**凌墨**：很有可能。

**旁白**：凌墨的語氣變得凝重。

**凌墨**：如果他發現筆電裡什麼都沒有，又回到建築發現同伴被打倒、人質逃走...

**旁白**：他沒有說完，但每個人都明白那意味著什麼。

**清水浩介**：那...那他們會不會...會不會找到這裡來？

**旁白**：浩介的聲音再次充滿恐懼。

**凌墨**：我不確定。

**旁白**：凌墨坦誠地說道。

**凌墨**：這個公寓是我租的，沒有用真名。但如果他們有足夠的資源和時間...

**旁白**：他停頓了一下。

**凌墨**：不過至少現在，這裡應該是安全的。他們需要時間追查線索。

**李杰**：我們應該報警吧？

**旁白**：李杰突然想到這點。

**凌墨**：我已經報警了。

**旁白**：凌墨平靜地說道。

**凌墨**：在帶你們離開之後，我用自己的手機匿名報案。

**凌墨**：警察現在應該正在那棟建築調查。

**清水浩介**：那太好了！他們會被抓起來的，對吧？

**旁白**：浩介的聲音充滿希望。

**凌墨**：我不太樂觀。

**旁白**：凌墨搖了搖頭。

**凌墨**：我們沒有人真正受傷，頂多只有一些瘀青和綁痕。

**凌墨**：他們可以說是誤會，或是編造其他理由。沒有確鑿的證據。

**旁白**：他看向李杰和浩介。

**凌墨**：而且，如果我們出面作證，反而會暴露自己的位置。

**凌墨**：對方顯然有一定的背景和資源。在沒有足夠證據的情況下，警察能做的很有限。

**李杰**：所以...他們可能很快就會被放出來？

**凌墨**：很有可能。

**旁白**：凌墨的語氣帶著無奈。

**凌墨**：但至少，這會讓他們暫時分心，給我們爭取一些時間。

**旁白**：房間陷入沉重的沉默。

**旁白**：那個黑西裝男現在已經發現了李杰的謊言嗎？

**旁白**：他們會如何追查？會找到這個公寓嗎？

**旁白**：警察的介入能阻止他們多久？

**旁白**：太多未知的問題懸在空中，沒有人能給出答案。
