# 第四十三幕：隱匿與接觸

**旁白**：逃離那棟廢棄建築後，四個人決定暫時待在一起。

**旁白**：凌墨的公寓不大，客廳、廚房、臥室連在一起，擠進四個人顯得格外狹窄。

**旁白**：但沒有人抱怨。比起自由和安全，空間狹小根本不算什麼。

**旁白**：他們制定了簡單的策略——恵美和浩介暫時待在公寓裡，避免外出。

**旁白**：凌墨負責採買食物和日用品，也負責觀察外面的動靜。

**旁白**：而李杰...他必須回去上班。

**李杰**：我不能一直躲著。

**旁白**：李杰在第二天早晨說道，語氣堅定。

**凌墨**：你確定嗎？那些人可能還在找你。

**李杰**：正因如此，我才要回去。

**旁白**：李杰看向凌墨。

**李杰**：我們現在什麼都不知道——他們下一步會做什麼？有多少人在找我們？他們掌握了多少情報？

**李杰**：躲在這裡只會讓我們一直處於被動。

**旁白**：他深吸一口氣，眼神變得銳利。

**李杰**：與其等著他們找上門，不如我主動出去，看看他們會有什麼反應。

**李杰**：如果他們出現，至少我們能掌握主動權，了解他們的動向。

**李杰**：如果他們沒有出現...那也說明了些什麼，不是嗎？

**凌墨**：那我跟著你。

**旁白**：凌墨的語氣堅定。

**凌墨**：我會保持距離，暗中觀察。如果那些黑西裝男再次出現，至少我能及時反應。

**李杰**：這樣...不會太危險嗎？

**凌墨**：比讓你一個人去安全。

**旁白**：凌墨從口袋裡掏出一個小巧的裝置，看起來像是鑰匙圈。

**凌墨**：而且，我會在你身上放一個GPS追蹤器。

**凌墨**：這樣即使我跟丟了，或是你被帶走，我也能立刻找到你的位置。

**清水恵美**：這個...跟我放在浩介身上的一樣嗎？

**旁白**：恵美小聲問道。

**凌墨**：對。同樣的裝置。

**旁白**：凌墨把追蹤器遞給李杰。

**凌墨**：放在你的錢包裡，或是口袋裡。只要隨身攜帶就好。

**李杰**：好。

**旁白**：李杰接過追蹤器，放進西裝外套的內袋。

**凌墨**：還有一點。如果遇到危險，立刻打電話給我。

**凌墨**：不要試圖自己解決。記住，他們人多勢眾，而且有武器。

**李杰**：我明白。

**旁白**：李杰點點頭，認同凌墨的安排。

**旁白**：於是，李杰回到了三鷹的家，也回到了公司。

**旁白**：而凌墨則像影子一樣，在遠處暗中跟隨。

**旁白**：隔天早上，李杰走進辦公室。

**旁白**：同事們看到他都紛紛打招呼，問他這幾天去哪了。

**旁白**：李杰只是含糊地說家裡有些急事要處理。

**旁白**：他才剛在座位上坐下，還沒來得及打開電腦，中野就快步走了過來。

**中野海斗**：杰！

**旁白**：中野的聲音壓得很低，但能聽出急切。

**中野海斗**：你...你真的沒事嗎？

**李杰**：我沒事。

**旁白**：李杰環顧四周，其他同事都在忙著自己的工作。

**李杰**：等一下午休時間，我們去外面談。

**旁白**：中野點點頭，回到了自己的座位。

**旁白**：上午的時間在處理堆積的郵件和文件中度過。

**旁白**：李杰試著專注在工作上，但腦中仍然迴盪著這幾天發生的一切。

**旁白**：終於到了午休時間。

**旁白**：李杰和中野走出辦公大樓，來到附近一個較為安靜的公園。

**旁白**：兩人坐在長椅上，周圍只有幾個遛狗的老人和推嬰兒車的母親。

**中野海斗**：杰，那天晚上...到底發生了什麼？

**旁白**：中野直接切入主題，語氣充滿擔憂。

**中野海斗**：我拖住了其中一個黑西裝男一陣子，但當我趕到巷子裡時，你已經不見了。

**中野海斗**：地上只有你的公事包...我打了好幾通電話給你，但都沒人接...

**李杰**：我被他們抓走了。

**旁白**：李杰深吸一口氣，環顧四周確認沒有人注意。

**李杰**：另一個黑西裝男用電擊棒擊倒了我，然後把我帶到某個地方。

**中野海斗**：什麼？！

**旁白**：中野的聲音提高了，但立刻又壓低。

**中野海斗**：天啊...那你現在...你是怎麼逃出來的？

**李杰**：有人救了我。

**旁白**：李杰沒有說出凌墨的名字。

**李杰**：一個也經歷過那班列車的人。

**中野海斗**：我明白了...那那些黑西裝男呢？他們還在追你嗎？

**李杰**：不確定。這幾天我沒有發現跟蹤的跡象。

**李杰**：但我還是很小心。每天都在警戒著。

**旁白**：中野沉默了幾秒，臉上浮現愧疚的表情。

**中野海斗**：對不起...那天我沒能保護好你。

**中野海斗**：如果我動作快一點，或許就能阻止他們把你帶走...

**李杰**：別這麼說。

**旁白**：李杰搖了搖頭。

**李杰**：你已經幫了我很多了。如果不是你當時拖住另一個人...

**李杰**：我可能連反應的時間都沒有。

**李杰**：中野，你做得夠多了。而且你也要小心。

**李杰**：那些人...他們不是普通的跟蹤狂。很危險。

**中野海斗**：我知道。但如果再遇到麻煩，一定要聯絡我。

**旁白**：中野的語氣變得堅定。

**中野海斗**：我們是同事，也是朋友。不要一個人扛著。

**李杰**：謝謝，中野。

**旁白**：李杰感到一陣溫暖，但同時也感到沉重。

**旁白**：中野是個好人，一個願意冒險幫助他的同事。

**旁白**：但現在的情況太複雜了，李杰不想把更多人拉進這個危險的漩渦。

**旁白**：兩人又聊了幾句，便回到辦公室繼續工作。

**旁白**：日子在這種緊繃的狀態下度過。

**旁白**：白天，李杰在辦公室處理著工作，假裝一切正常。

**旁白**：晚上下班後，他會繞一些路，確認沒有被跟蹤，才回到凌墨的公寓。

**旁白**：又過了幾天。

**旁白**：某個傍晚，凌墨從外面回來，手裡提著超市的塑膠袋。

**凌墨**：我打聽到一些消息。

**旁白**：他把東西放在小桌上，表情凝重。

**李杰**：什麼消息？

**凌墨**：警察逮捕了那個被我打倒的黑西裝男。

**清水浩介**：真的嗎？那太好了！

**旁白**：浩介的臉上閃過一絲希望。

**凌墨**：別高興得太早。

**旁白**：凌墨搖了搖頭。

**凌墨**：那個人什麼都不說，警察沒有足夠的證據拘留他。他已經被釋放了。

**旁白**：房間裡的空氣瞬間凝固。

**李杰**：那...另一個人呢？那個去我家的黑西裝男？

**凌墨**：他出了車禍。

**旁白**：凌墨的語氣帶著一絲意外。

**凌墨**：在前往三鷹的路上，他的車子和一輛卡車相撞。現在人在醫院。

**清水恵美**：車禍...是意外嗎？

**旁白**：恵美輕聲問道，聲音裡帶著疑惑。

**凌墨**：我不知道。可能是意外，也可能..。

**旁白**：他沒有說完，但每個人都明白那個可能性。

**旁白**：也許有其他勢力介入了。

**旁白**：日子繼續在狹小的公寓裡流逝。

**旁白**：白天，陽光從窗簾的縫隙透進來，在地板上投下細長的光線。

**旁白**：夜晚，四個人擠在小桌旁吃飯，電視裡播放著無關緊要的新聞。

**旁白**：李杰在客廳的角落鋪了睡墊，凌墨和浩介輪流睡沙發，恵美則睡在臥室裡。

**旁白**：空間雖然狹窄，但至少是安全的。

**旁白**：一週過去了，然後兩週、三週...

**旁白**：一個月後的某個下午，凌墨再次從外面回來。

**旁白**：這次他的表情比之前輕鬆一些。

**凌墨**：這一個月來，我沒有發現任何跟蹤的跡象。

**旁白**：他環視著其他三人。

**凌墨**：那些黑西裝男...似乎放棄了。至少暫時是這樣。

**清水浩介**：真的嗎？我們...我們安全了？

**凌墨**：不能百分之百確定，但至少現在看起來是安全的。

**旁白**：凌墨坐下來，語氣變得嚴肅。

**凌墨**：但我們也不能永遠躲在這裡。你們都有自己的生活要過。

**李杰**：你的意思是...？

**凌墨**：我們需要回歸正常生活。但要做好準備。

**旁白**：他從口袋裡掏出手機。

**凌墨**：我在想，我們可以用手機的GPS追蹤器互相分享位置。

**凌墨**：這樣每個人都能知道其他人的狀態。如果有人遇到危險，其他人能立刻知道。

**清水恵美**：這個方法...聽起來不錯。

**旁白**：恵美點點頭。

**李杰**：好，就這麼辦。

**旁白**：四個人互相交換了手機，設定好GPS追蹤和位置分享。

**旁白**：在手機螢幕上，四個小點代表著四個人的位置，全都聚集在中野的這個小公寓裡。

**旁白**：但很快，這些點就會分散到東京的各個角落。

**凌墨**：明天開始，我們試著恢復正常生活。

**凌墨**：但要隨時保持警覺。如果發現任何可疑的人或事，立刻聯絡其他人。

**旁白**：三人都點了點頭。

**旁白**：隔天，李杰第一次在一個月後離開那間公寓。

**旁白**：陽光灑在臉上，街道上的聲音、來往的行人、商店的招牌...一切都是如此熟悉又陌生。

**旁白**：他深吸一口氣，感受著自由的氣息。

**旁白**：浩介和恵美姐弟回到了清水家，與母親團聚。

**旁白**：凌墨則繼續住在中野的公寓。

**旁白**：李杰也回到三鷹的家住。

**旁白**：其實在這一個月期間，他曾經回家確認過狀況。

**旁白**：公寓的門鎖沒有被破壞的痕跡，房間裡一切如常。

**旁白**：客廳的茶几上，他的筆記型電腦就安靜地放在那裡，螢幕闔上，從未被人觸碰過。

**旁白**：那個黑西裝男在去三鷹的路上就出了車禍，根本沒到過這裡。

**旁白**：至少，家裡是安全的。

**旁白**：現在回到自己的公寓居住，李杰終於能稍微放鬆一些。

**旁白**：不用再擠在狹小的空間裡，不用再輪流睡沙發和地板。

**旁白**：手機上的四個小點，每天在東京的地圖上移動著，互相確認著彼此的安全。

**旁白**：又過了幾天。

**旁白**：一個週四的下午，李杰正在辦公室裡整理文件，手機突然震動了一下。

**旁白**：一則來自陌生號碼的簡訊。

**旁白**：他點開訊息，螢幕上顯示著簡短的文字。

**旁白**：「李先生，我是麻布警察署的宇佐美誠一郎，警部。」

**旁白**：「我想和您談談。關於您最近遭遇的綁架事件，以及那些黑西裝男的事。」

**旁白**：「如果您有興趣，請回覆這則訊息。我可以提供協助。」

**旁白**：李杰盯著螢幕，心臟開始加速跳動。

**旁白**：麻布警察署...警部...

**旁白**：這個人怎麼知道他被綁架的事？他怎麼知道黑西裝男？

**旁白**：他怎麼拿到自己的電話號碼？

**旁白**：而且...警察會主動聯繫他嗎？

**旁白**：李杰的手指懸在螢幕上方，猶豫著該不該回覆。

**旁白**：這可能是一個陷阱，也可能是一個轉機。

**旁白**：辦公室裡的日光燈發出微弱的嗡嗡聲。

**旁白**：螢幕上的文字靜靜地等待著回應。

**旁白**：李杰深吸一口氣，開始打字。
