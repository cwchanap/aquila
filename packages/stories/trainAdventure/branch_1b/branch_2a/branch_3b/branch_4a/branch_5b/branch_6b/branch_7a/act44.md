# 第四十四幕：麻布警部的邀約

**旁白**：李杰盯著手機螢幕上的訊息，深吸一口氣。

**旁白**：一個警部...主動聯繫他...

**旁白**：他的手指在螢幕上停留了幾秒，然後開始打字。

**旁白**：「宇佐美警部，您好。請問您聯繫我的目的是什麼？」

**旁白**：訊息送出後，李杰緊盯著螢幕。

**旁白**：不到一分鐘，對方就回覆了。

**旁白**：「李先生，我目前正在調查一起失蹤案件。」

**旁白**：「最近我與一位失蹤乘客的親屬取得了聯繫，在他的協助下，我得知了黑西裝男攻擊您的事件。」

**旁白**：「我也從高尾警察署那邊獲得了一些相關資訊，最終找到了您。」

**旁白**：李杰皱起眉头。

**旁白**：失蹤乘客的親屬...是誰？

**旁白**：高尾警察署...他們又知道些什麼？

**旁白**：他繼續看著螢幕，又一則訊息傳來。

**旁白**：「如果方便的話，我們可以見面詳談。我想這對您和對我的調查都會有幫助。」

**旁白**：「明天晚上，六本木有一家安靜的餐廳，我們可以在那裡談談。您覺得如何？」

**旁白**：李杰沒有立刻回覆。

**旁白**：他站起身，環顧辦公室。

**旁白**：中野正坐在自己的座位上，專注地看著電腦螢幕。

**旁白**：李杰走到中野的座位旁。

**李杰**：中野，能借一步說話嗎？

**旁白**：中野抬起頭，看到李杰嚴肅的表情，立刻點了點頭。

**旁白**：兩人走到茶水間，這裡沒有其他人。

**中野海斗**：怎麼了，杰？

**李杰**：我收到一個自稱是麻布警察署警部的訊息。

**旁白**：李杰把手機遞給中野看。

**李杰**：他說在調查失蹤案件，想和我見面談談。

**旁白**：中野仔細看著螢幕上的對話記錄，表情變得凝重。

**中野海斗**：麻布警察署...警部...

**旁白**：他沉思了幾秒。

**中野海斗**：這可能是個機會。如果他真的是警察，也許能幫上忙。

**李杰**：但也可能是陷阱。那些黑西裝男或許會偽裝成警察。

**中野海斗**：那就讓我陪你去。

**旁白**：中野的語氣堅定。

**中野海斗**：餐廳是開放的場所，會有其他客人在。而且如果我們兩個人一起去，至少比你一個人安全。

**旁白**：李杰看著中野認真的表情。

**李杰**：你確定嗎？這可能很危險...

**中野海斗**：我確定。而且...

**旁白**：中野露出一個輕鬆的笑容。

**中野海斗**：如果真的是陷阱，至少我能再幫你拖住幾個人，對吧？

**旁白**：李杰忍不住笑了出來。

**李杰**：好，那就麻煩你了。

**旁白**：兩人回到座位。

**旁白**：李杰拿起手機，開始打字。

**旁白**：「宇佐美警部，我可以赴約。但我想帶一位朋友同行，他也和這件事有關。可以嗎？」

**旁白**：很快，回覆來了。

**旁白**：「當然沒問題。明天晚上七點，六本木『靜月亭』餐廳。期待與您見面。」

**旁白**：李杰把餐廳名稱記下來，然後鎖上手機。

**旁白**：但他還有一個人需要聯繫。

**旁白**：下班後，李杰回到家中。

**旁白**：他坐在客廳的沙發上，拿出手機，找到凌墨的聯絡方式。

**旁白**：他打了一段文字訊息，把宇佐美警部的事情簡單說明。

**旁白**：幾分鐘後，手機震動。

**旁白**：「我也想去。」

**旁白**：凌墨的回覆簡潔明瞭。

**旁白**：李杰想了想，回覆道：「好，明天晚上六點半在六本木站見面。」

**旁白**：「明白。」

**旁白**：李杰放下手機，靠在沙發上。

**旁白**：明天...他就要見到那個神秘的宇佐美警部了。

**旁白**：這個人到底是敵是友？他真的能幫上忙嗎？

**旁白**：還是...這又是一個新的陷阱？

**旁白**：窗外的東京夜景燈火通明，但李杰的心中依然充滿了不安。

---

**旁白**：第二天傍晚，李杰和中野一起離開公司。

**旁白**：兩人沒有直接前往六本木，而是先在新宿轉了幾圈，確認沒有被跟蹤。

**旁白**：夕陽漸漸西沉，天空染上一片橙紅色。

**旁白**：他們搭上地鐵，前往六本木站。

**旁白**：六本木站的人潮熙熙攘攘，上班族、觀光客、年輕人混雜在一起。

**旁白**：李杰和中野走出檢票口，在出口處等待。

**旁白**：幾分鐘後，一個熟悉的身影出現了。

**旁白**：凌墨穿著黑色的連帽外套和牛仔褲，背著一個小背包。

**旁白**：他的表情依然冷靜，眼神銳利地掃視著周圍。

**凌墨**：李杰。

**旁白**：他走到李杰面前，點了點頭。

**李杰**：這位是我的同事，中野海斗。

**旁白**：李杰介紹道。

**中野海斗**：你好，我是中野。

**旁白**：中野伸出手。

**凌墨**：凌墨。

**旁白**：凌墨簡短地回應，和中野握了握手。

**旁白**：他的握手很有力，但很快就鬆開了。

**李杰**：餐廳在這附近，走路大約十分鐘。

**旁白**：三人一起走出車站。

**旁白**：六本木的街道充滿了都市的活力。

**旁白**：霓虹燈招牌閃爍著，餐廳和酒吧的門口站著招攬客人的服務生。

**旁白**：遠處可以看到東京鐵塔，在暮色中亮起燈光。

**旁白**：三人沿著較為安靜的巷弄前進。

**旁白**：凌墨一直保持警覺，不時回頭確認後方。

**旁白**：走了大約八分鐘，他們來到一條狹窄的巷子。

**旁白**：巷子深處，一個木製的招牌上寫著「靜月亭」三個字。

**旁白**：這是一家傳統的日式料理店，外觀低調而優雅。

**旁白**：紙糊的拉門透出溫暖的黃色燈光。

**旁白**：李杰深吸一口氣，伸手推向店門。

**旁白**：門後，等待他們的會是什麼？
