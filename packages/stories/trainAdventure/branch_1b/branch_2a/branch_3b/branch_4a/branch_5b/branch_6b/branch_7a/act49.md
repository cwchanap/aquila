# 第四十九幕：醉後真言

**旁白**：週五下班後，大輔突然提議。

**齋藤大輔**：欸欸！我們要不要找個時間聚一聚？

**旁白**：他興奮地看著李杰和中野。

**齋藤大輔**：我來公司也一個多月了，想說大家一起放鬆一下！

**中野海斗**：好主意！那就這個週末吧！

**旁白**：中野立刻贊同。

**中野海斗**：來我家怎麼樣？我那裡空間大，還有遊戲機。

**齋藤大輔**：太好了！我帶啤酒和零食！

**旁白**：大輔開心地拍手。

**中野海斗**：杰，你也一定要來啊！

**李杰**：好，那就週六晚上？

**中野海斗**：沒問題！我傳地址給你們！

---

**旁白**：週六傍晚，李杰拎著一袋零食來到中野家。

**旁白**：這是一間兩房一廳的公寓，整潔而溫馨。

**旁白**：客廳的電視櫃下擺著遊戲主機，茶几上已經放了幾包洋芋片。

**中野海斗**：杰，歡迎歡迎！

**旁白**：中野打開門，笑容滿面。

**中野海斗**：大輔還沒到，應該快了。

**李杰**：你家還真不錯。

**旁白**：李杰環顧四周。

**中野海斗**：還好啦，一個人住的話夠用了。

**旁白**：中野接過李杰手中的袋子。

**中野海斗**：咦，你買了這麼多？

**李杰**：隨便買的，不知道你們喜歡吃什麼。

**旁白**：門鈴響起。

**中野海斗**：應該是大輔到了！

**旁白**：中野打開門，大輔抱著一大箱啤酒站在門口。

**齋藤大輔**：我來啦！

**旁白**：他的臉上掛著燦爛的笑容。

**齋藤大輔**：中野，杰先生，晚上好！

**中野海斗**：哇，大輔，你買這麼多啤酒？

**旁白**：中野看著那一大箱啤酒，有些驚訝。

**齋藤大輔**：今晚要盡興嘛！不醉不歸！

**旁白**：大輔笑著走進來，把啤酒放在茶几上。

**齋藤大輔**：還有還有，我帶了炸雞和薯條！

**旁白**：他從另一個袋子裡拿出熱騰騰的炸雞。

**中野海斗**：太棒了！那我們開始吧！

---

**旁白**：三人圍坐在茶几前，電視螢幕上顯示著遊戲畫面。

**旁白**：中野和大輔操作著手把，李杰在旁邊觀戰。

**齋藤大輔**：哈哈！海斗，你的角色死了！

**中野海斗**：等等等等！我還能復活！

**旁白**：兩人專注地盯著螢幕。

**李杰**：中野，你左邊有敵人。

**中野海斗**：看到了！

**旁白**：激烈的戰鬥持續了幾分鐘。

**齋藤大輔**：耶！我們贏了！

**旁白**：大輔興奮地舉起手。

**中野海斗**：配合得不錯！杰，下一局換你上！

**李杰**：好啊。

**旁白**：李杰接過手把。

**旁白**：遊戲一局接著一局，三人輪流上場。

**旁白**：茶几上的零食越來越少，啤酒罐越堆越高。

**齋藤大輔**：來來來，休息一下，喝一杯！

**旁白**：大輔拿起啤酒罐，遞給李杰和中野。

**中野海斗**：乾杯！

**旁白**：三人碰杯，一口氣喝下大半罐。

**齋藤大輔**：哈～爽！

**旁白**：大輔滿足地嘆氣。

**齋藤大輔**：說真的，能來這家公司工作真的太好了！

**中野海斗**：怎麼突然這麼感慨？

**齋藤大輔**：因為你們人都很好啊！

**旁白**：大輔的臉已經有些紅了。

**齋藤大輔**：不像我之前那家公司，老闆超怪的！

**中野海斗**：怎麼個怪法？

**旁白**：中野好奇地問。

**齋藤大輔**：就是...很難形容啦！

**旁白**：大輔抓了抓頭。

**齋藤大輔**：做事方式很奇怪，講話也莫名其妙的。

**齋藤大輔**：總之就是個怪人！

**李杰**：但你還是待了那麼久？

**齋藤大輔**：因為工作本身還好啊，不會太忙。

**旁白**：大輔喝了一口啤酒。

**齋藤大輔**：而且我弟弟失蹤了，我需要穩定的收入去找他...

**旁白**：大輔的聲音變得有些低落。

**齋藤大輔**：雖然薪水普通，但至少穩定嘛。

**中野海斗**：那後來為什麼離職？

**齋藤大輔**：因為...那個老闆後來也不見了。

**旁白**：大輔的語氣變得有些複雜。

**齋藤大輔**：剛開始大家都很開心，工作變輕鬆了！

**齋藤大輔**：但是...沒有老闆以後，業務就開始走下坡...

**旁白**：他嘆了口氣。

**齋藤大輔**：客戶一個一個流失，案子越來越少。

**齋藤大輔**：我看情況不對，趕快找新工作，免得等公司倒閉就來不及了！

**中野海斗**：原來如此...那你來我們公司真是太好了！

**旁白**：中野拍了拍大輔的肩膀。

**齋藤大輔**：對啊！幸好有你推薦！

**旁白**：大輔露出感激的笑容。

**齋藤大輔**：幸好我跑得快，在公司倒閉前就找到新工作了！

**旁白**：大輔露出慶幸的笑容。

**中野海斗**：哈哈，那你來我們公司真是太好了！

**旁白**：中野拍了拍大輔的肩膀。

**齋藤大輔**：對啊！幸好有你推薦！

**旁白**：大輔露出感激的笑容。

**中野海斗**：為你的新生活乾杯！

**旁白**：三人再次碰杯。

**旁白**：他打了個酒嗝。

**旁白**：三人繼續喝著啤酒，話題又回到遊戲和工作。

**旁白**：時間一分一秒過去，桌上的啤酒罐越來越多。

---

**旁白**：已經是深夜十一點了。

**旁白**：三人都已經喝得有些醉了，臉紅通通的。

**齋藤大輔**：欸欸...你們說...如果有平行世界該有多好...

**旁白**：大輔靠在沙發上，眼神迷離。

**中野海斗**：平行世界？

**齋藤大輔**：對啊...在那個世界，蓮還在...我們一家人還在一起...

**旁白**：他的聲音變得哽咽。

**中野海斗**：大輔...

**齋藤大輔**：而且...而且那個怪老闆...就讓他永遠困在另一個世界吧...

**旁白**：他喃喃自語著，聲音越來越小。

**齋藤大輔**：反正他...也困在那裡了...回不來了...

**旁白**：李杰突然清醒了一些。

**旁白**：他看向大輔，眉頭微皺。

**李杰**：大輔先生...你剛才說什麼？

**旁白**：李杰的語氣變得認真。

**李杰**：你的老闆...困在另一個世界？

**旁白**：大輔愣了一下，似乎意識到自己說了什麼。

**齋藤大輔**：啊...沒...沒什麼啦...

**旁白**：他揮了揮手，試圖轉移話題。

**齋藤大輔**：我...我只是喝醉了...胡說八道...

**李杰**：大輔先生，你是說你老闆也...

**旁白**：李杰想要追問。

**齋藤大輔**：沒有沒有...我什麼都沒說...

**旁白**：大輔搖著頭，聲音變得含糊不清。

**齋藤大輔**：我只是...太累了...想睡了...

**旁白**：他的眼皮越來越重。

**李杰**：等等，大輔先生...

**旁白**：但大輔已經沒有反應了。

**齋藤大輔**：嗯...

**旁白**：他的頭垂了下來，靠在沙發上。

**旁白**：幾秒後，傳來了輕微的鼾聲。

**旁白**：李杰想要繼續追問，但大輔已經完全睡著了。

**中野海斗**：杰...我也...好睏...

**旁白**：中野也靠在沙發上，眼皮越來越重。

**中野海斗**：今天...玩得...好開心...

**旁白**：他的聲音越來越小，很快也睡著了。

**旁白**：李杰看著兩個已經睡著的朋友。

**旁白**：腦中不斷回想著大輔剛才說的話。

**旁白**：「讓他永遠困在另一個世界」...「也困在那裡了」...

**旁白**：大輔是什麼意思？

**旁白**：那個機車老闆...真的在另一個世界？

**旁白**：還是只是大輔喝醉後的胡言亂語？

**旁白**：但為什麼大輔意識到後要急著否認？

**旁白**：李杰想要再思考下去，但酒精的作用讓他的意識越來越模糊。

**旁白**：眼皮變得沉重，頭也開始昏沉。

**李杰**：先...睡一下吧...明天再...

**旁白**：他趴在茶几上，還沒說完就閉上了眼睛。

**旁白**：客廳裡只剩下三個人平穩的呼吸聲。

**旁白**：電視螢幕還亮著，顯示著遊戲的暫停畫面。

**旁白**：茶几上散落著空啤酒罐和零食袋。

**旁白**：三個人就這樣歪七扭八地睡在客廳裡。

**旁白**：窗外的東京夜色依然閃爍。

**旁白**：而一個模糊的線索，就在這個醉酒的夜晚若隱若現。

**旁白**：大輔究竟知道些什麼？

**旁白**：那個老闆，真的和那個世界有關嗎？
