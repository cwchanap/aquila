# 第五十三幕：春日來臨

**旁白**：這是一個平凡的週五下午。李杰坐在辦公桌前,專注地檢視著程式碼。

**旁白**：窗外的陽光灑進辦公室,東京的三月已經開始有了春天的氣息。

**旁白**：從那個九月的夜晚到現在,已經過了兩年半。李杰依然在這家公司工作,依然和中野、大輔一起處理專案。生活平靜而規律。

**旁白**：部門經理山田部長走到李杰的座位旁。

**山田部長**：李君,有件事要跟你說。

**李杰**：什麼事？

**旁白**：李杰抬起頭。

**山田部長**：下週我們團隊會有一位新的初級業務分析師加入,是個應屆畢業生,女孩子。到時候可能需要你幫忙帶一下新人。

**李杰**：好的,了解。沒問題。

**旁白**：山田部長點點頭,走開了。

**旁白**：李杰轉頭繼續工作,但旁邊的中野和大輔已經湊了過來。

**中野海斗**：欸欸！新人是女生耶！而且是應屆畢業生,那應該二十二、二十三歲左右吧？

**齋藤大輔**：說不定很可愛！我們公司女生本來就不多,尤其是工程部門！

**旁白**：兩人越說越激動。

**李杰**：你們...冷靜一點。

**旁白**：李杰無奈地說。

**李杰**：工作是工作,別想太多。

**中野海斗**：欸,杰你不期待嗎？說不定是個大美女！

**齋藤大輔**：對啊！

**李杰**：我只是覺得...不要像變態一樣想東想西比較好。

**旁白**：李杰淡淡地說。

**旁白**：雖然他也從未交過女朋友,但他知道在職場上保持專業很重要。

**中野海斗**：哈哈哈！杰你太認真了！我們只是期待一下而已嘛！

**齋藤大輔**：對啊對啊！反正就是希望能有個好相處的新同事！

**旁白**：兩人笑著說,但眼中的期待掩飾不住。

**旁白**：李杰搖搖頭,繼續工作。

---

**旁白**：一週後的週一早晨,辦公室裡瀰漫著一股微妙的氣氛。中野和大輔比平時還要早到,兩人都穿得比平常整齊。大輔甚至特地理了髮,噴了古龍水。

**李杰**：你們兩個...

**旁白**：李杰看著兩人,不知道該說什麼。

**中野海斗**：欸,今天是新人報到的日子嘛！稍微注重一下形象！

**齋藤大輔**：對啊對啊！第一印象很重要的！

**旁白**：就在這時,山田部長帶著一個女孩走了過來。

**山田部長**：大家早安。來,我來介紹一下新成員。

**旁白**：女孩站在山田身旁,微笑著向大家鞠躬。她穿著簡單的白色襯衫和黑色長褲,長髮紮成馬尾。五官精緻,皮膚白皙,臉上掛著明朗而活潑的笑容。

**原田繪里花**：大家好～我叫原田繪里花！是上智大學商業管理專業的應屆畢業生！以後請多多指教！

**旁白**：她的聲音清脆而充滿活力,說話時眼睛亮晶晶的。

**旁白**：中野和大輔瞬間眼睛發亮。

**山田部長**：原田同學會擔任我們團隊的初級業務分析師。李君,你來當她的入職夥伴,帶她熟悉環境和業務流程。

**李杰**：好的,沒問題。

**旁白**：李杰站起身,向繪里花點頭。

**李杰**：我是李杰,負責後端開發。有什麼問題隨時可以問我。

**原田繪里花**：哇！李前輩好！

**旁白**：繪里花立刻笑著鞠躬。

**原田繪里花**：那就麻煩您了！我會很乖的！啊不對,應該說我會很努力的！

**旁白**：她吐了吐舌頭,顯得有些俏皮。

**中野海斗**：我是中野海斗！也是做後端開發的！

**旁白**：中野立刻站起來,笑容燦爛。

**齋藤大輔**：我是齋藤大輔！負責運維和技術支援！原田同學有什麼需要幫忙的,儘管說！

**原田繪里花**：哇～謝謝中野前輩、齋藤前輩！大家都好熱情啊！感覺來對地方了！

**旁白**：繪里花笑得很開心,眼睛彎成了月牙形。

**旁白**：李杰看了看身旁兩個過度興奮的同事,嘆了口氣。

**旁白**：今天...可能會很漫長。

---

**旁白**：上午的時間,李杰帶著繪里花熟悉辦公環境。介紹各個部門的位置、會議室的使用規則、系統的登入方式。繪里花很認真地聽,偶爾做筆記,問問題時也很有條理。

**李杰**：這是我們的專案管理系統,所有的任務都會在這裡追蹤。

**原田繪里花**：哇～看起來好專業！請問如果要新增任務,流程是什麼？我想盡快熟悉整個工作流程！

**李杰**：你先提出需求,然後我們會評估可行性和工作量。最後在系統裡建立任務。

**原田繪里花**：明白了！謝謝李前輩！

**旁白**：繪里花點點頭,在筆記本上寫下重點。

**原田繪里花**：欸對了,這個系統可以自己改顏色標籤嗎？我喜歡用不同顏色分類,這樣看起來比較可愛！

**李杰**：可以,在設定裡面。

**原田繪里花**：太好了！我最喜歡粉紅色和淺藍色！

**旁白**：她興奮地說著,眼睛閃閃發光。

**旁白**：中野和大輔時不時會找藉口過來「關心」新人。

**中野海斗**：原田同學,需要幫忙嗎？有什麼不懂的可以問我們！

**齋藤大輔**：對啊！

**原田繪里花**：謝謝前輩們！不過李前輩已經解釋得很清楚了！而且我發現李前輩超有耐心的耶！

**旁白**：繪里花笑著說。

**原田繪里花**：雖然看起來有點嚴肅,但其實人很好！

**李杰**：...我沒有很嚴肅。

**原田繪里花**：有啦有啦！不過這樣反差很可愛！

**旁白**：她調皮地笑著。

**旁白**：李杰有些無奈,但也沒反駁。

**旁白**：終於到了午休時間。

**中野海斗**：原田同學,一起去吃午餐吧！我們帶你去附近好吃的店！

**齋藤大輔**：對啊對啊！

**原田繪里花**：好啊好啊！我最喜歡探索新餐廳了！前輩們都知道哪些好吃的店嗎？我超愛吃的！尤其是甜點！

**旁白**：四人一起走出辦公室。

---

**旁白**：他們來到附近一家定食店。這是團隊常去的餐廳,價格合理,份量也足。四人點完餐,坐在靠窗的位置。

**中野海斗**：原田同學,今天第一天感覺怎麼樣？

**原田繪里花**：超好的！大家都很親切！而且辦公室的氣氛也很棒！比我想像中還要輕鬆耶！我本來還擔心會不會很嚴肅,結果完全不會！

**旁白**：繪里花笑得很開心。

**齋藤大輔**：那就好！我們團隊氣氛很不錯的！

**中野海斗**：原田同學...有男朋友嗎？

**旁白**：中野突然問道。

**旁白**：李杰差點被茶嗆到。

**原田繪里花**：欸？怎麼突然問這個！

**旁白**：繪里花笑了起來,臉微微紅了。

**原田繪里花**：沒有啦,我單身！而且最近都專注在找工作和畢業論文上,哪有時間談戀愛！

**齋藤大輔**：真的嗎？這麼可愛居然沒有男朋友！

**李杰**：喂,你們兩個,別嚇到新人。

**李杰**：你們聽起來像變態一樣。

**中野海斗**：欸？！我們只是關心而已啊！

**齋藤大輔**：對啊對啊！就是想了解一下新同事嘛！

**原田繪里花**：哈哈,沒關係的李前輩！我不介意啦！而且我也很好奇前輩們的事情呢！

**旁白**：繪里花笑著說,眼神閃爍著好奇的光芒。

**原田繪里花**：那...前輩們呢？你們有女朋友嗎？我也可以問吧？公平起見！

**中野海斗**：我...我沒有。

**齋藤大輔**：我也沒有...

**旁白**：兩人尷尬地笑著。

**原田繪里花**：李前輩呢？

**李杰**：也沒有。

**旁白**：李杰簡短地回答。

**原田繪里花**：欸？三位前輩都沒有女朋友？這也太巧了吧！還是說...你們三個其實是好兄弟聯盟？發誓一起單身到底那種？

**旁白**：繪里花眨了眨眼,語氣俏皮。

**中野海斗**：哈哈...才沒有那種東西啦...我們都比較宅啦...平常不是工作就是打遊戲...

**齋藤大輔**：對啊...沒什麼機會認識女生...

**原田繪里花**：哇～前輩們都喜歡打遊戲啊？打什麼遊戲？手機遊戲還是電腦遊戲？

**旁白**：餐點送上來了,四人開始用餐。

**中野海斗**：我們主要玩電腦遊戲！

**中野海斗**：最近在玩《幻想編年史》的新版本！那個新副本超難的！

**齋藤大輔**：對對對！上次我們三個組隊打了三個小時才過關！

**原田繪里花**：哇～聽起來很厲害！我不太會玩遊戲耶...只會玩一些簡單的手機遊戲。像是消消樂那種！

**中野海斗**：沒關係！以後可以教你！我們可以組隊一起玩！

**齋藤大輔**：對啊！

**李杰**：你們別給新人壓力。

**旁白**：李杰無奈地說。

**原田繪里花**：哈哈,沒關係的！如果有機會的話,我也想試試看！不過我可能會很菜喔！到時候不要嫌棄我！

**旁白**：她笑著說,表情有些調皮。

**中野海斗**：不會不會！我們會保護你的！

**原田繪里花**：那前輩們平常還會做什麼？除了打遊戲之外？會去哪裡玩嗎？

**齋藤大輔**：呃...有時候會去遊戲中心...或者去秋葉原逛逛...

**原田繪里花**：秋葉原啊！我也喜歡去那裡！雖然我不太懂動漫,但是那裡的女僕咖啡廳超有趣的！而且電器也很便宜！

**中野海斗**：對對對！我們也常去那裡買遊戲和周邊商品！

**原田繪里花**：欸那以後可以一起去嗎？我想看看你們都買些什麼！一定很有趣！

**旁白**：繪里花興奮地說著,眼睛閃閃發光。

**齋藤大輔**：當然可以！

**中野海斗**：好啊好啊！

**李杰**：...你們不要太興奮。

**旁白**：李杰在旁邊淡淡地說。

**原田繪里花**：李前輩也一起來嘛！不然我一個女生跟兩個大叔去,感覺怪怪的！

**齋藤大輔**：欸？！大叔？！我們才三十歲而已！

**中野海斗**：對啊！還很年輕好嗎！

**原田繪里花**：哈哈哈！開玩笑的啦！前輩們看起來都很年輕！

**旁白**：繪里花笑得很開心,氣氛變得更加輕鬆。

**原田繪里花**：不過說真的,李前輩你也會去秋葉原嗎？

**李杰**：偶爾。

**原田繪里花**：哇～那你也喜歡動漫嗎？看起來不像耶！感覺你比較像那種...嗯...比較酷酷的類型？

**李杰**：...我就是普通的宅男。

**原田繪里花**：哈哈哈！原來如此！

**原田繪里花**：那李前輩最喜歡什麼動畫？

**旁白**：李杰簡短地回答了幾個動畫名稱。繪里花則繼續問東問西,話題從動漫聊到美食,從美食聊到旅遊。她的話很多,但不會讓人覺得煩躁。反而有一種自然的親和力,讓氣氛變得很輕鬆。

**旁白**：午餐在歡快的氣氛中結束。四人一起走回辦公室。春天的陽光灑在街道上,櫻花樹的枝頭已經冒出了花苞。辦公室裡多了一個新成員。雖然興趣不同,生活方式也不同,但這樣的差異反而讓團隊的氣氛變得更加有趣。

**中野海斗**：原田同學,下午也要加油喔！有問題隨時問我們！

**齋藤大輔**：對啊！

**原田繪里花**：好的！謝謝前輩們！不過我覺得應該不會有太大問題啦！因為李前輩教得很好！

**旁白**：繪里花笑著說。

**原田繪里花**：雖然看起來有點冷淡,但其實超溫柔的！

**李杰**：...我沒有冷淡。

**原田繪里花**：有啦有啦！不過我喜歡這種反差！

**旁白**：她調皮地眨了眨眼。

**旁白**：李杰走在最後面,看著前面三個人的背影。嘴角不禁露出一絲笑容。平靜的生活,似乎又增添了一些新的色彩。
