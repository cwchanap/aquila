# 第五十四幕：聖誕聚會

**旁白**：繪里花加入團隊後，辦公室的氣氛變得更加活潑。

**旁白**：作為初級業務分析師，她的工作是在專案經理和工程師之間建立溝通橋樑。

**旁白**：這意味著她需要頻繁地和李杰討論技術需求和實作細節。

**旁白**：幾乎每天，她都會來到李杰的座位旁。

**原田繪里花**：李前輩，打擾一下！專案經理說客戶希望在報表頁面加入匯出PDF的功能，這個技術上可行嗎？

**李杰**：可以，用現有的函式庫就能做到。

**原田繪里花**：太好了！那大概需要多久時間？我要跟經理回報。

**李杰**：如果優先處理的話，大概三天。

**原田繪里花**：了解！那我先跟經理說，等他確認後我再來找你建立任務！

**旁白**：繪里花轉身要走，突然又回頭。

**原田繪里花**：對了李前輩，你中午想吃什麼？我要去買便當，可以順便幫你帶！

**李杰**：...不用了，我自己去買就好。

**原田繪里花**：欸～不要客氣嘛！反正我也要去，幫你帶一下很簡單的！

**李杰**：那...雞肉便當吧。

**原田繪里花**：好的！交給我！

**旁白**：她蹦蹦跳跳地走開了。

**旁白**：中野在旁邊笑著說道。

**中野海斗**：杰，繪里花對你真好啊！每天都幫你買便當。

**李杰**：...她只是順路而已。

**中野海斗**：哈哈，你就是太遲鈍了！

**旁白**：另一天，李杰正在檢查程式碼。

**旁白**：繪里花突然出現在他身後，探頭看著螢幕。

**原田繪里花**：哇～李前輩你在寫什麼？看起來好複雜！

**李杰**：資料庫查詢的優化。

**原田繪里花**：這麼多英文和符號...你們工程師真的好厲害！我完全看不懂！

**旁白**：她歪著頭，表情有些困惑。

**李杰**：這只是基本的SQL語法。

**原田繪里花**：對你來說是基本，對我來說就像外星文字啦！

**旁白**：她笑著說。

**原田繪里花**：不過李前輩工作的時候好專注喔！每次都皺著眉頭，看起來超認真的！

**李杰**：...妳有事嗎？

**原田繪里花**：啊，對對對！我是來問專案進度的！經理想知道登入功能改善的部分完成了沒？

**李杰**：完成了，已經部署到測試環境。

**原田繪里花**：太好了！那我馬上去跟經理報告！李前輩辛苦了！

**旁白**：她又蹦蹦跳跳地離開了。

**旁白**：大輔從座位上探出頭來。

**齋藤大輔**：繪里花真的很有活力啊！每天都這麼開朗！

**中野海斗**：對啊，有她在，辦公室都變得熱鬧多了！

**李杰**：...是很吵。

**中野海斗**：欸～你這麼說太過分了吧！人家明明很可愛啊！

**旁白**：李杰沒有回應，繼續專注在工作上。

**旁白**：但他不得不承認，繪里花的出現確實讓平淡的工作日變得不太一樣了。

---

**旁白**：時光飛逝，轉眼間繪里花加入團隊已經一個月了。

**旁白**：十二月的東京街頭掛滿了聖誕裝飾，商店櫥窗裡閃爍著溫暖的燈光。

**旁白**：這天是十二月二十四日，聖誕夜。

**旁白**：下午四點，辦公室裡的人已經開始收拾東西準備下班。

**旁白**：大輔走到李杰和中野的座位旁。

**齋藤大輔**：欸欸，今天聖誕夜耶！我們要不要一起去吃飯慶祝一下？

**中野海斗**：好主意！去哪裡？

**齋藤大輔**：去居酒屋吧！六本木那邊有一家不錯的店，我之前去過！氣氛很好，東西也好吃！

**中野海斗**：可以啊！杰，你也一起來吧？

**李杰**：好。

**齋藤大輔**：那我去問問繪里花要不要一起來！

**旁白**：大輔走向繪里花的座位。

**齋藤大輔**：繪里花，今天聖誕夜！我們要去六本木的居酒屋聚餐，妳要不要一起來？

**原田繪里花**：欸？聖誕聚餐嗎？

**旁白**：繪里花抬起頭，臉上露出有些為難的表情。

**原田繪里花**：不好意思...我今天晚上已經約了朋友了...沒辦法去耶...

**齋藤大輔**：這樣啊...那沒辦法了...

**中野海斗**：沒關係沒關係！下次再一起吧！

**旁白**：中野笑著說，但語氣中帶著一絲失望。

**原田繪里花**：真的很抱歉！下次一定要約我喔！

**齋藤大輔**：沒問題！那我們就先走了！繪里花也要玩得開心喔！

**原田繪里花**：好的！前輩們也是！聖誕快樂！

**旁白**：三人收拾好東西，一起離開了辦公室。

---

**旁白**：傍晚六點，三人來到六本木。

**旁白**：街道上擠滿了人潮，到處都是準備慶祝聖誕夜的情侶和朋友。

**旁白**：霓虹燈在寒冷的冬夜中閃爍，營造出熱鬧而溫暖的氛圍。

**旁白**：他們走進一家名為「月見屋」的居酒屋。

**旁白**：店內裝潢走傳統和風路線，木製的桌椅散發著歲月的氣息。

**旁白**：牆上掛著燈籠，吧台後方擺滿了各式各樣的日本酒。

**旁白**：雖然是聖誕夜，但這家店並沒有特別的聖誕裝飾，反而保持著純粹的日式風格。

**旁白**：三人找了個角落的位置坐下。

**齋藤大輔**：來來來！先點啤酒！今天是聖誕夜，要喝個痛快！

**中野海斗**：沒問題！我請客！

**李杰**：你們不要喝太多。

**齋藤大輔**：哈哈哈！放心啦！不會像上次那樣喝到斷片的！

**旁白**：服務生送來三杯生啤酒和一盤毛豆。

**旁白**：三人舉杯碰了碰。

**齋藤大輔**：乾杯！聖誕快樂！

**中野海斗**：聖誕快樂！

**李杰**：聖誕快樂。

**旁白**：三人喝了一大口啤酒。

**齋藤大輔**：啊～爽！果然下班後喝啤酒最棒了！

**中野海斗**：對啊！而且今天是聖誕夜，感覺特別有氣氛！

**旁白**：他們陸續點了炸雞、烤魚、炒麵、煎餃等料理。

**旁白**：店內的氣氛熱鬧而溫馨，其他桌的客人也都在開心地聊天喝酒。

**齋藤大輔**：欸，說到聖誕夜...你們有沒有想過要找女朋友啊？

**中野海斗**：突然說什麼啦！

**齋藤大輔**：因為啊，像今天這種日子，情侶都在外面約會，我們三個大男人在這裡喝酒...感覺有點...

**中野海斗**：...有點孤單？

**齋藤大輔**：對啊！

**旁白**：兩人嘆了口氣。

**李杰**：我覺得這樣挺好的。

**中野海斗**：杰，你真的不想交女朋友嗎？

**李杰**：沒有特別想。

**齋藤大輔**：欸～你這樣不行啦！要積極一點！

**中野海斗**：對啊！像繪里花那樣開朗可愛的女生，不考慮看看嗎？

**李杰**：...她只是同事。

**齋藤大輔**：可是她對你很好啊！每天幫你買便當，還常常來找你聊天！

**中野海斗**：對啊！我看她蠻喜歡你的！

**李杰**：那只是工作需要。她是業務分析師，本來就要常常和工程師溝通。

**齋藤大輔**：你就是太遲鈍了啦！

**旁白**：大輔喝了一口啤酒，繼續說道。

**齋藤大輔**：不過說真的，繪里花真的很不錯！個性開朗，又很努力工作！

**中野海斗**：而且很可愛！笑起來眼睛會彎成月牙形！

**齋藤大輔**：對對對！超可愛的！

**李杰**：...你們兩個冷靜點。

**旁白**：李杰淡淡地說。

**中野海斗**：哈哈哈！我們只是實話實說而已！

**旁白**：三人繼續喝著酒，聊著工作、遊戲、還有最近看的動畫。

**旁白**：時間一分一秒過去，桌上的空酒杯越來越多。

**旁白**：雖然沒有女生同行，但三個男人的聚會也有它獨特的樂趣。

**旁白**：沒有需要顧慮形象，沒有需要小心翼翼的話題。

**旁白**：只有最純粹的友誼和輕鬆的氛圍。

---

**旁白**：時間來到晚上九點半。

**旁白**：三人決定結束今晚的聚會。

**齋藤大輔**：差不多該回家了！明天還要上班呢！

**中野海斗**：對啊，今天喝得很開心！

**李杰**：走吧。

**旁白**：三人結完帳，走出居酒屋。

**旁白**：冬夜的冷風吹在臉上，讓微醺的頭腦清醒了一些。

**齋藤大輔**：哈～今天喝得好開心啊！

**中野海斗**：對啊！下次再來！

**齋藤大輔**：那你們路上小心！我家就在附近，走路就到了！

**中野海斗**：你住六本木真好，都不用搭電車！

**齋藤大輔**：哈哈，這倒是！那我先走了！你們也早點回家！

**旁白**：大輔揮揮手，轉身走進巷子裡。

**中野海斗**：杰，我們去車站吧。

**李杰**：嗯。

**旁白**：兩人一起走向六本木站。

**旁白**：六本木的夜晚依然熱鬧，聖誕夜的氣氛還沒散去。

**旁白**：街道上到處都是打扮時髦的年輕人，情侶手牽手走在燈火通明的街道上。

**旁白**：兩人進入六本木站，刷卡進站。

**中野海斗**：我們都搭大江戶線，不過方向不一樣。

**李杰**：你往哪邊？

**中野海斗**：我往大門那邊，然後換淺草線去大森町。你呢？

**李杰**：我直接往新宿，然後換中央線。

**中野海斗**：那我們在這裡就分開了。路上小心，明天見。

**李杰**：嗯，你也是。

**旁白**：兩人在閘口道別。

**旁白**：中野走向往大門方向的月台，李杰則走向往新宿方向的月台。

**旁白**：李杰搭上往新宿方向的大江戶線。

**旁白**：晚上十點的車廂裡人不算多，大部分都是準備回家的上班族。

**旁白**：李杰找了個座位坐下，看著窗外飛逝的隧道景色。

**旁白**：今晚的聚會很愉快，雖然只有三個男人。

**旁白**：他想起中野和大輔調侃他對繪里花的感覺。

**旁白**：她只是同事...對吧？

**旁白**：列車搖晃著前進，經過好幾站後，終於到達新宿站。
