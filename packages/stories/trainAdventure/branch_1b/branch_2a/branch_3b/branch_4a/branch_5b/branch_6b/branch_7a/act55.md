# 第五十五幕：新宿站的秘密

**旁白**：列車到達新宿站，車門緩緩打開。李杰下了車走向出口。新宿站是東京最繁忙的車站之一，即使是晚上十點，站內依然人來人往。

**旁白**：他刷卡出站，走進夜晚的新宿街頭。聖誕夜的氣氛還很濃厚，街道上擠滿了人群。

**旁白**：情侶手牽手走在燈火通明的街道上，年輕人三五成群地往居酒屋和卡拉OK走去。霓虹燈在冬夜中閃爍，營造出熱鬧而溫暖的氛圍。

**旁白**：李杰感覺有點口渴。剛才在居酒屋喝了不少啤酒，現在需要補充一些水分。

**旁白**：他走進一家7-11便利商店。店內的暖氣讓他感到舒適，冷藏櫃裡擺滿了各式各樣的飲料。

**旁白**：李杰拿了一瓶綠茶，走到櫃檯結帳。

**店員**：謝謝，一百三十日圓。

**旁白**：李杰付了錢，接過找零和收據。

**旁白**：他走出便利商店，打開瓶蓋，喝了一大口。冰涼的綠茶順著喉嚨滑下，讓他清醒了不少。

**旁白**：李杰沒有馬上回車站。不知道為什麼，他突然不想那麼快回家。

**旁白**：也許是因為今晚的聚會讓他心情不錯，也許是因為聖誕夜的氛圍感染了他。

**旁白**：他沿著街道慢慢走著，一邊喝著綠茶，一邊看著來往的人群。

**旁白**：新宿的夜晚充滿了活力。高樓大廈的燈光，商店的霓虹招牌，街頭藝人的音樂——這一切構成了東京獨特的夜景。

**旁白**：李杰在人群中慢慢走著，沒有特定的方向。經過幾家餐廳，幾間商店，還有一個小小的公園。

**旁白**：大約走了十分鐘後，他決定該回去了。明天還要上班，不能太晚睡。

**旁白**：他轉身，準備往車站的方向走。就在這時，他經過一家星巴克咖啡店。

**旁白**：透過玻璃窗，他不經意地瞥了一眼店內的景象。然後，他的腳步停住了。

**旁白**：在靠窗的座位上，他看到了一個熟悉的身影——繪里花。

**旁白**：她穿著一件淺色的大衣，頭髮放了下來，化著比平常更精緻的妝。

**旁白**：和辦公室裡那個活潑開朗的樣子不同，此刻的她看起來更加成熟。

**旁白**：李杰愣住了。他沒想到會在這裡遇到她。她不是說和朋友有約嗎？

**旁白**：在她旁邊，並排坐著另一個女孩。那個女孩有著一頭烏黑的長髮垂在肩上，五官精緻，氣質冷豔，穿著深色的長外套。

**旁白**：她的表情冷靜而專注，和繪里花開朗的氣質形成鮮明對比。

**旁白**：這應該就是繪里花說的朋友吧？但為什麼感覺兩人的氣質差這麼多？

**旁白**：兩個女孩的面前坐著一個男人。那個男人背對著街道，所以李杰看不到他的臉。

**旁白**：但從身形來看，那是個相當高大的男人。寬闊的肩膀撐起筆挺的黑色西裝，後頸的線條顯示他可能是個外國人。

**旁白**：從坐姿和體型來判斷，他應該超過一百八十公分，身材強壯。

**旁白**：李杰下意識地往旁邊移了一步，躲到柱子後面。他不想被發現在看她們。

**旁白**：從柱子後面，他繼續觀察著店內的情況。

**旁白**：繪里花和那個長髮女孩似乎正在認真地聽那個男人說話，偶爾點頭，偶爾交換眼神。

**旁白**：桌上放著幾杯咖啡，還有一些文件。那個男人似乎在解釋著什麼，偶爾用手比劃著。

**旁白**：繪里花的表情很認真，和平常在辦公室裡笑嘻嘻的樣子完全不同。

**旁白**：那個長髮女孩則更加冷靜，偶爾在筆記本上寫著什麼。

**旁白**：氣氛看起來有些正式，不像是普通的朋友聚會，更像是某種會面或者面談。

**旁白**：李杰感到一陣困惑。繪里花到底在做什麼？那個男人是誰？

**旁白**：為什麼會是這樣正式的會面？桌上的那些文件又是什麼？

**旁白**：他站在柱子後面，看著這一幕。繪里花專注地聽著那個男人說話，完全沒有注意到窗外的他。

**旁白**：李杰感到一種奇怪的感覺，就像是無意間看到了不該看的東西。

**旁白**：他知道自己不應該繼續看下去了，這是繪里花的私事，和他無關。而且，如果被發現在偷看，會很尷尬。

**旁白**：李杰深吸一口氣，轉身離開。他加快腳步，往車站的方向走去。

**旁白**：手中的綠茶瓶已經空了，他隨手丟進路邊的回收桶。腦中不斷浮現剛才的畫面。

**旁白**：那個黑西裝的男人是誰？為什麼氣氛看起來那麼正式？

**旁白**：桌上的文件是什麼？那個長髮女孩又是誰？

**旁白**：她和繪里花的氣質完全不同，但她們似乎很熟悉。

**旁白**：李杰走進新宿站，刷卡進站，直接走向中央線的月台。

**旁白**：月台上人不多，冷冽的風從隧道吹來。他站在月台邊緣，腦中還在想著剛才的畫面。

**旁白**：為什麼他會在意這件事？繪里花只是同事，她私下見誰、做什麼，和他沒有關係。

**旁白**：但不知道為什麼，他就是覺得有點在意。

**旁白**：也許是因為這一個月來，繪里花每天都會來找他聊天、幫他買便當。

**旁白**：她開朗的笑容、活潑的個性，已經成為他日常生活的一部分。

**旁白**：所以當他看到她和陌生男人在一起，而且氣氛如此正式時，心中會產生一種微妙的感覺。

**旁白**：好奇？擔心？還是⋯⋯李杰搖了搖頭。

**旁白**：他不應該多想，那是她的私事。而且，她並沒有發現他。

**旁白**：如果明天問她，反而會讓她知道自己看到了那一幕。還是假裝什麼都不知道比較好。

**旁白**：列車進站的聲音打斷了他的思緒。橘色的中央線列車緩緩停靠在月台邊。

**旁白**：李杰搖了搖頭，走進車廂。他找了個座位坐下，看著窗外飛逝的夜景。

**旁白**：列車駛離新宿，經過中野、高圓寺、阿佐谷，最後到達三鷹。

**旁白**：東京的夜晚依然燈火通明，每一盞燈後面都有著不同的故事。

**旁白**：而他只是其中一個普通人，過著平凡的生活。

**旁白**：列車搖晃著前進，帶著他回到三鷹的小公寓。但李杰的思緒，還停留在那家咖啡店的畫面。

**旁白**：繪里花坐在那個高大男人對面的樣子，認真而專注的表情。那是他從未在辦公室裡看過的繪里花。

**旁白**：還有那個長髮女孩，冷豔而神秘，和繪里花完全不同的氣質。

**旁白**：她們是什麼關係？那個黑西裝男人又是誰？桌上的那些文件又是關於什麼？

**旁白**：李杰嘆了口氣，閉上眼睛。

**旁白**：也許明天，他會找機會問問繪里花。也許不會。

**旁白**：但不管怎樣，那個畫面已經深深印在他的腦海中。

**旁白**：這個聖誕夜，留下了一個小小的疑問——一個關於繪里花的秘密。

**旁白**：列車到達三鷹站，李杰下了車。冬夜的寒風吹在臉上，讓他完全清醒了。

**旁白**：他走出車站，沿著熟悉的街道回家。小公寓的燈還是熄著的，一如既往的安靜。

**旁白**：李杰打開門，脫下外套，倒在沙發上。他閉上眼睛，腦中卻還在回想著剛才的畫面。

**旁白**：繪里花坐在那個高大男人對面的樣子，那個冷豔的長髮女孩，桌上的文件，正式而嚴肅的氛圍。

**旁白**：那究竟是什麼樣的會面？為什麼繪里花會和那樣的人見面？

**旁白**：李杰嘆了口氣。也許他不應該問，畢竟那是她的私事。

**旁白**：而且，她並沒有發現他。如果問了，反而會讓她知道自己在偷看。

**旁白**：還是假裝什麼都不知道比較好。

**旁白**：疲憊感逐漸襲來，李杰很快就睡著了。

**旁白**：窗外的東京夜色依舊繁華，無數的燈光在黑暗中閃爍。

**旁白**：每個人都有自己的秘密。而今晚，李杰無意間窺見了繪里花的一角。
