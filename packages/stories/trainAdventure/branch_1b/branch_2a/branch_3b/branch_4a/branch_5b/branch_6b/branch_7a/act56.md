# 第五十六幕：意外的約會

**旁白**：第二天早上，李杰像往常一樣來到辦公室。昨晚的畫面還在腦中徘徊，但他決定假裝什麼都不知道。

**旁白**：大約九點半，繪里花走進辦公室。她穿著簡單的白色襯衫和黑色長褲，頭髮紮成馬尾，臉上掛著一如既往的笑容。

**原田繪里花**：大家早安～！

**旁白**：她向大家打招呼，聲音依然充滿活力。

**中野海斗**：早安！繪里花，昨天聖誕夜玩得開心嗎？

**原田繪里花**：嗯...還不錯啊！

**旁白**：繪里花笑著說，但語氣有些含糊。

**齋藤大輔**：跟朋友去哪裡玩啊？有幾個男生幾個女生？

**旁白**：大輔湊過來，眼中閃爍著八卦的光芒。

**原田繪里花**：就...就普通聚餐而已啦！沒什麼特別的！

**旁白**：繪里花的笑容變得有些僵硬，眼神閃爍了一下。

**齋藤大輔**：欸～不要這麼保密嘛！男生多還是女生多？有沒有看上的對象？

**旁白**：李杰看著繪里花略顯不自在的表情，腦中閃過昨晚的畫面——那個黑西裝男人，那個冷豔的長髮女孩，還有桌上的文件。

**旁白**：她確實在隱瞞什麼。但他不打算揭穿她。

**李杰**：大輔，別煩繪里花了。她都說了只是普通聚餐，你幹嘛一直追問？

**旁白**：李杰抬起頭，語氣平靜地說道。

**齋藤大輔**：欸？杰你幹嘛突然這麼維護她？

**旁白**：大輔轉頭看向李杰，嘴角露出一抹壞笑。

**齋藤大輔**：你又不是繪里花的男朋友，怎麼比我還在意她的事啊？

**旁白**：李杰的臉微微一紅。

**李杰**：我只是覺得你太煩了。

**旁白**：繪里花也紅著臉，低頭看著自己的桌面。

**原田繪里花**：謝...謝謝李前輩...

**旁白**：她小聲地說道，聲音帶著一絲羞澀。

**中野海斗**：好啦好啦，大輔你別鬧了。

**旁白**：海斗笑著打圓場。

**中野海斗**：說到聚會，我想問你們一件事。跨年夜那天，我打算在家裡辦個通宵派對，你們要不要來？

**齋藤大輔**：哇！通宵派對！這個好啊！

**旁白**：大輔立刻興奮起來。

**中野海斗**：對啊！我還邀請了隔壁姊妹組的人一起來。雖然他們也都是男的就是了。

**旁白**：海斗笑著說。姊妹組是公司另一個開發團隊，因為工作內容相似，兩個團隊關係很好。

**齋藤大輔**：太棒了！這樣人多熱鬧！杰，你來吧？繪里花妳也一起來啊！

**旁白**：大輔轉頭看向李杰和繪里花。

**旁白**：李杰正想開口說好，但繪里花突然搶先說話。

**原田繪里花**：不好意思...我那天晚上...已經約了李前輩了...

**旁白**：繪里花的聲音不大，但在安靜的辦公室裡卻格外清晰。

**齋藤大輔**：欸？

**中野海斗**：什麼？

**旁白**：大輔和海斗同時愣住了，兩人睜大眼睛看著繪里花。

**原田繪里花**：我們...打算一起去看跨年煙火...所以沒辦法參加派對...

**旁白**：繪里花繼續說道，臉上帶著靦腆的笑容。

**旁白**：李杰也愣住了。他完全沒想到繪里花會突然說出這樣的話。

**齋藤大輔**：等等等等！你們什麼時候...關係發展到這個階段了？！

**旁白**：大輔幾乎是喊出來的，語氣充滿震驚。

**中野海斗**：對啊！我們完全不知道耶！什麼時候開始的？

**旁白**：海斗也湊過來，眼中充滿好奇。

**原田繪里花**：就...最近啦...

**旁白**：繪里花笑著說，然後轉頭看向李杰。她的眼神帶著一絲期待，彷彿在等他的反應。

**旁白**：李杰的腦中一片混亂。繪里花到底在做什麼？為什麼突然說他們有約？

**旁白**：但看著繪里花那雙期待的眼睛，他突然明白了——她需要一個藉口。一個不去派對的藉口。

**旁白**：也許...她那天晚上還有其他事情要做？就像昨天聖誕夜那樣的會面？

**旁白**：李杰深吸一口氣，決定配合她。

**李杰**：嗯...是啊...

**旁白**：他簡短地回答，臉上帶著一絲不自然。

**齋藤大輔**：天啊！我好嫉妒！為什麼你們在一起了，就剩我一個人單身！

**李杰**：等等，我們沒有...

**旁白**：李杰想要解釋，但大輔根本沒在聽。

**齋藤大輔**：算了算了！我還是乖乖去海斗家的派對，繼續當單身狗好了！

**旁白**：大輔誇張地嘆了口氣，帶著一臉失落的表情回到自己的座位。

**中野海斗**：杰啊，跨年夜可是很重要的日子喔。記得要做好安全措施啊！

**旁白**：海斗壞笑著說，語氣帶著調侃。

**李杰**：你...！

**旁白**：李杰的臉瞬間漲紅。

**原田繪里花**：海斗前輩！你在說什麼啦！

**旁白**：繪里花也紅著臉，聲音帶著一絲嬌嗔。

**中野海斗**：哈哈哈！開玩笑的啦！你們要好好享受喔！

**旁白**：海斗笑著走回自己的座位，留下臉紅的兩人。

**旁白**：辦公室恢復了平靜。大家各自回到工作崗位，只剩下偶爾傳來的鍵盤敲擊聲。

**旁白**：過了大約十分鐘，繪里花走到李杰的座位旁。

**原田繪里花**：李前輩...可以出來一下嗎？

**旁白**：她小聲地說道。

**旁白**：李杰點點頭，跟著她走到茶水間。茶水間裡沒有其他人，只有咖啡機發出低沉的嗡嗡聲。

**原田繪里花**：那個...剛才真的很抱歉...我不應該擅自說我們有約的...

**旁白**：繪里花低著頭，聲音充滿歉意。

**原田繪里花**：其實...我只是突然很想和李前輩一起去看跨年煙火...

**旁白**：她停頓了一下，抬起頭看向李杰。

**原田繪里花**：但又不知道該怎麼開口...所以才會在剛才...那樣說...

**旁白**：她的眼神真誠而期待，臉上帶著一絲羞澀。

**原田繪里花**：李前輩...你願意陪我一起去嗎？

**旁白**：李杰沉默了幾秒，看著眼前這個充滿活力的女孩。

**旁白**：昨晚的畫面再次浮現——她坐在那個黑西裝男人對面，表情認真而專注。那個他從未見過的繪里花。

**旁白**：她身上肯定有什麼秘密。但...那又如何呢？

**李杰**：我有選擇嗎？

**旁白**：李杰嘆了口氣，語氣帶著一絲無奈。

**原田繪里花**：欸？

**李杰**：妳都已經當著大家的面說了，我現在反悔的話，不就很尷尬嗎？

**旁白**：李杰看著繪里花，嘴角微微上揚。

**原田繪里花**：所以...你答應了？

**旁白**：繪里花的眼睛亮了起來。

**李杰**：算是吧。反正我本來也沒什麼計劃。

**原田繪里花**：太好了！謝謝李前輩！

**旁白**：繪里花開心地笑了起來。

**原田繪里花**：那我們約跨年夜晚上六點，在品川站見面！可以嗎？

**李杰**：品川？

**原田繪里花**：對啊！因為我們要去橫濱看煙火，從品川出發比較方便！而且我也住在品川附近！

**李杰**：好，那就品川站。

**原田繪里花**：太棒了！那就這麼說定了！

**旁白**：繪里花蹦蹦跳跳地離開茶水間，留下李杰一個人站在原地。

**旁白**：李杰看著她離開的背影，心中有些複雜。

**旁白**：他不知道繪里花為什麼突然邀請他，也不知道她身上藏著什麼秘密。

**旁白**：但不知道為什麼，他並不討厭這種感覺。

**旁白**：平靜的生活，似乎又出現了一些漣漪。

**旁白**：李杰喝了一口水，轉身走回辦公室。窗外的東京冬日陽光灑進來，帶著一絲溫暖。

**旁白**：跨年夜...還有幾天就到了。
