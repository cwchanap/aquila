# 第五十七幕：跨年之夜

**旁白**：十二月三十一日，跨年夜。傍晚五點半，李杰從三鷹的公寓出發。

**旁白**：他穿著深灰色的大衣，裡面是黑色的毛衣和牛仔褲。這是他少數幾件比較正式的衣服。

**旁白**：雖然說是約會，但他還是不太習慣這種場合。走在前往車站的路上，他的心跳比平常快了一些。

**旁白**：搭上中央線，轉乘山手線，大約四十分鐘後，列車到達品川站。

**旁白**：品川站是東京的主要轉運站之一，即使在傍晚時分，人潮依然洶湧。跨年夜的氣氛讓整個車站都充滿了活力。

**旁白**：李杰走出改札口，在約定的中央剪票口附近等待。他看了看手錶，五點五十五分。還有五分鐘。

**旁白**：他環顧四周，人來人往。情侶、朋友、家人，每個人臉上都帶著期待的笑容。

**旁白**：就在這時，他看到了她。

**旁白**：繪里花從人群中走來。她穿著一件米白色的長大衣，裡面是淺粉色的毛衣和深色的長裙。

**旁白**：長髮披散在肩上，化著比平常更精緻的淡妝。耳邊戴著一對小巧的耳環，在燈光下閃爍著微光。

**旁白**：和辦公室裡那個紮著馬尾、穿著制式襯衫的她完全不同。此刻的繪里花，看起來就像是要去參加正式約會的女孩。

**原田繪里花**：李前輩！

**旁白**：繪里花揮著手，快步走向他。臉上帶著燦爛的笑容。

**李杰**：妳來了。

**旁白**：李杰點點頭，視線在她身上停留了一秒。

**原田繪里花**：對不起！有讓你等很久嗎？

**李杰**：沒有，我也剛到。

**原田繪里花**：太好了！那我們走吧！我知道附近有一家很不錯的餐廳！

**旁白**：繪里花笑著說。

**旁白**：兩人走出車站，走進品川的夜色中。冬夜的寒風吹在臉上，但街道上的人群和燈光營造出溫暖的氛圍。

---

**旁白**：繪里花帶著李杰來到一家洋食餐廳。餐廳裝潢溫馨，暖黃色的燈光讓人感到放鬆。

**旁白**：跨年夜的關係，餐廳裡幾乎客滿，大部分都是情侶或小團體。

**店員**：歡迎光臨。兩位嗎？

**原田繪里花**：對，我有預約。名字是原田。

**店員**：好的，請跟我來。

**旁白**：店員帶著他們走到靠窗的位置。透過玻璃窗可以看到外面的街景。

**旁白**：兩人坐下後，店員遞上菜單。

**原田繪里花**：李前輩，你想吃什麼？這家的漢堡排和奶油燉菜都很好吃喔！

**李杰**：那我點漢堡排好了。

**原田繪里花**：我也要漢堡排！還有...要不要點個沙拉分著吃？

**李杰**：好啊。

**旁白**：繪里花向店員點了餐，然後開心地看著窗外。

**原田繪里花**：今天天氣真好！等一下去橫濱應該能看到很棒的煙火！

**李杰**：橫濱的跨年煙火很有名嗎？

**原田繪里花**：超有名的！每年都有好多人去看！而且橫濱的夜景也很漂亮！

**旁白**：繪里花興奮地說著，雙手比劃著。

**原田繪里花**：李前輩以前有去過橫濱嗎？

**李杰**：工作的時候去過幾次，但都只是去開會，沒有好好逛過。

**原田繪里花**：那今天就當作是觀光啦！雖然時間不多，但至少能看到煙火！

**旁白**：她笑得很開心，眼中閃爍著期待的光芒。

**旁白**：李杰看著她，嘴角微微上揚。不知道為什麼，看到她這麼開心，他的心情也變好了。

**旁白**：這時，李杰的手機震動了。他拿出來看，是大輔傳來的訊息。

**旁白**：「杰！你現在在幹嘛！跟繪里花在一起嗎！好羨慕啊！」

**旁白**：接著又是海斗的訊息。

**旁白**：「我們這邊超無聊的...大輔一直在抱怨...你們要好好享受啊！」

**旁白**：李杰看著訊息，忍不住笑了。

**原田繪里花**：怎麼了嗎？

**李杰**：大輔和海斗在抱怨。

**原田繪里花**：哈哈！他們在派對上不開心嗎？

**李杰**：好像是大輔一直在說他很孤單。

**原田繪里花**：欸～大輔前輩明明很受歡迎的啊！怎麼會孤單！

**旁白**：繪里花笑著說。

**李杰**：大概是看到別人成雙成對，他一個人單身的關係吧。

**原田繪里花**：那...我們等一下拍張照片傳給他們好了！讓他們更嫉妒！

**旁白**：繪里花調皮地眨了眨眼。

**李杰**：...妳還真是壞心眼。

**原田繪里花**：哪有～這叫分享快樂啦！

**旁白**：兩人笑了起來。餐點也在這時送上來了。

**旁白**：漢堡排滋滋作響，淋著濃郁的醬汁。奶油燉菜散發著誘人的香氣。

**原田繪里花**：看起來好好吃！我要開動了！

**李杰**：開動了。

**旁白**：兩人開始用餐。繪里花切了一小塊漢堡排放進嘴裡,臉上露出滿足的表情。

**原田繪里花**：嗯～好好吃！李前輩你覺得呢？

**李杰**：很不錯。

**原田繪里花**：對吧對吧！我就說這家很好吃！

**旁白**：繪里花開心地說著，然後又切了一塊，這次她用叉子指向李杰。

**原田繪里花**：李前輩，要不要嚐嚐我的？

**旁白**：李杰愣了一下。

**李杰**：...不用了，我們點的是一樣的。

**原田繪里花**：欸～可是我想跟你分享嘛！

**旁白**：她笑著說，依然把叉子伸向李杰。

**旁白**：李杰猶豫了一秒，最後還是就著她的叉子吃了那塊漢堡排。

**李杰**：...嗯，很好吃。

**原田繪里花**：對吧！那我也要吃你的！

**旁白**：繪里花笑嘻嘻地伸出叉子，從李杰的盤子裡叉了一塊。

**旁白**：李杰看著她，搖了搖頭，但嘴角帶著笑意。

**旁白**：晚餐在輕鬆愉快的氣氛中結束。兩人走出餐廳，回到品川的街道上。

---

**旁白**：現在是晚上七點半。距離橫濱的煙火還有一段時間。

**原田繪里花**：李前輩，我們在附近走走吧！消化一下！

**李杰**：好。

**旁白**：兩人沿著品川站附近的街道慢慢走著。這一帶有許多商店和餐廳，跨年夜的氣氛讓街道顯得格外熱鬧。

**旁白**：冬夜的寒風吹來，繪里花不自覺地縮了縮脖子。

**李杰**：冷嗎？

**原田繪里花**：還好啦！只是風有點大！

**旁白**：她笑著說，但手卻藏進了大衣口袋裡。

**旁白**：李杰看著她，想了想，脫下自己的圍巾。

**李杰**：拿去用。

**原田繪里花**：欸？可是李前輩你自己...

**李杰**：我不冷。

**旁白**：李杰把圍巾遞給她。繪里花遲疑了一下，接過圍巾，圍在脖子上。

**原田繪里花**：謝謝...好溫暖...

**旁白**：她小聲地說，臉上帶著一絲紅暈。不知道是因為寒風，還是因為害羞。

**旁白**：兩人繼續往前走。經過一家甜點店時，繪里花停下了腳步。

**原田繪里花**：哇！這家的蛋糕看起來好好吃！

**旁白**：她看著櫥窗裡的蛋糕，眼睛發亮。

**李杰**：妳還吃得下？

**原田繪里花**：甜點是另一個胃啦！

**旁白**：繪里花理直氣壯地說。

**李杰**：...妳想吃的話就買吧。

**原田繪里花**：真的嗎！那我們進去看看！

**旁白**：兩人走進甜點店。店內擺滿了各式各樣的蛋糕和甜點。

**旁白**：繪里花在櫃檯前猶豫了很久，最後選了一個草莓塔。

**原田繪里花**：我們可以在橫濱看煙火的時候吃！

**李杰**：好。

**旁白**：李杰結了帳。繪里花提著甜點盒，開心地笑著。

**原田繪里花**：謝謝李前輩！等一下我們一起吃！

**旁白**：又走了一會兒，兩人回到品川站。

**原田繪里花**：差不多該出發去橫濱了！煙火十點半開始，我們現在過去剛好！

**李杰**：嗯。

**旁白**：兩人搭上往橫濱的京急線。列車裡擠滿了準備去看煙火的人群，氣氛熱鬧而興奮。

**旁白**：繪里花和李杰並肩站在車廂裡。列車搖晃著前進，繪里花的手不小心碰到了李杰的手。

**旁白**：她愣了一下，臉微微紅了。

**原田繪里花**：對...對不起...

**李杰**：沒關係。

**旁白**：李杰平靜地說，但心跳卻快了一拍。

**旁白**：大約二十分鐘後，列車到達橫濱。

---

**旁白**：橫濱的夜景果然名不虛傳。高樓大廈的燈光倒映在港灣的水面上，形成一片璀璨的景象。

**旁白**：遠處可以看到紅磚倉庫、摩天輪，還有橫濱地標大廈。

**旁白**：兩人沿著港邊走著，尋找一個適合看煙火的位置。

**原田繪里花**：哇～好漂亮！李前輩你看！那邊就是山下公園！

**旁白**：繪里花興奮地指著前方。

**李杰**：嗯，夜景確實很美。

**旁白**：他們找到一個視野開闊的地方站定。周圍已經聚集了不少人，大家都在等待煙火的開始。

**旁白**：繪里花從包包裡拿出剛才買的草莓塔。

**原田繪里花**：李前輩，我們吃蛋糕吧！

**李杰**：現在吃？

**原田繪里花**：對啊！反正還有一點時間！而且...這樣感覺很浪漫耶！

**旁白**：她笑著說，眼中閃爍著調皮的光芒。

**旁白**：兩人分著吃草莓塔。甜而不膩的奶油，新鮮的草莓，酥脆的塔皮。在冬夜的寒風中，這份甜點帶來了溫暖。

**旁白**：這時，李杰的手機又震動了。他拿出來看，又是大輔的訊息。

**旁白**：「杰！現在怎麼樣了！有牽手嗎！有親親嗎！快回報狀況啊！」

**旁白**：李杰看著訊息，無奈地搖了搖頭。

**原田繪里花**：又是大輔前輩嗎？

**李杰**：嗯...他問我們有沒有牽手。

**原田繪里花**：欸？！

**旁白**：繪里花的臉瞬間紅了。

**原田繪里花**：大輔前輩...真是的...問這種問題...

**旁白**：她低著頭，聲音越來越小。

**旁白**：李杰看著她害羞的樣子，嘴角微微上揚。

**李杰**：不用理他。

**旁白**：他簡短地回覆了一個「很好」，然後把手機收起來。

**旁白**：時間來到十點半。人群開始騷動起來。

**旁白**：突然，一聲巨響。

**旁白**：煙火在夜空中綻放，五彩繽紛的光芒照亮了整個港灣。

**原田繪里花**：哇啊！開始了！

**旁白**：繪里花興奮地拉著李杰的袖子。

**旁白**：一朵又一朵的煙火在夜空中盛開。紅色、藍色、金色、綠色，各種顏色交織成一幅壯麗的畫面。

**旁白**：每一次爆炸，都伴隨著震耳欲聾的聲響。周圍的人群發出驚嘆的歡呼聲。

**原田繪里花**：好漂亮...

**旁白**：繪里花仰望著天空，眼中映著煙火的光芒。

**旁白**：李杰也抬頭看著煙火，但他的視線很快就移到了身旁的繪里花身上。

**旁白**：她的側臉在煙火的光芒中顯得格外動人。眼睛睜得大大的，嘴角帶著開心的笑容。

**旁白**：他突然覺得，煙火雖然美麗，但不如她的笑容來得吸引人。

**旁白**：煙火持續了將近十五分鐘。最後一波煙火在空中綻放，形成巨大的圓形圖案。

**旁白**：當最後的光芒消失時，人群爆發出熱烈的掌聲和歡呼聲。

**原田繪里花**：太美了...李前輩，謝謝你今天陪我來！

**旁白**：繪里花轉頭看向李杰，眼中帶著真誠的感謝。

**李杰**：不用謝。我也...玩得很開心。

**旁白**：李杰難得地坦誠地說道。

**原田繪里花**：真的嗎！太好了！

**旁白**：繪里花開心地笑了起來。

**旁白**：人群開始散去。兩人也隨著人流慢慢往車站走去。

**旁白**：這個跨年夜，將會成為他們共同的美好回憶。
