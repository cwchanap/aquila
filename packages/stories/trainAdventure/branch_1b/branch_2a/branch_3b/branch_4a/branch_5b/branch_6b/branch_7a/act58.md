# 第五十八幕：夜路的陰影

**旁白**：回到品川站已經接近十二點。街道上依然熱鬧，跨年的鐘聲即將響起。

**原田繪里花**：李前輩，我家就在附近。你送我到路口就好了。

**李杰**：我送妳到家門口吧。

**原田繪里花**：欸？可是...

**李杰**：都這麼晚了，妳一個人走不安全。

**旁白**：李杰堅持道。

**原田繪里花**：...好吧。那就麻煩你了。

**旁白**：繪里花點點頭，臉上帶著一絲羞澀。

**旁白**：兩人沿著住宅區的街道走著。品川這一帶雖然靠近車站，但住宅區卻很安靜。

**旁白**：走了大約十分鐘，繪里花停在一棟公寓前。

**原田繪里花**：就是這裡了。

**李杰**：嗯。

**原田繪里花**：今天...真的很開心。謝謝你，李前輩。

**旁白**：繪里花認真地看著李杰。

**李杰**：我才要謝謝妳。今天過得...很充實。

**原田繪里花**：那...以後還可以一起出來玩嗎？

**旁白**：繪里花小心翼翼地問道，眼中帶著期待。

**李杰**：...可以啊。

**旁白**：李杰點點頭。

**原田繪里花**：太好了！那就這麼說定了！

**旁白**：繪里花開心地笑了起來。

**原田繪里花**：對了，你的圍巾...

**旁白**：她解下圍巾，準備還給李杰。

**李杰**：明天上班再還我就好。這麼晚了，妳快回去吧。

**原田繪里花**：嗯...那我進去了。路上小心！

**李杰**：妳也是。晚安。

**原田繪里花**：晚安！

**旁白**：繪里花揮了揮手，走進公寓大門。在門關上前，她回頭看了李杰最後一眼，臉上帶著燦爛的笑容。

**旁白**：李杰站在原地，直到她的身影完全消失，才轉身離開。

---

**旁白**：回到品川站，李杰看了看時刻表。這個時間，他需要搭山手線到新宿，然後轉中央線回三鷹。

**旁白**：他搭上往新宿方向的山手線。車廂裡人不多，大部分都是剛結束跨年活動準備回家的人。

**旁白**：列車搖晃著前進，經過五反田、目黑、惠比壽、涉谷...

**旁白**：李杰坐在座位上，腦中還在回想今晚的畫面。繪里花的笑容，她害羞的表情，還有那雙在煙火光芒下閃閃發亮的眼睛。

**旁白**：他的嘴角不自覺地上揚。

**旁白**：也許...他開始明白，為什麼大家都說戀愛的感覺很美好。

**旁白**：大約二十五分鐘後，列車到達新宿站。

---

**旁白**：新宿站即使在接近午夜時分，依然人潮洶湧。跨年夜讓這個東京最繁忙的車站更加熱鬧。

**旁白**：李杰下了車，走向中央線的月台。他需要穿過幾條通道才能到達。

**旁白**：就在他走過第二條通道時，一種奇怪的感覺突然襲來。

**旁白**：有人在跟蹤他。

**旁白**：這不是錯覺，而是一種直覺。一種在危險時刻變得敏銳的本能。

**旁白**：李杰的腳步沒有停，但他的身體緊繃了起來。

**旁白**：他用餘光掃視周圍。人群熙熙攘攘，到處都是趕著換車的乘客。

**旁白**：但他無法確定是誰在跟蹤他。

**旁白**：突然，他想起了那些黑西裝男人。

**旁白**：兩年多前，他剛從那個世界回來後不久，和清水浩介一起被那群黑西裝男人綁架了。

**旁白**：他們被帶到郊區一個廢棄建築的金屬房間裡，被綁在椅子上。

**旁白**：那些人詢問他關於中村隼人和失蹤列車的事情，甚至用槍威脅。

**旁白**：最後是凌墨喬裝成那個組織的成員，趁機救出了他們。

**旁白**：那次經歷讓他明白——有些組織對那個世界非常感興趣。

**旁白**：而他作為回來的人之一，自然成為了目標。

**旁白**：那之後的日子似乎很平靜，他以為那些人已經不再關注他了。

**旁白**：但現在...這種被跟蹤的感覺...

**旁白**：李杰加快了腳步。他穿過人群，往中央線的月台方向走去。

**旁白**：那種被注視的感覺依然存在，如影隨形。

**旁白**：他再次用餘光掃視周圍，但依然看不出任何異樣。周圍都是普通的乘客——上班族、學生、情侶、家庭。

**旁白**：沒有人看起來可疑。沒有人穿著黑西裝。沒有人盯著他看。

**旁白**：但那種感覺不會錯。有人在看著他。

**旁白**：李杰走下樓梯，來到中央線的月台。月台上也擠滿了人。

**旁白**：他站在月台邊緣，假裝在看手機，但實際上是在用手機的反光觀察身後的情況。

**旁白**：人群中沒有任何異樣。

**旁白**：幾分鐘後，往三鷹方向的中央線列車進站。車門打開，人群湧入。

**旁白**：李杰也跟著上了車。他故意等到最後一秒才上車，然後迅速轉身看向月台。

**旁白**：月台上的人群正在散去，準備搭下一班車。沒有人急著跟上來。

**旁白**：車門關上，列車開始移動。

**旁白**：李杰在車廂裡環顧四周。乘客們都很正常——有人在滑手機，有人在打瞌睡，有人在聊天。

**旁白**：沒有人注意他。

**旁白**：那種被跟蹤的感覺消失了。

**旁白**：李杰鬆了口氣，但心中依然充滿疑問。

**旁白**：剛才到底是誰在跟蹤他？是那些黑西裝男人的同夥？還是其他對那個世界感興趣的組織？

**旁白**：或者...只是他太敏感了？

**旁白**：但他相信自己的直覺。在那個危險的世界裡，正是這種直覺救了他無數次。

**旁白**：李杰找了個座位坐下，看著窗外飛逝的夜景。

**旁白**：列車經過中野、高圓寺、阿佐谷，最後到達三鷹。

**旁白**：李杰下了車，走出車站。三鷹的街道很安靜，和新宿的喧囂形成鮮明對比。

**旁白**：他沿著熟悉的街道往家的方向走。這次，他沒有感覺到任何異樣。

**旁白**：也許...剛才只是錯覺？或者跟蹤的人已經放棄了？

**旁白**：李杰搖了搖頭。不要想太多了。今天是跨年夜，是和繪里花一起度過的美好時光。

**旁白**：他不應該讓這種可能只是錯覺的感覺破壞了今晚的美好回憶。

**旁白**：回到公寓，李杰打開門，脫下外套。

**旁白**：他倒在沙發上，閉上眼睛。腦中浮現的是繪里花的笑容，而不是剛才那種被跟蹤的不安感。

**旁白**：也許只是他多慮了。也許只是跨年夜人潮太多，讓他產生了錯覺。

**旁白**：這時，手機震動了。李杰拿出手機，看到繪里花傳來的訊息。

**原田繪里花**：『李前輩，到家了嗎？』

**旁白**：李杰看著螢幕上的訊息，嘴角不自覺地上揚。剛才的不安感似乎也消散了一些。

**旁白**：他用拇指快速打字回覆。

**李杰**：『剛到。妳也安全到家了吧？』

**旁白**：幾秒鐘後，繪里花的回覆就來了。

**原田繪里花**：『嗯！我已經洗好澡了！今天真的很開心！謝謝李前輩！』

**旁白**：接著又是一則訊息。

**原田繪里花**：『橫濱的煙火超美的對吧！而且那個草莓塔也好好吃！』

**旁白**：李杰看著訊息，想起她在煙火下興奮的樣子，還有一起吃草莓塔時她滿足的表情。

**李杰**：『嗯，很美。妳喜歡就好。』

**原田繪里花**：『李前輩明天有什麼計劃嗎？』

**旁白**：李杰想了想。明天元旦，他通常都是在家裡休息，打打遊戲，或是去附近的神社參拜。

**李杰**：『還沒想好。可能去明治神宮初詣吧。』

**原田繪里花**：『哇！明治神宮！那一定很多人！我也想去耶！』

**旁白**：李杰看著這則訊息，猶豫了一下。她是在暗示什麼嗎？

**旁白**：正當他思考該怎麼回覆時，繪里花又傳來一則訊息。

**原田繪里花**：『不過明天人太多了...李前輩還是好好休息吧！我們之後再約！』

**旁白**：接著是一個可愛的笑臉貼圖。

**旁白**：李杰鬆了口氣，但心中也有一絲微妙的失落感。

**李杰**：『好。那妳也早點休息。晚安。』

**原田繪里花**：『晚安！李前輩～做個好夢喔！』

**旁白**：又是一個笑臉貼圖，這次旁邊還有小星星。

**旁白**：李杰把手機放在茶几上，整個人往後靠在沙發上。

**旁白**：剛才那種被跟蹤的緊張感已經完全消失了，取而代之的是一種溫暖而放鬆的感覺。

**旁白**：也許...今天真的只是他想太多了。

**旁白**：疲憊感逐漸襲來，李杰很快就睡著了。

**旁白**：窗外的東京夜色依舊繁華，無數的燈光在黑暗中閃爍。

**旁白**：新的一年已經到來。而他的平靜生活，似乎又開始出現了一些不安的波瀾。

**旁白**：但不管怎樣，這個跨年夜，依然是美好的。
