# 第六十二幕：父子間的秘密

**旁白**：四月三十日，上午十點。林婉如提著購物袋出門去菜市場。她說中午要做糖醋小排，得去買新鮮的豬肋排。

**林婉如**：小杰，你和爸爸在家等著。我很快就回來。

**李杰**：好，媽慢走。

**旁白**：大門關上的聲音在公寓裡迴盪。李杰坐在客廳的沙發上，手裡拿著手機，隨意地滑著新聞。李明遠剛從陽台走進來，他穿著灰色的舊T恤和運動褲，手裡拿著一本報紙。

**旁白**：父親在對面的單人沙發坐下，展開報紙開始閱讀。客廳裡很安靜，只有翻報紙的沙沙聲，還有遠處傳來的汽車聲。

**旁白**：李杰偷偷瞄了父親一眼。李明遠戴著老花眼鏡，專注地看著報紙。臉上的皺紋在晨光中顯得更深了，兩鬢的白髮也比上次回來時多了一些。

**旁白**：這個畫面讓李杰感到有些不真實。他和父親單獨相處的時間並不多。平時在家裡，大部分時候都是母親在說話，父親只是偶爾應聲。而李杰自己，也更習慣和母親聊天。和父親說話總是有些尷尬，不知道該說什麼。

**旁白**：沉默持續著。李杰看著手機螢幕，但腦中想的卻是別的事——兩年多前，他被黑西裝男人綁架的那一天。那個冰冷的男人說的話，至今還清晰地迴盪在他的記憶裡。

**旁白**：「渡邊陸...和你的父親有聯繫。你的父親,李明遠。」

**旁白**：「你的父親李明遠,和渡邊陸曾經在中國的同一家公司工作。前後大約四年時間,他們有過密切的工作往來。」

**旁白**：李杰當時太震驚，太害怕，沒有多想這件事的含義。後來他成功逃脫，回到日常生活，這段記憶就被他刻意壓在心底。但現在，和父親單獨在一個空間裡，那些話又浮現了出來。

**旁白**：渡邊陸...父親真的認識他嗎？那個在20年前搭上列車失蹤，又在8年前回到這個世界，最後又再次搭上列車的男人...父親和他到底是什麼關係？

**旁白**：李杰深吸一口氣。也許...可以試探一下？

**李杰**：爸。

**旁白**：李明遠抬起頭,從報紙上方看向兒子。

**李明遠**：嗯?

**李杰**：我...想問你一件事。

**李明遠**：什麼事?

**旁白**：李明遠放下報紙,摘下老花眼鏡,認真地看著李杰。

**旁白**：李杰猶豫了一下,努力讓自己的語氣聽起來隨意而自然。

**李杰**：你以前...工作的時候,有沒有認識一個叫渡邊陸的日本人?

**旁白**：問題說出口的瞬間，李杰緊張地盯著父親的反應。李明遠的表情僵了一下——那是一個很微小的變化，眉頭輕輕皺起，眼神閃過一絲困惑，然後是...警戒？但很快，他的表情又恢復了平靜。

**李明遠**：渡邊陸?

**旁白**：父親重複著這個名字，聲音平淡。

**李明遠**：沒聽過這個名字。我不認識什麼渡邊陸。

**旁白**：他的語氣很肯定,但眼神卻變得銳利起來。

**李明遠**：你怎麼會問起這個人?你從哪裡聽來的這個名字?

**旁白**：父親的語氣變得有些嚴肅,甚至帶著一絲質問的意味。

**旁白**：李杰沒想到父親的反應會這麼強烈。他的心跳加速，努力保持鎮定。

**李杰**：沒什麼...就是...之前在公司整理一些舊資料的時候,偶然看到這個名字。

**旁白**：李杰說得含糊其辭,試圖把話題帶過去。

**李杰**：我看資料上好像提到這個人以前在中國有工作往來,所以隨口問問你有沒有印象。

**李明遠**：渡邊陸?

**旁白**：父親重複著這個名字,稍微皺了皺眉,像是在回想。

**旁白**：但李杰注意到,那個反應來得太快了。幾乎是在他說完名字的瞬間,父親就重複了一遍。

**旁白**：不像是在努力回想,更像是...在掩飾什麼。

**李明遠**：沒聽過。我不認識這個人。

**旁白**：父親的語氣很平淡,然後拿起報紙,重新戴上老花眼鏡。

**旁白**：彷彿這只是一個無關緊要的問題,不值得多想。

**旁白**：但李杰看到,父親握著報紙的手指,在那一瞬間用力了一下。

**旁白**：紙張的邊緣被捏得微微皺起。

**李明遠**：怎麼突然問這個?

**旁白**：父親的語氣依然平淡,眼睛看著報紙,沒有抬頭看李杰。

**旁白**：這個問題聽起來很隨意,像是閒聊時的好奇。

**旁白**：但李杰感覺到,空氣中有一種微妙的緊繃感。

**李杰**：就...就是公司裡整理資料時看到的。我也不太清楚是誰。

**旁白**：李杰努力讓自己的聲音聽起來輕鬆。

**李明遠**：哦。

**旁白**：父親只是簡短地應了一聲,沒有再追問。

**旁白**：他翻了一頁報紙,表情平靜,彷彿已經忘記了這個話題。

**旁白**：但李杰注意到,父親盯著報紙的眼睛,好幾秒都沒有移動。

**旁白**：他根本沒在讀報紙。他在思考。

**旁白**：李杰低頭看著手機,假裝若無其事地滑著螢幕。

**旁白**：但他的心跳有些快。

**旁白**：父親的反應太奇怪了。

**旁白**：如果真的不認識這個人,為什麼要這麼快就否認?為什麼握報紙的手會用力?為什麼假裝在看報紙卻眼神呆滯?

**旁白**：父親在說謊。而且,他不想讓李杰知道他在說謊。

**旁白**：客廳又陷入了沉默。

**旁白**：李明遠看著報紙,李杰看著手機,但兩個人都心不在焉。

**旁白**：空氣中瀰漫著一種微妙的緊繃感。

**旁白**：李杰的腦中只有一個念頭在不斷盤旋——黑西裝男人說的是真的,父親和渡邊陸確實有聯繫。

**旁白**：但父親選擇對他撒謊。為什麼?他在隱瞞什麼?

**旁白**：渡邊陸到底是誰？他和父親之間有什麼秘密？這一切和那班神秘的列車又有什麼關係？

**旁白**：就在這時,大門傳來鑰匙轉動的聲音。

**林婉如**：我回來了!豬肋排很新鮮,老闆特地幫我挑的!

**旁白**：母親的聲音打破了客廳裡凝重的氣氛。

**旁白**：林婉如提著購物袋走進來,臉上帶著笑容。

**林婉如**：小杰,來幫媽媽拿一下東西。

**李杰**：好。

**旁白**：李杰站起身,走向母親。

**旁白**：在轉身的瞬間,他回頭看了父親一眼。

**旁白**：李明遠依然坐在沙發上,表情平靜地看著報紙。

**旁白**：但李杰注意到,報紙的頁面,從剛才到現在都沒有翻過。

---

**旁白**：午飯時間,三人坐在餐桌前。

**旁白**：桌上擺著林婉如做的糖醋小排,還有清炒時蔬、番茄蛋花湯。

**林婉如**：來來來,多吃點!這個糖醋小排是我的拿手菜!

**旁白**：母親熱情地給李杰夾菜。

**林婉如**：小杰,下午有什麼打算?要不要出去走走?

**李杰**：嗯...我約了朋友。

**林婉如**：好啊!多和朋友聚聚!年輕人就該多出去玩!

**旁白**：母親笑著說。

**旁白**：李明遠默默地吃著飯,沒有說話。

**旁白**：整個午餐時間,父親都很安靜。

**旁白**：他只是偶爾應和母親的話,或者點點頭表示同意,但眼神始終沒有看向李杰。

**旁白**：李杰也沒有主動和父親說話。兩人之間，彷彿隔著一道無形的牆。而這道牆上，刻著一個名字——渡邊陸。

---

**旁白**：午飯後，李杰回到房間。他坐在書桌前，看著窗外的天空。腦中不斷回想著剛才和父親的對話。

**旁白**：父親的反應，那種緊張，那種警戒，那種急於知道消息來源的態度...這些都證明了一件事——黑西裝男人沒有說謊。父親確實認識渡邊陸。但父親對他撒了謊。

**旁白**：這讓李杰感到一種難以言喻的失落。父子之間本來就不親近，現在又多了一層秘密和謊言。

**旁白**：李杰嘆了口氣。也許不該問的。也許有些事情，不知道比較好。

**旁白**：他拿出手機，打開微信。張偉在群組裡發了訊息。

**張偉**：『下午三點,徐家匯見!我們去打撞球!』

**王浩**：『收到!』

**陳宇**：『沒問題!』

**旁白**：李杰看著訊息,嘴角勉強勾起一個笑容。

**李杰**：『好,等等見。』

**旁白**：他站起身，換上外出的衣服。走出房間時，父親還坐在客廳的沙發上。李明遠看著電視，但眼神有些渙散，像是在想著什麼心事。

**李杰**：爸,媽,我出去了。

**林婉如**：好!路上小心!晚上回來吃飯嗎?

**李杰**：可能會晚一點。

**林婉如**：那我留飯菜給你!

**李杰**：謝謝媽。

**旁白**：李杰看向父親。李明遠轉過頭,兩人的視線短暫地交會。

**旁白**：那一瞬間,李杰在父親的眼中看到了複雜的情緒——擔憂?警戒?還是...愧疚?

**旁白**：但很快,父親就移開了視線。

**李明遠**：路上小心。

**李杰**：嗯。

**旁白**：李杰轉身離開,關上大門。

**旁白**：走在樓梯間裡,他的腳步聲在狹小的空間裡迴盪。

**旁白**：腦中依然盤旋著那些疑問。

**旁白**：渡邊陸是誰?父親和他到底是什麼關係?這一切和那班列車有什麼聯繫?

**旁白**：但現在不是深究的時候。

**旁白**：李杰搖了搖頭,試圖把這些疑問拋在腦後。

**旁白**：他要去見朋友。要享受假期。要暫時忘記那些複雜的事。

**旁白**：至少...現在先這樣吧。

**旁白**：走出公寓大樓,上海的陽光灑在他身上。

**旁白**：街道上人來人往,充滿了生活的氣息。

**旁白**：李杰深吸一口氣,往地鐵站的方向走去。
