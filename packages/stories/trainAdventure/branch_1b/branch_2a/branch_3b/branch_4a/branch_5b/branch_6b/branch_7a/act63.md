# 第六十三幕：意外的線索

**旁白**：五月三日，晚上七點。李杰和三個朋友約在徐家匯的一家KTV。

**旁白**：包廂裡燈光昏暗，音響播放著節奏感強烈的音樂。牆上的大螢幕顯示著歌詞，茶几上擺著啤酒、小吃和果盤。

**旁白**：張偉正拿著麥克風唱著一首老歌，聲音走音得厲害，但唱得很投入。王浩和陳宇坐在沙發上，一邊喝啤酒一邊起鬨。

**王浩**：偉哥！唱得太好了！再來一首！

**陳宇**：哈哈哈！這高音簡直要破音了！

**張偉**：去去去！你們行你們來！

**旁白**：張偉唱完最後一句，把麥克風扔給王浩。

**張偉**：輪到你了！

**王浩**：好！看我的！

**旁白**：王浩接過麥克風，開始點歌。

**旁白**：李杰坐在角落的沙發上，手裡拿著一罐啤酒，笑著看朋友們鬧騰。

**旁白**：這種氛圍讓他感到放鬆，暫時忘記了繪里花的秘密，忘記了父親的謊言，忘記了那些複雜的事。

**陳宇**：杰哥，你怎麼不唱？

**李杰**：等等再說。你們先唱。

**張偉**：杰哥這是在日本待久了，變得矜持了！

**旁白**：眾人笑了起來。

**旁白**：王浩開始唱歌，選的是一首搖滾老歌，唱得比張偉好一些，至少沒那麼走音。

**旁白**：李杰喝了一口啤酒，靠在沙發上。陳宇坐到他旁邊，也拿起一罐啤酒。

**陳宇**：杰哥，在日本的公司怎麼樣？同事好相處嗎？

**李杰**：還不錯。大家都挺友善的。

**陳宇**：那就好。我之前聽說日本職場文化挺嚴格的，還擔心你不習慣。

**李杰**：一開始確實需要適應，不過現在已經習慣了。

**陳宇**：你們公司技術團隊怎麼樣？有什麼厲害的工程師嗎？

**李杰**：有啊。我們團隊裡有幾個資深工程師，技術都挺強的。

**陳宇**：欸，對了！前幾天刷Twitter，看到一個以前一起工作過的人發了你們公司的照片。

**旁白**：陳宇拿出手機，滑了幾下螢幕。

**陳宇**：你看，這個人。你認識嗎？

**旁白**：陳宇把手機螢幕轉向李杰。

**旁白**：李杰看向螢幕，愣了一下。

**旁白**：照片上是一個熟悉的笑臉——齋藤大輔，和他同期進公司的同事，平時總是笑嘻嘻的，看起來有點傻傻的。

**李杰**：大輔？這是...我同事。

**陳宇**：真的？！那應該就是他！齋藤大輔！

**旁白**：陳宇興奮地說。

**陳宇**：這傢伙超厲害的！當時我們都覺得他是天才！他以前在我們公司待過一陣子。

**李杰**：等等...你說他是天才？

**旁白**：李杰的語氣充滿了懷疑。

**旁白**：在他印象中，大輔雖然技術不錯，但絕對稱不上天才。而且那傢伙平時看起來有點呆呆的，做事也常常出些小差錯。

**李杰**：你確定我們說的是同一個人？

**陳宇**：應該是吧！齋藤大輔，日本人，做資安的對吧？

**李杰**：嗯...是做資安的。

**陳宇**：那就是他！這傢伙幾年前被挖角到中國，在我們公司工作過一段時間。

**旁白**：李杰皺了皺眉。

**李杰**：你們公司？

**陳宇**：對啊，訊聯科技。就是伯父以前工作的那家公司。

**旁白**：李杰的心臟猛地跳了一下。

**旁白**：訊聯科技——父親工作了很多年的公司。他當然知道這家公司，父親直到四年前才離開那裡。

**旁白**：但齋藤大輔...也在那家公司工作過？

**李杰**：大輔...在訊聯科技工作過？

**陳宇**：對啊！他當時在我們那邊當首席安全工程師。雖然只待了幾個月，但我老闆說這個人是真正的天才駭客。

**旁白**：陳宇喝了一口啤酒，繼續說。

**陳宇**：不過那時候我還只是個初級工程師，根本沒機會跟他這種等級的人接觸。我也是後來聽老闆提起，才知道他有多厲害。

**陳宇**：聽說他解決了我們系統裡一個存在好幾年的安全漏洞，只花了兩週。而且還重新設計了整個防火牆架構。

**李杰**：...是嗎。

**旁白**：李杰的語氣有些複雜。

**旁白**：父親在訊聯科技當保安，齋藤大輔是首席安全工程師。兩個人的工作性質完全不同，應該不會有什麼交集才對。

**旁白**：但為什麼...為什麼那個黑西裝男人會說父親和渡邊陸有聯繫？而大輔又恰好在同一家公司工作過？

**李杰**：我只知道他技術不錯，但不知道他是...天才。

**陳宇**：哈哈！也許他在日本公司比較低調吧！反正我老闆對他評價超高的！

**旁白**：陳宇拍了拍李杰的肩膀。

**陳宇**：可惜他後來又回日本了。我老闆還想留他，但他說家裡有事，必須回去。

**陳宇**：說起來也是緣分，沒想到他後來進了你的公司。伯父當時還在那邊上班吧？也許他們見過面呢！

**旁白**：李杰的腦中突然閃過一個念頭。

**李杰**：大輔是什麼時候在訊聯科技的？

**陳宇**：嗯...讓我想想...應該是2023年吧。大概五年前。

**旁白**：李杰的手指緊緊握住啤酒罐。

**旁白**：2023年。那時候父親還在訊聯科技工作，他是2024年才離開的。

**旁白**：齋藤大輔在那段時間也在那家公司...和父親在同一家公司。

**旁白**：雖然父親只是保安，大輔是首席安全工程師，兩人的工作性質完全不同，按理說不會有什麼交集。

**旁白**：但這真的只是巧合嗎？還有渡邊陸...黑西裝男人說父親和渡邊陸有聯繫...

**陳宇**：怎麼了？杰哥你臉色有點不好。

**旁白**：陳宇關心地問道。

**李杰**：...沒事。只是有點驚訝。

**旁白**：李杰努力讓自己的聲音聽起來正常。

**李杰**：沒想到大輔以前在我爸工作的公司待過。真是巧啊。

**陳宇**：對啊！說不定伯父還見過他呢！不過那是個大公司，幾千個員工，伯父在門口當保安，大輔在樓上寫程式，應該也沒什麼機會碰到吧。

**李杰**：...也是。

**旁白**：李杰停頓了一下，像是想起了什麼。

**李杰**：對了，你在訊聯科技的時候，有沒有聽說過一個叫渡邊陸的人？

**旁白**：陳宇愣了一下，皺起眉頭想了想。

**陳宇**：渡邊陸？沒聽過欸。是日本人嗎？

**李杰**：嗯，應該是。

**陳宇**：沒印象。我那時候雖然在公司待了幾年，但就是個初級工程師，職位太低了，接觸的圈子也小。這個人是做什麼的？

**李杰**：不，沒什麼。只是隨口問問。

**旁白**：李杰搖搖頭，努力讓自己看起來不在意。

**旁白**：陳宇看著李杰，似乎想問什麼，但最後只是笑了笑。

**陳宇**：杰哥你今天怎麼怪怪的？是不是在日本工作太累了？

**李杰**：可能吧。最近確實有點累。

**旁白**：陳宇拍了拍李杰的肩膀。

**陳宇**：那趁著回來好好休息！別想太多工作的事！

**旁白**：這時，王浩唱完了歌。

**王浩**：下一個誰來？杰哥，該你了吧！

**張偉**：對啊！杰哥快來一首！

**李杰**：好。

**旁白**：李杰站起身，接過麥克風。

---

**旁白**：李杰點了一首老歌，開始唱。他努力讓自己專注在歌詞上，專注在音樂上。

**旁白**：朋友們在旁邊起鬨，拍手，跟著節奏搖晃。包廂裡充滿了歡笑聲和音樂聲。

**旁白**：但李杰的心思已經不在這裡了。

**旁白**：他想起了公司裡的齋藤大輔——那個總是笑嘻嘻的、看起來有點傻的同事，那個被陳宇稱為「天才駭客」的人，那個曾經在父親工作的公司待過的人。

**旁白**：李杰唱完了歌，把麥克風遞給張偉，然後坐回沙發上，拿起啤酒喝了一大口。

**旁白**：接下來的時間，李杰繼續和朋友們唱歌、聊天、開玩笑。他表現得和平常一樣，笑著、鬧著，沒有讓任何人看出異樣。

**旁白**：但在心裡，他已經把這件事記了下來——訊聯科技、齋藤大輔、2023年、父親的保安工作。

**旁白**：父親只是個保安，大輔是首席安全工程師。兩個人的職位天差地別，應該不會有什麼交集。

**旁白**：但這真的只是巧合嗎？為什麼黑西裝男人會說父親和渡邊陸有聯繫？

---

**旁白**：晚上十一點，聚會結束。四個人走出KTV，站在街道上。

**旁白**：夜晚的上海依然燈火通明，街上還有不少人。

**張偉**：今天真開心！好久沒這麼放鬆了！

**王浩**：對啊！下次杰哥回來，我們再約！

**陳宇**：沒問題！杰哥，你什麼時候回日本？

**李杰**：週日。五月七號的飛機。

**張偉**：這麼快？

**李杰**：嗯。公司還有工作。

**陳宇**：那好吧。那就祝你工作順利！

**王浩**：記得常聯絡啊！別又消失半年！

**李杰**：會的。

**旁白**：四人互相道別，然後各自離開。

**旁白**：李杰搭上地鐵，一個人坐在車廂裡。列車搖晃著前進，窗外的隧道一片漆黑。

**旁白**：李杰看著玻璃上自己的倒影，腦中不斷思考著——齋藤大輔、訊聯科技、父親、渡邊陸。

**旁白**：父親是保安，大輔是首席工程師。一個在門口守衛，一個在樓上寫程式。兩個人的世界根本不會重疊。

**旁白**：但為什麼...為什麼他們都和那個神秘的渡邊陸扯上關係？

**旁白**：這些名字、這些地點，像拼圖的碎片散落在他面前。但他還看不出完整的圖案，還有太多空白，太多未知。

**旁白**：但至少，他現在多了一些線索。回到日本後，也許可以試著從齋藤大輔那裡了解更多。

**旁白**：當然，得小心一點，不能讓對方察覺自己在調查什麼。

**旁白**：李杰深吸一口氣，閉上眼睛。列車繼續前進，帶著他回家。

**旁白**：假期還剩下最後幾天。週日晚上，他就要回到東京，回到那個充滿秘密的城市。
