# 第六十四幕：最後的家庭時光

**旁白**：五月六日，早上九點。

**旁白**：李杰收拾好行李，把行李箱放在客廳角落。明天晚上就要去機場，今天是在上海的最後一天。

**林婉如**：小杰，準備好了嗎？我們出發吧！

**旁白**：母親從臥室走出來，換上了一件淺藍色的連衣裙，臉上化了淡妝，看起來精神奕奕。父親也換上了乾淨的襯衫和休閒褲，難得沒有穿那件灰色舊T恤。

**李杰**：好。

**旁白**：李杰看著父母，心中湧起一種奇妙的感覺。

**旁白**：父母今天都休息，說要陪他好好逛逛上海。這種三個人一起出去玩的機會，李杰已經記不清上次是什麼時候了。

**旁白**：也許是他高中畢業那年？還是更早？

**李明遠**：走吧。今天天氣不錯。

**旁白**：父親難得主動開口，聲音裡帶著一絲輕鬆。

**旁白**：李杰跟著父母走出家門。晨光灑在街道上，空氣中有種清新的味道。

---

**旁白**：他們先去了豫園附近的老街。週末的老街人潮洶湧，到處都是遊客和攤販。

**旁白**：古色古香的建築、掛著紅燈籠的店鋪、賣著各種小吃的攤位，到處都充滿了生活氣息。

**林婉如**：小杰，你看！這家店還在！你小時候最喜歡吃他們家的小籠包！

**旁白**：母親指著一家小店，臉上露出懷念的笑容。

**李杰**：真的嗎？我都不記得了。

**林婉如**：你當然不記得！那時候你才五六歲。每次來豫園，你都吵著要吃小籠包。

**李明遠**：記得。那時候他吃得滿臉都是湯汁。

**旁白**：父親難得地笑了笑，眼神變得柔和。

**林婉如**：對啊！每次都要我給他擦嘴。

**旁白**：李杰聽著父母回憶他小時候的事，心中有種溫暖的感覺。

**旁白**：這種感覺很陌生，卻又很舒服。

**林婉如**：走，我們進去吃點東西！

---

**旁白**：他們在老街上逛了一個多小時，嘗了各種小吃——生煎包、糖葫蘆、臭豆腐。

**旁白**：中午，他們去了一家老字號的本幫菜餐廳。餐廳裡裝潢古典，掛著書法字畫，空氣中飄著菜香。

**林婉如**：小杰，你想吃什麼？儘管點！

**李杰**：紅燒肉吧。還有糖醋小排。

**林婉如**：好！再來個清蒸鱸魚！明遠，你呢？

**李明遠**：我都可以。

**林婉如**：那就這樣。服務員！

**旁白**：菜很快就上來了。紅燒肉色澤紅亮，肥而不膩。糖醋小排酸甜可口。清蒸鱸魚鮮嫩無比。

**林婉如**：小杰，多吃點！在日本應該吃不到這麼正宗的中國菜吧？

**李杰**：嗯，確實。

**旁白**：李杰夾了一塊紅燒肉，放進嘴裡。熟悉的味道在口中散開，讓他想起小時候每逢過年過節，母親都會做這道菜。

**李明遠**：在日本工作還習慣嗎？

**旁白**：父親突然開口問道。

**李杰**：還好。已經習慣了。

**李明遠**：那就好。年輕人在外面打拼不容易，要照顧好自己。

**旁白**：父親的聲音平穩，但李杰聽得出其中的關心。

**李杰**：我會的。

**林婉如**：對了，小杰，你在日本有沒有交女朋友啊？

**旁白**：母親突然問道，眼神中帶著期待。

**李杰**：...還沒有。

**林婉如**：你都二十八了！也該考慮這些事了！

**李杰**：我知道，媽。

**旁白**：李杰有些尷尬地笑了笑。

**旁白**：他沒有告訴母親，其實他心裡一直放不下一個人——原田繪里花。雖然她似乎有些不願告訴他的秘密，但李杰相信，總有一天她會願意說的。

**李明遠**：不用急。緣分到了自然就有了。

**旁白**：父親替兒子解了圍。

**林婉如**：你就是太縱容他了！

**旁白**：母親嗔怪地看了父親一眼，但臉上帶著笑容。

**旁白**：李杰看著父母，心中湧起一種溫暖的感覺。至少今天，他只想好好享受和家人在一起的時光。

---

**旁白**：下午，他們去了世紀公園。公園很大，綠樹成蔭，湖水清澈。

**旁白**：五月的上海天氣正好，不冷不熱。陽光透過樹葉灑下斑駁的光影，微風吹過，帶來草木的清香。

**旁白**：他們沿著湖邊的步道慢慢走著。湖面上有人在划船，岸邊有老人在釣魚，還有孩子在草地上放風箏。

**林婉如**：小杰，你看那個風箏！

**旁白**：母親指著天空中一個彩色的蝴蝶風箏。

**李杰**：嗯，挺漂亮的。

**林婉如**：你小時候也喜歡放風箏。記得嗎？

**李杰**：記得一點。

**林婉如**：那時候你爸爸還帶你去買了一個龍形的風箏。結果第一次放就掛在樹上了。

**旁白**：母親笑著說。

**李明遠**：那次是風太大。

**旁白**：父親難得地辯解了一句。

**林婉如**：哈哈！還狡辯！明明就是你技術不好！

**旁白**：看著父母拌嘴，李杰也忍不住笑了。

**旁白**：這種輕鬆愉快的氛圍，讓他感覺自己好像又回到了童年。那時候他還小，不懂得世界的複雜，只知道跟著父母到處玩，覺得每一天都很快樂。

**旁白**：他們在公園裡找了一張長椅坐下。父親去附近的小店買了三瓶水回來。

**李明遠**：喝點水吧。

**旁白**：父親遞給李杰一瓶水。

**李杰**：謝謝爸。

**旁白**：李杰接過水，擰開瓶蓋喝了一口。

**旁白**：他偷偷看了父親一眼。陽光下，父親的側臉顯得有些蒼老。兩鬢的白髮在光線中格外明顯。

**林婉如**：小杰，來！我們拍張照！

**旁白**：母親拿出手機，笑容滿面。

**林婉如**：好久沒拍全家福了！

**旁白**：母親招手叫來一個路人，請他幫忙拍照。

**林婉如**：來來來！小杰站中間！你爸站這邊！

**旁白**：三個人站在湖邊，背景是碧綠的湖水和藍天。

**路人**：準備好了嗎？笑一個！一、二、三！

**旁白**：快門聲響起。那一瞬間，李杰覺得時間好像靜止了。

**旁白**：陽光、湖水、父母的笑容，這一切都被定格在照片裡。

**林婉如**：讓我看看！哎呀拍得真好！

**旁白**：母親看著手機螢幕，滿意地點點頭。

**林婉如**：小杰，我等等傳給你！

**李杰**：好。

---

**旁白**：傍晚，他們去了一家川菜館吃晚飯。

**旁白**：辣椒的香味和花椒的麻味混合在一起，讓人食慾大開。他們點了水煮魚、麻婆豆腐、宮保雞丁、還有一份清炒時蔬。

**林婉如**：在日本應該吃不到這麼辣的菜吧？

**李杰**：很少。日本料理以清淡為主。

**林婉如**：那你會想念這些味道嗎？

**李杰**：會。

**旁白**：李杰點點頭，夾了一塊水煮魚。魚肉鮮嫩，辣得剛剛好。

**李明遠**：那就趁現在多吃點。

**旁白**：父親也夾了一筷子菜，難得地多吃了兩碗飯。

**旁白**：吃完飯，天色已經暗了下來。街燈亮起，城市開始展現出另一種風貌。

**林婉如**：要不要去外灘走走？

**李杰**：會不會太晚了？

**林婉如**：不會！外灘晚上才漂亮！

**旁白**：於是他們又去了外灘。

**旁白**：黃浦江對岸，東方明珠塔和高樓大廈的燈光閃爍著，倒映在江面上。江邊人潮擁擠，到處都是拍照的遊客。

**旁白**：他們沒有擠進人群，只是在稍微遠一點的地方站著，靜靜地看著江景。

**李明遠**：上海變化很大。

**旁白**：父親突然開口說道。

**李明遠**：你小時候，這裡還沒這麼繁華。

**李杰**：是啊。

**林婉如**：時間過得真快。一轉眼，小杰都長這麼大了，還去日本工作了。

**旁白**：母親的聲音裡帶著一絲感慨。

**林婉如**：媽媽有時候還是會想，要是你留在上海工作就好了。至少我們可以常常見面。

**李杰**：媽...

**林婉如**：我知道我知道！年輕人要有自己的夢想！媽媽不會攔你的！

**旁白**：母親笑著擺擺手。

**林婉如**：只是...你要記得常回來看看我們。

**李杰**：我會的。

**旁白**：李杰感覺喉嚨有些發緊。

**李明遠**：在外面好好照顧自己。工作重要，但身體更重要。

**李杰**：嗯。我知道，爸。

**旁白**：三個人並肩站著，看著對岸的夜景。江風吹過，帶著一絲涼意。

**旁白**：這一刻，李杰只想珍惜和父母在一起的時光。

---

**旁白**：五月七日，下午四點。

**旁白**：李杰拖著行李箱，和父母一起搭計程車前往浦東機場。

**旁白**：下午的陽光溫暖而柔和，街道上車輛不少。一路上，大家都沒怎麼說話。空氣中有種離別前的沉默。

**旁白**：到了機場，李杰辦好登機手續，拿著登機證走向安檢口。

**旁白**：父母站在安檢口外，目送著他。

**林婉如**：小杰，到了日本記得報平安！

**李杰**：我會的，媽。

**林婉如**：還有，要記得按時吃飯！不要總是吃便利商店的東西！

**李杰**：好，我知道。

**林婉如**：還有——

**李明遠**：婉如，讓孩子走吧。要遲到了。

**旁白**：父親輕輕拉了拉母親的袖子。

**林婉如**：好好好。我知道了。

**旁白**：母親的眼眶有些紅了。

**李杰**：媽，爸，我走了。你們也保重身體。

**李明遠**：去吧。路上小心。

**旁白**：父親點點頭，聲音依然平穩，但眼神中有著不捨。

**李杰**：嗯。

**旁白**：李杰深深看了父母一眼，然後轉身走向安檢口。

**旁白**：走了幾步，他忍不住回頭。

**旁白**：父母還站在那裡，母親揮著手，父親雙手插在口袋裡，靜靜地看著他。

**旁白**：李杰揮了揮手,然後快步走進安檢通道。

**旁白**：他不敢再回頭。怕一回頭，眼淚就會掉下來。

---

**旁白**：飛機起飛了。透過舷窗，李杰看著上海越來越小,最後消失在雲層之中。

**旁白**：他靠在座位上,閉上眼睛。

**旁白**：腦中浮現出這幾天的畫面——和母親包餃子、和父親的對話、和朋友們的聚會、還有昨天一整天和父母的家庭時光。

**旁白**：這次回家,他度過了難得的溫馨時光,但心中也多了一些疑問。不過那些事,等回到東京再慢慢想吧。

**旁白**：李杰深吸一口氣,看向窗外。雲層之上,陽光明亮而刺眼。

**旁白**：飛機繼續向東飛行,飛向東京。
