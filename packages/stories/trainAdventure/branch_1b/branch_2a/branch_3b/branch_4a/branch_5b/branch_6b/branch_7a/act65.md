# 第六十五幕：失蹤

**旁白**：五月七日，晚上十點。李杰拖著行李箱走進三鷹的公寓。

**旁白**：房間裡靜悄悄的，空氣中有些悶。他打開窗戶，讓夜風吹進來。

**旁白**：李杰把行李箱放在角落，脫掉外套，癱坐在床上。

**旁白**：回到東京了。回到這個熟悉又陌生的城市。

**旁白**：他拿出手機，看了看訊息。這幾天在中國的時候，他沒有主動聯絡繪里花。他不知道該說什麼，也不知道她是否想聽他說話。

**旁白**：螢幕上有一則未讀訊息，是繪里花發來的。

**旁白**：「李桑，我決定再多休一週假。下週一才回去上班。抱歉沒能早點告訴你。」

**旁白**：訊息是昨天發的。李杰盯著螢幕，猶豫了一下，回覆了一句。

**旁白**：「知道了。好好休息。」

**旁白**：發送。

**旁白**：李杰把手機放在一旁，躺在床上，閉上眼睛。明天開始又要上班了。

---

**旁白**：五月八日，早上八點半。李杰走進公司的辦公室。

**旁白**：黃金週結束，同事們陸續回到工作崗位。辦公室裡充滿了閒聊聲和鍵盤敲擊聲。

**中野海斗**：喔！李桑！歡迎回來！

**旁白**：海斗坐在自己的位子上，笑著揮手。

**李杰**：早安，中野桑。

**旁白**：李杰走到自己的座位，放下包包。他環顧四周，繪里花的座位空著——這在預料之中。齋藤大輔的座位也是空的，不過大輔經常遲到，所以李杰沒有多想。

**旁白**：李杰坐下來，打開電腦。又是新的一週，又要開始處理累積的工作了。

---

**旁白**：上午的時間就在工作中度過。李杰處理著累積了一週的郵件，參加了一個會議，檢查了幾個程式碼。

**旁白**：十二點，午休時間。李杰和海斗一起去公司附近的便利商店買便當。

**中野海斗**：大輔那傢伙還沒來欸。

**旁白**：海斗拿著便當，隨口說道。

**李杰**：他還沒到嗎？

**中野海斗**：對啊。我上週六還跟他去了遊戲廳，他那時候精神還不錯。也沒說要請假什麼的。

**旁白**：海斗聳聳肩。

**中野海斗**：不過大輔經常遲到，也許下午會來吧。要是讓山田部長看到，又要被罵了。

**李杰**：是嗎。

**旁白**：兩人回到辦公室，坐在茶水間吃便當。李杰沒有多想，畢竟大輔經常遲到，這不是什麼新鮮事。

**旁白**：下午一點，李杰回到座位繼續工作。大輔的座位依然空著。

**旁白**：兩點左右，山田部長從會議室走出來，看到大輔的空位，臉色一沉。

**山田部長**：齋藤呢？他今天請假了嗎？

**旁白**：山田走到海斗的座位旁，語氣不悅。

**中野海斗**：呃...我不太清楚，部長。他沒有跟我說要請假。

**山田部長**：沒有請假？那他人呢？

**中野海斗**：我也不知道。我發訊息給他了，但他還沒回。

**山田部長**：打電話！

**旁白**：山田的聲音提高了一些。辦公室裡其他人都停下動作，看向這邊。

**中野海斗**：是！

**旁白**：海斗趕緊拿出手機，撥打大輔的號碼。

**旁白**：鈴聲響了很久，但沒有人接。

**中野海斗**：...沒人接，部長。

**山田部長**：繼續打！

**旁白**：海斗又打了兩次，結果都一樣。沒有人接聽。

**山田部長**：這傢伙到底在搞什麼！

**旁白**：山田怒氣沖沖地走回辦公室。辦公室裡的氣氛變得有些緊張。

**旁白**：李杰看著大輔的空位，心中的不安感越來越強。

**旁白**：不接電話。不回訊息。這不像是喝多了那麼簡單。

---

**旁白**：一天過去了。大輔沒有出現。

**旁白**：五月九日，週二。大輔還是沒來。

**旁白**：五月十日，週三。座位依然空著。

**旁白**：海斗每天都在發訊息、打電話，但沒有任何回應。

**旁白**：辦公室裡開始有人竊竊私語。

**同事A**：齋藤桑怎麼了？連續三天都沒來。

**同事B**：聽說部長超生氣的。

**同事A**：會不會是出了什麼事？

**旁白**：李杰坐在位子上，假裝專心工作，但耳朵聽著同事們的對話。

**旁白**：他想起了陳宇說的話——齋藤大輔曾經在訊聯科技工作過，和父親在同一家公司。那個曾經被稱為「天才駭客」的人，現在突然失蹤了。

**旁白**：這真的只是巧合嗎？

---

**旁白**：五月十二日，週五。

**旁白**：大輔依然沒有出現。辦公室裡的氣氛越來越沉重。

**旁白**：下午三點，山田部長把海斗叫進了辦公室。十分鐘後，海斗走出來，臉色有些難看。

**旁白**：李杰注意到海斗的表情，等山田離開後，走到海斗的座位旁。

**李杰**：中野桑，怎麼了？

**中野海斗**：部長說...如果大輔下週一還不來，就要解雇他了。

**旁白**：海斗的聲音有些低沉。

**中野海斗**：他說大輔連續曠工五天，又不申請假期，這已經嚴重違反公司規定了。

**李杰**：...這樣啊。

**中野海斗**：我真的不知道大輔怎麼了。上週六我們還一起去遊戲廳，他那時候還好好的啊。

**旁白**：海斗抓了抓頭髮，看起來很困擾。

**中野海斗**：我今天去報警了。

**李杰**：報警？

**中野海斗**：對。但警察說，大輔是成年人，而且我不是他的親屬，所以他們不會立案。除非有證據證明他遇到了危險。

**旁白**：海斗苦笑了一下。

**中野海斗**：我也不知道他家人的聯絡方式。他從來沒提過家裡的事。雖然知道他住在六本木，但不知道確切地址。

**李杰**：...你有在其他地方找過他嗎？

**中野海斗**：有。我在社交媒體上發了貼文，問有沒有人看到他。但到現在都沒有回應。

**旁白**：海斗嘆了口氣。

**中野海斗**：李桑，你說...大輔不會是出什麼事了吧？

**旁白**：海斗的眼神中帶著擔憂。

**李杰**：...我也不知道。

**旁白**：李杰不知道該怎麼回答。他不能告訴海斗自己心中的猜測——大輔的失蹤，也許和父親、和渡邊陸、和那些神秘的事情有關。

**旁白**：但他沒有證據，也不知道該從何說起。

**中野海斗**：唉...希望他沒事吧。

**旁白**：海斗轉回去繼續工作，但看得出來他心神不寧。

**旁白**：李杰回到自己的座位，看著大輔空蕩蕩的座位。

**旁白**：那個總是笑嘻嘻的、看起來有點傻的同事，那個被陳宇稱為「天才駭客」的人，那個曾經在父親工作的公司待過的人...就這樣消失了。

**旁白**：李杰的手指在鍵盤上停了下來。

**旁白**：齋藤大輔...你到底去哪裡了？

---

**旁白**：週末過去了。李杰在公寓裡度過了兩天，心情沉重。他試著搜尋關於大輔的消息，但什麼都沒找到。

**旁白**：五月十五日，週一。

**旁白**：李杰走進辦公室。大輔的座位上已經沒有了他的私人物品——電腦、馬克杯、桌上的小玩具，全都被收走了。

**旁白**：顯然，公司已經正式解雇了他。

**中野海斗**：李桑，早。

**旁白**：海斗的聲音聽起來有些疲憊。

**李杰**：早安。

**旁白**：李杰剛坐下，就聽到身後傳來熟悉的聲音。

**原田繪里花**：早安，大家。

**旁白**：李杰轉過頭。繪里花站在門口，穿著淺灰色的套裝,手裡拎著包包,臉上帶著淡淡的笑容。

**旁白**：她回來了。

**中野海斗**：原田桑！歡迎回來！

**原田繪里花**：我回來了。休息了這麼久，真是不好意思。

**旁白**：繪里花走到自己的座位，放下包包。她的目光掃過辦公室，落在了大輔空蕩蕩的座位上。

**原田繪里花**：齋藤桑呢？他還沒來嗎？

**旁白**：她問道，語氣平淡。

**中野海斗**：原田桑...妳不知道嗎？齋藤他...失蹤了。

**原田繪里花**：失蹤？

**旁白**：繪里花的表情變了一下，眉頭微微皺起。

**中野海斗**：對。他從上週一開始就沒來上班,電話也不接,訊息也不回。我們都聯絡不上他。

**旁白**：海斗簡單地解釋了情況。

**原田繪里花**：這樣啊...

**旁白**：繪里花的聲音變得有些低沉。她沒有再多問,只是轉身打開電腦,開始整理郵件。

**旁白**：李杰看著繪里花的背影。她的反應有些奇怪——不像是真的很驚訝,更像是...早就預料到會發生什麼事。

**旁白**：但也許只是他想太多了。

**旁白**：辦公室裡恢復了平靜。大家各自工作著,好像齋藤大輔從來沒有存在過一樣。

**旁白**：李杰盯著電腦螢幕,但腦中一片混亂。

**旁白**：渡邊陸、父親、訊聯科技、齋藤大輔...這些線索像散落的拼圖,但他還是看不出完整的圖案。

**旁白**：而現在,其中一塊拼圖消失了。

**旁白**：李杰深吸一口氣,繼續敲打著鍵盤。

**旁白**：但他知道,這件事還沒有結束。
