# 第六十九幕：猶豫的心

**旁白**：李杰踏出車廂，站在月台上。

**旁白**：他沒有立刻往出口走，而是假裝看手機，實際上在用餘光觀察車廂內的情況。

**旁白**：那個女孩...

**旁白**：她還站在原來的位置，低著頭玩手機。

**旁白**：沒有任何要下車的跡象。

**旁白**：車門關上的提示音響起。列車緩緩啟動，駛離月台，消失在夜色中。

**旁白**：李杰鬆了一口氣。

**李杰**：也許...真的只是我想太多了。

**旁白**：他深吸一口氣，轉身走向驗票閘門。

**旁白**：也許那個女孩只是剛好要去三鷹之後的站。也許她根本沒有注意到他。

**旁白**：也許這一切都只是他的過度敏感。

**旁白**：李杰走出車站，夜晚的涼風吹在臉上，讓他清醒了一些。

**旁白**：三鷹的夜晚安靜而平和。便利商店的燈光依然明亮，遠處偶爾有車輛駛過。

**旁白**：李杰沿著熟悉的路走回公寓。

**旁白**：回到家後，他直接走進浴室，讓熱水沖走一天的疲憊。

**旁白**：水蒸氣瀰漫在狹小的空間裡，溫暖而安心。

**旁白**：洗完澡，李杰倒在床上。

**旁白**：今天的聚會很愉快。見到老朋友，聊聊近況，享受美食...

**旁白**：但那個可疑的中年男人，還有電車上的女孩...

**旁白**：也許都只是巧合。

**旁白**：李杰閉上眼睛，很快就睡著了。

---

**旁白**：接下來的日子，生活恢復了平靜。

**旁白**：上班、寫程式、和繪里花一起吃午餐、下班回家...

**旁白**：日復一日，規律而安穩。

**旁白**：那種被跟蹤的感覺偶爾還會出現，但頻率似乎降低了。

**旁白**：也許真的只是他太敏感了。也許那些人已經不再關注他了。

**旁白**：李杰選擇不去深究。只要生活平靜，只要沒有實際的危險，他就不想打破這份安寧。

**旁白**：時間來到十月。

**旁白**：東京的秋天涼爽宜人，街道兩旁的樹木開始染上金黃和紅色。

---

**旁白**：十月十八日，星期四，上午十點。

**旁白**：李杰坐在辦公桌前，專注地看著螢幕上的代碼。

**旁白**：繪里花的座位是空的。她今天請了病假，說是感冒了有些不舒服。

**旁白**：辦公室裡安靜地只有鍵盤敲擊聲和偶爾的交談聲。

**旁白**：突然，一個身影在李杰旁邊坐下。

**中野海斗**：杰，在忙嗎？

**旁白**：是海斗。他拉了張椅子坐在李杰旁邊，臉上帶著八卦的笑容。

**李杰**：還好。怎麼了？

**中野海斗**：沒什麼，就是想問問...你和繪里花進展得怎麼樣了？

**旁白**：海斗壓低聲音，眼神中充滿好奇。

**李杰**：...還是老樣子。

**中野海斗**：老樣子？你們都在一起快一年了吧！怎麼還是老樣子？

**李杰**：我們沒有在一起。只是...關係比較好的同事。

**旁白**：李杰繼續看著螢幕，試圖迴避這個話題。

**中野海斗**：欸欸欸，別裝了！誰不知道你們兩個形影不離的！

**旁白**：海斗誇張地說。

**中野海斗**：每天一起搭電車上下班，午休一起吃飯，週末還一起出去...這不是情侶是什麼？

**李杰**：...真的沒有。

**中野海斗**：那你有沒有想過跟她告白？

**旁白**：海斗認真地問。

**李杰**：...

**旁白**：李杰停下手上的工作，沉默了幾秒。

**中野海斗**：杰，你不會還在猶豫吧？

**李杰**：我只是...不確定。

**中野海斗**：不確定什麼？不確定她喜不喜歡你？

**李杰**：...不確定我是不是真的喜歡她。

**旁白**：李杰說出了心中的疑問。

**旁白**：他和繪里花的關係確實很好。他們相處得很自然，聊天也很愉快，在一起的時候感覺很舒服。

**旁白**：但這算是愛情嗎？

**旁白**：李杰從來沒有戀愛過。他不知道愛情應該是什麼感覺。

**旁白**：心跳加速？臉紅？想要時時刻刻在一起？

**旁白**：他對繪里花...好像沒有這些感覺。

**旁白**：也許只是友情？也許他們只是很好的朋友？

**中野海斗**：欸，杰，你想太多了啦。

**旁白**：海斗拍了拍他的肩膀。

**中野海斗**：愛情不一定要轟轟烈烈的。有時候就是那種...很自然、很舒服的感覺。

**李杰**：...是嗎？

**中野海斗**：當然！你想想，你喜歡和她在一起嗎？

**李杰**：...喜歡。

**中野海斗**：看到她會覺得開心嗎？

**李杰**：會。

**中野海斗**：那不就得了！這就是喜歡啊！

**旁白**：海斗理所當然地說。

**中野海斗**：而且啊，你也不用擔心被拒絕。

**李杰**：為什麼？

**中野海斗**：因為公司裡大家都以為你們已經在交往了！

**旁白**：海斗笑著說。

**中野海斗**：連山田部長都以為你們是情侶！上次開會他還說「原田同學的男朋友也是我們部門的，真不錯」！

**李杰**：...他這樣說？

**中野海斗**：對啊！所以你根本不用怕被拒絕！她肯定也喜歡你啊！

**旁白**：海斗拍了拍李杰的背。

**中野海斗**：你就是想太多了！如果你再繼續這樣猶豫不決，最後就會像我一樣...

**旁白**：海斗突然嘆了口氣。

**中野海斗**：都三十多了還是單身...

**旁白**：海斗誇張地趴在桌上。

**中野海斗**：你知道嗎，上週我媽又打電話來催婚了。說什麼「隔壁老王的兒子都結婚了」、「你表妹的孩子都會走路了」...

**旁白**：海斗開始抱怨起來。

**中野海斗**：我也想交女朋友啊！但就是沒機會！每天都在加班，哪有時間認識新的人！

**李杰**：...辛苦了。

**中野海斗**：所以你啊，有機會就要好好把握！別像我一樣錯過了才後悔！

**旁白**：海斗認真地看著李杰。

**中野海斗**：繪里花是個很好的女孩。你們也很合適。不要再猶豫了，去告白吧！

**旁白**：李杰沉默了。

**旁白**：海斗說得對。他確實喜歡和繪里花在一起。她的笑容，她的聲音，她的存在...都讓他感到安心和快樂。

**旁白**：也許這就是愛情。

**旁白**：不需要轟轟烈烈，不需要驚天動地，只是想要和她在一起，想要看到她開心。

**旁白**：但他還是有些害怕。

**旁白**：如果告白失敗了怎麼辦？如果她只是把他當朋友怎麼辦？

**旁白**：那樣的話，他們連現在這種關係都維持不下去了。

**李杰**：...如果被拒絕了呢？

**中野海斗**：不會的啦！我跟你說，她絕對喜歡你！

**旁白**：海斗信心滿滿地說。

**中野海斗**：你就相信我一次！去告白吧！

**李杰**：...我再想想。

**中野海斗**：欸！別再想了！你都想了快一年了！

**旁白**：海斗有些無奈。

**中野海斗**：這樣吧，你給自己定個期限。比如...聖誕節怎麼樣？

**李杰**：聖誕節？

**中野海斗**：對啊！聖誕節多浪漫！在那天告白成功率超高的！

**旁白**：海斗興奮地說。

**中野海斗**：你可以帶她去看聖誕燈飾，然後在燈飾前告白！多完美啊！

**李杰**：...

**旁白**：李杰想像了一下那個畫面。

**旁白**：聖誕節的東京，到處都是漂亮的燈飾。他和繪里花站在燈光下，他鼓起勇氣說出那句話...

**旁白**：聽起來...好像不錯？

**李杰**：...好吧。我會考慮的。

**中野海斗**：不是考慮！是一定要做！

**旁白**：海斗伸出小指。

**中野海斗**：來，拉勾！聖誕節之前一定要告白！

**李杰**：...你幾歲了還拉勾。

**中野海斗**：欸！別管這麼多！快點！

**旁白**：李杰無奈地笑了，伸出小指和海斗勾在一起。

**中野海斗**：說定了喔！聖誕節之前！

**李杰**：...我知道了。

**旁白**：海斗滿意地點點頭，拍拍李杰的肩膀後回到自己的座位。

**旁白**：李杰看著螢幕上的代碼，但思緒已經飄遠。

**旁白**：聖誕節...那是兩個多月後的事了。

**旁白**：告白嗎？

**旁白**：他還是不確定。他不知道該怎麼說，不知道該選什麼時機，也不知道她會怎麼回答。

**旁白**：如果被拒絕了，他們連現在這種關係都維持不下去了吧？

**旁白**：但如果不說...他們就會一直這樣下去嗎？

**旁白**：李杰嘆了口氣。

**旁白**：也許到聖誕節的時候，他會有答案。

**旁白**：也許到時候，他會有勇氣。

**旁白**：也許...

**旁白**：窗外的秋日陽光灑進辦公室，溫暖而明亮。

**旁白**：但李杰的心中，依然充滿著猶豫和不安。
