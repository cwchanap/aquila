# 第七十二幕：歸途驚魂

**旁白**：兩人沿著海濱步道,往東京電訊站的方向走去。

**旁白**：夜晚的台場依然熱鬧,購物中心的燈光明亮,遠處傳來音樂聲和人們的歡笑聲。

**旁白**：但離開海邊後,海風就弱了許多,兩人都鬆了口氣。

**原田繪里花**：總算暖和一點了...

**李杰**：嗯。

**旁白**：十幾分鐘後,他們來到了東京電訊站。這是臨海線的車站,連接台場和大崎。兩人走進車站,搭上開往大崎方向的列車。

---

**旁白**：列車駛入地下隧道,窗外只剩下黑暗和偶爾閃過的燈光。臨海線從東京電訊站到大崎這一段都在地下行駛。

**旁白**：天王洲島、品川海濱...一站一站地經過。繪里花靠在座位上,看起來有些累了。

**原田繪里花**：今天走了好多路...腳有點酸...

**李杰**：辛苦了。

**原田繪里花**：不過很開心!

**旁白**：繪里花笑著說。

**旁白**：很快,列車抵達大崎站。這裡是臨海線和山手線的轉乘站。兩人走出車廂,來到山手線的月台。月台上分成兩個方向——外環往新宿、池袋,內環往品川、東京。

**李杰**：妳是往品川方向吧?

**原田繪里花**：對。李桑要往新宿?

**李杰**：嗯。

**旁白**：兩人站在月台中央,看著電子顯示板。品川方向的列車還有兩分鐘,新宿方向的還有五分鐘。

**原田繪里花**：看來我的車先來...

**旁白**：繪里花有些不捨地說。

**李杰**：路上小心。

**原田繪里花**：嗯!李桑也是!

**旁白**：兩人並肩站著,等待列車。月台上的人不多,大部分是剛從台場回來的情侶和朋友。遠處傳來列車進站的聲音,品川方向的列車緩緩駛入月台。

**原田繪里花**：那我先走了!今天真的很開心!

**李杰**：我也是。

**原田繪里花**：下週見!

**旁白**：繪里花揮揮手,轉身走向車廂。車門打開,她走了進去。

**旁白**：列車啟動前,繪里花透過車窗向他揮手,李杰也舉起手回應她的道別。

**旁白**：列車駛離月台,消失在夜色中。李杰站在原地,看著列車離去的方向,心中充滿了溫暖,但也有一絲寂寞。

---

**旁白**：幾分鐘後,往新宿方向的列車進站了。李杰上車,找了個座位坐下。

**旁白**：週末夜晚的山手線人不多,車廂裡很安靜。李杰拿出手機,習慣性地打開那個討論區。

**旁白**：螢幕上是各種討論串,大部分都是無聊的閒聊。

**旁白**：「今天吃了超好吃的拉麵w」

**旁白**：「推薦一下新番動畫」

**旁白**：「為什麼女生都不理我QQ」

**旁白**：李杰隨意滑動著,沒有特別想看什麼。就在這時,一個標題吸引了他的注意——「【速報】相模原發現男屍 疑似謀殺案」。李杰的手指停住了,他點開那個討論串。

**旁白**：「剛看到新聞,警方在相模原的山區發現一具男性屍體,據說已經死亡數月,疑似他殺。」

**旁白**：「警方根據現場發現的物品和初步DNA比對,懷疑死者為齋藤大輔,30歲,日本籍男性,軟體工程師,居住於東京都港區六本木。」

**旁白**：「據報導,齋藤大輔於今年五月上旬失蹤,當時有同事曾報警,但因無法證明有危險而未立案。」

**旁白**：李杰的心臟猛地一跳。齋藤大輔——他的同事，座位就在他旁邊的那個胖子。

**旁白**：那個總是笑嘻嘻、喜歡打遊戲的傢伙。那個五月突然失蹤、再也沒有出現的人。

**旁白**：原來...他被殺了?

**旁白**：李杰的手開始顫抖,他繼續往下滑,想看更多資訊。但接下來的留言都是一些無聊的廢話。

**旁白**：「又是殺人案...最近治安好差」

**旁白**：「相模原那邊本來就很偏僻啊」

**旁白**：「RIP」

**旁白**：「樓主有消息來源嗎?還是在瞎掰?」

**旁白**：「我也看到新聞了,應該是真的」

**旁白**：「齋藤大輔是誰啊?有人認識嗎?」

**旁白**：「半年前的失蹤案件,上次有人在這裡討論過」

**旁白**：「找到了!原來就是那個消失的傢伙!」

**旁白**：「所以確定是他殺?」

**旁白**：「新聞說疑似,還在調查中」

**旁白**：「話說相模原那邊是不是有什麼都市傳說?」

**旁白**：「樓上在說什麼鬼www」

**旁白**：李杰越看越煩躁。這些人只會胡亂猜測,完全沒有有用的資訊。

**旁白**：但齋藤大輔...他真的被殺了?

**旁白**：李杰想起五月的事。大輔突然不來上班,連續曠工好幾天。海斗每天發訊息、打電話,都沒有回應。

**旁白**：當時李杰還覺得奇怪——大輔雖然常遲到,但從沒有這樣消失過。海斗還去報了警,但警察說沒有證據,不受理。

**旁白**：然後大輔就這樣消失了。直到現在...原來他那時候已經死了,而且是被殺的。

**旁白**：李杰的背脊一陣發涼。

**旁白**：列車的廣播響起——下一站,新宿,新宿。李杰收起手機,站起身,腦中一片混亂。

---

**旁白**：新宿站,中央線月台。李杰機械地走向開往高尾方向的月台。

**旁白**：大輔被殺了。那個五月突然失蹤的同事,原來是被殺了。

**旁白**：這個事實在他腦中不斷迴響。

**旁白**：而他自己從五月開始也一直感覺被跟蹤,那個可疑的中年男人,那個在電車上的女孩...會不會他也有危險?

**旁白**：李杰的神經緊繃起來,他開始警覺地觀察周圍。月台上的每一個人,每一個動作,都讓他感到不安。

**旁白**：那個戴帽子的男人是在看他嗎?那個拿著公事包的上班族為什麼一直站在原地?

**旁白**：李杰深吸一口氣,試圖讓自己冷靜下來。不能太敏感,也許只是巧合。

**旁白**：列車進站了。李杰上車,找了個靠近車門的位置站著。他沒有坐下,而是保持警戒,方便隨時下車。

**旁白**：列車啟動,駛離新宿站。車廂裡人不多,大部分是回家的上班族和學生。

**旁白**：李杰的目光掃過每一個乘客,沒有什麼可疑的人。也許真的只是他想太多了。

**旁白**：列車繼續前進,經過中野、高圓寺...李杰稍微放鬆了一些。

**旁白**：就在這時,他注意到了。在車廂的另一端,靠近門口的位置,站著一個熟悉的身影。

**旁白**：李杰的心臟停了一拍——那個女孩,二十多歲,黑色長髮,牛仔褲,黑色夾克。

**旁白**：她低著頭,手裡拿著手機,手指在螢幕上滑動,表情冷淡,沒有任何笑容。就是上次在電車上看到的那個女孩。

**旁白**：李杰的血液凍結了。這不是巧合,絕對不是巧合。她在跟蹤他。

**旁白**：李杰努力保持鎮定,沒有直接看向她,只是假裝看手機,用餘光注意著她的動作。

**旁白**：她一直站在那裡,低頭玩手機,沒有任何異常。但李杰知道,她在監視他。

**旁白**：他應該怎麼辦?直接上前質問?但如果她真的是跟蹤者,那她的目的是什麼?

**旁白**：如果她有危險...齋藤大輔的屍體在腦中浮現。李杰的手心開始冒汗。

**旁白**：也許他應該先下車,去人多的地方?但如果她繼續跟著他呢?

**旁白**：列車繼續前進,經過吉祥寺...李杰的腦中一片混亂,他不知道該怎麼辦。

**旁白**：列車的廣播響起——下一站,三鷹,三鷹。

**旁白**：三鷹到了,這是他該下車的站。但如果她跟著他下車,如果她跟著他回家...

**旁白**：李杰的手握緊了手機,看著窗外,三鷹站的月台已經出現在視線中。列車開始減速。

**旁白**：他要下車嗎?還是繼續坐下去,看她會不會跟著?或者直接上前質問?

**旁白**：李杰的心臟狂跳。列車停靠在月台旁,車門打開的提示音響起。李杰站在車門前,猶豫著,手握緊包包帶。該怎麼辦?
