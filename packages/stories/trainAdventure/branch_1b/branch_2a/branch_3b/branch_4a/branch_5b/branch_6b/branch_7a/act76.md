# 第七十六幕：聖誕告白

**旁白**：時間飛逝，轉眼間已經來到十二月下旬。

**旁白**：街道上掛滿了聖誕裝飾，商店櫥窗裡閃爍著彩燈，空氣中瀰漫著節日的氣氛。

**旁白**：這段時間，辦公室裡的氣氛也漸漸恢復了正常。大輔的死依然是個謎，警察的調查似乎沒有進展，而那些黑西裝男人也沒有再出現。

**旁白**：李杰的生活回到了平靜。上班、下班、偶爾和海斗吃個飯，就像什麼都沒發生過一樣。

**旁白**：但他知道，有些事情已經改變了。

---

**旁白**：十二月二十二日，星期五。

**旁白**：辦公室裡，海斗正在整理桌上的東西，準備下班。

**李杰**：你明天就要出發了？

**旁白**：李杰問道。

**中野海斗**：對，明天早上的飛機。

**旁白**：海斗點點頭，臉上難得露出一絲笑容。

**中野海斗**：和大學同學約好了，去台灣玩七天。三十號回日本，然後休假到一月八號。

**李杰**：玩得開心。

**中野海斗**：謝了。

**旁白**：海斗停下手中的動作，看向李杰。

**中野海斗**：你呢？聖誕節有什麼計劃嗎？

**旁白**：李杰猶豫了一下。

**李杰**：我...打算約繪里花出來。

**中野海斗**：喔？

**旁白**：海斗挑了挑眉，笑容變得有些玩味。

**中野海斗**：終於要告白了？

**旁白**：李杰的臉有些發熱。

**李杰**：還...還不確定...

**中野海斗**：都約出來了還不確定？

**旁白**：海斗笑著拍了拍李杰的肩膀。

**中野海斗**：加油。你們挺配的。

**李杰**：謝謝...

**旁白**：李杰低下頭，心裡既緊張又期待。

---

**旁白**：十二月二十四日，平安夜。

**旁白**：晚上七點，新宿站東口。

**旁白**：李杰站在約定的地點，手心微微冒汗。他今天特地換了一套深灰色的休閒西裝外套，配上白色襯衫和深色牛仔褲。

**旁白**：周圍的人群熙熙攘攘，情侶們手牽手走過，臉上洋溢著幸福的笑容。街道兩旁的商店都掛著聖誕裝飾，燈光閃爍，播放著聖誕歌曲。

**旁白**：李杰看了看手錶——七點零三分。繪里花應該快到了。

**旁白**：他深吸一口氣，試圖讓自己冷靜下來。但心臟還是跳得很快。

**旁白**：這時，人群中出現了一個熟悉的身影。

**旁白**：繪里花穿著一件米色的長大衣，裡面是深紅色的毛衣和黑色長褲。頭髮放了下來，微微捲曲，在燈光下顯得柔順而有光澤。

**旁白**：她看到李杰，臉上露出笑容，快步走了過來。

**原田繪里花**：李桑，抱歉！讓你久等了。

**旁白**：她的臉頰微紅，不知道是因為冷風還是因為快步走過來。

**李杰**：沒有，我也剛到。

**旁白**：李杰笑著回答，但聲音有些緊張。

**原田繪里花**：那...我們走吧？

**李杰**：嗯。

---

**旁白**：兩人走在新宿的街道上，周圍是聖誕節的喧囂和燈光。

**旁白**：李杰訂了一家義式餐廳，位於新宿三丁目的一棟大樓八樓，是一家評價不錯的小餐廳。

**旁白**：電梯裡，兩人都有些沉默。繪里花低頭看著手機，李杰則盯著電梯的數字顯示。

**旁白**：氣氛有些微妙的尷尬。

**旁白**：電梯門打開，八樓是一條安靜的走廊，和樓下的喧囂形成鮮明對比。

**旁白**：餐廳的門口掛著溫暖的燈飾，玻璃窗上貼著聖誕貼紙。

**旁白**：李杰推開門，服務生立刻迎了上來。

**服務生**：晚上好，請問有預約嗎？

**李杰**：有，李先生，七點的預約。

**服務生**：好的，請跟我來。

---

**旁白**：餐廳裡的燈光溫暖而柔和，牆上掛著聖誕花環，每張桌上都放著小小的蠟燭。

**旁白**：背景音樂是輕柔的爵士樂，混雜著偶爾的聖誕歌曲。

**旁白**：服務生帶他們到靠窗的位置，可以看到樓下新宿街道的夜景。

**旁白**：繪里花坐下，脫下大衣，露出裡面深紅色的毛衣。她整理了一下頭髮，看向窗外。

**原田繪里花**：景色真漂亮...

**旁白**：她輕聲說道。

**李杰**：嗯，我特地找了靠窗的位置。

**旁白**：李杰回答，聲音還是有些緊張。

**旁白**：服務生遞上菜單，兩人開始點餐。

**旁白**：繪里花點了蘑菇燉飯和凱薩沙拉，李杰點了番茄肉醬義大利麵和一份烤雞。

**服務生**：飲料呢？

**原田繪里花**：我要一杯熱紅酒。

**李杰**：我也一樣。

**服務生**：好的，請稍等。

---

**旁白**：服務生離開後，兩人又陷入了短暫的沉默。

**旁白**：繪里花看著窗外的街景，李杰則看著桌上的蠟燭，不知道該說什麼。

**旁白**：最後，還是繪里花先開口了。

**原田繪里花**：李桑...

**李杰**：嗯？

**原田繪里花**：謝謝你今天約我出來。

**旁白**：她轉過頭，看著李杰，臉上帶著淺淺的笑容。

**原田繪里花**：我其實...有點意外。我以為你會和海斗一起過聖誕節。

**李杰**：他去台灣旅行了。

**原田繪里花**：是嗎...

**旁白**：繪里花點點頭。

**原田繪里花**：那...我很高興你約我。

**旁白**：她的聲音很輕，但李杰聽得很清楚。

**旁白**：李杰的心跳更快了。他深吸一口氣，決定說出心裡話。

**李杰**：繪里花...其實，我約妳出來，不只是因為海斗不在。

**旁白**：繪里花抬起頭，看著李杰。

**李杰**：我...一直想找個機會和妳說。

**旁白**：李杰的手握緊了桌布。

**李杰**：從第一次見到妳開始，我就覺得妳很特別。妳的笑容、妳說話的方式、妳對工作的認真...

**旁白**：繪里花的臉漸漸紅了起來。

**李杰**：這幾個月相處下來，我越來越確定...我喜歡妳。

**旁白**：李杰說完，緊張地看著繪里花。

**旁白**：繪里花低下頭，手指輕輕握著桌布。她的臉頰已經紅透了。

**旁白**：餐廳裡的音樂還在播放，周圍的客人還在聊天，但對李杰來說，整個世界都安靜了下來。

**旁白**：他在等待繪里花的回答。

**旁白**：幾秒鐘的沉默，感覺像是幾個小時。

**旁白**：然後，繪里花抬起頭。她的眼睛有些濕潤，但臉上帶著笑容。

**原田繪里花**：李桑...我也是。

**旁白**：她的聲音很小，但很堅定。

**原田繪里花**：我也一直...喜歡你。

**旁白**：李杰愣住了。他感覺心臟要跳出胸腔。

**原田繪里花**：從你第一天來公司開始，我就注意到你了。你很認真，也很溫柔。和你說話的時候，我總是很開心。

**旁白**：繪里花笑了，眼角有淚光閃爍。

**原田繪里花**：我一直在等...等你開口。

**旁白**：李杰也笑了，感覺胸口充滿了一種難以言喻的溫暖。

**李杰**：對不起，讓妳等這麼久。

**原田繪里花**：沒關係。

**旁白**：她擦了擦眼角，笑容更加燦爛。

**原田繪里花**：我很高興...你終於說出來了。

---

**旁白**：這時，服務生端著兩杯熱紅酒走了過來。

**服務生**：兩位的熱紅酒。

**旁白**：服務生把酒杯放在桌上，似乎察覺到了什麼特別的氣氛，微笑著退了下去。

**旁白**：李杰拿起酒杯，看著繪里花。

**李杰**：那...我們...算是在交往了嗎？

**旁白**：繪里花也拿起酒杯，臉上的笑容沒有消失。

**原田繪里花**：嗯。

**旁白**：她點點頭。

**原田繪里花**：從今天開始。

**旁白**：兩人舉起酒杯，輕輕碰了一下。

**旁白**：清脆的聲音在溫暖的餐廳裡響起，像是某種承諾。

**旁白**：窗外，新宿的街道燈火通明。聖誕歌曲在空氣中流淌，情侶們在街上漫步。

**旁白**：而在這個小小的餐廳裡，李杰和繪里花開始了他們的故事。

**旁白**：即使他們都不知道，未來還有多少未知的危險在等待著他們。

**旁白**：但至少在這一刻，一切都是美好的。

---

**旁白**：晚餐結束後，兩人在新宿街頭散步了一會兒。

**旁白**：聖誕夜的街道人潮洶湧，情侶們手牽手走過，商店裡播放著歡快的聖誕歌曲。

**旁白**：李杰和繪里花並肩走著，偶爾聊著工作、聊著最近看的電影、聊著未來的計劃。

**旁白**：氣氛很輕鬆，也很自然。兩人之間的距離比以前近了一些，但還沒有到牽手的程度。

**旁白**：晚上十點左右，兩人在新宿站分開。

**原田繪里花**：今天...真的很開心。

**旁白**：繪里花站在閘機前，臉上帶著笑容。

**李杰**：我也是。

**旁白**：李杰回答，聲音裡也帶著笑意。

**原田繪里花**：那...週一見？

**李杰**：嗯，週一見。

**旁白**：繪里花揮了揮手，轉身走進閘機。李杰目送著她的背影消失在人群中，心裡充滿了幸福感。

**旁白**：他深吸一口氣，也轉身走向月台。

**旁白**：今晚的一切，都像是夢一樣美好。
