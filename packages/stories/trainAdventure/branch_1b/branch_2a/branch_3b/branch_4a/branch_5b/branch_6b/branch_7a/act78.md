# 第七十八幕：意外的重逢

**旁白**：下班後，李杰和海斗一起離開公司。

**旁白**：兩人沒有直接回家，而是搭上地鐵，前往六本木。

**旁白**：傍晚的六本木站人潮洶湧，下班的上班族、購物的遊客、年輕情侶混雜在一起。

**旁白**：李杰和海斗穿過人群，走出車站。

**中野海斗**：你確定要去報警嗎？

**旁白**：海斗看向李杰。

**李杰**：嗯。這次我們不能再冒險了。上次被黑西裝襲擊的教訓已經夠了，這個網址可能是重要線索，應該交給警方處理。

**中野海斗**：你說得對。

**旁白**：海斗點點頭。

**旁白**：兩人沿著街道走了大約十分鐘，來到六本木警察署。

**旁白**：這是一棟現代化的建築，外牆是灰白色的，大門口掛著警察署的標誌。

**旁白**：李杰深吸一口氣，推開大門。

---

**旁白**：警察署的大廳很明亮，幾個警察正在櫃台後面處理文件。

**旁白**：李杰走到櫃台前。

**李杰**：不好意思，我想提交一些和謀殺案相關的證據。

**旁白**：櫃台後的年輕警察抬起頭。

**警察**：請問是哪個案件？

**李杰**：齋藤大輔的謀殺案。

**警察**：請稍等，我去通知負責的警官。

**旁白**：他轉身走進後方的辦公室。

**旁白**：李杰和海斗在大廳的長椅上坐下等待。

**旁白**：大廳裡有幾個人也在等待——一對中年夫婦、一個看起來很焦急的年輕女性、還有一個拿著公事包的西裝男。

**旁白**：李杰看了看手錶——晚上七點十五分。

**中野海斗**：不知道要等多久...

**旁白**：海斗小聲說道。

**李杰**：應該不會太久吧。

**旁白**：大約十分鐘後，一個熟悉的身影從辦公室走了出來。

**旁白**：李杰看到那個人的臉，愣住了。

**旁白**：那是一個四十多歲的中年男子，穿著深藍色的西裝，頭髮整齊地梳向後方。

**旁白**：臉型方正，眼神銳利而沉穩——正是宇佐美誠一郎。

**宇佐美誠一郎**：李先生？沒想到會在這裡再次見面。

**旁白**：宇佐美也露出驚訝的表情，然後很快恢復平靜，走了過來，伸出手。

**李杰**：宇佐美警部...我也沒想到。

**旁白**：李杰站起身，和宇佐美握手。

**中野海斗**：宇佐美警部，好久不見。

**宇佐美誠一郎**：中野先生，好久不見。來我的辦公室談吧。

---

**旁白**：宇佐美帶著李杰和海斗走進辦公區。

**旁白**：這裡有幾排辦公桌，幾個警察正在處理文件，敲打鍵盤的聲音此起彼落。

**旁白**：宇佐美的辦公室在最裡面，是一個小隔間，有一張辦公桌、幾把椅子和一個檔案櫃。

**旁白**：牆上掛著一幅東京地圖，桌上堆著一些文件和報告。

**宇佐美誠一郎**：請坐。你們說要提交和齋藤大輔案相關的證據？

**旁白**：宇佐美示意兩人坐下，自己則坐在辦公桌後，語氣恢復了專業。

**李杰**：是的。其實...我是昨天才發現的。這是大輔在三個月前寄給我的郵件，但因為沒有主旨，我一直沒注意到。

**旁白**：李杰從包包裡拿出手機，打開那封郵件，把手機遞給宇佐美。

**旁白**：宇佐美接過手機，仔細看著螢幕。

**旁白**：他的眼神越來越專注，眉頭也皺了起來。

**宇佐美誠一郎**：八月十五日...那是他失蹤三個月後。郵件裡有個網址？

**旁白**：宇佐美抬起頭看向李杰。

**李杰**：是的，在附件裡。一個文字檔，裡面只有一個網址。

**旁白**：李杰打開附件，讓宇佐美看。

**宇佐美誠一郎**：https://7f3a9b2c4d5e.xyz...你們試過訪問嗎？

**旁白**：宇佐美念出那個網址，聲音低沉。

**中野海斗**：試過了，但無法連線。顯示連線被拒絕。

**宇佐美誠一郎**：連線被拒絕...這個網址看起來像是某種哈希值，可能需要特殊的方式才能訪問。

**旁白**：他從抽屜裡拿出筆記本，開始記錄。

**宇佐美誠一郎**：李先生，可以請你把這封郵件轉發給我嗎？包括附件。

**李杰**：當然可以。

**旁白**：李杰操作手機，把郵件轉發給宇佐美。

**旁白**：宇佐美的手機震動了一下，他確認收到後點點頭。

**宇佐美誠一郎**：謝謝。這可能是個重要的線索。

**旁白**：他抬起頭，看著李杰和海斗。

**宇佐美誠一郎**：說起來也很巧...我昨天才被分配到齋藤大輔的案件。

**李杰**：昨天？

**旁白**：李杰有些驚訝。

**宇佐美誠一郎**：對。這個案件原本由相模原警察署負責，但因為某些原因，上級決定調給我們署。

**宇佐美誠一郎**：昨天下午，檔案才剛送到我手上，你們今天就來了，真的是很完美的時機。

**旁白**：宇佐美的語氣帶著一絲感慨。

**李杰**：那...調查有什麼進展嗎？

**宇佐美誠一郎**：坦白說，不太樂觀。

**旁白**：宇佐美嘆了口氣。

**宇佐美誠一郎**：屍體被發現時已經嚴重腐爛，很多證據都已經消失了。

**宇佐美誠一郎**：而且案發地點在山區，沒有監視器，也沒有目擊證人。

**宇佐美誠一郎**：目前我們能做的，就是追查他失蹤前的行蹤，還有他的人際關係。

**中野海斗**：那...他弟弟的案子呢？齋藤蓮的失蹤案。

**宇佐美誠一郎**：那個案子...沒有任何進展。齋藤蓮在九月二十三日失蹤後，就像人間蒸發一樣。沒有任何線索，沒有任何目擊，沒有任何痕跡。

**旁白**：宇佐美的表情變得更加沉重，聲音帶著一絲無奈。

**宇佐美誠一郎**：我們問過他的家人和朋友，也調查過他失蹤前的行蹤，但還是找不到任何有用的資訊。

**旁白**：李杰和海斗對視了一眼。

**旁白**：齋藤蓮...那個在列車上的眼鏡男。

**旁白**：李杰心裡清楚他去了哪裡，但這個真相無法告訴警方。

**旁白**：宇佐美把筆記本放在桌上。

**宇佐美誠一郎**：不過，你們今天提供的這個網址，可能會是個突破口。

**旁白**：宇佐美看著李杰和海斗，眼神認真。

**宇佐美誠一郎**：我會請技術部門的同事試著訪問這個網站，看看能不能找到什麼。

**宇佐美誠一郎**：如果有任何發現，我會通知你們。

**李杰**：謝謝。

**宇佐美誠一郎**：該謝的是我。謝謝你們願意提供這個線索，這對調查很有幫助。如果還有什麼發現，隨時聯絡我。

**旁白**：宇佐美站起身，伸出手。李杰和海斗也站起來，和宇佐美握手道別。

**李杰**：好的。

**旁白**：三人走出辦公室，宇佐美送兩人到大廳門口。

**宇佐美誠一郎**：路上小心。

**李杰**：您也是。

**旁白**：李杰和海斗走出警察署。

---

**旁白**：夜幕已經完全降臨，六本木的街道燈火通明。

**旁白**：李杰和海斗走在人行道上，沉默了一會兒。

**中野海斗**：沒想到會是宇佐美警部負責這個案子。不過，把線索交給他，我覺得挺放心的。上次見面，感覺他是個認真負責的人。

**旁白**：海斗先開口，看向李杰。

**李杰**：嗯，我也這麼覺得。

**旁白**：兩人走了一段路，海斗突然停下腳步，轉身看向李杰，臉上露出興奮的笑容，眼睛發亮。

**中野海斗**：對了，杰！既然都來六本木了，要不要去秋葉原？我們可以去那邊的女僕咖啡廳吃晚餐！

**中野海斗**：我很喜歡這種地方，但最近都一個人去有點無聊。正好你在，我們一起去吧！

**李杰**：女僕咖啡廳...

**中野海斗**：拜託啦！就當作慶祝我們順利把線索交給警方！

**旁白**：海斗雙手合十，擺出懇求的姿勢。李杰看著海斗期待的表情，忍不住笑了出來。

**李杰**：好吧，那就去吧。

**中野海斗**：太好了！

**旁白**：海斗高興地揮了揮拳頭。

**旁白**：兩人走向地鐵站，搭上前往秋葉原的電車。

---

**旁白**：秋葉原的夜晚充滿了活力。

**旁白**：巨大的電子看板閃爍著五顏六色的廣告，遊戲中心的音樂從門口傳出，動漫周邊商店的櫥窗裡擺滿了各種手辦。

**旁白**：海斗帶著李杰走進一條小巷，來到一家女僕咖啡廳門口。

**旁白**：門口站著兩個穿著粉紅色女僕裝的女孩，正在向路人招攬客人。

**女僕A**：歡迎光臨，主人！兩位主人要進來坐坐嗎？

**旁白**：看到李杰和海斗走近，其中一個女孩立刻笑容滿面地迎上來。

**中野海斗**：好啊好啊！

**旁白**：海斗興奮地點頭。

**旁白**：李杰跟在後面，感覺有些尷尬。

**旁白**：兩人走進咖啡廳，裡面的裝潢是粉紅色和白色為主調，牆上掛著可愛的裝飾，桌子上擺著蕾絲桌布。

**旁白**：幾個女僕在店裡忙碌著，招呼客人、端菜、唱歌。

**旁白**：店裡大約有十幾個客人，大多是年輕男性，也有幾對情侶。

**女僕B**：主人，這邊請!

**旁白**：另一個女僕帶著李杰和海斗到一個靠窗的位置。

**旁白**：兩人坐下，女僕遞上菜單。

**女僕B**：主人，這是我們的菜單!有什麼需要隨時叫我喔!

**旁白**：她甜甜地笑著，然後走開了。

**中野海斗**：哇，這裡真的很有氣氛耶！

**旁白**：海斗東看西看，臉上充滿興奮。

**李杰**：你還真的很喜歡這種地方...

**旁白**：李杰無奈地笑著，翻開菜單。

**中野海斗**：偶爾來體驗一下也不錯嘛！嗯...我要這個『愛心蛋包飯』！還有這個『萌萌熊掌雞塊』！

**旁白**：海斗認真地看著菜單。

**李杰**：我點個普通的咖哩飯就好。

**旁白**：李杰簡單地選了一個看起來最正常的餐點。

**旁白**：海斗招手叫來女僕，點了餐。

**旁白**：等待上菜的時候，海斗靠在椅背上，看著店裡的裝飾，突然問道。

**中野海斗**：杰，你對那個網址...還會好奇嗎？

**李杰**：說實話...會。

**旁白**：李杰停頓了一下，看著桌上的蕾絲桌布。

**李杰**：我很想知道那個網站裡到底有什麼，為什麼大輔要特地寄給我。

**旁白**：他抬起頭看向海斗。

**李杰**：但我覺得，交給警方處理是最好的選擇。

**中野海斗**：我也這麼認為。上次被黑西裝襲擊的事，真的太危險了，我不想再經歷一次。

**李杰**：而且...我現在有了繪里花。

**旁白**：李杰的聲音變得柔和。

**李杰**：我不想讓她擔心，也不想因為這些事而失去她。

**中野海斗**：說得對！你現在有女朋友了，當然要好好珍惜！

**旁白**：海斗笑著拍了拍李杰的肩膀。

**中野海斗**：不要再去冒險了！

**李杰**：嗯。所以，這次就讓警方去調查吧。

**旁白**：李杰點點頭。

**李杰**：我們只要過好自己的生活就好。

**中野海斗**：沒錯！為了平靜的生活！

**旁白**：海斗舉起桌上的水杯。

**李杰**：為了平靜的生活。

**旁白**：李杰也舉起水杯，和海斗輕輕碰杯。清脆的聲音在咖啡廳裡響起。

**旁白**：這時，女僕端著餐點走了過來。

**女僕B**：主人，您的餐點來囉！請慢用喔！♡

**旁白**：她把愛心蛋包飯和雞塊放在海斗面前，把咖哩飯放在李杰面前，擺出一個可愛的姿勢，然後走開了。

**中野海斗**：哇！看起來好可愛！我開動了！

**旁白**：海斗拿起手機，拍了張照片。

**旁白**：兩人開始吃起晚餐。

**旁白**：店裡的音樂輕快而歡樂，女僕們的笑聲和客人的談話聲混雜在一起。

**旁白**：李杰一邊吃著咖哩飯，一邊看著窗外秋葉原的夜景。

**旁白**：那個神秘的網址，那些詭異的事件，那些危險的黑西裝...

**旁白**：他決定把這些都忘掉。

**旁白**：讓警方去處理吧。他只想好好享受現在的生活——和繪里花在一起，和海斗做朋友，平靜而幸福地度過每一天。

**旁白**：至少，他是這麼希望的。

**旁白**：但他不知道，命運從來不會那麼輕易地放過他。

**旁白**：在看不見的地方，新的危機正在悄然逼近。
