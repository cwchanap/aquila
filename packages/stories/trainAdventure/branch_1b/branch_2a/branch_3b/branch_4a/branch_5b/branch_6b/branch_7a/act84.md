# 第八十四幕：逆轉

**旁白**：李杰的腦中飛快地運轉。他必須做點什麼。但他能做什麼？雙手被綁著，沒有工具，沒有武器。中村就坐在旁邊，隨時盯著他。

**旁白**：牆上的投影顯示，火焰已經蔓延到一半。距離繪里花和海斗越來越近。

**李杰**：長谷川！

**旁白**：李杰壓低聲音，朝隔壁房間喊道。

**李杰**：妳有辦法嗎？

**旁白**：隔壁傳來一陣窸窣聲，然後是澪的聲音。

**長谷川澪**：沒有。

**旁白**：澪的聲音依然平靜，但李杰能聽出其中的無奈。

**中村隼人**：別白費力氣了，李杰桑。沒有人能救你們。

**旁白**：中村笑著說，視線沒有離開投影。

**旁白**：時間一秒一秒地過去。李杰看著牆上的投影，心臟狂跳。四分鐘。三分鐘。兩分鐘。

**旁白**：火焰繼續蔓延。繪里花和海斗的掙扎越來越激烈，但繩子依然紋絲不動。李杰咬緊牙關。他不能就這樣看著他們死去。但他能做什麼？

**旁白**：一分鐘。

**旁白**：就在這時，投影畫面突然劇烈晃動。一個黑西裝男人闖進了房間——但不是中村的手下。那個男人動作迅速，衝向繪里花和海斗，開始解開綁著他們的繩子。

**李杰**：！

**旁白**：李杰的眼睛瞪大了。

**中村隼人**：什麼？

**旁白**：中村猛地站起來，臉上的笑容消失了，取而代之的是驚訝。

**中村隼人**：沒想到...他們動作這麼快...

**旁白**：中村盯著牆上的投影，然後突然笑了起來。那笑容中帶著一種奇怪的輕鬆感。

**中村隼人**：不過沒關係。我可以重新來過。

**李杰**：什麼？

**旁白**：李杰完全不明白中村在說什麼。重新來過？什麼意思？

**旁白**：就在這時——

**旁白**：一道黑影從隔壁房間破門而出。是澪。

**旁白**：她的動作快得不可思議。還沒等中村反應過來，她已經衝到他面前，一個乾淨俐落的動作將他摔倒在地。

**中村隼人**：嗚！

**旁白**：中村重重地摔在地上，後腦撞擊地板發出悶響。

**李杰**：長谷川？

**旁白**：李杰完全愣住了。這是怎麼回事？澪不是也被綁著嗎？

**旁白**：澪沒有回答。她迅速從中村的西裝口袋裡掏出一把手槍，然後轉身走向李杰。她用手槍的槍托敲碎綁著李杰的繩子。

**長谷川澪**：繪里花和海斗應該沒事。把他綁好。我要去清理外面的人。軍隊應該很快就會到。

**旁白**：澪的聲音依然平靜。

**李杰**：等等，這到底是怎麼回事——

**長谷川澪**：沒時間解釋。綁好他。等軍隊來。

**旁白**：澪打斷了他，把手槍塞進腰間，然後轉身走向門口。澪推開門，走出房間。

**旁白**：李杰聽到走廊上傳來兩聲悶響——那是身體撞擊牆壁的聲音。然後是兩聲重物倒地的聲音。門外的兩個黑西裝男人，被她瞬間擊倒了。

**旁白**：李杰呆呆地站在原地，看著倒在地上的中村。中村還有意識，但顯然被摔得不輕，掙扎著想爬起來。

**李杰**：...

**旁白**：李杰回過神來，走向牆角。那裡還有剛才綁著他和澪的繩子。

**旁白**：他拿起繩子，走向中村，開始綁他的雙手。

**中村隼人**：你...你以為這樣就結束了嗎...

**旁白**：中村的聲音虛弱，但依然帶著一種詭異的自信。

**中村隼人**：我還會再來的...一次又一次...直到成功為止...你什麼都不知道...

**李杰**：你在說什麼？

**旁白**：李杰皺起眉頭，但中村只是笑了笑，不再說話。

**旁白**：李杰用力拉緊繩子，確保中村動不了。

**旁白**：他看向牆上的投影。畫面中，那個黑西裝男人已經解開了繩子，正帶著繪里花和海斗離開房間。火焰在他們身後燃燒，但他們安全了。

**旁白**：李杰鬆了一口氣。但同時，他的腦中充滿了疑問。

**旁白**：澪到底是誰？她是怎麼掙脫繩子的？那個救人的黑西裝男人是誰？還有...軍隊？為什麼會有軍隊來？

**旁白**：還有中村說的那句話——「我可以重新來過」。那是什麼意思？

**旁白**：走廊上傳來更多的聲音——腳步聲、喊叫聲、物體碰撞的聲音。戰鬥還在繼續。

**旁白**：李杰坐在椅子上，看著被綁在地上的中村，等待著。等待著這一切的答案。
