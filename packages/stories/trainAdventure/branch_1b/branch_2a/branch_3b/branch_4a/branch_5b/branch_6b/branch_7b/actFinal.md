# 終幕：懦夫的結局

**旁白**：李杰閉上眼睛，深吸一口氣。

**旁白**：當他再次睜開眼時，眼神變得堅定起來。

**李杰**：我不會說的。

**旁白**：他的聲音顫抖著，但語氣出奇地堅定。

**李杰**：你們如果殺了他，警察不會放過你們的。

**旁白**：黑西裝男A愣了一秒，然後發出一聲大笑。

**旁白**：那笑聲在金屬房間裡迴盪，聽起來格外刺耳和詭異。

**黑西裝男A**：警察？

**旁白**：他笑得眼淚都快出來了，彷彿聽到了這輩子最好笑的笑話。

**黑西裝男A**：你太天真了，李杰。

**旁白**：他擦了擦眼角，笑容逐漸消失，眼神變得冰冷。

**黑西裝男A**：你以為警察有能力逮捕我們嗎？

**清水浩介**：不...不要...

**旁白**：浩介的聲音已經沙啞得幾乎聽不見，他趴在地上，身體劇烈顫抖。

**旁白**：黑西裝男A用槍口頂著浩介的太陽穴，食指扣在扳機上。

**旁白**：砰——！

**旁白**：一聲震耳欲聾的槍響在密閉的金屬房間裡炸開！

**李杰**：不——！

**旁白**：李杰的尖叫被槍聲淹沒。

**旁白**：浩介的身體猛地一震，然後軟軟地倒在地上。

**旁白**：鮮血從他的頭部湧出，在灰色的金屬地板上迅速擴散，形成一灘暗紅色的血泊。

**旁白**：他的眼睛睜得很大，但已經失去了焦距，嘴巴微微張開，像是還想說什麼。

**旁白**：但什麼都說不出來了。

**旁白**：永遠都說不出來了。

**李杰**：你...你們...

**旁白**：李杰的聲音卡在喉嚨裡，整個人像被雷劈中一樣僵在椅子上。

**旁白**：他不敢相信眼前的景象——浩介就這樣死了，就死在他面前。

**旁白**：毫無預警，毫不猶豫，就像捏死一隻蚊子一樣簡單。

**李杰**：你們...你們瘋了...

**旁白**：他的聲音在顫抖，恐懼像潮水一樣淹沒了他。

**黑西裝男A**：瘋了？

**旁白**：那個男人冷笑著，擦了擦槍口上的血跡。

**旁白**：然後他站起身，慢慢走向李杰，槍口對準了他的頭。

**旁白**：冰冷的金屬觸感抵在李杰的額頭上，那種感覺讓他全身的血液都凝固了。

**黑西裝男A**：現在，換你了。

**李杰**：等...等等...

**旁白**：李杰的聲音完全變了調，恐懼讓他幾乎無法呼吸。

**李杰**：我說...我說...

**旁白**：他崩潰了，眼淚不受控制地流下來。

**李杰**：我什麼都說...拜託...不要殺我...

**黑西裝男A**：那就說吧。

**旁白**：槍口依然抵在李杰的額頭上，冰冷而致命。

**李杰**：那天晚上...我搭上了末班車...

**旁白**：他的聲音顫抖著，語無倫次，但拼命地想把所有事情都說出來。

**李杰**：列車...列車開進隧道後...就到了另一個世界...

**李杰**：我在第一個車站下車...那裡一側是黑色的海，一側是森林...

**旁白**：他喘著氣，眼淚模糊了視線。

**李杰**：然後我沿著鐵軌一直走...走了很久...

**李杰**：經過了一個叫亡者峽谷的車站...然後是永恆森林...

**旁白**：他的聲音越來越急促，像是害怕說慢了就會被殺死。

**李杰**：在永恆森林站...我看到了中村隼人...

**李杰**：他死了...在列車車廂裡...胸口被劍刺穿...

**旁白**：黑西裝男A的眼神閃過一絲波動，但很快又恢復冷漠。

**李杰**：後來我遇到了精靈...他們帶我去了神樹...

**李杰**：神樹...神樹讓我回到了過去...回到一週前...

**旁白**：他哭得幾乎說不下去，整張臉都扭曲了。

**李杰**：我說了...我全都說了...拜託...放過我...

**旁白**：黑西裝男A沉默了幾秒鐘，眼神在李杰臉上掃視著。

**旁白**：然後他點了點頭，嘴角勾起一個冰冷的弧度。

**黑西裝男A**：很好。

**旁白**：他放下槍，後退了一步。

**黑西裝男A**：謝謝你的情報。

**旁白**：李杰鬆了一口氣，整個人癱軟在椅子上，以為自己得救了。

**黑西裝男A**：不過...

**旁白**：那個男人再次舉起槍，這次槍口對準了李杰的心臟。

**黑西裝男A**：現在你完全沒用了。

**李杰**：什——

**旁白**：砰——！

**旁白**：第二聲槍響。

**旁白**：李杰感到胸口傳來劇烈的衝擊，像是被一輛卡車撞上。

**旁白**：然後是火辣辣的疼痛，從胸口擴散到全身。

**旁白**：他低頭看去，看到胸口的衣服迅速被鮮血浸透。

**旁白**：暗紅色的血液不斷湧出，滴落在他的腿上，滴落在地板上。

**李杰**：啊...啊...

**旁白**：他想說什麼，但嘴裡只能發出含糊的聲音。

**旁白**：血液湧進喉嚨，讓他無法呼吸。

**旁白**：視野開始模糊，天花板上的燈光變得越來越暗。

**旁白**：耳邊的聲音也變得遙遠而空洞，像是隔著一層厚厚的水。

**旁白**：他能聽見黑西裝男們在說話，但已經聽不清內容。

**旁白**：冰冷的感覺從四肢蔓延，身體越來越沉重。

**旁白**：他想起了很多事情——

**旁白**：在異世界的經歷、神樹給的第二次機會、還有那個選擇逃避的一週。

**旁白**：他想起了恵美、大叔、大學生們、小林一家...

**旁白**：那些他選擇遺忘的人。

**旁白**：他想起了浩介的哭喊、那雙絕望的眼睛、還有倒在血泊中的屍體。

**旁白**：如果...如果當初他沒有選擇逃避...

**旁白**：如果他去調查真相、去阻止那班列車...

**旁白**：會不會有不同的結局？

**旁白**：但這些念頭很快就變得模糊不清。

**旁白**：意識像退潮的海水，一點一點離他遠去。

**旁白**：最後，一切陷入黑暗。

**旁白**：李杰死了。

**旁白**：他不知道接下來會發生什麼。

**旁白**：他也確實無法知道。

**旁白**：因為他已經死了。

**旁白**：這就是一個懦夫的結局。

**旁白**：他選擇了逃避，選擇了遺忘，選擇了把一切當作沒發生過。

**旁白**：而當危險真正降臨時，他的懦弱沒有能保護任何人。

**旁白**：也沒有保護他自己。

**旁白**：這個世界繼續運轉著。
