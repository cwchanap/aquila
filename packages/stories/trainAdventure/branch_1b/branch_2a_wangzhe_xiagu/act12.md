# 第十二幕：人去樓空

**旁白**：李杰在漆黑的森林裡狂奔。腿部的肌肉在燃燒,肺部像是要炸開,每一次呼吸都帶來刺痛。

**旁白**：但他不能停下。絕對不能停下。

**旁白**：腦海中不斷浮現那個劍傷——乾淨俐落,一擊致命。那個神秘的第三方,就在這片森林裡。而小林智子和孩子們,正待在那個人的家裡。

**李杰**：(內心,焦急)快⋯⋯必須快一點⋯⋯

**旁白**：他的視線開始模糊,雙腿幾乎失去知覺。體力早已透支,全憑一股意志力在支撐。

**旁白**：十分鐘。二十分鐘。三十分鐘。時間彷彿停滯,森林彷彿永無盡頭。

**旁白**：終於,在將近一個小時的狂奔後,那棟林中小屋再次出現在他的視野中。

**李杰**：(喘著氣)到了⋯⋯

**旁白**：他衝到門口,用力推門。

**旁白**：但門紋絲不動。

**李杰**：(大喊)小林女士!開門!是我!李杰!

**旁白**：裡面沒有任何聲音。

**李杰**：(再次用力推門,內心)怎麼回事?為什麼打不開?

**旁白**：他再次大喊。

**李杰**：小林女士!智子小姐!快開門!

**旁白**：依然是死一般的寂靜。

**旁白**：李杰強迫自己冷靜下來,開始仔細觀察屋內的每一個細節。

**旁白**：首先是正門。一根粗壯的木條橫在門後,牢牢地頂在門板和牆壁之間。這就是為什麼他推不開門——它被從內部閂上了。

**李杰**：(走近門,檢查木條)從裡面堵住的⋯⋯

**旁白**：然後是那張單人床。毯子有明顯被躺過的痕跡,凹陷處像是兩個小小的身軀蜷縮在一起。但毯子被凌亂地掀到一旁,像是有人匆忙起身。

**旁白**：李杰走到廚房。桌子上放著一塊麵包,咬了幾口,還有半杯水。旁邊的櫃子門開著——是他之前拿補給時打開的。

**旁白**：一切都顯示,這裡不久前還有人。他們吃了東西,孩子們睡過覺。但現在,他們消失了。

**李杰**：(站在屋子中央,環顧四周)到底⋯⋯發生了什麼?

**旁白**：他的腦中開始拼湊線索。一個巨大的、無法解釋的矛盾浮現出來。

**李杰**：(內心,思考)門從裡面閂上,這很明顯——他們在躲避來自屋外的危險。應該是聽從了我的建議,鎖好門,不給任何人開。

**旁白**：他走到窗戶前,看著那扇被他推開的窗戶。

**李杰**：(內心)但是⋯⋯他們人不見了。而門還是從裡面堵著的。唯一的出口,就只有這扇沒上鎖的窗戶。

**旁白**：他試圖理解這個矛盾。

**李杰**：(內心)如果是他們自己主動離開的⋯⋯為什麼要從窗戶爬出去?小林女士帶著兩個孩子,從窗戶離開既危險又費力。為什麼不直接開門?

**旁白**：但另一個可能性更讓他不安。

**李杰**：(內心)如果是被那個殺害小林先生的人闖入,強行帶走的⋯⋯

**旁白**：他的目光再次掃過屋內。桌椅擺放整齊,沒有翻倒。地板上沒有拖行的痕跡。牆上沒有撞擊的痕跡。

**李杰**：(內心)⋯⋯為什麼屋內沒有任何掙扎或打鬥的痕跡?

**旁白**：他走到床邊,看著那凌亂的毯子。以小林智子保護孩子的決心,如果有人闖入,她絕對會拼命反抗。但這裡什麼都沒有。

**李杰**：(內心)太安靜了。太⋯⋯平和了。就好像他們只是起身,然後⋯⋯

**旁白**：然後什麼?從窗戶離開?為什麼?去哪裡?

**李杰**：(低聲,困惑)這完全說不通⋯⋯

**旁白**：兩種可能性都存在著巨大的矛盾。屋內過於平靜的狀態,與他們離奇的消失方式格格不入。

**旁白**：他們就像是⋯⋯憑空蒸發了一樣。

**旁白**：李杰站在空蕩蕩的屋子裡,感到了前所未有的困惑。四周的寂靜壓迫著他,讓他幾乎無法思考。

**李杰**：(深吸一口氣)冷靜⋯⋯必須冷靜⋯⋯

**旁白**：他強迫自己混亂的思緒平靜下來。在原地猜測沒有任何意義。

**李杰**：(內心)唯一的辦法,就是出去尋找線索。看看他們有沒有留下任何蹤跡。

**旁白**：他回到廚房,將桌上剩下的半杯水倒進自己的水袋。然後檢查了一下背包裡的補給——麵包還有一些,水也夠。

**旁白**：李杰走到門口,移開那根沉重的木條。木條落在地上,發出沉悶的聲響。

**旁白**：他推開門,冰冷的夜風湧入屋內。外面的森林一片漆黑,只有紫月投下微弱的光芒。

**李杰**：(握緊手中的劍)我會找到你們的。

**旁白**：他走出這棟充滿謎團的空屋,消失在茫茫的夜色森林之中。

**旁白**：身後,小屋的門在風中輕輕搖晃,發出「吱呀」的聲響。彷彿在訴說著某個無人知曉的秘密。
