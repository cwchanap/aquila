# 第十一幕：寂靜的清晨

**旁白**：李杰回到木屋前，推開門走了進去。

**清水恵美**：（立刻抬起頭，期待地）李杰先生！找到大叔了嗎？

**李杰**：（搖頭）沒有。我在附近搜索了一圈，沒有看到任何人。

**旁白**：恵美的臉上閃過一絲失望，但她很快點了點頭，沒有多說什麼。

**清水恵美**：（小聲）這樣啊⋯⋯謝謝您願意去找⋯⋯

**旁白**：她轉過身，繼續照顧床上的女孩。李杰注意到，房間角落的小爐灶上正煮著什麼東西，散發出淡淡的食物香氣。

**清水恵美**：（邊照顧女孩邊說）我⋯⋯我在煮一些粥⋯⋯她需要補充體力⋯⋯

**李杰**：（走近床邊）讓我看看她的傷勢。

**旁白**：恵美讓開位置，讓李杰靠近。李杰仔細檢查了女孩的傷口。

**旁白**：右手臂的劍傷很深，從肩膀一直延伸到手肘，像是被利刃劈砍造成的。傷口邊緣很整齊，但失血量應該不小。

**旁白**：左大腿上的箭傷更加麻煩。箭桿已經被折斷，但箭頭還深深插在肉裡。周圍的皮膚紅腫，顯然很痛。

**李杰**：（評估）劍傷看起來可怕，但應該沒有傷到骨頭或重要血管。箭傷比較麻煩，箭頭還在裡面⋯⋯

**李杰**：（轉向恵美）不過都不是致命傷。只要好好處理，應該能撐過去。妳處理得不錯。

**清水恵美**：（鬆了一口氣）太好了⋯⋯我⋯⋯我真的很擔心⋯⋯我只能照著書上的圖示做⋯⋯

**旁白**：她擦了擦眼角，顯然壓力很大。李杰環顧房間，注意到櫥櫃裡還擺著其他醫療用品和食物。

**李杰**：這房子⋯⋯看起來有人居住的痕跡。而且還有這些東西。

**清水恵美**：（點頭）嗯⋯⋯我也覺得很奇怪⋯⋯這裡好像有人定期維護⋯⋯但現在沒有人⋯⋯

**旁白**：兩人沉默了片刻。房間裡只有爐灶上煮粥的聲音，還有床上女孩微弱的呼吸聲。

**清水恵美**：（小聲）李杰先生⋯⋯要不要⋯⋯一起吃點東西？

**旁白**：李杰這才意識到，自己從派對結束後就沒吃過任何東西。長時間的行走和緊張，讓他的體力消耗很大。

**李杰**：（點頭）好。

**旁白**：恵美從爐灶上端下一個小鍋，裡面是簡單的粥。她找到兩個木碗，盛了兩碗，遞給李杰一碗。

**旁白**：兩人坐在木桌旁，安靜地吃著。熱粥的溫暖讓疲憊的身體得到了一絲慰藉。

**李杰**：（打破沉默）這房子的東西很齊全。醫療用品、食物、甚至還有治療指南⋯⋯

**清水恵美**：（點頭）嗯⋯⋯我也覺得很奇怪⋯⋯好像有人特意準備的⋯⋯但是為什麼會放在森林裡？

**李杰**：（思考）也許是給迷路的人準備的。或者⋯⋯這個世界本來就有人居住。

**清水恵美**：（小聲）那些精靈⋯⋯會不會就是這個世界的居民？

**旁白**：李杰沉默了。他想起那個拿著黑劍的男人，想起恵美提到的從樹上跳下來的黑影。

**李杰**：（內心）這個世界⋯⋯到底有多少秘密？

**旁白**：兩人繼續默默地吃著粥。房間裡很安靜，只有爐火偶爾發出的劈啪聲。

**旁白**：吃完飯後，恵美開始收拾碗筷。李杰站起身，環顧房間。

**李杰**：今晚我們得想辦法休息。妳累了吧？

**清水恵美**：（點頭，有些不好意思）嗯⋯⋯有一點⋯⋯但是⋯⋯只有一張床⋯⋯

**李杰**：床讓給她。我們找其他地方睡。

**旁白**：李杰打開櫥櫃，翻找著裡面的物品。很快，他找到了幾塊厚實的地毯，還有一些看起來像是毛毯的布料。

**李杰**：這些應該可以用。

**旁白**：恵美幫忙將地毯鋪在地板上。兩人各選了房間的一角，中間隔著那張木桌。

**清水恵美**：（鋪好地毯後，小聲地）李杰先生⋯⋯真的⋯⋯真的謝謝您⋯⋯

**李杰**：（平靜地）別多想。好好休息吧。

**旁白**：恵美點點頭，裹著毛毯躺了下來。但她顯然很難入睡，不時翻身，發出輕微的聲響。

**旁白**：過了一會兒，她小聲開口。

**清水恵美**：（聲音在黑暗中顯得很小）李杰先生⋯⋯您還醒著嗎？

**李杰**：（低聲）嗯。

**清水恵美**：（猶豫）我⋯⋯我在想⋯⋯山本大叔會不會⋯⋯

**旁白**：她沒有說完，但意思已經很明顯了。

**李杰**：（沉默片刻）別想太多。他是個強壯的人，應該能照顧自己。

**清水恵美**：（聲音顫抖）可是⋯⋯可是這裡太危險了⋯⋯那些精靈⋯⋯還有那個黑影⋯⋯

**李杰**：擔心也沒用。我們現在能做的，就是照顧好自己，還有那個女孩。

**旁白**：恵美沒有再說話。房間裡又恢復了寂靜。

**旁白**：過了一會兒，恵美再次開口。

**清水恵美**：（更小聲）李杰先生⋯⋯您害怕嗎？

**李杰**：（誠實地）害怕。但恐懼解決不了問題。

**清水恵美**：（苦笑）我⋯⋯我一直都很害怕⋯⋯從上車開始⋯⋯到現在⋯⋯

**清水恵美**：（聲音哽咽）我不知道⋯⋯我不知道為什麼會遇到這種事⋯⋯我只是想回家⋯⋯

**李杰**：（沉默片刻）我也想。

**旁白**：兩人不再說話。房間裡只剩下床上女孩微弱的呼吸聲，還有窗外偶爾傳來的風聲。

**旁白**：李杰躺在地毯上，盯著天花板。疲憊和緊張讓他的身體沉重，但大腦卻異常清醒。

**李杰**：（內心）這個世界⋯⋯那個黑衣男⋯⋯精靈⋯⋯還有這座房子⋯⋯

**李杰**：（內心）到底要怎樣才能回去？

**旁白**：但思緒最終還是被疲憊打敗。不知過了多久，李杰終於沉入了夢鄉。

**旁白**：兩輪紫月依然懸掛在天空，靜靜注視著這座孤立的木屋。森林在夜色中沉睡，只有偶爾的風聲打破寂靜。

---

**旁白**：李杰睜開眼睛。

**旁白**：房間裡依然昏暗，紫色的月光透過窗戶灑進來，在地板上形成模糊的光影。

**旁白**：他不知道自己睡了多久。在這個沒有太陽的世界裡，時間變得毫無意義。

**李杰**：（坐起身，揉了揉眼睛）

**旁白**：他環顧房間。床上的女孩還在沉睡，呼吸平穩。但是——

**李杰**：（驚愕）清水？

**旁白**：恵美的地毯空了。毛毯整齊地疊在一旁，但人已經不見了。

**李杰**：（立刻站起來，快步走向門口）清水！

**旁白**：沒有回應。他推開門，衝到屋外。

**旁白**：濃霧依然籠罩著森林，能見度極低。周圍一片寂靜，沒有任何人影。

**李杰**：（大聲）清水！妳在哪裡？

**旁白**：他的聲音在霧中迴盪，卻沒有得到任何回應。

**李杰**：（內心，震驚）她去哪裡了？為什麼不叫醒我？

**旁白**：李杰開始在房子周圍搜索。他沿著昨天走過的路線檢查，希望能找到恵美的蹤跡。

**旁白**：但濃霧遮蔽了一切。地面上沒有明顯的腳印，也沒有任何打鬥的痕跡。

**李杰**：（內心）她是自己離開的？還是⋯⋯被什麼東西帶走了？

**旁白**：他繼續搜索，越走越遠。但隨著時間流逝，他逐漸意識到這樣下去沒有意義。

**旁白**：森林太大了，濃霧讓搜索變得幾乎不可能。而且⋯⋯那個黑衣男可能還在附近。

**李杰**：（停下腳步，內心）不行⋯⋯這樣太危險了。我不能為了找她而把自己也搭進去。

**李杰**：（內心）她是自己決定離開的？還是去找山本大叔了？

**旁白**：他站在霧中，猶豫了片刻。最後，他轉身往回走。

**李杰**：（內心）對不起⋯⋯但我沒辦法⋯⋯

**旁白**：他回到木屋，關上門。房間裡依然只剩下那個受傷的女孩，安靜地躺在床上。

**李杰**：（看著床上的女孩，沮喪地嘆氣）

**旁白**：他坐在木桌旁，雙手抱頭。

**李杰**：（內心）該死⋯⋯為什麼會變成這樣⋯⋯

**李杰**：（內心）山本大叔消失了，清水也不見了⋯⋯現在只剩下我和這個重傷的女孩⋯⋯

**旁白**：他抬起頭，看向床上的女孩。她的臉色依然蒼白，呼吸微弱。傷口需要定期照護，她需要食物和水。

**李杰**：（苦笑）真是⋯⋯麻煩透了⋯⋯

**旁白**：他站起身，走到櫥櫃前，開始檢查剩餘的物資。食物還有一些，水也夠用。醫療用品也算充足。

**李杰**：（內心）至少物資不是問題⋯⋯但是⋯⋯我要照顧她多久？她什麼時候會醒來？

**李杰**：（內心）而且⋯⋯我一個男人照顧一個年輕女孩⋯⋯萬一她醒來了⋯⋯

**旁白**：他搖了搖頭，試圖把這些想法趕出腦海。

**李杰**：（自言自語）算了⋯⋯先確保她活著吧。其他的⋯⋯之後再說。

**旁白**：他走到床邊，檢查女孩的傷口。繃帶需要更換了，手臂的傷口周圍還有些紅腫。

**李杰**：（拿起治療指南，翻閱著）照著書上做⋯⋯應該可以⋯⋯

**旁白**：他開始小心翼翼地更換繃帶。動作生疏，但盡力保持輕柔。

**旁白**：女孩在睡夢中微微皺眉，似乎感覺到了疼痛，但沒有醒來。

**李杰**：（內心）她到底是誰？那群大學生裡的哪一個？

**李杰**：（內心）其他人⋯⋯還活著嗎？

**旁白**：他處理完傷口，重新坐回桌旁。房間裡又恢復了寂靜。

**旁白**：窗外的濃霧依然沒有散去的跡象。兩輪紫月靜靜懸掛在天空，彷彿永恆。

**李杰**：（看向窗外，低聲）清水⋯⋯妳到底去哪裡了⋯⋯

**旁白**：但沒有人能回答他。

**旁白**：在這座孤立的木屋裡，李杰獨自守護著一個陌生的女孩，等待著不知何時會到來的轉機。
