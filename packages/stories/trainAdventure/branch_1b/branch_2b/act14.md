# 第十四幕：逃離與重建

**李杰**：（立刻轉身，壓低聲音但語氣急促）走！現在！

**高橋美咲**：（驚慌）什麼？

**李杰**：（快步走向她，不由分說地拉起她）別問了，我們必須離開這裡！

**旁白**：他扶著美咲站起來，但她的腿因為箭傷而無法快速移動。

**高橋美咲**：（痛苦地）好痛⋯⋯

**李杰**：（咬牙，轉過身蹲下）上來，我背妳。

**高橋美咲**：（猶豫）可是⋯⋯

**李杰**：（急切）快點！沒時間了！

**旁白**：美咲不再猶豫，趴在李杰背上。李杰一把抓起桌上的包裹和鏽劍，衝出了後門。

**旁白**：他沒有走大路，而是直接鑽進了木屋後方茂密的灌木叢。

**旁白**：剛離開木屋不到幾十米，身後就傳來了沈重的撞擊聲。

**旁白**：砰！

**旁白**：是木屋前門被踹開的聲音。

**李杰**：（咬緊牙關，拼命向前跑）

**旁白**：他在樹林間穿梭，不顧樹枝劃破皮膚的疼痛。背上的美咲緊緊抓著他的肩膀，呼吸急促而沈重。

**旁白**：身後的木屋方向傳來了微弱的火光。回頭一看，橘紅色的火焰正迅速吞噬著那棟曾經溫馨的房子。

**高橋美咲**：（低聲，帶著哭腔）房子⋯⋯被燒了⋯⋯

**李杰**：別看。抓穩了。

**旁白**：他在霧中狂奔了將近一個小時，直到確認身後沒有追兵，才在一個隱蔽的岩石縫隙下停了下來。

**李杰**：（放下美咲，大口喘著氣）

**旁白**：兩人癱坐在地上，肺部因為劇烈的呼吸而感到灼燒。

**高橋美咲**：（看著李杰被劃破的臉頰，小聲說）對不起⋯⋯

**李杰**：（搖頭，聲音沙啞）妳沒事就好。

**旁白**：他在附近找到了一些乾枯的枝葉，小心地生起了一堆極小的火，用石頭擋住光亮。

**李杰**：（沙啞地）妳的腿⋯⋯

**高橋美咲**：（搖頭）沒關係⋯⋯你為了我做了這麼多⋯⋯至少讓我做點什麼⋯⋯

**旁白**：她一拐一拐地端著兩個碗走過來，放在桌上。是簡單的粥，但散發出溫暖的香氣。

*(註：此處場景似乎跳轉到了逃離後的某個安全處)*

**高橋美咲**：（坐下）吃吧⋯⋯

**旁白**：李杰拿起筷子，機械地吃著。他太累了，甚至沒有力氣品嚐味道。

**旁白**：兩人安靜地吃了一會兒。房間裡只有筷子碰觸碗的聲音。

**高橋美咲**：（小聲）李杰先生⋯⋯剛才那個人⋯⋯是誰？

**李杰**：（停下筷子，搖頭）我不知道。

**高橋美咲**：（困惑）你不認識他？

**李杰**：（皺眉）我只記得⋯⋯在新宿站的月台上見過他。他站在立柱的陰影裡，一個人，一動也不動。

**李杰**：（沉默片刻）當時就覺得他很奇怪⋯⋯那種感覺⋯⋯他絕對不是什麼好人。

**高橋美咲**：（害怕地）他⋯⋯他很危險嗎？

**李杰**：（點頭）很危險。那把劍⋯⋯不是裝飾品。

**旁白**：美咲的臉色變得蒼白。她低下頭，緊緊握著筷子。

**高橋美咲**：（顫抖地）拓海他們⋯⋯如果遇到那個人⋯⋯

**李杰**：（看著她，沒有說話）

**高橋美咲**：（抬起頭，眼眶泛紅）他們⋯⋯他們還活著嗎⋯⋯

**李杰**：（沉默片刻）我不知道。

**高橋美咲**：（聲音哽咽）可是⋯⋯可是如果那個人⋯⋯

**李杰**：（打斷她，語氣平靜但冷酷）別太樂觀。那個黑衣男⋯⋯不是好惹的。

**旁白**：美咲愣住了。眼淚開始滑落臉頰。

**高橋美咲**：（哭泣）不⋯⋯不要⋯⋯他們⋯⋯他們是我最好的朋友⋯⋯

**旁白**：她放下筷子，雙手掩面。肩膀因為哭泣而顫抖。

**旁白**：李杰看著她，嘆了一口氣。

**李杰**：（站起來，走到她身邊）

**旁白**：美咲抬起頭，淚眼模糊地看著他。

**高橋美咲**：（哽咽）李杰先生⋯⋯

**旁白**：李杰沒有說話。他只是坐在她旁邊，讓她靠在自己的肩膀上。

**旁白**：美咲趴在他肩上，放聲哭了起來。所有的恐懼、悲傷、無助，都在這一刻爆發出來。

**李杰**：（輕輕拍著她的頭）

**旁白**：他的動作有些僵硬，不太習慣安慰人。但他沒有推開她。

**旁白**：房間裡只有美咲的哭泣聲。窗外的森林依然寂靜，濃霧籠罩著一切。

**旁白**：在這座新的木屋裡，兩個倖存者相依為命，在這個充滿危險的世界中尋找著一絲溫暖。

**旁白**：而在森林的某處，那個黑衣男⋯⋯不知道是否還在追尋著他們的蹤跡。
