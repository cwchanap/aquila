# 第十九幕：長老的召見

**旁白**：李杰和美咲被帶進建築內。

**旁白**：走廊很寬敞，牆壁是淡黃色的石材，天花板上掛著幾盞散發柔和光芒的水晶燈。地面鋪著光滑的石板，腳步聲在走廊中迴響。

**旁白**：兩旁是一扇扇木門，每扇門上都刻著精緻的花紋。建築內部乾淨整潔，但氣氛嚴肅。

**旁白**：守衛們帶著他們穿過幾條走廊，最後停在一扇厚重的木門前。

**精靈守衛A**：（打開門，精靈語）進去。

**旁白**：門內是一間牢房。

**旁白**：房間不大，大約三公尺見方。石牆，鐵柵欄，地上鋪著一些乾草和兩張簡陋的床墊。牆角有一個小窗戶，光線從外面透進來。

**旁白**：房間很簡陋，但至少乾淨。

**精靈守衛A**：（解開繩子，精靈語）老實待著。我們會向長老艾莉希雅報告。在她到來之前，不准離開這個房間。

**李杰**：（揉著手腕）艾莉希雅？

**精靈守衛B**：（點頭，精靈語）我們的長老。她會決定怎麼處理你們。

**旁白**：守衛們退出房間，關上門。

**旁白**：「咔嚓」一聲，鐵門上鎖。

**旁白**：牢房裡只剩下李杰和美咲。

**高橋美咲**：（揉著手腕，小聲）李杰先生，我們會怎麼樣？

**李杰**：（走到窗戶旁，看向外面）不知道。

**旁白**：窗外是精靈村莊的街道。陽光明媚，居民們來來往往。但每個經過牢房的精靈，都會投來好奇或警戒的眼神。

**高橋美咲**：（坐在床墊上，抱著膝蓋）他們會不會殺了我們？

**李杰**：（轉過身）不會。如果他們想殺我們，早就動手了。

**高橋美咲**：（低聲）那他們會把我們關多久？

**李杰**：（搖頭）我不知道。

**旁白**：兩人沉默了一會兒。

**高橋美咲**：（突然想到什麼）李杰先生，我們有多久沒吃東西了？

**李杰**：（想了想）從昨天早上到現在，快一天了吧。

**高橋美咲**：（摸著肚子）我好餓。

**李杰**：（走到門前，敲了敲鐵門）喂！

**旁白**：過了一會兒，門外傳來腳步聲。

**精靈守衛C**：（隔著門，精靈語）什麼事？

**李杰**：（精靈語）我們需要食物和水。

**精靈守衛C**：（猶豫了一下，精靈語）等一下。

**旁白**：腳步聲遠去。

**高橋美咲**：（期待地）他去拿食物了嗎？

**李杰**：（點頭）應該是。

**旁白**：過了大約十分鐘，腳步聲再次響起。

**旁白**：門打開一條縫，守衛遞進來一個木盤和兩個杯子。

**精靈守衛C**：（冷冷地，精靈語）拿著。

**旁白**：李杰接過木盤和杯子。門立刻關上，再次上鎖。

**李杰**：（把木盤放在地上）來吃吧。

**旁白**：木盤上有兩塊黑麵包和幾片乾肉。杯子裡裝著清水。

**高橋美咲**：（拿起麵包，咬了一口）好硬。

**李杰**：（喝了一口水）至少有東西吃。

**旁白**：麵包很硬，乾肉很鹹。水倒是乾淨，沒有異味。

**旁白**：典型的囚犯伙食。

**高橋美咲**：（勉強吃著麵包）這就是我們的晚餐嗎？

**李杰**：（點頭）看起來是。

**旁白**：兩人默默地吃完食物。

**旁白**：窗外的天色漸漸暗了下來。

**高橋美咲**：（看著窗外，輕聲）李杰先生，你說他們會對我們怎麼樣？

**李杰**：（靠在牆上）不知道。也許會審問我們，也許會把我們驅逐出去。

**高橋美咲**：（低聲）如果他們不讓我們離開呢？

**李杰**：（沉默）

**高橋美咲**：（抬起頭）如果我們再也回不了地球怎麼辦？

**李杰**：（看著她）不要想太多。

**高橋美咲**：（眼眶泛紅）可是我一直在想，我的家人怎麼辦？我爸媽，我姐姐，還有我的朋友們。他們一定以為我失蹤了，一定很擔心。

**李杰**：（輕聲）我也一樣。

**高橋美咲**：（看著李杰）你的家人呢？

**李杰**：（沉默片刻）我父母在上海。我已經很久沒回去看他們了。

**高橋美咲**：（小聲）他們知道你在日本嗎？

**李杰**：（點頭）知道。我每個月都會打電話回去。但如果我失蹤了，他們一定會很擔心。

**高橋美咲**：（眼淚滑落）我姐姐一定會瘋了。她一直很保護我，從小就是。如果我再也見不到她了。

**李杰**：（走過去，坐在她旁邊）別哭。我們一定會想辦法回去的。

**高橋美咲**：（抹著眼淚）可是我們連這裡是哪裡都不知道，怎麼回去？

**李杰**：（看著窗外）一步一步來。先活下去，然後想辦法。

**高橋美咲**：（低聲）李杰先生，你不害怕嗎？

**李杰**：（轉過頭）害怕。但害怕解決不了問題。

**高橋美咲**：（點頭）嗯。

**旁白**：兩人又沉默了一會兒。

**高橋美咲**：（輕聲）李杰先生，謝謝你。

**李杰**：（疑惑）謝什麼？

**高橋美咲**：（看著他）從遇到那輛火車開始，你一直在保護我。如果不是你，我可能早就死了。

**李杰**：（搖頭）我只是做了該做的事。

**高橋美咲**：（微笑）但你還是救了我。

**李杰**：（看著她，點頭）我們會一起活下去的。

**旁白**：窗外的天色完全暗了下來。

**旁白**：牢房裡只剩下微弱的月光。

**高橋美咲**：（躺在床墊上）晚安，李杰先生。

**李杰**：（靠在牆上）晚安。

**旁白**：美咲很快就睡著了。她的呼吸聲在寂靜的牢房中迴響，平穩而輕柔。

**旁白**：李杰沒有睡。他坐在床墊上，看著窗外的星空。

**李杰**：（內心）這裡到底是什麼地方？

**李杰**：（內心）為什麼會有精靈？

**李杰**：（內心）那輛火車到底把我們帶到了哪裡？

**旁白**：他想了很久，但沒有答案。

**旁白**：最終，他也閉上了眼睛。

**旁白**：在這個陌生的精靈村莊裡，兩個人類度過了他們的第一個夜晚。
