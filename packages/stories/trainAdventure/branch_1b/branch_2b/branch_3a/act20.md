# 第二十幕：與長老的對話

**旁白**：第二天早上。

**旁白**：陽光從窗戶照進來，溫暖而明亮。

**高橋美咲**：（睜開眼睛，揉著眼睛）早安，李杰先生。

**李杰**：（已經醒了，坐在窗戶旁）早安。

**高橋美咲**：（坐起來）你睡得好嗎？

**李杰**：（搖頭）還好。

**旁白**：就在這時，門外傳來腳步聲。

**旁白**：不止一個人。

**旁白**：「咔嚓」一聲，門鎖打開。

**精靈守衛A**：（推開門，精靈語）長老艾莉希雅要見你們。跟我們來。

**李杰**：（站起來）現在？

**精靈守衛A**：（點頭，精靈語）現在。

**高橋美咲**：（緊張地站起來）李杰先生，他們說什麼？

**李杰**：（對她）長老要見我們。

**高橋美咲**：（緊張）長老？

**旁白**：兩個守衛帶著李杰和美咲走出牢房。

**旁白**：他們穿過走廊，來到一扇雕刻著精緻花紋的木門前。

**精靈守衛A**：（敲了敲門，精靈語）長老，他們來了。

**旁白**：門內傳來一個溫和的女聲。

**艾莉希雅**：（精靈語）請進。

**旁白**：守衛推開門，示意李杰和美咲進去。

**旁白**：李杰和美咲走進房間。

**旁白**：房間很寬敞，牆上掛著幾幅畫，地上鋪著柔軟的地毯。中央擺著一張長桌，桌上放著各種食物——新鮮的水果、烤肉、麵包、還有幾壺飲料。

**旁白**：桌子的主位上，坐著一位精靈女性。

**旁白**：她看起來四十多歲，有著淡金色的長髮和溫柔的綠色眼睛。她穿著一件淡藍色的長袍，腰間繫著一條銀色的腰帶。

**旁白**：她的神情溫和，但眼神中透著智慧和威嚴。

**艾莉希雅**：（微笑，精靈語）歡迎。請坐。

**李杰**：（猶豫了一下，走向桌子）

**高橋美咲**：（跟在李杰身後，小聲）李杰先生？

**李杰**：（小聲）她讓我們坐下。

**旁白**：兩人在桌子旁邊坐下。守衛退出房間，關上了門。

**艾莉希雅**：（看著他們，溫和地，精靈語）我是艾莉希雅，這個村莊的長老。你們一定經歷了很辛苦的旅程。

**李杰**：（點頭，精靈語）是的。

**艾莉希雅**：（微笑）請享用這些食物。這是我們精靈的傳統，對遠道而來的客人，我們會準備最好的食物。

**旁白**：她推了推桌上的盤子。

**艾莉希雅**：（歉意地，精靈語）首先，我要向你們道歉。昨晚讓你們待在監牢裡，一定很不舒服。

**李杰**：（驚訝）你在道歉？

**艾莉希雅**：（點頭）是的。你們並沒有做錯什麼，但我們的守衛因為驚慌而把你們關起來。我為此感到抱歉。

**高橋美咲**：（小聲對李杰）她在說什麼？

**李杰**：（對她）她在為昨晚的事道歉。

**高橋美咲**：（驚訝）道歉？

**李杰**：（對艾莉希雅，皺眉，精靈語）我聽說精靈認為人類是邪惡的。

**艾莉希雅**：（微笑，精靈語）這是一個很長的故事。但在談論這些之前，請先享用食物吧。你們一定餓了。

**旁白**：她拿起一個盤子，遞給李杰。

**艾莉希雅**：（溫和地，精靈語）吃吧。之後我們可以好好談談。

**李杰**：（接過盤子，看著桌上的食物）

**旁白**：桌上的食物看起來很美味。新鮮的水果散發著香甜的氣息，烤肉金黃酥脆，麵包鬆軟可口。

**旁白**：和昨晚的囚犯伙食完全不同。

**高橋美咲**：（看著食物，吞了吞口水）

**李杰**：（對美咲）她讓我們吃。

**高橋美咲**：（小聲）真的可以嗎？

**李杰**：（點頭）應該沒問題。

**旁白**：兩人開始吃東西。

**旁白**：食物的味道很好。水果清甜多汁，烤肉香嫩可口，麵包鬆軟帶著淡淡的香草味道。

**旁白**：艾莉希雅微笑著看著他們，沒有說話。

**旁白**：她靜靜地等待著，等他們吃飽後，再開始談論正事。
