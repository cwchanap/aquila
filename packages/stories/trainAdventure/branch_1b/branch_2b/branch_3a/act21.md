# 第二十一幕：世界的真相

**旁白**：過了一會兒，李杰和美咲吃得差不多了。

**艾莉希雅**：（微笑，精靈語）吃飽了嗎？

**李杰**：（點頭，精靈語）是的，謝謝你的款待。

**艾莉希雅**：（溫和地，精靈語）很好。那麼，讓我先介紹一下這裡。

**旁白**：她拿起桌上的茶壺，給自己倒了一杯茶。

**艾莉希雅**：（精靈語）歡迎來到月華村。這是碧翠之島北部的精靈村莊。

**李杰**：（精靈語）碧翠之島？

**艾莉希雅**：（點頭，精靈語）是的。這座島上有三個精靈村莊。你們現在所在的月華村位於北部，南部是曦光之庭，中部是星辰谷。

**旁白**：她的神情變得有些嚴肅。

**艾莉希雅**：（精靈語）不過，這三個村莊現在處於戰爭狀態。村莊之間的往來被禁止，居民不允許自由流動。

**李杰**：（驚訝，精靈語）戰爭？

**艾莉希雅**：（點頭，精靈語）是的。這是一個很複雜的情況。但現在先不談這個。

**艾莉希雅**：（繼續，精靈語）我在一千年前被選為月華村的長老。在我之前的長老因為犯罪被驅逐出島，村民們選擇了我來領導這個村莊。

**李杰**：（驚訝，精靈語）一千年前？

**艾莉希雅**：（微笑，精靈語）是的。我知道這對你們來說可能很難理解。

**高橋美咲**：（小聲對李杰）她說了什麼？

**李杰**：（簡短地對她）她在介紹這個島和村莊的情況。她是一千年前當選的長老。

**高橋美咲**：（震驚）一千年？！

**艾莉希雅**：（看著他們的反應，溫和地，精靈語）你們看起來很驚訝。

**李杰**：（精靈語）是的。人類的壽命很短，通常只有七八十年。一千年對我們來說是難以想像的。

**艾莉希雅**：（點頭，理解地，精靈語）我明白了。精靈和人類確實很不一樣。

**旁白**：她喝了一口茶，然後繼續說。

**艾莉希雅**：（精靈語）其實，看到你們真的讓我很驚訝。我在這座島上出生，已經兩千五百年了，但從來沒有見過人類。

**李杰**：（震驚）兩千五百年？！

**艾莉希雅**：（微笑，精靈語）精靈的平均壽命大約是六千五百年。最長壽的精靈可以活到一萬年左右。

**高橋美咲**：（看著李杰震驚的表情，小聲）李杰先生？

**李杰**：（對她，難以置信地）她已經活了兩千五百年。精靈可以活六千五百年。

**高橋美咲**：（睜大眼睛）什麼？！

**旁白**：兩人都陷入了震驚。對於人類來說，這樣的壽命簡直超出了想像。

**艾莉希雅**：（繼續，精靈語）不過，這個村莊裡的大多數精靈都比我年輕。

**李杰**：（疑惑，精靈語）為什麼？

**艾莉希雅**：（神情變得嚴肅，精靈語）五千年前，在主島上發生了一場大戰。人類、精靈和獸人之間的戰爭。那場戰爭持續了很久，許多精靈在戰爭中死去。

**旁白**：她的眼神中透著悲傷。

**艾莉希雅**：（精靈語）戰爭結束後，大部分倖存的精靈選擇離開主島，搬到這座碧翠之島上，希望能遠離戰爭。所以這裡的精靈大多是在那之後出生的。

**李杰**：（內心）五千年前的戰爭？

**李杰**：（精靈語）你說主島上有人類？

**艾莉希雅**：（點頭，精靈語）是的。雖然我從未見過，但根據歷史記載，人類確實存在於這個世界。

**李杰**：（緊張地，精靈語）那現在呢？主島上還有人類嗎？

**艾莉希雅**：（搖頭，精靈語）我不太清楚。那場戰爭發生在我出生之前。我對主島的情況了解不多。

**旁白**：她停頓了一下。

**艾莉希雅**：（精靈語）不過，根據我們的歷史書記載，人類和獸人被描述為邪惡的種族，是侵略者。他們對精靈發動了戰爭，造成了巨大的傷亡。

**李杰**：（皺眉）侵略者？

**艾莉希雅**：（點頭，精靈語）是的。這就是為什麼我們的守衛看到你們時會如此緊張。在我們的教育中，人類是危險的。

**李杰**：（簡短地對美咲）她說這個世界有人類，五千年前發生過戰爭。精靈認為人類是侵略者。

**高橋美咲**：（不安）那我們怎麼辦？

**李杰**：（看向艾莉希雅，精靈語）但你並不認為我們是邪惡的？

**艾莉希雅**：（微笑，精靈語）歷史書記載的都是很久以前的事了。我更願意相信眼前看到的。

**旁白**：她看著李杰和美咲，眼神溫和。

**艾莉希雅**：（精靈語）你們看起來並不像那些故事裡描述的邪惡入侵者。

**李杰**：（精靈語）關於那條隧道，你知道些什麼嗎？

**艾莉希雅**：（搖頭，精靈語）說實話，我對那條隧道了解不多。就像村裡的其他精靈一樣，我們只知道它存在，但不知道它通往哪裡，也不知道是誰建造的。

**旁白**：她停頓了一下。

**艾莉希雅**：（認真地，精靈語）在談論這些之前，我想先問你們一個問題。你們是從哪裡來的？又為什麼會來到這裡？

**旁白**：李杰和美咲對視了一眼。

**李杰**：（深吸一口氣，精靈語）這說起來有些難以置信。我們來自另一個世界。

**艾莉希雅**：（挑眉）另一個世界？

**李杰**：（點頭，精靈語）是的。我們原本在一座叫東京的城市，晚上搭上了一輛開往郊區的列車。但那輛列車並沒有把我們帶到目的地，而是把我們帶到了一個完全陌生的地方。

**旁白**：他停頓了一下。

**李杰**：（精靈語）之後我們經歷了很多事情，最終找到了那條隧道，穿過去後就來到了你們的村莊。

**高橋美咲**：（安靜地聽著，雖然聽不懂，但感覺到氣氛很嚴肅）

**旁白**：艾莉希雅陷入了沉默。她的表情變得嚴肅。

**艾莉希雅**：（緩緩開口，精靈語）火車。

**李杰**：（精靈語）你知道那輛火車嗎？

**艾莉希雅**：（搖頭，精靈語）我對那輛火車了解不多。但我知道，每隔兩年，會有一輛火車從森林深處經過村莊北部，然後進入那條隧道。

**李杰**：（驚訝，精靈語）每隔兩年？

**艾莉希雅**：（點頭，精靈語）是的。村民們都知道這件事。那輛火車總是在深夜經過，沒有人知道它從哪裡來，要去哪裡。

**旁白**：她停頓了一下。

**艾莉希雅**：（精靈語）曾經有一些精靈試圖接近森林的核心，想要弄清楚火車的來源。但他們都被阻止了。

**李杰**：（疑惑，精靈語）被誰阻止？

**艾莉希雅**：（精靈語）石像守衛。在森林深處，有許多巨大的石像。它們像是守護者一樣，阻止任何人接近森林的核心。

**李杰**：（內心）石像守衛？

**艾莉希雅**：（精靈語）那些石像很古老。根據記載，它們在我們精靈搬到這座島之前就已經存在了。沒有人知道是誰建造的，也沒有人知道它們為什麼要守護那片森林。

**旁白**：李杰皺起眉頭。

**李杰**：（內心）石像？火車？隧道？這一切到底是怎麼回事？

**艾莉希雅**：（看著他們，精靈語）你們說你們來自另一個世界，被火車帶到了這裡。這聽起來很不可思議，但也許能解釋一些事情。

**李杰**：（精靈語）什麼事情？

**艾莉希雅**：（精靈語）那條隧道，那輛火車，還有那些石像守衛。也許它們都是連接不同世界的通道。

**旁白**：她的眼神變得深邃。

**艾莉希雅**：（精靈語）但這只是我的猜測。真相是什麼，我也不知道。

**旁白**：房間陷入了短暫的沉默。

**高橋美咲**：（小聲對李杰）李杰先生，她說了很多，對吧？

**李杰**：（對她，簡短地總結）她說這座島上有三個精靈村莊，現在處於戰爭狀態。她活了兩千五百年，精靈可以活六千五百年。五千年前這個世界發生過人類、精靈和獸人之間的大戰。

**旁白**：美咲睜大眼睛，難以置信地看著李杰。

**李杰**：（繼續）她說每隔兩年會有一輛火車從森林經過村莊進入隧道。森林深處有古老的石像守衛，阻止精靈接近。她猜測這些可能都是連接不同世界的通道。

**高橋美咲**：（震驚）連接不同世界？

**李杰**：（點頭）是的。但她也不確定。

**旁白**：美咲低下頭，消化著這些難以置信的信息。

**艾莉希雅**：（看著他們，精靈語）你們需要時間休息。這些事情一時半刻也說不清楚。我會安排房間給你們住，明天我們再繼續談。

**李杰**：（精靈語）謝謝你的款待。

**艾莉希雅**：（微笑）不客氣。歡迎來到月華村。
