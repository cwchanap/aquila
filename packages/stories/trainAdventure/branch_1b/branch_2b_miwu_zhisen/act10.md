# 第十幕：木屋中的相遇

**旁白**：李杰握著劍，深吸一口氣，朝著木屋的方向走去。

**旁白**：腳下的土地變得平整，似乎曾有人打理過。就在他快要走到門廊前時，一陣急促的腳步聲從另一側的樹林裡傳來。

**李杰**：(立刻轉身，橫劍於身前)誰！

**清水恵美**：(驚叫一聲，猛地停住腳步)啊！

**旁白**：在紫月與窗戶透出的微光下，李杰認出了那個身影。

**李杰**：(驚訝)是你？那個畫畫的女孩？

**清水恵美**：(喘著氣，臉色蒼白，手中依然緊緊抓著那個素描本)李⋯⋯李杰先生？

**李杰**：(放下劍，但並未完全放鬆警惕)你怎麼會在這裡？你不是跟著大叔他們下車了嗎？

**清水恵美**：(聲音顫抖，眼神中充滿恐懼)我⋯⋯我們在月台上走散了⋯⋯霧太大了⋯⋯

**清水恵美**：(看了一眼李杰身後的木屋)你⋯⋯你也發現這裡了？

**李杰**：剛到。大叔和那些學生呢？

**清水恵美**：(搖頭)我不知道⋯⋯我們進了森林之後，霧突然變得好大，大家走得很快，我跟不上⋯⋯然後我就聽到了一些很可怕的叫聲⋯⋯

**李杰**：叫聲？

**清水恵美**：(抱緊素描本，身體微微發抖)嗯⋯⋯不像人類的聲音。大家開始亂跑，我⋯⋯我就拼命跑，然後就看到這道光亮⋯⋯

**李杰**：(皺眉)看來這森林比我想象的還要危險。

**清水恵美**：(看著李杰)那我們⋯⋯要進去嗎？

**李杰**：比起待在外面餵那種發出可怕叫聲的東西，我想進去是唯一的選擇。

**旁白**：李杰示意恵美跟在自己身後。他走上木製台階，來到那扇緊閉的大門前。

**旁白**：門上雕刻著複雜的花紋，中間鑲嵌著一個青銅製的獅頭敲門器。

**李杰**：(深吸一口氣，用力敲了敲門)

**旁白**：咚！咚！咚！

**旁白**：沈悶的聲音在寂靜的夜色中傳得很遠。

**旁白**：沒有回應。

**李杰**：有人在嗎？

**旁白**：依然是一片死寂。

**李杰**：(試著轉動門把)⋯⋯

**旁白**：門沒鎖。

**旁白**：大門緩緩向內滑開，發出輕微的「吱呀」聲。一股溫暖的、帶著淡淡薰衣草香氣的空氣從屋內湧出，瞬間驅散了身上的寒意。

**李杰**：(回頭看了恵美一眼，點了點頭)進去吧。

**旁白**：兩人走進屋內。眼前的景象讓他們感到無比震驚。

**旁白**：這不是一個荒廢的木屋。室內裝修得非常典雅，紅木的地板擦得發亮。壁爐裡燃燒著橘紅色的火焰，發出啪嗒啪嗒的聲響，將客廳照得通明。

**旁白**：舒適的沙發、擺滿書的書架、甚至桌上還放著一壺冒著熱氣的茶。

**清水恵美**：(愣在那裡)這裡⋯⋯這裡有人住？

**李杰**：(橫著劍，警惕地觀察四周)看起來是這樣。而且⋯⋯主人應該就在附近。

**李杰**：(大聲喊道)不好意思！有人在嗎？我們是迷路的旅人，想在這裡借宿一晚！

**旁白**：回應他的，只有木材在火中燃燒的聲音。

**清水恵美**：(指著桌上的茶壺)茶還熱著⋯⋯

**李杰**：(內心)太奇怪了。這裡的一切都像是剛準備好，卻偏偏一個人都沒有。

**旁白**：就在這時，二樓傳來了一聲細微的、像是重物落地的聲響。

**李杰**：(立刻抬起頭，眼神變得銳利)上面有人。

**清水恵美**：(嚇得抓住了李杰的衣袖)李杰先生⋯⋯

**李杰**：(低聲)你在這裡待著。如果發現不對勁，立刻往外面跑。

**清水恵美**：(點頭，聲音細若蚊蚋)好⋯⋯

**旁白**：李杰握緊手中的鏽劍，朝著那通向二樓的木製旋轉樓梯走去。
