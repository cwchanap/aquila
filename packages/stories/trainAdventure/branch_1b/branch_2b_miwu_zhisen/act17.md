# 第十七幕：隧道的抉擇

**旁白**：鐵門後的通道出乎意料地平整。

**旁白**：岩壁被人工修鑿過，呈現出一種冷硬的灰色。每隔一段距離，牆上就鑲嵌著一顆散發著微弱螢光的礦石，勉強照亮了前方的路。

**李杰**：（舉著火把，小心翼翼地走在前面）小心腳下，這裡可能有陷阱。

**高橋美咲**：（緊跟在他身後，聲音微弱）嗯⋯⋯

**旁白**：兩人沉默地走著。除了他們的腳步聲，只有火把燃燒時發出的劈啪聲。

**旁白**：大約走了十五分鐘，前方的通道開始變寬。

**旁白**：最後，他們來到了一個十字路口。

**旁白**：四條通道在他們面前交匯。每一條通道看起來都一模一樣，深不見底，充滿了未知的恐懼。

**李杰**：（停下腳步，環顧四周）這是⋯⋯迷宮嗎？

**高橋美咲**：（緊張地看著四周）我們該往哪邊走？

**李杰**：（拿出那張泛黃的地圖）地圖上⋯⋯

**旁白**：他低下頭，卻發現地圖上的路徑在進入洞穴後就消失了。取而代之的，是一片空白。

**李杰**：（咒罵）該死！地圖沒用了！

**高橋美咲**：（臉色蒼白）什麼？那我們怎麼辦？

**李杰**：（深吸一口氣，強迫自己冷靜下來）別慌。一定有線索。

**旁白**：他舉起火把，仔細觀察地面。

**旁白**：但地面是乾燥的石板，沒有留下任何腳印。

**旁白**：他轉向牆壁。牆上的螢光礦石依舊散發著微弱的光，沒有任何標記。

**李杰**：（內心）一定有什麼⋯⋯

**旁白**：就在這時，他注意到最左邊的那條通道裡，傳來了一種細微的聲音。

**旁白**：「滴答⋯⋯滴答⋯⋯」

**李杰**：（愣住）水聲？

**高橋美咲**：（也聽到了）和祭壇那裡的聲音一樣！

**李杰**：（走向那條通道，側耳傾聽）確實是水聲。

**旁白**：他轉向中間的那條通道。

**旁白**：在那裡，傳來了輕微的「嗚——」聲。

**李杰**：這是⋯⋯風聲？

**高橋美咲**：（點頭）嗯，我也聽到了。

**旁白**：最後，他走向最右邊的那條通道。

**旁白**：那裡一片死寂。什麼聲音都沒有。

**李杰**：（站在路口，眉頭緊鎖）三條路，三種情況。

**高橋美咲**：李杰先生，我們要選哪一條？

**李杰**：（思考）水聲代表生機，但也可能代表陷阱。風聲代表出口，但也有可能通往更深的地底。

**高橋美咲**：那死寂的那一條呢？

**李杰**：（看著最右邊的通道，眼神深邃）死寂⋯⋯代表未知。

**旁白**：兩人沉默了。

**旁白**：一條路代表著他們剛剛經歷過的「規律」——水與風。另一條路代表著徹底的「無」——寂靜。

**高橋美咲**：（猶豫）我們要回頭嗎？

**李杰**：（搖頭）鐵門已經關上了。我們只能往前走。

**旁白**：李杰看著眼前那三條深邃的通道。

**李杰**：（內心）左邊是水，中間是風，右邊是死寂。

**李杰**：（內心）如果這是一個測驗⋯⋯正確答案會在哪裡？

**旁白**：他的腦海中閃過新宿站月台上的那個黑衣男。閃過小林先生那致命的一劍。閃過那個活潑女大生的屍體。

**李杰**：（握緊鏽劍，眼神變得堅定）走。

**高橋美咲**：哪一邊？

**李杰**：我們走中間。

**高橋美咲**：（疑惑）風聲的那條？

**李杰**：（點頭）風代表流動，代表連通外界。如果這裡有風，就說明這條路一定能通往某個有空氣的地方。

**高橋美咲**：（深吸一口氣）好，我聽你的。

**旁白**：兩人邁步走進了中間的那條通道。

**旁白**：隨著他們的深入，風聲變得越來越大。火把的火焰被風吹得搖擺不定，隨時可能熄滅。

**李杰**：（護住火把）跟緊了！

**旁白**：又走了大約十分鐘，前方的光芒越來越亮。

**旁白**：當他們衝出通道時，眼前的景象讓他們再次愣住了。

**旁白**：這是一條隧道。

**旁白**：不是人工開鑿的石道，而是真正的、現代化的鐵路隧道。

**旁白**：兩側的牆壁是用灰色的石磚砌成，天花板呈拱形。而在隧道的中央，有一條延伸向黑暗深處的鐵軌。

**李杰**：（愣在原地，看著那條熟悉的鐵軌）這是⋯⋯

**高橋美咲**：（也愣住了，聲音顫抖）這是我們來的那條軌道嗎？

**李杰**：（搖頭）我不確定⋯⋯但看起來一模一樣。

**高橋美咲**：（驚喜）那我們是不是可以回家了？只要沿著這條軌道走⋯⋯

**李杰**：（冷淡地打斷她）別高興得太早。

**旁白**：他指著軌道延伸的方向。在隧道的盡頭，並不是城市的光芒，而是更深邃的黑暗。

**李杰**：妳聽。

**旁白**：隧道深處，傳來了一種低沈的、規律的震動聲。

**旁白**：那是列車行駛的聲音。

**旁白**：而且，正朝著他們的方向而來。

**高橋美咲**：（臉色驟變，驚恐地抓住李杰的衣袖）李杰先生！火車來了！

**李杰**：（臉色鐵青，握緊手中的鏽劍）該死！

**旁白**：遠處，兩盞巨大的頭燈劃破了隧道的黑暗。刺眼的光芒像是一頭沈睡的巨獸睜開了雙眼。

**旁白**：震動聲越來越大，軌道發出尖銳的摩擦聲。

**李杰**：（大喊）走！離開軌道！快！

**旁白**：他拉起美咲，拼命朝著隧道的一側跑去。

**旁白**：但在這狹窄的隧道裡，幾乎沒有躲避的空間。

**旁白**：巨大的列車呼嘯而至，帶著足以撕裂空氣的力量，朝著這兩個弱小的身影衝來。

**旁白**：在那刺眼的光芒中，李杰看到了車窗後那些模糊的臉孔。

**旁白**：那是之前在「寂海」站留下的那些乘客嗎？還是新的一批受害者？

**旁白**：列車的速度快得驚人，沒有絲毫減速的跡象。

**旁白**：在這千鈞一髮之際，李杰發現隧道的牆壁上，有一個極小的凹陷處。

**李杰**：（一把將美咲推了進去，自己也緊跟著擠了進去）躲好！

**旁白**：「轟隆隆——！！！」

**旁白**：列車從他們面前呼嘯而過。強大的風壓幾乎要將他們從凹陷處吸出去。李杰死死地按住牆壁，用身體擋住美咲。

**旁白**：一節、兩節、三節⋯⋯

**旁白**：列車源源不斷地駛過。在那令人窒息的轟鳴聲中，李杰感受到了一種原始的恐懼。

**旁白**：終於，最後一節車廂駛過。

**旁白**：紅色的尾燈在黑暗中漸漸縮小，最終消失在隧道的另一端。

**旁白**：隧道重新恢復了死寂。只有李杰和美咲急促而驚恐的喘息聲。

**李杰**：（癱坐在地上，全身被冷汗浸濕）

**高橋美咲**：（縮成一團，低聲抽泣）太可怕了⋯⋯太可怕了⋯⋯

**李杰**：（抬起頭，看著列車消失的方向，眼神中充滿了迷茫與憤怒）這到底是怎麼回事⋯⋯

**旁白**：就在這時，隧道深處又傳來了一種聲音。

**旁白**：不是列車的轟鳴聲，而是緩慢、沈重，且規律的腳步聲。

**旁白**：一步、一步。

**旁白**：那種壓迫感，正從黑暗中緩緩逼近。

**李杰**：（瞳孔驟然收縮，緩緩站起身，橫劍於身前）

**李杰**：（內心，咬牙切齒）又是他。

**旁白**：在隧道的盡頭，一個黑色的身影緩緩浮現。

**旁白**：那是穿著黑衣、握著黑劍的男人。

**旁白**：在這條通往未知的軌道上，最後的追逐，才剛剛開始。
