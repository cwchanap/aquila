# 第二十三幕：銀月湖的悠日

**旁白**：在前往神樹的路上,李杰和明日香在湖邊的一處空地上停了下來。

**旁白**：這裡視野開闊,可以清晰地看到整個銀月湖。湖面平靜如鏡,倒映著天空中的雙月。

**李杰**：我們在這裡休息一下吧。

**鈴木明日香**：好啊。

**旁白**：兩人坐下,看著湖面。

**鈴木明日香**：這裡真的很美。如果沒有那些危險的話,我真的想一直待在這裡。

**李杰**：(淡淡地)美景往往伴隨著危險。

**鈴木明日香**：(笑了笑)李先生,你總是這麼冷靜。

**李杰**：如果不冷靜,我們可能早就不在這裡了。

**鈴木明日香**：(點頭)也對。

**旁白**：明日香從背包裡拿出一些果實,遞給李杰。

**鈴木明日香**：給,這是我在村子裡摘的。

**李杰**：(接過果實,嚐了一口)味道不錯。

**鈴木明日香**：是吧？莉拉教我怎麼挑選熟透的。

**旁白**：兩人坐在草地上,享受著這難得的寧靜時光。

**鈴木明日香**：李先生,你還記得新宿站嗎？

**李杰**：記得。

**鈴木明日香**：我記得當時月台上好吵。那群大學生一直在笑,還有個眼鏡男一直在滑手機⋯⋯

**李杰**：(沈默了一下)嗯。

**鈴木明日香**：如果能回到那天晚上⋯⋯妳會上這輛車嗎？

**李杰**：(想了想)我不知道。

**鈴木明日香**：我會。

**李杰**：(有些意外)為什麼？

**鈴木明日香**：(看向李杰,微微一笑)因為如果我不上車,我就不會遇到你了。

**李杰**：(愣住了)明日香⋯⋯

**鈴木明日香**：(有些害羞地別過臉)雖然這個世界很可怕,但能遇到你,我覺得很幸運。

**李杰**：(輕聲說)我也一樣。

**旁白**：兩人在湖邊坐了很久,直到銀藍色的光芒漸漸變強。

**李杰**：我們走吧。

**鈴木明日香**：嗯。

**旁白**：李杰站起身,向明日香伸出手。

**鈴木明日香**：(握住他的手)李先生,等我們回去了⋯⋯你有什麼想做的事嗎？

**李杰**：我想請妳去吃頓大餐。

**鈴木明日香**：(開心地笑)好啊！那我要吃壽司！

**李杰**：沒問題。

**鈴木明日香**：那我們拉鉤。

**李杰**：(笑了笑,伸出手指)好,拉鉤。

**旁白**：兩人的小指勾在一起。

**鈴木明日香**：約好了喔。

**李杰**：約好了。

**旁白**：新的希望在心中萌芽。雖然前方依然充滿未知,但至少在這一刻,他們相信自己一定能找到回家的方法⋯⋯
