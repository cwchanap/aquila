# 第十幕：霧中棲所

**旁白**：李杰帶著明日香在森林裡又走了一個多小時。霧氣依然濃厚，但前方的樹木開始變得稀疏，隱約能看到一點光亮。

**鈴木明日香**：（緊張地抓住李杰的衣袖）李先生⋯⋯你看，前面好像有房子。

**李杰**：（點頭，手握緊了鏽劍）嗯，我看見了。小心點。

**旁白**：兩人放慢腳步，慢慢靠近。霧氣中，一棟石造的小屋緩緩浮現。屋子不大，牆壁是由乳白色的石塊砌成，在紫色月光下散發著淡淡的微光。

**李杰**：（在門前停下，側耳傾聽）

**旁白**：屋內一片寂靜。沒有燈光，也沒有人聲。

**李杰**：（輕輕推了推門把）

**旁白**：門沒鎖。

**旁白**：他示意明日香留在外面，自己先舉著劍走了進去。

**旁白**：屋子裡陳設簡單而雅致。一張圓形的石桌，兩把木質長椅。牆邊有一個小小的壁爐，裡面還殘留著一些餘燼，散發出微弱的熱量。

**李杰**：（轉了一圈，確認沒有危險後，走到門口招手）進來吧。

**鈴木明日香**：（戰戰兢兢地走進屋子，看到壁爐，眼睛亮了一下）好溫暖⋯⋯

**李杰**：看來這裡不久前還有人待過。

**旁白**：他走到壁爐前，添了幾塊木柴，火光漸漸亮了起來，照亮了整個客廳。

**鈴木明日香**：（坐在長椅上，整個人放鬆了一些）李先生，你說⋯⋯這房子的主人是誰？會是那些乘客嗎？

**李杰**：（搖頭）不，這裡的建築風格很不尋常。妳看那些裝飾和符號，不像是現代人的審美。

**旁白**：他指了指牆上刻著的一些像藤蔓又像文字的圖案。

**李杰**：反而更像是⋯⋯某種古老文明留下的遺跡。

**鈴木明日香**：（好奇地湊過去看）精靈？

**李杰**：（愣了一下）精靈？

**鈴木明日香**：（點頭，從隨身的包包裡掏出一本泛黃的冊子）這是我在剛才路過的一間廢棄房子裡找到的。虽然文字很奇怪，但我竟然能看懂一部分。這上面寫著關於「精靈族」的事。

**李杰**：（接過冊子，翻開看了看）妳能看懂？

**鈴木明日香**：（不好意思地笑了笑）嗯，我也覺得奇怪。可能是因爲我學過一些古文字研究吧？這上面的字符和某些失傳的語系很像。

**李杰**：（沉思）這本冊子先放妳那裡。妳有時間多研究一下，這可能是我們理解這個世界的關鍵。

**鈴木明日香**：（認真地點頭）好。

**旁白**：李杰站起身，在屋子裡搜索了一番。他在後面的小房間裡找到了一些乾淨的衣物和毛巾，還有一間帶有浴缸的浴室。

**李杰**：鈴木小姐，這裡有浴室和衣服。妳要不要先去整理一下？

**鈴木明日香**：（驚喜地叫出聲）真的嗎？太好了！我感覺自己快要發霉了！

**旁白**：她興沖沖地接過衣服進了浴室。不一會兒，裡面傳來了嘩啦啦的水聲。

**李杰**：（坐在壁爐前，看著跳動的火焰，內心）精靈族⋯⋯如果這本冊子是真的，那這個世界可能比我想象的還要瘋狂。

**旁白**：大約半小時後，浴室門開了。明日香穿著一件寬大的銀白色長袍走了出來，頭髮濕漉漉的披在肩上，臉頰因爲熱水的浸泡而顯得紅潤。

**鈴木明日香**：（有些侷促地抓著衣角）那个⋯⋯李先生，我洗好了。衣服有點大，不過挺舒服的。

**李杰**：（迅速別過臉，心跳莫名加速）咳，妳沒事就好。既然有熱水，我也去沖一下。

**旁白**：李杰走進浴室，簡單地沖洗了一下。溫暖的水流帶走了多日的疲憊，讓他感到神清氣爽。

**旁白**：當他換上同樣款式的長袍走出浴室時，看見明日香已經在石桌上擺好了幾樣東西。

**鈴木明日香**：（笑著招手）李先生快來！我在廚房的櫃子裡找到了一些看起來像乾果和肉乾的東西，味道竟然還不錯！

**李杰**：（走過去坐下，嚐了一塊乾果）嗯，甜甜的。

**旁白**：兩人在溫暖的火光下，分享著這頓簡單卻無比珍貴的晚餐。屋外的迷霧和危險彷彿被擋在了那扇石門之外。

**旁白**：兩個陌生人,因為命運的安排相遇,在這座霧中的小屋裡,共享著一份脆弱而珍貴的溫暖。
