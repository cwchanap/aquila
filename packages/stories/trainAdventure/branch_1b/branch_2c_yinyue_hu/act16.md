# 第十六幕：血染的村落

**旁白**：李杰和明日香在森林裡走了一個多小時。隨著他們的深入，原本茂密的樹林漸漸變得稀疏，前方隱約出現了一些建築的輪廓。

**鈴木明日香**：（緊張地小聲問）李先生⋯⋯那裡就是精靈的村落嗎？

**李杰**：（握緊鏽劍，壓低聲音）應該是。保持安靜。

**旁白**：兩人放慢腳步，藉著紫色月光的掩護，慢慢靠近。

**旁白**：這是一座規模不大的村莊。建築風格與之前的小屋相似，都是由白色岩石和活生生的樹木交織而成，透出一種自然而神祕的美感。

**旁白**：然而，當他們踏入村莊範圍的那一刻，一股濃重的血腥味撲面而來。

**鈴木明日香**：（驚叫一聲，捂住口鼻）咳⋯⋯好臭！

**李杰**：（臉色驟變，一把將她拉到身後）小心！

**旁白**：眼前的景象慘絕人寰。

**旁白**：原本寧靜的廣場上，橫七豎八地躺著幾十具屍體。那些屍體有著修長的四肢、尖尖的耳朵、以及精緻的五官——正是傳說中的精靈。

**旁白**：他們的銀藍色血液在大地上流淌，在紫月的映照下閃爍著詭異的光芒，宛如一條死亡之河。

**鈴木明日香**：（臉色慘白，雙腿劇烈顫抖）怎⋯⋯怎麼會這樣⋯⋯

**李杰**：（強忍著胃部的翻騰，警惕地環視四周）看來這裡剛發生過一場大屠殺。

**旁白**：他在廣場中央停下。那裡矗立著一尊巨大的女性精靈石像，石像的頭部已經碎裂，半邊身體被燒得焦黑。

**旁白**：李杰注意到，這些精靈的死狀都非常悽慘。有的被利刃穿心，有的被箭矢貫穿，還有的身體呈現出不自然的扭曲。

**李杰**：（走近一具屍體，蹲下查看）傷口很新鮮⋯⋯襲擊者可能剛離開不久。

**旁白**：就在這時，廣場另一側的屋子裡傳來了一聲輕微的呻吟。

**李杰**：（瞬間站起身，橫劍於身前）誰！

**旁白**：一個渾身是血的小精靈從門口爬了出來。他的右腿被箭射穿，臉上滿是恐懼和絕望。

**小精靈**：（用微弱的、李杰聽不懂的語言哀求著）⋯⋯艾拉如⋯⋯艾拉如大人⋯⋯

**鈴木明日香**：（看到受傷的孩子，母性本能戰勝了恐懼，衝了過去）李先生！他在求救！

**李杰**：（猶豫了一下，還是跟了上去）小心陷阱。

**旁白**：明日香蹲在小精靈身邊，從口袋裡拿出僅剩的一點繃帶，試圖幫他止血。

**鈴木明日香**：（溫柔地安慰著）別怕⋯⋯沒事了⋯⋯

**旁白**：小精靈看著這兩個外來者，眼神中閃過一絲疑惑，但更多的是深深的哀痛。他伸出顫抖的手，指向村莊後方最高的那棟房子。

**小精靈**：（吐出最後一口氣，聲音微弱得幾乎聽不見）⋯⋯艾拉如⋯⋯快救⋯⋯

**旁白**：話音剛落，小精靈的眼神漸漸涣散，手臂無力地垂落在血泊中。

**鈴木明日香**：（愣在那裡，淚水奪眶而出）不⋯⋯

**李杰**：（站起身，目光深沈地望向那棟大房子）艾拉如。看來那就是我們要找的人。

**旁白**：他想起冊子上提到的「精靈領袖艾拉如」。

**李杰**：（內心）如果他還活著，或許能告訴我們發生了什麼。

**旁白**：然而，當他們推開那棟大房子的門時，裡面的景象讓李杰的心沈到了谷底。

**旁白**：房間中央的椅子上，坐著一個身材高大、面容威嚴的老年精靈。他身上穿著華麗的祭袍，但胸口插著一把黑色的長劍。

**旁白**：黑色血液順著祭袍流下，染黑了地面上的符號。

**李杰**：（瞳孔縮小）艾拉如⋯⋯

**旁白**：在艾拉如的屍體旁，還躺著一個讓李杰感到心驚的身影。

**旁白**：那是那個總是帶著溫和笑容、給人可靠感覺的慷慨大叔。

**旁白**：他的臉上還帶著平靜的表情,彷彿只是睡著了。但胸口那道致命的傷痕,清楚地說明了他的命運。

**李杰**：(低聲)是他⋯⋯那位大叔⋯⋯

**鈴木明日香**：(驚恐地)他⋯⋯他是在車站下車的那個人⋯⋯

**李杰**：是的⋯⋯他應該和其他乘客一起行動⋯⋯但為什麼只有他一個人來到這裡?

**旁白**：他小心翼翼地將白布蓋回去,退後幾步。

**旁白**：明日香的眼眶泛紅,李杰則緊緊握著拳頭。

**李杰**：(低聲)他和我們一樣,只是普通人。只是想活下去,想回家⋯⋯卻在這裡失去了生命。

**莉拉**：(低聲)他很勇敢。在襲擊發生時,他沒有逃跑。他和艾拉如大人一起戰鬥,想要保護我們⋯⋯

**莉拉**：你們⋯⋯認識他嗎?

**李杰**：算是吧。我們曾經搭乘同一輛列車⋯⋯只是沒想到,會在這種情況下再見面。

**旁白**：莉拉點點頭,眼中滿是悲傷與敬意。

**旁白**：廣場上的精靈們開始吟唱起低沉的歌謠,那是精靈族的安魂曲。歌聲如同微風,溫柔而悲傷,在村莊上空迴盪。

**旁白**：明日香雖然聽不懂歌詞,但那旋律中的悲傷卻是如此清晰。她的眼淚不自覺地流了下來。

**旁白**：李杰和明日香靜靜地站在一旁,看著那些覆蓋著白布的遺體。

**旁白**：天空中的雙月依然冷漠地懸掛著,銀藍色的光芒灑在這片哀傷的土地上。

**旁白**：他們本以為來到月光林地能找到答案,能找到艾拉如,能了解更多關於這個世界的秘密。但現在,他們只看到了死亡和悲傷。

**旁白**：而那個掌握著關鍵資訊的精靈領袖,已經永遠無法回答他們的問題了。

**旁白**：李杰看向遠方的森林,心中充滿了疑問。那個持黑劍的襲擊者⋯⋯他是誰?為什麼要襲擊這個和平的村莊?

**旁白**：而其他和大叔一起下車的乘客們⋯⋯他們現在在哪裡?

**旁白**：一個個謎團,在這片銀藍色的光芒中,變得更加深沉。
