# 第十七幕：艾拉如的計劃

**旁白**：葬禮結束後，精靈們緩緩散去。銀藍色的光芒依然籠罩著村莊，但空氣中的悲傷卻更加濃重。

**旁白**：李杰看著那些精靈們離開廣場，然後轉身走向莉拉。

**李杰**：莉拉，我能和妳單獨談談嗎？

**旁白**：莉拉愣了一下，眼神中閃過一絲不安，但很快她點了點頭。

**莉拉**：好⋯⋯請跟我來。

**旁白**：她帶著李杰和明日香沿著一條小徑，走向村莊邊緣。他們來到一片被巨樹環繞的空地，這裡很安靜，沒有其他精靈。

**李杰**：莉拉，妳真的相信艾拉如說的話嗎？

**莉拉**：我相信他。艾拉如大人是我們的領袖，他帶領我們在這個世界生存了那麼久。他從來沒有背叛過我們，從來沒有對我們說過謊。

**莉拉**：如果他說妳的朋友會沒事，那她就一定會沒事。

**李杰**：但他已經死了。如果他的計劃還沒完成⋯⋯

**莉拉**：我知道⋯⋯艾拉如大人已經不在了。但我相信，他一定已經安排好了一切。

**旁白**：李杰看著莉拉那堅定的眼神，心中卻更加困惑。

**李杰**：（對明日香說）明日香，妳問問她，艾拉如提到的「那個計劃」到底是什麼？

**鈴木明日香**：（愣了一下，然後點頭，嘗試用意義和手勢與莉拉溝通）⋯⋯那個⋯⋯艾拉如⋯⋯計劃？

**旁白**：莉拉似乎理解了明日香的意思。她沈默了片刻，然後緩緩開口。

**莉拉**：艾拉如大人一直希望能找到一個能讓我們所有人離開這個世界的方法。

**莉拉**：他說這個世界是一個牢籠，我們是被詛咒的種族。唯有通過「終焉祭壇」的考驗，我們才能回到真正的家園。

**李杰**：（心中一震）終焉祭壇？地圖上那個地方？

**莉拉**：（點頭）是的。但祭壇需要「引導者」和「祭品」。

**李杰**：（眼神變得銳利）祭品？

**莉拉**：（眼神有些躲閃）我不知道具體細節。大人只說，當列車再次到來時，時機就成熟了。

**李杰**：（內心）列車⋯⋯艾拉如早就知道我們會來？

**李杰**：那他爲什麼要把乘客分散開？

**莉拉**：他說是爲了測試每個人。每個人都要選擇自己的道路。

**旁白**：李杰深吸一口氣，腦海中的疑問越來越多。

**旁白**：艾拉如認識那個黑劍襲擊者。艾拉如知道列車的存在，知道會有乘客來。他刻意分散乘客，卻又說會有人去救治受傷者。

**旁白**：這一切背後，到底隱藏著什麼樣的真相？

**李杰**：還有其他妳能告訴我們的嗎？任何艾拉如提到過的事情？

**莉拉**：對不起⋯⋯艾拉如大人真的沒有告訴我更多。他總是獨自承擔一切，不願意讓我們知道太多。

**李杰**：我明白了。謝謝妳告訴我們這些。

**莉拉**：我只是按照命令行事。

**旁白**：明日香雖然聽不懂她們的對話，但能感受到談話已經接近尾聲。

**鈴木明日香**：李先生⋯⋯她還說了什麼嗎？

**李杰**：（對明日香說）艾拉如沒有告訴她更多。但她說相信艾拉如一定安排好了一切。

**鈴木明日香**：我只希望⋯⋯美咲還活著⋯⋯

**旁白**：莉拉說完，轉身離開了這片空地。只剩下李杰和明日香站在原地，看著那兩輪紫色的月亮。

**鈴木明日香**：李先生⋯⋯你相信她說的話嗎？

**李杰**：我不知道。但至少⋯⋯我們現在知道了一些線索。艾拉如早就知道列車的存在，知道我們會來。他有某種計劃，但現在他已經死了，我們可能永遠無法知道全部的真相。

**鈴木明日香**：那我們接下來該怎麼辦？

**李杰**：先在村莊休息，補充物資。然後⋯⋯我們需要想辦法找到其他的倖存者，或者找到離開這個世界的方法。

**李杰**：莉拉給的這張地圖⋯⋯也許能幫我們找到美咲的下落。

**鈴木明日香**：如果美咲真的還活著⋯⋯我一定要找到她。

**李杰**：我們會的。但現在，我們需要保持冷靜，一步一步來。

**旁白**：兩人並肩站在樹下，看著遠方黑暗的森林。

**旁白**：李杰將莉拉畫的地圖小心地收進口袋，那上面標記的路線，也許能引領他們找到答案。

**旁白**：艾拉如的計劃依然是個謎。那個持黑劍的襲擊者依然下落不明。而在這片充滿危險的異世界裡，他們必須找到活下去的方法。

**旁白**：以及⋯⋯回家的希望。
