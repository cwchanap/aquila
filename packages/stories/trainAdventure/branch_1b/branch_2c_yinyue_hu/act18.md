# 第十八幕：殘酷的真相

**旁白**：清晨——雖然這個世界沒有真正的清晨，但李杰和明日香還是習慣用這個詞來標記他們休息後的醒來時刻。

**旁白**：兩人在月光林地的客房中整理好行裝，準備出發前往莉拉所說的襲擊地點。

**旁白**：村莊裡的精靈們依然籠罩在哀悼的氣氛中，但生活還是要繼續。

**旁白**：李杰將莉拉給的地圖仔細收好，確認腰間的鏽劍依然別在原位。

**李杰**：準備好了嗎？

**鈴木明日香**：嗯⋯⋯我準備好了。

**旁白**：明日香的眼神中帶著堅定，但也隱藏著一絲不安。她不知道在那個地方會發現什麼，但她必須去尋找答案。

**旁白**：他們來到村莊邊緣的一間小屋。莉拉推開門,裡面停放著兩具覆蓋著白布的遺體。

**莉拉**：就是他們⋯⋯我們在迷霧之森西側發現的。

**旁白**：李杰深吸一口氣,走上前。他伸手掀開第一具遺體的白布。

**旁白**：那是一張年輕女性的臉,長長的黑髮散落在一旁,和血跡混在一起。她穿著淺米色的連衣裙和素色外套,臉色蒼白如紙,眼睛緊閉。

**旁白**：李杰認出了她——那個在車站和他們一起下車的文靜少女。

**李杰**：是那個⋯⋯文靜的女孩⋯⋯

**旁白**：他掀開第二具遺體的白布。一個戴著眼鏡的年輕男性,胸口有明顯的傷口,臉上帶著驚恐的表情。

**李杰**：還有⋯⋯那個膽小的眼鏡男⋯⋯

**旁白**：明日香看到不是健太,整個人鬆了一口氣。但隨即,悲傷和愧疚感湧上心頭。

**鈴木明日香**：那個女孩⋯⋯還有那個眼鏡男⋯⋯

**旁白**：她的聲音哽咽,眼淚再次流下。

**旁白**：李杰蹲下來仔細查看兩人的傷口,眉頭緊鎖。

**李杰**：這些傷口⋯⋯是劍傷。美咲脖子上的傷口，是一刀斃命。傷口很整齊，應該是用非常鋒利的劍造成的。

**李杰**：這個男孩胸口的貫穿傷⋯⋯也是劍造成的。

**旁白**：他停頓了一下，神情變得更加凝重。

**李杰**：這種傷口⋯⋯和我之前看到的精靈劍士傷口很相似。

**鈴木明日香**：是⋯⋯是那個黑劍男嗎？

**李杰**：很有可能。這種俐落的傷口，還有這樣的貫穿力⋯⋯在月光林地，我看到犧牲的精靈劍士們身上也有類似的傷口。應該就是同一個人造成的。

**旁白**：明日香的眼淚止不住地流下來，身體開始顫抖。

**鈴木明日香**：為什麼⋯⋯為什麼會這樣⋯⋯美咲⋯⋯她只是想活下去⋯⋯我們只是想回家⋯⋯

**旁白**：她的聲音哽咽，雙腿一軟，跪倒在地上。李杰走到她身邊，輕輕扶住她的肩膀。

**李杰**：明日香⋯⋯

**鈴木明日香**：我以為⋯⋯我以為美咲被艾拉如救走了⋯⋯莉拉說艾拉如會派人去救她⋯⋯可是⋯⋯可是為什麼⋯⋯

**旁白**：她的身體劇烈顫抖著，眼淚不停地流下。

**李杰**：也許⋯⋯艾拉如確實派人來救她了，但那個黑劍男也來了。從傷口來看，他們應該是在這裡遇到了襲擊。

**旁白**：李杰環顧房間，試圖尋找更多線索。地板上有打鬥的痕跡，一些家具被推倒，牆上也有劍痕。

**李杰**：這裡發生過戰鬥。看這些痕跡，應該有人試圖保護他們。但⋯⋯還是沒能成功。

**鈴木明日香**：拓海⋯⋯他為什麼會在這裡⋯⋯

**李杰**：也許他和美咲在一起。也許他試圖保護她。

**旁白**：李杰看著拓海的遺體，那個沉穩的年輕人，即使在生命的最後一刻，依然選擇了保護同伴。

**李杰**：他很勇敢。

**旁白**：明日香的哭聲在小小的房間裡迴盪。李杰靜靜地陪在她身邊，給她時間發洩悲痛。

**旁白**：過了許久，明日香的哭聲漸漸平息。

**鈴木明日香**：我們⋯⋯我們不能就這樣離開他們⋯⋯

**李杰**：我知道。我們會好好安葬他們。

**旁白**：兩人走出房屋，在附近找到一片較為平坦的空地。李杰用劍挖掘泥土,明日香則用雙手幫忙清理石塊。

**旁白**：他們沒有說話，只是默默地工作著。挖掘了兩個墓穴後，他們小心翼翼地將美咲和拓海的遺體搬出來。

**李杰**：(註：此處修正，死者應為文靜少女和眼鏡男)

**旁白**：李杰將他們輕輕放入墓穴中，然後開始填土。明日香跪在墓前，雙手合十，低聲祈禱。

**鈴木明日香**：願你們⋯⋯在另一個世界能找到平安⋯⋯

**旁白**：兩座簡陋的墳墓，在紫色月光的照耀下，顯得格外孤寂。

**旁白**：埋葬完畢後，兩人在墓前又站了許久。明日香的眼睛已經哭得紅腫，但她努力控制著情緒。

**李杰**：我們該走了。

**鈴木明日香**：嗯⋯⋯

**旁白**：她最後看了一眼那兩座墳墓，然後跟著李杰離開。當他們走回交叉路口時，明日香突然停下腳步。

**鈴木明日香**：李先生⋯⋯我們還要去右邊那棟房子看看嗎？

**李杰**：⋯⋯要。也許那裡會有其他線索。

**鈴木明日香**：那我們走吧。

**旁白**：她深吸一口氣，努力讓自己堅強起來。兩人轉向右側，朝著另一棟房屋走去。

**旁白**：天空中的雙月依然冷漠地注視著這片土地。而在這片永恆的黑暗中，死亡的陰影，似乎無處不在。
