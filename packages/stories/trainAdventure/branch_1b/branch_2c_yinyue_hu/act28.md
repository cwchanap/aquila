# 第二十八幕：真相的重量

**旁白**：浩介看著明日香,眼中滿是愧疚。

**鈴木浩介**：姊,對不起。我剛才忘記告訴妳們一件很重要的事了。

**旁白**：他放下茶杯,認真地看著李杰和明日香。

**鈴木浩介**：你們剛才問我姊姊失蹤多久了,我說三十八年。

**鈴木浩介**：但你們可能還不知道⋯⋯現在是什麼年份。

**旁白**：李杰的心臟開始加速跳動。一種不祥的預感湧上心頭。

**李杰**：(聲音低沈)現在是哪一年？

**鈴木浩介**：(沈默了一會兒,緩緩說道)現在是二零六四年。

**旁白**：李杰和明日香同時愣住了。

**鈴木明日香**：(尖叫)二零六四年？！

**李杰**：(難以置信)怎麼可能⋯⋯我們明明是在二零二六年上車的⋯⋯

**鈴木浩介**：(點頭)是的。這就是我為什麼說你們失蹤了三十八年。

**旁白**：李杰的大腦飛快地運轉著。二零二六年到二零六四年⋯⋯真的是三十八年。

**鈴木明日香**：(哭著說)三十八年⋯⋯我失蹤了十八年⋯⋯(註：此處可能是角色情緒激動下的口誤或筆誤，應為三十八年)

**旁白**：她的聲音顫抖得厲害。

**鈴木明日香**：那⋯⋯那我爸爸媽媽⋯⋯他們⋯⋯

**鈴木浩介**：鈴木先生和鈴木太太都還好。他們一直在等你回來。

**旁白**：聽到這句話,明日香鬆了一口氣。但浩介的目光轉向李杰,眼神帶著一絲歉意。

**鈴木浩介**：李先生⋯⋯你母親還好,但你父親⋯⋯

**鈴木浩介**：八年前在一場意外中過世了。很抱歉。

**旁白**：李杰的身體微微一顫。

**李杰**：⋯⋯我知道了。

**旁白**：他的聲音很平靜,但明日香能感受到他手心微微發涼。她輕輕握住李杰的手,給他無聲的支持。李杰轉頭看向她,眼神中帶著感激。

---

**旁白**：明日香深吸一口氣,眼淚滑落臉頰。

**鈴木明日香**：十八年⋯⋯弟弟現在都三十三歲了⋯⋯(註：再次出現十八年/三十三歲的邏輯，可能角色在說話時還在混亂中)

**鈴木明日香**：我錯過了他的成長⋯⋯錯過了這麼多⋯⋯

**旁白**：李杰緊握著她的手,但他自己也感到一陣眩暈。

**旁白**：十八年⋯⋯他應該四十六歲了。

**旁白**：父親已經不在了,母親現在該有多老？他的朋友、同事,所有人的生活都繼續向前。只有他們,被困在時間之外。

---

**旁白**：浩介站起身,走到一旁的櫃子前。他拿出一盒面紙,遞給明日香。

**鈴木浩介**：我知道這很難接受。但至少⋯⋯你們回來了。這比我姊姊幸運多了。

**旁白**：他的聲音帶著一絲苦澀。

**鈴木浩介**：她失蹤的時候,我才十五歲。等了三十八年,我從少年變成中年人。而她⋯⋯永遠停留在二十歲。

**旁白**：客廳裡只剩下明日香的哭泣聲和窗外鳥兒的鳴叫。陽光依然溫暖地灑進來。但在這溫暖的光線中,卻藏著無法彌補的時光裂痕。

**旁白**：十八年的空白,就這樣橫亙在他們與這個世界之間。

**旁白**：李杰看著窗外的陽光,心中感到一陣莫名的荒涼。

**李杰**：(內心)這就是回家的代價嗎⋯⋯

**旁白**：他想起神樹的話。

**旁白**：兩人在客廳裡坐了很久。浩介一直靜靜地陪著他們。

**鈴木浩介**：休息一下吧。我已經安排好了旅館。

**鈴木明日香**：(點頭,聲音微弱)好⋯⋯

**旁白**：李杰也站起身。他知道,現在最需要的是冷靜。

**李杰**：謝謝你,浩介。

**鈴木浩介**：(拍了拍李杰的肩膀)別客氣。我們是家人。

**旁白**：家人。

**旁白**：這個詞讓李杰感到一陣溫暖。雖然父親已經不在了,但至少他還有母親。還有明日香。

**旁白**：他們會一起努力。

**旁白**：重新開始新的生活。

**旁白**：一定會的。
