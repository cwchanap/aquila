# 第三十一幕：旅館的悠日

**旁白**：在旅館的日子過得很快。

**旁白**：李杰和明日香每天都會一起去泡溫泉,一起在山間散步。

**旁白**：雖然身處在這個充滿未知的世界,但兩人的心卻前所未有地貼近。

**鈴木明日香**：(走在林間小徑上,看著李杰)杰君,你在想什麼？

**李杰**：(轉頭看向她,微笑)我在想,如果我們能在這裡一直待下去,好像也不錯。

**鈴木明日香**：(笑了笑)我也這麼想過。但我們還有家人在等我們,不是嗎？

**李杰**：(點頭)嗯。家人最重要。

**鈴木明日香**：那你呢？

**李杰**：我也要去看母親。還要處理很多事情。

**李杰**：工作、住處⋯⋯可能要花不少時間。

**鈴木明日香**：那⋯⋯我們多久能再見面？

**旁白**：明日香有些不安地問道。

**李杰**：應該不會太久。一個月？兩個月？

**李杰**：等我們都安頓好了，我就去找妳。

**鈴木明日香**：嗯⋯⋯那我等你。

**旁白**：明日香伸出小指。

**鈴木明日香**：拉勾勾，不許反悔。

**旁白**：李杰笑著，也伸出小指和她勾在一起。

**李杰**：好，拉勾勾。

---

**旁白**：那天晚上，兩人最後一次一起泡溫泉。

**旁白**：在庭院的長椅上，他們並肩坐著看星空。

**旁白**：沒有說太多話，只是靜靜地陪伴著彼此。

**旁白**：手牽著手，感受著彼此的溫度。

**旁白**：這一週的時光，對他們來說太過珍貴。

**旁白**：在這個遠離塵囂的溫泉小鎮，他們找到了內心的平靜。

**旁白**：也確認了彼此的心意。

**旁白**：明天，他們就要離開這裡，回到各自的生活。

**旁白**：但他們知道，這不是結束。

**旁白**：而是新的開始。

---

**旁白**：就這樣，一週的時間很快過去了。
