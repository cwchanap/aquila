# 第三十二幕：告別與叮囑

**旁白**：離開的那天，天氣晴朗。

**旁白**：浩介親自開車送他們前往機場。

**鈴木浩介**：(握著方向盤,認真地說)李先生,明日香姊,我再叮囑一次。

**鈴木浩介**：關於那個世界的事情,請務必守口如瓶。

**鈴木浩介**：這不僅是為了妳們的安全,也是為了妳們家人的安全。

**李杰**：我們明白。

**鈴木明日香**：嗯。

**鈴木浩介**：我已經為你們安排好了新的身份和背景。

**鈴木浩介**：李先生,你的新身份是一位在國外工作多年的工程師,因為身體原因回國修養。

**鈴木浩介**：姊,妳的新身份是在北海道一所偏遠小學任教的老師,因為學校倒閉而回到家中。

**鈴木浩介**：這些資料我都已經放進你們的行李箱裡了。請務必背熟。

**李杰**：謝謝你,浩介。

**鈴木浩介**：(微微一笑)不客氣。我們是家人。

**鈴木浩介**：(看了後照鏡一眼)還有,如果你們發現有任何不對勁的地方,或者感覺被人跟蹤⋯⋯

**鈴木浩介**：立刻聯繫我。

**李杰**：好。

**鈴木明日香**：浩介⋯⋯謝謝你。

**鈴木浩介**：(沈默了一會兒,輕聲說)姊,妳能回來,我真的很開心。

**鈴木明日香**：(眼眶泛紅)我也是。

**李杰**：我明白。如果能幫上忙，我們很願意配合。

**鈴木明日香**：是的。如果有我們能做的⋯⋯

---

**旁白**：浩介的表情變得更加嚴肅。

**清水浩介**：還有很重要的一點。

**清水浩介**：請不要對任何人提起那個世界的事。

**旁白**：李杰和明日香都認真地看著浩介。

**清水浩介**：包括你們的家人、朋友，任何人都不要說。

**清水浩介**：如果有人問起你們這段時間去了哪裡⋯⋯

**清水浩介**：就說你們在靜養院接受治療。

**清水浩介**：這不是強制的。
