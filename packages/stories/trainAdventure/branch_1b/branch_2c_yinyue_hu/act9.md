# 第九幕：銀月之途

**溫柔母親**：（看向李杰）你⋯⋯要跟我們一起走嗎？

**旁白**：李杰沉默地看著眼前這對母子三人。母親膝蓋上的傷口還在滲血，兩個孩子緊緊抓著她的手，眼神中滿是恐懼與不安。

**李杰**：（內心）跟著他們去亡者峽谷？一個受傷的母親，兩個年幼的孩子⋯⋯在這種鬼地方，這只會成為拖累。

**旁白**：他想起剛才在車站內找到的地圖。三條路線清晰地標示著——北邊的「亡者峽谷」、南邊的「迷霧之森」、還有西邊的「銀月湖」。

**李杰**：（內心）我們才剛認識不到幾個小時，說到底只是陌生人。在這種隨時可能喪命的情況下，帶著他們只會讓我自己也陷入危險。

**旁白**：李杰深吸一口氣，做出了決定。他從口袋裡掏出那張泛黃的地圖，展開給那位母親看。

**李杰**：（平靜地說）這是我在車站裡找到的地圖。沿著鐵軌走確實能到亡者峽谷，但那條路完全暴露在外，沒有任何遮蔽。

**溫柔母親**：（抬起頭，眼神中閃過一絲希望）那⋯⋯還有別的路嗎？

**李杰**：（指著地圖上的一條細線）這裡有一條小路，雖然繞了一點，但穿過了一片森林。在那裡走，至少比在鐵軌上安全得多。

**溫柔母親**：（感激地）謝謝你⋯⋯真的太謝謝你了。你⋯⋯你不跟我們一起去嗎？

**李杰**：（搖了搖頭，語氣堅定）我打算去西邊的「銀月湖」。地圖上顯示那裡可能會有更多線索。我們就在這裡分開吧。

**旁白**：那位母親的眼神暗淡了下去，但她還是點了點頭，勉強露出一個微笑。

**溫柔母親**：我明白了。李先生，謝謝你剛才救了我們。請你⋯⋯請你也一定要保重。

**李杰**：（點頭）妳也是。帶著孩子，小心一點。

**旁白**：李杰沒有再停留，他握緊手中的鏽劍，轉身朝著西邊的森林走去。身後，傳來了母子三人踩在碎石上的腳步聲，漸漸遠去。

**旁白**：李杰走進了西邊的森林。這裡的霧氣比車站周圍還要濃，兩輪紫色的殘月在樹梢間時隱時現，將森林染成一片詭異的紫灰色。

**李杰**：（內心）呼⋯⋯總算是一個人了。雖然有點愧疚，但這才是最理智的選擇。在這種地方，先活下去才是最重要的。

**旁白**：他沿著地圖上標示的小徑緩慢前進。森林裡一片死寂，只有他自己的腳步聲和偶爾傳來的樹枝斷裂聲。

**旁白**：走了大約一個小時，前方的霧氣中突然出現了一個模糊的身影。

**李杰**：（立刻停下腳步，橫劍於身前，壓低聲音）誰！

**旁白**：那身影似乎也被嚇了一跳，猛地停住了。

**神祕聲音**：（帶著顫抖的哭腔）誰、是誰在那裡？

**李杰**：（愣了一下，這個聲音聽起來很耳熟）你是⋯⋯？

**旁白**：那身影緩緩靠近，在微弱的月光下，李杰認出了那張臉。

**李杰**：（驚訝）鈴木小姐？

**鈴木明日香**：（看到李杰，整個人差點癱倒在地上，哇地一聲哭了出來）李先生！真的是你！太好了⋯⋯嗚嗚嗚⋯⋯我還以為我要死在這裡了⋯⋯

**旁白**：鈴木明日香，那个在新宿月台上和朋友們走在一起的、活潑好動的大學生。此刻她穿著一件淡藍色的連衣裙，長髮散亂，裙擺上沾滿了泥土和乾枯的草葉。

**李杰**：（收起劍，走上前）妳怎麼會在這裡？妳不是跟著那群學生進森林了嗎？

**鈴木明日香**：（一邊抹眼淚一邊哽咽）我⋯⋯我和他們走散了。霧太大了，大家都在跑，我跟不上⋯⋯然後我就一個人迷路了⋯⋯

**鈴木明日香**：（抽泣著）我好害怕⋯⋯到處都是怪物的叫聲⋯⋯我一直跑，不知道跑了多久，最後就看到你了⋯⋯

**李杰**：（看著眼前這個驚魂未定的女孩，嘆了口氣）好了，別哭了。既然遇到了，就先跟著我吧。

**鈴木明日香**：（點頭，死死抓住李杰的衣袖，像是抓住了救命稻草）嗯⋯⋯請千萬不要丟下我⋯⋯求你了⋯⋯

**李杰**：（有些無奈）我知道了。走吧，我們得找個安全的地方休息。

**旁白**：兩人並肩走向森林深處。明日香依然不停地抽泣，李杰則保持著沈默，目光警惕地掃視著四周。

**旁白**：而在森林的某處，那雙漆黑的眼睛依然在冷冷地注視著他們。
