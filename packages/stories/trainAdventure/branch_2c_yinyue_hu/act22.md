# 第二十二幕：停止搜尋

**旁白**：告別之後,兩人的心情依然沈重,但至少明日香不再像之前那樣崩潰了。

**鈴木明日香**：李先生,我們接下來該怎麼辦？

**李杰**：我打算去神樹那裡看看。莉拉說神樹能給我們答案。

**鈴木明日香**：答案⋯⋯是回家的答案嗎？

**李杰**：(點頭)希望是。

**鈴木明日香**：那我也去。

**李杰**：可是妳的身體⋯⋯

**鈴木明日香**：(打斷他,眼神堅定)我沒事。我不想再留在這裡等待了。我也想知道真相。

**李杰**：(看著她,最終點了點頭)好。那我們一起去。

**旁白**：兩人開始整理物資,準備出發。

**鈴木明日香**：(一邊收拾一邊小聲說)李先生,你說⋯⋯我們真的能回家嗎？

**李杰**：(沈默了片刻)我不知道。但我會盡全力去尋找。

**鈴木明日香**：(看著他)如果⋯⋯如果真的回不去了呢？

**李杰**：(看著窗外的紫月)那就想辦法在這裡活下去。

**鈴木明日香**：(低下頭,聲音微弱)在這裡活下去⋯⋯兩個人嗎？

**李杰**：(沒有正面回答,只是輕聲說)我們先找到答案再說吧。

**鈴木明日香**：(點了點頭)嗯。

**旁白**：兩人走出小屋,來到廣場上。莉拉已經在那裡等他們了。

**莉拉**：你們決定出發了嗎？

**李杰**：是的。我們想去神樹那裡。

**莉拉**：(點頭)我明白了。我會帶你們去。

**鈴木明日香**：(對莉拉鞠躬)莉拉小姐,謝謝妳這段時間的照顧。

**莉拉**：(微笑)別客氣。我們是同伴。

**旁白**：三人走出村莊,再次踏入那片被銀藍色光芒籠罩的森林。

**旁白**：走了一段路後,李杰停下腳步。

**李杰**：莉拉。

**莉拉**：(回頭)嗯？

**李杰**：我想請妳幫個忙。

**莉拉**：什麼事？

**李杰**：幫我停止搜尋。

**莉拉**：(驚訝)停止搜尋？為什麼？

**李杰**：(沈聲)搜尋已經沒有意義了。我們已經知道美咲和拓海他們遭到了襲擊。健太下落不明,如果他還活著,一定會來找我們。如果他⋯⋯

**李杰**：(嘆了口氣)繼續搜尋只會讓妳們的族人也陷入危險。那個黑劍男還在附近。

**莉拉**：(沈默了一會兒,點頭)我明白了。我會轉告族人的。

**鈴木明日香**：(擔心地看著李杰)李先生⋯⋯

**李杰**：(握住她的手)明日香,我們不能再讓更多人犧牲了。

**鈴木明日香**：(低下頭)我明白⋯⋯

**李杰**：我們會一直在一起。我會一直陪著妳。

**鈴木明日香**：一起⋯⋯對，一起。

**旁白**：她站起身，走到李杰面前。

**鈴木明日香**：至少⋯⋯我還有你。

**旁白**：李杰伸出手，輕輕握住她的手。

**李杰**：我會一直陪著妳。

**旁白**：窗外的紫月依然高懸，銀藍色的光芒灑滿村莊。在這片永恆的夜色中，兩個倖存者緊緊依靠著彼此。墓地裡的墳墓越來越多，每一座都是一個破碎的希望。但至少，他們還有彼此。在這個殘酷的世界裡，這也許就是唯一的慰藉。

**旁白**：搜尋停止了，但生存還在繼續。而在遠方的黑暗中，未知的危險依然潛伏著。等待著下一個獵物的到來。
