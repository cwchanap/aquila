# 第二十九幕：溫泉鄉的休憩

**旁白**：浩介帶領兩人來到這座位於半山腰的溫泉旅館。

**旁白**：旅館古色古香,白色的牆壁配上黑色的瓦頂,隱約能聽見遠處溪流的聲音。

**鈴木浩介**：這家旅館是我的一個朋友開的,很安靜,適合休息。

**李杰**：謝謝你,浩介。

**鈴木浩介**：(微笑)別客氣。

**旁白**：浩介轉向明日香。

**鈴木浩介**：姊,妳先跟這位女將去房間休息。李先生,我們在那邊的客廳談談？

**鈴木明日香**：(有些猶豫地看向李杰)

**李杰**：(點頭)去吧。我等一下就過去。

**鈴木明日香**：(小聲)嗯。

**旁白**：看著明日香跟著女將離開,浩介轉身帶著李杰走向大廳一側的茶室。

**旁白**：茶室很雅致,中間擺著一個矮几,兩側是柔軟的坐墊。

**鈴木浩介**：李先生,請坐。

**李杰**：謝謝。

**旁白**：兩人坐下後,浩介為李杰倒了一杯茶。

**鈴木浩介**：李先生,雖然我姊姊剛才說你們遇到了很多奇怪的事,但我還是想親耳聽你說一遍。

**鈴木浩介**：(認真地看著李杰)你們到底去了哪裡？

**旁白**：李杰接過茶杯,沈默了片刻。

**李杰**：我們去了一個⋯⋯我們無法理解的世界。

**旁白**：他開始斷斷續續地講述這段旅程。從那輛末班車開始,到那個充滿紫色月光的森林,再到銀月湖和最後的祭壇。

**旁白**：浩介靜靜地聽著,臉上的表情從驚訝到凝重,最後變成了深深的思索。

**鈴木浩介**：(長嘆一口氣)精靈⋯⋯異世界⋯⋯時間裂痕⋯⋯這聽起來真的很像科幻小說。

**鈴木浩介**：但如果你們真的是那輛失蹤列車的乘客,那就解釋得通了。

**李杰**：(驚訝)你知道那輛列車？

**鈴木浩介**：(點頭)嗯。這三十八年來,那輛列車一共出現過五次。

**鈴木浩介**：每次都會在新宿站的月台上消失,然後在幾年後又突然出現。

**鈴木浩介**：而每次出現,都會帶回一兩個像你們這樣的「歸還者」。

**李杰**：那其他人呢？那些沒回來的人呢？

**鈴木浩介**：(眼神黯淡)失蹤了。官方的說法是遭遇了嚴重的交通事故,但我們這些家屬都知道⋯⋯事情沒那麼簡單。

**李杰**：(沈思)所以,我們並不是唯一的受害者。

**鈴木浩介**：(點頭)是的。這就是為什麼政府會設立這個「靜養區」。專門為像你們這樣的人提供保護和治療。

**李杰**：(皺眉)保護？還是監視？

**鈴木浩介**：(苦笑)都有吧。畢竟你們帶回來的訊息,對於科學界來說太過驚人。

**鈴木浩介**：不過別擔心。我是這方面的負責人之一。我會保證你們的安全。

**李杰**：謝謝你。

**鈴木浩介**：(站起身)好了,不說這些沈重的話題了。

**鈴木浩介**：你一定累了。先去泡個溫泉,放鬆一下吧。

**鈴木浩介**：晚上我們再一起吃飯。

**李杰**：好。

**旁白**：李杰站起身,走出茶室。

**旁白**：走廊裡散發著淡淡的木香。他走向自己的房間,心中卻依然充滿了疑問。

**李杰**：(內心)那輛列車⋯⋯到底是誰在操控？神樹說的代價⋯⋯難道僅僅是時間嗎？

**旁白**：他在房間門口停下。對面房間的紙門緊閉著。

**李杰**：(輕聲)明日香⋯⋯妳還好嗎？

**旁白**：沒有回應。想必她是太累,已經睡著了。

**李杰**：(微微一笑)好好睡吧。

**旁白**：他推開自己的房門,走了進去。

---

**旁白**：士兵帶領兩人來到這座位於深山中的溫泉鄉。

**旁白**：周圍雲霧繚繞，空气中瀰漫著淡淡的硫磺味。

**士兵**：李杰先生，鈴木小姐。到了。這家旅館已經被我們包下來了，你們可以在這裡安心休息。

**李杰**：謝謝。

**鈴木明日香**：(有些緊張)這裡⋯⋯只有我們嗎？

**士兵**：(微笑)別擔心。還有一些搜救隊的同伴,以及專門為你們準備的醫療團隊。

**旁白**：三人走進旅館大廳。女將已經在門口等候。

**女將**：歡迎兩位。房間已經準備好了,請跟我來。

**旁白**：士兵和女將帶領兩人來到二樓的房間。

**士兵**：兩位請好好休息。如需協助，請按房內的呼叫鈴。晚餐時間是六點，會有人來通知。

**李杰**：知道了，謝謝。

**旁白**：士兵和女將鞠躬後離開，紙門輕輕關上。房間裡只剩下李杰和明日香。

**鈴木明日香**：(興奮地在房間裡走來走去)真不敢相信⋯⋯我們真的回來了。而且還能住溫泉旅館！一週後⋯⋯一週後我就能見到爸爸媽媽和弟弟了！

**旁白**：她的臉上洋溢著幸福的笑容。經歷了那麼多磨難，他們終於能在這樣寧靜的地方休息。

**旁白**：李杰站在窗邊，靜靜地看著外面的山景。溫暖的陽光，和煦的微風，還有遠處的溫泉蒸氣⋯⋯一切都如此平靜而美好。

**鈴木明日香**：(走到李杰身邊，關切地看著他)李先生？你在想什麼？看起來心事重重的⋯⋯

**李杰**：沒什麼。只是還不太習慣回到這裡。還有⋯⋯父親的事。

**鈴木明日香**：對不起⋯⋯

**旁白**：明日香輕輕握住李杰的手。

**鈴木明日香**：但至少⋯⋯我們平安回來了。伯母一定很想見你。

**李杰**：(看著明日香真誠的眼神，輕輕點了點頭)⋯⋯嗯，你說得對。

**旁白**：明日香靠在李杰的肩膀上，兩人靜靜地看著窗外的景色。

**旁白**：窗外，夕陽漸漸西沉。溫暖的橘色光芒灑在山巒上，將整片天空染成金黃色。遠處的溫泉蒸氣在夕陽下閃爍著淡淡的金光。一切都如此美麗而寧靜。在這個遠離塵囂的溫泉小鎮，他們終於找到了片刻的安寧。一週後，他們就能與家人團聚。回到真正屬於自己的生活。這段漫長而艱辛的旅程，終於迎來了結局。
